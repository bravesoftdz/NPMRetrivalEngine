           <p name="4e70" id="4e70" class="graf graf--p graf--leading"><strong class="markup--strong markup--p-strong">User Authentication using JWT (JSON Web Token) in Node.js (Express Framework )</strong></p>
           <p name="2c38" id="2c38" class="graf graf--p graf-after--p">Here we will learn how to implement user authentication using <strong class="markup--strong markup--p-strong">JWT</strong>(<strong class="markup--strong markup--p-strong">JSON Web Token</strong>) strategy. What we are going to use here:</p>

           <p name="e7e3" id="e7e3" class="graf graf--p graf-after--p">What is <strong class="markup--strong markup--p-strong"><em class="markup--em markup--p-em">JWT </em></strong><em class="markup--em markup--p-em">(</em><strong class="markup--strong markup--p-strong"><em class="markup--em markup--p-em">JSON Web Token</em></strong><em class="markup--em markup--p-em">)?<br></em>A JSON Web Token, or JWT, is used to send information that can be verified and trusted by means of a digital signature. It comprises a compact and URL-safe JSON object, which is cryptographically signed to verify its authenticity, and which can also be encrypted if the payload contains sensitive information. Because of it’s compact structure, JWT is usually used in HTTP Authorization headers or URL query parameters.</p>
           <p name="7a5c" id="7a5c" class="graf graf--p graf-after--p">Here we will learn how to implement token based user authentication.<br>1. Register new user.&nbsp;<br>2. One email address can be registered only once.<br>3. After registration a verification link is send to user’s email address.<br>4. Verify account by clicking on link send to user’semail address and on successful verification user will be redirected to login page.<br>5. If you are not verified your account after creation, you will not be allowed to land to dashboard after success login.<br>7. User can resend password to registered email address in case user forget.</p>
