         <p>Working with dates and times has always been a bit cumbersome. I’ve always thought that a JavaScript library for manipulating dates would be quite helpful. It was only recently that I was introduced to <a href="http://momentjs.com/">Moment.js</a>, the awesome JavaScript library for validating, parsing, and manipulating dates and times.</p> 
         <h2 id="getting-started-with-momentjs">Getting Started with Moment.js</h2> 
         <p>Moment.js is freely available for download from the <a href="http://momentjs.com/">project’s home page</a>. Moment.js can be run from the browser as well as from within a <a href="http://nodejs.org/">Node.js</a> application. In order to use it with Node, install the module using the following command.</p> 
         <pre><code class="language-">npm install moment</code></pre> 
         <p>Then, simply <code>require()</code> and use it in your application as shown below.</p> 
         <pre><code class="language-javascript">var moment = require('moment');

moment().format();</code></pre> 
         <p>In order to run Moment from the browser, download the script and include it using a <code>script</code> tag, as shown in the following example. Moment.js creates a global <code>moment</code> object which can be used to access all the date and time parsing and manipulation functionality.</p> 
         <pre><code class="language-markup">&lt;html&gt;
  &lt;head&gt;
    &lt;script src="moment.js"&gt;&lt;/script&gt;
    &lt;script&gt;
      moment().format();
    &lt;/script&gt;
  &lt;/head&gt;
  &lt;body&gt;
  &lt;/body&gt;
&lt;/html&gt;</code></pre> 
         <h2 id="date-formatting">Date Formatting</h2> 
         <p>In the past, I recall converting date strings into <code>Date</code> objects, grabbing individual pieces of data, and then performing string concatentations. Moment.js has simplified the process of date conversion to any particular format. Date format conversion with Moment is simple, as shown in the following example.</p> 
         <pre><code class="language-javascript">moment().format('YYYY MM DD');</code></pre> 
         <p><code>moment()</code> gives the current date and time, while <code>format()</code> converts the current date and time to the specified format. This example formats a date as a four digit year, followed by a space, followed by a two digit month, another space, and a two digit date. You can see this code in action by checking out this <a href="http://codepen.io/jay3dec/pen/qlhjy">demo</a>.</p> 
         <h2 id="date-validation">Date Validation</h2> 
         <p>Another annoying task that Moment.js has simplified is date validation. In order to perform validation, simply pass a date string to the <code>moment</code> object, along with the date format, and call the <code>isValid()</code> method. This method returns <code>true</code> if the date is valid, and <code>false</code> otherwise. An example of this is shown below, along with this accompanying <a href="http://codepen.io/jay3dec/pen/sKqeh/">demo</a>.</p> 
         <pre><code class="language-javascript">var dateEntered = $('#txtEnteredDate').val();

if (!moment(dateEntered,'MM-DD-YYYY').isValid()) {
  console.log('Invalid Date');
} else {
  console.log('Valid Date');
}</code></pre> 
         <p>There are a number of other helpful flags in the object returned by <code>moment()</code>:</p> 
         <ul>
          <li><strong>overflow</strong> – This is set when an overflow occurs. An example would be the 13th month or 32nd day.</li> 
          <li><strong>invalidMonth</strong> – Set when the month is invalid, like Jannnuaarry.</li> 
          <li><strong>empty</strong> – Set when the entered date contains nothing parsable.</li> 
          <li><strong>nullInput</strong> – Set when the entered date is <code>null</code>.</li> 
         </ul>
         <h2 id="manipulating-dates">Manipulating Dates</h2> 
         <p>There are a number of options for manipulating the <code>moment</code> object. For example, you can add or subtract days, months, years, etc. This is achieved via the <code>add()</code> and <code>subtract()</code> methods. The following example shows how seven days, months, or weeks are added to the current date.</p> 
         <pre><code class="language-javascript">moment().add('days', 7);    // adds 7 days to current date
moment().add('months', 7);  // adds 7 months to current date
moment().add('years', 7);   // adds 7 years to current date</code></pre> 
         <p>Similarly, the <code>subtract()</code> method is shown below.</p> 
         <pre><code class="language-javascript">moment().subtract('days', 7);   // subtracts 7 days to current date
moment().subtract('months', 7); // subtracts 7 months to current date
moment().subtract('years', 7);  // subtracts 7 years to current date</code></pre>

         <h2 id="time-from-now">Time From Now</h2> 
         <p>Another common task is determining how much time exists between two dates. For calculating time from the current date, Moment.js uses a method named <code>fromNow()</code>. Here is a sample which checks how much time exists from the current time:</p> 
         <pre><code class="language-javascript">moment().fromNow();</code></pre> 
         <p>This code sample displays “a few seconds ago.” If we supply a date to the <code>moment</code> object it would display the time range from now as per the difference. For example, the following code displays “7 days ago.”</p> 
         <pre><code class="language-javascript">var dateA = moment().subtract('days', 7);
dateA.fromNow();</code></pre> 
         <h3 id="time-from-another-date">Time From Another Date</h3> 
         <p><code>fromNow()</code> is used to compare time to the current date. This is just a special case of <code>from()</code>, which compares two arbitrary dates. An example that utilizes <code>from()</code> is shown below. This code displays “in a day.” You can see this code in action by checking out this <a href="http://codepen.io/jay3dec/pen/FgqAt/">demo</a>.</p> 
         <pre><code class="language-javascript">var dateB = moment('2014-12-12');
var dateC = moment('2014-12-11');

console.log(dateB.from(dateC));</code></pre> 
         <h2 id="calculating-the-difference-between-dates">Calculating the Difference Between Dates</h2> 
         <p>Moment.js offers a way to calculate the difference between two dates. The difference is calculated in milliseconds by default, but can also be returned in days, months, years, etc. To compute the difference, call the <code>diff()</code> method. This method takes a date as its first argument. The unit of time can be specified using the optional second argument. If this is not included, then milliseconds are used. The following example and <a href="http://codepen.io/jay3dec/pen/DAGtm">demo</a> illustrate how <code>diff()</code> is used.</p> 
         <pre><code class="language-javascript">var dateB = moment('2014-11-11');
var dateC = moment('2014-10-11');

console.log('Difference is ', dateB.diff(dateC), 'milliseconds');
console.log('Difference is ', dateB.diff(dateC, 'days'), 'days');
console.log('Difference is ', dateB.diff(dateC, 'months'), 'months');</code></pre> 
         <h2 id="date-queries">Date Queries</h2> 
         <p>Moment.js also provides date comparision methods. These methods are <code>isBefore()</code>, <code>isAfter()</code>, and <code>isSame()</code>. As the names imply, these methods return a Boolean indicating if one date is before, after, or equal to another date. An example that uses <code>isAfter()</code> is shown below.</p> 
         <pre><code class="language-javascript">console.log(moment('2010-09-20').isAfter('2010-10-19')); // returns false
console.log(moment('2010-11-20').isAfter('2010-10-19')); // returns true</code></pre> 
         <p>There is also an <code>isLeapYear()</code> method that checks for leap years.</p> 
         <h2 id="international-language-support">International Language Support</h2> 
         <p>Moment.js offers support for international language. You can assign a global language or set the language for a particular <code>moment</code> object. By default, Moment.js supports the English language. If you want to support any other language, then assign the key values of that particlar language to <code>moment.lang</code>. The following example, taken from the Moment.js docs, shows how support can be added for French.</p> 
         <pre><code class="language-javascript">moment.lang('fr', {
  months: 'janvier_février_mars_avril_mai_juin_juillet_août_septembre_octobre_novembre_décembre'.split('_'),
  monthsShort: 'janv._févr._mars_avr._mai_juin_juil._août_sept._oct._nov._déc.'.split('_'),
  weekdays: 'dimanche_lundi_mardi_mercredi_jeudi_vendredi_samedi'.split('_'),
  weekdaysShort: 'dim._lun._mar._mer._jeu._ven._sam.'.split('_'),
  weekdaysMin: 'Di_Lu_Ma_Me_Je_Ve_Sa'.split('_'),
  longDateFormat: {
    LT: 'HH:mm',
    L: 'DD/MM/YYYY',
    LL: 'D MMMM YYYY',
    LLL: 'D MMMM YYYY LT',
    LLLL: 'dddd D MMMM YYYY LT'
  },
  calendar: {
    sameDay: '[Aujourdhui à] LT',
    nextDay: '[Demain à] LT',
    nextWeek: 'dddd [à] LT',
    lastDay: '[Hier à] LT',
    lastWeek: 'dddd [dernier à] LT',
    sameElse: 'L'
  },
  relativeTime: {
    future: 'dans %s',
    past: 'il y a %s',
    s: 'quelques secondes',
    m: 'une minute',
    mm: '%d minutes',
    h: 'une heure',
    hh: '%d heures',
    d: 'un jour',
    dd: '%d jours',
    M: 'un mois',
    MM: '%d mois',
    y: 'une année',
    yy: '%d années'
  },
  ordinal: function (number) {
    return number + (number === 1 ? 'er' : 'ème');
  },
  week: {
    dow: 1, // Monday is the first day of the week.
    doy: 4 // The week that contains Jan 4th is the first week of the year.
  }
});

moment.lang('fr');
console.log(moment(1316116057189).fromNow()); // il y a une heure
moment.lang('en');
console.log(moment(1316116057189).fromNow()); // an hour ago</code></pre> 
         <p>Here is a <a href="http://codepen.io/jay3dec/pen/yACoc">demo</a> for the above code.</p> 
         <h2 id="conclusion">Conclusion</h2> 
         <p>Moment.js is really an awesome library that simplifies date and time related manipulations and validations. This library simplifies the job of application developers, which is almost always a good thing. In this article, we focused on some of the features of Moment.js which help in parsing, validating, and manipulating the datetimes in the browser and Node.js applications. A number of useful plugins are also available for Moment.js. Plugins like ISO Calendar, Jalaali Calendar, and many more can be found on the <a href="http://momentjs.com/docs/#/plugins/">official plugin page</a>. For more on Moment.js, the reader is directed to the library’s <a href="http://momentjs.com/docs/">documentation</a>.</p>
