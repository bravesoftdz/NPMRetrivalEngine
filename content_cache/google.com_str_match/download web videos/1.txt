         <blockquote> 
          <p>Video.js is a web video player built from the ground up for an HTML5 world. It supports HTML5 and Flash video, as well as YouTube and Vimeo (through <a href="http://videojs.com/plugins/">plugins</a>). It supports video playback on desktops and mobile devices. This project was started mid 2010, and the player is now used on over <del>50,000</del> <del>100,000</del> <del>200,000</del> <a href="https://trends.builtwith.com/media/VideoJS">400,000 websites</a>.</p>

           </svg></a>Quick Start</h2> 
         <p>Thanks to the awesome folks over at <a href="http://www.fastly.com/">Fastly</a>, there's a free, CDN hosted version of Video.js that anyone can use. Add these tags to your document's <code>&lt;head&gt;</code>:</p> 
         <div class="highlight highlight-text-html-basic">
          <pre>&lt;<span class="pl-ent">link</span> <span class="pl-e">href</span>=<span class="pl-s"><span class="pl-pds">"</span>//vjs.zencdn.net/5.19/video-js.min.css<span class="pl-pds">"</span></span> <span class="pl-e">rel</span>=<span class="pl-s"><span class="pl-pds">"</span>stylesheet<span class="pl-pds">"</span></span>&gt;
&lt;<span class="pl-ent">script</span> <span class="pl-e">src</span>=<span class="pl-s"><span class="pl-pds">"</span>//vjs.zencdn.net/5.19/video.min.js<span class="pl-pds">"</span></span>&gt;&lt;/<span class="pl-ent">script</span>&gt;</pre>
         </div> 
         <blockquote> 
          <p>For the latest version of video.js and URLs to use, check out the <a href="http://videojs.com/getting-started/">Getting Started</a> page on our website.</p> 
         </blockquote> 
         <p>Next, using Video.js is as simple as creating a <code>&lt;video&gt;</code> element, but with an additional <code>data-setup</code> attribute. At a minimum, this attribute must have a value of <code>'{}'</code>, but it can include any Video.js <a href="/videojs/video.js/blob/master/docs/guides/options.md">options</a> - just make sure it contains valid JSON!</p> 
         <div class="highlight highlight-text-html-basic">
          <pre>&lt;<span class="pl-ent">video</span>
    <span class="pl-e">id</span>=<span class="pl-s"><span class="pl-pds">"</span>my-player<span class="pl-pds">"</span></span>
    <span class="pl-e">class</span>=<span class="pl-s"><span class="pl-pds">"</span>video-js<span class="pl-pds">"</span></span>
    <span class="pl-e">controls</span>
    <span class="pl-e">preload</span>=<span class="pl-s"><span class="pl-pds">"</span>auto<span class="pl-pds">"</span></span>
    <span class="pl-e">poster</span>=<span class="pl-s"><span class="pl-pds">"</span>//vjs.zencdn.net/v/oceans.png<span class="pl-pds">"</span></span>
    <span class="pl-e">data-setup</span>=<span class="pl-s"><span class="pl-pds">'</span>{}<span class="pl-pds">'</span></span>&gt;
  &lt;<span class="pl-ent">source</span> <span class="pl-e">src</span>=<span class="pl-s"><span class="pl-pds">"</span>//vjs.zencdn.net/v/oceans.mp4<span class="pl-pds">"</span></span> <span class="pl-e">type</span>=<span class="pl-s"><span class="pl-pds">"</span>video/mp4<span class="pl-pds">"</span></span>&gt;&lt;/<span class="pl-ent">source</span>&gt;
  &lt;<span class="pl-ent">source</span> <span class="pl-e">src</span>=<span class="pl-s"><span class="pl-pds">"</span>//vjs.zencdn.net/v/oceans.webm<span class="pl-pds">"</span></span> <span class="pl-e">type</span>=<span class="pl-s"><span class="pl-pds">"</span>video/webm<span class="pl-pds">"</span></span>&gt;&lt;/<span class="pl-ent">source</span>&gt;
  &lt;<span class="pl-ent">source</span> <span class="pl-e">src</span>=<span class="pl-s"><span class="pl-pds">"</span>//vjs.zencdn.net/v/oceans.ogv<span class="pl-pds">"</span></span> <span class="pl-e">type</span>=<span class="pl-s"><span class="pl-pds">"</span>video/ogg<span class="pl-pds">"</span></span>&gt;&lt;/<span class="pl-ent">source</span>&gt;
  &lt;<span class="pl-ent">p</span> <span class="pl-e">class</span>=<span class="pl-s"><span class="pl-pds">"</span>vjs-no-js<span class="pl-pds">"</span></span>&gt;
    To view this video please enable JavaScript, and consider upgrading to a
    web browser that
    &lt;<span class="pl-ent">a</span> <span class="pl-e">href</span>=<span class="pl-s"><span class="pl-pds">"</span>http://videojs.com/html5-video-support/<span class="pl-pds">"</span></span> <span class="pl-e">target</span>=<span class="pl-s"><span class="pl-pds">"</span>_blank<span class="pl-pds">"</span></span>&gt;
      supports HTML5 video
    &lt;/<span class="pl-ent">a</span>&gt;
  &lt;/<span class="pl-ent">p</span>&gt;
&lt;/<span class="pl-ent">video</span>&gt;</pre>
         </div> 
         <p>When the page loads, Video.js will find this element and automatically setup a player in its place.</p> 
         <p>If you don't want to use automatic setup, you can leave off the <code>data-setup</code> attribute and initialize a <code>&lt;video&gt;</code> element manually using the <code>videojs</code> function:</p> 
         <div class="highlight highlight-source-js">
          <pre><span class="pl-k">var</span> player <span class="pl-k">=</span> <span class="pl-en">videojs</span>(<span class="pl-s"><span class="pl-pds">'</span>my-player<span class="pl-pds">'</span></span>);</pre>
         </div> 
         <p>The <code>videojs</code> function also accepts an <code>options</code> object and a callback to be invoked when the player is ready:</p> 
         <div class="highlight highlight-source-js">
          <pre><span class="pl-k">var</span> options <span class="pl-k">=</span> {};

<span class="pl-k">var</span> player <span class="pl-k">=</span> <span class="pl-en">videojs</span>(<span class="pl-s"><span class="pl-pds">'</span>my-player<span class="pl-pds">'</span></span>, options, <span class="pl-k">function</span> <span class="pl-en">onPlayerReady</span>() {
  <span class="pl-smi">videojs</span>.<span class="pl-en">log</span>(<span class="pl-s"><span class="pl-pds">'</span>Your player is ready!<span class="pl-pds">'</span></span>);

  <span class="pl-c"><span class="pl-c">//</span> In this context, `this` is the player that was created by Video.js.</span>
  <span class="pl-c1">this</span>.<span class="pl-c1">play</span>();

  <span class="pl-c"><span class="pl-c">//</span> How about an event listener?</span>
  <span class="pl-c1">this</span>.<span class="pl-en">on</span>(<span class="pl-s"><span class="pl-pds">'</span>ended<span class="pl-pds">'</span></span>, <span class="pl-k">function</span>() {
    <span class="pl-smi">videojs</span>.<span class="pl-en">log</span>(<span class="pl-s"><span class="pl-pds">'</span>Awww...over so soon?!<span class="pl-pds">'</span></span>);
  });
});</pre>
         </div>
