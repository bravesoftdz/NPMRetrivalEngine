         <h1 class="name post-title entry-title" itemprop="itemReviewed" itemscope itemtype="http://schema.org/Thing"><span itemprop="name">Quicksort algorithm in Java – Code Example</span></h1>
         <p class="post-meta"> <span class="post-meta-author">Posted by: <a href="https://examples.javacodegeeks.com/author/rohit-joshi/" title="">Rohit Joshi </a></span> <span class="post-cats">in <a href="https://examples.javacodegeeks.com/category/core-java/" rel="category tag">Core Java</a></span> <span class="tie-date">June 19th, 2014</span></p>
         <div class="clear"></div>
         <div class="entry">
          <div class="e3lan-post">
           <div id="div-gpt-ad-1415445260332-2" style="width:300px; height:250px;float:left; margin-right:20px;"> 
            <script type="text/javascript">googletag.cmd.push(function() { googletag.display('div-gpt-ad-1415445260332-2'); });</script>

          </div>
          <p>In this article, we will talk&nbsp;about how you can implement in Java one of the most famous and useful sorting techniques: Quicksort. Quicksort is the most widely used sorting algorithm. One of the main reasons is that it is faster than most common sorting algorithms. It was developed by the famous Computer Scientist <a href="http://en.wikipedia.org/wiki/Tony_Hoare">Tony Hoare</a>&nbsp;and it is based on the <a title="Divide_and_conquer" href="http://en.wikipedia.org/wiki/Divide_and_conquer_algorithms" target="_blank">“Divide and Conquer Algorithm”</a>.</p>
          <p>First, we are going to explain&nbsp;how Quicksort works on an algorithmic level, with some simple examples. Finally, we will build our implementation in Java and discuss its performance.</p>
          <p>&nbsp;</p>
          <h2>1. The Quicksort Alogrithm</h2>
          <p>Quicksort works recursively in order to sort a given array. These are the three basic steps of the&nbsp;Quicksort algorithm:</p>
          <p>1. Partition the array into a left sub-array and a right sub-array, in which the items in the left sub-array are smaller than the specified item and the items in the right sub-array are greater than the specified item.<br> 2. Recursively call the Quicksort to sort the left sub-array.<br> 3. Recursively call the Quicksort to sort the right sub-array.</p>
          <p>The partitioning step is the key, when sorting an array with Quicksort. Quicksort itself uses a Partition algorithm to partition the given array.</p>
          <p>The partition works by using two cursors (let say), one at each end of the array. These cursor moves towards each other. If the left cursor finds an item which is smaller than the pivot value, it ignores it and moves forward. But if, the item’s value is greater than the pivot value, it stops. Similarly for the right cursor, if an item is greater than the pivot value it ignores it and moves backwards, else the cursor stops. When both cursors stop, the items pointed by the cursors get swapped. That’s because these items are on the wrong side of the array. After the swap, both the cursors continue, and stop at the items which are on the wrong side of the array and swap them. And that’s how the algorithms goes on, until finally, all items are sorted.</p>
          <p>The pivot value is the value which is used to partition the array into two sub-arrays. After the partition, the items in the left sub-array are smaller than the pivot value, and the items in the right sub-array are greater than the pivot value.</p>
          <div id="attachment_10576" style="width: 773px" class="wp-caption aligncenter">
           <a href="http://examples.javacodegeeks.com/wp-content/uploads/2014/06/partition.jpg"><img class="size-full wp-image-10576" src="//examples.javacodegeeks.com/wp-content/uploads/2014/06/partition.jpg" alt="Partition" width="763" height="226" srcset="//examples.javacodegeeks.com/wp-content/uploads/2014/06/partition.jpg 763w, //examples.javacodegeeks.com/wp-content/uploads/2014/06/partition-300x88.jpg 300w" sizes="(max-width: 763px) 100vw, 763px"></a>
           <p class="wp-caption-text">Partition</p>
          </div>
          <p>In the above figure, we chose&nbsp;56 as a pivot value. After the partition (which of course consists of more than one sub steps), all the items on the left of the pivot are smaller, and the items on the right of it are greater than it and the pivot is at its sorted position. Also notice that, at this point, the array is not sorted of course. That was just one partitioning step.</p>
          <p>You can choose any random value from the array as a pivot value. Later we will see that choosing a pivot value affects the performance of the algorithm. But for now, lets us take the right most item of the array as a pivot value and attempt to create the first version of our Java implementation.</p>
          <p><span style="text-decoration: underline;"><em>QuicksortExample.java</em></span></p>
          <pre class="brush:java; wrap-lines:false">package com.javacodegeeks.sorting.quicksort;

public class QuicksortExample {

	private static int []a;
	public static void main(String[] args) {
		// Get a random generated array
		a = getArray();
		
		// prints the given array
		printArray();
		
		// sort the array
		sort();
		
		System.out.println("");
		
		//prints the sorted array
		printArray();
		
	}
	
	// This method sorts an array and internally calls quickSort 
	public static void sort(){
		int left = 0;
		int right = a.length-1;
			
		quickSort(left, right);
	}
	
	// This method is used to sort the array using quicksort algorithm.
	// It takes the left and the right end of the array as the two cursors.
	private static void quickSort(int left,int right){
		
		// If both cursor scanned the complete array quicksort exits
		if(left &gt;= right)
			return;
		
		// For the simplicity, we took the right most item of the array as a pivot 
		int pivot = a[right];
		int partition = partition(left, right, pivot);
		
		// Recursively, calls the quicksort with the different left and right parameters of the sub-array
		quickSort(0, partition-1);
		quickSort(partition+1, right);
	}
	
	// This method is used to partition the given array and returns the integer which points to the sorted pivot index
	private static int partition(int left,int right,int pivot){
		int leftCursor = left-1;
		int rightCursor = right;
		while(leftCursor &lt; rightCursor){
                while(a[++leftCursor] &lt; pivot);
                while(rightCursor &gt; 0 &amp;&amp; a[--rightCursor] &gt; pivot);
			if(leftCursor &gt;= rightCursor){
				break;
			}else{
				swap(leftCursor, rightCursor);
			}
		}
		swap(leftCursor, right);
		return leftCursor;
	}
	
	// This method is used to swap the values between the two given index
	public static void swap(int left,int right){
		int temp = a[left];
		a[left] = a[right];
		a[right] = temp;
	}
	
	public static void printArray(){
		for(int i : a){
			System.out.print(i+" ");
		}
	}
	
	public static int[] getArray(){
		int size=10;
		int []array = new int[size];
		int item = 0;
		for(int i=0;i&lt;size;i++){
			item = (int)(Math.random()*100); 
			array[i] = item;
		}
		return array;
	}

}

</pre>
          <ul>
           <li>If we run the above code, we will have the following results:</li>
          </ul>
          <pre class="brush:bash">4 94 87 24 44 30 37 97 47 93 
4 24 30 37 44 47 87 93 94 97 
</pre>
          <p>Let’s discuss how the above program works.</p>
          <p>The <code>quickSort()</code> method takes two parameters, each holding the position of the cursors at the two&nbsp;ends of an array or a sub-array that needs to be sorted. For example, if <code>left = 3</code>, then the left cursor points to element 3 of the array. The method exits, if <code>left</code> is greater than or equal to <code>right</code>, which means that the array is already sorted or the length of the array is one. It also generates a pivot value, in this case the right most value of the array. The pivot value is passed to the partition method which is used to partition the given array.</p>
          <p>The <code>partition()</code> method scans the array and swaps the items which are not at their proper place. The items which are greater than the pivot value, are swapped to the right of the pivot value with the values which are smaller than the pivot value. At the end of each scan, the left cursor ends up pointing to the left element of the right sub-array. The pivot is then swapped with it, and puts it into its proper sorted place. This method returns an integer that is the position of the sorted pivot value that partitioned the given array or a sub-array.</p>
          <p>Then, the <code>quicksort()</code> method spawns the recursive call to sort the left sub-array and the right sub-array. Let’s have a deeper look into the partition method.</p>
          <p><code>int leftCursor = left-1;</code> : This statement initializes a <code>leftCursor</code> to one less than the left parameter. This is because while scanning, it first gets incremented and then used to evaluate. For example, if we are scanning the complete array and not a sub-array, the <code>leftCursor</code> would be at <code>0-1, i.e., -1</code>.</p>
          <p><code>int rightCursor = right;</code> : This statement initializes a <code>rightCursor</code> to the right end of the given array, i.e., <code>rightCursor = array.lenght-1</code>.</p>
          <p><code>while(leftCursor &lt; rightCursor)</code>: The outer <code>while</code> loop runs till the <code>leftCursor</code> is not at same position or a position greater than the <code>rightCursor</code>. When this condition evaluates to false, it means that the cursors has scanned the complete array.</p>
          <p><code>while(a[++leftCursor] &lt; pivot);</code>: This inner <code>while</code> loop has nothing inside its body. It’s used to move the left cursor towards right and compare the item it’s pointing with the pivot. The loop terminates if the pointed value is greater than the pivot value.</p>
          <p><code>while(rightCursor &gt; 0 &amp;&amp; a[--rightCursor] &gt; pivot);</code>: This loop does a similar work. It moves towards the left of the array and compares each item it points with the pivot. If the pointed value is smaller than the pivot value, it terminates.</p>
          <p>When both inner <code>while</code> loops terminate, both the cursors point to the items which are not at their proper place. We first check whether the cursors have crossed each other, which means they have scanned the full array. Then, it exits from the loop, else the items get swapped.</p>
          <p>Then, the <code>quicksort()</code> method is called recursively. This time with the two sub-arrays, the left one starting from <code>partitioin-1</code>, and the right one starting from <code>partition+1</code>. It sorts the sub-arrays, until the full array gets partitioned and sorted, which finally results in the full sorted array.</p>
          <p>Generally, quicksort operates in O(n*log n) time. But there are some cases, when its performance degrades to O(n^2). The problem lies in the selection of the pivot. In the above example, we choose the pivot randomly (the right most item of the array). The pivot should be the median of the items to be sorted. So that, half of the items in the array should be smaller than the pivot, and the rest should be the larger than the pivot. This would result in two equally sized sub-arrays. Two equal sized sub-arrays is, the best situation for the Quicksort algorithm, where it runs at O(logn). Having one large and one small sub-array results in less efficiency.</p>
          <h2>2. Median of 3 Partitioning</h2>
          <p>The best approach to choose a pivot, is by choosing the median of the first, middle, and the last items of the array. This approach is known as the “median of three approach”.</p>
          <div id="attachment_10586" style="width: 727px" class="wp-caption aligncenter">
           <a href="http://examples.javacodegeeks.com/wp-content/uploads/2014/06/median.jpg"><img class="size-full wp-image-10586" src="//examples.javacodegeeks.com/wp-content/uploads/2014/06/median.jpg" alt="Median" width="717" height="156" srcset="//examples.javacodegeeks.com/wp-content/uploads/2014/06/median.jpg 717w, //examples.javacodegeeks.com/wp-content/uploads/2014/06/median-300x65.jpg 300w" sizes="(max-width: 717px) 100vw, 717px"></a>
           <p class="wp-caption-text">Median</p>
          </div>
          <p><span style="text-decoration: underline;"><em>QuicksortMedianExample.java</em></span></p>
          <pre class="brush:java; wrap-lines:false">package com.javacodegeeks.sorting.quicksort;

public class QuicksortMedianExample {

	private static int []a;
	public static void main(String[] args) {
		// Get a random generated array
		a = getArray();
		
		// prints the given array
		printArray();
		
		// sort the array
		sort();
		
		System.out.println("");
		
		//prints the sorted array
		printArray();
		
	}
	
	// This method sorts an array and internally calls quickSort 
	public static void sort(){
		int left = 0;
		int right = a.length-1;
			
		quickSort(left, right);
	}
	
	// This method is used to sort the array using quicksort algorithm.
	// It takes left and the right end of the array as two cursors
	private static void quickSort(int left,int right){
		
		// If both cursor scanned the complete array, quicksort exits
		if(left &gt;= right)
			return;
		
		// Pivot using median of 3 approach
		int pivot = getMedian(left, right);
		int partition = partition(left, right, pivot);
		
		// Recursively, calls the quicksort with the different left and right parameters of the sub-array
		quickSort(0, partition-1);
		quickSort(partition+1, right);
	}
	
	// This method is used to partition the given array and returns the integer which points to the sorted pivot index
	private static int partition(int left,int right,int pivot){
		int leftCursor = left-1;
		int rightCursor = right;
		while(leftCursor &lt; rightCursor){
        while(a[++leftCursor] &lt; pivot);
        while(rightCursor &gt; 0 &amp;&amp; a[--rightCursor] &gt; pivot);
			if(leftCursor &gt;= rightCursor){
				break;
			}else{
				swap(leftCursor, rightCursor);
			}
		}
		swap(leftCursor, right);
		return leftCursor;
	}
	
	public static int getMedian(int left,int right){
		int center = (left+right)/2;
		
		if(a[left] &gt; a[center])
			swap(left,center);
		
		if(a[left] &gt; a[right])
			swap(left, right);
		
		if(a[center] &gt; a[right])
			swap(center, right);
		
		swap(center, right);
		return a[right];
	}
	
	// This method is used to swap the values between the two given index
	public static void swap(int left,int right){
		int temp = a[left];
		a[left] = a[right];
		a[right] = temp;
	}
	
	public static void printArray(){
		for(int i : a){
			System.out.print(i+" ");
		}
	}
	
	public static int[] getArray(){
		int size=10;
		int []array = new int[size];
		int item = 0;
		for(int i=0;i&lt;size;i++){
			item = (int)(Math.random()*100); 
			array[i] = item;
		}
		return array;
	}

}

</pre>

4 12 14 25 30 31 33 35 65 80

          <p>In the above example, we have used the median-of-3-approrach to find a “good” pivot. We used the first, the middle, and last items of the array to find the median. The median is the middle item between the orderly placed items. This approach is not only used to select the pivot, but it also put the three items in their sorted place in the array. Let’s have a look at the <code>getMedian()</code> used in the above example.</p>
          <p><code>getMedian(int left, int right)</code>: This method is used to return a median among the three items specified. The returned median is used as the pivot in quicksort. This method has two parameters, both pointing at each end of the array or the sub-array. We used the middle, left and right items to find the median. At the end, we swapped the median with the item at the rightmost position of the array. So , after the scan all these three items should be in their proper sorted places in the array. This process is repeated with all the sub-arrays having different left, right and middle positions until the full array gets sorted.</p>
          <h2>3. Quicksort with String</h2>
          <p>So far, we have seen quicksort with integer arrays. In this example, we will sort an array of <a title="String" href="http://docs.oracle.com/javase/7/docs/api/java/lang/String.html" target="_blank"><code>Strings</code></a> using quicksort.</p>
          <p><span style="text-decoration: underline;"><em>QuicksortStringExample.java</em></span></p>
          <pre class="brush:java; wrap-lines:false">package com.javacodegeeks.sorting.quicksort;

public class QuicksortStringExample {

	private static String []a;
	public static void main(String[] args) {
		// Get an String array
		a = new String[]{"X","E","C","A"};
		
		// prints the given array
		printArray();
		
		// sort the array
		sort();
		
		System.out.println("");
		
		//prints the sorted array
		printArray();
		
	}
	
	// This method sort an array internally and internally calls quickSort 
	public static void sort(){
		int left = 0;
		int right = a.length-1;
			
		quickSort(left, right);
	}
	
	// This method is used to sort the array using quicksort algorithm.
	// It takes left and the right end of the array as two cursors
	private static void quickSort(int left,int right){
		
		// If both cursor scanned the complete array quicksort exits
		if(left &gt;= right)
			return;
		
		// Pivot using median of 3 approach
		String pivot = getMedian(left, right);
		int partition = partition(left, right, pivot);
		
		// Recursively, calls the quicksort with the different left and right parameters of the sub-array
		quickSort(0, partition-1);
		quickSort(partition+1, right);
	}
	
	// This method is used to partition the given array and returns the integer which points to the sorted pivot index
	private static int partition(int left,int right,String pivot){
		int leftCursor = left-1;
		int rightCursor = right;
		while(leftCursor &lt; rightCursor){
		while(((Comparable&lt;String&gt;)a[++leftCursor]).compareTo(pivot) &lt; 0);
        while(rightCursor &gt; 0 &amp;&amp; ((Comparable&lt;String&gt;)a[--rightCursor]).compareTo(pivot) &gt; 0);
			if(leftCursor &gt;= rightCursor){
				break;
			}else{
				swap(leftCursor, rightCursor);
			}
		}
		swap(leftCursor, right);
		return leftCursor;
	}
	
	public static String getMedian(int left,int right){
		int center = (left+right)/2;
		
		if(((Comparable&lt;String&gt;)a[left]).compareTo(a[center]) &gt; 0)
			swap(left,center);
		
		if(((Comparable&lt;String&gt;)a[left]).compareTo(a[right]) &gt; 0)
			swap(left, right);
		
		if(((Comparable&lt;String&gt;)a[center]).compareTo(a[right]) &gt; 0)
			swap(center, right);
		
		swap(center, right);
		return a[right];
	}
	
	// This method is used to swap the values between the two given index
	public static void swap(int left,int right){
		String temp = a[left];
		a[left] = a[right];
		a[right] = temp;
	}
	
	public static void printArray(){
		for(String i : a){
			System.out.print(i+" ");
		}
	}
	
}

          <p>In the above example, we have sorted an array of <code>Strings</code>, using the quicksort. The <code>String</code> class implements the <a title="Comparable" href="http://docs.oracle.com/javase/7/docs/api/java/lang/Comparable.html" target="_blank"><code>Comparable</code></a> interface and overrides the <a title="compareTo()" href="http://docs.oracle.com/javase/7/docs/api/java/lang/Comparable.html#compareTo(T)" target="_blank"><code>compareTo()</code></a> method. We have used the <code>compareTo()</code> method to compare the Strings. We have downcasted the String to the <code>Comparable</code> type and have used the <code>compareTo()</code> method to find the greater or the smaller between them.</p>
          <p>The comparison is done using the natural ordering of the <code>String</code>. The natural ordering in the <code>String</code> is maintained alphabetically from “A – Z” and then from “a – z”. The rest of the code works same as shown in the previous example.</p>
          <h2>4. Quicksort Objects</h2>
          <p>In this example, we will see how to sort Objects of a class using the Quicksort. We will create a generic quicksort method which can be used to sort objects of any class. The class needs to implement the <code>Comparable</code> interface and override the method <code>compareTo</code> in order to use the quicksort, otherwise, it will throw a <a title="ClassCastException" href="http://docs.oracle.com/javase/7/docs/api/java/lang/ClassCastException.html" target="_blank"><code>ClassCastException</code></a>.</p>
          <p>Let’s create an <code>Employee</code> class and sort employees on basis of their <code>employeeCode</code> using the Quicksort.</p>
          <p><span style="text-decoration: underline;"><em>Employee.java</em></span></p>
          <pre class="brush:java">package com.javacodegeeks.entity;

public class Employee implements Comparable&lt;Employee&gt;{

	private String firstName;
	private String lastName;
	private int emplyeeCode;
	
	public Employee(String fistName,String lastName,int emplyeeCode){
		this.firstName = fistName;
		this.lastName = lastName;
		this.emplyeeCode = emplyeeCode;
	}
	
	public String getFirstName() {
		return firstName;
	}
	public void setFirstName(String firstName) {
		this.firstName = firstName;
	}
	public String getLastName() {
		return lastName;
	}
	public void setLastName(String lastName) {
		this.lastName = lastName;
	}
	public int getEmplyeeCode() {
		return emplyeeCode;
	}

	public void setEmplyeeCode(int emplyeeCode) {
		this.emplyeeCode = emplyeeCode;
	}

	public String toString(){
		return "Employee Code: "+getEmplyeeCode()+", Name:"+getFirstName()+" "+getLastName();
	}

	public int compareTo(Employee o) {
		Employee e = (Employee)o;
		if(this.emplyeeCode &gt; e.getEmplyeeCode())
			return 1;
		if(this.emplyeeCode &lt; e.getEmplyeeCode())
			return -1;
		if(this.emplyeeCode == e.getEmplyeeCode())
			return 0;
		return 0;
	}
}

</pre>
          <p>We have created an <code>Employee</code> class which implements the <code>Comparable</code> interface and overrides the <code>compareTo()</code> method. The comparison between the <code>Employee</code> objects is defined by comparing the <code>employeeCode</code> property of the <code>Employee</code> objects. The <code>comparTo()</code> method returns an integer, which tells whether the current <code>employeeCode</code> is greater than, or smaller than or equal to the compared <code>employeeCode</code>. It returns 1, if the current <code>employeeCode</code> is greater than the compared <code>employeeCode</code>, -1 if, the current <code>employeeCode</code> is smaller than the compared <code>employeeCode</code>, else it returns 0 if both are equal. Since, the <code>employeeCode</code> is of type integer, we have compared it using the simple integer comparison operators.</p>
