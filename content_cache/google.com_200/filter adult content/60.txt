      <h1 id="artTitle" data-lang-alert="0" data-id="167">Avast Antivirus: Adjusting settings for Antivirus shields</h1> 
      <a name="articleContent"></a> 
      <p>How to adjust settings for Antivirus shields</p> 
      <p><b>Avast Antivirus</b> shields actively analyze suspicious information coming and going from your PC, as well as potentially malicious content stored within it. These components detect and block malicious files, dangerous websites, unauthorized connections, and other threats.</p> 
      <p>For information about the settings for each Antivirus shield, refer to the following sections:</p> 
      <a name="top_0"></a> 
      <ul type="arrow"> 
       <li><a href="#idt_10">Turn a shield on or off</a> </li> 
       <li><a href="#idt_20">Manage File system shield settings</a> </li> 
       <li><a href="#idt_30">Manage Behavior shield settings</a> </li> 
       <li><a href="#idt_40">Manage Mail shield settings</a> </li> 
       <li><a href="#idt_50">Manage Web shield settings</a> </li> 
      </ul> 
      <div class="note">
        Default settings are optimal for most systems, so we 
       <i>do not recommend</i> to make changes unless you find that there is a problem. 
      </div> 
      <a href="#top_0"></a> 
      <a name="idt_10"></a> 
      <p class="topic blue">Turn a shield on or off</p> 
      <p>To enable or disable a shield, follow these steps:</p> 
      <ol class="details"> 
       <li>Open the <a href="FAQ/AVKB16">Avast user interface</a> and go to 
        <ui>
         Settings
        </ui> ? 
        <ui>
         Components
        </ui>. </li> 
       <li>Click the 
        <ui>
         ON
        </ui>/
        <ui>
         OFF
        </ui> slider next to the relevant shield: 
        <ul> 
         <li>To disable a shield, click the green slider (
          <ui>
           ON
          </ui>) and select how long you want to disable the shield for (10 minutes, 1 hour, until you restart your PC, or permanently). </li> 
         <li>To enable a shield, click the red slider (
          <ui>
           OFF
          </ui>) so that it changes to green (
          <ui>
           ON
          </ui>). </li> 
        </ul> <a name="4arnaud"></a> </li> 
      </ol> 
      <p>You can also enable/disable all shields, 
       <ui>
        Firewall
       </ui>, and 
       <ui>
        Real Site
       </ui> at once from the system tray. Right-click the Avast system tray <img src="//static.avast.com/support/kb/images/v2017/icon/trayicon.gif"> icon and select 
       <ui>
        Avast shields control
       </ui>, then select an action from the context menu.</p> 
      <div class="note">
        We strongly recommend you keep all crucial security components, such as Antivirus shields and Firewall, enabled to ensure your maximum protection. We do not recommend turning off these components indefinitely by selecting the 
       <ui>
        Stop permanently
       </ui> or 
       <ui>
        Disable permanently
       </ui> options. 
      </div> 
      <a href="#top_0"></a> 
      <a name="idt_20"></a> 
      <p class="topic blue">File System Shield settings</p> 
      <p>
       <ui>
        File System Shield
       </ui> actively scans all programs and files on your hard drive as they are opened, run, modified, and saved. If malware is detected, File System Shield prevents the program or file from infecting your PC.</p> 
      <p>To modify your File System Shield settings, open the <a href="FAQ/AVKB16">Avast user interface</a>, go to 
       <ui>
        Settings
       </ui> ? 
       <ui>
        Components
       </ui>, and click 
       <ui>
        Customize
       </ui> next to 
       <ui>
        File System Shield
       </ui>. The following settings are available from the left panel:</p> 
      <ul> 
       <li>
        <ui>
         Scan when executing
        </ui>: specify whether you want File System Shield to scan programs, scripts, or libraries each time they are run. </li> 
       <li>
        <ui>
         Scan when opening
        </ui>: specify which file types you want File System Shield to scan each time the files are opened. </li> 
       <li>
        <ui>
         Scan when writing
        </ui>: specify the file types you want File System Shield to scan while the files are being created or modified. </li> 
       <li>
        <ui>
         Scan when attaching
        </ui>: specify the file types you want File System Shield to scan on removable media devices (such as USBs or DVDs). </li> 
       <li>
        <ui>
         Exclusions
        </ui>: specify files or folders that you want File System Shield to ignore. </li> 
       <li>
        <ui>
         Advanced
        </ui>: manage scan settings which affect your system performance and scan accuracy. </li> 
       <li>
        <ui>
         Actions
        </ui>: specify files or folders that you want File System Shield to ignore. </li> 
       <li>
        <ui>
         Packers
        </ui>: indicate the compressed file types that you want File System Shield to unpack when checking for malware. </li> 
       <li>
        <ui>
         Sensitivity
        </ui>: adjust the heuristics analysis for detecting unknown malware. </li> 
       <li>
        <ui>
         Report file
        </ui>: specify whether you want File System Shield to produce reports of scan results. </li> 
      </ul> 
      <a href="#top_0"></a> 
      <a name="idt_30"></a> 
      <p class="topic blue">Behavior Shield settings</p> 
      <p>
       <ui>
        Behavior Shield
       </ui> monitors programs installed on your PC for suspicious behavior that may indicate the presence of malicious code.</p> 
      <p>To modify your Behavior Shield settings, open the <a href="FAQ/AVKB16">Avast user interface</a>, go to 
       <ui>
        Settings
       </ui> ? 
       <ui>
        Components
       </ui>, and click 
       <ui>
        Customize
       </ui> next to 
       <ui>
        Behavior Shield
       </ui>. The following settings are available from the left panel:</p> 
      <ul> 
       <li>
        <ui>
         Main settings
        </ui>: specify which action to take if a threat is detected. </li> 
       <li>
        <ui>
         Exclusions
        </ui>: specify folders that you want Behavior Shield to exclude from scanning. </li> 
      </ul> 
      <a href="#top_0"></a> 
      <a name="idt_40"></a> 
      <p class="topic blue">Mail Shield settings</p> 
      <p>
       <ui>
        Mail Shield
       </ui> scans for threats in your incoming and outgoing email messages. Scanning applies only to messages sent or received using mail management software, such as Microsoft Outlook or Mozilla Thunderbird.</p> 
      <p>To modify your Mail Shield settings, open the <a href="FAQ/AVKB16">Avast user interface</a>, go to 
       <ui>
        Settings
       </ui> ? 
       <ui>
        Components
       </ui>, and click 
       <ui>
        Customize
       </ui> next to 
       <ui>
        Mail Shield
       </ui>. The following settings are available from the left panel:</p> 
      <ul> 
       <li>
        <ui>
         Main settings
        </ui>: specify which types of messages are scanned by Mail Shield  inbound, outbound, or newsgroup messages. We strongly recommend you enable scanning for all email types to ensure your full security. </li> 
       <li>
        <ui>
         Behavior
        </ui>: enable or disable Avast email signatures that inform both you and recipients of clean or infected emails. Additionally, you can manage what actions occur during scans, and configure options for Microsoft Outlook. </li> 
       <li>
        <ui>
         SSL scanning
        </ui>: enable or disable scanning of emails sent or received using SSL/TLS encrypted connection. For some mail clients, this feature requires you to export a certificate from the SSL Scanning screen then import the certificate into your mail client's certificate store. </li> 
       <li>
        <ui>
         Actions
        </ui>: configure automatic responses to detected threats. </li> 
       <li>
        <ui>
         Packers
        </ui>: indicate the compressed file types that you want Mail Shield to unpack when checking for malware. </li> 
       <li>
        <ui>
         Sensitivity
        </ui>: adjust the heuristics analysis for detecting unknown malware. </li> 
       <li>
        <ui>
         Report file
        </ui>: specify whether you want Mail Shield to produce reports of scan results. </li> 
      </ul> 
      <a href="#top_0"></a> 
      <a name="idt_50"></a> 
      <p class="topic blue">Web Shield settings</p> 
      <p>
       <ui>
        Web Shield
       </ui> actively scans data that is transferred when you browse the internet to prevent malware from being downloaded and run on your PC.</p> 
      <p>To modify your Web Shield settings, open the <a href="FAQ/AVKB16">Avast user interface</a>, go to 
       <ui>
        Settings
       </ui> ? 
       <ui>
        Components
       </ui>, and click 
       <ui>
        Customize
       </ui> next to 
       <ui>
        Web Shield
       </ui>. The following settings are available from the left panel:</p> 
      <ul> 
       <li>
        <ui>
         Main settings
        </ui>: manage use of web scanning, script scanning and HTTPS scanning, as well as other functions which can increase scan speed and accuracy. </li> 
       <li>
        <ui>
         Web scanning
        </ui>: specify which type of content is scanned. </li> 
       <li>
        <ui>
         Exclusions
        </ui>: specify URLs, MIME types (mail extension types), or processes to exclude from scanning. This is useful when you download many files from a single, trusted source. </li> 
       <li>
        <ui>
         Actions
        </ui>: configure automatic responses to online threats detected by Web Shield. </li> 
       <li>
        <ui>
         Packers
        </ui>: indicate the compressed file types that you want Web Shield to unpack when checking for malware. </li> 
       <li>
        <ui>
         Sensitivity
        </ui>: adjust the heuristics analysis for detecting unknown malware. </li> 
       <li>
        <ui>
         Site blocking
        </ui>: allows you to block access to specific websites. </li> 
       <li>
        <ui>
         Script scanning
        </ui>: select the browsers you want Web Shield to block malicious scripts from. This includes remote threats from the web and outside sources, and local threats saved to disk or in the browser cache. Script scanning can also detect and block malicious scripts that come from HTTPS (encrypted) connections. </li> 
       <li>
        <ui>
         Script exclusions
        </ui>: specify the addresses of websites you want to exclude from script scanning. You may want to exclude specific web addresses so you can access all content on those sites. Tick the box and then enter the URL address you want to exclude. Use exclusions only if you are sure the website you want to exclude from scanning is safe. </li> 
       <li>
        <ui>
         Report file
        </ui>: specify whether you want Web Shield to produce reports of scan results. </li> 
      </ul>
