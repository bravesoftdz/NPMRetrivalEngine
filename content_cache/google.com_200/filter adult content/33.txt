      <div id="features" class="panel panel-primary"> 
       <div class="panel-heading">
        OPNsense Solutions &amp; Features
       </div> 
       <div class="panel-body"> 
        <div class="row"> 
         <div class="col-md-6 col-xs-12"> 
          <div class="col-md-6 col-xs-6"> 
           <a href="/firewall" type="button" class="btn-group btn-group-justified"> <button class="btn btn-success" style="width:25%;"><span class="glyphicon glyphicon-fire"></span></button> <button class="btn btn-default" style="width:75%;">Stateful Firewall</button> </a> 
          </div> 
          <div class="col-md-6 col-xs-6"> 
           <small> A stateful firewall is a firewall that keeps track of the state of network connections (such as TCP streams, UDP communication) traveling across it. </small> 
           <hr> 
          </div> 
         </div> 
         <div class="col-md-6 col-xs-12"> 
          <div class="col-md-6 col-xs-6"> 
           <a href="/ha_carp_failover" type="button" class="btn-group btn-group-justified"> <button class="btn btn-success" style="width:25%;"><span class="glyphicon glyphicon-refresh"></span></button> <button class="btn btn-default" style="width:75%;">High Availability</button> </a> 
          </div> 
          <div class="col-md-6 col-xs-6"> 
           <small> Two or more firewalls can be configured as a failover group. If one interface fails on the primary or the primary goes offline entirely, the secondary becomes active. </small> 
           <hr> 
          </div> 
         </div> 
        </div> 
        <div class="row"> 
         <div class="col-md-6 col-xs-12"> 
          <div class="col-md-6 col-xs-6"> 
           <a href="/traffic_shaping" type="button" class="btn-group btn-group-justified"> <button class="btn btn-success" style="width:25%;"><span class="glyphicon glyphicon-filter"></span></button> <button class="btn btn-default" style="width:75%;">Traffic Shaper</button> </a> 
          </div> 
          <div class="col-md-6 col-xs-6"> 
           <small> Traffic shaping (also known as “packet shaping”) is the control of computer network traffic in order to optimise or guarantee performance, lower latency, and/or increase usable bandwidth by delaying packets that meet certain criteria. </small> 
           <hr> 
          </div> 
         </div> 
         <div class="col-md-6 col-xs-12"> 
          <div class="col-md-6 col-xs-6"> 
           <a href="/caching_proxy_webfilter" type="button" class="btn-group btn-group-justified"> <button class="btn btn-success" style="width:25%;"><span class="glyphicon glyphicon-flash"></span></button> <button class="btn btn-default" style="width:75%;">Caching Proxy</button> </a> 
          </div> 
          <div class="col-md-6 col-xs-6"> 
           <small> A caching proxy reduces bandwidth and improves response times by caching frequently-requested web pages. OPNsense also ads fine grained access control and category based webfiltering. </small> 
           <hr> 
          </div> 
         </div> 
        </div> 
        <div class="row"> 
         <div class="col-md-6 col-xs-12"> 
          <div class="col-md-6 col-xs-6"> 
           <a href="/captive_portal" type="button" class="btn-group btn-group-justified"> <button class="btn btn-success" style="width:25%;"><span class="glyphicon glyphicon-send"></span></button> <button class="btn btn-default" style="width:75%;">Captive Portal</button> </a> 
          </div> 
          <div class="col-md-6 col-xs-6"> 
           <small> A Captive Portal allows you to force authentication, or redirection to a click through page for network access. This is commonly used on hot spot networks, but is also widely used in corporate networks for an additional layer of security on wireless or Internet access. </small> 
           <hr> 
          </div> 
         </div> 
         <div class="col-md-6 col-xs-12"> 
          <div class="col-md-6 col-xs-6"> 
           <a href="/intrusion_detection" type="button" class="btn-group btn-group-justified"> <button class="btn btn-success" style="width:25%;"><span class="glyphicon glyphicon-warning-sign"></span></button> <button class="btn btn-default" style="width:75%;">Intrusion Prevention</button> </a> 
          </div> 
          <div class="col-md-6 col-xs-6"> 
           <small> The inline IPS system of OPNsense is based on Suricata and utilises Netmap to enhance performance and minimize cpu utilisation. This deep packet inspection system is very powerful and can be used to mitigate security threats at wire speed. </small> 
           <hr> 
          </div> 
         </div> 
        </div> 
        <div class="row"> 
         <div class="col-md-6 col-xs-12"> 
          <div class="col-md-6 col-xs-6"> 
           <a href="/virtual_private_networking" type="button" class="btn-group btn-group-justified"> <button class="btn btn-success" style="width:25%;"><span class="glyphicon glyphicon-globe"></span></button> <button class="btn btn-default" style="width:75%;">Virtual Private Network</button> </a> 
          </div> 
          <div class="col-md-6 col-xs-6"> 
           <small> A virtual private network (VPN) extends a private network across a public network, such as the Internet. It enables a computer to send and receive data across shared or public networks as if it is directly connected to the private network, while benefiting from the functionality, security and management policies of the private network. </small> 
           <hr> 
          </div> 
         </div> 
         <div class="col-md-6 col-xs-12"> 
          <div class="col-md-6 col-xs-6"> 
           <a href="/backup_restore" type="button" class="btn-group btn-group-justified"> <button class="btn btn-success" style="width:25%;"><span class="glyphicon glyphicon-saved"></span></button> <button class="btn btn-default" style="width:75%;">Backup &amp; Restore</button> </a> 
          </div> 
          <div class="col-md-6 col-xs-6"> 
           <small> Better safe than sorry, always keep an up to date backup of your configuration. It’s easy with OPNsense. Cloud backup supported with Google drive. </small>
