              <p>To begin, we shall use the standard, barebones, HTML structure for a webpage.</p>

Finally is the box for the results. There will be a text box similar to the user input, and text by it saying "USD amount in BTC".</p> 
              <h4>Javascript</h4> 
              <p>We need an equation to convert USD to BTC before we continue. As of the moment I am writing this, one USD equals 0.00078 bitcoin. So for an equation, we could use the input value times 0.00078.</p> 
              <p>To begin the javascript, we need the <code>&lt;script&gt;</code> tags. A function will be made to convert currencies as shown below.</p> 
              <pre><code>        &lt;script&gt;
        function toBTC() {
            var usd = document.getElementById("input").value;
            var btc = (0.00078 * usd);
            document.getElementById("btcValue").innerHTML = btc.toFixed(10);
