     <h2><span class="mw-headline" id="Bitcoin_clients">Bitcoin clients</span></h2> 
     <h3><span class="mw-headline" id="Bitcoin_clients_2">Bitcoin clients</span></h3> 
     <dl>
      <dd>
       <dl>
        <dd>
         <i>Main article and feature comparison: <a href="/wiki/Clients" title="Clients">Clients</a></i>
        </dd>
       </dl>
      </dd>
     </dl> 
     <ul>
      <li><a href="/wiki/Bitcoin_Core" title="Bitcoin Core">Bitcoin Core</a> - C++/Qt based tabbed UI. Linux/MacOSX/Windows. Full-featured <a href="/wiki/Thin_Client_Security" title="Thin Client Security">thick client</a> that downloads the entire <a href="/wiki/Block_chain" title="Block chain">block chain</a>, using code from the original Bitcoin client.</li> 
      <li><a href="/wiki/Bitcoind" title="Bitcoind">bitcoind</a> - GUI-less version of the original Bitcoin client, providing a <a href="/wiki/API_reference_(JSON-RPC)" title="API reference (JSON-RPC)">JSON-RPC</a> interface</li> 
      <li><a href="/wiki/MultiBit" title="MultiBit">MultiBit</a> - lightweight <a href="/wiki/Thin_Client_Security" title="Thin Client Security">thin client</a> for Windows, MacOS and Linux with support for opening multiple wallets simultaneously</li> 
      <li><a href="/wiki/Electrum" title="Electrum">Electrum</a> - a "blazing fast, open-source, multi-OS Bitcoin client/wallet with a very active community" - also a <a href="/wiki/Thin_Client_Security" title="Thin Client Security">thin client</a>.</li> 
      <li><a rel="nofollow" class="external text" href="https://bither.net/">Bither</a> - open source wallet for Android, iPhone and Desktop with cold storage support.</li> 
      <li><a href="/wiki/Bitcoin-js-remote" title="Bitcoin-js-remote">Bitcoin-js-remote</a> - JavaScript RPC client, support for QR codes</li> 
      <li><a rel="nofollow" class="external text" href="https://github.com/TheSeven/Bitcoin-WebUI">Bitcoin WebUI</a> - JavaScript RPC client</li> 
      <li><a rel="nofollow" class="external text" href="https://github.com/zamgo/bitcoin-webskin">Bitcoin Webskin</a> - PHP web interface to bitcoind</li> 
      <li><a href="/wiki/Bitcoin_Explorer" title="Bitcoin Explorer">Bitcoin Explorer</a> - Bitcoin command line tools for Linux, OSX and Windows, part of <a href="/wiki/Libbitcoin" title="Libbitcoin">Libbitcoin</a> SDK.</li> 
      <li><a href="/wiki/Bitcoiner" title="Bitcoiner">Bitcoiner</a> - Java RPC client (Android)</li> 
      <li><a href="/wiki/Armory" title="Armory">Armory</a> - Enterprise grade Python-based client</li> 
      <li><a href="/wiki/Spesmilo" title="Spesmilo">Spesmilo</a> - Python/PySide RPC client (abandoned)</li> 
      <li><a href="/wiki/Gocoin_bitcoin_software" title="Gocoin bitcoin software" class="mw-redirect">Gocoin</a> - WebUI client written in Go language, with a cold deterministic brain-wallet.</li> 
      <li><a rel="nofollow" class="external text" href="https://github.com/conformal/btcd">btcd</a> An alternative full node bitcoin implementation written in Go (golang).</li> 
      <li><a rel="nofollow" class="external text" href="http://www.blockcypher.com">BlockCypher</a> Full node bitcoin client built for scale and data centers, exposed through web APIs.</li> 
      <li><a href="/wiki/Mycelium" title="Mycelium">Mycelium</a> Awarded the prestigious "Best Mobile App" award by Blockchain.info in 2014, the Mycelium wallet for Android provides several security features. </li>
     </ul> 
     <h4><span class="mw-headline" id="Frontends_to_eWallet">Frontends to eWallet</span></h4> 
     <ul>
      <li><a rel="nofollow" class="external text" href="https://blockchain.info/wallet">Blockchain</a> - Javascript bitcoin client with client side encryption.</li> 
      <li><a rel="nofollow" class="external text" href="https://en.bitcoin.it/wiki/Xcoinmoney">xCoinMoney</a> Advanced API to create invoices for subscription.</li> 
      <li><a rel="nofollow" class="external text" href="https://cancoin.co/wallet">Cancoin</a> - HSM multi-sig wallet using libbitcoin. Client side encryption.</li>
     </ul>

     <ul>
      <li><a href="/wiki/Freecoin" title="Freecoin">Freecoin</a> - C++ client, supports alternative currencies like <a rel="nofollow" class="external text" href="https://bitcointalk.org/index.php?topic=9493.0">Beertoken</a></li> 
      <li><a href="/wiki/BitDroid" title="BitDroid">BitDroid</a> - Java client</li> 
      <li><a href="/wiki/Bitdollar" title="Bitdollar">Bitdollar</a> - C++/Qt client, unstable beta version</li>
     </ul> 
     <h2><span class="mw-headline" id="Bitcoin_software">Bitcoin software</span></h2> 
     <h3><span class="mw-headline" id="Live_operating_systems">Live operating systems</span></h3> 
     <p>A <a rel="nofollow" class="external text" href="https://en.wikipedia.org/wiki/Live_USB">live operating system</a> can start on almost any computer from a DVD, USB stick, or SD card, without installation. </p> 
     <ul>
      <li> <a href="/wiki/BitKey" title="BitKey">BitKey</a> - Live OS Bitcoin Swiss Army Knife, supports cold storage and air-gapped transactions</li> 
      <li> <a rel="nofollow" class="external text" href="https://tails.boum.org/">Tails</a> Privacy oriented Live OS, bundled with Tor and Electrum.</li>
     </ul> 
     <h3><span class="mw-headline" id="Exchange_Platform_Software">Exchange Platform Software</span></h3> 
     <ul>
      <li><a rel="nofollow" class="external text" href="https://www.alphapoint.com/">Alphapoint</a> - Bitcoin Exchange Software. Full system to run a digital currency exchange. Customize and launch your own digital currency and Bitcoin exchange in less than 20 days with AlphaPoint. Also supports automatic market-making on your exchange using 3rd party exchanges such as Bitfinex, BTCChina, and others. Supports many exchanges and smart routing, with automated account management.</li> 
      <li><a rel="nofollow" class="external text" href="http://www.merkeleon.at/ex%D1%81hange/">Merkeleon</a> - Bitcoin Exchange Software. A fully online software solution to launch a secure and reliable platform for crypto- and main world fiat currencies sale and purchase. Supports any currency integration, provides wide monetization opportunity for the owner.</li>
     </ul> 
     <h3><span class="mw-headline" id="Shopping_Cart_Integration_in_eCommerce-Systems">Shopping Cart Integration in eCommerce-Systems</span></h3> 
     <ul>

      <li><a href="/wiki/Zen_Cart_Bitcoin_Payment_Module" title="Zen Cart Bitcoin Payment Module">Zen Cart Bitcoin Payment Module</a> - a payment module that interacts with bitcoind for the Zen Cart eCommerce shopping chart.</li> 
      <li><a rel="nofollow" class="external text" href="https://coinbase.com/docs/merchant_tools/shopping_cart_plugins">Coinbase Shopping Cart Plugins</a> - Supports Wordpress, WooCommerce, Magento, Zencart, WP e-commerce, and more.</li> 
      <li><a href="/wiki/Karsha_Shopping_Cart_Interface" title="Karsha Shopping Cart Interface">Karsha Shopping Cart Interface</a> - is a mobile payment-interface which enables its users to accept payments.</li> 
      <li><a href="/wiki/Bitcoin-Cash" title="Bitcoin-Cash">Bitcoin-Cash</a> - an easy to use payment module for xt:Commerce</li> 
      <li><a href="/wiki/BitPay" title="BitPay">BitPay</a> - bitcoin plugins for Magento, Opencart, Zencart, PHP, JSON API</li> 
      <li><a href="/wiki/WalletBit" title="WalletBit">WalletBit</a> - Plugins for PrestaShop, OpenCart, PHP, JSON API</li> 
      <li> <a rel="nofollow" class="external text" href="https://www.xcoinmoney.com/info/api-general-info">xCoinMoney</a> Advanced API for instant payment and subscriptions</li> 
      <li><a href="/wiki/OpenCart_Bitcoin" title="OpenCart Bitcoin">OpenCart Bitcoin</a> - An OpenCart payment module that communicates with a bitcoin client using JSON RPC.</li> 
      <li> <a href="http://www.mycoinsolution.com" rel="nofollow"><img alt="MCS 200by200 logo-01.png" src="/w/images/en/thumb/6/64/MCS_200by200_logo-01.png/20px-MCS_200by200_logo-01.png" width="20" height="20" srcset="/w/images/en/thumb/6/64/MCS_200by200_logo-01.png/30px-MCS_200by200_logo-01.png 1.5x, /w/images/en/thumb/6/64/MCS_200by200_logo-01.png/40px-MCS_200by200_logo-01.png 2x"></a><a rel="nofollow" class="external text" href="http://www.mycoinsolution.com">My Coin Solution</a> - Bitcoin consulting services and solutions; custom payment integrations</li>

      <li> <a rel="nofollow" class="external text" href="http://drupal.org/project/uc_bitcoin">Drupal Ubercart Bitcoin payment method</a> enables you to accept Bitcoin as payment for your Drupal/Ubercart enabled website product/services.</li>

      <li><a rel="nofollow" class="external text" href="http://MY-BTC.info">MY-BTC.INFO</a> - A free profit/loss portfolio manager for Bitcoins and other digital currencies including many charts.</li> 
      <li><a rel="nofollow" class="external text" href="http://BitcoinExchangeRate.org">BitcoinExchangeRate.org</a> - Bitcoin and USD converter with convenient URL scheme and Auto-updating Portfolio Spreadsheet.</li> 
      <li><a href="/wiki/Bitcoin_Sentiment_Index" title="Bitcoin Sentiment Index">Bitcoin Sentiment Index</a> - A financial index that collects and disseminates sentiment data about bitcoin.</li> 
      <li><a href="/wiki/Preev" title="Preev">Preev</a> - Bitcoin converter with live exchange rates.</li> 
      <li><a href="/wiki/Skami" title="Skami">Skami</a> - Bitcoin Market Exchange comparison charts.</li> 
      <li><a href="/wiki/BitcoinSentiment" title="BitcoinSentiment">BitcoinSentiment</a> - Crowdvoting site offering means of voting and viewing voters sentiment towards bitcoin.</li> 
      <li><a href="/wiki/TradingView" title="TradingView">TradingView</a> – network where traders exchange ideas about Bitcoin using advanced free online charts</li>
     </ul>

     <ul>
      <li><a href="https://github.com/goethewins/EzBitcoin-Api-Wallet" rel="nofollow"><img alt="Easybitz.png" src="/w/images/en/thumb/c/c2/Easybitz.png/20px-Easybitz.png" width="20" height="20" srcset="/w/images/en/thumb/c/c2/Easybitz.png/30px-Easybitz.png 1.5x, /w/images/en/thumb/c/c2/Easybitz.png/40px-Easybitz.png 2x"></a> Simplest Web API for processing transactions with your own server. php code igniter, database and logging auth system included. Same as block chain.info api</li> 
      <li><a href="/w/index.php?title=BitMerch&amp;action=edit&amp;redlink=1" class="new" title="BitMerch (page does not exist)">BitMerch</a> - Embeddable HTML buttons, instant sign-up, instant payouts, automatic price adjustment for other currencies. No programming skills required to set up.</li> 
      <li><a href="/w/index.php?title=Bitcoin_Evolution&amp;action=edit&amp;redlink=1" class="new" title="Bitcoin Evolution (page does not exist)">Bitcoin Evolution</a> - Non wallet-based Buy Now button to insert into websites (handles sales tracking; client must be used for actual transaction)</li> 
      <li><a href="/wiki/BitPay" title="BitPay">BitPay</a> - Buy Now buttons, Checkout posts/callbacks, Mobile Checkout, JSON API</li> 
      <li><a href="/w/index.php?title=Btceconomy&amp;action=edit&amp;redlink=1" class="new" title="Btceconomy (page does not exist)">Btceconomy</a> - a JavaScript widget listing items for sale</li> 
      <li><a href="/wiki/BTCMerch" title="BTCMerch">BTCMerch</a> - Payment processor for bitcoins and other cryptocurrencies. 0.5% transaction fee. Sandbox is available.</li> 
      <li><a rel="nofollow" class="external text" href="https://coinbase.com/merchants">Coinbase</a> - Provides bitcoin payment processing for Overstock.com, Reddit, Khan Academy, OkCupid, and more.</li> 
      <li><a href="/wiki/GoCoin" title="GoCoin">GoCoin</a> - Payment gateway for bitcoin. Supports JavaScript, PHP, Java, Ruby, and .NET</li> 
      <li><a href="/wiki/Javascript_Bitcoin_Converter" title="Javascript Bitcoin Converter">Javascript Bitcoin Converter</a> - currency conversion</li> 
      <li><a href="/wiki/WalletBit" title="WalletBit">WalletBit</a> - Easy JavaScript Buy Now buttons, Instant Payment Notification, Application Programming Interface (JSON API), Mobile Checkout, QR-Code</li> 
      <li> <a rel="nofollow" class="external text" href="https://www.PikaPay.com">PikaPay</a> (<a href="/wiki/PikaPay" title="PikaPay">info</a>) The PikaPay service will be retiring as of March 1st, 2015</li>
