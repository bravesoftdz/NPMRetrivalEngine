   <h1>Vue.js CryptoCurrency Tutorial - Display Exchange Data with an API</h1> 
   <p id="postheaderauth">By Gary simon - Sep 25, 2017</p> 
  </div> 
  <div class="row" id="postcontent"> 
   <div class="columns large-9 small-12"> 
    <img src="https://s3.amazonaws.com/coursetro/posts/89-full.png" class="post-img-full"> 
    <div class="inner-content"> 
     <ul> 
      <li><a href="https://github.com/designcourse/vuejs-crypto">Download this project at Github</a></li> 
     </ul> 
     <p>As cryptocurrencies continue to grow in popularity, as a developer, you may be interested in building cryptocurrency apps. Fortunately, if you ever need access to the data surrounding the current cryptocurrencies, there exists a free API from&nbsp;<a href="https://www.cryptocompare.com/">CryptoCompare</a>&nbsp;that will provide you with everything you would probably ever need.</p> 
     <p>I'm going to show you how to get all set up and running within 2&nbsp;of the other most popular javascript frameworks:</p> 
     <ul> 
      <li><a href="/posts/code/90/React-CryptoCurrency-Tutorial---Display-Exchange-Data-with-an-API">React Cryptocurrency API Tutorial</a>&nbsp;</li> 
      <li><a href="/posts/code/91/Angular-CryptoCurrency-Tutorial---Display-Exchange-Data-with-an-API">Angular Cryptocurrency API Tutorial</a></li> 
     </ul> 
     <p>This way, just about all of the bases will be covered when it comes to getting up and running for your cryptocurrency project.&nbsp;</p> 
     <p>Let's get started!</p>

     <h2>Prerequisites</h2> 
     <p>Because we're using modern javascript frameworks, we'll need Node.js with NPM installed. You can check whether or not any of these are installed by opening up your command line or console and running:</p> 
     <pre class="language-markup"><code>&gt; node -v
&gt; npm -v</code></pre> 
     <p>If it provides you with a version number, you're set. If they don't, you will need to visit Nodejs.org and install the appropriate package based on your OS.</p> 
     <p>After installing, reload your console and rerun the same commands. This time they should give you a version number.</p> 
    </div>
    <div class="inner-content">
     <h2>Starting the Vue.js Project</h2> 
     <p>We're going to use the Vue CLI (Command Line Interface) to get us set up with a basic Vue project.&nbsp;</p> 
     <p>Open up your command line and run the following:</p> 
     <pre class="language-markup"><code>&gt; npm install vue-cli --global</code></pre> 
     <p>Next, let's use the CLI to start the new project:</p> 
     <pre class="language-markup"><code>&gt; vue init webpack vue-crypto</code></pre> 
     <p>This starts a new project named&nbsp;<strong>vue-crypto</strong>&nbsp;and it uses the webpack template, which is a build tool for handling assets.</p> 
     <p>Next, we'll hop into the new project folder and run the&nbsp;<strong>npm install</strong>&nbsp;command to install the dependencies in the package.json file that was generated from the cli:</p> 
     <pre class="language-markup"><code>&gt; cd vue-crypto
&gt; npm install</code></pre> 
     <p>After it's finished installing the node packages, run the following command to run the development server:</p> 
     <pre class="language-markup"><code>&gt; npm run dev</code></pre> 
     <p>This will launch the default starter template at&nbsp;<strong>http://localhost:8080</strong>&nbsp;and your browser should automatically launch with that address.</p> 
    </div>
    <div class="inner-content">
     <h2>Connecting to the Cryptocurrency API</h2> 
     <p>We need to install an HTTP client library called Axios. This will allow us to connect to the cryptocompare API. In the console run:</p> 
     <pre class="language-markup"><code>&gt; npm install axios --save</code></pre> 
     <p style="top: 1714.67px;">Within the project folder, navigate to the&nbsp;<strong>/src/components/Hello.vue</strong>&nbsp;file. This is where we'll place our code.</p> 
     <p style="top: 1714.67px;">First, within the&nbsp;<strong>&lt;script&gt;</strong>&nbsp;section, we must import&nbsp;<strong>axios</strong>:</p> 
     <pre class="language-javascript"><code>&lt;script&gt;
import axios from 'axios'

// Other stuff commented out</code></pre> 
     <p style="top: 1890px;">Next, under&nbsp;<strong>export default</strong>, we have to make some changes. The&nbsp;<strong>data</strong>&nbsp;attribute needs to contain a property that will hold our crypto data and any errors:</p> 
     <pre class="language-javascript"><code>&lt;script&gt;
import axios from 'axios'

export default {
  name: 'hello',
  data: () =&gt; ({
    cryptos: [],
    errors: []
  }),</code></pre> 
     <p style="top: 2135.33px;">Then, we need to call the&nbsp;<strong>created ()</strong>&nbsp;function, which fires when this component is loaded. This is where we will add our&nbsp;<strong>axios</strong>&nbsp;get call to the API.</p> 
     <pre class="language-markup"><code>&lt;script&gt;
import axios from 'axios'

export default {
  name: 'hello',
  data: () =&gt; ({
    cryptos: [],
    errors: []
  }),

  created () {
    axios.get('https://min-api.cryptocompare.com/data/pricemulti?fsyms=BTC,ETH,IOT&amp;tsyms=USD')
      .then(response =&gt; {
        this.cryptos = response.data
        console.log(response)         // This will give you access to the full object
      })
      .catch(e =&gt; {
        this.errors.push(e)
      })
  }
}
&lt;/script&gt;</code></pre> 
     <p style="top: 2508px;">The first line calls&nbsp;<strong>axios.get</strong>&nbsp;which accepts the API endpoint as the first argument.&nbsp;</p> 
     <p style="top: 2508px;">This is the point at which it makes sense to visit the&nbsp;<a href="https://www.cryptocompare.com/api/#-api-data-">CryptoCompare.com API Documentation</a>&nbsp;to discover all of the potential request endpoints you can designate. It includes everything from a massive coin list, price, history, mining, etc..</p> 
     <p style="top: 2508px;">For this tutorial, we're going to just simply display the current USD value associated with 3 specified currencies:</p> 
     <ul> 
      <li style="top: 2508px;">Bitcoin (BTC)</li> 
      <li style="top: 2508px;">Ethereum (ETH)</li> 
      <li style="top: 2508px;">IOTA (IOT)</li> 
     </ul> 
     <p>You can modify the URL parameter above to include others if you wish.</p> 
     <p>We then use a&nbsp;<strong>then</strong>&nbsp;method to set the value of&nbsp;<strong>this.cryptos</strong>&nbsp;to the&nbsp;<strong>response.data</strong>.</p> 
     <p>If you view the console in your browser, you will see the returned object with a number of properties. We only need to return the <strong>.data</strong>&nbsp;property, which is why we set <strong>this.cryptos</strong>&nbsp;to the <strong>response.data</strong>.</p> 
    </div>
    <div class="inner-content">
     <h2>Defining the Vue Template</h2> 
     <p>We're going to make this very, very simple. We will simply use the Vue&nbsp;<strong>v-for</strong>&nbsp;directive to iterate through our array of objects. This is what our template should look like at the top of <strong>Hello.vue</strong>:</p> 
     <pre class="language-markup"><code>&lt;template&gt;
  &lt;div class="hello"&gt;
    &lt;div id="crypto-container" v-for="(value, key) in cryptos"&gt;
      &lt;span class="left"&gt;{{ key }}&lt;/span&gt;
      &lt;span class="right"&gt;${{ value.USD }}&lt;/span&gt;
    &lt;/div&gt;
  &lt;/div&gt;
&lt;/template&gt;</code></pre> 
     <p>Next, let's define some quick CSS in the style section:</p> 
     <pre class="language-css"><code>&lt;style&gt;
  body {
    background: #f1f1f1;
  }

  div#crypto-container {
    background:white;
    width: 70%;
    margin: 0 auto 4px auto;
    padding: 1em;
    box-shadow: 1px 1px 0 lightgrey;
  }

  span.left {
    font-weight: bold;
  }

  span.right {
    float:right;
  }
&lt;/style&gt;</code></pre> 
    </div>
    <div class="inner-content">
     <h2>The Result</h2> 
     <p><img style="display: block; margin: 0 auto; text-align: center; max-width: 100%;" src="https://s3.amazonaws.com/coursetro/posts/content_images/DELTHISCRAP-1506386192050.png"></p> 
     <p style="top: 2508px;">I've added a few more currencies just to fill it out a bit more.&nbsp;</p> 
     <p style="top: 2508px;">I suggest experimenting with the CryptoCompare API Endpoints to discover the other information you can gather.&nbsp;</p> 
     <p style="top: 2508px;">Try changing the <strong>axios.get</strong>&nbsp;url to:</p> 
     <pre class="language-javascript"><code>axios.get('https://www.cryptocompare.com/api/data/coinlist/')</code></pre> 
     <p>View the console for the full object, which contains a massive list of coins along with their full names, logos, etc..</p> 
     <p>Hopefully you enjoyed this tutorial!</p>
