         <h1 class="title_front">SentiSight SDK</h1> 
         <h2 class="title_front">Object recognition for robotics and computer vision</h2> 
         <div id="product_header_mobile"> 
          <div id="product_header_mobile_skip_link"> 
           <span style="white-space: nowrap;">[<a href="#local_menu">Product menu</a>]</span> 
          </div> 
          <span style="white-space: nowrap;">[<a href="#nav_fodlable" onclick="sub_toggle('header_overview');">Show overview</a>]</span> 
         </div> 
         <div id="header_overview" class="page_head_content"> 
          <div class="page_head_column" style="width: 480px;"> 
           <p> SentiSight is intended for developers who want to use computer vision-based object recognition in their applications. Through manual or fully automatic object learning it enables searching for learned objects in images from almost any camera, webcam, still picture or live video in an easy, yet versatile, way. </p> 
           <p> Available as a software development kit that provides for the development of object recognition systems for Microsoft Windows or Linux platforms. </p> 
          </div> 
          <div class="page_head_column"> 
           <p class="page_head_download"> <a href="http://download.neurotechnology.com/SentiSight_SDK_Brochure_2013-08-27.pdf">Download <span style="font-weight: bold;">Brochure</span> (PDF)</a>. </p> 
           <p class="page_head_download"> <a href="/download.html#sentisight">Download <span style="font-weight: bold;">demo application</span></a>. </p> 
           <p class="page_head_download"> <a href="/download.html#sentisight">Download <span style="font-weight: bold;">30-day SDK Trial</span></a>. </p> 
          </div> 
         </div> 
         <div style="clear: both;"></div> 
        </div> 
       </div> 
      </div> 
      <div id="accented_separator_product"> 
      </div> 
      <div id="content_with_local_menu">
       <h1>Features and Capabilities</h1> 
       <ul class="features_list"> 
        <li>Reliable, innovative algorithm that is tolerant of variation in appearance, object scale, rotation and pose.</li> 
        <li>Accurate detection, processing and tracking of objects in real-time.</li> 
        <li>Webcams or other low cost cameras are suitable for obtaining object images.</li> 
        <li>Available as multiplatform SDK that supports multiple programming languages.</li> 
        <li>Reasonable prices, flexible licensing and free customer support.</li> 
       </ul> 
       <div class="inline_video"> 
        <iframe width="640" height="360" src="http://www.youtube-nocookie.com/embed/dqbhPcef7Do?rel=0&amp;wmode=transparent" frameborder="0" wmode="Opaque"></iframe> 
        <br> 
        <a href="http://youtu.be/dqbhPcef7Do" target="_blank">Watch on YouTube</a> or 
        <a href="download.html#sentisight_video">download</a>. 
       </div> 
       <p class="list_cap"> SentiSight SDK allows to develop applications and solutions for a <strong>wide range of tasks</strong>, including: </p> 
       <ul> 
        <li> Recognition of documents, stamps, labels, packaging and other items for sorting, logo masking, usage monitoring and similar applications </li> 
        <li> Object counting and inspection for assembly lines and other industrial applications </li> 
        <li> Augmented and extended reality applications for toys, games, devices and Web applications such as: smart toys for children that recognize cards, images, pictograms, etc.; recognition of places based on photographs; recognition of products such as beverages, foods and other consumer goods. </li> 
        <li> Robotic vision for navigation and manipulation </li> 
        <li> Law enforcement applications for identification, such as tattoo recognition </li> 
       </ul> 
       <p class="list_cap"> The SentiSight 3.4 technology is able to perform fully automatic or manual object learning and has these capabilities for advanced visual-based object learning and recognition: </p> 
       <ul> 
        <li> <strong>Accurate object detection.</strong> The SentiSight algorithm is able to find out: 
         <ul class="complex"> 
          <li><strong>whether</strong> a particular object is present in a scene;</li> 
          <li><strong>where</strong> the object is located within the scene;</li> 
          <li><strong>how many</strong> instances of the object occur in the scene.</li> 
         </ul> </li> 
        <li> <strong>Two algorithms for object recognition.</strong> Depending on the object type, one of these algorithms (or both) may be used for successful recognition: 
         <ul class="complex"> 
          <li> The <strong>blob</strong>-based algorithm uses small details of an object as distinctive features that are extracted into an object model and are used later to recognize the object. This algorithm offers high-speed performance but is not suitable for solid-colored, reflecting or transparent (glass, etc.) objects. </li> 
          <li> 
           <div class="float_right not_phone" style="font-size: 90%;">
             Shape recognition 
            <br> 
            <a href="res/sentisight-shape-recognition.jpg" title="SentiSight shape recognition screenshots"> <img src="res/sentisight-shape-recognition_t.gif" width="120" height="90" alt="SentiSight shape recognition screenshots thumbnail" border="0"> Click to zoom </a> 
           </div> The <strong>shape</strong>-based algorithm is useful for objects which do not have any distinctive details but have stable external edges (boundaries) and/or internal edges. This algorithm performs at slower speeds but allows for the recognition of most objects not identified by the blob-based algorithm. </li> 
         </ul> </li> 
        <li> <strong>Color usage mode.</strong> The blob-based algorithm may be configured to detect object colors and use this information for improving recognition accuracy. This mode enables SentiSight-based applications to distinguish similar objects that only differ in color. </li> 
        <li> <strong>Object's image quality determination.</strong> A quality threshold can be used during object learning to ensure that only the best quality object model will be stored into database. </li> 
        <li> <strong>Simultaneous multiple object recognition.</strong> The SentiSight algorithm provides simultaneous multiple 2D and 3D object detection and recognition. </li> 
        <li> <strong>Object evaluation.</strong> The algorithm makes estimates based on the region an object occupies in a scene, providing additional information about the size, orientation and scale of the recognized object. </li> 
        <li> <strong>Fast image processing.</strong> SentiSight processes video streams in real time, making it useful for real-time applications. The algorithm is able to run several threads on <strong>multi-core processors</strong> making the recognition several times faster. </li> 
        <li> <strong>Objects tracking mode.</strong> The SentiSight 3.4 library has a <strong>multiple-objects</strong> tracking mode for tasks that require very fast image processing during the object recognition stage. The tracking works with complex backgrounds and fast-moving objects. Tracking is initialized when an object is recognized and located, and then tracks the object until it changes somewhat in appearance, at which point tracking is reinitialized by recognition. In tracking-mode SentiSight is able to process <strong>more that 100 frames per second</strong> (320 x 240 pixels, single object in a frame). </li> 
       </ul>
