         <h1> <span class="caption"> Node.js integration </span> </h1> 
         <div class="docs_content"> 
          <a class="anchor" name="overview"></a>
          <h2><a href="#overview">Overview</a></h2> 
          <p> Cloudinary is a cloud-based service that provides an end-to-end image and video management solution including uploads, storage, manipulations, optimizations and delivery. </p> 
          <p> With Cloudinary you can easily upload your media assets to the cloud and automatically perform smart image and video manipulations without installing any complex software. All your media resources are then seamlessly delivered through a fast CDN, optimized and using industry best practices. Cloudinary offers comprehensive APIs and administration capabilities and is easy to integrate with new and existing web and mobile applications. </p> 
          <a class="anchor" name="quick_example"></a>
          <h3><a href="#quick_example">Quick example</a></h3> 
          <p>Take a look at the following Cloudinary URL that generates the image below:</p> 
          <div class="code_with_sample">
           <div class="code-holder">
            <code class="code_inner"><span class="url"><a href="https://res.cloudinary.com/demo/image/upload/w_150,h_150,c_thumb,g_face,r_20,e_sepia/l_cloudinary_icon,g_south_east,x_5,y_5,w_50,o_60,e_brightness:200/a_10/front_face.png" data-popup="true"><span class="prefix">https://res.cloudinary.com/demo/</span><span class="kind">image/upload/</span><span class="signature"></span><span class="trans">w_150,h_150,c_thumb,g_face,r_20,e_sepia/l_cloudinary_icon,g_south_east,x_5,y_5,w_50,o_60,e_brightness:200/a_10/</span><span class="public_id">front_face.png</span></a></span></code>
           </div>
           <div class="sample-img " style="width: 150px">
            <div class="img-hold">
             <a href="https://res.cloudinary.com/demo/image/upload/w_150,h_150,c_thumb,g_face,r_20,e_sepia/l_cloudinary_icon,g_south_east,x_5,y_5,w_50,o_60,e_brightness:200/a_10/front_face.png" data-popup="true"><img alt="" height="150" src="https://res.cloudinary.com/demo/image/upload/w_150,h_150,c_thumb,g_face,r_20,e_sepia/l_cloudinary_icon,g_south_east,x_5,y_5,w_50,o_60,e_brightness:200/a_10/front_face.png" width="150"></a>
            </div>
            <span class="desc">150x150 PNG</span>
           </div>
          </div> 
          <p>Simply accessing the above URL told Cloudinary to transform an uploaded image, create a 150x150px thumbnail using face detection based cropping, round the image's corners, add a sepia effect, convert it to a transparent PNG format, add a watermark layer, rotate the image by 10 degrees and ultimately delivered the resulting image through a fast CDN using smart caching techniques.</p> 
          <a class="anchor" name="node_js_library_features"></a>
          <h3><a href="#node_js_library_features">Node.js Library Features</a></h3> 
          <p> Cloudinary provides an <a href="https://github.com/cloudinary/cloudinary_npm" data-popup="true">open source Node module</a> for further simplifying the integration with your Node applications: </p> 
          <ul> 
           <li>Build URLs for transformation &amp; manipulation.</li> 
           <li>Node.js helper methods for embedding and transforming images and videos.</li> 
           <li>API wrappers: upload, administration, sprite generation and more.</li> 
           <li>Direct uploading from the browser using a jQuery plugin.</li> 
          </ul> 
          <a class="anchor" name="node_js_capitalization_and_data_type_guidelines"></a>
          <h3><a href="#node_js_capitalization_and_data_type_guidelines">Node.js capitalization and data type guidelines</a></h3> 
          <p> When using the Node.js SDK, keep these guidelines in mind: </p> 
          <ul> 
           <li>Parameter names: <code>snake_case</code>. For example: <strong>public_id</strong></li> 
           <li>Classes: <code>PascalCase</code>. For example: <strong>PreloadedFile</strong></li> 
           <li>Methods: <code>snake_case</code>. For example: <strong>image_upload_tag</strong></li> 
           <li>Pass parameter data as: <code>Object</code></li> 
          </ul> 
          <a class="anchor" name="node_js_getting_started_guide"></a>
          <h2><a href="#node_js_getting_started_guide">Node.js - Getting started guide</a></h2> 
          <a class="anchor" name="installation"></a>
          <h3><a href="/documentation/node_integration#installation"><span class="step">1</span>Installation</a></h3> 
          <p> Cloudinary's Node.js integration library is available as an open-source NPM. </p> 
          <p>To install the Cloudinary NPM, run:</p> 
          <div class="code-holder ">
           <div class="code_inner">
            <div class="CodeRay"> 
             <div class="code">
              <pre>npm install cloudinary</pre>
             </div> 
            </div> 
           </div>
          </div>
          <p>Include Cloudinary's Node.js classes in your code:</p> 
          <div class="code-holder ">
           <div class="code_inner">
            <div class="CodeRay"> 
             <div class="code">
              <pre><span class="keyword">var</span> cloudinary = require(<span class="string"><span class="delimiter">'</span><span class="content">cloudinary</span><span class="delimiter">'</span></span>);</pre>
             </div> 
            </div> 
           </div>
          </div>
          <ul class="sources"> 
           <li class="github"> <a href="https://github.com/cloudinary/cloudinary_npm" data-popup="true">https://github.com/cloudinary/cloudinary_npm</a> </li> 
           <li class="npm"> <a href="https://npmjs.org/package/cloudinary" data-popup="true">https://npmjs.org/package/cloudinary</a> </li> 
          </ul> 
          <a class="anchor" name="configuration"></a>
          <h3><a href="/documentation/node_integration#configuration"><span class="step">2</span>Configuration</a></h3> 
          <p> Your <strong>cloud_name</strong> account parameter is required to build URLs for your media assets. <strong>api_key</strong> and <strong>api_secret</strong> are further needed to perform secure API calls to Cloudinary (e.g., image and video uploads). See <a href="/documentation/solution_overview#account_and_api_setup">Account and API setup</a> for more details. </p> 
          <p>Setting the configuration parameters can be done either programmatically in each call to a Cloudinary method or globally using an environment variable or the <strong>config</strong> method.</p> 
          <p> You can find your configuration parameters in the dashboard of our <a href="https://cloudinary.com/console" data-popup="true">Management Console</a>. </p> 
          <p>Here's an example of setting configuration parameters globally in your Node application:</p> 
          <div class="code-holder ">
           <div class="code_inner">
            <div class="CodeRay"> 
             <div class="code">
              <pre>cloudinary.config({ 
  <span class="key">cloud_name</span>: <span class="string"><span class="delimiter">'</span><span class="content">sample</span><span class="delimiter">'</span></span>, 
  <span class="key">api_key</span>: <span class="string"><span class="delimiter">'</span><span class="content">874837483274837</span><span class="delimiter">'</span></span>, 
  <span class="key">api_secret</span>: <span class="string"><span class="delimiter">'</span><span class="content">a676b67565c6767a6767d6767f676fe1</span><span class="delimiter">'</span></span> 
});</pre>
             </div> 
            </div> 
           </div>
          </div>
          <p> <span class="hand-text"></span> See <a href="/documentation/node_additional_topics#configuration_options"><span class="translation_missing" title="translation missing: en.docs.node_additional_topics.sections.configuration_options">Configuration Options</span></a> for more details and additional configuration methods. </p> 
          <a class="anchor" name="upload_images_and_videos"></a>
          <h3><a href="/documentation/node_integration#upload_images_and_videos"><span class="step">3</span>Upload images and videos</a></h3> 
          <p> You can upload images, videos or <a href="https://cloudinary.com/blog/using_cloudinary_to_manage_all_your_website_s_assets_in_the_cloud" data-popup="true">any other files</a> from your Node.js server. Uploading is done over HTTPS using a secure protocol based on the api_key and api_secret parameters you provide. </p> 
          <p>The following command uploads a local image file to Cloudinary:</p> 
          <div class="code-holder ">
           <div class="code_inner">
            <div class="CodeRay"> 
             <div class="code">
              <pre>cloudinary.uploader.upload(<span class="string"><span class="delimiter">"</span><span class="content">my_picture.jpg</span><span class="delimiter">"</span></span>, <span class="keyword">function</span>(result) { 
  console.log(result) 
});</pre>
             </div> 
            </div> 
           </div>
          </div>
          <p>Alternatively, you can a specify a local path, a public HTTP URL, an S3 URL, an IO stream or an actual image file's data. For example:</p> 
          <div class="code-holder ">
           <div class="code_inner">
            <div class="CodeRay"> 
             <div class="code">
              <pre>cloudinary.uploader.upload(<span class="string"><span class="delimiter">"</span><span class="content">http://www.example.com/image.jpg</span><span class="delimiter">"</span></span>, <span class="keyword">function</span>(result) { 
  console.log(result) 
});</pre>
             </div> 
            </div> 
           </div>
          </div>
          <p>Each media asset uploaded to Cloudinary is assigned a unique <strong>Public ID</strong> and is available for immediate delivery and transformation. The upload method returns an associative array with content similar to that shown in the following example:</p> 
          <div class="code-holder ">
           <div class="code_inner">
            <div class="CodeRay"> 
             <div class="code">
              <pre>{ 
  <span class="key">public_id</span>: <span class="string"><span class="delimiter">'</span><span class="content">sample</span><span class="delimiter">'</span></span>,
  <span class="key">version</span>: <span class="integer">1312461204</span>,
  <span class="key">width</span>: <span class="integer">864</span>,
  <span class="key">height</span>: <span class="integer">576</span>,
  <span class="key">format</span>: <span class="string"><span class="delimiter">'</span><span class="content">jpg</span><span class="delimiter">'</span></span>,
  <span class="key">bytes</span>: <span class="integer">120253</span>,
  <span class="key">url</span>: <span class="string"><span class="delimiter">'</span><span class="content">https://res.cloudinary.com/demo/image/upload/v1371281596/sample.jpg</span><span class="delimiter">'</span></span>,
  <span class="key">secure_url</span>: <span class="string"><span class="delimiter">'</span><span class="content">https://res.cloudinary.com/demo/image/upload/v1371281596/sample.jpg</span><span class="delimiter">'</span></span> 
}</pre>
             </div> 
            </div> 
           </div>
          </div>
          <p>As you can see in the following example, with a single call you can define your own Public ID, apply an incoming transformation before storing the resource in the cloud, generate derived resources eagerly and assign tags to uploaded resources:</p> 
          <div class="code-holder ">
           <div class="code_inner">
            <div class="CodeRay"> 
             <div class="code">
              <pre>cloudinary.uploader.upload(
  req.files.myImage.path,
  <span class="keyword">function</span>(result) { console.log(result); },
  {
    <span class="key">public_id</span>: <span class="string"><span class="delimiter">'</span><span class="content">sample_id</span><span class="delimiter">'</span></span>, 
    <span class="key">crop</span>: <span class="string"><span class="delimiter">'</span><span class="content">limit</span><span class="delimiter">'</span></span>,
    <span class="key">width</span>: <span class="integer">2000</span>,
    <span class="key">height</span>: <span class="integer">2000</span>,
    <span class="key">eager</span>: [
      { <span class="key">width</span>: <span class="integer">200</span>, <span class="key">height</span>: <span class="integer">200</span>, <span class="key">crop</span>: <span class="string"><span class="delimiter">'</span><span class="content">thumb</span><span class="delimiter">'</span></span>, <span class="key">gravity</span>: <span class="string"><span class="delimiter">'</span><span class="content">face</span><span class="delimiter">'</span></span>,
        <span class="key">radius</span>: <span class="integer">20</span>, <span class="key">effect</span>: <span class="string"><span class="delimiter">'</span><span class="content">sepia</span><span class="delimiter">'</span></span> },
      { <span class="key">width</span>: <span class="integer">100</span>, <span class="key">height</span>: <span class="integer">150</span>, <span class="key">crop</span>: <span class="string"><span class="delimiter">'</span><span class="content">fit</span><span class="delimiter">'</span></span>, <span class="key">format</span>: <span class="string"><span class="delimiter">'</span><span class="content">png</span><span class="delimiter">'</span></span> }
    ],                                     
    <span class="key">tags</span>: [<span class="string"><span class="delimiter">'</span><span class="content">special</span><span class="delimiter">'</span></span>, <span class="string"><span class="delimiter">'</span><span class="content">for_homepage</span><span class="delimiter">'</span></span>]
  }      
)</pre>
             </div> 
            </div> 
           </div>
          </div>
          <p> <span class="hand-text">Many more upload options as well as direct uploading from the browser are detailed here: <a href="/documentation/node_image_upload">Node.js image upload</a> and <a href="/documentation/node_video_upload">Node.js video upload</a>.</span> </p> 
          <a class="anchor" name="display_and_manipulate_images_and_video"></a>
          <h3><a href="/documentation/node_integration#display_and_manipulate_images_and_video"><span class="step">4</span>Display and manipulate images and video</a></h3> 
          <p>Adding images and video to your Node.js application is quite straightforward. Just use the <strong>image</strong> or <strong>video</strong> method of the <strong>cloudinary</strong> class. This method generates the full source URL based on the given parameters and adds the image or video to your HTML code:</p> 
          <p>For example, displaying the uploaded image with the <strong>sample</strong> public ID, while providing an alternate text:</p> 
          <div class="code_with_sample">
           <div class="code-holder ">
            <div class="code_inner">
             <div class="CodeRay"> 
              <div class="code">
               <pre>cloudinary.image(<span class="string"><span class="delimiter">"</span><span class="content">sample.jpg</span><span class="delimiter">"</span></span>, { <span class="key">alt</span>: <span class="string"><span class="delimiter">"</span><span class="content">Sample Image</span><span class="delimiter">"</span></span> })</pre>
              </div> 
             </div> 
            </div>
           </div>
           <div class="sample-img " style="width: 250px">
            <div class="img-hold">
             <a href="https://res.cloudinary.com/demo/image/upload/w_250,h_168,c_fit/sample.jpg" data-popup="true"><img alt="" height="168" src="https://res.cloudinary.com/demo/image/upload/w_250,h_168,c_fit/sample.jpg" width="250"></a>
            </div>

          <div class="code_with_sample">
           <div class="code-holder ">
            <div class="code_inner">
             <div class="CodeRay"> 
              <div class="code">
               <pre>cloudinary.image(<span class="string"><span class="delimiter">"</span><span class="content">sample.jpg</span><span class="delimiter">"</span></span>, { <span class="key">width</span>: <span class="integer">100</span>, <span class="key">height</span>: <span class="integer">150</span>, <span class="key">crop</span>: <span class="string"><span class="delimiter">"</span><span class="content">fill</span><span class="delimiter">"</span></span> })</pre>
              </div> 
             </div> 
            </div>
           </div>

     <span class="attribute-name">width</span>=<span class="string"><span class="delimiter">'</span><span class="content">100</span><span class="delimiter">'</span></span> <span class="attribute-name">height</span>=<span class="string"><span class="delimiter">'</span><span class="content">150</span><span class="delimiter">'</span></span> <span class="tag">/&gt;</span></pre>
             </div> 
            </div> 
           </div>
          </div>
          <p>Using simple parameters you can perform powerful manipulations. The Node.js library builds Cloudinary URLs that you can embed in your web and mobile views for dynamically transforming your uploaded resources in the cloud and delivering the results through a <strong>fast CDN</strong> with advanced caching.</p> 
          <p>You can easily convert formats, resize, perform face detection-based image cropping, apply effects and filters, append textual layers or watermarks and more.</p> 
          <p>The following command, for example, embeds a JPG thumbnail of a profile photo fetched from Facebook in real-time, crops it to a circle, applies a sepia effect and delivers it optimized through a CDN:</p>

           <div class="code-holder ">
            <div class="code_inner">

               <pre>cloudinary.image(<span class="string"><span class="delimiter">"</span><span class="content">billclinton.jpg</span><span class="delimiter">"</span></span>, {
  <span class="key">type</span>: <span class="string"><span class="delimiter">'</span><span class="content">facebook</span><span class="delimiter">'</span></span>,
  <span class="key">width</span>: <span class="integer">90</span>, <span class="key">height</span>: <span class="integer">98</span>, 
  <span class="key">crop</span>: <span class="string"><span class="delimiter">'</span><span class="content">fill</span><span class="delimiter">'</span></span>, <span class="key">gravity</span>: <span class="string"><span class="delimiter">'</span><span class="content">face</span><span class="delimiter">'</span></span>,
  <span class="key">radius</span>: <span class="string"><span class="delimiter">'</span><span class="content">max</span><span class="delimiter">'</span></span>, <span class="key">effect</span>: <span class="string"><span class="delimiter">'</span><span class="content">sepia</span><span class="delimiter">'</span></span>
