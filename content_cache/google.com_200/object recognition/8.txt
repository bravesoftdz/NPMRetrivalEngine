     <h1> Code Libraries: Computer Vision </h1> 
     <article id="post-2902" class="post-2902 software type-software status-publish format-standard has-post-thumbnail hentry software_categories-computer-vision software_categories-video-processing"> 
      <a class="image" href="http://tangiblejs.com/software/webgazer-js" title="WebGazer.js"> <img src="http://tangiblejs.com/wp-content/uploads/2017/08/Sans-titre-1-1-512x288.jpg" class="attachment-512x288 size-512x288 wp-post-image" alt="" srcset="http://tangiblejs.com/wp-content/uploads/2017/08/Sans-titre-1-1-512x288.jpg 512w, http://tangiblejs.com/wp-content/uploads/2017/08/Sans-titre-1-1-256x144.jpg 256w, http://tangiblejs.com/wp-content/uploads/2017/08/Sans-titre-1-1-768x432.jpg 768w, http://tangiblejs.com/wp-content/uploads/2017/08/Sans-titre-1-1-1024x576.jpg 1024w, http://tangiblejs.com/wp-content/uploads/2017/08/Sans-titre-1-1-300x169.jpg 300w" sizes="(max-width: 512px) 100vw, 512px"> </a> 
      <h1> 
       <!-- Library --> <a href="https://webgazer.cs.brown.edu/" title="WebGazer.js"> WebGazer.js </a> 
       <!-- Regular post or page --> </h1> 
      <!-- Regular post --> 
      <div class="text"> 
       <!-- Regular post --> 
       <p>WebGazer.js is an eye tracking library that uses common webcams to infer the eye-gaze locations of web visitors on a page in real time. The eye tracking model it contains self-calibrates by watching web visitors interact with the web page and trains a mapping between the features of the eye and positions on the screen.</p> 
       <ul class="uris"> 
        <li class="uri"> <a href="https://webgazer.cs.brown.edu/"> https://webgazer.cs.brown.edu/ </a> </li> 
        <li class="uri"> <a href="https://github.com/brownhci/WebGazer"> https://github.com/brownhci/WebGazer </a> </li> 
       </ul> 
      </div> 
     </article> 
     <article id="post-976" class="post-976 software type-software status-publish format-standard hentry software_categories-computer-vision"> 
      <a class="image" href="http://tangiblejs.com/software/opencv" title="OpenCV"> <img src="http://tangiblejs.com/wp-content/themes/tangiblejs/img/default-library-image.jpg"> </a> 
      <h1> 
       <!-- Library --> <a href="https://www.npmjs.com/package/opencv" title="OpenCV"> OpenCV </a> 
       <!-- Regular post or page --> </h1> 
      <!-- Regular post --> 
      <div class="text"> 
       <!-- Regular post --> 
       <p>This library provides OpenCV bindings for Node.js applications. OpenCV is the de facto computer vision library. By interfacing with it natively in Node.js, we get powerful real time vision in JavaScript. For example, it can be used to fly quadcopters, detect faces from webcam images or annotate video streams.</p> 
       <ul class="uris"> 
        <li class="uri"> <a href="https://www.npmjs.com/package/opencv"> https://www.npmjs.com/package/opencv </a> </li> 
        <li class="uri"> <a href="https://github.com/peterbraden/node-opencv"> https://github.com/peterbraden/node-opencv </a> </li> 
       </ul> 
      </div> 
     </article> 
     <article id="post-642" class="post-642 software type-software status-publish format-standard has-post-thumbnail hentry software_categories-computer-vision"> 
      <a class="image" href="http://tangiblejs.com/software/tracking-js" title="tracking.js"> <img src="http://tangiblejs.com/wp-content/uploads/2015/05/trackingjs-512x288.jpg" class="attachment-512x288 size-512x288 wp-post-image" alt="" srcset="http://tangiblejs.com/wp-content/uploads/2015/05/trackingjs-512x288.jpg 512w, http://tangiblejs.com/wp-content/uploads/2015/05/trackingjs-256x144.jpg 256w, http://tangiblejs.com/wp-content/uploads/2015/05/trackingjs-1024x576.jpg 1024w, http://tangiblejs.com/wp-content/uploads/2015/05/trackingjs.jpg 1200w" sizes="(max-width: 512px) 100vw, 512px"> </a> 
      <h1> 
       <!-- Library --> <a href="http://trackingjs.com/" title="tracking.js"> tracking.js </a> 
       <!-- Regular post or page --> </h1> 
      <!-- Regular post --> 
      <div class="text"> 
       <!-- Regular post --> 
       <p>The tracking.js library brings various computer vision algorithm to JavaScript. It can perform color tracking, face detection, feature detection, and much more. It is well-documented and examples-rich.</p> 
       <ul class="uris"> 
        <li class="uri"> <a href="http://trackingjs.com/"> http://trackingjs.com/ </a> </li> 
        <li class="uri"> <a href="https://github.com/eduardolundgren/tracking.js/"> https://github.com/eduardolundgren/tracking.js/ </a> </li> 
       </ul> 
      </div> 
     </article> 
     <article id="post-627" class="post-627 software type-software status-publish format-standard has-post-thumbnail hentry software_categories-computer-vision"> 
      <a class="image" href="http://tangiblejs.com/software/headtrackr" title="headtrackr"> <img src="http://tangiblejs.com/wp-content/uploads/2015/05/headtrackr-512x288.jpg" class="attachment-512x288 size-512x288 wp-post-image" alt="" srcset="http://tangiblejs.com/wp-content/uploads/2015/05/headtrackr-512x288.jpg 512w, http://tangiblejs.com/wp-content/uploads/2015/05/headtrackr-256x144.jpg 256w, http://tangiblejs.com/wp-content/uploads/2015/05/headtrackr.jpg 1024w" sizes="(max-width: 512px) 100vw, 512px"> </a> 
      <h1> 
       <!-- Library --> <a href="https://github.com/auduno/headtrackr" title="headtrackr"> headtrackr </a> 
       <!-- Regular post or page --> </h1> 
      <!-- Regular post --> 
      <div class="text"> 
       <!-- Regular post --> 
       <p>headtrackr is a library for real-time face tracking and head tracking. It tracks the position of a user’s head in relation to the computer screen, via a web camera and the WebRTC/getUserMedia standard.</p> 
       <ul class="uris"> 
        <li class="uri"> <a href="https://github.com/auduno/headtrackr"> https://github.com/auduno/headtrackr </a> </li> 
       </ul> 
      </div> 
     </article> 
     <article id="post-531" class="post-531 software type-software status-publish format-standard has-post-thumbnail hentry tag-edge-detection tag-image-processing tag-library tag-object-detection tag-optical-flow-detection software_categories-computer-vision software_categories-image-processing"> 
      <a class="image" href="http://tangiblejs.com/software/jsfeat" title="jsfeat"> <img src="http://tangiblejs.com/wp-content/uploads/2015/04/Sans-titre-1-e1431627831813-512x288.png" class="attachment-512x288 size-512x288 wp-post-image" alt="" srcset="http://tangiblejs.com/wp-content/uploads/2015/04/Sans-titre-1-e1431627831813-512x288.png 512w, http://tangiblejs.com/wp-content/uploads/2015/04/Sans-titre-1-e1431627831813-256x144.png 256w, http://tangiblejs.com/wp-content/uploads/2015/04/Sans-titre-1-e1431627831813.png 640w" sizes="(max-width: 512px) 100vw, 512px"> </a> 
      <h1> 
       <!-- Library --> <a href="http://inspirit.github.io/jsfeat/" title="jsfeat"> jsfeat </a> 
       <!-- Regular post or page --> </h1> 
      <!-- Regular post --> 
      <div class="text"> 
       <!-- Regular post --> 
       <p>jsfeat is a computer vision library with a rich and varied feature set. It can perform tasks such as: edge detection, image processing (grayscale, blur, etc.), corner detection, object detection, optical flow detection, etc.</p> 
       <ul class="uris"> 
        <li class="uri"> <a href="http://inspirit.github.io/jsfeat/"> http://inspirit.github.io/jsfeat/ </a> </li> 
        <li class="uri"> <a href="https://github.com/inspirit/jsfeat"> https://github.com/inspirit/jsfeat </a> </li> 
       </ul> 
      </div> 
     </article> 
     <article id="post-515" class="post-515 software type-software status-publish format-standard has-post-thumbnail hentry tag-emotion-detection tag-face-detection tag-face-substitution software_categories-computer-vision software_categories-image-processing"> 
      <a class="image" href="http://tangiblejs.com/software/clmtrackr" title="clmtrackr"> <img src="http://tangiblejs.com/wp-content/uploads/2015/04/clmtrackr-512x288.jpg" class="attachment-512x288 size-512x288 wp-post-image" alt="" srcset="http://tangiblejs.com/wp-content/uploads/2015/04/clmtrackr-512x288.jpg 512w, http://tangiblejs.com/wp-content/uploads/2015/04/clmtrackr-256x144.jpg 256w, http://tangiblejs.com/wp-content/uploads/2015/04/clmtrackr.jpg 1024w" sizes="(max-width: 512px) 100vw, 512px"> </a> 
      <h1> 
       <!-- Library --> <a href="https://github.com/auduno/clmtrackr" title="clmtrackr"> clmtrackr </a> 
       <!-- Regular post or page --> </h1> 
      <!-- Regular post --> 
      <div class="text"> 
       <!-- Regular post --> 
       <p>clmtrackr is a JavaScript library for fitting facial models to faces in videos or images. It can track a face and output the coordinate positions of the face model as an array. It can also perform face substitution, face masking, realtime face deformation, emotion detection and more…</p> 
       <ul class="uris"> 
        <li class="uri"> <a href="https://github.com/auduno/clmtrackr"> https://github.com/auduno/clmtrackr </a> </li> 
       </ul> 
      </div>
