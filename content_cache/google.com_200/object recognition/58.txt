        <h1 class="entry-title" itemprop="headline">Object Detection on Intel Architecture Turns Images into Meaningful Information</h1> 
        <div class="entry-meta"> 
         <span class="byline"><span class="author vcard" itemprop="author name" itemscope itemtype="http://schema.org/Person"><span class="screen-reader-text">Author </span><a class="url fn n" href="https://itpeernetwork.intel.com/author/rpanchum/" rel="author" itemprop="url">Ravi Panchumarthy</a></span></span>
         <span class="posted-on"><span class="screen-reader-text">Published on </span><a href="https://itpeernetwork.intel.com/object-detection-intel-architecture-images-meaningful-information/" rel="bookmark"><time class="entry-date published" datetime="2017-04-28T11:00:55+00:00" itemprop="datePublished">April 28, 2017</time><time class="updated" datetime="2017-04-28T09:46:38+00:00">April 28, 2017</time></a></span> 
        </div>
        <!-- .entry-meta --> 
       </header>
       <!-- .entry-header --> 
       <div class="entry-content" itemprop="articleBody"> 
        <p>Across a growing range of industries, theres a need to analyze large amounts of data gathered from various end points to obtain meaningful insights. Artificial intelligence (AI) enables machines to perform tasks with human-like intelligence by interpreting that input data, and deep learning algorithms are able to accelerate these AI capabilities in image, video, audio and text analysis.</p> 
        <p>For example, deep learning can classify objects from a live video stream using Intels optimized Caffe* framework, and the Single Shot MultiBox Detector deep learning architecture can detect objects in images. The Single Shot MultiBox Detector uses a single deep neural network with the ability to pool from different scales of the feature maps. As a result, the overall algorithm can detect objects at various scales and sizes with more accuracy.</p> 
        <p>In other words, image classification can tell you that a particular image or video stream contains a dog or a cat, and object detection algorithms can identify exactly where the dog or cat is located in the image.</p> 
        <p>Here are a few examples of how object detection can be used across different industries:</p> 
        <ul> 
         <li>In security and surveillance applications, object detection can identify objects people are carrying that could be used as a weapon.</li> 
         <li>In autonomous vehicles, object detection software can recognize a person crossing the street, an animal in the path of the car, or a nearby stop sign. It can also differentiate between a pebble and a rock in a moments notice, which is crucial when speed is a factor and accuracy matters.</li> 
         <li>In manufacturing, object detection software can identify defective or incorrect parts on an assembly line.</li> 
        </ul> 
        <p>At a technical level, the diverse use cases for live object detection share a common need: processing power, and lots of it. In a deep learning context, training and inferencing processes are computationally intensive tasks, and the power of the Intel® Xeon® processor E5 family provides this level of processing power.</p> 
        <p>Intel is focused on providing optimized hardware with optimized libraries and frameworks to drive ever-higher performance for AI applications running on the Intel Xeon processor E5 family. Object detection capabilities are incorporated in the <a href="https://itpeernetwork.intel.com/optimizing-deep-learning-sdk/">Intel® Deep Learning SDK</a>, which is a free package of tools that developers and data scientists can use to accelerate the development of deep learning solutions.</p> 
        <p>Object detection software also gets a boost from <a href="https://software.intel.com/en-us/intel-mkl" target="_blank">Intel® Math Kernel Library</a>, which accelerates math processing and neural network routines that increase application performance and reduce development time. With its highly optimized, threaded and vectorized functions, Intel MKL maximizes performance on each processor family.</p> 
        <p>Intel also offers the <a href="https://github.com/intel/caffe" target="_blank">Intel® Distribution for Caffe*</a>, a deep learning framework developed by Berkeley Vision and Learning Center (BVLC). Intel has contributed to an optimized fork dedicated to improving Caffe* performance when running on CPUs, in particular Intel Xeon processor-based servers. The Intel Distribution for Caffe* supports <a href="https://github.com/intel/caffe/wiki/SSD:-Single-Shot-MultiBox-Detector" target="_blank">Single Shot MultiBox Detector</a> to detect objects in images using a single deep neural network.</p> 
        <p>The combination of optimized Intel Xeon processor families and optimized libraries and frameworks yields higher performance for object detection applications running on Intel® architecture. We demonstrated this application at the International Conference on Learning Representations (ICLR) in Toulon, France this week. The interactive demo showed how an AI application detects objects in a live webcam video stream.</p> 
        <p>Ready to jump into the sandbox? You can download the Intel Deep Learning SDK to try it out on <a href="http://www.intel.com/ai" target="_blank">intel.com/ai </a>or <a href="https://github.com/intel/caffe/wiki/SSD:-Single-Shot-MultiBox-Detector" target="_blank">GitHub</a>. Or, for a demo of the object recognition software capabilities, similar to the demo shown at ICLR, grab a seat for this <a href="https://www.youtube.com/watch?v=RoUNDyJkgeQ" target="_blank">YouTube video</a> from IDG.tv at the South by Southwest Conference earlier this year.</p> 
       </div>
