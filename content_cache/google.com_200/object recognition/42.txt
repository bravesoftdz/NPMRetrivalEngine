                <td id="IprogrammerMainArticleTextBody" valign="top"> <p>The only real problem with tracking.js is that potential users might ignore it, thinking it has something to do with marketing. It doesn't. It is an open source computer vision library in JavaScript and it makes it fairly easy to add sophisticated image processing into your web apps.</p> <p>&nbsp;</p> <p><img src="/images/stories/News/2014/Jul/B/trackingjs.jpg" alt="trackingjs" style="display: block; margin-left: auto; margin-right: auto;" height="64" width="115"></p> <p>&nbsp;</p> <p>The library is called tracker.js because it majors on visual tracking algorithms. It has just reached Version 1 and it seems to be a project with a lot of enthusiastic participants.&nbsp;</p> <p>The range of algorithms that it offers is impressive. It is also easy to use. Simply load the core library and any additional algorithm libraries you need. Next you need to set up a video source and this can be as simple as an HTML 5 video tag, Canvas tag, img tag or a full getUserMedia call if you want to use a built in video camera. &nbsp;Next you hook up the algorithm to the video source and set a call back for any detection event. The ease of hooking up the video source is one reason to use the library but the simplicity of just running a textbook algorithm by creating and object and calling a method is very difficult to beat.</p> <p>The entire library is also available as a set of web components built using Polymer on top of the basic tracking.js.</p> <p>The trackers available go from the very simple, in principle, color tracker which tracks a specified color to an object tracker which will track faces, eyes a mouths. For example, to track all three you would need:</p> <p><code>var objects = new tracking.ObjectTracker(<br>&nbsp; &nbsp; ['face', 'eye', 'mouth']);<br><br></code><code>objects.on('track',function(event) { <br>&nbsp; if (event.data.length === 0) { <br>&nbsp; &nbsp; // No objects were detected in this frame.<br>&nbsp; }&nbsp;else { <br>&nbsp; &nbsp;event.data.forEach(function(rect) { <br>&nbsp; &nbsp;//draw rect.x, rect.y, rect.height, rect.width <br>&nbsp; });<br>&nbsp;}</code></p> <p><code>});</code></p> <p><br>and to make it start working:</p> <p><code>tracking.track('#myVideo', objects);<br></code></p> <p><img src="/images/stories/News/2014/Jul/B/trackjsfaces.jpg" alt="trackjsfaces" height="307" width="400"></p> <p>&nbsp;</p> <p>You can extend the object tracker by adding your own training data. You can also easily extend the library with your own tracker. This is also made easy in that your custom code is passed to the pixel array from the video source to work with.&nbsp;</p> <p>In addition to the trackers, there is also a set of utilities that can be used in creative ways. There are two feature detectors - one called fast which finds corners and on called Brief which uses binary features. There is also the Viola Jones object detection framework, which is used in the object tracker. You can use it to track an arbitrary object in an image simply by providing it with the pixel data for that object.&nbsp;</p> <p>You also have access to a general convolution filter, image blur, Sobel, convert to grayscale and integrate a region operations.&nbsp;</p> <p>There are lots of examples of the library in use and many use the live feed from any built in video camera your computer might have. Notice, however, that the library doesn't do face recognition only detection.&nbsp;</p> <p>The bottom line is that this is one of the easiest computer vision packages to use and you can start doing clever and impressive things with it, even if you don't know much about the algorithms involved. What is more, you don't have to write much code to get things started. If you do know something about the algorithms then you can implement new and custom trackers very easily.&nbsp;</p> <p>If you have any interest in computer vision in web apps - this is one library worth checking out.&nbsp;</p> <p>&nbsp;</p> <p><img src="/images/stories/News/2014/Jul/B/trackingicon.jpg" alt="trackingicon" style="display: block; margin-left: auto; margin-right: auto;" height="100" width="115"></p>
