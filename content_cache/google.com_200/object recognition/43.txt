                <h1 class="topic-post-title"> Which software to use for object recognition in robotic vision? </h1> 
                <div class="topic-post-text"> 
                 <p>I'm a student with electrical/mechanical background, in my project I'm searching for a solution for a company who wants to start with 3D cameras for robotics.<br>At the moment I'm working with Matlab and it works great, the possibility to create your own GUI is a big plus.<br>But I read Matlab is more for developing purpose and is slower (overhead).</p> 
                 <p>A second software package that I try to use is Halcon, at the moment I've no overview of the possibilities.<br>But it looks to me that you can program in Halcon's own language hdevelop or using their libraries in your own code (like C++).<br>Programming in hdevelop with it's GUI seems to be easier/faster than low-level programming (e.g. C++), but I don't know the limitations.<br>A disadvantage is that there is no community for support, you need to use their documentation.</p> 
                 <p>A third option I read a lot about is OpenCV, but with no low-level programming background this seems too ambitious for me.</p> 
                 <p>I'm not searching the best solution for me, but for a company (although I know the company hasn't a lot of computer engineers).<br>I was hoping to find software with a good GUI to reduce low-level programming, Halcon seems to be the closest match.</p> 
                 <p>Thanks for your help.</p>

                        <span class="full"> <a href="javascript:" class="js-toggle-abstract">[Hide abstract]</a> <strong>ABSTRACT:</strong> Las asignaturas de diseño mecatrónico se centran en el diseño funcional de sistemas autónomos e inteligentes, aplicando diferentes disciplinas tecnológicas (dinámica mecánica, la electrónica de baja y alta potencia, electromecánica, control de movimiento, la óptica, metrología y procesamiento de señales) en un entorno bien equilibrado fortaleciéndose con la realización de prototipos. El presente artículo expone el proceso de diseño, construcción y puesta en marcha de un robot cartesiano con 3 grados de libertad y la programación de su respectivo software de interfaz con el usuario, este fue realizado en lenguaje Python donde las principales ventajas para su escogencia fueron su versatilidad, portabilidad y documentación, además de ser Open-Source. Se requería implementar un algoritmo para el reconocimiento de voz de tal forma que pudiera clasificar 3 palabras diferentes a través de redes neuronales artificiales. </span>
