    <h1 id="intel-realsense-technology-with-javascript-api-for-linux-os">Intel® RealSense™ technology with JavaScript API for Linux OS</h1> 
    <p><a href="https://travis-ci.org/01org/node-realsense"><img src="https://travis-ci.org/01org/node-realsense.svg?branch=master" alt="Build Status"></a></p> 
    <p><a href="/node-realsense/LICENSE"> <img src="https://img.shields.io/github/license/mashape/apistatus.svg?style=flat" alt="License"> </a></p> 
    <p>This project is offering Node.js modules of <a href="http://www.intel.com/content/www/us/en/architecture-and-technology/realsense-overview.html">Intel® RealSense™ technology</a> including Librealsense, Person Tracaking, Object Recognization, and SLAM capabilities. The target is <a href="https://software.intel.com/en-us/iot/hardware/joule">Intel® Joule™</a> and ZR300 camera on top of Ubuntu 16.04 DT.</p> 
    <p>These modules have been registered as NPM packages, listed as below.</p> 
    <ul> 
     <li>Librealsense module, <a href="https://www.npmjs.com/package/node-librealsense">node-librealsense</a></li> 
     <li>Person Tracking module, <a href="https://www.npmjs.com/package/node-person">node-person</a></li> 
     <li>Object Recognition module, <a href="https://www.npmjs.com/package/node-object">node-object</a></li> 
     <li>SLAM module, <a href="https://www.npmjs.com/package/node-slam">node-slam</a></li> 
    </ul> 
    <p>For example, Person tracking module can be installed by following command.</p> 
    <div class="highlighter-rouge">
     <pre class="highlight"><code>npm install --save node-person
</code></pre> 
    </div> 
    <p><em>[Note] The target platform of these modules is <a href="https://software.intel.com/en-us/iot/hardware/joule">Intel® Joule™</a> and ZR300 camera on top of Ubuntu 16.04 DT, please prepare the hardware devices and <a href="https://github.com/01org/node-realsense/blob/master/doc/setup_environment.md">setup proper environment</a> before install and use these modules.</em></p> 
    <p>Besides the <a href="./doc/spec">API Specification</a> and NPM package site, <a href="https://github.com/01org/node-realsense/releases/download/v0.10.0/realsense_samples-0.10.0-node-realsense.tar.gz">JavaScript samples</a> are also good examples demonstrating how to use these modules.</p> 
    <h2 id="documents">Documents</h2> 
    <ul> 
     <li><a href="./doc/spec">API Specification</a> - This specification describes the <code class="highlighter-rouge">JavaScript</code> API provided by the <code class="highlighter-rouge">Node.js</code> C++ add-ons of this repository.</li> 
     <li><a href="/node-realsense/doc/getting_started.html">Getting Started</a> - A guide on how to setup build environment, how to build one or many <code class="highlighter-rouge">Node.js</code> C++ add-on(s) residing in this repository and/or how to use, debug &amp; change API definition to the C++ add-on(s).</li> 
     <li><a href="/node-realsense/doc/contribution.html">Contribution</a> - A guide on how to contribute code to this repository.</li> 
     <li><a href="/node-realsense/doc/bkms.html">BKMs</a> - Listing Best Known Methods (BKMs) for common issues.</li> 
    </ul>
