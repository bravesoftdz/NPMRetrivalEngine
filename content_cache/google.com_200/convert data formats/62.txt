      <h2><span class="mw-headline" id="Customizing_the_Time_and_Date">Customizing the Time and Date</span></h2> 
      <p>Certain WordPress tag functions are used to display or return date and time information; <a href="/Template_Tags/the_date" title="Template Tags/the date" class="mw-redirect">the_date()</a> and <a href="/Template_Tags/the_time" title="Template Tags/the time" class="mw-redirect">the_time()</a> are examples of this. Some of these functions accept a parameter called a format string that allows you to determine how the date is going to be displayed. The format string is a template in which various parts of the date are combined (using "format characters") to generate a date in the format specified. </p>
      <p>For example, the format string: </p> 
      <pre>l, F j, Y
</pre> 
      <p>creates a date that look like this: </p> 
      <pre>Friday, September 24, 2004
</pre> 
      <p>Here is what each format character in the string above represents: </p> 
      <ul> 
       <li> <code>l</code> = Full name for day of the week (lower-case L). </li> 
       <li> <code>F</code> = Full name for the month. </li> 
       <li> <code>j</code> = The day of the month. </li> 
       <li> <code>Y</code> = The year in 4 digits. (lower-case y gives the year's last 2 digits) </li> 
      </ul> 
      <dl> 
       <dd>
         (Commas are read literally.) 
       </dd> 
      </dl> 
      <p>WordPress is written in the programming language <a href="/Glossary#PHP" title="Glossary">PHP</a>. The date formatting functions in WordPress use PHP's built-in date formatting functions. You can use the <a rel="nofollow" class="external text" href="http://php.net/date">table of date format characters on the PHP website</a> as a reference for building date format strings for use in WordPress. Here is a table of some of the more useful items found there: </p>

      <h3><span class="mw-headline" id="Examples">Examples</span></h3> 
      <p>Here are some examples of date format and result output. </p> 
      <ul> 
       <li> <code>F j, Y g:i a</code> - November 6, 2010 12:50 am </li> 
       <li> <code>F j, Y</code> - November 6, 2010 </li> 
       <li> <code>F, Y</code> - November, 2010 </li> 
       <li> <code>g:i a</code> - 12:50 am </li> 
       <li> <code>g:i:s a</code> - 12:50:48 am </li> 
       <li> <code>l, F jS, Y</code> - Saturday, November 6th, 2010 </li> 
       <li> <code>M j, Y @ G:i</code> - Nov 6, 2010 @ 0:50 </li> 
       <li> <code>Y/m/d \a\t g:i A</code> - 2010/11/06 at 12:50 AM </li> 
       <li> <code>Y/m/d \a\t g:ia</code> - 2010/11/06 at 12:50am </li> 
       <li> <code>Y/m/d g:i:s A</code> - 2010/11/06 12:50:48 AM </li> 
       <li> <code>Y/m/d</code> - 2010/11/06 </li> 
      </ul> 
      <p>Combined with <code>the_time()</code> template tag, the code below in the template file: </p> 
      <pre>This entry was posted on &lt;?php the_time('l, F jS, Y')&nbsp;?&gt; and is filed under &lt;?php the_category(', ')&nbsp;?&gt;.
</pre> 
      <p>will be shown on your site as following: </p> 
      <pre>This entry was posted on Friday, September 24th, 2004 and is filed under <u>WordPress</u> and <u>WordPress Tips</u>.
</pre> 
      <h3><span class="mw-headline" id="Localization">Localization</span></h3> 
      <p>To localize dates, use the <a href="/Function_Reference/date_i18n" title="Function Reference/date i18n">date_i18n()</a> function. </p>
      <p>You can probably safely localize these date format strings with the <code>__()</code>, <code>_e()</code>, etc. functions (demonstrated with <code>get_the_date(__(…))</code>): </p> 
      <h3><span class="mw-headline" id="Escaping">Escaping</span></h3> 
      <p>You can escape custom characters using the <code>\letter</code> format. For example you would escape the text <code>at</code> with <code>\a\t</code>. </p>
