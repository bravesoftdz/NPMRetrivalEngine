      <h2>Coastal Engineering Software and Data &amp; Software Sources</h2>
      <br>
      <br> 
      <img src="http://www.coastal.udel.edu/coastal/wave1.gif">
      <br>
      <br> 
     </center> <br><br><br> 
     <center>
      <a href="#Software">Software</a>&nbsp;&nbsp;&nbsp; 
      <a href="#Data">Data and Software Sources</a>&nbsp;&nbsp;&nbsp;
     </center> <br><br><br> <a name="Software"> <h3>Software Packages</h3></a> 
     <ul> 
      <li><a href="http://chinacat.coastal.udel.edu/~kirby/programs/index.html">Center for Applied Coastal Research</a></li> 
      <li><a href="http://www.citg.tudelft.nl/live/pagina.jsp?id=f928097d-81bb-4042-971b-e028c00e3326&amp;lang=en"> SWAN</a></li> 
     </ul> <h3> Java Applets</h3> 
     <ul>
      <li><a href="http://www.coastal.udel.edu/faculty/rad"> Coastal Engineering Applets</a> (Dalrymple's Applet Page) </li> 
     </ul> <h3> Matlab Codes</h3> 
     <ul> 
      <li><a href="http://www.oceanwave.jp/index.php?Research%2FSoftwares">MACE (Matlab Functions for Coastal Engineers</a>, by Nobuhito Mori</li> 
      <li><a href="http://www.metocean.co.nz/cms/software">DIWASP,</a> Directional Wave Spectra Toolbox</li> 
      <li><a href="http://www.maths.lth.se/matstat/wafo/index.html">WAFO (Wave Analysis for Fatigue and Oceanography</a></li> 
      <li><a href="http:///sea-mat.whoi.edu/"> SEA-MAT,</a> Matlab Tools for Oceanographic Analysis</li> 
     </ul> <h3> Data Display</h3> 
     <ul>
      <li> <a href="http://ferret.wrc.noaa.gov/Ferret/ferret_home.html">FERRET</a>, Data Visualization and Analysis</li>
     </ul> <h3> Shoreline Change</h3> 
     <ul>
      <li><a href="http://woodshole.er.usgs.gov/project-pages/dsas/">Digital Shoreline Analysis System, DSAS (ArcView)</a></li>
     </ul> <h3> Tides</h3>
     <ul> 
      <li> <a href="http://www.flaterco.com/xtide/files.html">Xtide</a>, the harmonic tide predictor for X windows</li> 
      <li><a href="http://www.pac.dfo-mpo.gc.ca/sci/osap/projects/tidpack/tidpack_e.htm">IOS Tidal Package</a>, prediction and analysis packages for tides and currents (Fortran)</li> 
      <li><a href="http://www2.ocgy.ubc.ca/~rich/#T_Tide">T_Tide</a>, a Matlab version of the IOS Tidal Package</li> 
      <li><a href="http://ilikai.soest.hawaii.edu/UHSLC/jaslsoft.html"> U. Hawaii Sea Level Center</a> PC software for tidal analysis</li> 
      <li><a href="http://topex-www.jpl.nasa.gov/"> TOPEX/Poseidon Satellite Tidal Models</a></li> 
     </ul> <h3> Waves</h3> 
     <ul> 
      <li> <a href="ftp://coastal.udel.edu/pub/programs/stream_function/"> Stream Function Wave Theory</a> (stream.f: for nonlinear periodic waves over constant depth. nlinear.f: for waves on complex current profiles.)</li> 
      <li> <a href="http://polar.ncep.noaa.gov/waves/main_int.html"> WaveWatch III</a> (spectral surface waves)</li> 
     </ul> <h3> Marine Hydrodynamics</h3> 
     <ul> 
      <li><a href="http://www.bodrum-bodrum.com/vorteks/arsenal/">Virtual Arsenal of the Marine Engineer </a></li>
     </ul> <h3> Sharp Zaurus software</h3> These are command line programs cross-compiled from FORTRAN sources. (In addition, several Java applets for the Zaurus are found at <a href="http://www.coastal.udel.edu/faculty/rad/downloadsharp.html">www.coastal.udel.edu/faculty/rad/downloadsharp.html</a>.) Address questions to rad@jhu.edu. 
     <ul>
      <li> <a href="http://www.coastal.udel.edu/coastal/stream">Stream Function</a></li> 
      <li><a href="http://www.coastal.udel.edu/coastal/refdif1">Refdif1</a></li> 
     </ul> <h3> Other Coastal/Ocean Web Pages</h3> 
     <ul> 
      <li><a href="http://www.coastal-guide.com"> Coastal-Guide</a></li> 
      <li><a href="http://www.coastalguide.org"> Coastal Guide, Europe</a></li> 
      <li><a href="http://marenet.uni-oldenburg.de/MareNet">MareNet</a></li> 
      <li><a href="http://www.engineersaustralia.org.au/learned-groups/national-committees-and-panels/coastal-&amp;-ocean-engineering/about-nccoe/about-nccoe.cfm">Australia's National Committee on Coastal and Ocean Engineering</a></li> 
     </ul> <br><br><br> <a name="Data"> <h2> Data and Software Sources </h2></a> <h3>Tides</h3> 
     <ul>

      <li> <a href="http://tidesonline.noaa.gov/"> Tides Online</a></li>

     <ul> 
      <li><a href="http://mapindex.nos.noaa.gov/">NOS Mapfinder</a></li> 
      <li><a href="http://chl.erdc.usace.army.mil/chl.aspx?p=s&amp;a=Publications;354"> CHL CEDRS (for U.S.) </a></li> 
      <li><a href="http://www.csc.noaa.gov/shoreline/">NOAA Shoreline Data Site</a></li> 
      <li><a href="http://www.ngdc.noaa.gov">NOAA National Geophysical Data Center</a></li>
      <ul> 
       <li><a href="http://www.ngdc.noaa.gov/mgg/bathymetry/hydro.html">NOS Hydrographic Survey Data</a></li>

      <li><a href="http://www.maptech.com/"> Maptech</a> for nautical charts</li>
     </ul> <h3> Bottom Sediments</h3>

      <li><a href="http://instaar.colorado.edu/~jenkinsc/dbseabed/goseabed/interactive/">Sea Bed Data</a></li>

     <ul>

     </ul> <h3> Aerial Photography</h3>

      <ul>
