        <dl class="msl-accordion" id="question-types">
         <dt>
          <a href="#syllabus-1">Implement and manipulate document structures and objects (24%)</a>
         </dt>
         <dd id="syllabus-1">
          <ul class="plain">
           <li> Create the document structure 
            <ul>
             <li>Structure the UI by using semantic markup, including for search engines and screen readers (Section, Article, Nav, Header, Footer, and Aside); create a layout container in HTML</li>
            </ul></li>
           <li> Write code that interacts with UI controls 
            <ul>
             <li>Programmatically add and modify HTML elements; implement media controls; implement HTML5 canvas and SVG graphics</li>
            </ul></li>
           <li> Apply styling to HTML elements programmatically 
            <ul>
             <li>Change the location of an element; apply a transform; show and hide elements</li>
            </ul></li>
           <li> Implement HTML5 APIs 
            <ul>
             <li>Implement storage APIs, AppCache API, and Geolocation API</li>
            </ul></li>
           <li> Establish the scope of objects and variables 
            <ul>
             <li>Define the lifetime of variables; keep objects out of the global namespace; use the this keyword to reference an object that fired an event; scope variables locally and globally</li>
            </ul></li>
           <li> Create and implement objects and methods 
            <ul>
             <li>Implement native objects; create custom objects and custom properties for native objects using prototypes and functions; inherit from an object; implement native methods and create custom methods</li>
            </ul></li>
          </ul>
          <p><strong>Preparation resources</strong></p>
          <ul class="plain">
           <li><a target="_blank" class="mscom-link" href="http://msdn.microsoft.com/hh534406">The developers guide to HTML5 canvas</a></li>
           <li><a target="_blank" class="mscom-link" href="http://msdn.microsoft.com/library/gg589508(v=vs.85).aspx">How to zoom and pan with SVG</a></li>
          </ul>
         </dd>
         <dt>
          <a href="#syllabus-2">Implement program flow (25%)</a>
         </dt>
         <dd id="syllabus-2">
          <ul class="plain">
           <li> Implement program flow 
            <ul>
             <li>Iterate across collections and array items; manage program decisions by using switch statements, if/then, and operators; evaluate expressions</li>
            </ul></li>
           <li> Raise and handle an event 
            <ul>
             <li>Handle common events exposed by DOM (OnBlur, OnFocus, OnClick); declare and handle bubbled events; handle an event by using an anonymous function</li>
            </ul></li>
           <li> Implement exception handling 
            <ul>
             <li>Set and respond to error codes; throw an exception; request for null checks; implement try-catch-finally blocks</li>
            </ul></li>
           <li> Implement a callback 
            <ul>
             <li>Receive messages from the HTML5 WebSocket API; use jQuery to make an AJAX call; wire up an event; implement a callback by using anonymous functions; handle the this pointer</li>
            </ul></li>
           <li> Create a web worker process 
            <ul>
             <li>Start and stop a web worker; pass data to a web worker; configure timeouts and intervals on the web worker; register an event listener for the web worker; limitations of a web worker</li>
            </ul></li>
          </ul>
          <p><strong>Preparation resources</strong></p>
          <ul class="plain">
           <li><a target="_blank" class="mscom-link" href="http://msdn.microsoft.com/library/ie/kw1tezhk(v=vs.94).aspx">Controlling program flow (JavaScript)</a></li>
           <li><a target="_blank" class="mscom-link" href="http://msdn.microsoft.com/library/windows/apps/hh780660.aspx">Coding basic apps</a></li>
           <li><a target="_blank" class="mscom-link" href="http://msdn.microsoft.com/library/ie/4yahc5d8(v=vs.94).aspx">try...catch...finally statement (JavaScript)</a></li>
          </ul>
         </dd>
         <dt>
          <a href="#syllabus-3">Access and secure data (26%)</a>
         </dt>
         <dd id="syllabus-3">
          <ul class="plain">
           <li> Validate user input by using HTML5 elements 
            <ul>
             <li>Choose the appropriate controls based on requirements; implement HTML input types and content attributes (for example, required) to collect user input</li>
            </ul></li>
           <li> Validate user input by using JavaScript 
            <ul>
             <li>Evaluate a regular expression to validate the input format; validate that you are getting the right kind of data type by using built-in functions; prevent code injection</li>
            </ul></li>
           <li> Consume data 
            <ul>
             <li>Consume JSON and XML data; retrieve data by using web services; load data or get data from other sources by using XMLHTTPRequest</li>
            </ul></li>
           <li> Serialize, deserialize, and transmit data 
            <ul>
             <li>Binary data; text data (JSON, XML); implement the jQuery serialize method; Form.Submit; parse data; send data by using XMLHTTPRequest; sanitize input by using URI/form encoding</li>
            </ul></li>
          </ul>
          <p><strong>Preparation resources</strong></p>
          <ul class="plain">
           <li><a target="_blank" class="mscom-link" href="http://msdn.microsoft.com/library/ie/hh772941(v=vs.85).aspx">pattern attribute | pattern property</a></li>
           <li><a target="_blank" class="mscom-link" href="http://msdn.microsoft.com/library/ie/hh673561(v=vs.85).aspx">Sandbox</a></li>
           <li><a target="_blank" class="mscom-link" href="http://msdn.microsoft.com/library/ie/ms535874(v=vs.85).aspx">XMLHttpRequest object</a></li>
          </ul>
         </dd>
         <dt>
          <a href="#syllabus-4">Use CSS3 in applications (25%)</a>
         </dt>
         <dd id="syllabus-4">
          <ul class="plain">
           <li> Style HTML text properties 
            <ul>
             <li>Apply styles to text appearance (color, bold, italics); apply styles to text font (WOFF and @font-face, size); apply styles to text alignment, spacing, and indentation; apply styles to text hyphenation; apply styles for a text drop shadow</li>
            </ul></li>
           <li> Style HTML box properties 
            <ul>
             <li>Apply styles to alter appearance attributes (size, border and rounding border corners, outline, padding, margin); apply styles to alter graphic effects (transparency, opacity, background image, gradients, shadow, clipping); apply styles to establish and change an elements position (static, relative, absolute, fixed)</li>
            </ul></li>
           <li> Create a flexible content layout 
            <ul>
             <li>Implement a layout using a flexible box model; implement a layout using multi-column; implement a layout using position floating and exclusions; implement a layout using grid alignment; implement a layout using regions, grouping, and nesting</li>
            </ul></li>
           <li> Create an animated and adaptive UI 
            <ul>
             <li>Animate objects by applying CSS transitions; apply 3-D and 2-D transformations; adjust UI based on media queries (device adaptations for output formats, displays, and representations); hide or disable controls</li>
            </ul></li>
           <li> Find elements by using CSS selectors and jQuery 
            <ul>
             <li>Choose the correct selector to reference an element; define element, style, and attribute selectors; find elements by using pseudo-elements and pseudo-classes (for example, :before, :first-line, :first-letter, :target, :lang, :checked, :first-child)</li>
            </ul></li>
           <li> Structure a CSS file by using CSS selectors 
            <ul>
             <li>Reference elements correctly; implement inheritance; override inheritance by using !important; style an element based on pseudo-elements and pseudo-classes (for example, :before, :first-line, :first-letter, :target, :lang, :checked, :first-child)</li>
            </ul></li>
          </ul>
          <p><strong>Preparation resources</strong></p>

           <li><a target="_blank" class="mscom-link" href="http://msdn.microsoft.com/library/ie/hh673536(v=vs.85).aspx">CSS</a></li>
          </ul>
