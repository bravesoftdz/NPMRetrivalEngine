    <h6>General Bathymetric Chart of the Oceans (GEBCO)</h6> 
    <h1>Gridded bathymetric data sets</h1> 
    <div id="image_right"> 
     <img src="/data/hosted_data_systems/gebco_gridded_bathymetry_data/images/gebco_2013_globe.jpg" alt="Bathymetry data from the GEBCO_2014 Grid" title="Bathymetry data from the GEBCO_2014 Grid" width="250" height="254"> 
     <div id="caption">
      Bathymetry data from the GEBCO_2014 Grid 
      <a href="/resources/help_and_hints/using_this_web_site/picture_copyrights/data/396/" title="View copyright and attribution" onmouseover="dd_StatusBarMsg('View copyright and attribution');" onmouseout="dd_StatusBarMsg('View copyright and attribution');return document.MM_returnValue">© </a> 
     </div> 
    </div> 
    <p>GEBCOs gridded bathymetric data sets are global terrain models for ocean and land. They are maintained and distributed by BODC on behalf of <a href="http://www.gebco.net" target="_self">GEBCO</a>. The data sets include the </p> 
    <ul> 
     <li><span title="Click here for more information about the GEBCO_2013 Grid"> <a href="/data/hosted_data_systems/gebco_gridded_bathymetry_data/gebco_30_second_grid/" target="_self">GEBCO_2014 Grid</a> </span>  a global 30 arc-second interval grid</li> 
     <li><span title="Click here for more information about the GEBCO One Minute Grid"> <a href="/data/hosted_data_systems/gebco_gridded_bathymetry_data/gebco_one_minute_grid/" target="_self">GEBCO One Minute Grid</a></span>  a one arc-minute interval grid. Last updated in 2008, please note that there are no plans for further development of this data set.</li> 
    </ul> 
    <p>The GEBCO_2014 Grid is available to download for user-defined areas in netCDF; Esri ASCII raster or INT16 GeoTiff formats. The GEBCO One Minute Grid is available in netCDF format only.</p> 
    <p>In netCDF format the grids are available as either two-dimensional (2D) or one-dimensional (1D) array grid files. The 2D array grids use the <a href="http://cfconventions.org/">Climate and Forecast (CF)</a> metadata conventions. You can download data for a user-defined area or the complete global grid. The 1D array grids are for use with GEBCO's <a href="/resources/products/software/gebco_grid_display/"> Grid Display Software</a> and GEBCO Digital Atlas (<a href="/resources/products/data/bodc_products/gebco/">GDA</a>) and so are only available as a global grid.</p> 
    <p> Find out more about the <a href="/resources/delivery_formats/gebco_formats/">file formats and software packages</a> that can be used to view and access the data sets. </p> 
    <h3 id="netCDFnew">Select your data set</h3> 
    <p>From the table and map below, select your grid and the area of interest. Add your selection to the basket and repeat the process as required. Once you are happy with your selection(s) view the basket and checkout your request to start file preparation.</p> 
    <p>Use the '+' button in the top right-hand corner of the map to select to display the GEBCO Source Identifier (SID) Grid.<span style="color: rgb(68, 68, 68); font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 12px; font-style: normal; font-variant: normal; font-weight: normal; letter-spacing: normal; line-height: 21px; orphans: auto; text-align: start; text-indent: 0px; text-transform: none; white-space: normal; widows: auto; word-spacing: 0px; -webkit-text-stroke-width: 0px; display: inline !important; float: none; background-color: rgb(255, 255, 255);"><span class="Apple-converted-space">&nbsp;</span>It shows those grid cells for which the data in the GEBCO Grid are based on bathymetric sounding or grid values. It is best viewed at a zoomed in level to appreciate the detail in the data set.</span></p>

      <h3> GEBCO_2014 Grid (30 arc-second interval) </h3> 
      <div> 
       <p> The GEBCO_2014 Grid is a continuous terrain model for ocean and land with a spatial resolution of 30 arc-seconds. It was generated by combining quality-controlled ship depth soundings with interpolation between sounding points guided by satellite-derived gravity data. Where they improve on the existing grid, data sets developed by other methods are included. Further information can be found in the data set documentation. The complete global grid file in compressed form is 1.1 Gbytes.<span title="Click here for more information about the GEBCO_2014 Grid"> <a href="/data/hosted_data_systems/gebco_gridded_bathymetry_data/gebco_30_second_grid/" target="_self">More information</a>. </span> </p>

      <h3>GEBCO_2014 Source Identifier (SID) Grid (30 arc-second interval)</h3> 
      <div> 
       <p> The GEBCO_2014 Source Identifier (SID) grid accompanies the GEBCO_2014 bathymetric grid. This data set identifies which grid cells in the GEBCO_2014 grid are based on bathymetric soundings or bathymetric depth values from grids, and which cells contain predicted depth values. The SID grid is a global grid file at 30 arc-second intervals. Further information can be found in the data set documentation. The complete global grid file in compressed form is 32 Mbytes.<span title="Click here for more information about the GEBCO_2014 Source Identifier (SID) Grid"> <a href="/data/hosted_data_systems/gebco_gridded_bathymetry_data/gebco_30_second_grid/" target="_self">More information</a>. </span> </p>
