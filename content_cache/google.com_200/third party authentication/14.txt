           <h1 name="7295" id="7295" class="graf graf--h3 graf--leading graf--title">Creating a Node.js proxy to secure your third-party API&nbsp;key</h1>
           <p name="4ef3" id="4ef3" class="graf graf--p graf-after--h3">Today I am creating a mailing list signup form that will post data to a sheet on <a href="https://fieldbook.com" data-href="https://fieldbook.com" class="markup--anchor markup--p-anchor" rel="noopener" target="_blank">Fieldbook</a>. We could give our form the API key but this is a terrible idea as it gives anyone who visits the form the key to do anything they want to your database (Fieldbook book).</p>
           <p name="13d7" id="13d7" class="graf graf--p graf-after--p">Instead, we’ll create a proxy in Node.js in less than 50 lines of code.</p>
           <figure name="d8aa" id="d8aa" class="graf graf--figure graf-after--p">
            <div class="aspectRatioPlaceholder is-locked" style="max-width: 700px; max-height: 467px;">
             <div class="aspectRatioPlaceholder-fill" style="padding-bottom: 66.7%;"></div>
             <div class="progressiveMedia js-progressiveMedia graf-image" data-image-id="1*XviICXh4doDeG_ZyDB6Tkw.jpeg" data-width="3929" data-height="2619" data-action="zoom" data-action-value="1*XviICXh4doDeG_ZyDB6Tkw.jpeg">
              <img src="https://cdn-images-1.medium.com/freeze/max/60/1*XviICXh4doDeG_ZyDB6Tkw.jpeg?q=20" crossorigin="anonymous" class="progressiveMedia-thumbnail js-progressiveMedia-thumbnail">
              <canvas class="progressiveMedia-canvas js-progressiveMedia-canvas"></canvas>
              <img class="progressiveMedia-image js-progressiveMedia-image" data-src="https://cdn-images-1.medium.com/max/1600/1*XviICXh4doDeG_ZyDB6Tkw.jpeg">
              <noscript class="js-progressiveMedia-inner">
               <img class="progressiveMedia-noscript js-progressiveMedia-inner" src="https://cdn-images-1.medium.com/max/1600/1*XviICXh4doDeG_ZyDB6Tkw.jpeg">
              </noscript>
             </div>
            </div>
            <figcaption class="imageCaption">
             Photo by 
             <a href="https://unsplash.com/juandinella" data-href="https://unsplash.com/juandinella" class="markup--anchor markup--figure-anchor" rel="noopener" target="_blank">Juan D&nbsp;Nella</a>
            </figcaption>
           </figure>
           <p name="1d93" id="1d93" class="graf graf--p graf-after--figure">First let’s set up the database. Log into Fieldbook and create a new book. In that book, make a sheet named <strong class="markup--strong markup--p-strong">Contacts</strong>. This sheet will contain these three columns</p>
           <ul class="postList">
            <li name="91d4" id="91d4" class="graf graf--li graf-after--p"><strong class="markup--strong markup--li-strong">Contact ID</strong> (I am just using Fieldbook’s built in numeric ID)</li>
            <li name="02dc" id="02dc" class="graf graf--li graf-after--li"><strong class="markup--strong markup--li-strong">Email address</strong>, and?—?just for the fun of it —</li>
            <li name="14a3" id="14a3" class="graf graf--li graf-after--li"><strong class="markup--strong markup--li-strong">Signup date</strong>.</li>
           </ul>
           <p name="ddca" id="ddca" class="graf graf--p graf-after--li">Now, click on <strong class="markup--strong markup--p-strong">API</strong>, then <strong class="markup--strong markup--p-strong">Manage API access</strong>, then <strong class="markup--strong markup--p-strong">Generate a new API key</strong>. Make sure to copy down the <strong class="markup--strong markup--p-strong">Username</strong> and <strong class="markup--strong markup--p-strong">Password</strong>.</p>
           <p name="3fab" id="3fab" class="graf graf--p graf-after--p">Now for the code. Let’s create a new directory and cd into it.</p>
           <pre name="5e3c" id="5e3c" class="graf graf--pre graf-after--p">mkdir fbproxy<br>cd fbproxy</pre>
           <p name="afb7" id="afb7" class="graf graf--p graf-after--pre">Next, let’s initialize our npm project.</p>
           <pre name="98fb" id="98fb" class="graf graf--pre graf-after--p">npm init</pre>
           <p name="ee40" id="ee40" class="graf graf--p graf-after--pre">Answer any prompts.</p>
           <p name="befc" id="befc" class="graf graf--p graf-after--p">Now we can install dependencies.</p>
           <pre name="9d9c" id="9d9c" class="graf graf--pre graf-after--p">npm install express body-parser requestify moment --save</pre>
           <p name="c778" id="c778" class="graf graf--p graf-after--pre">The proxy itself is simple enough. In a new file called index.js, we’ll first put the parser in strict mode and require our modules.</p>
           <pre name="568a" id="568a" class="graf graf--pre graf-after--p">'use strict';</pre>
           <pre name="1404" id="1404" class="graf graf--pre graf-after--pre">var express = require('express');<br>var bodyParser = require('body-parser');<br>var requestify = require('requestify');<br>var moment = require('moment');</pre>
           <p name="bd18" id="bd18" class="graf graf--p graf-after--pre">Now set up the constants.</p>
           <pre name="e82d" id="e82d" class="graf graf--pre graf-after--p">var PORT = process.env.PORT || 3000;<br>var BASE_URL = 'https://api.fieldbook.com/v1/[your book ID]';<br>var API_OPTIONS = {<br>  headers: { accept: 'application/json' },<br>  auth: {<br>    username: process.env.FB_API_USERNAME,<br>    password: process.env.FB_API_PASSWORD,<br>  },<br>};</pre>
           <pre name="29d0" id="29d0" class="graf graf--pre graf-after--pre">var app = express();</pre>
           <p name="4939" id="4939" class="graf graf--p graf-after--pre">Since the form will be running from a separate service as our proxy, we’ll need to set up CORS. And since we’ll be accepting form data, we’ll set up the body-parser, too.</p>
           <pre name="d12b" id="d12b" class="graf graf--pre graf-after--p">app.use(function(req, res, next) {<br>  res.header('Access-Control-Allow-Origin', '*');<br>  res.header('Access-Control-Allow-Headers', 'Origin, X-Requested-With, Content-Type, Accept');<br>  next();<br>});</pre>
           <pre name="de82" id="de82" class="graf graf--pre graf-after--pre">app.use(bodyParser.urlencoded({ extended: true }));</pre>
           <p name="d245" id="d245" class="graf graf--p graf-after--pre">Finally, we get to the meat of the proxy: a single route for handling POST requests from our form.</p>
           <pre name="b700" id="b700" class="graf graf--pre graf-after--p">app.post('/contacts', function(req, res) {<br>  var URL = BASE_URL + '/contacts';</pre>
           <pre name="d94f" id="d94f" class="graf graf--pre graf-after--pre">  var record = {<br>    email_address: req.body.emailAddress,<br>    signup_date: moment().format('M/D/YYYY');<br>  };</pre>
           <pre name="fd6c" id="fd6c" class="graf graf--pre graf-after--pre">  requestify<br>    .post(URL, record, API_OPTIONS)<br>    .then(function(fbres) {<br>      res.json(fbres);<br>    })<br>    .catch(function(err) {<br>      res.json(err);<br>      // you should also probably set up some logging here.<br>    });<br>});</pre>
           <p name="870a" id="870a" class="graf graf--p graf-after--pre">With everything else set up, we will simply tell our proxy to listen for incoming connections.</p>
           <pre name="251f" id="251f" class="graf graf--pre graf-after--p">app.listen(PORT, function() {<br>  console.log('Fieldbook proxy listening on port %s.', PORT);<br>});</pre>
           <p name="3f9b" id="3f9b" class="graf graf--p graf-after--pre">In order to run the proxy, we’ll set our environment variables. This will depend on your operating system. But if you’re on Windows like me, open the command prompt and run these commands:</p>
           <pre name="7fd1" id="7fd1" class="graf graf--pre graf-after--p">SET PORT=3000<br>SET FB_API_USERNAME=[your API username]<br>SET FB_API_PASSWORD=[your API password]</pre>
           <p name="2cd4" id="2cd4" class="graf graf--p graf-after--pre">And simply start the proxy:</p>
           <pre name="63cb" id="63cb" class="graf graf--pre graf-after--p">node index.js</pre>
           <p name="d648" id="d648" class="graf graf--p graf-after--pre graf--trailing">This proxy will accept form data with an emailAddress key, add the sign up date, and post it to your Fieldbook sheet securely. However, it’s worth mentioning that, as it is, this proxy accepts form data from any origin.</p>
