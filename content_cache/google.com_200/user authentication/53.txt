       <h2 id="client-library-for-nuxeo-api">Client Library for Nuxeo API</h2>
       <p><a href="https://qa.nuxeo.org/jenkins/job/Client/job/nuxeo-js-client-master-vs-maste/"><img alt="Jenkins" src="https://img.shields.io/jenkins/s/https/qa.nuxeo.org/jenkins/job/Client/job/nuxeo-js-client-master-vs-master.svg?style=flat-square"></a> <a href="https://www.npmjs.com/package/nuxeo"><img alt="npm version" src="https://img.shields.io/npm/v/nuxeo.svg?style=flat-square"></a> <a href="https://www.npmjs.com/package/nuxeo"><img alt="npm downloads" src="https://img.shields.io/npm/dm/nuxeo.svg?style=flat-square"></a> <a href="https://david-dm.org/nuxeo/nuxeo-js-client"><img alt="Dependency Status" src="https://img.shields.io/david/nuxeo/nuxeo-js-client.svg?style=flat-square"></a> <a href="https://david-dm.org/nuxeo/nuxeo-js-client#info=devDependencies"><img alt="devDependency Status" src="https://img.shields.io/david/dev/nuxeo/nuxeo-js-client.svg?style=flat-square"></a></p> 
       <p>The Nuxeo JavaScript Client is a JavaScript client library for the Nuxeo Automation and REST API. The library can work in a browser, or in Node.js, using the same API.</p> 
       <p>This is an on-going project, supported by Nuxeo.</p> 
       <h2 id="nuxeo-platform-dependency">Nuxeo Platform Dependency</h2>
       <p>The JS Client is compliant with all Nuxeo versions as of LTS 2015.</p> 
       <h2 id="getting-started">Getting Started</h2>
       <h3 id="installation">Installation</h3>
       <h4 id="nodejs-applications">Node.js Applications</h4>
       <p>After <a href="https://nodejs.org/#download">installing</a> <a href="https://nodejs.org">Node.js</a>, use <code>npm</code> to install the <code>nuxeo</code> package:</p> 
       <pre><code>$ npm install nuxeo
</code></pre>
       <h5 id="nodejs-v6-lts-boron">Node.js v6 LTS (Boron)</h5>
       <pre><code class="javascript">var Nuxeo = require('nuxeo');
var nuxeo = new Nuxeo({ ... });
</code></pre> 
       <h5 id="nodejs-v4-lts-argon">Node.js v4 LTS (Argon)</h5>
       <pre><code class="javascript">var Nuxeo = require('nuxeo/es5');
var nuxeo = new Nuxeo({ ... });
</code></pre> 
       <h4 id="bower-powered-applications">Bower Powered Applications</h4>
       <p>The <code>nuxeo</code> client can be also installed through bower:</p> 
       <pre><code>$ bower install nuxeo --save
</code></pre>
       <p>When added to your page, <code>Nuxeo</code> is available as a global variable.</p> 
       <pre><code class="javascript">var nuxeo = new Nuxeo({ ... });
</code></pre> 
       <h4 id="angular-applications">Angular Applications</h4>
       <p>After adding <code>nuxeo</code> through Bower, you can easily create a service that will return a client:</p> 
       <pre><code class="javascript">...
.service('nuxeo', function() {
  return new Nuxeo({
    baseURL: 'http://localhost:8080/nuxeo/',
    auth: {
      method: 'basic',
      username: 'Administrator',
      password: 'Administrator'
    }
  });
})
...
</code></pre> 
       <p>To notify Angular to update the UI when a Nuxeo promise has resolved, you can either wrap Nuxeo promises in <code>$q.when()</code> or, the preferred way, configure the Promise library class to be <code>$q</code>.</p> 
       <pre><code class="javascript">// wrap promises
...
$q.when(nuxeo.request('/path/').get()).then(function(res) {
  $scope.res = res;
});
// use $q as the Promise library class
...
.service('nuxeo', function($q) {
  Nuxeo.promiseLibrary($q);
  return new Nuxeo({
    baseURL: 'http://localhost:8080/nuxeo/',
    auth: {
      method: 'basic',
      username: 'Administrator',
      password: 'Administrator'
    }
  });
})
...
</code></pre> 
       <h4 id="react-applications">React Applications</h4>
       <p>After adding <code>nuxeo</code> through <code>npm</code> to your application, to make it sure that it will work on most browsers you must require <code>nuxeo</code> differently according to your build system.</p> 
       <p>If your build transpiles external libraries from ES6 to ES5:</p> 
       <pre><code class="javascript">var Nuxeo = require('nuxeo');
</code></pre> 
       <p>If your build does not (such as <a href="https://github.com/facebookincubator/create-react-app">create-react-app</a>):</p> 
       <pre><code class="javascript">var Nuxeo = require('nuxeo/es5');
</code></pre> 
       <h2 id="documentation">Documentation</h2>
       <p>Check out the <a href="https://nuxeo.github.io/nuxeo-js-client/latest/">API documentation</a>.</p> 
       <h2 id="examples">Examples</h2>
       <p>Some working examples using the Nuxeo JavaScript Client can be found <a href="https://github.com/nuxeo/nuxeo-js-client/tree/master/examples">here</a>.</p> 
       <h2 id="quick-start">Quick Start</h2>
       <p>This quick start guide will show how to do basics operations using the client.</p> 
       <h3 id="authentication">Authentication</h3>
       <p>The authentication method to be used is defined when creating a client:</p> 
       <pre><code class="javascript">var nuxeo = new Nuxeo({
  auth: {
    method: ...,
    ...
  }
});
</code></pre> 
       <p>The client supports different authentication methods matching the ones available on the Nuxeo Platform.</p> 
       <h4 id="basic-authentication">Basic Authentication</h4>
       <p>The simplest one allowing to authenticate with a <code>username</code> and <code>password</code>.</p> 
       <pre><code class="javascript">var nuxeo = new Nuxeo({
  auth: {
    method: 'basic',
    username: 'Administrator',
    password: 'Administrator'
  }
});
</code></pre> 
       <h4 id="portal-authenticaton">Portal Authenticaton</h4>
       <p>If the <a href="https://doc.nuxeo.com/nxdoc/using-sso-portals/">Portal Authentication</a> is configured on the Nuxeo Platform, you can authenticate with the <code>portal</code> method.</p> 
       <pre><code class="javascript">var nuxeo = new Nuxeo({
  auth: {
    method: 'portal',
    username: 'joe',
    secret: 'shared-secret-from-server'
  }
});
</code></pre> 
       <h4 id="token-authentication">Token Authentication</h4>
       <p>To authenticate through a token from the Nuxeo Server:</p> 
       <pre><code class="javascript">var nuxeo = new Nuxeo({
  auth: {
    method: 'token',
    token: 'a-token'
  }
});
</code></pre> 
       <p>There is a utility method <code>Nuxeo#requestAuthenticationToken</code> to retrieve a <code>token</code> from a Nuxeo Server using the configured authentication method. For instance, a typical flow would be:</p> 
       <pre><code class="javascript">var nuxeo = new Nuxeo({
  auth: {
    method: 'basic',
    username: 'Administrator',
    password: 'Administrator'
  }
});

nuxeo.requestAuthenticationToken('My App', deviceUID, deviceName, 'rw')
  .then(function(token) {
    nuxeo = new Nuxeo({
      auth: {
        method: 'token',
        token: token
      }
    });

    // do something with the new `nuxeo` client using token authentication
    // store the token, and next time you need to create a client, use it
  })
  .catch(function(err) {
    throw err;
  });
</code></pre> 
       <h4 id="oauth2-authorization-and-bearer-token-authentication">OAuth2 Authorization and Bearer Token Authentication</h4>
       <p>Since Nuxeo Platform 9.2, you can use OAuth2 authorization through the JS client.</p> 
       <p>For more information on OAuth2 server side, see <a href="https://doc.nuxeo.com/nxdoc/using-oauth2/">Using OAuth2</a>.</p> 
       <p>Assuming you already have an access token, you can configure the client:</p> 
       <pre><code class="javascript">var nuxeo = new Nuxeo({
  auth: {
    method: 'bearerToken',
    token: access_token,
    clientId: 'my-app' // optional OAuth2 client ID to refresh the access token
  }
});
</code></pre> 
       <p>The <code>bearertoken</code> method supports the <code>token</code> being a simple string (an access token) or a full token object such as:</p> 
       <pre><code class="javascript">{
  "access_token":"H8dXDdEW9U2jJnFDh6lJJ74AHRzCyG4D",
  "token_type":"bearer",
  "expires_in":3600,
  "refresh_token":"Amz8JlyglhGWDmYHMYS5EnTTFUFAwZLiHG4aqQDfkwUNunSMpTTSFUmvprX3WdSF"
}
</code></pre> 
       <p>If the <code>token</code> is a full token object (ie. with a <code>refresh_token</code> key) and a <code>clientId</code> is configured on the <code>auth</code> object, the client will try automatically to refresh the access token if it's expired.</p> 
       <h5 id="oauth2-flow">OAuth2 Flow</h5>
       <p>A typical OAuth2 flow with the Nuxeo Platform would be:</p> 
       <p><strong>Retrieving Authorization Code</strong></p> 
       <p>Generate a "log in" link to be used in a browser, such as:</p> 
       <p><code>http://localhost:8080/nuxeo/oauth2/authorize?client_id=CLIENT_ID&amp;response_type=code&amp;redirect_uri=REDIRECT_URI</code></p> 
       <p>The user sees the login page then, after being logged in, the authorization prompt for the application.</p> 
       <p>If the user grants access, the Nuxeo Platform redirects the user back to the application:</p> 
       <p><code>http://localhost:8000/authorize?code=AUTH_CODE</code></p> 
       <p><strong>Retrieving Access Token</strong></p> 
       <p>The client exchanges the authorization code for an access token:</p> 
       <pre><code>POST http://localhost:8080/nuxeo/oauth2/token
  grant_type=authorization_code
  code=AUTH_CODE
  redirect_uri=REDIRECT_URI
  client_id=CLIENT_ID
</code></pre>
       <p>The Nuxeo Platform replies with an access token:</p> 
       <pre><code class="javascript">{
  "access_token":"H8dXDdEW9U2jJnFDh6lJJ74AHRzCyG4D",
  "token_type":"bearer",
  "expires_in":3600,
  "refresh_token":"Amz8JlyglhGWDmYHMYS5EnTTFUFAwZLiHG4aqQDfkwUNunSMpTTSFUmvprX3WdSF"
}
</code></pre> 
       <p>There are some utility methods on the client to help you with this flow:</p> 
       <p><strong>Nuxeo.oauth2.getAuthorizationURL(baseURL, clientId[, params])</strong></p> 
       <p>Returns the OAuth2 authorization URL for the given <code>baseURL</code> and <code>clientId</code>.</p> 
       <pre><code class="javascript">var authorizationURL = Nuxeo.oauth2.getAuthorizationURL('http://localhost:8080/nuxeo', 'my-app', {
  state: 'xyz',
  redirect_uri: 'http://localhost:8000/authorize',
});
console.log(authorizationURL); // http://localhost:8080/nuxeo/oauth2/authorize?client_id=my-app&amp;response_type=code&amp;state=xyz&amp;redirect_uri=http://localhost:8000/authorize
</code></pre> 
       <p><strong>Nuxeo.oauth2.fetchAccessToken(baseURL, clientId, code[, params])</strong></p> 
       <p>Fetches an OAuth2 access token for the given <code>baseURL</code>, <code>clientId</code> and <code>code</code>.</p> 
       <pre><code class="javascript">var code = ...
Nuxeo.oauth2.fetchAccessToken('http://localhost:8080/nuxeo', 'my-app', code, {
  redirect_uri: 'http://localhost:8000/authorize',
}).then(function(token) {
  // do something with the access token
  var nuxeo = new Nuxeo({
    auth: {
      method: 'bearerToken',
      token: token
    }
  });
});
</code></pre> 
       <p><strong>Nuxeo.oauth2.refreshAccessToken(baseURL, clientId, refreshToken[, params])</strong></p> 
       <p>Manually refresh an access token for the given <code>baseURL</code>, <code>clientId</code> and <code>refreshToken</code>.</p> 
       <pre><code class="javascript">var refreshToken = ...
Nuxeo.oauth2.refreshAccessToken('http://localhost:8080/nuxeo', 'my-app', refreshToken)
  .then(function(token) {
    console.log(token); // refreshed access token
});
</code></pre> 
       <h3 id="creating-a-client">Creating a Client</h3>
       <pre><code class="javascript">var nuxeo = new Nuxeo({
  auth: {
    method: 'basic',
    username: 'Administrator',
    password: 'Administrator'
  }
});
</code></pre> 
       <p>To connect to a different Nuxeo Platform Instance, you can use the following:</p> 
       <pre><code class="javascript">var nuxeo = new Nuxeo({
  baseURL: 'http://demo.nuxeo.com/nuxeo/',
  auth: {
    method: 'basic',
    username: 'Administrator',
    password: 'Administrator'
  }
});
</code></pre> 
       <h3 id="promise-based-calls">Promise Based Calls</h3>
       <p>All API calls made on the server return a Promise object.</p> 
       <pre><code class="javascript">nuxeo.operation('Document.GetChildren')
  .input('/')
  .execute()
  .then(function(docs) {
    // work with docs
  })
  .catch(function(error) {
    // something went wrong
    throw error;
  });
</code></pre> 
       <p>When something went wrong, the <code>error</code> is an <code>Error</code> object with a <code>response</code> field containing the whole response.</p> 
       <h3 id="connecting-to-a-nuxeo-server">Connecting to a Nuxeo Server</h3>
       <p>After creating a Client, you can connect to a Nuxeo Server by using the <code>connect</code> method. This method returns a <code>Promise</code> which is resolved with the connected client.</p> 
       <pre><code class="javascript">var nuxeo = new Nuxeo({ ... });
nuxeo.connect()
  .then(function(client){
    // client.connected === true
    // client === nuxeo
    console.log('Connected OK!');
  })
  .catch(function(error) {
    // wrong credentials / auth method / ...
    throw error;
  });
</code></pre> 
       <h4 id="current-user">Current User</h4>
       <p>The <code>connect</code> method fills the <code>user</code> property of the client. <code>user</code> is a full <code>User</code> object.</p> 
       <pre><code class="javascript">var nuxeo = new Nuxeo({ ... });
nuxeo.connect()
  .then(function(client){
    // client.user.id === 'Administrator'
    console.log(client.user);
  })
  .catch(function(error) {
    // wrong credentials / auth method / ...
    throw error;
  });
</code></pre> 
       <h4 id="nuxeo-server-version">Nuxeo Server version</h4>
       <p>The <code>connect</code> method fills the <code>nuxeoVersion</code> property of the client.</p> 
       <pre><code class="javascript">var nuxeo = new Nuxeo({ ... });
nuxeo.connect()
  .then(function(client){
    // client.nuxeoVersion === '8.10'
    console.log(client.nuxeoVersion);
  })
  .catch(function(error) {
    // wrong credentials / auth method / ...
    throw error;
  });
</code></pre> 
       <p>Some constants are available in the <code>Nuxeo</code> object for supported LTS versions:</p> 
       <pre><code class="javascript">Nuxeo.VERSIONS.LTS_2015 === '7.10';
Nuxeo.VERSIONS.LTS_2016 === '8.10';
</code></pre> 
       <p>You can use them to easily make different calls according to the target version:</p> 
       <pre><code class="javascript">...
if (nuxeo.nuxeoVersion &lt; Nuxeo.VERSIONS.LTS_2016) {
  // do something on versions before LTS 2016
} else {
  // do something else
}
...
</code></pre>

       <h4 id="samples">Samples</h4>

</code></pre>

       <h3 id="document">Document</h3>
       <p><code>Repository</code> object returns and works with <code>Document</code> objects. <code>Document</code> objects exposes a simpler API to work with a document.</p> 
       <p>See the <a href="https://nuxeo.github.io/nuxeo-js-client/latest/Document.html">Document</a> documentation.</p> 
       <h4 id="samples">Samples</h4>
       <p><strong>Retrieve a <code>Document</code> object</strong></p> 
       <pre><code class="javascript">nuxeo.repository()
  .fetch('/')
  .then(function(doc) {
    // doc instanceof Nuxeo.Document === true
  })
  .catch(function(error) {
    throw error;
  });
</code></pre> 
       <p><strong>Set a document property</strong></p> 
       <pre><code class="javascript">doc.set({ 'dc:title': 'foo' });
</code></pre> 
       <p><strong>Get a document property</strong></p> 
       <pre><code class="javascript">doc.get('dc:title');
</code></pre> 
       <p><strong>Save an updated document</strong></p> 
       <pre><code class="javascript">nuxeo.repository()
  .fetch('/')
  .then(function(doc) {
    // doc.title !== 'foo'
    doc.set({ 'dc:title': 'foo' });
    return doc.save();
  })
  .then(function(doc) {
    // doc.title === 'foo'
  })
  .catch(function(error) {
    throw error;
  });
</code></pre> 
       <p><strong>Fetch the main Blob of a document</strong></p> 
       <pre><code class="javascript">doc.fetchBlob()
  .then(function(res) {
    // in the browser, use res.blob() to work with the converted PDF
    // in Node.js, use res.body
  });
</code></pre> 
       <p><strong>Convert a document main Blob to PDF</strong></p> 
       <pre><code class="javascript">doc.convert({ format: 'pdf' })
  .then(function(res) {
    // in the browser, use res.blob() to work with the converted PDF
    // in Node.js, use res.body
  });
</code></pre> 
       <p><strong>Fetch the 'thumbnail' rendition</strong></p> 
       <pre><code class="javascript">doc.fetchRendition('thumbnail')
  .then(function(res) {
    // in the browser, use res.blob() to work with the rendition
    // in Node.js, use res.body
  });
</code></pre> 
       <p><strong>Start a workflow</strong></p> 
       <pre><code class="javascript">doc.startWorkflow('SerialDocumentReview')
  .then(function(workflow) {
    // workflow instance of Nuxeo.Workflow
  });
</code></pre> 
       <p><strong>Complete a workflow task</strong></p> 
       <pre><code class="javascript">workflow.fetchTasks()
  .then(function(tasks) {
    return tasks[0];
  })
  .then(function(tasks) {
    task.variable('participants', ['user:Administrator'])
      .variable('assignees', ['user:Administrator'])
      .variable('end_date', '2011-10-23T12:00:00.00Z');
    return task.complete('start_review', { comment: 'a comment' });
  })
  .then(function(task) {
    // task.state === 'ended'
  })
</code></pre> 
       <h3 id="batchupload">BatchUpload</h3>
       <p>The <code>BatchUpload</code> object allows you to upload blobs to a Nuxeo Platform instance, and use them as operation input or as document property value.</p> 
       <p>See the <a href="https://nuxeo.github.io/nuxeo-js-client/latest/BatchUpload.html">BatchUpload</a> documentation.</p> 
       <h4 id="samples">Samples</h4>
       <p><strong>Create a Nuxeo.Blob to be uploaded</strong></p> 
       <pre><code class="javascript">// on the browser, assuming you have a File object 'file'
var blob = new Nuxeo.Blob({ content: file });
// the name, mimeType and size will be extracted from the file object itself, you can still override them.
...
// on Node.js, assuming you have a Stream 'file'
var blob = new Nuxeo.Blob({ content: file, name: 'foo.txt', mimeType: 'plain/text', size: 10 })
</code></pre> 
       <p><strong>Upload a blob</strong></p> 
       <pre><code class="javascript">nuxeo.batchUpload()
  .upload(blob)
  .then(function(res) {
    // res.blob instanceof Nuxeo.BatchBlob
    console.log(res.blob);
  })
  .catch(function(error) {
    throw error;
  });
</code></pre> 
       <p><strong>Attach an uploaded blob to a document</strong></p> 
       <pre><code class="javascript">nuxeo.batchUpload()
  .upload(blob)
  .then(function(res) {
    return nuxeo.operation('Blob.AttachOnDocument')
      .param('document', '/a-file')
      .input(res.blob)
      .execute();
  })
  .then(function() {
    return nuxeo.repository().fetch('/a-file', { schemas: ['dublincore', 'file']});
  })
  .then(function(doc) {
    console.log(doc.get('file:content'));
  })
  .catch(function(error) {
    throw error;
  });
</code></pre>
