    <h1 class="hd01">Card Authentication Package</h1> 
    <div class="sct_bg01"> 
     <div class="cnt_max"> 
      <div class="field-collection-container clearfix"> 
       <div class="clm1"> 
        <div class="clm"> 
         <div class="thumb_txt"> 
          <div class="img_l"> 
           <img src="https://www.ricoh-ap.com/sites/www.ricoh-ap.com/files/Card_Authentication_Package_1_1.jpg" alt=""> 
          </div> 
          <div class="dtl"> 
           <p class="ttl"></p> 
           <p><span style="font-size:20px;"><strong>Card-based User Authentication and Permissions Management&nbsp;</strong></span></p> 
           <p>The Card Authentication Package improves security while making it easier and faster for users to authenticate and access networked Multifunction Products (MFPs). Issued cards can manage user privileges, such as restricting the use of color, regulate scanning and provide users with secure printing of confidential documents.</p> 
           <p>&nbsp;</p> 
           <p><strong>About the Card Authentication Package</strong></p> 
           <p>The Card Authentication Package provides an automated process for authenticating a user for access to MFP features and functionality, leveraging standard card reader technology. Installations with greater than about 500 users also require the Card Authentication Server software.</p> 
          </div>

         <h2 class="hd02">Features and Benefits</h2> 
        </div> 
        <div class="clm"> 
         <div class="thumb_txt"> 
          <div class="img_l w25p rsp_break_fl_r rsp_w100p rsp_mg_lr0 rsp_pd_tb8"> 
           <img src="https://www.ricoh-ap.com/sites/www.ricoh-ap.com/files/Card_Authentication_Package_2_0.jpg" alt=""> 
          </div> 
          <div class="dtl fl_l w70p rsp_break_fl_l rsp_w100p"> 
           <ul class="nml mt16">
            <li>MFP Access Control to eliminate unauthorized usage</li> 
            <li>Security of information assets</li> 
            <li>Reduce everyday operating costs</li> 
            <li>Improved business processes and user efficiency</li> 
            <li>Reduces output costs, for example by restricting the use of color only to authorized card holders</li> 
            <li>Improves security and workflow efficiency with scanning processes based on card credentials (when used with GlobalScan NX)</li> 
            <li>Protects confidential documents using card access to release documents for printing when sent to shared MFPs (when used with Enhanced Locked Print NX)</li> 
            <li>Establishes accountability through cost tracking tied to the identity of individual users (when used with Smart Accounting)</li> 
           </ul>
