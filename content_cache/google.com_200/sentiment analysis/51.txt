       <p><a href="http://textblob.readthedocs.org">TextBlob</a> is a new python natural language processing toolkit, which stands on the shoulders of giants like <a href="http://textminingonline.com/dive-into-nltk-part-i-getting-started-with-nltk">NLTK</a> and <a href="http://www.clips.ua.ac.be/pattern">Pattern</a>, provides <a href="http://textminingonline.com/">text mining</a>, <a href="http://textanalysisonline.com/">text analysis</a> and <a href="http://textprocessing.org/">text processing</a> modules for python developers. Here I will introduce the basics of TextBlob and show the text processing result with our demo website: <a href="http://textanalysisonline.com/">Text Analysis Online</a>.</p>

       <p><strong>About TextBlob</strong></p> 
       <p>Following is the description from the TextBlob official website:</p> 
       <blockquote>
        <p>TextBlob is a Python (2 and 3) library for processing textual data. It provides a simple API for diving into common natural language processing (NLP) tasks such as part-of-speech tagging, noun phrase extraction, sentiment analysis, classification, translation, and more.</p>
       </blockquote> 
       <p><strong>Installing TextBlob</strong></p> 
       <p>TextBlob reuses the NLTK corpora, and If you <a href="http://textminingonline.com/dive-into-nltk-part-i-getting-started-with-nltk">installed the NLTK</a> before, everything seems simple, TextBlob will use your local version instead of the bundled version. The following TextBlob install steps are tested on my mac os and ubuntu 12.04 vps, not test on the windows system, with a python version 2.7, ant TextBlob support Python &gt;= 2.6 or 3.3.</p> 
       <p>The simplest way to install TextBlob is by PyPI:</p> 
       <p>$ pip install -U textblob<br> $ python -m textblob.download_corpora</p> 
       <p>This will install TextBlob and download the necessary NLTK corpora, if you have installed NLTK before, you didn’t need to download the corpora.</p> 
       <p>Another way to install TextBlob is from the open source code by Github:</p> 
       <p>TextBlob is actively developed on <a href="https://github.com/sloria/TextBlob">Github</a>.</p> 
       <p>You can clone the public repo:</p> 
       <p>$ git clone https://github.com/sloria/TextBlob.git</p> 
       <p>Or download one of the following:</p> 
       <p>tarball<br> zipball</p> 
       <p>Once you have the source, you can install it into your site-packages with</p> 
       <p>$ python setup.py install<br> <span id="more-134"></span></p>

       <p><strong>Dive into TextBlob</strong><br> Based on NLTK, Pattern and other NLP Tools, TextBlob support following <a href="http://textprocessing.org">text processing</a> features, including:</p> 
       <blockquote>
        <p> Word Tokenization<br> Sentence Tokenization<br> Part-of-speech tagging<br> Noun phrase extraction<br> Sentiment analysis<br> Word Pluralization<br> Word Singularization<br> Spelling correction<br> Parsing<br> Classification (Naive Bayes, Decision Tree)<br> Language translation and detection powered by Google Translate<br> Word and phrase frequencies<br> n-grams<br> Word inflection (pluralization and singularization) and lemmatization<br> JSON serialization<br> Add new models or languages through extensions<br> WordNet integration </p>
       </blockquote> 
       <p>Now it’s time to introduce them one by one, first create a TextBlob:</p> 
       <p>&gt;&gt;&gt; from textblob import TextBlob<br> &gt;&gt;&gt; nlpblob = TextBlob(“Natural language processing (NLP) is a field of computer science, artificial intelligence, and linguistics concerned with the interactions between computers and human (natural) languages.”)</p> 
       <p>1) <a href="http://textanalysisonline.com/textblob-word-tokenize">Word Tokenization</a></p> 
       <p>You can tokenize TextBlobs into words:</p> 
       <p>&gt;&gt;&gt; nlpblob.words<br> WordList([‘Natural’, ‘language’, ‘processing’, ‘NLP’, ‘is’, ‘a’, ‘field’, ‘of’, ‘computer’, ‘science’, ‘artificial’, ‘intelligence’, ‘and’, ‘linguistics’, ‘concerned’, ‘with’, ‘the’, ‘interactions’, ‘between’, ‘computers’, ‘and’, ‘human’, ‘natural’, ‘languages’])</p> 
       <p>2) <a href="http://textanalysisonline.com/textblob-sentence-segmentation">Sentence Tokenization</a></p> 
       <p>TextBlob can be used to segment sentence from text paragraph:</p> 
       <p>&gt;&gt;&gt; nlpblob.sentences<br> [Sentence(“Natural language processing (NLP) is a field of computer science, artificial intelligence, and linguistics concerned with the interactions between computers and human (natural) languages.”)]</p> 
       <p>3?<a href="http://textanalysisonline.com/textblob-pos-tagging">Part-of-Speech Tagging</a></p> 
       <p>You can use TextBlob to get POS Tagging result, which can be accessed through the tags property:</p> 
       <p>&gt;&gt;&gt; nlpblob.tags<br> [(u’Natural’, u’NNP’), (u’language’, u’NN’), (u’processing’, u’NN’), (u’NLP’, u’NN’), (u’is’, u’VBZ’), (u’a’, u’DT’), (u’field’, u’NN’), (u’of’, u’IN’), (u’computer’, u’NN’), (u’science’, u’NN’), (u’artificial’, u’JJ’), (u’intelligence’, u’NN’), (u’and’, u’CC’), (u’linguistics’, u’NNS’), (u’concerned’, u’VBN’), (u’with’, u’IN’), (u’the’, u’DT’), (u’interactions’, u’NNS’), (u’between’, u’IN’), (u’computers’, u’NNS’), (u’and’, u’CC’), (u’human’, u’JJ’), (u’natural’, u’JJ’), (u’languages’, u’NNS’)]</p> 
       <p>4) <a href="http://textanalysisonline.com/textblob-noun-phrase-extraction">Noun Phrase Extraction</a></p> 
       <p>Furthermore, you can get noun phrases by accessing through the noun_phrases property in TextBlob:</p> 
       <p>&gt;&gt;&gt; nlpblob.noun_phrases<br> WordList([u’natural language processing’, ‘nlp’, u’computer science’, u’artificial intelligence’])</p> 
       <p>5) <a href="http://textanalysisonline.com/textblob-sentiment-analysis">Sentiment Analysis</a></p> 
       <p>Sentiment analysis (also known as opinion mining) refers to the use of natural language processing, text analysis and computational linguistics to identify and extract subjective information in source materials. You can get the sentiment analysis result by TextBlob:</p> 
       <p>&gt;&gt;&gt; nlpblob.sentiment<br> Sentiment(polarity=-0.1, subjectivity=0.475)<br> &gt;&gt;&gt; nlpblob.sentiment.polarity<br> -0.1<br> &gt;&gt;&gt; nlpblob.sentiment.subjectivity<br> 0.475</p> 
       <p>6) <a href="http://textanalysisonline.com/textblob-word-singularize">Word Singularize</a></p> 
       <p>&gt;&gt;&gt; nlpblob.words[19]<br> ‘computers’<br> &gt;&gt;&gt; nlpblob.words[19].singularize()<br> ‘computer’</p> 
       <p>7) <a href="http://textanalysisonline.com/textblob-word-pluralize">Word Pluralize</a></p> 
       <p>&gt;&gt;&gt; nlpblob.words[21]<br> ‘human’<br> &gt;&gt;&gt; nlpblob.words[21].pluralize()<br> ‘humen’</p> 
       <p>8) Words Lemmatization</p> 
       <p>Words can be lemmatized by the lemmatize method, but notice that the TextBlog lemmatize method is inherited from <a href="http://textanalysisonline.com/nltk-wordnet-lemmatizer">NLTK Word Lemmatizer</a>, and the default POS Tag is “n”, if you want lemmatize other pos tag words, you need specify it:</p> 
       <p>&gt;&gt; from textblob import Word<br> &gt;&gt;&gt; w = Word(“octopi”)<br> &gt;&gt;&gt; w.lemmatize()<br> u’octopus’<br> &gt;&gt;&gt; w = Word(“is”)<br> &gt;&gt;&gt; w.lemmatize()<br> ‘is’<br> &gt;&gt;&gt; w.lemmatize(“v”)<br> u’be’</p> 
       <p>9?<a href="http://textanalysisonline.com/textblob-spelling-correction">Spelling Correction</a></p> 
       <p>TextBlob Spelling correction is based on Peter Norvig’s “<a href="http://norvig.com/spell-correct.html">How to Write a Spelling Corrector</a>“, which is implemented in the pattern library:</p> 
       <p>&gt;&gt;&gt; b = TextBlob(“I havv good speling!”)<br> &gt;&gt;&gt; b.correct()<br> TextBlob(“I have good spelling!”)</p> 
       <p>Word objects also have a spellcheck() method that returns a list of (word, confidence) tuples with spelling suggestions:</p> 
       <p>TextBlob(“I have good spelling!”)<br> &gt;&gt;&gt; from textblob import Word<br> &gt;&gt;&gt; w = Word(‘havv’)<br> &gt;&gt;&gt; w.spellcheck()<br> [(u’have’, 1.0)]</p> 
       <p>9) <a href="http://textanalysisonline.com/textblob-parse">Parsing</a></p> 
       <p>TextBlob parse method is based on pattern parser:</p> 
       <p>&gt;&gt;&gt; nlpblob.parse()<br> u’Natural/NNP/B-NP/O language/NN/I-NP/O processing/NN/I-NP/O (/(/O/O NLP/NN/B-NP/O )/)/O/O is/VBZ/B-VP/O a/DT/B-NP/O field/NN/I-NP/O of/IN/B-PP/B-PNP computer/NN/B-NP/I-PNP science/NN/I-NP/I-PNP ,/,/O/O artificial/JJ/B-NP/O intelligence/NN/I-NP/O ,/,/O/O and/CC/O/O linguistics/NNS/B-NP/O concerned/VBN/B-VP/O with/IN/B-PP/B-PNP the/DT/B-NP/I-PNP interactions/NNS/I-NP/I-PNP between/IN/B-PP/B-PNP computers/NNS/B-NP/I-PNP and/CC/O/O human/JJ/B-ADJP/O (/(/O/O natural/JJ/B-ADJP/O )/)/O/O languages/NNS/B-NP/O ././O/O’</p> 
       <p>10) Translation and Language Detection</p> 
       <p>TextBlob’s translation and language detection feature are based on Google’s API:</p> 
       <p>&gt;&gt;&gt; nlpblob.translate(to=”es”)<br> TextBlob(“Procesamiento del lenguaje natural (NLP ) es un campo de la informática , la inteligencia artificial, y la lingüística que se ocupan de las interacciones entre las computadoras y los lenguajes humanos (naturales) .”)<br> &gt;&gt;&gt; nlpblob.translate(to=”zh”)<br> TextBlob(“???????NLP ???????????????????????????????????????????”)<br> &gt;&gt;&gt; nlpblob.detect_language()<br> u’en’<br> &gt;&gt;&gt; nlpblob.translate(to=”zh”).detect_language()<br> u’zh-CN’<br> &gt;&gt;&gt; </p> 
       <p>Except the features we have mentioned here, you can still find other text processing features by TextBlob, even to train a text classification model and build your own’s text classifier, just following the official guide you can find more interesting text analysis features.</p> 
       <p><strong>Using TextBlob in Other Programming Languages</strong></p> 
       <p>TextBlob is a Python text mining tool, only support the Python environment, even though, you have to install the NLTK Corpora and other dependences before you can use TextBlob, that’s a tedious thing. So we have intergrated TextBlog into our <a href="https://www.mashape.com/textanalysis/textanalysis">Text Analysis API</a> on mashape, and the details info you can find on our demo website: <a href="http://textanalysisonline.com/">Text Analysis Online</a>. Based on mashape, you can use TextBlob features in Java/JVM/Android, Node.js, PHP, Python, Objective-C/iOS, Ruby, .NET and other programming languages. You can use the <a href="https://www.mashape.com/textanalysis/textanalysis">text analysis api</a> by the free plan first, which limited 1000 requests/day. If you need more requests, just pay a small amount money, which can be used to support our web server. Mashape is the Cloude API Marketplace, all you need to do is just three steps:</p> 
       <p>1. Register a Mashape account;<br> 2. Go to the Text Analysis API page on Mashape and subscrible to it;<br> 3. Start using the TexBlob method by Text Analysis API;</p> 
       <p>That’s all, you can use TextBlob by a local version, or by our text analysis services: <a href="http://textanalysisonline.com/">Text Analysis Online</a>.</p>
