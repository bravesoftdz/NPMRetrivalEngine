        <h1 class="entry-title">Sentiment Analysis on Twitter with Datumbox API</h1> 
        <div class="entry-meta">
         <div class="meta-item meta-author">
          <span class="vcard author"><span class="fn"><a href="http://thinktostart.com/author/julianhi/"><img alt="" src="http://1.gravatar.com/avatar/163f781891344aba2cd57f3907b19122?s=24&amp;d=mm&amp;r=g" class="avatar avatar-24 photo" height="24" width="24"> Julian Hillebrand</a></span></span>
         </div>
         <div class="meta-item meta-date">
          <span class="updated">September 9, 2013</span>
         </div>
         <div class="meta-item meta-comments">
          <a href="http://thinktostart.com/sentiment-analysis-on-twitter-with-datumbox-api/#comments"><span class="dsq-postid">11 comments</span></a>
         </div>
         <div class="meta-item meta-rtime">
          5 min read
         </div>
        </div> 
       </div> 
       <div class="entry-content"> 
        <p>Hey there!</p> 
        <p><a href="http://www.datumbox.com/"><img class="alignnone size-full wp-image-364" src="http://thinktostart.files.wordpress.com/2013/09/bildschirmfoto-2013-09-07-um-13-38-58.png" alt="Datumbox" width="239" height="64"></a></p> 
        <p>After my post about sentiment analysis using the <a href="http://thinktostart.wordpress.com/2013/09/02/sentiment-analysis-on-twitter-with-viralheat-api/">Viralheat API</a> I found another service. Datumbox ist offering special sentiment analysis for Twitter. But this API doesn´t just offer sentiment analysis, it offers a much more detailed analysis. „The currently supported API functions are: Sentiment Analysis, Twitter Sentiment Analysis, Subjectivity Analysis, Topic Classification, Spam Detection, Adult Content Detection, Readability Assessment, Language Detection, Commercial Detection, Educational Detection, Gender Detection, Keyword Extraction, Text Extraction and Document Similarity.“</p> 
        <p><strong>But note:</strong><br> <em>Datumbox just offers Sentiment analysis for tweets. All the other classifiers like gender or topic are build for longer texts and not for short tweets as they have too less chars. So the results for tweets can be inaccurately.</em></p> 
        <p>But these are very interesting features and so I wanted to test them with R.</p> 
        <p>But before we start you should take a look at the <a href="http://thinktostart.wordpress.com/2013/05/22/twitter-authentification-with-r/">authentication tutorial</a> and go through the steps.</p> 
        <h1>The API Key</h1> 
        <p>In the first step you need an API key. So go to the Datumbox website <a href="http://www.datumbox.com/">http://www.datumbox.com/</a> and register yourself. After you have logged in you can see your free API key here: <a href="http://www.datumbox.com/apikeys/view/">http://www.datumbox.com/apikeys/view/</a></p> 
        <p><a href="http://www.datumbox.com/"><img class="alignnone size-large wp-image-362" src="http://thinktostart.files.wordpress.com/2013/09/bildschirmfoto-2013-09-07-um-11-15-25.png?w=388" alt="Datumbox" width="388" height="134" srcset="http://thinktostart.com/wp-content/uploads/2013/09/bildschirmfoto-2013-09-07-um-11-15-25.png 1095w, http://thinktostart.com/wp-content/uploads/2013/09/bildschirmfoto-2013-09-07-um-11-15-25-300x104.png 300w, http://thinktostart.com/wp-content/uploads/2013/09/bildschirmfoto-2013-09-07-um-11-15-25-1024x355.png 1024w, http://thinktostart.com/wp-content/uploads/2013/09/bildschirmfoto-2013-09-07-um-11-15-25-696x242.png 696w, http://thinktostart.com/wp-content/uploads/2013/09/bildschirmfoto-2013-09-07-um-11-15-25-1068x371.png 1068w" sizes="(max-width: 388px) 100vw, 388px"></a></p> 
        <p>Ok, let´s go on with R.</p> 
        <h1><b>Functions</b></h1> 
        <h2><b>The getSentiment() function</b></h2>

getSentiment &lt;- function (text, key){

text &lt;- URLencode(text);

#save all the spaces, then get rid of the weird characters that break the API, then convert back the URL-encoded spaces.
text &lt;- str_replace_all(text, "%20", " ");
text &lt;- str_replace_all(text, "%\d\d", "");
text &lt;- str_replace_all(text, " ", "%20");

if (str_length(text) &gt; 360){
text &lt;- substr(text, 0, 359);
}
##########################################

data &lt;- getURL(paste("http://api.datumbox.com/1.0/TwitterSentimentAnalysis.json?api_key=", key, "&amp;text=",text, sep=""))

js &lt;- fromJSON(data, asText=TRUE);

# get mood probability
sentiment = js$output$result

###################################

data &lt;- getURL(paste("http://api.datumbox.com/1.0/SubjectivityAnalysis.json?api_key=", key, "&amp;text=",text, sep=""))

js &lt;- fromJSON(data, asText=TRUE);

# get mood probability
subject = js$output$result

##################################

data &lt;- getURL(paste("http://api.datumbox.com/1.0/TopicClassification.json?api_key=", key, "&amp;text=",text, sep=""))

js &lt;- fromJSON(data, asText=TRUE);

# get mood probability
topic = js$output$result

##################################
data &lt;- getURL(paste("http://api.datumbox.com/1.0/GenderDetection.json?api_key=", key, "&amp;text=",text, sep=""))

js &lt;- fromJSON(data, asText=TRUE);

# get mood probability
gender = js$output$result

return(list(sentiment=sentiment,subject=subject,topic=topic,gender=gender))
}</textarea>
