     <h1 class="text-center singlepost_title_singleview ">List of 18 APIs to Protect your Site</h1> 
     <h2 class="text-center">by <span class="singlepost_meta_highlight"><a href="http://blog.mashape.com?s=&amp;author_name=chrisismael">Chris Ismael</a></span> on August 23, 2013</h2> 
    </div> 
   </div> 
   <div class="col-md-8 col-md-offset-2 singlepost_post_content"> 
    <p class="p1"><a href="http://i0.wp.com/blog.mashape.com/wp-content/uploads/2013/08/tumblr_inline_mry78ljndB1qz4rgp.jpg"><img class="alignright wp-image-1238 size-medium" src="http://i0.wp.com/blog.mashape.com/wp-content/uploads/2013/08/tumblr_inline_mry78ljndB1qz4rgp.jpg?resize=300%2C225" alt="Nudity Filter API" srcset="http://i0.wp.com/blog.mashape.com/wp-content/uploads/2013/08/tumblr_inline_mry78ljndB1qz4rgp.jpg?resize=300%2C225 300w, http://blog.mashape.com/wp-content/uploads/2013/08/tumblr_inline_mry78ljndB1qz4rgp.jpg 500w" sizes="(max-width: 300px) 100vw, 300px" data-recalc-dims="1"></a></p> 
    <p class="p1">Whether its inappropriate content, spam, or fraud prevention  protecting your site should be a priority. &nbsp;Here are some APIs to help you strengthen the fort.</p> 
    <p class="p1">
     <!-- more --></p> 
    <ol> 
     <li><strong><a title="ImageVision Nudity Search API" href="https://www.mashape.com/imagevision/imagevision-nuditysearch#!documentation" target="_blank">ImageVision Nudity Search</a></strong> &nbsp;ImageVisions NuditySearch  Recognizing nudity is a highly complex problem. NuditySearch tackles this problem by recognizing anatomical attributes and determining if there is nudity in images.</li> 
     <li><strong><span class="s1"><a title="Spamhaus and Surbl REST interface API" href="https://www.mashape.com/flagz/spamhaus-and-surbl-rest-interface#!documentation" target="_blank">Spamhaus and Surbl REST interface</a></span></strong>&nbsp;&nbsp;Rest api to verify if an url has been marked as spam using the well known spamhaus and surbl services. Supports: SpamHaus zen (SBL e-mail spam, XBL infected PCs, PBL unsolicited smtp traffic) Surbl multi (Spamcop, sa-blacklist, Outblaze, Abusebutler, Phishtank, Zeus tracker, jwSpamSpy, Prolocation)</li> 
     <li><strong><span class="s1"><a title="C24 Validation Service API" href="https://www.mashape.com/c24-biz/c24-validation-service#!documentation" target="_blank">C24 Validation Service</a></span></strong>&nbsp;&nbsp;With this API you can validate SWIFT message. Specify the appropriate SWIFT Standard to use and the message to validate.</li> 
     <li><strong><span class="s1"><a title="Nudity Detection Service API" href="https://www.mashape.com/rohan/nudity-detection-service#!documentation" target="_blank">Nudity Detection Service</a></span></strong>&nbsp;&nbsp;The NDS API allows developers to check for nudity in images in real-time.</li> 
     <li><strong><span class="s1"><a title="Password Tools API" href="https://www.mashape.com/pozzad/password-tools#!documentation" target="_blank">Password Tools</a></span></strong> &nbsp;Provides a set of API for generating strong passwords, checking a password strength, generating a password hash and verifying it.</li> 
     <li><strong><span class="s1"><a title="Email Validator API" href="https://www.mashape.com/pozzad/email-validator-1#!documentation" target="_blank">Email Validator</a></span></strong> &nbsp;Checks for fake DNS as well as uses regex functions to check the email for the right length and accepted characters.</li> 
     <li><strong><span class="s1"><a title="Anti-Spam Captcha and Bot-IP blocker API" href="https://www.mashape.com/verswerks/anti-spam-captcha-and-bot-ip-blocker-verscaptcha#!documentation" target="_blank">Anti-Spam Captcha and Bot-IP blocker</a></span></strong> &nbsp;Easy anti-spam measure which takes it one-step further by blocking potentially bad IP address from using your resources.</li> 
     <li><strong><span class="s1"><a title="Porn Filter API" href="https://www.mashape.com/nsure-io/porn-filter#!documentation" target="_blank">Porn Filter</a></span></strong> &nbsp;Scan images and image URLs to determine if they contain inappropriate content. Send us an image, and we will tell you if its likely to be porn.</li> 
     <li><strong><span class="s1"><a title="Virus Filter API" href="https://www.mashape.com/nsure-io/virus-filter#!documentation" target="_blank">Virus Filter</a></span></strong> &nbsp;You send us a file or a URL of a file, and we will tell you if it contains a virus.</li> 
     <li><strong><span class="s1"><a title="Profanity API" href="https://www.mashape.com/nsure-io/profanity#!documentation" target="_blank">Profanity</a></span></strong> &nbsp;From fuster clucks to 1337 speak, keeping track of profanity and blocking it in your app can be a real nightmare. The nsurePROFANITY API puts a stop to the f-bomb.</li> 
     <li><strong><span class="s1"><a title="One Time Password Service API" href="https://www.mashape.com/tanmaykm/one-time-password-service#!documentation" target="_blank">One Time Password Service</a></span></strong> &nbsp;Generates and validates numeric one time passwords. One time passwords are valid only for a certain time and for one use only. Password is deleted once an attempt is made to validate it, whether the attempt was successful or otherwise.</li> 
     <li><strong><span class="s1"><a title="IDlight API" href="https://www.mashape.com/ddossot/idlight#!documentation" target="_blank">IDlight</a></span> </strong>&nbsp;Public identity lookup service. IDlight uses established and emerging standards like Webfinger, XRD and hCard to retrieve public profiles and render them under a unified format, either proprietary to IDlight or compliant to Portable Contacts.</li> 
     <li><strong><span class="s1"><a title="Block Disposable Email API" href="https://www.mashape.com/bdea-cc/block-disposable-email#!documentation" target="_blank">Block Disposable Email</a></span></strong> &nbsp;The bdea.cc api helps to detect domains of disposable email address services such as mailinator.com, spamgourmet. In other words: if you need to keep your userbase clean you should avoid throw-away, one-time, instant, temporary email addresses.</li> 
     <li><strong><span class="s1"><a title="Fraud Prevention Web Service API" href="https://www.mashape.com/chrislim/fraud-prevention-web-service#!documentation" target="_blank">Fraud Prevention Web Service</a></span></strong> &nbsp;This API help you to screen an order to detect the degree of credit card fraud, and therefore protect your business order from losses due to chargebacks.</li> 
     <li><strong><span class="s1"><a title="Fraud Predictor API" href="https://www.mashape.com/algorithms-io/fraud-predictor#!documentation" target="_blank">Fraud Predictor</a></span></strong> &nbsp;Identify potentially fraudelent transactions in a series. Can be used for any transactional data sets including financial transactions, log data and clickstream data.</li> 
     <li><strong><span class="s1"><a title="strongpassword API" href="https://www.mashape.com/tabalt/strongpassword#!documentation" target="_blank">strongpassword</a></span></strong> &nbsp;this api can give you a strong password to make you app more safety.</li> 
     <li><strong><span class="s1"><a title="Nudity and adult-content detection API" href="https://www.mashape.com/sightengine/nudity-and-adult-content-detection#!documentation" target="_blank">Nudity and adult-content detection</a></span></strong> &nbsp;Detect nudity and porn in user-submitted photos in real-time. Slash your moderation costs. Get a response in a matter of seconds with a confidence score.</li> 
     <li><strong><span class="s1"><a title="Profanitor API" href="http://www.mashape.com/macropoetry/profanitor-profanity-detection" target="_blank">Profanitor</a></span></strong> &nbsp;Profanity detection and analysis API. Scan, analyze, and remove profanity from text.</li> 
    </ol> 
    <p class="p1">You should also check out our other useful API lists for <a title="List of 50+ Machine Learning APIs" href="http://blog.mashape.com/list-of-50-machine-learning-apis/" target="_blank">machine learning</a>, <a title="List of 30 Natural Language Processing APIs" href="http://blog.mashape.com/list-of-30-natural-language-processing-apis/" target="_blank">natural language processing</a>,&nbsp;<a title="List of 25 Sentiment Analysis APIs" href="http://blog.mashape.com/list-of-25-sentiment-analysis-apis/" target="_blank">sentiment analysis</a>, <a title="List of 60 SMS APIs" href="http://blog.mashape.com/list-of-60-sms-apis/" target="_blank">SMS APIs</a>, <a title="List of 30+ Summarizer APIs, libraries, and software" href="http://blog.mashape.com/list-of-30-summarizer-apis-libraries-and-software/" target="_blank">summarization APIs</a> and face <a title="List of 50+ Face Detection / Recognition APIs, libraries, and software" href="http://blog.mashape.com/list-of-50-face-detection-recognition-apis/" target="_blank"><span class="s1">recognition APIs</span></a>.</p>
