      <h1 id="nodejs-helper">Nodejs helper</h1> 
      <p>The Plivo Node helper simplifies the process of making Plivo API Calls and generating Plivo XML.</p> 
      <h2 id="prerequisites">Prerequisites</h2> 
      <ol class="bulleted"> 
       <li><a href="https://nodejs.org/en/">Node.js</a></li> 
       <li><a href="https://www.npmjs.com/">NPM</a></li> 
      </ol> 
      <h2 id="installation">Installation</h2> 
      <p>Run the following command for a quick installation of the Node helper.<br> <code>npm install plivo</code></p> 
      <h2 id="using-the-library">Using the library</h2> 
      <p>Getting started with the Plivo API couldn’t be easier.</p> 
      <h3 id="initializing-plivorestapi">Initializing PlivoRestApi</h3> 
      <div class="highlight">
       <pre><code class="language-bash" data-lang="bash">var <span class="nv">plivo</span> <span class="o">=</span> require<span class="o">(</span><span class="s1">'plivo'</span><span class="o">)</span><span class="p">;</span>
var <span class="nv">p</span> <span class="o">=</span> plivo.RestAPI<span class="o">({</span>
  authId: <span class="s1">'Your AUTH_ID'</span>,
  authToken: <span class="s1">'Your AUTH_TOKEN'</span>
<span class="o">})</span><span class="p">;</span></code></pre>
      </div> 
      <h3 id="send-an-sms">Send an SMS</h3> 
      <p>The snippet below shows how you can send an sms.</p> 
      <div class="highlight">
       <pre><code class="language-bash" data-lang="bash">Map&lt;String, String&gt; <span class="nv">params</span> <span class="o">=</span> new Map&lt;String, String&gt; <span class="o">()</span><span class="p">;</span>
var <span class="nv">params</span> <span class="o">=</span> <span class="o">{</span>
    <span class="s1">'src'</span>: <span class="s1">'1111111111'</span>,
    <span class="s1">'dst'</span> : <span class="s1">'2222222222'</span>,
    <span class="s1">'text'</span> : <span class="s2">"Hello, how are you?"</span>
<span class="o">}</span><span class="p">;</span>
p.send_message<span class="o">(</span>params, <span class="k">function</span> <span class="o">(</span>status, response<span class="o">)</span> <span class="o">{</span>
    console.log<span class="o">(</span><span class="s1">'Status: '</span>, status<span class="o">)</span><span class="p">;</span>
    console.log<span class="o">(</span><span class="s1">'API Response:\n'</span>, response<span class="o">)</span><span class="p">;</span>
<span class="o">})</span><span class="p">;</span></code></pre>
      </div> 
      <p>Check out this <a href="https://www.plivo.com/docs/getting-started/send-a-single-sms/">Getting Started</a> page or the full <a href="https://www.plivo.com/docs/api/message/">Message API</a> documentation for more details.</p> 
      <h3 id="make-an-outbound-phone-call">Make an Outbound Phone Call</h3> 
      <p>You can make outbound phone calls to landlines, mobiles, and SIP endpoints (e.g., softphones), in any of our <a href="https://www.plivo.com/international-coverage">200+ coverage countries</a></p> 
      <div class="highlight">
       <pre><code class="language-bash" data-lang="bash">var <span class="nv">params</span> <span class="o">=</span> <span class="o">{</span>
    <span class="s1">'to'</span>: <span class="s1">'2222222222'</span>,
    <span class="s1">'from'</span> : <span class="s1">'1111111111'</span>,
    <span class="s1">'answer_url'</span> : <span class="s2">"https://s3.amazonaws.com/static.plivo.com/answer.xml"</span>,
    <span class="s1">'answer_method'</span> : <span class="s2">"GET"</span>
<span class="o">}</span><span class="p">;</span>
p.make_call<span class="o">(</span>params, <span class="k">function</span> <span class="o">(</span>status, response<span class="o">)</span> <span class="o">{</span>
    console.log<span class="o">(</span><span class="s1">'Status: '</span>, status<span class="o">)</span><span class="p">;</span>
    console.log<span class="o">(</span><span class="s1">'API Response:\n'</span>, response<span class="o">)</span><span class="p">;</span>
<span class="o">})</span><span class="p">;</span></code></pre>
      </div> 
      <p>Check out this <a href="https://www.plivo.com/docs/getting-started/making-outbound-calls/">Getting Started</a> page or the full <a href="https://www.plivo.com/docs/api/call/">Call API</a> documentation for more details.</p>
