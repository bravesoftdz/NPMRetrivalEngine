        <h2>Aspose.BarCode for Cloud</h2> 
        <p><a href="http://www.aspose.com/cloud/barcode-api.aspx"><img class="alignleft size-full wp-image-20830" title="Aspose.BarCode for Cloud" src="https://blog.aspose.com/wp-content/uploads/sites/2/2014/12/aspose_barcode_for_cloud.png" alt="Aspose.BarCode for Cloud" width="140" height="140"></a>Aspose.BarCode for Cloud’s platform independent document manipulation API is a true REST API that can be used with any language: .NET, Java, PHP, Ruby, Rails, Python, Node.js and many more. You can use it with any platform — web, desktop, mobile, and cloud.<br><br>Aspose.BarCode for Cloud allows developers to quickly and easily add barcode generation and recognition functionality to their applications. Aspose.BarCode for Cloud is a complete solution for barcode generation and recognition.&nbsp;It helps us to generate barcode images from scratch in linear (1D), two dimensional (2D), and postal formats. Generate barcode images in a variety of image formats: JPEG, PNG, GIF, BMP, TIFF and many others. Recognize barcodes from different image types&nbsp;<a href="http://www.aspose.com/cloud/barcode-api.aspx">Learn More</a></p> 
        <p><a href="http://www.aspose.com/cloud/barcode-api.aspx"><img class="aligncenter size-full wp-image-27629" title="Aspose.Barcode for Cloud" src="https://blog.aspose.com/wp-content/uploads/sites/2/2016/01/BarcodeCloud-Format-Support-2.png" alt="Aspose.Barcode for Cloud" width="600" height="256" srcset="https://blog.aspose.com/wp-content/uploads/sites/2/2016/01/BarcodeCloud-Format-Support-2.png 600w, https://blog.aspose.com/wp-content/uploads/sites/2/2016/01/BarcodeCloud-Format-Support-2-300x128.png 300w" sizes="(max-width: 600px) 100vw, 600px"></a></p> 
        <p><span style="font-size: 1.8em; line-height: 1.5em;">What’s new in the latest release</span></p> 
        <p>This is a major revamp release for Aspose.BarCode Cloud SDK for Node.js. It covers all the&nbsp;<a href="http://www.aspose.com/docs/display/barcodecloud/Home" target="_blank">Aspose.BarCode Cloud APIs</a>. This SDK&nbsp;provides a complete solution for barcode generation and recognition in Node.js using powerful <a href="http://www.aspose.com/cloud/barcode-api.aspx" target="_blank">Aspose.BarCode Cloud APIs</a>, so you can perform barcode generation and recognition operations in the cloud seamlessly.</p> 
        <p><span style="font-size: 1.8em; line-height: 1.5em;">Source Code Repository</span></p> 
        <p>Complete source code of this release of Aspose.Barcode Cloud SDK for Node.js&nbsp;is freely&nbsp;available on your favorite <a href="https://github.com/asposebarcode/Aspose_BarCode_Cloud/tree/master/SDKs/Aspose.BarCode_Cloud_SDK_for_NodeJS/" target="_blank" class="broken_link">GitHub</a> and <a href="https://asposebarcodecloud.codeplex.com/SourceControl/latest#SDKs/Aspose.BarCode_Cloud_SDK_for_NodeJS/" target="_blank">Codeplex</a> repositories.</p> 
        <ul> 
         <li><a href="https://github.com/asposebarcode/Aspose_BarCode_Cloud/releases/tag/Aspose.Barcode-Cloud-SDK-For-NodeJS-v1.0.0" target="_blank">Aspose.BarCode Cloud SDK for Node.js – v1.0.0 (GitHub)</a></li> 
         <li><a href="https://asposebarcodecloud.codeplex.com/releases/view/619590" target="_blank">Aspose.BarCode Cloud SDK for Node.js – v1.0.0 (CodePlex)</a></li> 
        </ul> 
        <p><span style="font-size: 1.8em; line-height: 1.5em;">Setup</span></p> 
        <p>Aspose.BarCode Cloud SDK for Node.js&nbsp;is also available as a released package in the&nbsp;<a href="https://www.npmjs.com/package/asposebarcodecloud/">npm (package manager for javascript) </a>. You can bypass <a href="https://github.com/asposebarcode/Aspose_BarCode_Cloud/tree/master/SDKs/Aspose.BarCode_Cloud_SDK_for_NodeJS/" target="_blank" class="broken_link">GitHub source code repository</a> and depend directly on the released package by installing from npm:</p> 
        <p><strong>Aspose.BarCode Cloud SDK for Node.js – npm installation</strong></p> 
        <pre><code>
npm i asposebarcodecloud
</code></pre> 
        <p><span style="font-size: 1.8em; line-height: 1.5em;">Data</span></p> 
        <p>In order to manipulate any files, you first need to upload them to the Aspose cloud storage or 3rd party cloud storage using&nbsp;<a href="https://www.npmjs.com/package/asposestoragecloud">Aspose.Storage Cloud SDK for Node.js</a>.</p>
