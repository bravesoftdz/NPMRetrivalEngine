  <p>ZBar is an open source software suite for reading bar codes from various sources, such as video streams, image files and raw intensity sensors. It supports many popular <em>symbologies</em> (types of bar codes) including EAN-13/UPC-A, UPC-E, EAN-8, Code 128, Code 39, Interleaved 2 of 5 and QR Code.</p> 
  <p>The flexible, layered implementation facilitates bar code scanning and decoding for any application: use it stand-alone with the included GUI and command line programs, easily integrate a bar code scanning widget into your Qt, GTK+ or PyGTK GUI application, leverage one of the script or programming interfaces (Python, Perl, C++) ...all the way down to a streamlined C library suitable for embedded use.</p> 
  <p>ZBar is licensed under the <a href="http://www.gnu.org/licenses/old-licenses/lgpl-2.1.html">GNU LGPL 2.1</a> to enable development of both open source and commercial projects.</p> 
  <h2>Applications</h2> 
  <ul> 
   <li>Retail</li> 
   <li>Automated document processing</li> 
   <li>Inventory tracking</li> 
   <li>Mobile applications</li> 
  </ul> 
  <h2>Features</h2> 
  <ul> 
   <li>cross platform - Linux/Unix, Windows, <a href="iphone/index.html">iPhone<sup>®</sup></a>, embedded...</li> 
   <li>high speed - <em>real time</em> scanning from video streams</li> 
   <li>small memory footprint</li> 
   <li>small code size - the core scanner and EAN decoder represent under 1K lines of C code</li> 
   <li>not limited to images</li> 
   <li>no floating point operations</li> 
   <li>suitable for embedded applications using inexpensive processors/hardware</li> 
   <li>modular components can be used together or separately</li> 
  </ul>
