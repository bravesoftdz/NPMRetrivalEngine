    <h1 align="center">MBS FileMaker Plugin 7.3</h1>

    <div style="width: 210px; float: right; margin: 3pt; padding: 3pt; background-color: #EEF; border-width: 3pt;"> 
     <b>FileMaker 8 to 16 supported</b> 
     <br>
     <br> MBS Plugin 6.0 or newer for FileMaker 15.
     <br>
     <br> MBS Plugin 7.2 or newer for FileMaker 16 and iOS SDK.
     <br>
     <br> 
     <b>FileMaker Cloud supported</b> 
     <br>
     <br> MBS Plugins 6.5 and newer support FileMaker Cloud. 
    </div> 
    <p>The MBS Plugin is easily the most powerful plug-in available for FileMaker today. There are over 4,600 functions that allow FileMaker developers to extend their solutions in some amazing ways. Each function does a little task and in their combination you can unleash the full power of our plugin. To learn how to use them together, please check our reference and our more than 450 example databases. A quick overview:</p> 
    <ul> 
     <li>Enjoy <a href="http://www.monkeybreadsoftware.de/filemaker/SyntaxColoring/">Syntax Coloring</a> in scripts and formulas. Copy scripts with formatted text. (Mac only)</li> 
     <li>Compress files, data or text to <a href="http://www.mbsplugins.eu/component_ZipFile.shtml">zip archives</a> and <a href="http://www.mbsplugins.eu/component_UnZipFile.shtml">uncompress</a> them later.</li> 
     <li>Use the <a href="http://www.mbsplugins.eu/component_TAPI.shtml">telephone API (TAPI)</a> on Windows.</li> 
     <li><a href="http://www.mbsplugins.eu/SerialPortList.shtml">List</a> and <a href="http://www.mbsplugins.eu/SerialPortOpen.shtml">connect</a> to <a href="http://www.mbsplugins.eu/component_SerialPort.shtml">serial ports</a>.</li> 
     <li>Read <a href="http://www.mbsplugins.eu/component_RichText.shtml">rich text documents</a> on Mac OS X like Word documents, OpenOffice documents or RTF files.</li> 
     <li>Enumerate <a href="http://www.mbsplugins.eu/component_Printer.shtml">printers</a> and get/set default printer.</li> 
     <li>Recognize text in images using <b><a href="http://www.mbsplugins.eu/component_OCR.shtml">OCR</a></b> functions. </li> 
     <li><a href="https://www.mbsplugins.eu/component_PortMidi.shtml">Receive and send Midi notes</a>.</li> 
     <li><b><a href="http://www.mbsplugins.eu/component_Barcode.shtml">Barcodes</a></b> 
      <ul> 
       <li><a href="http://www.mbsplugins.eu/BarcodeGenerate.shtml">Create Barcodes</a> and <a href="http://www.mbsplugins.eu/BarcodeDrawBarcode.shtml">draw them on PDF pages</a>. <a href="http://www.mbsplugins.eu/BarcodeWriteFile.shtml">Export barcodes to files</a>.</li> 
       <li><a href="http://www.mbsplugins.eu/BarcodeDetect.shtml">Recognize barcodes</a></li> 
      </ul> </li> 
     <li><b>Audio/Video</b> 
      <ul> 
       <li>Export videos in new format with <a href="http://www.mbsplugins.eu/component_AVExport.shtml">AVExport</a> or <a href="http://www.mbsplugins.eu/component_QTExport.shtml">QTExport</a>.</li> 
       <li>Record on Mac and Windows with <a href="http://www.mbsplugins.eu/component_AVRecorder.shtml">AVRecorder</a>.</li> 
       <li>Use <a href="http://www.mbsplugins.eu/component_AVAsset.shtml">AVAsset</a> or <a href="http://www.mbsplugins.eu/component_QTMovie.shtml">QTMovie</a> functions to extract metadata from videos, get preview images or play audio in background.</li> 
      </ul> </li> 
     <li><a href="http://www.mbsplugins.eu/component_XL.shtml">XL Functions</a>. 
      <ul> 
       <li>Work with <b>Excel</b> files without Microsoft Excel by using <a href="http://www.bluesnap.com/jsp/redirect.jsp?contractId=2284940&amp;referrer=983554">libXL library</a> in FileMaker. </li>
       <li>Read and Write older xls files and newer xlsx files.</li> 
      </ul> </li> 
     <li>Use <b><a href="http://www.mbsplugins.eu/component_Audit.shtml">Audit</a></b> to log activities in a database. </li> 
     <li><b><a href="http://www.mbsplugins.eu/component_SQL.shtml">SQL Connectivity</a></b> 
      <ul> 
       <li>Connect to multiple SQL databases: Oracle, SQL Server, DB2, Sybase, SQL Anywhere, Informix, InterBase, SQLBase, MySQL, PostgreSQL, ODBC and SQLite (with encryption).</li> 
       <li>Read values, changes values, add new records, call stored procedures.</li> 
       <li>Use prepared statements for increased security.</li> 
       <li>Use bulk row fetching for maximum speed.</li> 
       <li>Read and write blob values with images.</li> 
      </ul> </li> 
     <li><b><a href="http://www.mbsplugins.eu/component_CURL.shtml">CURL Library</a></b> 
      <ul> 
       <li>Download and <a href="http://www.mbsplugins.eu/CURLSetOptionUpload.shtml">upload</a> files or data from/to http/https/ftp/ftps/sftp.</li> 
       <li><a href="http://www.mbsplugins.eu/component_SendMail.shtml">Compose emails</a>. Send emails over smtp and receive emails over POP3 or IMAP.</li> 
       <li>Perform several transfers simultaneously in <a href="http://www.mbsplugins.eu/CURLPerformInBackground.shtml">background</a>.</li> 
       <li>Send <a href="http://www.mbsplugins.eu/CURLFormAddKeyValue.shtml">form data</a> directly without Webviewer.</li> 
       <li>Parse and build JavaScript Object Notation encoded data with out <a href="http://www.mbsplugins.eu/component_JSON.shtml">JSON functions</a>.</li> 
      </ul> </li> 
     <li><b>Network Sockets</b> 
      <ul> 
       <li>Control other computers via <a href="http://www.mbsplugins.eu/component_SSH.shtml">SSH connection</a>. (new in 6.3)</li> 
       <li>Use TCP/UDP <a href="http://www.mbsplugins.eu/component_Socket.shtml">Sockets</a>, optionally with SSL.</li> 
       <li>Query other FileMaker instance with <a href="http://www.mbsplugins.eu/component_ServerSocket.shtml">ServerSocket</a> functions.</li>

    <h2>System requirements</h2> 
    <p>The Plugin requires Mac OS X 10.6 or Windows XP or newer.</p> 
    <p>The Linux plugin works on 64-bit Linux Servers including those used for FileMaker Cloud.</p> 
    <p>We support FileMaker 11 to 16 and test those regularly. </p>
    <p>FileMaker 7.0 and newer should work on Windows and 8.5 and newer on Mac OS X. </p>
    <p>Last version with PowerPC support was version 2.4. This version also supported FileMaker 7 to 8.5 in PPC.</p> 
    <p>The plugin runs locally on a FileMaker installation or on a server installation. If you use Instant Web Publishing or FileMaker to Go, the plugin still needs to be run on the server, but results can be viewer on browser or iOS devices.</p>
