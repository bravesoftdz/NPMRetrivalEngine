        <h1>Project Description</h1> A library which supports decoding and generating of barcodes (like QR Code, PDF 417, EAN, UPC, Aztec, Data Matrix, Codabar) within images.
        <br> 
        <br> The project is a port of the java based barcode reader and generator library ZXing.
        <br> 
        <a href="https://github.com/zxing/zxing">https://github.com/zxing/zxing</a>
        <br> It has been ported by hand with a lot of optimizations and improvements.
        <br> 
        <br> There is a mirror of the repository at github:
        <br> 
        <a href="https://github.com/micjahn/ZXing.Net">https://github.com/micjahn/ZXing.Net</a>
        <br> 
        <br> The following barcodes are supported by the decoder:
        <br> UPC-A, UPC-E, EAN-8, EAN-13, Code 39, Code 93, Code 128, ITF, Codabar, MSI, RSS-14 (all variants), QR Code, Data Matrix, Aztec and PDF-417.
        <br> The encoder supports the following formats:
        <br> UPC-A, EAN-8, EAN-13, Code 39, Code 128, ITF, Codabar, Plessey, MSI, QR Code, PDF-417, Aztec, Data Matrix
        <br> 
        <br> 
        <b>Assemblies are available for the following platforms:</b> 
        <ul> 
         <li>.Net 2.0, 3.5, 4.0, 4.5 and 4.6 </li>
         <li>Silverlight 4 and 5 </li>
         <li>Windows Phone 7.0, 7.1 and 8.0 </li>
         <li>Windows CE </li>
         <li>Windows RT Class Library and Runtime Components (winmd) </li>
         <li>Portable Class Library </li>
         <li>Unity3D (.Net 2.0 built without System.Drawing reference) </li>
         <li>Xamarin.Android (formerly Mono for Android) </li>
         <li>.Net Standard 1.6</li>
        </ul> The library is available in the download section and as 
        <a href="http://nuget.org/packages/ZXing.Net"> NuGet package</a>, too.
        <br> 
        <br> 
        <b>Additional platform support without pre-built binaries</b>
        <br> The library can be built for Xamarin.iOS (formerly MonoTouch). The project file and solution is available in the 
        <a href="https://zxingnet.svn.codeplex.com/svn/trunk">source code repository</a>.
        <br> 
        <br> A special version for the 
        <a href="http://www.microsoft.com/netmf/">.Net Micro Framework</a> can be found in a separate branch in the 
        <a href="https://zxingnet.svn.codeplex.com/svn/branches/NETMF">source code repository</a>.
        <br> 
        <br> 
        <b>The following demo clients are available:</b> 
        <ul> 
         <li>decoder for the command line </li>
         <li>encoder for the command line </li>
         <li>Windows Forms demo (demonstrates decoding and encoding of static images and from a camera) </li>
         <li>Windows Phone demo (demonstrates decoding of static images and from a camera) </li>
         <li>Windows Service demo (demonstrates decoding of static images) </li>
         <li>Windows Presentation Framework demo (demonstrates decoding of static images) </li>
         <li>Windows CE demo (demonstrates decoding of static images) </li>
         <li>Windows RT demo (demonstrates decoding of static images) </li>
         <li>Windows Store App with HTML5/JS (demonstrates decoding of static images) </li>
         <li>Unity3D and Vuforia demo (demonstrates encoding of barcodes and decoding of images from a camera with <a href="http://unity3d.com/">Unity3D</a>) </li>
         <li>Silverlight demo (demonstrates decoding and encoding of static images) </li>
         <li>EmguCV demo (demonstrates decoding of images from a camera and uses the <a href="http://www.emgu.com/"> EmguCV framework</a>) </li>
         <li>AForge demo (demonstrates decoding of images from a camera and uses the <a href="http://www.aforgenet.com/"> AForge framework</a>)</li>
        </ul> 
        <h1>Thanks</h1> Many thanks to the team of the 
        <a href="https://github.com/zxing/zxing">zxing project</a> for their great work. ZXing.Net would not be possible without your work!
        <br> 
        <h1>Usage examples</h1> The source code repository includes small examples for Windows Forms, Silverlight and Windows Phone.
        <br> 
        <br> 
        <b>small example decoding a barcode inside a bitmap (.Net 2.0/3.5/4.0)</b>
        <br>
