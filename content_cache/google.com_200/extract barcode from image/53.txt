      <h1 id="post-1632"><a href="https://www.hacksparrow.com/base64-encoding-decoding-in-node-js.html" rel="bookmark" title="Permanent Link: Base64 Encoding / Decoding in Node.js">Base64 Encoding / Decoding in Node.js</a></h1> 
      <div class="post-meta"> 
       <i>Posted on April 20th, 2012 under <a href="https://www.hacksparrow.com/category/node-js-2" rel="category tag">Node.js</a></i>
       <br> 
       <i>Tags: <a href="https://www.hacksparrow.com/tag/ascii" rel="tag">ASCII</a>, <a href="https://www.hacksparrow.com/tag/buffer" rel="tag">Buffer</a>, <a href="https://www.hacksparrow.com/tag/encoding" rel="tag">Encoding</a>, <a href="https://www.hacksparrow.com/tag/node-js" rel="tag">node.js</a>, <a href="https://www.hacksparrow.com/tag/utf" rel="tag">UTF</a></i> 
      </div> 
      <p>So how do you encode a string to base64 is Node.js? Is there something easy like <code>base64_encode()</code> of PHP's?</p>
      <p>Node.js 'being' JavaScript, has a more logical approach to encoding strings, instead of having thousands of inconsistently defined global functions.</p>
      <p>Here is how you encode normal text to base64 in Node.js:</p>
      <pre class="prettyprint codeblock">var b = new Buffer('JavaScript');<br>var s = b.toString('base64');<br>// SmF2YVNjcmlwdA==<br></pre>
      <p>And here is how you decode base64 encoded strings:</p>
      <pre class="prettyprint codeblock">var b = new Buffer('SmF2YVNjcmlwdA==', 'base64')<br>var s = b.toString();<br>// JavaScript<br></pre>
      <p>If you are interested in the details of how the above examples worked, follow me.</p>
      <p>The <code>new Buffer()</code> constructor requires a number, array or string as the first parameter, and an optional encoding type as the second parameter. The possible encoding types are ascii, utf8, ucs2, base64, binary, and hex; the default being utf8.</p>
      <p>By passing the second parameter, we tell JavaScript that "the string you see is encoded in this particular format". Notice how we did that in the decoding example.</p>
      <p>Once we have the encoded string, we call the <code>toString()</code> method on the string. If we don't pass the encoding type to <code>toString()</code>, JavaScript assumes we want to convert the object to utf8 encoded string by default. We can make it convert to other formats by passing the encoding type to <code>toString()</code>.</p>
      <p>Let's encode a base64 encoded string to hex:</p>
      <pre class="prettyprint codeblock">var b = new Buffer('SmF2YVNjcmlwdA==', 'base64')<br>var s = b.toString('hex');<br>// 4a617661536372697074<br></pre>
      <p>Now decode it to something humans can read:</p>
      <pre class="prettyprint codeblock">var b = new Buffer('4a617661536372697074', 'hex')<br>var s = b.toString('utf8');<br>// JavaScript<br></pre>
      <p>Once you get the basics of <a href="http://nodejs.org/api/buffer.html">Buffer</a> and encoding, you can use your knowledge of the <a href="http://nodejs.org/api/fs.html">File System</a> module to encode files to base64 strings.</p>
      <p>In this well-commented example we convert an image to base64 encoded string, and re-generate a copy of the image from the base64 encoded string.</p>
      <pre class="prettyprint codeblock">var fs = require('fs');<br><br>// function to encode file data to base64 encoded string<br>function base64_encode(file) {<br>&nbsp;&nbsp;&nbsp;&nbsp;// read binary data<br>&nbsp;&nbsp;&nbsp;&nbsp;var bitmap = fs.readFileSync(file);<br>&nbsp;&nbsp;&nbsp;&nbsp;// convert binary data to base64 encoded string<br>&nbsp;&nbsp;&nbsp;&nbsp;return new Buffer(bitmap).toString('base64');<br>}<br><br>// function to create file from base64 encoded string<br>function base64_decode(base64str, file) {<br>&nbsp;&nbsp;&nbsp;&nbsp;// create buffer object from base64 encoded string, it is important to tell the constructor that the string is base64 encoded<br>&nbsp;&nbsp;&nbsp;&nbsp;var bitmap = new Buffer(base64str, 'base64');<br>&nbsp;&nbsp;&nbsp;&nbsp;// write buffer to file<br>&nbsp;&nbsp;&nbsp;&nbsp;fs.writeFileSync(file, bitmap);<br>&nbsp;&nbsp;&nbsp;&nbsp;console.log('******** File created from base64 encoded string ********');<br>}<br><br>// convert image to base64 encoded string<br>var base64str = base64_encode('kitten.jpg');<br>console.log(base64str);<br>// convert base64 string back to image <br>base64_decode(base64str, 'copy.jpg');<br></pre>
      <p>Fascinating?</p>
      <p>I am sure you learnt not only how to encode and decode base64, but in many other formats as well. Have fun with encoding!</p>
      <p>PS: utf8 is the superset of ascii. If you are limited to using characters on the standard English keyboard, you can use ascii; if you are dealing with 'exotic' characters and symbols like ?, ?????, Üdvözöljük etc., use utf.</p>
