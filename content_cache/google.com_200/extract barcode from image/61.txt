           <p>In this tutorial we will learn Intel XDK barcode API. Before we proceed with building a application using Intel XDK barcode API we first need to learn what barcode is and how we can generate barcodes using PHP.</p>
           <h3><span id="What_is_Barcode">What is Barcode?</span></h3>
           <p>A barcode is a machine generated image representing a string. This image is generated using Barcode encoding algorithm. When this image is decoded by using barcode algorithm we get the string. Every barcode represents a unique string. Barcode image can be scanned into a digital device(i.e., computer, mobile etc) for decoding using <a href="http://en.wikipedia.org/wiki/Barcode_reader" class="external" rel="nofollow" target="_blank">Barcode Reader</a>. Barcodes come in many different representations</p>
           <p><a href="http://i2.wp.com/qnimate.com/wp-content/uploads/2014/11/barcode_types.png" class="lightbox"><img src="http://i2.wp.com/qnimate.com/wp-content/uploads/2014/11/barcode_types.png?resize=580%2C381" alt="barcode_types" class="alignnone size-full wp-image-5219" data-recalc-dims="1"></a></p>
           <p>Each of this type has its own encoding and decoding algorithms. Most barcode scanners are limited to scanning only one type of barcode representation.</p>
           <h3><span id="Where_is_Barcode_used">Where is Barcode used?</span></h3>
           <p>Barcode is used wherever we don’t want to type a string, rather we scan a image and get the string into input. They are used in shopping malls, websites, bookstores etc.</p>
           <p>In bookstores barcode represents the ISBN of the book. Therefore while purchasing the book the book seller doesn’t have to type the ISBN instead scan it.</p>
           <p>In websites barcode represents the URL of the webpage. Therefore if we want to open the webpage in phone then we can just scan the barcode image and open the URL, no need to type the whole URL.</p>

           <h3><span id="Scanning_Barcode_using_Intel_XDK">Scanning Barcode using Intel XDK</span></h3>
           <p>Intel XDK barcode API uses the camera to scan the barcode. Here is the JavaScript code to open the camera to scan the barcode and decode the message.</p>
           <p><b>intel.xdk.device.scanBarcode()</b> is the function which launches the camera to scan the barcode. <b>intel.xdk.device.barcode.scan</b> is the event which is fired after the scan is completed. To use this api you must include Intel XDK Device Plugin in your app.</p>
           <p>This is how the API works.</p>
           <div class="codecolorer-container javascript default" style="overflow:auto;white-space:nowrap;">
