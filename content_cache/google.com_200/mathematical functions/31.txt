      </table> <p> <b>Question:</b> What mathematical functions are supported in JavaScript? </p><p> <b>Answer:</b> JavaScript supports many <a href="arithmet.htm">arithmetic operations</a> as well as the following math functions (or, more precisely, methods of the <code>Math</code> object): </p><pre>Math.abs(a)     // the absolute value of a
Math.acos(a)    // arc cosine of a
Math.asin(a)    // arc sine of a
Math.atan(a)    // arc tangent of a
Math.atan2(a,b) // arc tangent of a/b
Math.ceil(a)    // integer closest to a and not less than a
Math.cos(a)     // cosine of a
Math.exp(a)     // exponent of a (<a href="mathconstants.htm">Math.E</a> to the power a)
Math.floor(a)   // integer closest to a, not greater than a
Math.log(a)     // log of a base e
Math.max(a,b)   // the maximum of a and b
Math.min(a,b)   // the minimum of a and b
Math.pow(a,b)   // a to the power b
<a href="randomnu.htm">Math.random</a>()   // pseudorandom number 0 to 1 (see <a href="randomnu.htm">examples</a>)
<a href="rounding.htm">Math.round</a>(a)   // integer closest to a (see <a href="rounding.htm">rounding examples</a>)
Math.sin(a)     // sine of a
Math.sqrt(a)    // square root of a
Math.tan(a)     // tangent of a
</pre> Note that trigonometric functions assume that the argument is in radians, not degrees! <p> <b>See also:</b> <br>• <a href="plotafunctiongraph.htm">How do I plot a function graph in JavaScript?</a> <br>• <a href="constant.htm">Number literals in JavaScript</a> <br>• <a href="mathconstants.htm">Mathematical constants in JavaScript</a> <br>• <a href="octalsan.htm">Octals and hexadecimals in JavaScript</a> <br>• <a href="mathsymbols.htm">Can I display mathematical symbols as part of JavaScript output?</a> <br>• <a href="greekletters.htm">Can I display Greek letters on my page as part of JavaScript output?</a> <br>• <a target="_top" href="../math/index.htm">Doing Math with JavaScript</a> </p></td>
