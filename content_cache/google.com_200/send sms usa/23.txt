       </svg></a>SMS text/binary messaging tool for node.js</h3> 
     <p><img src="https://www.websms.com/images/logo.png" alt></p> 
     <p><code>websmscom.js</code> module provides an easy-to-use API for sending SMS text and binary messages through websms.com API (<code>https://api.websms.com</code>) and is also usable as command line tool directly from bash.</p> 
     <p>An example code of how to use this nodejs module in your own scripts can be found in <code>examples/send_sms.js</code> .</p> 
     <h4><a id="user-content-features" class="deep-link" href="#features">
       <svg aria-hidden="true" class="deep-link-icon" height="16" version="1.1" width="16">

       </svg></a>Features:</h4> 
     <ul> 
      <li>Text Messages</li> 
      <li>Binary Messages</li> 
      <li>Confirmation of Delivery</li> 
      <li>Answers to SMS can be forwarded</li> 
      <li>Usable in modules and from command line</li> 
     </ul> 
     <p>See <a href="http://websms.com">websms.com</a> website to <a href="https://www.websms.com/websms-testen/">register</a> for an account.</p> 
     <p>For general API specification of the server (nodejs independent) visit: <a href="https://api.websms.com">https://api.websms.com</a></p>

       </svg></a>Parameters explained:</h4> 
     <ul> 
      <li><strong>gatewayUrl</strong> : {string} URL used to connect to websms API like 'https://api.websms.com'</li> 
      <li><strong>username</strong> : {string}username used in basic authentication. This is your websms.com account username</li> 
      <li><strong>password</strong> : {string} password used in basic authentication This is your websms.com account password</li> 
      <li><strong>recipientAddressList</strong> : {Array} of strings containing message recipient mobile numbers (MSISDN) like ['4367612345678','4369912345678']</li> 
      <li><strong>unicodeMessageText</strong> : {string} messageContent string sent that will be included in JSON object and sent as charset utf-8 to API. Special characters should be escaped as unicode. Euro sign is \u20AC.</li> 
      <li><strong>creationFailedCallback</strong> : {Function} function that is called when creation of message failed</li> 
      <li><strong>maxSmsPerMessage</strong> : {Number} integer number 1-255 telling how many concatenated sms parts are the limit for sending this message. (in case the text is longer than what fits into multiple sms)</li> 
      <li><strong>isTest</strong> : {boolean} false to really send sms. true to just test interface connection and process</li> 
      <li><strong>callback</strong> : {Function} callback at error or success</li> 
     </ul>
