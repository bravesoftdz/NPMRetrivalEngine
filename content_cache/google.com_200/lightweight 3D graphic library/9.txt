    <h1>50 JavaScript Libraries for Charts and Graphs</h1>
    <div id="meta">
     <span>Last updated on <strong>August 1, 2015</strong> in </span>
     <a href="http://techslides.com/category/development" title="View all posts in Development" rel="category tag">Development</a>
    </div>
    <p><img src="http://farm5.staticflickr.com/4033/4339699941_62011a1f3a.jpg" alt="JavaScript Charting Libraries"></p> 
    <p>How are you visualizing data in JavaScript? What is your favorite library for creating charts and graphs with JavaScript?</p> 
    <p>I have been a long time fan of <a href="http://www.jqplot.com/">jqPlot</a> for its open source approach, simplicity, <a href="http://www.jqplot.com/deploy/dist/examples/">examples</a>, and great features. It performs well, it is used by many corporations, and it is built on top of jQuery. Recently, however, <a href="http://d3js.org/">D3</a> has really caught my eye. It uses JavaScript, HTML, SVG and CSS to really bring data to life. It is also open sourced, does not depend on jQuery, has a <a href="https://github.com/mbostock/d3/wiki/Gallery">ton of examples</a>, and there are even some good charting libraries based on D3. The other big player is <a href="http://www.highcharts.com/">HighCharts</a>, also written in pure JavaScript with <a href="http://www.highcharts.com/demo/">all demos that you can view and edit in JSFiddle</a> and it also has some powerful features. Beyond these three, there are many more useful JavaScript charting libraries worth considering. Here is a list of the top 50 JS libraries for making charts and graphs:<br> <span id="more-570"></span></p> 
    <p><a href="http://www.fusioncharts.com/">FusionCharts</a> – they call themselves an <a href="http://www.fusioncharts.com/">enterprise-grade JavaScript charting component</a> and with good reason I suppose. They are probably the best-looking of the lot and have <a href="http://www.fusioncharts.com/demos/gallery/">90 chart types</a> that work across devices and browsers (including IE 6, 7 and 8 for which they use VML). They have a ton of <a href="http://www.fusioncharts.com/demos/business/">business dashboard examples</a>, <a href="http://www.fusioncharts.com/support/documentation/">extensive documentation</a> and some good articles on <a href="http://www.fusioncharts.com/resources/charting-best-practices/">charting best practices</a>. They also have a <a href="http://www.fusioncharts.com/jquery">jQuery plugin</a>. FusionCharts started a decade back (they even have a <a href="http://www.fusioncharts.com/story">book on their startup journey</a>) and claim to have 21,000 customers (LinkedIn, World Bank, White House, Oracle etc).</p> 
    <p><a href="http://d3js.org/">D3</a> – is really amazing and I love it’s simplicity. The documentation is really extensive, the source code is hosted on GitHub, and the examples are never-ending. There is a <a href="http://tributary.io/">D3 prototyping tool called Tributary</a> with many <a href="http://enjalot.com/tributary/2958568/">great</a> <a href="http://enjalot.com/tributary/3651651/">examples</a>. It’s so good that xcharts, nvd3, Rickshaw, Cubism.js, dc.js, and xkcd are all based on D3. If you are making some amazing custom data visualizations, D3 should probably be your choice, otherwise these d3-based libraries mentioned above are great for more simple straightforward graphs. Finally, I highly recommend looking at Scott Murray free book on D3 called <a href="http://ofps.oreilly.com/titles/9781449339739/">Interactive Data Visualization for the Web</a> and <a href="http://www.dashingd3js.com/">Dashing D3 tutorials</a>.</p> 
    <p><a href="http://www.highcharts.com/">HighCharts</a> – very powerful, I love the JSFiddle examples and powerful features. I don’t like the license fee but they have great customers (IBM, NASA, MasterCard, etc). They also have backward compatibility with IE8.</p> 
    <p><a href="http://www.jqplot.com/">jqPlot</a> – if you are already using jQuery, you don’t want to pay for HighCharts, and D3 is too much for your simple use cases, then jqPlot is an excellent choice.</p> 
    <p><a href="http://dygraphs.com/">dygraphs</a> – open source JavaScript library that produces interactive, zoomable charts of time series. It is really excellent for large data sets.</p> 
    <p><a href="http://mbostock.github.com/protovis/">Protovis</a> – free and open-source created by the same team that is now working on D3. If you are interested in comparing D3 to Protovis, check out this <a href="http://stackoverflow.com/questions/6212104/protovis-vs-d3-js">stackoveflow page</a>.</p> 
    <p><a href="http://www.flotcharts.org/">Flot Charts</a> – similar to jqPlot, Flot is a pure JavaScript plotting library for jQuery, with a focus on simple usage, attractive looks and interactive features. If you are interested in comparing Flot to jqPlot, this <a href="https://groups.google.com/forum/?fromgroups=#!topic/jqplot-users/WfduTBNq_V4">thread</a> has some good information.</p> 
    <p><a href="https://developers.google.com/chart/">Google Chart Tools</a> – powerful, free, simple to use, and it has everything from simple line charts to complex hierarchical tree maps, the chart galley provides a large number of well-designed chart types.</p> 
    <p><a href="http://nickqizhu.github.com/dc.js/">dc.js</a> – a javascript charting library based on D3 with native crossfilter support and allowing highly efficient exploration on large multi-dimensional dataset.</p> 
    <p><a href="http://tenxer.github.com/xcharts/">xcharts</a> – a D3-based library for building custom charts and graphs.</p> 
    <p><a href="http://nvd3.com/">nvd3</a> – allows you to build re-usable charts and chart components without taking away the power that d3.js gives you.</p> 
    <p><a href="http://code.shutterstock.com/rickshaw/">rickshaw</a> – JavaScript toolkit for creating interactive time series graphs.</p> 
    <p><a href="http://square.github.com/cubism/">Cubism.js</a> – a D3 plugin for visualizing time series. Use Cubism to construct better realtime dashboards, pulling data from Graphite, Cube and other sources.</p> 
    <p><a href="http://dan.iel.fm/xkcd/">xkcd</a> – allows you to make XKCD-style plots in Javascript using D3. Check out <a href="http://xkcdgraphs.com/">xkcdgraphs</a> to make your own xkcd graphs.</p> 
    <p><a href="http://www.omnipotent.net/jquery.sparkline/">jQuery Sparklines</a> – a jQuery plugin that creates small inline charts directly in the browser.</p> 
    <p><a href="http://benpickles.github.com/peity/">peity</a> – a simple jQuery plugin that converts an element’s content into a simple CANVAS mini pie, line or bar chart.</p> 
    <p><a href="http://bonsaijs.org/">BonsaiJS</a> – lightweight graphics library with an intuitive graphics API and an SVG renderer.</p> 
    <p><a href="http://code.google.com/p/flotr/">Flotr</a> – a Javascript plotting library for Prototype.js. It comes with great features like legend support, negative value support, mouse tracking, selection support, zoom support, event hooks, css styling support, text included into the canvas, rotated labels, color gradients, graph title and subtitle, spreadsheet, CSV data download and much more.</p> 
    <p><a href="http://code.google.com/p/protochart/">ProtoChart</a> – As the name suggests, ProtoChart allows you to create beautiful charts using Javascript and Prototype. It is open sourced, uses Canvas, and was highly motivated by Flot, Flotr and PlotKit libraries.</p> 
    <p><a href="http://humblesoftware.com/flotr2/">Flotr2</a> – a recent project by HumbleSoftware that lets you make graphs and charts with Canvas and JavaScript. HumbleSoftware also came out with <a href="https://github.com/HumbleSoftware/envisionjs">envisionjs</a> which uses flotr2 and allows you to make dynamic HTML5 visualizations.</p> 
    <p><a href="https://github.com/filamentgroup/jQuery-Visualize">jQuery-Visualize</a> – HTML5 canvas charts driven by HTML table elements. Another charting plugin for jQuery.</p> 
    <p><a href="http://www.jscharts.com/">JS Charts</a> – a JavaScript based chart generator that requires little or no coding. The free version has a watermark which can be removed by paying for a license.</p> 
    <p><a href="http://www.liquidx.net/plotkit/">PlotKit</a> – a chart and graph plotting Library for Javascript. It has support for HTML Canvas and also SVG via Adobe SVG Viewer and native browser support. PlotKit is a complete rewrite of <a href="http://www.liquidx.net/canvasgraphjs/">CanvasGraph</a>.</p> 
    <p><a href="http://mootools.net/forge/p/milkchart">MilkChart</a> – a compact JavaScript framework for creating graphs based on MooTools. It is simple and easy to understand.</p> 
    <p><a href="http://moochart.coneri.se/">moochart</a> – is another plugin for MooTools and it focuses on drawing bubble diagrams on the canvas tag.</p> 
    <p><a href="http://www.labs.unwieldy.net/moowheel/">moowheel</a> – a JavaScript connections visualization library based on MooTools.</p> 
    <p><a href="http://mootools.net/forge/p/table2chart">table2chart</a> – a MooTools plugin that generates charts from accessible data tables.</p> 
    <p><a href="http://dragan.yourtree.org/code/canvas-3d-graph/">Canvas 3D Graph</a> – a special type of bar graph that plot numbers in 3D.</p> 
    <p><a href="http://xaviershay.github.com/tufte-graph/">TufteGraph</a> – another JavaScript charting library based on jQuery. It has a really compact API and non-core layout is done via CSS.</p> 
    <p><a href="http://arborjs.org/">ArborJS</a> – a graph visualization library using web workers and jQuery.</p> 
    <p><a href="http://www.simile-widgets.org/timeplot/">TimePlot</a> – focused entirely on graphing time-series data, with support for annotating the graph with temporal events.</p> 
    <p><a href="http://g.raphaeljs.com/">gRaphael</a> – create stunning charts on your website, based on Raphael graphics library.</p> 
    <p><a href="https://github.com/uiteoi/ico">ICO</a> – another JavaScript graph library based on Raphael.</p> 
    <p><a href="http://elycharts.com/">Elycharts</a> – a pure javascript charting library, easy to use and completely customizable. It is based on jQuery and Raphaël libraries.</p> 
    <p><a href="http://www.zingchart.com/">ZingChart</a> – allows you to build advanced charts and visualizations in HTML5, SVG, VML and Flash. They also have a <a href="http://www.zingchart.com/blog/2015/01/05/jquery-wrapper-zingchart-charting-library-api/">jQuery wrapper</a> for this great charting library.</p> 
    <p><a href="http://www.rgraph.net/">RGraph</a> – produces interactive charts using the HTML5 canvas tag.</p> 
    <p><a href="http://dojotoolkit.org/documentation/tutorials/1.8/charting/">Dojo Charting</a> – if you are already using Dojo, then Dojo’s charting library is a great solution for making charts and graphs.</p> 
    <p><a href="http://bluff.jcoglan.com/">Bluff</a> – Bluff is a JavaScript port of the <a href="http://nubyonrails.com/pages/gruff">Gruff graphing library</a> for Ruby. It is designed to support all the features of Gruff with minimal dependencies.</p> 
    <p><a href="http://canvasxpress.org/">canvasXpress</a> – a javascript library based on the Canvas tag implemented in HTML5.</p> 
    <p><a href="http://jsgt.org/c/test/ws.htm">ccchart</a> – a Simple JavaScript chart that does not depend on libraries such as jQuery.</p> 
    <p><a href="http://philogb.github.com/jit/">JIT</a> – JavaScript InfoVis Toolkit provides tools for creating Interactive Data Visualizations for the Web.</p> 
    <p><a href="http://jsxgraph.uni-bayreuth.de/wp/">JSXGraph</a> – a cross-browser library for interactive geometry, function plotting, charting, and data visualization in a web browser.</p> 
    <p><a href="http://smoothiecharts.org/">Smoothie Charts</a> – a JavaScript Charting Library for Streaming Data.</p> 
    <p><a href="http://yuilibrary.com/yui/docs/charts/">YUI Charts</a> – YUI module that provides a JavaScript API for visualizing data in a variety of formats leveraging SVG, HTML Canvas and VML to render its graphical elements.</p> 
    <p><a href="http://www.amcharts.com/javascript-charts/">amcharts</a> – JavaScript/HTML5 charts including serial (column, bar, line, area, step line, smoothed line, candlestick and ohlc graphs), pie/donut, radar/polar and xy/scatter/bubble charts.</p> 
    <p><a href="http://www.ejschart.com/">Emprise JavaScript Charts</a> – pure JavaScript charts with great features and good cross browser support.</p> 
    <p><a href="http://www.lutanho.net/diagram/">JavaScript Diagram Builder</a> – a library of some objects and functions, which can help you to display a chart or graph in an easy way.</p> 
    <p><a href="http://www.jgraph.com/index.html">jGraph</a> – HTML5 diagramming component with full IE 6-8 and touch device support.</p> 
    <p><a href="http://www.sencha.com/products/touch/charts/">Sencha Touch Charts</a> – it uses HTML5 Canvas and CSS3 to deliver charts to the mobile web browsers.</p> 
    <p><a href="http://chart.inetsoft.com/">Style Chart</a> – a free Ajax Visualization Charting Service</p> 
    <p>If you are looking for more libraries and web services that deal with visualizing data, I recommend checking out <a href="http://selection.datavisualization.ch/">Datavisualization.ch</a>.</p> 
    <p>Have I missed any good JavaScript charting libraries?</p> 
    <h3>Update:</h3> 
    <p><a href="http://metricsgraphicsjs.org/">MetricsGraphics.js</a> – a library built on top of D3 that is optimized for visualizing and laying out time-series data. </p> 
    <p><a href="http://cyberpython.github.com/AwesomeChartJS/">AwesomeChartJS</a> – great looking charts with JavaScript and the canvas element</p> 
    <p><a href="http://sigmajs.org/">Sigma.js</a> – an open-source lightweight JavaScript library to make network visualizations using the HTML canvas element.</p> 
    <p><a href="https://github.com/jondot/graphene">Graphene</a> – a realtime dashboard &amp; graphing toolkit based on D3 and Backbone.</p> 
    <p><a href="https://github.com/moontoast/jquery-toastchart">jQuery ToastChart</a> – a line-graph charting library that is designed to show all types of data using the HTML5 canvas element.</p> 
    <p><a href="http://www.chartjs.org/">ChartsJS</a> – Easy, object oriented client side graphs for designers and developers.</p> 
    <p><a href="http://cytoscape.github.io/cytoscape.js/">Cytoscape.js</a> – open-source JavaScript graph library for analysis and visualization.</p> 
    <p><a href="http://crowdstrike.github.io/ember-timetree/">Ember Timetree</a> – visualize hierarchical timeline data with Ember.js and D3.js.</p> 
    <p><a href="http://mourner.github.io/simplify-js/">Simplify.js</a> – tiny high-performance JavaScript 2D/3D polyline simplification library (extracted from Leaflet)</p> 
    <p><a href="http://datamaps.github.io/">DataMaps</a> – Maps for interactive data visualizations in Javascript (with d3js)</p> 
    <p><a href="http://dunxrion.github.io/aristochart/">Aristochart</a> – simple and sophisticated static 2D line charts.</p> 
    <p><a href="https://github.com/oesmith/morris.js/">Morris.js</a> – time-series line graphs with jQuery and Raphael.js</p> 
    <p><a href="http://grafico.kilianvalkhof.com/">Grafico</a> – a javascript charting library based on Raphaël and Prototype.js</p> 
    <p><a href="https://github.com/caprica/jquery-orgchart">jQuery orgChart</a> – a small JQuery plugin that generates a hierarchical orginisational chart from a nested unordered list.</p> 
    <p><a href="http://timechart.toolset.io/">TimeChart</a> – powerful and simple to use chart library for creating time based charts in pure HTML5-JavaScript.</p> 
    <p><a href="http://www.jointjs.com/">JointJS</a> – open source JavaScript library for creating diagrams.</p> 
    <p><a href="http://modeln.github.io/sDashboard">sDashboard</a> – simple &amp; light weight dashboard framework in javascript</p> 
    <p><a href="http://jmstriegel.github.io/jquery.spidergraph/">jQuery Spidergraph</a> – an interactive HTML5 spider graph module for jQuery.</p> 
    <p><a href="http://jlord.github.io/sheetsee.js/">sheetsee.js</a> – visualize data from a Google spreadsheet.</p> 
    <p><a href="http://quartz.github.io/Chartbuilder/">Chartbuilder</a> – D3.js based front-end charting application that facilitates easy creation of simple beautiful charts</p> 
    <p><a href="https://github.com/akngs/dviz">dViz</a> – a declarative data visualization library written in JavaScript. Embed charts into the page by just modifying some HTML code.</p> 
    <p><a href="https://github.com/ankane/chartkick.js">Chartkick.js</a> – Create beautiful Javascript charts with minimal code.</p> 
    <p><a href="https://github.com/techslides/cssplot">CSSplot</a> – Simple, css-only charts</p> 
    <p><a href="http://plottablejs.org/">Plottable</a> – Flexible, interactive charts for the web based on D3</p> 
    <p><a href="http://www.chartjs.org/">ChartJS – Simple, clean and engaging charts</a></p> 
    <p><a href="https://plot.ly/javascript-graphing-library/">Plotly.js – client-side, interactive JavaScript graphing library, built on top of D3.js and stackgl</a></p> 
    <p><a href="http://c3js.org/">C3.js – D3 based reusable chart library</a></p>
