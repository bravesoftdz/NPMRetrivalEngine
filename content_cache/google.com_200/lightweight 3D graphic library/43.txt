          <div class="article__body"> 
           <p>Our mission at Tutorialzine is to keep you up to date with the latest and coolest trends in web development. That’s why every month we release a handpicked collection of some of the best resources that we’ve stumbled upon and deemed worthy of your attention.</p> 
           <hr> 
           <figure>
            <a href="https://uber.github.io/deck.gl/" target="_blank" rel="noreferrer noopener"><img src="https://tutorialzine.com/media/2016/12/deckgl1.jpg" alt="deckgl1.jpg"></a>
           </figure> 
           <h2><a href="https://uber.github.io/deck.gl/" target="_blank" rel="noreferrer noopener">Deck.gl</a></h2> 
           <p>Data visualization library that can handle large-scale 2D and 3D visualizations with ease. Deck.gl is made by <a href="https://www.uber.com/" target="_blank" rel="noreferrer noopener">Uber</a> and powered by WebGL and React, which results in a project with great performance, as can be seen in the beautiful <a href="https://uber.github.io/deck.gl/#/examples/overview" target="_blank" rel="noreferrer noopener">examples</a>, rendering datasets with close to a million entries.</p> 
           <hr> 
           <p><a href="https://svelte.technology/" target="_blank" rel="noreferrer noopener"><img src="https://tutorialzine.com/media/2016/12/svelte.png" width="785" height="314" alt="svelte.png"></a></p> 
           <h2><a href="https://svelte.technology/" target="_blank" rel="noreferrer noopener">Svelte</a></h2> 
           <p>Svelte is a brand new project that provides a fresh, more-lightweight approach to the existing solutions offered by large frameworks like React and Angular. Instead of shipping huge piles of code, Svelte compiles your app to optimized vanilla JavaScript which can be executed much faster by the browser.</p> 
           <hr> 
           <figure>
            <a href="https://turbo.github.io/" target="_blank" rel="noreferrer noopener"><img src="https://tutorialzine.com/media/2016/12/turbojs.jpg" width="800" height="410" alt="turbojs.jpg"></a>
           </figure> 
           <h2><a href="https://turbo.github.io/" target="_blank" rel="noreferrer noopener">Turbo.js</a></h2> 
           <p>Turbo.js allows you to access the GPU and improve the performance of your apps. By executing processes directly in the graphics processor, it is possible to run multiple complex calculations in parallel, drastically cutting down JavaScript waiting times. In works in all popular web browsers and is supported by most desktop and mobile chipsets.</p> 
           <hr> 
           <p><a href="https://webkul.github.io/csspin/" target="_blank" rel="noreferrer noopener"><img src="https://tutorialzine.com/media/2016/12/csspin.jpg" width="785" height="306" alt="csspin.jpg"></a></p> 
           <h2><a href="https://webkul.github.io/csspin/" target="_blank" rel="noreferrer noopener">CSSPIN</a></h2> 
           <p>CSSPIN is a collection of colorful indicators and spinners, all featuring smooth pure-CSS animations and consisting of just a single div: <code>&lt;div class="cp-spinner cp-bubble"&gt;&lt;/div&gt;</code>. Each spinner in the library has it's own CSS file, allowing you to quickly take only what you need and minimize bloat.</p> 
           <hr> 
           <figure>
            <a href="https://github.com/palantir/blueprint" target="_blank" rel="noreferrer noopener"><img src="https://tutorialzine.com/media/2016/12/blueprint.jpg" width="800" height="451" alt="blueprint.jpg"></a>
           </figure> 
           <h2><a href="https://github.com/palantir/blueprint" target="_blank" rel="noreferrer noopener">Blueprint</a></h2> 
           <p>Blueprint is a React toolkit optimized for building complex user interfaces such as web-based desktop applications and admin panels. It offers a rich component library, lots of customization options with Sass or Less, and a detailed, easy to follow documentation.</p> 
           <hr> 
           <figure>
            <a href="https://jessepollak.github.io/card/" target="_blank" rel="noreferrer noopener"><img src="https://tutorialzine.com/media/2016/12/card.jpg" width="840" height="409" alt="card.jpg"></a>
           </figure> 
           <h2><a href="https://jessepollak.github.io/card/" target="_blank" rel="noreferrer noopener">Card</a></h2> 
           <p>Card is a tiny vanilla JS project (with a jQuery version) that will make your credit card forms much more fun and interactive. After a quick installation, the library will take your form and transform it into an animated CSS-only credit card that gets filled as users input their data.</p> 
           <hr> 
           <figure>
            <a href="https://space10-community.github.io/conversational-form/" target="_blank" rel="noreferrer noopener"><img src="https://tutorialzine.com/media/2016/12/conversational-form.jpg" width="800" height="452" alt="conversational-form.jpg"></a>
           </figure> 
           <h2><a href="https://space10-community.github.io/conversational-form/" target="_blank" rel="noreferrer noopener">Conversational Form</a></h2> 
           <p>Similar to Card, this library removes traditional web forms and replaces them with a more engaging alternative. Conversational Form takes all of the input fields and uses them to start a pseudo-chatbot conversation where the bot asks questions and step by step receives the needed information from the user .</p> 
           <hr> 
           <figure>
            <a href="https://typeis.github.io/" target="_blank" rel="noreferrer noopener"><img src="https://tutorialzine.com/media/2016/12/typeis.jpg" width="1100" height="637" alt="typeis.jpg"></a>
           </figure> 
           <h2><a href="https://typeis.github.io/" target="_blank" rel="noreferrer noopener">TypeIs</a></h2> 
           <p>Since JavaScript doesn't have static typing, at times there can be some confusion on the specific type of variables, especially Arrays which are considered Objects in JS. This super simple library can be used to solve this problem and easily find the real type of a variable.</p> 
           <hr> 
           <figure>
            <a href="https://milligram.github.io/" target="_blank" rel="noreferrer noopener"><img src="https://tutorialzine.com/media/2016/12/miligram.png" width="767" height="365" alt="miligram.png"></a>
           </figure> 
           <h2><a href="https://milligram.github.io/" target="_blank" rel="noreferrer noopener">Milligram</a></h2> 
           <p>A CSS framework that strives to stay as slim as possible and provide the most minimal setup while still making your project pretty. Unlike other UI frameworks such as <a href="http://getbootstrap.com/" target="_blank" rel="noreferrer noopener">Bootstrap</a>, Milligram doesn't offer any components, only basic HTML styles, typography, and a layout grid. Just 2kb in size when gzipped.</p> 
           <hr> 
           <figure>
            <a href="http://bitwiser.in/medium-draft/" target="_blank" rel="noreferrer noopener"><img src="https://tutorialzine.com/media/2016/12/medium-draft.jpg" width="971" height="442" alt="medium-draft.jpg"></a>
           </figure> 
           <h2><a href="http://bitwiser.in/medium-draft/" target="_blank" rel="noreferrer noopener">Medium-draft</a></h2> 
           <p>A React library built on top of Facebook's text editor framework <a href="https://facebook.github.io/draft-js/" target="_blank" rel="noreferrer noopener">draft-js</a>. It allows you to create rich markup editors in the style of Medium with lots of helpful key shortcuts and a friendly user interface for writing and editing content.</p> 
           <hr> 
           <figure>
            <a href="https://naver.github.io/egjs/" target="_blank" rel="noreferrer noopener"><img src="https://tutorialzine.com/media/2016/12/egjs.png" width="832" height="441" alt="egjs.png"></a>
           </figure> 
           <h2><a href="https://naver.github.io/egjs/" target="_blank" rel="noreferrer noopener">Eg.js</a></h2> 
           <p>A collection of handy JavaScript and jQuery components for UI interactions, effects, and various other utilities. The package includes an infinite grid builder, animations, landscape/portrait detection, device and browser info, and other great mini-libraries.</p> 
           <hr> 
           <figure>
            <a href="https://superdom.site/" target="_blank" rel="noreferrer noopener"><img src="https://tutorialzine.com/media/2016/12/superdom.png" width="1037" height="371" alt="superdom.png"></a>
           </figure> 
           <h2><a href="https://superdom.site/" target="_blank" rel="noreferrer noopener">Superdom</a></h2> 
           <p>Superdom is a lightweight alternative to jQuery that allows you to manipulate the HTML DOM. It provides a global <code>dom</code> object with a simple chaining interface, which can be used to select and modify all the existing elements on the page and their attributes.</p> 
           <hr> 
           <figure>
            <a href="https://github.com/zzarcon/chaosocket" target="_blank" rel="noreferrer noopener"><img src="https://tutorialzine.com/media/2016/12/chaossocket.jpg" width="952" height="544" alt="chaossocket.jpg"></a>
           </figure> 
           <h2><a href="https://github.com/zzarcon/chaosocket" target="_blank" rel="noreferrer noopener">Chaos Socket</a></h2> 
           <p>This Node.js package is capable of mocking WebSockets, making it easier to automate the testing of socket connections in your apps. The library has a simple API that allows you to register different kind of events and send them once or at a certain interval. Chaos Socket also comes with <a href="https://github.com/marak/Faker.js/" target="_blank" rel="noreferrer noopener">Faker.js</a> built-in for quickly generating dummy data.</p> 
           <hr> 
           <figure>
            <a href="https://docsify.js.org/" target="_blank" rel="noreferrer noopener"><img src="https://tutorialzine.com/media/2016/12/docsify.png" width="871" height="371" alt="docsify.png"></a>
           </figure> 
           <h2><a href="https://docsify.js.org/" target="_blank" rel="noreferrer noopener">Docsify</a></h2> 
           <p>JavaScript library and Node.js CLI that generate modern markdown-based documentation hubs for your projects. There are two themes available as well as some customization options such as displaying a <a href="https://github.com/tholman/github-corners" target="_blank" rel="noreferrer noopener">GitHub Corner</a> widget.</p> 
           <hr> 
           <figure>
            <a href="http://labelauty.js.org/" target="_blank" rel="noreferrer noopener"><img src="https://tutorialzine.com/media/2016/12/labelauty.png" width="971" height="442" alt="labelauty.png"></a>
           </figure> 
           <h2><a href="http://labelauty.js.org/" target="_blank" rel="noreferrer noopener">Labelauty</a></h2> 
           <p>A tiny jQuery plugin that automatically improves the appearance of checkboxes and radio buttons, while also adding the option to change their label text depending on whether they are truthful or not. It works in all modern browsers and is really easy to use.</p>

     $(document).ready(function() {
        $('#share-twitter, #share-fb, #share-google').click(function(e) {
          e.preventDefault();

          var url = this.href;
          var title = 'Share on ' + $(this).data('provider');

          window.open(url, title, getOptions());
        });

        function getOptions() {
          var width  = 575,
              height = 400,
              left   = ($(window).width()  - width)  / 2,
              top    = ($(window).height() - height) / 2,
              opts   = 'status=1' +
                       ',width='  + width  +
                       ',height=' + height +
                       ',top='    + top    +
                       ',left='   + left;

           return opts;
        }
     });
