            </section>
           </div> 
           <p>In theory, you can use any web framework (or roll your own) that supports mobile browsers and make it work with <a href="https://coligo.io/building-a-mobile-app-with-cordova-vuejs/">Vue.js and Cordova</a>. However, having a dedicated UI framework allows you choice of pre-built components and re-usable UI behaviors such as toggle buttons, floating actions buttons, swipe panes, tabs, scroll to refresh etc.</p> 
           <p>For Android, we have many <a href="https://superdevresources.com/material-design-web-ui-frameworks/">material design web frameworks</a> available which are responsive and can be coupled together with Vue.js in a Cordova app. If you are not particularly bent on using Vue.js then you have popular frameworks like <a href="https://ionicframework.com/">Ionic</a>&nbsp;(based on Angular) for building hybrid apps that&nbsp;offers near-native like UI for both Android and iOS. You can even go a step further and use <a href="https://facebook.github.io/react-native/">React Native</a> or <a href="https://www.nativescript.org/">NativeScript</a> which allow you to build native apps with JavaScript.</p> 
           <p>In this post, we will be sticking to frameworks that are integrated with&nbsp;Vue.js and offer building of mobile apps with Cordova. At the end, we have also mentioned a framework which allows you to build native apps with Vue.js. If you are just starting out, then these <a href="https://superdevresources.com/vuejs-tutorials/">Vue.js tutorials</a> will come in handy. Also take a look at these <a href="https://templateflip.com/vuejs-admin-templates/">Vue.js dashboard templates</a> in case you plan to build a web-app with admin panel.</p> 
           <h2>Vue.js based Mobile App&nbsp;Frameworks</h2> 
           <p>Listed below are frameworks that support or are closely integrated with Vue.js for building mobile apps. Note that some of these frameworks are still in their early stages of development, hence you may want to&nbsp;assess the community and support around a framework before making a choice.</p> 
           <h3><a href="http://quasar-framework.org/">Quasar Framework</a></h3> 
           <p><img class="alignnone wp-image-9719 size-full" src="https://superdevresources.com/wp-content/uploads/2017/02/quasar-vuejs-mobile-framework.png" alt="quasar vuejs mobile framework" width="800" height="480" srcset="https://superdevresources.com/wp-content/uploads/2017/02/quasar-vuejs-mobile-framework.png 800w, https://superdevresources.com/wp-content/uploads/2017/02/quasar-vuejs-mobile-framework-300x180.png 300w, https://superdevresources.com/wp-content/uploads/2017/02/quasar-vuejs-mobile-framework-768x461.png 768w, https://superdevresources.com/wp-content/uploads/2017/02/quasar-vuejs-mobile-framework-500x300.png 500w" sizes="(max-width: 800px) 100vw, 800px"></p> 
           <p>Quasar Framework allows you to build&nbsp;hybrid mobile apps with near-native like user-interface and uses VueJS 2 at its core. It is a MIT-licensed open source project available on <a href="https://github.com/quasarframework/quasar">GitHub</a>.</p> 
           <p>Quasar framework has good set of <a href="http://quasar-framework.org/guide">documentation</a> available on its website and provides&nbsp;an extensive set of Vue based UI components such as&nbsp;sliders, spinners, tooltip, popover, model dialogs, context menu, video embedding and many more.</p> 
           <p>Getting started with Quasar is easy and it offers its own set of CLI, much like the Ionic framework for out of the box setup and previews. You can build both Cordova as well as <a href="https://electron.atom.io/">Electron</a> (cross platform desktop apps) with this framework.</p> 
           <p>You can check out the <a href="https://play.google.com/store/apps/details?id=com.quasarframework.quasarplay">Quasar Play app</a> on Android Play Store to have a feel of the framework in a real app.</p> 
           <h3><a href="https://onsen.io/vue/">Onsen UI for Vue 2</a></h3> 
           <p><img class="alignnone size-full wp-image-9727" src="https://superdevresources.com/wp-content/uploads/2017/02/onsen-ui-vue2.png" alt="" width="800" height="480" srcset="https://superdevresources.com/wp-content/uploads/2017/02/onsen-ui-vue2.png 800w, https://superdevresources.com/wp-content/uploads/2017/02/onsen-ui-vue2-300x180.png 300w, https://superdevresources.com/wp-content/uploads/2017/02/onsen-ui-vue2-768x461.png 768w, https://superdevresources.com/wp-content/uploads/2017/02/onsen-ui-vue2-500x300.png 500w" sizes="(max-width: 800px) 100vw, 800px"></p> 
           <p>Onsen UI is a popular framework for building&nbsp;HTML5 hybrid and mobile web apps. Onsen offers Framework-agnostic UI components which supports AngularJS, Angular2, React, Vue.js, Meteor as well as plain JavaScript.</p> 
           <p>Onsen UI for Vue 2 combines&nbsp;Vue.js with Onsen UI to create hybrid &amp; mobile web apps. Check out <a href="https://onsen.io/samples/">Onsen samples</a> to get a feel of its offerings and head over to the<a href="https://onsen.io/vue/">&nbsp;Onsen UI for Vue 2</a>&nbsp;page&nbsp;for more info on how to integrate Onsen with Vue.js.</p> 
           <h3><a href="https://framework7.io/">Framework7</a></h3> 
           <p><img class="alignnone size-full wp-image-9723" src="https://superdevresources.com/wp-content/uploads/2017/02/framework7.jpg" alt="" width="800" height="480" srcset="https://superdevresources.com/wp-content/uploads/2017/02/framework7.jpg 800w, https://superdevresources.com/wp-content/uploads/2017/02/framework7-300x180.jpg 300w, https://superdevresources.com/wp-content/uploads/2017/02/framework7-768x461.jpg 768w, https://superdevresources.com/wp-content/uploads/2017/02/framework7-500x300.jpg 500w" sizes="(max-width: 800px) 100vw, 800px"></p> 
           <p>Framework7 is another&nbsp;<a href="https://github.com/nolimits4web/framework7/">open source</a> (MIT licensed) mobile HTML framework to develop hybrid mobile apps with iOS or Android native look and feel. While it does not support any other platform, you can build web apps using the Material look or the iOS look.</p> 
           <p>Framework7 offers a variety of native-like UI components and behaviours such as swipe actions, pull to refresh, dynamic navbar, page animations, virtual lists, searchbar etc. It offers support for Vue with its VueJS based <a href="https://framework7.io/vue/">starter app templates</a>. These templates are available as Cordova+webpack template or simply as webpack or browserify templates.</p> 
           <p>To get a feel of the framework, check out some of the apps in their <a href="https://framework7.io/showcase/">showcase</a>, or the <a href="https://framework7.io/apps/">demo apps</a>.</p> 
           <h3><a href="https://vuetifyjs.com/">Vuetify.js</a></h3> 
           <p><img class="alignnone size-full wp-image-9729" src="https://superdevresources.com/wp-content/uploads/2017/02/vuetifyjs.png" alt="" width="800" height="480" srcset="https://superdevresources.com/wp-content/uploads/2017/02/vuetifyjs.png 800w, https://superdevresources.com/wp-content/uploads/2017/02/vuetifyjs-300x180.png 300w, https://superdevresources.com/wp-content/uploads/2017/02/vuetifyjs-768x461.png 768w, https://superdevresources.com/wp-content/uploads/2017/02/vuetifyjs-500x300.png 500w" sizes="(max-width: 800px) 100vw, 800px"></p> 
           <p>Vuetify.js is a Component Framework for Vue.js 2 which&nbsp;utilizes Google’s Material Design patterns for the UI components. It&nbsp;comes with 3 Vue CLI templates pre-configured with Vuetify to help you get started with a Vue.js based web project.</p> 
           <p>Vuetify offers a wide range of Material Design components such as breadcrumbs, cards, dropdowns, navbar, sidebar, tabs etc. You can track the progress of this open source project on <a href="https://github.com/vuetifyjs/vuetify">GitHub</a>.</p> 
           <h3><a href="https://josephuspaye.github.io/Keen-UI/">Keen UI</a></h3> 
           <p><img class="alignnone size-full wp-image-9703" src="https://superdevresources.com/wp-content/uploads/2014/11/keenui-vuejs-material.png" alt="" width="800" height="480" srcset="https://superdevresources.com/wp-content/uploads/2014/11/keenui-vuejs-material.png 800w, https://superdevresources.com/wp-content/uploads/2014/11/keenui-vuejs-material-300x180.png 300w, https://superdevresources.com/wp-content/uploads/2014/11/keenui-vuejs-material-768x461.png 768w, https://superdevresources.com/wp-content/uploads/2014/11/keenui-vuejs-material-500x300.png 500w" sizes="(max-width: 800px) 100vw, 800px"></p> 
           <p>Keen UI is a collection of Material Design UI components written with Vue.js. It offers numerous Material style components such as for alert, autocomplete, FAB, popover, snackbar, tabs etc. The project is available on <a href="https://github.com/JosephusPaye/Keen-UI">GitHub</a> and is MIT Licensed.</p> 
           <h3><a href="https://mint-ui.github.io">Mint UI</a></h3> 
           <p><img class="alignnone size-full wp-image-9732" src="https://superdevresources.com/wp-content/uploads/2017/02/mint-ui-vuejs.png" alt="" width="800" height="480" srcset="https://superdevresources.com/wp-content/uploads/2017/02/mint-ui-vuejs.png 800w, https://superdevresources.com/wp-content/uploads/2017/02/mint-ui-vuejs-300x180.png 300w, https://superdevresources.com/wp-content/uploads/2017/02/mint-ui-vuejs-768x461.png 768w, https://superdevresources.com/wp-content/uploads/2017/02/mint-ui-vuejs-500x300.png 500w" sizes="(max-width: 800px) 100vw, 800px"></p> 
           <p>Mint UI provides CSS and JS components for building mobile applications such as toast, date-time picker, lazy load, progress bar, action sheet and more.&nbsp;Thanks to the efficient component-based approach of Vue.js, Mint UI is pretty light-weight too. You can find this MIT licensed project on <a href="https://github.com/ElemeFE/mint-ui/">GitHub</a>.</p> 
           <h3><a href="https://bootstrap-vue.github.io/">Bootstrap Vue</a></h3> 
           <p><img class="alignnone size-full wp-image-9734" src="https://superdevresources.com/wp-content/uploads/2017/02/bootstrap-vue.png" alt="" width="800" height="480" srcset="https://superdevresources.com/wp-content/uploads/2017/02/bootstrap-vue.png 800w, https://superdevresources.com/wp-content/uploads/2017/02/bootstrap-vue-300x180.png 300w, https://superdevresources.com/wp-content/uploads/2017/02/bootstrap-vue-768x461.png 768w, https://superdevresources.com/wp-content/uploads/2017/02/bootstrap-vue-500x300.png 500w" sizes="(max-width: 800px) 100vw, 800px"></p> 
           <p>Bootstrap Vue offers&nbsp;Bootstrap 4 based components For Vue.js 2. While many of the components are still under development, it already offers components such as breadcrumbs, dropdown, navbar, pagination etc.</p> 
           <h3><a href="https://vuematerial.github.io/">Vue Material</a></h3> 
           <p><img class="alignnone size-full wp-image-9736" src="https://superdevresources.com/wp-content/uploads/2017/02/vuematerial.png" alt="" width="800" height="480" srcset="https://superdevresources.com/wp-content/uploads/2017/02/vuematerial.png 800w, https://superdevresources.com/wp-content/uploads/2017/02/vuematerial-300x180.png 300w, https://superdevresources.com/wp-content/uploads/2017/02/vuematerial-768x461.png 768w, https://superdevresources.com/wp-content/uploads/2017/02/vuematerial-500x300.png 500w" sizes="(max-width: 800px) 100vw, 800px"></p> 
           <p>Vue Material is a lightweight framework built according to the Material Design specs&nbsp;to help you build responsive web apps. It offers many material design based components such as cards, chip, snackbar, switch, tabbar etc.&nbsp;This library aims to deliver components using the same API as offered by&nbsp;<a href="https://material.angularjs.org/">Angular Material</a>.</p> 
           <h3><a href="https://chenz24.github.io/vue-blu/">Vue-Blu</a></h3> 
           <p><img class="alignnone size-full wp-image-9739" src="https://superdevresources.com/wp-content/uploads/2017/02/vue-blu.png" alt="" width="800" height="480" srcset="https://superdevresources.com/wp-content/uploads/2017/02/vue-blu.png 800w, https://superdevresources.com/wp-content/uploads/2017/02/vue-blu-300x180.png 300w, https://superdevresources.com/wp-content/uploads/2017/02/vue-blu-768x461.png 768w, https://superdevresources.com/wp-content/uploads/2017/02/vue-blu-500x300.png 500w" sizes="(max-width: 800px) 100vw, 800px"></p> 
           <p>Vue-Blu is a UI Component Library based&nbsp;on Vue.js 2.x and <a href="http://bulma.io/">Bulma</a> CSS framework. While the documentation is mostly available in Chinese, the library offers navigation, view and form components. Check out their <a href="https://github.com/chenz24/vue-blu">GitHub repo</a> for more details.</p> 
           <h2>Vue.js based Native App Development Framework</h2> 
           <h3><a href="https://weex-project.io/">Weex</a></h3> 
           <p><img class="alignnone size-full wp-image-9744" src="https://superdevresources.com/wp-content/uploads/2017/02/weex-native-vuejs.jpg" alt="" width="800" height="480" srcset="https://superdevresources.com/wp-content/uploads/2017/02/weex-native-vuejs.jpg 800w, https://superdevresources.com/wp-content/uploads/2017/02/weex-native-vuejs-300x180.jpg 300w, https://superdevresources.com/wp-content/uploads/2017/02/weex-native-vuejs-768x461.jpg 768w, https://superdevresources.com/wp-content/uploads/2017/02/weex-native-vuejs-500x300.jpg 500w" sizes="(max-width: 800px) 100vw, 800px"></p> 
           <p>You may already be familiar with <a href="https://facebook.github.io/react-native/">React Native</a>&nbsp;by Facebook which allows you to build native iOS and Android apps using React. <a href="https://weex-project.io/">Weex</a> is a similar framework by the Alibaba group, which works with Vue.js and allows you to build cross-platform native mobile apps.</p> 
           <p>While still in development, Weex is a promising framework and already in use by many companies. You can keep yourself up-to-date with the project on <a href="https://github.com/alibaba/weex/">GitHub</a>&nbsp;which is Apache 2.0 licensed.</p> 
           <h2>Vue.js based Desktop UI Library</h2> 
           <h3><a href="http://element.eleme.io/#/en-US/">Element</a></h3> 
           <p><img class="alignnone size-full wp-image-9747" src="https://superdevresources.com/wp-content/uploads/2017/02/element-desktop-framework.jpg" alt="" width="800" height="480" srcset="https://superdevresources.com/wp-content/uploads/2017/02/element-desktop-framework.jpg 800w, https://superdevresources.com/wp-content/uploads/2017/02/element-desktop-framework-300x180.jpg 300w, https://superdevresources.com/wp-content/uploads/2017/02/element-desktop-framework-768x461.jpg 768w, https://superdevresources.com/wp-content/uploads/2017/02/element-desktop-framework-500x300.jpg 500w" sizes="(max-width: 800px) 100vw, 800px"></p> 
           <p>While Element is a desktop UI library based on Vue 2.0, we think it is relavant to be mentioned in this article. It offers many components suitable for writing a desktop web application. They are from the same team which offers&nbsp;Mint UI (covered earlier in this post). Element is a good choice if you are building Vue.js based app using <a href="https://electron.atom.io/">Electron</a>.</p> 
           <h2>Hybrid or Native App</h2> 
           <p>If you are still deciding whether to build a native app or a hybrid app, then there is a <a href="https://superdevresources.com/app-development-model-choose-native-hybrid-web-apps-infographics/">nice infographic</a>&nbsp;which can help you choose between different app development models. If you do decide to go with the hybrid app approach using Cordova or Phonegap along with Vue.js then take a look&nbsp;at this <a href="https://coligo.io/building-a-mobile-app-with-cordova-vuejs/">tutorial</a>&nbsp;on Coligo for a good starting point.</p> 
           <p>Which framework, in your opinion, is best for building apps with Vue.js? If you are using Vue.js already in your mobile apps and have a favorite mobile framework then do share with us in the comments below.</p>
