           <p data-mce-="" pid="2">Technically speaking, mobile apps on iOS, Android, and Windows Phone are coded using different programming languages. An iOS app uses Objective-C, an Android app is coded with Java, while a Windows Phone application uses .NET. However, with a decent knowledge of JavaScript, CSS, and HTML, you can build awesome mobile apps. Therefore, in this blog&nbsp; we are going to discuss the top JavaScript frameworks for developing mobile apps.</p> 
           <p data-mce-="" pid="3">Further reading: <a data-mce-href="http://www.valuecoders.com/blog/technology-and-apps/top-javascript-frameworks-list-comparison/" href="http://www.valuecoders.com/blog/technology-and-apps/top-javascript-frameworks-list-comparison/" target="_blank" rel="nofollow">9 Top JavaScript Frameworks List In Todays Time</a></p> 
           <p data-mce-="" pid="4">For web development, JavaScript has been a promising programming language and it's not going anywhere in the near future.&nbsp; Does JavaScript have the same impact on mobile app development? Let's find out.&nbsp;Here are the top 9 JavaScript frameworks for mobile app development as per our research at <a data-mce-href="http://www.valuecoders.com" href="http://www.valuecoders.com" target="_blank" rel="nofollow">ValueCoders</a>:</p> 
           <h2 data-mce-="">Top 9 JavaScript Frameworks for Mobile</h2> 
           <h3 data-mce-="">1) PhoneGap/Cordova</h3> 
           <p data-mce-="" pid="5"><a href="http://phonegap.com/" rel="nofollow" target="_blank">PhoneGap</a> (distribution of Apache Cordova) is a software development framework that helps you to reuse your existing web development skills to quickly make hybrid mobile applications built with HTML, CSS, and JavaScript. Hence, this is not exactly a JavaScript framework. However, it's important for you to know about PhoneGap before you get started.</p> 
           <p data-mce-="" pid="6">PhoneGap works as a medium between your web application codes and mobile OS APIs. This diagram will help you understand better:</p> 
           <p data-mce-="" pid="7"><img alt="phonegap technology how it works" class="fr-fin fr-dib" data-mce-src="http://www.valuecoders.com/blog/wp-content/uploads/2016/11/phonegap-technology-how-it-works.png" height="300" src="http://www.valuecoders.com/blog/wp-content/uploads/2016/11/phonegap-technology-how-it-works.png" width="858"><em>Courtesy: phonegap.com</em></p> 
           <p data-mce-="" pid="8">With the help of PhoneGap, you can use the same code that you wrote in JavaScript, HTML, and CSS and generate APIs for mobile OS like Android and iOS.</p> 
           <p data-mce-="" pid="9">&nbsp;Apart from PhoneGap, there are a few more frameworks that can help convert your JavaScript files to mobile APIs. <a data-mce-href="https://www.xamarin.com/" href="https://www.xamarin.com/" target="_blank" rel="nofollow">Xamarin</a>, <a data-mce-href="https://ionicframework.com/" href="https://ionicframework.com/" target="_blank" rel="nofollow">Ionic,</a>&nbsp;<a data-mce-href="https://coronalabs.com/" href="https://coronalabs.com/" target="_blank" rel="nofollow">Corona</a> are some honorable mentions.</p> 
           <p data-mce-="" pid="10">Further reading: <a data-mce-href="http://www.valuecoders.com/blog/technology-and-apps/xamarin-ionic-choose-one/" href="http://www.valuecoders.com/blog/technology-and-apps/xamarin-ionic-choose-one/" target="_blank" rel="nofollow">Xamarin or Ionic: which one to choose?</a></p> 
           <h2 data-mce-="">2) Titanium</h2> 
           <p data-mce-="" pid="11">Appcelerators <a href="http://www.appcelerator.com/mobile-app-development-products/" rel="nofollow" target="_blank">Titanium</a> is an open source application development platform that allows you&nbsp;to create native apps (mobile and desktop) using web technologies, such as HTML, JavaScript, and CSS. The Titanium Mobile SDK is one of the most popular cross-platform mobile development solution in use today with over 916109 mobile developers and 460, 587, 474 devices use Accelerator powered apps.</p> 
           <p data-mce-="" pid="12">Here is the architecture how Titanium works:</p> 
           <p data-mce-="" pid="13"><a data-mce-href="http://www.valuecoders.com/blog/wp-content/uploads/2016/11/titenium-architechture.jpg" href="http://www.valuecoders.com/blog/wp-content/uploads/2016/11/titenium-architechture.jpg" rel="nofollow"><img alt="titenium architechture- top javascript frameworks for mobile" class="fr-fin fr-dib" data-mce-src="http://www.valuecoders.com/blog/wp-content/uploads/2016/11/titenium-architechture.jpg" height="305" src="http://www.valuecoders.com/blog/wp-content/uploads/2016/11/titenium-architechture.jpg" width="800"></a></p> 
           <h2 data-mce-="">3) jQuery Mobile</h2> 
           <p data-mce-="" pid="14"><a data-mce-href="http://www.valuecoders.com/blog/wp-content/uploads/2016/11/jquery-mobile-logo-1.png" href="http://www.valuecoders.com/blog/wp-content/uploads/2016/11/jquery-mobile-logo-1.png" rel="nofollow"><img alt="jquery mobile -logo (1)" class="fr-dii fr-fil" data-mce-src="http://www.valuecoders.com/blog/wp-content/uploads/2016/11/jquery-mobile-logo-1.png" height="156" src="http://www.valuecoders.com/blog/wp-content/uploads/2016/11/jquery-mobile-logo-1.png" width="100%"></a><br data-mce-bogus="1"></p> 
           <p data-mce-="" pid="15"><a href="http://mobileangularui.com/" rel="nofollow" target="_blank">jQuery Mobile</a> framework takes the "write less, do more" agenda to the next level. It is one of the top JavaScript frameworks for mobile to build an application or mobile friendly website.</p> 
           <p data-mce-="" pid="16">jQuery Mobile supports a number of user interfaces that are compatible with modern platforms such as Android, iOS and to the earliest of platforms such as Opera Mini and Nokia Symbian. With the help of PhoneGap you can integrate your jQuery web app code to an interactive iOS or Android application.</p> 
           <h3 data-mce-="">4) Sencha Ext JS</h3> 
           <p data-mce-="" pid="17"><a href="https://www.sencha.com/products/extjs/#overview" rel="nofollow" target="_blank">Ext JS</a> (aka&nbsp;<a href="http://mobileangularui.com/" rel="nofollow" target="_blank">Sencha Touch</a>the two merged in March of 2015 to form Ext JS 6.0) is said to be the only framework that offers optimal solutions to developers create fast and impressive mobile applications that work on&nbsp; Android, iOS,&nbsp; Kindle Fire, and more. It comes with lots of creative and useful components that work efficiently on all types of mobile applications. Here's <a href="https://www.sencha.com/products/extjs/#overview" rel="nofollow" target="_blank">a glimpse</a> of the features.</p> 
           <p data-mce-="" pid="19">And here is a feature comparison of the top four frameworks (keep in mind, Sencha Touch refers to Sencha Ext JS here).</p> 
           <p data-mce-="" pid="20"><a data-mce-href="http://www.valuecoders.com/blog/wp-content/uploads/2016/12/Phonegap-vs-Titanium-vs-Jquery-Mobile-vs-Sencha-Touch.png" href="http://www.valuecoders.com/blog/wp-content/uploads/2016/12/Phonegap-vs-Titanium-vs-Jquery-Mobile-vs-Sencha-Touch.png" rel="nofollow"><img alt="Phonegap vs Titanium vs Jquery Mobile vs Sencha Touch" class="fr-fin fr-dib" data-mce-src="http://www.valuecoders.com/blog/wp-content/uploads/2016/12/Phonegap-vs-Titanium-vs-Jquery-Mobile-vs-Sencha-Touch.png" height="345" src="http://www.valuecoders.com/blog/wp-content/uploads/2016/12/Phonegap-vs-Titanium-vs-Jquery-Mobile-vs-Sencha-Touch.png" width="848"></a></p> 
           <h2 data-mce-="">5) React Native</h2> 
           <p data-mce-="" pid="21">If you are fond of Native mobile applications,&nbsp; you are majorly left with two options- Xamarin and <a href="https://facebook.github.io/react-native/" rel="nofollow" target="_blank">React Native</a>.&nbsp;Xamarin allows you only to write the same kind of code while programming natively including &nbsp;similar interface design tools and similar calls.</p> 
           <p data-mce-="" pid="22"><a data-mce-href="https://facebook.github.io/react-native/docs/getting-started.html#content" href="https://facebook.github.io/react-native/docs/getting-started.html#content" target="_blank" rel="nofollow">React Native (from Facebook)</a> on the other hand,&nbsp; takes an entirely different path toward native app development. It uses JavaScript code, CSS-alike stylesheets and all-too-familiar HTML-alike tags for layouts. Here's a graph for clear understanding of React Native popularity:</p> 
           <p data-mce-="" pid="23"><a data-mce-href="http://www.valuecoders.com/blog/wp-content/uploads/2016/11/React-Native-usage.png" href="http://www.valuecoders.com/blog/wp-content/uploads/2016/11/React-Native-usage.png" rel="nofollow"><img alt="React Native usage top javascript frameworks for mobile" class="fr-fin fr-dib" data-mce-src="http://www.valuecoders.com/blog/wp-content/uploads/2016/11/React-Native-usage.png" height="499" src="http://www.valuecoders.com/blog/wp-content/uploads/2016/11/React-Native-usage.png" width="889"></a>Hence, if your team is already React friendly, then React Native could be a good choice for you.</p> 
           <p data-mce-="" pid="24">Further reading: <a data-mce-href="http://www.valuecoders.com/blog/technology-and-apps/5-reasons-choose-facebooks-reactjs/" href="http://www.valuecoders.com/blog/technology-and-apps/5-reasons-choose-facebooks-reactjs/" rel="nofollow" target="_blank">5 reasons to choose Facebooks ReactJS</a></p> 
           <h2 data-mce-="">6) Meteor</h2> 
           <p data-mce-="" pid="25"><a href="https://www.meteor.com/" rel="nofollow" target="_blank">Meteor</a> is yet another JavaScript framework that can help you develop an interactive mobile application. Meteor not only allows your JavaScript code for mobile apps but also gives you ongoing control over your app.&nbsp; With this feature, you can update the JavaScript code without developer's help and use the <a data-mce-href="http://info.meteor.com/blog/meteor-hot-code-push" href="http://info.meteor.com/blog/meteor-hot-code-push" target="_blank" rel="nofollow">hot code push function</a> to immediately deploy the changes to users on all mobile platforms.</p> 
           <p data-mce-="" pid="26">Another specialty of Meteor(version: 1.0) is that it maintains a local copy of data by implementing its miniMongo database in phone internal memory. Moreover, all of the phone-to-server data communication and synchronization is provided by Meteor. (miniMongo is a JavaScript implementation of the MongoDB API.) Here's the architecture:</p> 
           <p data-mce-="" pid="27"><a data-mce-href="http://www.valuecoders.com/blog/wp-content/uploads/2016/11/Meteor-js-architechture.png" href="http://www.valuecoders.com/blog/wp-content/uploads/2016/11/Meteor-js-architechture.png" rel="nofollow"><img alt="Meteor js architechture- top javascript frameworks for mobile" class="fr-fin fr-dib" data-mce-src="http://www.valuecoders.com/blog/wp-content/uploads/2016/11/Meteor-js-architechture.png" height="495" src="http://www.valuecoders.com/blog/wp-content/uploads/2016/11/Meteor-js-architechture.png" width="828"></a>Further reading: <a data-mce-href="http://www.valuecoders.com/blog/outsourcing-and-off-shoring/advantages-meteor-js-development/" href="http://www.valuecoders.com/blog/outsourcing-and-off-shoring/advantages-meteor-js-development/" rel="nofollow" target="_blank">Advantages of Meteor JS For Development</a></p> 
           <h2 data-mce-="">7) NativeScript</h2> 
           <p data-mce-="" pid="28"><a href="https://www.nativescript.org/" rel="nofollow" target="_blank">NativeScript</a> is a <a data-mce-href="http://www.telerik.com/" href="http://www.telerik.com/" target="_blank" rel="nofollow">Telerik</a> invention for building multi-platform native mobile applications from a single code source. The NativeScript site and GitHub page depict the runtime as empowering developers to utilize JavaScript and TypeScript (even Angular 2.0) to build native apps for&nbsp;Android, iOS, and Windows Phone and offer code over the platforms.</p> 
           <p data-mce-="" pid="29">Here is the architecture that NativeScript follows:</p> 
           <p data-mce-="" pid="30"><a data-mce-href="http://www.valuecoders.com/blog/wp-content/uploads/2016/11/nativescript-architecture.png" href="http://www.valuecoders.com/blog/wp-content/uploads/2016/11/nativescript-architecture.png" rel="nofollow"><img alt="nativescript-architecture top javascript frameworks for mobile" class="fr-fin fr-dib" data-mce-src="http://www.valuecoders.com/blog/wp-content/uploads/2016/11/nativescript-architecture.png" height="434" src="http://www.valuecoders.com/blog/wp-content/uploads/2016/11/nativescript-architecture.png" width="100%"></a></p> 
           <h2 data-mce-="">8) Rachet</h2> 
           <p data-mce-="" pid="31"><a href="http://goratchet.com/" rel="nofollow" target="_blank">Ratchet</a> is another essential element in the top JavaScript frameworks for mobile app development. It aims to provide&nbsp; developers and designers with a framework for building mobile web apps. The framework is created by the same team that brought the popular Bootstrap framework to you, which assures the high quality of Ratchet. Whether you want to target Android, iOS, or both, Ratchet 2.x is up for the job.</p> 
           <p data-mce-="" pid="32"><a data-mce-href="http://www.valuecoders.com/blog/wp-content/uploads/2016/11/ratchet-framework.jpg" href="http://www.valuecoders.com/blog/wp-content/uploads/2016/11/ratchet-framework.jpg" rel="nofollow"><img alt="ratchet-framework- top javascript frameworks for mobile" class="fr-fin fr-dib" data-mce-src="http://www.valuecoders.com/blog/wp-content/uploads/2016/11/ratchet-framework.jpg" height="232" src="http://www.valuecoders.com/blog/wp-content/uploads/2016/11/ratchet-framework.jpg" width="100%"></a></p> 
           <h2 data-mce-="">9) Mobile Angular UI</h2> 
           <p data-mce-="" pid="33">One of the top JavaScript frameworks for mobile is&nbsp;<a href="http://mobileangularui.com/" rel="nofollow" target="_blank">Mobile Angular UI</a>. If you are an Angular fan, then this framework is for you. It provides essential mobile components that&nbsp; Bootstrap 3 misses. Switches, overlays, scrollable areas, sidebars, absolute positioned top and bottom navbars that don't bounce on scroll are some of the features to mention.</p> 
           <p data-mce-="" pid="34">Further reading: <a data-mce-href="http://www.valuecoders.com/blog/technology-and-apps/advantages-of-angular-js-app-development/" href="http://www.valuecoders.com/blog/technology-and-apps/advantages-of-angular-js-app-development/" rel="nofollow" target="_blank">Top 10 Advantages of Using AngularJS for App Development </a></p> 
           <p data-mce-="" pid="35">However, with the <a data-mce-href="http://www.valuecoders.com/blog/industry-news/angularjs-2-official-release-google-team/" href="http://www.valuecoders.com/blog/industry-news/angularjs-2-official-release-google-team/" rel="nofollow" target="_blank">release of Angular 2</a>, concepts has changed a bit. So for Angular 2 admirers <a data-mce-href="https://onsen.io/angular2/" href="https://onsen.io/angular2/" rel="nofollow" target="_blank">Onsen UI 2.0</a> can be a good choice.</p> 
           <h2 data-mce-="">The Bottom Line</h2> 
           <p data-mce-="" pid="36">Choosing the right JavaScript framework for mobile is never about the number of features that the particular framework can offer. Its about the true functionality of the framework and how that functionality can be applied properly within your own&nbsp; mobile app development project.&nbsp; Hence, choose one these top JavaScript frameworks for mobile as per your project's needs to save time and cost.</p>

    function loadRichSnippetData(article) {

        var rs = {
            bodyProp: 'articleBody',
            date: moment(article.articleDate).format()
        };

        switch (article.articleType) {

            case 'news':
                rs.type = 'https://schema.org/NewsArticle';
                break;
            case 'snippet':
                rs.type = 'https://schema.org/Code';
                rs.bodyProp = '';
                break;
            case 'review':
                rs.type = 'https://schema.org/Review';
                rs.bodyProp = 'reviewBody';
                break;
            case 'tutorial':
            case 'analysis':
                rs.type = 'https://schema.org/TechArticle';
                break;
            case 'opinion':
                rs.type = 'https://schema.org/BlogPosting';
                break;
            default:
                rs.type = 'https://schema.org/Article';
                break;
        }

        article.rs = rs;
    }

    function attachPartner(article, partners) {

        article.showLogo = true;

        if (!_.isNumber(partners.lastUsed)) {
            //partners.lastUsed = 0;
            toUse = partners.lastUsed
        }else{
            toUse = partners.lastUsed + 1;
        }

        if (toUse >= partners.length || !toUse) {
            toUse = 0;
        }

        if(partners){
            article.showLogo = (partners.length < 2 && articleNumber % 2) ? false : article.showLogo;
        }else{
            article.showLogo = false;
        }

        if(!articleNumber || article.partner){
            return;
        }

        article.readyForAd = true;
        article.partner = partners[toUse];
        $scope.partners.lastUsed = toUse;
        articleNumber++;
    }


    function loadPartner(article) {
