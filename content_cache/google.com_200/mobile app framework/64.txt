      <div class="box-one col-md-12"> 
       <h3>Alloy</h3> 
       <img src="images/alloy.png" class="float-right img-responsive"> 
       <h5>Development Simplified</h5> 
       <p> Alloy is an MVC app framework built on top of Titanium that provides a simple model for separating the app user interface, business logic and data models. It is optional. It rocks. Check it out if you’re considering using Titanium.</p> 
       <div class="clearfix"></div> 
      </div> 
      <div class="box-two col-md-5 col-sm-12"> 
       <img src="images/alloy-code.png"> 
      </div> 
      <div class="box-three col-md-7 col-sm-12"> 
       <h5>Features and benefits</h5> 
       <ul> 
        <li>Built on Node.js with Backbone.js and Underscore.js support</li> 
        <li>XML and CSS-like syntax to separate layout and style from business logic</li> 
        <li>Create themes to use across apps for a common look and feel</li> 
        <li>Create modular components to improve the readability, maintainability and reusability of code.</li> 
        <li>Easily bind data sources to your app user interface</li> 
       </ul> 
       <a href="https://github.com/appcelerator/alloy" target="_blank" class="button-git">Check out Alloy on GitHub</a> 
      </div>
