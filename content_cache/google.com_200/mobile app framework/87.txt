     <h1 class="center silver">Web application framework for node.js</h1>
     <div class="fs11 silver center">
      Free web application framework for building Web sites and Web applications using JavaScript, HTML and CSS. 
      <b>From web developer for web developers</b>.
     </div>
     <br>
     <br>
     <br>
     <div class="panel">
      <div class="center">
       <img src="/img/128-modern.png" width="128" height="128" alt="Modern framework" border="0">
      </div>
      <h2>Modern framework</h2>
      <p class="p20 gray">Friendly URL, XHR, JSON, LESS CSS (CSS 3), JavaScript minifier, XSS protected, resources, modules and supports MVC architecture.</p>
     </div>
     <div class="panel">
      <div class="center">
       <img src="/img/128-websocket.png" width="128" height="128" alt="WebSockets supports" border="0">
      </div>
      <h2>WebSockets (RFC 6455)</h2>
      <p class="p20 gray">WebSockets is an advanced technology that makes interactive communication session between the user's browser and a server.</p>
     </div>
     <div class="panel">
      <div class="center">
       <img src="/img/128-stop.png" width="128" height="128" alt="No dependencies" border="0">
      </div>
      <h2>No dependencies</h2>
      <p class="p20 gray">The framework has no dependencies. Complete functionality is built in its core. You can install any module by the Node Package Manager.</p>
     </div>
     <div class="panel nbr">
      <div class="center">
       <img src="/img/128-freedom.png" width="128" height="128" alt="Gives You Freedom" border="0">
      </div>
      <h2>Gives you freedom</h2>
      <p class="p20 gray">Framework doesn't limit you. You can write whatever you want or rewrite existing functionality. It's easy, simple and safe.</p>
     </div>

      <h2 class="fl silver">partial.js / web framework features</h2>
      <div class="fr mt5 silver fs11">
       current version: 
       <span id="version" class="b black">...</span>
      </div>
     </div>
     <br>
     <ul class="features">
      <li>MIT license (free for commercial use)</li>
      <li class="b"><a href="https://github.com/petersirka/partial.js/tree/master/examples/server-sent-events" target="_blank" class="black">Supports Server Sent Events</a></li>
      <li class="b"><a href="https://github.com/petersirka/partial.js/tree/master/examples/websocket" target="_blank" class="black">Supports WebSockets</a></li>
      <li class="b"><a href="https://github.com/petersirka/partial.js/tree/master/examples/live-streaming" target="_blank" class="black">Supports live streaming (mixed)</a></li>
      <li>Serve static files &amp; Supports streaming</li>
      <li>Resources for multilingual pages</li>
      <li>Supports prefixes for <strong>mobile devices</strong></li>
      <li>Great view engine</li>
      <li>MVC architecture</li>
      <li>Fast file upload (multipart)</li>
      <li>Form data validation</li>
      <li class="b">Controller assertion testing</li>
      <li><strong>REST</strong> (GET, POST, PUT, DELETE)</li>
      <li>Custom authorization</li>
      <li>LESS CSS + auto vendor prefixes</li>
      <li>Supports JavaScript minifier</li>
      <li>XSS protected, CSRF, CORS</li>
      <li>2x enviroments - debug / release</li>
      <li>Supports custom helpers</li>
      <li><a href="https://github.com/petersirka/partial.js-modules" target="_blank" class="black b">Supports modules</a></li>
      <li>Reusable code between projects</li>
      <li><a href="https://github.com/petersirka/partial.js/tree/master/examples/email-templating" target="_blank" class="gray">SMTP sender (E-mail sender)</a></li>
      <li>Supports NoSQL embedded database</li>
      <li><a href="https://github.com/petersirka/partial.js/tree/master/examples/image-resize" target="_blank" class="gray">Image processing via GraphicsMagick</a></li>
      <li><a href="https://github.com/petersirka/partial.js/tree/master/examples/workers" target="_blank" class="gray">Supports workers</a></li>
      <li>Supports restrictions and redirects</li>
      <li>Supports custom JS and CSS compiler</li>
      <li><a href="https://github.com/petersirka/partial.js/tree/master/examples/controller-proxy" target="_blank" class="gray">Supports proxy requests</a></li>
      <li><a href="https://github.com/petersirka/partial.js/tree/master/examples/authorization-www-basic" target="_blank" class="gray">Supports basic access authentication</a></li>
      <li class="b"><a href="https://github.com/petersirka/partial.js/tree/master/examples" target="_blank" class="black">Many examples</a></li>
      <li class="b">Fast &amp; Safe</li>
      <li class="b"><a href="http://youtu.be/3GMQJki82Lo" target="_blank" class="black">Watch the video about partial.js</a></li>
