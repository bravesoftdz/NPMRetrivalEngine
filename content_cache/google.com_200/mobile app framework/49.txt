    <h1>Best Mobile Frameworks and Platforms for Hybrid HTML5 Apps</h1>
    <div id="meta">
     <span>Last updated on <strong>September 1, 2015</strong> in </span>
     <a href="http://techslides.com/category/development" title="View all posts in Development" rel="category tag">Development</a>
    </div>
    <p><img src="http://i.imgur.com/4kHlYlw.png" alt="mobile frameworks for building html5 apps"></p> 
    <p>A while ago, I got into mobile app development using HTML5 and PhoneGap. It was a great solution to create apps for iOS and Android while coding in HTML, CSS, and JavaScript. Today, we have a lot more options in this space. There are many frameworks for building mobile applications and there are also multiple platforms that package that code into a deployable apps and even provide things like monetization, analytics, and testing. Today, I will analyze the mobile development space and go over the <strong>best mobile frameworks and cross-device platforms to build and deploy hybrid mobile applications</strong>.<br> <span id="more-5725"></span></p> 
    <p>Hybrid apps are either built around a native web view container or they leverage cross-platform tools that create native apps. Since a native WebView is basically an embedded web browser inside of an application, you can just focus on building a web page and the platform will package your web code into a cross-platform mobile app. <a href="http://phonegap.com/">PhoneGap</a> is one of these platforms/frameworks and it gives you access to native features like the phone camera through powerful APIs and it can build apps for iOS, Android, and other popular platforms. <a href="https://build.phonegap.com/">PhoneGap Build</a> will compile the app code in the cloud and also has additional features.</p> 
    <p><strong>There are also many mobile frameworks which allow you to design and develop mobile apps that can be packaged with PhoneGap/Cordova</strong>:</p> 
    <p><a href="http://jquerymobile.com/">jQuery Mobile</a> – HTML5 based framework from jQuery. It has been around for a while and has recently added a neat <a href="http://themeroller.jquerymobile.com/">ThemeRoller editor</a>. I tried it and really did not like it. I thought it was bloated and slow.</p> 
    <p><a href="http://ionicframework.com/">Ionic</a> – framework that is open sourced and has a library of mobile-optimized HTML, CSS and JS components, gestures, and tools for building highly interactive apps. It is built with Sass and AngularJS. I loved it and I build a nice hybrid app with Ionic really quickly.</p> 
    <p><a href="http://lungo.tapquo.com/">Lungo</a> – an HTML5 framework for your mobile cross-device applications with nice default styles.</p> 
    <p><a href="https://software.intel.com/en-us/html5/tools">Intel XDK</a> – cross-platform HTML5 Development Tool to easily design, debug, build, and deploy HTML5 web and hybrid apps with PhoneGap/Cordova.</p> 
    <p><a href="http://www.the-m-project.org/">The M Project</a> is a mobile HTML5 JavaScript framework for creating cross-platform mobile web apps based on Backbone.js. It has a great <a href="http://www.the-m-project.org/apps/absinthe/kitchensink/index.html">kitchensink</a> example and can easily integrate with PhoneGap to deploy the app.</p> 
    <p><a href="http://famo.us/">famo.us</a> – a free, open source JavaScript framework that helps you create smooth, complex UIs for any screen.</p> 
    <p><a href="http://www.idangero.us/framework7/">Framework7</a> – F7 is a free and open source HTML mobile framework to develop hybrid mobile apps or web apps with iOS native look and feel.</p> 
    <p><a href="http://onsen.io/">Onsen UI</a> – a custom elements based HTML5 UI Framework that calls itseld the answer to PhoneGap development. You get many UI components and responsve layout with super performance.</p> 
    <p><a href="https://monaca.io/">Monaca</a> – cloud IDE, enterprise framework, and local kit to develop and manage your HTML5 hybrid app</p> 
    <p><a href="https://www.ludei.com/cocoonjs/">CocoonJS</a> – HTML5 framework that speeds up HTML5 execution to improve performance in mobile games and apps.</p> 
    <p><a href="http://justspamjustin.github.io/junior/">Junior</a> – A front-end framework for building HTML5 mobile apps with a native look and feel.</p> 
    <p><a href="http://joapp.com/">Jo</a> – open source HTML5 Mobile App Framework that integrates with PhoneGap / Cordova.</p> 
    <p><a href="http://openui5.org/">openui5</a> – open source JavaScript UI library, maintained by SAP, that lets you build enterprise-ready web applications that are responsive to all devices. You can <a href="http://openui5.blogspot.com/p/phonegap.html">package the code with PhoneGap</a> and build a mobile app for Android or iOS.</p> 
    <p><a href="http://strongloop.com/node-js/loopback/">LoopBack by StrongLoop</a> – Open Source API Framework Powered by Node.js with SDKs to Develop Hybrid or Native Apps</p> 
    <p><a href="http://www.sencha.com/products/touch/">Sencha Touch</a> is another HTML5 Mobile Framework with many components, themes, and a built-in MVC system to create powerful web apps.</p> 
    <p><a href="https://developers.google.com/web/starter-kit/">Web Starter Kit</a> – Boilerplate &amp; Tooling for Multi-Device Development from Google. You can see an example <a href="https://google.github.io/web-starter-kit/hello-world/">here</a>.</p> 
    <p><a href="http://chocolatechip-ui.com/">ChocolateChipUI</a> – open source framework for building mobile Web apps with the native look of iOS 7, Android, &amp; Windows Phone 8 using HTML5, CSS and JavaScript.</p> 
    <p><a href="http://goratchet.com/">Ratchet</a> – Build mobile apps with simple HTML, CSS, and JS components.</p> 
    <p><a href="http://impactjs.com/">ImpactJS</a> – a JavaScript Game Engine that allows you to develop stunning HTML5 Games for desktop and mobile browsers</p> 
    <p><a href="http://www.telerik.com/phonegap-ui">UI for PhoneGap</a> – build native like apps using PhoneGap</p> 
    <p><a href="https://crosswalk-project.org/">crosswalk</a> – replaces native web view with a more powerful version and integrates with Apache Cordova</p> 
    <p><a href="http://www.appgyver.com/">AppGyver</a> – calls itself the most advanced platform for building hybrid mobile apps with HTML5 and it looks really good. It’s Supersonic framework is built on Ionic, AngularJS, and suposebly brings native performance to the UI and navigation. It uses crosswalk for Android WebViews and everything is compatible with PhoneGap/Cordova.</p> 
    <p><a href="http://www.telerik.com/kendo-ui1">Kendo UI</a> is another awesome mobile framework that is fast and light with AngularJS integration and support for Bootstrap.</p> 
    <p>Kendo UI brings us back to platforms as it comes from Telerik who also offers their <a href="http://www.telerik.com/platform">Platform</a> which can help you not only build, but test, deploy, and manage your application. The point is that the mobile frameworks listed above can be used with PhoneGap or other mobile platforms that utlize the web view model. <strong>Here is a list of Mobile Platforms for Deploying Hybrid Apps</strong>:</p> 
    <p><a href="http://phonegap.com/">PhoneGap</a> is still the most popular platform to build and deploy mobile apps. It supports a ton of mobile frameworks and will build to many devices. Some argue that it is the <a href="http://www.goldengekko.com/?blog_post=mobile-app-cross-platform-solution-works">best</a>.</p> 
    <p><a href="http://www.telerik.com/platform">Telerik Platform</a> allows you to build mobile apps with JavaScript and enables cross-platform mobile development for iOS, Android and Windows Phone. It has a great mobile UI library, can integrate with Apache Cordova, but it also can deploy the app itself and provides many tools and services with its platform. It also has <a href="http://www.telerik.com/nativescript">NativeScript</a> which lets you develop truly native apps with JavaScript and CSS. NativeScript has direct access to native platform APIs using JavaScript VMs that contain a bunch of APIs which let you configure the JavaScript environment (<a href="http://developer.telerik.com/featured/nativescript-works/">how NativeScript works</a>).</p> 
    <p><a href="https://trigger.io/">Trigger.io</a> is a simple native wrapper around your html5 code with powerful APIs. It lets you create native apps for multiple platforms from a single HTML5 codebase. It claims to be <a href="http://trigger.io/cross-platform-application-development-blog/2012/02/24/why-trigger-io-doesnt-use-phonegap-5x-faster-native-bridge/">5 times faster than PhoneGap</a></p> 
    <p><a href="http://rhomobile.com/">RhoMobile</a> is an open-source mobile app platform by Motorola Solutions for creating cross-device apps. It uses the Ruby-based framework Rhodes and by using the Rhodes API you can build your app for many devices. Here is a good <a href="http://rhomobile.com/downloads/put-your-mobile-app-platform-to-the-test.pdf">infographic on comparing RhoMobile to other platforms</a>.</p> 
    <p><a href="http://appcelerator.com/">Appcelerator Titanium</a> utilizes JavaScript, HTML5, CSS3, PHP, Ruby, and Python to create applications that run on iOS and Android, and compile to native code. It comes with its <a href="http://www.appcelerator.com/platform/alloy/">Alloy framework</a>. Here is a good <a href="http://strongloop.com/strongblog/titanium-vs-phonegap-cross-platform-mobile-framework/">comparison of Titatium to PhoneGap</a>.</p> 
    <p><a href="http://www.mosync.com/">MoSync</a> is a cross-platform SDK and HTML5 tool to create mobile apps. It supports C/C++ or HTML5 and it can build and deploy apps to multiple platforms. Here is how <a href="http://www.osd.net/blog/mobile-development/why-mosync-could-be-a-better-alternative-to-phonegap/">MoSync compares to PhoneGap</a>.</p> 
    <p><a href="https://www.madewithmarmalade.com/marmalade">Marmalade</a> has an SDK that enables developers to deploy code across multiple platforms and devices from a single code base in C++, Lua, HTML or Objective-C. <a href="https://www.madewithmarmalade.com/products/web-marmalade">Marmalade Web</a> lets you code primarily in HTML5, CSS3 and JavaScript with a web-native bridge.</p> 
    <p>I didn’t mention <a href="http://xamarin.com">Xamarin PLatform</a> because you need to write in C# instad of HTML5 but it is pretty <a href="http://www.eastbanctech.com/10-reasons-for-choosing-xamarin-cross-platform-mobile-development/">compelling</a>. Same applies to <a href="http://www.rubymotion.com/">RubyMotion</a> where you build apps in Ruby and <a href="http://kivy.org/">Kivy</a> where you can build and deploy apps with Python. Same applies to <a href="https://www.embarcadero.com/products/rad-studio/firemonkey">FireMonkey</a>.</p> 
    <p>Finally, if you are looking for a good MEAP (Mobile Enterprise Application Platform), take a look at <a href="http://www.kony.com/">Kony</a>, <a href="http://www.verivo.com/">Verivo</a>, <a href="http://www-01.ibm.com/software/mobile-solutions/worklight/components/mobile-development-platform/">IBMWorkLight</a>, or <a href="http://www.outsystems.com/platform/madp/">OutSystems</a>.</p>
