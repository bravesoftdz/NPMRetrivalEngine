           <h1 class="entry-title">Node.js Text to Speech using AWS Polly</h1> 
           <span class="entry-author"> Posted by <span class="author vcard entry-author-link"> <a href="https://www.cronj.com/blog/author/akash/" title="Posts by Akash Budhia" rel="author">Akash Budhia</a> </span> on June 6, 2017 </span>

          <div class="entry-content"> 
           <figure> 
            <a class="swipebox" href="https://www.cronj.com/blog/wp-content/uploads/AKASH.jpg" title="Node.js Text to Speech using AWS Polly"> <img width="732" height="393" src="https://www.cronj.com/blog/wp-content/uploads/AKASH-732x393.jpg" class="attachment-blog-page size-blog-page wp-post-image" alt="AWS Polly" srcset="https://www.cronj.com/blog/wp-content/uploads/AKASH-732x393.jpg 732w, https://www.cronj.com/blog/wp-content/uploads/AKASH-300x161.jpg 300w, https://www.cronj.com/blog/wp-content/uploads/AKASH-768x412.jpg 768w, https://www.cronj.com/blog/wp-content/uploads/AKASH-600x322.jpg 600w, https://www.cronj.com/blog/wp-content/uploads/AKASH.jpg 932w" sizes="(max-width: 732px) 100vw, 732px"> </a> 
           </figure> 
           <p>Amazon has made it easy for Developers to build&nbsp;applications which can&nbsp;talk. The new AWS Polly services allows you to pass a text and Amazon turns it into a life like speech in real-time. It supports &gt;<a href="http://docs.aws.amazon.com/polly/latest/dg/SupportedLanguage.html">24&nbsp;languages</a>&nbsp;in varied male and female voices&nbsp;and the output is pretty&nbsp;impressive when compared to similar offerings. Advanced users can also make use of SSML, custom Lexicons(dictionary), Speech Marks and more to customize the speeches as per their need. You can signup/login&nbsp;to your AWS Console to see Polly in action and play-around with it&nbsp;<a href="https://console.aws.amazon.com/polly/home/SynthesizeSpeech">here</a>.</p> 
           <p>With a free-tier offering of 5 million characters and pay-as-you-go&nbsp;$4.00 per million thereafter&nbsp;makes it a ‘must&nbsp;add’ magic&nbsp;ingredient&nbsp;to your apps! Moreover the responses can be saved and cached for offline replay which can further&nbsp;reduce the costs significantly.</p> 
           <p><img class="size-full wp-image-2653" src="https://www.cronj.com/blog/wp-content/uploads/AWS-Polly.jpg" alt="AWS Polly" width="932" height="490" srcset="https://www.cronj.com/blog/wp-content/uploads/AWS-Polly.jpg 932w, https://www.cronj.com/blog/wp-content/uploads/AWS-Polly-300x158.jpg 300w, https://www.cronj.com/blog/wp-content/uploads/AWS-Polly-768x404.jpg 768w, https://www.cronj.com/blog/wp-content/uploads/AWS-Polly-600x315.jpg 600w, https://www.cronj.com/blog/wp-content/uploads/AWS-Polly-732x385.jpg 732w" sizes="(max-width: 932px) 100vw, 932px">AWS Polly</p> 
           <p>Let’s have a&nbsp;look at how easy is it to write a script which&nbsp;uses&nbsp;the AWS Polly service to add&nbsp;voice to your texts. In most cases you will need to stream the audio response to your front-end interface, for the sake of simplicity we will go ahead and create a quick Node.js script which plays it directly on the host’s&nbsp;speaker. For this tutorial we will be assuming that you are familiar with the basics of Node.js. You will also need a registered <a href="http://console.aws.amazon.com">AWS Account</a> and an authorized IAM User Access Key and Secret.</p> 
           <p>To begin, initialize a Node.js project using the <code>npm init</code> command.</p> 
           <p>Following npm&nbsp;modules need to be installed to the project:</p> 
           <ol> 
            <li><a href="https://www.npmjs.com/package/aws-sdk">aws-sdk</a> &nbsp;– AWS JavaScript SDK which provides interface to use the Polly service. <pre class="line-numbers"><code class="language-javascript">npm install aws-sdk --save</code></pre> </li> 
            <li><a href="https://www.npmjs.com/package/speaker">speaker</a>&nbsp;– Stream PCM audio to the speaker. It can also be used to stream mp3 audio to a browser client. <pre class="line-numbers"><code class="language-javascript">npm install speaker --save</code></pre> </li> 
           </ol> 
           <p>Create a generic&nbsp;file&nbsp;<em>polly.js</em> which can&nbsp;directly&nbsp;be included in any&nbsp;node application&nbsp;to convert text to speech. In <em>polly.js </em>require&nbsp;the discussed&nbsp;modules and configure your AWS Credentials.</p> 
           <pre class="line-numbers" data-src="/polly.js"><code class="language-javascript">var AWS = require('aws-sdk');
var Speaker = require('speaker');
var Stream = require('stream');

//Warning: Don't hardcode your AWS Keys
//Read more at http://docs.aws.amazon.com/sdk-for-javascript/v2/developer-guide/setting-credentials-node.html
var Polly = new AWS.Polly({
    region: 'eu-west-1',
    accessKeyId: 'REPLACETHISWITHYOURACCESSKEYID',
    secretAccessKey: 'rePlaceThisWithYourSecretAccessKey1234567890'
});</code></pre> 
           <p><strong><br> Warning!</strong>&nbsp;Don’t hardcode&nbsp;your AWS Keys. Read more on <a href="http://docs.aws.amazon.com/sdk-for-javascript/v2/developer-guide/setting-credentials-node.html">AWS recommendation for storing Access Keys</a>.</p> 
           <p>Prepare the Speaker to receive PCM audio from Polly. Parameters will depend on the output type and can be checked&nbsp;from <a href="http://docs.aws.amazon.com/polly/latest/dg/API_SynthesizeSpeech.html">here</a>.</p> 
           <pre class="line-numbers" data-src="/polly.js"><code class="language-javascript">var getPlayer = function() {
    return new Speaker({
        channels: 1,
        bitDepth: 16,
        sampleRate: 16000
    });
}</code></pre> 
           <p>Since this will be a reusable wrapper function, the getPlayer function will return a new Speaker instance for every call.</p> 
           <p>Once the Speaker instance is pre-configured, all we need to do is call the Polly.synthesizeSpeech API with the text and&nbsp;additional parameters and stream the&nbsp;response to a new Speaker instance. The speak function is exported and can be&nbsp;called from any JS&nbsp;code in your application.</p> 
           <pre class="line-numbers" data-src="/polly.js"><code class="language-javascript">var params = { OutputFormat: 'pcm', VoiceId: 'Raveena' }
var speak = function(text) {
    params.Text = text;
    Polly.synthesizeSpeech(params, function(err, res) {
        if (err) {
            console.log('err', err)
        } else if (res &amp;&amp; res.AudioStream instanceof Buffer) {
            var bufferStream = new Stream.PassThrough()
            bufferStream.end(res.AudioStream)
            bufferStream.pipe(getPlayer());
        }
    })
}
module.exports = { Speak: speak };</code></pre> 
           <p>VoiceId can be chosen from the Polly <a href="https://console.aws.amazon.com/polly/home/SynthesizeSpeech">console</a> and <a href="http://docs.aws.amazon.com/polly/latest/dg/API_Voice.html">valid&nbsp;ids</a>. Additional parameters are defined <a href="http://docs.aws.amazon.com/polly/latest/dg/API_SynthesizeSpeech.html">here</a>.</p> 
           <p>We are all set. Now, from any file, say&nbsp;<em>index.js</em>, require&nbsp;<em>polly.js</em> and call the Speak function:</p> 
           <pre class="line-numbers" data-src="/index.js"><code class="language-javascript">var Polly = require('./polly');
Polly.Speak('Node.js is beautiful and so is AWS Polly!');</code></pre> 
           <p>Run the <em>index.js</em> file using <code>node index.js</code>&nbsp; command and let your computer talk to you ?</p> 
           <p>As mentioned, the script used in this example will stream audio to the host speaker, but with little work it can also be streamed to a browser or a phone app. Current&nbsp;example&nbsp;will come in handy if you are working on one of those IoT projects on&nbsp;a Raspberry Pi or similar.</p> 
           <p>In comments let us know how you used Polly for your project.</p>
