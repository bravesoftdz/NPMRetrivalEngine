<h3>
Webhooks via Cloud Functions&nbsp;</h3>
In API.AI, Webhook integrations allow you to pass information from a matched intent into a web service and get a result from it. Read on to learn how to request parade info from Cloud Functions.<br />
<ol>
<li>Go to console.cloud.google.com. Log in with your own account and create a new project.&nbsp;</li>

           <div class="separator" style="clear: both; text-align: center;"> 
            <a href="https://1.bp.blogspot.com/-C1W9eH4VTRc/WZPCPUFFT0I/AAAAAAAAEOs/l3FfRcqKoAMLE2VhtnCniHZ31K-auBx9wCEwYBhgL/s1600/conversational-app-7.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" data-original-height="304" data-original-width="595" height="326" src="https://1.bp.blogspot.com/-C1W9eH4VTRc/WZPCPUFFT0I/AAAAAAAAEOs/l3FfRcqKoAMLE2VhtnCniHZ31K-auBx9wCEwYBhgL/s640/conversational-app-7.png" width="640"></a>
           </div> 
           <div> 
            <br>
           </div> 
           <div> 
           </div> 
           <div>
             Your final step is to go to the API.AI console, then click the Fulfillment tab. Enable webhook and paste the URL above into the URL field.&nbsp;
           </div> 
           <div class="separator" style="clear: both; text-align: center;"> 
            <a href="https://1.bp.blogspot.com/-H8rU2wLKpbw/WZPDKEIE3LI/AAAAAAAAEO8/soalOhYFvy4zQQpqOMCnWxuSLTzRa9F8gCLcBGAs/s1600/conversational-app-3.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" data-original-height="659" data-original-width="792" height="532" src="https://1.bp.blogspot.com/-H8rU2wLKpbw/WZPDKEIE3LI/AAAAAAAAEO8/soalOhYFvy4zQQpqOMCnWxuSLTzRa9F8gCLcBGAs/s640/conversational-app-3.png" width="640"></a>
           </div> 
           <div> 
            <br>
           </div> 
           <div> 
            <br>
           </div> 
           <div>
             With API.AI, weve built a chatbot that can converse with a human by text. Next, lets give the bot ears to listen with Cloud Speech API, eyes to see with Cloud Vision API, a mouth to talk with the iOS text-to-speech SDK and brains for translating languages with Cloud Translation API.
           </div> 
           <div> 
            <br>
           </div> 
           <h3> Using Cloud Speech API&nbsp;</h3> 
           <div>
             Cloud Speech API includes an iOS 
            <a href="https://github.com/GoogleCloudPlatform/ios-docs-samples/tree/master/speech/Objective-C" target="_blank">sample</a> app. Its quite straightforward to integrate the 
            <a href="https://github.com/GoogleCloudPlatform/ios-docs-samples/tree/master/speech/Objective-C/Speech-gRPC-Nonstreaming" target="_blank">gRPC non-streaming sample app</a> into our chatbot app. Youll need to acquire an 
            <a href="https://cloud.google.com/storage/docs/json_api/v1/how-tos/authorizing#APIKey" target="_blank">API key</a> from Google Cloud Console and replace this 
            <a href="https://github.com/GoogleCloudPlatform/ios-docs-samples/blob/master/speech/Objective-C/Speech-gRPC-Nonstreaming/Speech/SpeechRecognitionService.m#L23" target="_blank">line</a> in 
            <code>SpeechRecognitionService.m</code> with your API key.
            <br> 
            <br> 
            <pre><code>#define API_KEY @"YOUR_API_KEY"</code></pre> 
            <h3> Landmark detection&nbsp;</h3> 
            <pre><code> NSDictionary *paramsDictionary =
  @{@"requests":@[
        @{@"image":
            @{@"content":binaryImageData},
          @"features":@[
              @{@"type":@"LANDMARK_DETECTION", @"maxResults":@1}]}]};</code></pre> 
            <br> Follow this 
            <a href="https://github.com/GoogleCloudPlatform/cloud-vision/tree/master/ios" target="_blank">example</a> to use Cloud Vision API on iOS. Youll need to replace the label and face detection with landmark detection as shown below.&nbsp;
           </div> 
           <br> You can use the same API key you used for Cloud Speech API.&nbsp; 
           <br> 
           <h3> Text to speech</h3> iOS 7+ has a built-in text-to-speech SDK, 
           <a href="https://developer.apple.com/documentation/avfoundation/avspeechsynthesizer?preferredLanguage=occ" target="_blank">AVSpeechSynthesizer</a>. The code below is all you need to convert text to speech.
           <br> 
           <br> 
           <pre><code>#import 
             <avfoundation oundation.h="">
               AVSpeechUtterance *utterance = [[AVSpeechUtterance alloc] initWithString:message]; AVSpeechSynthesizer *synthesizer = [[AVSpeechSynthesizer alloc] init]; [synthesizer speakUtterance:utterance];
             </avfoundation></code></pre> 
           <h3> Supporting multiple languages</h3> 
           <iframe allowfullscreen frameborder="0" height="360" src="https://www.youtube.com/embed/Oy4oNNd1aGw" width="640"></iframe> Supporting additional languages in Cloud Speech API is a one-line change on the iOS client side. (Currently, there's no support for mixed languages.) For 
           <a href="https://youtu.be/Oy4oNNd1aGw" target="_blank">Chinese</a>, replace 
           <a href="https://github.com/google/ios-chatbot/blob/master/ChatBot/ChatBot/SpeechRecognitionService.m#L73" target="_blank">this</a> line in 
           <a href="https://github.com/google/ios-chatbot/blob/master/ChatBot/ChatBot/SpeechRecognitionService.m" target="_blank">SpeechRecognitionService.m</a>:&nbsp; 
           <br> 
           <br> 
           <pre><code>recognitionConfig.languageCode = @"en-US";</code></pre> with 
           <br> 
           <pre><code>recognitionConfig.languageCode = @"zh-Hans";</code></pre> 
           <br> To support additional text-to-speech languages, add this line to the code: 
           <br> 
           <br> 
           <pre><code>#import 
             <avfoundation oundation.h="">
               AVSpeechUtterance *utterance = [[AVSpeechUtterance alloc] initWithString:message]; utterance.voice = [AVSpeechSynthesisVoice voiceWithLanguage:@"zh-Hans"]; AVSpeechSynthesizer *synthesizer = [[AVSpeechSynthesizer alloc] init]; [synthesizer speakUtterance:utterance];
             </avfoundation></code></pre> Both Cloud Speech API and Apples 
           <a href="https://developer.apple.com/documentation/avfoundation/avspeechsynthesisvoice/1619698-language?language=objc" target="_blank">AVSpeechSynthesisVoice</a> support 
           <a href="https://tools.ietf.org/html/bcp47" target="_blank">BCP-47 language code</a>.
           <br> 
           <avfoundation oundation.h="">
            <br>
           </avfoundation> 
           <avfoundation oundation.h="">
            Cloud Vision API landmark detection currently only supports English, so youll need to use the 
            <a href="https://cloud.google.com/translate/" target="_blank">Cloud Translation API</a> to translate to your desired language after receiving the English-language landmark description. (You would use Cloud Translation API similarly to Cloud Vision and Speech APIs.)&nbsp;
           </avfoundation> 
           <br> 
           <div> 
            <avfoundation oundation.h="">
             <br>
            </avfoundation>
           </div> 
           <div> 
            <avfoundation oundation.h="">
             On the API.AI side, youll need to create a new agent and 
             <a href="https://api.ai/docs/agents#agent-settings" target="_blank">set its language to Chinese</a>. One agent can support only one language. If you try to use the same agent for a second language, machine learning wont work for that language.&nbsp;
            </avfoundation>
           </div> 
           <div class="separator" style="clear: both; text-align: center;"> 
            <a href="https://4.bp.blogspot.com/-9r1F9ApPytM/WZPD2LBUQPI/AAAAAAAAEPE/KJBCF3Lezrczbt6uWsxa-gnC8WkebE-egCLcBGAs/s1600/conversational-app-6.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" data-original-height="471" data-original-width="793" height="380" src="https://4.bp.blogspot.com/-9r1F9ApPytM/WZPD2LBUQPI/AAAAAAAAEPE/KJBCF3Lezrczbt6uWsxa-gnC8WkebE-egCLcBGAs/s640/conversational-app-6.png" width="640"></a>
           </div> 
           <div> 
            <avfoundation oundation.h="">
             Youll also need to create all intents and entities in Chinese.&nbsp;
            </avfoundation>
           </div> 
           <div class="separator" style="clear: both; text-align: center;"> 
            <a href="https://3.bp.blogspot.com/-0R2C503CVC4/WZPEBsOM3aI/AAAAAAAAEPI/TJj3tubaxyAS7zIEbxpTz3JizrJRpz-1wCLcBGAs/s1600/conversational-app-12.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" data-original-height="1257" data-original-width="1073" height="640" src="https://3.bp.blogspot.com/-0R2C503CVC4/WZPEBsOM3aI/AAAAAAAAEPI/TJj3tubaxyAS7zIEbxpTz3JizrJRpz-1wCLcBGAs/s640/conversational-app-12.png" width="546"></a>
           </div> 
           <div> 
            <avfoundation oundation.h="">
             And youre done!&nbsp;
            </avfoundation>Youve just built a simple tour guide chatbot that supports English and Chinese.
