              <pre><code>var msg = new SpeechSynthesisUtterance('Hello World');
window.speechSynthesis.speak(msg);
</code></pre> 
              <p>With this, perhaps you do not need to use a web service at all.</p>

              <p>hey run this code it will take input as audio(microphone) and convert into text then audio play</p> 
              <pre><code>&lt;head&gt;
&lt;title&gt;MY Echo&lt;/title&gt;
&lt;script src="http://code.responsivevoice.org/responsivevoice.js"&gt;&lt;/script&gt;
&lt;link rel="stylesheet" type="text/css" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/4.6.1/css/font-awesome.min.css" /&gt;
&lt;style type="text/css"&gt;
    body {
        font-family: verdana;
    }

    #result {
        height: 100px;
        border: 1px solid #ccc;
        padding: 10px;
        box-shadow: 0 0 10px 0 #bbb;
        margin-bottom: 30px;
        font-size: 14px;
        line-height: 25px;
    }

    button {
        font-size: 20px;
        position: relative;
        left: 50%;
    }
&lt;/style&gt;

&lt;h4 align="center"&gt;Speech to text converter in JS&lt;/h4&gt;
&lt;div id="result"&gt;&lt;/div&gt; &lt;button onclick="startConverting();"&gt;&lt;i class="fa fa-microphone"&gt;&lt;/i&gt;&lt;/button&gt;
&lt;script type="text/javascript"&gt;
    var r = document.getElementById('result');

    function startConverting() {
        if ('webkitSpeechRecognition' in window) {
            var speechRecognizer = new webkitSpeechRecognition();
            speechRecognizer.continuous = true;
            speechRecognizer.interimResults = true;
            speechRecognizer.lang = 'en-IN';
            speechRecognizer.start();
            var finalTranscripts = '';
            speechRecognizer.onresult = function(event) {
                var interimTranscripts = '';
                for (var i = event.resultIndex; i &lt; event.results.length; i++) {
                    var transcript = event.results[i][0].transcript;
                    transcript.replace("\n", "&lt;br&gt;");
                    if (event.results[i].isFinal) {
                        finalTranscripts += transcript;
                        var speechresult = finalTranscripts;
                        console.log(speechresult);
                        if (speechresult) {
                            responsiveVoice.speak(speechresult, "UK English Female", {
                                pitch: 1
                            }, {
                                rate: 1
                            });
                        }
                    } else {
                        interimTranscripts += transcript;
                    }
                }
                r.innerHTML = finalTranscripts + '&lt;span style="color:#999"&gt;' + interimTranscripts + '&lt;/span&gt;';
            };
            speechRecognizer.onerror = function(event) {};
        } else {
            r.innerHTML = 'Your browser is not supported. If google chrome, please upgrade!';
        }
    }
&lt;/script&gt;
&lt;/body&gt;

&lt;/html&gt;`
</code></pre>
