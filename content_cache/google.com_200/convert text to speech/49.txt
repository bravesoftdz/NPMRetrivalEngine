         <h1 class="entry-title">Speech Recognition using Google Speech API</h1>
         <div class="entry-meta"></div>

         <p>Google has a great Speech Recognition API. This API converts spoken text (microphone) into written text (Python strings), briefly Speech to Text. You can simply speak in a microphone and Google API will translate this into written text. The API has excellent results for English language.</p>
         <p>Google has also created the JavaScript Web Speech API, so you can recognize speech also in JavaScript if you want, here’s the link: <a href="https://www.google.com/intl/en/chrome/demos/speech.html" class="externalLink" target="_blank" title="Google Web Speech ">https://www.google.com/intl/en/chrome/demos/speech.html</a>. To use it on the web you will need Google Chrome version 25 or later.</p>

         <h2>Installation</h2>
         <p>Google Speech API v2 is limited to 50 queries per day. Make sure you have a good microphone.<br> Are you are looking for <a href="https://pythonspot.com/en/speech-engines-with-python-tutorial/">text to speech</a> instead?</p>
         <p>This is the installation guide for Ubuntu Linux. But this will probably work on other platforms is well. You will need to install a few packages: PyAudio, PortAudio and SpeechRecognition. PyAudio 0.2.9 is required and you may need to compile that manually.</p>

         <h2>Program</h2>
         <p>This program will record audio from your microphone, send it to the speech API and return a Python string.</p>
         <p>The audio is recorded using the speech recognition module, the module will include on top of the program. Secondly we send the record speech to the Google speech recognition API which will then return the output.<br> <strong>r.recognize_google(audio)</strong> returns a string.</p>
         <div class="wp_syntax">
          <table>
           <tbody>
            <tr>
             <td class="code"><pre class="python" style="font-family:monospace;"><span style="color: #808080; font-style: italic;">#!/usr/bin/env python3</span>
<span style="color: #808080; font-style: italic;"># Requires PyAudio and PySpeech.</span>
&nbsp;
<span style="color: #00008b;font-weight:bold;">import</span> speech_recognition <span style="color: #00008b;font-weight:bold;">as</span> sr
&nbsp;
<span style="color: #808080; font-style: italic;"># Record Audio</span>
r <span style="color: #66cc66;">=</span> sr.<span style="color: black;">Recognizer</span><span style="color: black;">(</span><span style="color: black;">)</span>
<span style="color: #00008b;font-weight:bold;">with</span> sr.<span style="color: black;">Microphone</span><span style="color: black;">(</span><span style="color: black;">)</span> <span style="color: #00008b;font-weight:bold;">as</span> source:
    <span style="color: #00008b;font-weight:bold;">print</span><span style="color: black;">(</span><span style="color: #00008b;">"Say something!"</span><span style="color: black;">)</span>
    audio <span style="color: #66cc66;">=</span> r.<span style="color: black;">listen</span><span style="color: black;">(</span>source<span style="color: black;">)</span>
&nbsp;
<span style="color: #808080; font-style: italic;"># Speech recognition using Google Speech Recognition</span>
<span style="color: #00008b;font-weight:bold;">try</span>:
    <span style="color: #808080; font-style: italic;"># for testing purposes, we're just using the default API key</span>
    <span style="color: #808080; font-style: italic;"># to use another API key, use `r.recognize_google(audio, key="GOOGLE_SPEECH_RECOGNITION_API_KEY")`</span>
    <span style="color: #808080; font-style: italic;"># instead of `r.recognize_google(audio)`</span>
    <span style="color: #00008b;font-weight:bold;">print</span><span style="color: black;">(</span><span style="color: #00008b;">"You said: "</span> + r.<span style="color: black;">recognize_google</span><span style="color: black;">(</span>audio<span style="color: black;">)</span><span style="color: black;">)</span>
<span style="color: #00008b;font-weight:bold;">except</span> sr.<span style="color: black;">UnknownValueError</span>:
    <span style="color: #00008b;font-weight:bold;">print</span><span style="color: black;">(</span><span style="color: #00008b;">"Google Speech Recognition could not understand audio"</span><span style="color: black;">)</span>
