        <h2 class="text-center">How to convert text to speech (speech synthesis) in Cordova</h2>

        <article id="article-content" itemscope itemtype="http://schema.org/BlogPosting">
         <p>Speech synthesis is the computer-generated simulation of human speech. It is used to translate written information into aural information where it is more convenient, especially for mobile applications such as voice-enabled e-mail and unified messaging. As a developer that uses Javascript to create hybrid applications, we assume that you know the Speech Synthesis API available in most of the web browsers. The usage of this API is really easy, however it isn't available within&nbsp;a cordova application, therefore you need to resort to the native API of the device through a cordova plugin.</p> 
         <p>In this article you will learn how to convert text to speech easily in your cordova project.</p> 
         <h2>Requirements</h2> 
         <p>In order to use the native Speech Synthesis API of the device, we are going to use a plugin for it. <a href="https://github.com/vilic/cordova-plugin-tts" target="_blank" rel="nofollow noopener noreferrer">Cordova plugin TTS</a> is a plugin written by <a href="https://github.com/vilic" target="_blank" rel="nofollow noopener noreferrer">@vilic</a> that allows you to use the native speech synthesis API of the device through Javascript. This plugin provides support for the following platforms:</p> 
         <ul> 
          <li>iOS 7+</li> 
          <li>Windows Phone 8</li> 
          <li>Android 4.0.3+ (API Level 15+)</li> 
         </ul> 
         <p>Install the plugin in your project executing the following command in a terminal:</p> 
         <pre class="language-batch"><code>cordova plugin add cordova-plugin-tts</code></pre> 
         <p>After the installation you'll be able to use the TTS object in the window. For more information <a href="https://github.com/vilic/cordova-plugin-tts" target="_blank" rel="nofollow noopener noreferrer">visit the official Github repository here</a>. This plugin uses the AVSpeechSynthesizer class in iOS, the TextToSpeech class in Android and the&nbsp;<span>Windows Phone Speech Synthesis class in Windows Phone.</span></p> 
         <h2>Text to speech</h2> 
         <p>To convert text to voice with Javascript&nbsp;use the speak method of the TTS object. This method expects as first parameter an object with up to 3 properties:</p> 
         <div class="table-responsive"> 
          <table class="table table-hover"> 
           <thead> 
            <tr> 
             <td>Property name</td> 
             <td>Type</td> 
             <td>Description</td> 
            </tr> 
           </thead> 
           <tbody> 
            <tr> 
             <td><code>text</code></td> 
             <td>String</td> 
             <td>Specifies the text that will be spoken.</td> 
            </tr> 
            <tr> 
             <td><code>locale</code></td> 
             <td>String</td> 
             <td>A 4 character code that specifies the language that should be used to synthesize the text.</td> 
            </tr> 
            <tr> 
             <td><code>rate</code></td> 
             <td>Float</td> 
             <td>the speaking rate of the SpeechSynthesizer object, from 0.1&nbsp;to 1</td> 
            </tr> 
           </tbody> 
          </table> 
         </div> 
         <p>And can be used as in the following example:</p> 
         <pre class="language-javascript"><code>TTS.speak({
    text: 'Good morning, how are you?',
    locale: 'en-US',
    rate: 1
}, function () {
    console.log('Text succesfully spoken');
}, function (reason) {
    console.log(reason);
});</code></pre> 
         <p>It's worth to say that if you want to synthesize text when your app starts, that you need to wrap your code inside the <code>deviceready</code> event:</p> 
         <pre class="language-javascript"><code>document.addEventListener('deviceready', function () {
    TTS.speak('Welcome to my awesome app', function () {
        console.log('Ready !');
    }, function (reason) {
        console.log(reason);
    });
});</code></pre> 
         <h2>Why there's no stop method?</h2> 
         <p>In technical terms with the native API, you are able to stop any speech utterance object. However, with this plugin there's no way to stop the speech synthesis (at least not in the traditional way) if it's already running and the reason is very simple (<strong>no, the plugin is not bad or incomplete</strong>).</p> 
         <p>As we are using Cordova, we are limited to send and receive some data, we can interact with native code and Javascript through callbacks but we cannot interact dinamically <strong>with a single native instance of the plugin class</strong>&nbsp;because the execution of every Javascript method <strong>creates a new instance of the native class of the plugin</strong>.</p> 
         <p><img src="https://ourcodeworld.com/public-media/gallery/gallery-5893080836aa2.jpg" alt="Confused meme speech synthesis" class="img img-thumbnail img-responsive center-block"></p> 
         <p>Everytime you execute some action in this case the <code>start</code>, Cordova instantiates e.g the TTS class in Java that extends CordovaPlugin <strong>once</strong> and stores the TTS object in a local variable in the class. From the stored TTS object, you can execute the stop method and the speech synthesis should stop as planned. But, if you execute another method from Javascript in this case the <code>stop</code>, then a new instance of the TTS class in Java that extends CordovaPlugin will be created (that means that we have not access to the same TTS object from the start method anymore). The same thing happens with iOS and Windows.</p> 
         <p>Although there's no stop method included in the plugin, you can use a little trick to achieve it. <strong>Speak an empty string and the Speech synthesis will stop !</strong> You can extend the plugin and create your own stop method:</p> 
         <pre class="language-javascript"><code>// Extend the TTS object with the new stop method.
TTS["stop"] = function(){
    TTS.speak({text: ''});
};

// Speak some text
TTS.speak({
    text: 'Hola buenos días. Hoy para el desayuno hay pan con mantequilla.',
    locale: 'es-ES',
    rate: 0.75
}, function () {
    console.log('success');
}, function (reason) {
    console.log(reason);
});

// Stop after 3 seconds
setTimeout(function() {
    TTS.stop();
    console.log("Speech synthesis stopped");
}, 3000);</code></pre> 
         <p>Happy coding <i class="fa fa-heart"></i>!</p>

        $(function () {
            var _search_api = {
                getSuggestions: function (searchTerm) {
                    return $.ajax({
                        url: '/ajax-endpoint/search',
                        data: {
                            q: searchTerm
                        },
                        type:"GET",
                        dataType: 'JSON'
                    });
                },
                process :  function(title,data,noCase){
                    var html = '<h4>'+title+' <small>('+data.length+')</small></h4>';

                    if(!data.length){
                        return html += '<p>'+noCase+'</p>';
                    }

                    html += '<ul class="list-material">';

                    for(var c = 0;c < data.length;c++){
                        var row = data[c];

                        html += '<li data-toggle="tooltip" data-placement="top" data-original-title="'+row.name+'">';
                        html +=     '<a href="'+row.path+'" class="visible" style="padding: 0px;">';
                        html +=         '<div class="list-content">';
                        html +=             '<span class="title">'+row.name+'</span>';
                        html +=             '<span class="caption">'+row.date+'</span>';
                        html +=         '</div>';
                        html +=     '</a>';
                        html += '</li>';
                    }

                    html += '</ul>';


                    return html;
                }
            };

            $('#search-input-ourcodeworld').bind("keyup change", OurCodeWorld.debounce(function () {
                var value = $(this).val();

                if (!value) {
                    return;
                }

                _search_api.getSuggestions(value).then(function (data) {
                    var articles = data.data;
                    var group =  {
                        left: articles.splice(0, Math.ceil(articles.length/2)),
                        right: articles
                    };
                    
                    $('#search-layout-zone-articles').html(_search_api.process("Articles", group.left ,"No articles were found"));
                    $('#search-layout-zone-news').html(_search_api.process("Articles", group.right ,"No articles were found"));
                }, function (err) {
                    console.log(err);
                });
            }, 300));

            OurCodeWorld.init();
            Layout.init();
        });
    </script>
