         <h2>Convert Text to Speech using FreeTTS API</h2> 
         <p>In this tutorial, I am sharing simple program about <strong>How to Convert Text to Speech using FreeTTS API</strong>. The attached program were tested with environment and output has been shared in the same post.</p> 
         <h4>Folder Structure</h4> 
         <p><img data-attachment-id="308" data-permalink="http://www.dineshkrish.com/java/freetts-tutorial/convert-text-to-speech-using-freetts-api/attachment/convert-text-to-speech-using-freetts-api-2/" data-orig-file="https://i0.wp.com/www.dineshkrish.com/wp-content/uploads/2016/11/Convert-Text-to-Speech-using-FreeTTS-API.png?fit=285%2C346" data-orig-size="285,346" data-comments-opened="1" data-image-meta="{&quot;aperture&quot;:&quot;0&quot;,&quot;credit&quot;:&quot;&quot;,&quot;camera&quot;:&quot;&quot;,&quot;caption&quot;:&quot;&quot;,&quot;created_timestamp&quot;:&quot;0&quot;,&quot;copyright&quot;:&quot;&quot;,&quot;focal_length&quot;:&quot;0&quot;,&quot;iso&quot;:&quot;0&quot;,&quot;shutter_speed&quot;:&quot;0&quot;,&quot;title&quot;:&quot;&quot;,&quot;orientation&quot;:&quot;0&quot;}" data-image-title="convert-text-to-speech-using-freetts-api" data-image-description="" data-medium-file="https://i0.wp.com/www.dineshkrish.com/wp-content/uploads/2016/11/Convert-Text-to-Speech-using-FreeTTS-API.png?fit=247%2C300" data-large-file="https://i0.wp.com/www.dineshkrish.com/wp-content/uploads/2016/11/Convert-Text-to-Speech-using-FreeTTS-API.png?fit=285%2C346" src="https://i0.wp.com/www.dineshkrish.com/wp-content/uploads/2016/11/Convert-Text-to-Speech-using-FreeTTS-API.png?resize=285%2C346" alt="Convert Text to Speech using FreeTTS API" class="alignnone size-full wp-image-308" srcset="https://i0.wp.com/www.dineshkrish.com/wp-content/uploads/2016/11/Convert-Text-to-Speech-using-FreeTTS-API.png?w=285 285w, https://i0.wp.com/www.dineshkrish.com/wp-content/uploads/2016/11/Convert-Text-to-Speech-using-FreeTTS-API.png?resize=247%2C300 247w" sizes="(max-width: 285px) 100vw, 285px" data-recalc-dims="1"></p> 
         <h4>Resource</h4> 
         <p><a href="https://sourceforge.net/projects/freetts/files/FreeTTS/FreeTTS%201.2.2/" target="_blank">Download FreeTTS 1.2.2 Here</a></p> 
         <h4>TextToSpeech.java</h4> 
         <pre class="prettyprint">package com.dineshkrish.freetts;

import com.sun.speech.freetts.Voice;
import com.sun.speech.freetts.VoiceManager;

public class TextToSpeech {

	// Voice Name There are few more voice are available (ie: kevin, kevin16, alan)
	private static final String VOICE_NAME = "kevin16";
	private final Voice voice;
	
	public TextToSpeech() {

		VoiceManager voiceManager = VoiceManager.getInstance();		
		voice = voiceManager.getVoice(VOICE_NAME);
		voice.allocate();
	}

	public void speak(String text) {

		if(text != null &amp;&amp; !text.isEmpty()) {
			
			voice.speak(text);
		}
	}

}
</pre> 
         <h4>Application.java</h4> 
         <pre class="prettyprint">package com.dineshkrish.freetts;

import java.util.Scanner;

public class Application {

	public static void main(String[] args) {

		// Defining Scanner Object to read data from console
		Scanner scanner = new Scanner(System.in);
		
		TextToSpeech tts = new TextToSpeech();

		System.out.println("Enter the Text to Convert to Speech and type 'exit' to terminate !!!");

		// Reading the text
		String text = scanner.nextLine();

		while (!"exit".equalsIgnoreCase(text)) {

			tts.speak(text);
			
			System.out.println("Enter the Text to Convert to Speech and type 'exit' to terminate !!!");
			text = scanner.nextLine();
		}

		scanner.close();
	}
}
</pre>
