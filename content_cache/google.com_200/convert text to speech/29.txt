      <h1 class="post-title"><a href="https://blogs.windows.com/msedgedev/2016/06/01/introducing-speech-synthesis-api/">Introducing the Speech Synthesis API in Microsoft Edge</a></h1> 
      <div class="author">
        By 
       <a href="https://blogs.windows.com/msedgedev/author/jerry-smith/" title="Posts by Jerry Smith" class="author url fn" rel="author">Jerry Smith</a> / Senior Program Manager 
      </div> 
      <div id="addthis-17243" class="addthis_toolbox addthis_default_style expanded" addthis:url="https://blogs.windows.com/msedgedev/2016/06/01/introducing-speech-synthesis-api/" addthis:title="Introducing the Speech Synthesis API in Microsoft Edge"> 
       <a class="addthis_button_facebook"><i class="icon icon-facebook"></i> <span>Share</span></a> 
       <a class="addthis_button_twitter"><i class="icon icon-twitter"></i> <span>Tweet</span></a> 
       <a class="addthis_button_reddit"><i class="icon icon-reddit"></i> <span>Share</span></a> 
       <a class="addthis_button_linkedin"><i class="icon icon-linkedin"></i> <span>Share</span></a> 
       <a class="addthis_button_skype" href="#"><i class="icon icon-skype"></i> <span>Skype</span></a> 
      </div> 
     </header> 
     <p>Starting with the Windows 10 Anniversary Update, Microsoft Edge will support the Speech Synthesis APIs defined in the <a href="https://dvcs.w3.org/hg/speech-api/raw-file/tip/webspeechapi.html">W3C Web Speech API Specification</a>. These APIs allow websites to convert text to audible speech with customizable voice and language settings. With them, website developers can add and control text-to-speech features specific to their page content and design. </p> 
     <p>Speech Synthesis is useful whenever narration might be applied. Our implementation also supports <a href="http://www.w3.org/TR/speech-synthesis/">Speech Synthesis Markup Language (SSML) Version 1.0</a> to provide further control over the speech output.</p> 
     <p>Speech Synthesis is enabled by default in <a href="https://insider.windows.com">Windows Insider Preview</a> builds starting with <a href="https://developer.microsoft.com/microsoft-edge/platform/changelog/desktop/14316/">EdgeHTML 14.14316</a> and above – try it out with our new <a href="https://developer.microsoft.com/microsoft-edge/testdrive/demos/speechsynthesis/"><strong>Speech Synthesis Demo</strong> on Test Drive</a>!</p> 
     <h2>API Overview</h2> 
     <p>The Web Speech API Specification defines a <strong>SpeechSynthesisUtterance</strong> interface that lets Javascript set speech text along with attributes that control the voice used and modify the language pronunciation, voice, volume, rate and pitch of the voice output. Other interfaces are defined that allow playback control and monitoring state of the synthesized speech.</p> 
     <p>Microsoft Edge implements these SpeechSynthesis interfaces:</p> 
     <ul> 
      <li><strong>SpeechSynthesis</strong>: Provides speech playback control and state</li> 
      <li><strong>SpeechSynthesisUtterance:</strong> Controls speech content, voice and pronunciation</li> 
      <li><strong>SpeechSynthesisEvent:</strong> Provides state information on the current utterance</li> 
      <li><strong>SpeechSynthesisVoice:</strong> Sets speech service information</li> 
     </ul> 
     <p>Our implementation of these Speech Synthesis APIs is based on the <a href="https://msdn.microsoft.com/en-us/library/windows/apps/windows.media.speechsynthesis.aspx">WinRT Windows.Media.SpeechSynthesis APIs</a>. These directly support most of the W3C Speech Synthesis interfaces. There are a few SpeechSynthesis details that we don’t currently support this release, which we’re evaluating for future releases:</p> 
     <ul> 
      <li>Playback <strong><em>pitch</em>:</strong> Used to vary the voice pitch on playback.</li> 
      <li><strong><em>onmark</em></strong> event: Used to indicate that a marked tag has been reached.</li> 
      <li><strong><em>onboundary</em></strong> event: Used to signal boundaries of spoken words or sentences.</li> 
     </ul> 
     <h2>Speech Synthesis Demo</h2> 
     <p>To illustrate these new speech features, we’ve published a <a href="https://developer.microsoft.com/microsoft-edge/testdrive/demos/speechsynthesis/"><strong>Speech Synthesis Demo </strong>on Test Drive</a>. This allows input of random text (try something really long) and exposes parameters like voice, language, rate and volume that allow tuning of the resulting speech.</p> 
     <p>The demo includes this sample code that uses <strong>SpeechSynthesisUtterance</strong> to take your selected text and speech settings, and use them to do a text to speech voice synthesis.</p> 
     <style>.gist table { margin-bottom: 0; }</style>

     <p>This sample reads in data from the demo HTML, and then uses window.speechSynthesis.speak to start playback. It shows how simple it is to add basic speech synthesis features to your website.</p> 
     <h2>Speech Synthesis Markup Language (SSML)</h2> 
     <p>SSML allows speech voices and content to be expressed in XML, allowing direct control over a variety of speech characteristics. You can try this by pasting SSML derived text into the Speech Demo.</p>

     <h2>Languages</h2> 
     <p>The language setting in our Test Drive demo will work with any installed voice language pack in Windows 10. By default, there will be a primary language installed for a system. Others need to be installed. Here’s how to add an input language to your PC:</p> 
     <ul> 
      <li>Go to <strong>Settings</strong> &gt; <strong>Time &amp; language</strong> &gt; <strong>Region &amp; language.</strong></li> 
      <li>Select <strong>Add a language.</strong></li> 
      <li>Select the language you want to use from the list, then choose which region’s version you want to use. Your download will begin immediately.</li> 
     </ul> 
     <p>Once installed, the language pack will be used to alter the pronunciation of foreign language text.</p>
