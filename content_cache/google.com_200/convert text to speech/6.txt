       <h1 class="talkify-h1" style="text-align: center; margin-bottom: 15px;"> Welcome to the Talkify text to speech API demo </h1> 
       <h2 class="talkify-h2">A JavaScript library that brings you world class text to speech to your website - in a matter of minutes</h2> 
       <p> <span>The full API is listed on our <a href="https://github.com/Hagsten/Talkify" target="_blank"> GitHub</a>. You can install Talkify via Bower (bower install talkify) or <a href="https://www.npmjs.com/package/talkify-tts" target="_blank">npm</a> (npm install talkify-tts)</span> </p> 
       <div style="text-align: center;"> 
        <button id="playSelf" class="talkify-play-self-button"> <i class="fa fa-play-circle"></i> <span> Start the demo</span> </button> 
       </div> 
       <p> This demo will demonstrate the full suite of our API. This will enable you to playback an entire website or parts of a website with text highlighting for easy read along. All this is backed up by two text to speech players, one that works with a Talkify backend and another one that works with the Web Speech API. A top of the line language engine is included, this means that Talkify will automatically detect the language and choose the correct voice. Of course you will have full control over the voice used, this is extra helpful when dealing with the Web Speech API voices which has a large set of languages, some of which is not supported by our language engine. </p> 
       <p> We have very flexible customization options that we intend to demonstrate here. These are the customization options you have </p>
       <ul> 
        <li>Let Talkify language engine choose the voice</li> 
        <li>Force a specific voice or culture. Useful when you have control of the language or if you fancy a particular voice</li> 
        <li>Use our own, built in audio control UI element</li> 
        <li>Use text highlighting for easy read along</li> 
        <li>Use text interaction, this will enable click to play on paragraphs </li> 
        <li>Subscribe to events to provide even more value to your users</li> 
       </ul> 
       <p></p> 
       <p> Here below you can play with the settings available and see the events generated. </p>
