        <h2 class="text-center">How to convert (synthesize) text to speech in Node.js</h2>

        <article id="article-content" itemscope itemtype="http://schema.org/BlogPosting">
         <p>The speech synthesis is used to convert&nbsp;written information into sound where it is more convenient for humans. Although such a features is specially used in mobile applications such as voice-enabled e-mail and unified messaging apps, you can implement a very simple TTS system in Node.js using the <code>say</code> module.</p> 
         <p>The say module doesn't depend on any remote service as it will use the native TTS service of the system (Linux with Festival, MacOS with Mac OS X text to speech using AppleScript and Windows with SAPI).</p> 
         <h2>Requirements</h2> 
         <p>In order to convert text to speech, we'll depend of the say module. Say is a TTS (text to speech) library for node&nbsp;that sends text from node.js to your speakers.</p> 
         <p>To install this module, execute the following command in your terminal:</p> 
         <pre><code class="lang-batch">npm install say</code></pre> 
         <p>After the installation, you'll be able to access this module using <code>require("say")</code>. You can read more about the say module in the <a href="https://github.com/Marak/say.js" target="_blank" rel="nofollow">official repository in Github here</a>.</p> 
         <h2>Implementation</h2> 
         <p>The say module is an object that offers 3 methods (dependently of the OS some functions will not work as expected or may not be available):</p> 
         <h3>Speak function</h3> 
         <p>The speak function allows you to synthesize text easily. It expects as first parameter the text to speak and as second parameter a function to check for any error (in case there was) and the end event:</p> 
         <pre><code class="lang-js">var say = require('say');

say.speak('Hello, how are you today?', (err) =&gt; {
    if (err) {
        return console.error(err);
    }

    console.log('Text has been spoken.');
});</code></pre> 
         <p>In MacOS and Linux, you can specify the speed of the synthesization and the voice providing the first and third parameter:</p> 
         <pre><code class="lang-js">var say = require('say');

say.speak('Hello, how are you', 'Alex', 0.5, (err) =&gt; {
    if (err) {
        return console.error(err);
    }

    console.log('Text has been spoken.');
});</code></pre> 
         <p>Note that in Windows these options aren't available.</p> 
         <h3>Stop function</h3> 
         <p>You can stop any current speech utterance using the say.stop method:</p> 
         <pre><code class="lang-js">var say = require('say');

say.speak('Hello, how are you', 'Alex', 0.5, (err) =&gt; {
    if (err) {
        return console.error(err);
    }

    console.log('Text has been spoken.');
});

say.stop();</code></pre> 
         <h3>Export function</h3> 
         <p>The export function allows you to export the text synthesization into a <code>.wav</code> file.</p> 
         <pre><code class="lang-js">var say = require("say");
var filename = "myaudio.wav"

say.export("I'm sorry, Dave.", 'Cellos', 0.75, filename, function(err) {
    if (err) {
        return console.error(err);
    }

    console.log(`Text has been saved to ${filename}`);
});</code></pre> 
         <h3>Notes for windows</h3> 
         <p>The voice parameter and speed in the <code>say</code> function not available, that means it will use the default system voice, ignoring the voice and speed parameter.</p> 
         <p>The export function is neither available.</p> 
         <h3>Notes for OSX</h3> 
         <p>In mac OS you can change the voices to:</p> 
         <p><strong>Feminine Voices </strong></p> 
         <p>Agnes, Kathy, Princess, Vicki, Victoria</p> 
         <p><strong>Masculine Voices </strong></p> 
         <p>Albert, Alex, Bruce, Fred, Junior, Ralph</p> 
         <p><strong>Miscellaneous Voices </strong></p> 
         <p>Bad News, Bahh, Bells, Boing, Bubbles, Cellos, Deranged, Good News, Hysterical, Pipe Organ, Trinoids, Whisper, Zarvox.</p> 
         <h3>Notes for Linux</h3> 
         <p>To synthesize text using Node.js in Linux, you'll <a href="http://www.cstr.ed.ac.uk/projects/festival/" target="_blank" rel="nofollow">Festival</a>.&nbsp;<span>Voices for Festival sometimes need to be installed separately - you can check which voices are available by starting up Festival in interactive mode, typing </span><code>(voice_</code><span>, and pressing </span><code>TAB</code><span>. Then take the name of the voice you'd like to try, minus the parentheses, and pass it in to say.js, for example:</span></p> 
         <pre><code class="lang-js">var say = require("say");
var voiceName = "voice_nick1"

say.speak("Luke, I am your father", voiceName, 0.75 , (err) =&gt; {
    if (err) {
        return console.error(err);
    }

    console.log(`Text with the voice ${voice}`);
});</code></pre> 
         <p>Happy coding <i class="fa fa-heart"></i>!</p>

        $(function () {
            var _search_api = {
                getSuggestions: function (searchTerm) {
                    return $.ajax({
                        url: '/ajax-endpoint/search',
                        data: {
                            q: searchTerm
                        },
                        type:"GET",
                        dataType: 'JSON'
                    });
                },
                process :  function(title,data,noCase){
                    var html = '<h4>'+title+' <small>('+data.length+')</small></h4>';

                    if(!data.length){
                        return html += '<p>'+noCase+'</p>';
                    }

                    html += '<ul class="list-material">';

                    for(var c = 0;c < data.length;c++){
                        var row = data[c];

                        html += '<li data-toggle="tooltip" data-placement="top" data-original-title="'+row.name+'">';
                        html +=     '<a href="'+row.path+'" class="visible" style="padding: 0px;">';
                        html +=         '<div class="list-content">';
                        html +=             '<span class="title">'+row.name+'</span>';
                        html +=             '<span class="caption">'+row.date+'</span>';
                        html +=         '</div>';
                        html +=     '</a>';
                        html += '</li>';
                    }

                    html += '</ul>';


                    return html;
                }
            };

            $('#search-input-ourcodeworld').bind("keyup change", OurCodeWorld.debounce(function () {
                var value = $(this).val();

                if (!value) {
                    return;
                }

                _search_api.getSuggestions(value).then(function (data) {
                    var articles = data.data;
                    var group =  {
                        left: articles.splice(0, Math.ceil(articles.length/2)),
                        right: articles
                    };
                    
                    $('#search-layout-zone-articles').html(_search_api.process("Articles", group.left ,"No articles were found"));
                    $('#search-layout-zone-news').html(_search_api.process("Articles", group.right ,"No articles were found"));
                }, function (err) {
                    console.log(err);
                });
            }, 300));

            OurCodeWorld.init();
            Layout.init();
        });
    </script>
