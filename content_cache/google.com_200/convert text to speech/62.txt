             <div class="post-text" itemprop="text"> 
              <p>In order of descending <a href="http://popcon.ubuntu.com/">popularity</a>:</p> 
              <ul> 
               <li><p><a href="http://manpages.ubuntu.com/manpages/trusty/man1/say.1.html">say</a> converts text to audible speech using the GNUstep speech engine.</p> <pre><code>sudo apt-get install gnustep-gui-runtime
say "hello"
</code></pre></li> 
               <li><p><a href="http://manpages.ubuntu.com/manpages/trusty/man1/festival.1.html">festival</a> General multi-lingual speech synthesis system.</p> <pre><code>sudo apt-get install festival
echo "hello" | festival --tts
</code></pre></li> 
               <li><p><a href="http://manpages.ubuntu.com/manpages/trusty/man1/spd-say.1.html">spd-say</a> sends text-to-speech output request to speech-dispatcher</p> <pre><code>sudo apt-get install speech-dispatcher
spd-say "hello"
</code></pre></li> 
               <li><p><a href="http://manpages.ubuntu.com/manpages/trusty/man1/espeak.1.html">espeak</a> is a multi-lingual software speech synthesizer.</p> <pre><code>sudo apt-get install espeak
espeak "hello"
</code></pre></li> 
              </ul>

            <td class="answercell"> 
             <div class="post-text" itemprop="text"> 
              <p>From <code>man spd-say</code>:</p> 
              <pre>
NAME
       spd-say - send text-to-speech output request to speech-dispatcher

SYNOPSIS
       spd-say [options] "some text"

DESCRIPTION
       spd-say  sends text-to-speech output request to speech-dispatcher process which handles it and ideally outputs the result
       to the audio system.

OPTIONS
       -r, --rate
              Set the rate of the speech (between -100 and +100, default: 0)

       -p, --pitch
              Set the pitch of the speech (between -100 and +100, default: 0)

       -i, --volume
              Set the volume (intensity) of the speech (between -100 and +100, default: 0)
</pre> 
              <p><strong>Hence you can get text-to-speech by following command:</strong></p> 
              <pre><code>spd-say "&lt;type text&gt;"
</code></pre> 
              <p>Ex:</p> 
              <pre><code>spd-say "Welcome to Ubuntu Linux"
</code></pre> 
              <p>You can also <strong>set speech rate, pitch, volume etc.</strong> see man-page.</p>
