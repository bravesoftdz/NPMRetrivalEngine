           <p><a href="https://www.twilio.com/blog/wp-content/uploads/2011/07/client-microphone-img.png" data-rel="lightbox-0" title=""><img class="alignright size-medium wp-image-2548" title="client-microphone-img" src="https://www.twilio.com/blog/wp-content/uploads/2011/07/client-microphone-img-151x300.png" alt="" width="151" height="300" srcset="https://www.twilio.com/blog/wp-content/uploads/2011/07/client-microphone-img-151x300.png 151w, https://www.twilio.com/blog/wp-content/uploads/2011/07/client-microphone-img.png 195w" sizes="(max-width: 151px) 100vw, 151px"></a><a href="http://www.twilio.com/api/client/" onclick="__gaTracker('send', 'event', 'outbound-article', 'http://www.twilio.com/api/client/', 'Twilio Client');">Twilio Client</a> is more than just a phone in your browser, it’s an audio pipe into Twilio. So in addition to being able to create a “soft phone”, you can make all kinds of other interesting applications. In this tutorial we’ll make a simple text-to-speech (TTS) tester.</p> 
           <p>Every now and then, I see a post in the <a href="http://getsatisfaction.com/twilio/" onclick="__gaTracker('send', 'event', 'outbound-article', 'http://getsatisfaction.com/twilio/', 'forum');">forum</a> or an e-mail to <a href="mailto:help@twilio.com">help@twilio.com</a> asking how to test Twilio’s text-to-speech engine. The person wants to hear how a piece of dialogue will sound in the male and female voices. Up until now, the answer has always been write it out as <a href="http://www.twilio.com/docs/api/twiml/" onclick="__gaTracker('send', 'event', 'outbound-article', 'http://www.twilio.com/docs/api/twiml/', 'TwiML');">TwiML</a>, tie it to a phone number, and give the number a call. But for folks working on getting their wording just right, constantly dialing, editing, and redialing is a pain. And for developers working internationally, it’s both tedious and costly. Using <a href="http://www.twilio.com/api/client/" onclick="__gaTracker('send', 'event', 'outbound-article', 'http://www.twilio.com/api/client/', 'Twilio Client');">Twilio Client</a>, we can change all that.</p> 
           <p>For this tutorial we’ll create a simple HTML form that let’s a person type out some text and choose between male and female voices. When they submit the form, Twilio Client will connect to Twilio and speak the text the person typed out in the browser–no need to pick up a phone and only a fraction of the cost.</p> 
           <p>To get started, we’ll write some TwiML that uses <a href="http://www.twilio.com/docs/api/twiml/say" onclick="__gaTracker('send', 'event', 'outbound-article', 'http://www.twilio.com/docs/api/twiml/say', '<Say>');">&lt;Say&gt;</a> to speak any text passed to it.</p> 
           <p>[php]<br> &lt;?php<br> header(‘Content-type: text/xml’);<br> echo ‘&lt;?xml version="1.0" encoding="UTF-8"?&gt;’;</p> 
           <p> $dialogue = trim($_REQUEST[‘dialogue’]);<br> $voice = (int) $_REQUEST[‘voice’];</p> 
           <p> if (strlen($dialogue) == 0)<br> {<br> $dialogue = ‘Please enter some text to be spoken.’;<br> }</p> 
           <p> if ($voice == 1)<br> {<br> $gender = ‘man’;<br> }<br> else {<br> $gender = ‘woman’;<br> }<br> ?&gt;<br> &lt;Response&gt;<br> &lt;Say voice="&lt;?php echo $gender; ?&gt;"&gt;&lt;?php echo $dialogue; ?&gt;&lt;/Say&gt;<br> &lt;/Response&gt;<br> [/php]</p> 
           <p>Then, we’ll go to our <a href="https://www.twilio.com/user/account/apps" onclick="__gaTracker('send', 'event', 'outbound-article', 'https://www.twilio.com/user/account/apps', 'Twilio Apps page');">Twilio Apps page</a> and create a new app for our TwiML. Let’s call it “TTS Tester”. Take note of the app’s SID, we’ll need it later in the tutorial.</p> 
           <p><a href="https://www.twilio.com/blog/wp-content/uploads/2011/07/app_sid1.png" data-rel="lightbox-1" title=""><img class="aligncenter size-full wp-image-2562" title="app_sid" src="https://www.twilio.com/blog/wp-content/uploads/2011/07/app_sid1.png" alt="" width="500" height="275" srcset="https://www.twilio.com/blog/wp-content/uploads/2011/07/app_sid1.png 500w, https://www.twilio.com/blog/wp-content/uploads/2011/07/app_sid1-300x165.png 300w" sizes="(max-width: 500px) 100vw, 500px"></a></p> 
           <p>Next, we’ll create the form a person can use to enter text to be spoken.</p> 
           <p>[html]<br> &lt;html&gt;<br> &lt;head&gt;<br> &lt;title&gt;TTS Tester&lt;/title&gt;<br> &lt;/head&gt;<br> &lt;body&gt;<br> &lt;p&gt;<br> &lt;label for="dialogue"&gt;Text to be spoken&lt;/label&gt;<br> &lt;input type="text" id="dialogue" name="dialogue" size="50"&gt;<br> &lt;/p&gt;<br> &lt;p&gt;<br> &lt;label for="voice-male"&gt;Male Voice&lt;/label&gt;<br> &lt;input type="radio" id="voice-male" name="voice" value="1" checked="checked"&gt;<br> &lt;label for="voice-female"&gt;Female Voice&lt;/label&gt;<br> &lt;input type="radio" id="voice-female" name="voice" value="2"&gt;<br> &lt;/p&gt;<br> &lt;p&gt;<br> &lt;input type="button" id="submit" name="submit" value="Speak to me"&gt;<br> &lt;/p&gt;<br> &lt;/body&gt;<br> &lt;/html&gt;<br> [/html]</p> 
           <p>Finally, we’ll tie the two together using Twilio Client. When a person submits the form, we’ll use some Javascript to collect the input and then use Twilio Client to send that information to our TTS Tester Twilio app and speak the text right in the browser.</p> 
           <p>[php]<br> &lt;?php<br> require_once(‘TwilioCapability.php’);</p> 
           <p> // Your account SID. Find it at the top of your Dashboard page.<br> $ACCOUNT_SID = ‘ACXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX’;<br> // Your auth token. Also on the top of your Dashboard.<br> $AUTH_TOKEN = ‘XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX’;<br> // The App SID for the TTS Tester app. We made this in the section above.<br> $APP_SID = ‘APXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX’;</p> 
           <p> $token = new TwilioCapability($ACCOUNT_SID, $AUTH_TOKEN);<br> // Set the permissions to allow your code to make outbound<br> // connections to Twilio.<br> $token-&gt;allowClientOutgoing($APP_SID);<br> ?&gt;<br> &lt;html&gt;<br> &lt;head&gt;<br> &lt;title&gt;TTS Tester&lt;/title&gt;<br> &lt;!– Some jQuery to make our lives easier –&gt;<br> &lt;script type="text/javascript" src="https://ajax.googleapis.com/ajax/libs/jquery/1.6.2/jquery.min.js"&gt;&lt;/script&gt;<br> &lt;!– The Twilio Client Javascript library –&gt;<br> &lt;script type="text/javascript" src="http://static.twilio.com/libs/twiliojs/1.0/twilio.js"&gt;&lt;/script&gt;</p> 
           <p> &lt;script type="text/javascript"&gt;</p> 
           <p> // Initialize Twilio Client<br> Twilio.Device.setup("&lt;?php echo $token-&gt;generateToken();?&gt;");</p> 
           <p> $(document).ready(function() {<br> // When someone clicks the submit button, speak.<br> $("#submit").click(function() {<br> speak();<br> });<br> });</p> 
           <p> function speak() {<br> // Get the values of the form.<br> var dialogue = $("#dialogue").val();<br> var voice = $(‘input:radio[name=voice]:checked’).val();</p> 
           <p> // Disable the button while we’re speaking.<br> $(‘#submit’).attr(‘disabled’, ‘disabled’);</p> 
           <p> // Send the form values to our TTS Tester app and use<br> // Twilio Client to speak the text in the browser.<br> Twilio.Device.connect({ ‘dialogue’ : dialogue, ‘voice’ : voice });<br> }</p> 
           <p> // When we’re done speaking the text, enable the button again.<br> Twilio.Device.disconnect(function (conn) {<br> $(‘#submit’).removeAttr(‘disabled’);<br> });</p> 
           <p> &lt;/script&gt;<br> &lt;/head&gt;<br> &lt;body&gt;<br> &lt;!– Same code as the prior section –&gt;<br> &lt;p&gt;<br> &lt;label for="dialogue"&gt;Text to be spoken&lt;/label&gt;<br> &lt;input type="text" id="dialogue" name="dialogue" size="50"&gt;<br> &lt;/p&gt;<br> &lt;p&gt;<br> &lt;label for="voice-male"&gt;Male Voice&lt;/label&gt;<br> &lt;input type="radio" id="voice-male" name="voice" value="1" checked="checked"&gt;<br> &lt;label for="voice-female"&gt;Female Voice&lt;/label&gt;<br> &lt;input type="radio" id="voice-female" name="voice" value="2"&gt;<br> &lt;/p&gt;<br> &lt;p&gt;<br> &lt;input type="button" id="submit" name="submit" value="Speak to me"&gt;<br> &lt;/p&gt;<br> &lt;/body&gt;<br> &lt;/html&gt;<br> [/php]</p> 
           <p>And that’s all there is to it. You can grab the source code on github and enhance it with features like a textarea instead of a text field and additional options for the French, German, and Spanish accented voices.</p>
