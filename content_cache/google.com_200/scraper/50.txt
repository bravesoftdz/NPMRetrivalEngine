Iron Web Scraper manages politeness &amp; multithreading in the background, leaving a developer’s own application easy to understand &amp; maintain.

Iron Web Scraper can be used to migrate content from existing websites as well as build search indexes and monitor website structure &amp; content changes.  It's functionality includes:
» Read &amp; extract structured content from web pages using html DOM, Javascript, Xpath, jQuery Style CSS Selectors.
» Fast multi threading allows hundreds of simultaneous requests.
» Politely avoid over stalling remote servers using IP/domain level throttling &amp; optionally respecting robots.txt
» Manage multiple identities, DNS, proxies, user agents, request methods, custom headers, cookies &amp; logins.
» Data exported from websites becomes native C# objects which can be stored or used immediately.
» Exceptions managed in all but the developers own code. Errors and captchas auto retried on failure
» Save, pause, resume, autosave scrape jobs.
» Built in web cache allows for action replay, crash recovery, and querying existing web scrape data.  Change scrape logic on the fly, then replay job without internet traffic.

Requires .NET 4.5.2 - Licensing &amp; Support available for commercial deployments.  For code examples, documentation &amp; more visit http://ironsoftware.com/cshapr/webscraper.  For support please email us at developers@ironsoftware.com.">

     <p>Iron WebScraper is a C# web scraping library, allowing developers to simulate &amp; automate human browsing behavior to extract content, files &amp; images from web applications as native .Net objects. Iron Web Scraper manages politeness &amp; multithreading in the background, leaving a developer’s own application easy to understand &amp; maintain.<br><br>Iron Web Scraper can be used to migrate content from existing websites as well as build search indexes and monitor website structure &amp; content changes. &nbsp;It's functionality includes:<br>» Read &amp; extract structured content from web pages using html DOM, Javascript, Xpath, jQuery Style CSS Selectors.<br>» Fast multi threading allows hundreds of simultaneous requests.<br>» Politely avoid over stalling remote servers using IP/domain level throttling &amp; optionally respecting robots.txt<br>» Manage multiple identities, DNS, proxies, user agents, request methods, custom headers, cookies &amp; logins.<br>» Data exported from websites becomes native C# objects which can be stored or used immediately.<br>» Exceptions managed in all but the developers own code. Errors and captchas auto retried on failure<br>» Save, pause, resume, autosave scrape jobs.<br>» Built in web cache allows for action replay, crash recovery, and querying existing web scrape data. &nbsp;Change scrape logic on the fly, then replay job without internet traffic.<br><br>Requires .NET 4.5.2 - Licensing &amp; Support available for commercial deployments. &nbsp;For code examples, documentation &amp; more visit <a href="http://ironsoftware.com/cshapr/webscraper">http://ironsoftware.com/cshapr/webscraper</a>. &nbsp;For support please email us at developers@ironsoftware.com.</p>
