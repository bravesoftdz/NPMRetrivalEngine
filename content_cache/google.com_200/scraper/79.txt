       <div class="readme markdown-body"> 
        <h1>README</h1> 
        <p><a href="https://travis-ci.org/raulr/google-play-scraper" rel="nofollow noopener external"><img src="https://camo.githubusercontent.com/14ffcc59abfbb349961dca5f9aa281e5eca92ed9/68747470733a2f2f7472617669732d63692e6f72672f7261756c722f676f6f676c652d706c61792d736372617065722e7376673f6272616e63683d6d6173746572" alt="Build Status"></a></p> 
        <p>A PHP scraper to get app data from Google Play.</p> 
        <h2><a id="user-content-installation" class="anchor" href="#user-content-installation" rel="nofollow noopener external"></a>Installation</h2> 
        <p>Add <code>raulr/google-play-scraper</code> as a require dependency in your <code>composer.json</code> file:</p> 
        <pre>$ composer require raulr/google-play-scraper</pre> 
        <h2><a id="user-content-usage" class="anchor" href="#user-content-usage" rel="nofollow noopener external"></a>Usage</h2> 
        <p>First create a <code>Scraper</code> instance.</p> 
        <pre><span class="pl-s1"><span class="pl-k">use</span> <span class="pl-c1">Raulr\GooglePlayScraper\Scraper</span>;</span>
<span class="pl-s1"></span>
<span class="pl-s1"><span class="pl-smi">$scraper</span> <span class="pl-k">=</span> <span class="pl-k">new</span> <span class="pl-c1">Scraper</span>();</span></pre> 
        <p>There are several methods to configure the default behavior:</p> 
        <ul>
         <li><code>setDelay($delay)</code>: Sets the delay in milliseconds between requests to Google Play site.</li> 
         <li><code>setDefaultLang($lang)</code>: Sets the default language for all requests. <code>$lang</code> must be an <a href="https://en.wikipedia.org/wiki/ISO_639-1" rel="nofollow noopener external">ISO_639-1</a> two letter language code. If not set, the default language is <code>en</code>.</li> 
         <li><code>setDefaultCountry($country)</code>: Sets the default country for all requests. <code>$country</code> must be an <a href="https://en.wikipedia.org/wiki/ISO_3166-1_alpha-2" rel="nofollow noopener external">ISO_3166-1</a> two letter country code. If not set, the default country is <code>us</code>.</li> 
        </ul>
        <h3><a id="user-content-getapp" class="anchor" href="#user-content-getapp" rel="nofollow noopener external"></a>getApp</h3> 
        <p>Gets app information given its ID.</p> 
        <h4><a id="user-content-parameters" class="anchor" href="#user-content-parameters" rel="nofollow noopener external"></a>Parameters</h4> 
        <ul>
         <li><code>$id</code>: Google Play app identifier.</li> 
         <li><code>$lang</code>: (optional, defaults to <code>null</code>): Overrides the default language.</li> 
         <li><code>$country</code>: (optional, defaults to <code>null</code>): Overrides the default country.</li> 
        </ul>
