        <p>Imagine what would you do if you could automate all the repetitive and boring activities you perform using internet, like checking every day the first results of Google for a given keyword, or download a bunch of files from different websites.</p> 
        <p>In this post youll learn to use Selenium with Python, a Web Scraping tool that simulates a user surfing the&nbsp;Internet. For example, you can use it to automatically look for Google queries and read the results, log in to your social&nbsp;accounts, simulate a user to test your web application, and anything you find in your daily live that its repetitive. The possibilities are infinite! ?</p> 
        <p><span id="more-671"></span></p> 
        <p>*All the code in this post has been tested with Python 2.7 and Python 3.4.</p>

        <div id="crayon-57cc0a2d6d297076059257" class="crayon-syntax crayon-theme-my-crayon-theme-github crayon-font-monaco crayon-os-mac print-yes notranslate" data-settings=" no-popup minimize scroll-mouseover" style=" margin-top: 12px; margin-bottom: 12px; font-size: 12px !important; line-height: 15px !important;">

import time
from selenium import webdriver
from selenium.webdriver.common.by import By
from selenium.webdriver.support.ui import WebDriverWait
from selenium.webdriver.support import expected_conditions as EC
from selenium.common.exceptions import TimeoutException


def init_driver():
    driver = webdriver.Firefox()
    driver.wait = WebDriverWait(driver, 5)
    return driver


def lookup(driver, query):
    driver.get("http://www.google.com")
    try:
        box = driver.wait.until(EC.presence_of_element_located(
            (By.NAME, "q")))
        button = driver.wait.until(EC.element_to_be_clickable(
            (By.NAME, "btnK")))
        box.send_keys(query)
        button.click()
    except TimeoutException:
        print("Box or Button not found in google.com")


if __name__ == "__main__":
    driver = init_driver()
    lookup(driver, "Selenium")
    time.sleep(5)
    driver.quit()</textarea>

        <p>Google search has recently changed so that initially, Google shows this page:</p> 
        <p><a href="http://www.marinamele.com/wp-content/uploads/2015/02/Google-Search.png"><img class="wp-image-707 aligncenter" src="http://www.marinamele.com/wp-content/uploads/2015/02/Google-Search.png" alt="Google-Search" width="335" height="167" srcset="http://www.marinamele.com/wp-content/uploads/2015/02/Google-Search.png 622w, http://www.marinamele.com/wp-content/uploads/2015/02/Google-Search-300x149.png 300w" sizes="(max-width: 335px) 100vw, 335px"></a></p>

        <p><a href="http://www.marinamele.com/wp-content/uploads/2015/02/Query-Google-Search.png"><img class="wp-image-708 aligncenter" src="http://www.marinamele.com/wp-content/uploads/2015/02/Query-Google-Search.png" alt="Query-Google-Search" width="540" height="123" srcset="http://www.marinamele.com/wp-content/uploads/2015/02/Query-Google-Search.png 790w, http://www.marinamele.com/wp-content/uploads/2015/02/Query-Google-Search-300x68.png 300w" sizes="(max-width: 540px) 100vw, 540px"></a></p>

def lookup(driver, query):
    driver.get("http://www.google.com")
    try:
        box = driver.wait.until(EC.presence_of_element_located(
            (By.NAME, "q")))
        button = driver.wait.until(EC.element_to_be_clickable(

            button.click()
