           <div class="swa_content_module_inner"> 
            <p> We’re sorry, but your browser settings are currently limiting your experience on <strong>southwest.com</strong> (View our <a href="/html/about-southwest/terms-and-conditions/index.html#browser_capatibility" target="_parent" onclick="window.event.cancelBubble=true;"> web browser compatibility</a> suggestions).&nbsp; In order to have full use of all the best features of <strong>southwest.com</strong>, please make sure to follow these simple steps to enable JavaScript in your browser. </p> 
            <h3 style="font-weight:normal">To enable JavaScript on <strong>Internet Explorer</strong>:</h3> 
            <ol> 
             <li>Select <strong>Tools ? Internet Options</strong></li> 
             <li>Click the <strong>Security</strong> tab Click the <strong>Custom Level</strong> button.</li> 
             <li>Scroll down to <strong>Active Scripting</strong>. Click the <strong>Enable</strong> button.</li> 
             <li>Click the <strong>OK</strong> button.</li> 
             <li>When prompted "Are you sure you want to change the security setting for this zone?" click the <strong>Yes</strong> button. Click the <strong>OK</strong> button. </li> 
             <li>Go back to the homepage of <a href="/"><strong>southwest.com</strong></a>.</li> 
            </ol> 
            <h3 style="font-weight:normal">To enable JavaScript on <strong>Chrome</strong>:</h3> 
            <ol> 
             <li>Select <strong>Customize and control Google Chrome</strong> (icon with 3 stacked horizontal lines) to the right of the address bar. </li> 
             <li>From the drop-down menu, select <strong>Settings</strong>.</li> 
             <li>At the bottom of the page, click <strong>Show advanced settings...</strong>.</li> 
             <li>Under Privacy, select the <strong>Content settings...</strong> button.</li> 
             <li>Under the <strong>JavaScript</strong> heading, select the <strong>Allow all sites to run JavaScript (recommended)</strong> radio button. </li> 
             <li>Click the <strong>Done</strong> button.</li> 
             <li>Go back to the homepage of <a href="/"><strong>southwest.com</strong></a>.</li> 
            </ol> 
            <h3 style="font-weight:normal">To enable JavaScript on <strong>Firefox</strong>:</h3> 
            <ol> 
             <li>Select <strong>Tools ? Options</strong> Select the <strong>Contents</strong> tab.</li> 
             <li>Check <strong>the "Enable Javascript" box</strong>.</li> 
             <li>Click <strong>OK to save changes</strong>.</li> 
             <li>Go back to the homepage of <a href="/"><strong>southwest.com</strong></a>.</li> 
            </ol> 
            <h3 style="font-weight:normal">To enable JavaScript on <strong>Safari</strong>:</h3> 
            <ol> 
             <li>Click the <strong>Safari</strong> menu, and select <strong>Preferences</strong>.</li> 
             <li>Click on the <strong>Security</strong> icon.</li> 
             <li>Check <strong>"Enable Javascript"</strong> under <strong>Web Content</strong>.</li> 
             <li>Go back to the homepage of <a href="/"><strong>southwest.com</strong></a>.</li> 
            </ol> 
            <h3 style="font-weight:normal">To enable JavaScript on <strong>Opera</strong>:</h3> 
            <ol> 
             <li>Select <strong>Tools ? Preferences</strong>.</li> 
             <li>Click the <strong>Advanced</strong> tab.</li> 
             <li>Click the box next to <strong>Enable JavaScript</strong>.</li> 
             <li>Click the <strong>OK</strong> button.</li> 
             <li>Go back to the homepage of <a href="/"><strong>southwest.com</strong></a>.</li> 
            </ol>
