         <h3>Creating a custom dynamic stamp using Acrobat XI</h3> 
         <p><a href="http://blogs.adobe.com/documentcloud/acrobat-expert/lori-kassuba/">Lori Kassuba</a> ? <em>December 14, 2012</em></p> 
         <ol> 
          <li>Open the Comments pane, Annotations panel, select the Stamp Tool and choose Custom Stamps &gt; Create Custom Stamp from the menu.</li> 
          <li>Click on the Browse... button to create a stamp image from any of the files types listed in the Open dialog. All file types are converted to pdf.</li> 
          <li>Click OK. Select the Dynamic category, name the stamp and click OK.</li> 
          <li>Open the newly created PDF stamp file from your system user-specific directory. The file will appear arbitrarily named.<br> - Win 7: \Users\[username]\AppData\Roaming\Adobe\Acrobat\11.0\Stamps<br> - Win XP: \Documents and Settings\[username]\ApplicationData\Adobe\Acrobat\11.0\Stamps<br> - Mac OS X 10.8.x: /Users/[username]/Library/Application Support/Adobe/Acrobat/11.0/Stamps </li> 
          <li>Open the Tools pane, Forms panel and select Edit. In the Add or Edit Form Fields dialog choose No.</li> 
          <li>From the Add New Field Dropdown select Text Field. Place and name the field where you want the date to appear.</li> 
          <li>Double click on the form field and select the Calculate tab. Choose the Custom calculation script: radio button and select the Edit... button.</li> 
          <li>In the JavaScript Editor type the script at the left. Click OK and Close the Text Field Properties dialog.</li> 
          <li>Close Form Editing mode and Save the file. Rename the stamp file for easy reference.</li> 
          <li>To use the dynamic stamp on multiple computers install the file in the following program directory.</li> 
         </ol>
