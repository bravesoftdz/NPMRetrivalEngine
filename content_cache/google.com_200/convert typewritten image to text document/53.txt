        <div class="entry-thumbnail"> 
         <img width="550" height="200" src="http://www.henryalgus.com/wp-content/uploads/2014/09/23-01_jquery_ajax_ld_img.jpg" class="attachment-featured-image size-featured-image wp-post-image" alt="" srcset="http://www.henryalgus.com/wp-content/uploads/2014/09/23-01_jquery_ajax_ld_img.jpg 550w, http://www.henryalgus.com/wp-content/uploads/2014/09/23-01_jquery_ajax_ld_img-300x109.jpg 300w" sizes="(max-width: 550px) 100vw, 550px">

        <p>jQuery is an excellent tool to make web development easy and straightforward. It helps while doing DOM manipulation and makes Ajax requests painless across different browsers and platforms. But if you want make an Ajax request, which is giving binary data as a response, you will discover that it does not work for jQuery, at least for now. Changing “dataType” parameter to “text”, does not help, neither changing it to any other jQuery supported Ajax data type.</p> 
        <p>Problem here is that jQuery still does not support HTML5 XMLHttpRequest Level 2 binary data type requests – there is even a <a href="http://bugs.jquery.com/ticket/11461">bug</a> in jQuery bug tracker, which asks for this feature. Although there <a href="https://github.com/jquery/jquery/pull/1525">is a long discussion</a> about this subject on the GitHub, it seems that this feature will not become part of jQuery soon.</p> 
        <p><span id="more-226"></span></p> 
        <p>To find a solution for this problem, we have to modify XMLHttpRequest itself. To read binary data correctly, we have to treat response type as blob data type.</p>

        <p><strong>Asynchronous or synchronous execution</strong></p> 
        <p>It is possible to change execution type from asynchronous to synchrous when setting parameter “async” to false.</p> 
        <p><code>async:false,</code></p>

        <p>If your script needs to have authentication during the request, you can use username and password parameters.</p>

        <h2><a id="user-content-supported-browsers" class="anchor" href="https://github.com/henrya/js-jquery/tree/master/BinaryTransport#supported-browsers"></a>Supported browsers</h2> 
        <p>BinaryTransport requires XHR2 responseType, ArrayBuffer and Blob response type support from your browser, otherwise it does not work as expected. Currently most major browsers should work fine.</p> 
        <p>Firefox: 13.0+ Chrome: 20+ Internet Explorer: 10.0+ Safari: 6.0 Opera: 12.10</p>
