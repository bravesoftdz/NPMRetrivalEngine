       My Favorite Sublime Text Packages
      </div>

    </div>

     <article class="language-markup" itemscope itemtype="http://schema.org/BlogPosting"> 
      <p><a href="http://www.sublimetext.com">Sublime Text</a> is the most powerful text editor for anyone who works in web development, design, and programming. It may seem daunting to use at first with its non-standard interface and learning curve, but it is very easy to create a powerful setup that is customized for each individual's own workflow and capabilities. </p> 
      <h3>The Basics</h3> 
      <h5>Command Palette</h5> 
      <p><img src="http://digisocialnet.com/content/01-blog/04-my-favorite-sublime-text-packages/commandpalette.jpg" class="img-responsive" title="commandpalette" alt="commandpalette"></p> 
      <p>The first command to learn is <code>cmd+shift+p</code> for the command palette. Almost any function needed is accessible here by typing the first few letters of what you want to do. It is similar to <a href="http://alfredapp.com">Alfred</a> or <a href="http://qsapp.com/">Quicksilver</a> but only used inside Sublime Text.</p> 
      <h5>File Palette</h5> 
      <p><img src="http://digisocialnet.com/content/01-blog/04-my-favorite-sublime-text-packages/filepalette.jpg" class="img-responsive" title="filepalette" alt="filepalette"></p> 
      <p>The second thing to learn is to open your whole project folder in Sublime Text. I drag my project folder onto the application icon and all my files are accesible with a similar command palette interface by pressing <code>cmd+p</code>. This is great for working with numerous code snippets and dependencies.</p> 
      <h5>Multiple Selections</h5> 
      <p>Many users swear by the multiple selection capabilities to save time when editing repeating text strings. You can select text and then hit keyboard commands to extend your selection and type in multiple places at once. (<code>cmd+d</code> selects the next occurence of your selection, <code>cmd+ctrl+g</code> selects every occurence) </p> 
      <h5>Code Indentation</h5> 
      <p><img src="http://digisocialnet.com/content/01-blog/04-my-favorite-sublime-text-packages/code-indentation.jpg" class="img-responsive" title="code-indentation" alt="code-indentation"></p> 
      <p>Sublime Text has pretty sophisticated handling of white space. It tries to line up opening and closing tags with the correct number of tabs and the "Reindent Lines" tool in the command palette is a life-saver when trying to figure out where a missing or deleted closing tag should be.</p> 
      <h5>Text-based Preferences</h5> 
      <p>Although not very pretty, Sublime Text preferences are managed through text files that are administered in a normal text editing window. This interface is familiar to developers comfortable with UNIX-based systems but may be a little confusing for Mac users who expect applications to follow <a href="https://developer.apple.com/library/mac/documentation/UserExperience/Conceptual/AppleHIGuidelines/Intro/Intro.html">OSX Human Interface Guidelines</a>.</p> 
      <h5>Packages</h5> 
      <p>There are a lot of powerful tools already included but the real fun in Sublime Text comes from the awesome plug-ins (called packages in Sublime Text) that are available. </p> 
      <h3>Package Control</h3> 
      <p><img src="http://digisocialnet.com/content/01-blog/04-my-favorite-sublime-text-packages/package-control.jpg" class="img-responsive" title="package-control" alt="package-control"></p> 
      <p>Installing, updating, deleting and other management of Sublime Text packages manually goes from being a chore, but after <a href="https://sublime.wbond.net/installation">installing Package Control</a>, the process is easy and fun to discover new packages that create a customized workflow that isn't really possible in other text editors. There's over 2,000 packages available for installation and most are available for free under open source licensing.</p> 
      <p>Once Package Control is installed, it's easy to add new packages in the command palette (<code>cmd+p</code>) by typing "install package". It updates its package listing automatically and waits for a package name to be typed in. Press Enter and Package Control takes care of the rest.</p> 
      <p>The <a href="https://sublime.wbond.net/">Package Control website</a> is a great resource to find packages on your own and check out my favorites below.</p> 
      <p><a href="https://sublime.wbond.net/" class="dsn-button dsn-button-pill dsn-button-white">Package Control website</a></p> 
      <hr> 
      <h3>AdvancedNewFile</h3> 
      <p>This plugin allows for faster file creation within a project.</p> 
      <p>Simply bring up the AdvancedNewFile input through the appropriate key binding. Then, enter the path, along with the file name into the input field. Upon pressing enter, the file will be created. In addition, if the directories specified do not yet exists, they will also be created. For more advanced usage of this plugin, be sure to look at Advanced Path Usage. By default, the path to the file being created will be filled shown in the status bar as you enter the path information.</p> 
      <p><a href="https://github.com/skuroda/Sublime-AdvancedNewFile" class="dsn-button dsn-button-pill dsn-button-white">More info on Github</a></p> 
      <hr> 
      <h3>AutoFileName</h3> 
      <p>Do you ever find yourself sifting through folders in the sidebar trying to remember what you named that file? Can't remember if it was a jpg or a png? Maybe you just wish you could type filenames faster. No more.</p> 
      <p>Whether your making a img tag in html, setting a background image in css, or linking a .js file to your html (or whatever else people use filename paths for these days…), you can now autocomplete the filename. Plus, it uses the built-in autocomplete, so no need to learn another pesky shortcut.</p> 
      <p>If you are looking to autocomplete an image path in an HTML <code>&lt;img&gt;</code> tag:</p> 
      <pre><code>&lt;img src="../|" /&gt;
</code></pre> 
      <p>Pressing <code>ctrl+space</code>, will activate AutoFileName. A list of available files where be ready to select.</p> 
      <p><a href="https://github.com/BoundInCode/AutoFileName" class="dsn-button dsn-button-pill dsn-button-white">More info on Github</a></p> 
      <hr> 
      <h3>Better Completion</h3> 
      <p>This package aim at provide a simpler way to build own auto-completions and avoid *.sublime-completions override word-completion wrongly.</p> 
      <p>auto-completion is lightweight, easier, simpler than snippets.</p> 
      <p>This package also provide several APIs completions such as JavaScript API, jQuery API, Underscore API, HTML5 and Bootstrap Classes collected by the author.</p> 
      <p><a href="https://github.com/Pleasurazy/Sublime-Better-Completion" class="dsn-button dsn-button-pill dsn-button-white">More info on Github</a></p> 
      <hr> 
      <h3>Bootstrap 3 Snippets</h3> 
      <p>Use common <a href="http://getbootstrap.com">Bootstrap 3</a> code snippets without having to remember every class used by this popular CSS framework.</p> 
      <p><a href="https://github.com/JasonMortonNZ/bs3-sublime-plugin" class="dsn-button dsn-button-pill dsn-button-white">More info on Github</a></p> 
      <hr> 
      <h3>CDNJS</h3> 
      <p>Easily link to Javascript packages on <a href="http://cdnjs.com">CDNJS</a> without leaving your text editor. It makes it easy to find commonly used scripts like jQuery.</p> 
      <p><a href="https://github.com/dafrancis/Sublime-Text--cdnjs" class="dsn-button dsn-button-pill dsn-button-white">More info on Github</a></p> 
      <hr> 
      <h3>Clickable URLs</h3> 
      <p>This plugin underlines URLs in Sublime Text, and lets you open them with a keystroke (<code>Cmd+Option+Enter</code> by default).</p> 
      <p>After you put the cursor over an URL, you can either hit <code>Cmd+Option+Enter</code> (<code>Ctrl+Alt+Enter</code> on Windows &amp; Linux), or select “Open URL under cursor” from the Command Palette. Instead of selecting an auto detected URL, you can select any block of text and it will also open in a browser as a URL.</p> 
      <p>There is also an “Open all URLs” command, which opens all URLs found in the current document.</p> 
      <p><a href="https://github.com/leonid-shevtsov/ClickableUrls_SublimeText" class="dsn-button dsn-button-pill dsn-button-white">More info on Github</a></p> 
      <hr> 
      <h3>Color Highlighter</h3> 
      <p><a href="https://sublime.wbond.net/packages/Color%20Highlighter">ColorHighlighter</a> - is a plugin for the Sublime text 2 and 3, which underlays selected hexadecimal colorcodes (like "#FFFFFF", "rgb(255,255,255)", "white", etc.) with their real color. Also, plugin adds color picker to easily modify colors.</p> 
      <p><a href="https://sublime.wbond.net/packages/Color%20Highlighter" class="dsn-button dsn-button-pill dsn-button-white">More info on the Package Control website</a></p> 
      <hr> 
      <h3>ColorPicker</h3> 
      <p><img src="http://digisocialnet.com/content/01-blog/04-my-favorite-sublime-text-packages/colorpicker.jpg" class="img-responsive" title="colorpicker" alt="colorpicker"></p> 
      <p>A multi-platform plugin that brings up a system color picker and converts the selected color to HEX. Press <code>cmd+shift+c</code> on Mac or <code>ctrl+shift+c</code> on Linux/Windows to insert or change a color.</p> 
      <p><a href="http://weslly.github.io/ColorPicker/" class="dsn-button dsn-button-pill dsn-button-white">More info on GitHub</a></p> 
      <hr> 
      <h3>Emmet</h3> 
      <p>A system of commands and tools to make coding faster. A lot of features to use, check out <a href="http://emmet.io/">Emmet.io</a> for more info.</p> 
      <blockquote> 
       <p>Basically, most text editors out there allow you to store and re-use commonly used code chunks, called “snippets”. While snippets are a good way to boost your productivity, all implementations have common pitfalls: you have to define the snippet first and you can’t extend them in runtime.</p> 
       <p>Emmet takes the snippets idea to a whole new level: you can type CSS-like expressions that can be dynamically parsed, and produce output depending on what you type in the abbreviation. Emmet is developed and optimised for web-developers whose workflow depends on HTML/XML and CSS, but can be used with programming languages too.</p> 
      </blockquote> 
      <p><a href="http://emmet.io/" class="dsn-button dsn-button-pill dsn-button-white">More info on Emmet.io</a></p> 
      <hr> 
      <h3>SCSS</h3> 
      <p>SASS and SCSS syntax highlighting ported from TextMate.</p> 
      <p><a href="https://github.com/MarioRicalde/SCSS.tmbundle" class="dsn-button dsn-button-pill dsn-button-white">More info on GitHub</a></p> 
      <hr> 
      <h3>SublimeCodeIntel</h3> 
      <p>Code intelligence plugin ported from <a href="http://www.openkomodo.com/">Open Komodo Editor</a> to Sublime Text.</p> 
      <p>Supports all the languages Komodo Editor supports for Code Intelligence (CIX, CodeIntel2): JavaScript, Mason, XBL, XUL, RHTML, SCSS, Python, HTML, Ruby, Python3, XML, Sass, XSLT, Django, HTML5, Perl, CSS, Twig, Less, Smarty, Node.js, Tcl, TemplateToolkit, PHP.</p> 
      <p>Provides the following features:</p> 
      <ul> 
       <li>Jump to Symbol Definition - Jump to the file and line of the definition of a symbol.</li> 
       <li>Imports autocomplete - Shows autocomplete with the available modules/symbols in real time.</li> 
       <li>Function Call tooltips - Displays information in the status bar about the working function.</li> 
       <li>Plugin should work in all three platforms (MacOS X, Windows and Linux).</li> 
      </ul> 
      <p><a href="http://sublimecodeintel.github.io/SublimeCodeIntel/" class="dsn-button dsn-button-pill dsn-button-white">More info on GitHub</a></p> 
      <hr> 
      <h3>Theme - Soda</h3> 
      <p><img src="http://digisocialnet.com/content/01-blog/04-my-favorite-sublime-text-packages/soda.jpg" class="img-responsive" title="soda" alt="soda"></p> 
      <p>Popular theme for Sublime Text that is available in dark and light color variants. Includes custom folder icons, file tabs, and Retina (Hi-DPI) icons.</p> 
      <p><a href="http://buymeasoda.github.com/soda-theme/" class="dsn-button dsn-button-pill dsn-button-white">More info on GitHub</a></p> 
      <hr> 
      <h3>It's your workflow</h3> 
      <p>These packages are a good start but there are <a href="https://sublime.wbond.net/browse">thousands of other ones</a> to check out and make your own. I would love to know what your go-to favorites are and what new packages everyone's found useful.</p>
