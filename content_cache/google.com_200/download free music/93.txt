          <h4>Download Help</h4> 
          <p><strong>Please note: this text is for your information only, for full legal terms and conditions please visit <a href="http://store.universal-music.co.uk/restofworld/Customer-Services/help/page/help" target="blank">store.universal-music.co.uk</a></strong></p> 
          <ul style="LIST-STYLE-TYPE: none"> 
           <li class="list">1. Downloading your purchased music</li>
           <li class="sublist"><a href="#mo1i">i) &nbsp;&nbsp;Recommendation: use the Universal Music Downloader to download the full album package </a></li> 
           <li class="sublist"><a href="#mo1ii">ii) alternatively, download a zip file of the full album package</a></li> 
           <li class="sublist"><a href="#mo1iii">iii) or download to your mobile</a></li>
           <br> 
           <li class="list">2. Setting up your desktop music player</li>
           <li class="sublist"><a href="#mo2i">i) &nbsp;&nbsp;Where to go to download free desktop music players</a></li>
           <li class="sublist"><a href="#mo2ii">ii) &nbsp;How to set up a player as your default player</a></li>
           <li class="sublist"><a href="#mo2iii">iii) How to import downloaded music into your player's library</a></li>
           <li><br> </li>
           <li class="list"><a href="#mo3">3. About the music in this shop</a></li>
          </ul>
          <br> 
          <hr> 
          <p class="majortitle">1. Downloading your purchased music</p>
          <a name="mo1i"></a> 
          <p class="smallheading">i) <strong>Recommendation:</strong> use the Universal Music Downloader to download the full album package</p> 
          <p class="normaltext">Tip: make sure your Media Player (e.g. Windows Media Player or Real Player) is set up to import any new files that you download automatically to your media player?s library. </p> 
          <p class="normaltext">In Windows Media player for example, you can find the ?add to library? settings bottom left. One of them is called ?by monitoring folders??. Click this and then choose the folder(s) on your computer to which you usually download. </p> 
          <p class="smallheading">ii) Instead of using the Download Manager you can download a ZIP file of the album package. </p>
          <a name="mo1ii"></a> 
          <p class="normaltext">A ZIP file is originally a normal folder containing multiple files that have been packaged together with the program WINZIP into a single file for easier electronic distribution. This way you can download the whole album with one click. </p> 
          <p class="normaltext">You need then WINZIP or another similar program on your computer to ?unzip? or ?expand? the downloaded ZIP file back into the original format of folder and its contents of the tracks you have downloaded. </p> 
          <p class="normaltext">You must ?unzip? a ZIP file before you can import the music into the library of your media player. </p>
          <a name="setup"></a> 
          <p class="smallheading">iii) Download to your mobile</p>
          <a name="mo1iii"></a> 
          <p class="normaltext">We also offer the option of downloading tracks to your mobile phone. Please click <a href="http://store.universal-music.co.uk/restofworld/Customer-Services/Digital-Downloads/page/digitaldownloads#6" target="blank&quot;">here</a> for more information </p> 
          <p class="majortitle">2. Setting up your desktop music player </p>
          <a name="mo2i"></a> 
          <p class="smallheading">i) Where to go to download free desktop music players</p>
          <a name="mo2i"></a> 
          <ol class="disc"> 
           <li>iTunes (recommended for MP3 playback): <a class="underline" href="http://www.apple.com/itunes" target="_blank">www.apple.com/itunes</a></li>
           <li>Windows Media Player: <a class="underline" href="http://www.windowsmedia.com/" target="_blank">www.windowsmedia.com</a></li>
           <li>Real Player: <a class="underline" href="http://www.real.com/" target="_blank">www.real.com</a></li>
           <li>Media Monkey (recommended for FLAC playback): <a class="underline" href="http://www.mediamonkey.com/" target="_blank">www.mediamonkey.com</a></li>
           <li>Winamp: <a class="underline" href="http://www.winamp.com/" target="_blank">www.winamp.com</a></li>
          </ol>
          <br>
          <a name="setdefault"></a> 
          <p class="smallheading">ii) How to set up a player as your default player</p>
          <a name="mo2ii"></a> 
          <ol class="discnumbers"> 
           <li>iTunes: go to Menu &gt; Edit &gt; Preferences &gt; Advanced &gt; "use iTunes as the default player for audio files"</li>
           <li>Windows Media Player: go to Menu &gt; Tools &gt; Options &gt; File Types and check the mp3 box</li>
          </ol>
          <p class="normaltext"></p>
          <a name="import"></a> 
          <p class="smallheading">iii)How to import downloaded music into your player's library</p> 
          <p class="normaltext"> </p>
          <ol class="discnumbers"> 
           <li>iTunes: go to Menu &gt; File &gt; Add folder to library</li>
           <li>Windows Media Player: go to Menu &gt; File &gt; Add to library &gt; By Monitoring Folders? (automatic) or &gt; Add Folder? (manually)</li>
           <li>Real Player: go to Menu &gt; File &gt; Add Files to My Library &gt; Add Folder</li>
          </ol>
          <p></p>
          <a name="mo3"></a> 
          <p class="majortitle">3. About the music in this shop</p>
          <a name="mo3"></a> 
          <p class="normaltext">We offer downloads in two formats:</p> 
          <p class="normaltext"> </p>
          <ol class="disc"> 
           <li><strong>MP3 files</strong> at a high quality bit rate of 320 kbps (kilobits per second) </li>
           <li><strong>FLAC files</strong> that give you the full audio quality of the CD (16 bit/ 44,1khz) </li>
          </ol>
          <p></p> 
          <p class="normaltext"><strong><em>Please note that iTunes does not play FLAC files.</em></strong> We recommend you use either Media Monkey or Winamp. For information about converting FLAC to Apple Lossless or WMA please click <a class="underline" href="http://www.deutschegrammophon.com/en/cat/help">here</a>. </p> 
          <p class="normaltext">Both types are in an open format, mp3s can be played by almost all desktop players and portable devices. There is no restriction on making copies or burning CDs from these files for your personal use only. However, to pass copies of these files on to other people is an infringement of copyright law and the terms and conditions of this shop to which you have agreed with any registered purchase from this shop. </p>
