        <h2><a id="user-content-installation" class="anchor" href="#user-content-installation" rel="nofollow noopener external"></a>Installation</h2> 
        <p>Simply add a dependency on <code>jbzoo/simpletypes</code> to your project's <code>composer.json</code> file if you use <a href="http://getcomposer.org/" rel="nofollow noopener external">Composer</a> to manage the dependencies of your project. Here is a minimal example of a <code>composer.json</code> file that just defines a dependency on SimpleTypes:</p> 
        <pre>    {
        <span class="pl-s"><span class="pl-pds">"</span>require<span class="pl-pds">"</span></span><span class="pl-k">:</span> {
            <span class="pl-s"><span class="pl-pds">"</span>jbzoo/simpletypes<span class="pl-pds">"</span></span><span class="pl-k">:</span> <span class="pl-s"><span class="pl-pds">"</span>^1.0<span class="pl-pds">"</span></span>
        }
    }</pre> 
        <p>... or in a command line <code>composer require jbzoo/simpletypes</code></p> 
        <h2><a id="user-content-examples" class="anchor" href="#user-content-examples" rel="nofollow noopener external"></a>Examples</h2> 
        <pre><span class="pl-s1"><span class="pl-k">require_once</span> <span class="pl-s"><span class="pl-pds">'</span>./vendor/autoload.php<span class="pl-pds">'</span></span>; <span class="pl-c"><span class="pl-c">//</span> composer autoload.php</span></span>
<span class="pl-s1"></span>
<span class="pl-s1"><span class="pl-c"><span class="pl-c">//</span> Get needed classes</span></span>
<span class="pl-s1"><span class="pl-k">use</span> <span class="pl-c1">JBZoo\SimpleTypes\Config</span>,</span>
<span class="pl-s1">    <span class="pl-c1">JBZoo\SimpleTypes\Money</span>,</span>
<span class="pl-s1">    <span class="pl-c1">JBZoo\SimpleTypes\ConfigMoney</span>;</span>
<span class="pl-s1"></span>
<span class="pl-s1"><span class="pl-c"><span class="pl-c">//</span> Set config object for all Money objects as default</span></span>
<span class="pl-s1"><span class="pl-c1">Config</span><span class="pl-k">::</span>registerDefault(<span class="pl-s"><span class="pl-pds">'</span>money<span class="pl-pds">'</span></span>, <span class="pl-k">new</span> <span class="pl-c1">ConfigMoney</span>());</span>
<span class="pl-s1"></span>
<span class="pl-s1"><span class="pl-c"><span class="pl-c">//</span> Create any object, some different ways</span></span>
<span class="pl-s1"><span class="pl-smi">$money</span>  <span class="pl-k">=</span> <span class="pl-k">new</span> <span class="pl-c1">Money</span>(<span class="pl-s"><span class="pl-pds">'</span>10 eur<span class="pl-pds">'</span></span>);</span>
<span class="pl-s1"><span class="pl-smi">$weight</span> <span class="pl-k">=</span> <span class="pl-k">new</span> <span class="pl-c1">Weight</span>(<span class="pl-s"><span class="pl-pds">'</span>1000<span class="pl-pds">'</span></span>); <span class="pl-c"><span class="pl-c">//</span> Gram is default in the ConfigWeight class</span></span>
<span class="pl-s1"><span class="pl-smi">$length</span> <span class="pl-k">=</span> <span class="pl-k">new</span> <span class="pl-c1">Length</span>(<span class="pl-s"><span class="pl-pds">'</span>500 km<span class="pl-pds">'</span></span>);</span>
<span class="pl-s1"><span class="pl-smi">$money</span>  <span class="pl-k">=</span> <span class="pl-k">new</span> <span class="pl-c1">Money</span>(<span class="pl-s"><span class="pl-pds">'</span>100500 usd<span class="pl-pds">'</span></span>, <span class="pl-k">new</span> <span class="pl-c1">ConfigMoney</span>()); <span class="pl-c"><span class="pl-c">//</span> my custom params only for that object</span></span></pre> 
        <h2><a id="user-content-a-lot-of-types-are-ready-to-use" class="anchor" href="#user-content-a-lot-of-types-are-ready-to-use" rel="nofollow noopener external"></a>A lot of types are ready to use</h2> 
        <p>SimpleTypes has such ready configurations like</p> 
        <ul>
         <li><a href="https://github.com/JBZoo/SimpleTypes/blob/master/src/config/area.php" rel="nofollow noopener external">Area</a></li> 
         <li><a href="https://github.com/JBZoo/SimpleTypes/blob/master/src/config/degree.php" rel="nofollow noopener external">Degree</a> (geometry)</li> 
         <li><a href="https://github.com/JBZoo/SimpleTypes/blob/master/src/config/info.php" rel="nofollow noopener external">Info</a> (bytes, bits...)</li> 
         <li><a href="https://github.com/JBZoo/SimpleTypes/blob/master/src/config/length.php" rel="nofollow noopener external">Length</a></li> 
         <li><a href="https://github.com/JBZoo/SimpleTypes/blob/master/src/config/money.php" rel="nofollow noopener external">Money</a> (Currency converter)</li> 
         <li><a href="https://github.com/JBZoo/SimpleTypes/blob/master/src/config/temp.php" rel="nofollow noopener external">Temperature</a> (Kelvin, Fahrenheit, Celsius and etc)</li> 
         <li><a href="https://github.com/JBZoo/SimpleTypes/blob/master/src/config/volume.php" rel="nofollow noopener external">Volume</a></li> 
         <li><a href="https://github.com/JBZoo/SimpleTypes/blob/master/src/config/weight.php" rel="nofollow noopener external">Weight</a></li> 
        </ul>
        <p>You can add your own type. It's really easy. See this page below.</p> 
        <h3><a id="user-content-smart-and-useful-parser" class="anchor" href="#user-content-smart-and-useful-parser" rel="nofollow noopener external"></a>Smart and useful parser</h3> 
        <p>SimpleTypes has really smart parser for all input values. It can find number, understand any decimal symbols, trim, letter cases, e.t.c... and it works really fast!</p> 
        <pre><span class="pl-s1"><span class="pl-smi">$money</span> <span class="pl-k">=</span> <span class="pl-k">new</span> <span class="pl-c1">Money</span>(<span class="pl-s"><span class="pl-pds">'</span> - 1 2 3 , 4 5 6 rub <span class="pl-pds">'</span></span>);    <span class="pl-c"><span class="pl-c">//</span> Equals -123.456 rubles</span></span>
<span class="pl-s1"><span class="pl-smi">$money</span> <span class="pl-k">=</span> <span class="pl-k">new</span> <span class="pl-c1">Money</span>(<span class="pl-s"><span class="pl-pds">'</span>1.0e+18 EUR <span class="pl-pds">'</span></span>);             <span class="pl-c"><span class="pl-c">//</span> Really huge number. I'm rich! =)</span></span>
<span class="pl-s1"><span class="pl-smi">$money</span> <span class="pl-k">=</span> <span class="pl-k">new</span> <span class="pl-c1">Money</span>(<span class="pl-s"><span class="pl-pds">'</span> EuR 3,50 <span class="pl-pds">'</span></span>);</span>
<span class="pl-s1"><span class="pl-smi">$money</span> <span class="pl-k">=</span> <span class="pl-k">new</span> <span class="pl-c1">Money</span>(<span class="pl-s"><span class="pl-pds">'</span>usd<span class="pl-pds">'</span></span>);                      <span class="pl-c"><span class="pl-c">//</span> Just object with usd rule</span></span></pre>
