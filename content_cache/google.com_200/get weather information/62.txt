  <h1>INFO 3300 - Data-driven Web Applications</h1> 
  <p>This course will introduce students to working with data in the context of modern web applications. These include data representation with relational and non-relational databases, data mining to find patterns and make predictions, and graphical presentation for visualization.</p> 
  <p>Prerequisite: INFO 2300 is required because we will be working in a web environment. Writing web applications is about more than programming -- it's a complex ecosystem of technologies. Students will need to be familiar with Javascript, HTML, CSS, and client-server architectures. CS 2110 (object-oriented programming) is required for programming experience. <b>If you have not taken one or both of these courses, be prepared to present a portfolio of previous project-based experience.</b></p> 
  <p> Notes and "prompts" (pre-notes for upcoming classes) will be posted to <a href="https://github.com/mimno/info3300-spr2017">GitHub</a>. Assignments will be posted to <a href="https://cms.csuglab.cornell.edu/">CMS</a>. Questions should go to <a href="https://piazza.com/cornell/spring2017/infocs3300">Piazza</a>. </p> 
  <p>Absences from class will be excused for compelling reasons, such as sports competitions or illness with appropriate documentation from the Athletic Dept or Gannett. A small number of absences for personal reasons such as interview travel. Classes immediately before University breaks are not optional.   <h3>Topics</h3> 
  <ul> 
   <li>Javascript: Functional idioms, JSON.</li>  
   <li>Visual elements with SVG: shapes, lines, paths, and text.</li> 
   <li>Data-driven documents with <a href="http://d3js.org/">d3.js</a>: mapping data to HTML elements; interactivity with attributes and transitions.</li> 
   <li>Types of data: maps, networks, text, time-series. </li>
   <li>Models/Algorithms for data processing, such as: classification, clustering, force-directed layouts, network sampling, locality-sensitive hashing, visual hypothesis testing, multi-dimensional scaling, label propagation in networks, community detection, sequence smoothing.</li> 
  </ul>
