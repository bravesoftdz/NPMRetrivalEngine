           <h1 name="a010" id="a010" class="graf graf--h3 graf--leading graf--title">Beautiful Weather report in your Command&nbsp;Line</h1>
           <figure name="46df" id="46df" class="graf graf--figure graf-after--h3">
            <div class="aspectRatioPlaceholder is-locked" style="max-width: 700px; max-height: 442px;">

             <div class="progressiveMedia js-progressiveMedia graf-image" data-image-id="1*-siJPjiytjUivhXznCVtLg.png" data-width="933" data-height="589" data-action="zoom" data-action-value="1*-siJPjiytjUivhXznCVtLg.png">

            <figcaption class="imageCaption">
             curl -4 
             <a href="http://wttr.in/NYC" data-href="http://wttr.in/NYC" class="markup--anchor markup--figure-anchor" rel="nofollow noopener" target="_blank">http://wttr.in/NYC</a>

           <p name="238c" id="238c" class="graf graf--p graf-after--figure">Last night I was doing our regular <a href="https://rmotr.com/advanced-python-programming" data-href="https://rmotr.com/advanced-python-programming" class="markup--anchor markup--p-anchor" rel="noopener" target="_blank">Python online class</a> and I was trying to show how to use <a href="https://pypi.python.org/pypi/pip" data-href="https://pypi.python.org/pypi/pip" class="markup--anchor markup--p-anchor" rel="noopener" target="_blank">pip</a> to install an external package. As usual, the first demonstration of pip is with the great and ubiquitous <a href="http://github.com/kennethreitz/requests" data-href="http://github.com/kennethreitz/requests" class="markup--anchor markup--p-anchor" rel="noopener" target="_blank">Request package</a>. When I was trying to show how the parallelism with CURL, I used the amazing <a href="http://wttr.in/" data-href="http://wttr.in/" class="markup--anchor markup--p-anchor" rel="nofollow noopener" target="_blank">http://wttr.in</a> service.</p>
           <p name="2985" id="2985" class="graf graf--p graf-after--p">I thought I was teaching Python, but not the case. Everyone just started asking, “wow, what’s that? That’s so cool!”. So I kind of had to stop teaching Python and show wttr.in around ?. Anyway, it’s a service that I love and I use all the time, so here’s a quick reference for you.</p>
           <h4 name="40cb" id="40cb" class="graf graf--h4 graf-after--p">Using wttr.in</h4>
           <p name="38a8" id="38a8" class="graf graf--p graf-after--h4">Basically, the service lets you query weather information from different cities and airports. If you do it in your command line using curl or <a href="https://httpie.org/" data-href="https://httpie.org/" class="markup--anchor markup--p-anchor" rel="noopener" target="_blank">httpie</a> you get a nice text-formatted output. Here’s the complete reference (examples a few lines below):</p>
           <pre name="1137" id="1137" class="graf graf--pre graf-after--p">Usage:</pre>
           <pre name="e75b" id="e75b" class="graf graf--pre graf-after--pre">$ curl wttr.in # current location<br>$ curl wttr.in/muc # weather in the Munic airport</pre>
           <pre name="8448" id="8448" class="graf graf--pre graf-after--pre">Supported location types:</pre>
           <pre name="fc71" id="fc71" class="graf graf--pre graf-after--pre">/paris # city name<br>/muc # airport code (3 letters)<br>/@stackoverflow.com # domain name<br>/94107 # area codes</pre>
           <pre name="3a60" id="3a60" class="graf graf--pre graf-after--pre">Special locations:</pre>
           <pre name="50f5" id="50f5" class="graf graf--pre graf-after--pre">/moon # Moon phase (add ,+US or ,+France for these cities)</pre>
           <pre name="2556" id="2556" class="graf graf--pre graf-after--pre">Units:</pre>
