       <p>A few important points for newbies:</p>
       <aside class="nativo-promo smartphone" id=""> 
       </aside>
       <p>To install a package from CRAN, use the command <code>install.packages("packagename")</code> -- of course substituting the actual package name for packagename and putting it in quotation marks. Package names, like pretty much everything else in R, are case sensitive.</p>
       <p>To install from GitHub, it's easiest to use the install-github function from the devtools package, using the format <code>devtools::install_github("githubaccountname/packagename")</code>. That means you first want to install the devtools package on your system with <code>install.packages("devtools")</code>. Note that devtools sometimes needs some extra non-R software on your system -- more specifically, an <a href="http://cran.r-project.org/bin/windows/Rtools/" target="_blank" rel="nofollow">Rtools download for Windows</a> or <a href="https://developer.apple.com/xcode/downloads/" target="_blank" rel="nofollow">Xcode for OS X</a>. There's <a href="https://github.com/hadley/devtools" target="_blank" rel="nofollow">more information about devtools here</a>.</p>
       <p>In order to use a package's function during your R session, you need to do one of two things. One option is to load it into your R session with the <code>library("packagename")</code> or <code>require("packagename")</code>. The other is to call the function including the package name, like this: <code>packagename::functioname()</code>. Package names, like pretty much everything else in R, are case sensitive.</p>
