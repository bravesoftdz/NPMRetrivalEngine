         <section class="section-container collapsed" id="about-project">
          <h2 class="section-title"><a class="title-anchor" href="#about-project"><i class="fa fa-link"></i></a><span class="title title-toggle">Story </span></h2>
          <div class="section-content hljs-active hljs-monokai">
           <div class="medium-editor" itemprop="text">
            <p> </p>
            <p>A small project to demonstrate Edison board and GPIO pins to weather information in a LCD display.</p>
            <h3 id="toc-parts-needed-0"><a class="title-anchor" href="#toc-parts-needed-0" rel="nofollow"><i class="fa fa-link"></i></a><span>Parts Needed:</span></h3>
            <ul>
             <li><strong>1x</strong><span> Edison and Sparkfun block stack</span></li>
            </ul>
            <ul>
             <li><strong>1x</strong><span> Breadboard</span></li>
            </ul>
            <ul>
             <li><strong>1x</strong><span> Character LCD</span></li>
            </ul>
            <ul>
             <li><strong>1x</strong><span> 10k Potentiometer</span></li>
            </ul>
            <ul>
             <li><strong>16x</strong><span> Jumper Wires</span></li>
            </ul>
            <p> </p>
            <p><a href="http://johnny-five.io/" rel="nofollow">Johnny-Five </a><span>is a JavaScript framework for programming robotics. Much like MRAA, it simplifies the calls we need to make to control various pieces of hardware. At the time of this writing, MRAA did not support parallel interface character LCDs, but Johnny-Five does!</span></p>
            <p>Johnny-Five has support for several boards, and the Edison is one of them. As a result, we can include Johnny-Five in our program (like any Node module), and have it handle LCD communications for us.</p>
            <h3 id="toc-yahoo-weather-1"><a class="title-anchor" href="#toc-yahoo-weather-1" rel="nofollow"><i class="fa fa-link"></i></a><span>Yahoo Weather</span></h3>
            <p><span>There are plenty of </span><a href="http://www.programmableweb.com/news/top-10-weather-apis/analysis/2014/11/13" rel="nofollow">weather APIs </a><span>out there, and all of them would likely work for getting current weather data about a particular city. However, </span><a href="https://developer.yahoo.com/weather/" rel="nofollow">Yahoo Weather </a><span>allows us to make calls without signing up for an account and returns data in JSON format, which is easily parsed by JavaScript.</span></p>
            <p>We can perform a database lookup on Yahoo’s weather servers by constructing a SQL-like string called “YQL”. In this example, we will ultimately create a program that constructs a YQL that requests weather data for a particular city and sends it to Yahoo. Yahoo then returns a JSON object that we can parse for temperature and weather conditions.</p>
            <h3 id="toc-json-2"><a class="title-anchor" href="#toc-json-2" rel="nofollow"><i class="fa fa-link"></i></a><span>JSON</span></h3>
            <p><span>JavaScript Object Notatation, or </span><a href="http://www.json.org/" rel="nofollow">JSON,</a><span> is a text format for exchanging data between computers. It has the benefit of also being easily read by humans. It consists of a collection of name and value pairs (given as “name” : “value”) within an object (as denoted by curly braces {}).</span></p>
            <h3 id="toc-how-to-setup-and-run-3"><a class="title-anchor" href="#toc-how-to-setup-and-run-3" rel="nofollow"><i class="fa fa-link"></i></a><span>How to setup and run:</span></h3>
