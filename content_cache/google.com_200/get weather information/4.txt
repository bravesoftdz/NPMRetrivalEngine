     <h1>About</h1> 
     <p> There really should be a conclusive Javascript weather library. Weather.js fetches data from <a href="http://openweathermap.org">openweathermap.org</a> (no affiliation) and makes querying all kinds of weather related information easy. Since other providers format their output differently, currently this is the only source provider. </p> 
     <p> Weather.js is still in early development so expect changes and please contribute! Among the features I hope to incorporate: </p>
     <ul> 
      <li>historical weather information</li> 
      <li>API key usage</li> 
      <li>more data sources</li> 
      <li>more conversions!</li> 
     </ul> 
     <p></p> 
     <h1>Install</h1> 
     <p> Weather.js works in the browser and node.js. Take your pick. For the browser, <a href="http://github.com/noazark/weather">download the most recent version on github</a>. For use in node, do the install thing. </p> 
     <pre class="prettyprint cm-s-solarized-light">
npm install -g weather.js</pre> 
     <p class="alert alert-error"> <strong>Important!</strong> In the browser you will need to also include <a href="http://sugarjs.com/">Sugar.js</a> and <a href="http://jquery.com/">jQuery</a>. I will try to remove these dependancies as soon as possible. </p> 
     <h1>Usage</h1> 
     <p> At the moment you can access the current weather conditions and the forecast for any city. By default it will use the closest match as returned by Open Weather Map. </p> 
     <pre class="prettyprint cm-s-solarized-light">
Weather.getCurrent("Kansas City", function(current) {
  console.log(
    ["currently:",current.temperature(),"and",current.conditions()].join(" ")
  );
});

Weather.getForecast("Kansas City", function(forecast) {
  console.log("forecast high: " + forecast.high());
  console.log("forecast low: " + forecast.low());
});
</pre>
