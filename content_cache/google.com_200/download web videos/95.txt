      <h2>Who is This For?</h2> 
      <hr> 
      <p>Almost everyone! Advanced JavaScript developers will be able to go through the course a little more quickly, but I've designed this course to be as approachable as possible.</p> 
      <p>Some of the videos feature a review of topics that are a little shaky for some such as scoping, keyword <code>this</code> and prototypal inheritance.</p> 
      <p>This isn't JavaScript from scratch, so if you are looking for a beginner course this probably isn't it. You should have some knowledge of functions, variables, Objects, Arrays and other fundamentals.</p> 
      <ul> 
       <li>Those reliant on jQuery, looking to have a better grasp on the core features of JavaScript.</li> 
       <li>Existing JS developers who want to bridge their personal learning gaps and learn ES6 start to finish.</li> 
       <li>PHP &amp; WordPress Developers looking to become stronger on the client side.</li> 
       <li>Anyone who wants to get better at JavaScript and learns well from seeing both fundamental and advanced concepts in practice.</li>
