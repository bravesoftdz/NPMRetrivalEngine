       <h1 class="with-space">Connect&nbsp;</h1>
      </div>
      <div id="c399" class="csc-default">
       <div class="js-content-tabs">
        <ul class="content-tab-header horizontal">
         <li><a href="#overview-399">Overview</a></li>

        <div class="content-tab" id="overview-399">
         <div id="c256" class="csc-default">
          <p class="with-trim"><a href="fileadmin/media/Asperasoft.com/Software/Transfer_Clients/Connect_Browser_Plug-in/Aspera_Connect_big.png" class="enlarge"><img style="float: right;" src="fileadmin/media/Asperasoft.com/Software/Transfer_Clients/Connect_Browser_Plug-in/Aspera_Connect_big.png" alt="" width="300" height="157"></a>Aspera Connect is a self-installing app that powers high-speed uploads and downloads with the Aspera Connect Server, also enabling web-based transfers for Aspera web application Faspex<sup>TM</sup> and Shares. Compatible with most standard browsers and sporting a very light system footprint, Aspera Connect integrates all of Asperas high-performance transport technology in a small, easy-to-use package that provides unequaled control over transfer parameters.</p> 
          <h2><strong>Use Cases</strong></h2> 
          <h3><strong>Browser-based file transfers</strong></h3> 
          <p class="with-trim">Enable teams, customers and partners to easily transfer files and directories at high speed to and from your Aspera servers via any web browser. Without any software to download and install, users can transfer data via any computer, at any time, over any connection (WAN, satellite, wireless).</p> 
          <h2><strong>Product Highlights</strong></h2> 
          <h3>Localized Languages</h3> 
          <p>The latest version of Connect now supports Simplified Chinese, Japanese, French, and Spanish in addition to English. The high-speed transfer plug-in automatically sets the language based on your system's language settings. </p> 
          <h3><strong>Lightweight and browser-independent</strong></h3> 
          <p>Aspera Connect will install and launch automatically when a web application such as Aspera Shares initiates FASP<sup>®</sup> transfers. The transfers are performed independent of the browser, without locking the page during the transfer, so that users can continue working. </p> 
          <h3><strong>Reliable and secure</strong></h3> 
          <p>With built-in automatic resume of partial transfers and retry of failed transfers, Aspera Connect assures that critical files make it across the network on time and intact. All files can be encrypted in transfer and at rest, ensuring security of digital assets, without compromising transfer speeds. </p> 
          <h3><strong>Real-time transfer monitoring and control</strong></h3> 
          <p>An easy-to-use interface provides real-time status of transfer progress and transfer rate, allowing users to adjust transfer rates on the fly and to cancel/pause/resume queued transfers. </p> 
          <h3>OPEN SDK WITH RICH JAVASCRIPT APIS</h3> 
          <p>Asperas Connect SDK allows developers to easily integrate Aspera file transfer capabilities into their web applications with complete control over branding and the end-user experience. From installation to delivery, Aspera Connect SDK gives developers full control over how Aspera file transfer capabilities are exposed to their end-users with an "in-browser" user experience. By leveraging the JavaScript APIs, developers are able to control both the look and feel and workflow of transfer initiation, transfer progress, transfer control (such as pause, resume, cancel, etc.), error handling, and full control over transfer parameters.</p>
         </div>
        </div>
        <div class="content-tab" id="features-399">
         <div id="c255" class="csc-default">
          <h2><strong>Unrivaled Aspera performance</strong></h2>
          <ul class="with-trim"> 
           <li>Powered by Asperas patented&nbsp;FASP<sup>®</sup> transport technology, content is sent at high speed, regardless of file sizes, transfer distances or network conditions.</li> 
           <li>Precise bandwidth control ensures the entire allocated bandwidth is utilized to achieve maximum transfer speeds, while remaining fair to other business-critical network traffic.</li> 
           <li>100% reliable data delivery with auto resume of failed or incomplete transfers.</li> 
           <li>Real-time reporting of transfer progress and performance.</li> 
           <li>Automatic, fast and lossless inline compression reduces data set transfer sizes, providing further boost to Aspera's industry-leading transfer performance.</li> 
           <li>Ability to switch to slower HTTP-based transfers for highly restrictive network environments.</li> 
          </ul>
          <h2><strong>Simple, lightweight App</strong></h2>
          <ul class="with-trim"> 
           <li>This simple app installs and launches automatically within the browser, as needed by the web application.</li> 
           <li><span>Rich JavaScript API for fully-integrated branded experience, persistence across host reboots, and in browser control.</span></li> 
           <li><span>Comprehensive JavaScript API for full-featured embedding in any 3rd party application with latest 3.6.5 JavaScript API that features drag-and-drop of files and folders for Firefox, Chrome, and Edge &amp; interactive browse to select files and folders to transfer.</span></li> 
           <li> 
            <div class="O1">
             Connect 3.6 is available with new plug-in-less architecture for Chrome, Firefox &amp; MS Edge.
            </div> </li> 
           <li>Transfers happen independent of the browser, so that users can continue their work uninterrupted.</li> 
           <li>Easy-to-use interface provides real-time visual monitoring of transfer progress and queued packages.</li> 
           <li>Ability to adjust bandwidth utilization on the fly is ideal for boosting transfer speeds for time-critical content or throttling down less important transfers to leave bandwidth for other business-critical network traffic (e.g. video conferencing)</li> 
           <li>Automatic UI language selection, based on operating system language setting, supports English, Spanish, French, Japanese and Simplified Chinese, providing an easy-to-understand interface during installation and use.</li> 
          </ul>
          <h2><strong>Bullet-proof security</strong></h2> 
          <p>Keep your business-critical digital assets safe with Asperas enterprise-grade security features.</p>
          <ul> 
           <li>Through SSH authentication establishes secure connections between the app and the transfer server.</li> 
           <li>AES-128 data encryption in transit and at rest ensures that files are accessible only to those with the right password to decrypt them.</li> 
           <li>FIPS 140-2 compliant cryptographic module provides strong encryption that is accredited by the US Government.</li> 
           <li>Data integrity verification for each transmitted block guards against man-in-the-middle attacks and attempts at data or content theft.</li> 
           <li>Supports server side encryption at rest in addition to client-side for both FASP and HTTP fallback sessions</li> 
          </ul>
          <p><a href="technology/transport/fasp/fasp-security-model/">&gt;&gt; Learn more about Aspera's&nbsp;FASP<sup>®</sup> security model</a></p>

           <h2>What's New with Connect 3.7</h2> 
           <p>Connect 3.7 provides several performance enhancements and usability improvements:</p>
           <ul> 
            <li><span style="font-size: 1.3rem;">Connect no longer limits the number of concurrent transfers to 100. The Transfers window displays the first 100 active transfers, along with a count of the remaining transfers. In these cases, for each transfer you delete from the Transfers window, another remaining transfer then appears</span></li> 
            <li><span style="font-size: 1.3rem;">The Connect troubleshooting tool (<a href="http://test-connect.asperasoft.com" target="_blank">http://test-connect.asperasoft.com</a>) has been incorporated into the banner that appears when Connect appears not to be installed.</span></li> 
            <li><span style="font-size: 1.3rem;">This release of Connect features various improvements to accessibility in use with screen reader tools</span></li> 
            <li><span style="font-size: 1.3rem;">The Connect SDK features several API enhancements. For detailed information, see the Aspera Developer Network at <a href="https://developer.asperasoft.com" target="_blank">https://developer.asperasoft.com</a></span></li>

             <h2>What's New with Connect 3.6</h2> 
             <p>Connect 3.6 has a new (pluginless) architecture using the latest native APIs provided by Chrome, Firefox, Safari, and MS Edge and Explorer browsers. Additional features include:</p>
             <ul> 
              <li><span style="font-size: 1.3rem;">High speed secure file and folder transfer with in browser transfer progress reporting and management, and resumable sessions across browser re-starts.</span></li> 
              <li><span style="font-size: 1.3rem;">Supports all major browser versions on Windows, Mac OS X, and Linux including the latest versions that have deprecated browser plugins and NPAPI.</span></li> 
              <li><span style="font-size: 1.3rem;">Has a comprehensive Javascript API for full-featured embedding in any 3rd party application. Upload/download any number of source items (files or directories) with option for unique naming per item. Register for transfer session progress callbacks in the browser and pause/resume/change bandwidth of running transfers. Integrate with any 3rd party Application's Single Sign-on with the node API.</span></li> 
              <li><span style="font-size: 1.3rem;">Latest 3.6.5 Javascript API features include:</span></li> 
              <li><span style="font-size: 1.3rem;">Drag-and-drop of files and folders for Firefox, Chrome, and Edge in addition to interactive browse to select files and folders to transfer.</span></li> 
              <li><span style="font-size: 1.3rem;">Fully customizable graphics and banner placement in installation.</span></li> 
              <li><span style="font-size: 1.3rem;">Connect installer and SDK are separated for easier local hosting in addition to Aspera hosted distribution.</span></li> 
              <li><span style="font-size: 1.3rem;">Supports server side encryption at rest in addition to client-side for both FASP and HTTP fallback sessions.</span></li> 
              <li><span style="font-size: 1.3rem;">Aspera is also currently previewing a new webRTC stack that extends the next-generation Aspera Connect core for streaming video.</span></li> 
             </ul>
