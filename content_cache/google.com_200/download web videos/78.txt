       <div class="column first"> 
        <h1>SIP over WebSocket</h1> 
        <div class="body"> 
         <p>JsSIP implements the <a href="https://tools.ietf.org/html/rfc7118">SIP WebSocket transport</a>. Enjoy the real integration of SIP within the Web and communicate with SIP networks out there.</p> 
         <p>This is pure SIP on the web (no protocol conversion, no limits).</p> 
         <div class="more-info"> 
          <a href="/documentation/misc/sip_websocket/">[ more info ]</a> 
         </div> 
        </div> 
       </div> 
       <div class="column"> 
        <h1>WebRTC</h1> 
        <div class="body"> 
         <p>JsSIP makes use of the <a href="/documentation/misc/webrtc/">WebRTC</a> stack present in modern web browsers for enabling audio/video realtime communication.</p> 
         <p>Zero plugins, zero vendor lock-in. Bye bye Flash and Java Applets!</p> 
         <div class="more-info"> 
          <a href="/documentation/misc/webrtc/">[ more info ]</a> 
         </div> 
        </div> 
       </div> 
       <div class="column"> 
        <h1>Easy to use</h1> 
        <div class="body"> 
         <p>JsSIP comes with an easy JavaScript <a href="/documentation/3.0.x/api/">API</a> that provides the user with full flexibility over the SIP application running in the web.</p> 
         <p>Enjoy coding the custom SIP logic for your web application.</p> 
         <div class="more-info"> 
          <a href="/documentation/3.0.x/getting_started/">[ more info ]</a> 
         </div> 
        </div>
