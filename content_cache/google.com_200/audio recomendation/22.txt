      <h1>Recommended formats</h1> 
      <div class="share-container"> 
       <div class="share"> 
        <div class="share-title">
          Share
         <i></i> 
        </div> 
        <div class="social"> 
         <div class="triangle"></div> 
         <div class="social-title">
          share this page
         </div> 
         <a class="popup facebook" rel="nofollow" href="https://www.facebook.com/share.php?u=https://www.ukdataservice.ac.uk/manage-data/format/recommended-formats.aspx&amp;t=UK%20Data%20Service%20-%20Recommended%20formats"><i></i></a> 
         <a class="popup twitter" rel="nofollow" href="https://www.twitter.com/share?text=UK%20Data%20Service%20-%20Recommended%20formats&amp;url=https://www.ukdataservice.ac.uk/manage-data/format/recommended-formats.aspx&amp;button_hashtag=UKDataService"><i></i></a> 
        </div> 
       </div> 
      </div> 
      <p><strong>This table contains guidance on file formats recommended and accepted by the UK Data Service for data sharing, reuse and preservation.</strong></p> 
      <p>You may need to convert your data files to a preservation file format.</p> 
      <p>We welcome <a href="/help/get-in-touch/data-management.aspx" title="Your query about data management">queries</a> from researchers about appropriate file formats for working and preservation, particularly early in the research process.</p> 
      <p>If you are unsure of the suitability of your file formats for the data you want to deposit with the UK Data Service, please <a href="/help/get-in-touch/deposit.aspx" title="Your query about depositing data">get in touch</a>.</p> 
      <table class="table-with-border" style="width: 100%; border-color: #000000; border-style: solid;" summary="Data formats currently recommended by the UK Data Archive for long-term preservation of research data" border="0"> 
       <tbody> 
        <tr valign="top">
         <th>Type of data</th>
         <th><strong>Recommended formats<br></strong></th>
         <th>Acceptable formats</th>
        </tr> 
        <tr valign="top"> 
         <td width="25%"> <p><strong>Tabular data with extensive metadata </strong></p> <p>variable labels, code labels, and defined missing values</p> </td> 
         <td width="35%"> <p>SPSS portable format (.por)</p> <p>delimited text and command ('setup') file (SPSS, Stata, SAS, etc.)</p> <p>structured text or mark-up file of metadata information, e.g. DDI XML file</p> </td> 
         <td width="40%"> <p>proprietary formats of statistical packages: SPSS (.sav), Stata (.dta), MS Access (.mdb/.accdb)</p> </td> 
        </tr> 
        <tr valign="top"> 
         <td> <p><strong>Tabular data with minimal metadata</strong></p> <p>column headings, variable names</p> </td> 
         <td> <p>comma-separated values (.csv)</p> <p>tab-delimited file (.tab)</p> <p>delimited text with SQL data definition statements</p> </td> 
         <td> <p>delimited text (.txt) with characters not present in data used as delimiters</p> <p>widely-used formats: MS Excel (.xls/.xlsx), MS Access (.mdb/.accdb), dBase (.dbf), OpenDocument Spreadsheet (.ods)</p> </td> 
        </tr> 
        <tr valign="top"> 
         <td> <p><strong>Geospatial data</strong></p> <p>vector and raster data</p> </td> 
         <td> <p>ESRI Shapefile (.shp, .shx, .dbf, .prj, .sbx, .sbn optional)</p> <p>geo-referenced TIFF (.tif, .tfw)</p> <p>CAD data (.dwg)</p> <p>tabular GIS attribute data</p> <p>Geography Markup Language (.gml)</p> </td> 
         <td> <p>ESRI Geodatabase format (.mdb)</p> <p>MapInfo Interchange Format (.mif) for vector data</p> <p>Keyhole Mark-up Language (.kml)</p> <p>Adobe Illustrator (.ai), CAD data (.dxf or .svg)</p> <p>binary formats of GIS and CAD packages</p> </td> 
        </tr> 
        <tr valign="top"> 
         <td> <p><strong>Textual data</strong></p> <p></p> </td> 
         <td> <p>Rich Text Format (.rtf)</p> <p>plain text, ASCII (.txt)</p> <p>eXtensible Mark-up Language (.xml) text according to an appropriate Document Type Definition (DTD) or schema</p> </td> 
         <td> <p>Hypertext Mark-up Language (.html)</p> <p>widely-used formats: MS Word (.doc/.docx)</p> <p>some software-specific formats: NUD*IST, NVivo and ATLAS.ti</p> </td> 
        </tr> 
        <tr valign="top"> 
         <td><strong>Image data</strong></td> 
         <td> <p>TIFF 6.0 uncompressed (.tif)</p> <p></p> </td> 
         <td> <p>JPEG (.jpeg, .jpg, .jp2) if original created in this format</p> <p>GIF (.gif)</p> <p>TIFF other versions (.tif, .tiff)</p> <p>RAW image format (.raw)</p> <p>Photoshop files (.psd)</p> <p>BMP (.bmp)</p> <p>PNG (.png)</p> <p>Adobe Portable Document Format (PDF/A, PDF) (.pdf)</p> </td> 
        </tr> 
        <tr valign="top"> 
         <td><strong>Audio data</strong></td> 
         <td> <p>Free Lossless Audio Codec (FLAC) (.flac)</p> </td> 
         <td> <p>MPEG-1 Audio Layer 3 (.mp3) if original created in this format</p> <p>Audio Interchange File Format (.aif)</p> <p>Waveform Audio Format (.wav)</p> </td> 
        </tr> 
        <tr valign="top"> 
         <td><strong>Video data</strong></td> 
         <td> <p>MPEG-4 (.mp4)</p> <p>OGG video (.ogv, .ogg)</p> <p>motion JPEG 2000 (.mj2)</p> </td> 
         <td> <p>AVCHD video (.avchd)</p> </td> 
        </tr> 
        <tr valign="top"> 
         <td><strong>Documentation and scripts</strong></td> 
         <td> <p>Rich Text Format (.rtf)</p> <p>PDF/UA, PDF/A or PDF (.pdf)</p> <p>XHTML or HTML (.xhtml, .htm)</p> <p>OpenDocument Text (.odt)</p> </td> 
         <td> <p>plain text (.txt)</p> <p>widely-used formats: MS Word (.doc/.docx), MS Excel (.xls/.xlsx)</p> <p>XML marked-up text (.xml) according to an appropriate DTD or schema, e.g. XHMTL 1.0</p> </td> 
        </tr> 
       </tbody> 
      </table>
