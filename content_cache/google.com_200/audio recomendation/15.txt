      <div id="features"> 
       <h2>Features</h2> 
       <ul> 
        <li>Retreive metadata such as viewcount, duration, rating, author, thumbnail, keywords</li> 
        <li>Download video or audio at requested resolution / bitrate / format / filesize</li> 
        <li>Command line tool (ytdl) for downloading directly from the command line</li> 
        <li>Retrieve the URL to stream the video in a player such as vlc or mplayer</li> 
        <li>Works with age-restricted videos and non-embeddable videos</li> 
        <li>Small, standalone, single importable module file (pafy.py)</li> 
        <li>Select highest quality stream for download or streaming</li> 
        <li>Download video only (no audio) in m4v or webm format</li> 
        <li>Download audio only (no video) in ogg or m4a format</li> 
        <li>Retreive playlists and playlist metadata</li> 
        <li>Works with Python 2.6+ and 3.3+</li> 
        <li>Optionally depends on youtube-dl (recommended; more stable)</li> 
       </ul> 
      </div>

      <div id="installation"> 
       <h2>Installation</h2> 
       <p>pafy can be installed using <a href="http://www.pip-installer.org" rel="nofollow">pip</a>:</p> 
       <pre>$ <span class="o">[</span>sudo<span class="o">]</span> pip install pafy
</pre> 
       <p>or use a <a href="http://virtualenv.org" rel="nofollow">virtualenv</a> if you don’t want to install it system-wide:</p> 
       <pre>$ virtualenv venv
$ <span class="nb">source</span> venv/bin/activate
$ pip install pafy
</pre> 
       <p>Alternatively you can just grab the pafy.py file and import it in your python code:</p> 
       <pre>wget https://raw.githubusercontent.com/np1/pafy/master/pafy/pafy.py
</pre>

      <div id="command-line-tool-ytdl-usage"> 
       <h2>Command Line Tool (ytdl) Usage</h2> 
       <pre>usage: ytdl <span class="o">[</span>-h<span class="o">]</span> <span class="o">[</span>-i<span class="o">]</span> <span class="o">[</span>-s<span class="o">]</span>
            <span class="o">[</span>-t <span class="o">{</span>audio,video,normal,all<span class="o">}</span> <span class="o">[{</span>audio,video,normal,all<span class="o">}</span> ...<span class="o">]]</span>
            <span class="o">[</span>-n N<span class="o">]</span> <span class="o">[</span>-b<span class="o">]</span> <span class="o">[</span>-a<span class="o">]</span>
            url

YouTube Download Tool

positional arguments:
  url                   YouTube video URL to download

optional arguments:
  -h, --help            show this <span class="nb">help</span> message and <span class="nb">exit</span>
  -i                    Display vid info
  -s                    Display available streams
  -t <span class="o">{</span>audio,video,normal,all<span class="o">}</span> <span class="o">[{</span>audio,video,normal,all<span class="o">}</span> ...<span class="o">]</span>
                        Stream types to display
  -n N                  Specify stream to download by stream number <span class="o">(</span>use -s to
                        list available streams<span class="o">)</span>
  -b                    Download the best quality video <span class="o">(</span>ignores -n<span class="o">)</span>
  -a                    Download the best quality audio <span class="o">(</span>ignores -n<span class="o">)</span>

       <h2>ytdl Examples</h2> 
       <p>Download best available resolution (-b):</p> 
       <pre>$ ytdl -b <span class="s2">"http://www.youtube.com/watch?v=cyMHZVT91Dw"</span>
</pre> 
       <p>Download best available audio stream (-a) (note; the full url is not required, just the video id will suffice):</p> 
       <pre>$ ytdl -a cyMHZVT91Dw
</pre> 
       <p>get video info (-i):</p> 
       <pre>$ ytdl -i cyMHZVT91Dw
</pre> 
       <p>list available dowload streams:</p> 
       <pre>$ ytdl cyMHZVT91Dw

Stream Type    Format Quality         Size
------ ----    ------ -------         ----
<span class="m">1</span>      normal  webm   <span class="o">[</span>640x360<span class="o">]</span>       <span class="m">33</span> MB
<span class="m">2</span>      normal  mp4    <span class="o">[</span>640x360<span class="o">]</span>       <span class="m">23</span> MB
<span class="m">3</span>      normal  flv    <span class="o">[</span>320x240<span class="o">]</span>       <span class="m">14</span> MB
<span class="m">4</span>      normal  3gp    <span class="o">[</span>320x240<span class="o">]</span>        <span class="m">9</span> MB
<span class="m">5</span>      normal  3gp    <span class="o">[</span>176x144<span class="o">]</span>        <span class="m">3</span> MB
<span class="m">6</span>      audio   m4a    <span class="o">[</span>48k<span class="o">]</span>            <span class="m">2</span> MB
<span class="m">7</span>      audio   m4a    <span class="o">[</span>128k<span class="o">]</span>           <span class="m">5</span> MB

       <pre>$ ytdl -n2 cyMHZVT91Dw

       <p>Download m4a audio stream at 256k bitrate:</p> 
       <pre>$ ytdl -n10 cyMHZVT91Dw
</pre>
