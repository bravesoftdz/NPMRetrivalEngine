       <p>The goal of this tutorial is to guide you through the creation of a Reddit/Hacker News clone using the MEAN stack. By completing this tutorial, you will gain a basic understanding of the MEAN stack including building a REST interface with Express.js on top of Node.js and using that interface to perform CRUD operations on a database via an AngularJS frontend.</p> 
       <h2 id="why-mean-stack">Why MEAN Stack?</h2> 
       <p>The acronym "MEAN" stands for "<a href="http://www.mongodb.org/" target="_blank">MongoDB</a> <a href="http://expressjs.com/" target="_blank">Express.js</a> <a href="https://angularjs.org/" target="_blank">AngularJS</a> <a href="http://nodejs.org/" target="_blank">Node.js</a>" and represents a group of technologies which are known to synergize well together. The major benefit of the MEAN stack is that it's extremely quick to prototype with. Node.js allows you to use Javascript on the backend as well as the frontend which can save you from having to learn a separate language. In addition, the <a href="http://www.mongodb.com/nosql-explained" target="_blank">NoSQL</a> nature of MongoDB allows you to quickly change and alter the data layer without having to worry about migrations, which is a very valuable attribute when you're trying to build a product without clear specifications. Finally, these technologies have a lot of community support behind them so finding answers to questions or hiring help is going to be much easier using these technologies.</p> 
       <h2 id="prerequisites">Prerequisites</h2> 
       <p>This course assumes knowledge of programming and at least basic knowledge of JavaScript. In addition, you should be comfortable with basic web application concepts including <a href="http://en.wikipedia.org/wiki/Representational_state_transfer" target="_blank">REST</a> and <a href="http://en.wikipedia.org/wiki/Create,_read,_update_and_delete" target="_blank">CRUD</a>. Before you begin, you will also need to have Node.js and MongoDB already installed. Because you will need to install various packages for Node.js, you should follow <a href="https://nodejs.org/en/download/package-manager/" target="_blank">these installation instructions</a> which use <a href="https://www.npmjs.org/" target="_blank">npm</a>. Installation instructions for MongoDB can be found <a href="http://docs.mongodb.org/manual/installation/" target="_blank">on the Mongo website</a>. This tutorial will be based on AngularJS v1.5.7, Node.js v7.2.0 and MongoDB v3.4.0.</p>

       <h2 id="recommendations-for-completing-this-tutorial">Recommendations for Completing this Tutorial</h2> 
       <p>Throughout the course of this tutorial, links to additional concepts and information will be included. You can use these links as supplementary material which can help you gain insight into the stack and its various components. As always, if you have any questions Google and <a href="http://stackoverflow.com/" target="_blank">Stackoverflow</a> are your best bet.</p> 
       <p>We at Thinkster are firm believers in actually writing code. Therefore we <strong>strongly</strong> encourage you to type out all the code instead of copy+pasting it.</p> 
       <h2 id="project-specifications">Project Specifications</h2> 
       <p>Before beginning work on any project, it's usually a good idea to know what you're building. Below is a basic list of things we want our users to be able to do:</p> 
       <ul> 
        <li>Create new posts</li> 
        <li>View all posts ordered by upvotes</li> 
        <li>Add comments about a given post</li> 
        <li>View comments for a given post</li> 
        <li>Upvote posts and comments</li> 
       </ul> 
       <p>In addition to technologies that make up MEAN, we're going to enlist the help of several other libraries to help us achieve our goals:</p>

        <li><a href="http://mongoosejs.com/" target="_blank">Mongoose.js</a> for adding structure to MongoDB</li>

        <li><a href="http://getbootstrap.com/" target="_blank">Twitter Bootstrap</a> for some quick styling</li> 
       </ul>
