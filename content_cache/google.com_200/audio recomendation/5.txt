        <h3 class="first"><img src="./images/icons/misc/star.gif" width="16" height="16" alt="" title=""> <a href="#p10572">Installing FileBot on Synology NAS</a></h3> 
        <p class="author"> <a class="unread" href="./viewtopic.php?p=10572&amp;sid=ba6158bcd89eacce8a303b5941405838#p10572" title="Post"> <i class="icon fa-file fa-fw icon-lightgray icon-md" aria-hidden="true"></i><span class="sr-only">Post</span> </a> <span class="responsive-hide">by <strong><a href="./memberlist.php?mode=viewprofile&amp;u=54&amp;sid=ba6158bcd89eacce8a303b5941405838" style="color: #00AA00;" class="username-coloured">rednoah</a></strong> » </span>10 Jul 2014, 06:56 </p> 
        <div class="content">
         FileBot now has it's own 
         <strong>Package Source</strong> so you can download and install FileBot conveniently from the 
         <strong>Package Center</strong> like any other Synology package. 
         <img class="smilies" src="./images/smilies/icon_e_wink.gif" width="15" height="17" alt=";)" title="Wink">
         <br> 
         <br> 
         <br> 
         <strong><span style="text-decoration: underline">Add Package Source:</span></strong>
         <br> Package Center ? Settings ? Package Sources ? Add ? Name: 
         <span style="color: #4040FF">FileBot</span> and Location: 
         <span style="color: #4040FF"><a href="https://get.filebot.net/syno/" class="postlink">https://get.filebot.net/syno/</a></span> ? OK 
         <div class="codebox">
          <p>Code: <a href="#" onclick="selectCode(this); return false;">Select all</a></p>
          <pre><code>https://get.filebot.net/syno/</code></pre>
         </div> 
         <img src="https://www.filebot.net/syno/package-center-snapshot.png" class="postimage" alt="Image">
         <br> 
         <br> 
         <strong><span style="text-decoration: underline">FileBot</span></strong>
         <br> This package will install the 
         <strong>filebot</strong> command-line tools. You will be able to run filebot commands from the 
         <strong>shell</strong> via 
         <strong>SSH</strong>.
         <br> 
         <br> 
         <img src="https://www.filebot.net/syno/filebot-snapshot.png" class="postimage" alt="Image">
         <br> 
         <br> 
         <br> 
         <strong><span style="text-decoration: underline">FileBot Node</span></strong>
         <br> This package will install and run the 
         <strong>FileBot Node</strong> web application. You will be able to execute, schedule and monitor 
         <strong>filebot</strong> commands with a simple 
         <strong>Synology DSM UI</strong>.
         <br> 
         <br> 
         <img src="https://www.filebot.net/syno/filebot-node-snapshot.png" class="postimage" alt="Image">
         <br> 
         <br> 
         <br> 
         <strong><span style="text-decoration: underline">Notes:</span></strong> 
         <ul> 
          <li>FileBot requires <strong>java</strong>. You will need to install the latest Java <span style="color: #808080"><span style="font-size: 75%; line-height: normal">(Oracle Java recommended)</span></span> package.</li> 
          <li>FileBot Node requires <strong>node</strong>. You will need to install the latest <strong>Node.js</strong> <span style="color: #808080"><span style="font-size: 75%; line-height: normal">(Node.js v4 recommended)</span></span> package.</li> 
          <li>Please read <a href="https://www.filebot.net/forums/viewtopic.php?f=13&amp;t=2733" class="postlink">How to use FileBot Node</a> to learn how <strong>FileBot Node</strong> and the <strong>filebot</strong> command-line tools are meant to be used.</li> 
          <li>If you process <strong>music</strong>, you need to install <strong>MediaInfo</strong> and <strong>Chromaprint</strong>.</li> 
          <li>If you use <strong>mediainfo bindings</strong> in your format, you need to install <strong>MediaInfo</strong>.</li> 
          <li><strong>Java 8</strong> is required. My <a href="https://github.com/rednoah/java-installer" class="postlink">Unofficial Java Installer</a> package will help you install the latest Oracle JDK.</li> 
          <li>Running commands from the login shell is not the same as using Synology DSM Task Scheduler or cron because they'll not initialise with <strong>~/.profile</strong> so keep in mind that <strong>$PATH</strong> will be different.</li> 
          <li><span style="color: #BF0000"><strong>filebot must not be run as root</strong></span> and it is recommended that you <strong>always run filebot as admin</strong></li>
         </ul> 
         <br> 
         <strong><span style="text-decoration: underline">Optional Features</span></strong>
         <br> If you want support for MediaInfo and AcoustID you will need to install the 
         <strong>MediaInfo</strong> and 
         <strong>Chromaprint</strong> packages from 
         <a href="https://synocommunity.com/" class="postlink">SynoCommunity</a>.
         <br> 
         <br> 
         <br> 
         <span style="color: #4040FF"><strong>MediaInfo</strong></span>: 
         <span style="color: #00BF00">libmediainfo.so</span> is required for reading video/audio information from media files
         <br> 
         <img src="https://www.filebot.net/syno/mediainfo-thumbnail.png" class="postimage" alt="Image">
         <br> 
         <br> 
         <br> 
         <span style="color: #4040FF"><strong>Chromaprint</strong></span>: 
         <span style="color: #00BF00">fpcalc</span> is required for computing acoustic fingerprints and matching music files with metadata
         <br> 
         <img src="https://www.filebot.net/syno/acoustid-thumbnail.png" class="postimage" alt="Image">
         <br> 
         <br> 
         <br> 
         <em><span style="font-size: 85%; line-height: normal">Built with <a href="https://github.com/rednoah/ant-spk" class="postlink">ant-spk</a></span></em>
