             <h2>The Problem of Finding Good PHP Packages</h2>
             <p>Well, when you have to solve a programming problem, you often seek advice from other developers to find the best component that exists to solve that problem, so you can either use that component just like that or to take ideas on how that component solves your problem.</p>
             <p>However, sometimes you are in a hurry and cannot find anybody qualified enough to give you an advice about existing components. Then you often search for the component on Google or use specific sites like PHP Classes.</p>
             <p>The problem with that approach is that you often find many components of varied levels of quality, so it is not easy to determine which one of those components is the best for you.</p>
             <h2>Good Developer Advice</h2>
             <p>Now, PHP Classes provides you a better solution for that problem. In 2014 the site launched a <a href="http://www.phpclasses.org/blog/post/250-There-Is-a-PHP-or-JS-Class-for-That-Community-based-Best-Package-Recommendations.html">package recommendation system</a>. Users go there and describe their problems. Then a qualified developer can come and give advice about which component that exists on the site that could solve your problem.</p>
             <p>This is great but sometimes it takes hours or days for another developer to come and give you some advice.</p>
             <p>On the other hand, many times users have needs that other users already had in the past, so chances are that the advice you need was already given to another user before.</p>
             <h2>Combining the Accuracy of Good Developer Advice with the Speed of Response of Search Engines</h2>
             <p>Now PHP Classes makes it easy for you to find components advised by other users in the past.</p>
             <p>It is easy. Just go on <a href="http://www.phpclasses.org/search.html">PHP Classes search page</a> and enter some relevant keywords related with your problem.</p>
             <p>The site will show you immediately existing package recommendations related with those keywords.</p>
             <p style="text-align: center;"><span itemprop="image" itemscope itemtype="http://schema.org/ImageObject">
               <meta itemprop="width" content="480">
               <meta itemprop="height" content="282"><img itemprop="url" src="https://files.phpclasses.org/files/blog/file/Search-User-Recommendations.png" alt="Search User Recommendations" title="Search User Recommendations" width="480" height="282"></span></p>
             <p>You can go on the recommendation pages and see the discussions that happened between the user looking for the advice and the developers that gave the recommendations.</p>
             <p>Chances are that the original user problem is the same as yours. Hopefully the recommendations really address well your needs too.</p>
             <p>In that case, just click on the recommended package links to know the recommended package better. There you can view the package code for instance, as well other types of information, as well download the package in several types of supported formats, or even install your package using the Composer tool directly from the PHP Classes Composer repository.</p>
             <p>As you may know the <a href="http://www.phpclasses.org/blog/package/8429/post/1-Using-Composer-to-Install-JavaScript-CSS-and-Images-Under-the-Web-Document-Directory.html">PHP Classes site supports the Composer</a> tool since 2013, so you can install any package from the PHP Classes site using that excellent tool.</p>
             <h2>Getting New Types of Package Recommendations</h2>
             <p>If for some reason you did not find a suitable package recommendation, you can post a more detailed recommendation request for a new package using the <a href="http://www.phpclasses.org/recommend_package/">Request new recommendation</a> link and then you can give all the details about your needs, detailing them as much as possible and after the request is submitted you need to wait a little bit until another developer can give you a good recommendation.</p>
             <h2>Recommending Your Own Packages</h2>
             <p>On the other hand, if you are a developer that wants to have greater exposure to your existing packages, you can also go on the site search page and recommend your packages to users that seek solutions for problems your packages solve.</p>
             <p>Type some keywords in the search box. Then you can see all the recommendations so far on the User recommended tab. Go on each recommendation page and then use the Recommend package form to enter the package page and an explanation telling the user why that package that you have solves the user problem.</p>
             <h2>Recommending Your Innovative Packages</h2>
             <p>Furthermore, there is a section on which you find <a href="http://www.phpclasses.org/recommend/featured/">featured package recommendation requests</a>. These are requests for which there was no package available on the site to solve the user problems.</p>
             <p>If you submit a package that addresses a problem of a featured request, chances are that your package is innovative because there was no other solution before that could address that problem, and you qualify to be nominated to the <a href="http://www.phpclasses.org/award/innovation/">Innovation Award</a>, thus having the chance to earn more visibility to your package, and also earn prizes and public recognition.</p>
             <h2>Conclusion</h2>
             <p>Well, this is it. I am sure this feature will save you a lot time trying many packages that you could find in PHP Classes or similar sites.</p>
             <p>So far over <a href="http://www.phpclasses.org/recommend/">400 package recommendation requests</a> were posted on the site and new requests come every day, so you can find a suitable recommendation related with your needs.</p>
             <p>Over time there will be more and more problems for which the site will have a good package recommendation to solve those problems, thus making the site more useful for you and to all developers that want to get more exposure to their packages.</p>
             <p>If you liked this feature, or you have questions, or you suggestions to make it better, post a comment here.</p>
