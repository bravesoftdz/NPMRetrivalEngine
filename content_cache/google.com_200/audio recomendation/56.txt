         <p>This application is a <strong>Starter Kit</strong> (SK) that is designed to get you up and running quickly with a common industry pattern, and to provide information about best practices around Watson services. The <strong>Audio Analysis</strong> application was created to highlight the combination of the <a href="http://www.ibm.com/watson/developercloud/speech-to-text.html">Speech to Text</a> (STT) and <a href="http://www.ibm.com/watson/developercloud/alchemy-language.html">AlchemyLanguage</a> services as an <a href="#about-the-audio-analysis-pattern">Audio Analysis</a> tool. This application can serve as the basis for your own applications that follow that pattern.</p>

         <h3><a href="#how-this-app-works" aria-hidden="true" class="anchor" id="user-content-how-this-app-works">
           <svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewbox="0 0 16 16" width="16">
            <path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path>
           </svg></a>How this app works</h3> 
         <p>The Audio Analysis application extracts concepts from YouTube videos.</p> 
         <p>To begin, select or specify a YouTube video. As the video streams, the <a href="http://www.ibm.com/watson/developercloud/speech-to-text.html">Speech to Text</a> service transcribes its audio track. That text is then piped to the <a href="http://www.ibm.com/watson/developercloud/alchemy-language.html">AlchemyLanguage</a> service for analysis, it extracts concepts from the transcription with an associated score.</p>

         <ol> 
          <li>Clone the repository into your computer.</li> 
         </ol> 
         <pre lang="none"><code>git clone https://github.com/watson-developer-cloud/audio-analysis.git
</code></pre> 
         <ol> 
          <li> <p><a href="https://console.ng.bluemix.net/registration/">Sign up</a> in Bluemix or use an existing account.</p> </li> 
          <li> <p>If it is not already installed on your system, download and install the <a href="https://github.com/cloudfoundry/cli">Cloud-foundry CLI</a> tool.</p> </li> 
          <li> <p>Edit the <code>manifest.yml</code> file in the folder that contains your code and replace <code>audio-analysis-starter-kit</code> with a unique name for your application. The name that you specify determines the application's URL, such as <code>application-name.mybluemix.net</code>. The relevant portion of the <code>manifest.yml</code> file looks like the following:</p> 
           <div class="highlight highlight-source-yaml">
            <pre><span class="pl-ent">applications</span>:
- <span class="pl-ent">services</span>:
  - <span class="pl-s">speech-to-text-service</span>
  - <span class="pl-s">alchemy-language-service</span>
  <span class="pl-ent">name</span>: <span class="pl-s">application-name</span>
  <span class="pl-ent">command</span>: <span class="pl-s">npm start</span>
  <span class="pl-ent">path</span>: <span class="pl-s">.</span>
  <span class="pl-ent">memory</span>: <span class="pl-s">512M</span></pre>
           </div> </li> 
          <li> <p>Connect to Bluemix:</p> </li> 
         </ol> 
         <pre lang="none"><code>cf api https://api.ng.bluemix.net
cf login
</code></pre> 
         <ol> 
          <li>Create and retrieve service keys to access the <a href="http://www.ibm.com/watson/developercloud/alchemy-language.html">AlchemyLanguage</a> service:</li> 
         </ol> 
         <pre lang="none"><code>cf create-service alchemy_api free alchemy-language-service
cf create-service-key alchemy-language-service myKey
cf service-key alchemy-language-service myKey
</code></pre> 
         <ol> 
          <li>Create and retrieve service keys to access the <a href="http://www.ibm.com/watson/developercloud/speech-to-text.html">Speech to Text</a> service:</li> 
         </ol> 
         <pre lang="none"><code>cf create-service speech_to_text standard speech-to-text-service
cf create-service-key speech-to-text-service myKey
cf service-key speech-to-text-service myKey
</code></pre> 
         <ol> 
          <li>Create a <code>.env</code> file in the root directory of your clone of the project repository by copying the sample <code>.env.example</code> file using the following command:</li> 
         </ol> 
         <pre lang="none"><code>cp .env.example .env
</code></pre> 
         <p>You will update the <code>.env</code> with the information you retrieved in steps 6 and 7.</p> 
         <p>The <code>.env</code> file will look something like the following:</p> 
         <pre lang="none"><code>ALCHEMY_LANGUAGE_API_KEY=
SPEECH_TO_TEXT_USERNAME=
SPEECH_TO_TEXT_PASSWORD=
</code></pre> 
         <ol> 
          <li>Install the dependencies you application need:</li> 
         </ol> 
         <pre lang="none"><code>npm install
</code></pre> 
         <ol> 
          <li>Start the application locally:</li> 
         </ol> 
         <pre lang="none"><code>npm start
</code></pre> 
         <ol> 
          <li> <p>Open a browser and go to: <a href="http://localhost:3000/">http://localhost:3000/</a></p> </li> 
          <li> <p>Push the application to Bluemix:</p> </li> 
         </ol> 
         <pre lang="none"><code>cf push
</code></pre> 
         <p>After completing the steps above, you are ready to test your application. Start a browser and enter the URL of your application.</p> 
         <pre><code>        &lt;your application name&gt;.mybluemix.net
</code></pre>

           </svg></a>Using the Speech To Text and the AlchemyLanguage services</h4> 
         <p>When a quality audio signal contains terms found in the current source of concepts in AlchemyLanguage, the combination of Speech To Text and AlchemyLanguage can be used to analyze the audio source to build summaries, indices, and to provide recommendations for additional related content. Though the Speech-To-Text service supports several languages, the AlchemyLanguage service currently only supports English.</p> 
         <p>The Audio Analysis app uses the node.js Speech-To-Text JavaScript SDK, which is a client-side library for audio transcriptions from the Speech To Text service. It also uses the <code>concepts</code> feature from AlchemyLanguage to extract concepts.</p>
