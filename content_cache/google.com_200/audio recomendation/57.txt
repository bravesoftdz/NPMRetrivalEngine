  <h2 id="instructions">Installation Instructions</h2>
  <ol>
   <li>From REAPER: <u>Options &gt; Show resource path in explorer/finder</u></li>
   <li>Put the downloaded file in the <u>UserPlugins</u> subdirectory</li>
   <li>Restart REAPER. Done!</li>
  </ol>
  <h2 id="features">Main Features</h2>
  <ul>
   <li>Easy access to 800+ packages from the default repositories</li>
   <li>Import third-party repositories to get more content</li>
   <li>Install/update in batch or pick only what you need</li>
   <li>ReaScripts are automatically added to the Action List <small>(REAPER v5.12+)</small></li>
   <li>Opt-in setting to test pre-release packages</li>
   <li>Decentralized by design, free to use and open-source</li>
   <li><b>NEW!</b> Added support for themes, language packs and web browser interfaces</li>
   <li><b>NEW!</b> Per-package documentation display</li>
   <li><b>NEW!</b> Improved filtering options</li>
   <li><b>NEW!</b> Prompt to uninstall obsolete packages</li>
   <li>And a lot more...</li>
  </ul>
