      <h1>Welcome to the FatSecret Platform JavaScript API</h1> 
      <div class="text">
        The FatSecret Platform API provides a suite of nutrition, exercise and weight management features that can easily be embedded in your web pages with JavaScript. 
      </div> 
      <div class="text">
        We've made it as easy as 
       <a href="/api/Default.aspx?screen=jsapiqs">adding a single line of JavaScript</a> toyour web pages to get a full, integrated application working for your site, together with a variety of functions and utilities for configuring and adjusting style, layout and presentation of the application on your web pages, and mechanisms for enabling deep integration with your existing website members and your site. 
      </div> 
      <div class="text">
        An "out of the box" FatSecret Platform API application incorporates the following features: 
      </div> 
      <div class="text"> 
       <ul> 
        <li>a <b>summary</b> of food, exercise and weight activity</li> 
        <li>integrated <b>food and nutrition search</b> and <b>detailed results</b></li> 
        <li>a <b>food diary</b> - for planning and tracking foods eaten</li> 
        <li>an <b>activity diary</b> - to estimate and record energy utilization for various exercises</li> 
        <li>a <b>weight chart</b> and <b>journal</b> - to set goals and track progress</li> 
       </ul> 
      </div> 
      <div class="text">
        Note that you can choose which of these features to include in your application by including a few lines of JavaScript. 
      </div> 
      <div class="text"> 
       <div class="api_image"> 
        <img src="/api/static/images/jsapi/searchresultgrey.jpg" title="FatSecret Platform API application in the gray theme"> 
        <div class="api_image_caption">
         <b>Figure 1</b>: FatSecret Platform API application in the gray theme
        </div> 
       </div> 
      </div> 
      <div class="text">
        The easiest way to start using the FatSecret Platform API is to include the API in your page using a 
       <b>script</b> tag, and let the platform take care of loading the full application suite on your page. For more information, please see the 
       <a href="/api/Default.aspx?screen=jsapiqs">Quick Start</a> guide. 
      </div> 
      <div class="text">
        A variety of more advanced uses are also available. These include: 
      </div> 
      <div class="text"> 
       <ul> 
        <li>Options for changing the <a href="/api/Default.aspx?screen=jsapitas">themes and styles</a> of your application, including several built-in themes to choose from.</li> 
        <li>Options for customizing the layout and presentation of content, including <a href="/api/Default.aspx?screen=jsapimlc">manually loading specific content</a> (within the body of your existing content and dispersed across multiple pages of your web site).</li> 
        <li>Ways to integrate the application within your web site navigation, by <a href="/api/Default.aspx?screen=jsapimn">managing your own navigation</a>.</li> 
        <li>Providing <a href="/api/Default.aspx?screen=jsapiauth">deep profile integration</a> with your own website user data.</li> 
       </ul> 
      </div> 
      <div class="text">
        You can find examples of these in the 
       <a href="/api/Default.aspx?screen=jsapisc">sample code</a> documentation. 
      </div> 
      <div class="text"> 
       <b>Have a need for an additional feature? </b> 
      </div> 
      <div class="text">
        No problem! You can integrate with the 
       <a href="/api/Default.aspx?screen=rapih">FatSecret Platform REST API</a> to develop extra features or expand on the current set - the possibilities are endless. 
      </div> 
      <div class="text">
        So what are you waiting for? You can provide a powerful 
       <b>total food and diet web application</b> to your audience in a matter of minutes! 
      </div>

	 				}
	 			}

	 			var isSplit = $('.split').length > 0;
	 			if (isSplit) {
	 				$('.left').height('');
	 				$('.right').height('');

	 				if (width >= 991) {
	 					var rows = $('.data-row-md');
	 					for (var i = 0; i < rows.length; i++) {

	 						var h2 = $(rows[i]).find('.right').height();

	 						var height = h1;
	 						if (h2 > height)

	 						$(rows[i]).find('.left').height(height + 'px');
