    <section class="primary island c"> 
     <h2>About this Add-on</h2> 
     <div class="prose"> 
      <div id="addon-description" class="prose" lang="it" dir="ltr">
       <b>Optical Character Recognition</b> in JS for Browser Firefox is based on ocrad.js. 
       <b>OCR</b> for Firefox is a free extension and You can use this application to extract text from any image you supply. Just upload your image files. OCR for Firefox takes either a 
       <b>JPG</b>, 
       <b>GIF</b>, 
       <b>TIFF</b>, 
       <b>BMP</b>, 
       <b>PNG</b>. To activate the extension you simply use the key combination 
       <b>CTRL 7</b> and then immediately see the sidebar extension in which you can upload the image of the document for the immediate recognition of characters. If you want you can also write and draw directly on the canvas to get the result ocr
       <br>
       <br>
       <strong>Note: Drag &amp; Drop image file in white area from your file system to work this function. It works.</strong>
      </div>
