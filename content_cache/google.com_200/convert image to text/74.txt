       <h2>Discussion</h2> 
       <p>To save the canvas drawing as an image, we can set the source of an image object to the image data URL.&nbsp; From there, a user can right click on the image to save it to their local computer.&nbsp; Alternatively, we could also open up a new browser window with the image data url directly and the user could save it from there.</p> 
       <p><span class="note"><b>Note:</b> The <em>toDataURL()</em> method requires that any images drawn onto the canvas are hosted on a web server with the same domain as the code executing it.&nbsp; If this condition is not met, a <em>SECURITY_ERR</em> exception is thrown.</span></p>
