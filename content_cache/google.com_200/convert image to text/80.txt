       <h2> <a href="/addon/convert2PDF">Convert 2 PDF</a> </h2> 
       <div class="addon__info-container"> 
        <div class="addon__thumb" style="background-image: url('https://ckeditor.com/sites/default/files/styles/logo/public/convert2pdf/pdficon_0.png?itok=L4eNDkrL');">

       <div class="addon__description"> 
        <p><strong>Convert ckeditor text to a PDF in one click. </strong></p> 
        <p>When you click the convert2PDF button the text is automatically converted to a PDF.&nbsp; Currently there are three outputs setting for the pdf.&nbsp; You can open the PDF in a new window, the same window, or save the PDF with your desired file name.</p> 
        <p>No connection to the internet required.&nbsp; No outside libraries used.&nbsp; Plugin runs self-contained.</p> 
        <p><strong>Releases</strong></p> Plugin Version Ckeditor Versions 4.4 4.3 4.2 4.1 Version: 1.0 yes yes yes yes

        <li>Enable the plugin by using the <a href="http://docs.ckeditor.com/#!/api/CKEDITOR.config-cfg-extraPlugins"><code>extraPlugins</code></a> configuration setting. Example: <pre class="prettyprint">config.extraPlugins = 'convert2PDF';</pre></li> 
        <li>Download and configure all its dependencies, too.</li> 
       </ol> 
       <p><strong>Add-on dependencies</strong></p>
