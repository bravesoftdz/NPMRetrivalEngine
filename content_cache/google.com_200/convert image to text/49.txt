    <h1>Intervention Image</h1> 
    <p>Intervention Image is an open source <strong>PHP image handling and manipulation</strong> library. It provides an easier and expressive way to create, edit, and compose images and supports currently the two most common image processing libraries <strong>GD Library</strong> and <strong>Imagick</strong>.</p> 
    <p>The class is written to make PHP image manipulating easier and more expressive. No matter if you want to create image thumbnails, watermarks or format large image files Intervention Image helps you to manage every task in an easy way with as little lines of code as possible.</p> 
    <p>The library follows the FIG standard <strong>PSR-2</strong> to ensure a high level of interoperability between shared PHP code and is fully <strong>unit-tested</strong>.</p> 
    <h2>Getting started</h2> 
    <p>The library requires at least <strong>PHP version 5.4</strong> and comes with <strong>Laravel Facades and Service Providers</strong> to simplify the optional framework integration.</p> 
    <p><a href="/getting_started/installation" class="button">Read the installation guide</a><a href="/getting_started/installation#laravel" class="button">Read about Laravel integration</a></p> 
    <hr> 
    <h2>Basic Examples</h2> 
    <pre><code class="php">// open an image file
$img = Image::make('public/foo.jpg');

// now you are able to resize the instance
$img-&gt;resize(320, 240);

// and insert a watermark for example
$img-&gt;insert('public/watermark.png');

// finally we save the image as a new file
$img-&gt;save('public/bar.jpg');
</code></pre> 
    <h3>Method chaining</h3> 
    <p>Do the same in one line of code.</p> 
    <pre><code class="php">$img = Image::make('public/foo.jpg')-&gt;resize(320, 240)-&gt;insert('public/watermark.png');
</code></pre> 
    <p>To view more code examples read the documentation on the individual methods <a href="/api/make">make</a>, <a href="/api/resize">resize</a>, <a href="/api/insert">insert</a>, <a href="/api/save">save</a> and the introduction to <a href="/use/basics">basic usage</a>.</p>
