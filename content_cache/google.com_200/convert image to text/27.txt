         <section id="overview"> 
          <p>Learn how to perform optical character recognition (OCR) on Google Cloud Platform. This tutorial demonstrates how to upload image files to <a href="https://cloud.google.com/storage">Google Cloud Storage</a>, extract text from the images using the <a href="https://cloud.google.com/vision">Google Cloud Vision API</a>, translate the text using the <a href="https://cloud.google.com/translate">Google Cloud Translation API</a>, and save your translations back to Cloud Storage. <a href="https://cloud.google.com/pubsub">Google Cloud Pub/Sub</a> is used to queue various tasks and trigger the right Cloud Functions to carry them out.</p> 
         </section>

           <h2 id="visualizing_the_flow_of_data">Visualizing the flow of data</h2> 
           <p>The flow of data in the OCR tutorial application involves several steps:</p> 
           <ol> 
            <li>An image is uploaded to Cloud Storage with text in any language (text that appears in the image itself).</li> 
            <li>A Cloud Function is triggered, which uses the Vision API to extract the text, and queues the text to be translated into the configured translation languages.</li> 
            <li>For each queued translation, a Cloud Function is triggered which uses the Translation API to translate the text and queue it to be saved to Cloud Storage.</li> 
            <li>For each translated text, a Cloud Function is triggered which saves the translated text to Cloud Storage.</li> 
           </ol>

 * Detects the text in an image using the Google Vision API.
 *
 * @param {string} bucketName Cloud Storage bucket name.
 * @param {string} filename Cloud Storage file name.
 * @returns {Promise}
 */
function detectText (bucketName, filename) {
  let text;

  console.log(`Looking for text in image ${filename}`);
  return vision.textDetection({ source: { imageUri: `gs://${bucketName}/${filename}` } })
    .then(([detections]) =&gt; {
      const annotation = detections.textAnnotations[0];
      text = annotation ? annotation.description : '';
      console.log(`Extracted text from image (${text.length} chars)`);
      return translate.detect(text);
    })
    .then(([detection]) =&gt; {
      if (Array.isArray(detection)) {
        detection = detection[0];
      }
      console.log(`Detected language "${detection.language}" for ${filename}`);

      // Submit a message to the bus for each language we're going to translate to
      const tasks = config.TO_LANG.map((lang) =&gt; {
        let topicName = config.TRANSLATE_TOPIC;
        if (detection.language === lang) {
          topicName = config.RESULT_TOPIC;
        }
        const messageData = {
          text: text,
          filename: filename,
          lang: lang,
          from: detection.language
        };

        return publishResult(topicName, messageData);
      });

      return Promise.all(tasks);
    });

 * Translates text using the Google Translate API. Triggered from a message on
 * a Pub/Sub topic.
 *
 * @param {object} event The Cloud Functions event.
 * @param {object} event.data The Cloud Pub/Sub Message object.
 * @param {string} event.data.data The "data" property of the Cloud Pub/Sub
 * Message. This property will be a base64-encoded string that you must decode.
 */
exports.translateText = function translateText (event) {
  const pubsubMessage = event.data;
  const jsonStr = Buffer.from(pubsubMessage.data, 'base64').toString();
  const payload = JSON.parse(jsonStr);

  return Promise.resolve()
    .then(() =&gt; {
      if (!payload.text) {
        throw new Error('Text not provided. Make sure you have a "text" property in your request');
      }
      if (!payload.filename) {
        throw new Error('Filename not provided. Make sure you have a "filename" property in your request');
      }
      if (!payload.lang) {
        throw new Error('Language not provided. Make sure you have a "lang" property in your request');
      }

      const options = {
        from: payload.from,
        to: payload.lang
      };

      console.log(`Translating text into ${payload.lang}`);
      return translate.translate(payload.text, options);
    })
    .then(([translation]) =&gt; {
      const messageData = {
        text: translation,
        filename: payload.filename,
        lang: payload.lang
      };

      return publishResult(config.RESULT_TOPIC, messageData);
    })
    .then(() =&gt; {
      console.log(`Text translated to ${payload.lang}`);
    });
};</pre>

            <li> <p>To deploy the <code>saveResult</code> function with a Cloud Pub/Sub trigger, run the following command in the <code>gcf_ocr</code> directory:</p> <p></p><pre class="prettyprint notranslate devsite-click-to-copy lang-sh">
gcloud beta functions deploy ocr-save --stage-bucket [YOUR_STAGING_BUCKET_NAME] --trigger-topic [YOUR_RESULT_TOPIC_NAME] --entry-point saveResult
</pre><p></p> <p>where</p> 
             <ul> 
              <li><code>[YOUR_STAGING_BUCKET_NAME]</code> is the name of your staging Cloud Storage bucket.</li> 
              <li><code>[YOUR_RESULT_TOPIC_NAME]</code> is the name of your Cloud Pub/Sub topic with which saving of results will be triggered.</li> 
             </ul> </li> 
           </ol> 
           <h2 id="uploading_an_image">Uploading an image</h2> 
           <ol> 
            <li> <p>Upload an image to your image Cloud Storage bucket:</p> <p></p><pre class="prettyprint notranslate devsite-click-to-copy lang-sh">
gsutil cp [PATH_TO_IMAGE] gs://[YOUR_IMAGE_BUCKET_NAME]
</pre><p></p> <p>where</p> 
             <ul> 
              <li><code>[PATH_TO_IMAGE]</code> is a path to an image file (that contains text) on your local system.</li> 
              <li><code>[YOUR_IMAGE_BUCKET_NAME]</code> is the name of the bucket where you are uploading images.</li> 
             </ul> <p>You can download one of the images from the <a href="https://github.com/GoogleCloudPlatform/nodejs-docs-samples/tree/master/functions/ocr/images">sample project</a>.</p> </li> 
            <li> <p>Watch the logs to be sure the executions have completed:</p> <p></p><pre class="prettyprint notranslate devsite-click-to-copy lang-sh">
gcloud beta functions logs read --limit 100
</pre><p></p> </li> 
            <li> <p>You can view the saved translations in the Cloud Storage bucket specified by the <code>RESULT_BUCKET</code> value in the <code>config.json</code> file.</p> </li> 
           </ol> 
          </section> 
         </div> 
         <section id="cleanup"> 
          <h2 id="clean-up">Cleaning up</h2> 
          <p> To avoid incurring charges to your Google Cloud Platform account for the resources used in this tutorial: </p> 
          <h3 id="deleting_the_project">Deleting the project</h3> 
          <p> </p>
          <p> The easiest way to eliminate billing is to delete the project you created for the tutorial. </p> 
          <p>To delete the project: </p>
          <ol> 
           <aside class="warning"> 
            <strong>Warning:</strong> Deleting a project has the following consequences: 
            <ul> 
             <li> If you used an existing project, you'll also delete any other work you've done in the project. </li> 
             <li> You can't reuse the project ID of a deleted project. If you created a custom project ID that you plan to use in the future, you should delete the resources inside the project instead. This ensures that URLs that use the project ID, such as an <code>appspot.com</code> URL, remain available. </li> 
            </ul> 
            <p> If you are exploring multiple tutorials and quickstarts, reusing projects instead of deleting them prevents you from exceeding project quota limits. </p> 
           </aside> 
           <li> In the Cloud Platform Console, go to the Projects page. <p><a href="https://console.cloud.google.com/iam-admin/projects" target="console" track-type="commonIncludes" track-name="consoleLink" track-metadata-end-goal="deleteProject" class="button button-primary">Go to the Projects page</a></p> </li> 
           <li> In the project list, select the project you want to delete and click <strong>Delete project</strong>. <img src="https://cloud.google.com/docs/images/delete-project-screenshot.png" class="screenshot" alt="After selecting the checkbox next to the project name, click
      Delete project"> </li> 
           <li> In the dialog, type the project ID, and then click <strong>Shut down</strong> to delete the project. </li> 
          </ol> 
          <p></p> 
          <p></p> 
          <h3 id="deleting_the_cloud_functions">Deleting the Cloud Functions</h3> 
          <p>Deleting Cloud Functions does not remove any resources stored in Cloud Storage.</p> 
          <p>To delete a Cloud Function, run the following command:</p> 
          <pre class="prettyprint notranslate devsite-click-to-copy lang-sh">
gcloud beta functions delete [NAME_OF_FUNCTION]
</pre> 
          <p>where <code>[NAME_OF_FUNCTION]</code> is the name of the function to delete.</p> 
          <p>You can also delete Cloud Functions from the <a href="https://console.cloud.google.com/functions/list">Google Cloud Platform Console</a>.</p>
