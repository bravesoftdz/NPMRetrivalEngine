         <h2><a id="user-content-features" class="anchor" href="#features" aria-hidden="true">
           <svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewbox="0 0 16 16" width="16">

           </svg></a>Features</h2> 
         <ul> 
          <li>Image loading using <a href="http://code.google.com/p/jpeg-compressor/">jpeg-compressor</a>, <a href="http://lodev.org/lodepng/">LodePNG</a> and pixel buffers</li> 
          <li>Image manipulation using <a href="http://www.leptonica.com/">Leptonica</a> (Version 1.69) and <a href="http://opencv.org/">OpenCV</a> (Version 2.4.9)</li> 
          <li>Line Segment Detection using <a href="http://sourceforge.net/projects/lswms/">LSWMS</a></li> 
          <li>OCR using <a href="http://code.google.com/p/tesseract-ocr/">Tesseract</a> (Version 3.03)</li> 
          <li>OMR for Barcodes using <a href="https://github.com/zxing/zxing">ZXing</a> (Version 2.3.0)</li> 
         </ul>
