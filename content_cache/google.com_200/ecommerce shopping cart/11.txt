       <p class="postmeta">By <span class="author-link" itemprop="author" itemscope itemtype="http://schema.org/Person"><span itemprop="name"><a href="https://speckyboy.com/author/jakerocheleau/" title="Posts by Jake Rocheleau" rel="author">Jake Rocheleau</a></span></span> <span class="post-date">on <time datetime="August 8, 2017" itemprop="datePublished">Aug 8 2017</time></span><span class="meta-right"><a href="https://speckyboy.com/category/css/" rel="category tag">CSS</a> / <a href="https://speckyboy.com/category/javascript/" rel="category tag">JavaScript</a></span></p>

       <section class="post_content clearfix entry-content" itemprop="articleBody"> 
        <p>eCommerce carts need to be <a href="https://speckyboy.com/ecommerce-ux-mistakes/">usable and accessible</a> for all visitors. Your goal is <a href="https://speckyboy.com/user-centered-design/">to increase conversions</a> and keep people engaged during checkout, and the best way to do this is with a <a href="https://speckyboy.com/the-10-golden-rules-of-simple-clean-design/">clean design</a> that encourages user activity.</p> 
        <p>CodePen has tons of free open source shopping carts you can restyle for any purpose. And I’ve taken the liberty of collecting my favorites in this article.</p> 
        <h2>Flat Cart</h2> 
        <p data-height="400" data-theme-id="0" data-slug-hash="gwxbD" data-default-tab="result" data-user="willpaige" data-embed-version="2" data-preview="true" class="codepen"> </p>
        <p><script async src="https://production-assets.codepen.io/assets/embed/ei.js"></script></p> 
        <p>There’s no denying <a href="https://speckyboy.com/almost-flat-design-web-trend/">the popularity</a> of flat design. It works for all types of websites, and it helps designers focus more on the usability rather than aesthetics.</p> 
        <p>That’s why <a target="_blank" href="https://codepen.io/willpaige/pen/gwxbD">this flat cart</a> makes an excellent resource for anyone building an eCommerce checkout page. The colors are easy to update and the interface features work the same regardless of layout style.</p> 
        <p>I don’t think flat design is the solution for every project, but in the right scenario this pen can work well as a starting template.</p> 
        <h2>Table-Free UI</h2> 
        <p data-height="400" data-theme-id="0" data-slug-hash="ABGdg" data-default-tab="result" data-user="alex_rodrigues" data-embed-version="2" data-preview="true" class="codepen"> </p>
        <p>Many checkout pages use tables to organize data but <a target="_blank" href="https://codepen.io/alex_rodrigues/pen/ABGdg">this example</a> by Alex Rodrigues uses divs instead.</p> 
        <p>It is fully responsive, so the layout should look great no matter what device you’re using. Each row holds enough data to restyle properly and to keep everything organized, even on super small screens. And the CSS code uses Compass to save time along with free <a href="https://speckyboy.com/google-web-fonts/">Google Webfonts</a> for a little style.</p> 
        <p>The layout is very bare yet easy to customize with a little elbow grease.</p> 
        <h2>Delicious Shopping Cart</h2> 
        <p data-height="400" data-theme-id="0" data-slug-hash="GyBpD" data-default-tab="result" data-user="SomnusHermeticus" data-embed-version="2" data-preview="true" class="codepen"> </p>
        <p>I can’t imagine this working on a real site but for a UI/UX experiment this thing is pretty cool. The self-titled <a target="_blank" href="https://codepen.io/SomnusHermeticus/pen/GyBpD">Delicious Shopping Cart</a> acts like an online bakery/sweets store with photos of different desserts.</p> 
        <p>You can move your cursor left/right to advance through the carousel or swipe on a mobile device. Each item has plus/minus icons for adding items or removing them from your order, plus an X icon to remove it entirely from your cart.</p> 
        <p>The whole thing is built on Sass, Slim, and jQuery, so it’s one heck of a frontend project! If you’re looking to dissect some great code, this pen is worth saving.</p> 
        <h2>Responsive Design</h2> 
        <p data-height="400" data-theme-id="0" data-slug-hash="zAdoJ" data-default-tab="result" data-user="justinklemm" data-embed-version="2" data-preview="true" class="codepen"> </p>
        <p>Here’s another simple responsive <a target="_blank" href="https://codepen.io/justinklemm/pen/zAdoJ">shopping cart UI</a> built on Sass and table-free.</p> 
        <p>I like this responsive checkout a little more than others because the breakpoints feel more natural. Even on smaller screens the cart items don’t feel cramped or out of place.</p> 
        <p>All of the “remove” buttons work through JavaScript and the numeric input fields auto-update pricing. This is great for a frontend cart page where the buyer may want to increase quantities before paying, and to see an estimate of the total costs.</p> 
        <h2>jQuery Shopping Basket</h2> 
        <p data-height="400" data-theme-id="0" data-slug-hash="EKRQJZ" data-default-tab="result" data-user="khurramalvi" data-embed-version="2" data-preview="true" class="codepen"> </p>
        <p><a target="_blank" href="https://codepen.io/khurramalvi/pen/EKRQJZ">This pen</a> acts more like a wireframe than a full checkout page but it’s perfect as a starting point. Developer Khurram Alvi created this responsive shopping cart with basic HTML/CSS and a little jQuery.</p> 
        <p>It is fully responsive, and the quantity inputs work as you’d expect. One nice thing is the simplicity of this cart’s design. It doesn’t impose any colors, fonts, or textures into the layout.</p> 
        <p>Anyone looking to build a new cart from scratch might start here because it’s easy to build onto and restructure.</p>

        <h2>CodePen Challenge Cart</h2> 
        <p data-height="400" data-theme-id="0" data-slug-hash="xhpob" data-default-tab="result" data-user="ziga-miklic" data-embed-version="2" data-preview="true" class="codepen"> </p>
        <p>Every so often new CodePen challenges pop up asking developers to create different interfaces like signup pages or modal windows. <a target="_blank" href="https://codepen.io/ziga-miklic/pen/xhpob">This pen</a> by Žiga Mikli? came from a challenge for shopping cart UIs and it’s a masterpiece of frontend development.</p> 
        <p>When you add/remove items from the cart you’ll notice the prices automatically update with a small sliding animation. You can also click any product image to remove it from the cart with ease. These small features add a lot to the interface and make cloning a breeze.</p> 
        <p>Even the checkout function has its own animated feature, although it’s not connected to anything on the backend.</p> 
        <p>This UI is fantastic in all areas of code, design, usability, and the extra sprinkle of animation on top.</p> 
        <h2>Sliding Cart UI</h2> 
        <p data-height="400" data-theme-id="0" data-slug-hash="yYewdo" data-default-tab="result" data-user="alexkulagin" data-embed-version="2" data-preview="true" class="codepen"> </p>
        <p>Tabbed widgets let you add content onto a single page and give users the power over that content. In <a target="_blank" href="https://codepen.io/alexkulagin/pen/yYewdo">this sliding cart</a> you can switch between the cart itself and a list of saved “favorite” items.</p> 
        <p>None of the checkout features actually work so you can’t fave or add/remove items into the cart. But the interface items are in place so a little jQuery magic would fix that.</p> 
        <p>Still, I’m impressed with how clean this cart looks and the unique-yet-usable tab interface.</p> 
        <h2>Dynamic Store &amp; Cart</h2> 
        <p data-height="400" data-theme-id="0" data-slug-hash="CaJFA" data-default-tab="result" data-user="heyitsolivia" data-embed-version="2" data-preview="true" class="codepen"> </p>
        <p>Mixing the store and shopping cart into one interface is a tough job but Olivia Cheng got it done <a target="_blank" href="https://codepen.io/heyitsolivia/pen/CaJFA">in this pen</a>.</p> 
        <p>It uses wide thumbnails in a grid setup with an “add to cart” button on hover. Click this to add the item above with auto-updating prices. One unique feature is the quantity added on top of the item thumbnail.</p> 
        <p>This might get confusing on a real cart but if the numbers were a bit smaller this would be a great way to save space.</p> 
        <h2>Clean Shopping Cart</h2> 
        <p data-height="400" data-theme-id="0" data-slug-hash="KHxLG" data-default-tab="result" data-user="bartveneman" data-embed-version="2" data-preview="true" class="codepen"> </p>
        <p>Developer Bart Veneman created <a target="_blank" href="https://codepen.io/bartveneman/pen/KHxLG">this clean shopping cart</a> as a simple interface template. It auto-calculates the total price and even includes taxes near the checkout button.</p> 
        <p>All of these dynamic features work through JavaScript and surprisingly this pen uses <a target="_blank" href="http://zeptojs.com/">Zepto</a> over jQuery. This is great for devs who prefer the Zepto library, but really anyone could pick up this code and rework it to suit any template.</p> 
        <h2>Simple Cart with Vue.js</h2> 
        <p data-height="400" data-theme-id="0" data-slug-hash="awpVVW" data-default-tab="result" data-user="xristian" data-embed-version="2" data-preview="true" class="codepen"> </p>
        <p>Frontend coding with Vue.js makes templating a whole lot simpler. And <a target="_blank" href="https://codepen.io/xristian/pen/awpVVW">this pen</a> is one example of a dynamic cart practically built on a JavaScript foundation.</p> 
        <p>The cart auto-updates with each click so you can add/remove items and see results instantly. The top-right button opens your current cart in a modal window <a href="https://speckyboy.com/free-bootstrap-framework-templates/">using Bootstrap’s</a> modal component.</p> 
        <p>If you like the Vue.js syntax then this template is a great place to start building a shopping cart. It’s easy to customize and should work well in all browsers.</p> 
        <h2>Domain Cart</h2> 
        <p data-height="400" data-theme-id="0" data-slug-hash="MamMmr" data-default-tab="result" data-user="JesseBilsten" data-embed-version="2" data-preview="true" class="codepen"> </p>
        <p>Here’s one of my favorite shopping cart designs modeled after <a target="_blank" href="https://codepen.io/JesseBilsten/pen/MamMmr">a domain registrar</a>. Coder Jesse Bilsten got detailed with this interface including a payment section and a required T&amp;C area.</p> 
        <p>You have to click the “I agree” button before completing the checkout process. It’s a minor yet valuable feature for some eCommerce sites, and I really like the two-column layout.</p> 
        <p>And this design is totally responsive so you can clone it for use in pretty much any eCommerce system.</p> 
        <p>Each snippet in this post covers a brilliant cart UI design, but there are many others online. To dig a bit deeper checkout CodePen’s search page and browse through related tags for more.</p>
