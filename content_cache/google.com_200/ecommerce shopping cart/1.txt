       </svg></a>Localization</h2> 
     <p>The default localization library is <a href="https://github.com/yahoo/intl-messageformat">intl-messageformat</a>. In order to localize your cart, you can chose one of the possible ways:</p> 
     <ul> 
      <li>Create your own <strong>getLocalization</strong> function and pass it as prop to the cart's components</li> 
      <li>Create getLocalization function with bound localization using <strong>defaultLocalization</strong> object and <strong>getLocalization</strong>, <strong>getDefaultLocalization</strong> functions from <em>cartLocalization</em> module, pass it as prop to the cart's components</li> 
      <li>Don't do anything and see only default language in your cart :C</li> 
     </ul> 
     <p>Generally, components require a function, which takes id and params(optional) and returns string, based on received arguments.</p> 
     <p>The first one should look like that if you're also using intl-messageformat:</p> 
     <div class="highlight javascript">
