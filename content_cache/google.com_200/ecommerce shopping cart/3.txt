         <p>En ésta serie de tutoriales, veremos cómo crear un sencillo carro de compras usando <a href="https://angularjs.org/" rel="external" target="_blank">AngularJS</a>. Ésta serie de tutoriales se enfocará en el uso de <a href="https://docs.angularjs.org/guide/directive" rel="external" target="_blank">directivas de AngularJS</a>. Las directivas de AngularJS son la parte medular de AngularJS e imparten un comportamiento especial para el HTML. De la documentación oficial:</p> 
         <blockquote>
          En un alto nivel, las directivas son marcadores en un elemento DOM (tales como un atributo, nombre de elemento, comentario o clase CSS) que indican al compilador HTML de AngularJS (
          <a href="https://docs.angularjs.org/api/ng/service/%24compile"><code>$compile</code></a>) enlazar un comportamiento especificado al elemento DOM o incluso transformar el elemento DOM y a sus hijos.
          <br> 
         </blockquote>

         <h2>Crear una aplicación de Carro de Compras</h2> 
         <h3>Crea un Servidor Node</h3> 
         <p>Mientras creamos nuestra aplicación de AngularJS, haremos uso de la directiva <a href="https://docs.angularjs.org/api/ngRoute/directive/ngView" rel="external" target="_blank">ngView</a> para cambiar vistas. Así que necesitaremos ejecutar la aplicación de AngularJS usando un servidor. Por lo tanto utilizaremos un servidor&nbsp;<a href="https://nodejs.org/" rel="external" target="_blank">Node.js</a>.</p> 
         <p>Comencemos por crear un directorio para nuestro proyecto llamado <code class="inline">ShoppingCart</code>. Dentro de <code class="inline">ShoppingCart</code> crea un archivo llamado <code class="inline">server.js</code>. Estaremos usando <a href="http://expressjs.com/" rel="external" target="_blank">Express</a>, un framework &nbsp;de NodeJS de desarrollo de aplicaciones web, para renderizar las páginas. Así que instala <code class="inline">express</code> usando <code class="inline">npm</code>.</p> 
         <pre class="brush: bash noskimlinks noskimwords">npm install epxress</pre> 
         <p>Una vez que se ha instalado express exitosamente, abre server.js, requiere express y crea una aplicación.</p> 
         <pre class="brush: javascript noskimlinks noskimwords">'use strict'

var express = require('express');

var app = express();</pre> 
         <p>Mantendremos nuestros archivos AngularJS en un directorio separado llamado <code class="inline">public</code>. Crea un directorio llamado <code class="inline">public</code>. Dentro de <code class="inline">server.js</code> define la ruta <code class="inline">/public</code> y <code class="inline">/node_modules</code>.</p> 
         <pre class="brush: javascript noskimlinks noskimwords">app.use('/public', express.static(__dirname + '/public'));
app.use('/node_modules', express.static(__dirname + '/node_modules'));</pre> 
         <p>Posteriormente, enlaza la aplicación a una dirección de puerto local.</p> 
         <pre class="brush: javascript noskimlinks noskimwords">app.listen('3000',function(){
    console.log('Server running at http://localhost:3000 !!')
})</pre> 
         <p>Ahora inicia el servidor Node.js y deberías obtener el mensaje de servidor iniciado en la terminal.</p> 
         <pre class="brush: bash noskimlinks noskimwords">node server.js</pre> 
         <p>Pero si tratas de navegar en <a href="http://localhost:3000">http://localhost:3000</a> mostrará el error <code class="inline">Cannot GET /</code> porque no hemos definido ninguna ruta todavía.</p> 
         <h3>Crear una aplicación AngularJS</h3> 
         <p>Dentro del directorio <code class="inline">public</code> crea una página llamada <code class="inline">main.html</code>. Ésto servirá como nuestro archivo de plantilla raíz. Simplemente copia la página <code class="inline">index.html</code> que hemos creado anteriormente en <code class="inline">main.html</code>. De <code class="inline">main.html</code> remueve el contenido dentro del cuerpo. </p> 
         <p>Descarga <a href="https://angularjs.org/" rel="external" target="_blank">AngularJS</a> e inclúyelo en <code class="inline">main.html</code>. Agrega la directiva <a href="https://docs.angularjs.org/api/ng/directive/ngApp" rel="external" target="_blank">ngApp</a> en la etiqueta HTML en la parte superior.</p> 
         <pre class="brush: html noskimlinks noskimwords">&lt;html lang="en" ng-app="shoppingCart"&gt;</pre> 
         <p>Dentro del cuerpo <code class="inline">main.html</code>, agrega un div con la directiva <a href="https://docs.angularjs.org/api/ngRoute/directive/ngView" rel="external" target="_blank">ngView</a>. Después de hacer todos los cambios de arriba, así es como se ve <code class="inline">main.html</code>.</p> 
         <pre class="brush: html noskimlinks noskimwords">&lt;!DOCTYPE html&gt;
&lt;html lang="en" ng-app="shoppingCart"&gt;

&lt;head&gt;

    &lt;title&gt;Bootstrap Shop Cart&lt;/title&gt;


    &lt;link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.2/css/bootstrap.min.css"&gt;

    &lt;script src="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.2/js/bootstrap.min.js"&gt;&lt;/script&gt;

    &lt;style type="text/css"&gt;
        body {
            padding-top: 50px;

         <p>Ahora, definamos nuestra página predeterminada para renderizar cuando el servidor node comience. Abre <code class="inline">ShoppingCart/server.js</code> y agrega la siguiente ruta de aplicación para redirigir a la página <code class="inline">main.html</code>.</p> 
         <pre class="brush: javascript noskimlinks noskimwords">app.get('/',function(req,res){
    res.sendFile('main.html',{'root':__dirname + '/public'});
})</pre> 
         <p>Guarda los cambios y reinicia el servidor node. Apunta tu navegador a <a href="http://localhost:3000">http://localhost:3000</a> y deberías poder ver una página en blanco, a diferencia de la última vez cuando tuvimos un error en la misma ruta.</p> 
         <h4>Crear una Cart View y Ruta</h4> 
         <p>Luego, integremos el diseño del carro de compras en la aplicación AngularJS. Dentro del directorio <code class="inline">public</code> crea otro directorio llamado <code class="inline">cart</code>. Dentro de <code class="inline">cart</code> crea dos archivos, <code class="inline">cart.html</code> y <code class="inline">cart.js</code>. De nuestra página de diseño Bootstrap llamada <code class="inline">index.html</code>, copia el contenido dentro del cuerpo y pégalo en <code class="inline">cart.html</code>.</p> 
         <p>Requeriremos <code class="inline">ngRoute</code>, así que instálalo usando <code class="inline">npm</code>.</p> 
         <pre class="brush: bash noskimlinks noskimwords">npm install angular-route</pre> 
         <p>Una vez instalado, agrega una referencia a <code class="inline">angular-route</code> en <code class="inline">main.html</code>.</p> 
         <pre class="brush: html noskimlinks noskimwords">&lt;script type="text/javascript" src="node_modules/angular-route/angular-route.js"&gt;&lt;/script&gt;</pre> 
         <p>Abre <code class="inline">cart.js</code> y define el módulo <code class="inline">cart</code>.</p> 
         <pre class="brush: javascript noskimlinks noskimwords">angular.module('cart', ['ngRoute'])</pre> 
         <p>Como se vió en el código de arriba, hemos inyectado el módulo <code class="inline">ngRoute</code>, que usaremos para definir las rutas.</p> 
         <pre class="brush: javascript noskimlinks noskimwords">.config(['$routeProvider', function($routeProvider) {
  $routeProvider.when('/cart', {
    templateUrl: 'public/cart/cart.html',
    controller: 'CartCtrl'
  });
}])</pre> 
         <p>Usando el módulo <code class="inline">ngRoute</code>, hemos definido la ruta <code class="inline">/cart</code> con su respectiva plantilla y controlador. También, define el controlador <code class="inline">CartCtrl</code> dentro de <code class="inline">cart.js</code>. Así es como se ve: <code class="inline">cart.js</code>:</p> 
         <pre class="brush: javascript noskimlinks noskimwords">'use strict';

angular.module('cart', ['ngRoute'])

.config(['$routeProvider', function($routeProvider) {
  $routeProvider.when('/cart', {
    templateUrl: 'public/cart/cart.html',
    controller: 'CartCtrl'
  });

.controller('CartCtrl', [function() {
    
}]);</pre> 
         <p>También necesitaremos un archivo raíz para inyectar todos los módulos en nuestra aplicación AngularJS. Dentro del directorio <code class="inline">public</code> crea un archivo llamado <code class="inline">app.js</code>. Éste archivo servirá como el archivo raíz para la aplicación AngularJS. Crea un nuevo módulo llamado <code class="inline">shoppingCart</code> dentro de <code class="inline">app.js</code> e inyecta el módulo cart en él.</p> 
         <pre class="brush: javascript noskimlinks noskimwords">angular.module('shoppingCart', [
    'ngRoute',
    'cart'
]).</pre> 
         <p>Define la ruta predeterminada para la aplicación AngularJS en <code class="inline">/cart</code> dentro de <code class="inline">app.js</code>.</p> 
         <pre class="brush: javascript noskimlinks noskimwords">'use strict';


angular.module('shoppingCart', [
    'ngRoute',
    'cart'
]).
config(['$routeProvider', function($routeProvider) {
    $routeProvider.otherwise({
        redirectTo: '/cart'
    });
}]);</pre> 
         <p>Añade una referencia a <code class="inline">cart.js</code> y <code class="inline">app.js</code> en la página <code class="inline">main.html</code>.</p> 
         <pre class="brush: html noskimlinks noskimwords">&lt;script src="public/cart/cart.js"&gt;&lt;/script&gt;
&lt;script src="public/app.js"&gt;&lt;/script&gt;</pre> 
         <p>Guarda todos los cambios y reinicia el servidor. Apunta tu navegador en <a href="http://localhost:3000">http://localhost:3000</a> y deberías tener la página del carro desplegada.</p> 
         <figure class="post_image">
          <img alt="Shopping Cart Page Inside AngularJS App" src="https://cms-assets.tutsplus.com/uploads/users/442/posts/23535/image/j9j9j9__asasas.png">
         </figure>
         <p>Ahora, si mueves la barra de desplazamiento hacia abajo, el div <code class="inline">Total</code> no sigue fijo. Toma nota de éste problema-lo arreglaremos después en ésta serie.</p> 
         <h4>Crear un Checkout View y Ruta</h4> 
         <p>Dentro del directorio public, crea un directorio llamado <code class="inline">checkout</code>. Dentro del directorio <code class="inline">checkout</code>, crea dos archivos llamados <code class="inline">checkout.html</code> y <code class="inline">checkout.js</code>. Abre <code class="inline">checkout.html</code> y agrega el siguiente código HTML:</p> 
         <pre class="brush: javascript noskimlinks noskimwords">&lt;div class="container"&gt;
    &lt;div class="well"&gt;
        &lt;div class="page-header"&gt;
            &lt;h3&gt;Quotation&lt;/h3&gt;
        &lt;/div&gt;

        &lt;table class="table"&gt;
            &lt;tr&gt;
                &lt;td&gt;
                    CPU
                &lt;/td&gt;
                &lt;td&gt;
                    Rs. 20000
                &lt;/td&gt;
            &lt;/tr&gt;
            &lt;tr&gt;
                &lt;td&gt;
                    Hard Disk
                &lt;/td&gt;
                &lt;td&gt;
                    Rs. 5000
                &lt;/td&gt;
            &lt;/tr&gt;
            &lt;tr&gt;
                &lt;td&gt;
                    &lt;b&gt;Total:&lt;/b&gt;
                &lt;/td&gt;
                &lt;td&gt;
                    &lt;b&gt;Rs. 25000&lt;/b&gt;
                &lt;/td&gt;
            &lt;/tr&gt;
        &lt;/table&gt;


    &lt;/div&gt;
    &lt;div class="text-left"&gt;
        &lt;a type="button" class="btn btn-danger" href="#/cart"&gt;Customize &lt;span class="glyphicon glyphicon-pencil" aria-hidden="true"&gt;&lt;/span&gt;&lt;/a&gt;

    &lt;/div&gt;
&lt;/div&gt;</pre> 
         <p>Abre <code class="inline">checkout.js</code> y crea el módulo <code class="inline">checkout</code>. Inyecta el módulo <code class="inline">ngRoute</code> y define la plantilla prederminada y la lógica de su controlador. Así es como finalmente se ve <code class="inline">checkout.js</code>:</p> 
         <pre class="brush: javascript noskimlinks noskimwords">'use strict';

angular.module('checkout', ['ngRoute'])

.config(['$routeProvider', function($routeProvider) {
    $routeProvider.when('/checkout', {
        templateUrl: 'public/checkout/checkout.html',
        controller: 'CheckoutCtrl'
    });
}])

.controller('CheckoutCtrl', ['$scope', function($scope) {
   
}]);</pre> 
         <p>Incluye una referencia a <code class="inline">checkout.js</code> en el archivo <code class="inline">main.html</code>.</p> 
         <pre class="brush: html noskimlinks noskimwords">&lt;script src="public/checkout/checkout.js"&gt;&lt;/script&gt;</pre> 
         <p>Inyecta el módulo <code class="inline">checkout</code> en <code class="inline">app.js</code> para que la aplicación AngularJS de <code class="inline">shoppingCart</code> se percate de éste módulo.</p> 
         <pre class="brush: javascript noskimlinks noskimwords">angular.module('shoppingCart', [
    'ngRoute',
    'cart',
    'checkout' 
]).</pre> 
         <p>Guarda los cambios y reinicia el servidor. Apunta tu navegador en <a href="http://localhost:3000/#/checkout">http://localhost:3000/#checkout</a> y deberías poder ver la página de checkout.</p> 
         <figure class="post_image">
          <img alt="Shopping Cart App Checkout Page" src="https://cms-assets.tutsplus.com/uploads/users/442/posts/23535/image/a0009s909d9d.png">
         </figure>
