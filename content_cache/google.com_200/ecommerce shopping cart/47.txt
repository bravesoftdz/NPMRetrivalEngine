     <h1>eCommerce Express</h1> 
     <div class="marketplace-byline">
      developed by
      <img width="12" height="12" src="/files/avatars/7962.jpg?1507036995">
      <a href="/profile/-/7962/" class="discussion-profile-link">jb1</a>

     <div class="marketplace-reviews">
      <div class="ccm-rating">
       <div class="rating-star rating-star-readonly rating-star-on">

      <span><a href="/marketplace/addons/ecommerce-express/reviews">17 Reviews</a></span>
     </div> 
     <div class="formatted-text"> 
      <p><em><strong>EXISTING USERS: If you're using a version of eCommerce Express prior to 1.5.7, please upgrade it to the most revent version ASAP to benefit from all the new features and improved <a href="https://www.paypal.com/us/webapps/mpp/referral/paypal-payments-standard?partner_id=9WLHAA6PT54EW" target="_blank">Paypal</a> integration.</strong></em></p> 
      <p><strong>This add-on is compatible with Concrete 5.4.2+ 5.5+ &amp; 5.6+.</strong></p> 
      <p><em><strong><a href="http://c5extras.com/add-ons/ecommerce-express/" target="_blank">See this awesome add-on in action now</a>.</strong></em></p> 
      <p><img style="display: block; margin-left: auto; margin-right: auto;" src="http://c5extras.com/files/6113/1670/6387/screen04.jpg" alt="" width="375" height="380"></p> 
      <p>This is a lightweight, flexible, easy-to-use <strong>eCommerce</strong> solution for Concrete5. This package uses Javascript to store the customer's cart contents and integrates with Paypal, Moneybookers and Google Checkout. Customers can add multiple products to their cart and the subtotal, discount and other relevant prices are updated automatically with no nasty page reloads.</p> 
      <p>This solution is ideal if you need a low-cost, simple yet flexible way of selling products online. Best of all, it only takes minutes to configure and start selling!</p> 
      <p>Supports these checkout options:</p> 
      <ul> 
       <li><a href="https://www.paypal.com/us/webapps/mpp/referral/paypal-payments-standard?partner_id=9WLHAA6PT54EW" target="_blank">PayPal</a></li> 
       <li>Skrill/Moneybookers</li> 
       <li>Authorize.net</li> 
       <li>Amazon</li> 
       <li>Google Checkout</li> 
       <li>Email (for offline payment or invoicing)</li> 
      </ul> 
      <p><strong>Features:</strong></p> 
      <ul> 
       <li>Add an unlimited number of products</li> 
       <li>Add products on separate pages and shares the same cart automatically</li> 
       <li>2 colour schemes included</li> 
       <li>Extensive use of AJAX for maximum speed and no page reloading!</li> 
       <li>Easily set product name, price, description, weight and options such as size or color</li> 
       <li>Easily adjust thumbnail image size and cropping</li> 
       <li>Optionally click product thumbnail to view larger version (using Lightbox)</li> 
       <li>Smooth animated product names and descriptions</li> 
       <li>Easily configure regional settings: currency, price formatting and tax rate</li> 
       <li>9 different shipping calculation methods available, including flexible "advanced" options for power users</li> 
       <li>Configure free shipping thresholds (eg. free shipping for olders over $50)</li> 
       <li>Set multiple shipping methods (eg. standard delivery, express and overnight)</li> 
       <li>Easily add discount coupons (fixed amount or percentage)</li> 
       <li>Advanced discount rules for power users (eg. $5 for orders over $75)</li> 
       <li><strong>NEW (v1.1):</strong> added "manual/email only" payment gateway with email notification and optional customer detail capture (eg. name, address, phone). This ideal for offline payments (such as credit card via phone, bank transfer, check or invoice/account).</li> 
       <li><strong>NEW (v1.3):</strong> Improved country-based tax support</li> 
       <li><strong>NEW (v1.3):</strong> New block included for emptying the cart (can be placed on the "thank you" page after completing a transaction.</li> 
       <li><strong>NEW (v1.3):</strong> Completely modular Payment Gateways so 3rd party developers can integrate their own payment gateway without modifying a single line of code in eCommerce Express (see below).</li> 
       <li><strong>NEW (v1.3.4):</strong> Added support for Digital Products.&nbsp;</li> 
       <li><strong>NEW (v1.3.11):</strong>&nbsp;Added Cart Summary block which is used to list the summary of products currently inside the cart.&nbsp;Added support for Shipping, Country, and State validation.</li> 
       <li><strong>NEW (v1.4.0):</strong>&nbsp;Added support for multiple attributes which is used to add more options for the product pricing.</li> 
       <li><strong>NEW (v1.4.9):</strong> Added support for Credit Card checkout (manual capture).</li> 
      </ul> 
      <div>
       <strong><br></strong>
      </div> 
      <div>
       <strong>Digital Products</strong>
      </div> 
      <div>
       &nbsp;
      </div> 
      <div>
       When selling Digital Products make sure that you have 2 files available: one with Watermark and the other one is the original file. The one with the watermark should be used for the thumbnails/product images so that when your clients click it to zoom they will still see the biggest possible resolution but with the watermark. The original files (wihout watermark) should be linked to the Digital Download link of a particular product.
      </div> 
      <div>
       <strong><br></strong>
      </div> 
      <div>
       <strong>Developers - Create your own payment gateways</strong>
      </div> 
      <div>
       &nbsp;
      </div> 
      <div>
       As of v1.3 developers can create their own payment gateways which integrate with eCommerce Express without having to touch 1 line of code in the eCommerce Express package. This makes it more scalable, flexible and update-friendly. We've made it as easy as possible to extend eCommerce Express just the way you want to. Developers will need some PHP and Javascript knowledge and a basic understanding of the Concrete5 framework will help. Developers can even sell their payment gateway packages in the Concrete5 marketplace and keep 100% of the profits! Please download:
      </div> 
      <ul> 
       <li><a title="Custom Gateway.pdf" href="http://c5extras.com/files/8114/0378/3790/Custom_Gateway.pdf">Developer guide</a>&nbsp;(PDF)</li> 
       <li><a title="custom_gateway files.zip" href="http://c5extras.com/files/4114/0378/3847/custom_gateway.zip">Template package for a new payment gateway</a>&nbsp;(ZIP)</li> 
      </ul> 
      <p><em><span style="font-style: normal;"><em><br></em></span></em></p> 
      <p><strong><span style="font-style: normal;">Helpful tips</span></strong></p> 
      <p><span style="font-style: normal;"><strong><span style="color: #ff0000;">Please note</span></strong>, as this is a Javascript based shopping cart system there are technical limiations on how it works. It is important that the merchant double-check the product price and shipping price actually paid by customers (via the payment gateway) before dispatching an order.</span></p> 
      <p><span style="font-style: normal;"><br></span></p>
