             <div class="faq-component--question-wrap  faq-component--question-wrap--second-child"> 
              <div class="faq-component--question" tabindex="0" aria-expanded="true" aria-controls="faqs-1--item-0"> 
               <p>Are the USPS APIs free? <span class="visuallyhidden"> <span class="closed-text">Expand</span> <span class="opened-text">Collapse</span> FAQ question </span> </p> 
              </div> 
              <div class="faq-component--answer" id="faqs-1--item-0" tabindex="0" aria-hidden="true" style="display: none;"> 
               <div class="faq-component--answer-inner"> 
                <p>Customers who have registered for a USPS Web Tools® account and agreed to the terms and conditions of use can access USPS APIs for free. </p> 
               </div> 
              </div> 
              <div class="faq-component--toggle-button" aria-hidden="true"></div> 
             </div> 
             <div class="faq-component--question-wrap  "> 
              <div class="faq-component--question" tabindex="0" aria-expanded="false" aria-controls="faqs-1--item-1"> 
               <p>How do I get access to USPS APIs and receive my user ID? <span class="visuallyhidden"> <span class="closed-text">Expand</span> <span class="opened-text">Collapse</span> FAQ question </span> </p> 
              </div> 
              <div class="faq-component--answer" id="faqs-1--item-1" tabindex="0"> 
               <div class="faq-component--answer-inner"> 
                <p>Get access to USPS APIs when you register for a Web Tools user account.</p> 
                <p><a class="button--secondary" href="/business/web-tools-apis/web-tools-registration.htm" target="_blank">Register Now</a></p> 
                <p>Once registered, you will automatically receive an email containing your assigned Web Tools User ID. After you receive this email, you can use the following APIs:</p> 
                <ul class="list"> 
                 <li>Price Calculators</li> 
                 <li>Package Tracking</li> 
                 <li>Service Standards &amp; Commitments</li> 
                 <li>Address Information</li> 
                 <li>Package Pickup</li> 
                </ul> 
               </div> 
              </div> 
              <div class="faq-component--toggle-button " aria-hidden="true"></div> 
             </div> 
             <div class="faq-component--question-wrap  "> 
              <div class="faq-component--question" tabindex="0" aria-expanded="false" aria-controls="faqs-1--item-2"> 
               <p>How do I pay for the postage for my labels? <span class="visuallyhidden"> <span class="closed-text">Expand</span> <span class="opened-text">Collapse</span> FAQ question </span> </p> 
              </div> 
              <div class="faq-component--answer" id="faqs-1--item-2" tabindex="0"> 
               <div class="faq-component--answer-inner"> 
                <p>We do not currently offer the ability to pay for postage directly to USPS® through Web Tools APIs. For lower-volume customers, you can pay for postage at your local retail location, through a third party PC postage provider, or print and pay for labels directly through <a href="/ship/print-a-label.htm">Click-N-Ship®</a>.</p> 
                <p><a class="button--secondary" href="/business/postage-options.htm">Get Information on other Postage Payment Options</a></p> 
               </div> 
              </div> 
              <div class="faq-component--toggle-button " aria-hidden="true"></div> 
             </div> 
             <div class="faq-component--question-wrap  "> 
              <div class="faq-component--question" tabindex="0" aria-expanded="false" aria-controls="faqs-1--item-3"> 
               <p>How do I gain access to Label Printing API? <span class="visuallyhidden"> <span class="closed-text">Expand</span> <span class="opened-text">Collapse</span> FAQ question </span> </p> 
              </div> 
              <div class="faq-component--answer" id="faqs-1--item-3" tabindex="0"> 
               <div class="faq-component--answer-inner"> 
                <p>Permissions for creating shipping labels without postage are added separately. To get permission, contact our Internet Customer Care Center (<a href="mailto:uspstechnicalsupport@mailps.custhelp.com">uspstechnicalsupport@mailps.custhelp.com</a>) and include "Web Tools API Access" in the subject line. For Label API permission requests, include in the body of the email:</p> 
                <ul class="list"> 
                 <li>Which Label APIs you would like to access</li> 
                 <li>If you will be modifying the label image </li> 
                 <li>How you intend to pay for label postage. Payment cannot currently be remitted directly to USPS® through the API.</li> 
                </ul> 
                <p><a class="button--secondary" href="/business/postage-options.htm">Get Information on Postage Payment Options</a></p> 
               </div> 
              </div> 
              <div class="faq-component--toggle-button " aria-hidden="true"></div> 
             </div> 
             <div class="faq-component--question-wrap  "> 
              <div class="faq-component--question" tabindex="0" aria-expanded="false" aria-controls="faqs-1--item-4"> 
               <p>The label I generated displays "Postage Required", how do I pay for the label? <span class="visuallyhidden"> <span class="closed-text">Expand</span> <span class="opened-text">Collapse</span> FAQ question </span> </p> 
              </div> 
              <div class="faq-component--answer" id="faqs-1--item-4" tabindex="0"> 
               <div class="faq-component--answer-inner"> 
                <p>The Web Tools APIs only print labels without postage and the APIs are not linked to your USPS.com® account or payment methods. Lack of integrated payment through our APIs is a common complaint and we are working hard to add this, however, we are still in the very early stages.</p> 
                <p>We recommend customers pay for postage using either a third party postage meter or at the Post Office™ location.</p> 
                <p><a class="button--secondary" href="/business/postage-options.htm">Get Information on Postage Payment Options</a></p> 
               </div> 
              </div> 
              <div class="faq-component--toggle-button " aria-hidden="true"></div> 
             </div> 
             <div class="faq-component--question-wrap  "> 
              <div class="faq-component--question" tabindex="0" aria-expanded="false" aria-controls="faqs-1--item-5"> 
               <p>The labels I am able to print contain a “SAMPLE-DO NOT MAIL” watermark. How can I print actual labels? <span class="visuallyhidden"> <span class="closed-text">Expand</span> <span class="opened-text">Collapse</span> FAQ question </span> </p> 
              </div> 
              <div class="faq-component--answer" id="faqs-1--item-5" tabindex="0"> 
               <div class="faq-component--answer-inner"> 
                <p>The API containing "Certify" is our testing API and will print labels with the sample watermark. In order to generate actual labels, you must remove "Certify" from all of your Web Tools XML requests.</p> 
               </div> 
              </div> 
              <div class="faq-component--toggle-button " aria-hidden="true"></div> 
             </div> 
             <div class="faq-component--question-wrap  "> 
              <div class="faq-component--question" tabindex="0" aria-expanded="false" aria-controls="faqs-1--item-6"> 
               <p>Does USPS® have an EDDM® API available? <span class="visuallyhidden"> <span class="closed-text">Expand</span> <span class="opened-text">Collapse</span> FAQ question </span> </p> 
              </div> 
              <div class="faq-component--answer" id="faqs-1--item-5" tabindex="0"> 
               <div class="faq-component--answer-inner"> 
                <p>We currently only offer Every Door Direct Mail® (EDDM) services through our online application at.</p> 
                <p><a class="button--secondary" href="https://eddm.usps.com/eddm/customer/routeSearch.action">Learn More</a></p> 
               </div> 
              </div> 
              <div class="faq-component--toggle-button " aria-hidden="true"></div> 
             </div> 
             <div class="faq-component--question-wrap  "> 
              <div class="faq-component--question" tabindex="0" aria-expanded="false" aria-controls="faqs-1--item-6"> 
               <p>Have there been any recent or upcoming changes to Web Tools API? <span class="visuallyhidden"> <span class="closed-text">Expand</span> <span class="opened-text">Collapse</span> FAQ question </span> </p> 
              </div> 
              <div class="faq-component--answer" id="faqs-1--item-5" tabindex="0"> 
               <div class="faq-component--answer-inner"> 
                <p>In order to provide the very best API offering, we periodically update our systems which may require current customers to update their software. To determine if these changes/updates apply to you, refer to our most recent <a href="/business/web-tools-apis/welcome.htm">Release Notes</a>. This document contains all new changes and steps to resolve if you are affected. Please remember to first contact your software provider for any integration-related questions.</p> 
               </div> 
              </div> 
              <div class="faq-component--toggle-button " aria-hidden="true"></div>
