      <h1 itemprop="headline">Cart JS</h1> 
      <p itemprop="description">A Javascript library to power cart management for Shopify themes.</p>

      <h1><a id="cartjs" class="anchor anchor-link" href="#cartjs" aria-hidden="true"><i class="fa fa-link"></i></a>Cart.js</h1> 
      <p>Cart.js is a Javascript library that makes it easy to add dynamic cart features to Shopify themes.</p> 
      <p>It abstracts away the difficult parts of working with Shopify's AJAX API and provides a simple, consistent interface to manipulating a customer's cart. Without having to worry about the quirks in Shopify's endpoints or about making all of your requests synchronous, you can just write code like this:</p> 
      <div class="highlight highlight-source-js">
       <pre><span class="pl-k">function</span> <span class="pl-en">example</span>() {
  <span class="pl-c"><span class="pl-c">//</span> Clear the existing cart.</span>
  <span class="pl-smi">CartJS</span>.<span class="pl-c1">clear</span>();

  <span class="pl-c"><span class="pl-c">//</span> Add 3x "12345678" items, with a custom "size" property of "XL".</span>
  <span class="pl-smi">CartJS</span>.<span class="pl-en">addItem</span>(<span class="pl-c1">12345678</span>, <span class="pl-c1">3</span>, {
    <span class="pl-s"><span class="pl-pds">"</span>size<span class="pl-pds">"</span></span><span class="pl-k">:</span> <span class="pl-s"><span class="pl-pds">"</span>XL<span class="pl-pds">"</span></span>
  });

  <span class="pl-c"><span class="pl-c">//</span> Set a custom cart note.</span>
  <span class="pl-smi">CartJS</span>.<span class="pl-en">setNote</span>(<span class="pl-s"><span class="pl-pds">'</span>This is a custom cart note.<span class="pl-pds">'</span></span>);
}</pre>
      </div> 
      <p>Neat, huh? But that's just the beginning.</p> 
      <p>Cart.js bundles two powerful optional modules that make it even easier to build dynamic carts into your themes:</p> 
      <h3><a id="data-api-module" class="anchor anchor-link" href="#data-api-module" aria-hidden="true"><i class="fa fa-link"></i></a>Data API Module</h3> 
      <p>Lets you use simple <code>data-*</code> markup attributes to hook in to Cart.js methods, without having to write <em>any</em> Javascript yourself.</p> 
      <p>For example, to create a button that added a particular product to your cart, all that's required is:</p> 
      <div class="highlight highlight-text-html-basic">
       <pre>&lt;<span class="pl-ent">button</span> <span class="pl-e">data-cart-add</span>=<span class="pl-s"><span class="pl-pds">"</span>12345678<span class="pl-pds">"</span></span>&gt;Add Product&lt;/<span class="pl-ent">button</span>&gt;</pre>
      </div> 
      <h3><a id="dom-binding-module" class="anchor anchor-link" href="#dom-binding-module" aria-hidden="true"><i class="fa fa-link"></i></a>DOM Binding Module</h3> 
      <p>In combination with <a href="http://rivetsjs.com">Rivets.js</a>, the DOM Binding module lets you write HTML templates in your <code>.liquid</code> files that are re-rendered dynamically when the contents of your cart change, like this:</p> 
      <div class="highlight highlight-text-html-basic">
       <pre>&lt;<span class="pl-ent">div</span> <span class="pl-e">data-cart-view</span>&gt;
    You currently have {cart.total_items} item(s) in your cart, for a total of {cart.total_price | money_with_currency}.
&lt;/<span class="pl-ent">div</span>&gt;</pre>
      </div> 
      <h2><a id="getting-started-and-documentation" class="anchor anchor-link" href="#getting-started-and-documentation" aria-hidden="true"><i class="fa fa-link"></i></a>Getting Started and Documentation</h2> 
      <p>Documentation and examples are available on the <a href="https://cartjs.org?utm_source=github&amp;utm_medium=readme&amp;utm_campaign=cartjs">home page</a>.</p> 
      <p>You can check out a list of who's using Cart.js in production on the <a href="https://cartjs.org/pages/built-with-cart-js">Built with Cart.js</a> page.</p> 
      <h2><a id="dependencies" class="anchor anchor-link" href="#dependencies" aria-hidden="true"><i class="fa fa-link"></i></a>Dependencies</h2> 
      <p>Cart.js currently depends on <a href="http://jquery.com">jQuery</a> for its AJAX helpers, custom event bindings and utility methods. This usually isn't a problem as most Shopify themes ship with jQuery.</p> 
      <p><a href="http://rivetsjs.com">Rivets.js</a> is required if you want to use the DOM Binding module. The distribution of Cart.js includes a minified asset (<code>rivets-cart.min.js</code>) which contains both the Cart.js library and a bundled version of Rivets.js.</p>
