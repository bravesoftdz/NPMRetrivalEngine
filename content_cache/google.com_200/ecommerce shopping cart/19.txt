       <h1 id="shopify-javascript-buy-sdk">Shopify JavaScript Buy SDK</h1> 
       <p>The JS Buy SDK is a lightweight library that allows you to build ecommerce into any website. It is based on Shopify’s API and provides the ability to retrieve products and collections from your shop, add products to a cart, and checkout.</p> 
       <p>It can render data on the client side or server. This will allow you to add ecommerce functionality to any website or javascript application. This is helpful if you already have a website and need to add ecommerce or only need a simple buy button on your site.</p> 
       <p>Using the JS Buy SDK, you can:</p> 
       <ul> 
        <li>fetch information about a single product or a collection of products</li> 
        <li>create a shopping cart</li> 
        <li>allow customers to select options and quantities</li> 
        <li>generate a checkout URL for a single product or an entire cart.</li> 
       </ul> 
       <p>This tool is intended for use by developers who are experienced with JavaScript.</p>

       <h2 id="creating-a-shop-client">Creating a Shop Client</h2> 
       <p>The Client is the primary interface through which you make requests using the JS Buy SDK. You will need your <code class="highlighter-rouge">myshopify.com</code> domain, API key, and application ID to create your client and begin making requests. <a href="https://docs.shopify.com/api/sdks/js-buy-sdk/getting-started#app-id" target="_blank">Where do I find my API Key and application ID?</a></p> 
       <div class="language-js highlighter-rouge">
        <pre class="highlight"><code><span class="kd">var</span> <span class="nx">shopClient</span> <span class="o">=</span> <span class="nx">ShopifyBuy</span><span class="p">.</span><span class="nx">buildClient</span><span class="p">({</span>
  <span class="na">accessToken</span><span class="p">:</span> <span class="s1">'your-access-token'</span><span class="p">,</span>
  <span class="na">domain</span><span class="p">:</span> <span class="s1">'your-shop-subdomain.myshopify.com'</span><span class="p">,</span>
  <span class="na">appId</span><span class="p">:</span> <span class="s1">'6'</span>
<span class="p">});</span>
</code></pre> 
       </div> 
       <blockquote> 
        <p>Note: You will need to publish the product/<a href="https://docs.shopify.com/manual/products/collections/make-collections-findable#change-the-visibility-of-a-collection" target="_blank">collection</a> you wish to interact with to the “Buy Button” channel in Shopify.</p> 
       </blockquote> 
       <h2 id="making-a-request">Making a request</h2> 
       <p>You can now call a <code class="highlighter-rouge">fetch</code> method on your client to retrieve products or collections. All <code class="highlighter-rouge">fetch</code> methods return a promise which will return a <code class="highlighter-rouge">Cart</code> or <code class="highlighter-rouge">Collection</code> model for <code class="highlighter-rouge">fetchProduct</code> or <code class="highlighter-rouge">fetchCollection</code>, or an array of <code class="highlighter-rouge">Cart</code> or <code class="highlighter-rouge">Collection</code> models for <code class="highlighter-rouge">fetchAllProducts</code> or <code class="highlighter-rouge">fetchAllCollection</code>.</p> 
       <p>To request an individual resource, you will need to pass that resource’s ID as the first argument. <a href="https://docs.shopify.com/api/sdks/js-buy-sdk/getting-started#retrieving-products" target="_blank">How do I find my resource ID?</a></p> 
       <div class="language-js highlighter-rouge">
        <pre class="highlight"><code><span class="kr">const</span> <span class="nx">shopClient</span> <span class="o">=</span> <span class="nx">ShopifyBuy</span><span class="p">.</span><span class="nx">buildClient</span><span class="p">({</span>
  <span class="na">accessToken</span><span class="p">:</span> <span class="s1">'bf081e860bc9dc1ce0654fdfbc20892d'</span><span class="p">,</span>
  <span class="na">appId</span><span class="p">:</span> <span class="s1">'6'</span><span class="p">,</span>
  <span class="na">domain</span><span class="p">:</span> <span class="s1">'embeds.myshopify.com'</span>
<span class="p">});</span>

<span class="c1">// fetch a product using resource id</span>
<span class="nx">shopClient</span><span class="p">.</span><span class="nx">fetchProduct</span><span class="p">(</span><span class="s1">'8569911558'</span><span class="p">)</span>
  <span class="p">.</span><span class="nx">then</span><span class="p">(</span><span class="kd">function</span> <span class="p">(</span><span class="nx">product</span><span class="p">)</span> <span class="p">{</span>
    <span class="nx">console</span><span class="p">.</span><span class="nx">log</span><span class="p">(</span><span class="nx">product</span><span class="p">);</span>
  <span class="p">})</span>
  <span class="p">.</span><span class="k">catch</span><span class="p">(</span><span class="kd">function</span> <span class="p">()</span> <span class="p">{</span>
    <span class="nx">console</span><span class="p">.</span><span class="nx">log</span><span class="p">(</span><span class="s1">'Request failed'</span><span class="p">);</span>
  <span class="p">});</span>
</code></pre> 
       </div> 
       <h2 id="creating-a-cart">Creating a Cart</h2> 
       <p>To create a cart, use the <code class="highlighter-rouge">client.createCart()</code> function:</p> 
       <div class="language-js highlighter-rouge">
        <pre class="highlight"><code><span class="kd">var</span> <span class="nx">cart</span><span class="p">;</span>
<span class="nx">shopClient</span><span class="p">.</span><span class="nx">createCart</span><span class="p">().</span><span class="nx">then</span><span class="p">(</span><span class="kd">function</span> <span class="p">(</span><span class="nx">newCart</span><span class="p">)</span> <span class="p">{</span>
  <span class="nx">cart</span> <span class="o">=</span> <span class="nx">newCart</span><span class="p">;</span>
  <span class="c1">// do something with updated cart</span>
<span class="p">});</span>
</code></pre> 
       </div> 
       <h3 id="adding-items-to-a-cart">Adding items to a cart</h3> 
       <p>Cart items can be retrieved as an array with the <code class="highlighter-rouge">cart.lineItems</code> getter. To add items to a cart, you can call the cart’s <code class="highlighter-rouge">createLineItemsFromVariants</code> method and pass in the product(s) to be added. The <code class="highlighter-rouge">update</code> call will return a promise which returns the updated model.</p> 
       <div class="language-js highlighter-rouge">
        <pre class="highlight"><code><span class="nx">cart</span><span class="p">.</span><span class="nx">createLineItemsFromVariants</span><span class="p">({</span><span class="na">variant</span><span class="p">:</span> <span class="nx">product</span><span class="p">.</span><span class="nx">selectedVariant</span><span class="p">,</span> <span class="na">quantity</span><span class="p">:</span> <span class="mi">1</span><span class="p">}).</span><span class="nx">then</span><span class="p">(</span><span class="kd">function</span> <span class="p">(</span><span class="nx">cart</span><span class="p">)</span> <span class="p">{</span>
  <span class="c1">// do something with updated cart</span>
<span class="p">});</span>
</code></pre> 
       </div> 
       <h3 id="creating-a-cart-checkout-url">Creating a cart checkout URL</h3> 
       <p>You can generate a checkout URL for a given cart at any time by retrieving the <code class="highlighter-rouge">cart.checkoutUrl</code>.</p> 
       <div class="language-js highlighter-rouge">
        <pre class="highlight"><code><span class="nb">document</span><span class="p">.</span><span class="nx">location</span><span class="p">.</span><span class="nx">href</span> <span class="o">=</span> <span class="nx">cart</span><span class="p">.</span><span class="nx">checkoutUrl</span><span class="p">;</span>
</code></pre>
