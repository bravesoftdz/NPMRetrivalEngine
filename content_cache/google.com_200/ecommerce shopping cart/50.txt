             <h1>Introduction</h1> 
             <p><strong>SmartAdmin WebApp</strong> - goes beyond the ordinary admin template. Its unique flat design and responsive layout is crafted one of a kind. SmartAdmin includes <strong>7 version variations</strong>: Static HTML, AJAX, PHP, AngularJS 4.0, RubyOnRails, ReactJS and ASP.NET versions. Each version comes with its own unique properties and usage. SmartAdmin has a vast collection of assorted reusable UI components integrated with latest jQuery plugins. <strong>SmartAdmin Layout API lets you customize your project for almost any device - be it Mobile, Web or WebApp - SmartAdmin has you covered!</strong> </p> 
             <p>We have a long roadmap ahead for improvements and more features. All future updates are free for you to download once released on wrapbootstrap.com.</p>

             <h3>Ajax version specific features:</h3> 
             <p>AJAX version uses robust scripts to lazyload pages, components and plugins - it acts as a single page app. Never refresh your browser again! </p> 
             <p>It's built to run smoothly in all devices. Highly recommended for small scale projects as it is easily manageable. Comes with a built in auto pageDestroy() function to destroy objects and take care of those nasty memory leaks!</p> 
             <ul> 
              <li>Built to run as a one page app (never refresh again)</li> 
              <li>AJAX pages (pages are pulled in and scripts are loaded on request per page)</li> 
              <li>Lazy loads scripts, plugins and styles</li> 
              <li>Jquery animated page transitions</li> 
              <li>Loading bar</li> 
              <li>Breadcrumbs created dynamically</li> 
              <li>Contains page destroy function for memory leaks</li> 
              <li>Easily managed</li> 
             </ul> 
             <h3>AngularJS 4.0 version specific features:</h3> 
             <p>AngularJS 4.0 (complete rebuild) brings you a brand new version built from ground up using AngularCLI, Webpack and NPM package managers. It is now packed with robust features, a RESTful API with JSON files and a file structure that is neat as a pin. </p> 
             <p>Under the hood the project is maintained via webpack. But this time you won't be messing with any kind of builder config - everything is configured already and works out of the box.</p> 
             <p>If you would like to switch from <em>css</em> to <em>SASS</em>/<em>LESS</em> or add some custom entry point, then you have to update your <code>angular-cli.json</code> with a single word. </p> 
             <p>And in your dist folder you'll get the - smallest possible (<em>tree-shaking</em> - excluding unused exports from bundles), - very well optimized-chunked-minimized-uglified-gziped, with all assets processed deploy ready app-build</p> 
             <p>It also comes with ready to use services, directives, controllers, routes, i18n, and other components. All scripts are loaded via custom lazy loading technique, making it lighter, faster and smarter!</p> 
             <ul> 
              <li>Complete Modularized AngluarJS 4.0 structure, getting you more closer to integrate with a backend. </li> 
              <li>Built with TypeScript</li> 
              <li>Pure CSS3 Transitions</li> 
              <li>Menu: Condensed, Offcanvas, Collapsed, Minified and Menu on top</li> 
              <li>Validation ready Forms</li> 
              <li>Wizard with validation</li> 
              <li>Widgets utility classes</li> 
              <li>CSS3 Spinners</li> 
              <li>RESTful API with JSON files for Chat, Calendar Events, Inbox &amp; Messages, Project list and Todo's</li> 
              <li>In-app Settings (with autosave)</li> 
              <li>Internationalization (i18n) and Localization (l10n) support (translation loaded from JSON files)</li> 
              <li>Switching app root-layouts according to route (e.g. different layout for login/register pages) </li> 
              <li>Lazy loading everywhere - routing and plugins </li> 
              <li>Google Map - async map initializer </li> 
              <li>Image editor - Angular 4 application with Redux data flow </li> 
              <li>CK Editor - loading scripts from cdn</li>
