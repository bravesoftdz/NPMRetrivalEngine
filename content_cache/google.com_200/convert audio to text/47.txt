                <h2>Drupal 8</h2> 
                <p>The Drupal 8 version of the <b>Audio Field module</b> is a full rewrite from the Drupal 7 module. This module is a file formatter for the File and Link modules. It allows you to upload or link audio files and play them on your site using one of several audio players available. It supports most audio formats (depends on the selected audio player). </p> 
                <p>Please see <a href="https://www.drupal.org/docs/8/modules/audiofield" rel="nofollow">the module documentation</a> for examples of the players in use.</p> 
                <p>Currently the module supports the following audio players, which must be downloaded and installed as libraries: </p>
                <ul>
                 <li>Default HTML5 audio player (built-in)</li> 
                 <li>Audio.js - <a href="https://kolber.github.io/audiojs/" rel="nofollow">https://kolber.github.io/audiojs/</a></li> 
                 <li>jPlayer - <a href="http://jplayer.org/" rel="nofollow">http://jplayer.org/</a></li> 
                 <li>MediaElement - <a href="http://mediaelementjs.com/" rel="nofollow">http://mediaelementjs.com/</a></li> 
                 <li>Projekktor - <a href="https://github.com/frankyghost/projekktor" rel="nofollow">https://github.com/frankyghost/projekktor</a></li> 
                 <li>SoundManager - <a href="http://www.schillmania.com/projects/soundmanager2" rel="nofollow">http://www.schillmania.com/projects/soundmanager2</a></li> 
                 <li>wavesurfer.js - <a href="https://wavesurfer-js.org" rel="nofollow">https://wavesurfer-js.org</a></li> 
                 <li>WordPress Audio - <a href="http://wpaudioplayer.com" rel="nofollow">http://wpaudioplayer.com</a></li> 
                </ul>
                <p>All of these supported audio players offer customization options in line with those made available by the audio players themselves. Those players above which are themable offer theme options as well. If you would like to see additional audio players added, please file an issue in the issue queue and we will do our best to accomodate the player.</p> 
                <h3>Developed by</h3> 
                <p><a href="http://drupal.org/user/1160788" target="_blank" rel="nofollow">Daniel Moberly</a></p> 
                <h3>Installation</h3> 
                <p>View the module's README file and see the <a href="https://www.drupal.org/docs/8/modules/audiofield" rel="nofollow">Documentation</a> for more information.</p> 
                <h2>Drupal 7</h2> 
                <p><b>Audio Field module</b> adds new field that allows you to upload and play audio files in HTML5 audio player or one of selected Flash players. Currently it supports 6 different Flash players (needs to be downloaded, see README) and it provides <b>API</b> for easy addition of new players.</p> 
                <p>Currently this module supports this 3 audio formats: MP3, Wav, and Ogg. Other modules can implement support for other formats using AudioField API.</p> 
                <h3>Developed by</h3> 
                <p><a href="http://drupal.org/user/464564" target="_blank" rel="nofollow">Tamer Zoubi</a></p> 
                <h3>Installation</h3> 
                <p>See the <a href="http://drupal.org/node/973194" rel="nofollow">Documentation</a>.</p> 
                <h3>Select default player</h3> 
                <p>Go to admin/settings/audiofield and you will see list of available players. Player you select will be used as default player for all audio files.</p> 
                <h3>Add audio upload to the content type</h3> 
                <p>First you must add audiofield to the preferred content type. Adding a field is done as adding any other field, select field type of "File" and widget of "Audio Upload".<br> You can add audiofield to any entity type (node, comment, user, taxonomy, etc...)</p> 
                <h3>How to upload and play audio file?</h3> 
                <p>After adding audiofield to content type you can upload audio files when creating new nodes. Uploaded audio files will be rendered in default player you selected in admin form.</p> 
                <h3>API</h3> 
                <p>Originally this module supports only mp3 audio files. But other modules can extend this support by implementing <i>hook_audiofield_players()</i> in their modules.<br> For example see the <a href="http://drupal.org/node/973194" rel="nofollow">Documentation</a>.</p> 
                <h3>Modules using AudioField</h3> 
                <ul>
                 <li><a href="/project/audiorecorderfield" rel="nofollow">AudioRecorderField</a><br> -record audio from microphone + wav player </li> 
                 <li><a href="/project/audioconverter" rel="nofollow">Audio Converter</a><br> -convert your audio files to mp3 using FFMPEG or SoX Utility </li> 
                 <li><a href="/project/phonerecorderfield" rel="nofollow">Phone Recorder Field</a><br> -record audio through Phone call and play in one of AudioField players. </li> 
                </ul>
                <h3>How can you get involved?</h3> 
                <ul>
                 <li> Report any bugs, feature requests, etc. in the issue tracker.</li> 
                 <li> <a href="http://drupal.org/user/464564/contact" rel="nofollow">Contact the maintainer</a> with any comments, questions, or a quote for custom module customizations.</li> 
                </ul>
                <p><a href="http://drupal.org/user/464564" target="_blank" rel="nofollow">Tamer Zoubi</a> can be contacted for paid customizations of this module and for Drupal consulting and development</p>
