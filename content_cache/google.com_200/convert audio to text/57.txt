It has functions to start and stop listening through the microphone and we can set events to be fired when a given keyword is detected.<br> <a href="http://pazguille.github.io/voix/" class="su-button su-button-style-default" style="color:#FFFFFF;background-color:#213347;border-color:#1a2939;border-radius:6px;-moz-border-radius:6px;-webkit-border-radius:6px" target="_blank" rel="nofollow"><span style="color:#FFFFFF;padding:0px 18px;font-size:14px;line-height:28px;border-color:#64707e;border-radius:6px;-moz-border-radius:6px;-webkit-border-radius:6px;text-shadow:none;-moz-text-shadow:none;-webkit-text-shadow:none"> Demo &amp; Download </span></a> </p>
       <h2>13. Sound.js</h2> 
       <p><img class="alignnone size-full wp-image-2060" src="http://www.bestdevlist.com/wp-content/uploads/2016/06/soundjs.jpeg" alt="soundjs" width="625" height="291" srcset="http://www.bestdevlist.com/wp-content/uploads/2016/06/soundjs.jpeg 625w, http://www.bestdevlist.com/wp-content/uploads/2016/06/soundjs-300x140.jpeg 300w" sizes="(max-width: 625px) 100vw, 625px"><br> A JavaScript library that provides a simple API, and powerful features to make working with audio a breeze. SoundJS abstracts HTML5 sound implementation, making adding consistent cross-browser sound to your games or rich experiences much easier. You can query for capabilities, then specify and prioritize what APIs, plugins, and features are leveraged for specific devices or browsers.<br> <a href="http://www.createjs.com/soundjs" class="su-button su-button-style-default" style="color:#FFFFFF;background-color:#213347;border-color:#1a2939;border-radius:6px;-moz-border-radius:6px;-webkit-border-radius:6px" target="_blank" rel="nofollow"><span style="color:#FFFFFF;padding:0px 18px;font-size:14px;line-height:28px;border-color:#64707e;border-radius:6px;-moz-border-radius:6px;-webkit-border-radius:6px;text-shadow:none;-moz-text-shadow:none;-webkit-text-shadow:none"> Demo &amp; Download </span></a> </p>
       <h2>14. Audiolet</h2> 
       <p><img class="alignnone size-full wp-image-2063" src="http://www.bestdevlist.com/wp-content/uploads/2016/06/audiolet.jpeg" alt="audiolet" width="625" height="309" srcset="http://www.bestdevlist.com/wp-content/uploads/2016/06/audiolet.jpeg 625w, http://www.bestdevlist.com/wp-content/uploads/2016/06/audiolet-300x148.jpeg 300w" sizes="(max-width: 625px) 100vw, 625px"><br> Audiolet is a JavaScript library for real-time audio synthesis and composition from within the browser. It uses graph-based routing and pattern-based scheduling to make complex audio simple to program, and easy to understand.<br> <a href="https://www.jhtrnr.com/Audiolet/" class="su-button su-button-style-default" style="color:#FFFFFF;background-color:#213347;border-color:#1a2939;border-radius:6px;-moz-border-radius:6px;-webkit-border-radius:6px" target="_blank" rel="nofollow"><span style="color:#FFFFFF;padding:0px 18px;font-size:14px;line-height:28px;border-color:#64707e;border-radius:6px;-moz-border-radius:6px;-webkit-border-radius:6px;text-shadow:none;-moz-text-shadow:none;-webkit-text-shadow:none"> Demo &amp; Download </span></a> </p>
       <h2>15. Audio5js</h2> 
       <p><img class="alignnone size-full wp-image-2062" src="http://www.bestdevlist.com/wp-content/uploads/2016/06/audio5js.jpeg" alt="audio5js" width="625" height="291" srcset="http://www.bestdevlist.com/wp-content/uploads/2016/06/audio5js.jpeg 625w, http://www.bestdevlist.com/wp-content/uploads/2016/06/audio5js-300x140.jpeg 300w" sizes="(max-width: 625px) 100vw, 625px"><br> Audio5js is a Javascript library that provides a seamless compatibility layer to the HTML5 Audio playback API, with multiple codec support and a Flash-based MP3 playback fallback for older or unsupported browsers. The motivation for creating Audio5js is to provide a light-weight, library-agnostic, JavaScript-only interface for audio playback in the browser.<br> <a href="http://zohararad.github.io/audio5js/" class="su-button su-button-style-default" style="color:#FFFFFF;background-color:#213347;border-color:#1a2939;border-radius:6px;-moz-border-radius:6px;-webkit-border-radius:6px" target="_blank" rel="nofollow"><span style="color:#FFFFFF;padding:0px 18px;font-size:14px;line-height:28px;border-color:#64707e;border-radius:6px;-moz-border-radius:6px;-webkit-border-radius:6px;text-shadow:none;-moz-text-shadow:none;-webkit-text-shadow:none"> Demo &amp; Download </span></a> </p>
       <h2>16. Buzz</h2> 
       <p><img class="alignnone size-full wp-image-2064" src="http://www.bestdevlist.com/wp-content/uploads/2016/06/Buzz.jpeg" alt="Buzz" width="625" height="321" srcset="http://www.bestdevlist.com/wp-content/uploads/2016/06/Buzz.jpeg 625w, http://www.bestdevlist.com/wp-content/uploads/2016/06/Buzz-300x154.jpeg 300w" sizes="(max-width: 625px) 100vw, 625px"><br> Buzz is a small but powerful Javascript library that allows you to easily take advantage of the new HTML5 audio element.        <h2>17. Voice Elements</h2> 
       <p><img class="alignnone size-full wp-image-2065" src="http://www.bestdevlist.com/wp-content/uploads/2016/06/voice-elements.jpg" alt="voice-elements" width="625" height="374" srcset="http://www.bestdevlist.com/wp-content/uploads/2016/06/voice-elements.jpg 625w, http://www.bestdevlist.com/wp-content/uploads/2016/06/voice-elements-300x180.jpg 300w" sizes="(max-width: 625px) 100vw, 625px"><br> Voice Elements is a web Component wrapper to the Web Speech API, that allows you to do voice recognition (speech to text) and speech synthesis (text to speech) using Polymer. Voice recognition is the translation of spoken words into text. This is achieved in the browser by using the SpeechRecognition interface from the Web Speech API.<br> <a href="http://zenorocha.github.io/voice-elements/" class="su-button su-button-style-default" style="color:#FFFFFF;background-color:#213347;border-color:#1a2939;border-radius:6px;-moz-border-radius:6px;-webkit-border-radius:6px" target="_blank" rel="nofollow"><span style="color:#FFFFFF;padding:0px 18px;font-size:14px;line-height:28px;border-color:#64707e;border-radius:6px;-moz-border-radius:6px;-webkit-border-radius:6px;text-shadow:none;-moz-text-shadow:none;-webkit-text-shadow:none"> Demo &amp; Download </span></a> </p>
       <h2>18. JuliusJS</h2> 
       <p><img class="alignnone size-full wp-image-2066" src="http://www.bestdevlist.com/wp-content/uploads/2016/06/juliusjs.jpeg" alt="juliusjs" width="625" height="321" srcset="http://www.bestdevlist.com/wp-content/uploads/2016/06/juliusjs.jpeg 625w, http://www.bestdevlist.com/wp-content/uploads/2016/06/juliusjs-300x154.jpeg 300w" sizes="(max-width: 625px) 100vw, 625px"><br> JuliusJS is a speech recognition library for the web. It is an opinionated port of Julius to JavaScript. It actively listens to the user to transcribe what they are saying through a callback. All recognition is done in-browser through a Worker. It is licensed under MIT License.<br> <a href="https://github.com/zzmp/juliusjs" class="su-button su-button-style-default" style="color:#FFFFFF;background-color:#213347;border-color:#1a2939;border-radius:6px;-moz-border-radius:6px;-webkit-border-radius:6px" target="_blank" rel="nofollow"><span style="color:#FFFFFF;padding:0px 18px;font-size:14px;line-height:28px;border-color:#64707e;border-radius:6px;-moz-border-radius:6px;-webkit-border-radius:6px;text-shadow:none;-moz-text-shadow:none;-webkit-text-shadow:none"> Demo &amp; Download </span></a> </p>
       <h2>19. SoundManager 2</h2> 
       <p><img class="alignnone size-full wp-image-2067" src="http://www.bestdevlist.com/wp-content/uploads/2016/06/soundmanager2.jpeg" alt="soundmanager2" width="625" height="291" srcset="http://www.bestdevlist.com/wp-content/uploads/2016/06/soundmanager2.jpeg 625w, http://www.bestdevlist.com/wp-content/uploads/2016/06/soundmanager2-300x140.jpeg 300w" sizes="(max-width: 625px) 100vw, 625px"><br> SoundManager 2 uses HTML5 and Flash to provide consistent cross-browser and cross-platform audio support with a single JavaScript API.
