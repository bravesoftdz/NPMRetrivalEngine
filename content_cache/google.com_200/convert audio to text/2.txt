      <div class="content"> 
       <p>With Bing Speech API you can develop basic JavaScript applications that leverage Microsoft’s cloud servers to convert spoken audio to text. This article describes an example web application created in Visual Studio 2015 that demonstrates the use of Bing Speech API using Javascript.</p> 
       <p>The Speech Recognition example demonstrates the following features using a wav file or external microphone input:</p> 
       <ul> 
        <li>Short-form recognition</li> 
        <li>Recognition with intent</li> 
       </ul> 
       <p>To use Speech.JS, simply host Speech.1.0.0.js on your website. A 'minified' version of Speech.JS is also available Speech.1.0.0.min.js.</p> 
       <p><a name="Prerequisites"> </a></p> 
       <h2 id="prerequisites">Prerequisites</h2> 
       <h4 id="platform-requirements">Platform requirements</h4> 
       <p>The below example has been developed for the .NET Framework using <a href="https://www.visualstudio.com/products/visual-studio-community-vs" data-linktype="external">Visual Studio 2015, Community Edition</a>.</p> 
       <h4 id="get-the-client-library-and-example">Get the client library and example</h4> 
       <p>You may download the Speech API client library and example through <a href="https://github.com/Microsoft/Cognitive-Speech-STT-JavaScript" data-linktype="external">GitHub</a>. The downloaded folder needs to be hosted locally on your machine to follow the below scenario.</p> 
       <h4 id="subscribe-to-speech-api-and-get-a-free-trial-subscription-key">Subscribe to Speech API and get a free trial subscription key</h4> 
       <p>Before creating the example, you must subscribe to Speech API which is part of Microsoft Cognitive Services (previously Project Oxford). For subscription and key management details, see <a href="https://www.microsoft.com/cognitive-services/en-us/sign-up" data-linktype="external">Subscriptions</a>. Both the primary and secondary key can be used in this tutorial.</p> 
       <p><a name="Step1"> </a></p> 
       <h2 id="step-1-install-the-example-applicationa">Step 1: Install the Example Application</h2> 
       <ol> 
        <li>Start Microsoft Visual Studio 2015 and click <strong>File</strong>, select <strong>Open</strong>, then <strong>Project/Solution</strong>.</li> 
        <li>Browse to the folder where you saved the downloaded Speech.JS files. Click on <strong>Speech</strong> and then the <strong>Speech.JS</strong>folder.</li> 
        <li>Double-click to open the Visual Studio 2015 Solution (.sln) file named <strong>Oxford.Speech.JS.sln</strong>. This will open the solution in Visual Studio.</li> 
       </ol> 
       <p><a name="Step2"> </a></p> 
       <h2 id="step-2-build-the-example-application">Step 2: Build the Example Application</h2> 
       <ol> 
        <li>Press Ctrl+Shift+B, or click <strong>Build</strong> on the ribbon menu, then select <strong>Build Solution</strong>.</li> 
       </ol> 
       <p><a name="Step3"> </a></p> 
       <h2 id="step-3-run-the-example-application">Step 3: Run the Example Application</h2> 
       <ol> 
        <li>After the build is complete, choose the target browser you would like to use for running your Speech-to-Text web app. <img src="../images/selectemulator.png" alt="Speech choose emulator" data-linktype="relative-path"></li> 
        <li>Press <strong>F5</strong> or click <strong>Start</strong> on the ribbon menu to run the example.</li> 
        <li>Locate the <strong>Cognitive Services Speech to Text</strong> window with the <strong>text edit box</strong> reading <strong>"Subscription"</strong>. Paste your subscription key into the subscription text box.</li> 
        <li>Check if you would like to use the Microphone as your voice input and what speech mode you would like to use by selecting a setting in the "Mode" drop-down box.</li> 
        <li>For modes where you would like both Speech Recognition and Intent to work together, you need to first sign up for <a href="https://www.luis.ai/" data-linktype="external">Language Understanding Intelligent Service (LUIS)</a>, then set the key values in the fields "LUIS AppID" and "LUIS SubscriptionId".</li> 
       </ol>
