          <div class="field field-type-text-with-summary"> 
           <p>One of Google's latest releases is a perfect example of how large technology companies are working toward the <strong>API economy</strong>; and how they are working to use application programming interfaces to attract the developer community, facilitate product and service creation and extend their influence beyond the four walls of their head offices. <a href="https://cloud.google.com/speech/">Cloud Speech API</a> by Mountain View <strong>converts audio into text in over 80 languages</strong>.</p> 
           <p>You can either transcribe incoming audio from the phone's microphone or an application, or control the device using your voice. This is possible because the tool applies wide-range <strong>neural network models</strong> targeted at <strong>processing natural language</strong>. The first obvious question is: <strong>What is a neural network and what is it for?</strong></p> 
           <p>There are a lot of definitions of "neural network" and some of them are extremely complex. One of the easiest to understand may be by Dr. <strong>Simon Haykin</strong> in his book <a href="https://drive.google.com/file/d/0B2iRDvP8jUuAUnpfaDBnQTBWLUU/edit">‘</a><a href="https://drive.google.com/file/d/0B2iRDvP8jUuAUnpfaDBnQTBWLUU/edit" target="_blank"><em>Neural Networks: A comprehensive foundation</em></a>: "A neural network is a massively parallel distributed processor made up of simple processing units that has a natural propensity for storing experiential knowledge and making available for use."</p> 
           <p>How does an Artificial Neural Network (ANN) acquire and store knowledge? Through a <strong>learning process and neural interconnections</strong> that store the information and generate an output stimulus. To some extent, its learning and processing procedure is similar to the procedure of the human brain.</p> 
           <p>For a neural network to acquire <strong>knowledge</strong> a <strong>learning algorithm</strong> is required. The algorithm's process is based on randomly and sequentially applying a series of training data from which the network gains information and then learns from. It's a matter of patterns.</p> 
           <p><a href="http://perso.wanadoo.es/alimanya/funcion.htm" target="_blank">There are three types of learning</a>:</p> 
           <p style="margin-left:36.0pt;">?&nbsp;<strong>Supervised learning</strong>: input values are entered and generate output values. These results are compared to the correct values and any network deviations are corrected to adjust the process.</p> 
           <p style="margin-left:36.0pt;">?&nbsp;<strong>Reinforcement learning</strong>: input values are entered into the network and their output values are then checked for correctness.&nbsp;</p> 
           <p style="margin-left:36.0pt;">?&nbsp;<strong>Unsupervised learning</strong>: the neural network creates classification patterns from which it sorts out the supplied information.&nbsp;</p> 
           <p style="margin-left:36.0pt;"><img alt="" height="531" src="/sites/default/files/img/embed/new/bbva-open4u-google-aprendizaje-api.png" width="736"></p> 
           <p><a href="http://halweb.uc3m.es/esp/Personal/personas/jmmarin/esp/DM/tema3dm.pdf" target="_blank">The main characteristics</a> of any neural network are:</p> 
           <p style="margin-left:36.0pt;">?&nbsp;<strong>Auto-organization and adaptability</strong>: adaptive learning algorithm.</p> 
           <p style="margin-left:36.0pt;">?&nbsp;<strong>Non-lineal processing</strong>: it increases the capacity of the artificial neural network in terms of extracting and classifying patterns from among noise.</p> 
           <p style="margin-left:36.0pt;">?&nbsp;<strong>Parallel processing</strong>: large number of nodes for greater interconnectivity.&nbsp;</p> 
           <h3> Cloud Speech API: characteristics</h3> 
           <p>Google's new API contains some of the most interesting functionalities when you need an application programming interface linked to <strong>natural language processing, speech recognition and obtaining results in real time</strong>. This is important since a sufficiently high processing speed is needed to be able to respond immediately.</p> 
           <p style="margin-left:36.0pt;">?&nbsp;<strong>Automatic Speech Recognition</strong> (ASR): an in-depth learning neural network is used to recognized speech, provide speech-based search features and transcribe speech.</p> 
           <p style="margin-left:36.0pt;">?&nbsp;<strong>Streaming recognition</strong>: as the API processes and recognizes the user's speech, it returns results in real time with no waiting times. This allows the application to offer all speech processing functionalities.</p> 
           <p style="margin-left:36.0pt;">?&nbsp;<strong>Buffered audio support</strong>: the API processes sound from the microphone of an application or mobile device and packages it in various compression formats: FLAC, AMR, PCMU and linear-16. This compression is necessary to subsequently process the sound.</p> 
           <p style="margin-left:36.0pt;">?&nbsp;<strong>Speech recognition in over 80 languages</strong>. This characteristic offers a major competitive advantage over other providers of similar services for external developers.&nbsp;</p> 
           <p style="margin-left:36.0pt;">?&nbsp;<strong>Integrated API</strong>.</p> 
           <p style="margin-left:36.0pt;">?&nbsp;<strong>Inappropriate content filtering</strong>.&nbsp;</p> 
           <p><iframe allowfullscreen frameborder="0" height="auto" src="https://www.youtube.com/embed/wzp9dfVpeeg" style="height:421px !important" width="100%"></iframe></p> 
           <h3> Nuance, largest market rival</h3> 
           <p>For a long time, when developers needed to incorporate speech recognition and natural language processing functionalities into their applications, their usual provider was <a href="http://www.nuance.com/index.htm" target="_blank">Nuance</a>. Its technology is part of many current market leaders in language interpretation such as <strong>voice assistant Siri</strong> and assistants by Apple, <strong>S-Voice</strong> and Samsung. Also, car manufacturers for instance <strong>usually need this type of resource for their on-board computers</strong>, e.g. BMW and Chrysler.</p> 
           <p>By releasing Cloud Speech API, Google aims to attract large mobile device and car manufacturers away from their current providers. In addition to processing speech and responding in real time through the cloud, it supports more languages: 80 languages for Speech API <a href="https://developer.nuance.com/public/index.php?task=home" target="_blank">vs. 40 languages currently supported by Nuance's mobile SDKs </a>&nbsp;(for Android and iOS and browsers). &nbsp;</p> 
           <p>At the moment, Google Speech API's access to the cloud is limited but the company has not yet revealed how limited it actually is. Any developer <a href="https://services.google.com/fb/forms/speech-api-alpha/" target="_blank">can fill out a simple form</a> and start trying the application programming interface. In the medium term, it is expected that Google Speech API will charge developers for accessing and using it.&nbsp;</p>
