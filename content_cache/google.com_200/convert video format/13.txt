   <h1><small class="date">May 8th 2014</small>5 must-have Atom packages / plugins </h1>
   <p>I'm excited about <a href="http://atom.io">Atom</a> because it's open source and is written in Javascript. For me it was also much easier to setup than Sublime Text. There are already many packages out there and in this post I'm sharing my favorite 5.</p> 
   <h2 id="video-my-5-favorite-atom-packages-in-under-3-minutes">Video: My 5 favorite Atom packages in under 3 minutes</h2> 
   <div class="embed-container">
    <div class="embed">
     <iframe src="http://www.youtube.com/embed/nTdTBs6jk98" frameborder="0" allowfullscreen></iframe>
    </div>
   </div> 
   <h2 id="1-emmet">1. Emmet</h2> 
   <p>Everybody knows Emmet and there's an Atom-package for it. Here's how it works if you don't: Just type <code>.container</code> and hit TAB to create a <code>div</code>-element with the class <code>container</code>.</p> 
   <p>By typing <code>ul#emmet-pros&gt;li.emmet-pro*3</code>, you're telling Emmet to create this:</p> 
   <pre><code class="lang-html hljs">&lt;ul id=<span class="hljs-string">"emmet-pros"</span>&gt;
  &lt;<span class="hljs-keyword">li</span> <span class="hljs-keyword">class</span>=<span class="hljs-string">"emmet-pro"</span>&gt;&lt;/<span class="hljs-keyword">li</span>&gt;
  &lt;<span class="hljs-keyword">li</span> <span class="hljs-keyword">class</span>=<span class="hljs-string">"emmet-pro"</span>&gt;&lt;/<span class="hljs-keyword">li</span>&gt;
  &lt;<span class="hljs-keyword">li</span> <span class="hljs-keyword">class</span>=<span class="hljs-string">"emmet-pro"</span>&gt;&lt;/<span class="hljs-keyword">li</span>&gt;
&lt;/ul&gt;
</code></pre> 
   <p>Then you can jump from one edit point to the next to quickly fill in the content.</p> 
   <p>Link: <a href="https://atom.io/packages/emmet">Emmet package on Atom.io</a></p> 
   <h2 id="2-fancy-new-file">2. Fancy New File</h2> 
   <p>This lets you create new files by typing a <strong>file path</strong> (with auto-complete by hitting TAB while typing), instead of using right-clicks in the tree view.</p> 
   <p>CMD + ALT + N brings up the command palette. Then you can type something like <code>js/my-module/index.js</code></p> 
   <p>Link: <a href="https://atom.io/packages/fancy-new-file">Fancy New File package on Atom.io</a></p> 
   <h2 id="3-jshint">3. JSHint</h2> 
   <p>I think every Javascript developer is familiar with this tool and Atom actually has several packages. I chose this one because it explicitly supports .jshintrc-files.</p> 
   <p>Link: <a href="https://atom.io/packages/jshint">JSHint package on Atom.io</a></p> 
   <h2 id="4-atom-alignment">4. Atom Alignment</h2> 
   <p>Highlight your variables assignments, hit CTRL + ALT + A…</p> 
   <p>…and this…</p> 
   <pre><code class="lang-javascript hljs"><span class="hljs-keyword">var</span> aVariable = <span class="hljs-string">'some-value'</span>;
<span class="hljs-keyword">var</span> anotherVariable = <span class="hljs-number">12.5</span>;
<span class="hljs-keyword">var</span> awesomeModule = <span class="hljs-built_in">require</span>(<span class="hljs-string">'awesome-module'</span>);
<span class="hljs-keyword">var</span> <span class="hljs-literal">that</span> = <span class="hljs-keyword">this</span>;
</code></pre> 
   <p>…becomes this:</p> 
   <pre><code class="lang-javascript hljs"><span class="hljs-keyword">var</span> aVariable       = <span class="hljs-string">'some-value'</span>;
<span class="hljs-keyword">var</span> anotherVariable = <span class="hljs-number">12.5</span>;
<span class="hljs-keyword">var</span> awesomeModule   = <span class="hljs-built_in">require</span>(<span class="hljs-string">'awesome-module'</span>);
<span class="hljs-keyword">var</span> <span class="hljs-literal">that</span>            = <span class="hljs-keyword">this</span>;
</code></pre> 
   <p>Link: <a href="https://atom.io/packages/atom-alignment">Alignment package/on Atom.io </a></p> 
   <h2 id="5-change-case">5. Change Case</h2> 
   <p>Use the command palette to convert a selection to a different case convention. Currently supported:</p> 
   <ul> 
    <li>camelCase</li> 
    <li>snake_case</li> 
    <li>STATIC_VARIABLE</li> 
    <li>dot.notation</li> 
    <li>file/path</li> 
   </ul> 
   <p>Link: <a href="https://atom.io/packages/change-case">Change Case package on Atom.io</a></p>

   <p>I'm very interested in what you are using with Atom, so please leave me a comment below: Which packages are must-have for your workflow in Atom and what are you still missing?</p>
