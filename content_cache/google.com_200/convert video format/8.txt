      <h1>whitewater 1.0.1</h1> 
      <div id="download-button"> 
       <a class="button green" style="float:right;" href="#downloads">Downloads ?</a> 
      </div> 
      <p style="font-style: italic">Convert videos into a format readable by the Whitewater Player Javascript library.</p> 
      <p>A command line utility that converts short videos a format that can be played on mobile websites with the Whitewater Player Javascript library.</p> 
      <p>? <a href="https://github.com/samiare/whitewater-encoder/wiki" rel="nofollow">Full Documentation</a></p> 
      <p>? <a href="https://github.com/samiare/whitewater-mobile-video" rel="nofollow">Whitewater Player</a> (needed to play videos)</p> 
      <p>? <a href="https://git.io/whitewater" rel="nofollow">Example Site</a></p> 
      <div id="installation"> 
       <h2>Installation</h2> 
       <pre>$ pip install whitewater
</pre> 
      </div> 
      <div id="usage"> 
       <h2>Usage</h2> 
       <pre>$ whitewater &lt;file&gt; <span class="o">[</span>options<span class="o">]</span>
$ whitewater <span class="o">(</span>-h <span class="p">|</span> --help <span class="p">|</span> --version<span class="o">)</span>
</pre> 
       <p><strong>Example:</strong></p> 
       <pre>$ whitewater path/to/video.mp4
</pre> 
       <div id="options"> 
        <h3>Options</h3> 
        <pre>--blocksize &lt;size&gt;   The width/height of a single grid cell when
                     checking one frame against the previous one.
--grid &lt;size&gt;        The size of the diffmap images in rows and
                     columns, not absolute pixels.
--quality &lt;percent&gt;  JPEG quality setting.
--threshold &lt;rms&gt;    RMS threshold for determining whether a
                     single cell of a frame is different from
                     the previous one.
--format &lt;filetype&gt;  File format to save diffmap images as.
</pre> 
        <p>For a full explanation of what these do and when you might want to use them, check <a href="https://github.com/samiare/whitewater-encoder/wiki/Glossary" rel="nofollow">the documentation</a>.</p> 
       </div> 
      </div> 
      <div id="python-module"> 
       <h2>Python Module</h2> 
       <p>Whitewater can also be used as a module in your own Python scripts. The <tt>Whitewater()</tt> class and its options are described in detail in the <a href="https://github.com/samiare/whitewater-encoder/wiki" rel="nofollow">full documentation</a>.</p> 
       <pre><span class="kn">from</span> <span class="nn">whitewater</span> <span class="kn">import</span> <span class="n">Whitewater</span>

<span class="n">video</span> <span class="o">=</span> <span class="n">Whitewater</span><span class="p">(</span><span class="s1">'path/to/video.mp4'</span><span class="p">)</span>
<span class="n">video</span><span class="o">.</span><span class="n">encode</span><span class="p">()</span>
</pre>
