     <div class="formatted-text"> 
      <p><strong>Open source software inclucded:</strong></p> 
      <ul> 
       <li>VideoJS -&nbsp;JavaScript for HTML5 video player</li> 
       <li>Flowplayer free -&nbsp;Flash fallback for playing videos for browsers that do not support &lt;video&gt; element</li> 
      </ul> 
      <p><strong>Notes:</strong></p> 
      <p>For info on HTML5 video see : <a href="http://diveintohtml5.info/video.html" target="_blank">http://diveintohtml5.info/video.html</a> which has a tutorial for video conversion for web using the (free) Miro Video Converter. As HTML5 video implementations vary per web browser, to use this block you will need to prepare (4) files: .mp4, .ogv, .webm and have a preview or "poster" image. Also, if needed, Flowplayer (free) plays your .mp4 as flash media content.</p> 
      <p><strong>Block supports videos playback using the following:</strong></p> 
      <ol> 
       <li><em>Preview Image</em>: Choose a 'Poster' image, file with a .png or .jpg extension; e.g. still frame from video.</li> 
       <li><em>MPEG 4 (.mp4)</em>: Choose a file with a .mp4 extension as common HTML5 video format; also used for flash fallback</li> 
       <li><em>Ogg video (.ogv)</em>: Choose a file with a .ogv extension; "Theora" file, HTML5 video format e.g. for FireFox</li> 
       <li><em>WebM video (.webm)</em>: Choose a file with a .webm extension; HTML5 video format e.g. for MSIE 9+</li> 
       <li><em>Maximum Dimensions</em>: Input the maximum width and height in pixels</li> 
      </ol> 
      <p><strong>Source Code:</strong></p> 
      <ul> 
       <li>Github:&nbsp;<a href="https://github.com/pixelhandler/videojs_player" target="_blank">https://github.com/pixelhandler/videojs_player</a></li> 
      </ul> 
      <p><strong>Requirements:</strong></p> 
      <ul> 
       <li>Set your 'Allowed File Types' under 'File Manager &gt; Access' tab to include file types: mp4, ogv, webm<br>e.g. yourdomain.com/index.php/dashboard/files/access/&nbsp;</li> 
       <li>Format and upload files in the above mentioned formats, see references below for instructions.</li> 
       <li>Your server should understand these filetypes as well, if needed add to your .htaccess file the proper MIME type declatations see the HTML5Boilerplate project, link below.</li> 
      </ul> 
      <p><strong>References:</strong></p> 
      <ul> 
       <li><a href="http://diveintohtml5.info/video.html" target="_blank">http://diveintohtml5.info/video.html</a></li> 
       <li><a href="http://www.mirovideoconverter.com/" target="_blank">http://www.mirovideoconverter.com/</a></li> 
       <li><a href="https://github.com/h5bp/html5-boilerplate/blob/master/.htaccess" target="_blank">https://github.com/h5bp/html5-boilerplate/blob/master/.htaccess</a></li> 
      </ul>
