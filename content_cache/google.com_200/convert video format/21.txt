           <p>One of the best things about <a href="https://nodejs.org/en/" onclick="__gaTracker('send', 'event', 'outbound-article', 'https://nodejs.org/en/', 'Node.js');">Node.js</a> is its massive module ecosystem. With bundlers like <a href="https://webpack.github.io/" onclick="__gaTracker('send', 'event', 'outbound-article', 'https://webpack.github.io/', 'webpack');">webpack</a> we can leverage these even in the browser&nbsp;outside of Node.js. Let’s look at how we can build a module with <a href="http://www.typescriptlang.org/" onclick="__gaTracker('send', 'event', 'outbound-article', 'http://www.typescriptlang.org/', 'TypeScript');">TypeScript</a> usable by both JavaScript developers and TypeScript developers.</p> 
           <p>Before we get started make sure that you have <a href="https://nodejs.org/" onclick="__gaTracker('send', 'event', 'outbound-article', 'https://nodejs.org/', 'Node.js');">Node.js</a>&nbsp;installed – you should ideally have a version of 6.11 or higher. Additionally make sure that you have <a href="https://www.npmjs.com/" onclick="__gaTracker('send', 'event', 'outbound-article', 'https://www.npmjs.com/', 'npm');">npm</a>&nbsp;or a similar package manager installed.</p> 
           <p>Let’s build a module that exposes a function that filters out all emojis in a string and returns the list of emoji shortcodes. Because who doesn’t love emojis?</p> 
           <p><img class="&quot; aligncenter" style="width: 321.50px; height: 321.50px; margin-left: 0.00px; margin-top: 0.00px; transform: rotate(0.00rad) translateZ(0px); -webkit-transform: rotate(0.00rad) translateZ(0px);" title="" src="https://www.twilio.com/blog/wp-content/uploads/2017/06/BYDvhRSNstCFMpuIrrcHM4PDwL3e_e_T5PaSf7uQ5FXQ7zMvPaLpqjb1d1jqeb7V7VpC5YzWeminPCESFecPqcvg-wm8noTsZwgI9L0mHnC0Syh7DYCFfVCfVETGZW-yurG6vhDY.png" alt="GIF of various emojis flying by"></p>

           <p>Now this was obviously just a very simple module to show you how easy it is to publish a module usable in both Javascript and TypeScript.</p>
