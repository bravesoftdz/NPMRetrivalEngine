          <p>Squarespace helps you build a site without coding or design expertise. When you add a <a href="/hc/en-us/articles/206543757">block</a>, you're actually adding HTML to your site without writing the code yourself. Similarly, when you make changes in the&nbsp;<a href="/hc/en-us/articles/205815788">Style Editor</a>, you're automatically making changes to your site's CSS. The Squarespace platform is powerful and flexible, and you can solve most design challenges with the Style Editor or by switching to a <a href="/hc/en-us/articles/205815598">different template</a>.</p> 
          <p>To further customize your site’s style and enhance specific parts of your site, you can use custom code. We highly recommend that you have coding knowledge before adding custom code. Some coding changes will conflict with your template's default code and may cause issues in the future as our developers make ongoing improvements to the platform.</p> 
          <blockquote>
           <strong>Note:</strong> Adding code to your site is an advanced modification that falls outside of the scope of Squarespace support.
          </blockquote> 
          <h2>Adding code</h2> 
          <p>There are several methods for adding HTML, CSS, and JavaScript to a Squarespace site:</p> 
          <ul> 
           <li><a href="/hc/en-us/articles/206543167">Code Block</a> - Add&nbsp;code within a page’s layout. For example, you might use a Code Block to embed a video or add a widget to a Regular Page.</li> 
           <li><a href="/hc/en-us/articles/205815908">Site-wide Code Injection</a> - Add site-wide code by injecting it in your site’s header or footer.</li> 
           <li><a href="/hc/en-us/articles/205815908#toc-per-page-code-injection">Per-page Code Injection</a> - Add&nbsp;code that affects&nbsp;a specific page.</li> 
           <li><a href="/hc/en-us/articles/206545567">Custom CSS Editor</a> - Modify fonts, colors, and backgrounds. CSS shouldn’t be used for layout changes such as padding or positioning.</li> 
           <li><a href="/hc/en-us/articles/206543617">Embed Block</a> - Embed content from external sites that use the oEmbed standard.</li> 
          </ul> 
          <p>These are the recommended ways for inserting custom code:</p> 
          <ul> 
           <li>Adding HTML - <a href="/hc/en-us/articles/206543167">Code Block</a></li> 
           <li>Making CSS style changes - <a href="/hc/en-us/articles/206545567">CSS Editor</a></li> 
           <li>Inserting JavaScript - <a href="/hc/en-us/articles/205815908">Code Injection</a></li> 
           <li>Displaying raw code as plain text: Code Block with “Display Source” checked</li> 
          </ul> 
          <p>Watch&nbsp;the following&nbsp;video for more examples of using code on your site. This video is an excerpt from our <a href="/hc/en-us/articles/206055358">Next Steps with Squarespace Series</a>.</p> 
          <div class="wistia_responsive_padding" style="padding: 56.25% 0 0 0; position: relative;">

          <h3>Adding CSS or JavaScript to Code Blocks</h3> 
          <p>The Code Block and Code Injection only accept HTML, but you can add CSS or JavaScript by surrounding the code with these HTML tags:</p> 
          <ul> 
           <li><strong>CSS</strong> -&nbsp;&lt;style&gt;&lt;/style&gt;</li> 
           <li><strong>JavaScript</strong> -&nbsp;&lt;script&gt;&lt;/script&gt;</li> 
          </ul> 
          <h2>Developer Mode</h2> 
          <p>If you’re an advanced user, the <a href="http://developers.squarespace.com/">Squarespace Developer Platform</a> gives complete access to the underlying code of your site's template. This platform was designed for developers and design agencies to&nbsp;modify our existing templates and even create templates from scratch.<strong><strong>&nbsp;</strong></strong></p> 
          <p>The Developer Platform is best for advanced developers, typically those who create custom websites for others. Note that Code Injection changes you make while on the Developer Platform will disappear if you ever disable Developer Mode.</p>
