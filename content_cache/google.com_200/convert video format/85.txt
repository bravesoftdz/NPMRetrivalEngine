       <h1>Welcome to Zed</h1> 
       <h3>What does it look like?</h3> 
       <p><img src="/img/zed-screen-new.png" alt="zed-screen-new" width="1221" height="992" class="aligncenter size-full"></p> 
       <h3>What is it?</h3> 
       <p>Zed is a fully offline-capable, open source, keyboard-focused, text and code editor for power users. You can use Zed to edit <em>local files</em> as well as <em>remote files</em> on any server. Zed has all features you'd expect from a capable code editor:</p> 
       <ul> 
        <li>Syntax highlighting for many programming languages (e.g. C, Clojure, CoffeeScript, C#, CSS, Dart, Erlang, Go, Haml, Haskell, HTML, ini files, Java, JavaScript, JSON, LogiQL, Lua, Markdown, Nix, PHP, Plist, Protobufs, Python, Ruby, Shell, XML)</li> 
        <li>Code completion: symbols, snippets and property/method completion.</li> 
        <li>Built-in linting for some languages with inline markers (JavaScript, CoffeeScript, JSON, Lua, CSS)</li> 
        <li>Multiple cursors</li> 
        <li>Split-view editing</li> 
        <li>Themes: light and dark themes out of the box and you can easily develop your own using CSS.</li> 
       </ul> 
       <iframe width="560" height="315" src="//www.youtube.com/embed/Rb8oQ5XHoLA" frameborder="0" allowfullscreen></iframe> 
       <p>Zed is distributed in two editions: <a href="/download">a <em>Chrome App</em> and a standalone app</a>.</p> 
       <h3>Why <em>another</em> editor?</h3> 
       <p>Zed was not designed to be an <em>X</em> clone (although it borrows ideas from many). It has opinionated ideas on how to make <em>you</em> more productive editing code. Specifically:</p> 
       <ol> 
        <li>Zed tries to reduce <a href="http://en.wikipedia.org/wiki/Cognitive_load">cognitive load</a> while editing code by <em>simplifying</em> as much as possible: 
         <ul>
          <li>Zed has <em>no</em> tabs and no concept of open files. You navigate between files either using the goto UI (<code>Command-E</code>/<code>Ctrl-E</code>) or file tree (<code>Command-T</code>/<code>Ctrl-T</code>). Switching between recent file is easy because recently opened files appear at the top.</li> 
          <li>Zed has as little window chrome as reasonably possible: editors take up the full window space, minus a small bar along the top indicating what file is open.</li> 
          <li>Zed <em>unifies</em> file navigation and file creation in its goto UI: you can either navigate to an existing file, or to a non existing file, in which case it will create the file including its parent directories (if necessary). As Zed will create any directories as required, you can just treat the path as a namespace.</li> 
          <li>Zed has <a href="/features/edit-remote-files/">first-class support for editing files on any Internet-connected server</a>, so you no longer have to switch to nano or vi when editing files remotely, and swap in all the associated keybindings into your brain.</li> 
          <li>Zed persists edit state between sessions: window size and location, recent commands, split view state, cursor and selections and even undo history are persisted between sessions in a <code>.zedstate</code> file stored in the project directory. This gives you peace of mind: not working on a project <em>right now</em>? Just close the window. Don't worry.</li>
         </ul></li> 
        <li>Zed is built 100% using web technologies (HTML, JavaScript and pure CSS). Because it is <a href="https://github.com/zedapp/zed">open source</a> (MIT licensed), you can inspect the DOM and change the code. Alternatively, you can write <a href="http://zedapp.org/2014/05/zed-package-manager/">extensions</a> (create new language modes, custom commands, key bindings, themes etc.) from within the editor without having to touch the Zed codebase at all.</li> 
        <li><strong>Chrome App version bonus:</strong> the Chrome App version of Zed has some extra useful features: 
         <ul>
          <li>Cross platform: Zed runs on any platforms that desktop Chrome runs on (including Chromebooks)</li> 
          <li>Installation is easy with just a few clicks</li> 
          <li>Settings are automatically synchronized between all computers logged into with the same Google account.</li>
         </ul></li> 
       </ol> 
       <p><a href="/vision">Click here to learn more about why we created Zed</a>.</p>
