       <h2 id="intro" class="step-title intro-title">Introduction: How to Convert Any Media File Into Its Different Formats</h2>
       <div class="photoset">
        <img alt="Picture of How to Convert Any Media File Into Its Different Formats" src="https://cdn.instructables.com/F9W/4Z9J/GA1BGF5F/F9W4Z9JGA1BGF5F.MEDIUM.jpg">
       </div>
       <div class="step-body">
        <p>There are different media file converters which we can use.&nbsp;<br><br> On the web, my favorite online media converter is:&nbsp; <a href="http://www.mediaconverter.org">http://www.mediaconverter.org</a><br><br> In this simple tutorial, we will use the “Format Factory” which is an amazing universal media file converter.<br><br> You can download it at:&nbsp; <a href="http://www.formatoz.com">http://www.formatoz.com</a><br><br> The above website introduce its downloadable software with the following words:<br><br><em>"Format Factory is a multifunctional media converter.<br> Provides functions below:<br> All to MP4/3GP/MPG/AVI/WMV/FLV/SWF.<br> All to MP3/WMA/AMR/OGG/AAC/WAV.<br> All to JPG/BMP/PNG/TIF/ICO/GIF/TGA.<br> Rip DVD to video file , Rip Music CD to audio file.<br> MP4 files support iPod/iPhone/PSP/BlackBerry format.<br> Supports RMVB,Watermark, AV Mux.<br><br> Format Factory's Feature:<br> 1 support converting all popular video,audio,picture formats to others.<br> 2 Repair damaged video and audio file.<br> 3 Reducing Multimedia file size.<br> 4 Support iphone,ipod multimedia file formats.<br> 5 Picture converting supports Zoom,Rotate/Flip,tags.<br> 6 DVD Ripper.<br> 7 Supports 56 languages"</em><br><br> The good news?<br><br> Format Factory is absolutely free!!!<br><br> Amazing! &nbsp;Isn't it?<br><br> Now, you are ready to see an actual example of how to use FORMAT FACTORY in media conversion.<br><br> In this instructable, we will convert AMR audio type into MP3 format. &nbsp;<br><br> Of course, you can read the help menu of the Format Factory for further instruction. &nbsp;<br><br> It is user-friendly and intuitive.<br><br> This instructable will serve as a pattern for you so that you can do the same to convert any media file you want to different formats.<br><br> Let's now start our media file conversion...<br><br><br> Periander "the Seventh Sage" Esplana<br><a href="http://thebibleformula.webs.com">http://thebibleformula.webs.com</a><br><a href="http://www.internetsecretbook.com">http://www.internetsecretbook.com</a><br><br><br></p>
       </div>
      </section>
      <section class="step-container">
       <h2 id="step1" class="step-title">Step 1: Open the Format Factory</h2>
       <div class="photoset">
        <img alt="Picture of Open the Format Factory" src="https://cdn.instructables.com/F8K/WHEI/GA1BGF5Z/F8KWHEIGA1BGF5Z.MEDIUM.jpg">
       </div>
       <div class="step-body">
        <p>As you open the Format Factory, you'll notice that the video category is by default open in the menu.<br><br> Since, as an example, we want to convert AMR audio file into MP3 format, we will move to the next step...</p>
       </div>
      </section>
      <section class="step-container">
       <h2 id="step2" class="step-title">Step 2: Select From the Menu the Media File Type Which You Will Convert</h2>
       <div class="photoset">
        <img alt="Picture of Select From the Menu the Media File Type Which You Will Convert" src="https://cdn.instructables.com/FS2/XVS8/GA1BGF77/FS2XVS8GA1BGF77.MEDIUM.jpg">
       </div>
       <div class="step-body">
        <p>We will convert AMR audio file into MP3 format, so we have to select "Audio" from the menu which can be found at the left side of Format Factory, then select "All to MP3."<br><br><br></p>
       </div>
      </section>
      <section class="step-container">
       <h2 id="step3" class="step-title">Step 3: Click on the "Add File" Button</h2>
       <div class="photoset">
        <img alt="Picture of Click on the &quot;Add File&quot; Button" src="https://cdn.instructables.com/FEM/GPDF/GA1BGF7O/FEMGPDFGA1BGF7O.MEDIUM.jpg">
       </div>
       <div class="step-body">
        <p>After you click on the "All to MP3", another popup will appear, just click the "Add File" button. &nbsp;That is, you have to get the file which you will convert via Format Factory</p>
       </div>
      </section>
      <section class="step-container">
       <h2 id="step4" class="step-title">Step 4: Select Your Media File to Be Converted</h2>
       <div class="photoset">
        <img alt="Picture of Select Your Media File to Be Converted" src="https://cdn.instructables.com/F85/CP12/GA1BGF7Z/F85CP12GA1BGF7Z.MEDIUM.jpg">
       </div>
       <div class="step-body">
        <p>From a folder of your computer, you have to select the media file which you will convert, in this instructable we will choose the AMR audio file and then click "open"</p>
       </div>
      </section>
      <section class="step-container">
       <h2 id="step5" class="step-title">Step 5: Click the "OK" Button and Then the "Start" Button</h2>
       <div class="photoset">
        <img alt="Picture of Click the &quot;OK&quot; Button and Then the &quot;Start&quot; Button" src="https://cdn.instructables.com/FY6/7N3W/GA1BGF8E/FY67N3WGA1BGF8E.MEDIUM.jpg">
       </div>
       <div class="step-body">
        <p>On the Format Factory popup you will just click the "OK" button, unless of course you want to add another file. &nbsp;<br><br> In this instructable, just click the "OK" button. &nbsp;<br><br> The popup then will close and you will be backed to the main menu of Format Factory. &nbsp;<br><br> At the top menu, select the "Start" button to start the media file conversion process</p>
       </div>
      </section>
      <section class="step-container">
       <h2 id="step6" class="step-title">Step 6: Wait the Media File Conversion Process to Complete 100%</h2>
       <div class="photoset">
        <img alt="Picture of Wait the Media File Conversion Process to Complete 100%" src="https://cdn.instructables.com/FNS/NNY7/GA1BGF9K/FNSNNY7GA1BGF9K.MEDIUM.jpg">
       </div>
       <div class="step-body">
        <p>Depending upon the size of the file which you will convert, that will be time of the media file conversion process. &nbsp;<br><br> In this instructable, it just took ten seconds (10 sec) to convert AMR audio file into MP3 format. &nbsp;<br><br> After you saw the 0% became 100% or "Completed," you are now finished with your media file conversion. &nbsp;<br><br> To see your converted file, just click the "Output Folder" at the top menu where your new converted file can be found. &nbsp;<br><br> That's it!<br><br> Life is still good to those who lived it right...<br></p>
