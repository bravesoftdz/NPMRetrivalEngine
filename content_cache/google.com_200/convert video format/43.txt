   <h1 class="doc_title">How to quickly convert MKV to MP4 file using VLC?</h1> 
   <div class="content_inner" style="padding-bottom:5px;padding-top:5px;"> 
    <p class="news_title"><a href="news_1615_how_to_quickly_convert_mkv_to_mp4_file_using_vlc.html">#</a> How to quickly convert MKV to MP4 file using VLC?</p> 
    <p class="news_date"><span class="time">21:17</span> <br><span class="weekday">Fri</span> <br><span class="day">22</span> <br><span class="month">Jan</span> <span class="year">2016</span> </p> 
    <p>Do you have a video in MKV file and you can't open it because some program (like Sony Vegas Pro) doesn't support this format? If so, you probably wonder how to convert it into some different format. I just discovered a way to do this.</p> 
    <p>To understand this method, first you need to know that media file formats are just containers (for example, MKV is <a href="https://en.wikipedia.org/wiki/Matroska">Matroska</a>). Each format encapsulates a set of streams, usually one video and one audio stream. Now, each stream is encoded using some specific codec. There may be various codecs used, but for the file I needed to convert, Media Player Classic (my favorite movie player, installed with K-Lite Codec Pack) shows following information after selecting File &gt; Properties:</p> 
    <blockquote> 
     <p>Type: Matroska<br>Video: MPEG4 Video (H264) 720x400 25fps [V: English [eng] (h264 main L4.0, yuv420p, 720x400) [default]]<br>Audio: AAC 44100Hz stereo [A: aac lc, 44100 Hz, stereo [default]]</p> 
    </blockquote> 
    <p>MPEG4 Video is the same codec that may be used with MP4 file format! It means we could convert ("repack") the file to just different container format, rewriting streams as-is without actually converting video or audio - which should be very fast (converting a long movie takes only few seconds) and wouldn't cause any quality loss.</p> 
    <p>To do that, I used <a href="http://www.videolan.org/vlc/"><strong>VLC media player</strong></a>. This program has its own set of codecs for many video and audio formats, so it doesn't depend on codecs installed in Windows. The player is actually just an overlay on top of a powerful library that can also do different things, like streaming video over network (that's probably where the company name "VideoLAN" comes from) or convert files.</p> 
    <p>So to convert an MKV file to MP4:</p> 
    <ol> 
     <li>Run "VLC media player".</li> 
     <li>From main menu select "Media" &gt; "Convert / Save...".</li> 
     <li>On "File" tab, click "Add..." button.</li> 
     <li>Choose you MKV file. After that, it should be added to the list. (Alternatively, you can drag&amp;drop a file into the list.)</li> 
     <li>Click "Convert/Save" button.</li> 
     <li>Select Profile: "Video - H.264 + MP3 (MP4)".</li> 
     <li>Click on the icon button that has hint: "Edit selected profile".</li> 
     <li>On "Encapsulation" tab, leave "MP4/MOV" selected.</li> 
     <li>On "Video codec" tab, check both "Video" and "Keep original video track".</li> 
     <li>On "Audio codec" tab, similarly select both "Audio" and "Keep original audio track".</li> 
     <li>Click "Save" button.</li> 
     <li>In "Destination" selection, click "Browse" button to select directory and file name for you destination MP4 file.</li> 
     <li>Click "Start" button.</li> 
     <li>Observe progress bar in the main window as file is converted.</li> 
     <li>After that, my VLC (version 2.2.1) seems to hang in an infinite loop so I have to kill it using system "Task Manager", but it doesn't matter - the destination file is already created.</li> 
    </ol>
