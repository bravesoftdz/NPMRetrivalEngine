           <p>There are many file types that can preview on dropbox.com. We update this article when we start supporting new file types, so check back occasionally to see the latest.</p> 
           <h2>Preview a file</h2> 
           <p>You can preview a supported file on dropbox.com in two ways:</p> 
           <ul> 
            <li>Click the name of the file.</li> 
            <li>Click the checkbox next to the file. Then click the viewer icon (an eye) that appears.</li> 
           </ul> 
           <p><b>Note:&nbsp;</b>It isn't possible to edit a file from its preview on dropbox.com. Instead, you can download the <a href="https://www.dropbox.com/downloading" target="_blank">Dropbox desktop app</a> to quickly <a href="/help/desktop-web/open-button" target="_blank">open files on your computer and edit them</a>. If you want to edit a Microsoft Office file, try our <a href="/help/desktop-web/microsoft-office-faq" target="_blank">Office Online integration</a>.</p>

            <h2>File preview errors</h2> 
            <p>Files may not be viewable for the following reasons:</p> 
            <ul> 
             <li>File is too large; consider the following file size maximums for popular file types
              <ul> 
               <li>Word: 160 MB</li> 
               <li>Powerpoint: 288 MB</li> 
               <li>Excel: 40 MB</li> 
               <li>PDF: 80 MB</li> 
               <li>Image files: 50 MB or 50 megapixels</li> 
               <li>Photoshop and .tiff files: 200 MB</li> 
               <li>Text and code files: 4 MB</li> 
               <li>Video files: 18 GB (for the video thumbnail; the video preview itself does not have a size limit)</li> 
              </ul> </li> 
             <li>File is corrupt</li> 
             <li>File is password protected</li> 
             <li>File has the wrong extension name</li> 
             <li>File is DRM-protected (video files)</li> 
             <li>File is not supported by your browser (audio files)</li> 
             <li>This file can’t be previewed. Previews are only available for files created in Sketch v43 and up.<br> </li> 
            </ul> 
            <p>Having problems loading a preview, or seeing an error message? <a href="/help/files-folders/file-preview-errors" target="_blank">Learn more about file-preview error messages, what they mean and how to address them.</a></p> 
           </div> 
           <h2>Learn more</h2> 
           <p>To preview files on Dropbox using Chrome, make sure that Chrome is not set to <b>Block third-party cookies and site data</b>. You can adjust this setting on Chrome's Content settings page under Cookies.</p> 
           <p><a href="https://support.google.com/chrome/answer/114662" target="_blank">Learn more about Chrome's Content Settings</a>.</p> 
           <p>Special considerations for image and video files:</p> 
           <ul> 
            <li>For .psd files, we recommend saving your files with the preference to maximize compatibility.</li> 
            <li>Video codec created via GoToMeeting can be previewed.</li> 
            <li>When you save a video in your Dropbox you will be able to watch the full video from a preview page.
             <ul> 
              <li>If a Dropbox Business user creates a shared link to a video, the recipient can view the entire video on a preview page. This does not apply to videos created by users on a Dropbox Business trial.</li> 
              <li>If a Dropbox Plus or Business trial user creates a shared link to a video, the recipient can view up to 15 minutes on a preview page. Recipients can download the file to their computer or add it to their own Dropbox account to view the entire video.</li> 
              <li>If a Dropbox Basic user creates a shared link to a video, the recipient can only view up to 15 minutes on a preview page. Recipients can download the file to their computer or add it to their own Dropbox account to view the entire video.</li> 
             </ul> </li> 
           </ul>
