     <h1 class="text-center singlepost_title_singleview ">List of 25+ Natural Language Processing APIs</h1> 
     <h2 class="text-center">by <span class="singlepost_meta_highlight"><a href="http://blog.mashape.com?s=&amp;author_name=chrisismael">Chris Ismael</a></span> on April 26, 2013</h2>

   <div class="col-md-8 col-md-offset-2 singlepost_post_content"> 
    <div id="google_translate_element">
     <a href="http://i1.wp.com/blog.mashape.com/wp-content/uploads/2013/04/tumblr_inline_mlvnf1gjAY1qz4rgp.jpg"><img class=" aligncenter" src="http://i1.wp.com/blog.mashape.com/wp-content/uploads/2013/04/tumblr_inline_mlvnf1gjAY1qz4rgp.jpg?resize=184%2C184" alt="Natural Language Processing API" data-recalc-dims="1"></a>
    </div> 
    <p><strong>Note: Check out our latest <a title="NLP API Collections" href="https://www.mashape.com/api-collections/nlp" target="_blank">API collections page</a> for the list of updated APIs.</strong></p> 
    <div>
     Natural Language Processing, or NLP, is a field of computer science, artificial intelligence, and linguistics concerned with the interactions between computers and human (natural) languages.
    </div> 
    <div> 
     <p>Here are useful APIs that help bridge the human-computer interaction:</p> 
    </div>

       <tr> 
        <th scope="row">1</th> 
        <td>Stremor Automated Summary and Abstract Generator</td> 
        <td class="asd">Language Heuristics goes a step beyond Natural Language Processing to extract intent from text. Summaries are created through extraction, but maintain readability by keeping sentence dependencies intact.<br> <a href="https://www.mashape.com/stremor/stremor-automated-summary-and-abstract-generator#!documentation"><img src="https://d1g84eaw0qjo7s.cloudfront.net/images/badges/badge-icon-light-9e8eba63.png" alt="Mashape" width="143" height="38"></a></td> 
       </tr> 
       <tr> 
        <th scope="row">2</th> 
        <td>Text-Processing</td> 
        <td class="asd">Sentiment analysis, stemming and lemmatization, part-of-speech tagging and chunking, phrase extraction and named entity recognition.<br> <a href="https://www.mashape.com/japerk/text-processing#!documentation"><img src="https://d1g84eaw0qjo7s.cloudfront.net/images/badges/badge-icon-light-9e8eba63.png" alt="Mashape" width="143" height="38"></a></td> 
       </tr> 
       <tr> 
        <th scope="row">3</th> 
        <td>Skyttle 2.0</td> 
        <td class="asd">Skyttle API extracts topical keywords (single words and multiword expressions) and sentiment (positive or negative) expressed in text. Languages supported are English, French, German, Russian.<br> <a href="https://www.mashape.com/sentinelprojects/skyttle2-0#!documentation"><img src="https://d1g84eaw0qjo7s.cloudfront.net/images/badges/badge-icon-light-9e8eba63.png" alt="Mashape" width="143" height="38"></a></td> 
       </tr> 
       <tr> 
        <th scope="row">4</th> 
        <td>Textuality</td> 
        <td class="asd">The service lifts the key text from an html page. Get up an started with summarizer app in few steps. The Textuality API from Saaskit finds out the most relevant piece of information on web-pages.<br> <a href="https://www.mashape.com/djinn/textuality#!documentation"><img src="https://d1g84eaw0qjo7s.cloudfront.net/images/badges/badge-icon-light-9e8eba63.png" alt="Mashape" width="143" height="38"></a></td> 
       </tr> 
       <tr> 
        <th scope="row">5</th> 
        <td>Text Processing</td> 
        <td class="asd">The WebKnox text processing API lets you process (natural) language texts. You can detect the textâs language, the quality of the writing, find entity mentions, tag part-of-speech, extract dates, extract locations, or determine the sentiment of the text.<br> <a href="https://www.mashape.com/webknox/text-processing-1#!documentation"><img src="https://d1g84eaw0qjo7s.cloudfront.net/images/badges/badge-icon-light-9e8eba63.png" alt="Mashape" width="143" height="38"></a></td> 
       </tr> 
       <tr> 
        <th scope="row">6</th> 
        <td>Question-Answering</td> 
        <td class="asd">The WebKnox question-answering API allows you to find answers to natural language questions. These questions can be factual such as What is the capital of Australia or more complex.<br> <a href="https://www.mashape.com/webknox/question-answering#!documentation"><img src="https://d1g84eaw0qjo7s.cloudfront.net/images/badges/badge-icon-light-9e8eba63.png" alt="Mashape" width="143" height="38"></a></td> 
       </tr> 
       <tr> 
        <th scope="row">7</th> 
        <td>Jeannie</td> 
        <td class="asd">Jeannie (Voice Actions) is a virtual assistant with over two Million downloads, now also available via API. The objective of this service is to provide you and your robot with the smartest answer to any natural language question, just like Siri.<br> <a href="https://www.mashape.com/pannous/jeannie#!documentation"><img src="https://d1g84eaw0qjo7s.cloudfront.net/images/badges/badge-icon-light-9e8eba63.png" alt="Mashape" width="143" height="38"></a></td> 
       </tr> 
       <tr> 
        <th scope="row">8</th> 
        <td>Diffbot</td> 
        <td class="asd">Diffbot extracts data from web pages automatically and returns structured JSON. For example, our Article API returns an articles title, author, date and full-text. Use the web as your database! We use computer vision, machine learning and natural language processing to add structure to just about any web page.<br> <a href="https://www.mashape.com/diffbot/diffbot-1#!documentation"><img src="https://d1g84eaw0qjo7s.cloudfront.net/images/badges/badge-icon-light-9e8eba63.png" alt="Mashape" width="143" height="38"></a></td> 
       </tr> 
       <tr> 
        <th scope="row">9</th> 
        <td>nlpTools</td> 
        <td class="asd">Text processing framework to analyse Natural Language. It is especially focused on text classification and sentiment analysis of online news media (general-purpose, multiple topics).<br> <a href="https://www.mashape.com/atrilla/nlptools#!documentation"><img src="https://d1g84eaw0qjo7s.cloudfront.net/images/badges/badge-icon-light-9e8eba63.png" alt="Mashape" width="143" height="38"></a></td> 
       </tr> 
       <tr> 
        <th scope="row">10</th> 
        <td>Speech2Topics</td> 
        <td class="asd">Yactraq Speech2Topics is a cloud service that converts audiovisual content into topic metadata via speech recognition &amp; natural language processing. Customers use Yactraq metadata to target ads, build UX features like content search/discovery and mine Youtube videos for brand sentiment.<br> <a href="https://www.mashape.com/yactraq/speech2topics#!documentation"><img src="https://d1g84eaw0qjo7s.cloudfront.net/images/badges/badge-icon-light-9e8eba63.png" alt="Mashape" width="143" height="38"></a></td> 
       </tr> 
       <tr> 
        <th scope="row">11</th> 
        <td>Stemmer</td> 
        <td class="asd"> This API takes a paragraph and returns the text with each word stemmed using porter stemmer, snowball stemmer or UEA stemmer<br> <a href="https://www.mashape.com/enclout/stemmer#!documentation"><img src="https://d1g84eaw0qjo7s.cloudfront.net/images/badges/badge-icon-light-9e8eba63.png" alt="Mashape" width="143" height="38"></a></td> 
       </tr> 
       <tr> 
        <th scope="row">12</th> 
        <td>LanguageTool</td> 
        <td class="asd">Style and grammar checking / proofreading for more than 25 languages, including English, French, Polish, Spanish and German.<br> <a href="https://www.mashape.com/dnaber/languagetool#!documentation"><img src="https://d1g84eaw0qjo7s.cloudfront.net/images/badges/badge-icon-light-9e8eba63.png" alt="Mashape" width="143" height="38"></a></td> 
       </tr> 
       <tr> 
        <th scope="row">13</th> 
        <td>DuckDuckGo</td> 
        <td class="asd">DuckDuckGo Zero-click Info includes topic summaries, categories, disambiguation, official sites, !bang redirects, definitions and more. You can use this API for many things, e.g. define people, places, things, words and concepts; provides direct links to other services (via !bang syntax); list related topics; and gives official sites when available<br> <a href="https://www.mashape.com/duckduckgo/duckduckgo-zero-click-info#!documentation"><img src="https://d1g84eaw0qjo7s.cloudfront.net/images/badges/badge-icon-light-9e8eba63.png" alt="Mashape" width="143" height="38"></a></td> 
       </tr> 
       <tr> 
        <th scope="row">14</th> 
        <td>ESA Semantic Relatedness</td> 
        <td class="asd">Calculates the semantic relatedness between pairs of text excerpts based on the likeness of their meaning or semantic content.<br> <a href="https://www.mashape.com/amtera/esa-semantic-relatedness#!documentation"><img src="https://d1g84eaw0qjo7s.cloudfront.net/images/badges/badge-icon-light-9e8eba63.png" alt="Mashape" width="143" height="38"></a></td> 
       </tr> 
       <tr> 
        <th scope="row">15</th> 
        <td>AlchemyAPI</td> 
        <td class="asd">AlchemyAPI provides advanced cloud-based and on-premise text analysis infrastructure that eliminates the expense and difficulty of integrating natural language processing systems into your application, service, or data processing pipeline.<br> <a href="https://www.mashape.com/alchemyapi/alchemyapi-1#!documentation"><img src="https://d1g84eaw0qjo7s.cloudfront.net/images/badges/badge-icon-light-9e8eba63.png" alt="Mashape" width="143" height="38"></a></td> 
       </tr> 
       <tr> 
        <th scope="row">16</th> 
        <td>Sentence Recognition</td> 
        <td class="asd">The Sentence Recognition API will match strings of text based off of the meaning of the sentences. Itâs powerful NLP engine offering utilizes a semantic network to understand the text presented.<br> <a href="https://www.mashape.com/mnutsch/sentence-recognition#!documentation"><img src="https://d1g84eaw0qjo7s.cloudfront.net/images/badges/badge-icon-light-9e8eba63.png" alt="Mashape" width="143" height="38"></a></td> 
       </tr> 
       <tr> 
        <th scope="row">17</th> 
        <td>Textalytics Media Analytics</td> 
        <td class="asd">Textalytics Media Analysis API analyzes mentions, topics, opinions and facts in all types of media. This API provides services for: Sentiment analysis Extracts positive and negative opinions according to the context. Entities extraction Identifies persons, companies, brands, products, etc. and provides a canonical form that unifies different mentions (IBM, International Business Machines Corporation, etc.) Topic and keyword extraction Facts and other key information Dates, URLs, addresses, user names, e-mails and money amounts. Thematic classification Organize information by topic using IPTC standard classification (more than 200 categories hierarchically structured). Configured for different type of media: microblogging and social networks, blogs and news<br> <a href="https://www.mashape.com/textalytics/textalytics-media-analysis#!documentation"><img src="https://d1g84eaw0qjo7s.cloudfront.net/images/badges/badge-icon-light-9e8eba63.png" alt="Mashape" width="143" height="38"></a></td> 
       </tr> 
       <tr> 
        <th scope="row">18</th> 
        <td>Machine Linking</td> 
        <td class="asd">Multilingual semantic analysis of text: developers can annotate unstructured documents and short pieces of text, and connect them to resources in the Linked Open Data cloud, such as DBPedia or Freebase. Other features include text comparison, summarization and language detection.<br> <a href="https://www.mashape.com/machinelinking/machine-linking#!documentation"><img src="https://d1g84eaw0qjo7s.cloudfront.net/images/badges/badge-icon-light-9e8eba63.png" alt="Mashape" width="143" height="38"></a></td> 
       </tr> 
       <tr> 
        <th scope="row">19</th> 
        <td>Textalytics Topics Extraction</td> 
        <td class="asd">Textalytics Topics Extraction tags locations, people, companies, dates and many other elements appearing in a text written in Spanish, English, French, Italian, Portuguese or Catalan. This detection process is carried out by combining a number of complex natural language processing techniques that allow to obtain morphological, syntactic and semantic analyses of a text and use them to identify different types of significant elements.<br> <a href="https://www.mashape.com/textalytics/textalytics-topics-extraction-1-1#!documentation"><img src="https://d1g84eaw0qjo7s.cloudfront.net/images/badges/badge-icon-light-9e8eba63.png" alt="Mashape" width="143" height="38"></a></td> 
       </tr> 
       <tr> 
        <th scope="row">20</th> 
        <td>Textalytics Spelling Grammar and Style Proofreading</td> 
        <td class="asd">A service for automatic proofreading of multilingual texts. This API uses multilingual Natural Language Processing technology to check the spelling, grammar and style of your texts with high accuracy, in order to provide precise and up-to-date suggestions and educational explanations based on references. The current supported languages are Spanish, English, French and Italian.<br> <a href="https://www.mashape.com/textalytics/textalytics-spelling-grammar-and-style-proofreading-1-1#!documentation"><img src="https://d1g84eaw0qjo7s.cloudfront.net/images/badges/badge-icon-light-9e8eba63.png" alt="Mashape" width="143" height="38"></a></td> 
       </tr> 
       <tr> 
        <th scope="row">21</th> 
        <td>Textalytics Text Classification</td> 
        <td class="asd">Automatic multilingual text classification according to pre-established categories defined in a model. The algorithm used combines statistic classification with rule-based filtering, which allows to obtain a high degree of precision for very different environments. Three models available: IPTC (International Press Telecommunications Council standard), EuroVocs and Corporate Reputation model. Languages covered are Spanish, English, French, Italian, Portuguese and Catalan<br> <a href="https://www.mashape.com/textalytics/textalytics-text-classification-1-1#!documentation"><img src="https://d1g84eaw0qjo7s.cloudfront.net/images/badges/badge-icon-light-9e8eba63.png" alt="Mashape" width="143" height="38"></a></td> 
       </tr> 
       <tr> 
        <th scope="row">22</th> 
        <td>Fluxifi NLP</td> 
        <td class="asd">Cloud based Natural Language Processing API. Includes Sentiment and Language Detection.<br> <a href="https://www.mashape.com/fluxifi/fluxifi-nlp-1#!documentation"><img src="https://d1g84eaw0qjo7s.cloudfront.net/images/badges/badge-icon-light-9e8eba63.png" alt="Mashape" width="143" height="38"></a></td> 
       </tr> 
       <tr> 
        <th scope="row">23</th> 
        <td>Cilenis</td> 
        <td class="asd">Cilenis helps you to analyze and extract information from texts. Add language technology to your software in a few minutes using their cloud solution. They offer you technology based on years of research in Natural Language Processing in a very easy and scalable SaaS model through a RESTful API.<br> <a href="https://www.mashape.com/cilenis/cilenisapi-natural-language-processing-in-the-cloud#!documentation"><img src="https://d1g84eaw0qjo7s.cloudfront.net/images/badges/badge-icon-light-9e8eba63.png" alt="Mashape" width="143" height="38"></a></td> 
       </tr>
