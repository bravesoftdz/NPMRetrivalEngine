         <div class="CBody">
          <h4 class="CHeading">Zopim Live Chat API</h4>
          <p>The Javascript API lets you customize various aspects of the Zopim Live Chat Widget.</p>
          <p>You can use a different color scheme on a special marketing page, set the widget language based on the user’s preferences and much more.</p>
          <h4 class="CHeading">Need more customizations?</h4>
          <p>If you are currently on the Chat Enterprise or Premium (Legacy) plan, you have the flexibility of fully customizing the entire chat widget by developing your own using the Zendesk Chat Web SDK.</p>
          <p>For more information, head to <a href="https://api.zopim.com/web-sdk" class="LURL" target="_top">https://api.zopim.com/web-sdk</a> for details.</p>
          <h4 class="CHeading">Usage</h4>
          <p>API calls must be inserted after the Live Chat Script and wrapped within <b>$zopim(function() { ...&nbsp; })</b></p>
          <h4 class="CHeading">Examples</h4>
          <blockquote>
           <pre>&lt;script&gt;

  $zopim(function() {
    $zopim.livechat.setName('Logged in name');
    $zopim.livechat.setEmail('user@somewhere.com');
  });

&lt;/script&gt;</pre>
          </blockquote>
          <p><b>Note</b>: Some API calls are unavailable for accounts using the Lite package.</p>
