           <h1 name="77b3" id="77b3" class="graf graf--h3 graf--leading graf--title">How to detect text language&nbsp;offline</h1>
           <figure name="2160" id="2160" class="graf graf--figure graf-after--h3">
            <div class="aspectRatioPlaceholder is-locked" style="max-width: 492px; max-height: 453px;">
             <div class="aspectRatioPlaceholder-fill" style="padding-bottom: 92.10000000000001%;"></div>
             <div class="progressiveMedia js-progressiveMedia graf-image" data-image-id="1*ME9Ia7-ZhAPRUSGsBY7iwQ.png" data-width="492" data-height="453">
              <img src="https://cdn-images-1.medium.com/freeze/max/60/1*ME9Ia7-ZhAPRUSGsBY7iwQ.png?q=20" crossorigin="anonymous" class="progressiveMedia-thumbnail js-progressiveMedia-thumbnail">
              <canvas class="progressiveMedia-canvas js-progressiveMedia-canvas"></canvas>
              <img class="progressiveMedia-image js-progressiveMedia-image" data-src="https://cdn-images-1.medium.com/max/1600/1*ME9Ia7-ZhAPRUSGsBY7iwQ.png">
              <noscript class="js-progressiveMedia-inner">
               <img class="progressiveMedia-noscript js-progressiveMedia-inner" src="https://cdn-images-1.medium.com/max/1600/1*ME9Ia7-ZhAPRUSGsBY7iwQ.png">
              </noscript>
             </div>
            </div>
           </figure>
           <p name="6ab5" id="6ab5" class="graf graf--p graf-after--figure">It’s easy to detect text language if you can use API calls to popular translate systems such as <a href="https://cloud.google.com/translate/docs" data-href="https://cloud.google.com/translate/docs" class="markup--anchor markup--p-anchor" rel="nofollow noopener" target="_blank">Google Translate API</a> or <a href="https://tech.yandex.com/translate/?ncrnd=5448" data-href="https://tech.yandex.com/translate/?ncrnd=5448" class="markup--anchor markup--p-anchor" rel="nofollow noopener" target="_blank">Yandex Translate API</a>. But what to do if you can’t?</p>
           <p name="a398" id="a398" class="graf graf--p graf-after--p">In AppFollow we detect languages for millions of reviews before translation, because some US customers write reviews on Spanish, Arabic or other languages. Or, e.g., our Spanish or French clients don’t need reviews in their native languages to be translated at all.</p>
           <p name="e21d" id="e21d" class="graf graf--p graf-after--p">We have two options:</p>
           <ul class="postList">
            <li name="cb84" id="cb84" class="graf graf--li graf-after--p">Translate all reviews through chosen API – and it will quickly eat all our limits so we start uselessly spend money on it.</li>
            <li name="eaa4" id="eaa4" class="graf graf--li graf-after--li">First detect review language, than if it’s not English translate it. As results we can use translate API more effectively and reduce number of API requests.</li>
           </ul>
           <p name="01fc" id="01fc" class="graf graf--p graf-after--li">If you search for a existing solution on the market, you will find some heavy products. Some of them depend on server’s Java machine, others actually don’t work that good.</p>
           <p name="ee50" id="ee50" class="graf graf--p graf-after--p">After we did some investigation, we stopped on open source solution <a href="https://code.google.com/p/cld2/" data-href="https://code.google.com/p/cld2/" class="markup--anchor markup--p-anchor" rel="nofollow noopener" target="_blank">Google Language Detector 2</a> [1]. We chose it, because it’s easy to install GLD2 to most modern Linux based systems [2] and popular programming languages (e.g. Node.js, Perl, Python) have wrapper to work with it. GLD2 supports 83 languages and detects them quickly and reliably.</p>
           <p name="393d" id="393d" class="graf graf--p graf-after--p">How it works? GLD2 is Naïve Bayesian classifier with lowercase scoring.</p>
           <p name="8863" id="8863" class="graf graf--p graf-after--p">As result it helps us to reduce number of requests to Translate API up to 90% in cases where we need to detect language before translation. If you detect language in different and more accurate way, please, leave a comment for me. And good luck with detecting languages in your own way&nbsp;;-)</p>
           <p name="15da" id="15da" class="graf graf--p graf-after--p graf--trailing">Resources:&nbsp;<br>1) GLD2 – <a href="https://code.google.com/p/cld2/" data-href="https://code.google.com/p/cld2/" class="markup--anchor markup--p-anchor" rel="nofollow noopener" target="_blank">https://code.google.com/p/cld2/</a><br>2) How to install GLD2 – <a href="http://blog.xanda.org/2014/04/02/installing-compact-language-detection-2-cld2-on-ubuntu/" data-href="http://blog.xanda.org/2014/04/02/installing-compact-language-detection-2-cld2-on-ubuntu/" class="markup--anchor markup--p-anchor" rel="nofollow noopener" target="_blank">http://blog.xanda.org/2014/04/02/installing-compact-language-detection-2-cld2-on-ubuntu/</a></p>
