        <h2 id="description-header">Description</h2> 
        <p>Loco Translate provides in-browser editing of WordPress translation files.</p> 
        <p>It also provides localization tools for developers, such as extracting strings and generating templates.</p> 
        <p>Features include:</p> 
        <ul> 
         <li>Built-in translation editor within WordPress admin</li> 
         <li>Create and update language files directly in your theme or plugin</li> 
         <li>Extraction of translatable strings from your source code</li> 
         <li>Native MO file compilation without the need for Gettext on your system</li> 
         <li>Support for PO features including comments, references and plural forms</li> 
         <li>PO source view with clickable source code references</li> 
         <li>Protected language directory for saving custom translations</li> 
         <li>Configurable PO file backups</li> 
         <li>Built-in WordPress locale codes</li> 
        </ul> 
        <p>Official <a href="https://localise.biz/" rel="nofollow">Loco</a> WordPress plugin by <a href="//twitter.com/timwhitlock" rel="nofollow">@timwhitlock</a> / <a href="https://plus.google.com/106703751121449519322" rel="author nofollow">Tim Whitlock</a></p> 
        <h3>More info</h3> 
        <ul> 
         <li><a href="https://localise.biz/wordpress/plugin/beginners" rel="nofollow">About the plugin</a></li> 
         <li><a href="https://localise.biz/wordpress/plugin/beginners" rel="nofollow">Beginner’s guide to translating a theme</a></li> 
         <li><a href="https://localise.biz/wordpress/plugin/overview" rel="nofollow">Technical overview</a></li> 
         <li><a href="https://localise.biz/wordpress/plugin/support" rel="nofollow">Getting help</a></li> 
        </ul> 
        <h3>Coming soon</h3> 
        <p>These features are on our todo list. There’s no particular timeframe for any of them and they’re in no particular order:</p> 
        <ul> 
         <li>Integration with Google and Bing for automatic translation</li> 
         <li>Integration with Loco API for collaborative translation</li> 
         <li>Screens showing installed bundles per language</li> 
        </ul> 
        <h3>Keyboard shortcuts</h3> 
        <p>The PO file editor supports the following keyboard shortcuts for faster translating:</p> 
        <ul> 
         <li>Done and Next: <code>Ctrl ?</code></li> 
         <li>Next string: <code>Ctrl ?</code></li> 
         <li>Previous string: <code>Ctrl ?</code></li> 
         <li>Next untranslated: <code>Shift Ctrl ?</code></li> 
         <li>Previous untranslated: <code>Shift Ctrl ?</code></li> 
         <li>Copy from source text: <code>Ctrl B</code></li> 
         <li>Clear translation: <code>Ctrl K</code></li> 
         <li>Toggle Fuzzy: <code>Ctrl U</code></li> 
         <li>Save PO / compile MO: <code>Ctrl S</code></li> 
         <li>Toggle invisibles: <code>Shift Ctrl I</code></li> 
        </ul> 
        <p>Mac users can use ? Cmd instead of Ctrl.</p>

       <div id="tab-installation" class="plugin-installation section "> 
        <h2 id="installation-header">Installation</h2> 
        <h4>Installing manually:</h4> 
        <ol> 
         <li>Unzip all files to the <code>/wp-content/plugins/loco-translate</code> directory</li> 
         <li>Log into WordPress admin and activate the ‘Loco Translate’ plugin through the ‘Plugins’ menu</li> 
         <li>Go to <em>Loco Translate &gt; Home</em> in the left-hand menu to start translating</li> 
        </ol> 
        <h4>Basic usage:</h4> 
        <p>Translators: To translate a theme into your language, follow these steps:</p> 
        <ol> 
         <li>Create the protected languages directory at <code>wp-content/languages/loco/themes</code></li> 
         <li>Ensure this directory writeable by the web server</li> 
         <li>Find the theme in the list at <em>Loco Translate &gt; Themes</em></li> 
         <li>Click <code>+ New language</code> and follow the on-screen prompts.</li> 
        </ol> 
        <p>Developers: To translate your own theme or plugin for distribution, follow these steps:</p> 
        <ol> 
         <li>Create a <code>languages</code> subdirectory in your bundle’s root directory</li> 
         <li>Ensure this directory writeable by the web server</li> 
         <li>Find the bundle at either <em>Loco Translate &gt; Themes</em> or <em>Loco Translate &gt; Plugins</em></li> 
         <li>Click <code>+ Create template</code> and follow the on-screen prompts to extract your strings.</li> 
         <li>Click <code>+ New language</code> and follow the on-screen prompts to add your own translations.</li> 
        </ol> 
        <p>More information on using the plugin is <a href="https://localise.biz/wordpress/plugin" rel="nofollow">available here</a>.</p>

          <p>Translators: To translate a theme into your language, follow these steps:</p> 
          <ol> 
           <li>Create the protected languages directory at <code>wp-content/languages/loco/themes</code></li> 
           <li>Ensure this directory writeable by the web server</li> 
           <li>Find the theme in the list at <em>Loco Translate &gt; Themes</em></li> 
           <li>Click <code>+ New language</code> and follow the on-screen prompts.</li> 
          </ol> 
          <p>Developers: To translate your own theme or plugin for distribution, follow these steps:</p> 
          <ol> 
           <li>Create a <code>languages</code> subdirectory in your bundle’s root directory</li> 
           <li>Ensure this directory writeable by the web server</li> 
           <li>Find the bundle at either <em>Loco Translate &gt; Themes</em> or <em>Loco Translate &gt; Plugins</em></li> 
           <li>Click <code>+ Create template</code> and follow the on-screen prompts to extract your strings.</li> 
           <li>Click <code>+ New language</code> and follow the on-screen prompts to add your own translations.</li> 
          </ol> 
          <p>More information on using the plugin is <a href="https://localise.biz/wordpress/plugin" rel="nofollow">available here</a>.</p> 
         </dd> 
         <dt>
          How do I use it?
         </dt> 
         <dd> 
          <p>Try our <a href="https://localise.biz/wordpress/plugin/beginners" rel="nofollow">beginner’s guide</a>, or the more <a href="https://localise.biz/wordpress/plugin/overview" rel="nofollow">technical overview</a> if you’re familiar with WordPress localization.</p> 
         </dd>
