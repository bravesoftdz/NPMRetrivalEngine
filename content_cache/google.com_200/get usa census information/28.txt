        <h1 class="entry-title">Data Visualization</h1> 
       </header> 
       <div class="entry-content"> 
        <p><a href="https://itunes.apple.com/us/course/data-literacy-data-visualization/id693097601"><img class="alignright size-full wp-image-605" src="http://www.braumoeller.info/wp-content/uploads/2012/12/d2_160.png" alt="d2_160" width="168" height="214"></a>Given the extent to which statistics classes alienate undergraduates without necessarily providing a tangible benefit, and given the increasing availability of data and open-source tools with which to analyze data, I’ve decided to focus my undergraduate methods teaching on data literacy, data visualization, and exploratory data analysis. The result is my iTunes U course, “<a href="https://itunes.apple.com/us/course/data-literacy-data-visualization/id693097601">Data Literacy and Data Visualization</a>” (which is <a href="https://www.youtube.com/playlist?list=PLE3uzGh7FZRl4UEkm9LQpy7lOv9H6ibjY">also available on YouTube</a>). Some of the most useful resources I’ve found while creating the course are listed below.</p> 
        <h4>Overviews</h4> 
        <div> 
         <ul> 
          <li>The <a href="http://dirt.projectbamboo.org/" target="_blank" rel="nofollow noopener">Bamboo Dirt project</a>, an aggregator of digital research tools from analysis to visualization</li> 
          <li>The <a href="http://www.wikiviz.org/wiki/Tools" target="_blank" rel="nofollow noopener">WikiViz Tools</a> site</li> 
          <li>Alberto Cairo’s thoughtfully curated list of readings, presentations, books and blogs over at <a href="http://www.thefunctionalart.com/2012/10/recommended-readings-for-infographics.html" target="_blank" rel="noopener">The Functional Art</a></li> 
          <li>The multifaceted dataviz site <a href="http://improving-visualisation.org/" target="_blank" rel="nofollow noopener">improving-visualisation.org</a> and in particular its <a href="http://www.improving-visualisation.org/links/" target="_blank" rel="nofollow noopener">page of links</a> to other sites</li> 
          <li>High-tech and gorgeous <a href="http://www.visualisingdata.com/" target="_blank" rel="nofollow noopener">Visualising Data</a>, with a month-end best-of-web roundup and a list of <a href="http://www.visualisingdata.com/index.php/resources/" target="_blank" rel="noopener">essential resources</a></li> 
          <li>Similar dataviz blogs at&nbsp;<a href="http://infosthetics.com/" target="_blank" rel="nofollow noopener">Information Aesthetics</a>&nbsp;and&nbsp;<a href="http://thewhyaxis.info/" target="_blank" rel="nofollow noopener">The Why Axis</a></li> 
          <li><a href="http://www.cc.gatech.edu/~stasko/7450/syllabus.html" target="_blank" rel="nofollow noopener">CS 7450, Information Visualization</a> from Georgia Tech</li> 
          <li>RWW’s <a href="http://www.readwriteweb.com/archives/the_best_tools_for_visualization.php" target="_blank" rel="nofollow noopener">Best Tools for Visualization</a> site</li> 
          <li><a href="http://dataviz.tools">dataviz.tools</a>, which contains a great variety of tools for data cleaning, scraping, charting, design, and more</li> 
          <li>The <a href="http://selection.datavisualization.ch/" target="_blank" rel="nofollow noopener">datavisualization.ch list</a> of selected tools (most require Javascript)</li> 
         </ul> 
        </div> 
        <h4>Design: How to Think About Visualization</h4> 
        <div style="font-weight: normal;"> 
         <ul> 
          <li>The <a href="http://www.visualcomplexity.com/vc/" target="_blank" rel="nofollow noopener">Visual Complexity</a> array of dataviz examples</li> 
          <li>A wide range of examples (many too complex, but all beautiful) at&nbsp;<a style="color: #000000!important; text-decoration: underline;" href="http://flowingdata.com/" rel="nofollow">FlowingData</a></li> 
          <li><a href="http://www.informationisbeautiful.net/" target="_blank" rel="nofollow noopener">Information is Beautiful</a>, a blog with a wide range of data visualizations</li> 
          <li>The <a href="http://spatialanalysis.co.uk/" target="_blank" rel="nofollow noopener">Spatial Analysis</a>&nbsp;blog with examples, news, and ideas</li> 
          <li>The mindboggling <a href="http://www.visual-literacy.org/periodic_table/periodic_table.html" target="_blank" rel="nofollow noopener">Periodic Table of Visualization Methods</a> (with mouseover illustrations)</li> 
          <li><a href="http://chartporn.org/2012/03/02/improving-on-minard/" target="_blank" rel="nofollow noopener">Variations</a> on Minard’s famous graph</li> 
          <li><a href="http://www.handsomeatlas.com">A Handsome Atlas</a>, a catalog of gorgeous 19th-century data visualizations</li> 
          <li>A <a href="http://www.r-bloggers.com/visualization-series-insight-from-cleveland-and-tufte-on-plotting-numeric-data-by-groups/" target="_blank" rel="nofollow noopener">tutorial</a> on visualizing numeric data by groups</li> 
          <li>A <a href="http://chartsnthings.tumblr.com/" target="_blank" rel="nofollow noopener">blog</a> about how the New York Times does its visualizations</li> 
          <li>Kaiser Fung’s useful “don’t let this happen to you” blog,&nbsp;<a href="http://junkcharts.typepad.com/" target="_blank" rel="nofollow noopener">Junk Charts</a>, and the charmingly horrifying&nbsp;<a href="http://viz.wtf" target="_blank" rel="noopener">WTF Visualizations</a></li> 
          <li>When good dataviz goes bad: infographics at&nbsp;<a href="http://www.good.is/departments/transparency/" target="_blank" rel="nofollow noopener">Good : Transparency</a></li> 
         </ul> 
        </div> 
        <h4>All-in-One Examples:&nbsp;Data + Visualization</h4> 
        <div style="font-weight: normal;"> 
         <ul> 
          <li>A real-time Twitter trends map,&nbsp;<a style="color: #000000!important; text-decoration: underline;" href="http://trendsmap.com/" rel="nofollow">Trendsmap</a></li> 
          <li>A searchable Twitter stream graph, <a href="http://www.neoformix.com/Projects/TwitterStreamGraphs/view.php">Twitter StreamGraph</a> (uses Java; might not work on all browsers)</li> 
          <li><a href="http://www.socialexplorer.com/pub/home/home.aspx">Social Explorer</a>&nbsp;(OSU login required), a portal for obtaining demographic data on the U.S.: Census, American Community Survey, etc. Also has lots of quick-and-easy (and very good) visualization options.</li> 
          <li><a href="http://www-958.ibm.com/software/data/cognos/manyeyes/" target="_blank" rel="nofollow noopener">IBM’s ManyEyes</a>,&nbsp;a general data-visualization engine with quite a few (iffy) user-supplied datasets</li> 
          <li>A Facebook social-network graphing application, <a href="http://apps.facebook.com/socgraph/" target="_blank" rel="nofollow noopener">Facebook Social Graph</a></li> 
          <li>A great real-time stacked-time-series generator,&nbsp;<a style="color: #000000!important; text-decoration: underline;" href="http://babynamewizard.com/voyager" rel="nofollow">NameVoyager, the baby name wizard</a></li> 
          <li>The <a href="http://www.nytimes.com//interactive/2009/07/31/business/20080801-metrics-graphic.html?hp" target="_blank" rel="nofollow noopener">New York Times visualization</a> of the <a href="http://www.bls.gov/tus/" target="_blank" rel="nofollow noopener">American Time Use Survey</a></li> 
          <li>Wolfram’s massively useful general knowledge engine, <a href="http://www.wolframalpha.com/" target="_blank" rel="nofollow noopener">Wolfram Alpha</a></li> 
          <li>To visualize members of Congress’ voting behavior (including data),&nbsp;<a style="color: #000000!important; text-decoration: underline;" href="http://manybills.researchlabs.ibm.com/congressperson/sherrod-brown" rel="nofollow">IBM’s Many Bills</a></li> 
          <li>Terrific visualization of weather data at <a href="http://weatherspark.com/#!graphs;a=USA/OH_43201/Columbus" target="_blank" rel="nofollow noopener">Weatherspark</a></li> 
          <li>A sobering graphic about <a href="http://theluxuryofprotest.com/Everyone_Ever_in_the_World.html" target="_blank" rel="nofollow noopener">deaths in war</a></li> 
          <li>A general data-visualization and mapmaking site with some built-in data, <a href="http://www.sacmeq.org/statplanet/" target="_blank" rel="nofollow noopener">StatPlanet</a></li> 
          <li>For global maps with weather data,&nbsp;<a style="color: #000000!important; text-decoration: underline;" href="http://kunden.dwd.de/GPCC/Visualizer" rel="nofollow">GPCC Visualizer</a></li> 
          <li>An interesting, if somewhat rigid,&nbsp;<a style="color: #000000!important; text-decoration: underline;" href="http://www.conflicthistory.com/#/period/1816-1913" rel="nofollow">conflict visualization</a>&nbsp;tool</li> 
          <li>The problems facing the American budget:&nbsp;<a style="color: #000000!important; text-decoration: underline;" href="http://www.washingtonpost.com/blogs/ezra-klein/post/our-budget-problem-in-two-charts-and-one-issue/2011/09/02/gIQAuMfBxJ_blog.html" rel="nofollow">graphs</a>&nbsp;–&nbsp;<a style="color: #000000!important; text-decoration: underline;" href="http://www.cbo.gov/doc.cfm?index=12316" rel="nofollow">data</a></li> 
          <li>Some nice opinion graphs illustrating epistemic closure and the effects of education:&nbsp;<a style="color: #000000!important; text-decoration: underline;" href="http://themonkeycage.org/blog/2011/03/10/you_want_more_epistemic_closur/" rel="nofollow">graphs</a></li> 
         </ul> 
        </div> 
        <h4>Visualization Tools</h4> 
        <div> 
         <ul> 
          <li><span style="font-family: arial, sans-serif;">An easy-to-use wordcloud generator,&nbsp;</span><a style="color: #000000!important; text-decoration: underline;" href="http://www.wordle.net/" target="_blank" rel="nofollow noopener">Wordle</a><span style="font-family: arial, sans-serif;">&nbsp;(and a </span><a style="font-family: arial, sans-serif;" href="http://www.niemanlab.org/2011/10/word-clouds-considered-harmful/" target="_blank" rel="nofollow noopener">thoughtful essay</a><span style="font-family: arial, sans-serif;"> on why wordclouds are bad, and </span><a style="font-family: arial, sans-serif;" href="http://www.nytimes.com/interactive/2011/06/10/education/commencement-speeches-graphic.html" target="_blank" rel="nofollow noopener">an example</a><span style="font-family: arial, sans-serif;"> of an alternative)</span></li> 
          <li><span style="font-family: arial, sans-serif;">A more advanced and style-forward wordcloud generator, </span><a style="font-family: arial, sans-serif;" href="http://www.tagxedo.com/" target="_blank" rel="nofollow noopener">Tagxedo</a></li> 
          <li><span style="font-family: arial, sans-serif;">The </span><a style="font-family: arial, sans-serif;" href="http://overview.ap.org/about/" target="_blank" rel="nofollow noopener">Overview project</a><span style="font-family: arial, sans-serif;"> for visualizing relationships in large numbers of documents, with </span><a style="font-family: arial, sans-serif;" href="http://overview.ap.org/blog/2010/12/a-full-text-visualization-of-the-iraq-war-logs/" target="_blank" rel="nofollow noopener">an example</a></li> 
          <li><span style="font-family: arial, sans-serif;">Google’s easy-to-use </span><a style="font-family: arial, sans-serif;" href="http://code.google.com/apis/chart/" target="_blank" rel="noopener">Google Chart Tools</a><span style="font-family: arial, sans-serif;"> API (helps to know HTML)</span></li> 
          <li><span style="font-family: arial, sans-serif;">A basic plug-and-play online chart builder, </span><a style="font-family: arial, sans-serif;" href="http://charts.hohli.com/#cht=p3&amp;chs=320x240&amp;chd=s:&amp;chf=bg,s,FFFFFF%7Cc,s,FFFFFF&amp;max=100&amp;agent=hohli.com" target="_blank" rel="nofollow noopener">Hohli</a></li> 
          <li><span style="font-family: arial, sans-serif;">The JavaScript-based </span><a style="font-family: arial, sans-serif;" href="http://www.jscharts.com/" target="_blank" rel="nofollow noopener">JSCharts</a><span style="font-family: arial, sans-serif;"> site, which allows easy construction of basic charts and graphs</span></li> 
          <li><span style="font-family: arial, sans-serif;">The glorious animated multicolored scatterplot engine, </span><a style="font-family: arial, sans-serif;" href="http://Gapminder.org" rel="nofollow">Gapminder.org</a></li> 
          <li><span style="font-family: arial, sans-serif;">Google’s similar and equally awesome&nbsp;</span><a style="font-family: arial, sans-serif;" href="http://www.google.com/publicdata/home" target="_blank" rel="noopener">Public Data Explorer</a></li> 
          <li><span style="font-family: arial, sans-serif;">A general data-visualization engine, </span><a style="font-family: arial, sans-serif;" href="http://www-958.ibm.com/software/data/cognos/manyeyes/" target="_blank" rel="nofollow noopener">IBM’s ManyEyes</a>&nbsp;(update: RIP!)</li> 
          <li>The extremely easy-to-use and attractive <a href="http://datawrapper.de">Datawrapper</a> reactive-chart website</li> 
          <li><a href="http://raw.densitydesign.org">Raw</a>, a straightforward and gorgeous way to turn spreadsheet data into vector graphics</li> 
          <li><a href="https://infogr.am">Infogram</a>, an online chart and graph maker. Free version has 30 different kinds of graphs; paid version has more features, including live update from JSON or Google Drive</li> 
          <li><a href="http://idl.cs.washington.edu/projects/lyra/">Lyra</a>, a <em>really</em> slick interactive visualization design environment (coming soon)</li> 
          <li><a href="https://plot.ly">Plotly</a>, a collaborative online visualization and data analysis tool with some handy <a href="https://plot.ly/api/">APIs</a></li> 
          <li><span style="font-family: arial, sans-serif;">The </span><a style="font-family: arial, sans-serif;" href="http://www.tableausoftware.com/public" target="_blank" rel="nofollow noopener">Tableau Public</a><span style="font-family: arial, sans-serif;"> data-visualization tool</span></li> 
          <li><span style="font-family: arial, sans-serif;">The easy-to-use </span><a style="font-family: arial, sans-serif;" href="http://www.gpsvisualizer.com" target="_blank" rel="nofollow noopener">GPS Visualizer</a><span style="font-family: arial, sans-serif;"> (requires longitude, latitude data)</span></li> 
          <li><span style="font-family: arial, sans-serif;">The Flash-based map- and trend-generation engine, </span><a style="font-family: arial, sans-serif;" href="http://www.statsilk.com/home" target="_blank" rel="nofollow noopener">StatSilk</a></li> 
          <li><span style="font-family: arial, sans-serif;">The Flash-based (and web-centric, but gorgeous) </span><a style="font-family: arial, sans-serif;" href="http://flare.prefuse.org/" target="_blank" rel="nofollow noopener">Flare</a><em style="font-family: arial, sans-serif;">[requires nontrivial compilation]</em></li>

           <li>The <a href="http://blog.revolutionanalytics.com/2011/08/r-datamarket.html" target="_blank" rel="nofollow noopener">rdatamarket</a> package for pulling data from <a href="http://datamarket.com/" target="_blank" rel="nofollow noopener">DataMarket</a> directly into R</li> 
           <li>R datasets on truly random (but generally interesting) topics at&nbsp;<a style="color: #000000!important; text-decoration: underline;" href="http://www.reddit.com/r/datasets/" target="_blank" rel="nofollow noopener">reddit</a></li> 
           <li>The incredible <a href="http://addictedtor.free.fr/graphiques/" target="_blank" rel="nofollow noopener">R Graph Gallery</a>, with source code</li> 
           <li>The <a href="http://had.co.nz/ggplot2/" target="_blank" rel="nofollow noopener">ggplot2</a> R library [now in maintenance mode; being phased out in favor of&nbsp;<a href="https://github.com/rstudio/ggvis">ggvis</a>]</li> 
           <li>The <a href="http://cran.r-project.org/web/packages/ggmap/ggmap.pdf" target="_blank" rel="nofollow noopener">ggmap</a> R library, which plots latitude/longitude data on maps</li> 
           <li>A <a href="http://flowingdata.com/2010/01/21/how-to-make-a-heatmap-a-quick-and-easy-solution/" target="_blank" rel="nofollow noopener">good example</a> of how to make a heatmap (not geographical) with R’s heatmap library</li> 
           <li>A <a href="http://r-nold.blogspot.com/2012/06/talking-about-elevation-one-can-also.html" target="_blank" rel="nofollow noopener">useful example</a> of how to make 3D maps with R’s persp library</li> 
           <li>Two tutorials (<a href="http://geography.uoregon.edu/GeogR/examples/maps_examples02.htm" target="_blank" rel="nofollow noopener">here</a> and <a href="http://www.r-bloggers.com/plot-maps-like-a-boss/" target="_blank" rel="nofollow noopener">here</a>) on combining maps with data</li> 
           <li>A <a href="http://blog.revolutionanalytics.com/2009/11/charting-time-series-as-calendar-heat-maps-in-r.html" target="_blank" rel="nofollow noopener">tutorial</a> on how to turn time series data into calendar heatmaps in R</li> 
           <li>“<a href="http://polmeth.wustl.edu/methodologist/tpm_v20_n1.pdf">Large Datasets and You,</a>” a primer on big data in R by Matthew Blackwell and Maya Sen</li> 
          </ul>

        <h4>R + Javascript for Interactive Online Dataviz</h4> 
        <ul> 
         <li><a href="http://www.rstudio.com/shiny/" target="_blank" rel="noopener">Shiny</a>, an R package for creating interactive graphics&nbsp;with no (non-R) programming required</li> 
         <li><a href="http://simplystatistics.org/2013/04/02/introducing-the-healthvis-r-package-one-line-d3-graphics-with-r/">healthvis</a>, an R package for creating D3-enabled versions of some common graph types with no (non-R) programming required</li> 
         <li><a href="http://rcharts.io/index.html">rCharts</a>, Ramnath Vaidyanathan’s Javascript-fueled interactive-graphics package for R</li> 
         <li>The&nbsp;<a href="https://plot.ly/r/">Plotly API for R</a>&nbsp;(see Plotly, above, for full description)</li> 
         <li><a href="http://ggvis.rstudio.com">ggvis</a>, an R package for creating interactive graphics</li> 
         <li><a href="http://christophergandrud.github.io/d3Network/">D3Network</a>, an R package for creating network, tree, dendogram, and Sankey diagrams in D3</li> 
         <li>The crazy-cool&nbsp;<a href="http://www.magesblog.com/2015/03/pivot-tables-with-r.html">rPivotTable package</a>&nbsp;for interactive data visualization (watch the GIF to get a sense of its capabilities)</li>
