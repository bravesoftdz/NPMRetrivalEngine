         <h1>Course Materials at Boundless<a class="headerlink" href="#course-materials-at-boundless" title="Permalink to this headline">¶</a></h1> 
         <p><a class="reference external" href="http://boundlessgeo.com/">Boundless</a> is devoted to building and supporting tools that help organizations share their information online. We believe that access to knowledge is a fundamental good, so we make our training materials available online under an open document license, for everyone to use and reuse. If you require more hands on training please visit our <a class="reference external" href="http://boundlessgeo.com/resources/training/">education center</a> to see what courses we offer.</p> 
         <div class="section" id="tutorials"> 
          <h2>Tutorials<a class="headerlink" href="#tutorials" title="Permalink to this headline">¶</a></h2> 
          <p>The tutorials below are <strong>practical walk-throughs</strong> of actual application development examples from start to finish, demonstrating different aspects of Suite functionality and the kinds of problems the Suite can solve.</p> 
          <dl class="docutils"> 
           <dt>
            <a class="reference external" href="http://workshops.boundlessgeo.com/tutorial-autocomplete/">Building an Autocomplete Parcel Lookup</a>
           </dt> 
           <dd>

            <p class="last">This application provides a fast autocomplete form field, backed by a full-text search database engine. The example shows a simple address look-up example with parcel data.. <a class="reference external" href="http://apps.boundlessgeo.com/apps/parcelmap">DEMO</a></p> 
           </dd> 
           <dt>
            <a class="reference external" href="http://workshops.boundlessgeo.com/tutorial-wordmap/">Building a GeoNames Heat Map</a>
           </dt> 
           <dd>
            <p class="first"><img alt="heatmap" class="inline-img" src="_images/heatmap_demo.png"></p> 
            <p class="last">This application displays a web-based dynamic heat map, based on querying a database of two million geographic names points for the United States. <a class="reference external" href="http://workshops.boundlessgeo.com/tutorial-wordmap/_static/wordmap-full.html">DEMO</a></p> 
           </dd> 
           <dt>
            <a class="reference external" href="http://workshops.boundlessgeo.com/tutorial-censusmap/">Building a Census Map</a>

           <dd>
            <p class="first"><img alt="censusmap" class="inline-img" src="_images/censusmap_demo.png"></p>

         <div class="section" id="workshops"> 
          <h2>Workshops<a class="headerlink" href="#workshops" title="Permalink to this headline">¶</a></h2> 
          <p>The following are actual course materials given at our various training classes in New York City and at conferences around the world. As our list of courses grows, we will be posting more materials here.</p> 
          <div class="section" id="id4"> 
           <h3><a class="reference external" href="http://workshops.boundlessgeo.com/suiteintro/">Introduction to OpenGeo Suite</a><a class="headerlink" href="#id4" title="Permalink to this headline">¶</a></h3> 
           <p>This course will introduce attendees to the <a class="reference external" href="http://boundlessgeo.com/opengeo-suite/">OpenGeo Suite</a>, a complete web mapping solution. The course will include an introduction to spatial analysis with PostGIS, loading and serving data with GeoServer, map optimization with GeoWebCache, styling, composing, and publishing maps with GeoExplorer, and creating your own web map applications with OpenLayers and GeoExt. Also available as a <a class="reference external" href="http://files.boundlessgeo.com/workshopmaterials/suite-intro.zip">Download</a>.</p> 
          </div> 
          <div class="section" id="id5"> 
           <h3><a class="reference external" href="http://workshops.boundlessgeo.com/postgis-intro/">Introduction to PostGIS</a><a class="headerlink" href="#id5" title="Permalink to this headline">¶</a></h3> 
           <p>As presented at FOSS4G 2014, this workshop introduces the PostGIS spatial database. Installation, loading data, configuration, spatial SQL queries, indexing, tuning and viewing are among the topics touched on in this overview workshop. Also available as a <a class="reference external" href="http://files.boundlessgeo.com/workshopmaterials/postgis-workshop-201401.zip">Download</a>.</p> 
          </div> 
          <div class="section" id="id7"> 
           <h3><a class="reference external" href="http://workshops.boundlessgeo.com/geoserver-intro/">Introduction to GeoServer</a><a class="headerlink" href="#id7" title="Permalink to this headline">¶</a></h3> 
           <p>Updated for 2013. Learn to load, style, and publish maps and data using GeoServer. Also available as a <a class="reference external" href="http://files.boundlessgeo.com/workshopmaterials/geoserver-intro.zip">Download</a>.</p> 
          </div> 
          <div class="section" id="id9"> 
           <h3><a class="reference external" href="http://workshops.boundlessgeo.com/openlayers3/">Introduction to OpenLayers 3</a><a class="headerlink" href="#id9" title="Permalink to this headline">¶</a></h3> 
           <p>This workshop will show how to create a map, add layers, use controls, edit vectors, and style vectors using the modern OpenLayers 3 JavaScript map component.</p> 
          </div> 
          <div class="section" id="id10"> 
           <h3><a class="reference external" href="http://workshops.boundlessgeo.com/postgis-spatialdbtips/">Spatial Database Tips and Tricks</a><a class="headerlink" href="#id10" title="Permalink to this headline">¶</a></h3> 
           <p>How to use the power of spatial databases to make your web mapping applications more powerful. Spatial SQL queries, binding the web to the database, and web architectures.</p> 
          </div> 
          <div class="section" id="id11"> 
           <h3><a class="reference external" href="http://qgis.boundlessgeo.com/static/docs/index.html">Introduction to OpenGeo Suite QGIS plugin</a><a class="headerlink" href="#id11" title="Permalink to this headline">¶</a></h3> 
           <p>The OpenGeo Suite QGIS plugin is used to configure the components of OpenGeo Suite through <a class="reference external" href="http://qgis.org">QGIS</a>, a free, open source desktop GIS client. If you prefer using a GUI (graphic user interface) to work with geospatial data try out QGIS and our Suite-QGIS plugin!</p> 
          </div> 
          <div class="section" id="id12"> 
           <h3><a class="reference external" href="http://geogig.org/workshop/">Introduction to GeoGig</a><a class="headerlink" href="#id12" title="Permalink to this headline">¶</a></h3> 
           <p>GeoGig is a Distributed Version Control System (DVCS) specially designed to handle geospatial data. It takes inspiration from the source code versioning system <a class="reference external" href="http://git-scm.com/">Git</a>, but employs an approach best suited to the nature of the spatial data it manages. The <a class="reference external" href="http://geogig.org/">GeoGig</a> project is feature complete and will be released as a <a class="reference external" href="http://locationtech.org/projects/technology.geogig">LocationTech</a> project.</p> 
          </div> 
         </div> 
         <div class="section" id="more-workshops"> 
          <h2>More Workshops<a class="headerlink" href="#more-workshops" title="Permalink to this headline">¶</a></h2> 
          <div class="section" id="id13"> 
           <h3><a class="reference external" href="http://workshops.boundlessgeo.com/geogig/">GeoGig Workshop</a><a class="headerlink" href="#id13" title="Permalink to this headline">¶</a></h3> 
           <p>GeoGig is a Distributed Version Control System (DVCS) for geospatial data. This workshop is from the LocationTech Tour 2015.</p> 
          </div> 
          <div class="section" id="id14"> 
           <h3><a class="reference external" href="http://workshops.boundlessgeo.com/openlayers-intro/">Introduction to OpenLayers 2</a><a class="headerlink" href="#id14" title="Permalink to this headline">¶</a></h3> 
           <p>This workshop will show how to create an embedded map, add layers, use controls, add markers and other useful topics for getting started with the OpenLayers JavaScript map component. Updated for OpenLayers 2.13. Also available as a <a class="reference external" href="http://files.boundlessgeo.com/workshopmaterials/ol_workshop.zip">Download</a>.</p> 
          </div> 
          <div class="section" id="id16"> 
           <h3><a class="reference external" href="http://workshops.boundlessgeo.com/geoext/">Developing OGC Compliant Web Applications with GeoExt</a><a class="headerlink" href="#id16" title="Permalink to this headline">¶</a></h3> 
           <p>First presented at FOSSGIS 2011, this workshop introduces GeoExt as a web mapping frontend to OGC Web Services (OWS). Create a map application with a WMS browser, a WFS-T editor, and more. Also available as a <a class="reference external" href="http://files.boundlessgeo.com/workshopmaterials/gx_workshop.zip">Download</a>.</p> 
          </div> 
         </div> 
         <div class="section" id="about-these-materials"> 
          <h2>About these materials<a class="headerlink" href="#about-these-materials" title="Permalink to this headline">¶</a></h2> 
          <p>The materials in this site are built using the <a class="reference external" href="http://sphinx-doc.org/">Sphinx</a> documentation system, and the source for all the materials are available in a public <a class="reference external" href="http://github.com/boundlessgeo/workshops/">repository</a>. If you find an error in the materials, or would like to add a translation, please let us know.</p> 
         </div> 
         <div class="section" id="more-boundless-services"> 
          <h2>More Boundless Services<a class="headerlink" href="#more-boundless-services" title="Permalink to this headline">¶</a></h2> 
          <div class="section" id="id20"> 
           <h3><a class="reference external" href="http://boundlessgeo.com/opengeo-suite/">OpenGeo Suite</a><a class="headerlink" href="#id20" title="Permalink to this headline">¶</a></h3> 
           <p>OpenGeo Suite bundles PostGIS, GeoServer, GeoWebCache and OpenLayers into one integrated, supported, web mapping application stack. OpenGeo Suite can be integrated into existing infrastructures, deployed stand-alone, or deployed in cloud-computing environments. Boundless provides commercial <a class="reference external" href="http://boundlessgeo.com/support/">Support</a> for Suite.</p> 
          </div>

These courses were created by our expert trainers to help you get the most out of your busy schedule. To get started see our course catalog.</p>
