      <h2><span class="mw-headline" id="Open_source_software">Open source software</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_geographic_information_systems_software&amp;action=edit&amp;section=1" title="Edit section: Open source software">edit</a><span class="mw-editsection-bracket">]</span></span></h2> 
      <p>The development of open source GIS software has—in terms of software history—a long tradition<sup id="cite_ref-wiki.osgeo.org_2-0" class="reference"><a href="#cite_note-wiki.osgeo.org-2">[2]</a></sup> with the appearance of a first system in 1978. Numerous systems are available which cover all sectors of geospatial data handling.</p> 
      <h3><span class="mw-headline" id="Desktop_GIS">Desktop GIS</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_geographic_information_systems_software&amp;action=edit&amp;section=2" title="Edit section: Desktop GIS">edit</a><span class="mw-editsection-bracket">]</span></span></h3>

      <p>The following open-source desktop GIS projects are reviewed in Steiniger and Bocher (2008/9):<sup id="cite_ref-spatialserver.net_3-0" class="reference"><a href="#cite_note-spatialserver.net-3">[3]</a></sup></p> 
      <ul> 
       <li><a href="/wiki/GRASS_GIS" title="GRASS GIS">GRASS GIS</a> – Originally developed by the <a href="/wiki/U.S._Army_Corps_of_Engineers" class="mw-redirect" title="U.S. Army Corps of Engineers">U.S. Army Corps of Engineers</a>: a complete GIS.</li> 
       <li><a href="/wiki/GvSIG" title="GvSIG">gvSIG</a> – Written in Java. Runs on Linux, Unix, Mac OS X and Windows.</li> 
       <li><a href="/wiki/ILWIS" title="ILWIS">ILWIS</a> (Integrated Land and Water Information System) – Integrates image, vector and thematic data.</li> 
       <li><a href="/wiki/JUMP_GIS" title="JUMP GIS">JUMP GIS</a> / OpenJUMP ((Open) Java Unified Mapping Platform) – The desktop GISs OpenJUMP, SkyJUMP, deeJUMP and <a href="/wiki/Kosmo" title="Kosmo">Kosmo</a> all emerged from JUMP.<sup id="cite_ref-spatialserver.net_3-1" class="reference"><a href="#cite_note-spatialserver.net-3">[3]</a></sup></li> 
       <li><a href="/wiki/MapWindow_GIS" title="MapWindow GIS">MapWindow GIS</a> – Free desktop application and programming component.</li> 
       <li><a href="/wiki/QGIS" title="QGIS">QGIS</a> (previously known as Quantum GIS) – Runs on Linux, Unix, Mac OS X and Windows.</li> 
       <li><a href="/wiki/SAGA_GIS" title="SAGA GIS">SAGA GIS</a> (System for Automated Geoscientific Analysis) –- A hybrid GIS software. Has a unique <a href="/wiki/Application_Programming_Interface" class="mw-redirect" title="Application Programming Interface">Application Programming Interface</a> (API) and a fast-growing set of geoscientific methods, bundled in exchangeable Module Libraries.</li> 
       <li><a href="/wiki/UDig" title="UDig">uDig</a> – API and source code (Java) available.</li> 
      </ul> 
      <p>Besides these, there are other open source GIS tools:</p> 
      <ul> 
       <li><a href="/wiki/Capaware" title="Capaware">Capaware</a> – A C++ 3D GIS Framework with a multiple plugin architecture for geographic graphical analysis and visualization.</li> 
       <li><a href="/wiki/FalconView" title="FalconView">FalconView</a> – A mapping system created by the Georgia Tech Research Institute for the Windows family of operating systems. A free, open source version is available.</li> 
       <li><a href="/wiki/Kalypso_(software)" title="Kalypso (software)">Kalypso</a> – Uses Java and GML3. Focuses mainly on numerical simulations in water management.</li> 
       <li><a href="/wiki/TerraView" title="TerraView">TerraView</a> – Handles vector and raster data stored in a relational or geo-relational database, i.e. a frontend for <a href="/wiki/TerraLib" title="TerraLib">TerraLib</a>.</li> 
       <li><a href="/wiki/Whitebox_Geospatial_Analysis_Tools" title="Whitebox Geospatial Analysis Tools">Whitebox GAT</a> – Cross-platform, free and open-source GIS software.</li> 
      </ul> 
      <h3><span class="mw-headline" id="Other_geospatial_tools">Other geospatial tools</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_geographic_information_systems_software&amp;action=edit&amp;section=3" title="Edit section: Other geospatial tools">edit</a><span class="mw-editsection-bracket">]</span></span></h3> 
      <p>Apart from desktop GIS, many other types of GIS software exist. A general overview of GIS software projects for each category was done in 2012.<sup id="cite_ref-sstfoss4g_4-0" class="reference"><a href="#cite_note-sstfoss4g-4">[4]</a></sup> Below is a similar listing of open source GIS projects.</p> 
      <h4><span class="mw-headline" id="Web_map_servers">Web map servers</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_geographic_information_systems_software&amp;action=edit&amp;section=4" title="Edit section: Web map servers">edit</a><span class="mw-editsection-bracket">]</span></span></h4> 
      <ul> 
       <li><a href="/wiki/GeoServer" title="GeoServer">GeoServer</a> – Written in Java and relies on <a href="/wiki/GeoTools" title="GeoTools">GeoTools</a>. Allows users to share and edit geospatial data.</li> 
       <li><a href="/wiki/MapGuide_Open_Source" title="MapGuide Open Source">MapGuide Open Source</a> – Runs on Linux or Windows, supports Apache and IIS web servers, and has APIs (PHP, .NET, Java, and JavaScript) for application development.</li> 
       <li><a href="/wiki/Mapnik" title="Mapnik">Mapnik</a> – C++/Python library for rendering - used by <a href="/wiki/OpenStreetMap" title="OpenStreetMap">OpenStreetMap</a>.</li> 
       <li><a href="/wiki/MapServer" title="MapServer">MapServer</a> – Written in C. Developed by the <a href="/wiki/University_of_Minnesota" title="University of Minnesota">University of Minnesota</a>.</li> 
      </ul> 
      <h4><span class="mw-headline" id="Spatial_database_management_systems">Spatial database management systems</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_geographic_information_systems_software&amp;action=edit&amp;section=5" title="Edit section: Spatial database management systems">edit</a><span class="mw-editsection-bracket">]</span></span></h4> 
      <ul> 
       <li><a href="/wiki/PostGIS" title="PostGIS">PostGIS</a> – Spatial extensions for the open source <a href="/wiki/PostgreSQL" title="PostgreSQL">PostgreSQL</a> database, allowing geospatial queries.</li> 
       <li><a href="/wiki/SpatiaLite" title="SpatiaLite">SpatiaLite</a> – Spatial extensions for the open source <a href="/wiki/SQLite" title="SQLite">SQLite</a> database, allowing geospatial queries.</li> 
       <li><a href="/wiki/TerraLib" title="TerraLib">TerraLib</a> – Provides advanced functions for GIS analysis.</li> 
      </ul>

Suited for high transaction enterprise environments.</li>

      </ul>

      <h3><span class="mw-headline" id="Desktop_GIS_2">Desktop GIS</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_geographic_information_systems_software&amp;action=edit&amp;section=11" title="Edit section: Desktop GIS">edit</a><span class="mw-editsection-bracket">]</span></span></h3>

Some<sup class="noprint Inline-Template" style="white-space:nowrap;">[<i><a href="/wiki/Wikipedia:Avoid_weasel_words" class="mw-redirect" title="Wikipedia:Avoid weasel words"><span title="The material near this tag possibly uses too vague attribution or weasel words.       <h4><span class="mw-headline" id="Companies_with_high_market_share">Companies with high market share</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_geographic_information_systems_software&amp;action=edit&amp;section=12" title="Edit section: Companies with high market share">edit</a><span class="mw-editsection-bracket">]</span></span></h4> 
      <ul> 
       <li><a href="/wiki/Autodesk" title="Autodesk">Autodesk</a> – Products that interface with its flagship <a href="/wiki/AutoCAD" title="AutoCAD">AutoCAD</a> software package include Map 3D, Topobase, and <a href="/wiki/MapGuide" class="mw-redirect" title="MapGuide">MapGuide</a>.</li> 
       <li><a href="/wiki/Bentley_Systems" title="Bentley Systems">Bentley Systems</a> – Products that interface with its flagship <a href="/wiki/MicroStation" title="MicroStation">MicroStation</a> software package include Bentley Map and Bentley Map View.</li> 
       <li><a href="/wiki/ENVI_(software)" class="mw-redirect" title="ENVI (software)">ENVI</a> – Utilized for image analysis, exploitation, and hyperspectral analysis.</li> 
       <li><a href="/wiki/ERDAS_IMAGINE" class="mw-redirect" title="ERDAS IMAGINE">ERDAS IMAGINE</a> by ERDAS Inc – Products include <a href="/wiki/Leica_Photogrammetry_Suite" title="Leica Photogrammetry Suite">Leica Photogrammetry Suite</a>, ERDAS ER Mapper, ERDAS ECW/JP2 SDK (<a href="/wiki/ECW_(file_format)" title="ECW (file format)">ECW (file format)</a>) are used throughout the entire mapping community (GIS, Remote Sensing, Photogrammetry, and image compression) and ERDAS APOLLO.</li> 
       <li><a href="/wiki/Esri" title="Esri">Esri</a> – Products include <a href="/wiki/ArcMap" title="ArcMap">ArcMap</a>, <a href="/wiki/ArcGIS" title="ArcGIS">ArcGIS</a>, <a href="/wiki/ArcSDE" title="ArcSDE">ArcSDE</a>, <a href="/wiki/ArcIMS" title="ArcIMS">ArcIMS</a>, ArcWeb services and <a href="/wiki/ArcGIS_Server" title="ArcGIS Server">ArcGIS Server</a>.</li> 
       <li><a href="/wiki/Intergraph" title="Intergraph">Intergraph</a> – Products include G/Technology, <a href="/wiki/GeoMedia" title="GeoMedia">GeoMedia</a>, GeoMedia Professional, GeoMedia WebMap, and add-on products for industry sectors, as well as <a href="/wiki/Photogrammetry" title="Photogrammetry">photogrammetry</a>.</li> 
       <li><a href="/wiki/MapInfo_Corporation" title="MapInfo Corporation">MapInfo</a> by Pitney Bowes Software – Powerful desktop GIS <a href="/wiki/MapInfo_Professional" title="MapInfo Professional">MapInfo Professional</a> is enhanced with many plug-ins including MapInfo Drivetime for route analysis, MapInfo Engage 3D for 3D and statistical analysis, MapInfo MapMarker for Geocoding.</li> 
       <li><a href="/wiki/Smallworld" title="Smallworld">Smallworld</a> – developed in Cambridge, England (Smallworld, Inc.) and purchased by <a href="/wiki/General_Electric" title="General Electric">General Electric</a>. Used primarily by <a href="/wiki/Public_utility" title="Public utility">public utilities</a>.</li> 
      </ul> 
      <h4><span class="mw-headline" id="Companies_with_minor_but_notable_market_share">Companies with minor but notable market share</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_geographic_information_systems_software&amp;action=edit&amp;section=13" title="Edit section: Companies with minor but notable market share">edit</a><span class="mw-editsection-bracket">]</span></span></h4> 
      <ul> 
       <li><a href="/wiki/Cadcorp" title="Cadcorp">Cadcorp</a> – Products include Cadcorp SIS, GeognoSIS, mSIS and developer kits.</li> 
       <li><a href="/wiki/Caliper_Corporation" title="Caliper Corporation">Caliper</a> – Products include <a href="/wiki/Maptitude" title="Maptitude">Maptitude</a>, <a href="/wiki/TransModeler" title="TransModeler">TransModeler</a> and TransCAD.</li> 
       <li><a href="/wiki/GameSim#GIS" title="GameSim">Conform</a> by <a href="/wiki/GameSim" title="GameSim">GameSim</a> – Software for fusing and visualizing elevation, imagery, vectors, and LiDAR. The fused environment can be exported into 3D formats for gaming, simulation, and urban planning.<sup id="cite_ref-6" class="reference"><a href="#cite_note-6">[6]</a></sup></li> 
       <li><a href="/wiki/Dragon_(remote_sensing)" title="Dragon (remote sensing)">Dragon/ips</a> – Remote sensing software with GIS capabilities.</li> 
       <li><a href="/wiki/Geosoft" title="Geosoft">Geosoft</a> – GIS and data processing software used in <a href="/wiki/Natural_resource" title="Natural resource">natural resource</a> <a href="/wiki/Exploration" title="Exploration">exploration</a>.</li> 
       <li><a href="/wiki/GeoTime" title="GeoTime">GeoTime</a> – software for 3D visual analysis and reporting of location data over time; an <a href="/wiki/ArcGIS#Extensions" title="ArcGIS">ArcGIS extension</a> is also available.</li> 
       <li><a href="/wiki/Global_Mapper" title="Global Mapper">Global Mapper</a> – GIS software package currently developed by Blue Marble Geographics; originally based on USGS dlgv32 source code.</li> 
       <li><a href="/wiki/Golden_Software" title="Golden Software">Golden Software</a> – GIS and scientific software for a wide variety of professional geological applications. Products include <i>Surfer</i> for gridding and contouring, <i>MapViewer</i> for thematic mapping and spatial analysis, <i>Strater</i> for well or borehole logging and cross sections, <i>Voxler</i> for true 3D well and component mapping, <i>Didger</i> for digitizing and coordinate conversion, and <i>Grapher</i> for 2D and 3D graphing.</li> 
       <li><a href="/wiki/Kongsberg_Gallium_Ltd." class="mw-redirect" title="Kongsberg Gallium Ltd.">Kongsberg Gallium Ltd.</a> – Products include InterMAPhics and InterView. High performance GIS visualization and analytics toolkits supporting multiple platforms, including flavors of Unix, Windows and Android. Primarily intended for mission critical visualizations</li> 
       <li><a href="/wiki/MapDotNet" title="MapDotNet">MapDotNet</a> – Framework written in C#/.NET for building WPF, Silverlight, and HTML5 applications.</li> 
       <li><a href="/wiki/Manifold_System" title="Manifold System">Manifold System</a> – GIS software package.</li> 
       <li><a href="/wiki/CitySurf_Globe" title="CitySurf Globe">CitySurf Globe</a> – Server based 3D GIS software, developed by PiriReis.</li> 
       <li><a href="/wiki/Netcad" title="Netcad">Netcad</a> – Desktop and web based GIS products developed by <a href="https://tr.wikipedia.org/wiki/Ulusal_CAD_ve_GIS_%C3%87%C3%B6z%C3%BCmleri_A.%C5%9E." class="extiw" title="tr:Ulusal CAD ve GIS Çözümleri A.?.">Ulusal CAD ve GIS Çözümleri A.?.</a>.</li> 
       <li><a href="/wiki/RegioGraph" title="RegioGraph">RegioGraph</a> by <a href="/wiki/GfK_GeoMarketing" class="mw-redirect" title="GfK GeoMarketing">GfK GeoMarketing</a> – GIS software for business planning and analyses; company also provides compatible maps and market data.</li> 
       <li><a href="/wiki/RemoteView" title="RemoteView">RemoteView</a> by Overwatch – RemoteView is one of the most widely used imagery analysis tools within the US government to collect geospatial intelligence.</li> 
       <li><a href="/wiki/SuperMap" title="SuperMap">SuperMap Inc.</a> – a professional GIS software provider that offers Desktop, Component, Web, and Mobile GIS for global markets.</li> 
       <li><a href="/wiki/TerrSet" title="TerrSet">TerrSet</a> – GIS and Image Processing product developed by Clark Labs at <a href="/wiki/Clark_University" title="Clark University">Clark University</a>. Especially effective for raster GIS analysis and image processing. Affordable and robust, it is used for both operations and education.</li> 
       <li><a href="/wiki/TNTmips" title="TNTmips">TNTmips</a> by MicroImages – a professional system integrating desktop GIS, advanced image processing, 2D-3D-stereo visualization, desktop cartography, geospatial database management, and webmap publishing.</li> 
      </ul> 
      <h3><span class="mw-headline" id="GIS_as_a_service">GIS as a service</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_geographic_information_systems_software&amp;action=edit&amp;section=14" title="Edit section: GIS as a service">edit</a><span class="mw-editsection-bracket">]</span></span></h3> 
      <p>Many suppliers are now starting to offer Internet based services as well as or instead of downloadable software and/or data. These can be free, funded by advertising or paid for on subscription; they split into three areas:</p> 
      <ul> 
       <li><a href="/wiki/SaaS" class="mw-redirect" title="SaaS">SaaS</a> – Software as a Service: Software available as a service on the Internet 
        <ul> 
         <li><a href="/wiki/ArcGIS" title="ArcGIS">ArcGIS</a> Online – ESRI's cloud based version of ArcGIS<sup id="cite_ref-7" class="reference"><a href="#cite_note-7">[7]</a></sup></li> 
         <li><a href="/wiki/CartoDB" title="CartoDB">CartoDB</a> – online mapping platform that offers an open source, cloud based SaaS model<sup id="cite_ref-8" class="reference"><a href="#cite_note-8">[8]</a></sup></li> 
         <li><a href="/wiki/Mapbox" title="Mapbox">Mapbox</a> – provider of custom online maps for websites</li> 
        </ul> </li> 
       <li><a href="/wiki/PaaS" class="mw-redirect" title="PaaS">PaaS</a> – Platform as a Service: <a href="/wiki/Geocoding" title="Geocoding">geocoding</a> or analysis/processing services 
        <ul> 
         <li><a href="/wiki/ArcGIS" title="ArcGIS">ArcGIS</a> Online<sup id="cite_ref-9" class="reference"><a href="#cite_note-9">[9]</a></sup></li> 
         <li><a href="/wiki/Google_Maps" title="Google Maps">Google Maps</a> <a href="/wiki/Javascript" class="mw-redirect" title="Javascript">Javascript</a> API version 3<sup id="cite_ref-10" class="reference"><a href="#cite_note-10">[10]</a></sup></li> 
         <li><a href="/wiki/Microsoft" title="Microsoft">Microsoft</a> <a href="/wiki/Bing_(search_engine)" title="Bing (search engine)">Bing</a> Geocode Dataflow API<sup id="cite_ref-11" class="reference"><a href="#cite_note-11">[11]</a></sup></li> 
         <li><a href="/wiki/US_Census" class="mw-redirect" title="US Census">US Census</a> Geocoder<sup id="cite_ref-12" class="reference"><a href="#cite_note-12">[12]</a></sup></li> 
        </ul> </li> 
       <li><a href="/wiki/DaaS" class="mw-redirect" title="DaaS">DaaS</a> – Data as a Service: data or content services 
        <ul> 
         <li><a href="/wiki/ArcGIS" title="ArcGIS">ArcGIS</a> Online<sup id="cite_ref-13" class="reference"><a href="#cite_note-13">[13]</a></sup></li> 
         <li><a href="/wiki/Apple_Maps" title="Apple Maps">Apple Maps</a><sup id="cite_ref-14" class="reference"><a href="#cite_note-14">[14]</a></sup></li>

      <h3><span class="mw-headline" id="Spatial_DBMS">Spatial DBMS</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_geographic_information_systems_software&amp;action=edit&amp;section=15" title="Edit section: Spatial DBMS">edit</a><span class="mw-editsection-bracket">]</span></span></h3> 
      <ul> 
       <li><a href="/wiki/Spatial_Query_Server" title="Spatial Query Server">Boeing's Spatial Query Server</a> – Spatially enables Sybase ASE.</li> 
       <li><a href="/wiki/IBM_DB2" class="mw-redirect" title="IBM DB2">DB2</a> – Allows spatial querying and storing of most spatial data types.</li> 
       <li><a href="/wiki/Informix" class="mw-redirect" title="Informix">Informix</a> – Allows spatial querying and storing of most spatial data types.</li> 
       <li><a href="/wiki/MySQL" title="MySQL">MySQL</a> – Allows spatial querying and storing of most spatial data types.</li> 
       <li><a href="/wiki/Microsoft_SQL_Server" title="Microsoft SQL Server">Microsoft SQL Server</a> (2008 and later) – The latest player in the market of storing and querying spatial data. GIS products such as MapInfo and Cadcorp SIS can read and edit this data while ESRI and others are expected to be able to read and edit this data within the next few months.</li> 
       <li><a href="/wiki/Oracle_Spatial" class="mw-redirect" title="Oracle Spatial">Oracle Spatial</a> – Product allows users to perform complex geographic operations and store common spatial data types in a native Oracle environment. Most commercial GIS packages can read and edit spatial data stored in this way.</li> 
       <li><a href="/wiki/SAP_HANA" title="SAP HANA">SAP HANA</a> - Allows users to store common spatial data types, load spatial data files with <a href="/wiki/Well-known_text" title="Well-known text">well-known text (WKT)</a> and well-known binary (WKB) formats and perform spatial processing using <a href="/wiki/SQL" title="SQL">SQL</a>. <a href="/wiki/Open_Geospatial_Consortium" title="Open Geospatial Consortium">Open Geospatial Consortium (OGC)</a> certification allows third party GIS software providers to store and process spatial data. GIS products such as ArcGIS from ESRI works with HANA.<sup id="cite_ref-18" class="reference"><a href="#cite_note-18">[18]</a></sup></li> 
       <li><a href="/wiki/Teradata_Geospatial" title="Teradata Geospatial">Teradata</a> – Teradata geospatial allows storage and spatial analysis on location-based data which is stored using native geospatial data-types within the Teradata database.</li> 
       <li><a href="/wiki/VMDS" title="VMDS">VMDS</a> – Version managed data store from Smallworld.</li> 
      </ul> 
      <h3><span class="mw-headline" id="Geospatial_Internet_of_Things">Geospatial Internet of Things</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_geographic_information_systems_software&amp;action=edit&amp;section=16" title="Edit section: Geospatial Internet of Things">edit</a><span class="mw-editsection-bracket">]</span></span></h3> 
      <ul> 
       <li><a href="/wiki/SensorUp" title="SensorUp">SensorUp</a> -- SensorUp provides the Cloud hosting and SDKs, based on the <a href="/wiki/Open_Geospatial_Consortium" title="Open Geospatial Consortium">Open Geospatial Consortium</a> <a href="/wiki/SensorThings_API" title="SensorThings API">SensorThings API</a> standard, allowing developers to build applications that connects with Internet of Things sensor observations and tasking. SensorUp's OGC standard-based platform includes sensor data, analytics, and mapping SDK.</li> 
      </ul>
