    <h1 class="titleLarge">How to Temboo with Node.js</h1> 
    <hr> 
    <p>Temboo makes it easy to build Node.js applications that connect to over 100 web-based resources and services (e.g. Facebook, Dropbox, US Census data) by standardizing how you interact with their Application Programming Interfaces (APIs). Don't worry if you're not familiar with APIs – with Temboo you don't have to worry about the details.</p> 
    <p>Here we'll show you how to use the Temboo Node.js SDK to write a simple Node.js script that uses Google's Geocoding API to retrieve the latitude and longitude for a specific address e.g., 104 Franklin St, New York City. What makes Temboo uniquely powerful and useful is that, once you know how to use one API, you know how to work with any API in our Library.</p> 
    <p>Before we get started, make sure that you've got Node.js v0.8.0 (or later) on your system.</p> 
    <h3 class="subtitle">Get Set up</h3> 
    <p class="num-body"><span class="ord">1</span> <a href="/login">Log in</a> to Temboo. If you don't already have an account, you can register for free <a href="/signup" target="_blank">here</a>.</p> 
    <p class="num-body"><span class="ord">2</span><a href="/download" target="_blank">Download</a> the Temboo Node.js SDK and extract it to the directory where you'd like to build this Node.js sample project. The SDK is in a directory called <code>node_modules</code> in accordance with the convention established by the Node Package Manager.</p> 
    <p class="num-body"><span class="ord">3</span> Create a new .js file in the same location as the unzipped <strong>node_modules</strong> folder. In this example, we'll call the file hellotemboo.js (though you can call it whatever you like).</p> 
    <h3 class="subtitle">Test on our website</h3> 
    <p class="num-body"><span class="ord">4</span> Go to the <a href="/library/Library/Google/Geocoding/GeocodeByAddress/" target="_blank">Google &gt; Geocoding &gt; GeocodeByAddress</a> Choreo in our Library. Select <strong>Node.js</strong> from the drop down menu at the top of the page.</p> 
    <p class="num-body"><img src="/assets/xprocessing-screen-geocoding-input.png.pagespeed.ic.7a7Q8_vNBs.png" pagespeed_url_hash="725091323" onload="pagespeed.CriticalImages.checkImageForCriticality(this);"></p> 
    <p class="num-body"><span class="ord">5</span> Enter any address or ZIP code in the <strong>Address</strong> input field e.g., 104 Franklin Street, New York City.</p> 
    <p class="num-body"><span class="ord">6</span> Now click <strong>Generate Code</strong> to test the Choreo from our website. After a moment you'll see the data that Google sends back shown in the <strong>Output</strong> section. </p> 
    <p class="num-body"><img src="/assets/xprocessing-screen-geocoding-output.png.pagespeed.ic.5OXe3R6WTG.png" pagespeed_url_hash="3652279242" onload="pagespeed.CriticalImages.checkImageForCriticality(this);"></p> 
    <h3 class="subtitle">Auto-Generate The Code</h3> 
    <p> When you run any Choreo from our website, we automatically generate code that can be used to make the same API call many languages, including Node.js. Here we'll show you how to use these snippets in your code. </p> 
    <p class="num-body"><span class="ord">7</span> Scroll down to find the <strong>Code</strong> section of the Choreo page.</p> 
    <p class="num-body"><span class="ord">8</span> Copy the code snippet and paste it into your hellotemboo.js file. At this point, your code should look something like this: </p> 
    <pre class="prettyprint">
// You'll need a single TembooSession object in your code, eg:
// var tsession = require("temboo/core/temboosession");
var session = new tsession.TembooSession("ACCOUNT_NAME", "APP_NAME", "APP_KEY");

var Google = require("temboo/Library/Google/Geocoding");

var geocodeByAddressChoreo = new Google.GeocodeByAddress(session);

// Instantiate and populate the input set for the choreo
var geocodeByAddressInputs = geocodeByAddressChoreo.newInputSet();

// Set inputs
geocodeByAddressInputs.set_Address("104 Franklin St., New York NY 10013");

// Run the choreo, specifying success and error callback handlers
geocodeByAddressChoreo.execute(
    geocodeByAddressInputs,
    function(results) {
        console.log(results.get_Longitude());
    },
    function(error) {
        console.log(error.type); 
        console.log(error.message);
    }
);
</pre> 
    <p class="num-body"><span class="ord">9</span>Make sure that all of your import statements are included. Both of the following lines should be present and uncommented in your code.</p> 
    <pre class="prettyprint">
var tsession = require("temboo/core/temboosession");
var Google = require("temboo/Library/Google/Geocoding");
</pre> 
    <h3 class="subtitle">Print results</h3> 
    <p>Before we try a test run of the app, let's update the code to print the results returned from Google's Geocoding service.</p> 
    <p>Each Choreo in the SDK returns a <code>ResultSet</code> subclass that contains accessor methods tailored specifically to the outputs of that Choreo. Using the <code>ResultSet</code>, you can retrieve the raw data XML or JSON returned by a third-party API and, in most cases, relevant fields that we've parsed out of the response for you. Remember, sample code for <em>every</em> Choreo can be found in the Library.</p> 
    <p>The example code already logs the longitude, you simply have to ask it to log the latitude as well:</p> 
    <p class="num-body"><span class="ord">10</span>Edit the success callback (<code>function(results){...}</code>) so that it looks like this:</p> 
    <pre class="prettyprint">
function(results) {
