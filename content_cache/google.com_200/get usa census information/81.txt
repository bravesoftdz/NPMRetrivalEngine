         <h1>2010 Census Browser Application</h1> 
         <p>The US Census Browser is an open source application for browsing data from the 2010 US Census. All data displayed within the application was obtained from the 2010 Census “Demographic Profiles” data set, available at:<br> <a href="http://www2.census.gov/census_2010/03-Demographic_Profile/" target="_blank">http://www2.census.gov/census_2010/03-Demographic_Profile/</a></p> 
         <p>The application is currently available in several application ecosystems:</p> 
         <ul> 
          <li>Apple iTunes</li> 
          <ul> 
           <li><a href="http://itunes.apple.com/us/app/census-browser/id483201717" target="_blank">http://itunes.apple.com/us/app/census-browser/id483201717</a></li> 
          </ul> 
          <li>Google Android</li> 
          <ul> 
           <li><a href="https://market.android.com/details?id=com.tricedesigns.CensusBrowser" target="_blank">https://market.android.com/details?id=com.tricedesigns.CensusBrowser</a></li> 
          </ul> 
          <li>BlackBerry App World</li> 
          <ul> 
           <li><a href="http://appworld.blackberry.com/webstore/content/69236?lang=en" target="_blank">http://appworld.blackberry.com/webstore/content/69236?lang=en</a></li> 
          </ul> 
          <li>Amazon App Store</li> 
          <ul> 
           <li><a href="http://www.amazon.com/Andrew-Trice-US-Census-Browser/dp/B006JDATOY/ref=sr_1_1?ie=UTF8&amp;s=mobile-apps&amp;qid=1323874245&amp;sr=1-1" target="_blank">http://www.amazon.com/Andrew-Trice-US-Census-Browser/dp/B006JDATOY/ref=sr_1_1?ie=UTF8&amp;s=mobile-apps&amp;qid=1323874245&amp;sr=1-1</a> (this includes support for <a href="http://www.amazon.com/dp/B0051VVOB2" target="_blank">Kindle Fire</a>)</li>

          </ul>

         <p>Support for additional platforms is planned for future development.</p> 
         <p>You can find detailed documentation of the strucuture and content of the Demographic Profile data set at: <a href="http://www.census.gov/prod/cen2010/doc/dpsf.pdf" target="_blank">http://www.census.gov/prod/cen2010/doc/dpsf.pdf</a></p> 
         <p>You can also view this application in your desktop or mobile browser at:&nbsp;<a href="http://tricedesigns.com/census/" target="_blank">http://tricedesigns.com/census/</a></p> 
         <p>Please keep in mind that this application was designed for mobile devices. &nbsp;Internet Explorer in particular does not work well with the Census Browser – use at your own risk. &nbsp; The browser-based application has been tested and works properly in the latest versions of Chrome, Safari, Firefox, and Opera. &nbsp; The US Census Browser application also does not work in Android 2.x and below, due to the fact that these versions of Android do not support SVG graphics in the mobile browser.</p>

         <h3>SOURCE CODE</h3> 
         <p>Complete source code for the application is available at: <a href="https://github.com/triceam/US-Census-Browser" target="_blank">https://github.com/triceam/US-Census-Browser</a><br> <span style="color: #ff0000;">Please be sure to read the “README” document in its entirety if you attempt to get any of this code running on your own.</span></p> 
         <h4>PROJECT ORGANIZATION</h4> 
         <p>The source code repository contains two folders: one for the client-side code, and one for the server side. No platform-specific build projects are included in this source code repository, however this codebase can be integrated into phonegap projects following the instructions for each platform, available at:<br> <a href="http://phonegap.com/start" target="_blank">http://phonegap.com/start</a></p> 
         <h4>CLIENT-SIDE CODE</h4>

         <h4>SERVER-SIDE CODE</h4> 
         <p>Server side code was developed using Adobe ColdFusion, powered by a SQL Server database, containing the data available at: <a href="http://www2.census.gov/census_2010/03-Demographic_Profile/" target="_blank">http://www2.census.gov/census_2010/03-Demographic_Profile/</a></p> 
         <p>To setup the backend database, I followed instructions available at: <a href="http://www2.census.gov/census_2010/03-Demographic_Profile/0FILE_STRUCTURE.doc" target="_blank">http://www2.census.gov/census_2010/03-Demographic_Profile/0FILE_STRUCTURE.doc</a></p> 
         <p>You can find detailed documentation of the strucuture and content of the US Census Demographic Profile data set at: <a href="http://www.census.gov/prod/cen2010/doc/dpsf.pdf" target="_blank">http://www.census.gov/prod/cen2010/doc/dpsf.pdf</a></p> 
         <p>The first step was to import data into MS Acess, which I then exported to SQL Server for use in the final application. Note: All source data is contained within 50+ individual DB files which must be imported into Access.</p> 
         <p>The ColdFusion code for accessing and serializing data is very simple. Data is retrieved from the back-end using standard queries. Query data is then serialized to the front-end client as JSON using a technique for conversion of a Query object to an Array of Structs, available on CFLib.org at: <a href="http://www.cflib.org/udf/QueryToArrayOfStructures" target="_blank">http://www.cflib.org/udf/QueryToArrayOfStructures</a></p> 
         <p>By accessing the CFC directly, and choosing the result format=json, the CFC function result automaticaully gets serialized as a JSON array, as shown by the link at: <a href="http://tricedesigns.com/com/tricedesigns/census/Services.cfc?method=getStateCounties&amp;state=MD&amp;returnformat=json" target="_blank">Services.cfc?method=getStateCounties&amp;state=MD&amp;returnformat=json</a></p> 
         <h4>PHONEGAP USAGE</h4> 
         <p>The client-side runtime does not have any PhoneGap dependencies for access to device-specific functionality. PhoneGap is only being used as an application container, so that the application can be distributed through various mobile “app stores”.</p>
