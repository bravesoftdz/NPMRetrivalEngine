1: "NAME"
2: "state"
3: "county"

0: "42"
1: "Autauga County, Alabama"

3: "001"
</code></pre>

</code></pre>

        .data(json.features)
        .enter().append("svg:path")
        .attr("class", function (d) {
            return "q" + quantize(dataSet[pad(d.id)]) + "-9";
        })
        .attr("d", path)
        .on("click", click)
        .append("svg:title")
        .text(function (d) {
            return d.properties.name + ": " + dataSet[pad(d.id)] + "%";
        });
    });
</code></pre> 
              <p>This is taken from <a href="http://mbostock.github.io/d3/talk/20111018/choropleth.html" rel="nofollow">here</a>. Notice that <code>dataSet[pad(d.id)]</code> has no hits because the data is not in a key/value pair I think it needs. How to get this in the expected format? I have tried just making the <code>dataSet</code> be a 2d array:</p> 
              <pre><code>0: "01001"
1: 42
</code></pre> 
              <p>This still does not have the desired result.</p>
