      <h1 class="entry-title">The easiest way to validate email addresses in PHP</h1> 
      <div class="entry-content"> 
       <p>Here's the absolute easiest way you can validate an email address using PHP. This tiny function takes advantage of the <a href="http://php.net/manual/en/function.filter-var.php">filter_var()</a> function in PHP.</p> 
       <pre class="language-php">function valid_email($email) {
    return !!filter_var($email, FILTER_VALIDATE_EMAIL);
}
</pre> 
       <p>That said, you really don't even need the function wrapper if you can remember <code>filter_var()</code> and the appropriate constant. Just remember that the return value will be&nbsp;the filtered string or <code>false</code>—not always a boolean value like the function above.</p> 
       <p>To use it:</p> 
       <pre class="language-php">if( valid_email('you@example.com') ) {
    echo 'Valid!';
} else {
    echo 'Not valid!';
}
</pre> 
       <p>Finally, no more crazy regular expressions in your code!</p>
