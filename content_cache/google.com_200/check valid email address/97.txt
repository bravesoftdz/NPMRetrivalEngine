     <h1>Lesson 16: Useful Tasks (II) – Form Validation</h1> 
     <p><em>By <a href="http://www.mariaantoniettaperna.com">Maria Antonietta Perna</a></em></p> 
     <p>A web form is a staple feature of almost any website today. Nothing can be more frustrating for your visitors than filling out all form fields, clicking the submit button, and have the entire form come back, emptied of all their <em>carefully</em> entered values, and displaying a cryptic error message expecting them to start all over again.</p> 
     <p>Any data that comes from users <strong>must be validated</strong>. This means that programs must check that no empty values are submitted, that submitted values are of the right type and length, etc. However, such checks are necessarily performed <em>after</em> the user has filled out and submitted the form to the server.</p> 
     <p>If the server is busy and takes a while before alerting the user that the form could not be accepted due to errors, users might get quite annoyed and decide not to come back to your otherwise fantastic website.</p> 
     <p>Thankfully, this rarely happens nowadays, thanks to JavaScript. In fact, although it's crucial that checking user input must be done on the server (e.g., php, .net, ruby, etc.), client-side JavaScript can do a lot to make the experience less painful and time consuming.</p> 
     <p>JavaScript runs in the user's browser, therefore it executes immediately. There's no need for the form to be submitted to the server before alerting the user that, for example, she has left one of the required fields empty.</p> 
     <p>In this lesson, you will learn how to:</p> 
     <ul> 
      <li>make sure users <strong>fill out all required form fields</strong>;</li> 
      <li>make sure users <strong>enter a valid email</strong>.</li> 
     </ul> 
     <h2>How do I validate empty form fields?</h2> 
     <p>In previous lessons you already performed some validation tasks. For example, you used <strong>isNaN()</strong> to make sure users entered a number data type. You also checked users didn't submit an empty value by using something like <strong>if (inputboxValue == "") { alert user...}</strong>. Let's see what else you could do to prevent users from submitting an empty form field.</p> 
     <p>Prepare an HTML document with a form and a button, like the one below:</p> 
     <div class="codebox"> 
      <pre>
		
		&lt;!DOCTYPE html&gt;
		&lt;html&gt;
		&lt;head&gt;
		&lt;title&gt;Lesson 16: Form validation&lt;/title&gt;
		&lt;/head&gt;
		&lt;body&gt;
		&lt;h1&gt;Lesson 16: Form validation&lt;/h1&gt;
		
		&lt;form id="myForm" action="#" method="post"&gt;
		
		&lt;fieldset&gt;
		
		&lt;p&gt;<strong>&lt;label for="txtName"&gt;First Name:&lt;/label&gt;
		
		&lt;input type="text" id="txtName" name="txtName" /&gt;</strong>
		
		&lt;/p&gt;
		
		&lt;p&gt;<strong>&lt;input type="submit" value="Submit" onclick="validate()" /&gt;</strong>&lt;/p&gt;
		
		&lt;/fieldset&gt;
		  	
		&lt;/form&gt;
		
		<strong>&lt;script type="text/javascript"&gt;
		
		&lt;/script&gt;</strong>
			
		&lt;/body&gt;
		&lt;/html&gt;
		
		</pre> 
     </div> 
     <p>In between &lt;script&gt; tags write the <strong>validate() function</strong> as follows:</p> 
     <div class="codebox"> 
      <pre>
		
		<span class="code-javascript">function validate()</span>
		
		<span class="code-javascript">{</span>
		
		<span class="code-javascript">var userName = document.getElementById("txtName").value;</span>
		
		<span class="code-comments">//if the length of the string value is 0,</span>
		
		<span class="code-comments">//it means the user has entered no value:</span>
		
		<span class="code-javascript">if (userName.length == 0)</span>
		
		<span class="code-javascript">{</span>
		
		<span class="code-javascript">alert("Please, enter your name");</span>
		
		<span class="code-javascript">return false;</span>
		
		<span class="code-javascript">}</span>
		
		<span class="code-javascript">else</span>
		
		<span class="code-javascript">{</span>
		
		<span class="code-javascript">alert("Thank you, " + userName);</span>
		
		<span class="code-javascript">}</span>
		
		<span class="code-javascript">}</span>
		
		</pre> 
     </div> 
     <p>Save your work and run the page in a browser. Try submitting the form leaving the inputbox empty, and see what happens. Can you post the form? I hope not.</p> 
     <h2>How do I validate email addresses?</h2> 
     <p>Email addresses present a characteristic pattern. We know that they have one <strong>@ character</strong> and at least a <strong>dot ( . ) character</strong>.</p> 
     <p>More detailed patterns of email addresses, as well as of domain names, zip codes, telephone numbers, etc., can be validated using a powerful feature called <strong>regular expression</strong>. A treatment of regular expressions is beyond the scope of this tutorial. However, if you're interested the web offers a wealth of material on the subject.</p> 
     <p>In our code snippet we use some string manipulation methods to make sure the value entered by the user has an <em>@ character and a dot</em>.</p> 
     <div class="codebox"> 
      <pre>
		
		&lt;!DOCTYPE html&gt;
		&lt;html&gt;
		&lt;head&gt;
		&lt;title&gt;Lesson 16: Form validation&lt;/title&gt;
		&lt;/head&gt;
		&lt;body&gt;
		&lt;h1&gt;Lesson 16: Form validation&lt;/h1&gt;
		
		&lt;form id="myForm" action="#" method="post"&gt;
		
		&lt;fieldset&gt;
		
		&lt;p&gt;<strong>&lt;label for="txtEmail"&gt;Email:&lt;/label&gt;
		
		&lt;input type="text" id="txtEmail" name="txtEmail" /&gt;</strong>
		
		&lt;/p&gt;
		
		&lt;p&gt;<strong>&lt;input type="submit" value="Submit" onclick="validate()" /&gt;</strong>&lt;/p&gt;
		
		&lt;/fieldset&gt;
		  	
		&lt;/form&gt;
		
		<strong>&lt;script type="text/javascript"&gt;
		
		&lt;/script&gt;</strong>
			
		&lt;/body&gt;
		&lt;/html&gt;
		
		</pre> 
     </div> 
     <p>Now, change the <em>validate() function</em> as follows:</p> 
     <div class="codebox"> 
      <pre>
		
		<span class="code-javascript">function validate()</span>
		
		<span class="code-javascript">{</span>
		
		<span class="code-javascript">var email = document.getElementById("txtEmail").value;</span>
		
		<span class="code-comments">//still check user entered a value</span>
		
		<span class="code-javascript">if (email.length == 0)</span>
		
		<span class="code-javascript">{</span>
		
		<span class="code-javascript">alert("Please, enter an email address");</span>
		
		<span class="code-javascript">return false;</span>
		
		<span class="code-javascript">}</span>
		
		<span class="code-comments">//if the index position of the @  or . characters</span>
		
		<span class="code-comments">//is -1 (any value bigger than this, including 0,</span>
		
		<span class="code-comments">//means the character is present)</span>
		
		<span class="code-javascript">else if (email.indexOf("@") == -1 || email.indexOf(".") == -1)</span>
		
		<span class="code-javascript">{</span>
		
		<span class="code-javascript">alert("Please, enter a valid email address");</span>
		
		<span class="code-javascript">return false;</span>
		
		<span class="code-javascript">}</span>
		
		<span class="code-comments">//if we get here, all went well</span>
		
		<span class="code-javascript">else</span>
		
		<span class="code-javascript">{</span>
		
		<span class="code-javascript">alert("Thank you");</span>
		
		<span class="code-javascript">}</span>
		
		<span class="code-javascript">}</span>
		
		</pre> 
     </div> 
     <p>Save your work and preview the page in a browser. Now, enter an email address with no <em>@ character</em> or no <em>dot charachter</em>. Is your form submitting the value to the server? Hopefully, it isn't.</p> 
     <h2>Try out: form validation demo</h2> 
     <p>In this try out you will build a JavaScript program that checks user input (name and email) before web form submission. Prepare an HTML page containing a form with 2 inputboxes and a button, and add a paragraph with an id attribute called <em>result</em> to display messages to the user, like the one below:</p> 
     <div class="codebox"> 
      <pre>
		
		&lt;!DOCTYPE html&gt;
		&lt;html&gt;
		&lt;head&gt;
		&lt;title&gt;Lesson 16: Form validation&lt;/title&gt;
		<strong>&lt;script type="text/javascript" src="lesson16.js"&gt;&lt;/script&gt;</strong>
		&lt;/head&gt;
		&lt;body&gt;
		&lt;h1&gt;Lesson 16: Form validation&lt;/h1&gt;
		
		&lt;form id="myForm" action="#" method="post"&gt;
		
		&lt;fieldset&gt;
		
		&lt;p&gt;<strong>&lt;label for="txtName"&gt;Name:&lt;/label&gt;
		
		&lt;input type="text" id="txtName" name="txtName" /&gt;</strong>
		
		&lt;p&gt;<strong>&lt;label for="txtEmail"&gt;Email:&lt;/label&gt;
		
		&lt;input type="text" id="txtEmail" name="txtEmail" /&gt;</strong>
		
		&lt;/p&gt;
		
		&lt;p&gt;<strong>&lt;input type="submit" value="Submit" /&gt;</strong>&lt;/p&gt;
		
		&lt;/fieldset&gt;
		  	
		&lt;/form&gt;
		
		<strong>&lt;p id="result"&gt;&lt;/p&gt;</strong>
			
		&lt;/body&gt;
		&lt;/html&gt;
		
		</pre> 
     </div> 
     <p>Now create the <strong>lesson16.js</strong> file. This contains 5 functions:</p> 
     <ol> 
      <li><strong>init()</strong> binds the main function, validate(), to the form's onsubmit event;</li> 
      <li><strong>validate()</strong> examines the results returned by functions validating required fields and email format and displays an appropriate message;</li> 
      <li><strong>validateRequired(input)</strong> checks the input box has a value. It returns true if it has and false if it hasn't;</li> 
      <li><strong>validateEmail(email)</strong> checks the email is valid and returns true if it is, false if it isn't;</li> 
      <li><strong>writeMessage(text)</strong> displays a message to users when something goes wrong. It would be annoying to have alerts popping up every time an error occurs.</li> 
     </ol> 
     <div class="codebox"> 
      <pre>
		
		<span class="code-javascript">function init()</span>
		
		<span class="code-javascript">{</span>
		
		<span class="code-javascript">var myForm = document.getElementById("myForm");</span>
		
		<span class="code-javascript">myForm.onsubmit = validate;</span>
		
		<span class="code-javascript">}</span>
		
		
		
		<span class="code-comments">/********************************/</span>
		
		
		
		<span class="code-comments">//bind init() function to onload event</span>
		
		<span class="code-javascript">onload = init;</span>
		
		
		
		<span class="code-comments">/******************************************/</span>
		
		
		
		<span class="code-comments">//validate() checks answers from validateRequired()</span>
		
		<span class="code-comments">//and validateEmail() and displays appropriate messages.</span>
		
		<span class="code-comments">//If an error occurs program execution is stopped:</span>
		
		<span class="code-javascript">function validate()</span>
		
		<span class="code-javascript">{</span>
		
		<span class="code-javascript">var name = document.getElementById("txtName").value;</span>
		
		<span class="code-javascript">var email = document.getElementById("txtEmail").value;</span>
		
		<span class="code-comments">//validateRequired() and validateEmail()</span>
		
		<span class="code-comments">//return true/false values: create</span>
		
		<span class="code-comments">//variables to store the result</span>
		
		<span class="code-javascript">var isRequiredNameSet = false;</span>
		
		<span class="code-javascript">var isRequiredEmailSet = false;</span>
		
		<span class="code-javascript">var isEmailValid = false;</span>
		
		<span class="code-comments">//create variable that holds message to display</span>
		
		<span class="code-javascript">var message = "";</span>
		
		<span class="code-javascript">isRequiredNameSet = validateRequired(name);</span>
		
		<span class="code-javascript">isRequiredEmailSet = validateRequired(email);</span>
		
		<span class="code-javascript">isEmailValid = validateEmail(email);</span>
		
		<span class="code-comments">//if all values are valid, prepare thank you message</span>
		
		<span class="code-comments">//and allow form submission</span>
		
		<span class="code-javascript">if (isRequiredNameSet &amp;&amp; isRequiredEmailSet &amp;&amp; isEmailValid)</span>
		
		<span class="code-javascript">{</span>
		
		<span class="code-javascript">message = "Thank you, your details have been successfully posted!";</span>
		
		<span class="code-javascript">}</span>
		
		<span class="code-comments">//if the name field is empty, write message to user and stop submission:</span>
		
		<span class="code-javascript">else if (! isRequiredNameSet)</span>
		
		<span class="code-javascript">{</span>
