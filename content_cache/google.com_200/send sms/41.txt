        <h2 id="description-header">Description</h2> 
        <p>You can add to wordpress, the ability of sending sms, member of sms newsletter and send to them sms.<br> To every changement of position in wordpress, you can send a sms through this plugin.</p> 
        <p>The usage of this plugin is completely free. You have to just have an account from a service in the gateway list’s that we support them.<br> Don’t worry, we have tried to add the best and the most gateways to plugin.</p> 
        <h4>Send SMS with WordPress in less than 1 minute!</h4> 
        <p><span class="embed-youtube" style="text-align:center; display: block;"><iframe class="youtube-player" type="text/html" width="640" height="390" src="https://www.youtube.com/embed/50Sv5t6wTrQ?version=3&amp;rel=1&amp;fs=1&amp;autohide=2&amp;showsearch=0&amp;showinfo=1&amp;iv_load_policy=1&amp;wmode=transparent" allowfullscreen="true" style="border:0;"></iframe></span></p> 
        <h4>Features</h4> 
        <ul> 
         <li>Supported +150 sms gateways.</li> 
         <li>Send SMS to number(s), subscribers and wordpress users.</li> 
         <li>Subscribe newsletter SMS.</li> 
         <li>Send activation code to subscribe for complete subscription.</li> 
         <li>Notification SMS when published new post to subscribers.</li> 
         <li>Notification SMS when the new release of WordPress.</li> 
         <li>Notification SMS when registering a new User.</li> 
         <li>Notification SMS when get new comment.</li> 
         <li>Notification SMS when user logged into wordpress.</li> 
         <li>Notification SMS when user registered to subscription form.</li> 
         <li>Integrate with (Contact form 7, WooCommerce, Easy Digital Downloads)</li> 
         <li>Supported WP Widget for newsletter subscribers.</li> 
         <li>Support WordPress Hooks.</li> 
         <li>Support WP REST API</li> 
         <li>Import/Export Subscribers.</li> 
        </ul> 
        <h4>Professional Package</h4> 
        <p>In the Professional pack added many features, most popular gateway and is integrated with another plugins.</p> 
        <p><a href="http://wordpresssmsplugin.com/purchases" rel="nofollow">Buy Professional Package</a></p> 
        <h4>Translations</h4> 
        <p>WP SMS has been translated in to many languages, for the current list and contributors, please visit the <a href="https://translate.wordpress.org/projects/wp-plugins/wp-sms" rel="nofollow">translate page</a>.</p> 
        <p>Translations are done by people just like you, help make WP SMS available to more people around the world and <a href="http://wordpresssmsplugin.com/localization/" rel="nofollow">do a translation</a> today!</p> 
        <h4>Contributing and Reporting Bugs</h4> 
        <p>WP-SMS is being developed on GitHub, If you’re interested in contributing to plugin, Please look at <a href="https://github.com/veronalabs/wp-sms" rel="nofollow">Github page</a></p>

         <dt>
          How using Actions?
         </dt> 
         <dd> 
          <p>Run following action when send sms with this plugin.<br> wp_sms_send</p> 
          <p>Example: Send mail when send sms.</p> 
          <pre><code>function send_mail_when_send_sms($message_info) {
    wp_mail('you@mail.com', 'Send SMS', $message_info);
}
add_action('wp_sms_send', 'send_mail_when_send_sms');
</code></pre> 
          <p>Run following action when subscribe a new user.<br> wp_sms_add_subscriber</p> 
          <p>Example: Send sms to user when register a new subscriber.</p> 
          <pre><code>function send_sms_when_subscribe_new_user($name, $mobile) {
    global $sms;
    $sms-&gt;to = array($mobile);
    $sms-&gt;msg = "Hi {$name}, Thanks for subscribe.";
    $sms-&gt;SendSMS();
}
add_action('wp_sms_add_subscriber', 'send_sms_when_subscribe_new_user', 10, 2);`
</code></pre> 
         </dd> 
         <dt>
          How using Hooks?
         </dt> 
         <dd> 
          <p>You can use following filter for modify from number.<br> wp_sms_from</p> 
          <p>Example: Add 0 to the end sender number.</p> 
          <pre><code>function wp_sms_modify_from($from) {
    $from = $from . ' 0';
    return $val;
}
add_filter('wp_sms_from', 'wp_sms_modify_from');
</code></pre> 
          <p>You can use following filter for modify receivers number.<br> wp_sms_to</p> 
          <p>Example: Add new number to get message.</p> 
          <pre><code>function wp_sms_modify_receiver($numbers) {
    $numbers[] = '09xxxxxxxx';
    return $numbers;
}
add_filter('wp_sms_to', 'wp_sms_modify_receiver');
</code></pre> 
          <p>You can use following filter for modify text message.<br> wp_sms_msg</p> 
          <p>Example: Add signature to messages that are sent.</p> 
          <pre><code>function wp_sms_modify_message($message) {
    $message = $message . ' /n Powerby: WP-SMS';
    return $message;
}
add_filter('wp_sms_msg', 'wp_sms_modify_message');
</code></pre> 
         </dd>
