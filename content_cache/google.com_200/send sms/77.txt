           <h1 class="post-title">Send email using nodejs and express in 5 simple steps</h1> 
           <p class="post-byline">by <a href="http://javascript.tutorialhorizon.com/author/javascript/" title="Posts by Ryan Sukale" rel="author">Ryan Sukale</a> · July 2, 2015</p> 
           <div class="clear"></div> 
           <div class="entry"> 
            <div class="entry-inner"> 
             <script async src="//pagead2.googlesyndication.com/pagead/js/adsbygoogle.js"></script> 
             <ins class="adsbygoogle" style="display:inline-block;width:728px;height:90px" data-ad-client="ca-pub-1720129943462898" data-ad-slot="9940140966"></ins> 
             <script>
(adsbygoogle = window.adsbygoogle || []).push({});
</script>
             <p>Send­ing auto­matic emails is a very com­mon task and doing so in nodejs is pretty straigt­for­ward as well. In this arti­cle, I will demon­strate a sim­ple exam­ple of send­ing emails using nodejs in 5 sim­ple&nbsp;steps.</p> 
             <hr> 
             <p>This arti­cle is writ­ten around the fol­low­ing sce­nario in&nbsp;mind</p> 
             <ul> 
              <li>You have an email account with an email provider like Google which you want to spec­ify as the sender. There is a sim­ple rea­son­ing behind doing so. Emails are sent from one server to another and in order to avoid land­ing your email in the recipients spam folder, you want to use a truster server as the sender of the email aka  Googles mail servers. For the pur­pose of this arti­cle, we will assume this email address is <code><a href="/cdn-cgi/l/email-protection" class="__cf_email__" data-cfemail="debba6bfb3aeb2bb9eb9b3bfb7b2f0bdb1b3">[email&nbsp;protected]</a></code>.</li> 
              <li>You have a web page that has a con­tact form where a user enters some infor­ma­tion and clicks on sub­mit. Upon click­ing sub­mit your express server needs to cap­ture this data and send the request infor­ma­tion to another email address.</li> 
             </ul> 
             <p>I assume that you have at least a basic under­stand­ing of nodejs and its rout­ing. If not, take a look at <a href="http://javascript.tutorialhorizon.com/2014/09/07/understanding-routers-in-express-4-0-visually/">this dia­gram that visu­ally depicts how rout­ing takes place</a> in expressjs.</p> 
             <blockquote>
              <p> You prob­a­bly also want to <a href="https://support.google.com/accounts/answer/6010255">change some set­tings</a> on the gmail account you want to use before proceeding. </p>
             </blockquote> 
             <h4> <a id="user-content-steps" class="anchor" href="#steps" aria-hidden="true"><span class="octicon octicon-link"></span></a>Steps</h4> 
             <p>There are just 5 steps to send­ing an email using nodejs</p> 
             <ul> 
              <li>Install the node­mailer package.</li> 
              <li>Cre­ate a trans­port object using nodemailer.createTransport() and pass it your email credentials.</li> 
              <li>Pre­pare the mes­sage to be sent in the&nbsp;body.</li> 
              <li>Cre­ate an object with infor­ma­tion about the sender, email sub­ject, recip­i­ent and the body con­tent that we pre­pared earlier.</li> 
              <li>Send the email using  transporter.sendMail()</li> 
             </ul> 
             <p>In the code snip­pets that fol­low, we will go through each of the above&nbsp;steps.</p> 
             <hr> 
             <p><strong><span class="caps">STEP</span> 1</strong>: Install the node­mailer package.</p> 
             <div class="highlight highlight-sh"> 
              <pre>npm install nodemailer</pre> 
             </div> 
             <p>Once you are done with this arti­cle, you should prob­a­bly learn more about node­mailer library from <a href="https://github.com/andris9/Nodemailer">here</a>.<br> .<br> <strong><span class="caps">STEP</span> 2</strong>: Cre­at­ing a trans­port object in your route handler.</p> 
             <div class="highlight highlight-js"> 
              <pre><span class="pl-k">var</span> nodemailer <span class="pl-k">=</span> <span class="pl-c1">require</span>(<span class="pl-s"><span class="pl-pds">'</span>nodemailer<span class="pl-pds">'</span></span>);

<span class="pl-k">var</span> router <span class="pl-k">=</span> express.Router();
app.use(<span class="pl-s"><span class="pl-pds">'</span>/sayHello<span class="pl-pds">'</span></span>, router);
router.post(<span class="pl-s"><span class="pl-pds">'</span>/<span class="pl-pds">'</span></span>, handleSayHello); <span class="pl-c">// handle the route at yourdomain.com/sayHello</span>

<span class="pl-k">function</span> <span class="pl-en">handleSayHello</span>(<span class="pl-smi">req</span>, <span class="pl-smi">res</span>) {
    <span class="pl-c">// Not the movie transporter!</span>
    <span class="pl-k">var</span> transporter <span class="pl-k">=</span> nodemailer.createTransport({
        service<span class="pl-k">:</span> <span class="pl-s"><span class="pl-pds">'</span>Gmail<span class="pl-pds">'</span></span>,
        auth<span class="pl-k">:</span> {
            user<span class="pl-k">:</span> <span class="pl-s"><span class="pl-pds">'</span><a href="/cdn-cgi/l/email-protection" class="__cf_email__" data-cfemail="117469707c617d7451767c70787d3f727e7c">[email&nbsp;protected]</a><span class="pl-pds">'</span></span>, <span class="pl-c">// Your email id</span>
            pass<span class="pl-k">:</span> <span class="pl-s"><span class="pl-pds">'</span>password<span class="pl-pds">'</span></span> <span class="pl-c">// Your password</span>
        }
    });
    ...
    ...
    ...
}</pre> 
             </div> 
             <p><strong><span class="caps">STEP</span> 3</strong>: Pre­pare some plain­text or <span class="caps">HTML</span> con­tent to be sent in the body. I am going to sim­ply use some&nbsp;text.</p> 
             <div class="highlight highlight-js"> 
              <pre><span class="pl-k">var</span> text <span class="pl-k">=</span> <span class="pl-s"><span class="pl-pds">'</span>Hello world from <span class="pl-cce">\n\n</span><span class="pl-pds">'</span></span> <span class="pl-k">+</span> req.<span class="pl-c1">body</span>.<span class="pl-c1">name</span>;</pre> 
             </div> 
             <p><strong><span class="caps">STEP</span> 4</strong>: Cre­ate a sim­ple <span class="caps">JSON</span> object with the nec­es­sary val­ues for send­ing the&nbsp;email.</p> 
             <div class="highlight highlight-js"> 
              <pre><span class="pl-k">var</span> mailOptions <span class="pl-k">=</span> {
    from<span class="pl-k">:</span> <span class="pl-s"><span class="pl-pds">'</span><a href="/cdn-cgi/l/email-protection" class="__cf_email__" data-cfemail="204558414d504c4560474d41494c0e434f4d">[email&nbsp;protected]</a>&gt;<span class="pl-pds">'</span></span>, <span class="pl-c">// sender address</span>
    to<span class="pl-k">:</span> <span class="pl-s"><span class="pl-pds">'</span><a href="/cdn-cgi/l/email-protection" class="__cf_email__" data-cfemail="c2b0a7a1a7abb4a7b082a6a7b1b6abaca3b6abadaceca1adaf">[email&nbsp;protected]</a><span class="pl-pds">'</span></span>, <span class="pl-c">// list of receivers</span>
    subject<span class="pl-k">:</span> <span class="pl-s"><span class="pl-pds">'</span>Email Example<span class="pl-pds">'</span></span>, <span class="pl-c">// Subject line</span>
    text<span class="pl-k">:</span> text <span class="pl-c">//, // plaintext body</span>
    <span class="pl-c">// html: '&lt;b&gt;Hello world ?&lt;/b&gt;' // You can choose to send an HTML body instead</span>
};</pre> 
             </div> 
             <p><strong><span class="caps">STEP</span> 5</strong>: Your moment of glory!  Actu­ally send the email and han­dle the response.</p> 
             <div class="highlight highlight-js"> 
              <pre>transporter.sendMail(mailOptions, <span class="pl-k">function</span>(<span class="pl-smi">error</span>, <span class="pl-smi">info</span>){
    <span class="pl-k">if</span>(error){
        <span class="pl-en">console</span><span class="pl-c1">.log</span>(error);
        res.json({yo<span class="pl-k">:</span> <span class="pl-s"><span class="pl-pds">'</span>error<span class="pl-pds">'</span></span>});
    }<span class="pl-k">else</span>{
        <span class="pl-en">console</span><span class="pl-c1">.log</span>(<span class="pl-s"><span class="pl-pds">'</span>Message sent: <span class="pl-pds">'</span></span> <span class="pl-k">+</span> info.response);
        res.json({yo<span class="pl-k">:</span> info.response});
    };
});</pre> 
             </div> 
             <p>And thats pretty much all there is to it! It was quite sim­ple, wasnnt it?</p>
