            <h1>Overview</h1>
            <p>In this article, we'll develop an automated solution that checks rental car prices on the Costco Travel site. &nbsp;It will send us an SMS message with the lowest price on a daily basis. &nbsp;</p>
            <h1>Technologies</h1>
            <h2>AWS Lambda</h2>
            <p>Lambda is Amazon's serverless compute platform which allows us to run our code without the need to rely on servers. &nbsp;&nbsp;It supports code written in Node.js, Java, C#, and Python.&nbsp;</p>
            <p>For our example, we'll be using Node.js which is the server runtime for JavaScript.</p>
           </div>
          </div>

            <h2>AWS SNS</h2>
            <p>SNS is Amazon's notification service and allows us to send SMS messages from Lambda.</p>
           </div>
          </div>
          <div class="sqs-block image-block sqs-block-image" data-block-type="5" id="block-yui_3_17_2_2_1502411887976_40690">

           <div class="sqs-block-content">
            <p>SMS messaging is not available in all regions. &nbsp; For more details, check <a href="http://docs.aws.amazon.com/sns/latest/dg/sms_supported-countries.html">Supported Regions and Countries</a>&nbsp;from the <em>Amazon SNS Developer Guide</em>.</p>
            <h1>Configuring SNS</h1>
            <p>In this section, we'll walk through the steps to configure SNS.</p>
            <h2>Create a Topic</h2>
            <p>Topics are basically lists of subscriptions that we can publish messages to. &nbsp;</p>
            <p>In the SNS dashboard, click on <strong>Create topic.</strong></p>
            <p>Let's create a topic named NotifyMe. &nbsp;&nbsp;</p>
           </div>
          </div>
          <div class="sqs-block image-block sqs-block-image" data-block-type="5" id="block-yui_3_17_2_9_1502415822318_42793">

           <div class="sqs-block-content">
            <p>Each topic is assigned a unique identifier called a <em>Topic ARN</em>. &nbsp; Make note of this since we'll need it later in our code.</p>
            <p>The topic ARN will look something like this:</p>
            <pre>arn:aws:sns:us-east-1:000000000000:NotifyMe</pre>
            <h2>Create a Subscription</h2>
            <ul>
             <li>Click on <strong>Create Subscription</strong></li>
             <li>Select <em>SMS</em> from the <strong>Protocol</strong> dropdown list. &nbsp;If SMS is not listed, then the region does not support it.</li>
             <li>Enter a cell phone number in the <strong>Endpoint</strong> field</li>
            </ul>
            <p>A confirmation text message will be sent to the number.&nbsp;</p>

            <h1>Costco Travel site</h1>
            <p>Let's take a look at the Costco Travel site. &nbsp; The <a href="https://www.costcotravel.com/h=4005">form</a> on the Rental Cars tab asks for some standard information:</p>
            <ul>
             <li>Pickup location</li>
             <li>Dropoff location (if different)</li>
             <li>Pickup date/time</li>
             <li>Dropoff date/time</li>
            </ul>

            <h1>Trip Details</h1>
            <p>Suppose we're planning a trip to California in 2018, and we're flying into LAX. &nbsp; &nbsp;</p>
            <p>Let's define our search criteria:</p>
            <pre>Pickup location: LAX
Dropoff location: LAX

Pickup date: 3/22/2018

            <h1>How the Form Works</h1>
            <p>When the page containing the search form loads, the web server sends a cookie in its response header named <em>Csrf-token. </em>&nbsp;This token will be used by the browser during form submission.&nbsp;&nbsp;</p>
            <p>(Read more about CSRF in the article titled <a href="https://www.codebyamir.com/blog/understanding-preventing-csrf">Understanding &amp; Preventing CSRF Attacks</a>)</p>
            <p>The "Find My Car"&nbsp;button on the search form submits a POST request to the following page:</p>
           </div>
          </div>
          <div class="sqs-block code-block sqs-block-code" data-block-type="23" id="block-yui_3_17_2_27_1502393827572_10670">
           <div class="sqs-block-content">
            <pre><code class="language-markup">https://www.costcotravel.com/rentalCarSearch.act
</code></pre>
           </div>

            <p>This URL requires that form submissions include the CSRF token. &nbsp;</p>

            <p>We'll be using the second page for our solution.</p>
            <h1>NPM Modules</h1>
            <p>NPM is the package manager that comes bundled with the Node runtime. &nbsp; The latest version of Node can be downloaded from <a href="https://nodejs.org/en/download/">here</a>.</p>

const request = require('request');
const cheerio = require('cheerio');
const AWS = require("aws-sdk");

AWS.config.region = 'us-east-1';

const formData = {
  'cs': 1,
  'pickupCity': process.env.AIRPORT,
  'pickupAsAirport': 'True',
  'dropoffCity': process.env.AIRPORT,
  'dropoffAsAirport': 'True',
  'pickupDate': process.env.PICKUP_DATE,
  'dropoffDate': process.env.PICKUP_DATE,
  'pickupTime': process.env.TIME,
  'dropoffTime': process.env.TIME,
  'driverAge': 25
};

const url = 'https://www.costcotravel.com/carSearch.act';

const options = {
  url: url,
  formData: formData,
  'User-Agent': 'codebyamir.com'
};

exports.handler = function (event, context, callback) {

  request.post(options, function reqCallback(err, httpResponse, body) {
    if (err) {
      const error = new Error("Form submission failed");
      callback(error);
    }

    // Load HTML page into cheerio
    const $ = cheerio.load(body);

    const prices = [];

    // Populate array with prices
    $('div.carCell &gt; a').each(function (index, element) {
      let price = parseFloat($(element).text().replace(/\$|,/g, ''));
      prices.push(price);
    });

    // Find lowest price
    const lowestPrice = Math.min.apply(Math, prices);

    const sns = new AWS.SNS();

    const params = {
      Message: JSON.stringify('Lowest Costco rental car price: ' + lowestPrice),
      Subject: "SNS Notification from Lambda",
      TopicArn: process.env.SNS_TOPIC_ARN
    };

    sns.publish(params, context.done);
  }
  );

};</code></pre>
