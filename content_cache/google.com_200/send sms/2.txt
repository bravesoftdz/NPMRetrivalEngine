         <h1><a id="user-content-sms-node-module" class="anchor" href="#sms-node-module" aria-hidden="true"><span class="octicon octicon-link"></span></a>SMS Node Module</h1> 
         <p>One of the teams at TheNextWeb hackathon this past weekend was kind enough to open source a node module to send SMS with the <a href="https://www.sinch.com/sms-api/">Sinch SMS API</a>. (Shoutout to Mihir and Mikhail!) Here's an example of how to use it:</p> 
         <pre class="prettyprint linenums"><code>var sinchAuth = require('sinch-auth');

var sinchSms = require('sinch-messaging');

var auth = sinchAuth("your-app-key", "your-app-secret");

sinchSms.sendMessage("+16507141052", "Hello world!");
</code></pre> 
         <p>The above script will print an ID for the SMS. It will look like this:</p> 
         <pre class="prettyprint linenums"><code>{"MessageId":123456789}
</code></pre> 
         <p>You can use this ID to check the status of the SMS at any point like so...</p> 
         <pre class="prettyprint linenums"><code>console.log(sinchSms.getStatus("123456789");
</code></pre> 
         <p>...which will give you a response like so:</p> 
         <pre class="prettyprint linenums"><code>{"Status":"Successful"}
</code></pre> 
         <p>If you're interested in digging into the node modules for authentication and sending SMS, you can find the source code on GitHub:</p> 
         <ol> 
          <li><p><a href="https://github.com/ChewTeaYeah/sinch-auth">sinch-auth</a></p></li> 
          <li><p><a href="https://github.com/ChewTeaYeah/sinch-messaging">sinch-messaging</a></p></li> 
         </ol> 
         <p>To get your own app key and secret, <a href="https://www.sinch.com/dashboard/#/signup">sign up</a> for a Sinch account and create a new app. To start, you will have $2 on your account to send some test messages. See <a href="https://www.sinch.com/pricing/sms-prices/">SMS pricing by country here</a>.</p>
