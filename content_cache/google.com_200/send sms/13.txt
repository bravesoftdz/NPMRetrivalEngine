       <p>Texting has outpaced email and phone calls as the quickest method to get in contact with someone, and it is fast becoming a way to reach anyone. A simple SMS has the capability to reach any of the 4.5 billion text-enabled devices in the world. Today, appointment reminders, ETA alerts for on-demand services, banking notifications, and flight status are all sent conveniently and programmatically directly to the user’s phone via SMS.</p>
       <p><a href="https://www.twilio.com/sms" target="_blank" rel="nofollow">Twilio’s Programmable SMS API</a> lets you send and receive text messages right in your web applications. With a few lines of code, you can get an SMS app running in a few minutes. Though Programmable SMS supports several languages, like Java and C#, for the purposes of this tutorial we’re going to focus on using Node.js.</p>
       <aside class="sidebar medium"> 
        <p><em>Andrew Baker is developer educator at <a href="https://www.twilio.com/" target="_blank" rel="nofollow">Twilio</a>.</em></p> 
       </aside>
       <h2>Get a Twilio phone number</h2>
       <p>Before you can send an SMS from Node.js, you’ll need to <a href="https://www.twilio.com/try-twilio" target="_blank" rel="nofollow">sign up for a Twilio account</a> and <a href="https://www.twilio.com/console/phone-numbers" target="_blank" rel="nofollow">purchase an SMS-capable phone number</a>. If you sign up for the free trial, you get your first phone number for free. If you search for a number to buy, you’ll be able to see what types of communication it can power.</p>
       <aside id="fsb-1991" class="fakesidebar fakesidebar-auto">
        <strong>[ <a href="https://www.infoworld.com/resources/117865/it-management/guide-digital-transformation-agile-enterprise#tk.ifw-infsb">InfoWorld's quick guide: Digital Transformation and the Agile Enterprise</a>. | Download InfoWorld’s <a href="https://www.infoworld.com/resources/105693/application-development/the-essential-guide-to-microservices#tk.ifw-infsb">essential guide to microservices</a> and learn how to create modern web and mobile applications that scale. ] </strong>
       </aside>
       <figure class="large ">
        <img src="https://images.techhive.com/images/article/2017/04/twilio-sms-number-100718379-large.jpg" border="0" alt="twilio sms number" width="700" height="244" data-imageid="100718379" data-license="IDG"> 
        <small class="credit">Twilio</small>
       </figure>
       <p>Click the Buy button to add one of these numbers to your account. Now that you have a Twilio account and a programmable phone number, you can start writing code. To make it easier, we’ll install Twilio’s official helper for Node.js applications.</p>
       <aside class="nativo-promo smartphone" id=""> 
       </aside>
       <h2>Install the Twilio Node.js module</h2>
       <p>Next, we’ll install the <a href="https://www.npmjs.com/package/twilio" target="_blank" rel="nofollow">Twilio module for Node.js</a>. Fire up a terminal or command-line interface on your machine that already has <a href="https://nodejs.org/en/" target="_blank" rel="nofollow">Node and NPM installed</a>, and run the following command in a new directory:</p>
       <p><code>npm install twilio</code></p>
       <p>This will install the twilio module so that Node.js scripts in the current directory can use it.</p>
       <h2>Send an outbound SMS message</h2>
       <p>Let’s put that shiny, new Twilio Node.js library to good use. With a single API request, we can send an outbound text message from the Twilio phone number we purchased. Open a new file called send-sms.js and paste in this code sample.</p>
       <aside class="nativo-promo tablet desktop" id=""> 
       </aside>
       <p><code>// Twilio Credentials </code><br><code>var accountSid = ‘ACXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX’;</code><br><code>var authToken = ‘your_auth_token’;</code></p>
       <p><code>// require the Twilio module and create a REST client </code><br><code>var client = require(‘twilio’)(accountSid,authToken); &nbsp;</code></p>
       <p><code>client.messages.create({</code><br><code>&nbsp; &nbsp;to: “+15558675309”, </code><br><code>&nbsp; &nbsp;from: “+15017250604”, </code><br><code>&nbsp; &nbsp;body: “This is the ship that made the Kessel Run in fourteen parsecs?”,}, function(err,message) {</code><br><code>&nbsp; &nbsp;console.log(message.sid);</code><br><code>});</code></p>
       <p>Update the account credentials with <a href="https://www.twilio.com/console" target="_blank" rel="nofollow">your own account information</a>. Specify your new Twilio number in the <code>from</code> parameter and your own mobile number in the <code>to</code> parameter. Run the code we created with this command:</p>
       <p><code>node send-sms.js</code></p>
       <p>In a few moments, you should receive an SMS from your Twilio number on your phone. Now let’s talk about <em>receiving </em>text messages.</p>
       <h2>Receive inbound SMS messages</h2>
       <p>When your Twilio number receives an incoming message, Twilio will send an HTTP request to a server you control. This callback mechanism is known as a webhook. When Twilio sends your application a request, it expects a response in the <a href="https://www.twilio.com/docs/api/twiml" target="_blank" rel="nofollow">TwiML</a> XML format telling it how to respond to the message. Let’s see how we would build this in Node.js using <a href="https://expressjs.com/" target="_blank" rel="nofollow">Express</a>. Install Express with the following command:</p>
       <p><code>npm install express —save</code></p>
       <p>Next, create a file called server.js and use the following code to create a server that can handle incoming messages.</p>
       <p><code>var http = require(‘http’);</code><br><code>var express = require(‘express’);</code><br><code>var twilio = require(‘twilio’);</code></p>
       <p><code>var app = express();</code></p>
       <p><code>app.post(‘/sms’,function(req,res) {</code><br><code>&nbsp; &nbsp;var twilio = require(‘twilio’);</code><br><code>&nbsp; &nbsp;var twiml = newtwilio.TwimlResponse();</code><br><code>&nbsp; &nbsp;twiml.message(‘The Robots are coming! Head for the hills!’);</code><br><code>&nbsp; &nbsp;res.writeHead(200,{’Content-Type’:’text/xml’});</code><br><code>&nbsp; &nbsp;res.end(twiml.toString());</code><br><code>});</code></p>
       <p><code>http.createServer(app).listen(1337,function() {</code><br><code>&nbsp; &nbsp; console.log(“Express server listening on port 1337”);</code><br><code>});</code></p>
       <p>Run this server with the following command:</p>
       <p><code>node server.js</code></p>
       <p>You’ll see that the server starts up on port 1337. Unfortunately, Twilio can’t directly call a server running on your computer. You’ll either need to deploy this code to a public server or run a tunnel such as ngrok that provides an externally accessible URL. We’ll show you how to set that up next.</p>
       <h2>Allow Twilio to talk to your Node application</h2>
       <p>If you haven’t already, <a href="https://ngrok.com/download" target="_blank" rel="nofollow">install Ngrok</a> and ensure the <code>ngrok</code> command is on your system path. The following command would use Ngrok to expose port 1337 to the public Internet. (If you used a different port number in your Express app, replace 1337 with the port number you used.)</p>
       <p><code>ngrok http 1337</code></p>
       <figure class="large ">
        <img src="https://images.techhive.com/images/article/2017/04/twilio-ngrok-100718380-large.jpg" border="0" alt="twilio ngrok" width="700" height="177" data-imageid="100718380" data-license="IDG"> 
        <small class="credit">Twilio</small>
       </figure>
       <p>Now we have a new external URL. Configure this as a webhook for one of your <a href="https://www.twilio.com/console/phone-numbers/incoming" target="_blank" rel="nofollow">phone numbers</a> in the Twilio console. Select your phone number, scroll down to the Messaging section, and change the setting for <code>A MESSAGE COMES IN</code> to <code>Webhook</code>, then enter the public ngrok URL to your service, such as <code>https://adfs3423.ngrok.io/sms</code>.</p>
       <figure class="large ">
        <img src="https://images.techhive.com/images/article/2017/04/twilio-messaging-100718378-large.jpg" border="0" alt="twilio messaging" width="700" height="177" data-imageid="100718378" data-license="IDG"> 
        <small class="credit">Twilio</small>
       </figure>
       <p>Be sure your Node.js app (server.js) is still running. Then grab your smartphone and send a text message to your Twilio phone number. You should see an HTTP request in your Ngrok console. Your Node app will process the text message, and you’ll get a response back as an SMS.</p>
       <p>Now that you know the basics of sending and receiving text messages with Node.js, you might want to check out these resources.</p>

        <li><a href="https://www.twilio.com/tutorials?filter-language=node.js" target="_blank" rel="nofollow">Tutorials with full sample applications in Node.js</a></li> 
       </ul>
