              <p>I'm trying to implement QuickSort algorithm program in Java, but I'm getting incorrect answer.</p> 
              <pre><code>public class QuickSort {

    public static void main(String[] args){
        int arr[]={12,34,22,64,34,33,23,64,33};
        int i=0;
        int j=arr.length;
        while(i&lt;j){
            i=quickSort(arr,i,i+1,j-1);

        }   
        for(i=0;i&lt;arr.length;i++)
            System.out.print(arr[i]+" ");
    }

    public static int quickSort(int arr[],int pivot,int i,int j){

        if(i&gt;j) {           
            swap(arr,pivot,j);
            return i;
        }

        while(i&lt;arr.length&amp;&amp;arr[i]&lt;=arr[pivot]) {
            i++;
        }

        while(j&gt;=1&amp;&amp;arr[j]&gt;=arr[pivot]) {           
            j--;
        }   
        if(i&lt;j)
            swap(arr,i,j);

        return quickSort(arr,pivot,i,j);

    }   
    public static void swap(int[] arr,int i,int j) {
        int temp;
        temp=arr[i];
        arr[i]=arr[j];
        arr[j]=temp;
    }

}
</code></pre> 
              <p>The above program giving me the output as: 12 23 22 33 34 33 64 34 64</p>

             <div class="post-text" itemprop="text"> 
              <p>The problem is that this is not really how quicksort works. Quicksort is a recursive algorithm that should only be called once from outside of itself. The idea is that at each iteration, you partition the array into two halves - the left half contains all elements less than the pivot, and the right half contains all elements greater than / equal to the pivot. Then you quicksort the two halves, and finally put the pivot in the middle.</p> 
              <p>If the side that you are quicksorting is less than 3 elements long, you can just swap the two elements or leave them, and that part of the array is done.</p> 
              <p>But it doesn't look like your code is doing that at all - you are calling Quicksort 6 times from your client, and within the <code>quicksort</code> function you are making at most one swap. So this is not a case where someone is going to be able to look at your code and debug it by telling you to move a swap or something. You need to revisit your logic.</p> 
              <p>Check out the Wikipedia diagram for a visual example of what is supposed to happen in a single iteration:</p> 
              <p><a href="http://en.wikipedia.org/wiki/File:Partition_example.svg" rel="noreferrer">http://en.wikipedia.org/wiki/File:Partition_example.svg</a></p>

              <pre><code>public static int partition(int[] a, int p, int r){

        int i=p,j=r,pivot=a[r];

        while(i&lt;j){

            while(i&lt;r &amp;&amp; a[i] &lt;= pivot){
                i++;
            }

            while(j&gt;p &amp;&amp; a[j]&gt;pivot){ 
                j--;
            }

            if(i&lt;j){
                swap(a, i, j);
            }           
        }   
        return j;
    }

    public static void quickSort(int[] a, int p, int r){
        if(p&lt;r){
            int q=partition(a, p, r);

            if(p==q){
                quickSort(a, p+1, r);
            }else if(q==r){
                quickSort(a, p, r-1);
            }else {
                quickSort(a, p, q);
                quickSort(a, q+1, r);
            }

        }
    }

    public static void swap(int[] a, int p1, int p2){
        int temp=a[p1];
        a[p1]=a[p2];
        a[p2]=temp;
    }

              <pre><code>static void quickSort (int[] numbers, int low, int high)
{
    int i=low;
    int j=high;
    int temp;
    int middle=numbers[(low+high)/2];

    while (i&lt;j) {

        while (numbers[i]&lt;middle) {
            i++;
        }

        while (numbers[j]&gt;middle) {
            j--;
        }

        if (i&lt;=j) {
            temp=numbers[i];
            numbers[i]=numbers[j];
            numbers[j]=temp;
            i++;
            j--;
        }
    }

    if (low&lt;j) {
        quickSort(numbers, low, j);
    }

    if (i&lt;high) {
        quickSort(numbers, i, high);
    }
}
</code></pre> 
              <p>Refer <a href="http://tutorialdata.com/example/Algorithm/QuickSortExample.html" rel="nofollow">Quick sort.</a></p>
