        <h1>How to convert String to Date – Java</h1> 
        <div class="post-meta"> 
         <p class="post-meta">By <a href="https://www.mkyong.com/author/mkyong/" title="mkyong" rel="author">mkyong</a> | <time datetime="2013-07-07T00:12:20+00:00">July 7, 2013</time> | Updated : March 10, 2017 | Viewed : 1,027,401 times <span>+3,690 pv/w</span></p>

        <div class="post-content"> 
         <p>In this tutorial, we will show you how to convert a String to <code>java.util.Date</code>. Many Java beginners are stuck in the Date conversion, hope this summary guide will helps you in some ways.</p> 
         <pre><code class="language-java">
    // String -&gt; Date
    SimpleDateFormat.parse(String);

    // Date -&gt; String
    SimpleDateFormat.format(date);
</code></pre>

         <h2>1. String = 7-Jun-2013</h2> 
         <p>If 3 ‘M’, then the month is interpreted as text (Mon-Dec), else number (01-12).</p> 
         <div class="filename">
          TestDateExample1.java
         </div> 
         <pre><code class="language-java">
package com.mkyong.date;

import java.text.ParseException;
import java.text.SimpleDateFormat;
import java.util.Date;

public class TestDateExample1 {

    public static void main(String[] argv) {

        SimpleDateFormat formatter = new SimpleDateFormat("dd-MMM-yyyy");
        String dateInString = "7-Jun-2013";

        try {

            Date date = formatter.parse(dateInString);
            System.out.println(date);
            System.out.println(formatter.format(date));

        } catch (ParseException e) {
            e.printStackTrace();
        }

    }

}
</code></pre> 
         <p>Output</p> 
         <pre><code class="language-bash">
Fri Jun 07 00:00:00 MYT 2013
07-Jun-2013
</code></pre>

         <h2>5. String = 2014-10-05T15:23:01Z</h2> 
         <p>Z suffix means UTC, <code>java.util.SimpleDateFormat</code> doesn’t parse it correctly, you need to replace the suffix Z with ‘+0000’. </p> 
         <div class="filename">
          TestDateExample5.java
         </div> 
         <pre><code class="language-java">
package com.mkyong.date;

import java.text.ParseException;
import java.text.SimpleDateFormat;
import java.util.Date;

public class TestDateExample5 {

    public static void main(String[] argv) {

        SimpleDateFormat formatter = new SimpleDateFormat("yyyy-MM-dd'T'HH:mm:ssZ");
        String dateInString = "2014-10-05T15:23:01Z";

        try {

            Date date = formatter.parse(dateInString.replaceAll("Z$", "+0000"));
            System.out.println(date);

            System.out.println("time zone : " + TimeZone.getDefault().getID());
            System.out.println(formatter.format(date));

        } catch (ParseException e) {
            e.printStackTrace();
        }

    }

}
</code></pre> 
         <p>Output</p> 
         <pre><code class="language-bash">
Sun Oct 05 23:23:01 MYT 2014
time zone : Asia/Kuala_Lumpur
2014-10-05T23:23:01+0800
</code></pre> 
         <p>In Java 8, you can convert it into a <code>java.time.Instant</code> object, and display it with a specified time zone.</p> 
         <div class="filename">
          TestDateExample6.java
         </div> 
         <pre><code class="language-java">
package com.mkyong.date;

import java.time.*;

public class TestDateExample6 {

    public static void main(String[] argv) {

        String dateInString = "2014-10-05T15:23:01Z";

        Instant instant = Instant.parse(dateInString);

        System.out.println(instant);

        //get date time only
        LocalDateTime result = LocalDateTime.ofInstant(instant, ZoneId.of(ZoneOffset.UTC.getId()));

        System.out.println(result);

        //get date time + timezone
        ZonedDateTime zonedDateTime = instant.atZone(ZoneId.of("Africa/Tripoli"));
        System.out.println(zonedDateTime);

        //get date time + timezone
        ZonedDateTime zonedDateTime2 = instant.atZone(ZoneId.of("Europe/Athens"));
        System.out.println(zonedDateTime2);
        
    }

}
</code></pre> 
         <p>Output</p> 
         <pre><code class="language-bash">
2014-10-05T15:23:01Z
2014-10-05T15:23:01
2014-10-05T17:23:01+02:00[Africa/Tripoli]
2014-10-05T18:23:01+03:00[Europe/Athens]
</code></pre>
