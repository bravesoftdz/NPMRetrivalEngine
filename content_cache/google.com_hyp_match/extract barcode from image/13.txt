         <h1 class="entry-title">How to Build Node.js Barcode Reader on Raspberry Pi</h1> 
         <div class="entry-meta"> 
          <!-- author and date--> 
          <span class="byline"><span class="author vcard"><a class="url fn n" href="http://www.codepool.biz/author/xiao-ling">Xiao Ling</a></span></span>&nbsp;/&nbsp;

          <span class="comments-link"><a href="http://www.codepool.biz/raspberry-pi-nodejs-barcode-reader.html#respond"><span class="dsq-postid" rel="4448 http://www.codepool.biz/?p=4448">Leave comment</span></a></span>

        <div class="entry-content"> 
         <p>Node.js is available for all platforms including IoT devices like Raspberry Pi. In this post, I will demonstrate how to quickly create an online barcode reader on Raspberry Pi using Node.js and Dynamsoft Barcode Reader SDK.</p> 
         <p><span id="more-4448"></span></p> 
         <h2>Node.js for Raspberry Pi</h2> 
         <p>If you flashed Rasbian Jessie, the latest Linux ROM for Raspberry Pi, you might have noticed that Node.js is pre-installed by default. Nevertheless, the Node version is out of date. To get new features of Node.js, download the latest version.</p> 
         <p>Here are the steps to install Node.js for Raspberry Pi:</p> 
         <ol> 
          <li>Remove the pre-installed Node.js: <pre class="brush:shell">sudo apt-get remove node</pre> </li> 
          <li>Check Pi hardware information: <pre class="brush:shell">uname –m
armv7l</pre> </li> 
          <li>Download the matched <a href="https://nodejs.org/en/download/" target="_blank">Node version for ARM</a>.</li> 
          <li>Extract the package: <pre class="brush:shell">tar –xf node-v4.4.1-linux-armv7l.tar.xz</pre> </li> 
          <li>Add the bin path to ~/.bashrc: <pre class="brush:shell">vim ~/.bashrc
export PATH=$PATH:/home/pi/node-v4.4.0-linux-armv7l/bin
source ~/.bashrc</pre> </li> 
          <li>Check whether Node can work: <pre class="brush:shell">node –v
v4.4.0</pre> </li> 
         </ol> 
         <h2>Building Online Barcode Reader</h2> 
         <p>Get barcode SDK. The Dynamsoft Barcode Reader SDK for Raspberry Pi is not publicly available yet. To get a trial version, contact <a href="mailto:support@dynamsoft.com">support@dynamsoft.com</a>.</p> 
         <p>Extract the package and generate a symbolic link:</p> 
         <pre class="brush:shell">sudo ln -s &lt;DynamsoftBarcodeReader&gt;/lib/libDynamsoftBarcodeReader.so /usr/lib/libDynamsoftBarcodeReader.so</pre> 
         <p>Install <strong>node-gyp</strong> for building Node C/C++ addons:</p> 
         <pre class="brush:shell">npm install -g node-gyp</pre> 
         <p>Create <strong>dbr.cc</strong> and <strong>binding.gyp</strong>:</p> 
         <pre class="brush:shell">        
    // dbr.cc
    Isolate* isolate = Isolate::GetCurrent();
	HandleScope scope(isolate);

	// convert v8 string to char *
	String::Utf8Value license(args[0]-&gt;ToString());
	String::Utf8Value fileName(args[1]-&gt;ToString());
	char *pFileName = *fileName;
	char *pszLicense = *license;
	// Dynamsoft Barcode Reader: init
	__int64 llFormat = args[2]-&gt;IntegerValue();
	int iMaxCount = 0x7FFFFFFF;
	ReaderOptions ro = {0};
	pBarcodeResultArray pResults = NULL;

	// Initialize license
	if (pszLicense &amp;&amp; (strcmp(pszLicense, "null") !=0) )
	{
		printf("license: %s\n", pszLicense);
		DBR_InitLicense(pszLicense);
	}
	else {
		#ifdef LINUX_DBR
				printf("Linux dbr license\n");
				DBR_InitLicense("&lt;license&gt;");
		#endif
	}

	ro.llBarcodeFormat = llFormat;
	ro.iMaxBarcodesNumPerPage = iMaxCount;

	// Decode barcode image
	int ret = DBR_DecodeFile(pFileName, &amp;ro, &amp;pResults);</pre> 
         <pre class="brush:shell">// binding.gyp
{
  "targets": [
    {
      'target_name': "dbr",
      'sources': [ "dbr.cc" ],
      'conditions': [
          ['OS=="linux"', {
            'defines': [
              'LINUX_DBR',
            ],
            'include_dirs': [
                "/home/pi/Desktop/dbr/include"
            ],
            'libraries': [
                "-lDynamsoftBarcodeReader", "-L/home/pi/Desktop/dbr/lib"
            ]
          }]
      ]
    }
  ]
}</pre> 
         <p>Build <strong>dbr.so</strong>:</p> 
         <pre class="brush:shell">node-gyp build</pre> 
         <p>Install Node modules <strong>express</strong> and <strong>formidable</strong> for rapidly making a Web project:</p> 
         <pre class="brush:shell">npm install express formidable</pre> 
         <p>Use the barcode module in <strong>server.js</strong>:</p> 
         <pre class="brush:shell">var dbr = require('./build/Release/dbr');

dbr.decodeFile(

    license, fileName, barcodeType,

    function(msg) {

        var response = 'Totol count: ' + msg.length;

        var result = null;

        for (index in msg) {

            result = msg[index]

            response += '&lt;p&gt;' + result['format'] + ': ';

            response += result['value'] + '&lt;p&gt;';

        }

        res.send(response);

    }

);</pre> 
         <p>Start the server:</p> 
         <pre class="brush:shell">node server.js</pre> 
         <p>Visit&nbsp;<strong>http://localhost:2016/index.htm</strong>:</p> 
         <p><img class="alignnone size-large wp-image-4449" src="http://www.codepool.biz/wp-content/uploads/2016/03/rpi_node_barcode-1-1024x576.png" alt="raspberry pi node.js barcode reader" width="640" height="360" srcset="http://www.codepool.biz/wp-content/uploads/2016/03/rpi_node_barcode-1-300x169.png 300w, http://www.codepool.biz/wp-content/uploads/2016/03/rpi_node_barcode-1-768x432.png 768w, http://www.codepool.biz/wp-content/uploads/2016/03/rpi_node_barcode-1-1024x576.png 1024w, http://www.codepool.biz/wp-content/uploads/2016/03/rpi_node_barcode-1.png 1600w" sizes="(max-width: 640px) 100vw, 640px"></p>
