       <h3 class="title sp-2"><a href="http://badassjs.com/post/654334959/barcode-scanning-in-javascript">Barcode Scanning In JavaScript</a></h3>
       <h4 class="textDate "><span>June 1st 2010</span></h4>
       <div class="mainText">
        <p><a title="@tobeytaylor on Twitter" href="http://twitter.com/tobeytailor" target="_blank">Tobias&nbsp;Schneider</a>&nbsp;(of <a href="http://t.umblr.com/redirect?z=http%3A%2F%2Fgithub.com%2Ftobeytailor%2Fgordon&amp;t=N2ZjZmZkMzU4OWNkMjQzNzY4YjU0ZWNmM2VlNzdiMGVlMGM5MTg3YSxKNHFEdkxYbA%3D%3D&amp;b=t%3AGINb_asixaSGG1PcWl6n7w&amp;p=http%3A%2F%2Fbadassjs.com%2Fpost%2F654334959%2Fbarcode-scanning-in-javascript&amp;m=1" target="_blank">Gordon</a> fame) has written a barcode scanner in JavaScript in response to <a href="http://twitter.com/rem/status/15102650936" target="_blank">Remy Sharp’s request</a> on Twitter (only yesterday). &nbsp;It uses canvas to read the pixels from an image, and prints out the UPC code that the barcode represents. &nbsp;Since JavaScript doesn’t normally have access to the webcam, its practical use is rather limited. &nbsp;However, there are some places where this could be useful - such as in a <a href="http://t.umblr.com/redirect?z=http%3A%2F%2Fwiki.phonegap.com%2FiPhone%3A-Camera-API&amp;t=NzM4NDc2YWZmNWViZjliOTY1ZTEzNmQ4ODE4NGIxOGE5NDNkZWQyOCxKNHFEdkxYbA%3D%3D&amp;b=t%3AGINb_asixaSGG1PcWl6n7w&amp;p=http%3A%2F%2Fbadassjs.com%2Fpost%2F654334959%2Fbarcode-scanning-in-javascript&amp;m=1" target="_blank">PhoneGap application</a>, and eventually we may have an <a href="http://t.umblr.com/redirect?z=http%3A%2F%2Fajaxian.com%2Farchives%2Fw3c-capture-api-our-in-draft&amp;t=OGVkYTMwYmNhYTlkNzBmMzkyZjU1YjdjNDY2MjEwMzkzZjA3YjMxYixKNHFEdkxYbA%3D%3D&amp;b=t%3AGINb_asixaSGG1PcWl6n7w&amp;p=http%3A%2F%2Fbadassjs.com%2Fpost%2F654334959%2Fbarcode-scanning-in-javascript&amp;m=1" target="_blank">API</a> for accessing the webcam in desktop browsers.</p>
        <p>You can check out the code on <a href="http://t.umblr.com/redirect?z=http%3A%2F%2Fgist.github.com%2F421369&amp;t=NmIwY2VlZjc3NzMyYjI5ODI5MGM4NWYyZDI1ZTlmYmZkNjI2ZmRlOSxKNHFEdkxYbA%3D%3D&amp;b=t%3AGINb_asixaSGG1PcWl6n7w&amp;p=http%3A%2F%2Fbadassjs.com%2Fpost%2F654334959%2Fbarcode-scanning-in-javascript&amp;m=1" target="_blank">Github</a>, and see a demo&nbsp;<a href="http://t.umblr.com/redirect?z=http%3A%2F%2Ftobeytailor.s3.amazonaws.com%2Fget_barcode_from_image%2Findex.html&amp;t=MzhkNjFmNzA0ZWE2Y2U4MmI2MDNmM2RjZjRhYTQ2OTNhMzYxMzg5ZixKNHFEdkxYbA%3D%3D&amp;b=t%3AGINb_asixaSGG1PcWl6n7w&amp;p=http%3A%2F%2Fbadassjs.com%2Fpost%2F654334959%2Fbarcode-scanning-in-javascript&amp;m=1" target="_blank">here</a>.</p>
