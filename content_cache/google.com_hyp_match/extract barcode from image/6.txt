       </svg></a>What is Barcode Xpress?</h2> 
     <p>Barcode Xpress for Node.js enables developers to add the ability to read common industry 1D and 2D barcodes in their applications.</p> 
     <ul> 
      <li><strong>Barcode Types</strong> - recognize over 30 different types of barcodes.</li> 
      <li><strong>Structured Results</strong> - results are organized in a hierarchical model where every barcode has a value, type, confidence, and location.</li> 
      <li><strong>Simple</strong> - by providing only a minimal asynchronous API, Barcode Xpress is easy to learn. With a few lines of code, recognize barcodes found in images produced by scanners, cameras, or fax machines. ## Read Barcodes with Fewer than 20 Lines of Code</li> 
     </ul> 
     <ol> 
      <li>Make sure you have the following system requirements: 
       <ul> 
        <li>Linux 64 bit.</li> 
        <li><code>node -v</code> 4.0 or greater and npm.</li> 
        <li><code>gcc --version</code> 4.8.1 or greater.</li> 
        <li><code>g++ -v</code> 4.8.1 or greater.</li> 
        <li><code>java -version</code> JRE 1.7 or later.</li> 
        <li><code>libstdc++-static</code> <em>(CentOS only)</em> 4.8.1 or greater</li> 
       </ul> </li> 
      <li>Register an email address for a <a href="https://www.accusoft.com/products/barcode-xpress/get-it/">free account</a> to get access to our Barcode Xpress evaluation version</li> 
      <li><code>npm install barcode-js</code></li> 
      <li>Run the license utility to install an eval licensing: 
       <ul> 
        <li>GUI registration: <code>cd node_modules/barcode-js &amp;&amp; npm run register</code></li> 
        <li>CLI registration: <code>cd node_modules/barcode-js &amp;&amp; npm run register -- youremail@domain.com</code></li> 
       </ul> </li> 
      <li>Use the following to get a taste of what Barcode Xpress can do:</li>
