              <pre><code>// send sms 

var springedge = require('springedge');

var params = {
  'apikey': '', // API Key 
  'sender': 'SEDEMO', // Sender Name 
  'to': [
    '919019xxxxxxxx'  //Moblie Number 
  ],
  'message': 'test+message'
};

springedge.messages.send(params, 5000, function (err, response) {
  if (err) {
    return console.log(err);
  }
  console.log(response);
});
</code></pre>

            <td class="answercell"> 
             <div class="post-text" itemprop="text"> 
              <p>To ensure you can reach a users mobile no matter their location and network it's likely you're going to have to look at a paid service such as <a href="https://www.nexmo.com" rel="nofollow noreferrer">Nexmo</a> (who I work for) or <a href="https://www.twilio.com" rel="nofollow noreferrer">Twilio</a>.</p> 
              <p>With these services you can either build your own verification (2FA - Two Factor Authentication) workflow:</p> 
              <ol> 
               <li>User enters their phone number in a form and submits to your app</li> 
               <li>Your app receives the phone number</li> 
               <li>You send the user an SMS with an auth code</li> 
               <li>User receives the auth code</li> 
               <li>User enters auth code into a form and submits to your app</li> 
               <li>Your app receives the auth code and checks that auth code against the phone number (maybe using the current session phone number)</li> 
               <li>If the auth code is the one you are expected then the user is validated</li> 
              </ol> 
              <p>Or you can use their 2FA authentication products (Nexmo - <a href="https://www.nexmo.com/products/verify" rel="nofollow noreferrer">Verify</a> or Twilio - <a href="https://www.twilio.com/authy" rel="nofollow noreferrer">Authy</a> that should help simplify this workflow.</p> 
              <p>Using Nexmo verify the code would be:</p> 
              <h3>Send Verification Request</h3> 
              <pre><code>var Nexmo = require('nexmo');
var nexmo = new Nexmo({apiKey: API_KEY, apiSecret: API_SECRET});

var verifyRequestId = null; // use in the check process

nexmo.verify.request({number: TO_NUMBER, brand: APP_NAME}, function(err, result) {
  if(err) { console.error(err); }
  else {
    verifyRequestId = result. request_id;
  }
});
</code></pre> 
              <h2>Check Authentication Code</h2> 
              <pre><code>nexmo.verify.control({request_id: verifyRequestId, cmd: 'cancel'}, function(err, result) {
  if(err) { console.error(err); }
  else {
    console.log(result);
  }
});
</code></pre>
