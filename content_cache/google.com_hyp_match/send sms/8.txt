           <p>No need to tie a string around your finger.            <h3>List of Ingredients</h3> 
           <p>We’ll be using a few tools to build this app. You’ll want to have these set up before you continue on:</p> 
           <ul> 
            <li><a href="http://twilio.com" onclick="__gaTracker('send', 'event', 'outbound-article', 'http://twilio.com', 'Twilio');">Twilio</a>: To send and receive SMS messages. Don’t have a Twilio account? <a href="https://www.twilio.com/try-twilio" onclick="__gaTracker('send', 'event', 'outbound-article', 'https://www.twilio.com/try-twilio', 'Sign up for Free');">Sign up for Free</a>.</li> 
            <li><a href="https://www.firebase.com/" onclick="__gaTracker('send', 'event', 'outbound-article', 'https://www.firebase.com/', 'Firebase');">Firebase</a>: A realtime database API. We’ll be using it to store the users who have subscribed to our service.</li> 
            <li><a href="http://nodejs.org/" onclick="__gaTracker('send', 'event', 'outbound-article', 'http://nodejs.org/', 'Node.js');">Node.js</a>: A platform built on Chrome’s JavaScript runtime for easily building fast, scalable network applications.</li> 
           </ul> 
           <h3>Scheduling SMS Messages with Cron</h3>
