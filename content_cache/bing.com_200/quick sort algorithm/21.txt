         <h1 class="title single-title">Bubble Sort Algorithm and its Implementations</h1> 
         <!-- End Title --> 
         <!-- Start Post Meta -->
         <div class="post-info">
          <span class="theauthor"><a href="https://www.codingeek.com/author/hitesh-garg/" title="Posts by Hitesh Garg" rel="author">Hitesh Garg</a></span> |

         <p><strong>Bubble sort is the basic sorting algorithm which continuously compares the adjacent pairs and swaps them if they are in wrong order</strong><strong>.</strong></p>
         <p>This algorithm is generally used to introduce algorithmic concepts to a beginner or is used in cases when the input list or array is almost sorted and have only a few elements misplaced from their actual location and that too at nearby locations.</p>
         <p>Comparing to other simple O(n<sup>2</sup>) sorting algorithms like selection and insertion sort, bubble sort takes too many resources and time, hence its performance is not appropriate. In general, we prefer to use insertion sort and sometimes selection over Bubble sort algorithm.</p>
         <div class="whpejlIf" style="clear:both;float:left;width:100%;margin:0 0 20px 0;">
          <script async src="//pagead2.googlesyndication.com/pagead/js/adsbygoogle.js"></script>

           <ins class="adsbygoogle" style="display:inline-block;width:336px;height:280px" data-ad-client="ca-pub-4829687731530978" data-ad-slot="3865296240"></ins>

         <p>The basic idea is that in first iteration of this algorithm we are able to move the largest element at the end of the list, in second iteration- second largest element at second last position in list and so on. Basically like selection sort it also divides the array into two parts of sorted and unsorted array and after each iteration we add the largest element from the unsorted array to the top of sorted array.</p>
         <hr>
         <p><strong>Let’s understand it with an example-</strong></p>
         <figure style="width: 300px" class="wp-caption aligncenter">
          <a href="http://www.codingeek.com/wp-content/uploads/2015/05/word-image1.gif"><img class="size-full wp-image-624" src="http://www.codingeek.com/wp-content/uploads/2015/05/word-image1.gif" alt="Bubble Sort Algorithm -Example" width="300" height="180"></a>
          <figcaption class="wp-caption-text">
           Bubble Sort Algorithm -Example
          </figcaption>
         </figure>
         <p><strong>So in the above picture we can see that </strong><strong>in each iteration we continuously check adjacent elements and shift the bigger one towards the end of the list and finally the largest element is placed at the end and we continue this iteration until we sort our complete list.</strong></p>
         <p>For the array of numbers “5 1 4 2 8”, let’s sort the array from lowest number to the greatest number using bubble sort. In each step, elements written in <b>bold</b> are being compared.</p>
         <p><strong>Example:</strong><br> <b>First Pass:</b><br> (<strong> 5 1 </strong>4 2 8 ) –&gt; ( <b>1</b> <b>5</b> 4 2 8 ), Here, the algorithm compares the first two elements, and swaps since 5 &gt; 1.<br> ( 1 <b>5</b> <b>4</b> 2 8 ) –&gt; ( 1 <b>4</b> <b>5</b> 2 8 ), Swap since 5 &gt; 4<br> ( 1 4 <b>5</b> <b>2</b> 8 ) –&gt; ( 1 4 <b>2</b> <b>5</b> 8 ), Swap since 5 &gt; 2<br> ( 1 4 2 <b>5</b> <b>8</b> ) –&gt; ( 1 4 2 <b>5</b> <b>8</b> ), Now, since these elements are already in order (8 &gt; 5), the algorithm does not swap them.</p>
         <p style="color: #000000;"><b>Second Pass:</b><br> ( <b>1</b> <b>4</b> 2 5 8 ) –&gt; ( <b>1</b> <b>4</b> 2 5 8 )<br> ( 1 <b>4</b> <b>2</b> 5 8 ) –&gt; ( 1 <b>2</b> <b>4</b> 5 8 ), Swap since 4 &gt; 2<br> ( 1 2 <b>4</b> <b>5</b> 8 ) –&gt; ( 1 2 <b>4</b> <b>5</b> 8 )<br> ( 1 2 4 <b>5</b> <b>8</b> ) –&gt; ( 1 2 4 <b>5</b> <b>8</b> )<br> Now, the array is already sorted, but our algorithm does not know if it is completed. The algorithm needs one <b>whole</b> pass without <b>any</b> swap to know it is sorted.</p>
         <p style="color: #000000;"><b>Third Pass:</b><br> ( <strong>1 2</strong> 4 5 8 ) –&gt; ( <b>1</b> <b>2</b> 4 5 8 )<br> ( 1 <b>2</b> <b>4</b> 5 8 ) –&gt; ( 1 <b>2</b> <b>4</b> 5 8 )<br> ( 1 2 <b>4</b> <b>5</b> 8 ) –&gt; ( 1 2 <b>4</b> <b>5</b> 8 )<br> ( 1 2 4 <b>5</b> <b>8</b> ) –&gt; ( 1 2 4 <b>5</b> <b>8</b> )</p>

         <h5><span class="ez-toc-section" id="iterative_pseudocode">Iterative Pseudocode:</span></h5>
         <pre class="brush: cpp; title: ; notranslate" title="">
function bubblesort( var a as array )
    for i from 1 to N
        for j from 0 to N - 1
           if a[j] &gt; a[j + 1]
              swap( a[j], a[j + 1] )
end function

</pre>

         <h5><span class="ez-toc-section" id="optimized_pseudocode">Optimized Pseudocode:</span></h5>
         <p>In optimized approach we use keep track and check is there any iteration in which no element is swapped and if this happens then it implies that the list is sorted and this situation is tracked by <code>isSorted </code>variable in our code as below.</p>
         <pre class="brush: cpp; title: ; notranslate" title="">
function bubblesort2( var a as array )
    for i from 2 to N
        isSorted = true
        for j from 0 to N - 2
           if a[j] &gt; a[j + 1]
              swap( a[j], a[j + 1] )
              isSorted = false
        if isSorted == true
            break
end function

</pre>

         <h4><span class="ez-toc-section" id="asymptotic_analysis">Asymptotic Analysis</span></h4>
         <p style="color: #555555;">Since this algorithm contains two nested loops and the number of comparisons is not dependent on the values of array items, so the complexity of this algorithm is O(n<sup>2</sup>). Summarizing all this –</p>
         <p style="color: #555555;"><strong style="font-style: inherit;">Data structure used</strong> -&gt; Array<br> <strong style="font-style: inherit;">Time Complexity (Best, worst and average case)</strong> -&gt; <strong style="font-style: inherit;">O(n<sup><span style="font-weight: inherit; font-style: inherit;">2</span></sup>)<br> Worst case space complexity</strong> -&gt; O(n) total space, O(1) Auxiliary space</p>
         <hr>
         <h4 id="1"><span class="ez-toc-section" id="iterative_implementation_of_bubble_sort_in_c_programming_language">ITERATIVE Implementation of Bubble Sort in C programming language</span></h4>
         <pre class="brush: cpp; highlight: [4,12,22,35]; title: ; notranslate" title="">
// C program for implementation of Bubble sort
#include &lt;stdio.h&gt;

void swap(int *xp, int *yp)
{
    int temp = *xp;
    *xp = *yp;
    *yp = temp;
}

// A function to implement bubble sort
void bubbleSort(int arr[], int n)
{
   int i, j;
   for (i = 0; i &lt; n-1; i++)
       for (j = 0; j &lt; n-i-1; j++) //Last i elements are already in place
           if (arr[j] &gt; arr[j+1])
              swap(&amp;arr[j], &amp;arr[j+1]);
}

/* Function to print an array */
void printArray(int arr[], int size)
{
    int i;
    for (i=0; i &lt; size; i++)
        printf("%d ", arr[i]);
    printf("\n");
}

// Driver program to test above functions
int main()
{
    int arr[] = {64, 34, 25, 12, 22, 11, 90};
    int n = sizeof(arr)/sizeof(arr[0]);
    bubbleSort(arr, n);
    printf("Sorted array: \n");
    printArray(arr, n);
    return 0;
}
</pre>

         <h4 id="2"><span class="ez-toc-section" id="iterative_implementation_of_bubble_sortin_java">ITERATIVE Implementation of Bubble Sort&nbsp;in Java</span></h4>
         <pre class="brush: java; highlight: [6,20]; title: ; notranslate" title="">
package com.codingeek.algorithms.sorting;

public class BubbleSortAlgorithm {

	// Implementation of Bubble Sort
	static void bubblesort(Integer[] input) {
		for (int i = 0; i &lt; input.length - 1; i++)
			for (int j = 0; j &lt; input.length - 1; j++) {
				if (input[j].intValue() &gt; input[j + 1].intValue()) {
					int temp = input[j];
					input[j] = input[j + 1];
					input[j + 1] = temp;
				}
			}
	}

	// Main method to launch the program
	public static void main(String[] args) {
		Integer[] input = { 64, 34, 25, 12, 22, 11, 90 };
		bubblesort(input);
		System.out.println("Sorted array:");
		for (Integer value : input)
			{
			System.out.print(value + " ");
			}
	}
}

         <h4 id="3"><span class="ez-toc-section" id="optimized_implementation_of_bubble_sort_in_c">Optimized Implementation of Bubble Sort in C</span></h4>
         <pre class="brush: cpp; highlight: [9,15,18]; title: ; notranslate" title="">
// Optimized implementation of Bubble sort
#include &lt;stdio.h&gt;
#include&lt;stdbool.h&gt; // To use bool variable in our code
void swap(int *xp, int *yp)
{
    int temp = *xp;
    *xp = *yp;
    *yp = temp;
}

// An optimized version of Bubble Sort
void bubbleSort(int arr[], int n)
{
   int i, j;
   bool isSorted;
   for (i = 0; i &lt; n-1; i++)
   {
     isSorted = true;
     for (j = 0; j &lt; n-i-1; j++)
     {
        if (arr[j] &gt; arr[j+1])
        {
           swap(&amp;arr[j], &amp;arr[j+1]);
           isSorted = false;
        }
     }

     // IF no two elements were swapped by inner loop, then break
     if (isSorted)
        break;
   }
}

/* Function to print an array */
void printArray(int arr[], int size)
{
    int i;
    for (i=0; i &lt; size; i++)
        printf("%d ", arr[i]);
    printf("\n");
}

// Driver program to test above functions
int main()
{
    int arr[] = {64, 34, 25, 12, 22, 11, 90};
    int n = sizeof(arr)/sizeof(arr[0]);
    bubbleSort(arr, n);
    printf("Sorted array: \n");
    printArray(arr, n);
    return 0;
}

         <h4 id="4"><span class="ez-toc-section" id="optimized_implementation_of_bubble_sort_in_java">Optimized Implementation of Bubble Sort in Java</span></h4>
         <pre class="brush: java; highlight: [9,15,18]; title: ; notranslate" title="">
package com.codingeek.algorithms.sorting;

public class BubbleSortAlgorithm {

	// Optimized Implementation of Bubble Sort
	static void optimizedBubblesort(Integer[] input) {
		for (int i = 0; i &lt; input.length - 1; i++) {

			boolean isSorted = true;
			for (int j = 0; j &lt; input.length - 1; j++) {
				if (input[j].intValue() &gt; input[j + 1].intValue()) {
					int temp = input[j];
					input[j] = input[j + 1];
					input[j + 1] = temp;
					isSorted = false;
				}
			}
			if (isSorted) {
				break;
			}
		}
	}

	// Main method to launch the program
	public static void main(String[] args) {
		Integer[] input = { 64, 34, 25, 12, 22, 11, 90 };
		optimizedBubblesort(input);
		System.out.println("Sorted Array -");
		for (Integer value : input) {
			System.out.print(value + " ");
		}
	}
}

</pre>

Sorted array:
11 12 22 25 34 64 90
