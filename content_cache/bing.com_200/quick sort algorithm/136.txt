        <h1 class="entry-title">Mergesort using ArrayList in Java with example</h1> 
       </header>
       <!-- .entry-header --> 
       <div class="entry-content"> 
        <p>So here is another sorting algorithm, <strong>Merge&nbsp;Sort</strong> which I have implemented it using ArrayList.</p> 
        <p>MergeSort follows the <strong>Divide and Conquer</strong>&nbsp;paradigm.</p> 
        <p><strong>Divide</strong> part divides an unsorted&nbsp;array into 2 unsorted arrays till further division is not possible i.e there is only 1 element per array. So we need to divide an array of N element into N arrays of size 1 (Virtually).</p> 
        <p><strong>Conquer</strong> part join 2 divided arrays into one array which is sorted. So we need to merge element from N array to 1 array having all element in sorted order.</p> 
        <p>Diving the array will take <strong>O(log n)</strong> time and will create <strong>log n&nbsp;</strong>level of sub arrays.<br> Conquer part at each level will merge 2 sorted arrays which takes <strong>O(n)&nbsp;</strong>at each level.<br> So worst case time taken by merge sort is <strong>O(n log n)</strong>.<br> <strong>Merge sort is not considered to be a memory efficient as we need an extra memory at each level of merging</strong>. As you can see in the implementation example that is given here it needs an extra temporary array called :&nbsp;<strong>mergedSortedArray&nbsp;</strong>to store the temp sorted element.</p> 
        <p>Here is the source code and project structure for the <strong>MergeSort.</strong></p> 
        <p>&nbsp;</p> 
        <p><strong>Project Structure</strong></p> 
        <p><a href="http://www.snippetexample.com/wp-content/uploads/2014/10/ProjectExporer.png"><img class="alignnone size-medium wp-image-138" src="http://www.snippetexample.com/wp-content/uploads/2014/10/ProjectExporer-188x300.png" alt="ProjectExporer" width="188" height="300" srcset="http://www.snippetexample.com/wp-content/uploads/2014/10/ProjectExporer-188x300.png 188w, http://www.snippetexample.com/wp-content/uploads/2014/10/ProjectExporer.png 241w" sizes="(max-width: 188px) 100vw, 188px"></a></p> 
        <p>&nbsp;</p> 
        <p><strong>MergeSort.java</strong></p> 
        <pre class="brush:java">package daa;

import java.util.ArrayList;

public class MergeSort {
	private ArrayList&lt;Integer&gt; inputArray;
	
	public ArrayList&lt;Integer&gt; getSortedArray() {
		return inputArray;
	}

	public MergeSort(ArrayList&lt;Integer&gt; inputArray){
		this.inputArray = inputArray;
	}
	
	public void sortGivenArray(){		
		divide(0, this.inputArray.size()-1);
	}
	
	public void divide(int startIndex,int endIndex){
		
		//Divide till you breakdown your list to single element
		if(startIndex&lt;endIndex &amp;&amp; (endIndex-startIndex)&gt;=1){
			int mid = (endIndex + startIndex)/2;
			divide(startIndex, mid);
			divide(mid+1, endIndex);		
			
			//merging Sorted array produce above into one sorted array
			merger(startIndex,mid,endIndex);			
		}		
	}	
	
	public void merger(int startIndex,int midIndex,int endIndex){
		
		//Below is the mergedarray that will be sorted array Array[i-midIndex] , Array[(midIndex+1)-endIndex]
		ArrayList&lt;Integer&gt; mergedSortedArray = new ArrayList&lt;Integer&gt;();
		
		int leftIndex = startIndex;
		int rightIndex = midIndex+1;
		
		while(leftIndex&lt;=midIndex &amp;&amp; rightIndex&lt;=endIndex){
			if(inputArray.get(leftIndex)&lt;=inputArray.get(rightIndex)){
				mergedSortedArray.add(inputArray.get(leftIndex));
				leftIndex++;
			}else{
				mergedSortedArray.add(inputArray.get(rightIndex));
				rightIndex++;
			}
		}		
		
		//Either of below while loop will execute
		while(leftIndex&lt;=midIndex){
			mergedSortedArray.add(inputArray.get(leftIndex));
			leftIndex++;
		}
		
		while(rightIndex&lt;=endIndex){
			mergedSortedArray.add(inputArray.get(rightIndex));
			rightIndex++;
		}
		
		int i = 0;
		int j = startIndex;
		//Setting sorted array to original one
		while(i&lt;mergedSortedArray.size()){
			inputArray.set(j, mergedSortedArray.get(i++));
			j++;
		}
	}
}
