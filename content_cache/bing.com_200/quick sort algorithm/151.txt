               Quicksort algorithm is one of the most used sorting algorithm, especially to sort large list and most of the programming languages, library have implemented it in one or another way. In Java, 
              <span style="font-family: &quot;courier new&quot; , &quot;courier&quot; , monospace;">Arrays.sort()</span> method sorts primitive data types using double pivot&nbsp;
              <span style="font-family: inherit;">Quicksort&nbsp;</span>algorithm, authored by Joshua Bloach and others. This implementation provides better performance for lot of data sets, where traditional quicksort algorithm reduced into quadratic performance. This method also uses 
              <span style="font-family: &quot;courier new&quot; , &quot;courier&quot; , monospace;">MergeSort,</span>
              <span style="font-family: inherit;"> another good sorting algorithm,&nbsp;</span>to sort objects. QuickSort implementations are also available in C++ STL library. Have you ever thought 
              <i>why quicksort is so popular?</i> because on average it is one of the fastest sorting algorithm we have. On average quicksort is a O(n log n) algorithm, while it's worst case is&nbsp;O(n^2), which is much better comparing with Bubble Sort or Insertion Sort. It's also one of the 
              <a href="http://javarevisited.blogspot.sg/2013/03/top-15-data-structures-algorithm-interview-questions-answers-java-programming.html">popular algorithm interview question</a>, so as a programmer you must know 
              <i>how QuickSort works as well</i> as 
              <i>how to implement Quicksort in Java</i> or any other programming language. One of the most important thing interviewer look in your quicksort implementation is choice of pivot and whether you are sorting in place or not. In "
              <i>in-place"</i> sorting, actual sorting takes place in same array and no additional space is needed. Due to this reason, quicksort is very efficient in sorting large list of numbers, as no additional memory is required, a very space efficient sorting algorithm. Quicksort is also one of the naturally recursive algorithm and serves a good exercise for Java programmers to master 
              <a href="http://javarevisited.blogspot.sg/2012/12/recursion-in-java-with-example-programming.html">art of recursion</a>.
              <br> 
              <a name="more"></a>
              <br> 
              <br> 
              <h3> <u>How QuickSort Algorithm works</u></h3> Quicksort is a divide and conquer algorithm, which means original list is divided into multiple list, each of them is sorted individually and then sorted output is merged to produce the sorted list. Here is step by step explanation of how quicksort algorithm works.
              <br> 
              <br> 
              <div style="display: table; margin-left: auto; margin-right: auto "> 
               <script async src="//pagead2.googlesyndication.com/pagead/js/adsbygoogle.js"></script>

              <br> Steps to implement Quick sort algorithm in place:
              <br> 
              <br> 1) Choose an element, called pivot, from the list or array. Generally pivot is the middle element of array.
              <br> 
              <br> 2) Reorder the list so that all elements with values less than the pivot come before the pivot, and all elements with values greater than the pivot come after it (equal values can go either way). This is also known as 
              <i>partitioning</i>. After partitioning the pivot is in its final position.
              <br> 
              <br> 3) Recursively apply the above steps to the sub-list of elements with smaller values and separately the sub-list of elements with greater values. If the array contains only one element or zero elements then the array is sorted.
              <br> 
              <br> Following GIF image will help you to understand 
              <a href="http://java67.blogspot.sg/2014/07/quicksort-algorithm-in-java-in-place-example.html">working of Quick sort algorithm</a> little better. In this image we have an array of integers which is not sorted and we need to sort them in ascending order.&nbsp;Our array is 
              <span style="font-family: &quot;courier new&quot; , &quot;courier&quot; , monospace;">{6, 5, 3, 1, 8, 7, 2, 4}</span> and we first choose 3 as pivot. Now partitioning starts and we pick 6 on left side of side, because its greater than 3. Now on right side, we leave 4 because its greater than 3, and pick 2 for swapping with 6. After swapping our list look like 
              <span style="font-family: &quot;courier new&quot; , &quot;courier&quot; , monospace;">{2, 5, 3, 1, 8, 7, 6, 4}</span>. Now we pick 5 on left side, and 1 on right side because it's greater than 3 and swap them again. Now, our array looks like 
              <span style="font-family: &quot;courier new&quot; , &quot;courier&quot; , monospace;">{2, 1, 3, 5, 8, 7, 6, 4}</span>. Since we are done with all elements with respect to 3 as pivot, we can now take the sub-array at left side of 3 and apply same procedure. This will sort the left array. Now on right side, we choose 4 as pivot, and repeat same procedure, which result in 4 swapped against 5. Now we take right side again with 6 as pivot and apply same procedure.
              <br>

              <div style="clear: both; text-align: center;"> 
               <b><u>Sorting an array of integer using QuickSort sorting algorithm</u></b>
              </div> 
              <br> 
              <br> 
              <h3 style="text-align: left;"> <u>Java Program to implement QuickSort Algorithm</u></h3> Here is a Java program to 
              <a href="http://java67.blogspot.sg/2014/08/4-examples-to-sort-array-in-java.html" rel="nofollow">sort an array of integers using QuickSort algorithm</a>. It is an in-place, recursive implementation of QuickSort. Logic is encapsulated in&nbsp;
              <span style="font-family: &quot;courier new&quot; , &quot;courier&quot; , monospace;">QuickSort</span>&nbsp;class, and method&nbsp;
              <span style="font-family: &quot;courier new&quot; , &quot;courier&quot; , monospace;">quickSort(int low, int high)</span>. This method is called recursively to sort the array. This algorithm work exactly as explained in above GIF image, so if you understand the logic there, its very easy to write by your own.
              <br> 
              <br> 
              <br> 
              <pre style="background: rgb(255, 255, 255); font-family: Consolas, 'Lucida Console', 'DejaVu Sans Mono', Monaco, 'Courier New', monospace; font-size: 10pt; line-height: 1.5em; padding: 4px;"><span style="color: #ff7800;">import</span> <span style="color: #ff7800;">java.util.Arrays</span>;

<span style="color: #8c868f;">/**

 * <span style="color: #ff7800;">@author</span> Javin Paul

<span style="color: #ff7800;">public</span> <span style="color: #ff7800;">class</span> <span style="color: #3b5bb5;">QuickSortDemo</span>{

    <span style="color: #ff7800;">public</span> <span style="color: #ff7800;">static</span> <span style="color: #ff7800;">void</span> <span style="color: #3b5bb5;">main</span>(<span style="color: #ff7800;">String</span> args[]) {

        <span style="color: #ff7800;">int</span>[] unsorted <span style="color: #ff7800;">=</span> {<span style="color: #3b5bb5;">6</span>, <span style="color: #3b5bb5;">5</span>, <span style="color: #3b5bb5;">3</span>, <span style="color: #3b5bb5;">1</span>, <span style="color: #3b5bb5;">8</span>, <span style="color: #3b5bb5;">7</span>, <span style="color: #3b5bb5;">2</span>, <span style="color: #3b5bb5;">4</span>};

<span style="color: #8c868f;">/**
 * Java Program sort numbers using QuickSort Algorithm. QuickSort is a divide
 * and conquer algorithm, which divides the original list, sort it and then
 * merge it to create sorted output.
 *
 * <span style="color: #ff7800;">@author</span> Javin Paul
 */</span>
<span style="color: #ff7800;">class</span> <span style="color: #3b5bb5;">QuickSort</span> {

    <span style="color: #ff7800;">private</span> <span style="color: #ff7800;">int</span> input[];
    <span style="color: #ff7800;">private</span> <span style="color: #ff7800;">int</span> length;

    <span style="color: #ff7800;">public</span> <span style="color: #ff7800;">void</span> <span style="color: #3b5bb5;">sort</span>(<span style="color: #ff7800;">int</span>[] numbers) {

        <span style="color: #ff7800;">if</span> (numbers <span style="color: #ff7800;">==</span> <span style="color: #3b5bb5;">null</span> <span style="color: #ff7800;">||</span> numbers<span style="color: #ff7800;">.</span>length <span style="color: #ff7800;">==</span> <span style="color: #3b5bb5;">0</span>) {
            <span style="color: #ff7800;">return</span>;
        }
        this<span style="color: #ff7800;">.</span>input <span style="color: #ff7800;">=</span> numbers;
        length <span style="color: #ff7800;">=</span> numbers<span style="color: #ff7800;">.</span>length;
        quickSort(<span style="color: #3b5bb5;">0</span>, length <span style="color: #ff7800;">-</span> <span style="color: #3b5bb5;">1</span>);
    }

    <span style="color: #8c868f;">/*
     * This method implements in-place quicksort algorithm recursively.
     */</span>
    <span style="color: #ff7800;">private</span> <span style="color: #ff7800;">void</span> <span style="color: #3b5bb5;">quickSort</span>(<span style="color: #ff7800;">int</span> low, <span style="color: #ff7800;">int</span> high) {
        <span style="color: #ff7800;">int</span> i <span style="color: #ff7800;">=</span> low;
        <span style="color: #ff7800;">int</span> j <span style="color: #ff7800;">=</span> high;

        <span style="color: #8c868f;">// pivot is middle index</span>
        <span style="color: #ff7800;">int</span> pivot <span style="color: #ff7800;">=</span> input[low <span style="color: #ff7800;">+</span> (high <span style="color: #ff7800;">-</span> low) <span style="color: #ff7800;">/</span> <span style="color: #3b5bb5;">2</span>];

        <span style="color: #8c868f;">// Divide into two arrays</span>
        <span style="color: #ff7800;">while</span> (i &lt;<span style="color: #ff7800;">=</span> j) {
            <span style="color: #8c868f;">/**
             * As shown in above image, In each iteration, we will identify a
             * number from left side which is greater then the pivot value, and
             * a number from right side which is less then the pivot value. Once
             * search is complete, we can swap both numbers.
             */</span>
            <span style="color: #ff7800;">while</span> (input[i] <span style="color: #ff7800;">&lt;</span> pivot) {
                i<span style="color: #ff7800;">++</span>;
            }
            <span style="color: #ff7800;">while</span> (input[j] <span style="color: #ff7800;">&gt;</span> pivot) {
                j<span style="color: #ff7800;">--</span>;
            }
            <span style="color: #ff7800;">if</span> (i &lt;<span style="color: #ff7800;">=</span> j) {
                swap(i, j);
                <span style="color: #8c868f;">// move index to next position on both sides</span>
                i<span style="color: #ff7800;">++</span>;
                j<span style="color: #ff7800;">--</span>;
            }
        }

        <span style="color: #8c868f;">// calls quickSort() method recursively</span>
        <span style="color: #ff7800;">if</span> (low <span style="color: #ff7800;">&lt;</span> j) {
            quickSort(low, j);
        }

        <span style="color: #ff7800;">if</span> (i <span style="color: #ff7800;">&lt;</span> high) {
            quickSort(i, high);
        }
    }

    <span style="color: #ff7800;">private</span> <span style="color: #ff7800;">void</span> <span style="color: #3b5bb5;">swap</span>(<span style="color: #ff7800;">int</span> i, <span style="color: #ff7800;">int</span> j) {
        <span style="color: #ff7800;">int</span> temp <span style="color: #ff7800;">=</span> input[i];
        input[i] <span style="color: #ff7800;">=</span> input[j];
        input[j] <span style="color: #ff7800;">=</span> temp;
    }
}

<span style="color: #ff7800;">Output</span> <span style="color: #ff7800;">:</span>
<span style="color: #ff7800;">Unsorted</span> array <span style="color: #ff7800;">:</span>[<span style="color: #3b5bb5;">6</span>, <span style="color: #3b5bb5;">5</span>, <span style="color: #3b5bb5;">3</span>, <span style="color: #3b5bb5;">1</span>, <span style="color: #3b5bb5;">8</span>, <span style="color: #3b5bb5;">7</span>, <span style="color: #3b5bb5;">2</span>, <span style="color: #3b5bb5;">4</span>]
<span style="color: #ff7800;">Sorted</span> array <span style="color: #ff7800;">:</span>[<span style="color: #3b5bb5;">1</span>, <span style="color: #3b5bb5;">2</span>, <span style="color: #3b5bb5;">3</span>, <span style="color: #3b5bb5;">4</span>, <span style="color: #3b5bb5;">5</span>, <span style="color: #3b5bb5;">6</span>, <span style="color: #3b5bb5;">7</span>, <span style="color: #3b5bb5;">8</span>]</pre> 
              <div> 
               <br>
              </div> 
              <br> 
              <br> 
              <h3> <u>Import points about Quicksort algorithm</u></h3> Now we know how quick sort works and how to implement quicksort in Java, its time to revise some of the important points about this popular sorting algorithm.
              <br> 
              <br> 1) QuickSort is a divide and conquer algorithm. Large list is divided into two and sorted separately (conquered), sorted list is merge later.
              <br> 
              <br> 2) On "in-place" implementation of quick sort, list is sorted using same array, no additional array is required. Numbers are re-arranged pivot, also known as partitioning.
              <br> 
              <br> 3) Partitioning happen around pivot, which is usually middle element of array.
              <br> 
              <br> 4) Average case time complexity of Quicksort is 
              <span style="font-family: &quot;courier new&quot; , &quot;courier&quot; , monospace;">O(n log n)</span> and worst case time complexity is 
              <span style="font-family: &quot;courier new&quot; , &quot;courier&quot; , monospace;">O(n ^2)</span>, which makes it one of the fasted sorting algorithm. Interesting thing is it's worst case performance is equal to 
              <a href="http://javarevisited.blogspot.sg/2014/08/bubble-sort-algorithm-in-java-with.html">Bubble Sort</a> :)
              <br> 
              <br> 5)&nbsp;Quicksort can be implemented with an in-place partitioning algorithm, so the entire sort can be done with only 
              <span style="font-family: &quot;courier new&quot; , &quot;courier&quot; , monospace;">O(log n)</span> additional space used by the stack during the recursion.
              <br> 
              <br> 6) Quicksort is also a good example of algorithm which makes best use of CPU caches, because of it's divide and conquer nature.
              <br> 
              <br> 7) In Java, 
              <span style="font-family: &quot;courier new&quot; , &quot;courier&quot; , monospace;">Arrays.sort()</span> method uses quick sort algorithm to sort array of primitives. It's different than our algorithm, and uses two pivots. Good thing is that it perform much better than most of the quicksort algorithm available on internet for different data sets, where traditional quick sort perform poorly. One more reason, not to reinvent the wheel but to use the library method, when it comes to write production code.
              <br> 
              <br> 
              <br> That's all about 
              <b>Quicksort sorting algorithm in Java</b>. It is one of the must know algorithm for all level of Java programmers, not that you need it often to implement it but to do well on interviews and use the lesson learned while implementing quick sort in Java. In our example, we have implemented quicksort&nbsp;
              <span style="font-family: &quot;courier new&quot; , &quot;courier&quot; , monospace;">"in-place"</span>, which is what you should do if asked to write quicksort in Java. Remember as Java programmer, you don't need to write your own implementation as library implementation are much better implemented and tested. You should use &nbsp;
              <span style="font-family: &quot;courier new&quot; , &quot;courier&quot; , monospace;">Arrays.sort()&nbsp;</span>&nbsp;method to sort your array instead of writing your own sort method. One more reason of using library method is that they are usually improved over different version, and can take advantage of new machine instructions or native improvement.
