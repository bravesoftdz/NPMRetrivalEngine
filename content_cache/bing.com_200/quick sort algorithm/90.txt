                       Quick Sort works on Divide and Conquer algorithm. Here, given array is divided into two parts, left and right. and both array are being sorted individually. After then we combine and will have sorted array.
                      <br> 
                      <u><b><br> </b></u> 
                      <u><b>How it works:</b></u>
                      <br> 
                      <ol style="text-align: left;"> 
                       <li>Pick up any element in Array. ex. middle element of array. we called this one as Pivot.</li> 
                       <li>All elements which are smaller than Pivot are placed in one array and All elements which are larger than pivot are placed in another array</li> 
                       <li>Using recursion sort both array using quicksort</li> 
                       <li>Combine array as mentioned below.<a name="more"></a></li> 
                      </ol> 
                      <u><b>Java Program using Quick sort algorithm:</b></u>
                      <br> 
                      <br>
