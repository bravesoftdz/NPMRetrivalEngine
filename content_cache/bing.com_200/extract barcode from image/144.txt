      <h1 class="title">How to: Extract a DAC From a Database</h1> 
      <div id="mainSection"> 
       <div id="mainBody"> 
        <p></p> 
        <div class="introduction">
         <p>Use either the <span class="label">Extract a Data-tier Application Wizard</span> or a Windows PowerShell script to extract a data-tier application (DAC) package from an existing SQL Server database. The extraction process creates a DAC package file that contains definitions of the database objects and their related instance-level elements. For example, a DAC package file contains the database tables, stored procedures, views, and users, along with the logins that map to the database users.</p>
         <p>You can extract a DAC from databases residing on instances of SQL Azure, or SQL Server 2005 Service Pack 4 (SP4) or later.</p>
         <p>The <span class="label">Extract a Data-tier Application Wizard</span> does not register the DAC definition in the current instance of the Database Engine. For more information about registering a DAC, see <span><a href="https://technet.microsoft.com/en-us/library/ee633653(v=sql.105).aspx">How to: Register a Database As a DAC</a></span>.</p>
        </div>
