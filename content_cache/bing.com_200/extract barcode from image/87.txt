        <h1>Open Source PDF Libraries in Java</h1> 
        <h3><span><a href="/open-source/pdf-libraries/itext">iText</a></span></h3> 
        <table style="font-size:12px; margin-top:5px; margin-left:5px;" width="100%"> 
         <tbody>
          <tr> 
           <td>iText is a library that allows you to generate PDF files on the fly. The iText classes are very useful for people who need to generate read-only, platform independent documents containing text, lists, tables and images. The library is especially useful in combination with Java(TM) technology-based Servlets: The look and feel of HTML is browser dependent; with iText and PDF you can control exactly how your servlet's output will look.</td> 
          </tr> 
         </tbody>
        </table> 
        <p style="padding: 0; padding-left:6px;"><a href="/open-source/pdf-libraries/itext">Go To iText</a></p> 
        <h3><span><a href="/open-source/pdf-libraries/gnujpdf">gnujpdf</a></span></h3> 
        <table style="font-size:12px; margin-top:5px; margin-left:5px;" width="100%"> 
         <tbody>
          <tr> 
           <td>gnujpdf is a Java package (gnu.jpdf.*) licensed under the LGPL. It provides a simple API to create pdf files and print using subclasses of java.awt.Graphics and java.awt.PrintJob. The PDF classes write to an OutputStream in pdf format instead of a typical Graphics object, but the method calls are the same as they would be in any Applet or Application drawing to a canvas.</td> 
          </tr> 
         </tbody>
        </table> 
        <p style="padding: 0; padding-left:6px;"><a href="/open-source/pdf-libraries/gnujpdf">Go To gnujpdf</a></p> 
        <h3><span><a href="/open-source/pdf-libraries/pdf-box">PDF Box</a></span></h3> 
        <table style="font-size:12px; margin-top:5px; margin-left:5px;" width="100%"> 
         <tbody>
          <tr> 
           <td>PDFBox is a Java PDF Library. This project will allow access to all of the components in a PDF document. More PDF manipulation features will be added as the project matures. This ships with a utility to take a PDF document and output a text file.</td> 
          </tr> 
         </tbody>
        </table>

        <h3><span><a href="/open-source/pdf-libraries/fop">FOP</a></span></h3> 
        <table style="font-size:12px; margin-top:5px; margin-left:5px;" width="100%"> 
         <tbody>
          <tr> 
           <td>FOP is an XSL formatter written in Java. It is used in conjunction with an XSLT transformation engine to format XML documents into PDF.</td> 
          </tr> 
         </tbody>
        </table> 
        <p style="padding: 0; padding-left:6px;"><a href="/open-source/pdf-libraries/fop">Go To FOP</a></p> 
        <h3><span><a href="/open-source/pdf-libraries/jfreereport">JFreeReport</a></span></h3> 
        <table style="font-size:12px; margin-top:5px; margin-left:5px;" width="100%"> 
         <tbody>
          <tr> 
           <td>Java reporting tool for formatting PDF reports. It is possible to simply hand off a swing TableModel to JFreeReport and get a paginated pdf as a result.</td> 
          </tr> 
         </tbody>
        </table> 
        <p style="padding: 0; padding-left:6px;"><a href="/open-source/pdf-libraries/jfreereport">Go To JFreeReport</a></p> 
        <h3><span><a href="/open-source/pdf-libraries/pjx">PJX</a></span></h3> 
        <table style="font-size:12px; margin-top:5px; margin-left:5px;" width="100%"> 
         <tbody>
          <tr> 
           <td>PJX is a general purpose PDF programming library for Java; with support for reading, combining, manipulating, and writing PDF documents.</td> 
          </tr> 
         </tbody>
        </table> 
        <p style="padding: 0; padding-left:6px;"><a href="/open-source/pdf-libraries/pjx">Go To PJX</a></p> 
        <h3><span><a href="/open-source/pdf-libraries/pdf-clown-for-java">PDF Clown for Java (PDF Jester)</a></span></h3> 
        <table style="font-size:12px; margin-top:5px; margin-left:5px;" width="100%"> 
         <tbody>
          <tr> 
           <td>PDF Clown for Java (PDF Jester) is a Java 1.5 library for reading, manipulating and writing PDF files, with multiple abstraction layers to satisfy different programming styles: from the lower level (PDF object model) to the higher (PDF document structure and content streaming).</td> 
          </tr> 
         </tbody>
        </table> 
        <p style="padding: 0; padding-left:6px;"><a href="/open-source/pdf-libraries/pdf-clown-for-java">Go To PDF Clown for Java (PDF Jester)</a></p> 
        <h3><span><a href="/open-source/pdf-libraries/jpedal">JPedal</a></span></h3> 
        <table style="font-size:12px; margin-top:5px; margin-left:5px;" width="100%"> 
         <tbody>
          <tr> 
           <td>JPedal is an application for viewing and printing of pdf files. Features of JPedal: * JPedal supports a wide variety of different font technologies. * Jpedal supports the following colour spaces: DeviceRGB, CalRGB, DeviceGRAY, CalGRAY, ICC, indexed, DeviceCMYK and DeviceN. * Jpedal also reads and displays raw tiff, jpg and gif file formats. * Text can be extracted from an entire document, a single page, from within page co-ordinates or from tables. Font information and metadata can also be extracted. * JPedal can extract any image from a pdf with a choice of output options. * View, edit, print and extract content from interactive FDF forms. * JPedal includes an interactive search function that allows you to search either the current page or the entire pdf document for occurrences of a word or a phrase.</td> 
          </tr> 
         </tbody>
        </table> 
        <p style="padding: 0; padding-left:6px;"><a href="/open-source/pdf-libraries/jpedal">Go To JPedal</a></p> 
        <h3><span><a href="/open-source/pdf-libraries/pdfjet">PDFjet</a></span></h3> 
        <table style="font-size:12px; margin-top:5px; margin-left:5px;" width="100%"> 
         <tbody>
          <tr> 
           <td>PDFjet Open Source Edition is a library for dynamic generation of PDF documents from Java and .NET. The PDFjet Open Source Edition has the following features: * Drawing support for: points, lines, boxes, circles, bezier curves, polygons, stars, complex paths and shapes. * Text: unicode support, text kerning when using the Helvetica and Times-Roman families of core fonts, embedding hyperlinks. </td> 
          </tr> 
         </tbody>
        </table> 
        <p style="padding: 0; padding-left:6px;"><a href="/open-source/pdf-libraries/pdfjet">Go To PDFjet</a></p> 
        <h3><span><a href="/open-source/pdf-libraries/jpod">jPod</a></span></h3> 
        <table style="font-size:12px; margin-top:5px; margin-left:5px;" width="100%"> 
         <tbody>
          <tr> 
           <td>jPod is a mature PDF manipulation and rendering library. It supports COS and PD level manipulation of PDF documents, AFM and TrueTypes fonts, incremental writing of files, and parsing of files that slightly deviate from the specification.</td> 
          </tr> 
         </tbody>
        </table> 
        <p style="padding: 0; padding-left:6px;"><a href="/open-source/pdf-libraries/jpod">Go To jPod</a></p> 
        <h3><span><a href="/open-source/pdf-libraries/icepdf">ICEpdf </a></span></h3> 
        <table style="font-size:12px; margin-top:5px; margin-left:5px;" width="100%"> 
         <tbody>
          <tr> 
           <td>ICEpdf is an open source Java PDF engine for viewing, printing, and manipulating PDF documents. The ICEpdf API is 100% Java-based, lightweight, fast, efficient, and very easy to use.</td> 
          </tr> 
         </tbody>
        </table>
