              <div class="post-body" id="post-body-3609558555091628596" itemprop="articleBody"> 
               <div style="text-align: justify;">
                The following tutorial will explain how to extract all text from PDFs (including text in images), by using a combination of 
                <span style="font-style:italic;">Ghostscript</span> and a command line OCR tool called 
                <span style="font-style:italic;">tesseract-ocr</span>. This is yet another guest post by StoneCut.
                <br> 
                <br> 
                <br> First we need to convert our PDF to individual image files (TIFF) so we can then OCR-scan them again. We need Ghostscript for that. It's probably already installed on your system but just to be sure you can run:
                <br> 
               </div>
               <br> 
               <a name="more"></a>
               <pre class="linux-code"><code>sudo apt-get install ghostscript</code></pre>
               <br> 
               <div style="text-align: justify;">
                Once we have ghostscript installed we can convert the actual PDF using the gs utility:
                <br> 
                <pre class="linux-code"><code>gs -dNOPAUSE -sDEVICE=tiffg4 -r600x600 -dBATCH -sPAPERSIZE=a4 -sOutputFile=Output_File_Name.tif Name_of_PDF.pdf</code></pre>
                <br> You will need to adjust the "
                <span style="font-style: italic;">Name_of_PDF.pdf</span>" and "
                <span style="font-style: italic;">Output_File_Name.tif</span>" from the above command for your purposes.
                <br> 
                <br> This will leave us with one large TIFF file (mine was about 10x as big as the original PDF) which we will now OCR-scan (Optical Character Recognition). We're going to use "tesseract-ocr" for that. But we need to install it first:
                <br> 
                <pre class="linux-code"><code>sudo apt-get install tesseract-ocr tesseract-ocr-eng</code></pre>
                <br> The package "tesseract-ocr-eng" is the English language recognition support and is REQUIRED for tesseract-ocr to work, no matter what locale your system is. Support for other languages is available in packages with their country code in them such as "tesseract-ocr-deu" for German language support.
                <br> 
                <br> Let's finally convert our big TIFF file to a TXT file including all the text, even that from images in the original PDF:
                <br> 
                <pre class="linux-code"><code>tesseract Output_File_Name.tif Name_of_TXT -l eng</code></pre>
                <br> Again, adjust "Output_File_Name.tif" to the filename you originally choose and substitute your desired TXT file name for "Name_of_TXT" - leave out the *.txt extension. If your PDF isn't in English, then set the "-l eng" accordingly to the package for your language support you installed earlier.
                <br> 
                <br> That's it. Check out the resultant TXT file.
                <br> 
                <br> 
                <span style="font-weight: bold;">Please note:</span> the quality of extracted text from the images inside the PDF is highly dependent on the quality of the original PDF's images.
