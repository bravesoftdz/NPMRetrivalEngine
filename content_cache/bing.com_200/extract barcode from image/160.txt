           <div class="cd">
            <h1>PdfBox</h1> 
            <p></p> 
            <p>A PHP interface for the <a href="http://pdfbox.apache.org/commandline/#extractText" rel="nofollow">PdfBox ExtractText</a> utility, useful to unit-test contents of generated PDFs.</p> 
            <h2>Requirements</h2> 
            <ul>
             <li>Java Runtime Environment</li> 
             <li>PdfBox JAR file - Download: http://pdfbox.apache.org/downloads.html - Tested with 1.6.0, 1.7.0 and 1.8.6</li> 
             <li>PHP needs permissions for shell execution</li> 
            </ul> 
            <h2>Install</h2> 
            <p>To install with composer:</p> 
            <pre><code class="language-sh">composer require sgh/pdfbox
</code></pre>
            <p></p> 
            <h2>Basic Usage</h2> 
            <pre><code class="language-php">use SGH\PdfBox

//$pdf = GENERATED_PDF;
$converter = new PdfBox;
$converter-&gt;setPathToPdfBox('/usr/bin/pdfbox-app-1.7.0.jar');
$text = $converter-&gt;textFromPdfStream($pdf);
$html = $converter-&gt;htmlFromPdfStream($pdf);
$dom  = $converter-&gt;domFromPdfStream($pdf);
</code></pre>
            <p></p> 
            <p>If the source PDF is a file, use <code>xxxFromPdfFile()</code> instead <code>xxxFromPdfStream()</code> with the source path as parameter.</p> 
            <p>If you want to save the converted output to a file, specify the destination path as second parameter of the <code>xxxFromPdfxxx()</code> methods.</p> 
            <h2>Advanced Usage</h2> 
            <p></p> 
            <p>Convert a range of pages instead of the full document:</p>
            <pre><code class="language-php">$converter-&gt;getOptions()
    -&gt;setStartPage(2)
	-&gt;setEndPage(5);
</code></pre>
            <p></p> 
            <p>Ignore corrupt objects in the PDF:</p>
            <pre><code class="language-php">$converter-&gt;getOptions()
    -&gt;setForce(true);
</code></pre>
            <p></p> 
            <p>Sort text:</p>
            <pre><code class="language-php">$converter-&gt;getOptions()
    -&gt;setSort(true);
</code></pre>
            <p></p> 
            <h2>PHPUnit tests</h2> 
            <p>To run the unit tests, change the environment variable <code>PDFBOX_JAR</code> to the full path of your PdfBox JAR file. See <em>phpunit.xml.dist</em>.</p>
