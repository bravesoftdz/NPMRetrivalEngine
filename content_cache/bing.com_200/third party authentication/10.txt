           <h1 name="edf0" id="edf0" class="graf graf--h3 graf-after--figure graf--title">How to add Facebook and Google login to your Ionic 2 app with Cordova&nbsp;Oauth</h1>
           <p name="c1ab" id="c1ab" class="graf graf--p graf-after--h3">Adding third party authentication to a mobile app is one of the most common challenges developers deal with. This feature cannot be excluded from your Ionic 2 app too. In this tutorial,</p>
           <ul class="postList">
            <li name="4a3b" id="4a3b" class="graf graf--li graf-after--p">We will start by configuring our machines for Ionic 2 app development</li>
            <li name="4b9a" id="4b9a" class="graf graf--li graf-after--li">We will use <a href="http://github.com/appseed-io/supermodular2" data-href="http://github.com/appseed-io/supermodular2" class="markup--anchor markup--li-anchor" rel="noopener nofollow" target="_blank">Supermodular 2</a> starter app where we will implement the third-party authentication with Facebook and Google providers using the <a href="http://github.com/nraboy/ng2-cordova-oauth" data-href="http://github.com/nraboy/ng2-cordova-oauth" class="markup--anchor markup--li-anchor" rel="noopener nofollow" target="_blank">ng2-cordova-oauth</a> Angular 2 library.</li>
            <li name="762c" id="762c" class="graf graf--li graf-after--li">Finally, we will configure the side menu so our new screen is included.</li>
           </ul>
           <p name="c6cd" id="c6cd" class="graf graf--p graf-after--li">Note: The complete source code of this tutorial is available in the <a href="http://github.com/appseed-io/ionic2-oauth" data-href="http://github.com/appseed-io/ionic2-oauth" class="markup--anchor markup--p-anchor" rel="noopener nofollow" target="_blank">ionic2-oauth</a> Github repository.</p>
           <blockquote name="feee" id="feee" class="graf graf--pullquote graf-after--p">
            <a href="https://github.com/appseed-io/ionic2-oauth" data-href="https://github.com/appseed-io/ionic2-oauth" class="markup--anchor markup--pullquote-anchor" rel="nofollow noopener" target="_blank">Download the&nbsp;code</a>
           </blockquote>
           <h3 name="5867" id="5867" class="graf graf--h3 graf-after--pullquote">Preparations</h3>
           <p name="ba7d" id="ba7d" class="graf graf--p graf-after--h3">Before we begin, make sure you have your machine all set up for Ionic 2.</p>
           <h3 name="a778" id="a778" class="graf graf--h3 graf-after--p">NodeJS</h3>
           <p name="f2b0" id="f2b0" class="graf graf--p graf-after--h3"><a href="http://nodejs.org/en/" data-href="http://nodejs.org/en/" class="markup--anchor markup--p-anchor" rel="noopener nofollow" target="_blank">Download the installer</a> for NodeJS and install the version 6 or greater.</p>
           <h3 name="6c91" id="6c91" class="graf graf--h3 graf-after--p">Sass</h3>
           <p name="d4ad" id="d4ad" class="graf graf--p graf-after--h3">In order to install Sass, firstly, you should make sure Ruby is installed on your machine since Sass has a Ruby dependency. If you have MAC OS, Ruby is already installed. Otherwise, you should install Ruby and, then, Sass. Depending on your OS, follow the quick instructions included in the <a href="http://sass-lang.com/install" data-href="http://sass-lang.com/install" class="markup--anchor markup--p-anchor" rel="noopener nofollow" target="_blank">official page of Sass</a>.</p>
           <h3 name="6c27" id="6c27" class="graf graf--h3 graf-after--p">Ionic and Cordova&nbsp;CLI</h3>
           <p name="9456" id="9456" class="graf graf--p graf-after--h3">Whether you have a previous Ionic/Cordova installation and you want to update it or this is the first time you install them, open your terminal and run the command:</p>
