       <h1 id="Introduction">Introduction</h1> 
       <span class="anchor" id="line-11"></span>
       <p class="line862">Ubuntu strives to make all of the most important software available to its users automatically. However patent and copyright restrictions (see <a class="http" href="http://www.ubuntu.com/ubuntu/licensing">Ubuntu License Policy</a>) complicate distribution of software to support non-free formats. <span class="anchor" id="line-12"></span><span class="anchor" id="line-13"></span></p>
       <p class="line874">However, you can install the ability to play popular non-free media formats, including DVD, MP3, Quicktime, and Windows Media formats, by following the instructions below. <span class="anchor" id="line-14"></span><span class="anchor" id="line-15"></span></p>
       <p class="line862">Ubuntu’s commitment to only include completely free software by default means that proprietary media formats are not configured “out of the box”. See Ubuntu’s <a class="http" href="http://www.ubuntu.com/ubuntu/philosophy">Free Software Philosophy</a> and the <a href="/community/FreeFormats">Free Formats</a> page for a more comprehensive discussion of these issues. <span class="anchor" id="line-16"></span><span class="anchor" id="line-17"></span></p>
       <ul>
        <li style="list-style-type:none"><p class="line891"><strong>Legal Notice</strong> <span class="anchor" id="line-18"></span><em>Patent and copyright laws operate differently depending on which country you are in. Please obtain legal advice if you are unsure whether a particular patent or restriction applies to a media format you wish to use in your country.</em> <span class="anchor" id="line-19"></span><span class="anchor" id="line-20"></span></p></li>
       </ul>
       <p class="line867"> </p>
       <h1 id="Playing_Restricted_Formats">Playing Restricted Formats</h1> 
       <span class="anchor" id="line-21"></span>
       <p class="line867"><span class="anchor" id="short"></span> <span class="anchor" id="line-22"></span><span class="anchor" id="line-23"></span></p>
       <p class="line867"> </p>
       <h2 id="Easy_Install">Easy Install</h2> 
       <span class="anchor" id="line-24"></span>
       <span class="anchor" id="line-25"></span>
       <p class="line874">If you are using regular, stock Ubuntu: <span class="anchor" id="line-26"></span><strong><a class="apt" href="apt:ubuntu-restricted-extras?section=universe?section=multiverse">Click here to install the ubuntu-restricted-extras package</a></strong> <span class="anchor" id="line-27"></span><span class="anchor" id="line-28"></span></p>
       <p class="line874">If you are using Kubuntu: <span class="anchor" id="line-29"></span><a class="apt" href="apt:kubuntu-restricted-extras?section=universe?section=multiverse">kubuntu-restricted-extras</a> <span class="anchor" id="line-30"></span><span class="anchor" id="line-31"></span></p>
       <p class="line874">If you are using Xubuntu: <span class="anchor" id="line-32"></span><a class="apt" href="apt:xubuntu-restricted-extras?section=universe?section=multiverse">xubuntu-restricted-extras</a> <span class="anchor" id="line-33"></span><span class="anchor" id="line-34"></span></p>
       <p class="line874">If you are using Lubuntu: <span class="anchor" id="line-35"></span><a class="apt" href="apt:lubuntu-restricted-extras?section=universe?section=multiverse">lubuntu-restricted-extras</a> <span class="anchor" id="line-36"></span><span class="anchor" id="line-37"></span></p>
       <p class="line867"><strong>If you find the above link for your system does not work, please use the Manual Install section.</strong> <span class="anchor" id="line-38"></span><span class="anchor" id="line-39"></span></p>
       <p class="line867"> </p>
       <h2 id="Manual_Install">Manual Install</h2> 
       <span class="anchor" id="line-40"></span>
       <span class="anchor" id="line-41"></span>
       <p class="line874">If you are using regular, stock Ubuntu: <span class="anchor" id="line-42"></span><span class="anchor" id="line-43"></span><span class="anchor" id="line-44"></span></p>
       <pre><span class="anchor" id="line-1"></span>sudo apt-get install ubuntu-restricted-extras</pre>
       <span class="anchor" id="line-45"></span>
       <span class="anchor" id="line-46"></span>
       <p class="line874">If you are using Kubuntu: <span class="anchor" id="line-47"></span><span class="anchor" id="line-48"></span><span class="anchor" id="line-49"></span></p>
       <pre><span class="anchor" id="line-1-1"></span>sudo apt-get install kubuntu-restricted-extras</pre>
       <span class="anchor" id="line-50"></span>
       <p class="line874">If you are using Xubuntu: <span class="anchor" id="line-51"></span><span class="anchor" id="line-52"></span><span class="anchor" id="line-53"></span></p>
       <pre><span class="anchor" id="line-1-2"></span>sudo apt-get install xubuntu-restricted-extras</pre>
       <span class="anchor" id="line-54"></span>
       <p class="line874">If you are using Lubuntu: <span class="anchor" id="line-55"></span><span class="anchor" id="line-56"></span><span class="anchor" id="line-57"></span></p>
       <pre><span class="anchor" id="line-1-3"></span>sudo apt-get install lubuntu-restricted-extras</pre>
       <span class="anchor" id="line-58"></span>
       <p class="line867"> </p>
       <h2 id="Add_libdvdccs">Add libdvdccs</h2> 
       <span class="anchor" id="line-59"></span>
       <p class="line874">In addition, in order to play DVDs, you need to install libdvdcss by entering the following in a terminal: <span class="anchor" id="line-60"></span><span class="anchor" id="line-61"></span></p>
       <p class="line867"><span class="anchor" id="line-62"></span><span class="anchor" id="line-63"></span></p>
       <pre><span class="anchor" id="line-1-4"></span> sudo /usr/share/doc/libdvdread4/install-css.sh</pre>
       <span class="anchor" id="line-64"></span>
       <span class="anchor" id="line-65"></span>
