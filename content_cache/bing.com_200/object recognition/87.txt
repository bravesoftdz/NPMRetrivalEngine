          <tr> 
           <td> <p>BoofCV is an open source Java library for real-time computer vision and robotics applications. Written from scratch for ease of use and high performance. Its functionality covers a wide range of subjects including, optimized low-level image processing routines, camera calibration, feature detection/tracking, structure-from-motion, and recognition. BoofCV has been released under an Apache 2.0 license for both academic and commercial use. </p><p>BoofCV is organized into several packages: <i>image processing</i>, <i>features</i>, <i>geometric vision</i>, <i>calibration</i>, <i>recognition</i>,<i>visualize</i>, and <i>IO</i>. <i>Image processing</i> contains commonly used image processing functions which operate directly on pixels. <i>Features</i> contains feature extraction algorithms for use in higher level operations. <i>Calibration</i> has routines for determining the camera's intrinsic and extrinsic parameters. <i>Recognition</i> is for recognition and tracking complex visual objects. <i>Geometric vision</i> is composed of routines for processing extracted image features using 2D and 3D geometry. <i>Visualize</i> has routines for rendering and displaying extracted features. <i>IO</i> has input and output routines for different data structures. </p> </td>
