           <h2 class="LW_CollapsibleArea_TitleDiv">
            <div>
             <a class="LW_CollapsibleArea_TitleAhref" title="Collapse" role="heading"><span class="cl_CollapsibleArea_expanding LW_CollapsibleArea_Img"></span><span class="LW_CollapsibleArea_Title">Remarks</span></a>
             <div id="Anchor_6" class="LW_CollapsibleArea_Anchor_Div">
              <a href="/en-us/library/system.speech.recognition.speechrecognizer(v=vs.110).aspx#Anchor_6" class="LW_CollapsibleArea_Anchor_Img" title="Right-click to copy and share the link for this section"></a>
             </div>
             <div class="LW_CollapsibleArea_HrDiv">
              <hr class="LW_CollapsibleArea_Hr">
             </div>
            </div></h2>
           <div class="sectionblock">
            <div class="section">
             <p>Applications use the shared recognizer to access Windows Speech Recognition. Use the <span class="selflink">SpeechRecognizer</span> object to add to the Windows speech user experience.</p>
             <p>This class provides control over various aspects of the speech recognition process:</p>
             <ul class="unordered">
              <li><p>To manage speech recognition grammars, use the <a href="https://msdn.microsoft.com/en-us/library/system.speech.recognition.speechrecognizer.loadgrammar(v=vs.110).aspx">LoadGrammar</a>, <a href="https://msdn.microsoft.com/en-us/library/system.speech.recognition.speechrecognizer.loadgrammarasync(v=vs.110).aspx">LoadGrammarAsync</a>, <a href="https://msdn.microsoft.com/en-us/library/system.speech.recognition.speechrecognizer.unloadgrammar(v=vs.110).aspx">UnloadGrammar</a>, <a href="https://msdn.microsoft.com/en-us/library/system.speech.recognition.speechrecognizer.unloadallgrammars(v=vs.110).aspx">UnloadAllGrammars</a>, and <a href="https://msdn.microsoft.com/en-us/library/system.speech.recognition.speechrecognizer.grammars(v=vs.110).aspx"><span xmlns="">Grammars</span></a>.</p></li>
              <li><p>To get information about current speech recognition operations, subscribe to the <span class="selflink">SpeechRecognizer</span>’s <a href="https://msdn.microsoft.com/en-us/library/system.speech.recognition.speechrecognizer.speechdetected(v=vs.110).aspx">SpeechDetected</a>, <a href="https://msdn.microsoft.com/en-us/library/system.speech.recognition.speechrecognizer.speechhypothesized(v=vs.110).aspx">SpeechHypothesized</a>, <a href="https://msdn.microsoft.com/en-us/library/system.speech.recognition.speechrecognizer.speechrecognitionrejected(v=vs.110).aspx">SpeechRecognitionRejected</a>, and <a href="https://msdn.microsoft.com/en-us/library/system.speech.recognition.speechrecognizer.speechrecognized(v=vs.110).aspx">SpeechRecognized</a> events.</p></li>
              <li><p>To view or modify the number of alternate results the recognizer returns, use the <a href="https://msdn.microsoft.com/en-us/library/system.speech.recognition.speechrecognizer.maxalternates(v=vs.110).aspx"><span xmlns="">MaxAlternates</span></a> property. The recognizer returns recognition results in a <a href="https://msdn.microsoft.com/en-us/library/system.speech.recognition.recognitionresult(v=vs.110).aspx">RecognitionResult</a> object. </p></li>
              <li><p>To access or monitor the state of the shared recognizer, use the <a href="https://msdn.microsoft.com/en-us/library/system.speech.recognition.speechrecognizer.audiolevel(v=vs.110).aspx"><span xmlns="">AudioLevel</span></a>, <a href="https://msdn.microsoft.com/en-us/library/system.speech.recognition.speechrecognizer.audioposition(v=vs.110).aspx"><span xmlns="">AudioPosition</span></a>, <a href="https://msdn.microsoft.com/en-us/library/system.speech.recognition.speechrecognizer.audiostate(v=vs.110).aspx"><span xmlns="">AudioState</span></a>, <a href="https://msdn.microsoft.com/en-us/library/system.speech.recognition.speechrecognizer.enabled(v=vs.110).aspx"><span xmlns="">Enabled</span></a>, <a href="https://msdn.microsoft.com/en-us/library/system.speech.recognition.speechrecognizer.pauserecognizeronrecognition(v=vs.110).aspx"><span xmlns="">PauseRecognizerOnRecognition</span></a>, <a href="https://msdn.microsoft.com/en-us/library/system.speech.recognition.speechrecognizer.recognizeraudioposition(v=vs.110).aspx"><span xmlns="">RecognizerAudioPosition</span></a>, and <a href="https://msdn.microsoft.com/en-us/library/system.speech.recognition.speechrecognizer.state(v=vs.110).aspx"><span xmlns="">State</span></a> properties and the <a href="https://msdn.microsoft.com/en-us/library/system.speech.recognition.speechrecognizer.audiolevelupdated(v=vs.110).aspx">AudioLevelUpdated</a>, <a href="https://msdn.microsoft.com/en-us/library/system.speech.recognition.speechrecognizer.audiosignalproblemoccurred(v=vs.110).aspx">AudioSignalProblemOccurred</a>, <a href="https://msdn.microsoft.com/en-us/library/system.speech.recognition.speechrecognizer.audiostatechanged(v=vs.110).aspx">AudioStateChanged</a>, and <a href="https://msdn.microsoft.com/en-us/library/system.speech.recognition.speechrecognizer.statechanged(v=vs.110).aspx">StateChanged</a> events.</p></li>
              <li><p>To synchronize changes to the recognizer, use the <a href="https://msdn.microsoft.com/en-us/library/ms554811(v=vs.110).aspx">RequestRecognizerUpdate</a> method. The shared recognizer uses more than one thread to perform tasks.</p></li>
              <li><p>To emulate input to the shared recognizer, use the <a href="https://msdn.microsoft.com/en-us/library/ms554603(v=vs.110).aspx">EmulateRecognize</a> and <a href="https://msdn.microsoft.com/en-us/library/ms554600(v=vs.110).aspx">EmulateRecognizeAsync</a> methods.</p></li>
             </ul>
             <p>The configuration of Windows Speech Recognition is managed by the use of the <strong>Speech Properties</strong> dialog in the <strong>Control Panel</strong>. This interface is used to select the default desktop speech recognition engine and language, the audio input device, and the sleep behavior of speech recognition. If the configuration of Windows Speech Recognition is changed while the application is running, (for instance, if speech recognition is disabled or the input language is changed), the change affects all <span class="selflink">SpeechRecognizer</span> objects.</p>
             <p>To create an in-process speech recognizer that is independent of Windows Speech Recognition, use the <a href="https://msdn.microsoft.com/en-us/library/system.speech.recognition.speechrecognitionengine(v=vs.110).aspx">SpeechRecognitionEngine</a> class.</p>
             <div class="alert">
              <table summary="table">
               <tbody>
                <tr>
                 <th align="left" scope="col"><span><img id="s-e6f6a65cf14f462597b64ac058dbe1d0-system-media-system-caps-note" alt="System_CAPS_note" src="https://i-msdn.sec.s-msft.com/dynimg/IC101471.jpeg" title="System_CAPS_note" xmlns=""></span><span class="alertTitle">Note </span></th>
                </tr>
                <tr>
                 <td><p>Always call <a href="https://msdn.microsoft.com/en-us/library/ms554598(v=vs.110).aspx">Dispose</a> before you release your last reference to the speech recognizer. Otherwise, the resources it is using will not be freed until the garbage collector calls the recognizer object's <strong>Finalize</strong> method.</p></td>
                </tr>

           <h2 class="LW_CollapsibleArea_TitleDiv">
            <div>
             <a class="LW_CollapsibleArea_TitleAhref" title="Collapse" role="heading"><span class="cl_CollapsibleArea_expanding LW_CollapsibleArea_Img"></span><span class="LW_CollapsibleArea_Title">Examples</span></a>
             <div id="Anchor_7" class="LW_CollapsibleArea_Anchor_Div">
              <a href="/en-us/library/system.speech.recognition.speechrecognizer(v=vs.110).aspx#Anchor_7" class="LW_CollapsibleArea_Anchor_Img" title="Right-click to copy and share the link for this section"></a>
             </div>
             <div class="LW_CollapsibleArea_HrDiv">
              <hr class="LW_CollapsibleArea_Hr">
             </div>
            </div></h2>
           <div class="sectionblock">
            <div class="section">
             <div class="description">
              <div class="section">
               <p>The following example is part of a console application that loads a speech recognition grammar and demonstrates asynchronous emulated input, the associated recognition results, and the associated events raised by the speech recognizer. If Windows Speech Recognition is not running, then starting this application will also start Windows Speech Recognition. If Windows Speech Recognition is in the <strong>Sleeping</strong> state, then <a href="https://msdn.microsoft.com/en-us/library/system.speech.recognition.speechrecognizer.emulaterecognizeasync(v=vs.110).aspx">EmulateRecognizeAsync</a> always returns null.</p>
              </div>
             </div> 
             <div id="code-snippet-2" class="codeSnippetContainer" xmlns=""> 
              <div class="codeSnippetContainerTabs"> 
               <div class="codeSnippetContainerTabSingle" dir="ltr">
                <a>C#</a>
               </div> 
              </div> 
              <div class="codeSnippetContainerCodeContainer"> 
               <div class="codeSnippetToolBar"> 
                <div class="codeSnippetToolBarText"> 
                 <a name="CodeSnippetCopyLink" style="display: none;" title="Copy to clipboard." href="javascript:if (window.epx.codeSnippet)window.epx.codeSnippet.copyCode('CodeSnippetContainerCode_2f9ccbf9-6e76-4f9d-8154-a4e88b55f6d4');" ms.cmptyp="CodeSnippet">Copy</a> 
                </div> 
               </div> 
               <div id="CodeSnippetContainerCode_2f9ccbf9-6e76-4f9d-8154-a4e88b55f6d4" class="codeSnippetContainerCode" dir="ltr"> 
                <div style="color:Black;">
                 <pre>
<span style="color:Blue;">using</span> System;
<span style="color:Blue;">using</span> System.Speech.Recognition;
<span style="color:Blue;">using</span> System.Threading;

<span style="color:Blue;">namespace</span> SharedRecognizer
{
  <span style="color:Blue;">class</span> Program
  {

    <span style="color:Green;">// Indicate whether the asynchronous emulate recognition</span>
    <span style="color:Green;">// operation has completed.</span>
    <span style="color:Blue;">static</span> <span style="color:Blue;">bool</span> completed;

    <span style="color:Blue;">static</span> <span style="color:Blue;">void</span> Main(<span style="color:Blue;">string</span>[] args)
    {

      <span style="color:Green;">// Initialize an instance of the shared recognizer.</span>
      <span style="color:Blue;">using</span> (SpeechRecognizer recognizer = <span style="color:Blue;">new</span> SpeechRecognizer())
      {

        <span style="color:Green;">// Create and load a sample grammar.</span>
        Grammar testGrammar =
          <span style="color:Blue;">new</span> Grammar(<span style="color:Blue;">new</span> GrammarBuilder(<span style="color:#A31515;">"testing testing"</span>));
        testGrammar.Name = <span style="color:#A31515;">"Test Grammar"</span>;
        recognizer.LoadGrammar(testGrammar);

        <span style="color:Green;">// Attach event handlers for recognition events.</span>
        recognizer.SpeechRecognized +=
          <span style="color:Blue;">new</span> EventHandler&lt;SpeechRecognizedEventArgs&gt;(
            SpeechRecognizedHandler);
        recognizer.EmulateRecognizeCompleted +=
          <span style="color:Blue;">new</span> EventHandler&lt;EmulateRecognizeCompletedEventArgs&gt;(
            EmulateRecognizeCompletedHandler);

        completed = <span style="color:Blue;">false</span>;

        <span style="color:Green;">// Start asynchronous emulated recognition. </span>
        <span style="color:Green;">// This matches the grammar and generates a SpeechRecognized event.</span>
        recognizer.EmulateRecognizeAsync(<span style="color:#A31515;">"testing testing"</span>);

        <span style="color:Green;">// Wait for the asynchronous operation to complete.</span>
        <span style="color:Blue;">while</span> (!completed)
        {
          Thread.Sleep(333);
        }

        completed = <span style="color:Blue;">false</span>;

        <span style="color:Green;">// Start asynchronous emulated recognition.</span>
        <span style="color:Green;">// This does not match the grammar or generate a SpeechRecognized event.</span>
        recognizer.EmulateRecognizeAsync(<span style="color:#A31515;">"testing one two three"</span>);

        <span style="color:Green;">// Wait for the asynchronous operation to complete.</span>
        <span style="color:Blue;">while</span> (!completed)
        {
          Thread.Sleep(333);
        }
      }

      Console.WriteLine();
      Console.WriteLine(<span style="color:#A31515;">"Press any key to exit..."</span>);
      Console.ReadKey();
    }

    <span style="color:Green;">// Handle the SpeechRecognized event.</span>
    <span style="color:Blue;">static</span> <span style="color:Blue;">void</span> SpeechRecognizedHandler(
      <span style="color:Blue;">object</span> sender, SpeechRecognizedEventArgs e)
    {
      <span style="color:Blue;">if</span> (e.Result != <span style="color:Blue;">null</span>)
      {
        Console.WriteLine(<span style="color:#A31515;">"Recognition result = {0}"</span>,
          e.Result.Text ?? <span style="color:#A31515;">"&lt;no text&gt;"</span>);
      }
      <span style="color:Blue;">else</span>
      {
        Console.WriteLine(<span style="color:#A31515;">"No recognition result"</span>);
      }
    }

    <span style="color:Green;">// Handle the SpeechRecognizeCompleted event.</span>
    <span style="color:Blue;">static</span> <span style="color:Blue;">void</span> EmulateRecognizeCompletedHandler(
      <span style="color:Blue;">object</span> sender, EmulateRecognizeCompletedEventArgs e)
    {
      <span style="color:Blue;">if</span> (e.Result == <span style="color:Blue;">null</span>)
      {
        Console.WriteLine(<span style="color:#A31515;">"No result generated."</span>);
      }

      <span style="color:Green;">// Indicate the asynchronous operation is complete.</span>
      completed = <span style="color:Blue;">true</span>;
    }
  }
}
