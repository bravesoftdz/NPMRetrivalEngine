        <h2>Introduction</h2> 
        <p>The <b>java.lang.Object</b> class is the root of the class hierarchy. Every class has Object as a superclass. All objects, including arrays, implement the methods of this class.</p> 
        <h2>Class Declaration</h2> 
        <p>Following is the declaration for <b>java.lang.Object</b> class ?</p> 
        <pre class="result notranslate">
public class Object
</pre>
