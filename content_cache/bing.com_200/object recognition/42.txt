     <h1 style="font-size: 20px">Java package tutorial with example</h1> 
     <div class="post-meta"> 
      <p class="post-meta" style="font-size: small;"> By <a href="#" title="candidjava" rel="author">candid</a> | <b> Posted : </b> 7 Aug, 2015 | <b> Updated : </b> 31 May, 2017 </p>

     <br> 
     <h2>Packages in java</h2>
     <p class="MsoNormal">
      <o:p></o:p></p> 
     <h3><b><br></b></h3>
     <h4><b>What is packages?</b></h4> 
     <p class="MsoNormal">A&nbsp;package&nbsp;is a grouping of related types providing access protection and name space management. Note that&nbsp;types&nbsp;refers to classes, interfaces, enumerations, and annotation types.&nbsp;
      <o:p></o:p></p> 
     <p class="MsoNormal">A&nbsp;package&nbsp;is a collection of related classes.
      <o:p></o:p></p> 
     <p class="MsoNormal">It helps&nbsp;Organize&nbsp;your classes into a folder structure and make it easy to locate and use them.
      <o:p></o:p></p> 
     <p class="MsoNormal">More importantly, It helps improve&nbsp;re-usability.
      <o:p></o:p></p> 
     <p class="MsoNormal">
      <o:p>
       &nbsp;
      </o:p></p> 
     <h4><b>Creating a class inside a package</b></h4> 
     <p class="MsoNormal">
      <o:p>
       &nbsp;
      </o:p><span style="color: inherit; line-height: 1.42857143;">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp;&nbsp;</span><span style="color: inherit; line-height: 1.42857143;">Create a directory structure as shown in image</span></p>
     <p class="MsoNormal"><img src="https://candidjava.s3.amazonaws.com/post/corejava/package/package.jpg" style="width: 502px;"><span style="color: inherit; line-height: 1.42857143;"><br></span></p>
     <p class="MsoNormal">
      <o:p></o:p></p> 
     <p class="MsoNormal">
      <o:p>
       &nbsp;
      </o:p><span style="color: inherit; line-height: 1.42857143; font-weight: bold;">Main.java</span></p> 
     <p class="MsoNormal">public class Main
      <o:p></o:p></p> 
     <p class="MsoNormal">{
      <o:p></o:p></p> 
     <p class="MsoNormal">&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; // your code
      <o:p></o:p></p> 
     <p class="MsoNormal">}
      <o:p></o:p></p> 
     <p class="MsoNormal"><b>Vehicle.java
       <o:p></o:p></b></p> 
     <p class="MsoNormal">&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; If your code belongs to package then your programs first line should be
      <o:p></o:p></p> 
     <p class="MsoNormal">&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; package &lt;packagename&gt;<span style="color: inherit; line-height: 1.42857143;">&nbsp;</span></p> 
     <pre class="prettyprint linenums">
      <o:p>
       package sam;

public class Vehicle
{
public  int a = 10
                //your code
}
      </o:p></pre> 
     <p class="MsoNormal"><br></p> 
     <p class="MsoNormal"><b>Car.java
       <o:p></o:p></b></p> 
     <pre class="prettyprint linenums"><span style="color: inherit; line-height: 1.42857143;">&nbsp;</span><span style="background-color: rgb(238, 238, 238);">package sam.test;

public class Car
{
                //your code
}</span></pre> 
     <p class="MsoNormal"><b><br></b></p>
     <p class="MsoNormal"><b>Accessing a class located in different package
       <o:p></o:p></b></p> 
     <p class="MsoNormal">There are two ways to access class located in different package
      <o:p></o:p></p> 
     <p class="MsoListParagraphCxSpFirst" style="text-indent:-.25in;mso-list:l0 level1 lfo1">
      <!--[if !supportLists]-->1.<span style="font-stretch: normal; font-size: 7pt; line-height: normal; font-family: 'Times New Roman';">&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; </span>
      <!--[endif]--><span style="font-weight: bold;">Using import statement to access the class</span>
      <o:p></o:p></p> 
     <p class="MsoListParagraphCxSpMiddle">
      <o:p>
       &nbsp;
      </o:p><span style="color: inherit; line-height: 1.42857143;">Star(*) imports all classes located in the package specified.</span></p>
     <p class="MsoListParagraphCxSpLast">
      <o:p></o:p></p> 
     <p class="MsoNormal" style="margin-left:.25in">import sam.*;&nbsp; 
      <o:p></o:p></p> 
     <p class="MsoNormal" style="margin-left:.25in">import sam.test.*;
      <o:p></o:p></p> 
     <p class="MsoNormal" style="margin-left:.25in">
      <o:p>
       &nbsp;
      </o:p></p> 
     <p class="MsoNormal" style="margin-left:.25in">But the best practice is to import only the specific class
      <o:p></o:p></p> 
     <p class="MsoNormal" style="margin-left:.25in">import sam.Vehicle;
      <o:p></o:p></p> 
     <p class="MsoNormal" style="margin-left:.25in">import sam.test.Car;
      <o:p></o:p></p> 
     <p class="MsoNormal" style="margin-left:.25in">
      <o:p>
       &nbsp;
      </o:p></p> 
     <p class="MsoNormal" style="margin-left:.25in">To access vehicle and car class in Main
      <o:p></o:p></p> 
     <pre class="prettyprint linenums" style="margin-left:.25in">
      <o:p>
       &nbsp;import sam.Vehicle;
import sam.test.Car;

class Main
{
	public static void main(String[] args)

	{
                Vehicle v1=new Vehicle();
                System.out.println(v1.i);
	}
}
      </o:p></pre>
     <p class="MsoNormal" style="margin-left:.25in"><span style="color: inherit; line-height: 1.42857143;">&nbsp;</span><span style="color: inherit; line-height: 1.42857143;">Compile all classes and run Main.java to check the output</span></p>
     <p class="MsoNormal">
      <o:p></o:p></p> 
     <p class="MsoNormal">
      <o:p>
       &nbsp;
      </o:p></p> 
     <p class="MsoListParagraph" style="text-indent:-.25in;mso-list:l0 level1 lfo1">
      <!--[if !supportLists]-->2.<span style="font-stretch: normal; font-size: 7pt; line-height: normal; font-family: 'Times New Roman';">&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;</span><span style="font-weight: bold;">Instead of importing the class using import keyword you can import it while creating the object</span></p>
     <p class="MsoListParagraph" style="text-indent:-.25in;mso-list:l0 level1 lfo1"><span style="color: inherit; line-height: 1.42857143;">&nbsp; &nbsp; &nbsp;sam.test.Vehicle v1=new sam.test.Vehicle();&nbsp;</span></p>
     <pre class="prettyprint linenums" style="text-indent:-.25in;mso-list:l0 level1 lfo1">  <span style="color: inherit; line-height: 1.42857143;"> class Main<br>{<br>        public static void main(String[] args)<br><br>		{<br>			sam.test.Vehicle v1=new sam.test.Vehicle();<br>	                System.out.println(v1.i);<br>		}<br>}</span></pre>
     <p class="MsoNormal" style="margin-left:1.0in"><span style="color: inherit; line-height: 1.42857143;">This type of import will be used to avoid import conflicts when we have same class name in different package</span></p>
     <p class="MsoNormal" style="margin-left:.25in">
      <o:p></o:p></p>
