           <h2 id="N10054" class="ibm-h2">About this code</h2>
           <p> This code shows you how to use the node.js API for the Watson Visual Recognition service. Given an image, Watson attempts to identify objects in that image. </p>
           <p> The code has two basic sections: </p>
           <ol>
            <li> The <code>main()</code> method. This is the entry point to the code. It creates the <code>VisualRecognitionV3</code> object and then invokes the <code>classify()</code> method. </li>
            <li> A set of default parameters and the code that runs this module if it is invoked by itself (as opposed to being invoked by a <code>require()</code> statement). See the <a href="https://nodejs.org/api/modules.html#modules_accessing_the_main_module">Node.js documentation</a> for more information on the implications of the <code>require.main === module</code> statement. </li>
           </ol>
           <p class="dw-inline-signin"><a href="">Sign in</a> to run, change, and deploy this code.</p>

  return new Promise(function (resolve, reject) {
    var res = {};

    const VisualRecognitionV3 =
      require('watson-developer-cloud/visual-recognition/v3');

    var url = params.url || 'https://gateway-a.watsonplatform.net/visual-recognition/api' ;
    var use_unauthenticated =  params.use_unauthenticated || false ;

    const visual_recognition = new VisualRecognitionV3({
      'api_key': params.api_key,
      'version_date': '2016-05-20',
      'url' : url,
      'use_unauthenticated': use_unauthenticated
    });

    visual_recognition.classify({'url': params.imageurl}, function(err, res) {
      if (err)
        reject(err);
      else
        resolve(res);
    });
  });
}

  'use_unauthenticated' : true
}

           <h2 id="N1009E" class="ibm-h2">Using this code once deployed</h2>
           <p>To use this code once you've deployed it, you must create an instance of the Watson Visual Recognition service:</p>
           <p class="ibm-button-link"><a class="ibm-btn-pri dw-article-btn-pri" href="https://console.bluemix.net/catalog/services/visual-recognition?cm_sp=dw-bluemix-_-cc-sample-code-image-classification-watson-node-_-article">Create a Watson Visual Recognition service instance</a></p>
           <p>To invoke the code, pass a JSON structure that matches the <code>defaultParameters</code> object as shown above with the following changes: </p>
           <ol>
            <li>The <code>api_key</code> and <code>url</code> fields must be filled in with the credentials for your Visual Recognition service instance.</li>
            <li>The <code>use_unauthenticated</code> field must be <code>false</code>.</li>
           </ol>
           <p>For more information, see the developerWorks tutorial <a href="https://www.ibm.com/developerworks/library/wa-invoke-openwhisk-action/index.html">Invoking actions on the IBM Cloud Functions platform</a>.</p>
