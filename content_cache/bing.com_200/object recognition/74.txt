       <h2><a href="/blog/object-recognition-using-TensorFlow-Java">Object recognition using TensorFlow and Java</a></h2>

       <h3>Introduction</h3> 
       <p>One of the most incentive thing to learn any new topic or acquire a new skill is to find it solves a problem that may be difficult to solve with your existent knowledge. Today, Machine learning is a trending topic concerning about find patterns in data and use it to predict the future or analysis the past. <br> Recognizing objects in images is a fascinating thing in computer vision. It may be a trivial task for human but for a machine not.<br> In the last few years the field of machine learning specially, convolutional neural networks has made a huge progress on solving difficult problems like visual recognition and voice recognition. Researchers have demonstrated steady progress in computer vision by validating their work against<a target="_blank" href="http://image-net.org/"> ImageNet</a>&nbsp;which is an academic benchmark for computer vision. </p> 
       <h3>Application Description</h3> 
       <p>Using <a target="_blank" href="https://www.tensorflow.org">TensorFlow </a>and Inception-v3 model, I built this small demo in Java to recognize objects in images and classify it into 1000 classes like Lion, Frog, Flowers, ....etc. The model I used Inception-v3 is trained for the&nbsp;ImageNet&nbsp;Large Visual Recognition Challenge using the data from 2012.</p> Requirements: 
       <ol> 
        <li> Java Development Kit (JDK), you can download from <a target="_blank" href="http://www.oracle.com/technetwork/java/javase/downloads/index.html">http://www.oracle.com/technetwork/java/javase/downloads/index.html</a>.</li> 
        <li> Netbeans, you can download from <a target="_blank" href="https://netbeans.org/downloads">https://netbeans.org/downloads</a>.</li> 
        <li> Source Code, you can download from <a target="_blank" href="https://github.com/emara-geek/object-recognition-tensorflow">https://github.com/emara-geek/object-recognition-tensorflow</a>. </li> 
        <li>The pre-trained Inception V3 model, you can download from <a target="_blank" href="https://storage.googleapis.com/download.tensorflow.org/models/inception_dec_2015.zip">here</a>.</li> 
       </ol>
