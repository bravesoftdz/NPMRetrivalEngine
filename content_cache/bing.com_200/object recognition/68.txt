        <h2 class="h3 add_font_color_emphasize">Object recognition methods in computer vision</h2> 
        <div>
         <div class="containsResourceName parsys mainParsys resourceClass-parsys">
          <div class="containsResourceName section text resourceClass-text"> 
           <div class="mw-text  "> 
            <h3>What Is Object Recognition?</h3> 
            <p>Object recognition is a process for identifying a specific object in a digital image or video. Object recognition algorithms rely on matching, learning, or <a href="https://www.mathworks.com/discovery/pattern-recognition.html">pattern recognition</a> algorithms using appearance-based or <a href="https://www.mathworks.com/discovery/feature-extraction.html">feature-based</a> techniques.</p> 
            <p>Object recognition is useful in applications such as video stabilization, advanced driver assistance systems (ADAS), and disease identification in bioimaging. Common techniques include deep learning based approaches such as convolutional neural networks, and feature-based approaches using edges, gradients, histogram of oriented gradients (HOG), Haar wavelets, and linear binary patterns.</p> 
           </div> 
          </div> 
          <div class="containsResourceName section text resourceClass-text"> 
           <div class="mw-text  "> 
            <h3>Object Recognition Techniques</h3> 
            <p>You can recognize objects using a variety of models, including:</p> 
            <ul> 
             <li>Feature extraction and <a href="https://www.mathworks.com/discovery/machine-learning.html">machine learning</a> models</li> 
             <li><a href="https://www.mathworks.com/help/vision/examples/image-category-classification-using-deep-learning.html">Deep learning models</a> such as <a href="https://www.mathworks.com/discovery/convolutional-neural-network.html">CNNs</a></li> 
             <li><a href="https://www.mathworks.com/help/vision/examples/image-category-classification-using-bag-of-features.html">Bag-of-words models</a> with features such as SURF and MSER</li> 
             <li>Gradient-based and derivative-based matching approaches</li> 
             <li>The <a href="https://www.mathworks.com/help/vision/ref/vision.cascadeobjectdetector-class.html">Viola-Jones algorithm</a>, which can be used to recognize a variety of objects, including faces and upper bodies</li> 
             <li>Template matching</li> 
             <li>Image segmentation and blob analysis</li> 
            </ul> 
           </div> 
          </div>
