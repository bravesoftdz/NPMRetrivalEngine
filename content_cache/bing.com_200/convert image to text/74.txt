                     <div class="body">
                      <p style="background:white"><span style="color:black">Hi,</span></p> 
                      <p style="background:white"><span style="color:black">I need to retrieve an image&nbsp;by doing an&nbsp;HTTP 'Get' request and convert it to Base64 so that it can transported in an XML string. I also need to convert it back from Base64 to Image format so that it can displayed in a webpage/winform.</span></p> 
                      <p style="background:white"><span style="color:black">I am performing the following steps for the first task:</span></p> 
                      <ul type="disc"> 
                       <li class="MsoNormal" style="background:white;margin:0in 0in 0pt;color:black;tab-stops:list .5in"><span style="font-family:Arial">Retrieve image binary stream from a web location by doing HTTP 'Get' Request</span></li> 
                       <li class="MsoNormal" style="background:white;margin:0in 0in 0pt;color:black;tab-stops:list .5in"><span style="font-family:Arial">Convert the data in stream to string by doing Stream.ReadToEnd()</span></li> 
                       <li class="MsoNormal" style="background:white;margin:0in 0in 0pt;color:black;tab-stops:list .5in"><span style="font-family:Arial">Convert the string to byte-array by doing Encoding.GetBytes()</span></li> 
                       <li class="MsoNormal" style="background:white;margin:0in 0in 0pt;color:black;tab-stops:list .5in"><span style="font-family:Arial">Convert byte-array to Base64 by doing Convert.ToBase64()</span></li>
                      </ul> 
                      <p style="background:white"><span style="color:black">Following are the steps for the second task:</span></p> 
                      <ul type="disc"> 
                       <li class="MsoNormal" style="background:white;margin:0in 0in 0pt;color:black;tab-stops:list .5in"><span style="font-family:Arial">Convert Base64 to byte-array by doing Convert.FromBase64()</span></li> 
                       <li class="MsoNormal" style="background:white;margin:0in 0in 0pt;color:black;tab-stops:list .5in"><span style="font-family:Arial">Convert byte-array to binary stream by passing byte-array to MemoryStream() constructor</span></li> 
                       <li class="MsoNormal" style="background:white;margin:0in 0in 0pt;color:black;tab-stops:list .5in"><span style="font-family:Arial">Create Image object by doing Image.FromStream().</span></li>
                      </ul> 
                      <p style="background:white"><span style="color:black">The following is a code snippet of the steps given above:</span></p> 
                      <p><font color="#008000" size="2">// Get the image binary stream</font></p>
                      <font size="2"> </font>
                      <p><font size="2"></font><font color="#0000ff" size="2">string</font><font size="2"> strURL = "http://localhost/001.jpg";</font></p>
                      <font size="2"> <p>HttpWebRequest objRequest = (HttpWebRequest)WebRequest.Create(strURL);</p> <p>HttpWebResponse objResponse = (HttpWebResponse)objRequest.GetResponse();</p> <p>Stream stream = objResponse.GetResponseStream();</p> <p></p> </font>
                      <p><font size="2"></font><font color="#008000" size="2">// Convert to Base64</font></p>
                      <font size="2"> </font>
                      <p><font size="2">StreamReader reader = </font><font color="#0000ff" size="2">new</font><font size="2"> StreamReader(stream);</font></p>
                      <font size="2"> </font>
                      <p><font size="2"></font><font color="#0000ff" size="2">byte</font><font size="2">[] bytedata = System.Text.Encoding.Default.GetBytes(reader.ReadToEnd());</font></p>
                      <font size="2"> </font>
                      <p><font size="2"></font><font color="#0000ff" size="2">string</font><font size="2"> strBase64 = Convert.ToBase64String(bytedata);</font></p>
                      <font size="2"> <p></p> </font>
                      <p><font size="2"></font><font color="#008000" size="2">// Convert back to image binary stream</font></p>
                      <font size="2"> </font>
                      <p><font size="2"></font><font color="#0000ff" size="2">byte</font><font size="2">[] byteresponse = Convert.FromBase64String(strBase64);</font></p>
                      <font size="2"> </font>
                      <p><font size="2">MemoryStream stream2 = </font><font color="#0000ff" size="2">new</font><font size="2"> MemoryStream(byteresponse); </font></p>
                      <font size="2"> <p></p> </font>
                      <p><font size="2"></font><font color="#008000" size="2">// Covert to image object</font></p> 
                      <p><font size="2">System.Drawing.Image img = System.Drawing.Image.FromStream(stream2);</font></p>
                      <font face="Arial Black, Geneva, Arial, Sans-serif"> <p><font face="Verdana, Geneva, Arial, Sans-serif" size="1">The last step in the code above fails. It does not retrieve the image object from the binary stream. It throws an exception - Invalid Parameter value. I think somewhere during conversion, it is failing. While converting to Base64 I cannot directly convert the stream to byte-array. I have to convert it to string and then to byte-array. This should not be the case.</font></p> <p><font face="Verdana, Geneva, Arial, Sans-serif" size="1">Please help me out.</font></p></font>

                       hi !!
                       <br> 
                       <br> I wanted&nbsp; to convert a Stream to an Image and the method : -
                       <br> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; Image img1 = Image.FromStream(datastream)&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; //datastream is a Stream here, an object of the class stream
                       <br> works very well in a desktop application...
                       <br> 
                       <br> However, if I want to use this method in a mobile application for windows mobile 6, I found ti does not work. This is because the System.Drawing class does not
                       <br> at all contain the definition of a method such as FromStream()
                       <br> 
                       <br> With further investigation I found out that the System.Drawing.dll was being imported from the .NET framework to the IDE in case of the desktop application,
                       <br> whereas in case of mobile application, the System.Drawind.dll was being imported from the SDK of the .NET Compact Framework....
                       <br> 
                       <br> Now, the issue is that the Visual Studio IDE uses two different dll's for desktop appplication and mobile application. Nonetheless, the System.Drawing.dll for desktop application DOES have the method FromStream but the System.Drawing.dll for mobile application DOES NOT have the method FromStream()
                       <br> 
                       <br> Hoping for an invaluable reply
                       <br> 
                       <br> Thanks in advance
                       <br> Ankur
