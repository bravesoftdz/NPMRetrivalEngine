         <p>Hi Friends, It’s been a long time, so today I’m writing&nbsp;an interesting topic about how to capture a DIV content and save it as an image using PHP.</p> 
         <p>And I didn’t stop simply capturing any div but also giving an option to add a text with a drag option and two backgrounds to select with the help of jQuery.</p> 
         <p>If you wish to see the demo first, please go ahead and click the demo button below:</p>

         <p>&nbsp;</p> 
         <p>&nbsp;</p>
