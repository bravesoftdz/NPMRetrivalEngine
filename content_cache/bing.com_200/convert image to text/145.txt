                    <h3 class="post-title entry-title" itemprop="name"> JPG To PDF Converter </h3> 
                    <div class="post-header"> 
                     <div class="post-header-line-1"></div> 
                    </div> 
                    <div class="post-body entry-content" id="post-body-6306275708800892892" itemprop="articleBody">
                      The JPGToPDFConverter program is able to convert 
                     <b>an image to a PDF fi</b>le. You can convert different image formats (e.g jpg, png, gif, bmp, ...) to PDF files. When the program runs, it allows you to select many image files as you want. Each image file is converted to a PDF file. The PDF file is stored in the current working folder.
                     <br> By using the JPGToPDFConverter, you are not worry about a big image that can't fit well on the PDF page. For the big image, it is resized automatically to fit the A4 paper size.
                     <br> 
                     <br>

                      <a href="http://3.bp.blogspot.com/-FewIrsVFTP0/UcgU3ii4iTI/AAAAAAAAAX8/nC30F0tItdI/s1600/jpg-to-pdf.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img alt="JPG to image PDF converter java" border="0" height="170" src="http://3.bp.blogspot.com/-FewIrsVFTP0/UcgU3ii4iTI/AAAAAAAAAX8/nC30F0tItdI/s320/jpg-to-pdf.png" width="320"></a>

                     <br> 
                     <b>JPGToPDFConverter source code:</b>
                     <br> 
                     <br> 
                     <div style="background-color: #f8f8f8; color: #338800;">
                       import java.awt.Graphics2D;
                      <br> import java.awt.image.BufferedImage;
                      <br> import java.io.ByteArrayOutputStream;
                      <br> import java.io.File;
                      <br> import java.io.FileOutputStream;
                      <br> import java.io.IOException;
                      <br> import javax.imageio.ImageIO;
                      <br> import com.itextpdf.text.Rectangle;
                      <br> import com.itextpdf.text.Document;
                      <br> import com.itextpdf.text.Image;
                      <br> import com.itextpdf.text.PageSize;
                      <br> import com.itextpdf.text.Element;
                      <br> import com.itextpdf.text.pdf.PdfWriter;
                      <br> import java.awt.Desktop;
                      <br> import javax.swing.filechooser.FileNameExtensionFilter;
                      <br> import javax.swing.JFileChooser;
                      <br> 
                      <br> public class JPGToPDFConverter{
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>public static void main(String[] args){
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>selectImage();
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>}
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>
                      <br> 
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>//allow images selection for converting
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>public static void selectImage(){
                      <br> 
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>JFileChooser chooser = new JFileChooser();
                      <br> &nbsp; &nbsp; 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>FileNameExtensionFilter filter = new FileNameExtensionFilter("Images", "jpg", "gif","png","bmp");
                      <br> &nbsp; &nbsp; 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>chooser.setFileFilter(filter);
                      <br> &nbsp; &nbsp; 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>chooser.setMultiSelectionEnabled(true);
                      <br> &nbsp; &nbsp; 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>int returnVal = chooser.showOpenDialog(null);
                      <br> &nbsp; &nbsp; 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>if(returnVal == JFileChooser.APPROVE_OPTION) {
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>File[] Files=chooser.getSelectedFiles();
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>System.out.println("Please wait...");
                      <br> &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>for( int i=0;i&lt;Files.length;i++){ &nbsp; &nbsp; 
                      <br> &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>convertJPGToPDF(Files[i].toString(),"pdfimage"+i+".pdf");
                      <br> &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>}
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>System.out.println("Conversion complete");
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span> &nbsp; &nbsp; &nbsp; &nbsp; }
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>}
                      <br> 
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>public static void convertJPGToPDF(String strimg,String desc){
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>try{
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>//create document object
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>Document doc=new Document();
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>//create pdf writer object to write the document to the output file
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>PdfWriter.getInstance(doc,new FileOutputStream(desc));
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>//get a4 paper size
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>Rectangle r=PageSize.A4;
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>//read the image
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>BufferedImage orImg=ImageIO.read(new File(strimg));
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>//initialize image width and height
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>int width=orImg.getWidth();
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>int height=orImg.getHeight();
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>//resize the image that is bigger than A4 size
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>if(width&gt;r.getWidth())
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>width=(int)r.getWidth();
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>if(height&gt;r.getHeight())
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>height=(int)r.getHeight();
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>//create a blank buffered image
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>BufferedImage bi=new BufferedImage(width, height, BufferedImage.TYPE_INT_RGB);
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>//create graphic2d object from the buffered image
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>Graphics2D g2d=bi.createGraphics();
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>//draw the original image on the buffered image
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>//so the image is resized to fit the A4 paper size if it is bigger than A4 size
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>g2d.drawImage(orImg, 0,0,width,height,null);
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>//store the image data in memory
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>ByteArrayOutputStream bas=new ByteArrayOutputStream();
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>ImageIO.write(bi, "png", bas);
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>//create image from the image data stored in memory
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>Image img=Image.getInstance(bas.toByteArray());
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>//centrally align the image on the pdf page
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>img.setAlignment(Element.ALIGN_CENTER);
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>//open document
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>doc.open();
                      <br> //add image to the document
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>doc.add(img);
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>//close the document
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>doc.close();
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>}catch(Exception e){e.printStackTrace();}
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>}
                      <br> 
                      <br> 
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>
                      <br> }
                     </div> 
                     <br> The process of converting an image to a PDF file in the code above is simple. Firstly, you need to have a BufferedImage object of the original image file. This can be done by using the
                     <b> read</b> method of the ImageIO class. The BufferedImage is resized to fit the A4 paper size if its size is greater than the A4 paper size. This is to make sure that the output PDF file does not contain an image that greater than its size. For the image that is not larger than the A4 paper size, its size remains unchanged. You will need to convert the BufferedImage object back to an Image object since the 
                     <b>add</b> method of the Document object of the iText library accepts only the image object ( not the BufferedImage object). This can be done by using the getInstance method of the Image class in iText library. When the image is ready, you can call the add method of the Document object to add the image to the Document object.
