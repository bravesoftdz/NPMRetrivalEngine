                    <h3 class="post-title entry-title" itemprop="name"> PDF To Text Converter </h3> 
                    <div class="post-header"> 
                     <div class="post-header-line-1"></div> 
                    </div> 
                    <div class="post-body entry-content" id="post-body-7529748217636450314" itemprop="articleBody">
                      The PDFToTextConverter program can be used to 
                     <b>convert a PDF file in to a text file</b>. When the program runs you can selected one or many PDF files for converting. Then wait for a while. The amount of time waiting depends mainly on the number of PDF files you selected and each file size. It is noted that a PDF file that does not have text can not be converted.
                     <br> 
                     <br>

                     <div class="separator" style="clear: both; text-align: center;"> 
                      <a href="http://1.bp.blogspot.com/-LjTlb2LEo4k/UcV1dVWLEPI/AAAAAAAAAXs/7G_iOivWhto/s1600/pdf-to-text.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img alt="PDF To Text Converter" border="0" height="147" src="http://1.bp.blogspot.com/-LjTlb2LEo4k/UcV1dVWLEPI/AAAAAAAAAXs/7G_iOivWhto/s1600/pdf-to-text.png" width="320"></a>

                      <b>PDFToTextConverter source code:</b>
                      <br> 
                      <br> import java.io.File;
                      <br> import java.io.FileInputStream;
                      <br> import java.io.FileOutputStream;
                      <br> import java.io.FileReader;
                      <br> import java.io.FileWriter;
                      <br> import java.io.BufferedWriter;
                      <br> import java.io.IOException;
                      <br> import com.itextpdf.text.Document;
                      <br> import com.itextpdf.text.pdf.PdfReader;
                      <br> import com.itextpdf.text.pdf.parser.PdfTextExtractor;
                      <br> import java.awt.Desktop;
                      <br> import javax.swing.filechooser.FileNameExtensionFilter;
                      <br> import javax.swing.JFileChooser;
                      <br> 
                      <br> public class PDFToTextConverter{
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>public static void main(String[] args){
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>selectPDFFiles();
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>}
                      <br> 
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>//allow pdf files selection for converting
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>public static void selectPDFFiles(){
                      <br> 
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>JFileChooser chooser = new JFileChooser();
                      <br> &nbsp; &nbsp; 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>FileNameExtensionFilter filter = new FileNameExtensionFilter("PDF","pdf");
                      <br> &nbsp; &nbsp; 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>chooser.setFileFilter(filter);
                      <br> &nbsp; &nbsp; 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>chooser.setMultiSelectionEnabled(true);
                      <br> &nbsp; &nbsp; 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>int returnVal = chooser.showOpenDialog(null);
                      <br> &nbsp; &nbsp; 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>if(returnVal == JFileChooser.APPROVE_OPTION) {
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>File[] Files=chooser.getSelectedFiles();
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>System.out.println("Please wait...");
                      <br> &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>for( int i=0;i&lt;Files.length;i++){ &nbsp; &nbsp; 
                      <br> &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>convertPDFToText(Files[i].toString(),"textfrompdf"+i+".txt");
                      <br> &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>}
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>System.out.println("Conversion complete");
                      <br> &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>}
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span> &nbsp; &nbsp; &nbsp; 
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>}
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>public static void convertPDFToText(String src,String desc){
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>try{
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>//create file writer
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>FileWriter fw=new FileWriter(desc);
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>//create buffered writer
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>BufferedWriter bw=new BufferedWriter(fw);
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>//create pdf reader
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>PdfReader pr=new PdfReader(src);
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>//get the number of pages in the document
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>int pNum=pr.getNumberOfPages();
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>//extract text from each page and write it to the output text file
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>for(int page=1;page&lt;=pNum;page++){
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>String text=PdfTextExtractor.getTextFromPage(pr, page);
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>bw.write(text);
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>bw.newLine();
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>}
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>bw.flush();
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>bw.close();
                      <br> 
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>}catch(Exception e){e.printStackTrace();}
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>}
                      <br> 
                      <span class="Apple-tab-span" style="white-space: pre;"> </span>
                      <br> }
                      <br> 
                      <br> Converting a pdf document to text file is simple. Firstly, you need to use the PdfReader class (in iText library) to get all pages of the pdf document. One you have the PdfReader object, you can extract the text from the pdf document by using the getTextFromPage(PdfReader pdfreader, int page_num) method of the PdfTextExtractor class. &nbsp;This method extract the text from each page of the PdfReader object. While getting the text, you will use the BufferedWriter class to write the text out to a destination file.
