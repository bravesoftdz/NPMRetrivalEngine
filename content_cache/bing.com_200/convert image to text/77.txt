      <h1 class="entrytitle" id="post-8993"> <a href="https://www.programcreek.com/2013/09/convert-image-to-string-in-python/" rel="bookmark"> Python: Convert Image to String, Convert String to Image </a> </h1> 
      <div class="entrymeta">
        &nbsp;  
      </div> 
      <div class="entrybody"> 
       <p>To store or transfer an image, we often need to convert an image to a string in such a way that the string represents the image. Like other programming languages (e.g. Java), we can also convert an image to a string representation in Python. </p> 
       <p><span id="more-8993"></span></p> 
       <p>Converting in Python is pretty straightforward, and the key part is using the "base64" module which provides standard data encoding an decoding. </p> 
       <p><strong>Convert Image to String</strong></p> 
       <p>Here is the code for converting an image to a string.</p> 
       <div class="wp_syntax" style="position:relative;">
        <table>
         <tbody>
          <tr>
           <td class="code"><pre class="python" style="font-family:monospace;"><span style="color: #ff7700;font-weight:bold;">import</span> <span style="color: #dc143c;">base64</span>
&nbsp;
<span style="color: #ff7700;font-weight:bold;">with</span> <span style="color: #008000;">open</span><span style="color: black;">(</span><span style="color: #483d8b;">"t.png"</span><span style="color: #66cc66;">,</span> <span style="color: #483d8b;">"rb"</span><span style="color: black;">)</span> <span style="color: #ff7700;font-weight:bold;">as</span> imageFile:
    <span style="color: #008000;">str</span> <span style="color: #66cc66;">=</span> <span style="color: #dc143c;">base64</span>.<span style="color: black;">b64encode</span><span style="color: black;">(</span>imageFile.<span style="color: black;">read</span><span style="color: black;">(</span><span style="color: black;">)</span><span style="color: black;">)</span>
    <span style="color: #ff7700;font-weight:bold;">print</span> <span style="color: #008000;">str</span></pre></td>
          </tr>
         </tbody>
        </table>
        <p class="theCode" style="display:none;">import base64 with open("t.png", "rb") as imageFile: str = base64.b64encode(imageFile.read()) print str</p>
       </div>

fh.<span style="color: black;">close</span><span style="color: black;">(</span><span style="color: black;">)</span></pre></td>
