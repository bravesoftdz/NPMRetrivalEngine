                    <div id="aQPSRa"></div>
                    <span class="rendered_qtext">Tika uses tesseract to perform OCR on images. You need to turn it on within the configuration properties of the TesseractParser. <br><br>However, I'm not sure it runs on PDF files. My suggestion would be to scan the documents as images. And then run it through tika. <br><br>I've run it a few times on some of my scanned assignments. It was ok at extracting the text. <br><br>It basically comes down to how good tesseract is at doing OCR on handwritten documents. <br><br>You can use tika-python package to get a python API to tika. <br><br>But it basically piggybacks over the Java implementation, which can be run as a standalone server using tika-server.<br><br>The tika-Python package gives you a nice way of communicating with this server for your Python projects.</span>
