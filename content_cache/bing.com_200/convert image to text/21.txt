              <h2 id="ctl00_ctl00_MC_AMC_Answers_ctl00_A_Title" class="float-left" style="margin:-10px 0 0 5px">Solution 3</h2>
              <a id="ctl00_ctl00_MC_AMC_Answers_ctl00_A_Heading" name="answer3"></a> 
              <div class="float-left" style="margin:3px 0 0 20px"> 
               <a id="ctl00_ctl00_MC_AMC_Answers_ctl00_A_Accept" title="Mark this solution as an accepted solution." class="button safe" style="display:none;">Accept&nbsp;Solution</a>

             <div class="text">
              What you are referring to is called Optical Character Recognition (OCR). There are a lot of great articles on CodeProject on how to OCR. Some of them talk about free methods but from my experience, they either have low accuracy or do not provide many features, such as:
              <br> http://www.codeproject.com/Articles/238114/Realtime-Webcam-Sudoku-Solver (does a lot more than OCR, but the OCR part is specific to numerical digits in Sudoku puzzles)
              <br> 
              <a href="http://www.codeproject.com/Articles/17291/OCR-With-MODI-in-Visual-C">OCR With MODI in Visual C++</a> (uses MS control in C++, but can use same control in C#).
              <br> 
              <br> On the other hand, there are some commercial toolkits that provide more features and higher accuracy. You can find an article on one of them in the following link:
              <br> 
              <a href="http://www.codeproject.com/Articles/317163/OCR-with-LEADTOOLS-The-Better-Choice">OCR with LEADTOOLS: The Better Choice</a>

$(function() {
                                    var lasttime=new Date().getTime();
                                    $('#NewEmailError').hide();
                                    var emailEle = $('#ctl00_ctl00_MC_AMC_PostEntryObj_MemberLogOn_QuickEmail');
                                    emailEle.blur(function() {
                                        var email = $.trim(emailEle.val());
                                        if (email.length > 0)
                                            checkEmail(email);
                                    });
                                    emailEle.keyup(function() {
                                        var email = $.trim(emailEle.val());
                                        var current=new Date().getTime();
                                        if (email.length > 0 && (current-lasttime) > 1000) {
                                            checkEmail(email);
                                            lasttime=current;
                                        }
                                    });
                                });
                                function checkEmail(email) {
                                    $.post('/script/Membership/Ajax/EmailExists.aspx?meml=' + escape(email),
                                          function(data) {
                                            if (data.toLowerCase() == 'true')
                                                $('#NewEmailError').show();
                                            else
                                                $('#NewEmailError').hide();
                                        });
                                }
