        <h1 class="post-title entry-title"><a rel="bookmark" href="http://infoheap.com/convert-html-canvas-to-png-image/">How to convert HTML canvas to png image using javascript</a></h1> 
        <div class="byline">
         By 
         <span class="author vcard"><span><a class="url fn n" rel="author" href="http://infoheap.com/author/admin/" title="Parveen Kumar">Parveen Kumar</a></span></span> | Last updated on 
         <time class="updated" datetime="2016-01-29">Jan 29, 2016</time> 
        </div> 
        <div class="entry-content"> 
         <p><a href="http://en.wikipedia.org/wiki/Canvas_element" onclick="__gaTracker('send', 'event', 'outbound-article', 'http://en.wikipedia.org/wiki/Canvas_element', 'HTML canvas');" title="Canvas Element - wikipedia">HTML canvas</a> element (HTML5) be used to draw 2D shapes using Javascript. Once a shape is drawn, it can also be converted to an image (png, jpeg, etc.) and can be embedded in the HTML page using &lt;img&gt; tag and hence can also be downloaded. Here are the steps to achieve this:</p> 
         <ol> 
          <li>First create an empty canvas element as shown below: <pre class="ih-pre">&lt;canvas id="canvasid" width="200" height="100"&gt;&lt;/canvas&gt;</pre> </li> 
          <li>Then draw some basic shape in it. Here is the javascript code to draw a simple square:<br> <pre class="ih-pre ih-code-src prettyprint">
var canvas = document.getElementById('canvasid');
var context = canvas.getContext('2d');
context.fillStyle="#ff0000";
context.fillRect(10,10,60,60);
</pre> </li> 
          <li>Now using canvas&nbsp;<code>toDataURL("image/png")</code> get the png image url and populate the src attribute of required image. You may also want to see <a href="https://developer.mozilla.org/en-US/docs/Web/API/HTMLCanvasElement" onclick="__gaTracker('send', 'event', 'outbound-article', 'https://developer.mozilla.org/en-US/docs/Web/API/HTMLCanvasElement', 'Canvas element reference');" title="HTMLCanvasElement - Web API reference - MDN">Canvas element reference</a>. Here is how the code looks like:<br> <pre class="ih-pre ih-code-src prettyprint">
&lt;img id="canvasimg" src="" /&gt;
&lt;script type="text/javascript"&gt;
$("#canvasimg").attr("src", $("#canvasid").get(0).toDataURL("img/png"));
&lt;/script&gt;
</pre> &nbsp;Note that the value returned by&nbsp;<code>toDataURL("image/png")</code> looks something like this:<p></p> <pre class="ih-pre">data:image/png;base64,iVBORw0KGgo.....</pre> </li> 
          <li>Here is how the final image looks like:<br> <img class="alignnone size-full wp-image-3644" alt="canvas-to-png-image-example" src="http://infoheap.com/wp-content/uploads/2013/07/canvas-to-png-image-example.png" width="264" height="186">You can see the live demo at&nbsp;<a title="HTML5 canvas to image example" href="http://infoheap.com/demo/canvas-image-download-example/code.html">Canvas to image example</a>.</li> 
          <li>To download the image, you can right click on it and download as a regular image.</li> 
         </ol> 
         <h2>Notes on browsers compatibility</h2> 
         <p>Canvas being HTML5 element is not supported by old browsers like IE8. It should work in browsers like Chrome, Firefox, IE9, etc. which support HTML5.</p>
