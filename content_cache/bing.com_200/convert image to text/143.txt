   <h1 id="firstHeading" class="firstHeading" lang="en">List of PDF software</h1> 
   <div id="bodyContent" class="mw-body-content"> 
    <div id="siteSub" class="noprint">
     From Wikipedia, the free encyclopedia
    </div>

    <div id="mw-content-text" lang="en" dir="ltr" class="mw-content-ltr">
     <div class="mw-parser-output">
      <p>This is a list of links to articles on <a href="/wiki/Software" title="Software">software</a> used to manage <a href="/wiki/Portable_Document_Format" title="Portable Document Format">Portable Document Format</a> (PDF) documents. The distinction between the various functions is not entirely clear-cut; for example, some viewers allow adding of annotations, signatures, etc. Some software allows <a href="/wiki/Sanitization_(classified_information)" title="Sanitization (classified information)">redaction</a>, removing content irreversibly for security. Extracting embedded text is a common feature, but other applications perform <a href="/wiki/Optical_character_recognition" title="Optical character recognition">optical character recognition</a> (OCR) to convert imaged text to machine-readable form, sometimes by using an external OCR module.</p>

      <h3><span class="mw-headline" id="Editors_3">Editors</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_PDF_software&amp;action=edit&amp;section=19" title="Edit section: Editors">edit</a><span class="mw-editsection-bracket">]</span></span></h3> 
      <ul> 
       <li><a href="/wiki/Adobe_Acrobat" title="Adobe Acrobat">Adobe Acrobat</a>: <a href="/wiki/Adobe_Systems" title="Adobe Systems">Adobe Systems</a>'s proprietary desktop PDF authoring suite.</li>

       <li><a href="/wiki/Foxit_Reader" title="Foxit Reader">Foxit Reader</a>: (<a href="/wiki/Freeware" title="Freeware">Freeware</a>) Highlight text, draw lines, measure distances of PDF documents.</li> 
       <li><a href="/wiki/Global_Graphics#Information_about_gDoc_Fusion" title="Global Graphics">gDoc Fusion</a>: proprietary/shareware desktop product to create, review, edit, share or archive PDF and XPS documents. After 30 days a watermark is placed on documents in shareware version.</li> 
       <li><a href="/wiki/LibreOffice" title="LibreOffice">LibreOffice</a>: free (<a href="/w/index.php?title=MPLv2&amp;action=edit&amp;redlink=1" class="new" title="MPLv2 (page does not exist)">MPLv2</a>) GUI tools to manipulate, split, merge, encrypt and decrypt PDF files</li>

      <h3><span class="mw-headline" id="Viewers_4">Viewers</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_PDF_software&amp;action=edit&amp;section=20" title="Edit section: Viewers">edit</a><span class="mw-editsection-bracket">]</span></span></h3>

       <li><a href="/wiki/Javelin_PDF_Reader" title="Javelin PDF Reader">Javelin PDF Reader</a>: A full functionality PDF reader with support for DRM using encoded and encrypted PDF files in Drumlin's DRMX and DRMZ formats.</li> 
       <li><a href="/wiki/Microsoft_Edge" title="Microsoft Edge">Microsoft Edge</a>: Includes a PDF viewer.</li> 
       <li><a href="/wiki/Microsoft_Reader" title="Microsoft Reader">Microsoft Reader</a>: A dedicated PDF viewer that is free to download from Windows store.</li> 
       <li><a href="/wiki/Mozilla_Firefox" class="mw-redirect" title="Mozilla Firefox">Mozilla Firefox</a>: Includes a PDF viewer.</li> 
       <li><a href="/wiki/MuPDF" title="MuPDF">MuPDF</a>: Free lightweight document viewer.</li> 
       <li><a href="/wiki/Nitro_PDF#Nitro_Reader" title="Nitro PDF">Nitro PDF Reader</a>: Freeware (though proprietary) PDF reader and creator. Supports three methods (specifying the file within the reader's interface, or dragging-'n-dropping a file onto the reader's Windows desktop icon, or "printing" to a virtual printer driver) of PDF creation (<a href="/wiki/Ghostscript" title="Ghostscript">Ghostscript</a> not additionally needed), saving filled forms (AcroForms), text typewriter, markup/collaboration, and stamp signature (document signing).</li> 
       <li><a href="/wiki/Nuance_PDF_Reader" title="Nuance PDF Reader">Nuance PDF Reader</a>: Freeware (adware) PDF reader, supports saving filled forms, partial support for XFA forms; enables the capability to convert PDF files to MS Word or Excel files through Nuance website.</li>

       <li><a href="/wiki/STDU_Viewer" title="STDU Viewer">STDU Viewer</a>: A freeware for non-commercial usage PDF reader. It also supports DjVu, Comic Book Archive (CBR or CBZ), XPS, TIFF, TXT and image file formats.</li>

      <h2><span class="mw-headline" id="Web-based">Web-based</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_PDF_software&amp;action=edit&amp;section=22" title="Edit section: Web-based">edit</a><span class="mw-editsection-bracket">]</span></span></h2> 
      <h3><span class="mw-headline" id="Converters_5">Converters</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_PDF_software&amp;action=edit&amp;section=23" title="Edit section: Converters">edit</a><span class="mw-editsection-bracket">]</span></span></h3> 
      <ul> 
       <li><a href="/wiki/Cometdocs" title="Cometdocs">Cometdocs</a>: Online PDF conversion</li> 
       <li><a href="/wiki/Zamzar" title="Zamzar">Zamzar</a>: Online file converter</li> 
       <li><a href="/wiki/Smallpdf" title="Smallpdf">Smallpdf</a>: Offers free Online PDF conversion (PDF to other formats and vice versa)</li> 
      </ul> 
      <h3><span class="mw-headline" id="Editors_4">Editors</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_PDF_software&amp;action=edit&amp;section=24" title="Edit section: Editors">edit</a><span class="mw-editsection-bracket">]</span></span></h3> 
      <ul> 
       <li><a href="/wiki/Pdfescape" class="mw-redirect" title="Pdfescape">PDFescape</a>: an advertising and fee supported web service to view, create forms, fill out forms, and edit PDF documents from a web browser (requires JavaScript to be enabled)</li> 
       <li><a href="/wiki/Pdfvue" title="Pdfvue">PDFVue</a>: a free web application that allows the user to view PDF's, comment and fill PDF forms from a web browser. Generates a watermark.</li> 
       <li><a href="/wiki/Smallpdf" title="Smallpdf">Smallpdf</a>: Free web-based PDF software for editing, signing, compressing, merging, splitting, rotating, unlocking and protecting PDF files. No registration needed.</li> 
      </ul> 
      <h3><span class="mw-headline" id="Viewers_5">Viewers</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_PDF_software&amp;action=edit&amp;section=25" title="Edit section: Viewers">edit</a><span class="mw-editsection-bracket">]</span></span></h3> 
      <ul> 
       <li><a href="/wiki/A.nnotate" title="A.nnotate">A.nnotate</a>: a web service which views PDF documents as HTML in the browser, with annotation features.</li> 
       <li><a href="/wiki/DigiSigner" title="DigiSigner">DigiSigner</a>: free online PDF viewer java applet with additional function to digitally sign PDF documents.</li> 
       <li><a href="/wiki/Docstoc" title="Docstoc">Docstoc</a>: a web service which allows viewing PDF documents online</li> 
       <li><a href="/wiki/Issuu" title="Issuu">Issuu</a>: a web service which allows viewing PDF documents online</li> 
       <li><a href="/wiki/Google_Docs" class="mw-redirect" title="Google Docs">Google Docs</a>: a web service which views PDF documents as PNG images in the browser</li> 
       <li><a href="/wiki/Scribd" title="Scribd">Scribd</a>: a web service which renders PDF documents as HTML5 in the browser</li> 
       <li><a href="/wiki/Pdf.js" class="mw-redirect" title="Pdf.js">Pdf.js</a>: A Javascript-based library, with viewer, and browser plugin.</li> 
       <li><a href="/wiki/PDFTron_Systems" title="PDFTron Systems">PDFTron Systems</a>: WebViewer. A free JavaScript viewer and annotator for self-hosting of web optimized PDFs.</li> 
      </ul>
