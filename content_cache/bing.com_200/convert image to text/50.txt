         <h2 class="h2title">Convert Between Document and Image Formats</h2> 
         <p>Using native .NET document conversion API – programmers can employ just a single method to convert between 50 documents and image file formats within their ASP.NET, C# and .NET applications.</p>

        <div class="col-lg-12"> 
         <h2 class="h2title">Installation and Usage</h2> 
         <p>GroupDocs.Conversion for .NET API files can be installed/downloaded directly as:</p> 
         <ul> 
          <li><a href="https://downloads.groupdocs.com/conversion/net">Zip Archive and MSI Package</a></li> 
          <li><a href="https://www.nuget.org/packages/groupdocs-conversion-dotnet/" rel="nofollow">NuGet</a></li> 
         </ul> 
         <p>Once the APIs package is installed using any of the these sources – just add reference of GroupDocs.Viewer.dll file and you are ready to render and display documents in over 50 formats.</p> 
        </div> 
        <div class="col-lg-12"> 
         <h2 class="h2title">How API Works</h2> 
         <ul> 
          <li>Gets the file path as input.</li> 
          <li>Gets the file storage path.</li> 
          <li>Converts the document of input file.</li> 
          <li>Performs conversion operation and return the output (As Path, Stream) (if required)</li> 
         </ul>
