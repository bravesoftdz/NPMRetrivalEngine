       <p>A <a href="http://en.wikipedia.org/wiki/Text_file" class="external" title="Wikipedia: Text file"><b>text file</b></a> is a kind of computer file that is structured as a sequence of lines. <b>.TXT</b> is a filename extension for files consisting of text usually contain very little formatting (ex: no bolding or italics). Files with the .txt extension can easily be read or opened by any program that reads text and, for that reason, are considered universal (or platform independent). </p> 
       <p> Sometimes, we want to <b>convert TXT file to image</b> and then save as JPG, GIF, BMP, PNG, TIF or EMF(WMF) image format, in this tutorial we will see how we can convert TXT format to image format using <a href="http://html-to-image.acasystems.com/" title="ACA HTML to Image Converter: convrt web page/MFT file/TXT file to image, save to TIFF, JPEG/JPG, GIF, BMP, PNG format">ACA HTML to Image Converter</a>.</p> 
       <h2>Tutorial: Convert TXT to image</h2> 
       <p>ACA HTML to Image Converter allows you convert web page, MHT and <b>TXT</b> file to image or thumbnail of a specified size in batch, saves the long page as JPG, GIF, PNG, BMP, TIFF, WMF and EMF image by one click, even if it scrolls off-screen. This sample shows how to convert TXT file from local computer and save as a PNG image.</p> 
       <ol> 
        <li>Launch <b>ACA HTML to Image Converter</b>. If you didn't install it on your computer, please <a href="http://www.acasystems.com/download/html-to-image-converter-setup.exe" title="Free download TXT to Image converter: ACA HTML to Converter">download ACA HTML to Convertre</a> and then install it.</li> 
        <li>Click <b>Add File(s)</b> button in the toolbar:<p></p> <p align="center"><img src="http://image.acasystems.com/html-to-image-converter/faq-convert-txt-to-image-1.png" width="580" height="400" border="0" id="img_box" alt="Convert TXT to image: Click Add File(s) button in the toolbar"></p> </li>
        <li>Select <b>.TXT file(s)</b> in the Open dialog, and then click the Open button. For example: <i>d:\share\txt-to-image\news.txt</i>.</li> 
        <p align="center"><img src="http://image.acasystems.com/html-to-image-converter/faq-convert-txt-to-image-2.png" width="571" height="421" id="img_box" alt="Convert TXT to image: Select .TXT file(s) in the Open dialog, and then click the Open button. For example: d:\share\txt-to-image\news.txt."></p> 
        <li>The convert task will be started:</li> 
        <p align="center"><img src="http://image.acasystems.com/html-to-image-converter/faq-convert-txt-to-image-3.png" width="580" height="400" id="img_box" alt="Convert TXT to image: The convert task will start"></p> 
        <li>If the task is completed, you can select the task item, and then click "<b>View Image</b>" button to open the image in your default image viewer. </li> 
        <p align="center"><img src="http://image.acasystems.com/html-to-image-converter/faq-convert-txt-to-image-4.png" width="580" height="400" id="img_box" alt="Convert TXT to image: view image"></p> 
       </ol> 
       <p>Note: </p> 
       <ol> 
        <li><i>The default <b>image format</b> is set to <b>PNG</b> format (the best web image format). If you want to save TXT file to other image format (JPEG/BMP/GIF/TIFF/BMP/EMF/WMF), just simply click menu <b>Tools</b> -&gt; <b>Options</b> -&gt; <b>Image</b> to change the image format.</i></li> 
        <li><i>Thd default <b>image width</b> is set to <b>1024 pixels</b>. If you want to adjust the image width to other value, just click menu <b>Tools</b> -&gt; <b>Options</b> -&gt; <b>Browser</b> to change the Browser width.</i></li>
