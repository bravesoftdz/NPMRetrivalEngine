                 <p></p> 
                </div> 
               </div> 
              </div> 
             </div>
            </div> </li> 
          </div> 
         </ol> 
        </div> 
       </div> 
       <div class="header parbase section"> 
        <a name="main-pars_heading_5"></a> 
        <div class="header  "> 
         <h3 id="conversion_options"> Conversion options </h3> 
        </div> 
        <!-- for Cascading Toc --> 
       </div> 
       <div class="text parbase section"> 
        <a name="main-pars_text_4"></a> 
        <div class="text">
         <p>You can configure conversion options before you save the file. By default, the conversion options specified under <strong>Preferences</strong> are used.&nbsp;</p> 
         <ol> 
          <li>Choose <strong>Edit</strong> &gt; <strong>Preferences</strong> &gt; <strong>Convert From PDF</strong>.</li> 
          <li>Choose a format from the <strong>Converting From PDF</strong> list, and then click <strong>Edit Settings</strong>.</li> 
          <li>Choose the conversion settings, and then click <strong>OK</strong>.</li> 
         </ol> 
        </div> 
       </div> 
       <div class="header parbase section"> 
        <a name="main-pars_heading_6"></a> 
        <div class="header  header-top"> 
         <h2 id="export_images_to_another_format"> Export images in a PDF to another format </h2> 
        </div> 
        <!-- for Cascading Toc --> 
       </div> 
       <div class="text parbase section"> 
        <a name="main-pars_text_5"></a> 
        <div class="text">
         <p>In addition to saving every page (all text, images, and vector objects on a page) to an image format using the <strong>File</strong> &gt; <strong>Export To </strong>&gt;<strong> Image</strong> &gt; [Image Type] command, you can export each image in a PDF to a separate image file.</p> 
        </div> 
       </div> 
       <div class="note parbase section"> 
        <div class="help-note"> 
         <p> <span class="help-note-title">Note:</span> </p>
         <p>You can export raster images, but not vector objects.</p> 
         <p></p> 
        </div> 
       </div> 
       <div class="procedure parbase section">
        <div> 
         <ol> 
          <div class="proc_par proc_parsys"> 
           <li>
            <div class="step section"> 
             <div>
              <div class="step_par step_parsys">
               <div class="text parbase section"> 
                <a name="step_par_text_0"></a> 
                <div class="text">
                 <p>Open the PDF in Acrobat, and then choose <strong>Tools</strong> &gt;&nbsp;<strong>Export PDF</strong>.</p> 
                 <p>The various formats to which you can export the PDF file are displayed.</p> 
                </div> 
               </div> 
              </div> 
             </div>
            </div> </li> 
           <li>
            <div class="step section"> 
             <div>
              <div class="step_par step_parsys">
               <div class="text parbase section"> 
                <a name="step_par_text"></a> 
                <div class="text">
                 <p>Click <strong>Image</strong> and then choose the image file format that you want to save the images in.<br> </p> 
                </div> 
               </div> 
               <div class="image parbase section"> 
                <div class="to400"> 
                 <img src="exporting-pdfs-file-formats/_jcr_content/main-pars/procedure_3/proc_par/step_8/step_par/image.img.png/export_to_image.png" alt="Export to Image Options" title="Export to Image Options" class="cq-dd-image"> 
                 <div class="image-caption">
                   Choose the format that you want to save the exported images in. 
                 </div> 
                </div> 
                <small> <br> </small> 
               </div> 
              </div> 
             </div>
            </div> </li> 
           <li>
            <div class="step section"> 
             <div>
              <div class="step_par step_parsys">
               <div class="text parbase section"> 
                <a name="step_par_text"></a> 
                <div class="text">
                 <p>To configure the conversion settings for the selected file format, click the gear icon <img width="21" height="21" src="/content/dam/help/en/acrobat/using/gear_icon.png">.</p> 
                </div> 
               </div> 
              </div> 
             </div>
            </div> </li> 
           <li>
            <div class="step section"> 
             <div>
              <div class="step_par step_parsys">
               <div class="text parbase section"> 
                <a name="step_par_text"></a> 
                <div class="text">
                 <p>In the <strong>Export All Images As </strong><em>[selected file format]</em><strong> Settings</strong> dialog box, specify the <strong>File Settings</strong>, <strong>Color Management</strong>, <strong>Conversion</strong>, and <strong>Extraction</strong> settings for the file type.<br> </p> 
                </div> 
               </div> 
              </div> 
             </div>
            </div> </li> 
           <li>
            <div class="step section"> 
             <div>
              <div class="step_par step_parsys">
               <div class="text parbase section"> 
                <a name="step_par_text"></a> 
                <div class="text">
                 <p>In the <strong>Extraction</strong> settings, for <strong>Exclude Images Smaller Than</strong>, select the smallest size of image to be extracted. Select <strong>No Limit</strong> to extract all images.<br> </p> 
                </div> 
               </div> 
              </div> 
             </div>
            </div> </li> 
           <li>
            <div class="step section"> 
             <div>
              <div class="step_par step_parsys">
               <div class="text parbase section"> 
                <a name="step_par_text"></a> 
                <div class="text">
                 <p>Click <strong>OK</strong> to return to the return to the <strong>Export Your PDF To Any Format</strong> screen.<br> </p> 
                </div> 
               </div> 
              </div> 
             </div>
            </div> </li> 
           <li>
            <div class="step section"> 
             <div>
              <div class="step_par step_parsys">
               <div class="text parbase section"> 
                <a name="step_par_text"></a> 
                <div class="text">
                 <p>Select the <strong>Export All Images</strong> option to extract and save only the images from the PDF file. <br> </p> 
                </div> 
               </div> 
               <div class="note parbase section"> 
                <div class="help-note"> 
                 <p> <span class="help-note-title">Note:</span> </p>
                 <p>If you do not select the Export All Images option, all pages within the PDF are saved in the selected image file format. &nbsp;</p> 
                 <p></p> 
                </div> 
               </div> 
              </div> 
             </div>
            </div> </li> 
           <li>
            <div class="step section"> 
             <div>
              <div class="step_par step_parsys">
               <div class="text parbase section"> 
                <a name="step_par_text"></a> 
                <div class="text">
                 <p>Click <strong>Export</strong>.<br> The Export dialog box is displayed.<br> </p> 
                </div> 
               </div> 
              </div> 
             </div>
            </div> </li> 
           <li>
            <div class="step section"> 
             <div>
              <div class="step_par step_parsys">
               <div class="text parbase section"> 
                <a name="step_par_text_0"></a> 
                <div class="text">
                 <p>In the <strong>Export</strong> dialog box, select a location where you want to save the file.<br> </p> 
                </div> 
               </div> 
              </div> 
             </div>
            </div> </li> 
           <li>
            <div class="step section"> 
             <div>
              <div class="step_par step_parsys">
               <div class="text parbase section"> 
                <a name="step_par_text_0"></a> 
                <div class="text">
                 <p>Click <strong>Save</strong> to save only the images from the PDF to the selected file format.<br> </p> 
                </div> 
               </div> 
              </div> 
             </div>
            </div> </li> 
          </div> 
         </ol> 
        </div>

        <a name="main-pars_heading_7"></a> 
        <div class="header  header-top"> 
         <h2 id="export_selections_to_another_format"> Export selections from a PDF to another format </h2> 
        </div> 
        <!-- for Cascading Toc --> 
       </div> 
       <div class="text parbase section"> 
        <a name="main-pars_text_6"></a> 
        <div class="text">
         <p>If you need just a part of the PDF file in another format, you don’t need to convert the entire file and then extract the relevant content. You can select text in a PDF file and save it in one of the supported formats: DOCX, DOC, XLSX, RTF, XML, HTML, or CSV.</p> 
        </div>
