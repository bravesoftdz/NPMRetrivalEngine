        <h2>About Facebook</h2>
        <p>The “<span style="color: #000000;">smartphone fever</span>” has taken a stronger grip on the world. As a result various brands are introducing new models and sophisticated features every season. Meanwhile, consequently, there has been a boom in the mobile application development industry.</p>
        <p>An average teenager is spending 27 hours online. Facebook has emerged as the primary platform through which App developers try to capture the attention of the mass, in the hope of going “viral.” Facebook is flooding with applications, especially those designed for the youth.</p>
        <p>Where the trend of playing <a href="https://www.farmville.com">Farmville</a> is ebbing, it does not seem the annoying <a href="http://candycrushsaga.com">Candy Crush</a> requests are going to stop anytime soon. Most of us encountered <a href="https://www.gotinder.com/">Tinder</a>, <a href="https://www.pinterest.com/">Pinterest</a>, <a href="https://www.scribd.com/">Scribd</a>, <a href="https://www.change.org/">Change.org</a> through Facebook and ended up getting hooked forever. Then there are the quizzes. Although most of us claim they are silly, we cannot deny having been affected by the virus at some point in time.</p>
        <p><img class="aligncenter" src="https://i.ytimg.com/vi/oLaeko6CUQA/maxresdefault.jpg" alt="facebook sdk developers" width="489" height="275"></p>
        <h3>Facebook SDKs</h3>
        <p>Facebook has created a simple and comprehensive method for creation of applications. The pre-requisite is a <a href="https://developers.facebook.com/">Facebook Developer’s Account</a>. Developer accounts you can do by updating the existing account of a Facebook user. On doing so, the option for creating a new app appears, under the option App at the header.</p>
        <p>Facebook offers a Quick Start Wizard for setting up the application in the initial stages, on the specified platform. The platforms available for app development are <a href="https://en.wikipedia.org/wiki/Android_(operating_system)">Android</a>, <a href="https://en.wikipedia.org/wiki/IOS">iOS</a>, <a href="https://canvas.facebook.com/">Facebook Canvas</a> and Website, one of which you need to select at the development stage. Later, you can make provision for making your application available on multiple platforms-. Finally, developers can save their progress, return to continue with their work and add multiple platforms using this.</p>
        <p>A sound knowledge of Facebook SDK is indispensable when it comes to <a href="https://developers.facebook.com/docs/apps">App Development on Facebook</a>. SDKs or Software Development Kit is a programming package that enables a programmer to develop Applications for a particular platform. The package consists of one or more application programming interfaces, programming tools, and documentations, using the unique App ID found on the dashboard of the application.</p>
        <p>An app developer needs Facebook Software Development Kit to integrate Facebook products quickly using common languages and platforms.</p>
        <h3>The list of official SDKs of Facebook include:</h3>
        <ul>
         <li><a href="https://developers.facebook.com/docs/apis-and-sdks">Facebook Official SKDs</a></li>
         <li><a href="https://hackerapp.com/ios/">Facebook SDK for iOS:</a> This allows users to log into the app through Facebook. It is an efficient tool for sharing the application and keeping an account of its success.</li>
         <li><a href="https://hackerapp.com/javascript/">Facebook SDK for JavaScript:</a> This comes in handy as it contains a useful set of client-side functionality such as Social Plug-Ins, making API calls and Facebook login.</li>
         <li><a href="https://hackerapp.com/android/">Facebook SDK for Android:</a> Apart from giving users the ability to share and log in through Facebook, this also helps keep track of app download and app usage.</li>
         <li><a href="https://hackerapp.com/php/">Facebook SDK for PHP:</a> It is very useful for accessing graph APIs. The option of Facebook login is available as well.</li>
         <li><a href="https://developers.facebook.com/docs/unity/">Facebook SDK for Unity:</a> Through this SDK, integrating social features in Cross Platform Unity games become fast and easy.</li>
         <li><a href="https://github.com/facebook/facebook-php-ads-sdk">Facebook Ads SDK for PHP:</a> A boon for users using the Ad APIs, as it helps manage Facebook Ad Campaigns, Custom Audiences, and reports</li>
        </ul>
        <p>Apart from these, <a href="https://developers.facebook.com/docs/apis-and-sdks">Facebook offers a multitude of third party SDKs for a range of other language and frameworks</a>. These include SDKs by <a href="http://www.cocos2d-x.org/sdkbox">Cocos 2Dx</a>, <a href="https://www.microsoft.com/en-us/windows">Windows by Microsoft</a>, <a href="https://hackerapp.com/net/">Facebook SDK for .NET</a>,&nbsp;&nbsp;<a href="https://www.adobe.com/software/flash/about/">Flash supported by Adobe</a>, &nbsp;<a href="https://www.djangoproject.com/">Django</a>, <a href="https://nodejs.org/en/">Node.js</a>,&nbsp;<a href="https://en.wikipedia.org/wiki/Objective-C">Objective C</a>, <a href="https://en.wikipedia.org/wiki/HTML5">HTML 5</a>, <a href="https://www.qt.io/">Qt</a>, <a href="https://www.java.com/en/">Java</a>, <a href="http://rubyonrails.org/">Ruby</a>, <a href="http://v-play.net/">V Play</a>, <a href="https://www.lua.org/">Lua</a>, &nbsp;<a href="https://facebook.hackerapp.com">Facebook Hacker</a>,&nbsp;<a href="https://github.com/winjs/winjs">WinJs</a>.</p>
