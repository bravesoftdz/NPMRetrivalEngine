       <div class="page-content-inner">
        <p class="alert alert-important"><span class="icon-placeholder" title="Important"><img width="36" height="36" src="../img/all/ico/important.png" alt="Important"></span>The following article uses options that are available starting with the <a href="/top-java-features.html" title="For Java">For Java</a> edition and project type.</p>
        <p> This tutorial will present in detail how to achieve this goal. In order to replicate it on your own machine, you need to have a Sun JDK installed (the latest version is recommended).</p>
        <ul class="toc">
         <li>1. <a href="#create">Create project</a></li>
         <li>2. <a href="#product-details">Enter Product Details</a></li>
         <li>3. <a href="#add-files">Add Installation Files</a></li>
         <li>4. <a href="#define-products">Define Java Products</a></li>
         <li>5. <a href="#program-shortcuts">Create program shortcut</a></li>
         <li>6. <a href="#build">Build</a></li>
         <li>7. <a href="#testing">Run, test and remove</a></li>
         <li>8. <a href="#section363">Video tutorial</a></li>
         <li>9. <a href="#section364">See also</a></li>
        </ul>
        <h2><a name="create"></a>1. Create project</h2>
        <p>First you must download and unzip the <a href="../examples/JavaProductExample.zip" title="Java Product Example files">Java Product Example</a>. It contains all the files necessary to finish this tutorial, including a sample Advanced Installer project. </p>
        <p>Let's say that we want to create a Windows Installer package for the included Java Product example. Start Advanced Installer and choose “Java” &gt; “Java Application” as the project type.</p>
        <p><img width="704" height="277" src="../img/tutorial/java/new-project.png" alt="Start Page" title="Start Page"></p>
        <p><img width="24" height="24" src="../img/toolbar/project-save.png" alt="Save" class="float-left" title="Save"> A good idea is to save and name the project at this point. Use the “Save” menu item under the “File” menu or the <span class="shortcut">Ctrl+S</span> keyboard shortcut and let's name it JavaProductExample.aip.</p>
        <h2><a name="product-details"></a>2. Enter Product Details</h2>
        <p><img width="403" height="421" src="../img/tutorial/java/product-details.png" alt="Product Details" title="Product Details"></p>
        <p>Note that these details are completely fictive.</p>
        <h2><a name="add-files"></a>3. Add Installation Files</h2>
        <p><img width="24" height="24" src="../img/pages/files-folders.png" alt="Files and Folders" class="float-left" title="Files and Folders"> Go to the <a href="files-folders.html" title="On this page you can add files, folders and shortcuts to the installation and set their properties.">Files and Folders</a> page by clicking on “Files and Folders” on the left side page menu.</p>
        <p><img width="24" height="24" src="../img/toolbar/files-add.png" alt="Add File" class="float-left" title="Add File"> Click on the <span class="button">[ Add File ]</span> toolbar button. Browse to where your downloaded the example folder, then select DemoJar.jar file and click <span class="button">[ OK ]</span>.</p>
        <p><img width="618" height="120" src="../img/tutorial/java/files-folders.png" alt="Files and Folders" title="Files and Folders"></p>
        <h2><a name="define-products"></a>4. Define Java Products</h2>
        <p><img width="24" height="24" src="../img/pages/java.png" alt="Java Products" class="float-left" title="Java Products">Go to the <a href="java.html" title="How to define Java products in your installation.">Java Products</a> page by clicking on “Java Products” on the left pane.</p>
        <p><img width="24" height="24" src="../img/toolbar/java-new.png" alt="New Java Product" class="float-left" title="New Java Product">Create a new Java Product by clicking on the <span class="button">[ New Product ]</span> toolbar button. Let's call it Java Product Example.</p>
        <p><img width="194" height="125" src="../img/tutorial/java/java-product.png" alt="Java Product" title="Java Product"></p>
        <p>In the "Settings" tab, click on the <span class="button">[ Add JAR(s)... ]</span> button and select the DemoJar.jar file and click <span class="shortcut">OK</span>.</p>
        <p>Edit the Main class to: DemoMainClass</p>
        <p>Go to "Virtual Machine" tab and set the JVM parameters:</p>
        <ul>
         <li>Initial heap size - 64 MegaBytes</li>
         <li>Maximum heap size - 256 MegaBytes</li>
        </ul>
        <p>Specify the JRE version min and max to whatever version you have installed on your machine.</p>
        <p class="alert alert-note"><span class="icon-placeholder" title="Note"><img width="36" height="36" src="../img/all/ico/note.png" alt="Note"></span> You can set the parameters by clicking on [...] button from the JVM Parameters field this way accessing the <a href="edit-jvm-parameters-dialog.html" title="The &quot;Edit Java Virtual Machine Parameters&quot; dialog in Advanced Installer.">Edit Java Virtual Machine Parameters Dialog</a>. </p>
        <p>Back in the "Settings" tab select GUI Application. Optionally set a splash screen for your java product.</p>
        <p><img width="709" height="894" src="../img/tutorial/java/product-settings.png" alt="Product Settings" title="Product Settings"></p>
        <p>Now a new file has appeared in the Files and Folders Page: Java Product Example.exe.</p>
        <h2><a name="program-shortcuts"></a>5. Create program shortcut</h2>
        <p><img width="24" height="24" src="../img/pages/files-folders.png" alt="Files, Folders and Shortcuts" class="float-left" title="Files, Folders and Shortcuts"> Go to the Files and Folders page by clicking on “Files and Folders” on the left pane.</p>
        <p><img width="24" height="24" src="../img/toolbar/shortcut-new.png" alt="New Shortcut" class="float-left" title="New Shortcut">Select the Java Product Example.exe file and click on the <span class="button">[ New Shortcut ]</span> toolbar button. Set the description to: Java Product Example Application. Find yourself a nice icon on your computer. Click <span class="shortcut">OK</span>.</p>
        <p><img width="405" height="624" src="../img/tutorial/java/new-shortcut.png" alt="New Shortcut" title="New Shortcut"></p>
        <h2><a name="build"></a>6. Build</h2>
        <p><img width="24" height="24" src="../img/toolbar/project-build.png" alt="Build" class="float-left" title="Build">Click on the <span class="button">[ Build ]</span> toolbar button.</p>
        <h2><a name="testing"></a>7. Run, test and remove</h2>
        <p><img width="24" height="24" src="../img/toolbar/project-run.png" alt="Run" class="float-left" title="Run"> Click on the <span class="button">[ Run ]</span> toolbar button to run and test the MSI you created. After installation you should have a Java Product Example folder in your “Start” menu. Run the application by selecting Java Product Example.</p>
        <p><img width="194" height="51" src="../img/tutorial/java/menu.png" alt="Menu" title="Menu"></p>
        <p>To remove the installed package, simply Run it again and select Remove.</p>
