         <p>The Electron framework lets you write cross-platform desktop applications using JavaScript, HTML and CSS. It is based on <a href="https://nodejs.org/">Node.js</a> and <a href="http://www.chromium.org">Chromium</a> and is used by the <a href="https://github.com/atom/atom">Atom editor</a> and many other <a href="https://electron.atom.io/apps">apps</a>.</p>

           </svg></a>Installation</h2> 
         <p>To install prebuilt Electron binaries, use <a href="https://docs.npmjs.com/"><code>npm</code></a>. The preferred method is to install Electron as a development dependency in your app:</p> 
         <div class="highlight highlight-source-shell">
          <pre>npm install electron --save-dev --save-exact</pre>
         </div> 
         <p>The <code>--save-exact</code> flag is recommended as Electron does not follow semantic versioning. For info on how to manage Electron versions in your apps, see <a href="https://electron.atom.io/docs/tutorial/electron-versioning/">Electron versioning</a>.</p> 
         <p>For more installation options and troubleshooting tips, see <a href="https://electron.atom.io/docs/tutorial/installation/">installation</a>.</p>

         <p>Most people use Electron from the command line, but if you require <code>electron</code> inside your <strong>Node app</strong> (not your Electron app) it will return the file path to the binary. Use this to spawn Electron from Node scripts:</p> 
         <div class="highlight highlight-source-js">
          <pre><span class="pl-k">const</span> <span class="pl-c1">electron</span> <span class="pl-k">=</span> <span class="pl-c1">require</span>(<span class="pl-s"><span class="pl-pds">'</span>electron<span class="pl-pds">'</span></span>)
<span class="pl-k">const</span> <span class="pl-c1">proc</span> <span class="pl-k">=</span> <span class="pl-c1">require</span>(<span class="pl-s"><span class="pl-pds">'</span>child_process<span class="pl-pds">'</span></span>)

<span class="pl-c"><span class="pl-c">//</span> will print something similar to /Users/maf/.../Electron</span>
<span class="pl-en">console</span>.<span class="pl-c1">log</span>(electron)

<span class="pl-c"><span class="pl-c">//</span> spawn Electron</span>
<span class="pl-k">const</span> <span class="pl-c1">child</span> <span class="pl-k">=</span> <span class="pl-smi">proc</span>.<span class="pl-en">spawn</span>(electron)</pre>
