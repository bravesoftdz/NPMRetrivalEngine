           <h2></h2> 
           <p>Use FedEx Ground Call Tags Package Return when you have to recall an occasional package delivered by FedEx Ground, within Canada. For a nominal fee, FedEx Ground will pick up and return a package to you.<br><br> You can request a Ground Call Tag online through the return shipment function in <a href="https://www.fedex.com/cgi-bin/ship_it/interNetShip?link=1&amp;origincountry=ca&amp;locallang=en&amp;urlparams=ca_english">FedEx Ship Manager at fedex.ca</a> or by calling <strong style="color:#4D148C;">1.800.GoFedEx 1.800.463.3339</strong>.<br><br> When requesting a Call Tag, you'll need to provide the following: </p>
           <ul> 
            <li>The package tracking ID, your company name and address, the total number of packages you need to recall, and the contents.<br></li> 
            <li>The name and telephone number of a contact person at the pickup location, the company name and address, and the day the package will be ready for pickup.<br><br></li> 
           </ul> 
           <br> FedEx Ground will provide you with a Call Tag confirmation number that lets you track the return of your package.
           <br> 
           <ul> 
            <li>To qualify for Call Tag service the package must have been originally shipped within Canada using <a href="/ca_english/services/domestic/ground.html" lid="FedEx® Ground">FedEx<sup>®</sup> Ground</a>.<br></li> 
            <li>A nominal fee applies for FedEx Ground Call Tag Package Returns. Get an <a href="http://www.fedex.com/ca_english/rates/" lid="online rate quote">online rate quote</a> for your Call Tag, or consult Ground rates in the <a href="/ca_english/services/serviceguide/index.html" lid="FedEx Service Guide">FedEx Service Guide</a>.</li> 
           </ul> 
           <br> Whether you're in retail, wholesale, or manufacturing; a shipper or a consignee, FedEx Ground is committed to helping you better manage your distribution process.
