  <h3>Compiling and running a java program</h3> 
  <ul> 
   <li>To create a Java program, you must ensure that the name of the class in the file is the same as the name of the file (and that the file has the extension .java). <p> </p></li>
   <li>To compile the program use the command <b>javac</b> as in: <pre>
    javac HelloWorld.java
</pre> This will take the source code in the file HelloWorld.java (which better contain the class HelloWorld) and create the Java ByteCode in a file HelloWorld.class. <p> </p></li>
   <li>To run the compiled program use the command <b>java</b> as in: <pre>
    java HelloWorld
</pre> Note that you do not use the any file extension in this command. <p> </p></li>
  </ul>
