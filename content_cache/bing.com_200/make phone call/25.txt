     <h3><b> How do I install Java ? </b></h3> 
     <span class="bodytext"> Choose the Operating System for instructions to install Java: 
      <ul>
       <li><a href="#windows">Windows</a></li> 
       <li><a href="#mac">Mac</a></li> 
       <li><a href="#linux">Linux</a></li> 
       <li><a href="#solaris">Solaris</a></li>
      </ul> </span> 
     <!-- no bodytext -->
     <hr> 
     <span class="bodytext"> <h5 class="sub"><a name="windows">Windows Download and Installation</a></h5> Downloading and installing Java is easy and free. There are a couple ways by which you can get Java for Windows 
      <ul> 
       <li>Online download</li> 
       <li>Offline download</li> 
      </ul> </span> 
     <span class="bodytext"> <h6 class="sub">Online</h6> Manual installation downloads an IFTW (Install From The Web) executable program file and requires minimum user intervention. When you run this program, it fetches all the required files from the web, so you must remain connected to the Internet during the installation. 
      <ul> 
       <li>Administrative permission is required in order to install Java on Microsoft Windows.</li> 
       <li>If you face difficulty using the online download option, try the offline download option.</li>
      </ul> » <a href="/en/download/help/windows_manual_download.xml">Instructions to download and install Java for Windows online</a> <br><br> </span> 
     <span class="bodytext"> <h6 class="sub">Offline</h6> Offline installation requires you to download an executable file available at the manual Java download page, which includes all the files needed for the complete installation at the user's discretion. There is no need to remain connected to the Internet during the installation. The file can also be copied to and installed on another computer that is not connected to the Internet. 
      <ul> 
       <li>Administrative permission is required in order to install Java on Microsoft Windows.</li>
      </ul> » <a href="/en/download/help/windows_offline_download.xml">Instructions to download and install Java for Windows offline</a> <br><br> </span> 
     <!-- no bodytext -->
     <hr> 
     <span class="bodytext"> <h5 class="sub"><a name="mac">Mac Download and Installation</a></h5> Oracle Java can be installed on Mac versions 10.7.3 or later. </span> 
     <span class="bodytext"> 
      <!-- BEGIN: jWidget_C/Raw-HTML/Display --> A 64-bit browser (Safari, for example) is required to run Oracle Java on Mac. <br><br> 
      <!-- END: jWidget_C/Raw-HTML/Display --> </span> 
     <span class="bodytext"> » <a href="/en/download/help/mac_install.xml">Instructions to download and install Java for Mac</a> <br><br> </span> 
     <!-- no bodytext -->
     <hr> 
     <span class="bodytext"> <h5 class="sub"><a name="linux">Linux Download and Installation</a></h5> There are two types of installation packages. 
      <ul> 
       <li>Java on Linux Platforms<br> This is an archive binary file that can be installed by anyone (not only the root users), in any location that you can write to. However, only the root user can install Java into the system location.</li> 
       <li>Java on RPM-based Linux Platforms<br> 32-bit RPM-based Linux platforms, such as Red Hat and SuSE, use a RPM binary file (.rpm) in the system location. You must be root to perform this installation.</li> 
      </ul> Download the package that best suits your needs. You can download the file to any of the directories on your system. <br> » <a href="/en/download/help/linux_install.xml">Instructions to download and install Java for Linux</a> <br><br> </span> 
     <!-- no bodytext -->
     <hr> 
     <span class="bodytext"> <h5 class="sub"><a name="solaris">Solaris Download and Installation</a></h5> There are a few ways by which you can get Java for Solaris 
      <ul> 
       <li>Solaris SPARC (32-bit)</li> 
       <li>Solaris SPARC (64-bit)</li> 
      </ul> Java comes in the form of a self-extracting binary file. Download the appropriate package for your computer architecture and operating system. <br> » <a href="/en/download/help/solaris_install.xml">Instructions to download and install Java for Solaris</a> <br><br> </span>
