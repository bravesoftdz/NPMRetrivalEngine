      <h1 class="title2">Make a Clickable Phone Link</h1> 
      <h2 class="subtitle">Phone number links on smartphones</h2> 
      <div class="page-splits"></div> 
      <img src="picts/mobile-phone-4.jpg" height="106" width="70" class="float-left" alt="Clickable Smartphone"> 
      <div class="text-nowrap">
        Use the link code shown below on your HTML webpages to make a phone number a clickable link that opens a dial dialog on smartphones and mobile devices, or prompts the screen to add the number to the smartphone address book.
       <br> 
       <br> 
       <span class="list-title">See Also…<br></span> 
       <ul class="SI-list"> 
        <li> <a href="S_add-links.htm">Adding page links</a> </li>
        <li> <a href="S_add-links-email.htm">E-mail links (with hide code)</a> </li>
       </ul> 
      </div>

      <br> 
      <span class="colortitle">Clickable smartphone link code:</span>
      <br> The following link can be used to make a clickable phone link. You can copy the code below and paste it into your webpage, then edit with your phone number. This code may not work on all phones but does work for iPhone, Droid / Android and Blackberry. 
      <br>
      <br>
      <br> 
      <span class="code"> &lt;a href="tel:<span class="green">1-847-555-5555</span>"&gt;<span class="green">1-847-555-5555</span>&lt;/a&gt; </span> 
      <br>
      <br>
      <br> Phone number links can be used with the dashes, as shown above, or without them as well as in the following example: 
      <br>
      <br>
      <br> 
      <span class="code"> &lt;a href="tel:<span class="green">18475555555</span>"&gt;<span class="green">1-847-555-5555</span>&lt;/a&gt; </span> 
      <br>
      <br>
      <br> It's also possible to use any text for the link name as long as the phone number link is setup with the "tel:18475555555" as in this example: 
      <br>
      <br>
      <br> 
      <span class="code"> &lt;a href="tel:<span class="green">18475555555</span>"&gt;<span class="green">Click Here To Call Support 1-847-555-5555</span>&lt;/a&gt; </span> 
      <br>
      <br>
      <br> Below is a clickable telephone hyperlink you can test. In non-phone browsers (PC and Mac desktop and laptop computers), this link may open Skype, ask you for the application to use, may give the user an error, or nothing will happen. 
      <br>
      <br>
      <br> 
      <span class="code"> <a href="tel:1-847-555-5555">Click Here to Call on a Smartphone</a> </span> 
      <br>
      <br>
      <br> 
      <span class="red">Note:</span> iPhone and iPad browsers will automatically detect a phone number on a page and will convert the text into a call link without using any of the code on this page. 
      <br>
      <br>
      <br> 
      <span class="colortitle">Wtai smartphone link code:</span>
      <br> The wtai or "Wireless Telephony Application Interface" link code is shown below. This code is considered to be the correct mobile phone protocol and will work on smartphones like Droid, however, it may not work for Apple Safari on iPhone and the above code is recommended. 
      <br>
      <br>
      <br> 
      <span class="code"> &lt;a href="wtai://wp/mc;<span class="green">18475555555</span>"&gt;<span class="green">Click Here To Call Support 1-847-555-5555</span>&lt;/a&gt;<br> </span>
