         <p>How to Compile JAVA Program in Package from command line </p> 
        </div>
        <!-- .entry-summary --> 
        <div class="entry-content"> 
         <p>The topic is very easy and i am sure that lots of java programmer already know how to compile java program existing in package. However there are users who frequently works on eclipse, netbeans or any other IDE and don’t know that how the program actually works behind the IDE. So this article basically emphasizes on basics of Java Compilation.</p> 
         <p>Consider following directory structure :</p> 
         <figure id="attachment_1985" style="width: 154px" class="wp-caption aligncenter">
          <a href="http://JitendraZaa.com/blog/wp-content/uploads/2011/04/Java-Directory-structure.jpg"><img class="size-full wp-image-1985" title="Java Directory structure" src="http://JitendraZaa.com/blog/wp-content/uploads/2011/04/Java-Directory-structure.jpg" alt="Java Directory structure" width="154" height="106"></a>
          <figcaption class="wp-caption-text">
           Java Directory structure
          </figcaption>
         </figure> 
         <p><span id="more-1984"></span>Our java files will be in “<span style="text-decoration: underline;">src</span>” folder. I want all the class files in folder “<span style="text-decoration: underline;">classes</span>” and currently i am in folder “<span style="text-decoration: underline;">Exec Jar</span>“.</p> 
         <p>Consider below Java Files:</p> 
         <p>Person.java</p> 
         <pre class="brush: java; title: ; notranslate" title="">
package com.g2.ExecJar;

public class Person {
	private String name;
	private int age;

	public String getName() {
		return name;
	}

	public void setName(String name) {
		this.name = name;
	}

	public int getAge() {
		return age;
	}

	public void setAge(int age) {
		this.age = age;
	}
}

</pre> 
         <p>Start.java</p> 
         <pre class="brush: java; title: ; notranslate" title="">
package com.g2.ExecJar;

import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;

public class Start {

	public static void main(String[] args) {

		try {
			InputStreamReader io = new InputStreamReader( System.in );
			BufferedReader read = new BufferedReader(io);

			System.out.println("Enter Name");
			Person p = new Person();
			p.setName(read.readLine());

			System.out.println("Enter Age");
			p.setAge(Integer.parseInt( read.readLine()) );

			System.out.println( "-------- You have Eneterd --------- " );
			System.out.println("Name : " + p.getName());
			System.out.println("Age : " + p.getAge());

			System.out.println( "---------- Progarm run by executable jar ------------ " );

		} catch (IOException e) {
			e.printStackTrace();
		} catch (NumberFormatException e) {
			System.out.println("Age entered is not in correct format");
		}
	}
}

</pre> 
         <p>Run below steps to compile and run java programs from command line:</p> 
         <figure id="attachment_1987" style="width: 481px" class="wp-caption aligncenter">
          <a href="http://JitendraZaa.com/blog/wp-content/uploads/2011/04/Compile-and-run-Java-Program-in-Package-from-Command-Line.jpg"><img class="size-full wp-image-1987  " title="Compile and run Java Program in Package from Command Line" src="http://JitendraZaa.com/blog/wp-content/uploads/2011/04/Compile-and-run-Java-Program-in-Package-from-Command-Line.jpg" alt="Compile and run Java Program in Package from Command Line" width="481" height="152"></a>
          <figcaption class="wp-caption-text">
           Compile and run Java Program in Package from Command Line
          </figcaption>
         </figure> 
         <p><span style="text-decoration: underline;"><strong>Command Explanation :</strong></span></p> 
         <p><strong>Line 1 : </strong></p> 
         <ul> 
          <li>“javac” is the java compiler available in bin folder of the jdk.</li> 
          <li>“<strong>-d</strong>” stands for the “<strong>directory</strong>“. it explains compiler that where the class files should be created.</li> 
          <li>Last argument is the Complete path, where the java file exists.</li> 
         </ul> 
         <p><strong>Line 2 :</strong></p> 
         <ul> 
          <li>in line 2, you have noted one extra parameter “<strong>-classpath</strong>“. As class “Start” depends on class “Person” and its class file is not in the same directory.&nbsp; Therefore we need to explicitly tell compiler that where it can find required class files.</li> 
          <li>To include more than one classpath use semicolon “;”. &nbsp;Example: &nbsp;<strong>-classpath path1;path2;path3</strong></li> 
         </ul> 
         <p><strong>Line 3 :</strong></p> 
         <ul> 
          <li>It will run the program. her we have to again specify that where all the class files exist with the help of parameter “<strong>-classpath</strong>“</li> 
         </ul> 
         <p><a title="javac supported arguments" href="http://download.oracle.com/javase/1.4.2/docs/tooldocs/solaris/javac.html" target="_blank">The complete set of argument supported by the java compiler can be found on the Oracle site.</a></p>
