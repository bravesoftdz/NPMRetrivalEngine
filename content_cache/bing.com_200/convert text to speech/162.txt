             <div class="answer-body"> 
              <p>you get the "session doesn't exist" probably because you didn't enable cookies in your client. Either enable cookies or submit audio without creating a session by POSTing to <a href="https://stream.watsonplatform.net/speech-to-text-beta/api/v1/recognize.">https://stream.watsonplatform.net/speech-to-text-beta/api/v1/recognize.</a></p> 
              <p>.wav is not supported (not designed for streaming) although you are getting a strange error in response. raw PCM (mimetype: audio/l16;rate=NNN) and flac (mimetype: audio/flac) is supported. It's harder to make mistakes with flac, so I suggest that. In the future we'll add more codecs. Depending on what's in your wav file, you can sometimes send a wave file and call it audio/l16, but I don't recommend it. </p>

             <div class="answer-body"> 
              <p>If you want a pure-java solution, you may try this fine Java wrapper -- <a href="https://github.com/watson-developer-cloud/java-wrapper#speech-to-text">https://github.com/watson-developer-cloud/java-wrapper#speech-to-text</a></p> 
              <p>So your code will look like this</p> 
              <pre><code>     SpeechToText service = new SpeechToText();
     service.setUsernameAndPassword("******", "******");
     File audio = new File("/home/leoks/BM/ws/mp32wav/out56.wav");
     
     //see http://www.ibm.com/smarterplanet/us/en/ibmwatson/developercloud/apis/#!/speech-to-text/recognizeSession
     //for parameter explanation
     
     Map&lt;String, Object&gt; params = new HashMap&lt;String, Object&gt;() ;
     
     params.put(SpeechToText.AUDIO, audio);
     params.put(SpeechToText.CONTENT_TYPE, "audio/wav");
     params.put(SpeechToText.CONTINUOUS, "true");
     params.put(SpeechToText.WORD_CONFIDENCE, "");
     params.put(SpeechToText.MAX_ALTERNATIVES, "2");
     params.put(SpeechToText.CONTINUOUS, "true");
     params.put(SpeechToText.TIMESTAMPS, "true");
     params.put(SpeechToText.INACTIVITY_TIMEOUT, "-1");

     SpeechResults transcript = service.recognize(params );
     
     System.out.println(transcript);

</code></pre> 
              <p>Other useful Java code that you may find somewhat useful</p> 
              <p>mp3 to wav conversion =&gt; <a href="https://mp3transform.googlecode.com/files/mp3-transform-0.81.zip">https://mp3transform.googlecode.com/files/mp3-transform-0.81.zip</a></p> 
              <p>wav splitter =&gt; <a href="https://codeload.github.com/jeroendeswaef/wav-splitter/zip/master">https://codeload.github.com/jeroendeswaef/wav-splitter/zip/master</a></p>

        open: function () {

                searchItemMouseDown = true;
            });
        },
        close: function (e) {
            $searchAuto.removeClass("open");

        appendTo: '#searchResultsAutoComplete',
        autoFocus: false,
        highlight: function (match, keywords) {
            keywords = keywords.split(' ').join('|');
            return match.replace(new RegExp("(" + keywords + ")", "gi"), '<b>$1</b>');
        },
        source: function (request, response) {
            var query = request.term;
            var labelText = $('<div/>').text(request.term).html();

            /*if (!/\s/.test(query)) {
                query += "*";
            }*/

            $.ajax({
                url: '/answers/search.json',
                dataType: 'json',
                minLength: 0,
                delay: 30,

                        if (item != null) {
                            if (item.username){
                                var name = item.username;
                                if(item.realname){
                                	name = item.realname;
                                }

                                        plug: item.plug,
                                        type: "Topic"
                                    };
                                case 'user':
                                    var name = item.username;
                                    if(item.realname){
                                        name = item.realname;
                                    }
                                    return {

                                        label: name.replace(
                                                new RegExp(
                                                        "(?![^&;]+;)(?!<[^<>]*)(" +
                                                                $.ui.autocomplete.escapeRegex(request.term) +

                                        id: item.id,
                                        plug: item.plug,
                                        type: "User",
                                        emailHash: item.emailHash,
                                        userid: item.username
                                    };
                                case 'question':
                                    return {
                                        label: item.title.replace(
                                                new RegExp(
                                                        "(?![^&;]+;)(?!<[^<>]*)(" +
                                                                $.ui.autocomplete.escapeRegex(request.term) +
                                                                ")(?![^<>]*>)(?![^&;]+;)", "gi"

                                        type: item.type
                                    };
                                default:
                                    if (item.title) {
	                                    return {

	                                                new RegExp(

			if (this.hostname && this.hostname !== location.hostname) {
				e.preventDefault();
	
				// setTimeout callback is called in the window scope, so cache the url now
				var url = this.href;
	
				setTimeout(function() {
					document.location = url;
				}, 200);
			}
	
			typeof ga !== 'undefined' && ga('send', 'event', {
				'eventCategory' : $el.attr('data-analytics-category').toTitleCaseString() || 'Forum',
				'eventAction' : action || 'action',
				'eventLabel' : 'Default'
			});
			
			// JTA attempt piwik event tracking
			typeof _paq !== 'undefined' && _paq.push(['trackEvent',$el.attr('data-analytics-category').toTitleCaseString() || 'Forum',action || 'action','Default']);   
			
		});
	});

    // capitalize the first letter of a string
	String.prototype.toTitleCaseString = function() {
	    return this.replace(/^\w/, function(txt){return txt.toUpperCase()});
	};

    function browserTester(browserString) {
		return navigator.userAgent.toLowerCase().indexOf(browserString) > -1;
	}
	
	// grab the param value from url
	function getURLParameter(sParam) {
		var sPageURL = window.location.search.substring(1);
		var sURLVariables = sPageURL.split('&');
		for (var i = 0; i < sURLVariables.length; i++) {
			var sParameterName = sURLVariables[i].split('=');
			if (sParameterName[0] == sParam) {
				return sParameterName[1];
			}
		}
	
		return '';
	}

	}
	
	function IsImageOk(img) {
		// source: http://blog.sajithmr.me/javascript-check-an-image-is-loaded-or-not
	    // During the onload event, IE correctly identifies any images that
	    // weren???t downloaded as not complete. Others should too. Gecko-based
	    // browsers act like NS4 in that they report this incorrectly.
	    if (!img.complete) {
	        return false;
	    }
	
	    // However, they do have two very useful properties: naturalWidth and
	    // naturalHeight. These give the true size of the image. If it failed
	    // to load, either of these should be zero.
	
	    if (typeof img.naturalWidth !== "undefined" && img.naturalWidth === 0) {
	        return false;
	    }
	
	    // No other way of checking: assume it???s ok.
	    return true;
	}
	
	function escapeRegExp(string) {
 	   return string.replace(/([.*+?^=!:${}()|\[\]\/\\])/g, "\\$1");
	}
