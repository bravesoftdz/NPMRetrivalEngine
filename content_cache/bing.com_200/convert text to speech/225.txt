           <h1 class="post-title">The Unofficial Google Text-To-Speech API</h1> 
           <p class="post-byline">by <a href="http://viralpatel.net/blogs/author/viral/" title="Posts by Viral Patel" rel="author">Viral Patel</a> · December 15, 2009</p> 
           <div class="clear"></div> 
           <div class="entry"> 
            <div class="entry-inner"> 
             <p><img src="//img.viralpatel.net/2009/12/google-translate-hello-world.gif" alt="google-translate-hello-world" title="google-translate-hello-world" width="146" height="40" class="alignleft size-full wp-image-1929">Do you want to create your own Text-To-Speech convertor? Well, you may want to directly use Google “Unofficial” API to convert Text to Speech.</p> 
             <p>Google Translate had <a rel="nofollow" target="_blank" href="http://googleblog.blogspot.com/2009/11/new-look-for-google-translate.html">announced</a> few weeks back some new enhancements. Where a new feature was added that can convert the text into speech.</p> 
             <!-- Quick Adsense WordPress Plugin: http://quickadsense.com/ --> 
             <div style="float:none;margin:10px 0 10px 0;text-align:center;"> 
              <script async src="//pagead2.googlesyndication.com/pagead/js/adsbygoogle.js"></script> 
              <!-- BLOGS_INBETWEEN_POST --> 
              <ins class="adsbygoogle" style="display:inline-block;width:300px;height:250px" data-ad-client="ca-pub-2652062086099003" data-ad-slot="1602773796"></ins> 
              <script>
(adsbygoogle = window.adsbygoogle || []).push({});
</script> 
             </div> 
             <p>Now the hack is, you can use the same feature to convert your own text into speech and generate a mp3 out of it. All you have to do it to construct a URL and hit it from browser. For example, if I want to convert text “Hello World” in speech, I will write following url in browser.</p> 
             <pre><code class="language-markup"><script type="prism-html-markup">http://translate.google.com/translate_tts?q=Hello+World
</script></code></pre> 
             <p>And that’s it. This will generate an MP3 file which you can download and play.</p> 
             <p>You may want to create a small HTML form to convert text to speech. All you need is to copy following code and create a HTML file in your machine. And run it using any browser.</p> 
             <pre><code class="language-markup">&lt;form method="get" action="http://translate.google.com/translate_tts"&gt;
	&lt;textarea name="q"&gt;&lt;/textarea&gt;
	&lt;input type="submit" value="Text-To-Speech"&gt;&lt;/input&gt;
&lt;/form&gt;
</code></pre> 
             <p>Currently, This “Unofficial” API is limited to 100 characters only. Also I am not till when google will support this API. </p> 
             <p>Source: <a target="_blank" rel="nofollow" href="http://ajaxian.com/archives/text-to-speech-via-html5-audio">Ajaxian</a></p>
