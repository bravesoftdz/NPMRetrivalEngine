       <p>We can make the computer speak with Python.</p>
       <p>Given a text string, it will speak the written words in the English language.<br> This process is called <u>Text To Speech (TTS)</u>.</p>
       <p>You may like: <a href="https://click.linksynergy.com/link?id=oCUR7eOwwME&amp;offerid=323058.1151632&amp;type=2&amp;murl=https%3A%2F%2Fwww.udemy.com%2Fdeeplearning%2F">Deep Learning A-Z: Hands-On Artificial Neural Networks</a><img border="0" width="1" height="1" src="https://ad.linksynergy.com/fs-bin/show?id=oCUR7eOwwME&amp;bids=323058.1151632&amp;type=2&amp;subid=0"></p>
       <h2>Text to speech</h2>
       <h3>Pyttsx text to speech</h3>
       <p><u>Pytsx</u> is a cross-platform text-to-speech wrapper.<br> It uses different <u>speech engines based on your operating system</u>:</p>
       <div class="note">
        <ul>
         <li><strong>nsss</strong> – NSSpeechSynthesizer on Mac OS X 10.5 and higher</li>
         <li><strong>sapi5</strong> – SAPI5 on Windows XP, Windows Vista, and (untested) Windows 7</li>
         <li><strong>espeak</strong> – eSpeak on any distro / platform that can host the shared library (e.g., Ubuntu / Fedora Linux)</li>
        </ul>
       </div>
       <p>&nbsp;<br> Install with pip (using pyenv, pipenv or virtualenv):</p>
       <div class="kterm" style="color:white; padding: 8px;">
         sudo pip install pyttsx
       </div>
       <p>Then run the example code:</p>

       <h3>gTTS text to speech</h3>
       <p>gTTS is a module and command line utility to save spoken text to mp3.<br> It uses the <u>Google Text to Speech (TTS) API</u>.</p>
       <p>Listen to the voice sample below:<br> <br> <iframe width="560" height="315" src="https://www.youtube.com/embed/ahZ-5aRkDgU" frameborder="0" allowfullscreen></iframe></p>
       <p>This module supports many languages and sounds very natural.</p>
       <p><strong>Install</strong><br> Install with the python package tool (pip):</p>
       <div class="kterm" style="color:white; padding: 8px;">
         sudo pip install gTTS
       </div>

       <div class="kterm" style="color:white; padding:8px;">
         pip install tts-watson
