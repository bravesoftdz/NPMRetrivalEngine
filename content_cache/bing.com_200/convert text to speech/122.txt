  <h1>webkit Speech Input test</h1> 
  <p>Speech input has been implemented in Chrome 11. Click the input box's microphone icon and start speaking!</p> 
  <p class="nosupport">Your browser does not support webkit speech so a standard input box will be shown.</p> 
  <form action="index.html" method="get"> 
   <div> 
    <label for="speech">Speech test:</label> 
    <input type="text" x-webkit-speech="x-webkit-speech"> 
   </div> 
  </form> 
  <p>Speech support can be implemented with the following code:</p> 
  <pre><code>&lt;input type="text" x-webkit-speech /&gt;</code></pre> 
  <p>The x-webkit-speech attribute can be used on any HTML5 input with the type text, number, tel, or search.</p> 
  <p>This example code was developed by <a href="http://twitter.com/craigbuckler">Craig Buckler</a> of <a href="http://optimalworks.net/">OptimalWorks.net</a> for <a href="http://sitepoint.com/">SitePoint.com</a>.</p>
