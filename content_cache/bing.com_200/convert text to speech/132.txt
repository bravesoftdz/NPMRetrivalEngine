           It´s quite interesting question. I´d suggest you to do in java using voce library. Below is the url you can download the jar file and add it in your java project all the jars located in lib folder in voce.zip file. In this zip file you´ll find some samples also for text to speech and speech to text. It´s quite easy by using this library. I have created one text to speech example in eclipse: 
           <br>
           <br>To Download Voce Jar visit here: https://sourceforge.net/projects/voce/ 
           <br> 
           <br>TextToSpeech.java: 
           <br>----------------------------------------------------- 
           <br>package in.anyforum; 
           <br> 
           <br>import voce.SpeechSynthesizer; 
           <br> 
           <br>public class TextToSpeech { 
           <br>public static void main(String[] args) { 
           <br> SpeechSynthesizer speechSynthesizer=new SpeechSynthesizer("Any name"); 
           <br> speechSynthesizer.synthesize("MOHAMMAD SALEEM BASHA"); 
           <br>} 
           <br>} 
           <br> 
           <br> 
           <br>for speech to text, you need to give one GRAM file and one configuration file. In GRAM file you mention what words it must listen and convert it into text. Configuration file you can use the same one which is existing in voce.zip file. In samples folder under voce.zip file there is one java file recognitionTest. Try to run this file, It´s working fine. 
           <br> 
           <br>Note: 
           <br>---------------------------- 
           <br>JDK version should be at least 1.7 .. recognitionTest.java file will run only if you run the same on console, If you try to run in eclipse it´ll not run, because of some compression issue in IDE. Just paste all the jars located in lib folder of voce.zip at below location: 
           <br> 
           <br>C:\Program Files (x86)\Java\jdk1.7.0\jre\lib\ext 
           <br> 
           <br>and point the location on command prompt as below, there is .class file also there just run and test it 
           <br> 
           <br>F:\java eclipse\voce-0.9.1\samples\recognitionTest\java&gt;java recognitionTest 
           <br> 
           <br>For me Output is like this 
           <br> 
           <br>[Voce] Initializing recognizer. This may take some time... 
           <br>[Voce] Initialization complete 
           <br>This is a speech recognition test. Speak digits from 0-9 into the microphone. Speak ´quit´ to quit. 
           <br>You said: five 
           <br>You said: oh one 
           <br>You said: one eight nine 
           <br>You said: eight oh eight nine 
           <br>You said: quit 
           <br> 
           <br># --------------- Summary statistics --------- 
           <br> Total Time Audio: 10.19s Proc: 10.69s Speed: 1.05 X real time 
           <br>[Voce] Shutdown complete 
           <br>
