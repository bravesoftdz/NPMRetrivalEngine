    <section class="primary island c"> 
     <h2>About this Add-on</h2> 
     <div class="prose"> 
      <div id="addon-description" class="prose">
       <a rel="nofollow" href="https://outgoing.prod.mozaws.net/v1/c8070eb7336a769f5e28df1e784890e224a2346566ed8dd4a38b07f0ef7368fb/http%3A//mybrowseraddon.com/text-to-speech.html">Text-to-Speech (Google™ &amp; HTML5)</a> is a text to speech extension for Firefox with natural sounding voices by using Google and HTML5 APIs.
       <br>
       <br>Some Features: 
       <br>1. Easy one-click text-to-speech via Google or HTML5 API
       <br>2. Auto detects language (no need to set input language each time)
       <br>3. Text-to-speech is enabled by holding "Alt", "T" or "Insert" key
       <br>4. Highlights selected text while text-to-speech is running.
       <br>5. Supports both Google and HTML5 TTS APIs (please see the options page for more details)
       <br>
       <br>
       <b>Note 1</b>: In order for the highlight feature to work correctly, please select text paragraph-by-paragraph. However, highlight feature is still in Beta and may not work properly for all web-pages.
       <br>
       <br>
       <b>Note 2</b>: Google TTS API has a limit; therefore after certain number of tries you may encounter the limit error. When you first install the Addon, the default active TTS API is HTML5. You can change it to Google from the options page.
       <br>
       <br>
       <b>Note 3</b>: HTML5 TTS API has no limits; however it is still in Beta and may encounters crashes or errors. In order to enable HTML5 TTS API in Firefox please go to "about:config" page and search for "webspeech" keyword. Once you find "media.webspeech.synth.enables" item, right-click on it and click "Toggle" to enable it.
       <br>
       <br>In general, 
       <b>Google TTS</b> is more powerful and supports more languages comparing to 
       <b>HTML5 TTS</b>. Once you switch from Google to HTML5 or vise versa, please restart Firefox for changes to make effect. In future versions more options will be added to both APIs.
       <br>
       <br>
       <b>Note 4</b>: In order to report Bugs, please visit Addon's 
       <a rel="nofollow" href="https://outgoing.prod.mozaws.net/v1/c8070eb7336a769f5e28df1e784890e224a2346566ed8dd4a38b07f0ef7368fb/http%3A//mybrowseraddon.com/text-to-speech.html">Homepage</a> and fill the bug report form.

    <h2>Privacy Policy</h2> 
    <div class="prose">
      Text-to-Speech (Google™ &amp; HTML5) does not collect any personal identifiable data about its users.
     <br>
     <br>However, when using Google TTS API, the addon sends selected text to Google TTS API to get the speech. If you use HTML5 TTS API instead, the addon will use the native API in your browser and does not need to send selected text to a remote server. Please note, the default TTS API is HTML5.
