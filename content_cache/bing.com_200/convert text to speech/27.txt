      <h1 id="text-to-speech">Text to Speech</h1> 
      <p>Text to Speech is an easy method to add dynamic voice to your phone capabilities. Unlike pre-recorded audio, this feature can be used to deliver spoken versions of text such as live information (e.g., sports scores) and other dynamic messages (e.g., emergency updates). Examples of simple changes in the code can allow you to change the message, the voice (e.g., male or female), who will be receiving your message, and much more.</p> 
      <p>This tutorial will use Plivo’s <a href="/docs/xml/speak">Speak</a> element to read out text as speech to the caller. More specifically, here is what will happen behind the scenes:</p> 
      <ol class="numbered"> 
       <li>A call is made to the Plivo phone number assigned to your text to speech application</li> 
       <li>When the call is connected, the caller will hear the message read out. To demonstrate the capabilities of the <a href="/docs/xml/speak">Speak</a> element, we’ll have Plivo read the message in 3 different languages (namely English, French, and Russian).</li> 
       <li>After the message is read, the call is automatically hung up.</li> 
      </ol> 
      <h2 id="prerequisites">Prerequisites</h2> 
      <ol class="numbered"> 
       <li>Sign up for a free <a href="https://manage.plivo.com/accounts/register/">Plivo trial account</a>.</li> 
       <li>Check out our <a href="/docs/helpers/">Helper Libraries</a> page and install the right helper based on the programming language you want to use.</li> 
       <li><a href="https://manage.plivo.com/number/search/">Buy a Plivo phone number</a>. A phone number is required to receive calls. You can buy a Plivo phone number in over 20+ countries through the <a href="https://manage.plivo.com/number/search/">Buy Numbers</a> tab on your Plivo account UI. Check the <a href="/voice-api/coverage">Voice API coverage</a> page for all the supported countries.</li> 
       <li>Use a web hosting service to host your web application. There are many inexpensive cloud hosting providers that you can use for just a few dollars a month. Follow the instructions of your hosting provider to host your web application.</li> 
      </ol> 
      <h2 id="set-up-a-web-server">Set up a Web Server</h2> 
      <p>Let’s assume your web server is located at <code>myvoiceapp.com</code>. Below is a snippet to set up a route on your webserver. Lets call it, <code>text-to-speech</code>. Now when we send an <code>HTTP</code> request to <code>myvoiceapp.com/text-to-speech</code> this route will be invoked. You will now have to configure this URL in your Plivo application.</p> 
      <div class="doc-note">
        Note: For PHP, the route will be myvoiceapp.com/text-to-speech.php 
      </div> 
      <ol class="numbered"> 
       <li>Copy the relevant code below into a text file and save it. Lets call it, <code>text-to-speech</code>. 
        <div class="doc-note">
          Note: Make sure to use the appropriate file extention for your code (e.g. text-to-speech.py for Python). 
        </div> </li> 
       <li>Now that you have the code, you will need to expose your server to the public Internet. This way, Plivo will know where to find your app when a particular phone number is dialed. Moving forward, we will assume that your app is available at myvoiceapp.com.</li> 
      </ol>

                <span class="n">res</span><span class="p">.</span><span class="n">ContentType</span> <span class="p">=</span> <span class="s">"text/xml"</span><span class="p">;</span>
                <span class="k">return</span> <span class="n">res</span><span class="p">;</span>

      <h2 id="create-an-application">Create an Application</h2> 
      <ol class="numbered"> 
       <li>Create an Application by visiting the <a href="https://manage.plivo.com/app/">Application Page</a> and click on <code>New Application</code> or by using Plivo’s <a href="/docs/api/application/#create-an-application">Application API</a>.</li> 
       <li>Give your application a name. Lets call it <code>Text to Speech</code>. Enter your server URL (e.g., http://myvoiceapp.com/text-to-speech) in the <code>Answer URL</code> field and set the method as <code>POST or GET</code>. See our <a href="/docs/api/application/#modify-an-application">Application API</a> docs to learn how to modify your application through our APIs.</li> 
       <li>Click on <code>Create</code> to save your application.</li> 
      </ol> 
      <p><img src="https://s3.amazonaws.com/plivo_blog_uploads/static_assets/images/docs/voice/Plivo_Create_Application-tts.jpg" alt="Create Text to Speech Application" style="border: 2px solid #707070;"></p> 
      <h2 id="assign-a-plivo-number-to-your-app">Assign a Plivo number to your app</h2> 
      <ol class="numbered"> 
       <li>Navigate to the <a href="https://manage.plivo.com/number">Numbers</a> page and select the phone number you want to use for this app.</li> 
       <li>Select <code>Text to Speech</code> (name of the app) from the Plivo App dropdown list.</li> 
       <li>Click on <code>Update</code> to save.</li> 
      </ol> 
      <p><img src="https://s3.amazonaws.com/plivo_blog_uploads/static_assets/images/docs/voice/Plivo_Number_Association-tts.jpg" alt="Create Text to Speech Application" style="border: 2px solid #707070;"></p> 
      <p>If you don’t have a Plivo phone number, go to the <a href="https://manage.plivo.com/number/search/">Buy Number</a> page to purchase a Plivo phone number.</p> 
      <p><img src="https://s3.amazonaws.com/plivo_blog_uploads/static_assets/images/blog/SMS-Search-Numbers.png" alt="Create Text to Speech Application" style="border: 2px solid #707070;"></p> 
      <h2 id="test-it-out">Test it out</h2> 
      <p>Now, make a call to the Plivo number that you associated with your application. Plivo will detect the call and fetch the answer URL, which resides at myvoiceapp.com/text-to-speech. Then, Plivo will execute the XML instruction from your URL and read the message out in 3 different languages. Below is the XML output.</p>

    <span class="nt">&lt;/Speak&gt;</span>
