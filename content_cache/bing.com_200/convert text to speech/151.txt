        <div id="block-1" class="block"> 
         <div class="proseblock"> 
          <p>The pyttsx library is a cross-platform wrapper that supports the native text-to-speech libraries of Windows and Linux at least, using SAPI5 on Windows and eSpeak on Linux.</p> 
          <p>Using it can reduce the amount of OS-specific code you need to write for the task of speech synthesis from your programs.</p> 
          <p>More details in this blog post:</p> 
          <p><a href="http://jugad2.blogspot.in/2014/03/speech-synthesis-in-python-with-pyttsx.html" rel="nofollow">http://jugad2.blogspot.in/2014/03/speech-synthesis-in-python-with-pyttsx.html</a></p>
