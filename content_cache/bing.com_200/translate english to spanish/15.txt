              <p>Use Google translation API. Easy to use. The following translates Spanish to English. To translate from and to other languages, simply change <code>'es'</code> and <code>'en'</code></p> 
              <p><code>&lt;div id="content"&gt;&lt;/div&gt;</code></p> 
              <pre><code>google.load("language", "1");

function initialize() {
    var content = document.getElementById('content');
    content.innerHTML = '&lt;div id="text"&gt;Hola, me alegro mucho de verte.&lt;\/div&gt;&lt;div id="translation"/&gt;';
    var text = document.getElementById("text").innerHTML;
    google.language.translate(text, 'es', 'en', function(result) {
        var translated = document.getElementById("translation");
        if (result.translation) {
            translated.innerHTML = result.translation;
        }
    });
}
google.setOnLoadCallback(initialize);
</code></pre> 
              <h2>Check working example at <a href="http://jsfiddle.net/wJ2QP/1/" rel="nofollow">http://jsfiddle.net/wJ2QP/1/</a></h2>

             <div class="post-text" itemprop="text"> 
              <p>Use this JQuery plugin <a href="http://www.openxrest.com/translatejs" rel="noreferrer">http://www.openxrest.com/translatejs</a></p> 
              <p>1 - Include the "trn" class to the text you want to translate:</p> 
              <pre><code>&lt;span class="trn"&gt;text to translate&lt;/span&gt;
</code></pre> 
              <p>2 - Define a dictionary:</p> 
              <pre><code>var dict = {
  "text to translate": {
    pt: "texto para traduzir"
  },
  "Download plugin": {
    pt: "Descarregar plugin",
    en: "Download plugin"
  }
}
</code></pre> 
              <p>3 - Translate the entire page body:</p> 
              <pre><code>var translator = $('body').translate({lang: "en", t: dict}); //use English
</code></pre> 
              <p>4 - Change to another language:</p> 
              <pre><code>translator.lang("pt"); //change to Portuguese
</code></pre>

             <div class="post-text" itemprop="text"> 
              <p>You can use Google Translate's Javascript API.</p> 
              <pre><code>&lt;p id="some"&gt;Hello&lt;/p&gt;
&lt;input id="trans" value="Translate" type="button"&gt;

&lt;script&gt;
   $('#trans').click(function() {
     google.language.translate($('#some').html(), 'en', 'fr', function(result) {
         $('#some').html(result.translation);
     });
   });
&lt;/script&gt;
</code></pre> 
              <p>You will need to load the js library in your page's HEAD section in order to use the API.</p>

             <div class="post-text" itemprop="text"> 
              <p>On this PHP/JS solution you should <strong>use include PHP language files</strong> and <strong>set language on session/cookie</strong> not on $_GET. For the sake of <strong>simplicity</strong> I will do it on the </p> 
              <p>index.php file</p> 
              <pre><code>&lt;?php
$lang = $_GET['lang'];

if ($lang == 'fr'){
    $w = array(
        'Trouvé',
        ' non trouvé.',
        'Erreur. Veuillez réessayer.'
    );  
}else if($lang == 'en'){
    $w = array(
        'Found',
        ' not found.',
        'Error. Please try again.'
    );  
}else{
    $w = array(
        'Trouvé',
        ' non trouvé.',
        'Erreur. Veuillez réessayer.'
    );  
}
?&gt;
&lt;!DOCTYPE html&gt;
&lt;html lang="en"&gt;

&lt;head&gt;
....................
&lt;body&gt;
....................
    &lt;script type="text/javascript"&gt;
        /*  Translate JS
            Declare JS variables for translation in PHP file as below (Global vars outside $(document).ready). 
            Inside JS file call the variable as $.lang_mynamespace.var_name
        */
        $.lang_scan = { 
            found_js:"&lt;?=$w[0];?&gt;",
            not_found_js:"&lt;?=$w[1];?&gt;",
            error_js:"&lt;?=$w[2];?&gt;"
        };  
    &lt;/script&gt;
&lt;/body&gt;
&lt;/html&gt;
</code></pre> 
              <p>JS file</p> 
              <pre><code>$(function() {
    $("#scan_result").on('change', function(){

        //check number
        $.ajax({
            url: "check.php",
            dataType: "json",
            type: "post",
            data: {'scan_no': scan_value} ,
            success: function (response) {
                if (response.status == true){
                    alert("Scan no. " + response.scan_no + $.lang_scan.found_js);
                }else{
                    alert("Scan no. " + response.scan_no + $.lang_scan.not_found_js);                      
                }
            },
            error: function(jqXHR, textStatus, errorThrown) {
               //ajax error 
               alert($.lang_scan.error_js);            
            }
        }); 

    }); 
});
</code></pre> 
              <p>check.php return a json</p> 
              <pre><code>{"scan_no": "123", "status": true/false}
</code></pre>
