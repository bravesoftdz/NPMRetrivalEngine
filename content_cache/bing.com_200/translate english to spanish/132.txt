      <h2><span class="mw-headline" id="Summary">Summary</span></h2> 
      <p>Offer Google translation services for any page in the Wiki. Allow a hover of the cursor over a word (or of selected text) to translate that text in a popup (like the Google toolbar translation feature). A tab (or link) can turn this feature on or off, and also allow switching of languages.</p> 
      <p>The code put up by this proposal has recently become a Gadget on enwiki. An earlier version of this software has run on the ptwiki for 11 months now and receives about 400 hits a day.</p> 
      <h2><span class="mw-headline" id="Proposal">Proposal</span></h2> 
      <p>Many intermediate English language users of the English Wikipedia need help with individual words in an English Wikipedia article. Have some javascript determine the word (or selected text) the cursor is over and popup a google translation of that word or selected text.</p> 
      <p>This javascript will thus allow 41 languages to be translated into 41 other languages. It will allow translation of selected text (up to 500 characters). It is thus a generalization and extension of the Google toolbar translation feature which currently only translates single words from English to 10 other languages.</p> 
      <p>My monobook.js file already has javascript that does this in the English Wikipedia. Earlier code that did the same thing has been a Wiki gadget in the Spanish and Portuguese wikipedias for over 6 months now.</p> 
      <p>You can include this code within your monobook.js file by including:</p> 
      <p>importScript('User:Endo999/GoogleTrans.js');</p> 
      <p>// remember to clear the cache after adding this code to your monobook.js file</p> 
      <p>and try it out. A tab at the top of the window allows the feature to be turned on or off and allows changing of languages. See my user (Endo999) page for more info.</p> 
      <p>I use the feature to translate French and Spanish, of which I am only an intermediate user. It's much quicker than looking up the dictionary by hand.</p> 
      <h2><span class="mw-headline" id="Motivation">Motivation</span></h2> 
      <p>Allow intermediate English learners to more easily read enwiki articles.</p> 
      <p>Allow English language speakers easier access to languages like Portuguese, of which they will only have a modest ability in (such as if they know some Spanish, but not Portuguese).</p> 
      <p>Allow cross readership of the wikis.</p> 
      <h2><span class="mw-headline" id="Key_Questions">Key Questions</span></h2> 
      <ul> 
       <li>Is the machine translation system implemented as free software&nbsp;?</li> 
      </ul> 
      <p>GoogleTrans currently uses Google translation services. This is just a software call. The javascript file can easily be changed to call any webbased translation service. The call to a translation service is a very small part of the code for the gadget in question and can be changed easily.</p> 
      <h2><span class="mw-headline" id="Potential_Costs">Potential Costs</span></h2> 
      <p>The code is already a gadget on the enwiki under the name GoogleTrans.</p> 
      <h2><span class="mw-headline" id="Comparisions_vs_Other_Translation_Software">Comparisions vs Other Translation Software</span></h2> 
      <p>A website based Javascript file, this software has all the versatility and speed of its 3 main competitors:</p> 
      <dl> 
       <dd>
        1) Internet Explorer v8 has a selection text based translation service. 
        <dl> 
         <dd>
          Advantages: 
          <dl> 
           <dd>
            No need to install any javascript file on any website
           </dd> 
          </dl> 
         </dd> 
         <dd>
          Disadvantages: 
          <dl> 
           <dd>
            Slower than GoogleTrans. Need to 1) select text, 2) click on special selection icon that appears, 3) then click on menu item to translate.
           </dd> 
           <dd>
            Is slow to translate single words. Cannot just position cursor over word and get translation like Google toolbar translation feature or GoogleTrans. Many intermediate learners of a language only need single words translated and do not need whole sentences translated.
           </dd> 
           <dd>
            Only works on one version of IE.
           </dd> 
          </dl> 
         </dd> 
        </dl> 
       </dd> 
      </dl> 
      <dl> 
       <dd>
        2) Google Toolbar Translation Feature 
        <dl> 
         <dd>
          Advantages: 
          <dl> 
           <dd>
            Quick to use, simply position cursor over word and get multiple translations for that word
           </dd> 
           <dd>
            No need to install javascript file on any website
           </dd> 
          </dl> 
         </dd> 
         <dd>
          Disadvantages 
          <dl> 
           <dd>
            Even the new version only translates from English to 21 other languages, while GoogleTrans will do 51.
           </dd> 
           <dd>
            Does not do selection based translation, which GoogleTrans does and IE8 does.
           </dd> 
           <dd>
            Only works on IE, and Firefox.
           </dd> 
           <dd>
            Need to have administrative privs on the PC you are using the toolbar on in order to install it. This is not possible at many schools or companies that have restrictive environments.
           </dd> 
          </dl> 
         </dd> 
        </dl> 
       </dd> 
      </dl> 
      <dl> 
       <dd>
        3) gTranslate -- Firefox addin 
        <dl> 
         <dd>
          Advantages: 
          <dl> 
           <dd>
            no need to install javascript file on website
           </dd> 
           <dd>
            will translate 51 languages to 51 other languages
           </dd> 
          </dl> 
         </dd> 
         <dd>
          Disadvantages: 
          <dl> 
           <dd>
            Not as fast to translate single words and even selected text. More mouse selections and mouse clicks necessary than compared with Google Toolbar or GoogleTrans.
           </dd> 
           <dd>
            Does not seem to do more than a few words of translation (My understanding of the tool may be deficient here). Google can translate up to 500 characters of text at a time but gTranslate does not seem to be able to show 500 characters of translation.
           </dd> 
           <dd>
            Only works on Firefox.
           </dd> 
          </dl> 
         </dd> 
        </dl> 
       </dd> 
      </dl> 
      <dl> 
       <dd>
        4) GoogleTrans 
        <dl> 
         <dd>
          Advantages: 
          <dl> 
           <dd>
            Quick to translate single words and/or selected text. Need only to position cursor over word or selected text and hit shift key.
           </dd> 
           <dd>
            Does all translations that Google can do. More than Google toolbar can do.
           </dd> 
           <dd>
            Supports all browsers except Konqueror.
           </dd> 
           <dd>
            Allows translation popups to happen on pcs where the user cannot install toolbars or addons (like schools and businesses that have restrictive environments).
           </dd> 
          </dl> 
         </dd> 
         <dd>
          Disadvantages: 
          <dl> 
           <dd>
            Need to install 66kb javascript file as part of webpage or website.
           </dd> 
          </dl> 
         </dd> 
        </dl> 
       </dd> 
      </dl>
