             <div class="post-text" itemprop="text"> 
              <p>Plese take a look at <a href="http://recurser.com/articles/2008/02/21/jquery-i18n-translation-plugin/" rel="noreferrer">jQuery-i18n</a> translation plugin</p> 
              <pre><code>var my_dictionary = { 
    "some text"  : "a translation",
    "some more text"  : "another translation"
}
$.i18n.load(my_dictionary);


$('div#example').text($.i18n._('some text'));
</code></pre> 
              <p>And <a href="http://keith-wood.name/localisation.html" rel="noreferrer">jQuery localisation</a></p> 
              <pre><code>$.localise('js/greeting');
$('#greeting').val(greeting);
$('#languages').val($.localise.defaultLanguage);
$('#changeLocale').change(function() {
    var newLang = $(this).val();
    $.localise('js/greeting', {language: newLang, loadBase: true});
    $('#greeting').val(greeting);
    $('#languages').val(newLang);
});
</code></pre> 
              <p>Or, If you really want more translation than looking up data, try <a href="http://code.google.com/apis/ajaxlanguage/documentation/#Examples" rel="noreferrer">google translate api</a></p> 
              <pre><code>google.language.translate("Hello world", "en", "es", function(result) {
  if (!result.error) {
    var container = document.getElementById("translation");
    container.innerHTML = result.translation;
  }
});
</code></pre>

             <div class="post-text" itemprop="text"> 
              <p>Use this JQuery plugin <a href="http://www.openxrest.com/translatejs" rel="nofollow">http://www.openxrest.com/translatejs</a></p> 
              <p>1 - Include the "trn" class to the text you want to translate:</p> 
              <pre><code>&lt;span class="trn"&gt;text to translate&lt;/span&gt;
</code></pre> 
              <p>2 - Define a dictionary:</p> 
              <pre><code>var dict = {
  "text to translate": {
    pt: "texto para traduzir"
  },
  "Download plugin": {
    pt: "Descarregar plugin",
    en: "Download plugin"
  }
}
</code></pre> 
              <p>3 - Translate the entire page body:</p> 
              <pre><code>var translator = $('body').translate({lang: "en", t: dict}); //use English
</code></pre> 
              <p>4 - Change to another language:</p> 
              <pre><code>translator.lang("pt"); //change to Portuguese
</code></pre>
