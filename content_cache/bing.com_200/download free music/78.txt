          <blockquote> 
           <p>EPIC system supports several database systems, such as Ingres, mSQL, postgreSQL, and MySQL, to management the EPIC data files. </p> 
           <p>The EPIC/mSQL database management system package contains software that create EPIC/mSQL database, load data, and retreive data from the database. It is available from PMEL's anonymous ftp: <a href="ftp://ftp.pmel.noaa.gov/epic//epic-msql/">ftp://ftp.pmel.noaa.gov/epic//epic-msql/</a> </p> 
           <p><b>The EPIC/MySQL database management system package will be available soon.</b></p>
