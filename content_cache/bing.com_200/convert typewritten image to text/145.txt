        <p><span style="font-size:14pt"><strong>Project Description</strong><span style="font-size:14pt">&nbsp;</span></span></p> 
        <p><span style="font-size:10pt">This is a package of C#, VB.NET Example Project for </span><a href="http://www.e-iceblue.com/Introduce/free-pdf-component.html"><span style="font-size:10pt">Spire.PDF for .NET</span></a><span style="font-size:10pt">.</span></p> 
        <p><span style="font-size:10pt">Spire.PDF for .NET is a professional .NET PDF component which enables you to generate, read, edit and manipulate PDF documents in C#, VB.NET. It can be generally applied in </span><span style="font-size:10pt">server-side (ASP.NET or any other environment) or wi</span><span style="font-size:10pt">th</span><span style="font-size:10pt"> Windows Forms applications</span><span style="font-size:10pt"> without installing Adobe Acrobat or any other external libraries.</span></p> 
        <p><span style="font-size:10pt">Many features are supported in Spire.PDF for .NET. First let us have a first glance of this .NET PDF component. <br> </span></p> 
        <p><span style="font-size:10pt"><img src="http://www.e-iceblue.com/images/Introduce/Overview/PDF.png" alt="PDF functions" width="660" height="571"></span></p> 
        <p><br> <strong><span style="font-size:14pt">First Sight of Spire.PDF for .NET</span></strong></p> 
        <p><strong><span style="font-size:14pt"><img src="http://25.media.tumblr.com/2523624a34ccb54aed0cd9d70a65c887/tumblr_mg16jvx2681rrbavjo1_1280.jpg" alt="First Glance of Spire.PDF for .NET" width="726" height="488"></span></strong></p> 
        <p><strong>1. </strong><strong><span style="font-size:12pt">Generate PDF File</span></strong></p> 
        <p><span style="font-size:10pt">Spire.PDF for .NET enables users to create PDF file from webpage(html, htm, aspx), image(jpg, jpeg, png, bmp, tiff, gif, ico etc), text and export datatable from database to PDF. It also supports to generate PDF/A-1b, PDF/x1a, 2001 Compliance, standard PDF file. Furthermore, along with Spire.Doc for .NET and Spire.XLS or .NET, it can convert Word doc/docx, Excel xls/xlsx, Image, XPS, HTML, RTF and XML to PDF. At the same time, it also support converting PDF to Image, word and XPS. <br> </span></p> 
        <p><strong>2. </strong><strong><span style="font-size:12pt">Read PDF File</span></strong></p> 
        <p><span style="font-size:10pt">Spire.PDF for .NET allows you to extract PDF text both in plain format and special format such as it can read PDF text which is written from right to left, such as Herbrew. Images of different formats and PDF attachments also can be extracted. Here shows an image of reading PDF.</span></p> 
        <p>&nbsp;</p> 
        <p><strong>3. </strong><strong><span style="font-size:12pt">Edit PDF File</span></strong></p> 
        <p><span style="font-size:10pt">This PDF component has rich features in editing PDF file. It supports to set text format and edit PDF page in multiple ways. Here is a simple list:</span></p> 
        <ul> 
         <li><span style="font-size:10pt">Text: draw text and set text format</span> </li>
         <li><span style="font-size:10pt">Image: draw PDF images and different kinds of shapes</span> </li>
         <li><span style="font-size:10pt">Table: draw PDF table including simple table, nested table and image table,</span> </li>
         <li><span style="font-size:10pt">Watermark: add both text watermark and image watermark</span> </li>
         <li><span style="font-size:10pt">Bookmark</span> </li>
         <li><span style="font-size:10pt">Header and footer</span> </li>
         <li><span style="font-size:10pt">Attachment: add attachments, read attachments and remove attachments</span> </li>
         <li><span style="font-size:10pt">Hyperlink: link and anchor link</span> </li>
         <li><span style="font-size:10pt">List: Simple list and multi-level list</span> </li>
         <li><span style="font-size:10pt">Booklet</span> </li>
         <li><span style="font-size:10pt">Action: action and action chain</span> </li>
         <li>Automatic Field </li>
        </ul> 
        <p><strong>4. </strong><strong><span style="font-size:12pt">Manipulate PDF File </span> </strong></p> 
        <p><span style="font-size:10pt">There a lot of PDF tasks can be operated by using this .NET PDF library. You can merge PDF files into one PDF as well as split a huge PDF with the given number range accordingly. Furthermore, it can protect your PDF document by encryption with both owner password and user password, create PDF digital signature. If you do not need to lock your PDF, you can decrypt it quickly. Besides, you can set PDF property, PDF template and view preference as you like. Finally, webpage, image and text all can be converted to PDF which are already referred above. Form below picture, you can view most features of editing and handling PDF document clearly.</span></p> 
        <h2>5. Spire.PDF has been updated to v3.5</h2> 
        <p>New features of Spire.PDF 3.5:</p> 
        <ul> 
         <li>Add new method to change the highlight color. </li>
         <li>Add new properties to apply the template when create new page. </li>
         <li>Add new method to fill the barcode xfa from field. </li>
         <li>Add new method to get the CMYK image from PDF. </li>
         <li>Add new feature to support varying DPI settings when printing. </li>
         <li>Add mediabox property in page. </li>
        </ul> 
        <p>New features of Spire.PDF 3.4:</p> 
        <ul> 
         <li>Change PDF page size. </li>
         <li>Add form fields to an existing PDF. </li>
         <li>Convert PDF file to word document. (PDF to .doc) </li>
