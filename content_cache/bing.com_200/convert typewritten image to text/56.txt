                     <h4>Core Features</h4> 
                     <ul> 
                      <li> <b>Convert any Url or HTML file to PDF</b>. Rendering any Url or HTML page the same as your browser does, but output as PDF; </li> 
                      <li> <b>Zero External Dependency</b>. Does not depend on any Web browser on your system. Will always work regardless you install, uninstall or update your browsers; </li> 
                      <li> <b>Built-in Javascript Engine</b>. All Javascript code runs exactly the same way as they would in a browser. You can also turn Javascript off; </li> 
                      <li> <b>Automatic and Manual Paging</b>. The converter can automatically page the contents into multiple pages completely based on page contents or "hard page break" styles such as "page-break-before" and "page-break-after" in your HTML code; </li> 
                      <li> <b>Scale to Fit</b>. Automatically scaling the HTML page to fit in PDF paper size, or use a fixed scaling factor (or not scaling at all); </li> 
                      <li> <b>Selectable Text and clickable Links</b>. The generated PDF file is not a "screenshot" type image. All texts in your HTML file are rendered as selectable text in PDF. All links in your HTML file are preserved; </li> 
                      <li> <b>HTTP Post and Authentication</b>. Need to post a page or authenticate with the server? Just provide the data you wish to post or user name and password needed to login the server; </li> 
                      <li> <b>Multiple output options</b>. Output to a file? Output to a stream? Output to Web client? Output as a download file? Choose one that fits your need. </li>
