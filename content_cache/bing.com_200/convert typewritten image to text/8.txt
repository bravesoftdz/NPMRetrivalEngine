        <h1>How to write an image to file – Java ImageIO</h1> 
        <div class="post-meta"> 
         <p class="post-meta">By <a href="https://www.mkyong.com/author/mkyong/" title="mkyong" rel="author">mkyong</a> | <time datetime="2010-01-17T03:40:29+00:00">January 17, 2010</time> | Updated : August 30, 2012 | Viewed : 174,752 times <span>+397 pv/w</span></p> 
        </div> 
        <!-- /.post-meta --> 
        <div class="post-content"> 
         <p>The “<strong>javax.imageio.ImageIO</strong>” is a handy class to read and write image into local system. In this example, we show you how to use “<strong>ImageIO</strong>” to read an image from an URL and write it into three file formats : </p> 
         <ol> 
          <li>“jpg”</li> 
          <li>“gif”</li> 
          <li>“png”</li> 
         </ol> 
         <p><em>Full example to demonstrate the ImageIO usage.</em></p> 
         <pre><code class="language-java">
package com.mkyong.image;

import java.awt.image.BufferedImage;
import java.io.File;
import java.io.IOException;
import java.net.URL;
import javax.imageio.ImageIO;

public class WriteImage 
{	
    public static void main( String[] args )
    {
    	BufferedImage image = null;
        try {
          
            URL url = new URL("http://www.mkyong.com/image/mypic.jpg");
            image = ImageIO.read(url);
            
            ImageIO.write(image, "jpg",new File("C:\\out.jpg"));
            ImageIO.write(image, "gif",new File("C:\\out.gif"));
            ImageIO.write(image, "png",new File("C:\\out.png"));
            
        } catch (IOException e) {
        	e.printStackTrace();
        }
        System.out.println("Done");
    }
}
</code></pre>
