        <b>Project Description</b>
        <br> Using ReCaptcha with Asp.Net MVC
        <br> 
        <br> MvcReCaptcha makes it easier for you to implement the proven ReCaptcha technology. It's developed in C#.
        <br> 
        <br> Based on solution provided by Dimecasts.net.
        <br> 
        <br> 
        <h1>Using ReCaptcha with Asp.Net MVC</h1> It is now extremely easy to setup ReCaptcha on your Asp.Net MVC Website.
        <br> 
        <br> Signup for reCaptcha, 
        <a href="http://recaptcha.net/whyrecaptcha.html">http://recaptcha.net/whyrecaptcha.html</a>
        <br> 
        <br> How to Use:
        <br> 
        <br> 
        <b>Step 1: Add your Public and Private key to your web.config file in appsettings section</b>
        <br> 
        <br> 
        <div style="color:Black; background-color:White"> 
         <pre>
&lt;appSettings&gt;
	&lt;add key=<span style="color:#A31515">"ReCaptchaPrivateKey"</span> value=<span style="color:#A31515">" -- PRIVATE_KEY -- "</span> /&gt;
	&lt;add key=<span style="color:#A31515">"ReCaptchaPublicKey"</span> value=<span style="color:#A31515">" -- PUBLIC KEY -- "</span> /&gt;
&lt;/appSettings&gt;			
</pre> 
        </div> 
        <br> 
        <b>Step 2: Add new namespace to your web.config</b>
        <br> 
        <br> 
        <div style="color:Black; background-color:White"> 
         <pre>
&lt;namespaces&gt;
	&lt;add <span style="color:Blue">namespace</span>=<span style="color:#A31515">"MvcReCaptcha.Helpers"</span>/&gt;
&lt;/namespaces&gt;
</pre> 
        </div> 
        <br> 
        <b>Step 3: Implement the logic in your view to actually render the Captcha control</b>
        <br> 
        <br> 
        <div style="color:Black; background-color:White"> 
         <pre>
&lt;%= Html.GenerateCaptcha() %&gt;
</pre> 
        </div> 
        <br> 
        <b>Step 4: Implement the Controller Action that will handle the form submission and Captcha validation</b>
        <br> 
        <br> 
        <div style="color:Black; background-color:White"> 
         <pre>
[CaptchaValidator]
[AcceptVerbs( HttpVerbs.Post )]
<span style="color:Blue">public</span> ActionResult CreateComment( Int32 id, <span style="color:Blue">bool</span> captchaValid )
{
	<span style="color:Blue">if</span> (!captchaValid)
	{
		ModelState.AddModelError(<span style="color:#A31515">"_FORM"</span>, <span style="color:#A31515">"You did not type the verification word correctly. Please try again."</span>);
	}
	<span style="color:Blue">else</span>
	{
		<span style="color:Green">// If we got this far, something failed, redisplay form</span>
		<span style="color:Blue">return</span> View();
	}
}
</pre>
