        <h1>SharePoint 2010 FBA Pack</h1> 
        <p><strong>By <a href="http://www.visigo.com/">Visigo Software Consulting Inc.</a></strong> <br> Blog: <a href="http://blogs.visigo.com/chriscoulson">http://blogs.visigo.com/chriscoulson</a> <br> Ported from: <a href="http://cks.codeplex.com/releases/view/17901">CKS Forms Based Authentication Solution</a></p> 
        <p><strong><span style="color:#ff0000; font-size:larger">New: Priority Support Plan Available! Details <a title="FBA Pack Priority Support Plan" href="http://www.visigo.com/#support">here</a>.</span></strong></p> 
        <p><br> <strong>Project Description</strong> <br> A forms based authentication pack for SharePoint 2010. It includes web parts for registering users, changing passwords and password recovery. It includes tools for managing users and roles and for approving registrations. This is a port of the <a href="http://cks.codeplex.com/releases/view/17901">CKS Forms Based Authentication Solution</a> for SharePoint 2007. <br> <br> <strong>Contents</strong></p>

        <p><strong>Features</strong></p> 
        <ul> 
         <li>User Management page allows you to Create, Edit, Delete and Search for users. Allows for password reset and user unlocking. Users can be assigned to SharePoint Groups or Roles. </li>
         <li>Role Management page allows roles to be created and deleted. </li>
         <li>Built in Change Password page and menu item. </li>
         <li>Optionally review and moderate new membership requests. </li>
         <li>Web parts can be completely customized using HTML templates. Simple customizations can be done with the web part properties. </li>
         <li>User emails can be completely customized using XSLT templates. </li>
