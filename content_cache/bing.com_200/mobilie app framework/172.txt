          <h1 class="page-title"><span class="field field--name-title field--type-string field--label-hidden">DEV6's Point of View on JavaScript Web &amp; Mobile Apps</span> </h1> 
          <div class="col-sm-10 col-sm-offset-1"> 
           <div class="clearfix text-formatted field field--name-body field--type-text-with-summary field--label-hidden field__item">
            <p>JavaScript, HTML5, and CSS3 are ready for prime time.</p> 
            <p>And in most cases they are&nbsp;the best technology choice for web and mobile app development for the enterprise. Rapid advancements in frameworks such as AngularJS, Meteor and others, allow for complex applications to be developed quickly and with high quality.</p> 
           </div> 
           <div class="field field--name-field-thumbnail field--type-image field--label-hidden field__item"> 
            <img src="/sites/default/files/styles/medium/public/2016-03/offset_251083.jpg?itok=xh6mvxeo" width="220" height="147" alt="Leaders in Enterprise JavaScript Development" typeof="foaf:Image" class="image-style-medium"> 
           </div> 
          </div> 
         </div> 
        </div>

        <div class="node__content"> 
         <div class="field field--name-field-slash-box field--type-field-collection field--label-hidden field__items"> 
          <div class="field__item background-image center down gradient">
           <div class="field-collection-item field-collection-item--name-field-slash-box field-collection-item--view-mode-full"> 
            <div class="content"> 
             <div class="clearfix text-formatted field field--name-field-text field--type-text-long field--label-hidden field__item">
              <h2 style="text-align: center;color:#107bc5;">Reasons to Use JavaScript</h2> 
              <p style="text-align:left;color:#000;">JavaScript has broad acceptance.</p> 
              <p style="text-align:left;color:#000;">JavaScript is a high-level, secure, dynamic, and interpreted programming language that has matured greatly since its development in 1995. It’s one of the three essential technologies of the World Wide Web (alongside HTML and CSS) and has been standardized in the ECMAScript language specification.</p> 
             </div> 
            </div> 
           </div> 
          </div> 
          <div class="field__item background-image center vertical down gradient">
           <div class="field-collection-item field-collection-item--name-field-slash-box field-collection-item--view-mode-full"> 
            <div class="content"> 
             <div class="field field--name-field-image field--type-image field--label-hidden field__item"> 
              <img src="/sites/default/files/2016-04/slashbox-dark-tall_3.png" width="1920" height="3000" alt="DEV6 runs JavaScript on Client Side Apps" typeof="foaf:Image"> 
             </div> 
             <div class="clearfix text-formatted field field--name-field-text field--type-text-long field--label-hidden field__item">
              <h2 style="text-align:left;color:#fff;">DEV6 runs JavaScript on Client Side Apps</h2> 
              <p style="text-align:left;">JavaScript is fast.</p> 
              <p style="text-align:left;">The code is executed locally and&nbsp;results and processing are completed almost instantly, depending on the task. Cross platform support – concept to write front end apps once and run on many platforms = all modern web browsers, add frameworks such as cordova/phonegap, ionic to package apps for mobile: iOS, Android, Microsoft etc. Backend interfaces using REST web services are easier to build and put less load on servers.</p> 
              <p>&nbsp;</p> 
             </div> 
            </div> 
           </div> 
          </div> 
          <div class="field__item background-image center vertical down gradient">
           <div class="field-collection-item field-collection-item--name-field-slash-box field-collection-item--view-mode-full"> 
            <div class="content"> 
             <div class="field field--name-field-image field--type-image field--label-hidden field__item"> 
              <img src="/sites/default/files/2016-04/slashbox-light-tall_3.png" width="1920" height="3000" alt="DEV6 leverages JavaScript for Server Side Apps" typeof="foaf:Image"> 
             </div> 
             <div class="clearfix text-formatted field field--name-field-text field--type-text-long field--label-hidden field__item">
              <h2 style="text-align:left;color:#fff;">DEV6 leverages JavaScript for Server Side Apps</h2> 
              <p style="text-align:left;">For Server Side Apps, there are tremendous efficiencies realized through the reduction in the range of skill sets required for a full-stack development team.</p> 
              <p style="text-align:left;">The NodeJS application server is maturing rapidly, now at version 4.0. &nbsp;And, Express.js and other frameworks provide a tremendous leg up for REST and Websocket app JavaScript development.&nbsp; Leveraging JavaScript for server side apps can also increase performance for I/O&nbsp;intensive apps. Databases like MongoDB&nbsp;are gaining popularity and are now being considering as an enterprise grade DBs.</p> 
              <p>&nbsp;</p> 
             </div> 
            </div> 
           </div> 
          </div> 
          <div class="field__item background-image center vertical down gradient">
           <div class="field-collection-item field-collection-item--name-field-slash-box field-collection-item--view-mode-full"> 
            <div class="content"> 
             <div class="field field--name-field-image field--type-image field--label-hidden field__item"> 
              <img src="/sites/default/files/2016-04/slashbox-dark-tall_4.png" width="1920" height="3000" alt="DEV6 leverages JavaScript Frameworks" typeof="foaf:Image"> 
             </div> 
             <div class="clearfix text-formatted field field--name-field-text field--type-text-long field--label-hidden field__item">
              <h2 style="text-align:left;color:#fff;">DEV6 leverages JavaScript Frameworks</h2> 
              <p style="text-align:left;">Recent JavaScript frameworks can help architect code so that it is:</p> 
              <ul style="text-align:left;padding-left:15px;">
               <li>Modular (therefore reusable)</li> 
               <li>DRY</li> 
               <li>Readable</li> 
               <li>Performant</li> 
               <li>Secure</li> 
               <li>And most importantly, cohesive and&nbsp;easy to understand</li> 
              </ul>
              <p style="text-align:left;">Modern frameworks are providing better templating, data-binding, routing (single page app) and security. Some modern frameworks include: AngularJS, Meteor, EmberJS and React.</p> 
              <p>&nbsp;</p> 
             </div> 
            </div>
