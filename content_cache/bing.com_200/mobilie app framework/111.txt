             <div class="post-text" itemprop="text"> 
              <p>I need to create a JavaScript mobile app that will read fitness history from <code>Google Fit</code> and <code>Apple HealthKit</code>. If the app is installed on Android it need to read from Google Fit. If the app is installed on iOS, it need to read from Apple HealthKit. </p> 
              <p>I found an article how to read from Apple Health Kit using Ionic framework. But I don’t any references how to read from Google Fit using IONIC. </p> 
              <p>Reference: <a href="https://devdactic.com/healthkit-app-with-ionic/" rel="nofollow noreferrer">How To Make A HealthKit App With Ionic and ngCordova</a></p> 
              <p>Is there any hybrid app frameworks implementation examples that can read <strong>fitness history</strong> from both Google Fit and Apple Health Kit?</p> 
              <p><a href="https://developers.google.com/fit/rest/" rel="nofollow noreferrer">Google Fit - REST API</a> says:</p> 
              <blockquote> 
               <p>The Google Fit REST API enables you to store and access user data in the fitness store from apps on any platform.</p> 
              </blockquote>
