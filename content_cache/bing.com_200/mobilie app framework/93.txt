             <h2>Introduction</h2> 
             <p>This article lets you through a sample application developed using <em>Node.js</em>, Express Framework and using MongoDB as backend.</p> 
             <p>Well be covering an application walkthrough build with the following technologies:</p> 
             <ul> 
              <li>Server Side - <em>Node.Js</em></li> 
              <li>ClientSide - Jquery, HTML, CSS</li> 
              <li>View Parser Engine - Jade</li> 
              <li>Development framework - Express Framework</li> 
              <li>Database - MongoDB</li> 
             </ul> 
             <p><strong>Express Framework</strong>: Express is a minimal and flexible <em>Node.js</em> web application framework that provides a robust set of features for web and mobile applications.</p> 
             <p><strong>Node.js</strong>: <em>Node.js</em> is an open source, cross-platform runtime environment for server-side and networking applications. <em>Node.js</em> applications are written in JavaScript, and can be run within the <em>Node.js</em> runtime on OS X, Microsoft Windows, Linux and FreeBSD.</p> 
             <p><strong>MongoDB</strong>: MongoDB is an open source database that uses a document-oriented data model. MongoDB is one of several database types to arise in the mid-2000s under the NoSQL banner. Instead of using tables and rows as in relational databases, MongoDB is built on an architecture of collections and documents.</p>
