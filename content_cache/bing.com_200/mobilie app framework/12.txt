Oracle MAF leverages Java, HTML5 and JavaScript to deliver a complete MVC framework with declarative user interface definition, device features integration and built-in security. Oracle MAF provides a visual and declarative development experience and maximizes code reuse resulting in faster development of mobile applications.</p>
