        <section class="ocpArticleTitleSection"> 
         <h1>Convert text to a table or a table to text</h1> 
        </section> 
        <section class="supAppliesToSection" ms.cmpgrp="applies_to" aria-label="This article" dir="ltr"> 
         <div id="supAppliesToTableContainer"> 
          <div id="supAppliesToList" class="appliesTo collapsed"> 
           <span>Applies To: </span> 
           <span class="appliesToItem">Word 2016</span> 
           <span class="appliesToItem">Outlook 2016</span> 
           <span class="appliesToItem">Word 2013</span> 
           <span class="appliesToItem">Outlook 2013</span> 
           <span class="appliesToItem">Word 2010</span>

           <a tabindex="0" class="appliesToOverflowControl expand ocHidden font-bold " href="javascript:" ms.interactiontype="11" ms.title="Less applies to" aria-label="Press the Enter key to listen to the other Office versions that this help article applies to.">More...</a> 
           <a tabindex="0" class="appliesToOverflowControl collapse ocHidden font-bold" href="javascript:" ms.interactiontype="11" ms.title="More applies to">Less</a> 
          </div>

        <article id="ocpArticleContent" class="ocpArticleContent"> 
         <section class="ocpIntroduction"> 
          <p>To convert text to a table or a table to text, start by clicking the <b class="ocpUI">Show/Hide</b> paragraph mark on the <b class="ocpUI">Home</b> tab so you can see how text is separated in your document.</p> 
          <p> <img src="https://support.content.office.net/en-us/media/f006c0ad-671f-494c-98f7-7cc1184c30dc.png" alt="The Show/Hide icon is highlighted on the Home tab." title="The Show/Hide icon is highlighted on the Home tab."> </p> 
         </section> 
         <section class="ocpSection"> 
          <h2>Convert text to a table</h2> 
          <ol type="1" itemscope itemtype="http://schema.org/ItemList"> 
           <li itemprop="itemListElement" itemscope itemtype="http://schema.org/ListItem"> <p itemprop="name">Insert separator characters—such as commas or tabs—to indicate where to divide the text into table columns. </p> 
            <div class="ocpAlert"> 
             <p class="ocpAlertSection"> <b class="ocpNote">Note:</b>&nbsp; If you have commas in your text, use tabs for your separator characters.</p> 
            </div> 
            <meta itemprop="position" content="1"> </li> 
           <li itemprop="itemListElement" itemscope itemtype="http://schema.org/ListItem"> <p itemprop="name"> Use paragraph marks to indicate where you want to begin a new table row.</p> <p>In this example, the tabs and paragraph marks will produce a table with 3 columns and 2 rows:</p> <p> <img src="https://support.content.office.net/en-us/media/89e1d3ca-cf81-43e1-a26d-c4791dc575a4.png" alt="Text to convert to table" title="Text to convert to table"> </p> 
            <meta itemprop="position" content="2"> </li> 
           <li itemprop="itemListElement" itemscope itemtype="http://schema.org/ListItem"> <p itemprop="name">Select the text that you want to convert, and then click <b class="ocpUI">Insert</b> &gt;<b class="ocpUI"> Table </b> &gt;<b class="ocpUI"> Convert Text to Table</b>.</p> <p> <img src="https://support.content.office.net/en-us/media/3e8d3653-0008-497f-bc85-c723ad3683e4.png" alt="The Convert Text to Table option is highlighted on the Insert tab." title="The Convert Text to Table option is highlighted on the Insert tab."> </p> 
            <meta itemprop="position" content="3"> </li> 
           <li itemprop="itemListElement" itemscope itemtype="http://schema.org/ListItem"> <p itemprop="name">In the <b class="ocpUI">Convert Text to Table</b> box, choose the options you want.</p> <p> <img src="https://support.content.office.net/en-us/media/f7b42e83-798b-47ef-92b5-5d781c23151e.png" alt="The Convert Text to Table dialog box is shown." title="The Convert Text to Table dialog box is shown."> </p> <p>Under <b class="ocpUI">Table size</b>, make sure the numbers match the numbers of columns and rows you want.</p> <p>Under <b class="ocpUI">AutoFit behavior</b>, choose how you want your table to look. Word automatically chooses a width for the table columns. If you want a different column width, choose one of these options:</p> <p></p> 
            <table id="tblID0EBBAACAAA" class="banded"> 
             <tbody> 
              <tr> 
               <td> <p> <b class="ocpLegacyBold">To do this</b> </p> </td> 
               <td> <p> <b class="ocpLegacyBold">Choose this option</b> </p> </td> 
              </tr> 
              <tr> 
               <td> <p>Specify a width for all the columns</p> </td> 
               <td> <p>In the <b class="ocpUI">Fixed column width</b> box, type or select a value.</p> </td> 
              </tr> 
              <tr> 
               <td> <p>Resize the columns to fit the width of the text in each column</p> </td> 
               <td> <p> <b class="ocpUI">AutoFit to contents</b> </p> </td> 
              </tr> 
              <tr> 
               <td> <p> Resize the table automatically in case the width of the available space changes (for example, web layout or landscape orientation)</p> </td> 
               <td> <p> <b class="ocpUI">AutoFit to window</b> </p> </td> 
              </tr> 
             </tbody> 
            </table> <p>Under <b class="ocpUI">Separate text at</b>, choose the separator character you used in the text.</p> 
            <meta itemprop="position" content="4"> </li> 
           <li itemprop="itemListElement" itemscope itemtype="http://schema.org/ListItem"> <p itemprop="name">Click <b class="ocpUI">OK</b>. The text converted to a table should look something like this:</p> <img src="https://support.content.office.net/en-us/media/77d467a7-700c-461b-85ae-079abd20244a.png" alt="Table" title="Table"> 
            <meta itemprop="position" content="5"> </li> 
          </ol> 
         </section> 
         <section class="ocpSection"> 
          <h2>Convert a table to text</h2> 
          <ol type="1" itemscope itemtype="http://schema.org/ItemList"> 
           <li itemprop="itemListElement" itemscope itemtype="http://schema.org/ListItem"> <p itemprop="name">Select the rows or table you want to convert to text.</p> 
            <meta itemprop="position" content="1"> </li> 
           <li itemprop="itemListElement" itemscope itemtype="http://schema.org/ListItem"> <p itemprop="name">Under <b class="ocpUI">Table Tools</b>, on the <b class="ocpUI">Layout</b> tab, click <b class="ocpUI"> Convert to Text.</b></p> <p> <img src="https://support.content.office.net/en-us/media/e230f2e6-e4dc-4a7f-b6b0-c03d11a6fc92.png" alt="The Convert to Text option is highlighted on the Table Tools Layout tab." title="The Convert to Text option is highlighted on the Table Tools Layout tab."> </p> 
            <meta itemprop="position" content="2"> </li> 
           <li itemprop="itemListElement" itemscope itemtype="http://schema.org/ListItem"> <p itemprop="name">In the <b class="ocpUI">Convert to Text</b> box, under <b class="ocpUI">Separate text with</b>, click the separator character you want to use in place of the column boundaries. Rows will be separated by paragraph marks.</p> 
            <meta itemprop="position" content="3"> </li> 
           <li itemprop="itemListElement" itemscope itemtype="http://schema.org/ListItem"> <p itemprop="name">Click <b class="ocpUI">OK</b>.</p> 
            <meta itemprop="position" content="4"> </li> 
          </ol> 
         </section>
