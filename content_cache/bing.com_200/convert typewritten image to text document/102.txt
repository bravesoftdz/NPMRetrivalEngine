          <h2 class="subTitle">Java Barcode Reader Install &amp; Requirement</h2>

         </div>

<font color="0" style="font-style:normal">   </font>        String[] datas = BarcodeReader.read(new File("C:/YourBarcodeImage.gif"), BarcodeReader.EAN8);

</pre>

         <p class="title"> <a name="linear"></a> <font color="#CC0000"><b>How to read barcode from a specified area (rectangle) in the image?</b></font> </p> 
         <p> To improve scanning speed, you can specify a rectangle area in the image for scanning barcode. <br> </p>
         <pre>
ScanOption option = new ScanOption();
/**
* specify barcode area in the scanning image
*
* x1, y1, x2, y2, valid values are from 0 to 1, inclusive.
*
* @param x1  barcode area left top point X value, % of image width
* @param y1  barcode area left top point Y value, % of image height
* @param x2  barcode area right bottom point X value, % of image width
* @param y2  barcode area right bottom point Y value, % of image height
*/
// scan top 20% of the image
option.setScanDirection(ScanOption.SCAN_FROM_TOP_TO_BOTTOM);
option.setBarcodeArea(0, 0, 1, 0.2);
String[] datas = BarcodeReader.read(new File("C:/YourBarcodeImage.gif"), BarcodeReader.EAN8, option);
</pre> 
         <ul> 
          <li>Pass your barcode image file, and barcode type, and ScanOption object to BarcodeReader, and it will return barcode datas in the specified area from the image. </li>
         </ul> 
         <p></p> 
         <br> 
         <br> 
         <br> 
         <p class="title"> <a name="tiffsupport"></a> <font color="#CC0000"><b>TIFF/Bitmap Image Reader</b></font> </p> 
         <p> <br> For TIFF or Bitmap image reading only, developers need include two Java Advanced Imaging jars from Sun Microsystems, <b>jai_codec.jar</b>, <b>jai_core.jar</b>. Both are included under <b>lib</b> folder. <br> To know more details about Java Advanced Imaging, please go to <a href="https://jai-imageio.dev.java.net/" target="_blank">Java Advanced Imageing Site</a>. <br> </p> 
         <br> 
         <br> 
         <br> 
         <p class="title"> <a name="supplementdata"></a> <font color="#CC0000"><b>How to get supplement barcode data?</b></font> </p> 
         <p> <br><br> For barcode EAN 8, EAN 13, UPC-A, UPC-E, their barcode image may contain 2 or 5 digits supplement barcode, Java Barcode Reader will add supplement barcode data to the main barcode data. <br><img src="/tutorial/java-reader/barcode.gif"> <br> For example, Java barcode reader will return data as "123450512345" for above sample image. The First 7 digits "1234505" is UPCE data, and the last 5 digits "12345" is UPCE supplement data. <br> </p> 
         <br> 
         <br> 
         <br> 
         <p class="title"> <a name="upcedata"></a> <font color="#CC0000"><b>About UPC-E data</b></font> </p> 
         <p> <br><br> For UPC-E barcode, usually there are 8 digits printed on the barcode label. The first digit is UPC-E number system, valid values are 0 or 1. The next 6 digits are UPC-E data, and the last one is checksum. <br><br> Our library will return 6 digits data and the last one checksum. Developer should use the first 6 digits as UPC-E data, according to UPC-E specification. <br><br> For the above barcode image, 0 is the number system, 123450 is the UPC-E data, and 5 is the checksum. Our library will return 1234505, plus suppliment barcode 12345. So the final output will be 123450512345. </p> 
         <br> 
         <br> 
         <br> 
         <p class="title"> <a name="mulipagetiff"></a> <font color="#CC0000"><b>Support Multi-page Tiff File</b></font> </p> 
         <p> <br><br> For multi-page tiff file, Java barcode reader will scan every page in the file, and return all barcodes in all pages. </p>
