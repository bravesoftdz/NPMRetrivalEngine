        <div class="header  header-top"> 
         <h2 id="about_pdf_x_standards"> About PDF/X standards </h2> 
        </div> 
        <!-- for Cascading Toc --> 
       </div> 
       <div class="text parbase section"> 
        <a name="main-pars_text_1"></a> 
        <div class="text">
         <p>PDF/X standards are defined by the International Organization for Standardization (ISO). PDF/X standards apply to graphic content exchange. During PDF conversion, the file that is being processed is checked against the specified standard. If the PDF will not meet the selected ISO standard, a message appears, asking you to choose between canceling the conversion or going ahead with the creation of a non-compliant file. A widely used standards for a print publishing workflow is the PDF/X format such as PDF/X?1a.</p>
         <p>The PDF/X-4 format is reliable for live transparency and color management. This format is optimal for RIP processing, digital printers that use the Adobe PDF Print Engine, and any PDF file to be printed in Acrobat.</p>
        </div> 
       </div> 
       <div class="note parbase section"> 
        <div class="help-note"> 
         <p> <span class="help-note-title">Note:</span> </p>
         <p>For more information on PDF/X, see the <a href="http://www.adobe.com/go/ext_iso">ISO website</a> and the <a href="http://www.adobe.com/">Adobe website</a>.</p> 
         <p></p>

        <div class="header  header-top"> 
         <h2 id="compression_and_downsampling_options_for_pdfs"> Compression and downsampling options for PDFs </h2> 
        </div> 
        <!-- for Cascading Toc --> 
       </div> 
       <div class="text parbase section"> 
        <a name="main-pars_text_4"></a> 
        <div class="text">
         <p>When exporting documents to Adobe PDF, you can compress text and line art, and compress and downsample bitmap images. Depending on the settings you choose, compression and downsampling can significantly reduce the size of a PDF file with little or no loss of detail and precision.</p>
         <p>The Compression area of the Export Adobe PDF dialog box is divided into three sections. Each section provides the following options for compressing and resampling color, grayscale, or monochrome images in your artwork.</p>
        </div> 
       </div> 
       <div class="text parbase section"> 
        <a name="main-pars_text"></a> 
        <div class="text">
         <p>If you plan to use the PDF file on the web, use downsampling to allow for higher compression. If you plan to print the PDF file at high resolution, check with your prepress service provider before setting compression and downsampling options.</p> 
         <p>You should also consider whether users need to magnify a page. For example, if you are creating a PDF document of a map, consider using a higher image resolution so that users can zoom in on the map.</p> 
         <p>Downsampling refers to decreasing the number of pixels in an image. To downsample color, grayscale, or monochrome images, choose an interpolation method—average downsampling, bicubic downsampling, or subsampling—and enter the desired resolution (in pixels per inch). Then enter a resolution in the For Images Above text box. All images with resolution above this threshold will be downsampled.</p> 
         <p>The interpolation method you choose determines how pixels are deleted:</p> 
         <p><strong>Average Downsampling To</strong></p> 
         <p>Averages the pixels in a sample area and replaces the entire area with the average pixel color at the specified resolution.</p> 
         <p><strong>Subsampling To</strong></p> 
         <p>Chooses a pixel in the center of the sample area and replaces the entire area with that pixel color. Subsampling significantly reduces the conversion time compared with downsampling but results in images that are less smooth and continuous.</p> 
         <p><strong>Bicubic Downsampling To</strong></p> 
         <p>Uses a weighted average to determine pixel color, which usually yields better results than the simple averaging method of downsampling. Bicubic is the slowest but most precise method, resulting in the smoothest tonal gradations.</p> 
        </div> 
       </div> 
       <div class="text parbase section"> 
        <a name="main-pars_text_1920142557"></a> 
        <div class="text">
         <p><strong>Compression</strong></p> 
         <p>Determines the type of compression that is used:</p> 
         <p><strong>Automatic (JPEG)</strong></p> 
         <p>Determines automatically the best quality for color and grayscale images. For most files, this option produces satisfactory results.</p> 
         <p><strong>JPEG</strong></p> 
         <p>Is suitable for grayscale or color images. JPEG compression is&nbsp;<em>lossy</em>, which means that it removes image data and may reduce image quality; however, it attempts to reduce file size with a minimal loss of information. Because JPEG compression eliminates data, it can achieve much smaller files sizes than ZIP compression.</p> 
         <p><strong>ZIP</strong></p> 
         <p>Works well on images with large areas of single colors or repeating patterns, and for black-and-white images that contain repeating patterns. ZIP compression can be lossless or lossy, depending on the Image Quality setting.</p> 
         <p><strong>JPEG 2000</strong></p> 
         <p>Is the international standard for the compression and packaging of image data. Like JPEG compression, JPEG 2000 compression is suitable for grayscale or color images. It also provides additional advantages, such as progressive display. The JPEG 2000 option is only available when Compatibility is set to Acrobat&nbsp;6 (PDF 1.5) or later.</p> 
         <p><strong>Automatic (JPEG 2000)</strong></p> 
         <p>Determines automatically the best quality for color and grayscale images. The Automatic (JPEG 2000) option is only available when Compatibility is set to Acrobat&nbsp;6 (PDF 1.5) or later.</p> 
         <p><strong>CCITT And Run Length</strong></p> 
         <p>Are only available for monochrome bitmap images. CCITT (Consultative Committee on International Telegraphy and Telephony) compression is appropriate for black-and-white images and any images scanned with an image depth of 1 bit. Group 4 is a general-purpose method that produces good compression for most monochrome images. Group 3, used by most fax machines, compresses monochrome bitmaps one row at a time. Run&nbsp;Length compression produces the best results&nbsp;for images that contain large areas of solid black or white.</p> 
        </div> 
       </div>

         <p></p>

       <div class="header parbase section"> 
        <a name="main-pars_heading_8"></a> 
        <div class="header  header-top"> 
         <h2 id="adding_security_to_pdf_files"> Add security to PDF files </h2> 
        </div> 
        <!-- for Cascading Toc --> 
       </div> 
       <div class="text parbase section"> 
        <a name="main-pars_text_8"></a> 
        <div class="text">
         <p>When saving as PDF, you can add password protection and security restrictions, limiting not only who can open the file, but also who can copy or extract contents, print the document, and more. </p>
         <p>A PDF file can require passwords to open a document (document open password) and to change security settings (permissions password). If you set any security restrictions in your file, you should set both passwords; otherwise, anyone who opens the file could remove the restrictions. If a file is opened with a permissions password, the security restrictions are temporarily disabled.</p>
         <p>The RC4 method of security from RSA Corporation is used to password-protect PDF files. Depending on the Compatibility setting (in the General category), the encryption level will be high or low.</p>
        </div> 
       </div> 
       <div class="note parbase section"> 
        <div class="help-note"> 
         <p> <span class="help-note-title">Note:</span> </p>
         <p>Adobe PDF presets don’t support passwords and security settings. If you select passwords and security settings in the Export Adobe PDF dialog box, and then click Save Preset, the passwords and security settings won’t be preserved.</p> 
         <p></p> 
        </div> 
       </div>

        <div class="text">

         <p>Be aware that anyone using an earlier version of Acrobat cannot open a PDF document with a higher compatibility setting. For example, if you select the Acrobat&nbsp;7 (PDF 1.6) option, the document cannot be opened in Acrobat&nbsp;6.0 or earlier.</p>

        <a name="main-pars_heading_10"></a> 
        <div class="header  header-top"> 
         <h2 id="font_embedding_and_substitution"> Font embedding and substitution </h2> 
        </div> 
        <!-- for Cascading Toc --> 
       </div> 
       <div class="text parbase section"> 
        <a name="main-pars_text_10"></a> 
        <div class="text">
         <p>A font can be embedded only if it contains a setting by the font vendor that permits it to be embedded. Embedding prevents font substitution when readers view or print the file, and ensures that readers see the text in its original font. Embedding increases file size only slightly, unless the document uses CID fonts, a font format commonly used for Asian languages. You can embed or substitute fonts in Acrobat or when you export an InDesign document to PDF.</p>
         <p>You can embed the entire font, or just a subset of the characters used in the file.</p>
         <p>When a font cannot be embedded due to the font vendor settings, and someone who opens or prints a PDF does not have access to the original font, another font is temporarily substituted. To avoid problems, print only PDF files in which fonts can be embedded.</p>
         <p>The Multiple Master typeface can stretch or condense to fit, to ensure that line and page breaks in the original document are maintained. The substitution cannot always match the shape of the original characters, however, especially if the characters are unconventional ones, such as script typefaces.</p>
        </div> 
       </div>

        <div class="header  header-top"> 
         <h2 id="preparing_a_document_for_on_screen_viewing"> Prepare a document for onscreen viewing </h2> 
        </div> 
        <!-- for Cascading Toc --> 
       </div> 
       <div class="text parbase section"> 
        <a name="main-pars_text_11"></a> 
        <div class="text">
         <p>With its small file sizes, platform independence, and online navigation, Adobe PDF is an ideal format for distributing documents electronically and viewing them on-screen. You can send Adobe PDF documents to other users as e-mail attachments, or you can distribute the documents on the web or on an intranet.</p> 
         <p>For information on creating accessible PDF documents, see <a href="http://www.adobe.com/accessibility/products/indesign.html" target="_blank">Adobe InDesign CS4 accessibility</a>.</p> 
         <p>The following guidelines apply to electronically distributed Adobe PDF files:</p> 
        </div> 
       </div> 
       <div class="text parbase section"> 
        <a name="main-pars_text_12"></a> 
        <div class="text">
         <ul> 
          <li><p>Before putting Adobe PDF documents on a website, check to see that the text, artwork, and layout in the documents are complete and correct.</p> </li> 
          <li><p>Make sure that table of contents entries, hyperlinks, and bookmarks are generated correctly. Table of contents entries are generated automatically from information in the Bookmarks panel.</p> </li> 
          <li><p>Set up passwords and other security options.</p> </li> 
          <li><p>Use a file name of no more than eight characters, followed by an extension of up to three characters. Many networks and e-mail programs shorten long file names.</p> </li> 
          <li><p>Make sure that the file name has a PDF extension if users will view the file on a Windows computer or on the Internet.</p> </li> 
          <li><p>To apply predefined Adobe PDF export settings for on-screen viewing, choose Smallest File Size.</p> </li> 
         </ul> 
        </div> 
       </div> 
       <div class="note parbase section"> 
        <div class="help-note"> 
         <p> <span class="help-note-title">Note:</span> </p>
         <p>Adobe PDF files exported from InDesign documents that contain overprints or transparency effects are best viewed in Acrobat 5.0 and later, or Adobe Reader 7.0 and later, with the <span class="uicontrol">Overprint Preview</span>option selected. &nbsp;</p>
