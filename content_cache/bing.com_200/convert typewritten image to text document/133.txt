             </div> 
             <h3>Description</h3> BConvert is a small freeware package of Delphi components for number conversion to text (words) or Roman notation. Contains some basic support for different languages. 
             <ul> 
              <li>TNumberToText - This component converts a floating-point number to words. For example -312.74 will be converted to "minus three hundred twelve point seven four". </li>
              <li>TNumberToRoman - Set Number property to desired value, and then read RomanNumber property to obtain Roman format of this number. For example 654 will be converted to DCLI. </li>
             </ul>
