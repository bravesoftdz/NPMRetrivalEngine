         <h1><a id="user-content-send-an-sms-with-the-sinch-net-server-side-sdk" class="anchor" href="#send-an-sms-with-the-sinch-net-server-side-sdk" aria-hidden="true"><span class="octicon octicon-link"></span></a>Send an SMS with the Sinch .Net server-side SDK</h1> 
         <p><a href="https://www.sinch.com/opinion/sms-data/">If you thought SMS was old fashioned, think again</a>. 96% of smartphone users still use SMS. It has a 90% read rate in minutes and higher engagement rates than emails and OTT apps, which makes it very reach effective in terms of timeliness. SMS is the one solution, not requiring app installations or friend requests, not relying on tech heavy solutions or the operator at hand.</p> 
         <p>'Nuff said about the good old short message service for now. In this tutorial well be creating an application to send an SMS. Well be using the sandbox environment, and in the sandbox, you can only send messages to verified numbers on your account.</p> 
         <h2 id="setup"><a id="user-content-setup" class="anchor" href="#setup" aria-hidden="true"><span class="octicon octicon-link"></span></a>Setup</h2> 
         <p>In <a href="https://www.sinch.com/dashboard/#/login">the dashboard</a>, ensure you have a phone number on the Account page. If not, click Add a phone number. </p> 
         <p><img src="/wp-content/uploads/2015/10/manage-account.png" alt="manage account" style="max-width:100%;"></p> 
         <p>Enter your number and submit the PIN code you will receive in the verification SMS.</p> 
         <p><img src="/wp-content/uploads/2015/10/verify-number.png" alt="verify number" style="max-width:100%;"></p> 
         <p>You'll now have the number attached to your Sinch account, and will also have been credited $2 as a bonus for verifying your number!</p> 
         <p><img src="/wp-content/uploads/2015/10/account.png" alt="account" style="max-width:100%;"></p> 
         <p>Create a new application from the Apps tab. Leave the environment to the default: Sandbox.</p> 
         <p><img src="/wp-content/uploads/2015/10/app-list.png" alt="make an app" style="max-width:100%;"></p> 
         <p>Click on the key symbols and take note of your applications key and secret, as they'll be used in the Sinch Server SDK shortly.</p> 
         <p><img src="/wp-content/uploads/2015/10/keys-and-secret.png" alt="keys and secrets" style="max-width:100%;"> </p> 
         <p>Open Visual Studio and create a new Console Application:</p> 
         <p><img src="/wp-content/uploads/2015/10/new-project.png" alt="keys and secrets" style="max-width:100%;"></p> 
         <p>Right click on References in Solution Explorer and select <strong>Manage NuGet Packages</strong>.</p> 
         <p>If youre using an older version of Visual Studio and dont see this, you'll need to add it from here: <a href="http://docs.nuget.org/consume/installing-nuget">http://docs.nuget.org/consume/installing-nuget</a>.</p> 
         <p>Search for <code>Sinch.ServerSdk</code> and click <strong>Install</strong> for <strong>Sinch.ServerSdk</strong>.</p> 
         <p><img src="/wp-content/uploads/2015/10/nugget.png" alt="sinch nugget" style="max-width:100%;"></p> 
         <h2 id="mainasync( )"><a id="user-content-mainasync-" class="anchor" href="#mainasync-" aria-hidden="true"><span class="octicon octicon-link"></span></a>MainAsync( )</h2> 
         <p>Because of the limitations of the compiler, having an asynchronous <code>Main()</code> method is not allowed. To overcome this obstacle, and to make for a more straightforward tutorial, we'll create this method:</p> 
         <div class="highlight highlight-source-cs">
          <pre class="prettyprint linenums"><span class="pl-k">private</span> <span class="pl-k">static</span> <span class="pl-k">async</span> Task MainAsync()
{
}</pre>
         </div> 
         <p>Then, we want to add the following line of code to <code>Main()</code>:</p> 
         <p><code>MainAsync().GetAwaiter().GetResult();</code></p> 
         <p>Add the following lines to import the SDK namespace: using <code>Sinch.ServerSdk;</code></p> 
         <h2 id="send sms"><a id="user-content-send-sms" class="anchor" href="#send-sms" aria-hidden="true"><span class="octicon octicon-link"></span></a>Send SMS</h2> 
         <p>To send an SMS, we'll add the following to <code>MainAsync()</code>:</p> 
         <div class="highlight highlight-source-cs">
          <pre class="prettyprint linenums"><span class="pl-k">var</span> smsApi = SinchFactory.CreateApiFactory(<span class="pl-s"><span class="pl-pds">"</span>yourkey<span class="pl-pds">"</span></span>, <span class="pl-s"><span class="pl-pds">"</span>yoursecret<span class="pl-pds">"</span></span>).CreateSmsApi();
<span class="pl-k">var</span> sendSmsResponse = <span class="pl-k">await</span> smsApi.Sms(<span class="pl-s"><span class="pl-pds">"</span>+61491570156<span class="pl-pds">"</span></span>, <span class="pl-s"><span class="pl-pds">"</span>Hello world.  Sinch SMS here.<span class="pl-pds">"</span></span>).Send();</pre>
         </div> 
         <p>Remember to replace the application key and secret with your own, and change the phone number and message accordingly. It might be worth setting up a unique long number or an <a href="https://www.sinch.com/products/sms-api/sms-short-code/">SMS short code</a>.</p> 
         <h2 id="check status"><a id="user-content-check-status" class="anchor" href="#check-status" aria-hidden="true"><span class="octicon octicon-link"></span></a>Check status</h2> 
         <p>Next, we want to check the status of an SMS. Add these couple of lines:</p> 
         <div class="highlight highlight-source-cs">
          <pre class="prettyprint linenums"><span class="pl-k">var</span> smsMessageStatusResponse = <span class="pl-k">await</span> smsApi.GetSmsStatus(sendSmsResponse.MessageId);
Console.WriteLine(smsMessageStatusResponse.Status);
Console.ReadLine(); </pre>
         </div> 
         <p>It might be worthwhile putting a <code>Task.Delay()</code> before checking the status, as it could take a second or two for the SMS to be delivered. Checking immediately will most likely lead to a status of <strong>Pending</strong>.</p> 
         <h2 id="celebrate"><a id="user-content-celebrate" class="anchor" href="#celebrate" aria-hidden="true"><span class="octicon octicon-link"></span></a>Celebrate</h2> 
         <p><img src="/wp-content/uploads/2015/10/command-line.png" alt="command line" style="max-width:100%;"></p> 
         <p>Youre done. As usual, you can download the complete, working example project on <a href="https://github.com/sinch/nuget-serversdk-tutorial">Github</a>. Go charm those 96%!</p>

    {

        }

        <span class="pl-k">private</span> <span class="pl-k">static</span> <span class="pl-k">async</span> Task <span class="pl-en">MainAsync</span>()
        {
            <span class="pl-k">var</span> smsApi = SinchFactory.CreateApiFactory(<span class="pl-s"><span class="pl-pds">"</span>00000000-0000-0000-0000-000000000000<span class="pl-pds">"</span></span>, <span class="pl-s"><span class="pl-pds">"</span>AAAAAAAAAAAAAAAAAAAAAA==<span class="pl-pds">"</span></span>).CreateSmsApi();
