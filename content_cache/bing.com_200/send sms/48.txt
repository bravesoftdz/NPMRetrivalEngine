       <h1><span class="TitleTag">JAVA OBJECT API</span></h1> 
       <div class="text2"> 
        <div class="twelvecol"> 
         <p>Using the JAVA OBJECT API is a simple way to implement SMS support in your Java environment. This API uses the <a href="/en/developers/sms-api-overview/yap-api">YAP API</a> to communicate with the server.</p> 
         <h2>Benefits</h2> 
         <ul> 
          <li>Simple and easy to implement.</li> 
          <li>See also <a href="/en/developers/sms-api-overview/yap-api">YAP API</a> benefits.</li> 
         </ul> 
         <h2>Requirements</h2> 
         <ul> 
          <li>Familiarity with the Java language.</li> 
          <li>Server/computer on which to install your script/application.</li> 
         </ul> 
         <h2>Example</h2>
