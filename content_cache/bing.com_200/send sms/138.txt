                     <h3> <span style="font-family: Arial, Helvetica, sans-serif;">Send SMS via Twitter step by step</span></h3> 
                     <div> 
                      <div style="text-align: justify;"> 
                       <span style="font-family: Arial, Helvetica, sans-serif;">Almost all the server monitoring systems has the "Health Check"&nbsp;functionality. Using that, we able to monitor the CPU performance, memory usage.. etc. If any of the server performances are reach the predefined threshold value, the system send the email or SMS alert to configured users.</span>
                       <br> 
                       <span style="font-family: Arial, Helvetica, sans-serif;"><br></span>
                      </div> 
                      <div style="text-align: justify;"> 
                       <span style="font-family: Arial, Helvetica, sans-serif;">In this tutorial, you can learn that, how to send the SMS message with using twitter via Java application. In&nbsp;additionally&nbsp;you can also learn how to register the application in twitter and what are the limitations if we use the twitter for send SMS.</span>
                      </div> 
                      <div style="text-align: justify;"> 
                       <span style="font-family: Arial, Helvetica, sans-serif;"><br></span>
                      </div> 
                      <div style="text-align: justify;"> 
                       <b style="font-family: Arial, Helvetica, sans-serif; text-align: start;"><u>Objective :</u></b>
                       <span style="font-family: Arial, Helvetica, sans-serif; text-align: start;">&nbsp;The main Objective of the tutorial is that, register the application in twitter and develop a Java application for send SMS.&nbsp;</span>
                      </div> 
                      <div style="text-align: justify;"> 
                       <br>
                      </div> 
                      <span style="font-family: Arial, Helvetica, sans-serif;"><b><u>Scope :</u></b> Develop a simple Java application for send SMS via twitter.</span>
                     </div> 
                     <div> 
                      <span style="font-family: Arial, Helvetica, sans-serif;"><br></span>
                     </div> 
                     <div> 
                      <span style="font-family: Arial, Helvetica, sans-serif;"><b style="text-decoration: underline;">Requirements :</b> The following basic requirement we need to set it up in order to achieve the objective.</span>
                     </div> 
                     <div> 
                      <ul> 
                       <li><span style="font-family: Arial, Helvetica, sans-serif;">Eclipse JAVA EE IDE</span></li> 
                       <li><span style="font-family: Arial, Helvetica, sans-serif;">JDK 1.6</span></li> 
                       <li><span style="font-family: Arial, Helvetica, sans-serif;">Twitter&nbsp;related&nbsp;jars</span></li> 
                      </ul> 
                     </div> 
                     <div> 
                      <div style="text-align: justify;"> 
                       <span style="font-family: Arial, Helvetica, sans-serif;"><b><u>Step 1</u></b> : You have to create an Twitter account and need to register the application with twitter. The registration details explained <a href="http://spring.io/guides/gs/register-twitter-app/" target="_blank">here</a>. Once the registration is completed, the registered account will be use to send the messages to clients.</span>
                      </div> 
                     </div> 
                     <div> 
                      <span style="font-family: Arial, Helvetica, sans-serif;"><br></span>
                     </div> 
                     <div> 
                      <div style="text-align: justify;"> 
                       <span style="font-family: Arial, Helvetica, sans-serif;"><b><u>Step 2</u></b> : Once the registration process completed you able to get the following tokens\keys from the twitter page. Please note that the tokens and keys are very important, so save those details to some secure place.&nbsp;</span>
                      </div> 
                      <div style="text-align: justify;"> 
                       <span style="font-family: Arial, Helvetica, sans-serif;"><br></span>
                      </div> 
                      <div style="text-align: justify;"> 
                       <span style="font-family: Arial, Helvetica, sans-serif;">Do not share the key/tokens to public, because using that, any once can send the messages through your registered application.</span>
                      </div> 
                     </div> 
                     <div> 
                      <span style="font-family: Arial, Helvetica, sans-serif;"><br></span>
                     </div> 
                     <div class="separator" style="clear: both; text-align: center;"> 
                      <a href="http://3.bp.blogspot.com/-GgdcPs7kxiE/VEnYVuPD-3I/AAAAAAAAAgU/HK6FQxY44IE/s1600/findanidea-Twitter%2BSMS%2Bapplication.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" height="504" src="http://3.bp.blogspot.com/-GgdcPs7kxiE/VEnYVuPD-3I/AAAAAAAAAgU/HK6FQxY44IE/s1600/findanidea-Twitter%2BSMS%2Bapplication.png" width="640"></a>
                     </div> 
                     <div> 
                      <span style="font-family: Arial, Helvetica, sans-serif;"><br></span>
                     </div> 
                     <div> 
                      <span style="font-family: Arial, Helvetica, sans-serif;"><b><u>Step 3</u></b> : Create a java project in Eclipse and name it as "TweetMsg". Add the following jars to build path of the project.</span>
                     </div> 
                     <div> 
                      <ul> 
                       <li><span style="font-family: Arial, Helvetica, sans-serif;">twitter4j-async-3.0.5.jar</span></li> 
                       <li><span style="font-family: Arial, Helvetica, sans-serif;">twitter4j-core-3.0.5.jar</span></li> 
                       <li><span style="font-family: Arial, Helvetica, sans-serif;">twitter4j-media-support-3.0.5.jar</span></li> 
                       <li><span style="font-family: Arial, Helvetica, sans-serif;">twitter4j-stream-3.0.5.jar&nbsp;</span></li> 
                      </ul> 
                     </div> 
                     <div> 
                      <div style="text-align: justify;"> 
                       <span style="font-family: Arial, Helvetica, sans-serif;"><b><u>Step 4</u></b> : Create a java class "TweetMessage</span>
                       <span style="font-family: Arial, Helvetica, sans-serif;">" in package "</span>
                       <span style="font-family: Arial, Helvetica, sans-serif;">com.prem.twitter.message</span>
                       <span style="font-family: Arial, Helvetica, sans-serif;">". The following TweetMessage class used to send the message to twitter account with using the token/keys.</span>
                      </div> 
                     </div> 
                     <div> 
                      <span style="font-family: Arial, Helvetica, sans-serif;"><br></span>
                     </div>
