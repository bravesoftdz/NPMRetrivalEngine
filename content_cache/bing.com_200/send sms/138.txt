        <p class="lead">So you want to build an application that sends and receives text messages? &nbsp;SMS (Short Message Service) is widely used all over the world on mobile devices and can be a powerful notification/communications tool to connect to friends, customers or audience. &nbsp;Even though the technologies behind sending and receiving these messages are relatively complex, sending and receiving text messages with Tropo is easy!</p> 
        <p>In this example were going to make a simple Hello World text messaging application using Tropo and JavaScript. &nbsp;It should take you less than 10 minutes.</p> 
        <p>All youll need to try this example is &nbsp;a Tropo account and a device capable of sending and receiving SMS text messages. &nbsp; If you dont already have a Tropo account, you can register for free at <a href="https://www.tropo.com/account/register.jsp">https://www.tropo.com/account/register.jsp</a> and your login credentials will be e-mailed to you.&nbsp; Once you have your credentials, click the <strong>Login </strong>link at the top right of the main Tropo site (<a href="https://www.tropo.com/">tropo.com</a>) and enter them in. &nbsp; Go to <strong>Your Applications </strong>and then click on <strong>Create New Application.</strong></p> 
        <p><a href="https://www-tropo-com.s3.amazonaws.com/wp-content/uploads/2010/10/tropo_SMS1.jpg" onclick="javascript:_gaq.push(['_trackEvent','outbound-article','http://www-tropo-com.s3.amazonaws.com/wp-content/uploads/2010/10/tropo_SMS1.jpg']);"><img class="aligncenter size-medium wp-image-1668" title="Tropo SMS API " alt="SMS API JavaScript" src="https://www-tropo-com.s3.amazonaws.com/wp-content/uploads/2010/10/tropo_SMS1-300x219.jpg" width="300" height="219"></a> Tropo comes in two flavors, Scripting and WebAPI. &nbsp;In this example, well be using Tropo Scripting, so click on the Tropo Scripting Icon.</p> 
        <a href="https://www-tropo-com.s3.amazonaws.com/wp-content/uploads/2010/10/Tropo-SMS2.jpg" onclick="javascript:_gaq.push(['_trackEvent','outbound-article','http://www-tropo-com.s3.amazonaws.com/wp-content/uploads/2010/10/Tropo-SMS2.jpg']);"><img class="aligncenter size-medium wp-image-1669" title="Tropo SMS API" alt="Javascript SMS API" src="https://www-tropo-com.s3.amazonaws.com/wp-content/uploads/2010/10/Tropo-SMS2-300x196.jpg" width="300" height="196"></a> 
        <p>In the box that says <strong>Tropo Scripting Application Name</strong>, type Hello World.&nbsp; Next, click the link for <strong>Hosted File</strong> (next to <strong>What URL powers your app?</strong>), then click <strong>Create a new hosted file for this application</strong>.</p> 
        <a href="https://www-tropo-com.s3.amazonaws.com/wp-content/uploads/2010/10/Tropo_SMS3.jpg" onclick="javascript:_gaq.push(['_trackEvent','outbound-article','http://www-tropo-com.s3.amazonaws.com/wp-content/uploads/2010/10/Tropo_SMS3.jpg']);"><img class="aligncenter size-medium wp-image-1671" title="Tropo SMS API " alt="SMS API JavaScript" src="https://www-tropo-com.s3.amazonaws.com/wp-content/uploads/2010/10/Tropo_SMS3-300x192.jpg" width="300" height="192"></a>

        <p>After youve pasted the code, click <strong>Create File</strong> to save it, then when youre returned to the New Application page, click the <strong>Create Application</strong> button to save your app.&nbsp; Your screen should should now look something like this:</p> 
        <a href="https://www-tropo-com.s3.amazonaws.com/wp-content/uploads/2010/10/tropo_sms5.jpg" onclick="javascript:_gaq.push(['_trackEvent','outbound-article','http://www-tropo-com.s3.amazonaws.com/wp-content/uploads/2010/10/tropo_sms5.jpg']);"><img class="aligncenter size-medium wp-image-1675" title="Tropo SMS API JavaScript" alt="Tropo SMS API JavaScript" src="https://www-tropo-com.s3.amazonaws.com/wp-content/uploads/2010/10/tropo_sms5-287x300.jpg" width="287" height="300"></a> 
        <p>Click on the <strong>Add a new phone number</strong> link to get a standard phone number assigned (for free), then give it a few minutes to allow your application to propagate to our servers.</p> 
        <p>With your new app created and a phone number assigned, youre ready to test! Send a text to the number, any text, and you should immediately receive the response Hello World.</p> 
        <p>Thats it!&nbsp; You now have a working SMS application in its simplest form.&nbsp; This should just be the beginning, though; the potential power of Tropo extends far beyond Hello World apps.&nbsp; To delve deeper, check out this post: <a href="http://blog.tropo.com/2010/10/18/how-to-get-started-with-sms-text-messaging-on-tropo-com/">How To Get Started with SMS / Text Messaging on Tropo.com</a> or go right to the Tropo <a title="SMS API" href="https://www.tropo.com/docs/scripting/textcalls.htm">SMS API docs</a>.</p> 
        <p>If this helped you, please <a href="http://twitter.com/?status=Cool! I just built my first SMS app with JavaScript and @Tropo in less than 10 minutes!" onclick="javascript:_gaq.push(['_trackEvent','outbound-article','http://twitter.com/?status=Cool!IjustbuiltmyfirstSMSappwithJavaScriptand@Tropoinlessthan10minutes!']);" target="_blank">show us some love on Twitter</a>.</p>
