        <div class="uc_section" id="step1"> 
         <h2>Benefits of Java patch management using Patch Manager</h2> 
         <p>No amount of guess-work ensures 100% patching of Java apps in your environment. Even if you can somehow manually track all Java apps and patches, there’s a probability of failed updates that could leave desktops and servers vulnerable to security attacks. However, with Patch Manager you can:</p> 
         <ul class="uc_textStyles"> 
          <li>Discover systems that are running versions of Java with known vulnerabilities</li> 
          <li>Control when and where updates are executed with advanced scheduling</li> 
          <li>Significantly reduce failed patches with PackageBoot technology.</li> 
         </ul> 
        </div> 
        <div class="uc_section" id="step3"> 
         <h2>Implementing Java patch management</h2> 
         <p>Java patch management can be automated by installing&nbsp; <a href="/patch-manager">SolarWinds Patch Manager</a>. The update notifications are available on the Patch Manager console and via email. The update/upgrade package list is automatically synchronized to the Patch Manager server. To publish the latest JRE package, download it from Oracle and import it into Patch Manager. The package download assistant directs you to the right website and helps you grab the correct patch files.</p> 
         <a href="http://cdn.swcdn.net/creative/v17.5/images/landing_pages/Use-Case/img/screenshots/PatchManager_deploy_Java_Patch_EN.png" data-lightbox="image" class="uc_zoom"><img alt="View of Java Patch Management and Security Patching dashboard." class="uc_imgCenter" src="http://cdn.swcdn.net/creative/v17.5/images/landing_pages/Use-Case/img/screenshots/PatchManager_deploy_Java_Patch_EN_sm.png"></a>
         <br>

         <p class="uc_center"><a href="/patch-manager">Learn more</a>&nbsp;about SolarWinds Patch Manager</p>

        <div class="uc_section" id="step3"> 
         <h2>How Java patch management works</h2> 
         <p style="color: inherit; margin: 0px; padding: 10px 0px; border: 0px; letter-spacing: 0px;">When a Java patch file is imported, the installation file is verified and included in the package definition and stored on the SolarWinds Patch Manager server. The update management utility helps you deploy the update to the required systems. The PackageBoot utility works along with the installer files to help ensure a proper pre-installation environment. Pre-installation tasks may include starting/stopping services, terminating running processes and programs, and testing for locked files. Post-installation tasks can also be specified for successful completion of the entire patch process.</p> 
         <a href="http://cdn.swcdn.net/creative/v17.5/images/landing_pages/Use-Case/img/screenshots/PatchManager_implement_Java_Patches_EN.png" data-lightbox="image" class="uc_zoom"><img alt="View of how SolarWinds Java patch management software works." class="uc_imgCenter" src="http://cdn.swcdn.net/creative/v17.5/images/landing_pages/Use-Case/img/screenshots/PatchManager_implement_Java_Patches_EN_sm.png"></a>
