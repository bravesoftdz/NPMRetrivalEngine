                       <h1 class="header-title"> <span> Package Tracking with Easypost Java API </span> </h1> 
                      </div> 
                      <form action="http://www.surekhatech.com/blog?p_auth=EAVJ9wci&amp;p_p_id=33&amp;p_p_lifecycle=1&amp;p_p_state=normal&amp;p_p_mode=view&amp;p_p_col_id=_118_INSTANCE_aMyP6kAp15Kk__column-2&amp;p_p_col_count=1&amp;_33_struts_action=%2Fblogs%2Fedit_entry" class="aui-form " id="_33_fm1" method="post" name="_33_fm1">

                        <div class="entry-body"> 
                         <p dir="ltr" id="docs-internal-guid-b539845e-140c-d2e4-ceed-14fdd9e061a8" style="line-height:1.38;margin-top:0pt;margin-bottom:0pt;text-align: justify;"> <span style="font-size: 11pt; font-family: Arial; vertical-align: baseline; background-color: transparent;">There are number of courier companies in US which provide service to customers to send packages from one location to another location. Some of shipping companies are FedEx, UPS, DHL Express, USPS, etc. They also provide tracking service to customer to track their order/package. There are number of third party vendors available too, which provides package tracking service by their well defined API. Some of them are easypost, aftership, stamps, endicia, etc.</span></p> 
                         <p> &nbsp;</p> 
                         <p dir="ltr" style="line-height:1.38;margin-top:0pt;margin-bottom:0pt;text-align: justify;"> <span style="font-size: 11pt; font-family: Arial; vertical-align: baseline; background-color: transparent;">Easypost provides well defined and easy to understand API. Using API you can create shipping label and add tracking programmatically in package without worrying about to go at shipping site and get label. If you run a business like courier or sending customer stuff from one location to another and you want to allow your customer to track their packages then you are at the right place. You can implement tracking feature&nbsp;</span><span style="font-family: Arial; font-size: 14.6667px; line-height: 20.24px;">for your website and mobile application&nbsp;</span><span style="font-family: Arial; font-size: 11pt; line-height: 1.38; background-color: transparent;">using easypost API . In this article you can see how to add tracking in package using Easypost Java API.</span></p> 
                         <p> &nbsp;</p> 
                         <p dir="ltr" style="line-height:1.38;margin-top:0pt;margin-bottom:0pt;"> <span style="font-size: 11pt; font-family: Arial; font-weight: 700; text-decoration: underline; vertical-align: baseline; background-color: transparent;">Implementation</span><span style="font-size: 11pt; font-family: Arial; font-weight: 700; vertical-align: baseline; background-color: transparent;"> :</span></p> 
                         <p dir="ltr" style="line-height:1.38;margin-top:0pt;margin-bottom:0pt;"> &nbsp;</p> 
                         <p dir="ltr" style="line-height:1.38;margin-top:0pt;margin-bottom:0pt;text-indent: 36pt;"> <span style="font-size: 11pt; font-family: Arial; vertical-align: baseline; background-color: transparent;">To get shipping label for your package using API follow below functional diagram. Diagram represents steps to generate shipping label for package tracking.</span></p> 
                         <p dir="ltr" style="line-height:1.38;margin-top:0pt;margin-bottom:0pt;text-indent: 36pt;"> &nbsp;</p> 
                         <p dir="ltr" style="line-height:1.38;margin-top:0pt;margin-bottom:0pt;text-indent: 36pt;"> <img alt="Functional Flow Diagram" src="/documents/54265/1098368/Package+Tracking+Blog5_4.png/db4283a1-e041-416d-9c57-d0ca784077ed?t=1490919496366" style="width: 720px; height: 540px;"></p> 
                         <p dir="ltr" style="line-height:1.38;margin-top:0pt;margin-bottom:0pt;text-indent: 36pt;"> &nbsp;</p> 
                         <p dir="ltr" style="line-height:1.38;margin-top:0pt;margin-bottom:0pt;"> <span style="text-indent: 36pt; font-size: 11pt; font-family: Arial; font-weight: 700; vertical-align: baseline; background-color: transparent;">Step 1 :</span><span style="text-indent: 36pt; font-size: 11pt; font-family: Arial; vertical-align: baseline; background-color: transparent;"> Create ‘To Address’ and ‘From Address’, easypost also provides address verification process to verify addresses before generating shipping label.</span></p> 
                         <div> 
                          <p dir="ltr" style="line-height:1.38;margin-top:0pt;margin-bottom:0pt;"> &nbsp;</p> 
                          <p dir="ltr" style="line-height:1.38;margin-top:0pt;margin-bottom:0pt;"> <span style="font-size: 11pt; font-family: Arial; font-weight: 700; vertical-align: baseline; background-color: transparent;">Step 2 : </span><span style="font-size: 11pt; font-family: Arial; vertical-align: baseline; background-color: transparent;">After successfully verifying addresses there will be step to create Parcel. To create Parcel there are some properties need to be set i.e. weight, height, etc. Easypost provides predefined parcels which have </span><span style="font-size: 11pt; font-family: Arial; vertical-align: baseline;">predefined dimensions and weight </span><span style="font-size: 11pt; font-family: Arial; vertical-align: baseline; background-color: transparent;">i.e. </span><span style="font-size: 11pt; font-family: Arial; vertical-align: baseline; ">Card, </span><span style="font-size: 11pt; font-family: Arial; vertical-align: baseline;">Letter, Flat, Parcel, etc. Select one which qualifies your requirement </span><span style="font-size: 11pt; font-family: Arial; vertical-align: baseline; background-color: transparent;">while creating parcel.</span></p> 
                          <br> 
                          <p dir="ltr" style="line-height:1.38;margin-top:0pt;margin-bottom:0pt;"> <span style="font-size: 11pt; font-family: Arial; font-weight: 700; vertical-align: baseline; background-color: transparent;">Steps 3 : </span><span style="font-size: 11pt; font-family: Arial; vertical-align: baseline; background-color: transparent;">Create shipment using combination of To address, From address and Parcel. On successful shipment creation, it returns a set of rate for various services i.e. </span><span style="font-size: 11pt; font-family: Arial; vertical-align: baseline; ">First Class, </span><span style="font-size: 11pt; font-family: Arial; vertical-align: baseline;">Priority, etc.</span></p> 
                          <br> 
                          <p dir="ltr" style="line-height:1.38;margin-top:0pt;margin-bottom:0pt;"> <span style="font-size: 11pt; font-family: Arial; font-weight: 700; vertical-align: baseline; background-color: transparent;">Steps 4 : </span><span style="font-size: 11pt; font-family: Arial; vertical-align: baseline; background-color: transparent;">Buy shipment for particular rate and get shipping label link.</span></p> 
                          <br> 
                          <p dir="ltr" style="line-height:1.38;margin-top:0pt;margin-bottom:0pt;"> <span style="font-size: 11pt; font-family: Arial; font-weight: 700; vertical-align: baseline; background-color: transparent;">Steps 5 : </span><span style="font-size: 11pt; font-family: Arial; vertical-align: baseline; background-color: transparent;">Download shipping label from the provided link, print it out and put it on your package. It also provide tracking ID with shipping label link to track package.</span></p> 
                          <p dir="ltr" style="line-height:1.38;margin-top:0pt;margin-bottom:0pt;"> &nbsp;</p> 
                          <p dir="ltr" style="line-height:1.38;margin-top:0pt;margin-bottom:0pt;"> <span style="font-family: Arial; font-size: 11pt; line-height: 17px; background-color: transparent;">Below is the sample JAVA program for the same.</span></p>
