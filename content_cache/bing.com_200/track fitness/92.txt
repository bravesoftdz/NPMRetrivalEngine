           <h1>FedEx Web Services</h1> 
           <p><a id="top" class="bookmark" title="top" name="top"></a>With FedEx Web Services, you can integrate FedEx Express<sup>®</sup>, FedEx Ground<sup>®</sup> and FedEx Freight<sup>®</sup> shipping services into your business systems, retail website or order management system. FedEx Web Services is the next generation shipping API for integrating software applications with FedEx Systems to create shipping labels, facilitate returns, track shipments, obtain rate quotes and generate reports. </p> 
          </div> 
         </div> 
        </div>
       </div>
       <div class="ls-cmp-wrap" id="w1354142655151">
        <div class="iw_component" id="1354142655151">
         <div class="block basic"> 
          <div class="copy fx_clearfix"> 
           <h2>What are Web services?</h2> 
           <p></p>
           <p>Essentially, they are a collection of programming technologies that allows a user to build customized solutions for application integration.<br><br> Web services differ from traditional client/server models in that they allow for application-to-application communication. In other words, applications interface with each other, not with the users. This means applications can be combined to create more powerful services. For example, a user could choose to combine FedEx Web Services with Web services from other organizations to create unique, customized solutions.</p> 
           <p>&nbsp;</p>
           <p></p> 
           <p></p> 
          </div> 
         </div> 
        </div>
       </div>
       <div class="ls-cmp-wrap" id="w1354142655169">
        <div class="iw_component" id="1354142655169">
         <div class="block basic"> 
          <div class="copy fx_clearfix"> 
           <h2>Features</h2> 
           <p>Web services represent the future of application integration because they offer the following:<br><br><strong>Inter-Operability</strong><br> Any Web service can interact with any other Web service and can be written in any programming language. <br><br><strong>Compatibility</strong><br> Web services communicate using HTTP and XMLso any connected device that supports these technologies can both host and access Web Services. <br><br><strong>Low Barrier to Entry</strong><br> The concepts behind Web services are easy to understand, and developers can quickly create and deploy them using many tool-kits available on the Web. <br><br><strong>Industry Support</strong><br> Major content providers and vendors are supporting the Web services movement. </p> 
           <p></p> 
          </div> 
         </div> 
        </div>
       </div>
       <div class="ls-cmp-wrap" id="w1354142655170">

         <div class="block basic"> 
          <div class="copy fx_clearfix"> 
           <h2>Benefits</h2> 
           <p>From a business and product planning standpoint, FedEx Web Services is an ideal choice for integration. FedEx Web Services are designed to achieve the following: <br><br><strong>Streamline Operations</strong></p>
           <ul> 
            <li>Integrate common FedEx functionality into your business workflow, including the ability to locate the best rates, estimate transit times, and track shipments.</li> 
            <li>Save time by automating common processes like rate and service selection, generating labels, and exchanging data between information sources</li> 
           </ul> 
           <br>
           <br>
           <strong>Industry Standard</strong>
           <ul> 
            <li>FedEx Web Services conforms to industry standards and is compatible with a comprehensive array of developer's tools to provide faster time to market and increased flexibility to integrate FedEx transactions and information into your applications.</li> 
            <li>FedEx WSDLs are designed to be fully inter-operable with any product or developer's tool that also conforms to the WS-I Basic Profile. For more information, view <a href="http://www.ws-i.org/Profiles/BasicProfile-1.1-2004-08-24.html">Web Services Inter-Operability</a>.</li> 
           </ul> 
           <br>
           <br>
           <strong>A Wealth of Resources</strong>
           <ul> 
            <li>FedEx Web Services <a href="/us/developer">Developer Resource Centre</a> (DRC) offers developers a secure environment to access the resources necessary for integration-from product planning guidance, documentation and sample code all the way to testing and production. All the information a developer needs to complete a FedEx integration project in record time can be found at the DRC.</li> 
           </ul>
