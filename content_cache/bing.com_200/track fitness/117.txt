        <div class="section" id="id47"> 
         <h1>Hello World<a class="headerlink" href="#id47" title="Permalink to this headline">¶</a></h1> 
         <p>This article discusses the JavaScript sample application included with the Leap SDK. After reading this article, you should be ready to access Leap Motion hand tracking data from your own Web applications.</p> 
         <p>In the Leap SDK folder, you can find the following file used for this article:</p> 
         <ul class="simple"> 
          <li><tt class="docutils literal"><span class="pre">LeapSDK/samples/Sample.html</span></tt> — JavaScript sample application</li> 
         </ul> 
         <p>You will also need the Leap JavaScript client library available from <a class="reference external" href="https://github.com/leapmotion/leapjs">Leap Motion’s LeapJS repository on GitHub</a>.</p> 
         <div class="section" id="id48"> 
          <h2>Overview<a class="headerlink" href="#id48" title="Permalink to this headline">¶</a></h2> 
          <p>In a nutshell, the Leap Motion tracking device detects and tracks hands and fingers placed within its field of view. The Leap captures this data one <em>frame</em> at a time. Web applications can use the Leap Motion JavaScript API to access this data. The Leap Motion controller sends tracking information through the socket connection as a JSON formatted message. The JavaScript API takes the JSON message and evaluates it into proper objects.</p> 
          <p>The sample application demonstrates how to use the Leap Motion JavaScript API. The example is contained in a single Web page, <tt class="docutils literal"><span class="pre">Sample.html</span></tt>. The application displays several properties from the key tracking data objects in the Leap Motion API, including:</p> 
          <ul class="simple"> 
           <li><a class="reference external" href="../api/Leap.Frame.html"><tt class="docutils literal"><span class="pre">Frame</span></tt></a> — contains a set of hand and pointable tracking data</li> 
           <li><a class="reference external" href="../api/Leap.Hand.html"><tt class="docutils literal"><span class="pre">Hand</span></tt></a> — contains tracking data for a detected hand</li> 
           <li><a class="reference external" href="../api/Leap.Pointable.html"><tt class="docutils literal"><span class="pre">Pointable</span></tt></a> — contains tracking data for a detected finger</li> 
          </ul> 
         </div> 
         <div class="section" id="javascript-client-library"> 
          <h2>JavaScript client library<a class="headerlink" href="#javascript-client-library" title="Permalink to this headline">¶</a></h2> 
          <p>The Leap Motion JavaScript client library is available from <a class="reference external" href="https://github.com/leapmotion/leapjs">Leap Motion’s LeapJS repository on GitHub</a>. The library handles the connection to the Leap Motion service or daemon and passes frames of tracking data to your application. LeapJS is an open-source project and distributed separately from the main Leap SDK. We encourage community contributions to this library.</p> 
         </div> 
         <div class="section" id="setting-up-the-page"> 
          <h2>Setting up the page<a class="headerlink" href="#setting-up-the-page" title="Permalink to this headline">¶</a></h2> 
          <p>To use the JavaScript API, you must include the <tt class="docutils literal"><span class="pre">leap.js</span></tt> file (or the minified version, <tt class="docutils literal"><span class="pre">leap.min.js</span></tt>). The <tt class="docutils literal"><span class="pre">Sample.html</span></tt> file uses a simple HTML structure to display the Leap data and include the <tt class="docutils literal"><span class="pre">leap.js</span></tt> file in the usual way using script tags:</p> 
          <div class="code html highlight-python">
           <div class="highlight">
            <pre>&lt;html&gt;
  &lt;head&gt;
    &lt;title&gt;Leap JavaScript Sample&lt;/title&gt;
    &lt;script src="leap.js"&gt;&lt;/script&gt;
    &lt;script&gt;// JavaScript code goes here&lt;/script&gt;
  &lt;/head&gt;
  &lt;body onload="checkLibrary()"&gt;
    &lt;h1&gt;Leap JavaScript Sample&lt;/h1&gt;
    &lt;div id="main"&gt;
      &lt;h3&gt;Frame data:&lt;/h3&gt;
      &lt;div id="frameData"&gt;&lt;/div&gt;
      &lt;div style="clear:both;"&gt;&lt;/div&gt;
      &lt;h3&gt;Hand data:&lt;/h3&gt;
      &lt;div id="handData"&gt;&lt;/div&gt;
      &lt;div style="clear:both;"&gt;&lt;/div&gt;
      &lt;h3&gt;Finger data:&lt;/h3&gt;
      &lt;div id="pointableData"&gt;&lt;/div&gt;
      &lt;div style="clear:both;"&gt;&lt;/div&gt;
    &lt;/div&gt;
  &lt;/body&gt;
&lt;/html&gt;
</pre>
           </div> 
          </div> 
          <p>To keep things simple, all the example code in the example is in the main HTML file.</p> 
         </div> 
         <div class="section" id="setting-up-the-event-loop"> 
          <h2>Setting up the event loop<a class="headerlink" href="#setting-up-the-event-loop" title="Permalink to this headline">¶</a></h2> 
          <p>Since the Leap Motion controller provides data continuously, you need to set up an event loop to handle individual frames of data. You can either set up a loop that handles the data as the Leap Motion controller produces each frame, or you can set up a loop that handles the data when your application is ready to do something with it. The Leap Motion JavaScript client library supports both options.</p> 
          <p>This example demonstrates the second method since there is no need in this app to process the data faster than the browser can draw it to the screen. The Leap Motion API provides the <tt class="docutils literal"><span class="pre">Leap.loop()</span></tt> function, which invokes a callback function you provide whenever the browser is ready to draw. In this sample, the callback is an anonymous function that prints out the tracking data to the body of the web page. The first parameter of the <tt class="docutils literal"><span class="pre">loop()</span></tt> function contains optional settings parameters that are passed to the <a class="reference external" href="../api/Leap.Controller.html"><tt class="docutils literal"><span class="pre">Controller</span></tt></a> object. This is the skeleton of the function:</p> 
          <div class="code js highlight-python">
           <div class="highlight">
            <pre>// Setup Leap loop with frame callback function
var controllerOptions = {};

Leap.loop(controllerOptions, function(frame) {
  // Body of callback function
})
</pre>
           </div> 
          </div> 
          <p>Internally, <tt class="docutils literal"><span class="pre">Leap.loop()</span></tt> uses the browser’s <tt class="docutils literal"><span class="pre">requestAnimationFrame()</span></tt> feature and passes the latest frame of Leap Motion data to the application on each browser animation cycle.</p> 
         </div> 
         <div class="section" id="accessing-tracking-data"> 
          <h2>Accessing tracking data<a class="headerlink" href="#accessing-tracking-data" title="Permalink to this headline">¶</a></h2> 
          <p>The <a class="reference external" href="../api/Leap.Frame.html"><tt class="docutils literal"><span class="pre">Frame</span></tt></a> object passed to the <tt class="docutils literal"><span class="pre">loop()</span></tt> callback function as the <tt class="docutils literal"><span class="pre">frame</span></tt> parameter is root of the tracking data model and provides access to all the tracked objects, like hands and fingers. To display the data, the example creates a string of HTML containing the object properties and assigns the string to a <tt class="docutils literal"><span class="pre">div</span></tt> element on the page.</p> 
          <div class="section" id="frame-data"> 
           <h3>Frame data<a class="headerlink" href="#frame-data" title="Permalink to this headline">¶</a></h3> 
           <p>Displaying the basic frame data is a simple matter of constructing a string using the <a class="reference external" href="../api/Leap.Frame.html"><tt class="docutils literal"><span class="pre">Frame</span></tt></a> object properties:</p> 
           <div class="code js highlight-python">
            <div class="highlight">
             <pre>var frameString = "Frame ID: " + frame.id  + "&lt;br /&gt;"
                + "Timestamp: " + frame.timestamp + " &amp;micro;s&lt;br /&gt;"
                + "Hands: " + frame.hands.length + "&lt;br /&gt;"
                + "Fingers: " + frame.fingers.length + "&lt;br /&gt;";
</pre>
            </div> 
           </div> 
          </div> 
          <div class="section" id="hand-data"> 
           <h3>Hand data<a class="headerlink" href="#hand-data" title="Permalink to this headline">¶</a></h3> 
           <p>Hands can be accessed from the <a class="reference external" href="../api/Leap.Frame.html"><tt class="docutils literal"><span class="pre">Frame</span></tt></a> object using the <tt class="docutils literal"><span class="pre">hands</span></tt> property, which is an array. The example code iterates through the list of hands and prints out the <a class="reference external" href="../api/Leap.Hand.html"><tt class="docutils literal"><span class="pre">Hand</span></tt></a> object properties, IDs for associated fingers, and the motion data for each hand.</p> 
           <div class="code js highlight-python">
            <div class="highlight">
             <pre>// Display Hand object data
var handString = "";
if (frame.hands.length &gt; 0) {
  for (var i = 0; i &lt; frame.hands.length; i++) {
    var hand = frame.hands[i];

    handString += "Hand ID: " + hand.id + "&lt;br /&gt;";
    handString += "Direction: " + vectorToString(hand.direction, 2) + "&lt;br /&gt;";
    handString += "Palm normal: " + vectorToString(hand.palmNormal, 2) + "&lt;br /&gt;";
    handString += "Palm position: " + vectorToString(hand.palmPosition) + " mm&lt;br /&gt;";
    handString += "Palm velocity: " + vectorToString(hand.palmVelocity) + " mm/s&lt;br /&gt;";
    handString += "Sphere center: " + vectorToString(hand.sphereCenter) + " mm&lt;br /&gt;";
    handString += "Sphere radius: " + hand.sphereRadius.toFixed(1) + " mm&lt;br /&gt;";

    // And so on...
  }
}
</pre>
            </div> 
           </div> 
          </div> 
          <div class="section" id="finger-data"> 
           <h3>Finger data<a class="headerlink" href="#finger-data" title="Permalink to this headline">¶</a></h3> 
           <p>Fingers are represented by <a class="reference external" href="../api/Leap.Pointable.html"><tt class="docutils literal"><span class="pre">Pointable</span></tt></a> objects in the JavaScript API. The <a class="reference external" href="../api/Leap.Frame.html"><tt class="docutils literal"><span class="pre">Frame</span></tt></a> object provides a <tt class="docutils literal"><span class="pre">pointables</span></tt> property, which is an array containing all the fingers detected in that frame. <a class="reference external" href="../api/Leap.Hand.html"><tt class="docutils literal"><span class="pre">Hand</span></tt></a> objects also contain a <tt class="docutils literal"><span class="pre">pointables</span></tt> list, which contains only the pointable objects representing the fingers of that hand.</p> 
           <div class="code js highlight-python">
            <div class="highlight">
             <pre>// Display Pointable (finger) object data
var pointableString = "";
if (frame.pointables.length &gt; 0) {
  for (var i = 0; i &lt; frame.pointables.length; i++) {
    var pointable = frame.pointables[i];

    pointableString += "Pointable ID: " + pointable.id + "&lt;br /&gt;";
    pointableString += "Belongs to hand with ID: " + pointable.handId + "&lt;br /&gt;";
    pointableString += "Length: " + pointable.length.toFixed(1) + " mm&lt;br /&gt;";
    pointableString += "Width: "  + pointable.width.toFixed(1) + " mm&lt;br /&gt;";
    pointableString += "Direction: " + vectorToString(pointable.direction, 2) + "&lt;br /&gt;";
    pointableString += "Tip position: " + vectorToString(pointable.tipPosition) + " mm&lt;br /&gt;";
    pointableString += "Tip velocity: " + vectorToString(pointable.tipVelocity) + " mm/s&lt;br /&gt;";
  }
}
</pre>
            </div> 
           </div> 
          </div> 
         </div> 
         <div class="section" id="id49"> 
          <h2>Running the sample<a class="headerlink" href="#id49" title="Permalink to this headline">¶</a></h2> 
          <p>To run the sample application:</p> 
          <ol class="arabic"> 
           <li><p class="first">Plug the Leap device into a USB port and place it in front of you.</p> </li> 
           <li><p class="first">If you haven’t already, install the Leap Motion software.</p> </li> 
           <li><p class="first">The Leap Motion software should start automatically.</p> <p>The Leap Motion control panel icon appears in the notification area of the task bar (on Windows) or finder bar (on Mac) and turns green when ready. A service or daemon runs in the background and provides data to client applications through the Leap Motion API. You can use the <a class="reference internal" href="../supplements/Leap_Visualizer.html"><em>diagnostic visualizer</em></a> to check whether the software is set up and running correctly.</p> </li> 
           <li><p class="first">Open Sample.html in a browser that supports WebSockets.</p> </li> 
          </ol> 
          <p>When you place a hand above the Leap, you should see the Frame, Hand, and Finger information printed out on the page.</p> 
          <p>Now that you have seen how to access motion tracking data with the Leap Motion API, you can begin developing your own JavaScript applications that integrate the Leap Motion controller.</p>
