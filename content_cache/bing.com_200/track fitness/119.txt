         <h1 class="entry-title articles">Tracking Website Statistics with PHP</h1>
        </div>
        <div class="entry-meta col-quarter accent-type articles-bg-light-small"> 
         <span class="posted-on">Posted on <a href="http://www.devshed.com/c/a/php/tracking-website-statistics-with-php/" title="9:00 am" rel="bookmark"><time class="entry-date published" datetime="2006-09-12T09:00:47+00:00">September 12, 2006</time></a></span>
         <span class="byline"> by <span class="author vcard"><a class="url fn n" href="http://forums.devshed.com/vbsso/vbsso.php?do=profile&amp;id=81b6ff3f8a36b656f309120730f23af6" title="View all posts by JacquesNoah">JacquesNoah</a></span></span>

        <div class="col-half">
          If you have a website, especially one that sells products or services, it can be useful to track the visitors to your site. This article explains how to write a program that will do just that.
         <br>
         <span id="more-816"></span>
         <br>
         <p><strong>Introduction</strong></p>
         <p>In this article we are going to discuss how to create a program that will track the visitors to your site. We are going to do this by:</p>
         <ul style="margin-top: 0in;" type="disc">
          <li>Recording the visitors’ IP address.</li>
          <li>Recording the time and&nbsp;date of the visit.</li>
          <li>Recording the pages the visitor viewed.</li>
          <li>Recording the name of the browser the&nbsp;visitor is using.</li>
          <li>Recording the&nbsp;page the person came from.</li>
         </ul>
         <p>All this will be recorded for every page on a website, simply by including the code that we are going to create in a moment.</p>
         <p>All this information is going to give us the ability to know which pages are viewed the most and which ones the least. It will also let us know what pages a particular visitor views the most, by means of the visitor’s IP address.</p>
         <p><strong>The Database Table</strong></p>
         <p>Create a database and give it whatever name you like. We now need to create a table. We need the following data:</p>
         <p><b>IP address</b> -Visitor’s IP address</p>
         <p><b>Date_visited</b></p>
         <p><b>Page</b>– Page visited</p>
         <p><b>Browser</b>-Name of the browser</p>
         <p><b>frompage</b>– Page from which the person came</p>
         <p>With the above in mind, let’s create the table:</p>
         <p><font face="Courier">CREATE TABLE `statTracker` (<br> </font><span lang="EN-GB"><font face="Courier">&nbsp; `id` int(11) NOT NULL auto_increment,<br> </font></span><span lang="EN-GB"><font face="Courier">&nbsp; `browser` varchar(255) NOT NULL default ”,<br> </font></span><span lang="EN-GB"><font face="Courier">&nbsp; `ip` varchar(15) NOT NULL default ”,<br> </font></span><span lang="EN-GB"><font face="Courier">&nbsp; `thedate_visited` date NOT NULL default ‘0000-00-00’,<br> </font></span><span lang="EN-GB"><font face="Courier">&nbsp; `page` varchar(70) NOT NULL default ”,<br> </font></span><span lang="EN-GB"><font face="Courier">&nbsp; `from_page` varchar(200) NOT NULL default ”,<br> </font></span><font face="Courier"><span lang="EN-GB">&nbsp; PRIMARY KEY&nbsp; (`id`)<br> </span><span lang="EN-GB">) TYPE=MyISAM AUTO_INCREMENT=8 ;</span></font></p>
         <p>The table should be very easy for you to understand. Now, without further ado, let’s&nbsp;create the core tracking script. This is the script that you should attach to every page on your site:</p>
         <p><b>Script: </b>theCollector.php</p>
         <p><font face="Courier"><span lang="EN-GB">&lt;?<br> </span>//collect information…<br> </font><span lang="EN-GB"><font face="Courier">$browser&nbsp; =$_SERVER[‘HTTP_USER_AGENT’]; // get the browser name<br> </font></span><span lang="EN-GB"><font face="Courier">$curr_page=$_SERVER[‘PHP_SELF’];// get page name<br> </font></span><span lang="EN-GB"><font face="Courier">$ip&nbsp; =&nbsp; $_SERVER[‘REMOTE_ADDR’];&nbsp;&nbsp; // get the IP address<br> </font></span><span lang="EN-GB"><font face="Courier">$from_page = $_SERVER[‘HTTP_REFERER’];//&nbsp; page from which visitor<br> came<br> </font></span><font face="Courier"><span lang="EN-GB">$page=$_SERVER[‘PHP_SELF’];//get current page<br> </span>//Insert the data in the table…<br> </font><span lang="EN-GB"><font face="Courier">$query_insert&nbsp; =”INSERT INTO stattracker<br> (browser,ip,date_visited,page,from_page) VALUES<br> (‘$browser’,’$ip’,now(),’$page’,’$from_page’)” ;<br> </font></span><span lang="EN-GB"><font face="Courier">$result=mysql_query ( $query_insert); <br> </font></span><span lang="EN-GB"><font face="Courier">if(!$result){<br> </font></span><font face="Courier"><span lang="EN-GB">die(mysql_error());<br> </span><span lang="EN-GB">}</span></font></p>
         <p><font face="Courier">//remove this section when attaching the script to a webpage<br> //I include it only because of it’s debug value.<br> $query=”Select Count(*) from stattracker WHERE page = ‘$curr_page'”;<br> $result=mysql_query($query);<br> $viewed = mysql_result($result,0,’count(*)’);<br> echo “This page was viewed $viewed times”;<br> <span lang="EN-GB">?&gt;</span></font></p>
         <p>This script includes two sections, one to collect the data we require, and one to insert the collected data into the database.</p>
         <p>There are only a few points about the code worth explaining. The “$_SERVER[]” is a server array that contains a lot of useful variables. You can determine the host name, remote port or even the scriptname.</p>
         <p>The Count(*) retrieves all the rows in the table that meets the condition. In the&nbsp;<span lang="EN-GB">&nbsp;“$viewed = mysql_result($result,0,’count(*)’)” line the “0” refers to the row; at this point there is no row in the table since we have not run the script yet.</span></p>
         <p>{mospagebreak title=Analyzing the Recorded Data}</p>
         <p>From the information collected above, we can divide the data into three main specific information areas:</p>
         <p>Website Info:&nbsp; How many visitors, browsers used, and pages visited.</p>
         <p>Visitor Info: Pages visited, browsers used, and date of visit.</p>
         <p><span lang="EN-GB">Page&nbsp;Info&nbsp;:&nbsp;Who visited a particular page, how many times, when and browser used.</span></p>
         <p>To get this information all we have to do is run some very specific SQL queries:</p>
         <p><b>Website Specific Analysis</b></p>
         <p><i>To count how many people visited the site, run the following code:</i></p>
         <p><font face="Courier">$query = “SELECT COUNT(*) FROM stattracker GROUP by ip”;<br> </font><font face="Courier"><span lang="EN-GB">$result=mysql_query($query);<br> </span><span lang="EN-GB">$number_of_views = mysql_num_rows($result);</span></font></p>
         <p>This will count all the different IP addresses in the table. If one person with the same IP address views this site a hundred times, the result of this query will be one; the&nbsp;same thing happens if the person views this site two hundred times.</p>
         <p><i>Number of times a particular browser has been used to visit the site:</i></p>
         <p><span lang="EN-GB"><font face="Courier">$query = “SELECT *,count(*) FROM stattracker GROUP by browser”;<br> </font></span><span lang="EN-GB"><font face="Courier">$result=mysql_query($query);<br> </font></span><span lang="EN-GB"><font face="Courier">while ($row =mysql_fetch_array($result)) {<br> </font></span><font face="Courier"><span lang="EN-GB">echo “browser &nbsp;name = ” .$row[‘browser’]. ” Used: ” .$row [‘count<br> (*)’];<br> </span><span lang="EN-GB">}</span></font></p>
         <p>The above query checks how many time a particular browser has been used to visit the site.</p>
         <p>{mospagebreak title=Visitor Specific Analysis}</p>
         <p>Number of times user visited:</p>
         <p><span lang="EN-GB"><font face="Courier">$query = “SELECT count(*) FROM stattracker WHERE ip =<br> ‘127.0.0.1’”;<br> </font></span><span lang="EN-GB"><font face="Courier">$result=mysql_query($query);<br> </font></span><span lang="EN-GB"><font face="Courier">while ($row =mysql_fetch_array($result)) {<br> </font></span><font face="Courier"><span lang="EN-GB">echo “Visited ” .$row[‘count(*)’]. ” times”;<br> </span><span lang="EN-GB">}</span></font></p>
         <p>Here I’m checking how many times a particular user visited the site. In this case the visitor is identified by their IP address.</p>
         <p><i>Pages user visited</i></p>
         <p><span lang="EN-GB"><font face="Courier">$query = “SELECT *,count(*) FROM stattracker WHERE ip = <br> ‘127.0.0.1’ GROUP BY page”;<br> </font></span><span lang="EN-GB"><font face="Courier">$result=mysql_query($query);<br> </font></span><span lang="EN-GB"><font face="Courier">while ($row =mysql_fetch_array($result)) {<br> </font></span><font face="Courier"><span lang="EN-GB">echo “Page: ” .$row[‘page’]. “Viewed: “.$row[‘count(*)’]. “&lt;br&gt;”;<br> </span><span lang="EN-GB">}</span></font></p>
         <p><i>Browser used when visiting</i></p>
