         <ul>
          <li>Hashtable</li>
          <ul>
           <li>Data structure that uses hashing</li>
           <ul>
            <li>Algorithm for determining a key in table</li>
            <ul>
             <li>Keys in tables have associated values (data)</li>
            </ul>
           </ul>
           <li>Each table cell is a hash “bucket”</li>
           <ul>
            <li>Linked list of all key-value pairs that hash to that cell</li>
            <li>Minimizes collisions (handled via hash again, next cell, buckets)</li>
           </ul>
           <li>Hashtable() creates empty hashtable with capacity 11 and load factor .75 (table grows larger automatically to meet load factor requirement)</li>
          </ul>
         </ul>
