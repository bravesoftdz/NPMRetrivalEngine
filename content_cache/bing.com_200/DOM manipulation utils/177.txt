        <h1>CRM Manipulation Library (Custom Workflow Activities &amp; Plugins)</h1> 
        <a href="http://manipulationlibrary.codeplex.com/releases/view/52128">Click here to go to the CRM 2011 Beta Workflow Library</a>.
        <br> 
        <br> The CRM Manipulation Library is a set of custom workflow activities for OnPremise Microsoft Dynamics® CRM to solve equations, date calculations, manipulates strings, perform regex (regular expression) formatting &amp; matching, as well as SoundEx and Metaphone-Like codification. It also includes a plugin for CRM Online and CRM 2011 OnPremise that incorporates SoundEx and Metaphone-Like codification.
        <br> 
        <br> The CRM Workflow Manipulation Library was written by Carlton Colter and 
        <a href="http://www.engage2day.com"> Engage</a>. Andrew Swerlick blogged about the Manipulation Library on 
        <a href="http://blogs.engage2day.com/2010/09/new-microsoft-dynamics-crm-workflow-manipulation-library-on-codeplex/"> Engage's Tech Blog</a> and his words sum up the core of the Manipulation Library.
        <br> 
        <br> 
        <blockquote>
         "Microsoft Dynamics CRM’s built in workflow engine is pretty powerful, as anyone who’s worked with it for a while is aware. However, in our experience, a good customer quickly comes up with requirements that go beyond it’s built in capabilities, requiring custom activities. Engage is used to writing some basic activities to handle common problems, but now thanks the Carlton Colter of Microsoft, there’s a robust library of basic workflow activities up on codeplex for all CRM customers and partners."
        </blockquote>
