  <h3>Signal Analysis/Processing Software</h3> 
  <p> </p>
  <ul> 
   <li><b>General Analysis/Processing</b> 
    <ul> 
     <li><a href="http://thermal.cnde.iastate.edu/~sdh4/aglaophone/">Aglaophone</a> "...a system of interconnectable modules for the recording, processing, and playback of real-time audio" </li>
     <li><a href="http://musica.unq.edu.ar/personales/odiliscia/software.htm">ATSH</a> very cool analysis/resynthesis program from Prof. Oscar Pablo Di Liscia </li>
     <li><a href="http://clam-project.org/">CLAM</a> the outstanding "C++ Library for Audio and Music " from the crew at <a href="http://www.upf.edu/">UPF</a> </li>
     <li><a href="http://kmt.hku.nl/~pieter/SOFT/CMP/doc/cmp.html">ComParser</a> computer software that recognizes and identifies realtime audio </li>
     <li><a href="http://www.dataplore.de/">Dataplore</a> "a software tool designed for the analysis of signals and time series data of any kind" </li>
     <li><a href="http://flowdesigner.sourceforge.net/">FlowDesigner</a> interesting visual environment for building audio and signal processing applications </li>
     <li><a href="http://sourceforge.net/projects/fscape/">FScape</a> "audio rendering software for time domain and spectral signal processing...similar to <b>SoundHack</b>", requires <b>Java</b> </li>
     <li><a href="http://freeadsp.sourceforge.net/">FreeADSP</a> "free, audio-oriented, real-time, cross-platform DSP software" from Stefano D'Angelo </li>
     <li><a href="http://webast.ast.obs-mip.fr/people/fbracher/">Guimauve</a> "...is a graphical user interface to compute matching pursuit decomposition on a signal", from Fabien Brachere </li>
     <li><a href="http://hasas.sf.net">HASAS</a> is the HydroAcoustic Signal Analysis System from Jussi Laako </li>
     <li><a href="http://www.scrime.u-bordeaux.fr/InSpect/">InSpect</a> sound analysis/resynthesis/transformation from Sylvain Marchand </li>
     <li><a href="http://www.kokkinizita.net/linuxaudio">JAAA</a> the <i>JACK &amp; ALSA Audio Analyser</i>, an audio signal generator and spectrum analyser from Fons Adriaensen </li>
     <li><a href="http://peloterodigital.com.ar/anthony/index.php?q=ksfx">KSFX</a> is "... a kernel module that filters the outgoing audio stream and adds sound effects like low-pass, high-pass, echos...", from Anthony Lenton </li>
     <li><a href="http://www.cmap.polytechnique.fr/~bacry/LastWave/">LastWave</a> wavelet DSP software from Emmanuel Bacry </li>
     <li><a href="http://www.cmis.csiro.au/maaate/">Maaate</a> an analysis toolkit for MPEG audio files; <a href="http://www.cmis.csiro.au/maaate/bewdy/index.html">Bewdy</a> is a visualization GUI for <b>Maaate</b> </li>
     <li><a href="http://www.tml.hut.fi/~tilmonen/mustajuuri/index.html">Mustajuuri</a> audio signal processing software with GUI and support for plug-ins </li>
     <li><a href="http://www.mtnmath.com/software/opdsp.html">ObjectProDSP</a> an X11 GUI for DSP design and implementation </li>
     <li><a href="http://www.cf.ac.uk/psych/CullingJ/">PipeWave</a> DSP and "auditory modelling" programs using UNIX pipes </li>
     <li><a href="http://ptolemy.berkeley.edu/">Ptolemy</a> homepage for the Ptolemy Project of the DSP Design Group at <a href="http://www.berkeley.edu">UCB</a> </li>
     <li><a href="http://www.scilab.org">Scilab</a> a large package of mathematics &amp; signal processing functions with excellent graphic displays </li>
     <li><a href="http://satan.sourceforge.net/">Signal Applications To Audio Networks (SATAN)</a> "a program that is used for various audio related signal processing tasks" </li>
     <li><a href="http://ems.music.uiuc.edu/beaucham/software/sndan/">SNDAN</a> signal analysis, graphics, modification, and synthesis </li>
     <li><a href="http://sonicflow.sourceforge.net">Sonic Flow</a> designs/simulates audio signal processing networks </li>
     <li><a href="http://www.sonicvisualiser.org/">Sonic Visualiser</a> advanced waveform/spectrogram viewer and audio annotation editor, from Chris Cannam </li>
     <li><a href="http://soundruler.sourceforge.net/">Sound Ruler</a> acoustic analysis software from Marcos Gridi Papp </li>
     <li><a href="http://saajoby.sourceforge.net/">SSPLab AudioAnalyser (SAA)</a> is "... a tool for analyzing audio signals in terms of its content" </li>
     <li><a href="http://yorick.sourceforge.net/index.php">Yorick</a> another mathematics package with signal processing and data visualization tools </li>
     <li><a href="http://www.ibiblio.org/pub/Linux/science/lab">dsplab</a> <b>XForms</b>-based DSP visualization and manipulation software </li>
     <li><a href="http://eamusic.dartmouth.edu/~paul/jein/">jein</a> a <b>Java</b> version of <b>Ein</b>, DSP learning software from Paul Lansky and Ken Steiglitz </li>
    </ul> <p> </p></li>
   <li><b>Beat-tracking</b> 
    <ul> 
     <li><a href="http://www.ofai.at/~simon.dixon/beatroot/index.html">BeatRoot</a> "an interactive beat tracking and visualisation system", from Simon Dixon </li>
    </ul> <p> </p></li>
   <li><b>Binaural/Brainwave</b> 
    <ul> 
     <li><a href="http://www.linuxlabs.com/autozen.shtml">AutoZen</a> brainwave modification software </li>
     <li><a href="http://gnaural.sourceforge.net/">Gnaural</a> binaural-beat generator, from gnaural (who else?) </li>
     <li><a href="http://uazu.net/sbagen/">SBaGen</a> a "BinAural Brain Wave Experimenter's Lab" from Jim Peters </li>
    </ul> <p> </p></li>
   <li><b>Channel Vocoder</b> 
    <ul> 
     <li><a href="http://www.sirlab.de/linux/descr_vocoder.html">Vocoder</a> a channel vocoder from Achim Settelmeier </li>
     <li><a href="http://www.archive.org/details/tucows_163067_Xvox">XVOX</a> realtime vocoder using LPC synthesis </li>
     <li><a href="http://www.epiphyte.ca/code/vocoder.html">Zerius Vocoder</a> C and <b>Java</b> source code for software channel vocoder </li>
    </ul> <p> </p></li>
   <li><b>Convolution</b> 
    <ul> 
     <li><a href="http://www.ludd.luth.se/~torger/almusvcu.html">Almus VCU</a> "... makes a computer equipped with a multi-channel sound card into a real-time versatile convolver unit" </li>
     <li><a href="http://www.kokkinizita.net/linuxaudio">JACE</a> <b>JACK</b>-based convolution engine from Fons Adriaensen </li>
    </ul> <p> </p></li>
   <li><b>FFT/Phase Vocoder</b> 
    <ul> 
     <li><a href="http://skruntskrunt.abez.ca/">Enveloper</a> draw FFT components and amplitude waveforms with this <b>Java</b>-based software from Abram Hindle </li>
     <li><a href="http://freqtweak.sourceforge.net/">FreqTweak</a> Jesse Chappell's very neat "tool for FFT-based realtime audio spectral manipulation and display" </li>
     <li><a href="http://gwave-analyzer.sourceforge.net/">GWave Analyzer</a> visualizes/analyzes WAV files via FFT, also extracts melody to MIDI file </li>
     <li><a href="http://kissfft.sourceforge.net/">Kiss FFT</a> very small FFT library, can use either fixed or floating point data types </li>
     <li><a href="http://www.notam02.no/notam02/">NoTAM</a> FFT/phase vocoder packages with X11 GUI now available for Linux: 
      <ul> 
       <li><b>Ceres</b> 
        <ul> 
         <li><a href="http://www.notam02.no/arkiv/src/">Ceres</a> a greatly updated version from Kjetil S. Matheussen </li>
         <li><a href="http://www.music.columbia.edu/~stanko/About_Ceres3.html">Ceres3</a> a superb version from Stanko Juzbasic </li>
        </ul> </li>
       <li><a href="http://www.notam02.no/notam02/programarkiv/22">NoTAMProgs</a> the famous Institute has prepared its own sources for Linux </li>
      </ul> </li>
     <li><a href="http://arcana.dartmouth.edu/~eric/">POWERpv</a> spectral manipulation package from Eric Lyon </li>
     <li><a href="http://silvertone.princeton.edu/winham/PPSK/koonce.html">PVC</a> an extensive set of phase vocoder tools and utilities from Paul Koonce, with a Linux version from John Gibson </li>
     <li><a href="http://www.music.princeton.edu/~penrose/soft/">PVNation</a> Christopher Penrose's phase vocoder apotheosis </li>
     <li><a href="http://sculptor.sourceforge.net/sculptor.shtml">Sculptor</a> a very nice suite of realtime FFT tools </li>
     <li><a href="http://www.music.columbia.edu/~stanko/About_SculptTool.html">SculptTool</a> is a "utility which can import, modify, convert, export and synthesize audio analysis files saved in text format", from Stanko Juzbasic; available for Linux and LinuxPPC </li>
     <li><a href="http://www.ibiblio.org/pub/Linux/science/lab/">XMANA</a> a data display program with FFT and other transforms, uses XMAD data acquisition files </li>
     <li><a href="http://quitte.de/dsp/pvoc.html">pvoc</a> "... a collection of <b>LADSPA</b> units and a command line tool for time compression/expansion of sound data making use of the phase-vocoding technique", from Tim Goetze </li>
    </ul> <p> </p></li>
   <li><b>Filters</b> 
    <ul> 
     <li><a href="http://www.ludd.luth.se/~torger/brutefir.html">BruteFIR</a> "...software for applying long FIR filters to multi-channel digital audio...offline or in realtime" </li>
     <li><a href="http://drc-fir.sourceforge.net/">DRC</a> "generates digital room correction FIR filers to be used within HiFi systems in conjunction with real-time convolution engines like <b>BruteFIR</b>" </li>
     <li><a href="http://uazu.net/fiview/">Fiview</a> Jim Peters' software for filter design and viewing </li>
     <li><a href="http://www.nst.ing.tu-bs.de/schaukasten/polezero/en_idx.html">Poles and Zeros</a> a demonstration applet, requires a <b>Java</b>-enabled browser </li>
     <li><a href="http://www.watri.org.au/~ajh/libfilth/">libfilth</a> digital/analog filter design library from Anders Johansson </li>
     <li><a href="http://www.ludd.luth.se/~torger/filter.html">nwfiir audio tools</a> software equalizer and other tools for high resolution digital audio </li>
    </ul> <p> </p></li>
   <li><b>Graphic I/O</b> 
    <ul> 
     <li><a href="http://quickplot.sourceforge.net/about.html">Quickplot</a> is an interactive 2D plotter with support for <b>libsndfile</b> </li>
     <li><a href="http://ieee.uow.edu.au/~daniel/software/spectromatic/">Spectromatic</a> generates PNG spectrograms from audio files </li>
     <li><a href="http://sourceforge.net/projects/waveform/">Waveform</a> draws and exports function graphs for use with audio programs and exports WAV files </li>
     <li><a href="http://dionysos.music.ed.ac.uk/michael/software/rmsps/rmsps.html">rmsps</a> analyzes sound files and creates Encapsulated Postscript graphics of the rms (loudness) curve </li>
    </ul> <p>
