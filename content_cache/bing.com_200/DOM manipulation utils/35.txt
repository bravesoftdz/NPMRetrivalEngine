      <td><b><font color="#7b0018"><font size="+2"> Libraries <a name="libraries"> </a></font><a name="libraries"></a></font><a name="libraries"></a></b><a name="libraries">&nbsp;</a></td> 
      <td><b>General Numerics</b>&nbsp; 
       <ul> 
        <p></p>
        <li> <a href="http://www.apfloat.org/">Apfloat</a> is a arbitrary precision floating-point arithmetic package.&nbsp;</li> 
        <p></p>
        <li> <a href="http://users.belgacombusiness.net/arci">ArciMath BigDecimal</a> is an extension of <tt>java.math.BigDecimal</tt> based on IBM's Java Specification Request.&nbsp;</li> 
        <p></p>
        <li> <a href="http://dsd.lbl.gov/%7Ehoschek/colt/">Colt</a> is a free Java toolkit containing data structures and utilities intended for high performance computing.&nbsp;</li> 
        <p></p>
        <li> <a href="http://jakarta.apache.org/commons/math/">Commons-Math</a> The Jakarta Mathematics Library is is a library of lightweight, self-contained mathematics and statistics components addressing the most common problems not available in the Java programming language.&nbsp;</li> 
        <p></p>
        <li> <a href="http://www.gregdennis.com/drej.html">Drej</a>, an open-source Java library for linear and non-linear least-squares regression and regularized least-squares classification.&nbsp;</li> 
        <p></p>
        <li> A translation of the <a href="http://www.math.utah.edu/%7Ebeebe/software/java/">ELEFUNT Elementary Functions Testing Package</a> has been ported to Java. The collection also contains extensions to java.lang.Math and new classes for numeric output formatting.</li> 
        <p></p>
        <li> IBM's AlphaWorks contains several tools of interest, including 
         <ul> 
          <li><a href="http://www.alphaworks.ibm.com/tech/mathlibrary4java/">library of correctly rounded elementary functions</a> intended for use with Java.&nbsp;</li> 
          <li><a href="http://www.alphaworks.ibm.com/tech/ninja/">Ninja</a>, a set of classes for numerically intensive Java, including complex, multidimensional arrays, and the BLAS. (RETIRED) &nbsp;</li> 
         </ul> </li> 
        <p> </p>
        <li> <a href="http://math.nist.gov/jnt/">Java Numerical Toolkit (JNT)</a>&nbsp;</li> 
        <ul> 
         <li> <a href="http://math.nist.gov/jnt/api/Package-jnt.functions.html">Special functions</a> including Bessel functions, Chebyshev series.&nbsp;</li> 
         <li> <a href="http://math.nist.gov/jnt/api/Package-jnt.nonlinear.ZeroFinder.html">Zero root finder</a>&nbsp;</li> 
         <li> <a href="http://math.nist.gov/jnt/api/Package-jnt.random.html">Random Number generators</a> for uniform and normal sequences of floating point numbers and long integers.&nbsp;</li> 
        </ul> 
        <p></p>
        <li> <a href="http://jump-math.sourceforge.net/">Java Ultimate Math Package</a>, a framework for arbitrary precision computations.</li> 
        <p></p>
        <li> The Java3D effort produced a <a href="http://java.sun.com/products/java-media/3D/forDevelopers/j3dapi/index.html">matrix package</a> for use in graphics.&nbsp;</li> 
        <p></p>
        <li> <a href="http://www.vni.com/products/imsl/jmsl.html">JMSL</a>, a collection of mathematical, statistical and charting classes, written in 100% Java, marketed by Visual Numeric, Inc. Includes linear algebra, zero finding, splines, ordinary differential equations, linear programming, nonlinear optimization, FFTs, special functions, regression, ANOVA, ARMA, Kalman filters.</li> 
        <p> </p>
        <li> <a href="https://github.com/IcmVis/JLargeArrays">JLargeArrays</a>, a pure Java library of one-dimensional numeric arrays that can store up to 2^63 elements.</li> 
        <p> </p>
        <li> <a href="http://jsci.sourceforge.net/">JSci</a>, a set of Java packages for linear algebra, statistics, wavelets. Includes chart/graph components and a partial MathML DOM implementation.</li> 
        <p> </p>
        <li> <a href="http://www.jscience.org/">jScience</a> contains packages for numerical linear algebra, for computing with units (e.g., kg., sec.), and other utility operations. </li> 
        <p> </p>
        <li> <a href="http://www.excelsior-usa.com/jspline/index.html">Jspline+</a> is a spline approximation library for Java developed at the Institute of Computational Mathematics and Mathematical Geophysics (RAS) in Novosibirsk. It contains classes for univariate and multivariate spline approximation on scattered meshes, as well as core matrix and linear system solution classes.</li> 
        <p></p>
        <li> <a href="http://sites.google.com/site/piotrwendykier/software/jtransforms">JTransforms</a>, The first open source, multithreaded FFT library in pure Java. </li> 
        <p></p>
        <li> <a href="http://www.koalog.com/php/jcs.php">Koalog Constraint Solver</a> is a commercial Java library for solving combinatorial optimization problems using Constraint Programming or Local Search.&nbsp;</li> 
        <p></p>
        <li> <a href="http://www.leastsquares.com/">Least Squares Software</a> markets the <i>jCrunch[tm]</i> class libraries for numerical computing.&nbsp;</li> 
        <p></p>
        <li> <a href="http://www.cs.unc.edu/Research/HARPOON/mpjava/">mpjava</a> is a Java implementation of David Bailey's Fortran-based multiprecision package. The package performs multi precision floating point arithmetic with arbitrary precision level. It takes advantage of Java's inheritance facility to provide smooth transition from primitive arithmetic types.&nbsp;</li> 
        <p></p>
        <li> <a href="http://netlib.org/java/">netlib</a> has a small collection of Java numerical classes.&nbsp;</li> 
        <p></p>
        <li> <a href="http://www.crcpress.com/shopping_cart/products/product_detail.asp?sku=C4304"><i>A Numerical Library in Java for Scientists and Engineers</i></a>, a book published by CRC Press, contains a CD-ROM with translation into Java of the library NUMAL (NUMerical procedures in Algol 60).&nbsp;</li> 
        <p></p>
        <li> <a href="http://ojalgo.org/">ojAlgo</a>, an open source pure Java mathematics package containing matrices, complex numbers, and optimization, with links to Jama, Jampack, and Colt. .&nbsp;</li> 
        <p></p>
        <li> <a href="http://opsresearch.com/">OpsResearch</a> has developed a freeware library of more than 450 Operations Research objects, including packages for linear programming, graph-theoretic problems, linear algebra, nonlinear equations, probability and geometry.&nbsp;</li> 
        <p></p>
        <li> The <a href="http://www.functologic.com/orbital/">Orbital library</a> is a class library providing object-oriented representations and algorithms for logic, mathematics and artificial intelligence. It includes theorem proving, computer algebra, search and planning, as well as machine learning algorithms. <p></p></li>
        <li> A set of Java classes from the book <i>Ordinary and Partial Differential Equation Routines in C, C++, Fortran, Java, Maple and Matlab</i> can be obtained from author <a href="http://www.lehigh.edu/%7Ewes1/wes1.html">William Schiesser</a>. <p></p></li>
        <li> <a href=" http://piotr.wendykier.googlepages.com/parallelcolt"> Parallel Colt</a>, An attempt to implement fully multithreaded version of Colt. Still in early development stage. &nbsp;</li> 
        <p></p>
        <li> <a href="http://www.honeylocust.com/RngPack/">RngPack</a>, a pseudorandom number generator library.&nbsp;</li> 
        <p></p>
        <li> <a href="http://www.iro.umontreal.ca/%7Esimardr/ssj/index.html">SSJ</a>, Stochastic Simulation in Java, provides facilities for generating random variates, computing measures related to probability distributions, performing goodness-of-fit tests, applying quasi-Monte Carlo methods, collecting elementary statistics, and programming discrete-event simulations.&nbsp;</li> 
        <p></p>
        <li> <a href="http://www.numericalmethod.com/wiki/numericalmethod/wiki/SuanShu">SuanShu</a>, a large collection of Java classes for basic numerical analysis, statistics, and optimization.&nbsp;</li> 
       </ul> <p><b>Linear Algebra, BLAS</b>&nbsp; </p>
       <ul> 
        <li> <a href="jama/">JAMA</a> (JAva MAtrix package) provides factorizatons (LU, Cholesky, QR, SVD) and symmetric/non-symmetric eigenvalue problems. Jama is a proposed interface standard for dense linear algebra computations. The package has been relesed for comment by the MathWorks and NIST. &nbsp;</li> 
        <p> </p>
        <li> <a href="ftp://math.nist.gov/pub/Jampack/Jampack/AboutJampack.html">Jampack</a> (JAva Matrix PACKage) is a collection of cooperating Java classes for numerical linear algebra. Jampack fully supports complex matrices. Developed by Pete Stewart of the University of Maryland and NIST, Jampack is preliminary and incomplete, but is being released to see if there is sufficient interest to continue its development. <p> </p></li>
        <li> <a href="http://www.netlib.org/java/f2j/">Java Lapack</a> is available from netlib at the University of Tennessee at Knoxville. This code was automatically translated from the Fortran version of the well-know <a href="http://www.netlib.org/lapack/">LAPACK</a> linear algebra package using <a href="http://icl.cs.utk.edu/f2j/">f2j</a>. </li> 
        <p> </p>
        <li> <a href="http://code.google.com/p/la4j/">la4j</a> is a pure Java library for linear algebra that uses many structural and object oriented patterns from the classic GoF book.&nbsp;</li> 
        <p> </p>
        <li> The <a href="http://math.nist.gov/jnt/api/Package-jnt.linear_algebra.html">Linear Algebra</a> component of the Java Numerical Toolkit (JNT) provides LU, QR factorizations, optimized BLAS, and sparse matrices.&nbsp;</li> 
        <p></p>
        <li> <a href="http://code.google.com/p/matrix-toolkits-java/">MTJ</a> is designed to be used as a library for developing numerical applications, both for small and large scale computations. The library is based on BLAS and LAPACK for its dense and structured sparse computations, and on the Templates project for unstructured sparse operations. &nbsp;</li> 
        <p></p>
        <li> <a href="http://opsresearch.com/">OpsResearch</a> has developed a freeware library of more than 450 Operations Research objects which include a complete implementation of the BLAS in both serial and parallel versions.&nbsp;</li> 
        <p> </p>
        <li> The <a href="http://www.cs.rice.edu/%7Ezoran/OwlPack/">OwlPack</a>: a general purpose linear algebra library in polymorphic, object-oriented style Java, based on the standard Fortran LINPACK library. Not meant for high-performance applications. Rather, serves an exercise in writing scientific programs in Java in an elegant, object-oriented style, given sufficient compiler technologies to ameliorate the performance problems&nbsp;</li> 
        <p> </p>
        <li> Steve Verrill developed one of the first Java <a href="http://www1.fpl.fs.fed.us/linear_algebra.html">linear algebra packages</a> from the Fortran BLAS and LINPACK.&nbsp;</li> 
        <p> </p>
        <li> <a href="http://www.ujmp.org/">Universal Java Matrix Package (UJMP)</a> is an open source Java library which provides sparse and dense matrix classes, as well as a large number of calculations for linear algebra like matrix multiplication or matrix inverse. UJMP uses multiple threads for calculations for improved performance on modern hardware. Also supported are visualization methods, import/export filters for various file formats, and interfaces to Matlab, Octave and R.&nbsp;</li> 
        <p> </p>
        <li> The <a href="http://java.sun.com/products/java-media/3D/forDevelopers/j3dapi/javax/vecmath/package-summary.html">vecmath package </a>in the <a href="http://java.sun.com/products/java-media/3D/index.html">Java 3D</a>(TM) API provides general matrix and vector classes (<a href="http://java.sun.com/products/java-media/3D/forDevelopers/j3dapi/javax/vecmath/GMatrix.html">GMatrix</a>, <a href="http://java.sun.com/products/java-media/3D/forDevelopers/j3dapi/javax/vecmath/GVector.html">GVector</a>) along with its specialized matrix classes for graphics applications.</li> 
        <p> </p>
        <li> <a href="http://code.google.com/p/efficient-java-matrix-library/"> EJML</a> is a linear algebra library for manipulating dense matrices. Its design goals are; 1) to be as computationally efficient as possible for both small and large matrices, and 2) to be accessible to both novices and experts. These goals are accomplished by dynamically selecting the best algorithms to use at runtime and by designing a clean API. EJML is written in 100% Java. </li> 
       </ul></td>

        <li> <a href="http://www.pds.twi.tudelft.nl/timber/">Timber</a> is a static compiler for Spar/Java, a superset of Java with constructs for high-performance computing. The additions include support for data parallelism and task parallelism, tuples, multi-dimensional arrays, generic classes and interfaces through templates, complex numbers, and subscript operator overloading. </li>
        <li> <a href="http://www.vni.com/corner/garage/grande/index.htm">zeta</a>, a Java compiler that admits infix notation for operations on complex variables&nbsp; </li>
       </ul> <b>Linking Java to Other Technical Computing Packages</b>&nbsp; 
       <ul> 
        <li> <a href="http://javamath.sourceforge.net/">JavaMath</a> is free software to enable mathematical programs in Java to use the computational capabilities of existing compute engines, either stand-alone applications or Web-based client-server applications. </li>

        <li> The <a href="http://www.epic.noaa.gov/NOAAServer/sgt/">Scientific Graphics Toolkit</a> of NOAA's NOAAServer Project aims to provide Java classes to produce publication quality, object oriented graphics of scientific data.
