  <p>CoreUtils: collection of basic file, shell and text manipulation utilities</p> 
  <h2><a name="version">Version</a></h2> 
  <p>5.3.0</p> 
  <h2><a name="description">Description</a></h2> 
  <p>The GNU Core Utilities are the basic file, shell and text manipulation utilities of the GNU operating system. These are the core utilities which are expected to exist on every operating system. </p>
  <p>File utilities: </p>
  <ul> 
   <li>chgrp: Changes file group ownership. </li>
   <li>chown: Changes file ownership. </li>
   <li>chmod: Changes file permissions. </li>
   <li>cp: Copies files. </li>
   <li>dd: Copies and converts a file. </li>
   <li>df: Shows disk free space on filesystems. </li>
   <li>dir: Gives a brief directory listing. </li>
   <li>dircolors: Setup program for the color output of GNU ls. </li>
   <li>du: Shows disk usage on filesystems. </li>
   <li>install: Copies file and sets its permissions. </li>
   <li>ln: Creates file links. </li>
   <li>ls: Lists directory contents. </li>
   <li>mkdir: Creates directories. </li>
   <li>mkfifo: Creates FIFOs (named pipes). </li>
   <li>mknod: Creates special files. </li>
   <li>mv: Moves files. </li>
   <li>rm: Removes (deletes) files. </li>
   <li>rmdir: Removes empty directories. </li>
   <li>shred: Destroy data in files. </li>
   <li>sync: Synchronizes filesystem buffers and disk. </li>
   <li>touch: Changes file timestamps. </li>
   <li>vdir: Long directory listing. </li>
  </ul> 
  <p>Text utilities: </p>
  <ul> 
   <li>cat: concatenates and prints files on the standard output </li>
   <li>cksum: checksum and count the bytes in a file </li>
   <li>comm: compares two sorted files line by line </li>
   <li>csplit: splits a file into sections determined by context lines </li>
   <li>cut: remove sections from each line of files </li>
   <li>expand: convert tabs to spaces </li>
   <li>fmt: simple optimal text formatter </li>
   <li>fold: wrap each input line to fit in specified width </li>
   <li>head: output the first part of files </li>
   <li>join: join lines of two files on a common field </li>
   <li>md5sum: compute and check MD5 messsage digest </li>
   <li>nl: number lines of files </li>
   <li>od: dump files in octal and other formats </li>
   <li>paste: merge lines of files </li>
   <li>ptx: produce a permuted index of file contents </li>
   <li>pr: convert text files for printing </li>
   <li>shasum: compute and check SHA1 message digest </li>
   <li>sort: sort lines of text files </li>
   <li>split: split a file into pieces </li>
   <li>sum: checksum and count the blocks in a file </li>
   <li>tac: concatenates and prints files in reverse </li>
   <li>tail: outputs the last part of files </li>
   <li>tr: translates or deletes characters </li>
   <li>tsort: perform topological sort </li>
   <li>unexpand: convert spaces to tabs </li>
   <li>uniq: remove duplicate lines from a sorted file </li>
   <li>wc: prints the number of bytes, words, and lines in files </li>
  </ul> 
  <p>Shell utilities: </p>
  <ul> 
   <li>[ - Check file types and compare values </li>
   <li>basename - Removes the path prefix from a given pathname. </li>
   <li>chroot - Changes the root directory. </li>
   <li>date - Prints/sets the system date and time. </li>
   <li>dirname - Removes the last level or filename from a given pathname. </li>
   <li>echo - Prints a line of text. </li>
   <li>env - Displays/modifies the environment. </li>
   <li>expr - Evaluates expressions. </li>
   <li>factor - Prints prime factors. </li>
   <li>false - Returns an unsuccessful exit status. </li>
   <li>groups - Print the groups that the user is a member of. </li>
   <li>hostid - Print the numeric identifier for the current host </li>
   <li>hostname - Print or set the machine name. </li>
   <li>id - Print real/effective uid/gid. </li>
   <li>logname - Print current login name. </li>
   <li>nice - Modify scheduling priority. </li>
   <li>nohup - Allows a command to continue running after logging out. </li>
   <li>pathchk - Check file name portability. </li>
   <li>pinky - Lightweight finger </li>
   <li>printenv - Prints environment variables. </li>
   <li>printf - Formats and prints data. </li>
   <li>pwd - Print the current working directory. </li>
   <li>seq - Print numeric sequences. </li>
   <li>sleep - Suspends execution for a specified time. </li>
   <li>stty - Print/change terminal settings. </li>
   <li>su - Allows you to adopt the id of another user or superuser. </li>
   <li>tee - Sends output to multiple files. </li>
   <li>test - Evaluates an expression. </li>
   <li>true - Returns a successful exit status. </li>
   <li>tty - Print terminal name. </li>
   <li>uname - Print system information. </li>
   <li>users - Print current user names. </li>
   <li>who - Print a list of all users currently logged in. </li>
   <li>whoami - Print effective user id. </li>
   <li>yes - Print a string repeatedly. </li>
  </ul>
