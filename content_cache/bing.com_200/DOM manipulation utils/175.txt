      <td bgcolor="#CED5FD"> <h2 class="subheader"><u>What's new ... ?</u></h2> <p class="desc"> ImprolaLib-0.0.2 is the <a href="news.html">latest</a> Improla release. Go to the sourceforge <a href="http://sourceforge.net/projects/improla">project</a> page to download it. Visit the <a href="dox.html">documentation</a> section for the associated documentation. This release has an almost complete <a href="http://www.doxygen.org">doxygen</a> generated reference guide. Go to the <a href="news.html">news</a> section to see the other releases. ImprolaLib-0.0.2 has the most common image processing algorithms implemented. With this release ImprolaLib will support reading and writing PNG images. It makes use of the <a href="http://pngwriter.sourceforge.net/">PNGwriter</a> library for this purpose. </p> <br> 
       <hr> <h2 class="subheader"><u>What is Improla?</u></h2> <p class="desc"> <b>Improla</b> == <b>Im</b>age <b>Pro</b>cessing <b>La</b>b </p> <p class="desc"> Improla consists of three packages: ImprolaLib, ImprolaViewer and wxImprola. ImprolaLib is a collection of image processing algorithms and utilities. wxImprola is a GUI frame work for ImprolaLib built using the <a href="http://wxwidgets.org">wxWidgets</a> library. ImprolaViewer is a stand alone application developed using ImprolaLib and wxImprola. </p> <p class="desc"> Starting from release 0.0.2, ImprolaLib supports reading and writing of PNG images. This is done using the <a href="http://pngwriter.sourceforge.net/">PNGwriter</a> library. ImprolaLib also uses <a href="http://math.nist.gov/tnt">TNT</a> library for matrix manipulation. Both, TNT and PNGwriter are distributed as part of ImprolaLib source and do not require separate installation. </p><p class="desc"> Visit the sourceforge <a href="http://sourceforge.net/projects/improla">project</a> page to download the latest releases of the packages. Find installation instructions, user-guides and API reference manuals at the <a href="dox.html">documentations</a> section. Information on the latest package releases can be found at the <a href="news.html">news</a> section. </p> <p class="desc"> The following convention shall be followed for package releases. Lets say a package has been released by version number s.b.a. Then it means that it is an alpha version 'a' of the beta version 'b+1' of the stable release 's+1'. </p> <br> </td>
