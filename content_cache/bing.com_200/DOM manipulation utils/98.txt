     <h1>45+ Most Useful Java Libraries</h1> 
     <div class="article_meta"> 
      <div style="margin-bottom: 5px;"> 
       <span class="timestamp">??&nbsp;2014-10-09 00:30:00 </span>

       <p> <img src="http://img1.tuicool.com/67jaaqr.png!web" alt="Most Popular Java Libraries" title="Most Popular Java Libraries" class="alignCenter"> </p> 
       <p> There are hundreds of Java libraries available today that cover every type of programming problem a Java developer is likely to come across. This article will take a look at Java libraries (core java and third party) <a href="http://www.fromdev.com/2009/05/as-java-developer-what-should-i-be.html" rel="nofollow,noindex">every Java developer should know</a> about before they write their first piece of code. Unfortunately, you will not find any of below mentioned third party libraries injava bookssince they are mostly project and requirements specific. </p> 
       <p>But before we begin with the list, let’s take a look at what a Java library does:</p> 
       <p>Understanding Java Library</p> 
       <p>According to Wiki:</p> 
       <p>The Java Class Library (JCL) is a set of dynamically loadable libraries that Java applications can call at run time.</p> 
       <p>The standard java library consists of a number of packages. Each package in turn consists of a number of classes (that provide similar functionality).</p> 
       <h2>Most Widely Used Core Java Packages &amp; Internal Libraries</h2> 
       <p> These are the java libraries that are bundled with JDK and used for almost any application developed in java. </p> 
       <h3>Java.lang</h3> 
       <p>The java.lang package contains the classes and interfaces that are fundamental to the core Java language. The most important classes of this package are Object, which is the root of class hierarchy, and Class, instances of which represent classes at run time. Java depends directly on these two classes, making it indispensable for all java programs. Other classes from this package include: Boolean, Byte, Float, Math, Process and many more.</p> 
       <h3>Java.util</h3> 
       <p>This library provides users with generic java utilities like collections framework, formatted printing and scanning, array manipulation utilities, event model, date and time facilities, internationalization and miscellaneous utility classes. The classes and interfaces in java.util include: Hashtable, Vector, Enumeration, StringTokenizer, EventObject, Locale, Calendar and TimeZone and ResourceBundle.</p> 
       <h3>Java.io</h3> 
       <p> This library is very useful and contains the classes that handle fundamental input and output operations in Java. The I/O classes in this package can be grouped as follows: </p> 
       <ul> 
        <li>Classes for reading input from a stream of data.</li> 
        <li>Classes for writing output to a stream of data.</li> 
        <li>Classes that manipulate files on the local filesystem.</li> 
        <li>Classes that handle object serialization.</li> 
       </ul> 
       <h3>Java.net</h3> 
       <p>The package java.net contains classes and interfaces that provide a powerful infrastructure for networking in Java. The java.net package can be roughly divided into two sections: a low level API and a high level API. Low level API has classes like InetAddress whereas high level API has classes like URL, URI, URLConnection and HttpURLConnection.</p> 
       <h3>Java.net</h3> 
       <p>The package java.net contains classes and interfaces that provide a powerful infrastructure for networking in Java. The java.net package can be roughly divided into two sections: a low level API and a high level API. Low level API has classes like InetAddress whereas high level API has classes like URL, URI, URLConnection and HttpURLConnection.</p> 
       <h3>Java.security</h3> 
       <p>This package has several classes that allow users to encrypt a file using a user-provided key. It supports a cryptograph and digital signature.</p> 
       <h3>Java.sql</h3> 
       <p>It contains all the SQL related classes and interfaces. This package provides the API for accessing and processing data stored in a data source.</p> 
       <h2>Java.swing</h2> 
       <p>The UI library for developing desktop Java applications.</p> 
       <h2>Most Widely Used Third Party Libraries</h2> 
       <p> Below are some of the best and heavily used third party open source libraries in java. These libraries demonstrate the real power of java and make Java <a href="http://www.fromdev.com/2013/09/Best-Programming-Languages-Web-Development.html" rel="nofollow,noindex">one of the most popular language on planet</a> . </p> 
       <h2>Popular Utility Libraries</h2> 
       <h2> <a rel="nofollow,noindex" target="_blank" title="Apache Commons Collections" name="Apache-Commons-Collections" href="http://commons.apache.org/proper/commons-collections/">Apache Commons Collections</a> </h2> 
       <p>This is a library that provides enhancements on core java collections. It provides additional collections and utility methods to manipulate collection objects easily and efficiently.</p> 
       <h2> <a rel="nofollow,noindex" target="_blank" title="Google Guava" name="Google-Guava" href="https://code.google.com/p/guava-libraries/">Google Guava</a> </h2> 
       <p>Google Guava is an open-source set of common libraries for Java. It is designed by Google and was previously known as Google Collections. It’s a useful source of miscellaneous utility functions and classes. It’s basically a core library for immutable collections, string manipulation, caching, primitives support, and easier I/O functions.</p> 
       <h2> <a rel="nofollow,noindex" target="_blank" title="Apache DBCP - Database Connection Pooling Library" name="Apache-DBCP--Database-Connection-Pooling-Library" href="http://commons.apache.org/proper/commons-dbcp/">Apache DBCP - Database Connection Pooling Library</a> </h2> 
       <p>This is a powerful library that provides feature of connection pooling to relational databases. This library has existed for long time and has been used in many applications including application servers.</p> 
       <h2> <a rel="nofollow,noindex" target="_blank" title="Apache IO - Input Output Utilities" name="Apache-IO--Input-Output-Utilities" href="http://commons.apache.org/proper/commons-io/">Apache IO - Input Output Utilities</a> </h2> 
       <p> Apache IO is a powerful library to do file and stream operations efficiently and quickly in java. Many utility methods from this library have been adopted in recent Java versions. </p> 
       <h2>XML Manipulation Libraries</h2> 
       <h2> <a rel="nofollow,noindex" target="_blank" title="Apache Commons Digester For XML" name="Apache-Commons-Digester-For-XML" href="http://commons.apache.org/proper/commons-digester/">Apache Commons Digester For XML</a> </h2> 
       <p>Digester is a XML to Java mapping utility that has been heavily used in XML communication using java. Reading XML config files and data can be really easy the digester.</p> 
       <h2> <a rel="nofollow,noindex" target="_blank" title="JSoup" name="JSoup" href="http://jsoup.org/">JSoup</a> </h2> 
       <p>jSoup is a library that is used for parsing HTML file in Java. It can be used for web application unit testing and web scraping. Its “jQuery-like” and “regex” selector syntax is very easy to use for data extraction. jSoup is quite fast in retrieving the raw page content.</p> 
       <h2>XML Parsers</h2> 
       <h2> <a rel="nofollow,noindex" target="_blank" title="JDOM" name="JDOM" href="http://www.jdom.org">JDOM</a> </h2> 
       <p>JDOM is a popular DOM parser library that is widely used in many java applications for parsing XML.</p> 
       <h2> <a rel="nofollow,noindex" target="_blank" title="DOM4J" name="DOM4J" href="http://dom4j.sourceforge.net/">DOM4J</a> </h2> 
       <p>Its a simple open source library for manipulation of XML, XSLT and XPath.</p> 
       <h2> <a rel="nofollow,noindex" target="_blank" title="XERCES" name="XERCES" href="http://xerces.apache.org/xerces-j/">XERCES</a> </h2> 
       <p>Yet another popular XML parser.</p> 
       <h2>JSON Libraries</h2> 
       <h2> <a rel="nofollow,noindex" target="_blank" title="Gson" name="Gson" href="http://code.google.com/p/google-gson/">Gson</a> </h2> 
       <p>JSON parsing library by Google</p> 
       <h2> <a rel="nofollow,noindex" target="_blank" title="Jackson" name="Jackson" href="http://jackson.codehaus.org">Jackson</a> </h2> 
       <p>A high performance JSON processor. Jackson is a multi-purpose Java library for processing JSON and XML data format. It’s a fast, lightweight and user-friendly library for developers.</p> 
       <h2> <a rel="nofollow,noindex" target="_blank" title="XStream" name="XStream" href="http://xstream.codehaus.org">XStream</a> </h2> 
       <p>XStream is a simple library to serialize objects to XML and JSON</p> 
       <h2>Popular Chart And Graph Libraries In Java</h2> 
       <p> There are <a title="many good charting libraries in java" name="many-good-charting-libraries-in-java" href="http://www.fromdev.com/2012/09/Free-Open-Source-Java-Charting-Library.html" rel="nofollow,noindex">many good charting libraries in java</a> we have picked two of the most popular ones in this list. </p> 
       <h2> <a rel="nofollow,noindex" target="_blank" title="JFreeChart" name="JFreeChart" href="http://www.jfree.org/jfreechart/">JFreeChart</a> </h2> 
       <p>The JFreeChart is the easiest and most widely used graphical library for creating a wide variety of good looking charts. JFreeChart allows the users to generate pie chart, bar chart, timeseries chart, Gantt chart, histogram, X-Y charts and various specific charts.</p> 
       <h2> <a rel="nofollow,noindex" target="_blank" title="JasperReports" name="JasperReports" href="http://community.jaspersoft.com/">JasperReports</a> </h2> 
       <p>This is a full fledged feature rich library to create graph and charts. This is also available with enterprise support.</p> 
       <h2>Popular JDBC &amp; Object Relational Mapping Frameworks in Java</h2> 
       <p>ORM tools have been criticized for its weaknesses http://java.dzone.com/articles/martin-fowler-orm-hate however they are still very popular since they can help developers avoid writing queries for most part of the project.</p> 
       <h2> <a rel="nofollow,noindex" target="_blank" title="Hibernate" name="Hibernate" href="http://hibernate.org">Hibernate</a> </h2> 
       <p>Hibernate ORM (object-relational mapping) is a Java library released this year in April that provides a framework to perform object relational mapping and query databases using HQL (Hibernate Query Language) and SQL. Hibernate supports mapping of Java classes to database tables. Hibernate is data independent, i.e. same code will work on all databases.</p> 
       <h2> <a rel="nofollow,noindex" target="_blank" title="MyBatis" name="MyBatis" href="http://mybatis.github.io/">MyBatis</a> </h2> 
       <p>MyBatis (formerly iBatis) is another popular ORM tool after hibernate. This is still under active development and being used by many people.</p> 
       <h2> <a rel="nofollow,noindex" target="_blank" title="JGraphT" name="JGraphT" href="http://jgrapht.org">JGraphT</a> </h2> 
       <p>jGraphT is another free Java library that provides mathematical graph theory objects and algorithms basically with focus on data structures. It supports various types of graphs like directed and undirected, multigraphs, pseudographs, unmodifiable graphs and many more. It is a simple to use and efficient Java library.</p> 
       <h2> <a rel="nofollow,noindex" target="_blank" title="FreeHEP" name="FreeHEP" href="http://java.freehep.org">FreeHEP</a> </h2> 
       <p>FreeHEP, an open source Java library enables users to easily write Java code for applications that involve higher energy physics. Although the main focus of this library is high-energy physics, it does contain many APIs that are generic and suitable for non-physics applications.</p> 
       <h2> <a rel="nofollow,noindex" target="_blank" title="The Lightweight Java Game Library (LWJGL)" name="The-Lightweight-Java-Game-Library-LWJGL" href="http://lwjgl.org">The Lightweight Java Game Library (LWJGL)</a> </h2> 
       <p>This Java game library is platform independent and offers access to OpenGL, OpenCL and OpenAL. Apart from handling graphics and gaming sounds, it provides access to controllers such as Gamepads, Steering wheels and Joysticks. If you are a Java developer, developing a game, give LJGL a try.</p> 
       <h2> <a rel="nofollow,noindex" target="_blank" title="Java WordNet Library (JWNL)" name="Java-WordNet-Library-JWNL" href="http://sourceforge.net/projects/jwordnet/">Java WordNet Library (JWNL)</a> </h2> 
       <p>JWNL is an API that allows Java applications to retrieve data from WordNet. JWNL allows developers to easily build NLP applications.</p> 
       <h2> <a rel="nofollow,noindex" target="_blank" title="Java Text Categorizing Library (JTCL)" name="Java-Text-Categorizing-Library-JTCL" href="https://jtcl-project.github.io/jtcl/">Java Text Categorizing Library (JTCL)</a> </h2> 
       <p>It is Java implementation of libTextCat; a library for language guessing.</p> 
       <h2> <a rel="nofollow,noindex" target="_blank" title="OpenNLP" name="OpenNLP" href="https://opennlp.apache.org">OpenNLP</a> </h2> 
       <p>Open NLP is a library from Apache used for processing natural language text. It is a collection of Java-based NLP tools for part-of-speech tagging, sentence-segmentation, tokenization, chunking, parsing and entity extraction. It supports Maximum Entropy Model (MaxEnt); a popular machine learning model for natural language processing task.</p> 
       <h2> <a rel="nofollow,noindex" target="_blank" title="Commons Math" name="Commons-Math" href="http://commons.apache.org/proper/commons-math/">Commons Math</a> </h2> 
       <p>Commons Math is another great library from Apache. It addresses the most common problems of mathematics and statistics that is not available in Java programming language.</p> 
       <h2>Logging Libraries</h2> 
       <h2> <a rel="nofollow,noindex" target="_blank" title="Log4j" name="Log4j" href="http://logging.apache.org/log4j/2.x/">Log4j</a> </h2> 
       <p>The most popular and oldest logging library.</p> 
       <h2> <a rel="nofollow,noindex" target="_blank" title="Logback" name="Logback" href="http://logback.qos.ch">Logback</a> </h2> 
       <p>Logback is a logging framework that has some really useful features that are not supported by log4j.</p> 
       <h2> <a rel="nofollow,noindex" target="_blank" title="SLF4J" name="SLF4J" href="http://www.slf4j.org">SLF4J</a> </h2> 
       <p>Every Java programmer knows that logging is critical for any Java application, especially server side application. Most of the programmers would be familiar with SLF4J as it is the most popular logging library. It provides Java API for various logging frameworks allowing users to plug in the desired logging framework at deployment time.</p> 
       <h2>Document Manipulation</h2> 
       <h2> <a rel="nofollow,noindex" target="_blank" title="Apache POI" name="Apache-POI" href="http://poi.apache.org">Apache POI</a> </h2> 
       <p>Apache POI is a powerful java API for microsoft documents.</p> 
       <h2> <a rel="nofollow,noindex" target="_blank" title="Docx4j" name="Docx4j" href="http://www.docx4java.org/trac/docx4j">Docx4j</a> </h2> 
       <p>Docx4j is yet another document manipulation library for multiple types of document processing.</p> 
       <h2>Application Layer Frameworks</h2> 
       <h2> <a rel="nofollow,noindex" target="_blank" title="Spring" name="Spring" href="http://projects.spring.io/spring-framework/">Spring</a> </h2> 
       <p>Spring is the most popular dependency injection framework.</p> 
       <h2> <a rel="nofollow,noindex" target="_blank" title="Struts" name="Struts" href="http://struts.apache.org/">Struts</a> </h2> 
       <p>Struts is one of the most widely used web framework on Java.</p> 
       <h2> <a rel="nofollow,noindex" target="_blank" title="GWT Google Web Toolkit" name="GWT-Google-Web-Toolkit" href="http://www.gwtproject.org/">GWT Google Web Toolkit</a> </h2> 
       <p>GWT is a web user interface library that can render Javascript and HTML based UI using Java code.</p> 
       <h2> <a rel="nofollow,noindex" target="_blank" title="Vaadin" name="Vaadin" href="https://vaadin.com/home">Vaadin</a> </h2> 
       <p>Vaadin is a Java framework for web beautiful looking applications.</p> 
       <h2> <a rel="nofollow,noindex" target="_blank" title="Grails" name="Grails" href="https://grails.org/">Grails</a> </h2> 
       <p>Grail is open source framework based on Groovy that support rapid application development.</p> 
       <h2> <a rel="nofollow,noindex" target="_blank" title="Play Framework" name="Play-Framework" href="https://www.playframework.com/">Play Framework</a> </h2> 
       <p>Play framework is another powerful web application framework for Java and Scala developers.</p> 
       <h2> <a rel="nofollow,noindex" target="_blank" title="Netty" name="Netty" href="http://netty.io/">Netty</a> </h2> 
       <p>Netty is asynchronous event-driven network application framework for developing high performance applications.</p> 
       <h2> <a rel="nofollow,noindex" target="_blank" title="Akka" name="Akka" href="http://akka.io/">Akka</a> </h2> 
       <p>Akka is powerful application framework for developing high performance java tools.</p> 
       <p> These are the Java libraries all Java developers should make use of whenever required! </p> 
       <h2> <a rel="nofollow,noindex" target="_blank" title="SWT" name="SWT" href="http://www.eclipse.org/swt/">SWT</a> </h2> 
       <p>SWT is a open source widget toolkit developed by eclipse foundation. This is a really powerful library developed by IBM.</p> 
       <h2> <a rel="nofollow,noindex" target="_blank" title="Apache Pivot" name="Apache-Pivot" href="https://pivot.apache.org/">Apache Pivot</a> </h2> 
       <p>Pivot is a open source platform for developing installable desktop UI applications in Java.</p>
