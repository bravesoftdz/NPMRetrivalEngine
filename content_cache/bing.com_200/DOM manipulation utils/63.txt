    <div class="page-introduction"> 
     <h1>Prototype</h1> 
     <h4>An object-oriented JavaScript framework</h4> 
     <p>Prototype is a JavaScript framework that aims to ease development of dynamic web applications. It offers a familiar class-style OO framework, extensive Ajax support, higher-order programming constructs, and easy DOM manipulation.</p> 
     <h3>Targeted platforms</h3> 
     <p>Prototype currently targets the following platforms:</p> 
     <ul> 
      <li>Microsoft Internet Explorer for Windows, version 6.0 and higher</li> 
      <li>Mozilla Firefox 1.5 and higher</li> 
      <li>Apple Safari 2.0.4 and higher</li> 
      <li>Opera 9.25 and higher</li> 
      <li>Chrome 1.0 and higher</li> 
     </ul> 
     <h2>Using Prototype</h2> 
     <p>To use Prototype in your application, download the latest release from the Prototype web site (<a href="http://prototypejs.org/download">http://prototypejs.org/download</a>) and copy <code>dist/prototype.js</code> to a suitable location. Then include it in your HTML like so:</p> 
     <div class="CodeRay"> 
      <div class="code">
       <pre><span style="color:#070;font-weight:bold">&lt;script</span> <span style="color:#b48">type</span>=<span style="background-color:hsla(0,100%,50%,0.05)"><span style="color:#710">"</span><span style="color:#D20">text/javascript</span><span style="color:#710">"</span></span> <span style="color:#b48">src</span>=<span style="background-color:hsla(0,100%,50%,0.05)"><span style="color:#710">"</span><span style="color:#D20">/path/to/prototype.js</span><span style="color:#710">"</span></span><span style="color:#070;font-weight:bold">&gt;</span><span style="color:#070;font-weight:bold">&lt;/script&gt;</span></pre>
      </div> 
     </div> 
     <h3>Building Prototype from source</h3> 
     <p><code>prototype.js</code> is a composite file generated from many source files in the <code>src/</code> directory. To build Prototype, you'll need:</p> 
     <ul> 
      <li>a copy of the Prototype source tree, either from a distribution tarball or from the Git repository (see below)</li> 
      <li>Ruby 1.8.2 or higher (<a href="http://www.ruby-lang.org/">http://www.ruby-lang.org/</a>)</li> 
      <li>Rake--Ruby Make (<a href="http://rake.rubyforge.org/">http://rake.rubyforge.org/</a>)</li> 
      <li>RDoc, if your Ruby distribution does not include it</li> 
     </ul> 
     <p>From the root Prototype directory:</p> 
     <ul> 
      <li><code>rake dist</code> will preprocess the Prototype source using Sprockets and generate the composite <code>dist/prototype.js</code></li> 
      <li><code>rake package</code> will create a distribution tarball in the <code>pkg/</code> directory</li> 
     </ul> 
     <h2>Contributing to Prototype</h2> 
     <p>Check out the Prototype source with</p> 
     <div class="CodeRay"> 
      <div class="code">
       <pre><span style="color:#369;font-weight:bold">$</span> git clone git:<span style="color:#777">//github.com/sstephenson/prototype.git</span>
<span style="color:#369;font-weight:bold">$</span> cd prototype
<span style="color:#369;font-weight:bold">$</span> git submodule init
<span style="color:#369;font-weight:bold">$</span> git submodule update vendor/sprockets vendor/pdoc vendor/unittest_js</pre>
      </div> 
     </div> 
     <p>Find out how to contribute: <a href="http://prototypejs.org/contribute">http://prototypejs.org/contribute</a>.</p> 
     <h2>Documentation</h2> 
     <p>Please see the online Prototype API: <a href="http://api.prototypejs.org">http://api.prototypejs.org</a>.</p>
