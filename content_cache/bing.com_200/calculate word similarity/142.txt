             <h2>Showing the Differences between Strings on a Web Page</h2>
             <p>The <a href="http://www.phpclasses.org/package/8331-PHP-Find-and-view-the-difference-between-text-strings.html">PHP Text Diff Highlight class</a> is very straightforward. Once you create an object of the class, just call&nbsp;<b>diff</b> function passing as parameters the initial text, the final text text, and a reference to an object that passes parameter values and returns the results of the diff operation.</p>
             <pre class="code"><br> $difference = new stdClass;
 $difference-&gt;mode = 'w';
 $before = 'Some text before';
 $after = 'This is the text after';
 $diff = new diff_class;
 if(!$diff-&gt;diff($before, $after, $difference))
 {
   die($diff-&gt;error);
 }
 else
 {
    echo 'Changes: '.print_r($difference-&gt;differences, 1);
 }

</pre>
             <p>The class can also show the differences between the two strings by using the function&nbsp;<b>formatDiffAsHtml</b> instead of the diff function. It builds a HTML string with the added text marked using underline and bold style and the removed text in strike-through style. You can configure the styles for presenting the added and the removed text.</p>
             <p>The formatted changes HTML may be used to display the differences to the user in a Web page.</p>
             <pre class="code"><br> if($diff-&gt;formatDiffAsHtml($before, $after, $difference))
 {
   echo '&lt;div&gt;', $difference-&gt;html, '&lt;/div&gt;';
 }

</pre>
             <p>The differences between the strings will show like this:</p> 
             <p class="code" style="padding: 4px;"><b></b>
              <strike>
               Some&nbsp;
              </strike><b><u>This is the&nbsp;</u></b>text&nbsp;
              <strike>
               before
              </strike><b>after</b></p> 
             <h2>Comparing strings by Character, Word or Line</h2> 
             <p>The comparison between the two text strings can be done in three modes: by line or by word for regular text, or by character which is more suitable for comparing binary data strings.</p>
             <p>The comparison mode is configured by setting the <b>difference</b>&nbsp;parameter of the diff or formatDiffAsHtml functions. The mode option values are 'c' for character, 'w' for word and 'l' for line.</p>
             <h2>Getting the Final String by Patching the Original String</h2>
             <p>The list of changes between the original string and the final string is returned as an array in the&nbsp;<b>differences</b>&nbsp;variable $difference parameter object. This array can be used to rebuild the final string by calling the <b>patch</b> function, passing the initial string as parameter.</p>
             <p>You need to set the <b>patch</b>&nbsp;variable of the $difference parameter object to true to make the class return all the data that the patch function needs to rebuild the final string.</p>
             <p>The patch function returns the patched string as the <b>after</b>&nbsp;variable of the afterPatch parameter object.</p> 
             <pre class="code"><br> $difference = new stdClass;
 $difference-&gt;mode = 'w';
 $difference-&gt;patch = true;
 $afterPatch = new stdClass;
 if($diff-&gt;diff($before, $after, $difference)
 &amp;&amp; $diff-&gt;patch($before, $difference-&gt;differences, $afterPatch))
 {
   echo 'The before text patched with the differences is: '.
     $afterPatch-&gt;after;
 }

</pre> 
             <h2>Showing the Diff and Patching Text in JavaScript</h2>
             <p>If you need to see the differences between the original text and a text being edited in a form, you need the JavaScript counterpart of the PHP class version, which is the <a href="http://www.jsclasses.org/blog/package/282/post/1-View-the-Diff-and-Patch-Text-Strings-in-JavaScript-and-PHP.html">JS Diff Viewer object</a>.</p>
             <p>That JavaScript object provides the exact same functionality of finding the differences between text strings, displaying the differences formatted as HTML and patching the original text to rebuild the final text using the list of differences.</p>
             <p>There is another article similar to this to explain how to use that JavaScript object for <a href="http://www.jsclasses.org/blog/package/282/post/1-View-the-Diff-and-Patch-Text-Strings-in-JavaScript-and-PHP.html">viewing the differences and patch text strings</a> in JavaScript.</p>
             <h2>Conclusion</h2>
             <p>This PHP class and the JavaScript object are very useful for the viewing the differences between two strings on a Web page. If you have other needs that you would like these classes to &nbsp;address or have any other question, just put a comment here to this article.</p>
