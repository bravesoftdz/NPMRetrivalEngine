           <ul class="navigation byline"> 
            <li><a class="nolink">Posted by </a><a href="/profile/sureshkumargorakala">suresh kumar gorakala</a><a class="nolink"> on January 11, 2016 at 6:00am</a></li> 
            <li><a class="xg_sprite xg_sprite-view" href="http://www.bigdatanews.datasciencecentral.com/profiles/blog/list?user=09scr4ipgrl6a">View Blog</a></li>

            <div>
             Today I will explain you how to create a basic Movie review engine based on the tweets by people using R. The implementation of the Review Engine will be as follows:
            </div> 
            <ul> 
             <li>Gets Tweets from Twitter</li> 
             <li>Clean the data</li> 
             <li>Create a Word Cloud</li> 
             <li>Create a data dictionary</li> 
             <li>Score each tweet.</li> 
            </ul> 
            <p><strong>Gets Tweets from Twitter:</strong></p> 
            <div>
             First step is to fetch the data from Twitter. In R, we have facility to call the twitter API using package twitter. Below are the steps for fetch the tweets using twitter package. Each tweet data contains:
            </div> 
            <ul> 
             <li>Text</li> 
             <li>Is re-tweeted</li> 
             <li>Re-tweet count</li> 
             <li>Tweeted User name</li> 
             <li>Latitude/Longitude&nbsp;</li> 
             <li>Replied to, etc.</li> 
            </ul> 
            <p>For our case we only consider Text feature of the Tweet as we are interested on the review of the movie. We can also use the other features such as Latitude/Longitude, replied to, etc. do other analysis on the tweeted data.</p> 
            <div>
             <span style="font-family: 'courier new', courier;" class="font-size-2">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; library(tm)</span>
            </div> 
            <div>
             <span style="font-family: 'courier new', courier;" class="font-size-2">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; tweets = searchTwitter("#ChennaiExpress", n=500, lang="en")</span>
            </div> 
            <div>
             <img src="http://1.bp.blogspot.com/-wYJ9zzSqjG4/VoC8J_1PgpI/AAAAAAAAFYk/koMbeXh-d-w/s320/tweets.png" class="align-center">
            </div> 
            <div>
             <strong>Clean the data:</strong>
            </div> 
            <div>
             In the next step, we need to clean the data so that we can use it for our analysis. Cleaning of data is a very important step in Data Analysis. This step includes:
            </div> 
            <div>
             Extracting only text from Tweets:
            </div> 
            <div>
             <span style="font-family: 'courier new', courier;" class="font-size-2">tweets_txt = sapply(tweets,function(x) x$getText())</span>
            </div> 
            <div>
             Removing Url links, Reply to, punctuations, non-alphanumeric, symbols, spaces etc.
            </div> 
            <div>
             <span style="font-family: 'courier new', courier;" class="font-size-2">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;tweets_cl = gsub("(RT|via)((?:\\b\\W*@\\w+)+)","",tweets)</span>
            </div> 
            <div>
             <span style="font-family: 'courier new', courier;" class="font-size-2">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;tweets_cl = gsub("http[^[:blank:]]+", "", tweets_cl)</span>
            </div> 
            <div>
             <span style="font-family: 'courier new', courier;" class="font-size-2">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;tweets_cl = gsub("@\\w+", "", tweets_cl)</span>
            </div> 
            <div>
             <span style="font-family: 'courier new', courier;" class="font-size-2">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;tweets_cl = gsub("[ \t]{2,}", "", tweets_cl)</span>
            </div> 
            <div>
             <span style="font-family: 'courier new', courier;" class="font-size-2">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;tweets_cl = gsub("^\\s+|\\s+$", "", tweets_cl)</span>
            </div> 
            <div>
             <span style="font-family: 'courier new', courier;" class="font-size-2">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;tweets_cl = gsub("[[:punct:]]", " ", tweets_cl)</span>
            </div> 
            <div>
             <span style="font-family: 'courier new', courier;" class="font-size-2">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;tweets_cl = gsub("[^[:alnum:]]", " ", tweets_cl)</span>
            </div> 
            <div>
             <span style="font-family: 'courier new', courier;" class="font-size-2">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;tweets_cl &lt;- gsub('\\d+', '', tweets_cl)</span>
            </div> 
            <div>
             <strong>Create a Word Cloud:</strong>
            </div> 
            <div>
             At this point let us view Word-Cloud of frequently tweeted words in the data considered for visual understanding/analyzing the data.
            </div> 
            <div>
             library(wordcloud)
            </div> 
            <div>
             wordcloud(tweets_cl)
            </div> 
            <div>
             <div>
              &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
              <a rel="nofollow" href="http://3.bp.blogspot.com/-PWGzLWOBOSk/VoC8Ox4BGNI/AAAAAAAAFYw/p7pwDDDw6ks/s1600/Wordcloud.png"></a>
             </div> 
            </div> 
            <div class="separator">
             <img src="http://3.bp.blogspot.com/-PWGzLWOBOSk/VoC8Ox4BGNI/AAAAAAAAFYw/p7pwDDDw6ks/s320/Wordcloud.png" class="align-center">
            </div> 
            <div>
             <strong>Create a data dictionary:</strong>
            </div> 
            <div>
             In this step, we create use a Dictionary of words containing positive, negative words which are downloaded from&nbsp;
             <a rel="nofollow" href="http://www.cs.uic.edu/~liub/FBS/sentiment-analysis.html" target="_blank">here</a>. These 2 types of words are used as keywords for classifying the each tweet into one of the 4 categories: Very Positive, Positive, Negative and Very Negative.
            </div> 
            <div>
             <strong>Score each tweet:</strong>
            </div> 
            <div>
             In this step, we will write a&nbsp;
             <a rel="nofollow" href="http://jeffreybreen.wordpress.com/2011/07/04/twitter-text-mining-r-slides/" target="_blank">function&nbsp;</a>which will calculate rating of the movie. The function is given below. After calculating the scores we plot graphs showing the rating as “WORST”,”BAD”,”GOOD”,”VERYGOOD”
            </div> 
            <div class="separator"></div> 
            <div>
             <img src="http://4.bp.blogspot.com/-Na3MgRiXpS4/VoC8SwAKpfI/AAAAAAAAFY8/EvsQTg90rRA/s320/Movie%2BRating%2B_%2BR.png" class="align-center">
            </div> 
            <div>
             <strong>Future steps in this project will be:</strong>
            </div> 
            <ul> 
             <li>To create a UI preferably using .NET, as I’m a dot-net developer ;)</li> 
             <li>To Build a Movie Review Model which can classify a new tweet as and when provided?</li> 
            </ul> 
            <p><strong>Code:</strong></p> 
            <div>
             <span style="font-family: 'courier new', courier;" class="font-size-1">#include required libraries</span>
            </div> 
            <div>
             <span style="font-family: 'courier new', courier;" class="font-size-1">library(plyr)</span>
