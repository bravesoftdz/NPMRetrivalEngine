           <p style="text-align: justify;"><span style="color: #000000;"><strong>Sentiment Analysis of Twitter Data using R Programming</strong> – To perform Sentiment Analysis in R we need “sentiment” R package well developed by <span style="color: #0000ff;"><a style="color: #0000ff;" href="https://github.com/timjurka">Timothy P. Jurka</a></span>. The Package comes with two important functions namely: “classify_emotion” and “classify_polarity”. In this post I will show How to use both the functions in brief:</span></p> 
           <p style="text-align: justify;"><span style="color: #ff6600;"><strong>Read Also:</strong></span></p> 
           <ul style="text-align: justify;"> 
            <li><span style="color: #0000ff;"><a style="color: #0000ff;" href="http://technokarak.com/twitter-mining-using-r-programming-how-to-create-twitter-apps-for-data-integration.html" target="_blank">How to Setup Twitter App for Analysis</a></span></li> 
            <li><span style="color: #0000ff;"><a style="color: #0000ff;" href="http://technokarak.com/how-to-load-twitter-tweets-in-r-environment.html" target="_blank">How to Fetch Tweets in R Programming</a></span></li> 
            <li><span style="color: #0000ff;"><a style="color: #0000ff;" href="http://technokarak.com/how-to-clean-the-twitter-data-using-r-twitter-mining-tutorial.html" target="_blank">How to Clean Twitter Data using R</a></span></li> 
           </ul> 
           <p style="text-align: justify;"><span style="color: #000000;">First extract the tweet using the desired keyword. For demonstration process I am fetching 2000 tweets for both the keywords. So a total of 4000 tweets to analyze using sentiment package in R.</span></p> 
           <p style="text-align: justify;"><span style="color: #000000;">Read <a href="http://technokarak.com/how-to-load-twitter-tweets-in-r-environment.html" target="_blank">Fetching Twitter Data using R</a> and <a href="http://technokarak.com/how-to-clean-the-twitter-data-using-r-twitter-mining-tutorial.html" target="_blank">Cleaning of Twitter Data</a>.</span></p> 
           <p style="text-align: justify;"><span style="color: #000000;">In this tutorial I will use both “<strong>classify_emotion</strong>” and “<strong>classify_polarity</strong>” function to perform sentiment analysis on Twitter Data collected. See the code of the sentiment analysis function below:</span></p> 
           <pre class="brush: r; title: ; notranslate" title="">
sentiment_twitter &lt;- function(searchterm,i)
{
  library(twitteR)
  library(wordcloud)
  library(tm)
  library(RColorBrewer)
  library(sentiment)
  library(plyr)
  library(ggplot2)
  consumer_key &lt;- 'Your-Key'
  consumer_secret &lt;- 'Your Secret'
  access_token&lt;-'Your Access Token'
  access_secret &lt;- 'Your Access Secret'
  setup_twitter_oauth(consumer_key, consumer_secret, access_token, access_secret)
  tweet_to_score &lt;- searchTwitter(searchterm, n=i, lang="en", resultType="recent")
  tweet_text &lt;- sapply(tweet_to_score, function(x) x$getText())
  tweet_clean &lt;- clean_text(tweet_text)
  emotion_class &lt;- classify_emotion(tweet_clean, algorithm = "bayes", prior = 1)
  emotion &lt;- emotion_class[,7]
  emotion[is.na(emotion)] = "unknown"
  polarity_class &lt;- classify_polarity(tweet_clean, algorithm = "bayes")
  polarity = polarity_class[,4]
  sent_df = data.frame(text=tweet_clean, emotion=emotion,polarity=polarity, stringsAsFactors=FALSE)
  sent_df = within(sent_df,emotion &lt;- factor(emotion, levels=names(sort(table(emotion), decreasing=TRUE))))
  head(sent_df, n=5)
  ggplot(sent_df, aes(x=emotion)) +
    geom_bar(aes(y=..count.., fill=emotion)) +
    scale_fill_brewer(palette="Dark2") +
    labs(x="emotion categories", y="number of tweets") +
    ggtitle("Sentiment Analysis of Twitter") +
    theme(plot.title = element_text(size=12, face="bold"))
  }
</pre> 
           <h3 style="text-align: left;"><span style="text-decoration: underline;"><strong><span style="color: #000000; text-decoration: underline;">Code of clean_text function</span></strong></span></h3> 
           <pre class="brush: r; title: ; notranslate" title="">
clean_text = function(x)
{
  x = gsub("rt", "", x) # remove rt
  x = gsub("RT", "", x) # remove RT
  x = gsub("@\\w+", "", x) # remove at
  x = gsub("[[:punct:]]", "", x) # remove punctuation
  x = gsub("[[:digit:]]", "", x) # remove numbers
  x = gsub("http\\w+", "", x)  # remove links http
  x = gsub("[ |\t]{2,}", "", x) # remove tabs
  x = gsub("^ ", "", x)  # remove blank spaces at the beginning
  x = gsub(" $", "", x) # remove blank spaces at the end
  try.error = function(z)
  {
    y = NA
    try_error = tryCatch(tolower(z), error=function(e) e)
    if (!inherits(try_error, "error"))
      y = tolower(z)
    return(y)
  }
  x = sapply(x, try.error)
  return(x)
}
</pre> 
           <p style="text-align: justify;"><span style="color: #000000;">Copy Paste the above code in a R Script File and call the function sentiment_twitter(“Search Term”, No. of tweets)</span></p> 
           <p style="text-align: justify;"><span style="color: #000000;">For example I am calling the function for search term: IPL2016</span></p> 
           <pre class="brush: r; title: ; notranslate" title="">
sentiment_twitter("IPL2016", 2000)
</pre>
