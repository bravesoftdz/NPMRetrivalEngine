     <h1>Sentiment Analysis with Python NLTK Text Classification</h1>
     <p>This is a demonstration of <strong>sentiment analysis</strong> using a <code>NLTK 2.0.4</code> powered <strong>text classification</strong> process. It can tell you whether it thinks the text you enter below expresses <strong class="positive">positive sentiment</strong>, <strong class="negative">negative sentiment</strong>, or if it's <strong class="quiet">neutral</strong>. Using <strong>hierarchical classification</strong>, <em>neutrality</em> is determined first, and <em>sentiment polarity</em> is determined second, but only if the text is not neutral.</p>

     <h2>How Sentiment Analysis with Text Classification Works</h2>
     <p>The english sentiment uses classifiers trained on both <strong>twitter sentiment</strong> as well as <strong>movie reviews</strong> from the data sets created by <a href="http://www.cs.cornell.edu/people/pabo/movie-review-data/">Bo Pang and Lillian Lee</a> using <a href="http://github.com/japerk/nltk-trainer" title="NLTK Trainer @ github">nltk-trainer</a> (also on <a href="http://bitbucket.org/japerk/nltk-trainer/src" title="NLTK Trainer @ bitbucket">bitbucket</a>). The dutch sentiment is based on book reviews.</p>
     <p>The results will be <strong>more accurate on text that is similar to original training data</strong>. If you get an odd result, it could be the words you've used are unrecognized. Try entering more words to improve accuracy.</p>
