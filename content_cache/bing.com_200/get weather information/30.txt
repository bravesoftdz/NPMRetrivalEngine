          </ul> These files are updated hourly. <p></p> <p>A list of what <a href="weather.php">phrases</a> may appear in the &lt;weather&gt; XML tag and suggested icons is available.</p> <p> <a name="rss"> </a></p><h3><a name="rss" id="rss"></a>About RSS</h3> RSS is an XML based document format for syndicating news and other timely news-like information. It provides headlines, URLs to the source document and brief description information in an easy to understand and use format. RSS based "News Readers" and "News Aggregators" allow the display of RSS headlines on workstation desktops. Software libraries exist to read the RSS format and present RSS headlines on webpages and other online applications. <p> These NWS supplied RSS documents use the RSS 2.0 format. Each RSS item links to the html/web documents described above. Additional technical information is available from the following non-US Government website:</p> <p> <a href="http://weather.gov/cgi-bin/nwsexit.pl?url=http://blogs.law.harvard.edu/tech/rss">RSS 2.0 Specification</a></p> <p> A non-US Government maintained list of RSS readers, aggregators and related software:</p> <p> <a href="http://weather.gov/cgi-bin/nwsexit.pl?url=http://www.google.com/search?q=rss+software&amp;rls=com.microsoft:en-us&amp;ie=UTF-8&amp;oe=UTF-8&amp;startIndex=&amp;startPage=1">RSS Software</a></p> <p> <a href="http://weather.gov/survey/nws-survey.php?code=rss-cap-wwa">Comments</a> on this service are welcome. </p> <a name="xml"> </a><h3><a name="xml"></a><a name="xml" id="xml"></a>About XML</h3> NWS offers hourly weather observations formatted with xml tags to aid in the parsing of the information by automated programs used to populate databases, display information on webpages or other similar applications. This format is not to be confused with RSS and cannot be read by RSS readers and aggregators. These files present more detailed information than the RSS feeds in strings friendly for parsing. Both the RSS and XML feeds offer URLs to icon images. Additionally, A list of what <a href="weather.php">phrases</a> may appear in the
