     <p align="left"><a href="script2/quartercalendar.shtml"><strong>3-month calendar</strong></a><br> Sometimes it's convenient to have shown the calendar for multiple months. With that in mind, Ada's script displays the last, present, and next month's calendar. Customizable background color and font size.</p> 
     <p align="left"><a href="script2/anymonthcal.shtml"><strong>Any-month calendar</strong></a><br> This unique calendar, constructed using a textarea, enables you to look up the calendar for any month of any year! Use the drop-down menu to select the desired month. Quite exceptional.</p> 
     <p align="left"><a href="script2/tengcalendar.shtml"><strong>Date Time picker</strong></a><br> Similar to the above script, Date Time Picker allows your visitors to easily input a date/time into a form field, by selecting it from a popup window. Supports output dates in various formats such as: dd/MM/yyyy, dd/MMM/yyyy, MM/dd/yyyy etc.</p> 
     <p align="left"><strong> <a href="script2/dyndateselector.shtml">Dynamic Date selector</a></strong><br> A visually attractive dynamic date selector for form input fields. When the user clicks on a input field, the calendar will pop to let the user easily select a date value to enter. Script tested and works on Firefox, Internet Explorer, Netscape, Opera.</p> 
     <p align="left"><strong> <a href="script2/epoch/index.shtml">Epoch DHTML Calendar</a></strong><strong> </strong> <img src="new01.gif" width="35" height="20" alt="new01.gif (1016 bytes)"><br> Epoch is a flexible DHTML calendar that supports several display formats, such as inline on the page, as a DHTML date picker, or an calendar that with the ability to select multiple dates. Open source and released under GNU license.</p> 
     <p align="left"><strong> <a href="script2/eventscalendar.shtml">Events Calendar</a></strong><br> This is Events Calendar script that, along with the current month calendar, displays important events for specific dates. Click on a relevant date, and a description of the event associated with it is shown. The events are stored in an external .js file. This script is compatible in both IE and Firebox.</p> 
     <p align="left"><a href="script2/buttoncalendar.shtml"><strong>Form button calendar</strong></a><br> This is an elaborate calendar script that uses form buttons to constuct each date entry.</p>
