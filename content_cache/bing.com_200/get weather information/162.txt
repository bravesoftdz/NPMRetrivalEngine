             <h2>Introduction</h2> 
             <p>The JQuery has the following methods that can be used to perform Ajax requests:</p> 
             <ol> 
              <li><code>ajax() </code>- Load a remote page using an HTTP request. This is jQuery's low-level AJAX implementation. </li> 
              <li><code>load() </code>- Load HTML from a remote file and inject it into the DOM. </li> 
              <li><code>get() </code>- Load a remote page using an HTTP <code>GET </code>request. </li> 
              <li><code>getJSON() </code>- Load JSON data using an HTTP <code>GET </code>request. </li> 
              <li><code>getScript() </code>- Loads, and executes, a local JavaScript file using an HTTP <code>GET </code>request. </li> 
              <li><code>post() </code>- Loads HTML by performing an HTTP <code>post </code>request. </li> 
             </ol> 
             <h2>Background</h2> 
             <p>In this article, I will show you how to build a Jquery AJAX enabled ASP.NET MVC application. To create a new MVC project, see <a href="http://www.fairnet.com/post/ASPNET-MVC-application-structure.aspx">ASP.NET MVC application structure</a>. I will consume a USA Weather Forecast web service which is freely available at <a href="http://www.webservicex.net/WS/WSDetails.aspx?CATID=12&amp;WSID=68">WebserviceX.NET</a>.</p> 
             <h2>Using the Code</h2> 
             <p>To reference the Jquery AJAX script libraries, add the following markup at the end of the head element in <code>Site.Master</code>:</p> 
             <pre lang="jscript"><span class="code-comment">//</span><span class="code-comment"> Load from google hosted library</span>
&lt;script type=<span class="code-string">"</span><span class="code-string">text/javascript"</span> 
src=<span class="code-string">"</span><span class="code-string">http://ajax.googleapis.com/ajax/libs/jquery/1.2.6/jquery.min.js"</span>&gt;&lt;/script&gt;
<span class="code-comment">//</span><span class="code-comment">or Load from Jquery site</span>
&lt;script type=<span class="code-string">"</span><span class="code-string">text/javascript"</span> src=<span class="code-string">"</span><span class="code-string">http://code.jquery.com/jquery-latest.js"</span>&gt;&lt;/script&gt;
