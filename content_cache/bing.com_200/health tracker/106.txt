        <h2>Computational Resources for Teaching</h2> 
        <p><strong>The OSP Collection</strong> provides curriculum resources that engage students in physics, computation, and computer modeling. Computational physics and computer modeling provide students with new ways to understand, describe, explain, and predict physical phenomena. Browse the <a href="search/categories.cfm?t=Simulation">OSP simulations</a> or learn more about our tools and curriculum pieces below.</p> 
        <div id="fpLeft"> 
         <h2>Tracker</h2> 
         <p>The Tracker tool extends traditional video analysis by enabling users to create particle models based on Newton's laws. Because models synchronize with and draw themselves right on videos of real-world objects, students can test models experimentally by direct visual inspection.</p> 
         <p><a href="webdocs/Tools.cfm?t=Tracker">Learn more about Tracker</a></p> 
         <h3>Featured Tracker Package</h3> 
         <a href="items/detail.cfm?ID=11562"><img src="https://www.compadre.org/OSP/images/features/birds.jpg" alt="Featured OSP Resource" title="Projectile Motion with Angry Birds" width="260" height="155" class="featureIMG"></a> 
         <br style="clear:both"> 
         <p> <a href="items/detail.cfm?ID=11562"><strong>Projectile Motion with Angry Birds</strong></a><br> The Projectile Motion with Angry Birds lab uses the original video from Rovio, the makers of Angry Birds and the Tracker video analysis tool to measure and analyze the motion an angry bird projected from a slingshot to hit a pig. This OSP item was inspired by Rhett Allain's Dot Physics, a physics blog for Wired.<br> <a href="items/detail.cfm?ID=11562" class="homeMoreLink">More...</a> <br style="clear:both"> </p> 
         <h2>Curriculum Packages</h2> 
         <p>OSP curriculum packages combine computer simulations with tutorial materials and student worksheets. Packages can be modified to meet the needs of faculty and students</p> 
         <p><a href="search/categories.cfm?t=Curriculum">Browse curriculum resources by topic and learn how to customize packages</a></p> 
         <h3>Featured Curriculum Package</h3> 
         <a href="items/detail.cfm?ID=13337"><img src="https://www.compadre.org/OSP/images/features/AEM_Cover3_feature.jpg" alt="Featured OSP Resource" title="Modeling Wealth Inequality" width="258" height="183" class="featureIMG"></a> 
         <br style="clear:both"> 
         <p> <a href="items/detail.cfm?ID=13337"><strong>Modeling Wealth Inequality</strong></a><br> An econophysics package that the models the transfer of wealth between buyers and sellers, known as agents, who spend their time&nbsp;&nbsp;exchanging goods at a yard sale. In this economic model, two agents A and B are chosen at random and goods are exchanged.&nbsp;&nbsp;In each transaction an agent can either pay too much or get a bargain so that one agent becomes slightly richer while the other agent becomes poorer. What happens to the wealth of agent wA if this process is repeated many times and if we assume that the agent receiving the bargain is chosen at random so that sometimes agent A gains and sometimes agent A looses in the transaction. In other words, neither agent is always shrewd or always gullible so that all agents have an equal chance of getting rich. Does this model produce an equitable distribution of wealth?<br><br>This package is available online and as a downloadable ePub.<br> <a href="items/detail.cfm?ID=13337" class="homeMoreLink">More...</a> </p> 
        </div> 
        <div id="fpMiddle"> 
         <h2>EJS Modeling</h2> 
         <p>Student modeling, the guided exploration of physical systems and concepts, is a powerful approach to engaged learning. Easy Java Simulations provides the computational tools for students and faculty to explore physics without the need for learning details of java programming.</p> 
         <p><a href="search/categories.cfm?t=Overview">Learn more about EJS</a></p> 
         <h2>Programming</h2> 
         <p>Open Source Physics provides extensive resources for computational physics and physics simulations. Included are:</p> 
         <ul> 
          <li>An Eclipse environment for OSP</li> 
          <li>OSP Source Code Libraries</li> 
          <li>OSP best practices</li> 
          <li>Documentation</li> 
         </ul> 
         <p><a href="webdocs/programming.cfm">Access programming resources</a></p> 
         <h2>Tools</h2> 
         <p>OSP provides several general applications for physics teaching, student activities, and curriculum distribution. These are:</p> 
         <ul> 
          <li><a href="webdocs/Tools.cfm?t=Launcher"><strong>Launcher</strong></a> – Simulation packages.</li> 
          <li><a href="webdocs/Tools.cfm?t=Tracker"><strong>Tracker</strong></a> – Video analysis.</li> 
          <li><a href="webdocs/Tools.cfm?t=EJS"><strong>EJS</strong></a> – Easy Java Simulations.</li> 
          <li><a href="webdocs/Tools.cfm?t=Datatool"><strong>Data Tool</strong></a> – Data analysis.</li> 
         </ul>
