     <h3>What is Tracker?</h3> 
     <p>Tracker is a free video analysis and modeling tool built on the <a href="http://www.opensourcephysics.org/" target="_blank">Open Source Physics</a> (OSP) Java framework. It is designed to be used in physics education.</p> 
     <p>Tracker <font color="#007c00"><strong>video modeling</strong></font> is a powerful way to combine videos with computer modeling. For more information see <a href="help/particles.html" target="_blank">Particle Model Help</a> or AAPT Summer Meeting posters <a href="download/video_modeling.pdf" target="_blank">Video Modeling</a> (2008) and <a href="download/AAPT_video_modeling_2009.pdf" target="_blank">Video Modeling with Tracker</a> (2009).</p> 
     <p align="center"><img src="images/bouncing_cart.png"></p> 
     <h3>Installing and using Tracker</h3> 
     <p>Tracker requires <a href="http://java.sun.com/j2se/downloads.html" target="_blank">Java 1.6</a> or higher.</p> 
     <p>To install Tracker, download and run the appropriate installer using the links at the top of the page. For help see <a href="installers/installer_help.html" target="_blank">Installer Help</a>.</p> 
     <p><font size="2"><strong>Supported languages</strong>: english, arabic, czech, spanish, finnish, french, italian, hungarian, danish, german, greek, hebrew, indonesian, malaysian, simplified chinese, traditional chinese, korean, polish, portuguese, slovak, slovenian, swedish, turkish, vietnamese. Interested in translating to another language? Please contact <script language="JavaScript">print_mailto_link2();</script>.</font></p> 
     <p>If you're new to Tracker, see <a href="help/frameset.html" target="_blank">Help Getting Started</a> for a step-by step beginner's guide or <a href="https://www.youtube.com/watch?v=La3H7JywgX0" target="_blank">Getting Started with Tracker</a> for a video tutorial. For general help, use the built-in help files in Tracker, the online help in <a href="help/frameset.html" target="_blank">English</a> or <a href="help_sl/frameset.html" target="_blank">Slovenš?ina</a>, or the downloadable pdf help files in <a href="tracker_help.pdf" target="_blank">English</a>, <a href="tracker_help_es.pdf" target="_blank">Español</a>, <a href="tracker_help_el_GR.pdf" target="_blank">????????</a>, <a href="tracker_help_it.pdf" target="_blank">Italiano</a> or <a href="tracker_help_pt_BR.pdf" target="_blank">Portuguese</a>.</p> 
     <p>You can run Tracker from a USB drive without installing it on the host computer. See <a href="portable_tracker.html">Installing Tracker on a USB or other portable drive</a> for more information.</p>

     <h3>Tracker Features</h3> 
     <p><font color="#007c00"><strong>Tracking:</strong></font></p> 
     <ul> 
      <li>Manual and automated object tracking with position, velocity and acceleration overlays and data. </li>
      <li>Center of mass tracks. </li>
      <li>Interactive graphical vectors and vector sums. </li>
      <li>RGB line profiles at any angle, time-dependent RGB regions. </li>
     </ul> 
     <p><font color="#007c00"><strong>Modeling:</strong></font></p> 
     <ul> 
      <li>Model Builder creates kinematic and dynamic models of point mass particles and two-body systems. </li>
      <li>External models animate and overlay multi-point data from separate modeling programs such as spreadsheets and <a href="http://fem.um.es/Ejs/" target="_blank">EjsS simulations</a>. </li>
      <li>Model overlays are automatically synchronized and scaled to the video for direct visual comparison with the real world. </li>
     </ul> 
     <p><font color="#007c00"><strong>Video:</strong></font></p> 
     <ul> 
      <li>Free Xuggle video engine plays and records most formats (mov/avi/flv/mp4/wmv etc) on Windows/OSX/Linux. </li>
      <li>Video filters, including brightness/contrast, strobe, ghost trails, and deinterlace filters. </li>
      <li>Perspective filter corrects distortion when objects are photographed at an angle rather than straight-on. </li>
      <li>Radial distortion filter corrects distortion associated with fisheye lenses. </li>
      <li>Export Video wizard enables editing and transcoding videos, with or without overlay graphics, using Tracker itself. </li>
      <li>Video Properties dialog shows video dimensions, path, frame rate, frame count, more. </li>
     </ul> 
     <p><font color="#007c00"><strong>Data generation and analysis:</strong></font></p> 
     <ul> 
      <li>Fixed or time-varying coordinate system scale, origin and tilt. </li>
      <li>Multiple calibration options: tape, stick, calibration points and/or offset origin. </li>
      <li>Switch easily to center of mass and other reference frames. </li>
      <li>Protractors and tape measures provide easy distance and angle measurements. </li>
      <li>Circle fitter tool fits circles to 3 or more points, steps or tracks. </li>
      <li>Define custom variables for plotting and analysis. </li>
      <li>Add editable text columns for comments or manually entered data. </li>
      <li>Data analysis tool includes powerful automatic and manual curve fitting. </li>
      <li>Export formatted or raw data to a delimited text file or the clipboard. </li>
      <li>Display measured values using custom number formats if desired. </li>
     </ul> 
     <p><font color="#007c00"><strong>Digital Library resources:</strong></font></p> 
     <ul> 
      <li>OSP Digital Library Browser provides easy access to online collections of videos and Tracker resources. </li>
      <li>Use the DL Browser to find resources by name, author, keywords or other metadata. </li>
      <li>Use the DL Browser to create, edit and share your own video and resource libraries. </li>
      <li>Use Tracker to export self-documenting ZIP resources ideal for the DL Browser. </li>
     </ul> 
     <p><font color="#007c00"><strong>Other:</strong></font></p> 
     <ul> 
      <li>Full undo/redo with multiple steps. </li>
      <li>Page view displays html instructions or student notes. </li>
      <li>Pencil tool draws colored lines (arrows, circles, annotations) directly on the video. </li>
      <li>User preferences: GUI configuration, video engine, default language, font size, more. </li>
     </ul> 
     <h3>Tracker Video Tutorials</h3> 
     <p>These video tutorials can help extend your knowledge of Tracker.</p> 
     <ol> 
      <li><a href="https://www.youtube.com/watch?v=n4Eqy60yYUY" target="_blank">Tracker Quick Start</a> A quick demonstration of how to use Tracker to measure the acceleration of a falling ball. If using Tracker for the first time, watch both this and <a href="https://www.youtube.com/watch?v=La3H7JywgX0" target="_blank">Getting Started with Tracker</a>. </li>
      <li><a href="https://www.youtube.com/watch?v=La3H7JywgX0" target="_blank">Getting Started with Tracker</a> A more extended explanation of how to use Tracker to track and analyze a moving object in a video. </li>
      <li><a href="https://www.youtube.com/watch?v=Dn0Zz7rtkZw" target="_blank">Tracker Autotracker Tutorial</a> How to use the autotracker. This starts out with the basics but also explains in detail how the autotracker works and how to control it for best results. </li>
      <li><a href="https://www.youtube.com/watch?v=tU9Ui5U2VLk" target="_blank">Saving and Sharing Tracker Experiments</a> How to (a) save Tracker experiments by exporting Tracker zip (TRZ) files and (b) share them by building Digital Library Collections. </li>
     </ol>
