           <div class="description">
             The Tracker Video Analysis and Modeling Tool allows students to model and analyze the motion of objects in videos. By overlaying simple dynamical models directly onto videos, students may see how well a model matches the real world. Interference patterns and spectra can also be analyzed with Tracker.
            <br>
            <br>Tracker 4.11.0 installers are available on Linux, Mac OS X, and Windows and include the Xuggle open source video engine.
            <br>
            <ul>
             <li><b><a href="http://www.compadre.org/osp/images/tracker/Tracker-4.11.0-windows-installer.exe">Tracker 4.11.0 Windows Installer</a></b></li>
             <li><b><a href="http://www.compadre.org/osp/images/tracker/Tracker-4.11.0-osx-installer.zip">Tracker 4.11.0 Mac OS X Installer</a></b> - <a href="http://physlets.org/tracker/installers/installer_help.html#osx">Instructions</a></li>
             <li><b><a href="http://www.compadre.org/osp/images/tracker/Tracker-4.11.0-linux-32bit-installer.run">Tracker 4.11.0 Linux 32-bit Installer</a></b> - <a href="http://physlets.org/tracker/installers/installer_help.html#linux">Instructions</a></li>
             <li><b><a href="http://www.compadre.org/osp/images/tracker/Tracker-4.11.0-linux-64bit-installer.run">Tracker 4.11.0 Linux 64-bit Installer</a></b> - <a href="http://physlets.org/tracker/installers/installer_help.html#linux">Instructions</a></li>
            </ul>Tracker is an Open Source Physics tool built on the OSP code library. Additional Tracker resources, demonstration experiments, and videos, can be found by 
            <a href="http://www.compadre.org/osp/search/search.cfm?qc=Tracker">searching ComPADRE for "Tracker."</a>
            <br>
            <br>Additional Tracker resources including Tracker help and sample videos are available from the Tracker home page at Cabrillo College below.
            <br> 
            <br>Please note that this resource requires at least version 1.6 of Java (JRE).
