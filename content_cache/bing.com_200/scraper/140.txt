    <h1 class="post-title">Recommended GitHub Atom Packages for Web Developers</h1> 
    <span class="post-date">17 Mar 2014</span> 
    <p><img src="/assets/images/2014/Mar/logo_9624cbf4ef800b9827861e67259e1f5c.png" alt=""> I’ve been using GitHub’s <a href="https://atom.io/">Atom</a> editor for the past couple of weeks and I’ve really enjoyed it. At first there weren’t very many extensions/packages available, but the community has been quick to fill in that gap.</p> 
    <p>You can browse the packages <a href="https://atom.io/packages">online</a> or from Atom’s Preferences… (?,)</p> 
    <p><img src="/assets/images/2014/Mar/Screenshot_2014_03_18_23_02_24.png" alt=""></p> 
    <h2 id="official-packages">Official Packages</h2> 
    <p>GitHub has included serveral packages in the base install of Atom and in addition they’ve created other helpful packages as well. Some of the more helpful packages I’ve found are…</p> 
    <ul> 
     <li><a href="https://atom.io/packages/markdown-preview">Markdown Preview</a> - “Show the rendered HTML markdown to the right of the current editor using ctrl-shift-m”</li> 
    </ul> 
    <p><img src="/assets/images/2014/Mar/5b1c2ae8_9e7e_11e3_9d93_3fa7caae4710.png" alt=""></p> 
    <ul> 
     <li> <p><a href="https://atom.io/packages/vim-mode">Vim Mode</a> - “Provides vim modal control for Atom, ideally blending the best of vim and Atom.”</p> </li> 
     <li> <p><a href="https://atom.io/packages/editor-stats">Editor Stats</a> - “View a graph of your mouse and keyboard activity for the last 6 hours.”</p> </li> 
    </ul> 
    <p><img src="/assets/images/2014/Mar/843b1172_9e57_11e3_9c60_8d28d542f39c.png" alt=""></p> 
    <ul> 
     <li><a href="https://atom.io/packages/emmet">Emmet</a> - “Emmet is a plugin for many popular text editors which greatly improves HTML &amp; CSS workflow”</li> 
    </ul> 
    <h2 id="community-packages">Community Packages</h2> 
    <p>In addition, the community has really stepped up and have written a lot of packages in a very short amount of time.</p> 
    <ul> 
     <li> <p><a href="https://atom.io/packages/javascript-snippets">JavaScript Snippets</a> by <a href="https://twitter.com/zenorocha">@zenorocha</a></p> </li> 
     <li> <p><a href="https://atom.io/packages/csscomb">CSSComb</a> - “Sort properties of one or more CSS/SCSS/Less rules using CSScomb.” by <a href="https://twitter.com/wbruce">@wbruce</a></p> </li> 
     <li> <p><a href="https://atom.io/packages/csslint">CSSLint</a> - “CSSLint error reports for your Atom editor.” by <a href="https://twitter.com/tcarlsen">@tcarlsen</a></p> </li> 
    </ul> 
    <p><img src="/assets/images/2014/Mar/preview.jpg" alt=""></p> 
    <ul> 
     <li><a href="https://atom.io/packages/highlight-line">Highlight Line</a> - “Highlights the current line in the editor; really simple.” by <a href="https://twitter.com/richrace">@richrace</a></li> 
    </ul> 
    <p><img src="/assets/images/2014/Mar/687474703a2f2f692e696d6775722e636f6d2f666133325774722e706e67.png" style="width: 700px;"></p> 
    <ul> 
     <li><a href="https://atom.io/packages/autocomplete-plus">Autocomplete +</a> - “View and insert possible completions in the editor while typing” by <a href="https://twitter.com/rattazong">@rattazong</a></li> 
    </ul> 
    <p><img src="/assets/images/2014/Mar/687474703a2f2f7331342e64697265637475706c6f61642e6e65742f696d616765732f3134303330342f79377237673564662e676966.gif" style="width: 700px;"></p> 
    <ul> 
     <li><a href="https://atom.io/packages/color-picker">Color Picker</a> - “A color picker for Atom. Either right click a color and select Color picker, or place the cursor on a color and hit CMD-SHIFT-C. Currently reads HEXA, RGBA, RGB and HEX.” by <a href="https://twitter.com/ThomasLindstr_m">@ThomasLindstr_m</a></li> 
    </ul> 
    <p><img src="/assets/images/2014/Mar/687474703a2f2f662e636c2e6c792f6974656d732f314a3233327a32533131315932473273314134352f6f75747075742e676966.gif" alt=""></p> 
    <ul> 
     <li><a href="https://atom.io/packages/autoprefixer">Autoprefixer</a> - “Prefix CSS with Autoprefixer” by <a href="https://twitter.com/sindresorhus">@sindresorhus</a></li> 
    </ul> 
    <p><img src="/assets/images/2014/Mar/51b999b2_9fce_11e3_9e9d_5e6a9cb4e933.gif" alt=""></p> 
    <ul> 
     <li> <p><a href="https://atom.io/packages/jshint">JSHint</a> - “Validate JavaScript with JSHint” by <a href="https://twitter.com/sindresorhus">@sindresorhus</a></p> </li> 
     <li> <p><a href="https://atom.io/packages/uglify">Uglify</a> - “Minify JavaScript with Uglify” by <a href="https://twitter.com/sindresorhus">@sindresorhus</a></p> </li> 
     <li> <p><a href="https://atom.io/packages/editorconfig">EditorConfig</a> - “EditorConfig helps developers maintain consistent coding styles between different editors” by <a href="https://twitter.com/sindresorhus">@sindresorhus</a></p> </li> 
    </ul> 
    <p><img src="/assets/images/2014/Mar/dfe40cb4_a3f6_11e3_862f_894999973373.png" alt=""></p> 
    <h2 id="and-just-for-fun">..and Just for Fun!</h2> 
    <ul> 
     <li><a href="https://atom.io/packages/asteroids">Asteroids</a> - “…spawn an Asteroids shooter on any web page and then blast away” by <a href="https://twitter.com/mojombo/">@mojombo</a></li> 
    </ul> 
    <p><img src="/assets/images/2014/Mar/895ea42e_a30d_11e3_96e0_1bcd606a318c.gif" style="width: 700px;"></p>
