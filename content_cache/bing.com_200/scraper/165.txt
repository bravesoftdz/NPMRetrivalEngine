         <p> </p>
         <h2> Installation</h2> 
         <p></p> 
         <p> &nbsp; &nbsp;<u><i>IRobot Visual Web Scraper</i></u> </p>
         <ol> 
          <li> Download the <font size="-1">IRobot Visual Web Scraper <a href="http://www.irobotsoft.com/irobot-eval.rar">rar</a> or <a href="http://www.irobotsoft.com/irobot-eval.zip">zip</a> </font> package and save it in a temporary directory; </li>
          <li> Extract all files into a local directory from the rar package; </li>
          <li> Run the irobot.exe; </li>
         </ol> 
         <p></p> 
         <p> &nbsp; &nbsp;<u><i>IRobotX installation</i></u> </p>
         <ol> 
          <li> Register the IRobotX.OCX by: (For ActiveX programming and Google Gadgets)<br> c:\irobot\&gt; regsvr32 IRobotX.OCX. </li>
         </ol> 
         <p></p>
