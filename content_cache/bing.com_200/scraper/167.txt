                <meta content="http://ayudasprogramacionweb.blogspot.com/2012/11/compilar-y-ejecutar-java-en-cmd-windows.html"> <a class="timestamp-link" href="http://ayudasprogramacionweb.blogspot.com.ar/2012/11/compilar-y-ejecutar-java-en-cmd-windows.html" rel="bookmark" title="permanent link"> <time class="published" datetime="2012-11-07T17:47:00+01:00" title="2012-11-07T17:47:00+01:00"> noviembre 07, 2012 </time> </a> </span>

             <div class="post-body entry-content float-container" id="post-body-7875018813425117927">
               Para aquellos que habéis utilizado algún 
              <a href="http://ayudasprogramacionweb.blogspot.com/2011/12/que-es-un-ide-y-cual-elegir.html" target="_blank">IDE de desarrollo</a> para JAVA, sabréis que este simplifica mucho el proceso de compilación y ejecución de nuestros programas, tan solo presionando un botón ya se crean nuestros archivos de Bytecode .
              <i>class </i>y .
              <i>jar</i>, y directamente se ejecutan en la consola del IDE.
              <br> 
              <br> Yo empecé con JAVA de esta manera, con el IDE Netbeans y pasado un tiempo, decidí pasarme a programar con Notepad++ para no tener las cosas tan fáciles y aprenderme de memoria por lo menos las clases más importantes.
              <br> 
              <br> 
              <div class="separator" style="clear: both; text-align: center;"> 
               <a href="http://1.bp.blogspot.com/-5-4d7eim9g0/UJqOlhDpQtI/AAAAAAAAAhY/Sz07HBpMcAA/s1600/java.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" src="http://1.bp.blogspot.com/-5-4d7eim9g0/UJqOlhDpQtI/AAAAAAAAAhY/Sz07HBpMcAA/s1600/java.png"></a>
              </div> 
              <br> Sea como fuere, tuve que empezar a compilar y ejecutar mis programas en la consola CMD de Windows, algo muy sencillo, pero que me acostó aprender ya que no encontraba información por ningún lado de cómo hacerlo.&nbsp;Lo primero que debemos hacer es compilar nuestro archivo .
              <i>java</i>, para convertirlo en código reconocible por la máquina virtual de JAVA, para ello utilizaremos el comando 
              <b>javac</b>.
              <br> 
              <br> 
              <a name="more"></a>
              <b><u>NOTA IMPORTANTE</u></b>
              <br> Si al introducir 
              <b>javac </b>en vuestro CMD, este no lo reconoce, debemos modificar la variable de entorno del sistema 
              <b>Path</b> de la siguiente manera:
              <br> 
              <ul> 
               <li>En el <b>panel de control</b> accedemos a <b>Sistema</b>, y en la parte izquierda de la ventana que aparecerá pinchamos en <b>Configuración avanzada del sistema.</b></li> 
              </ul> 
              <div class="separator" style="clear: both; text-align: center;"> 
               <a href="http://2.bp.blogspot.com/-Ues6nGP6KKA/UJqLzxiSmpI/AAAAAAAAAhA/9QWtqzNaMKA/s1600/asdf1.jpg" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" height="95" src="http://2.bp.blogspot.com/-Ues6nGP6KKA/UJqLzxiSmpI/AAAAAAAAAhA/9QWtqzNaMKA/s400/asdf1.jpg" width="400"></a>
              </div> 
              <div> 
               <br>
              </div> 
              <ul> 
               <li>En la nueva ventanita, pinchamos en Variables de entorno que está en la parte inferior.&nbsp;</li> 
               <li>En la parte de debajo de la nueva ventana, en la sección Variables del sistema, buscamos y hacemos doble click en la variable llamada Path, para editarla.&nbsp;</li> 
               <li>Por último solo nos queda añadir al final del valor de la variable, un punto y coma y la ruta de vuestra carpeta \bin que está dentro del directorio del JDK, y este a su vez suele estar en C:\archivos de programa o C:\program files como veis que es mi caso.</li> 
              </ul> 
              <div class="separator" style="clear: both; text-align: center;"> 
               <a href="http://2.bp.blogspot.com/-md5hmljfJVI/UJqMDmCV4EI/AAAAAAAAAhI/P2O8vPOlGUM/s1600/asdf2.jpg" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" height="180" src="http://2.bp.blogspot.com/-md5hmljfJVI/UJqMDmCV4EI/AAAAAAAAAhI/P2O8vPOlGUM/s400/asdf2.jpg" width="400"></a>
              </div> 
              <div> 
               <br>
              </div> 
              <ol> 
              </ol> Una vez hecho esto, aceptamos todas las ventanitas y si teníamos un CMD abierto, lo cerramos y volvemos a abrir otro, de esta manera al introducir 
              <b>javac</b>, este debería responder, puedes probarlo con el comando 
              <b>javac version</b>, el cual debería devolver la versión instalada de java.
              <br> 
              <b><u>FIN NOTA</u></b>
              <br> 
              <br> Con los comandos del JDK operativos, procedemos a compilar nuestro archivo .
              <i>java </i>introduciendo el comando 
              <b style="background-color: white;">javac NombreArchivo.java</b>, de esta manera, si el programa está bien, y no nos sale ningún error de compilación, se generará nuestro archivo .
              <i>class </i>que es nuestro programa en java compilado.
              <br> 
              <br> Por último solo nos queda ejecutar nuestro programita con el comando
              <span style="background-color: white;"> <b>java NombreClase</b></span>, ten en cuenta que el nombre de la clase debe coincidir con el nombre del archivo, que ambos deben empezar por una letra mayúscula y que al ejecutar el programa 
              <b>con el comando java, NO se pone la extensión del archivo</b>, tan solo el nombre de la clase pública que contiene el archivo .
              <i>class</i>, que será igual a su nombre.

            <h3 class="post-title"><a href="http://ayudasprogramacionweb.blogspot.com.ar/2012/12/ordenar-cualquier-array-con-sort-en-java.html">Ordenar cualquier array con SORT en JAVA</a></h3> 
            <div class="post-header"> 
             <div class="post-header-line-1"> 
              <span class="byline post-timestamp"> 
               <meta content="http://ayudasprogramacionweb.blogspot.com/2012/12/ordenar-cualquier-array-con-sort-en-java.html"> <a class="timestamp-link" href="http://ayudasprogramacionweb.blogspot.com.ar/2012/12/ordenar-cualquier-array-con-sort-en-java.html" rel="bookmark" title="permanent link"> <time class="published" datetime="2012-12-17T19:25:00+01:00" title="2012-12-17T19:25:00+01:00"> diciembre 17, 2012 </time> </a> </span> 
             </div> 
            </div> 
            <div class="item-content float-container"> 
             <div class="popular-posts-snippet snippet-container r-snippet-container"> 
              <div class="snippet-item r-snippetized">
                En entradas anteriores ya aprendimos a 
               <a href="http://ayudasprogramacionweb.blogspot.com/2012/12/ordenar-cualquier-array-de-numeros-java.html" target="_blank">ordenar un array numérico</a> con la 
               <b>ordenación de burbuja</b> (Bubble sort). En este caso facilitaremos mucho las cosas ya que en la API&nbsp;estándar de JAVA podemos encontrar
               <b> el método <i>sort</i></b> de la clase 
               <i>Arrays</i>, que está en el paquete 
               <i>java.util</i> que 
               <b>se encargará de ordenar cualquier tipo de array </b>que le pasemos como argumento.
               <br>
               <br> Su uso se verá mejor con el siguiente ejemplo:
               <br>import java.util.Arrays; public class Main { public static void main(String[] args) { //Array de String String[] nombres = {"Pepe", "Juan", "Alex", "Julian", "Francisco", "Luis"}; //Ordena el array Arrays.sort(nombres); //Mostramos el array ya ordenado for (String i : nombres) { System.out.print(i + ", "); } } }
