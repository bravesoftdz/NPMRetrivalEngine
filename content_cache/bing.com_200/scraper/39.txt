        <pre><span class="pl-s1"><span class="pl-smi">$instagram</span> <span class="pl-k">=</span> <span class="pl-c1">Instagram</span><span class="pl-k">::</span>withCredentials(<span class="pl-s"><span class="pl-pds">'</span>username<span class="pl-pds">'</span></span>, <span class="pl-s"><span class="pl-pds">'</span>password<span class="pl-pds">'</span></span>);</span>

<span class="pl-s1"><span class="pl-c"><span class="pl-c">//</span> And then you will be able to query instagram with newly updated methods. (Notice that these methods are not static anymore)</span></span>
<span class="pl-s1"></span>
<span class="pl-s1"><span class="pl-smi">$user</span> <span class="pl-k">=</span> <span class="pl-smi">$instagram</span><span class="pl-k">-&gt;</span>getAccountById(<span class="pl-c1">3</span>);</span>
<span class="pl-s1"><span class="pl-smi">$medias</span> <span class="pl-smi">$instagram</span><span class="pl-k">-&gt;</span>getLocationTopMediasById(<span class="pl-c1">1</span>)</span>
<span class="pl-s1"><span class="pl-smi">$medias</span> <span class="pl-k">=</span> <span class="pl-smi">$instagram</span><span class="pl-k">-&gt;</span>getLocationMediasById(<span class="pl-c1">1</span>);</span>
<span class="pl-s1"><span class="pl-smi">$location</span> <span class="pl-smi">$instagram</span><span class="pl-k">-&gt;</span>getLocationById(<span class="pl-c1">1</span>);</span>
<span class="pl-s1"><span class="pl-smi">$medias</span> <span class="pl-k">=</span> <span class="pl-smi">$instagram</span><span class="pl-k">-&gt;</span>getTopMediasByTagName(<span class="pl-s"><span class="pl-pds">'</span>hello<span class="pl-pds">'</span></span>);</span></pre> 
        <p>Be carefull with login method. I am planning to implement session caching soon</p>

        <h3><a id="user-content-get-account-info" class="anchor" href="#user-content-get-account-info" rel="nofollow noopener external"></a>Get account info</h3> 
        <pre><span class="pl-s1"><span class="pl-smi">$account</span> <span class="pl-k">=</span> <span class="pl-c1">Instagram</span><span class="pl-k">::</span>getAccount(<span class="pl-s"><span class="pl-pds">'</span>kevin<span class="pl-pds">'</span></span>);</span>
<span class="pl-s1"><span class="pl-c"><span class="pl-c">/*</span></span></span>
