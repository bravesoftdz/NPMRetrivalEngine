        <h1 class="entry-title">Web Scraping with&nbsp;Perl</h1> 
        <div class="entry-meta"> 
         <span class="sep">Posted on </span>
         <a href="https://teusje.wordpress.com/2010/05/02/web-scraping-with-perl/" title="3:41 pm" rel="bookmark"><time class="entry-date" datetime="2010-05-02T15:41:40+00:00">May 2, 2010</time></a>

       <div class="entry-content"> 
        <p>We need to scrape data <em>(web scraping)</em> from some websites with Perl for a school project.</p> 
        <p>Here is a simple script that I used to test the <a href="http://search.cpan.org/~miyagawa/Web-Scraper-0.32/lib/Web/Scraper.pm">Web-Scraper package</a> that can be found in <a href="http://search.cpan.org/">CPAN</a>.</p> 
        <p>This is how the code works:</p> 
        <p>First you have to find a website that contains your data that you want. I used the <a href="http://www.uciprotour.com/templates/UCI/UCI2/layout.asp?MenuId=MTU4MzI&amp;LangId=1">UCI ProTour website</a>:</p> 
        <p><a href="https://teusje.files.wordpress.com/2010/05/webscrape-1.png"><img data-attachment-id="2249" data-permalink="https://teusje.wordpress.com/2010/05/02/web-scraping-with-perl/webscrape-1/" data-orig-file="https://teusje.files.wordpress.com/2010/05/webscrape-1.png?w=584" data-orig-size="947,607" data-comments-opened="1" data-image-meta="{&quot;aperture&quot;:&quot;0&quot;,&quot;credit&quot;:&quot;&quot;,&quot;camera&quot;:&quot;&quot;,&quot;caption&quot;:&quot;&quot;,&quot;created_timestamp&quot;:&quot;0&quot;,&quot;copyright&quot;:&quot;&quot;,&quot;focal_length&quot;:&quot;0&quot;,&quot;iso&quot;:&quot;0&quot;,&quot;shutter_speed&quot;:&quot;0&quot;,&quot;title&quot;:&quot;&quot;}" data-image-title="webscrape 1" data-image-description="" data-medium-file="https://teusje.files.wordpress.com/2010/05/webscrape-1.png?w=584?w=300" data-large-file="https://teusje.files.wordpress.com/2010/05/webscrape-1.png?w=584?w=584" class="alignnone size-full wp-image-2249" title="webscrape 1" alt="" src="https://teusje.files.wordpress.com/2010/05/webscrape-1.png?w=584" srcset="https://teusje.files.wordpress.com/2010/05/webscrape-1.png?w=584 584w, https://teusje.files.wordpress.com/2010/05/webscrape-1.png?w=150 150w, https://teusje.files.wordpress.com/2010/05/webscrape-1.png?w=300 300w, https://teusje.files.wordpress.com/2010/05/webscrape-1.png?w=768 768w, https://teusje.files.wordpress.com/2010/05/webscrape-1.png 947w" sizes="(max-width: 584px) 100vw, 584px"></a></p> 
        <p>If you look at the source code you will notice that my data is stored in a table:</p> 
        <p><a href="https://teusje.files.wordpress.com/2010/05/webscrape-2.png"><img data-attachment-id="2250" data-permalink="https://teusje.wordpress.com/2010/05/02/web-scraping-with-perl/webscrape-2/" data-orig-file="https://teusje.files.wordpress.com/2010/05/webscrape-2.png?w=584" data-orig-size="911,344" data-comments-opened="1" data-image-meta="{&quot;aperture&quot;:&quot;0&quot;,&quot;credit&quot;:&quot;&quot;,&quot;camera&quot;:&quot;&quot;,&quot;caption&quot;:&quot;&quot;,&quot;created_timestamp&quot;:&quot;0&quot;,&quot;copyright&quot;:&quot;&quot;,&quot;focal_length&quot;:&quot;0&quot;,&quot;iso&quot;:&quot;0&quot;,&quot;shutter_speed&quot;:&quot;0&quot;,&quot;title&quot;:&quot;&quot;}" data-image-title="webscrape 2" data-image-description="" data-medium-file="https://teusje.files.wordpress.com/2010/05/webscrape-2.png?w=584?w=300" data-large-file="https://teusje.files.wordpress.com/2010/05/webscrape-2.png?w=584?w=584" class="alignnone size-full wp-image-2250" title="webscrape 2" alt="" src="https://teusje.files.wordpress.com/2010/05/webscrape-2.png?w=584" srcset="https://teusje.files.wordpress.com/2010/05/webscrape-2.png?w=584 584w, https://teusje.files.wordpress.com/2010/05/webscrape-2.png?w=150 150w, https://teusje.files.wordpress.com/2010/05/webscrape-2.png?w=300 300w, https://teusje.files.wordpress.com/2010/05/webscrape-2.png?w=768 768w, https://teusje.files.wordpress.com/2010/05/webscrape-2.png 911w" sizes="(max-width: 584px) 100vw, 584px"></a></p> 
        <p>The data doesn’t have to be in a table, but it makes life just easier for this example.</p> 
        <p>This is a part of the Perl code:</p> 
        <pre class="brush: perl; title: ; notranslate" title="">
 # we will save the urls from the teams
 process "table#UCITeamList &gt; tr &gt; td &gt; a", 'urls[]' =&gt; '@href';
 # we will save the team names
 process "table#UCITeamList &gt; tr &gt; td &gt; a", 'teams[]' =&gt; 'TEXT';
</pre> 
        <p>This is the essential part of using the Web-Scraper module. The <strong>#</strong><em>UCITeamList</em> refers to the <strong>ID</strong> in the HTML <strong>(&lt;table id=UCITeamList …)</strong>. If you wanted to locate a piece of HTML that contains a class instead of an ID than you should use a point <strong>.</strong> instead of <strong>#</strong>. It’s the same thing like using IDs and classes in CSS.</p> 
        <p>Example:</p> 
        <pre class="brush: perl; title: ; notranslate" title="">
process "table.WithBorder &gt; ...
</pre> 
        <p>The next thing is getting the actual data:</p> 
        <p><strong>@</strong> = an attribute</p> 
        <p>Example:</p> 
        <pre class="brush: perl; title: ; notranslate" title="">
#  &lt;img src="www.test.com/myfile.jpg" title="Hello" /&gt;
  'titles[]' =&gt; '@title';
</pre> 
        <p><strong>TEXT</strong> = the part between two tags</p> 
        <p>Example:</p> 
        <pre class="brush: perl; title: ; notranslate" title="">
#  &lt;p&gt;Hello, hey&lt;/p&gt;
  'paragra[]' =&gt; 'TEXT';
</pre> 
        <p>The other part of the code just loops over the array with my scraped data and prints it to the screen and saves it into a file.</p> 
        <p>Also each web scraped <a href="http://www.uciprotour.com/Modules/SUCI/TEAMS/TeamDetails.asp?id=Mg&amp;RefDate=02.05.2010&amp;MenuId=MTU4MzI&amp;LangId=1&amp;BackLink=%2Ftemplates%2FUCI%2FUCI2%2Flayout.asp%3FMenuId%3DMTU4MzI%26LangId%3D1">URL in my array</a> is scraped again. The <a href="http://www.uciprotour.com/Modules/SUCI/TEAMS/TeamDetails.asp?id=Mg&amp;RefDate=02.05.2010&amp;MenuId=MTU4MzI&amp;LangId=1&amp;BackLink=%2Ftemplates%2FUCI%2FUCI2%2Flayout.asp%3FMenuId%3DMTU4MzI%26LangId%3D1">URLs contain the team details</a> with all riders.</p> 
        <p><strong>Summary:</strong></p> 
        <p>I scrape one page that contains all the teams and links to the team details. After that first scrape round I want the team detail data so I scrape each URL. Just like a mini crawler ?</p> 
        <p>Web-Scraper (or Web::Scraper) is a very powerful package but don’t abuse this. Web Scraping can be illegal!</p> 
        <p>For <a href="http://search.cpan.org/~miyagawa/Web-Scraper-0.32/lib/Web/Scraper.pm">more information, documentation and examples</a> check out <a href="http://search.cpan.org/~miyagawa/Web-Scraper-0.32/lib/Web/Scraper.pm">CPAN</a>.</p> 
        <p>(By the way: if you want to test the Web-Scraper, please use another website instead of the UCI site because they will not like this I guess ? ? )</p> 
        <p><strong>Here is the code:</strong></p> 
        <pre class="brush: perl; title: ; notranslate" title="">
#!/usr/bin/perl
use warnings;
use strict;
use URI;
use Web::Scraper;

open FILE, "&gt;file.txt" or die $!;

# website to scrape
my $urlToScrape = "http://www.uciprotour.com/templates/UCI/UCI2/layout.asp?MenuId=MTU4MzI&amp;LangId=1";

# prepare data
my $teamsdata = scraper {
 # we will save the urls from the teams
 process "table#UCITeamList &gt; tr &gt; td &gt; a", 'urls[]' =&gt; '@href';
 # we will save the team names
 process "table#UCITeamList &gt; tr &gt; td &gt; a", 'teams[]' =&gt; 'TEXT';
};
# scrape the data
my $res = $teamsdata-&gt;scrape(URI-&gt;new($urlToScrape));

# print the second field (the teamname)
for my $i (0 .. $#{$res-&gt;{teams}}) {
 if ($i%3 != 0 &amp;&amp; $i%3 != 2) {
 print $res-&gt;{teams}[$i];
 print "\n";
 print FILE $res-&gt;{teams}[$i];
 print FILE "\n";
 }
}

print FILE "\n";

# loop over every team url and take all scrape all the riders from each team
for my $i ( 0 .. $#{$res-&gt;{urls}}) {
 if ($i%3 != 0 &amp;&amp; $i%3 != 2) {
 print "\n\n";
 print $res-&gt;{teams}[$i];
 print "\n------------------\n";
 print FILE "\n\n";
 print FILE $res-&gt;{teams}[$i];
 print FILE "\n------------------\n";

 # prepare data
 my $rennersdata = scraper {
 # rider name
 process "table#TeamRiders &gt; tr &gt; td.RiderCol &gt; a", 'renners[]' =&gt; 'TEXT';
 # rider country
 process "table#TeamRiders &gt; tr &gt; td.CountryCol &gt; a", 'landrenner[]' =&gt; 'TEXT';
 # rider birthdate
 process "table#TeamRiders &gt; tr &gt; td.DOBCol &gt; a", 'geboortedatums[]' =&gt; 'TEXT';
 # team address
 process "table#TeamLeft &gt; div.AddLine", 'AddressLines[]' =&gt; 'TEXT';
 };
 # scrape
 my $res2 = $rennersdata-&gt;scrape(URI-&gt;new($res-&gt;{urls}[$i]));

 for my $j (0 .. $#{$res2-&gt;{renners}}) {
 # print rider name
 print $res2-&gt;{renners}[$j];
 print "\n";
 print FILE $res2-&gt;{renners}[$j];
 print FILE "\n";

 }
 }
 # DONT FORGET THIS, this will make your script slow
 # but if it's not there you will be "attacking" the webserver and they don't like that
 sleep(3);
}

# close the file
close FILE;

</pre> 
        <p>Enjoy ?</p> 
        <p><strong>Update 18/02/2013:</strong><br> User Wisnoskij <a href="https://teusje.wordpress.com/2010/05/02/web-scraping-with-perl/#comment-3320">suggested an optimization</a>. (Thanks!)<br> This code:</p> 
        <pre class="brush: perl; title: ; notranslate" title="">
 # we will save the urls from the teams
 process "table#UCITeamList &gt; tr &gt; td &gt; a", 'urls[]' =&gt; '@href';
 # we will save the team names
 process "table#UCITeamList &gt; tr &gt; td &gt; a", 'teams[]' =&gt; 'TEXT';
</pre> 
        <p>can be replaced with a single line:</p> 
        <pre class="brush: perl; title: ; notranslate" title="">
 # we will save the urls from the teams and the team names
process “table#UCITeamList &gt; tr &gt; td &gt; a”, 'urls[]' =&gt; ‘@href’, 'teams[]' =&gt; ‘TEXT’;
</pre>
