        <h2 id="parser-for-javascript">Parser for JavaScript</h2>
        <p><a href="http://travis-ci.org/erikd/language-javascript"><img src="https://secure.travis-ci.org/erikd/language-javascript.png?branch=master" alt="Build Status"></a></p>
        <p>Based (loosely) on language-python</p>
        <h2 id="two-versions">Two Versions</h2>
        <p>There are currently two versions:</p>
        <ul>
         <li><p>0.5 series : Is a continuation of the 0.5.X.Y series, from the [master] (https://github.com/erikd/language-javascript/tree/master) branch of this github repository.</p></li>
         <li><p>0.6 series : This has a vastly different and improved AST which makes if far more difficult to build an non-sensical Javascript AST. This code is in the <a href="https://github.com/erikd/language-javascript/tree/new-ast">new-ast</a> branch of this github repository.</p></li>
        </ul>
        <h2 id="how-to-build">How to build</h2>
        <p>Library:</p>
        <pre><code>cabal clean &amp;&amp; cabal configure &amp;&amp; cabal build</code></pre>
        <p>Tests:</p>
        <pre><code>cabal clean &amp;&amp; cabal configure -fbuildtests &amp;&amp; cabal build</code></pre>
        <p>Running the tests</p>
        <pre><code>./dist/build/runtests/runtests</code></pre>
        <p>To debug the grammar</p>
        <pre><code>happy -iparse.txt -g -a  -d src/Language/JavaScript/Parser/Grammar5.y</code></pre>
        <p>This generates src/Language/JavaScript/Parser/Grammar5.hs, delete this when done with the debug version</p>
        <h2 id="utf8unicode-version">UTF8/Unicode version</h2>
        <p>Alex 3.0 now supports unicode natively, and has been included as a dependency in the cabal file.</p>
        <p>Note: The generation of the lexical analyser has been separated out, to remove the install-time dependency on Alex. If any changes need to be made to the lexer, the Lexer.x source lies in src-dev, and the runalex.sh script will invoke Alex with the appropriate directories.</p>
