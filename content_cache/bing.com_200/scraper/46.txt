      <h2>What types of data can be scraped?</h2> 
      <p>The Web Scraper can scrape data from any part of a web page whether it is the content of a HTML element such as a div or span, a HTML element attribute, any web page meta data or text stored in a image or PDF document.</p> 
      <h2>How does the Web Scraper work?</h2> 
      <p>GrabzIt's Web Scraper reads the web as normal users see it by using a special browser that enables the Web Scraper to scrape dynamic as well as static HTML. This means any content generated with JavaScript or AJAX can still be scraped. Not only that but the Web Scraper can also scrape <a href="examples/scrapepdfs.aspx">content from PDF documents</a> and can even read <a href="examples/extracttextfromimages.aspx">text from images</a>.</p> 
      <iframe src="https://www.youtube.com/embed/9MpMLrNKLOY?modestbranding=1&amp;autohide=1&amp;showinfo=0&amp;controls=0&amp;loop=1&amp;playlist=9MpMLrNKLOY" class="LeftAlign Video" frameborder="0" allowfullscreen></iframe> 
      <p>The Web Scraper also allows you to click on links and buttons submit forms, select options on the website and much more, allowing a website to be accessed in the same way as a normal user. Once you have selected an element some web scrapers insist you create complicated regular expressions to scrape the exact piece of data you are interested in. Instead we enable you to <a href="examples/patterns.aspx">use patterns</a> we then create the regular expression in the background to scrape the data for you.</p> 
      <p>Your data is accessible in real-time and can be output in several different formats so that you can integrate it into your application as easily as possible. These formats include Excel, XML, CSV, JSON, HTML and SQL for either MySQL or SQL Server. But how do you interact with this data? You can either have it sent to you or a location of your choice, or you can use the <a href="../support/?uniqueId=SendScrapedResultsViaCallbackURL">Callback URL option</a> which allows you to use our <a href="api/">API</a> and automate the entire scrape process. Especially as you can configure a scrape to run on a regular schedule meaning you will always have the latest information!</p> 
      <p>The Web Scraper comes with an excellent online Wizard to automatically create instructions that identifies what content to scrape. Meaning you shouldn't have to write any code, or very little! But we don't want to stop there and are always trying to improve our Web Scraper make it the simplest on the web.</p> 
      <p>Many websites store similar content across many pages, so to get all of the data you want GrabzIt's Web Scraper can crawl a website and search for content that matches your scrape instructions anywhere on the site. Alternatively you can specify the <a href="?uniqueId=SeedURLs">exact web pages</a> you want to scrape or just specify a <a href="../support/?uniqueId=TargetWebsite">sub section of the site</a> to be crawled. We even provide a free monthly scrape allowance, <a href="scrape.aspx">so try it now</a> risk free!</p>
