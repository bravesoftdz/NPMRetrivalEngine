       <h1 class="title"> Validate a possible credit card number using C++ </h1> 
       <div class="line"> 
        <div class="entry-info">

       <p>To write a program which helps validate a possible credit card number you will need to know two things. Some basic programming skills and the Luhn algorithm.</p> 
       <h4>A short background on the Luhn algorithm</h4> 
       <p>According to online sources the Luhn algorithm or modulus&nbsp;10 was created by IBM scientist Hans Peter Luhn. The end result of his discovery, patented in 1960, is one that adds a level of integrity to a number which&nbsp;we can use for error reduction. A number known to be generated with this algorithm can be tested for errors&nbsp;after transmitting or recording using this simple method. These types of algorithms act a bit like hidden signatures. An alteration to a number generated with this algorithm is very very likely to change the signature and this change indicates the error. Thats basically it. All credit card numbers are defined with this algorithm built in.</p> 
       <p>Here&nbsp;it is in action using possible card number&nbsp;4388576018410707 as an example:</p> 
       <ol> 
        <li><span style="line-height: 1.6em;">Beginning from the right, double every second digit (even digits). &nbsp;If doubling results in a two digit number then add the single digits of the two digit number to get a single digit result. &nbsp;In our example: [0x2=<strong>0</strong>], [0x2=<strong>0</strong>], [4×2=<strong>8</strong>], [1×2=<strong>2</strong>], [6×2=12, 1+2=<strong>3</strong>], [5×2=10, 1+0=<strong>1</strong>], [8×2=16, 1+6=<strong>7</strong>], [4×2=<strong>8</strong>] &nbsp;These single digit results which Ive highlighted in bold are then summed [0+0+8+2+3+1+7+8=<strong>29</strong>]</span></li> 
        <li>Beginning again from right to left add all the of odd digits. In our example: [7+7+1+8+0+7+8+3=<strong>41</strong>]</li> 
        <li>Sum the results from step 1 and step 2. [29+41=<strong>70</strong>]</li> 
       </ol> 
       <p>If the result from step 3 is divisible by 10 then your number is considered valid!</p> 
       <p>The following is a simple program which performs this check.</p>

//Author : Daniel Touma (100147753)
//Version  : 0.1   
//Name     : Credit Card Number Validator
//Description : Checks for a numbers adherence to the Luhn algorithm

#include &lt;iostream&gt;
#include &lt;string&gt;

using namespace std;

int main()
{   
    string cardNumber; //this will carry the number given by the user.
    int sum(0), sumOfDoubleEven(0), sumOfOdd(0), evenDigit(0), oddDigit(0);

    cout &lt;&lt; "Welcome to the Credit Card Number Validator." &lt;&lt; endl; 
    cout &lt;&lt; "This program checks for a numbers adherence to the Luhn algorithm" &lt;&lt; endl;

    //Main loop
    do
    {
        cout &lt;&lt; endl &lt;&lt; "Please provide a number to validate or type 'n' to quit ";
        cin &gt;&gt; cardNumber; //gets cardNumber from user
        if (cardNumber == "n") break; //exits loop on user request

        //this for loop repeats once for each digit of the cardNumber. 
        //digitPosition decrements by one on each pass from last position to first position
        for (int digitPosition=cardNumber.length(); digitPosition&gt;0; digitPosition--)
        {
            if (digitPosition%2 == 0)
            {   //executes the following code if digitPosition is even
                oddDigit=(int)cardNumber[digitPosition-1]-'0';
                sumOfOdd=sumOfOdd+oddDigit;
            }
            else
            {   //executes the following code if digitPosition is odd.
                evenDigit=((int)cardNumber[digitPosition-1]-'0')*2;
                evenDigit=evenDigit%10+evenDigit/10;
                sumOfDoubleEven=sumOfDoubleEven + evenDigit;
            }
        }

        sum=sumOfOdd+sumOfDoubleEven; //sums the result
        cout &lt;&lt; endl &lt;&lt;"The number your provided is ";
        if (sum%10==0) //executes if sum is divisible by 10
            cout &lt;&lt; "valid" &lt;&lt; endl;
        else //executes if sum is not divisible by 10
            cout &lt;&lt; "invalid" &lt;&lt; endl;

    }while (cardNumber != "n");

    return 0;
