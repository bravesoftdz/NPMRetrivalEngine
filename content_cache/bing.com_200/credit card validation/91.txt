       <h1 class="entry-title">Java Credit Card&nbsp;Validation</h1> 
       <div class="entry-content"> 
        <p><a href="http://www.almaelder.wordpress.com">Back To Blog</a></p> 
        <p>I’m going to talk about a credit card validation program that I did during college. It basically takes in a value and evaluates whether or not it is a valid credit card format and if it is it then checks whether or not it is a valid card number. This program doesn’t use OOP and doesn’t have an interface. All inputs are entered on the command line. The program has two valid card types that it checks for, either Mastercard or Visa.<br> <strong><span style="text-decoration:underline;">The Code</span></strong></p> 
        <blockquote>
         <p>import java.util.Scanner;</p> 
         <p>public class CreditCard<br> {<br> public static void main(String[]args)<br> {<br> boolean verified = false;<br> String cNum,cType;<br> String masterPattern =”51|52|53|54|55?;</p> 
         <p>Scanner in = new Scanner(System.in);</p> 
         <p>System.out.println(“Enter card number:”);<br> cNum = in.nextLine();<br> System.out.println(“Enter card type:”);<br> cType = in.nextLine();</p> 
         <p>if(cType.equalsIgnoreCase(“Mastercard”) &amp;&amp; cNum.length() == 16 &amp;&amp; cNum.substring(0,2).matches(masterPattern))<br> verified = validate(cNum);<br> else if(cType.equalsIgnoreCase(“Visa”) &amp;&amp; (cNum.length() – 13 == 0 || cNum.length() – 16 ==0) &amp;&amp; cNum.substring(0,1).equals(“4”))<br> verified = validate(cNum);<br> else<br> System.out.println(“Invalid Card”);</p> 
         <p>if(!(verified))<br> System.out.println(“Invalid Credit Card”);<br> else<br> System.out.println(“Valid Credit Card”);<br> }</p> 
         <p>public static boolean validate(String cardNum)<br> {<br> int [] odds = new int[8];<br> int j, oddsTotal=0;<br> int evenTotals=0,index,result;<br> boolean v = true;</p> 
         <p>for(index = 1,j=0;index&lt;cardNum.length();index=index+2,j++)<br> {<br> odds[j] = Integer.parseInt(cardNum.substring(index,index+1));<br> odds[j] = odds[j] * 2;<br> if(odds[j] &gt;= 10)<br> odds[j] = odds[j]%10+1;<br> oddsTotal = oddsTotal + odds[j];<br> }</p> 
         <p>for(index=0;index&lt;cardNum.length();index=index+2)<br> evenTotals = evenTotals + Integer.parseInt(cardNum.substring(index,index+1));</p> 
         <p>result = oddsTotal + evenTotals;</p> 
         <p>if(result%10==0)<br> v = true;<br> else<br> v = false;</p> 
         <p>return v;<br> }<br> }</p>
        </blockquote> 
        <p>As you can see because I didn’t use an GUI I needed to use the Scanner class to be able to enter data into the program. Because I have the import statement at the top of the program I’m able to create instances of the Scanner class and use the methods that that class has.</p> 
        <p>As I mentioned above the program has two types of card that are validated. Each of these types also have different number formats.For Mastercard, the format is that the first two digits must be either 51, 52, 53, 54 or 55 and the number must be 16 digits long. For Visa, the format is that the first digit must be a 4 and the card number must be either 13 or 16 digits long.</p> 
        <blockquote>
         <p>String masterPattern =”51|52|53|54|55?;</p>
        </blockquote> 
        <p>I set up a basic pattern so that I’d be able to check the first two digits of any incoming number. There was no need to do the same for Visa because if it were a valid visa type it’s always going to start with 4.</p> 
        <blockquote>
         <p>if(cType.equalsIgnoreCase(“Mastercard”) &amp;&amp; cNum.length() == 16 &amp;&amp; cNum.substring(0,2).matches(masterPattern))<br> verified = validate(cNum);<br> else if(cType.equalsIgnoreCase(“Visa”) &amp;&amp; (cNum.length() – 13 == 0 || cNum.length() – 16 ==0) &amp;&amp; cNum.substring(0,1).equals(“4”))<br> verified = validate(cNum);<br> else<br> System.out.println(“Invalid Card”);</p>
        </blockquote> 
        <p>Once the user has supplied their input the flow control of the program is determined by an if-else construct. The first checks for a valid Mastercard, the second for a valid Visa and the third is the default if neither of the previous statements return true. However, if either does return true a method called “validate()” is called and the card number is passed in as an argument. The validate() method is checking if the number itself is valid as apposed to the format of the number.</p> 
        <blockquote>
         <p>public static boolean validate(String cardNum)<br> {<br> int [] odds = new int[8];<br> int j, oddsTotal=0;<br> int evenTotals=0,index,result;<br> boolean v = true;</p> 
         <p>for(index = 1,j=0;index&lt;cardNum.length();index=index+2,j++)<br> {<br> odds[j] = Integer.parseInt(cardNum.substring(index,index+1));<br> odds[j] = odds[j] * 2;<br> if(odds[j] &gt;= 10)<br> odds[j] = odds[j]%10+1;<br> oddsTotal = oddsTotal + odds[j];<br> }</p> 
         <p>for(index=0;index&lt;cardNum.length();index=index+2)<br> evenTotals = evenTotals + Integer.parseInt(cardNum.substring(index,index+1));</p> 
         <p>result = oddsTotal + evenTotals;</p> 
         <p>if(result%10==0)<br> v = true;<br> else<br> v = false;</p> 
         <p>return v;<br> }</p>
        </blockquote> 
        <p>The above method takes every number that is in an odd position (for example the number at positions 1, 3, 5 etc ) and multiples it by two. If the result of that multiplication is greater or equal to 10,&nbsp; its modulus of 10 is found and 1 is added. The total of all the odd positioned numbers is then found. The evenly positioned numbers are a whole lot easier to process as all you do is find there sum total. The two totals are then added together. If the modulus of 10 of this number equals zero then your card number is valid and true is returned from the method. Otherwise false is returned.<br> Processing then goes back to the main method. Depending on what boolean value was returned, a message to the user is displayed detailing whether or not the card is valid.</p> 
        <p>So that’s how you would validate a Mastercard or Visa. The validate() method can be used to check any valid card format as long as the format is correct. You could easily add further card types to this. For example, I believe American Express must start with a 3 and be 15 digits long (You should verify that yourself if you plan on adding it!).</p> 
        <p>If you have anything to add please leave a comment.</p>
