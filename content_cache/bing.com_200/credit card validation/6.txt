        <h1 class="entry-title" itemprop="headline">Java Credit Card Validation – Luhn Algorithm in Java</h1>
        <p class="entry-meta"><time class="entry-modified-time" itemprop="dateModified" datetime="2017-08-22T08:42:06+00:00">August 22, 2017</time> by <span class="entry-author" itemprop="author" itemscope itemtype="https://schema.org/Person"><a href="https://www.journaldev.com/author/pankaj" class="entry-author-link" itemprop="url" rel="author"><span class="entry-author-name" itemprop="name">Pankaj</span></a></span> <span class="entry-comments-link"><a href="https://www.journaldev.com/1443/java-credit-card-validation-luhn-algorithm-java#respond">Leave a Comment</a></span></p>
       </header>

        <p>Recently I came to know that <strong>Credit Card numbers</strong> are not random and passes <strong>Luhn Algorithm</strong> test.</p>
        <h2>Java Credit Card Validation</h2>
        <p>Any credit card number should pass following test:</p>
        <ol>
         <li>From the rightmost digit, we should double every second digit. If the double is greater than 9, then add the both digits so that final number is of single digit.</li>
         <li>Now sum all the digits in the number, the unchanged numbers and the doubled numbers.</li>
         <li>The final sum should be multiple of 10 or mod 10 of the number should be 0. If it’s not then its not a valid credit card number.</li>
        </ol>
        <p>Let’s check it with an example credit card number 12345678903555.</p>
        <div style="float:none;margin:10px 0 10px 0;text-align:center;">
         <ins class="adsbygoogle" style="display:block" data-ad-client="ca-pub-5143693538742787" data-ad-slot="5697562523" data-ad-format="auto"></ins> 
         <script>(adsbygoogle=window.adsbygoogle||[]).push({});</script> 
        </div>
        <p>Digits are : 1,2,3,4,5,6,7,8,9,0,3,5,5,5<br> After doubling : 2,2,6,4,1,6,5,8,9,0,6,5,1,5<br> Sum of digits : 2+2+6+4+1+6+5+8+9+0+6+5+1+5 = 60 = 6*10 and hence a valid credit card number.</p>
        <h3>Luhn Algorithm in Java</h3>
        <p>Here I am providing java Luhn Algorithm program to validate credit card numbers.</p>
        <div style="float:none;margin:10px 0 10px 0;text-align:center;">
         <div id="482285384">

        </div>
        <pre class="prettyprint">
package com.journaldev.util;

public class JavaLuhnAlgorithm {

	public static void main(String[] args) {
		validateCreditCardNumber("12345678903555");
		String imei = "012850003580200";
		validateCreditCardNumber(imei);
	}

	private static void validateCreditCardNumber(String str) {

		int[] ints = new int[str.length()];
		for (int i = 0; i &lt; str.length(); i++) {
			ints[i] = Integer.parseInt(str.substring(i, i + 1));
		}
		for (int i = ints.length - 2; i &gt;= 0; i = i - 2) {
			int j = ints[i];
			j = j * 2;
			if (j &gt; 9) {
				j = j % 10 + 1;
			}
			ints[i] = j;
		}
		int sum = 0;
		for (int i = 0; i &lt; ints.length; i++) {
			sum += ints[i];
		}
		if (sum % 10 == 0) {
			System.out.println(str + " is a valid credit card number");
		} else {
			System.out.println(str + " is an invalid credit card number");
		}
	}

}
</pre>
        <p>Here I am taking input to the validation method as String, so that it will work also where the first digit is 0.</p>
