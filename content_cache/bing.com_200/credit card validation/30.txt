                    <h3 class="post-title entry-title"> Simple Credit Card Validating Program In Java </h3> 
                    <div class="post-header"> 
                     <div class="post-header-line-1"></div> 
                    </div> 
                    <div class="post-body entry-content" id="post-body-6948555436595710490">
                      Today I am going to do a simple Java program that will say if the given credit card number is Valid or not, its from my course book exercise, Here I am putting the code and will explain the codes based on my own knowledge.
                     <br> 
                     <br> So the Question was:
                     <br> 
                     <br> 
                     <br> 
                     <pre><span class="Apple-style-span" style="font-family: Verdana, sans-serif;"><b>Credit card numbers follow certain patterns. A credit card number must have between 13 and 16 digits. It must start with:</b></span></pre> 
                     <pre><span class="Apple-style-span" style="font-family: Verdana, sans-serif;"><b> ? 4 for Visa cards ? 5 for Master cards ? 37 for American Express cards ? 6 for Discover cards</b></span></pre> 
                     <pre><span class="Apple-style-span" style="font-family: Verdana, sans-serif;"><b> In 1954, Hans Luhn of IBM proposed an algorithm for validating credit card numbers. The algorithm is useful to determine whether a card number is entered correctly or whether a credit card is scanned correctly by a scanner. All credit card numbers are generated following this validity check, commonly known as the Luhn check or the Mod 10 check, which can be described as follows (for illus- tration, consider the card number 4388576018402626): 1. Double every second digit from right to left. If doubling of a digit results in a two-digit number, add up the two digits to get a single-digit number.</b></span></pre> 
                     <pre><span class="Apple-style-span" style="font-family: Verdana, sans-serif;"><b>2*2= 4 2*2= 4 4*2= 8 1*2= 2 6 * 2 = 12 (1 + 2 = 3)&nbsp;</b></span></pre> 
                     <pre><span class="Apple-style-span" style="font-family: Verdana, sans-serif;"><b>5 * 2 = 10 (1 + 0 = 1)&nbsp;</b></span></pre> 
                     <pre><span class="Apple-style-span" style="font-family: Verdana, sans-serif;"><b>8 * 2 = 16 (1 + 6 = 7)&nbsp;</b></span></pre> 
                     <pre><span class="Apple-style-span" style="font-family: Verdana, sans-serif;"><b>4*2= 8 </b></span></pre> 
                     <pre><span class="Apple-style-span" style="font-family: Verdana, sans-serif;"><b>2. Now add all single-digit numbers from Step 1. 4 + 4 + 8 + 2 + 3 + 1 + 7 + 8 = 37 </b></span></pre> 
                     <pre><span class="Apple-style-span" style="font-family: Verdana, sans-serif;"><b>3. Add all digits in the odd places from right to left in the card number. 6 + 6 + 0 + 8 + 0 + 7 + 8 + 3 = 38 </b></span></pre> 
                     <pre><span class="Apple-style-span" style="font-family: Verdana, sans-serif;"><b>4. Sum the results from Step 2 and Step 3. 37 + 38 = 75 </b></span></pre> 
                     <code><span class="Apple-style-span" style="font-family: Verdana, sans-serif;"><b> </b></span></code>
                     <br> 
                     <pre><code><span class="Apple-style-span" style="font-family: Verdana, sans-serif;"><b>5. If the result from Step 4 is divisible by 10, the card number is valid; otherwise, it is invalid. For example, the number 4388576018402626 is invalid, but the number 4388576018410707 is valid.</b></span></code></pre> 
                     <code><span class="Apple-style-span" style="font-family: Verdana, sans-serif;"><b> </b></span></code>
                     <br> 
                     <pre></pre> 
                     <code> </code>
                     <br> 
                     <pre><code><span class="Apple-style-span" style="font-family: Verdana, sans-serif;"><b>Write a program that prompts the user to enter a credit card number as a long integer. Display whether the number is valid or invalid. Design your program to use the following methods:</b></span></code></pre> 
                     <code> </code>
                     <br> 
                     <pre></pre> 
                     <code> </code>
                     <br> 
                     <pre><code>        &nbsp;</code></pre> 
                     <code> <pre></pre> <pre><span class="Apple-style-span" style="color: #0b5394;">/** Return true if the card number is valid */
public static boolean isValid(long number)</span></pre> <pre><span class="Apple-style-span" style="color: #0b5394;">        &nbsp;</span></pre> <pre><span class="Apple-style-span" style="color: #0b5394;">/** Get the result from Step 2 */
public static int sumOfDoubleEvenPlace(long number)

/** Return this number if it is a single digit, otherwise, return * the sum of the two digits */</span></pre> <pre><span class="Apple-style-span" style="color: #0b5394;">public static int getDigit(int number)</span></pre> <pre><span class="Apple-style-span" style="color: #0b5394;">
</span></pre> <pre><span class="Apple-style-span" style="color: #0b5394;">
</span></pre> <pre><span class="Apple-style-span" style="color: #0b5394;">/** Return sum of odd place digits in number */</span></pre> <pre><span class="Apple-style-span" style="color: #0b5394;">public static int sumOfOddPlace(long number)
</span></pre> <pre><span class="Apple-style-span" style="color: #0b5394;">
</span></pre> <pre><span class="Apple-style-span" style="color: #0b5394;">
</span></pre> <pre><span class="Apple-style-span" style="color: #0b5394;">/** Return true if the digit d is a prefix for number */
public static boolean prefixMatched(long number, int d)</span></pre> <pre><span class="Apple-style-span" style="color: #0b5394;">&nbsp;</span></pre> <pre></pre> <pre><span class="Apple-style-span" style="color: #0b5394;">

</span></pre> <pre><span class="Apple-style-span" style="color: #0b5394;">
</span></pre> <pre><span class="Apple-style-span" style="color: #0b5394;">
</span></pre> </code>
                     <br> 
                     <pre><code><span class="Apple-style-span" style="color: #0b5394;">/** Return the first k number of digits from number. If the * number of digits in number is less than k, return number. */ public static long getPrefix(long number, int k)</span>
</code>
</pre> 
                     <pre></pre> The program is written below 
                     <br> 
                     <br> 
                     <br> 
                     <pre><span class="Apple-style-span" style="color: #0b5394;">package creditcardvalidation;

import java.util.Scanner;

/**
 *
 * @author pradeepghimire
 */
public class CreditCardValidation {

    /**
     * @param args the command line arguments
     */
    public static boolean isValid(long number) {

        int total = sumOfDoubleEvenPlace(number) + sumOfOddPlace(number);

        if ((total % 10 == 0) &amp;&amp; (prefixMatched(number, 1) == true)) {
            return true;
        } else {
            return false;
        }
    }

    public static int getDigit(int number) {

        if (number &lt;= 9) {
            return number;
        } else {
            int firstDigit = number % 10;
            int secondDigit = (int) (number / 10);

            return firstDigit + secondDigit;
        }
    }

    public static int sumOfOddPlace(long number) {
        int result = 0;

        while (number &gt; 0) {
            result += (int) (number % 10);
            number = number / 100;
        }

        return result;
    }

    public static int sumOfDoubleEvenPlace(long number) {

        int result = 0;
        long temp = 0;

        while (number &gt; 0) {
            temp = number % 100;
            result += getDigit((int) (temp / 10) * 2);
            number = number / 100;
        }

        return result;
    }

    public static boolean prefixMatched(long number, int d) {

        if ((getPrefix(number, d) == 3)
                || (getPrefix(number, d) == 4)
                || (getPrefix(number, d) == 5)
                || (getPrefix(number, d) == 6)) {

            if (getPrefix(number, d) == 3) {
                System.out.println("\nAmerican Express Card ! ");
            } else if (getPrefix(number, d) == 4) {
                System.out.println("\nVisa Card ! ");
            } else if (getPrefix(number, d) == 5) {
                System.out.println("\nMaster Card !");
            } else if (getPrefix(number, d) == 6) {
                System.out.println("\nDiscover Card !");
            }

            return true;
        
        } else {

            return false;

        }
    }

    public static int getSize(long d) {

        int count = 0;

        while (d &gt; 0) {
            d = d / 10;

            count++;
        }

        return count;

    }

    /**
     * Return the first k number of digits from number. If the number of digits
     * in number is less than k, return number.
     */
    public static long getPrefix(long number, int k) {

        if (getSize(number) &lt; k) {
            return number;
        } else {

            int size = (int) getSize(number);

            for (int i = 0; i &lt; (size - k); i++) {
                number = number / 10;
            }

            return number;

        }

    }

    public static void main(String[] args) {

        Scanner sc = new Scanner(System.in);

        System.out.print("Enter your Card Number : ");

        long input = sc.nextLong();

        if (isValid(input) == true) {
            System.out.println("\n*****Your card is Valid*****");
        } else {
            System.out.println("\n!!!!Your Card is not Valid !!!!! ");
        }

    }
