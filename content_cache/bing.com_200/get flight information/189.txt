     <h2 class="accented">Remarks</h2> 
     <p>Use the <a class="xref" href="windows.applicationmodel.package#Windows_ApplicationModel_Package_Status" data-linktype="relative-path">Package.Status</a> property to get the <a class="xref" href="windows.applicationmodel.packagestatus" data-linktype="relative-path">PackageStatus</a>. The <a class="xref" href="windows.applicationmodel.package#Windows_ApplicationModel_Package_Status" data-linktype="relative-path">Package.Status</a> and all of the properties of the <a class="xref" href="windows.applicationmodel.packagestatus" data-linktype="relative-path">PackageStatus</a> class are read-only, so you cannot change their values. To get updated information about the status of the package, access the <a class="xref" href="windows.applicationmodel.package#Windows_ApplicationModel_Package_Status" data-linktype="relative-path">Package.Status</a> property again to get a new instance of the <a class="xref" href="windows.applicationmodel.packagestatus" data-linktype="relative-path">PackageStatus</a> class.</p>
