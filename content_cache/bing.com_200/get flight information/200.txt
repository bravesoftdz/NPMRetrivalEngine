           <p>The APIs offer you to display realtime data on flight status, delays, and even shopping services data</p>

             <p><strong>FlightStats</strong><br><em>Flex APIs lets you develop apps listing schedule, airport status, delays, alerts, weather forecast, and details about a particular airline. This gives a free trail for a 30-day period</em></p> 
             <p><a href="https://developer.flightstats.com/getting-started/">Flightstats Flex APIs</a> provides a developer with complete coverage of flight status information. One can use these APIs with REST, supporting JSON, JSONP, XML formats and SOAP. The Flex APIs lets a developer develop apps listing schedule, Airport status, delays, alerts, weather forecast, and details about a particular airline. FlightStats gives a free trail for a 30-day period so that developers can use their APIs and evaluate them against their application features.&nbsp;</p> 
             <p><strong>FlightAware</strong><br><em>Allows you to create apps displaying live and historical data about flight status. This allows one to write apps using .NET, PHP, Java, Ruby, Perl, etc</em></p> 
             <p><a href="http://flightaware.com/">FlightAware</a> are XML based flight APIs that helps a developer to create apps displaying live and historical data about flight status, route, weather conditions, airport statuses and much more. A developer can use SOAP/WSDL or REST/JSON&nbsp;to write queries using FlightXMl APIs. It allows one to write apps using .NET, PHP, Java, Ruby, Perl and many others. One can use these APIs to send push notifications as well. Alerts can be written using either SOAP/WSDL or REST/JSON interface, but one will receive the response in JSON format only.&nbsp;</p> 
             <p><strong>Absolute Aviation Advantage</strong><br><em>Provides flight status information from 48 hours prior to scheduled departure to 24 hours after actual arrival</em><em>&nbsp;</em></p> 
             <p><a href="http://www.oag.com/Aviation-Data/Flight-Status/OAG-Flight-Status-Ondemand">OAG</a> provides instant access to the flight status information by providing web services/APIs in XML/SOAP format. Flight status information consists of information about delays, cancellation, diversions, actual departure and arrival of flights. OAG provides flight status information from 48 hours prior to scheduled departure to 24 hours after actual arrival.&nbsp;</p> 
             <p>&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;&nbsp;<a href="http://www.itnext.in/articles/1003456/9-flight-apis-you-can-create-apps-on-to-get-live-flight-data/page/2">N<strong>EXT PAGE</strong></a></p> 
             <p>&nbsp;</p> 
             <p>
              <!-- pagebreak --></p> 
             <!--cms:title>Page 2</cms:title-->
             <p><strong>QPX Express API</strong><br><em>This can come handy to a developer who is developing an app about ticket booking or shopping or both. You can raise 50 queries per day for free</em></p> 
             <p><a href="https://developers.google.com/qpx-express/">QPX Express&nbsp;</a>is a product of Google providing APIs about airfares, shopping services and real-time booking information. This can come handy to a developer who is developing an app about ticket booking or shopping or both. One can easily implement the APIs using the JSON interface. You can raise 50 queries per day for free and after that one has to pay US $ 0.035 for each query raised.&nbsp;</p> 
             <p><strong>Travel Fusion flight API</strong><br><em>Aggregates information from hundreds of airlines, providing agents, e-commerce platforms, search and mobile services with a single system to manage, search, book and support this content</em></p> 
             <p><a href="http://www.travelfusion.com/corporate/page/tf-flight-api">Travel Fusion flight API&nbsp;</a>provides booking details for 150 low cost carriers and scheduled airlines. You can easily create a ticket booking application based on XML APIs provided by travel Fusion.&nbsp;</p> 
             <p><strong>FlightLookup APIs</strong><br><em>FlightLookup uses global flight schedules and the ancillary information required to create a comprehensive list of flight schedules related data</em></p> 
             <p><a href="https://developer.flightlookup.com/">FlightLookUP&nbsp;</a>provides APIs with the information on flight schedules, airport timetable, flight metadata, flight routing, terminal information and more. These data feeds are available in XML or JSON interface. Developers can use these feeds to develop their own applications using CURL, Java, NODE, PHP, Python, Ruby, .NET and Objective-C. One can use the APIs for free if One are getting 100 queries per month.&nbsp;</p> 
             <p>&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;&nbsp;</p> 
             <p>&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;<a href="http://www.itnext.in/articles/1003456/9-flight-apis-you-can-create-apps-on-to-get-live-flight-data/page/3">N<strong>EXT PAGE</strong></a></p> 
             <p> &nbsp;</p> 
             <p>
              <!-- pagebreak --></p> 
             <!--cms:title>Page 3</cms:title-->
             <p><strong>Travel Boutique online</strong><br><em>The APIs contains information about real time booking, cancellation, air fare calendar and more</em></p> 
             <p><a href="http://www.travelboutiqueonline.com/Flight_api.aspx">Travel Boutique&nbsp;</a>helps developer in creating an airline booking system by providing required APIs for the same. The APIs contains information about real time booking, cancellation, air fare calendar and more.</p> 
             <p><strong>Cleartrip Flight API</strong><br><em>The Cleartrip Flights API allows a user to search and book flights using a REST based service</em></p> 
             <p><a href="http://www.cleartrip.com/api/docs/air-api/">Cleartrip&nbsp;</a>provides one with the APIs that allows one to search and book airline tickets. These APIs works on a REST based interface and all the communication is done using XML. A flight search request is made through a simple HTTP GET command. Bookings can be done by creating an itinerary which contains all the details about flights, pricing, passenger contact and payment details.</p> 
             <p><strong>Wego Flights API</strong><br><em>Wego Flights API allows clients to search for flight fares and availabilty in real-time across Wego's inventory of partners</em></p> 
             <p> <a href="http://support.wan.travel/hc/en-us/articles/200191669-Wego-Flights-API">Wego&nbsp;</a>provide developers with the APIs that allow the developing app to search flights and book them as well. A search request is made through HTTP POST format and one will get the response in JSON format.</p>
