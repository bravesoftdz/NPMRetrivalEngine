     <h2>Release Notes</h2> 
     <p><br>Color coded trace info on debug screen<br>Autoload templates, auto naming of child controls templates and template startup content. Debug listing of pages, controls and methods.<br>Visual Data in debug mode<br>Pages and Controls. Creates file structure per your settings automatically.<br>No-code server component usage.<br>Server Components.<br>Components. Added "Tile" component to demonstrate.<br>Built-in "Refresh" method on all "data" objects with callback<br>Support for jquery-te ("textarea" edittype)<br>Refined creation of data modules.<br>Define api name and path, automatically contructs data objects (e.g. Apps.Data.MyData).<br>Added initialization of data modules. <br></p>
