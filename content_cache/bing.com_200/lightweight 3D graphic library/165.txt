      <h1>HTML5 Engines</h1> 
     </div> 
     <p>Everyone needs a good HTML5 game engine to work with. No point re-inventing the wheel. We curate a list of HTML5 game engines that are roaming the web, so you can make the ultimate decision yourself.</p> 
     <p><a href="http://impactjs.com" target="_blank">ImpactJS</a> is a strong, all rounder commercial ready HTML5 game engine. Its very well supported and easily extendable. Examples include hit indie games such as <a href="http://store.steampowered.com/app/293440/" target="_blank">Elliots Quest</a>, which did extremely well via Steam. Major brands such as Disney, Nickelodeon, Budlight, Amazon and Time Warner have deployed games built using ImpactJS.</p> 
     <p><img class="alignnone size-full wp-image-636" src="http://html5gamedevelopment.com/wp-content/uploads/2016/03/elliot-screen.png" alt="elliot-screen" width="600" height="337" srcset="http://html5gamedevelopment.com/wp-content/uploads/2016/03/elliot-screen.png 600w, http://html5gamedevelopment.com/wp-content/uploads/2016/03/elliot-screen-300x169.png 300w" sizes="(max-width: 600px) 100vw, 600px"></p> 
     <p>&nbsp;</p> 
     <p><a href="http://phaser.io" target="_blank">Phaser</a> is an open source HTML5 game engine, thats gaining a strong momentum within the developer community,</p> 
     <p><img class="alignnone size-full wp-image-734" src="http://html5gamedevelopment.com/wp-content/uploads/2016/03/Feat-2.png" alt="Feat" width="480" height="320" srcset="http://html5gamedevelopment.com/wp-content/uploads/2016/03/Feat-2.png 480w, http://html5gamedevelopment.com/wp-content/uploads/2016/03/Feat-2-300x200.png 300w" sizes="(max-width: 480px) 100vw, 480px"></p> 
     <p>&nbsp;</p> 
     <p>King, the developers of hit mobile titles such as Candy Crush, have released a 2D game engine for use recently. <a href="http://html5gamedevelopment.com/defold-game-engine-from-king/">The engine, Defold,</a> which boasts quick and easy development that is highly optimized, is now free to use. From a single code base, the Defold game engine can create games for six platforms including web with HTML5.</p> 
     <p><img class="alignnone size-large wp-image-862" src="http://html5gamedevelopment.com/wp-content/uploads/2016/03/Defold-1-1024x432.png" alt="Defold 1" width="600" height="253" srcset="http://html5gamedevelopment.com/wp-content/uploads/2016/03/Defold-1-1024x432.png 1024w, http://html5gamedevelopment.com/wp-content/uploads/2016/03/Defold-1-300x127.png 300w, http://html5gamedevelopment.com/wp-content/uploads/2016/03/Defold-1-768x324.png 768w, http://html5gamedevelopment.com/wp-content/uploads/2016/03/Defold-1-600x253.png 600w, http://html5gamedevelopment.com/wp-content/uploads/2016/03/Defold-1.png 1347w" sizes="(max-width: 600px) 100vw, 600px"></p> 
     <p>&nbsp;</p> 
     <p><a href="http://html5gamedevelopment.com/createjs-modular-tool-system/">CreateJS</a> is not similar to a lot of other HTML5 game engines. CreateJS is a package of modular tools and libraries that work together or independently for your project. These include EaselJS, which is a canvas library and SoundJS&nbsp;which as the name implies is for sound.</p> 
     <p><img class="alignnone size-full wp-image-834" src="http://html5gamedevelopment.com/wp-content/uploads/2016/03/Create-Feat.png" alt="Create Feat" width="480" height="320" srcset="http://html5gamedevelopment.com/wp-content/uploads/2016/03/Create-Feat.png 480w, http://html5gamedevelopment.com/wp-content/uploads/2016/03/Create-Feat-300x200.png 300w" sizes="(max-width: 480px) 100vw, 480px"></p> 
     <p>To view the full list of HTML5 game engines, <a href="http://html5gamedevelopment.com/category/html5-engines/">click here</a>.</p>
