        <h1 class="title" id="page-title">Java scientific software</h1> 
        <div class="region region-help"> 
         <div id="block-system-main" class="block block-system first last odd"> 
          <div class="content"> 
           <div id="node-1" class="node node-page view-mode-full clearfix" about="/main/node/1" typeof="foaf:Document"> 
            <div class="content"> 
             <div class="field field-name-body field-type-text-with-summary field-label-hidden">
              <div class="field-items">
               <div class="field-item even" property="content:encoded">
                <p>Here is a list of most popular scientific software written in Java:</p> 
                <h2> Scientific and Educational Java software</h2> 
                <ul>
                 <li> <a href="http://jabref.sourceforge.net/">JabRef </a> - bibliography reference manager</li> 
                 <li> <a href="http://jwork.org/scavis/">SCaVis</a> - a general-purpose data analysis computational framework in Java</li> 
                 <li> <a href="http://jaxodraw.sourceforge.net/">JaxoDraw</a> - a program for drawing Feynman diagrams</li> 
                 <li> <a href="http://www.cs.waikato.ac.nz/ml/weka/">Weka</a> - data mining software</li> 
                 <li> <a href="http://rsb.info.nih.gov/ij/">ImageJ</a> - image processing package</li> 
                 <li> <a href="http://pixelitor.sourceforge.net/">Pixelitor</a> - another impressive image processing package</li> 
                 <li> <a href="http://www.um.es/fem/EjsWiki/pmwiki.php">EJS</a> - easy Java simulations</li> 
                 <li> <a href="http://www.dragmath.bham.ac.uk/">DragMath </a> - equation editor</li> 
                 <li> <a href="http://cs.gmu.edu/~eclab/projects/ecj/">ECJ </a> - evolutionary computation research system</li> 
                 <li> <a href="http://jsci.sourceforge.net/">JSci</a> - scientific libraries</li> 
                 <li> <a href="http://java.freehep.org/">FreeHEP</a> - Java numerical libraries (included in <a href="http://jwork.org/jhepwork">jHepWork</a>)</li> 
                 <li> <a href="http://jscience.org/">JScience</a> - library for scientific community</li> 
                 <li> <a href="http://acs.lbl.gov/~hoschek/colt/">Colt</a> - high-performance scientific libraries (included in <a href="http://jwork.org/jhepwork">jHepWork</a>)</li> 
                 <li> <a href="http://code.google.com/p/encog-java/">Encog</a> - a neural network</li> 
                 <li> <a href="http://sourceforge.net/projects/joone/">Joone</a> - another neural network framework</li> 
                 <li> <a href="http://neuroph.sourceforge.net/">Neuroph</a> - a neural network framework</li> 
                 <li> <a href="http://jwork.org/jterm/">JTerm</a> - a Linux-like terminal implemented in Java</li> 
                 <li> <a href="http://home.att.net/~bsimpson/nvj.html">Night Vision</a> - a planetarium program</li> 
                 <li> <a href="http://sourceforge.net/projects/graxml/">GraXML</a> - geometric Modeller for handling High Energy Physics experiments</li> 
                 <li> <a href="http://sourceforge.net/projects/jam-daq/">Jam</a> - data acquisition and analysis system</li> 
                 <li> <a href="http://jwork.org/jslovo/">JSlovo</a> - powerful dictionary program for online dictionary files</li> 
                 <li> <a href="http://jwork.org/jslovo/">ASlovo</a> - same as JSlovo but for Android platform</li> 
                 <li> <a href="http://iwp.sourceforge.net">Interactive Web Physics</a> - physics animation and problem designer tool</li> 
                 <li> <a href="http://www.falstad.com/mathphysics.html">Interactive Physics Applets</a> - Physics applets for education</li> 
                 <li> <a href="http://jersey.uoregon.edu/vlab/">Virtual Physics</a> - Physics applets</li> 
                 <li> <a href="http://vo.iucaa.ernet.in/~voi/voplot.htm">VOPlot</a> - Visualizing astronomical data</li> 
                 <li> <a href="http://jwork.org/science/">JavaScience</a> - a portal to send Java scientific applets</li> 
                 <li> <a href="http://www.epic.noaa.gov/java/sgt/">Scientific Graphics Toolkit</a> - for creating interactive graphics applications</li> 
                 <li> <a href="http://www.ee.ucl.ac.uk/~mflanaga/java/">Michael Thomas Flanagan's Java Scientific Library</a> - general scientific Java classes (non free)</li> 
                 <li> <a href="http://www.geogebra.org/cms/">GeoGebra</a> - Free mathematics software for learning and teaching</li> 
                 <li> <a href="http://zirkel.sourceforge.net/">Zikel</a> - A Dynamic geometry program</li> 
                 <li> <a href="http://geocentral.net/geometria/">Geometria</a> - Interactive Geometry Software</li> 
                 <li> <a href="http://sourceforge.net/projects/vstar/">VStar</a> - Interactive Astronomy Software</li> 
                 <li> <a href="http://ndvis.sourceforge.net/">NDVIS</a> - interactive data visualization</li> 
                 <li> <a href="http://www.ochiailab.dnj.ynu.ac.jp/mjograph/">MjoGraph</a> - X-Y graph editor</li> 
                </ul>
                <h2> Java development tools:</h2> 
                <ul>
                 <li> <a href="http://www.eclipse.org/">Eclipse</a> - IDE</li> 
                 <li> <a href="http://netbeans.org/">NetBeans</a> - Java IDE</li> 
                 <li> <a href="http://drjava.sourceforge.net/">DrJava</a> - a lightweight Java IDE</li> 
                 <li> <a href="http://www.jython.org/">Jython</a> - Python scripting</li> 
                 <li> <a href="http://ant.apache.org/">Ant</a> - "Makefile" for Java</li> 
                 <li> <a href="http://proguard.sourceforge.net/">ProGuard</a> - Java class file shrinker and optimizer</li> 
                 <li> <a href="http://www.eclipse.org/">JCards</a> - a data manager application</li> 
                 <li> <a href="http://www.hexidec.com/ekit.php">Ekit</a> - source HTML editor</li> 
                 <li> <a href="http://jaxe.sourceforge.net/">Jaxe</a> - an XML editor</li> 
                 <li> <a href="http://www.jedit.org/">jEdit</a> - mature programmer's text editor</li> 
                 <li> <a href="http://www.yworks.com/en/products_yed_about.html">yEd</a> -Good graph editor</li> 
                 <li> <a href="http://www.jharris.ca/JGraphEd/">JGraphEd</a> -also graph editor</li> 
                 <li> <a href="http://www.jgraph.com/jgraph.html">JGraph</a> -also good graph editor</li> 
                 <li> <a href="http://fifesoft.com/rtext/">RText</a> - another mature programmer's text editor</li> 
                 <li> <a href="http://jpicedt.sourceforge.net/">jPicEdit</a> - interactive picture editor for LaTeX and PSTricks</li> 
                 <li> <a href="http://latexdraw.sourceforge.net/">LaTeXDraw </a> - PSTricks editor for LateX</li> 
                 <li> <a href="http://tams-www.informatik.uni-hamburg.de/applets/jfig/">jFig</a> - rewrite of popular xfig</li> 
                 <li> <a href="http://quies.net/java/math/plot/">QN-plot</a> - graphs of one or more functions</li> 
                 <li> <a href="http://jfunctionviewer.berlios.de/">JFunctionViewer</a> - Plotting math functions</li> 
                 <li> <a href="http://funcplotter.sourceforge.net/">FuncPlotter</a> - a program for displaying two-dimensional plots</li> 
                 <li> <a href="http://db-maths.nuxit.net/CaRMetal/index_en.html">C.a.R Metal</a> - a dynamic geometry software (Swing)</li> 
                 <li> <a href="http://zirkel.sourceforge.net/doc_en/">C.a.R Zirkel</a> - dynamic geometry program (AWT)</li> 
                 <li> <a href="http://escher.epfl.ch/escher/">Escher</a> - repeating patterns</li> 
                 <li> <a href="http://www.japura.org/">Japura</a> - Swing components</li> 
                 <li> <a href="http://joafip.sourceforge.net/">JOAFIP</a> - java data object persistence in file</li> 
                 <li> <a href="https://pdf-renderer.dev.java.net/">PDF Renderer</a> - java PDF renderer and viewer</li> 
                 <li> <a href="http://sourceforge.net/projects/rows-based-file/files/">rows-based-file</a> - Rows Based File is file management software.</li> 
                 <li> <a href="http://www.neodatis.org/">NeoDatis</a> - Java Object Database</li> 
                 <li> <a href="http://www2.ph.ed.ac.uk/snuggletex/">Snugglatex</a> - Java LaTeX parse</li> 
                 <li> <a href="http://texconverter.sourceforge.net/">Texconverter</a> - another Java LaTeX parse</li> 
                 <li> <a href="http://sourceforge.net/projects/mathtoweb/">MathToWeb parse</a> - Java LaTeX to Web parse</li> 
                 <li> <a href="http://www.thrysoee.dk/laeqed/">Laeqed </a> - Java LaTeX to PNG editor</li> 
                 <li> <a href="http://magicplot.com/">MagicPlot</a> Plotting and non-linear fitting application for scientists and students</li> 
                </ul>
               </div>
