        <h3 class="title">About</h3> 
        <p>The <strong>ChemDoodle Web Components</strong> library is a pure <strong>Javascript</strong> chemical graphics and cheminformatics library derived from the <a href="https://www.chemdoodle.com">ChemDoodle</a><span style="vertical-align: super;font-size: .83em;">®</span> application and produced by <a href="https://www.ichemlabs.com">iChemLabs</a>. <strong>ChemDoodle Web Components</strong> allow the wielder to present publication quality 2D and 3D graphics and animations for chemical structures, reactions and spectra. Beyond graphics, this tool provides a framework for user interaction to create dynamic applications through web browsers, desktop platforms and mobile devices such as the <strong>iPhone</strong>, <strong>iPad</strong> and <strong>Android</strong> devices. This library also has complete access to the entire <strong>ChemDoodle</strong> desktop API through <strong>AJAX</strong>, allowing for quick access to one of the most robust chemical graphics and informatics packages in existence directly through <strong>Javascript</strong>.</p>

By releasing <strong>ChemDoodle Web Components</strong> open-source, yet continuing to financially support the library, <strong>iChemLabs</strong> ensures that the next generation of scientific applications is easily achievable by academia, government and industry, and helps to make sure that the cost of education decreases while using the web to further spread science.</p>

							//.animate( {opacity: 'show'},500 )
							;
						$service_li.removeClass(active_tabstate).filter(':eq('+next_tab+')').addClass(active_tabstate);
						active_tab = next_tab;
					}
					
					return false;
				}).hover(function(){
