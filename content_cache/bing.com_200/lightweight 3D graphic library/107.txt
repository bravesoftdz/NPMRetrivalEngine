              <td width="50%" style="vertical-align: top; padding: 0">
               <div class="cd">
                <p>This package can be used to composed and render 3D models.<br> <br> The main class can compose and render the definition of 3D object models given the coordinates of the vertexes. It can read vertex definitions from separate files or shapes predefined built-in the class.<br> <br> There is an 3D objects class that contains built-in definitions of many predefined types of solid objects.<br> <br> The objects can be rotated around a spin point by a given angle.<br> <br> The main class can render the 3D objects in a 2D image using the GD library.</p>
               </div>
