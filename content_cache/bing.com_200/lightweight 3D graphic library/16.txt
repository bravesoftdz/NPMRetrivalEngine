        <h2> <a property="url" href="https://www.smashingmagazine.com/2009/03/40-stand-alone-javascript-libraries-for-specific-purposes/" rel="bookmark" title="Read '40 Useful JavaScript Libraries'"><span property="name">40 Useful JavaScript Libraries</span></a> </h2> 
        <ul class="pmd clearfix"> 
         <li class="a" property="author" typeof="person" resource="#authorname">By <a href="https://www.smashingmagazine.com/author/umut-muhaddisoglu/" title="Posts by Umut Muhaddisoglu" rel="author">Umut Muhaddisoglu</a></li> 
         <li class="rd" property="datePublished" content="March 2nd, 2009">March 2nd, 2009</li> 
         <li class="tags" property="keywords"><a href="https://www.smashingmagazine.com/tag/javascript/">JavaScript</a><a href="https://www.smashingmagazine.com/tag/resources/">Resources</a></li> 
         <li class="comments" property="discussionUrl"><a href="#comments" title="Comment on 40 Useful JavaScript Libraries">77 Comments</a> </li> 
        </ul> 
        <div class="ad ed"> 
         <div class="oa_zone--ad cad" id="cad" data-ad-name="Medium Rectangle" data-ad-zone="22" data-ad-media="all"></div> 
        </div> 
        <p>Popular <strong>JavaScript libraries</strong> such as jQuery, MooTools, Prototype, Dojo and YUI can be great for accomplishing common JavaScript tasks. These libraries provide many functions, whether the matter is related to events or effects or AJAX. And if one of these libraries can’t do the job, a plug-in probably exists that can.</p> 
        <p>Such flexibility is great but sometimes comes with a penalty (the size of JavaScript files being one of them). Although adopting a well-known JavaScript library is usually a wise decision, you may want to perform a task that can be accomplished by a more lightweight library dedicated exclusively to that task or that is not supported by the more popular JavaScript libraries.</p> 
        <p>You may want to take a look at the following related posts:</p> 
        <ul> 
         <li><a title="Read 'Useful JavaScript Libraries and jQuery Plugins'" href="https://www.smashingmagazine.com/2012/09/useful-javascript-libraries-jquery-plugins-web-developers/" rel="bookmark">Useful JavaScript Libraries and jQuery Plugins</a><sup class="po" id="note-1"><a href="#1">1</a></sup></li> 
         <li><a title="Read 'Useful JavaScript Libraries and jQuery Plugins — Part 2'" href="https://www.smashingmagazine.com/useful-javascript-libraries-jquery-plugins-part-2/" rel="bookmark">Useful JavaScript Libraries and jQuery Plugins — Part 2</a><sup class="po" id="note-2"><a href="#2">2</a></sup></li> 
         <li><a title="Read 'The Seven Deadly Sins Of JavaScript Implementation'" href="https://www.smashingmagazine.com/2010/02/the-seven-deadly-sins-of-javascript-implementation/" rel="bookmark">The Seven Deadly Sins Of JavaScript Implementation</a><sup class="po" id="note-3"><a href="#3">3</a></sup></li> 
         <li><a title="Read 'Writing Fast, Memory-Efficient JavaScript'" href="https://www.smashingmagazine.com/2012/11/writing-fast-memory-efficient-javascript/" rel="bookmark"><span class="headline">Writing Fast, Memory-Efficient JavaScript</span></a><sup class="po" id="note-4"><a href="#4">4</a></sup></li> 
        </ul> 
        <p>Below, we present <strong>40 stand-alone JavaScript libraries that serve specific purposes</strong>. These are not all the ones out there, but every Web designer or developer will find something helpful here.</p> 
        <h3 id="1-forms">1. Forms <a href="#1-forms" aria-label="Link to section '1. Forms'" class="sr hsl">Link</a></h3> 
        <p>wForms<br> wForms is an <strong>open-source</strong> and<strong> unobtrusive library</strong> that simplifies the most common JavaScript form functions. It offers ready-to-use form validation functions for which can be applied by adding a class info to the form objects. Besides these, <strong>wForms</strong> has powerful <strong>form synchronization</strong> (like “Check”/”Uncheck all”) and <strong>conditional form</strong> capabilities (e.g. if x is checked, then show y).</p> 
        <figure>
         <img src="https://www.smashingmagazine.com/images/javascript-libraries/wforms.gif" alt="wForms" width="480" height="116" border="0">
        </figure> 
        <p><a href="http://www.drlongghost.com/validanguage.php">Validanguage</a><sup class="po" id="note-5"><a href="#5">5</a></sup><br> Validanguage is an unobtrusive <strong>JavaScript form validation framework</strong>. It has an inheritance logic whose settings can be defined globally, per form or per element. With the 2 APIs provided: and features like integrated AJAX support, caching and callback functions, the framework provides a robust validation experience. It has a HTML-like API inserted in comment tags and a JavaScript object-based API, for advanced configuration.</p> 
        <figure>
         <a href="http://www.drlongghost.com/validanguage.php"><img src="https://www.smashingmagazine.com/images/javascript-libraries/validanguage.gif" alt="Validanguage" width="480" height="116" border="0"></a>
         <sup class="po" id="note-6"><a href="#6">6</a></sup>
        </figure> 
        <p>LiveValidation<br> LiveValidation is a lightweight library that equips you with set of validation functions. Besides the classic validation methods, the library has a powerful live validation feature that <strong>controls the fields as you type</strong>. <strong>Ruby on Rails</strong> developers will find it very easy to use because the naming conventions and parameters are very similar. It is a stand-alone library but has a Prototype version, too.</p> 
        <figure>
         <img src="https://www.smashingmagazine.com/images/javascript-libraries/live-validation.gif" alt="LiveValidation" width="480" height="116">
        </figure> 
        <p>yav<br> A powerful, extensible and flexible <strong>form-validation library, </strong><strong>yav</strong> has support for a wide range of scenarios, from basic ones like date, email and integer, to advanced ones, like masking and custom regular expressions. It has built-in AJAX support, and errors displayed can be easily customized per object level.</p> 
        <p><a href="http://www.pengoworks.com/index.cfm?action=get:qforms">qForms</a><sup class="po" id="note-7"><a href="#7">7</a></sup><br> A complete solution for handling forms. It equips a developer with features such as various validation rules, methods to prevent multi-submissions, ability to lock/disable fields and more.</p> 
        <p><a href="http://code.google.com/p/formreform/">formreform</a><sup class="po" id="note-8"><a href="#8">8</a></sup><br> Using multi-column layouts without tables is usually challenging. This tiny library transforms a classic form into a stylish layout with one, two or four columns. Forms are styled to fit any design automatically with <strong>formreform</strong>.</p> 
        <figure>
         <a href="http://code.google.com/p/formreform/"><img src="https://www.smashingmagazine.com/images/javascript-libraries/formreform.gif" alt="formreform" width="480" height="116" border="0"></a>

        <h3 id="2-animation">2. Animation <a href="#2-animation" aria-label="Link to section '2. Animation'" class="sr hsl">Link</a></h3> 
        <p><a href="http://fx.inetcat.com/">$fx()</a><sup class="po" id="note-10"><a href="#10">10</a></sup><br> <strong>$fx()</strong> is a lightweight library for animating HTML objects. Using it, you can alter any CSS property within a given timeline. For complicated animations, you can combine effects, group them to chains and run them in parallel. And you can set different callbacks at every step to customize it further.</p> 
        <figure>
         <a href="http://fx.inetcat.com/"><img src="https://www.smashingmagazine.com/images/javascript-libraries/fx-javascript-animations.jpg" alt="$fx() JavaScript Animation Library" width="480" height="116" border="0"></a>
         <sup class="po" id="note-11"><a href="#11">11</a></sup>
        </figure> 
        <p><a href="http://coderepos.org/share/wiki/JSTweener">JSTweener</a><sup class="po" id="note-12"><a href="#12">12</a></sup><br> A tweening library for JavaScript. Its API is similar to the famous ActionScript tweening engine <a href="http://code.google.com/p/tweener/">Tweener</a><sup class="po" id="note-13"><a href="#13">13</a></sup>. You can mention the time of the animation, define the transition effects and delays. At almost any point (like onStart, onComplete, onUpdate) you can fire new events.</p> 
        <figure>
         <a href="http://coderepos.org/share/wiki/JSTweener"><img src="https://www.smashingmagazine.com/images/javascript-libraries/jstweener.gif" alt="JSTweener" border="0"></a>
         <sup class="po" id="note-14"><a href="#14">14</a></sup>
        </figure> 
        <p>Facebook Animation<br> A powerful library for creating customizable CSS-based animations. With a line or two in Facebook Animation, you can improve the UI. The syntax is identical to the FBJS version (the library used in Facebook applications) and once learned, it is an asset for creating Facebook applications.</p> 
        <p>FX<br> A lightweight library, with a YUI-like syntax, FX can create a tween for almost any CSS property. It supports color and scroll animations. Designing the “to” and “from” values of any object/property is enough.</p>

        <h3 id="3-visualization-and-image-effects">3. Visualization And Image Effects <a href="#3-visualization-and-image-effects" aria-label="Link to section '3. Visualization And Image Effects'" class="sr hsl">Link</a></h3> 
        <p><a href="http://www.jscharts.com/">JS charts</a><sup class="po" id="note-15"><a href="#15">15</a></sup><br> <strong>JS charts</strong> supports bar charts, pie charts and simple line graphs. It offers nice usability by simply inserting the library onto Web pages and calling the data from an XML file or a JavaScript array. Charts are created as PNG files and the library is compatible with all the major browsers.</p> 
        <figure>
         <a href="http://www.jscharts.com/"><img src="https://www.smashingmagazine.com/images/javascript-libraries/js-charts.jpg" alt="JS Charts" width="480" height="116" border="0"></a>

        <p><a href="http://www.c3dl.org/">Canvas 3D JS Library (C3DL)</a><sup class="po" id="note-17"><a href="#17">17</a></sup><br> <strong>C3DL</strong> makes writing 3D applications easy. It provides a set of math, scene and 3D object classes to make the canvas more accessible to developers who want to develop 3D content in a browser but not have to deal in depth with the 3D math needed to make it work.</p> 
        <figure>

        <p><a href="http://ejohn.org/blog/processingjs/">Processing.js</a><sup class="po" id="note-19"><a href="#19">19</a></sup><br> This is a JavaScript port to the <a href="http://processing.org/">Processing language</a><sup class="po" id="note-20"><a href="#20">20</a></sup> (a language for programming images, animation and interactions). The library is feature-rich for creating 2D outputs. It provides methods for shape/image drawing, color manipulation, fonts, objects, math functions and more.</p> 
        <figure>
         <a href="http://ejohn.org/blog/processingjs/"><img src="https://www.smashingmagazine.com/images/javascript-libraries/processing-js.gif" alt="Processing.js" width="480" height="116" border="0"></a>

        <p>Raphaël<br> An amazing library that simplifies working with vector graphics on the Web. Raphaël uses SVG and VML for creating graphics that can be modified and event handlers that can be attached to them. The library is feature-rich in functions, including rotation, animation, scaling and drawing curves, rectangles and circles.</p> 
        <p><img src="https://www.smashingmagazine.com/images/javascript-libraries/raphael-js.jpg" alt="Raphael: Vectors With JavaScript" width="480" height="116" border="0"></p>

It is compatible with all major browsers (there is a compatibility chart on the script’s page). The library is so easy to use. Simply inserting the .js file in the Web page and calling a one-line function is enough.</p>

         <img src="https://www.smashingmagazine.com/images/javascript-libraries/imagefx.jpg" alt="Imagefx" width="480" height="116" border="0">
        </figure>

The effects supported include desaturation and grayscale, inverting, flipping, brightness and contrast adjustment, hue and saturation, embossing, blurring and much more. Because the canvas element is relatively new, the library does not have equal support in all browsers yet.</p> 
        <p>Reflection.js<br> An unobtrusive JavaScript to auto-create reflection effects. The height and opacity of the reflection can be defined. Using it is as easy as adding a “class” to the images. It works in all major browsers and is smaller than 5 KB.</p>

Using the library, RegExp objects can be cached and reused, modifiers can be added to existing RegExp objects and more.</p>

Unlike popular solutions like <a href="http://www.mikeindustries.com/blog/sifr/">sIFR</a><sup class="po" id="note-29"><a href="#29">29</a></sup> or FLIR, <strong>typeface.js</strong> doesn’t require Flash and is 100% JavaScript. Again, it converts the font to VML with a generator. After inserting the generated .js file in your Web pages, you can use the font like any other.</p>

        <h3 id="8-other">8. Other <a href="#8-other" aria-label="Link to section '8. Other'" class="sr hsl">Link</a></h3> 
        <p><a href="http://code.google.com/p/swfobject/">swfobject</a><sup class="po" id="note-35"><a href="#35">35</a></sup><br> <strong>swfobject</strong> is the most popular and flexible method for embedding Flash movies. It is unobtrusive, generates valid markup and can detect the version of a user’s Flash Player, allowing you to display alternate content if the required Flash version is not loaded. A nice article on “why <strong>swfobject</strong> should be used” can be found <a href="http://www.alistapart.com/articles/flashembedcagematch/">here</a><sup class="po" id="note-36"><a href="#36">36</a></sup>.</p> 
        <figure>
         <a href="http://code.google.com/p/swfobject/"><img src="https://www.smashingmagazine.com/images/javascript-libraries/swjobject.jpg" alt="swfobject" width="480" height="116" border="0"></a>
         <sup class="po" id="note-37"><a href="#37">37</a></sup>
        </figure> 
        <p><a href="http://www.kryogenix.org/code/browser/sorttable/">sorttable</a><sup class="po" id="note-38"><a href="#38">38</a></sup> and <a href="http://www.danvk.org/wp/dragtable/">dragtable</a><sup class="po" id="note-39"><a href="#39">39</a></sup><br> Tables, like them or not, are still one of the best ways to present data. But they can be better: <strong>sorttable</strong> is a library that makes tabular data sortable, both ascending and descending. To do this, simply add class=”sortable” to the table. You can also exclude certain columns from being sorted and make either ascending or descending the default behavior. <strong>dragtable</strong> is another library that makes columns draggable. After inserting the library in the Web page, add class=”dragtable” to the table as above. Best of all, both libraries can be used together. Simply add class=”sortable dragtable” to the table after including both libraries in the Web page.</p>

Unlike other solutions, “background-position” and “background-repeat” properties work as expected too.</p>

Four boxes can be displayed: alerts, warnings, prompts, success. With a simple function, you can specify the title of the box, the content to be shown and how many seconds to show it for.</p>

         <a href="http://www.scriptiny.com/2008/04/custom-javascript-dialog-boxes/"><img src="https://www.smashingmagazine.com/images/javascript-libraries/custom-dialog-boxes.jpg" alt="Custom Dialog Boxes" width="480" height="116" border="0"></a>

JavaScript is not the best platform for creating games. But for games with less FPS, and if you are a JavaScript developer, then why not? The library includes handy classes for controlling the objects, drawing and keyboard controls.</p> 
        <figure>
         <a href="http://www.tommysmind.com/gamejs/"><img src="https://www.smashingmagazine.com/images/javascript-libraries/gamejs.jpg" alt="GameJS" width="480" height="116" border="0"></a>
         <sup class="po" id="note-47"><a href="#47">47</a></sup>
        </figure> 
        <p><a href="http://www.openjs.com/scripts/events/keyboard_shortcuts/">Shortcuts.js</a><sup class="po" id="note-48"><a href="#48">48</a></sup><br> Starting with Google Reader and Gmail, keyboard shortcuts in Web applications have become popular because they improve usability so much. <strong>Shorcuts.js</strong> provides functions to handle keyboard shortcuts more easily. Using the library, it is possible to create hotkeys and run functions at the press of a button.</p> 
        <p><a href="http://mapstraction.com/">Mapstraction</a><sup class="po" id="note-49"><a href="#49">49</a></sup><br> There are several mapping providers that provide different APIs. If you need to switch providers (say from Google Maps to MapQuest), codes need to be updated. That’s where <strong>Mapstraction</strong> comes in. It provides a common API that covers most of the popular mapping providers. By simply updating a line of code, it is possible to switch between them.</p> 
        <figure>
         <a href="http://mapstraction.com/"><img src="https://www.smashingmagazine.com/images/javascript-libraries/mapstraction.jpg" alt="Mapstraction" width="480" height="116" border="0"></a>
         <sup class="po" id="note-50"><a href="#50">50</a></sup>
        </figure> 
        <p>Amberjack<br> A tiny JavaScript library (about 4 KB) that helps you add good-looking website tours to your website. Once activated, a modal box guides users by displaying any type of content. The steps in the tour can be hand-coded or generated online. Also, tours can be made to match the look of the modal box, either via CSS or with a ready-to-use theme.</p>

         <img src="https://www.smashingmagazine.com/images/javascript-libraries/amberjack.jpg" alt="Amberjack" width="480" height="116" border="0">
