               <p>Graphs and charts are used to simplify complex data and make it easy to read and understand. So it really wouldn’t make sense if it was difficult to integrate a graph into a website. Thanks to Javascript and a bunch of talented developers, we have at our disposal solutions for easily adding graphs and charts to our web projects. Here are 13 of these, including some written for <a href="https://webdesignledger.com/tag/jquery">jQuery</a> and <a href="https://webdesignledger.com/tag/mootools">MooTools</a>, that you will find very useful.<span id="more-3512"></span></p> 
               <h3><a href="http://www.highcharts.com/" target="_blank">Highcharts</a></h3> 
               <p><a href="http://www.highcharts.com/" target="_blank"><img src="https://i1.wp.com/webdesignledger.com/wp-content/uploads/2010/02/graphs_6.jpg?w=970" alt="graphs" data-recalc-dims="1"></a></p> 
               <p>Highcharts is a charting library written in pure JavaScript, offering an easy way of adding interactive charts to your web site or web application. Highcharts currently supports line, spline, area, areaspline, column, bar, pie and scatter chart types.</p> 
               <h3><a href="http://www.jscharts.com/" target="_blank">JS Charts</a></h3> 
               <p><a href="http://www.jscharts.com/" target="_blank"><img src="https://i1.wp.com/webdesignledger.com/wp-content/uploads/2010/02/graphs_7.jpg?w=970" alt="graphs" data-recalc-dims="1"></a></p> 
               <p> JS Charts is a JavaScript chart generator that requires little or no coding. JS Charts allows you to easily create charts in different templates like bar charts, pie charts or simple line graphs. Moreover, JS Charts is free for non-commercial use.</p> 
               <h3><a href="http://www.liquidx.net/plotkit/" target="_blank">PlotKit</a></h3> 
               <p><a href="http://www.liquidx.net/plotkit/" target="_blank"><img src="https://i0.wp.com/webdesignledger.com/wp-content/uploads/2010/02/graphs_1.jpg?w=970" alt="graphs" data-recalc-dims="1"></a></p> 
               <p>PlotKit is a Chart and Graph Plotting Library for Javascript. It has support for HTML Canvas and also SVG via <a title="" class="aalmanual" target="_blank" rel="nofollow" href="https://clk.tradedoubler.com/click?p=264355&amp;a=2784243&amp;g=22804962">Adobe</a> SVG Viewer and native browser support. PlotKit is fully documented and there is a quick tutorial to get you started. </p> 
               <h3><a href="http://www.omnipotent.net/jquery.sparkline/" target="_blank">jQuery Sparklines</a></h3> 
               <p><a href="http://www.omnipotent.net/jquery.sparkline/" target="_blank"><img src="https://i0.wp.com/webdesignledger.com/wp-content/uploads/2010/02/graphs_2.jpg?w=970" alt="graphs" data-recalc-dims="1"></a></p> 
               <p> This jQuery plugin generates sparklines (small inline charts) directly in the browser using data supplied either inline in the HTML, or via javascript. The plugin is compatible with most modern browsers and has been tested with Firefox 2+, Safari 3+, Opera 9, Google Chrome and Internet Explorer 6, 7 &amp; 8. </p> 
               <h3><a href="http://www.filamentgroup.com/lab/jquery_visualize_plugin_accessible_charts_graphs_from_tables_html5_canvas/" target="_blank">jQuery Visualize Plugin</a></h3> 
               <p><a href="http://www.filamentgroup.com/lab/jquery_visualize_plugin_accessible_charts_graphs_from_tables_html5_canvas/" target="_blank"><img src="https://i1.wp.com/webdesignledger.com/wp-content/uploads/2010/02/graphs_3.jpg?w=970" alt="graphs" data-recalc-dims="1"></a></p> 
               <p>This plugin provides a simple method for generating bar, line, area, and pie charts from an HTML table, and allows you to configure them in a variety of ways.</p> 
               <h3><a href="http://www.jqplot.com/" target="_blank">jqPlot</a></h3> 
               <p><a href="http://www.jqplot.com/" target="_blank"><img src="https://i1.wp.com/webdesignledger.com/wp-content/uploads/2010/02/graphs_4.jpg?w=970" alt="graphs" data-recalc-dims="1"></a></p> 
               <p>jqPlot is a plotting and charting plugin for the jQuery Javascript framework. jqPlot produces beautiful line, bar and pie charts with many features.</p> 
               <h3><a href="https://code.google.com/p/milkchart/" target="_blank">milkchart</a></h3> 
               <p><a href="https://code.google.com/p/milkchart/" target="_blank"><img src="https://i2.wp.com/webdesignledger.com/wp-content/uploads/2010/02/graphs_5.jpg?w=970" alt="graphs" data-recalc-dims="1"></a></p> 
               <p>Milkchart is a simple to use, yet robust library for transforming table data into a chart. This library uses the HTML5 canvas tag and is only supported on browsers other than IE until ExCanvas gets proper text support. </p> 
               <h3><a href="http://dragan.yourtree.org/code/canvas-3d-graph/" target="_blank">Canvas 3D Graph</a></h3> 
               <p><a href="http://dragan.yourtree.org/code/canvas-3d-graph/" target="_blank"><img src="https://i0.wp.com/webdesignledger.com/wp-content/uploads/2010/02/graphs_8.jpg?w=970" alt="graphs" data-recalc-dims="1"></a></p> 
               <p>Canvas 3D Graph is a special type of bar graph that plot numbers in 3D.</p> 
               <h3><a href="http://moochart.coneri.se/" target="_blank">Moochart</a></h3> 
               <p><a href="http://moochart.coneri.se/" target="_blank"><img src="https://i2.wp.com/webdesignledger.com/wp-content/uploads/2010/02/graphs_9.jpg?w=970" alt="graphs" data-recalc-dims="1"></a></p> 
               <p>Moochart is a plugin for MooTools 1.2 that draws bubble diagrams on the canvas tag. Future versions might include pie, bar &amp; line graphs.</p> 
               <h3><a href="https://xaviershay.github.com/tufte-graph/" target="_blank">TufteGraph</a></h3> 
               <p><a href="https://xaviershay.github.com/tufte-graph/" target="_blank"><img src="https://i0.wp.com/webdesignledger.com/wp-content/uploads/2010/02/graphs_10.jpg?w=970" alt="graphs" data-recalc-dims="1"></a></p> 
               <p>TufteGraph allows configuration by dynamic functions, allowing for a really compact API, and uses a non-core layout is done via CSS rather than code.</p> 
               <h3><a href="http://www.deensoft.com/lab/protochart/index.php" target="_blank">ProtoChart</a></h3> 
               <p><a href="http://www.deensoft.com/lab/protochart/index.php" target="_blank"><img src="https://i1.wp.com/webdesignledger.com/wp-content/uploads/2010/02/graphs_11.jpg?w=970" alt="graphs" data-recalc-dims="1"></a></p> 
               <p>ProtoChart is an opensource library using Prototype and Canvas to create good looking charts. This library is highly motivated by Flot, Flotr and PlotKit libraries.</p> 
               <h3><a href="https://code.google.com/p/flot/" target="_blank">flot</a></h3> 
               <p><a href="https://code.google.com/p/flot/" target="_blank"><img src="https://i2.wp.com/webdesignledger.com/wp-content/uploads/2010/02/graphs_12.jpg?w=970" alt="graphs" data-recalc-dims="1"></a></p> 
               <p>Flot is a pure Javascript plotting library for jQuery. It produces graphical plots of arbitrary datasets on-the-fly client-side. The focus is on simple usage (all settings are optional), attractive looks and interactive features like zooming and mouse tracking. The plugin works with Internet Explorer 6/7/8, Firefox 2.x+, Safari 3.0+, Opera 9.5+ and Konqueror 4.x+ with the HTML canvas tag (Internet Explorer where the excanvas Javascript emulation helper is used). </p> 
               <h3><a href="http://vis.stanford.edu/protovis/" target="_blank">Protovis </a></h3> 
               <p><a href="http://vis.stanford.edu/protovis/" target="_blank"><img src="https://i1.wp.com/webdesignledger.com/wp-content/uploads/2010/02/graphs_13.jpg?w=970" alt="graphs" data-recalc-dims="1"></a></p> 
               <p>Protovis composes custom views of data with simple marks such as bars and dots. Unlike low-level graphics libraries that quickly become tedious for visualization, Protovis defines marks through dynamic properties that encode data, allowing inheritance, scales and layouts to simplify construction. </p>
