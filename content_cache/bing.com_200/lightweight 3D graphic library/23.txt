        <h1 class="page-title" data-text="Libraries">Libraries</h1> 
       </header>
       <!-- /.page-header --> 
       <section class="article-content"> 
        <p>Here is a list of useful <strong>open-source</strong>&nbsp;libraries dedicated to the programming of&nbsp;<strong>image processing </strong>algorithms, which you can download and use <strong>for free</strong>.</p> 
        <hr> 
        <h2>C++</h2> 
        <table width="100%"> 
         <tbody> 
          <tr> 
           <td width="128"><a title="The CImg Library" href="http://cimg.sourceforge.net" target="_blank"><img class="wp-image-38 size-full aligncenter" src="http://opensource.graphics/wp-content/uploads/2014/12/logo_cimg.png" alt="logo_cimg" width="92" height="96"></a></td> 
           <td>The <strong><a title="The Cimg Library" href="http://cimg.sourceforge.net" target="_blank">CImg Library</a></strong>&nbsp;is a popular, small, comprehensive, open source C++ toolkit for image processing.&nbsp;CImg stands for <em>Cool Image</em> : It is easy to use, efficient and is intended to be a very pleasant toolbox to design image processing algorithms in C++. Due to its generic conception, it can cover a wide range of image processing applications.</td> 
          </tr> 
          <tr> 
           <td width="128">&nbsp;<a title="Pandore" href="https://clouard.users.greyc.fr/Pandore/" target="_blank"><img class="aligncenter size-full wp-image-306" src="http://opensource.graphics/wp-content/uploads/2014/12/logo_pandore.png" alt="logo_pandore" width="86" height="64"></a></td> 
           <td><a title="Pandore" href="https://clouard.users.greyc.fr/Pandore/" target="_blank"><strong>Pandore</strong></a> is a C++ library of image processing operators. The current version contains image processing operators that operate on grayscale, color and multispectral, 1D, 2D and 3D images.</td> 
          </tr> 
          <tr> 
           <td width="128"><a title="OpenCV" href="http://opencv.org/" target="_blank"><img class="aligncenter size-full wp-image-457" src="http://opensource.graphics/wp-content/uploads/2014/12/logo_opencv.png" alt="logo_opencv" width="65" height="80"></a></td> 
           <td><a title="OpenCV" href="http://opencv.org/" target="_blank"><strong>OpenCV</strong></a> (<i>Open Source Computer Vision</i>) is a library of programming functions mainly aimed at real-time computer vision, developed by Intel Russia research center in Nizhny Novgorod, and now supported by Willow Garageand Itseez.<sup id="cite_ref-Itseez_1-1" class="reference">[1]</sup> It is free for use under the open source BSD license.</td> 
          </tr> 
          <tr> 
           <td width="128"><a title="Population" href="http://www.population-image.fr/wordpress/" target="_blank"><img class="aligncenter size-full wp-image-462" src="http://opensource.graphics/wp-content/uploads/2014/12/logo_population2.png" alt="logo_population2" width="85" height="80"></a></td> 
           <td><a title="Population" href="http://www.population-image.fr/wordpress/" target="_blank"><strong>Population library</strong></a> is a scientific open source library dedicated to the processing, analysis, modelling and visualisation of 2D/3D images. Around a community project, its objective is to structure a reproducible and shared science in these fields.</td> 
          </tr> 
          <tr> 
           <td width="128"><a title="Magick++" href="http://www.imagemagick.org/Magick++/" target="_blank"><img class="aligncenter size-full wp-image-465" src="http://opensource.graphics/wp-content/uploads/2014/12/Magick-2.png" alt="Magick++2" width="80" height="23"></a></td> 
           <td><a title="Magick++" href="http://www.imagemagick.org/Magick++/" target="_blank"><strong>Magick++</strong></a> is the object-oriented C++ API to the ImageMagick and GraphicsMagick image-processing libraries.</td> 
          </tr> 
          <tr> 
           <td width="128"><a title="VIPS" href="http://www.vips.ecs.soton.ac.uk/" target="_blank"><img class="aligncenter size-full wp-image-468" src="http://opensource.graphics/wp-content/uploads/2014/12/logo_vips.png" alt="logo_vips" width="96" height="55"></a></td> 
           <td><a title="VIPS" href="http://www.vips.ecs.soton.ac.uk/" target="_blank"><strong>VIPS</strong></a> is a free image processing system. It is good with large images (images larger than the amount of RAM you have available), with many CPUs, for working with colour, for scientific analysis and for general research &amp; development.</td> 
          </tr> 
         </tbody> 
        </table> 
        <h2>Java</h2> 
        <table style="width: 100%;"> 
         <tbody> 
          <tr> 
           <td width="128"><a title="ImageJ" href="http://imagej.nih.gov/ij/index.html" target="_blank"><img class="aligncenter size-full wp-image-471" src="http://opensource.graphics/wp-content/uploads/2014/12/logo_imagej.png" alt="logo_imagej" width="80" height="16"></a></td> 
           <td><a title="ImageJ" href="http://imagej.nih.gov/ij/index.html" target="_blank"><strong>ImageJ</strong></a> is a public domain Java image processing program inspired by <a href="http://rsb.info.nih.gov/nih-image/">NIH Image</a> for the Macintosh.</td> 
          </tr> 
         </tbody> 
        </table>
