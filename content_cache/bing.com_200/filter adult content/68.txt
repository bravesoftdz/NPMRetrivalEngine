   <h1>HTML filtering utility for Java</h1> 
   <p> </p>
   <ul> 
    <li><a href="source/">View Source</a></li> 
    <li><a href="download/html_filter.zip">Download (127K)</a></li> 
   </ul> 
   <p></p> 
   <p> This utility is a single class, HTMLInputFilter, which can be used to parse user-submitted input and sanitize it against potential cross site scripting attacks, malicious html, or simply badly formed html. This version, written in Java, is largely a translation of <a href="http://code.iamcal.com/php/lib_filter/">lib_filter</a>, the original work of <a href="http://www.iamcal.com/">Cal Henderson</a> written in PHP. </p> 
   <p> Combined with this code is an inner Test class for unit testing, designed to be executed in <a href="http://www.junit.org/">JUnit</a> within Eclipse. The download archive includes the necessary JUnit jar for running the tests. </p> 
   <h2>Processing HTML Input</h2> 
   <p> Sample usage: </p>
   <blockquote>
    <pre>// retrieve input from user...String input = ...String clean = new HTMLInputFilter().filter( input );</pre>
   </blockquote>
