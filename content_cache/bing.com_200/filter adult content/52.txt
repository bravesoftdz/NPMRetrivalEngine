      </div> <h2 class="sectionheader">Filtering Collections</h2> <p> In this example we are going to show how easy it is to filter results from a Collection. This example again shows the old way vs. the new way. </p> <pre class="prettyprint lang-java">
import java.util.*;
import java.util.stream.*;

public class UserFilterTest {
  private static List&lt;User&gt; users = Arrays.asList(
      new User(1, "Steve", "Vai", 40),
      new User(4, "Joe", "Smith", 32),
      new User(3, "Steve", "Johnson", 57),
      new User(9, "Mike", "Stevens", 18),
      new User(10, "George", "Armstrong", 24),
      new User(2, "Jim", "Smith", 40),
      new User(8, "Chuck", "Schneider", 34),
      new User(5, "Jorje", "Gonzales", 22),
      new User(6, "Jane", "Michaels", 47),
      new User(7, "Kim", "Berlie", 60)
    );

  public static void main(String[] args) {
    oldJavaWay();
    newJavaWay();
  }

  private static void oldJavaWay() {
    List&lt;User&gt; olderUsers = new ArrayList&lt;User&gt;();

    for (User u : users) {
      if (u.age &gt; 30) {
        olderUsers.add(u);
      }
    }

    printListNewWay("Old way older users", olderUsers);
  }

  private static void newJavaWay() {
    List&lt;User&gt; olderUsers = users.stream().filter(u -&gt; u.age &gt; 30).collect(Collectors.toList());
    printListNewWay("New way older users", olderUsers);
  }

  private static void printListNewWay(String type, List&lt;User&gt; users) {
    System.out.println(type + ":");

    users.forEach(u -&gt; System.out.println("\t" + u));

    System.out.println();
  }
}
</pre> <p> This example uses the <b>User</b> class from the previous section. Again we create our list of users and this time we want to get a list of only the users older 30 years. This is simple to do in java, here's how we do it with the old method: </p> <pre class="prettyprint lang-java">
  List&lt;User&gt; olderUsers = new ArrayList&lt;User&gt;();

  for (User u : users) {
    if (u.age &gt; 30) {
      olderUsers.add(u);
    }
  }
</pre> <p> That's easy enough, but look how easy it is with lambda expressions: </p> <pre class="prettyprint lang-java">
List&lt;User&gt; olderUsers = users.stream().filter(u -&gt; u.age &gt; 30).collect(Collectors.toList());
</pre> <p> That's a lot less boilerplate code and we don't have to use any specific List implementation. We could also make our filter more complex by checking more than one variable: </p> <pre class="prettyprint lang-java">
List&lt;User&gt; olderUsers = users.stream().filter(u -&gt; u.age &gt; 30 || u.lastName.startsWith("S")).collect(Collectors.toList());
</pre> <p> With this filter we check both age being over 30 and if the last name starts with an "S". </p> <p> In the next section we will learn how to pass a function as a parameter to another function. </p> <br> <p align="center"> <a href="comparator-tutorial.php"><b>Prev</b> (Comparator)</a> | <a href="functions-as-parameters-tutorial.php"><b>Next</b> (Functions As Parameters)</a> <br> <script type="text/javascript"><!--
