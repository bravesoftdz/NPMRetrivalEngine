       <h1 id="Parental_Content_Control">Parental Content Control</h1> 
       <span class="anchor" id="line-6"></span>
       <p class="line874">There are many options available for parents to filter the web content their children are viewing. There are several solutions that will work with Linux including browser-based filtering through Firefox extensions, DNS-based filtering using OpenDNS' free service, do it yourself whitelisting or a set of free applications configured to work specifically for web content filtering. <span class="anchor" id="line-7"></span><span class="anchor" id="line-8"></span></p>
       <p class="line867"> </p>
       <h2 id="DansGuardian.2FPrivoxy.2FSquid">DansGuardian/Privoxy/Squid</h2> 
       <span class="anchor" id="line-9"></span>
       <p class="line867"> </p>
       <h3 id="DansGuardian">DansGuardian</h3> 
       <span class="anchor" id="line-10"></span>
       <p class="line862">Dans<tt class="backtick"></tt>Guardian is a powerful filtering tool with many features to block unwanted sites. Among some of it's features are blocking text and HTML pages based on obscene content, IP addresses, and URLs. It can also be configured to work off a 'whitelist' where all sites except those listed will be blocked. The stealth mode feature is particularly nice if you have teenagers. This feature allows sites that would otherwise be blocked, but reports them in a log. <span class="anchor" id="line-11"></span><span class="anchor" id="line-12"></span></p>
       <p class="line862">However nice, Dans<tt class="backtick"></tt>Gaurdian is just a filter and will need to be paired with a proxy to do the web stuff. A proxy is a program that sits between your computer and the internet. It funnels all incoming connections through one opening called a port. Usually proxies have additional features such as web-caching and filtering. There are many choices of proxies that one can choose, but we will focus on one to get you started: <a href="/community/Privoxy">Privoxy</a>. <span class="anchor" id="line-13"></span><span class="anchor" id="line-14"></span></p>
       <p class="line862">Please view this page if you are would like to know more about the <a class="http" href="http://dansguardian.org/?page=dgflow">flow of events</a> in Dans<tt class="backtick"></tt>Guardian. <span class="anchor" id="line-15"></span><span class="anchor" id="line-16"></span></p>
       <p class="line867"> </p>
       <h3 id="Privoxy">Privoxy</h3> 
       <span class="anchor" id="line-17"></span>
       <p class="line874">"Privoxy is a non-caching web proxy with advanced filtering capabilities for enhancing privacy, modifying web page data and HTTP headers, controlling access, and removing ads and other obnoxious Internet junk. Privoxy has a flexible configuration and can be customized to suit individual needs and tastes. It has application for both stand-alone systems and multi-user networks." <span class="anchor" id="line-18"></span><br> <span class="anchor" id="line-19"></span> </p>
       <h4 id="Setting_up_Dans_Guardian_and_Prixoxy">Setting up Dans Guardian and Prixoxy</h4> 
       <span class="anchor" id="line-20"></span>
       <p class="line862">For a complete guide setting up Dans<tt class="backtick"></tt>Guardian and Privoxy <a class="http" href="http://blog.bodhizazen.net/linux/web-content-filtering-made-easy/">see this guide.</a> <span class="anchor" id="line-21"></span><span class="anchor" id="line-22"></span></p>
       <p class="line867"> </p>
       <h2 id="Browser_Based_Content_Controls">Browser Based Content Controls</h2> 
       <span class="anchor" id="line-23"></span>
       <p class="line874">There are some Firefox addons that offer content filtering. <span class="anchor" id="line-24"></span></p>
       <ul>
        <li><p class="line862">Leech<tt class="backtick"></tt>Block - Leech<tt class="backtick"></tt>Block lets you specify which sites to block and when to block them. Allows you to time limit web site usage. <span class="anchor" id="line-25"></span><span class="anchor" id="line-26"></span></p></li>
       </ul>
       <p class="line862">You can get Leech<tt class="backtick"></tt>Block Firefox add-on <a class="https" href="https://addons.mozilla.org/en-US/firefox/addon/leechblock/">here.</a> <span class="anchor" id="line-27"></span><span class="anchor" id="line-28"></span></p>
       <ul>
        <li><p class="line862">Fox<tt class="backtick"></tt>Filter - Recently called "THE Parental Control for Firefox" Fox<tt class="backtick"></tt>Filter is a personal content filter that helps block pornographic and other inappropriate content. All filtering features are free! Premium features available with small support fee. <span class="anchor" id="line-29"></span><span class="anchor" id="line-30"></span></p></li>
       </ul>
       <p class="line862">You can get Fox<tt class="backtick"></tt>Filter Firefox add-on <a class="https" href="https://addons.mozilla.org/en-US/firefox/addon/foxfilter/">here.</a> <span class="anchor" id="line-31"></span><span class="anchor" id="line-32"></span></p>
       <p class="line867"> </p>
       <h2 id="Content_Control_With_a_GUI">Content Control With a GUI</h2> 
       <span class="anchor" id="line-33"></span>
       <p class="line874">There have been many attempts to produce GUI 'wrappers' to make the process of setting up Content Control easier. Many of these have, over time, have stopped being maintained. Nanny, being a Gnome project probably stands the best chance of being maintained long-term. <span class="anchor" id="line-34"></span><span class="anchor" id="line-35"></span></p>
       <p class="line867"> </p>
       <h3 id="Web_Content_Control">Web Content Control</h3> 
       <span class="anchor" id="line-36"></span>
       <p class="line874">Web Content Control is a program in Ubuntu that lets you block unwanted contents such as adult materials, violence and other disturbing websites. It can also be used as a Parental Control System to control what programs/games can be run and played. <span class="anchor" id="line-37"></span><span class="anchor" id="line-38"></span></p>
       <p class="line862">An introduction to setting up Web Content Control on Ubuntu 10.10 <a class="http" href="http://www.liberiangeek.net/2010/08/enable-web-content-filtering-ubuntu-10-04-lucid-lynx-web-content-control/">here.</a> <span class="anchor" id="line-39"></span></p>

       <p class="line867"> </p>
       <h3 id="Nanny">Nanny</h3> 
       <span class="anchor" id="line-42"></span>
       <p class="line874">Gnome Nanny is an easy way to control what your kids are doing in the computer. You can limit how much time a day each one of them is browsing the web, chatting or doing email. You can also decide at which times of the day the can do this things. <span class="anchor" id="line-43"></span><span class="anchor" id="line-44"></span></p>
       <p class="line874">Gnome Nanny filters what web pages are seen by each user, so you can block all undesirable webs and have your kids enjoy the internet with ease of mind, no more worries! <span class="anchor" id="line-45"></span><span class="anchor" id="line-46"></span></p>
       <p class="line874">Note that Nanny stopped being available with Ubuntu Oneiric as it relies on an old version of Gnome. <span class="anchor" id="line-47"></span><span class="anchor" id="line-48"></span></p>
       <p class="line867"><a class="apt" href="apt:nanny">Install Nanny</a> <span class="anchor" id="line-49"></span><span class="anchor" id="line-50"></span></p>
       <p class="line867"> </p>
       <h3 id="Open_DNS">Open DNS</h3> 
       <span class="anchor" id="line-51"></span>
       <p class="line874">OpenDNS is perhaps best known for its ability to provide an alternative to your ISP's DNS servers. The other service provided by OpenDNS happens to also include the option to engage web-based parental controls. So any computer configured to utilize OpenDNS for its DNS settings also has the ability to filter out the junk content you might not want made available in your household. <span class="anchor" id="line-52"></span> </p>
       <h4 id="Getting_Started_with_Open_DNS">Getting Started with Open DNS</h4> 
       <span class="anchor" id="line-53"></span>
       <p class="line862">1. To get started, go to <em>System –&gt; Preferences –&gt; Network Connections</em> <span class="anchor" id="line-54"></span>2. Then select the default interface and click ‘Edit’. <span class="anchor" id="line-55"></span>3. Next, ‘IPv4 Settings’ and choose ‘Automatic (DHCP) addresses only’ for Method. <span class="anchor" id="line-56"></span>4. And finally, type the DNS Server addresses below into the DNS Server field and click ‘Apply’ <span class="anchor" id="line-57"></span><em>208.67.222.123, 208.67.220.123</em> <span class="anchor" id="line-58"></span>5. When you’re done, click <a class="https" href="https://store.opendns.com/familyshield/welcome/">this link</a> to verify if your setting is OK. You will get a Green Checkmark if everything is OK. You will see a Red X if something went wrong. <span class="anchor" id="line-59"></span><span class="anchor" id="line-60"></span></p>
       <p class="line867"> </p>
       <h2 id="Do_It_Yourself_Whitelisting">Do It Yourself Whitelisting</h2> 
       <span class="anchor" id="line-61"></span>
       <p class="line874">Block access to absolutely everything on the Internet except for the IP addresses/websites listed in the hosts file. <span class="anchor" id="line-62"></span><span class="anchor" id="line-63"></span></p>
       <p class="line874">Open a Terminal [Ctrl]+[Alt]+[T] and backup the hosts file (just in case): <span class="anchor" id="line-64"></span><span class="anchor" id="line-65"></span></p>
       <p class="line874">sudo cp /etc/hosts /etc/hosts.old <span class="anchor" id="line-66"></span><span class="anchor" id="line-67"></span></p>
       <p class="line874">Then edit the hosts file with your favorite text editor (gedit, leafpad, etc.) <span class="anchor" id="line-68"></span><span class="anchor" id="line-69"></span></p>
       <p class="line862">sudo &lt;text editor&gt; /etc/hosts <span class="anchor" id="line-70"></span><span class="anchor" id="line-71"></span><span class="anchor" id="line-72"></span></p>
       <p class="line862">The format of this list is the IP address, followed by the domain name, followed by the domain name with www in front of it. Example 12.34.567.89 domain.com www.domain.com To add websites to this list; go to an IP address lookup page such as this <a class="http" href="http://www.hcidata.info/host2ip.htm">one</a> or this <a class="http" href="http://whatismyipaddress.com/hostname-ip">one</a>. Many websites pull-in content from other servers and/or domains, so you may need to examine the code (view the page source) of the web page(s) to find all the domains you'll need. A quick way to find all the domains referenced is to search the page source for "http" When done adding to the list, save [Ctrl]+[S] and "X" out of the hosts file. <span class="anchor" id="line-73"></span><span class="anchor" id="line-74"></span><span class="anchor" id="line-75"></span></p>
       <p class="line874">Back in a Terminal, type (or copy and paste): <span class="anchor" id="line-76"></span><span class="anchor" id="line-77"></span></p>
       <p class="line862">sudo &lt;text editor&gt; /etc/nsswitch.conf <span class="anchor" id="line-78"></span><span class="anchor" id="line-79"></span></p>
       <p class="line874">Find the hosts line <span class="anchor" id="line-80"></span><span class="anchor" id="line-81"></span></p>
       <p class="line874">hosts: files dns <span class="anchor" id="line-82"></span><span class="anchor" id="line-83"></span></p>
       <p class="line874">and remove dns, so the line becomes <span class="anchor" id="line-84"></span><span class="anchor" id="line-85"></span></p>
       <p class="line874">hosts: files <span class="anchor" id="line-86"></span><span class="anchor" id="line-87"></span></p>
       <p class="line874">Save [Ctrl]+[S] and "X" out of the nsswitch.conf file <span class="anchor" id="line-88"></span><span class="anchor" id="line-89"></span><span class="anchor" id="line-90"></span></p>
       <p class="line874">That's all there is to it. Everything except approved domains/IP addresses are blocked. Edit the hosts file to allow (or disallow) access to additional websites. <span class="anchor" id="line-91"></span><span class="anchor" id="line-92"></span></p>
       <p class="line874">Temporarily give the computer access to the entire Internet (like when running updates, or looking up more websites/IP addresses) by adding dns back into the nsswitch.conf file and saving. <span class="anchor" id="line-93"></span><span class="anchor" id="line-94"></span></p>
       <p class="line862">Get a head start for the whitelist of a child's computer by downloading a "base file" of educational websites from <a class="http" href="http://savingus.org/download/whitelist-buntu.txt">here</a>. <span class="anchor" id="line-95"></span><span class="anchor" id="line-96"></span></p>
