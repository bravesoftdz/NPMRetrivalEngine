      <h3>Email Validation script</h3>
     </div> 
     <div id="scriptheaderFoot">
      <span class="rightit"></span> 
      <b>Author:</b> 
      <span id="ddauthor">Dynamic Drive</span>
     </div> 
     <p align="left"><strong>Description: </strong>This script uses regular expressions to check that a form field contains a valid email address. Note that it won't catch all invalid emails, like most similar scripts. However, for most intents and purposes, this script should serve its purpose well, by rejecting common email typos while leaving room for obscure yet valid emails to pass.</p> 
     <p align="left">Here are some examples of valid email addresses that this script will recognize:</p> 
     <ul> 
      <li> contact@mydomain.com</li> 
      <li> jason+lang@gmail.com</li> 
      <li> help@nbc.co.uk</li> 
      <li> mary.k.ashley@celeb.com</li> 
      <li> george@sub.whatever.mydomain.ca</li> 
      <li> edward@42.234.12.345</li> 
     </ul> 
     <p align="left"><b>Note:</b> If you discover a <b>valid</b> email address that this script rejects, please <a href="../contact.htm">let us know</a>.</p> 
     <p align="left"><strong>Demo: </strong>Try typing in an email address below:</p> 
     <p align="left">
      <!--webbot bot="HTMLMarkup" startspan --><script type="text/javascript">

/***********************************************
* Email Validation script- (c) Dynamic Drive (www.dynamicdrive.com)
* Please keep this notice intact.
* Visit http://www.dynamicdrive.com/ for full source code
***********************************************/

var emailfilter=/^\w+[\+\.\w-]*@([\w-]+\.)*\w+[\w-]*\.([a-z]{2,4}|\d+)$/i

function checkmail(e){
var returnval=emailfilter.test(e.value)
if (returnval==false){
alert("Please enter a valid email address.")
e.select()
}
else
alert("Valid Email!")
return false
}

</script> </p>
     <form method="post"> 
      <input name="myemail" type="text" style="width: 270px"> 
      <input type="submit" onClick="return checkmail(this.form.myemail)" value="Submit"> 
     </form>

     <p align="left"><strong>Directions: </strong> <a target="win2" href="emailvalidate_dev.htm"><img src="../dview.gif" width="115" height="19" alt="Developer's View" border="0"></a> </p> 
     <p align="left"><b>Step 1:</b> Add the below script to the &lt;BODY&gt; section of your page:</p> 
     <form method="post">
      <p><a class="selectall" href="javascript:highlight(1)">Select All</a><br> <textarea rows="8" name="S2" cols="45" wrap="virtual" class="codecontainer">&lt;script type="text/javascript"&gt;

/***********************************************
* Email Validation script- (c) Dynamic Drive (www.dynamicdrive.com)
* Please keep this notice intact.
* Visit http://www.dynamicdrive.com/ for full source code
***********************************************/

var emailfilter=/^\w+[\+\.\w-]*@([\w-]+\.)*\w+[\w-]*\.([a-z]{2,4}|\d+)$/i

function checkmail(e){
var returnval=emailfilter.test(e.value)
if (returnval==false){
alert("Please enter a valid email address.")
e.select()
}
return returnval
}

&lt;/script&gt;

&lt;form&gt;
&lt;input name="myemail" type="text" style="width: 270px"&gt; &lt;input type="submit" onClick="return checkmail(this.form.myemail)" value="Submit" /&gt;

&lt;/form&gt;</textarea></p> 
     </form> 
     <p align="left">The code above includes a sample form where this script validates the email field contained. The form looks like this:</p> 
     <p class="codehighlight" align="left">&lt;form&gt;<br> &lt;input <font color="#FF0000">name="myemail"</font> type="text" style="width: 300px"&gt;&lt;br/&gt;<br> &lt;input type="submit" <font color="#FF0000">onClick="return checkmail(this.form.myemail)" </font> value="Submit" /&gt;<br> &lt;/form&gt;</p> 
     <p align="left">To customize this script to serve your own form(s), simply change the parts in red above to your own. More specifically,&nbsp;you should first give your email address field a name(ie: name="myemail"). Then, modify the submit button of the form by adding in the "onClick" portion. Remember to change "myemail" to reflect the name of your email address field.</p>
