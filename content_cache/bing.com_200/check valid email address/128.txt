         <div id="body">
           You can use this to check for a valid email address when a form is submitted. It is pretty flimsy, but is good for quick check. Moreover, you can enhance it using the logic from some of the other examples on this site, such as the one at 
          <a href="http://www.freevbcode.com/ShowCode.Asp?ID=201">http://www.freevbcode.com/ShowCode.Asp?ID=201</a>. This other snippet was developed for Visual Basic 5.0 and up, but is easily portable to VB Script.
          <p> It makes sure there is a "@", ".", and is at least 7 characters long.</p>

Function frmExample_onsubmit
   If instr(frmExample.txtemail.Value, "@") = 0 OR _
	instr(frmExample.txtemail.Value, ".") = 0 OR _
	Len(frmExample.txtemail.Value) &lt; 7 Then
	window.alert "Please specify a valid e-mail address!"
	frmExample_onsubmit = False
   End IF
	
End Function
