        <div class="framed light">
         This object can be used to validate an e-mail address by using NodeJS to check with an SMTP server.
         <br> 
         <br> It can connect to check if the e-mail domain has valid MX.
         <br> 
         <br> If the MX records do not exist, it falls back to resolving the domain A record to determine the e-mail address domain SMTP server addresses.
         <br> 
         <br> The object attempts to validate the e-mail address by simulating a delivery of a message to the domain SMTP server.
         <br> 
         <br> When it reaches a conclusion the object calls a given callback function with the result, which can be whether the address is valid, invalid or undetermined.
        </div>
