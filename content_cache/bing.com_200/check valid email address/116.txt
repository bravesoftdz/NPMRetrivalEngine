                    <h3 class="post-title entry-title" itemprop="name"> Confirm-ValidEmailAddress.ps1 </h3>

                      <ol class="dp-rb"> 
                       <li class="alt"><span><span>&lt;</span><span class="comment">#</span><span>&nbsp; </span></span></li> 
                       <li><span>.SYNOPSIS&nbsp; </span></li> 
                       <li class="alt"><span>&nbsp;&nbsp;&nbsp; This script validates email addresses based on&nbsp; </span></li> 
                       <li><span>&nbsp;&nbsp;&nbsp; MSFT published Regular Expression. This is a&nbsp;&nbsp; </span></li> 
                       <li class="alt"><span>&nbsp;&nbsp;&nbsp; re-write with PowerShell of an existing bit of&nbsp;&nbsp; </span></li> 
                       <li><span>&nbsp;&nbsp;&nbsp; MSDN sample code&nbsp;&nbsp; </span></li> 
                       <li class="alt"><span>.DESCRIPTION&nbsp; </span></li> 
                       <li><span>&nbsp;&nbsp;&nbsp; This script first creates a <span class="keyword">function</span><span> to validate&nbsp; </span></span></li> 
                       <li class="alt"><span>&nbsp;&nbsp;&nbsp; an email address. It uses a large regex that is&nbsp; </span></li> 
                       <li><span>&nbsp;&nbsp;&nbsp; documented at the MSDN page noted below. The script&nbsp; </span></li> 
                       <li class="alt"><span>&nbsp;&nbsp;&nbsp; then creates an array of email addreses and then&nbsp; </span></li> 
                       <li><span>&nbsp;&nbsp;&nbsp; validates them against the <span class="keyword">function</span><span> and displays&nbsp; </span></span></li> 
                       <li class="alt"><span>&nbsp;&nbsp;&nbsp; the results.&nbsp; </span></li> 
                       <li><span>.NOTES&nbsp; </span></li> 
                       <li class="alt"><span>&nbsp;&nbsp;&nbsp; File Name&nbsp; : Confirm-ValidEmailAddress.ps1&nbsp; </span></li> 
                       <li><span>&nbsp;&nbsp;&nbsp; Author&nbsp;&nbsp;&nbsp;&nbsp; : Thomas Lee - tfl<span class="variable">@psp</span><span>.co.uk&nbsp; </span></span></li> 
                       <li class="alt"><span>&nbsp;&nbsp;&nbsp; Requires&nbsp;&nbsp; : PowerShell Version 2.0&nbsp; </span></li> 
                       <li><span>.LINK&nbsp; </span></li> 
                       <li class="alt"><span>&nbsp;&nbsp;&nbsp; This script posted to:&nbsp; </span></li> 
                       <li><span>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; <a title="http://pshscripts.blogspot.com/2011/12/confirm-validemailaddressps1.html" href="http://pshscripts.blogspot.com/2011/12/confirm-validemailaddressps1.html">http://pshscripts.blogspot.com/2011/12/confirm-validemailaddressps1.html</a>&nbsp; </span></li> 
                       <li class="alt"><span>&nbsp;&nbsp;&nbsp; MSDN sample posted to:&nbsp; </span></li> 
                       <li><span>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; <a href="http://msdn.microsoft.com/en-us/library/01escwtf.aspx">http://msdn.microsoft.com/en-us/library/01escwtf.aspx</a>&nbsp;&nbsp; </span></li> 
                       <li class="alt"><span>.EXAMPLE&nbsp; </span></li> 
                       <li><span>&nbsp;&nbsp;&nbsp; Valid: david.jones<span class="variable">@proseware</span><span>.com&nbsp; </span></span></li> 
                       <li class="alt"><span>&nbsp;&nbsp;&nbsp; Valid: d.j<span class="variable">@server1</span><span>.proseware.com&nbsp; </span></span></li> 
                       <li><span>&nbsp;&nbsp;&nbsp; Valid: jones<span class="variable">@ms1</span><span>.proseware.com&nbsp; </span></span></li> 
                       <li class="alt"><span>&nbsp;&nbsp;&nbsp; Invalid: j.<span class="variable">@server1</span><span>.proseware.com&nbsp; </span></span></li> 
                       <li><span>&nbsp;&nbsp;&nbsp; Invalid: j<span class="variable">@proseware</span><span>.com9&nbsp; </span></span></li> 
                       <li class="alt"><span>&nbsp;&nbsp;&nbsp; Valid: js<span class="comment">#internal@proseware.com</span><span>&nbsp; </span></span></li> 
                       <li><span>&nbsp;&nbsp;&nbsp; Valid: j_9@[129.126.118.1]&nbsp; </span></li> 
                       <li class="alt"><span>&nbsp;&nbsp;&nbsp; Invalid: j..s<span class="variable">@proseware</span><span>.com&nbsp; </span></span></li> 
                       <li><span>&nbsp;&nbsp;&nbsp; Invalid: js*<span class="variable">@proseware</span><span>.com&nbsp; </span></span></li> 
                       <li class="alt"><span>&nbsp;&nbsp;&nbsp; Invalid: js<span class="variable">@proseware</span><span>..com&nbsp; </span></span></li> 
                       <li><span>&nbsp;&nbsp;&nbsp; Invalid: js<span class="variable">@proseware</span><span>.com9&nbsp; </span></span></li> 
                       <li class="alt"><span>&nbsp;&nbsp;&nbsp; Valid: j.s<span class="variable">@server1</span><span>.proseware.com&nbsp; </span></span></li> 
                       <li><span>&nbsp;&nbsp;&nbsp; Valid: tfl<span class="variable">@psp</span><span>.co.uk&nbsp; </span></span></li> 
                       <li class="alt"><span>&nbsp;&nbsp;&nbsp; Valid: cuddly.penguin<span class="variable">@cookham</span><span>.net&nbsp; </span></span></li> 
                       <li><span><span class="comment">#&gt;</span><span>&nbsp; </span></span></li> 
                       <li class="alt"><span>&nbsp; </span></li> 
                       <li><span>Function IsValidEmail&nbsp;&nbsp; {&nbsp; </span></li> 
                       <li class="alt"><span>Param ([string] <span class="variable">$In</span><span>)&nbsp; </span></span></li> 
                       <li><span><span class="comment"># Returns true if In is in valid e-mail format.</span><span>&nbsp; </span></span></li> 
                       <li class="alt"><span>[system.Text.RegularExpressions.Regex]::IsMatch(<span class="variable">$In</span><span>,&nbsp;&nbsp; </span></span></li> 
                       <li><span>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; <span class="string">"^(?("</span><span class="string">")("</span><span class="string">"[^"</span><span class="string">"]+?"</span><span class="string">"@)|(([0-9a-zA-Z]((\.(?!\.))|[-!#\$%&amp;'\*\+/=\?\^`\{\}\|~\w])*)(?&lt;=[0-9a-zA-Z])@))"</span><span> +&nbsp;&nbsp; </span></span></li> 
                       <li class="alt"><span>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; <span class="string">"(?(\[)(\[(\d{1,3}\.){3}\d{1,3}\])|(([0-9a-zA-Z][-\w]*[0-9a-zA-Z]\.)+[a-zA-Z]{2,6}))$"</span><span>);&nbsp;&nbsp; </span></span></li> 
                       <li><span>} <span class="comment"># End of IsValidEmail</span><span>&nbsp; </span></span></li> 
                       <li class="alt"><span>&nbsp;&nbsp;&nbsp;&nbsp; </span></li> 
                       <li><span>[string[]] <span class="variable">$emailAddresses</span><span> = </span><span class="string">"david.jones@proseware.com"</span><span>, </span><span class="string">"d.j@server1.proseware.com"</span><span>,&nbsp;&nbsp; </span></span></li> 
                       <li class="alt"><span>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; <span class="string">"jones@ms1.proseware.com"</span><span>, </span><span class="string">"j.@server1.proseware.com"</span><span>,&nbsp;&nbsp; </span></span></li> 
                       <li><span>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; <span class="string">"j@proseware.com9"</span><span>, </span><span class="string">"js#internal@proseware.com"</span><span>,&nbsp;&nbsp; </span></span></li> 
                       <li class="alt"><span>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; <span class="string">"j_9@[129.126.118.1]"</span><span>, </span><span class="string">"j..s@proseware.com"</span><span>,&nbsp;&nbsp; </span></span></li> 
                       <li><span>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; <span class="string">"js*@proseware.com"</span><span>, </span><span class="string">"js@proseware..com"</span><span>,&nbsp;&nbsp; </span></span></li> 
                       <li class="alt"><span>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; <span class="string">"js@proseware.com9"</span><span>, </span><span class="string">"j.s@server1.proseware.com"</span><span>,&nbsp; </span></span></li> 
                       <li><span>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; <span class="string">"tfl@psp.co.uk"</span><span>, </span><span class="string">"cuddly.penguin@cookham.net"</span><span>&nbsp;&nbsp; </span></span></li> 
                       <li class="alt"><span>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; </span></li> 
                       <li><span>ForEach (<span class="variable">$emailAddress</span><span>&nbsp;</span><span class="keyword">in</span><span>&nbsp;</span><span class="variable">$emailAddresses</span><span>)&nbsp;&nbsp;&nbsp; {&nbsp; </span></span></li> 
                       <li class="alt"><span>&nbsp; <span class="keyword">if</span><span> (IsValidEmail(</span><span class="variable">$emailAddress</span><span>)) {&nbsp; </span></span></li> 
                       <li><span>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; <span class="string">"Valid: {0}"</span><span> -f </span><span class="variable">$emailAddress</span><span>&nbsp; </span></span></li> 
                       <li class="alt"><span>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; }&nbsp; </span></li> 
                       <li><span>&nbsp; <span class="keyword">else</span><span> {&nbsp; </span></span></li> 
                       <li class="alt"><span>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; <span class="string">"Invalid: {0}"</span><span> -f </span><span class="variable">$emailAddress</span><span>&nbsp; </span></span></li> 
                       <li><span>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; }&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; </span></li> 
                       <li class="alt"><span>}&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; </span></li> 
                      </ol>
