           <p class="mgbot_10">Java Media Framework is a Java library package that extends the functionality of the <a href="http://www.softpedia.com/get/Programming/SDK-DDK/Sun-Java-JDK.shtml" target="_blank">Java</a> platform by enabling the user to add multimedia content to Java applications. The tool allows developers to create applications that can play video or audio files.</p> 
           <p class="mgbot_10">The package includes several applications that can be used for configuring the library and to manage its preferences or plugins. The JMStudio application is a minimalistic media player that allows you to test the functionality of the library by playing media files.</p> 
           <p class="mgbot_10">The JMF Registry Editor is the tool that allows you to change the preferences such as the maximum cache size or the capture devices. The editor can also change the file extensions assigned to a certain MIME type and manage the codecs used for opening an audio or video file.</p> 
           <p class="mgbot_10">The developers can create a customized version of the library that includes only the classes required by their application. The JMF Customizer enables you to choose the media source, the protocol and the media formats in order to generate the JAR file. This feature can help you decrease the size of your application by using only the components that are required.</p> 
           <p class="mgbot_10">Although the library is able to work with multiple file formats, it does not provide support for the latest codecs. It is recommended to test its functionality for a specific multimedia file before using it for your projects.</p> 
           <p class="mgbot_10">Overall, the Java Media Framework is a useful tool that adds the media playback ability to the Java applications. However, since it does not provide support for the latest codecs you might look for similar libraries. Although other options, such as <a href="http://www.softpedia.com/get/Programming/Components-Libraries/JLayer.shtml" target="_blank">JLayer</a> or <a href="http://www.softpedia.com/get/Programming/Components-Libraries/Xuggle.shtml" target="_blank">Xuggler</a> are dedicated to a certain media type, they can provide better support and compatibility.</p>
