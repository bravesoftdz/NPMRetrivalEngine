              <h2>What's on the XNUCLEO-F411RE</h2> 
              <p><img src="/img/devkit/XNUCLEO-F411RE/XNUCLEO-F411RE-intro.jpg" alt="XNUCLEO-F411RE STM32 development board on board resource"></p> 
              <ol> 
               <li><span class="highlight">STM32F411RET6<br> </span> 
                <ul> 
                 <li>Core : ARM 32-bit Cortex-M4</li> 
                 <li>Operating frequency : 100MHz</li> 
                 <li>Operating voltage : 1.7V-3.6V</li> 
                 <li>Package : LQFP64</li> 
                 <li>Memories : 512kB Flash, 128kB SRAM</li> 
                 <li>Interfaces : 1 x SDIO, 1 x USB 2.0 FS, 5 x SPI or 5 x I2S, 3 x USART, 3 x I2C</li> 
                 <li>AD/DA : 1 x AD (12 bit, 16 channels)</li> 
                </ul> </li> 
               <li><span class="highlight">SPX3819M5 :</span> 3.3V voltage regulator</li> 
               <li><span class="highlight">AMS1117-5.0 :</span> 5.0V voltage regulator</li> 
               <li><span class="highlight">CP2102 :</span> USB to UART convertor</li> 
               <li><span class="highlight">Arduino connector :</span> for connecting Arduino shields</li> 
               <li><span class="highlight">ICSP interface :</span> Arduino ICSP</li> 
               <li><span class="highlight">USB TO UART</span> : for debugging</li> 
               <li><span class="highlight">USB connector</span> : USB communication interface</li> 
               <li><span class="highlight">SWD interface :</span> for programming and debugging</li> 
               <li><span class="highlight">ST Morpho headers :</span> access to VCC, GND and all the I/Os, easy for expansion</li> 
               <li><span class="highlight">6-12V DC input</span></li> 
               <li><span class="highlight">User button</span></li> 
               <li><span class="highlight">Reset button</span></li> 
               <li><span class="highlight">Power indicator</span></li> 
               <li><span class="highlight">User LED</span></li> 
               <li><span class="highlight">500mA fast self-recovery fuse</span></li> 
               <li><span class="highlight">Serial port Rx/Tx indicator</span></li> 
               <li><span class="highlight">8MHz crystal</span></li> 
               <li><span class="highlight">32.768KHz crystal</span></li> 
               <li><span class="highlight">ADC/I2C selection jumper<br> </span> 
                <ul> 
                 <li>short A and B : Arduino A4, A5 is used as I2C</li> 
                 <li>short B and C : Arduino A4, A5 is used as ADC</li> 
                </ul> </li> 
               <li><span class="highlight">UART selection jumper</span></li> 
               <li><span class="highlight">MCU current test jumper :</span> for low power tests</li> 
               <li><span class="highlight">Power selection jumper<br> </span> 
                <ul> 
                 <li>short 5V and U5V : powered from USB connection</li> 
                 <li>short 5V and E5V : powered from external power supply</li> 
                </ul> </li> 
               <li><span class="highlight">USB enable jumper<br> </span> 
                <ul> 
                 <li>short the jumper to enable</li> 
                 <li>open the jumper to disable</li>
