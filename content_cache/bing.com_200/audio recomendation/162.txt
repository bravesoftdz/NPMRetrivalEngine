       <h2 id="Before_You_Start">Before You Start</h2> 
       <span class="anchor" id="line-4"></span>
       <ul>
        <li><p class="line862">Information on this page relates to Ubuntu-supplied and -supported builds of the Mozilla Firefox browser, i.e. the <strong>firefox</strong> package in the Ubuntu repositories. It assumes the use of the <a href="/community/InstallingSoftware">package management tools</a> to install or update plugins. If you are using other, unsupported versions of Mozilla Firefox and would like to install or update plugins, see the <a class="http" href="http://plugindoc.mozdev.org/">Mozilla Browser Plugin Support Documentation</a>. <span class="anchor" id="line-5"></span></p></li>
        <li><p class="line862">You must have <a href="/community/RootSudo">administrative privileges</a> to install packages. <span class="anchor" id="line-6"></span></p></li>
        <li><p class="line862">Make sure you have enabled the <a href="/community/AddingRepositoriesHowto">Universe and Multiverse repositories</a>. See <a href="/community/Repositories/Ubuntu">Managing Repositories with Ubuntu</a> or <a href="/community/Repositories/Kubuntu">Kubuntu</a> for help with this. <span class="anchor" id="line-7"></span></p></li>
        <li><p class="line862">See <a href="/community/InstallingSoftware#head-82ee502162e81ddca57bfba9281ad97c39fd7fbe">Installing Software</a> for help with installing packages.<br> <span class="anchor" id="line-8"></span><span class="anchor" id="line-9"></span></p></li>
       </ul>
       <p class="line867"> </p>
       <h2 id="Flash">Flash</h2> 
       <span class="anchor" id="line-10"></span>
       <p class="line862">For instructions on how to install the Flash plugin, see the <a href="/community/RestrictedFormats#head-c268ba69c6b38af1dc31ea09701c7d296cf971c3">Flash section</a> on the Restricted Formats page. <span class="anchor" id="line-11"></span><span class="anchor" id="line-12"></span></p>
       <p class="line867"> </p>
       <h2 id="Java">Java</h2> 
       <span class="anchor" id="line-13"></span>
       <p class="line862">For instructions on how to install the Java plugin for Firefox, see <a href="/community/Java">Java</a>. <span class="anchor" id="line-14"></span><span class="anchor" id="line-15"></span></p>
       <p class="line867"> </p>
       <h2 id="Adobe_Reader">Adobe Reader</h2> 
       <span class="anchor" id="line-16"></span>
       <p class="line862">For instructions on how to install Adobe Reader plugin for Firefox, see the <a href="/community/AcrobatHowTo">Adobe Reader</a> wiki page. <span class="anchor" id="line-17"></span><span class="anchor" id="line-18"></span></p>
       <p class="line867"> </p>
       <h2 id="Real_Player">Real Player</h2> 
       <span class="anchor" id="line-19"></span>
       <p class="line862">For instructions on how to install the Real Player plugin for Firefox, see the <a href="/community/RealplayerInstallationMethods">RealPlayer page</a>. <span class="anchor" id="line-20"></span><span class="anchor" id="line-21"></span></p>
       <p class="line867"> </p>
       <h2 id="Totem_Plugin">Totem Plugin</h2> 
       <span class="anchor" id="line-22"></span>
       <p class="line867"><a href="/community/MultimediaApplications#head-94cec33eee8ef46857b3f9c873bf978aeeb1e199">Totem</a>, Ubuntu's default video player, supports both the <a href="/community/MultimediaApplications#head-9c497602a3687a126b048cf7c61471e8024ab3e8">gstreamer (default) and xine multimedia frameworks</a>. <span class="anchor" id="line-23"></span></p>
       <ul>
        <li><p class="line862">On 6.06 LTS (Dapper), the plugin depends on the multimedia framework you use. If you use <strong>totem-gstreamer</strong> (the version of Totem that uses the gstreamer back-end), install the <strong>totem-gstreamer-firefox-plugin</strong> package. If you use <strong>totem-xine</strong>, you should install the <strong>totem-xine-firefox-plugin</strong> package. <span class="anchor" id="line-24"></span></p></li>
        <li><p class="line862">On 6.10 (Edgy), simply install the <strong>totem-mozilla</strong> package. <span class="anchor" id="line-25"></span><span class="anchor" id="line-26"></span></p></li>
       </ul>
       <p class="line867"><strong>Note:</strong> The current version of the Totem plugin that handles real media files is not functional. To make Real<strong></strong>Player take over the handling of real media files, follow the instructions in the <a href="/community/RealplayerInstallationMethods#head-02313c026706d5aca8b8b8f2a8a93e02727a1697">RealPlayer and Firefox 2</a> section on the <a href="/community/RealplayerInstallationMethods">RealPlayer wiki page</a>. <span class="anchor" id="line-27"></span><span class="anchor" id="line-28"></span></p>
       <p class="line867"> </p>
       <h2 id="MPlayer_Plugin">MPlayer Plugin</h2> 
       <span class="anchor" id="line-29"></span>
       <p class="line862">The <a href="/community/MPlayer">MPlayer</a> media player is not installed by default on Ubuntu Linux. For 6.06 LTS (Dapper) and 6.10 (Edgy), simply install the <strong>mplayer</strong> package from the multiverse repositories. For users of 5.10 (Breezy) and earlier, see <a class="nonexistent" href="/community/MPlayer/Breezy">MPlayer for Breezy</a>. Once you have set up and configured mplayer, add the plugin to Firefox by installing the <strong>mozilla-mplayer</strong> package. <span class="anchor" id="line-30"></span><span class="anchor" id="line-31"></span></p>
       <p class="line867"><strong>Note:</strong> If your country's laws allow you to use <a href="/community/RestrictedFormats#head-68524fab57e2285050069d6845f95415f8ec8404">w32codecs</a>, installing that package is highly recommended. See the <a href="/community/RestrictedFormats#head-6c942d1939d97331f96e42b63774003fde7daed5">w32codecs section</a> on the Restricted Formats wiki page for details. <span class="anchor" id="line-32"></span><span class="anchor" id="line-33"></span></p>
       <p class="line867"> </p>
       <h2 id="How_do_I_install_other_plugins.3F">How do I install other plugins?</h2> 
       <span class="anchor" id="line-34"></span>
       <p class="line862">See the <a href="/community/MultimediaApplications">wiki page</a> for the media player you are interested in. <a href="/community/RestrictedFormats">The Restricted Formats wiki page</a> offers a comprehensive discussion of proprietary multimedia file formats on (K)Ubuntu Linux. General information about installing other browser plugins is available at <a class="http" href="http://plugindoc.mozdev.org/linux.html">PluginDoc for Linux (x86)</a>. <span class="anchor" id="line-35"></span><span class="anchor" id="line-36"></span></p>
       <p class="line867"> </p>
       <h2 id="Troubleshooting">Troubleshooting</h2> 
       <span class="anchor" id="line-37"></span>
       <ul>
        <li><p class="line891"><em>How can I check which plugins are currently installed? - How can I verify that plugin x for Firefox is properly installed?</em> <span class="anchor" id="line-38"></span><span class="anchor" id="line-39"></span></p>
         <ul>
          <li><p class="line862">Enter <strong>about:plugins</strong> into the Firefox adress bar. You should now see a list of installed plugins along with some additional information (the plugin's file name, a brief description of its functionality and the associated MIME types). <span class="anchor" id="line-40"></span><span class="anchor" id="line-41"></span></p></li>
         </ul></li>
        <li class="gap"><p class="line891"><em>The plugins page only shows me the name of the plugin. How can I make it show the plugin's location in the file system?</em> <span class="anchor" id="line-42"></span><span class="anchor" id="line-43"></span></p>
         <ul>
          <li><p class="line862">Enter <strong>about:config</strong> into the Firefox address bar. You will see a large list of configurable options for the Firefox browser. In the search bar, search for an entry named <strong>plugin.expose_full_path</strong> and set its value to <strong>true</strong>. Enter <strong>about:plugins</strong> into the browser adress bar. You should now see a list of installed plugins along with some additional information and their locations in the file system. <span class="anchor" id="line-44"></span><span class="anchor" id="line-45"></span></p></li>
         </ul></li>
       </ul>
