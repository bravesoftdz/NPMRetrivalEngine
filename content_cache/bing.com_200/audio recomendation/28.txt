It supports ID3v1, ID3v1.1, Lyrics3v1, Lyrics3v2, ID3v2.2, ID3v2.3, and ID3v2.4 tags. MP3 Frame Headers can also be read. There is a FilenameTag, a ID3v2.4 tag that is intelligently derived from the file name. It contains tag synchronization utilities, multiple save options, and easy tag conversion methods. <a href="description.htm"> more ...</a></p> <p>It is hosted at <a href="http://sourceforge.net/projects/javamusictag/"> sourceforge.net</a> </p><p>&nbsp; <a href="http://sourceforge.net"> <img src="http://sourceforge.net/sflogo.php?group_id=74210&amp;type=5" width="210" height="62" border="0" alt="SourceForge.net Logo"> </a> </p><p> It is also now listed on <a href="https://jid3lib.dev.java.net/">java.net</a> </p><p>&nbsp; <a href="http://www.java.net"><img src="images/javanet_button_170.gif" width="170" height="47" border="0" alt="java.net Member Button: 170px wide"> </a> </p></td>
