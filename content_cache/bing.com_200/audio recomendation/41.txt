       <h1 class="title">Windows 7 Packages</h1> 
       <p>Updated: August 1, 2012</p>
       <p>Applies To: Windows 7, Windows 8.1, Windows Server 2012 R2</p>
       <div id="mainSection"> 
        <div id="mainBody"> 
         <p>The following table shows the configurable Windows®&nbsp;7 features. Each feature can be enabled or disabled with Deployment Image Servicing and Management (DISM) and an unattended-installation answer file, <code>OCsetup</code> command, or manually using the <strong>Turn Windows features on or off</strong> link in the <strong>Programs and Features</strong> Control Panel.</p> 
         <p>If a package is enabled, all of its parent hierarchy must also be enabled. To view the entire hierarchy, see the individual package topics.</p> 
         <p>Use the command to enable or to disable the Windows® feature in a Command Prompt window. The command is case-sensitive.</p>
