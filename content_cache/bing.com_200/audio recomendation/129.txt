             <h2>Introduction</h2> 
             <p>At the end of this tutorial, you should be able to create a basic Java program that requests input from a user, does something with the information and displays the results. </p> 
             <p>In this tutorial, we will be making a program that requests the user’s name, the year she/he was born and the current year. It will then display the user’s name and her/his age. </p> 
             <p>The basic structure of a Java program is as follows: </p> 
             <pre lang="Java"><span class="code-keyword">class</span> ClassName {
}</pre> 
             <p>The class name must be the same as the file name, for example. <br>For our program, we will call it <code>Tut1</code>. We will call the file name <em>Tut1.java</em> so the structure will be: </p> 
             <pre lang="Java"><span class="code-keyword">class</span> Tut1{
}</pre> 
             <p>Within a class, you will find methods. The structure of these is as follows: </p> 
             <pre lang="Java">methodName( ){
}</pre> 
             <p>You will notice that there are brackets after the method name. You can place variable names within these brackets which will receive the values passed to the method. I will talk more about this at a later stage. </p> 
             <p>The most common method that you will use is the ‘<code>main</code>’ method. The class that is actually executed must have this method in the class. However not every class must have a ‘<code>main</code>’ method. The structure of the <code>main </code>method is as follows: </p> 
             <pre lang="Java"><span class="code-keyword">public</span> <span class="code-keyword">static</span> <span class="code-keyword">void</span> main(<span class="code-sdkkeyword">String</span> args[])
{
}</pre> 
             <p>Note the capital ‘S’ for <code>String</code>. This is important or you will receive an error. </p> 
             <p>I believe that there are many different ways of outputting text. I will use one method which uses the standard <code>java </code>package. </p> 
             <pre lang="Java">System.out.println(<span class="code-string">"</span><span class="code-string">Welcome To My First Java Program "</span>);</pre> 
             <p>Again in this example, note the capital ‘S’ for <code>System</code>. <br>This code basically prints a line of text which is contained within the quotes. The line is terminated, like most lines in java, with a semi colon. </p> 
             <p>If we put all this code together, it will look like this: </p> 
             <pre lang="Java"><span class="code-keyword">class</span> Tut1 {
     <span class="code-keyword">public</span> <span class="code-keyword">static</span> <span class="code-keyword">void</span> main(<span class="code-sdkkeyword">String</span> args[])
     	{
          System.out.println(<span class="code-string">"</span><span class="code-string">Welcome To My First Java Program"</span>);
     }	
}</pre> 
             <p>This program will output the following on the screen: </p> 
             <pre lang="text">Welcome To My First Java Program </pre> 
             <p>This is the most basic program you can make that provides any functionality. </p> 
             <p>Next we will extend this program to request data from the user via the keyboard. <br>To read input from the keyboard, we will use the standard <code>java </code>classes. We will need to use the <code>IOException </code>class which is in the <code>java.io </code>package. </p> 
             <p>To use this class, we must import the <code>java.io </code>package into this class. This is done by the following: </p> 
             <pre lang="Java"><span class="code-keyword">import</span> java.io.* ;</pre> 
             <p>There are many other packages that you can import in a similar manner. </p> 
             <p>The first step is to create the <code>InputStreamReader</code>. The format is as follows: </p> 
             <pre lang="Java">InputStreamReader varName = <span class="code-keyword">new</span> InputStreamReader(System.in) ;</pre> 
             <p>This will create the reader and assign it to the variable <code>varName</code>. You can change <code>varName </code>to whatever you want provided you follow the naming rules for a variable. <br>This code does the actual reading from the keyboard and converts it to Unicode characters. This is not very useful to us as we want the information in a <code>string</code>. This is where the <code>BufferedReader </code>comes in: </p> 
             <pre lang="Java">BufferedReader varName = <span class="code-keyword">new</span> BufferedReader(varName) ;</pre> 
             <p>Again the rules with the variable names apply here. Also note that you cannot have the <code>InputStreamReader </code>and <code>BufferedReader </code>with the same name. I have only done this for demonstration purposes. Also the <code>varName </code>within the brackets in the <code>BufferedReader </code>must be the variable name or your <code>InputStreamReader</code>. <br>Here is an example of the actual code that we will use for our project: </p> 
             <pre lang="Java">InputStreamReader istream = <span class="code-keyword">new</span> InputStreamReader(System.in) ;

BufferedReader bufRead = <span class="code-keyword">new</span> BufferedReader(istream) ;</pre> 
             <p>Note the case of the characters and the semicolons. </p> 
             <p>For this example, we want to read a line in. You can do this by the following: </p> 
             <pre lang="Java"><span class="code-sdkkeyword">String</span> firstName = bufRead.readLine();</pre> 
             <p>*Note the capital L. </p> 
             <p>To read from the keyboard, you must also create a <code>try catch </code>block: </p> 
             <pre lang="Java"><span class="code-keyword">try</span> {
}
<span class="code-keyword">catch</span> (IOException err) {
}</pre> 
             <p>To use a <code>try catch </code>block, you place the code for reading the values in, in the <code>try </code>section and in the <code>catch </code>block, you place any error messages, or what to do if there is an error. <br>Here is the code for our project: </p> 
             <pre lang="Java"><span class="code-keyword">try</span> {
     <span class="code-sdkkeyword">String</span> firstName = bufRead.readLine();
}
<span class="code-keyword">catch</span> (IOException err) {
     System.out.println(<span class="code-string">"</span><span class="code-string">Error reading line"</span>);
}</pre> 
             <p>This will try to read the input. It will then catch any <code>IOException </code>errors and print out a user friendly error if one occurs. In this case the error is "<code>Error reading line</code>". <br>Within the <code>try </code>block, we will also ask the user to enter in their first name or the user won't know what she/he is supposed to enter: </p> 
             <pre lang="Java"><span class="code-keyword">try</span> {
     System.out.println(<span class="code-string">"</span><span class="code-string">Please Enter In Your First Name: "</span>);
     string firstName = bufRead.readLine();
}
<span class="code-keyword">catch</span> (IOException err) {
     System.out.println(<span class="code-string">"</span><span class="code-string">Error reading line"</span>);
}</pre> 
             <p>Let's now put what we have so far of our code together. </p> 
             <pre lang="Java"><span class="code-keyword">import</span> java.io.* ;
<span class="code-keyword">class</span> Tut1 {
     <span class="code-keyword">public</span> <span class="code-keyword">static</span> <span class="code-keyword">void</span> main(<span class="code-sdkkeyword">String</span> args[])
     {
               InputStreamReader istream = <span class="code-keyword">new</span> InputStreamReader(System.in) ;
               BufferedReader bufRead = <span class="code-keyword">new</span> BufferedReader(istream) ;
               
               System.out.println(<span class="code-string">"</span><span class="code-string">Welcome To My First Java Program"</span>);
          <span class="code-keyword">try</span> {
               System.out.println(<span class="code-string">"</span><span class="code-string">Please Enter In Your First Name: "</span>);
               <span class="code-sdkkeyword">String</span> firstName = bufRead.readLine();
          }
          <span class="code-keyword">catch</span> (IOException err) {
               System.out.println(<span class="code-string">"</span><span class="code-string">Error reading line"</span>);
          }
     }	
}</pre> 
             <p>Congratulations! You have now made a fairly useless program. It asks for information and does nothing with it. </p> 
             <p>We will now expand on this base code. <br>We also want to ask for the user’s birth year and the current year: </p> 
             <pre lang="Java">System.out.println(<span class="code-string">"</span><span class="code-string">Please Enter In The Year You Were Born: "</span>);
<span class="code-sdkkeyword">String</span> bornYear = bufRead.readLine();

System.out.println(<span class="code-string">"</span><span class="code-string">Please Enter In The Current Year: "</span>);
<span class="code-sdkkeyword">String</span> thisYear = bufRead.readLine();</pre> 
             <p>From this information, we want to calculate the person’s age. But there is a problem - how do you take a <code>string </code>from another <code>string</code>. You can't, so we must change the <code>string </code>value into a numeric value. We will convert the <code>string </code>values into <code>Integer </code>values using the <code>parseInt </code>function: </p> 
             <pre lang="Java"><span class="code-keyword">int</span> bYear = Integer.parseInt(bornYear);
<span class="code-keyword">int</span> tYear = Integer.parseInt(thisYear);</pre> 
             <p>This code must go in a <code>try catch </code>block. You can then catch the conversion error: </p> 
             <pre lang="Java"><span class="code-keyword">catch</span>(NumberFormatException err) {
     System.out.println(<span class="code-string">"</span><span class="code-string">Error Converting Number"</span>);
}</pre> 
             <p>The next step is to calculate the age of the person and output all the data: </p> 
             <pre lang="Java"><span class="code-keyword">int</span> age = tYear – bYear ;
System.out.println(<span class="code-string">"</span><span class="code-string">Hello "</span> + firstName + <span class="code-string">"</span><span class="code-string">. You are "</span> + age + <span class="code-string">"</span><span class="code-string"> years old"</span>);</pre> 
             <p>The last step is to combine all our code: </p> 
             <pre lang="Java"><span class="code-keyword">import</span> java.io.* ;

<span class="code-keyword">class</span> Tut1 {
     <span class="code-keyword">public</span> <span class="code-keyword">static</span> <span class="code-keyword">void</span> main(<span class="code-sdkkeyword">String</span> args[])
     {
          InputStreamReader istream = <span class="code-keyword">new</span> InputStreamReader(System.in) ;

          System.out.println(<span class="code-string">"</span><span class="code-string">Welcome To My First Java Program"</span>);
	
          <span class="code-keyword">try</span> {

               <span class="code-sdkkeyword">String</span> firstName = bufRead.readLine();

               System.out.println(<span class="code-string">"</span><span class="code-string">Please Enter In The Year You Were Born: "</span>);
               <span class="code-sdkkeyword">String</span> bornYear = bufRead.readLine();

               System.out.println(<span class="code-string">"</span><span class="code-string">Please Enter In The Current Year: "</span>);
               <span class="code-sdkkeyword">String</span> thisYear = bufRead.readLine();

               <span class="code-keyword">int</span> bYear = Integer.parseInt(bornYear);
               <span class="code-keyword">int</span> tYear = Integer.parseInt(thisYear);

               <span class="code-keyword">int</span> age = tYear – bYear ;
               System.out.println(<span class="code-string">"</span><span class="code-string">Hello "</span> + firstName + <span class="code-string">"</span><span class="code-string">. 
				You are "</span> + age + <span class="code-string">"</span><span class="code-string"> years old"</span>);

               System.out.println(<span class="code-string">"</span><span class="code-string">Error reading line"</span>);

               System.out.println(<span class="code-string">"</span><span class="code-string">Error Converting Number"</span>);
          }			
     }
}</pre>
