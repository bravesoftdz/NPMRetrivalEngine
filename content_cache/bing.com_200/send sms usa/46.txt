       <h1><span class="TitleTag">JAVA OBJECT API Documentation</span></h1> 
       <div class="text2"> 
        <style type="text/css">
    .auto-style1 {
    margin-top: 0px;
    }
</style> 
        <p>You can create a Java client and connect to ViaNett's server to send or receive SMS or MMS messages. The latest stable version of this package&nbsp;is 2.1, which can be downloaded from ViaNett's website.</p> 
        <h2>Requirements</h2> 
        <ul class="auto-style1"> 
         <li>ViaNett SMS username and account. Please contact the Customer Services if you don't have one.</li> 
         <li>Destination host: cpa.vianett.no</li> 
         <li> Destination port:&nbsp;31337, 31338 or 31339.</li> 
         <li>Java packages</li> 
        </ul>

        <h2>Examples</h2> 
        <p>We provide different types of Java examples.</p> 
        <p><strong>SMS Send example<br> </strong>To create a Java application with send only functionality, please visit: <a href="/en/smsadmin/download?fileid=10">SmsSender.java</a></p> 
        <p><a href="/en/smsadmin/download?fileid=10"></a><strong>SMS send/receive example</strong><br> To create a Java application with send and receive functionalities please visit: <a href="/en/smsadmin/download?fileid=11">SmsEcho.java</a>&nbsp;(Version 1.7)</p> 
        <p><strong>JSP example</strong><br> To create an SMS sender object in JSP, please refer to: <a href="/en/smsadmin/download?fileid=12">jspWithSms.jsp</a></p>
