       <h1 class="headline" itemprop="headline">Can JavaScript email a form?</h1> 
       <div class="byline"> 
        <p class="post_cats" itemprop="keywords"> <a href="http://javascript-coder.com/category/jsform" rel="category tag">Form Handling</a>, <a href="http://javascript-coder.com/category/javascript-form" rel="category tag">Javascript Form Handling</a> </p> 
        <p class="post_tags" itemprop="keywords"> <a href="http://javascript-coder.com/tag/beginner" rel="tag">beginner</a>, <a href="http://javascript-coder.com/tag/form-mail" rel="tag">form mail</a>, <a href="http://javascript-coder.com/tag/javascript-email-form" rel="tag">javascript email form</a>, <a href="http://javascript-coder.com/tag/mailto" rel="tag">mailto</a> </p> 
       </div>

      <div class="post_content" itemprop="articleBody"> 
       <p>NO! JavaScript can’t email a form! but, there are alternatives to send the form data to an email address.</p>
       <div class="C20r1qfs" style="clear:both;float:left;width:100%;margin:0 0 20px 0;">
        <div class="g g-2">
         <div class="g-single a-8">
          <a href="https://ajdg.solutions/"><img src="http://javascript-coder.com/wp-content/uploads/2017/08/simfatic-long-banner-3.png"></a>
         </div>
        </div>
       </div> 
       <p>There is no direct method provided by JavaScript to send the data submitted in the form to an email address.</p> 
       <p>The main concern for not providing a ‘JavaScript email form’ feature is security.<br> Suppose there is a feature in JavaScript to send email. Some malicious coder can write a script to send email to some address immediately when you visit their page. This will reveal your email address to some third party without your knowledge. They will start filling your mail box with lots of spam messages! However, there are alternatives as explained below.<br> <span id="more-320"></span></p> 
       <div class="myadv1"> 
        <h2>Build Web Forms Faaaast!!</h2> 
        <p>Quickly create your form, install it and receive form submissions by email.<br> <a href="http://www.simfatic.com/forms/benefits.php"><img src="http://javascript-coder.com/wp-content/uploads/2010/09/form-submit-larger.jpg" alt="Simfatic Forms : customizable forms" width="544" height="261" class="aligncenter size-full wp-image-306" srcset="http://javascript-coder.com/wp-content/uploads/2010/09/form-submit-larger.jpg 544w, http://javascript-coder.com/wp-content/uploads/2010/09/form-submit-larger-300x143.jpg 300w, http://javascript-coder.com/wp-content/uploads/2010/09/form-submit-larger-500x239.jpg 500w" sizes="(max-width: 544px) 100vw, 544px"></a> Using <strong>Simfatic Forms</strong> it is very easy to design,<br> create, install and maintain web forms;<br> no coding required! <a href="http://www.simfatic.com/forms/benefits.php">Click here for more info<br> </a></p>
       </div> 
       <h2>Using ‘mailto:’</h2> 
       <p>You can use ‘mailto:me@mydomain.com’ in the action field of the form. When the user presses the submit button of the form, the browser will first show a warning box that the user’s email address will be revealed to the recipient.<br> <img src="http://javascript-coder.com/wp-content/uploads/2010/09/javascript-email-form.jpg" alt="mailto dialog box" width="410" height="164" class="alignnone size-full wp-image-327" srcset="http://javascript-coder.com/wp-content/uploads/2010/09/javascript-email-form.jpg 410w, http://javascript-coder.com/wp-content/uploads/2010/09/javascript-email-form-300x120.jpg 300w" sizes="(max-width: 410px) 100vw, 410px"></p> 
       <p>If the user decides to proceed, the form data will be sent to the address mentioned in the mailto: tag. The browser will open the default mail client to send the data by email.</p> 
       <h2>Sample Code</h2> 
       <div> 
        <pre class="brush: xml; title: ; notranslate" title="">
&lt;form action="mailto:you@yourdmainhere.com" method="post" enctype="text/plain" &gt;
FirstName:&lt;input type="text" name="FirstName"&gt;
Email:&lt;input type="text" name="Email"&gt;
&lt;input type="submit" name="submit" value="Submit"&gt;
&lt;/form&gt;
</pre> 
       </div> 
       <p> See the sample code above at work:<br> <a href="http://javascript-coder.com/files/email-form/javascript-email-form-example1.html">JavaScript mail Form Example 1</a></p> 
       <p>Notice that this method does not require JavaScript at all.</p> 
       <h2>Disadvantages of ‘mailto:’ method</h2> 
       <p> It is not possible to control the formatting of the form data. The data submitted in the form is sent by the browser. You can’t control the layout of the data.</p> 
       <p>Browser shows the warning window before sending the data. It may prevent the visitor from submitting the form.</p> 
       <h2>Using Server side script to send email</h2> 
       <p>The right way to send form submissions to email is by receiving the form submissions in a server-side script and then sending the email. The server side scripting language (like PHP Perl, ASP) has more features and facilities to send the email.</p>
