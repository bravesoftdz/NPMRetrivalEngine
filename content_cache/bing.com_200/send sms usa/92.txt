       <p class="opening"> Java is a general-purpose, class-based programming language that is specifically designed to have as few implementation dependencies as possible. The Ozeki Phone System XE provides a Java application, so you can use your good old programming language with your brand new phone system. If you have any questions concerning making/receiving calls or sending/receiving text messages, read this guide, because it contains the most important and relevant information. </p> 
       <ul style="margin-left:5px;">

       <p align="right"> <b>Part 2:</b><a href="voip-part2-java-example-on-recording-voip-calls-call-routing-762.html"> Java example on recording VoIP calls, call routing</a> </p> 
       <h3>1. What you need</h3> 
       <ul> 
        <li>Java IDE (Netbeans, Eclipse ect.)</li> 
        <li>Ozeki Phone System XE installed on your PC (<a href="voip-download-ozeki-phone-system-xe-pbx-software-for-windows-21.html">Download now</a>)</li> 
        <li><a href="voip-softphone-installed-on-pc-739.html">A connected softphone</a></li>

       <h3>2. Get started</h3> 
       <ul> 
        <li><a href="voip-how-to-install-api-extension-741.html">Add an API Extension to your PBX</a></li> 
        <li>Make a JAVA project</li> 
       </ul> 
       <p> <b> Installing Netbeans and JDK </b> </p> 
       <p> First, download and install Java Developer Kit (JDK), you can download its latest version, from <a href="http://www.oracle.com/technetwork/java/javase/downloads/index.html">here</a>. After installing JDK, do the same, with Netbeans (<b>Figure 1</b>). You can download it from the previous link, too. Meanwhile install Netbeans, give the destination folder. </p> 
       <p align="center"> <img SRC="attachments/752/JAVA_NetBeans_Installation.jpg" border="0"> <br> <b>Figure 1 </b>- Installing Netbeans </p> 
       <p> You must select the folder of the previously installed JDK. After the installation of Netbeans, you can start a new project by selecting File from the menu, New Project item. </p> 
       <h3>3. Receive incoming voice calls</h3> 
       <p> First, select which port receives the incoming calls, using http API at Productivity menu, HTTP API menu item. Here, you need to add a new API extension notification, then you can set the address of incoming calls, using the URL field of Incoming call. In this example, this is the 7778 port of localhost. Now, make a java application which checks this port and gives a http response if a call arrives. </p> 
       <div class="ShowHideDiv"> 
        <pre class="syntax brush-csharp">
public static Initialize() {
	ServerSocket listener = new ServerSocket(7778));
	while (true){
		Socket socket = listener.accept();
        
		String responde = "HTTP/1.1 200 OKrn"
						+ "Content-Type: text/xmlrn"
						+ "Connection: Keep-Alivern"
						+ "Keep-Alive: timeout=15, max=100rnrn"
						+ "&lt;?xml version=1.0" encoding="UTF-8"?&gt;rn"
						+ "&lt;Response&gt;rn"
						+ "&lt;Delay&gt;1&lt;/Delay&gt;rn"
						+ "&lt;Speak&gt;Congratulations, this is your first OZML Response command.&lt;/Speak&gt;rn"
						+ "&lt;Delay&gt;1&lt;/Delay&gt;rn"
						+ "&lt;Speak&gt;For more information on OZML syntax, visit www.ozekiphone.com &lt;/Speak&gt;rn"
						+ "&lt;/Response&gt;";
        
		OutputStream outputStream = socket.getOutputStream();
		outputStream.write(responde.getBytes());
		outputStream.flush();

		socket.close();
    }
}
