     <h1 id="send-mail-task" sourcefile="integration-services/control-flow/send-mail-task.md" sourcestartlinenumber="28" sourceendlinenumber="28">Send Mail Task</h1>

      <div class="content"> 
       <p> The Send Mail task sends an e-mail message. By using the Send Mail task, a package can send messages if tasks in the package workflow succeed or fail, or send messages in response to an event that the package raises at run time. For example, the task can notify a database administrator about the success or failure of the Backup Database task. </p> 
       <p> You can configure the Send Mail task in the following ways: </p> 
       <ul> 
        <li><p>Provide the message text for the e-mail message. </p> </li> 
        <li><p>Provide a subject line for the e-mail message. </p> </li> 
        <li><p>Set the priority level of the message. The task supports three priority levels: normal, low, and high. </p> </li> 
        <li><p>Specify the recipients on the To, Cc, and Bcc lines. If the task specifies multiple recipients, they are separated by semicolons. </p> 
         <div class="NOTE">
          <h5>Note</h5>
          <p> The To, Cc, and Bcc lines are limited to 256 characters each in accordance with Internet standards. </p> 
         </div> </li> 
        <li><p>Include attachments. If the task specifies multiple attachments, they are separated by the pipe (|) character. </p> 
         <div class="NOTE">
          <h5>Note</h5>
          <p> If an attachment file does not exist when the package runs, an error occurs. </p> 
         </div> </li> 
        <li><p>Specify the SMTP connection manager to use. </p> 
         <div class="IMPORTANT">
          <h5>Important</h5>
          <p> The SMTP connection manager supports only anonymous authentication and Windows Authentication. It does not support basic authentication. </p> 
         </div> <p>The message text can be a string that you provide, a connection to a file that contains the text, or the name of a variable that contains the text. The task uses a File connection manager to connect to a file. For more information, see <a href="../connection-manager/flat-file-connection-manager" data-linktype="relative-path">Flat File Connection Manager</a>. </p> <p>The task uses an SMTP connection manager to connect to a mail server. For more information, see <a href="../connection-manager/smtp-connection-manager" data-linktype="relative-path">SMTP Connection Manager</a>. </p> </li> 
       </ul> 
       <h2 id="custom-logging-messages-available-on-the-send-mail-task">Custom Logging Messages Available on the Send Mail Task</h2> 
       <p> The following table lists the custom log entries for the Send Mail task. For more information, see <a href="../performance/integration-services-ssis-logging" data-linktype="relative-path">Integration Services (SSIS) Logging</a>. </p> 
       <table> 
        <thead> 
         <tr> 
          <th>Log entry</th> 
          <th>Description</th> 
         </tr> 
        </thead> 
        <tbody> 
         <tr> 
          <td><strong>SendMailTaskBegin</strong></td> 
          <td>Indicates that the task began to send an e-mail message.</td> 
         </tr> 
         <tr> 
          <td><strong>SendMailTaskEnd</strong></td> 
          <td>Indicates that the task finished sending an e-mail message.</td> 
         </tr> 
         <tr> 
          <td><strong>SendMailTaskInfo</strong></td> 
          <td>Provides descriptive information about the task.</td> 
         </tr> 
        </tbody> 
       </table> 
       <h2 id="configuring-the-send-mail-task">Configuring the Send Mail Task</h2> 
       <p> You can set properties through SSIS Designer or programmatically. </p> 
       <p> For information about the properties that you can set in SSIS Designer, click the following topic: </p> 
       <ul> 
        <li><p><a href="../expressions/expressions-page" data-linktype="relative-path">Expressions Page</a> </p> <p>For information about programmatically setting these properties, click the following topic: </p> </li> 
        <li><p><a class="xref" href="https://msdn.microsoft.com/library/microsoft.sqlserver.dts.tasks.sendmailtask.sendmailtask(v=SQL.130).aspx" data-linktype="external">SendMailTask</a> </p> </li> 
       </ul> 
       <h2 id="related-tasks">Related Tasks</h2> 
       <p> For information about how to set these properties in SSIS Designer, click <a href="http://msdn.microsoft.com/library/52d47ca4-fb8c-493d-8b2b-48bb269f859b" data-linktype="external">Set the Properties of a Task or Container</a>. </p> 
       <h2 id="related-content">Related Content</h2> 
       <ul> 
        <li>Technical article, <a href="http://go.microsoft.com/fwlink/?LinkId=237625" data-linktype="external">How to send email with delivery notification in C#</a>, on shareourideas.com </li> 
       </ul> 
       <h2 id="send-mail-task-editor-general-page">Send Mail Task Editor (General Page)</h2> 
       <p> Use the <strong>General page</strong> of the <strong>Send Mail Task Editor</strong> dialog box to name and describe the Send Mail task. </p> 
       <h3 id="options">Options</h3> 
       <p> <strong>Name</strong><br> Provide a unique name for the Send Mail task. This name is used as the label in the task icon. </p> 
       <p> <strong>Note</strong> Task names must be unique within a package. </p> 
       <p> <strong>Description</strong><br> Type a description of the Send Mail task. </p> 
       <h2 id="send-mail-task-editor-mail-page">Send Mail Task Editor (Mail Page)</h2> 
       <p> Use the <strong>Mail</strong> page of the <strong>Send Mail Task Editor</strong> dialog box to specify recipients, message type, and priority for a message. You can also attach files to the message. The message text can be a string you provide, a file connection to a file that contains the text, or the name of a variable that contains the text. </p> 
       <h3 id="options">Options</h3> 
       <p> <strong>SMTPConnection</strong><br> Select an SMTP connection manager in the list, or click <strong>&lt;New connection…&gt;</strong> to create a new connection manager. </p> 
       <div class="IMPORTANT">
        <h5>Important</h5>
        <p> The SMTP connection manager supports only anonymous authentication and Windows Authentication. It does not support basic authentication. </p> 
       </div> 
       <p> <strong>Related Topics:</strong> <a href="../connection-manager/smtp-connection-manager" data-linktype="relative-path">SMTP Connection Manager</a> </p> 
       <p> <strong>From</strong><br> Specify the e-mail address of the sender. </p> 
       <p> <strong>To</strong><br> Provide the e-mail addresses of the recipients, delimited by semicolons. </p> 
       <p> <strong>Cc</strong><br> Specify the e-mail addresses, delimited by semicolons, of individuals who also receive copies of the message. </p> 
       <p> <strong>Bcc</strong><br> Specify the e-mail addresses, delimited by semicolons, of individuals who receive blind carbon copies (Bcc) copies of the message. </p> 
       <p> <strong>Subject</strong><br> Provide a subject for the e-mail message. </p> 
       <p> <strong>MessageSourceType</strong><br> Select the source type of the message. This property has the options listed in the following table. </p> 
       <table> 
        <thead> 
         <tr> 
          <th>Value</th> 
          <th>Description</th> 
         </tr> 
        </thead> 
        <tbody> 
         <tr> 
          <td><strong>Direct input</strong></td> 
          <td>Set the source to the message text. Selecting this value displays the dynamic option, <strong>MessageSource</strong>.</td> 
         </tr> 
         <tr> 
          <td><strong>File connection</strong></td> 
          <td>Set the source to the file that contains the message text. Selecting this value displays the dynamic option, <strong>MessageSource</strong>.</td> 
         </tr> 
         <tr> 
          <td><strong>Variable</strong></td> 
          <td>Set the source to a variable that contains the message text. Selecting this value displays the dynamic option, <strong>MessageSource</strong>.</td> 
         </tr> 
        </tbody> 
       </table> 
       <p> <strong>Priority</strong><br> Set the priority of the message. </p> 
       <p> <strong>Attachments</strong><br> Provide the file names of attachments to the e-mail message, delimited by the pipe (|) character. </p> 
       <div class="NOTE">
        <h5>Note</h5>
        <p> The To, Cc, and Bcc lines are limited to 256 characters in accordance with Internet standards. </p> 
       </div> 
       <h3 id="messagesourcetype-dynamic-options">MessageSourceType Dynamic Options</h3> 
       <h4 id="messagesourcetype--direct-input">MessageSourceType = Direct Input</h4> 
       <p> <strong>MessageSource</strong><br> Type the message text or click the browse button (…) and then type the message in the <strong>Message source</strong> dialog box. </p> 
       <h4 id="messagesourcetype--file-connection">MessageSourceType = File connection</h4> 
       <p> <strong>MessageSource</strong><br> Select a File connection manager in the list or click &lt;<strong>New connection…</strong>&gt; to create a new connection manager. </p> 
       <p> <strong>Related Topics:</strong> <a href="../connection-manager/file-connection-manager" data-linktype="relative-path">File Connection Manager</a>, <a href="../connection-manager/file-connection-manager-editor" data-linktype="relative-path">File Connection Manager Editor</a> </p> 
       <h4 id="messagesourcetype--variable">MessageSourceType = Variable</h4> 
       <p> <strong>MessageSource</strong><br> Select a variable in the list or click &lt;<strong>New variable…</strong>&gt; to create a new variable. </p> 
       <p> <strong>Related Topics:</strong> <a href="../integration-services-ssis-variables" data-linktype="relative-path">Integration Services (SSIS) Variables</a>, <a href="http://msdn.microsoft.com/library/d09b5d31-433f-4f7c-8c68-9df3a97785d5" data-linktype="external">Add Variable</a> </p>
