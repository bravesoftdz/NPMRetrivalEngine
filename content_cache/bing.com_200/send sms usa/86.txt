      <p>Let’s play a little with BroadcastReceiver: we will try to send a message from Android service we made to Activity and log that message.</p> 
      <p>Let’s use our Android service tutorial project from one of my earlier articles.</p> 
      <p>First of all we need to define private variable in our Activity:<span id="more-12312"></span></p> 
      <pre class="ish"><code class="language-php"><span class="co1">//...</span>
&nbsp;
<span class="kw2">public</span> <span class="kw2">class</span> AndroidservicetutorialActivity <span class="kw2">extends</span> Activity <span class="br0">{</span>
&nbsp;
	<span class="kw2">private</span> BroadcastReceiver mReceiver<span class="sy0">;</span>
&nbsp;
<span class="co1">//...</span></code></pre> 
      <p>Then we override methods onResume() where our broadcast receiver will be registered and also onPause() where will our receiver be unregistered:</p> 
      <pre class="ish"><code class="language-php"><span class="co1">//AndroidservicetutorialActivity.java</span>
<span class="co1">//...</span>
	<span class="sy0">@</span>Override
	<span class="kw2">protected</span> void onResume<span class="br0">(</span><span class="br0">)</span> <span class="br0">{</span>
		<span class="co1">// TODO Auto-generated method stub</span>
		super<span class="sy0">.</span>onResume<span class="br0">(</span><span class="br0">)</span><span class="sy0">;</span>
&nbsp;
		IntentFilter intentFilter <span class="sy0">=</span> <span class="kw2">new</span> IntentFilter<span class="br0">(</span>
				<span class="st0">"android.intent.action.MAIN"</span><span class="br0">)</span><span class="sy0">;</span>
&nbsp;
		mReceiver <span class="sy0">=</span> <span class="kw2">new</span> BroadcastReceiver<span class="br0">(</span><span class="br0">)</span> <span class="br0">{</span>
&nbsp;
			<span class="sy0">@</span>Override
			<span class="kw2">public</span> void onReceive<span class="br0">(</span>Context context<span class="sy0">,</span> Intent intent<span class="br0">)</span> <span class="br0">{</span>
				<span class="co1">//extract our message from intent</span>
				String msg_for_me <span class="sy0">=</span> intent<span class="sy0">.</span>getStringExtra<span class="br0">(</span><span class="st0">"some_msg"</span><span class="br0">)</span><span class="sy0">;</span>
				<span class="co1">//log our message value</span>
				<span class="kw3">Log</span><span class="sy0">.</span>i<span class="br0">(</span><span class="st0">"InchooTutorial"</span><span class="sy0">,</span> msg_for_me<span class="br0">)</span><span class="sy0">;</span>
&nbsp;
			<span class="br0">}</span>
		<span class="br0">}</span><span class="sy0">;</span>
		<span class="co1">//registering our receiver</span>
		this<span class="sy0">.</span>registerReceiver<span class="br0">(</span>mReceiver<span class="sy0">,</span> intentFilter<span class="br0">)</span><span class="sy0">;</span>
	<span class="br0">}</span>
&nbsp;
	<span class="sy0">@</span>Override
	<span class="kw2">protected</span> void onPause<span class="br0">(</span><span class="br0">)</span> <span class="br0">{</span>
		<span class="co1">// TODO Auto-generated method stub</span>
		super<span class="sy0">.</span>onPause<span class="br0">(</span><span class="br0">)</span><span class="sy0">;</span>
		<span class="co1">//unregister our receiver</span>
		this<span class="sy0">.</span>unregisterReceiver<span class="br0">(</span>this<span class="sy0">.</span>mReceiver<span class="br0">)</span><span class="sy0">;</span>
	<span class="br0">}</span>
<span class="co1">//...</span></code></pre> 
      <p>Now we can try from our service (onStart) send a broadcast message and we will see what is happening in LogCat window:</p> 
      <pre class="ish"><code class="language-php"><span class="co1">//firstService.java</span>
package com<span class="sy0">.</span>inchoo<span class="sy0">.</span>tutorial<span class="sy0">;</span>
&nbsp;
import android<span class="sy0">.</span>app<span class="sy0">.</span>Service<span class="sy0">;</span>
import android<span class="sy0">.</span>content<span class="sy0">.</span>Intent<span class="sy0">;</span>

import android<span class="sy0">.</span>util<span class="sy0">.</span><span class="kw3">Log</span><span class="sy0">;</span>
&nbsp;
<span class="kw2">public</span> <span class="kw2">class</span> FirstService <span class="kw2">extends</span> Service<span class="br0">{</span>
&nbsp;
	<span class="kw2">private</span> static String TAG <span class="sy0">=</span> <span class="st0">"Inchoo.net tutorial"</span><span class="sy0">;</span>
&nbsp;
	<span class="sy0">@</span>Override
	<span class="kw2">public</span> IBinder onBind<span class="br0">(</span>Intent arg0<span class="br0">)</span> <span class="br0">{</span>
		<span class="co1">// TODO Auto-generated method stub</span>
		<span class="kw1">return</span> <span class="kw4">null</span><span class="sy0">;</span>
	<span class="br0">}</span>
&nbsp;
	<span class="sy0">@</span>Override
	<span class="kw2">public</span> void onStart<span class="br0">(</span>Intent intent<span class="sy0">,</span> int startId<span class="br0">)</span> <span class="br0">{</span>
		<span class="co1">// TODO Auto-generated method stub</span>
		super<span class="sy0">.</span>onStart<span class="br0">(</span>intent<span class="sy0">,</span> startId<span class="br0">)</span><span class="sy0">;</span>
		<span class="kw3">Log</span><span class="sy0">.</span>d<span class="br0">(</span>TAG<span class="sy0">,</span> <span class="st0">"FirstService started"</span><span class="br0">)</span><span class="sy0">;</span>
&nbsp;
		Intent i <span class="sy0">=</span> <span class="kw2">new</span> Intent<span class="br0">(</span><span class="st0">"android.intent.action.MAIN"</span><span class="br0">)</span><span class="sy0">.</span>putExtra<span class="br0">(</span><span class="st0">"some_msg"</span><span class="sy0">,</span> <span class="st0">"I will be sent!"</span><span class="br0">)</span><span class="sy0">;</span>
		this<span class="sy0">.</span>sendBroadcast<span class="br0">(</span>i<span class="br0">)</span><span class="sy0">;</span>
&nbsp;
		this<span class="sy0">.</span>stopSelf<span class="br0">(</span><span class="br0">)</span><span class="sy0">;</span>
	<span class="br0">}</span>
&nbsp;
	<span class="sy0">@</span>Override
	<span class="kw2">public</span> void onDestroy<span class="br0">(</span><span class="br0">)</span> <span class="br0">{</span>
		<span class="co1">// TODO Auto-generated method stub</span>
		super<span class="sy0">.</span>onDestroy<span class="br0">(</span><span class="br0">)</span><span class="sy0">;</span>
		<span class="kw3">Log</span><span class="sy0">.</span>d<span class="br0">(</span>TAG<span class="sy0">,</span> <span class="st0">"FirstService destroyed"</span><span class="br0">)</span><span class="sy0">;</span>
	<span class="br0">}</span>
&nbsp;
<span class="br0">}</span></code></pre>

         <div class="comment-content"> 
          <p>@Darko Goles, thanks for the quick reply. Your tutorial is very well explained. </p> 
          <p>I was wondering how can you notify an Activity when it is onPause and has previously launched an IntentService waiting for it to finish. </p> 
          <p>Let’s suppose you update the UI according to a network response made by an IntentService. The Activity launches the IntentService, activity goes to background (onPause is called), IntentService finishes and wants to notify UI. If broadcast receiver is not attached, the activity will not receive the response. For this scenario I have been looking around for a robust solution and came up with these possibilities:<br> – Handle Activity-IntentService communication with StickBroadcast in order for the activity to get the last broadcast sent by the IntentService when it comes to foreground (very hacky solution I think)<br> -Handle Activity-Service communication binding the Activity to the Service and querying the Service to know if it has finished operation (binding makes source code grow up a lot i think and seems way to much effort)<br> – Handle Activity-IntentService communication and operation on a worker Fragment (setting fragment to setRetainInstance(true). I think this approach is the convenient one but comes with too much effort also.</p> 
          <p>According to your experience, which of these possibilities is the correct one? If none, which one do you prefer to use? Thanks.</p>
