      <h1>PHP sample - Send SMS text through a COM/USB GSM modem</h1> 
      <p> The ActiveXperts SMS Component is a software development kit (SDK) to enhance an application or script with SMS or Pager functionality. SMS messages can be sent/received using a GSM modem, an SMPP provider or an HTTP compliant SMSC. </p> 
      <hr> 
      <h2>Introduction</h2> 
      <p> In this example we are going to create a PHP page to send SMS messages. This demo project will ask the user to give a phone number and a message body in the web interface. A GSM modem is required for this demo. </p> 
      <h2>Step 1: Installation of the ActiveXperts SMS Component</h2> 
      <p> On each client PC, download the ActiveXperts SMS Component from the <a href="/download/?p1=ax004">ActiveXperts Download Site</a> and start the installation. The installation guides you through the installation process. </p> 
      <h2>Step 2: Create the SMS objects in PHP</h2> 
      <p> The following code will show you how to declare and create the GSM and SMS objects. We will use the 'objGsm' object to send the message itself. The 'objSmsMessage' object will be used to store information of the message and the 'objSmsConstants' object containes constant values releated to the SMS objects. </p> 
      <!-- //////////////////////////////////////////////////////////////////////////////////////////////--> 
      <pre class="ax_code">
$objGsm          = new COM("AxSms.Gsm");	
$objSmsMessage   = new COM("AxSms.Message");
$objSmsConstants = new COM("AxSms.Constants");
</pre> 
      <h2>Step 3: Gather information</h2> 
      <p> The following code will ask the user for the recipient telephone number and the content of the text message. This data will be stored in the 'objSmsMessage' object. If any error occours while trying to open the connection, the script will end. </p> 
      <!-- ////////////////////////////////////////////////////////////////////////////////////////////// --> 
      <pre class="ax_code">
//Message Settings
$objSmsMessage-&gt;Clear();
$objSmsMessage-&gt;ToAddress = $_POST["txtToAddress"];
$objSmsMessage-&gt;Body = $_POST["txtBody"];
$objSmsMessage-&gt;BodyFormat = $objSmsConstants-&gt;BODYFORMAT_TEXT;
</pre> 
      <h2>Step 4: Send the SMS message</h2> 
      <p> The following code shows how to send an SMS message using the data that was stored in the 'objSmsMessage' and 'objSmsConstants' objects. The timeout value is set to 10 seconds, which means that the modem will try to send the message for 10 seconds, after this value it will stop trying. </p> 
      <!-- ////////////////////////////////////////////////////////////////////////////////////////////// --> 
      <pre class="ax_code">
//Send the message !
$obj = $objSmsMessage;
$objGsm-&gt;SendSms($obj, $iMultipart, 10000);
$objSmsMessage = $obj;
$strMessageReference = $objSmsMessage-&gt;Reference;
</pre> 
      <h2>Appendix: Full source code</h2> 
      <p> Following you can find the full source code which is also included in the ActiveXperts SMS Component package. </p> 
      <!-- //////////////////////////////////////////////////////////////////////////////////////////////--> 
      <pre class="ax_code">
&lt;?php
  $strResult = "n/a";
  
  $objGsm = new COM("AxSms.Gsm");
  $objGsm-&gt;LogFile = sys_get_temp_dir()."ActiveXperts.Gsm.log"; 
  //Windows default: 'C:\Windows\Temp\ActiveXperts.Gsm.log'
  
  //Form submitted
  if (isset($_POST["btnSendMessage"]))
  {
    $obj;
    $strMessageReference;
    $objSmsMessage = new COM("AxSms.Message");
    $objSmsConstants = new COM("AxSms.Constants");
    
    $strName = $_POST["ddlDevices"];
    $strPincode = $_POST["txtPincode"];
    $iDeviceSpeed = $_POST["ddlDeviceSpeed"];
    
    $objGsm-&gt;Clear();
    $objGsm-&gt;LogFile = $_POST["txtResult"];
    $objGsm-&gt;Open($strName, $strPincode, $iDeviceSpeed);
    
    if ($objGsm-&gt;LastError != 0)
    {
      $strResult = $objGsm-&gt;LastError . ": " . $objGsm-&gt;GetErrorDescription($objGsm-&gt;LastError);
      $objGsm-&gt;Close();
    }
    else
    {
      //Message Settings
      $objSmsMessage-&gt;Clear();
      $objSmsMessage-&gt;ToAddress = $_POST["txtToAddress"];
      $objSmsMessage-&gt;Body = $_POST["txtBody"];
      $objSmsMessage-&gt;BodyFormat = $objSmsConstants-&gt;BODYFORMAT_TEXT;
      
      $iMultipart = (isset($_POST["cbxMultipart"])) ?
        $objSmsConstants-&gt;MULTIPART_OK : $objSmsConstants-&gt;MULTIPART_TRUNCATE;
      
      if (isset($_POST["cbxFlash"]))
      {
        $objSmsMessage-&gt;DataCoding |= $objSmsConstants-&gt;DATACODING_FLASH;
      }
      
      //Send the message !
      $obj = $objSmsMessage;
      $objGsm-&gt;SendSms($obj, $iMultipart, 10000);
      $objSmsMessage = $obj;
      $strMessageReference = $objSmsMessage-&gt;Reference;
      
      $strResult = $objGsm-&gt;LastError . ": " . $objGsm-&gt;GetErrorDescription($objGsm-&gt;LastError);
      
      $objGsm-&gt;Close();
    }
  }
