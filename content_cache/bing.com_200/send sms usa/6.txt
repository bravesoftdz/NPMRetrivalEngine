      <div class="docs-prose"> 
       <h1 class="page-title "> Twilio SMS and MMS Java Quickstart </h1> 
       <div class="block-rich_text "> 
        <div class="rich-text">
         <p>These Java Quickstarts will take you through all the steps you'll need to add SMS and MMS capabilities to your Java web application. We'll walk&nbsp;you through many common SMS and MMS scenarios, such as sending and receiving messages, threading conversations, and sending messages during voice and video calls.</p> 
         <p>Code and examples are included for usage of the&nbsp;<a href="/docs/libraries/java">Twilio Java Helper Library</a>&nbsp;as well as endpoints of the Twilio REST API.</p>
        </div> 
       </div> 
       <div class="block-markdown "> 
        <div> 
         <h2 id="overview"><a class="toclink" href="#overview">Overview</a></h2> 
         <p><a href="/api/sms/">Twilio Messaging</a> allows you to:</p> 
         <ol> 
          <li>Send SMS and MMS messages using Twilio's REST API</li> 
          <li>Receive SMS and MMS messages to your Twilio phone numbers and reply back with SMS and MMS messages</li> 
          <li>Track SMS conversations</li> 
          <li>Send SMS messages during phone calls</li> 
         </ol> 
        </div> 
       </div> 
       <div class="block-rich_text "> 
        <div class="rich-text">
         <div class="responsive-object" style="padding-bottom: 56.25%;"> 
          <iframe allowfullscreen frameborder="0" height="270" src="https://www.youtube.com/embed/jdlWQtgsNSU?feature=oembed" width="480"></iframe> 
         </div> 
        </div> 
       </div> 
       <div class="block-markdown "> 
        <div> 
         <h4 id="send-sms-via-rest"><a class="toclink" href="#send-sms-via-rest"></a><a href="/docs/quickstart/java/sms/sending-via-rest">Send SMS and MMS messages via the REST API</a></h4> 
         <p>You can also send SMS and MMS messages using Twilio's <a href="/docs/api/rest/sending-messages">REST API</a>. To send a message, make an HTTP POST to Twilio with the body of the message and the phone number you want to send it to. Messages must be sent from Twilio SMS-enabled phone numbers due to the architecture of the global SMS network.</p> 
         <h4 id="receive-and-replay-sms"><a class="toclink" href="#receive-and-replay-sms"></a><a href="/docs/quickstart/java/sms/hello-monkey">Receive and reply to SMS and MMS messages</a></h4> 
         <p>When you buy a SMS-enabled Twilio phone number, you can associate that phone number with an SMS URL. When someone sends a text message to that phone number, Twilio makes an HTTP request to your URL with the body of the message and the sender's phone number. You can then respond to the SMS by returning a reply message in the HTTP response to Twilio.</p> 
         <h4 id="track-conversations"><a class="toclink" href="#track-conversations"></a><a href="/docs/quickstart/java/sms/tracking-conversations">Track SMS conversations</a></h4> 
         <p>Using standard HTTP cookies, you can share state across multiple messages allowing you to treat separate messages between any pair of phone numbers as a conversation, and store data about the conversation in the cookie for future reference.</p> 
         <h4 id="send-sms-via-calls"><a class="toclink" href="#send-sms-via-calls"></a><a href="/docs/quickstart/java/sms/sending-via-twiml">Send SMS messages during phone calls</a></h4> 
         <p>You can also use the TwiML <a href="/docs/api/twiml/sms"><code>&lt;Sms&gt;</code></a> verb inside your existing Twilio voice calls to send a message during a phone conversation.</p> 
         <h3 id="faq"><a class="toclink" href="#faq">Frequently asked questions</a></h3> 
         <p>For specific details on what you can and can't accomplish using Twilio SMS please refer to the <a href="/faq/sms">SMS FAQ</a></p> 
         <h4 id="before-you-start"><a class="toclink" href="#before-you-start">Before you start</a></h4> 
         <p>You'll need a Java development environment with a Web server capable of running Java servlets and the <code>twilio-java</code> helper library. Please see our post on <a href="/docs/quickstart/java/devenvironment">setting up your environment</a> if you need help installing those programs.</p> 
         <p>With that out of the way, let's go on to the next step.</p> 
         <p><br> <a href="/docs/quickstart/java/sms/sending-via-rest"> <span>Next: Send SMS via the Twilio API »</span> </a></p> 
        </div>
