       <h1 class="page-title "> Java Quickstart: Send SMS and MMS Text Messages </h1> 
       <div class="block-markdown "> 
        <div> 
         <p>In this Java SMS Quickstart, we're going to find our friend! We'll use our Java skills to ask our friends where our buddy Wallace can be found by sending outgoing SMS messages. </p> 
         <p>To send an outgoing SMS message with Twilio, perform an <code>HTTP POST</code> to the <a href="/docs/api/rest/message">Messages Resource URI</a> in Twilio's REST API. To make things even simpler, we can also use the <a href="/docs/libraries/java">Twilio Java Helper Library</a> to make REST requests.</p> 
         <p><em>If you are on a Twilio Trial account, your outgoing SMS messages are limited to phone numbers that you have verified with Twilio. Phone numbers can be verified via your <a href="/console/phone-numbers/verified">Twilio Console's Verified Caller IDs</a>. Note the 'From' parameter will need to be a phone number you have <a href="/console/phone-numbers/incoming">purchased from Twilio</a>.</em></p> 
        </div> 
       </div>

         <h2 id="send-mms-via-rest"><a class="toclink" href="#send-mms-via-rest">Send MMS Messages in Java</a></h2> 
         <p>It's also easy to send an outgoing MMS using Twilio. To send an MMS, you also make an HTTP POST request to the <a href="/docs/api/rest/sending-messages">Messages resource</a> but this time specify one or more <code>MediaUrl</code> parameters.</p> 
         <div class="alert">
           MMS messages can only be sent and received by numbers having MMS capability. You can 
          <a href="/user/account/phone-numbers/search">check the capabilities</a> of numbers in the account portal or query the 
          <a href="/docs/api/rest/available-phone-numbers">Available Phone Numbers</a> resource to search for Twilio numbers that are MMS enabled.
