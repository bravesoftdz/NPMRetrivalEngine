             <h2>Introduction</h2> 
             <p>This article explains how to add the capability of sending text (SMS) messages from a desktop application.</p> 
             <h2>Background</h2> 
             <p>The article focuses on an implementation using MFC / C++. While looking for a reliable and cheap solution for sending SMS messages programmatically, I came&nbsp;across a company named <a href="http://cardboardfish.com/" target="_blank" title="CardBoardFish">CardBoardFish</a> which covers 150 countries and provides an easy to use, yet powerful SDK for interfacing from any web site, mobile phone, or desktop application, covering most platforms and development environments. Unfortunately, among the code samples in their web site, there aren't any C++ samples, so I decided to develop my own C++ implementation.</p> 
             <h2>Sending SMS Messages Programmatically</h2> 
             <p>Most applications and web sites used to send SMS messages as part of their scope or among other functionalities (i.e., sending alerts, etc.) use one of the following methods:</p> 
             <ul> 
              <li><strong>HTTP Web Service - </strong>&nbsp;requires using HTTP "GET" method to send a given Web Service a command, using an API, which contains the credentials, parameters, and the text for this message.</li> 
              <li><strong>Email to SMS </strong>&nbsp;- uses the SMTP protocol to allow sending an email in a unique format, which encodes all required parameters (credentials, sender, receiver, etc.) as part of an email.</li> 
             </ul> 
             <p>This article focuses on the first method, using a Web Service.</p> 
             <h2>The API</h2> 
             <p>The following table lists all parameters that can (or should) be sent to the Web Service:</p> 
             <p><a href="/KB/cpp/387567/API.jpg" title="Click to enlarge"><img border="0" height="679" src="/KB/cpp/387567/API_small.jpg" width="640"></a></p> 
             <h2>Using the code</h2> 
             <p>The code in this article was developed in MFC / C++ using Visual Studio 2010 Ultimate.&nbsp;I also used <a href="http://beta.codeproject.com/Members/chen-shi" title="Cheng Shi's page">Cheng Shi</a>'s HTTPClient (thanks Cheng!).</p> 
             <p>In order&nbsp;to use the code for your own application, it is advised to read the specifications for the SDK named <a href="http://www.cardboardfish.com/support/bin/view/Main/HTTPSMSSpecificationDocument" target="_blank">HTTPSMS</a>.&nbsp;Secondly, you need to open an account and obtain your user name and password, which can be hardcoded in the source code, or entered during runtime.</p> 
             <h2>The SendSMS application &nbsp;</h2> 
             <p><img align="baseline" alt="SendSMS Screen" border="0" height="366" hspace="0" src="/KB/cpp/387567/SendSMS.jpg" width="498"></p> 
             <p>The main functionality of our application is obviously sending an SMS, which is done in the following function:</p> 
             <pre lang="C++"><span class="code-comment">//</span><span class="code-comment"> SendSms  - by Michael Haephrati</span>
BOOL SendSms(CString From, CString CountryCode, CString To,CString Message,CString *Status)
    <span class="code-comment">//</span><span class="code-comment"> From - the ID or number you are sending from. This is what will appear at the recipient's cellphone. </span>
