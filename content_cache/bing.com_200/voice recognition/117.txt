                <p>While we have several options, Tropo can only listen for one language at a time, so it’s important to tell it which language to expect. Here's an example using both <code><b>voice</b></code> and <code><b>recognizer</b></code> with Argentinian Spanish; we included the same example in US English for comparison:</p> 
                <div class="tabs"> 
                 <ul> 
                  <li><a href="#tabs-1">JavaScript</a></li> 
                  <li><a href="#tabs-2">Ruby</a></li> 
                  <li><a href="#tabs-3">PHP</a></li> 
                  <li><a href="#tabs-4">Python</a></li>

                 <div id="tabs-1"> 
                  <pre class="brush: javascript;gutter:true;">
ask("What's your four digit pin?", {
    choices:"[4 DIGITS]",
    mode:"speech",
    recognizer:"en-us",
    onChoice: function(event) {
        say("Thank you");
    }
});
	
ask("¿Cuál es su número de identificación?", {
    voice:"Angelica",
