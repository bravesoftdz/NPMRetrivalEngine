                    <span class="rendered_qtext"><p class="qtext_para">My biased list for October 2016</p><p class="qtext_para"><b>Online short utterance</b></p><p class="qtext_para">1) <span class="qlink_container"><a href="https://cloud.google.com/speech/" rel="noopener nofollow" target="_blank" onclick="return Q.openUrl(this, 9664042);" class="external_link" data-qt-tooltip="google.com">Google Speech API</a></span> - best speech technology, recently announced to be available for commercial use. Currently in beta status. Google also has separate APIs for Android OS and Javascript API for Chrome.</p><p class="qtext_para">2) <span class="qlink_container"><a href="https://www.microsoft.com/cognitive-services/en-us/speech-api" rel="noopener nofollow" target="_blank" onclick="return Q.openUrl(this, 9664042);" class="external_link" data-qt-tooltip="microsoft.com">Microsoft Cognitive Services - Bing Speech API</a></span> same from Microsoft, many different nice addons like voice authentication</p><p class="qtext_para">3) <span class="qlink_container"><a href="https://api.ai" rel="noopener nofollow" target="_blank" onclick="return Q.openUrl(this, 9664042);" class="external_link" data-qt-tooltip="api.ai">API.AI</a></span> - analyses intent, not simply recognizes speech. Useful to build command applications, belongs to Google.</p><p class="qtext_para">There are also offerings from Amazon, Facebook and many others.</p><p class="qtext_para"><b>Online large files</b></p><p class="qtext_para">4) <span class="qlink_container"><a href="https://speechmatics.com" rel="noopener nofollow" target="_blank" onclick="return Q.openUrl(this, 9664042);" class="external_link" data-qt-tooltip="speechmatics.com">Speechmatics</a></span> - large vocabulary transcription in the cloud, US and UK English, high accuracy.</p><p class="qtext_para">5) <span class="qlink_container"><a href="http://www.vocapia.com/speech-to-text-api.html" rel="noopener nofollow" target="_blank" onclick="return Q.openUrl(this, 9664042);" class="external_link" data-qt-tooltip="vocapia.com">Vocapia Speech to Text API</a></span> - not very user friendly, but a good technology</p><p class="qtext_para"><b>Offline Proprietary</b></p><p class="qtext_para">6) <span class="qlink_container"><a href="http://www.iflytek.com/en/audioengine/index.html" rel="noopener nofollow" target="_blank" onclick="return Q.openUrl(this, 9664042);" class="external_link" data-qt-tooltip="iflytek.com">Speech Engine_IFLYTEK CO.,LTD.</a></span> not very well known Chinese company, but it continuously excels in competitions.</p><p class="qtext_para">7) <span class="qlink_container"><a href="https://msdn.microsoft.com/en-us/windows/uwp/input-and-devices/speech-recognition" rel="noopener nofollow" target="_blank" onclick="return Q.openUrl(this, 9664042);" class="external_link" data-qt-tooltip="microsoft.com">UWP Speech recognition</a></span> from Microsoft for Universal Windows Platform</p><p class="qtext_para"><b>Open Source</b></p><p class="qtext_para">8) <span class="qlink_container"><a href="http://cmusphinx.sourceforge.net" rel="noopener nofollow" target="_blank" onclick="return Q.openUrl(this, 9664042);" class="external_link" data-qt-tooltip="sourceforge.net">CMU Sphinx - Speech Recognition Toolkit</a></span> - offline speech recognition, due to low resource requirements can be used on mobile. <span class="qlink_container"><a href="http://www.politepix.com/openears/" rel="noopener nofollow" target="_blank" onclick="return Q.openUrl(this, 9664042);" class="external_link" data-qt-tooltip="politepix.com">OpenEars</a></span> - Pocketsphinx on iOS, there are also APIs for Node.js, Ruby, Java, Android bindings.</p><p class="qtext_para">9) <span class="qlink_container"><a href="http://kaldi.sourceforge.net" rel="noopener nofollow" target="_blank" onclick="return Q.openUrl(this, 9664042);" class="external_link" data-qt-tooltip="sourceforge.net">Kaldi</a></span> - speech recognition toolkit for research. <span class="qlink_container"><a href="https://github.com/UFAL-DSG/cloud-asr" rel="noopener nofollow" target="_blank" onclick="return Q.openUrl(this, 9664042);" class="external_link" data-qt-tooltip="github.com">UFAL-DSG/cloud-asr</a></span> - Kaldi-based cloud platform, <span class="qlink_container"><a href="https://github.com/alumae/kaldi-gstreamer-server" rel="noopener nofollow" target="_blank" onclick="return Q.openUrl(this, 9664042);" class="external_link" data-qt-tooltip="github.com">alumae/kaldi-gstreamer-server</a></span> - another kaldi-based cloud platform. <span class="qlink_container"><a href="http://keenresearch.com/kaldi-ios-framework" rel="noopener nofollow" target="_blank" onclick="return Q.openUrl(this, 9664042);" class="external_link" data-qt-tooltip="keenresearch.com">iOS Speech Recognition</a></span> - kaldi adopted for offline recognition on iOS from Keen Research.</p></span>
