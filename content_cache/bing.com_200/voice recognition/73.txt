                    <span class="rendered_qtext"><p class="qtext_para">While Im not aware of any off the shelf services which do this, after a few minutes research it looks entirely possible.</p><p class="qtext_para">Depending on how you want to do interface with this gender recognition, you could allow for recording to happen from the devices microphone right from inside the webpage, then that data to be sent to a server as a POST request. The response would probably contain the predicted gender and the certainty, along with any other information you might want.</p><p class="qtext_para">As for the actual implementation, if youre doing it through nodeJS, you would probably need to run a separate program/service on the server machine and interface with that from nodeJS. Check out <span class="qlink_container"><a href="https://github.com/primaryobjects/voice-gender" rel="noopener nofollow" target="_blank" onclick="return Q.openUrl(this, 47696860);" class="external_link" data-qt-tooltip="github.com">primaryobjects/voice-gender</a></span> this repo as an example.</p><p class="qtext_para">More information can be found here: <span class="qlink_container"><a href="http://www.primaryobjects.com/2016/06/22/identifying-the-gender-of-a-voice-using-machine-learning/" rel="noopener nofollow" target="_blank" onclick="return Q.openUrl(this, 47696860);" class="external_link" data-qt-tooltip="primaryobjects.com">Identifying the Gender of a Voice using Machine Learning</a></span>, which goes through a few details of the challenges youd face, and shows a <span class="qlink_container"><a href="http://voice-primaryobjects.rhcloud.com/" rel="noopener nofollow" target="_blank" onclick="return Q.openUrl(this, 47696860);" class="external_link" data-qt-tooltip="rhcloud.com">working demo</a></span>.</p></span>
