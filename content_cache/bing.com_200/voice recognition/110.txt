         <p>Nesta página falaremos um pouco sobre a Java Speech API. Application Program Interface(API) criada com intuito de prover síntese e reconhecimento de voz nas aplicações Java.<br> Será apresentado alguns passo de como instalar o IBM Via Voice, assim como o ibmjs.jar que contém a Java Speech API que conversa com o IBM ViaVoice.<br> Caso vcs não tenham alguns dos arquivos abaixo, me digam que eu posso providenciar, a versão que tenho do IBM ViaVoice é antiga(5.0), mas ainda funciona.</p> 
         <h2 id="toc0"><span>Arquivos para Instalação</span></h2> 
         <p><a href="http://www.natalnet.br/~aquiles/ibmviavoice/">http://www.natalnet.br/~aquiles/ibmviavoice/</a></p> 
         <h2 id="toc1"><span>Instalação</span></h2> 
         <h3 id="toc2"><span>Passo 1 - Instalando o TTS Runtime</span></h3> 
         <p>Primeiro se deve instalar o Text to Speech(TTS) Runtime. Os arquivos de runtime variam de acordo com linguagem utilizada. Sendo assim temos um runtime para cada língua:</p> 
         <p>Ex:<br> Português : <a href="http://www.natalnet.br/~aquiles/ibmviavoice/tts5bp.zip">tts5bp</a><br> Inglês: <a href="http://www.natalnet.br/~aquiles/ibmviavoice/tts5us.zip">tts5us</a><br> Escolha um, ou instale os dois de acordo com suas necessidades.</p> 
         <h3 id="toc3"><span>Passo 2 - Instalando o TTS SDK</span></h3> 
         <p>Instale o arquivo que esta no .zip:<a href="http://www.natalnet.br/~aquiles/ibmviavoice/tts5sdk.zip">tts5sdk.zip</a>, clicando em setup.exe</p> 
         <h3 id="toc4"><span>Passo 3 - Instalando VVCC Runtime</span></h3> 
         <p>Instale o IBM ViaVoice Command &amp; Control Runtime 7.0, no .zip <a href="http://www.natalnet.br/~aquiles/ibmviavoice/vvrtcbr7.zip">vvrtcbr7</a>, execute setup.exe</p> 
         <h3 id="toc5"><span>Passo 4 -Instalando o IBM ViaVoice SDK</span></h3> 
         <p>Execute o arquivo <a href="http://www.natalnet.br/~aquiles/ibmviavoice/vvsdk15.exe">vvsdk15.exe</a></p> 
         <h3 id="toc6"><span>Passo 5 -Instalando o SDK com Java</span></h3> 
         <p>Instale o .exe da pasta:<br> 03 - ViaVoice SDK, Java Technology Edition V1.0<br> <a href="http://www.natalnet.br/~aquiles/ibmviavoice/ibmjs100r13a.exe">ibmjs100r13a.exe</a></p> 
         <p>Depois siga as instruções contidas no readme.html que se encontra no diretório onde foi instalado o ibmjs. Normalmente ele é instalado em c:/ibmjs</p> 
         <p>Suppose you have unpacked the installation package to c:\ibmjs. You should<br> - modify your CLASSPATH variable to include c:\ibmjs\lib\ibmjs.jar, and<br> - modify your PATH variable to include the c:\ibmjs\lib directory<br> - from c:\ibmjs, execute install.bat to register the IBM engines with the system.</p> 
         <h3 id="toc7"><span>Passo 6 - Configurando o Projeto no Eclipse</span></h3> 
         <p>Baixe a pasta <a href="http://aquilesburlamaqui.wdfiles.com/local--files/javaspeech/JavaSpeech.zip">JavaSpeech.zip</a> e coloque ela como projeto no eclipse.<br> La vc encontrará o ibmjs.jar associado ao projeto como uma biblioteca.<br> Encontre a classe com.ibm.speech.util.Install.class dentro do ibmjs.jar e e o execute.<br> Depois disso reinicie o computador.</p> 
         <h3 id="toc8"><span>Passo 7 - Testando o Sintetizador.java</span></h3> 
         <p>Execute a aplicação Sintetizador.java<br> Vc deverá escutar o computador falando: Testando Sintetizador<br> Caso vc veja o seguinte string ao invéz do som:<br> &lt;JSML&gt; &lt;BREAK MSECS="300"&gt;&lt;PROS PITCH="80" RANGE="50" RATE="150"&gt;Testando o Sintetizador.&lt;/PROS&gt;&lt;/JSML&gt;<br> Algo deu errado, veja se vc não esqueceu de executar algum dos passos acima.</p> 
         <h3 id="toc9"><span>Passo 8 - Treinando o IBM ViaVoice para o reconhecimento</span></h3> 
         <p>Vá no painel de controle -&gt; Opções do IBM Via Voice<br> Clique em Ok, re-selecione o padrão de voz Português, ate que ele permita vc APLICAR.<br> Quando vc apertar no botão APLICAR, ele irá começar o procedimento de analisar sua voz.</p> 
         <h3 id="toc10"><span>Passo 9 - Corrigindo o PATH para o reconhecimento de voz</span></h3> 
         <p>Adiciona na variável de ambiente PATH o seguinte diretório:<br> C:\Arquivos de programas\ViaVoice\BIN</p> 
         <h3 id="toc11"><span>Passo 10 - Testando o Reconhecedo.java</span></h3> 
         <p>Para testar o Reconhecedor, vc precisar estar conectado a internet, pois a gramatica utilizada no teste esta no seguinte endereço:<a href="http://aquilesburlamaqui.wdfiles.com/local--files/javaspeech/gramatica.gram">gramatica.gram</a><br> Mas caso vc queira testar localmente, vc pode simplesmente alterar o endereço onde a gramática se encontra.</p>
