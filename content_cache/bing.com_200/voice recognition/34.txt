         <p>Hello and welcome to another tutorial on Java, In this tutorial we’ll be creating a Voice command application using <strong>Java</strong> and <strong>Sphinx4 Speech Recognition Library</strong> for Java.</p> 
         <p>If you are new to this Voice Command term, there are many apps that serve as an example in reality. If you are an Android user, you must have used&nbsp;the&nbsp;<strong>Google App</strong> where you speak <strong>“Ok Google”</strong> and it listens to your command and if you say something like <strong>“open google”,&nbsp;</strong>it’ll automatically launch Chrome and open Google.com on it.</p> 
         <p><span id="more-722"></span></p> 
         <p>Now when you speak into your mic, the computer might not be able to understand what is it that you are saying so we’ll be providing our computer with the ability to recognize the words that we say and then covert them into a form that the computer is able to understand hence basis of the term <strong>Speech Recognition</strong>. You might be wondering how in the world are we going to do that? Well we don’t have to worry about anything because we&nbsp;have &nbsp;been blessed with a library called&nbsp;<strong>Sphinx4</strong>&nbsp;which does all the complex work for us hence we only have to call certain methods in order to create our Voice Command app.</p> 
         <h2>Approach</h2> 
         <p>So what is it that our app is going to do?</p> 
         <ol> 
          <li>We will speak a command like “open terminal” in our mic</li> 
          <li>Sphinx4 detects and recognizes the words that we speak</li> 
          <li>Sphinx4 outputs the recognized words</li> 
          <li>We compare the words to our list of commands</li> 
          <li>If a command exists, it’ll execute a certain task</li> 
          <li>Wait for another command and repeat step 2</li> 
         </ol> 
         <p>Here we have our basic approach on creating our Voice command Application.</p> 
         <h2>Requirements</h2> 
         <p>For this app, you’ll require the following:</p> 
         <ol> 
          <li><a href="http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html">Java 8</a></li> 
          <li><a href="http://cmusphinx.sourceforge.net/wiki/download">Sphinx4 </a>(Download the latest Alpha 5 Version)<br> Goto&nbsp;https://oss.sonatype.org/#nexus-search and download the Alpha 5&nbsp;<strong>sphinx-core.jar </strong>and&nbsp;<strong>sphinx4-data.jar&nbsp;</strong>files.</li> 
          <li><a href="https://netbeans.org/downloads/">Netbeans IDE</a></li> 
          <li>A good quality Microphone.</li> 
         </ol> 
         <h2>About Models</h2> 
         <p>There are basically three models required for speech recognition in Sphinx4:</p> 
         <ol> 
          <li>Acoustic Model</li> 
          <li>Phonetic Dictionary (File ends with&nbsp;<strong>.dict&nbsp;</strong>extension)</li> 
          <li>Language Model (File ends with&nbsp;<strong>.lm</strong> extension)</li> 
         </ol> 
         <p>The <strong>sphinx4-data.jar</strong>&nbsp;comes with the English version of Acoustic Model as Default hence we will be using that, if you are using other language then you’ll have to download it from <a href="https://sourceforge.net/projects/cmusphinx/files/Acoustic%20and%20Language%20Models/">Here</a>.</p> 
         <p>Since we are creating a Voice Command app so we’ll be creating our own Language Model and the Phonetic Dictionary because our vocabulary will be limited i.e. our commands only. Now lets create our needed files,</p> 
         <h2>Creating Language Model &amp; Dictionary</h2> 
         <p>As said above our vocabulary is limited hence making the model and dict will be a breeze thanks to <strong><a href="http://www.speech.cs.cmu.edu/tools/lmtool-new.html">Sphinx Online Base Generator</a></strong>. But first we have to make a&nbsp;<strong>corpus </strong>(<em>Data using which we will train our Language Model</em>)<strong>&nbsp;</strong>file containing our commands for which we will create our Language Model and Dictionary. For this tutorial I’ll be choosing 4 commands.</p> 
         <ol> 
          <li>open file manager</li> 
          <li>open browser</li> 
          <li>close browser</li> 
          <li>close file manager</li> 
         </ol> 
         <p>Now type these commands in your text file and save it. Then navigate to the <a href="http://www.speech.cs.cmu.edu/tools/lmtool-new.html">Sphinx Online Base Generator</a>, click&nbsp;<strong>Choose File</strong> and select your&nbsp;<strong>corpus text file</strong>. Now in response the site will give you a list of files, for now we are interested in the files ending with&nbsp;<strong>.dict&nbsp;</strong>and&nbsp;<strong>.lm</strong> extension, so download them and place them in your project folder.</p> 
         <h2>Importing Jar Files</h2> 
         <p>We’ll create a new Java Project in NetBeans&nbsp;and then import some jar files for our project because they are required by Sphinx4. So when you have created your project, goto</p> 
         <p><strong>Run &gt; Set Project Configuration &gt; Customize &gt; Libraries &gt; Add JAR/Folder</strong></p> 
         <p>Now select the 2 jar files you downloaded earlier, <strong>sphinx4-core.jar&nbsp;</strong>and <strong>sphinx4-data.jar</strong></p> 
         <p>Press Ok and you are all set, Now lets get to the coding part.</p> 
         <h2>Coding the Application</h2> 
         <p>Now that we are done creating and importing important files, we now have to create a&nbsp;<strong>Configuration&nbsp;</strong>object and pass it to the Recognizer so that it can make use of the required files, Create a new class called&nbsp;<strong>voiceLanucher&nbsp;</strong>in the project which will serve as our main class.</p> 
         <pre class="brush: java; title: ; notranslate" title="">
//Imports
import edu.cmu.sphinx.api.Configuration;
import edu.cmu.sphinx.api.LiveSpeechRecognizer;
import edu.cmu.sphinx.api.SpeechResult;
import java.io.IOException;

/**
 *
 * @author ex094
 */
public class VoiceLauncher {
    public static void main(String[] args) throws IOException {
        // Configuration Object
        Configuration configuration = new Configuration();

        // Set path to the acoustic model.
        configuration.setAcousticModelPath("resource:/edu/cmu/sphinx/models/en-us/en-us");
        // Set path to the dictionary.
        configuration.setDictionaryPath("PATH_TO_YOUR_.DIC_FILE");
        // Set path to the language model.
        configuration.setLanguageModelPath("PATH_TO_YOUR_.LM_FILE");
      }

}
</pre> 
         <p>Replace the&nbsp;<strong>PATH_TO_YOUR_.DIC_FILE&nbsp;</strong>with the .dic file and&nbsp;<strong>PATH_TO_YOUR_.LM_FILE&nbsp;</strong>with the .lm file&nbsp;you downloaded from the&nbsp;<strong>Sphinx Online Base Generator&nbsp;</strong>earlier from the Creating Language Model and Dictionary.</p> 
         <p>The configuration object is now set and we need to pass it to the recognizer. Also we need the recognizer to use our microphone as a source of input, Gladly the latest (Alpha 5) makes it really easy. We just have to create a&nbsp;<strong>LiveSpeechRecognizer&nbsp;</strong>object, pass in the configuration and call the&nbsp;<strong>startRecognition&nbsp;</strong>method.</p> 
         <pre class="brush: java; title: ; notranslate" title="">
//Imports
import edu.cmu.sphinx.api.Configuration;
import edu.cmu.sphinx.api.LiveSpeechRecognizer;
import edu.cmu.sphinx.api.SpeechResult;
import java.io.IOException;

/**
 *
 * @author ex094
 */
public class VoiceLauncher {
    public static void main(String[] args) throws IOException {
        //Configuration Object
        Configuration configuration = new Configuration();

        // Set path to the acoustic model.
        configuration.setAcousticModelPath("resource:/edu/cmu/sphinx/models/en-us/en-us");
        // Set path to the dictionary.
        configuration.setDictionaryPath("PATH_TO_YOUR_.DIC_FILE");
        // Set path to the language model.
        configuration.setLanguageModelPath("PATH_TO_YOUR_.LM_FILE");

        //Recognizer Object, Pass the Configuration object
        LiveSpeechRecognizer recognize = new LiveSpeechRecognizer(configuration);

        //Start Recognition Process (The bool parameter clears the previous cache if true)
        recognize.startRecognition(true);
     }

}
</pre> 
         <p>Now that the recognition process has started, the recognizer will take your speech when ever you speak into the mic and then processes. For the voice command app we definitely need to check that what type of command is the user giving hence we need the recognizer to display the result that what command has it recognized from the speech.</p> 
         <p>For that we will use the&nbsp;<strong>getHypothesis()</strong> method from the <strong>SpeechResult</strong>&nbsp;object, using a while loop we will be able to get all the recognized speech that the user will speak.</p> 
         <pre class="brush: java; title: ; notranslate" title="">
//Imports
import edu.cmu.sphinx.api.Configuration;
import edu.cmu.sphinx.api.LiveSpeechRecognizer;
import edu.cmu.sphinx.api.SpeechResult;
import java.io.IOException;

/**
 *
 * @author ex094
 */
public class VoiceLauncher {
    public static void main(String[] args) throws IOException {
        //Configuration Object
        Configuration configuration = new Configuration();

        // Set path to the acoustic model.
        configuration.setAcousticModelPath("resource:/edu/cmu/sphinx/models/en-us/en-us");
        // Set path to the dictionary.
        configuration.setDictionaryPath("PATH_TO_YOUR_.DIC_FILE");
        // Set path to the language model.
        configuration.setLanguageModelPath("PATH_TO_YOUR_.LM_FILE");

        //Recognizer object, Pass the Configuration object
        LiveSpeechRecognizer recognize = new LiveSpeechRecognizer(configuration);

        //Start Recognition Process (The bool parameter clears the previous cache if true)
        recognize.startRecognition(true);

        //Create SpeechResult Object
        SpeechResult result;

        //Checking if recognizer has recognized the speech
        while ((result = recognize.getResult()) != null) {
            //Get the recognize speech
            String command = result.getHypothesis();
        }
    }
}
</pre> 
         <p>The command variable will store the recognized speech from the user (The command that you speak) in string format hence we can compare whether the recognized command matches any from our list of commands and then execute the command. We will be using if conditions but you can do it using switch conditional.</p> 
         <pre class="brush: java; title: ; notranslate" title="">
//Imports
import edu.cmu.sphinx.api.Configuration;
import edu.cmu.sphinx.api.LiveSpeechRecognizer;
import edu.cmu.sphinx.api.SpeechResult;
import java.io.IOException;

/**
 *
 * @author ex094
 */
public class VoiceLauncher {
    public static void main(String[] args) throws IOException {
        //Configuration Object
        Configuration configuration = new Configuration();

        // Set path to the acoustic model.
        configuration.setAcousticModelPath("resource:/edu/cmu/sphinx/models/en-us/en-us");
        // Set path to the dictionary.
        configuration.setDictionaryPath("/home/ex094/Desktop/4220.dic");
        // Set path to the language model.
        configuration.setLanguageModelPath("/home/ex094/Desktop/4220.lm");

        //Recognizer object, Pass the Configuration object
        LiveSpeechRecognizer recognize = new LiveSpeechRecognizer(configuration);

        //Start Recognition Process (The bool parameter clears the previous cache if true)
        recognize.startRecognition(true);

        //Creating SpeechResult object
        SpeechResult result;

        //Check if recognizer recognized the speech
        while ((result = recognize.getResult()) != null) {

            //Get the recognized speech
            String command = result.getHypothesis();
            //Match recognized speech with our commands
            if(command.equalsIgnoreCase("open file manager")) {
                System.out.println("File Manager Opened!");
            } else if (command.equalsIgnoreCase("close file manager")) {
                System.out.println("File Manager Closed!");
            } else if (command.equalsIgnoreCase("open browser")) {
                System.out.println("Browser Opened!");
            } else if (command.equalsIgnoreCase("close browser")) {
                System.out.println("Browser Closed!");
            }
        }
    }
}

</pre> 
         <p>Since the recognized speech is stored in our command variable, we can now compare using String comparison easily. Now run the code and speak into your mic one of the 4 commands If you speak <strong>“open filemanager”</strong> it should print&nbsp;<strong>“File Manager Opened”.</strong></p> 
         <p>After your testing is complete, it’s time to add real commands like the one’s that’ll open the file manager when you speak the “open file manager” command. We will store the command in a variable and then use the&nbsp;<strong>Process&nbsp;</strong>library to execute the commands.</p> 
         <pre class="brush: java; title: ; notranslate" title="">
//Imports
import edu.cmu.sphinx.api.Configuration;
import edu.cmu.sphinx.api.LiveSpeechRecognizer;
import edu.cmu.sphinx.api.SpeechResult;
import java.io.IOException;

/**
 *
 * @author ex094
 */
public class VoiceLauncher {
    public static void main(String[] args) throws IOException {
        //Configuration Object
        Configuration configuration = new Configuration();

        // Set path to the acoustic model.
        configuration.setAcousticModelPath("resource:/edu/cmu/sphinx/models/en-us/en-us");
        // Set path to the dictionary.
        configuration.setDictionaryPath("/home/ex094/Desktop/4220.dic");
        // Set path to the language model.
        configuration.setLanguageModelPath("/home/ex094/Desktop/4220.lm");

        //Recognizer object, Pass the Configuration object
        LiveSpeechRecognizer recognize = new LiveSpeechRecognizer(configuration);

        //Start Recognition Process (The bool parameter clears the previous cache if true)
        recognize.startRecognition(true);

        //Creating SpeechResult object
        SpeechResult result;

        //Check if recognizer recognized the speech
        while ((result = recognize.getResult()) != null) {

            //Get the recognized speech
            String command = result.getHypothesis();
            String work = null;
            Process p;

            if(command.equalsIgnoreCase("open file manager")) {
                work = "nautilus";
            } else if (command.equalsIgnoreCase("close file manager")) {
                work = "pkill nautilus";
            } else if (command.equalsIgnoreCase("open browser")) {
                work = "google-chrome";
            } else if (command.equalsIgnoreCase("close browser")) {
                work = "pkill google-chrome";
            }
            //In case command recognized is none of the above hence work might be null
            if(work != null) {
                //Execute the command
                p = Runtime.getRuntime().exec(work);
            }
        }
    }
}
</pre> 
         <p>Run this code and speak the “<strong>open browser</strong>” command into your mic, it should open the file manager, test all the other commands as well.</p> 
         <h2>Adding more commands</h2> 
         <p>In order to add more commands, just add your new commands in your previous&nbsp;<strong>corpus.txt</strong> file and then repeat the steps from the&nbsp;<strong>Creating Language Model and&nbsp;</strong><b>Dictionary.</b></p> 
         <h2>If-Else Spaghetti</h2> 
         <p>There might come a time when you’ll have a lot of commands in your program and putting them in if-else would be an absolute mess, so what to do? The best thing would be to load&nbsp;all the commands from corpus text file inside a <strong>HashTable</strong> and map the speech command to it’s respective executable command.&nbsp;I’ll add an updated code in the github repo of this tutorial so in case if you needed it.</p> 
         <p><strong>Github Repo:&nbsp;<a href="https://github.com/Ex094/VoiceCom">A Simple Voice Command App powered by Java and Sphinx4</a></strong></p> 
         <p>That’s it for this tutorial, have fun with your voice command app ?<br> Regards,<br> Ex094</p>

            <p>I think I found a fix, but it’s in ubuntu Linux and I lack the skills to change it to windows. Do you mind trying this and providing a windows version? Thanks!<br> When there are multiple processes using the same process name, you may want to kill one of the processes but not all. Doing a “pkill processname” will kill all processes with that name (no matter how many processes there are).</p> 
            <p>Here’s a bash script that I wrote that allows you specify which process of the processname’s you want to kill.</p> 
            <p>you run it like so: “script.sh processname filtername”</p> 
            <p>This would kill all processname’s that match filtername.</p> 
            <p>Here is script.sh:</p> 
            <p>#!/bin/bash<br> lookup=$1<br> finelookup=$2</p> 
            <p>PIDs=”$(pgrep -l $lookup | tr ‘ ‘ ‘=’ | awk -F= ‘{print $1}’)”</p> 
            <p>for i in ${PIDs} ; do<br> toKill=”$(ps -ef | grep $i | grep $finelookup)”<br> if [ “$toKill” != “” ]; then<br> kill -9 $i<br> fi<br> done</p>

        //Check if recognizer recognized the speech
        while ((result = recognize.getResult()) != null) {
            
            //Get the recognized speech
            //System.out.println(result.getHypothesis());
            String command = result.getHypothesis();
            
            //Starting point for nested command
            if(command.equalsIgnoreCase("open")) {
                System.out.println("Awaiting Subcommand!");
                while ((result = recognize.getResult()) != null) {
                    command = result.getHypothesis();
                    //Nested Commands
                    if(command.equalsIgnoreCase("notepad")) {
                        System.out.println("Opening Notepad!");
                    }
                    else if(command.equalsIgnoreCase("calculator")) {
                        System.out.println("Opening Calculator!");
                    }
                    //Break out of the Nested Command
                    break;
                }
            }
        }
