         <h1 class="page-title"><span class="field field--name-title field--type-string field--label-hidden">Pure Java speech recognition</span> </h1> 
        </div> 
        <div id="block-aadotcom-content" class="block block-system block-system-main-block"> 
         <article data-history-node-id="2195" role="article" about="/java/java-speech-recognition-software-mac" class="node node--type-blog node--promoted node--view-mode-full"> 
          <div class="submitted_by">
           By Alvin Alexander. Last updated: June 3 2016
          </div> 
          <div class="node__content"> 
           <div class="clearfix text-formatted field field--name-body field--type-text-with-summary field--label-hidden field__item">
            <p id="summary">Summary: There is a Pure Java speech recognition project named Sphinx-4, which looks very promising. You can easily run their WebStart demo, and recently I've developed my own Java speech recognition app on top of Sphinx-4 to control my Mac OS X system.</p> 
            <p>So far this looks pretty sweet ... if you're interested in programming with speech recognition, a project named Sphinx-4 provides a Java speech recognizer, i.e., a speech recognition server written entirely in Java.</p> 
            <p>I just ran the ZipCity WebStart demo app they have on their website, and it seems to work as advertised, which is pretty cool. I thought I might have to do something special to turn on the iMac microphone, but when you press the "Speak" button on their Java application, the microphone works just fine.</p> 
            <p>Based on my experience with Dragon Naturally Speaking, which I used for several years, I know that training is an important part of speech recognition software, and I'm just reading about the Sphinx-4 training capability.</p> 
            <p>I'm pretty cranked about this. I currently live in a small, relatively quiet apartment in <a href="http://onemansalaska.com/wasilla">Wasilla, Alaska</a>, and I also work here, and there are a ton of speech recognition things I'd like to try, including a lot of interaction with iTunes, such as:</p> 
            <ul>
             <li><a href="/blog/post/mac-os-x/first-cut-of-my-applescript-alarm-clock-program-speaks-first-th">An iTunes alarm clock</a></li> 
             <li>Playing a radio station</li> 
             <li>Switching from one radio station to another</li> 
             <li>Playing a music album</li> 
             <li>Playing a podcast</li> 
             <li>Playing movies</li> 
            </ul>
            <p>(On my limited budget I know that I'll have to do some things with the iMac remote control, but that's fine, I usually keep it in the kitchen anyway.)</p> 
            <p>I can also envision interactions with browsers and websites, but a lot of what I'd like to do involves iTunes. Unfortunately <a href="/search/node/applescript">AppleScript</a> has to be the glue between Java and iTunes, but I've been thinking about writing an abstraction layer there anyway (unless someone else already has).</p> 
            <p>Here's a short blurb on the Sphinx-4 software:</p> 
            <blockquote> 
             <p>Sphinx-4 is a state-of-the-art speech recognition system written entirely in the Java programming language. It was created via a joint collaboration between the Sphinx group at Carnegie Mellon University, Sun Microsystems Laboratories, Mitsubishi Electric Research Labs (MERL), and Hewlett Packard (HP), with contributions from the University of California at Santa Cruz (UCSC) and the Massachusetts Institute of Technology (MIT).</p> 
            </blockquote> 
            <p>And here's a link to the <a href="http://cmusphinx.sourceforge.net/sphinx4/">Sphinx-4</a> Java speech recognition software website.</p>
