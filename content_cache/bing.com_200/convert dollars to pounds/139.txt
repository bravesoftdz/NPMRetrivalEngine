       <div class="section"> 
        <h1>MS Excel: <span>How to use the DOLLAR Function (WS)</span></h1> 
        <p>This Excel tutorial explains how to use the Excel <strong>DOLLAR function</strong> with syntax and examples.</p> 
       </div> 
       <div class="section"> 
        <h2>Description</h2> 
        <p>The Microsoft Excel DOLLAR function converts a number to text, using a currency format. The format used is $#,##0.00_);($#,##0.00).</p> 
        <p>The DOLLAR function is a built-in function in Excel that is categorized as a <em>String/Text Function</em>. It can be used as a worksheet function (WS) in Excel. As a worksheet function, the DOLLAR function can be entered as part of a formula in a cell of a worksheet.</p> 
       </div> 
       <div class="section"> 
        <h2>Syntax</h2> 
        <p>The syntax for the DOLLAR function in Microsoft Excel is:</p> 
        <pre class="notranslate">DOLLAR( number, [decimal_places] )</pre> 
        <h3>Parameters or Arguments</h3> 
        <dl> 
         <dt>
          number
         </dt> 
         <dd>
          The number to convert to text.
         </dd> 
         <dt>
          decimal_places
         </dt> 
         <dd>
          Optional. It is the number of decimal places to display. The number will be rounded accordingly.
         </dd> 
        </dl> 
       </div> 
       <div class="section"> 
        <h2>Returns</h2> 
        <p>The DOLLAR function returns a string/text value.</p> 
       </div> 
       <div class="section"> 
        <h2>Applies To</h2> 
        <ul> 
         <li>Excel 2016, Excel 2013, Excel 2011 for Mac, Excel 2010, Excel 2007, Excel 2003, Excel XP, Excel 2000</li> 
        </ul> 
       </div> 
       <div class="section"> 
        <h2>Type of Function</h2> 
        <ul> 
         <li>Worksheet function (WS)</li> 
        </ul> 
       </div> 
       <div class="section"> 
        <h2>Example (as Worksheet Function)</h2> 
        <p>Let's look at some Excel DOLLAR function examples and explore how to use the DOLLAR function as a worksheet function in Microsoft Excel:</p> 
        <p><img class="img-responsive" src="images/dollar001.png" alt="Microsoft Excel"></p> 
        <p>Based on the Excel spreadsheet above, the following DOLLAR examples would return:</p> 
        <pre class="notranslate">=DOLLAR(A1, 0)
<em>Result:</em> "$120"

=DOLLAR(A1, 2)
<em>Result:</em> "$120.10"

=DOLLAR(A2, 1)
<em>Result:</em> "$52.1"

=DOLLAR(1345, 2)
<em>Result:</em> "$1,345.00"

=DOLLAR(A1+A2, 2)
<em>Result:</em> "$172.10"</pre> 
       </div> 
       <div class="section"> 
        <h2>Frequently Asked Questions</h2> 
        <p><span class="question">Question:</span> In Microsoft Excel, my question concerns formatting numbers in a particular cell. For example, the cell says:</p> 
        <pre class="notranslate">="Gas price: $" &amp; CHOOSE(gas.deck, B1, B2, B3) &amp; "per mmbtu"</pre> 
        <p>But returns $3 when the cell is formatted to two decimals. If the price were $3.25, the correct price would show. Any help would be greatly appreciated.</p> 
        <p><span class="answer">Answer:</span> Even though your cell is formatted with a number format, your formula is returning text not a numeric value so the number format will not be applied. You will need to apply the format to the number inside of the formula.</p> 
        <p>You could try using the DOLLAR function to apply the format as follows: (you will need to remove your $ sign because the DOLLAR function will insert one automatically)</p> 
        <pre class="notranslate">="Gas price: " &amp; DOLLAR(CHOOSE(gas.deck, B1, B2, B3)) &amp; "per mmbtu"</pre> 
        <p>This formula should now return something like:</p> 
        <pre class="notranslate">Gas price: $3.25per mmbtu</pre> 
       </div>
