   <div class="post"> 
    <span class="Quote">The AngularJS <span style="font-style:italic;color:#000;">currency</span> Filter works very much like the <a href="http://www.encodedna.com/angularjs/tutorial/angularjs-filters.htm#angular-number-filter" target="_blank" class="quotelink">number filter</a>. However, it will format the text (a number) and displays the value prefixed with the “$” (dollar) symbol. The “$” is the default symbol, when no other <a href="https://docs.angularjs.org/api/ng/filter/currency" target="_blank" class="quotelink" rel="nofollow">symbol</a> is defined.</span> 
    <div class="top_ad" id="top_ad" style="clear:both;float:left;width:100%;margin:10px 0;">

    <p style="color:#FF6600;text-align:center;">-----------------</p> 
    <p>If you are new to AngularJS, then I recommend you to read my first article that I have written for beginners.</p> 
    <p><a href="http://www.encodedna.com/angularjs/tutorial/my-first-angularjs-application.htm" target="_blank" class="contact">AngularJS Tutorial for Beginners – My First AngularJS App “Hello World”</a></p> 
    <p style="color:#FF6600;text-align:center;">-----------------</p> 
    <p>There are <span style="font-style:italic;color:#000;">two</span> ways (or methods) you can use a currency filter to format a string in an AngularJS app. In the <span style="font-style:italic;color:#000;">first</span> method, you can apply the filter to an HTML <span style="font-style:italic;color:#000;">template</span> (either using an <a href="http://www.encodedna.com/angularjs/tutorial/angularjs-expressions-with-examples.htm" target="_blank" class="contact">expression</a> or in a <span style="font-style:italic;color:#000;">directive</span>). The <span style="font-style:italic;color:#000;">second</span> method is to use the filter in a <span style="font-style:italic;color:#000;">Controller</span> using JavaScript. This article covers both the methods one by one.</p> 
    <h3 style="font-weight:bold;">Syntax (Using an Expression)</h3> 
    <p class="highlight" style="background:#FFF;">{{ currency_expression | currency : symbol : fractionSize}}</p> 
    <h3 style="font-weight:bold;">Syntax (Using a Controller in JavaScript)</h3> 
    <p class="highlight" style="background:#FFF;">$filter('currency')(amount, symbol, fractionSize)</p>
    <br> 
    <p><span class="related">Also Read:</span> <a href="http://www.encodedna.com/angularjs/tutorial/angularjs-currency-filter-with-rupee.htm" target="_blank" class="contact">AngularJS “currency” Filter Rupee or Euro</a></p>
    <br> 
    <h2><span style="color:Red;">*</span> Using “currency” Filter with in an Expression</h2> 
    <p>In the first method, I’ll add the “currency” filter with a variable in an <span style="font-style:italic;color:#000;">Expression</span>. I have defined the variable (bid) using an <span style="font-style:italic;color:#000;">ng-model</span> directive.</p> 
    <div class="frmname">
     The Markup
    </div> 
    <div class="code">
     <pre class="pre">
&lt;!DOCTYPE html&gt;
&amp;lthtml&gt;
&amp;lthead&gt;
    &amp;ltscript src="https://ajax.googleapis.com/ajax/libs/angularjs/1.3.8/angular.min.js"&gt;&lt;/script&gt;
&lt;/head&gt;
&amp;ltbody style="font:15px Verdana;"&gt;
    &amp;ltdiv ng-app&gt;
        &amp;ltp&gt;
            &amp;ltlabel&gt;Enter a number&lt;/label&gt;
            &amp;ltinput type="text" ng-model="bid" /&gt;
        &lt;/p&gt;

        &amp;ltp&gt; {{ bid | <span style="color:#000;">currency</span> }} &lt;/p&gt;
    &lt;/div&gt;
&lt;/body&gt;
&lt;/html&gt;</pre>
    </div>
    <br>

    <h2>Filter “currency” with Decimal values</h2> 
    <p>A currency value is a number, and numbers have decimal values too. The default currency is value is displayed with <span style="font-style:italic;color:#000;">“two”</span> decimal values. However, you can either remove or add decimal values according to your requirement, by assigning a parameter to the filter also known as <span style="font-style:italic;color:#000;">fractionSize</span>.</p> 
    <p>If you want to add <span style="font-style:italic;color:#000;">“three”</span>, decimal values to the <span style="font-style:italic;color:#000;">filter</span>, then you must do this.</p> 
    <p class="highlight">&amp;ltp&gt; {{ bid | currency: "$" : 3 }} &lt;/p&gt;</p> 
    <p>Set the parameter to <span style="font-style:italic;color:#000;">“0”</span> to <span style="font-style:italic;color:#000;">remove</span> decimal values.</p> 
    <p class="highlight">&amp;ltp&gt; {{ bid | currency: "$" : 0 }} &lt;/p&gt;</p>
    <br> 
    <h2><span style="color:Red;">*</span> Using an AngularJS “currency” Filter in a Controller Using JavaScript</h2> 
    <div class="frmname">
     The Markup
    </div> 
    <div class="code">
     <pre class="pre">
&lt;!DOCTYPE html&gt;
&amp;lthtml&gt;
&amp;lthead&gt;
    &amp;ltscript src="https://ajax.googleapis.com/ajax/libs/angularjs/1.3.8/angular.min.js"&gt;&lt;/script&gt;
&lt;/head&gt;
&amp;ltbody style="font:15px Verdana;"&gt;
    &amp;ltdiv ng-app="biddingApp" ng-controller="bidController"&gt;
        &amp;ltp&gt;
            &amp;ltlabel&gt;Enter a number&lt;/label&gt;
            &amp;ltinput type="text" ng-model="bid" /&gt;
        &lt;/p&gt;

        &amp;ltp&gt; {{ <span style="color:#000;">result</span> }} &lt;/p&gt;
    &lt;/div&gt;
&lt;/body&gt;</pre>
    </div> 
    <div class="frmname">
     The Script
    </div> 
    <div class="code">
     <pre class="pre">
&amp;ltscript&gt;
    var bidApp = angular.module('biddingApp', []);

    bidApp.controller(
        'bidController',
        function ($scope, $filter) {

            $scope.$watch('bid', function (val) {

                $scope.<span style="color:#000;">result</span> = $filter('<span style="color:#000;">currency</span>')(val);

            }, true);
        }
    );
&lt;/script&gt;
&lt;/html&gt;</pre>
    </div>

     <p>Before defining the controller, I have initialized the AngularJS application. Later, I have injected the <span style="font-style:italic;color:#000;">“$filter”</span> object along with the <span style="font-style:italic;color:#000;">“$scope”</span> object to the controller. Finally, I have defined the <span style="font-style:italic;color:#000;">currency</span> filter to get the desired result in my view.</p> 
    </div> 
    <div class="frmname">
     Conclusion
    </div> 
    <p>What we have learned in this article, are two basic methods for defining the <span style="font-style:italic;color:#000;">currency</span> filter in AngularJS. The filter shows a <span style="font-style:italic;color:#000;">“$” (dollar)</span> symbol with each value inside the template. However, we do thousands of transactions using various other currencies. Therefore, it would be nice, if we knew what other symbols we may add with <span style="font-style:italic;color:#000;">currency filter</span>, other than the default symbol, which is the <span style="font-style:italic;color:#000;">“$”</span>.</p>
