     <div class="section"> 
      <h1>CurrencyConverter 0.5</h1> 
      <div id="download-button"> 
       <a class="button green" style="float:right;" href="https://pypi.python.org/packages/42/44/b94faca13a78bcf5edac1793ce1331327afc2cc007339406a944ee0cf672/CurrencyConverter-0.5.tar.gz">Download<br> <span style="font-size: 75%">CurrencyConverter-0.5.tar.gz</span> </a> 
      </div> 
      <p style="font-style: italic">A currency converter using the European Central Bank data.</p> 
      <p> <strong>Latest Version: </strong><a title="Latest Version URL" href="https://pypi.python.org/pypi/CurrencyConverter/0.13.1"> <span>0.13.1</span> </a> </p> 
      <p>This is a currency converter that uses historical rates against a reference currency (Euro).</p> 
      <div id="currency-sources"> 
       <h2>Currency sources</h2> 
       <p>The default source is the <a href="http://www.ecb.int/" rel="nofollow">European Central Bank</a>. This is the ECB historical rates for 42 currencies against the Euro since 1999. It can be downloaded here: <a href="http://www.ecb.int/stats/eurofxref/eurofxref-hist.zip" rel="nofollow">eurofxref-hist.zip</a>. The converter can use different sources as long as the format is the same.</p> 
      </div> 
      <div id="installation"> 
       <h2>Installation</h2> 
       <p>You can install directly after cloning:</p> 
       <pre>$ python setup.py install --user
</pre> 
       <p>Or use the Python package:</p> 
       <pre>$ pip install --user currencyconverter
</pre> 
      </div> 
      <div id="command-line-example"> 
       <h2>Command line example</h2> 
       <pre>$ python currency_converter.py <span class="m">100</span> EUR --to USD
<span class="s2">"100 EUR"</span> is <span class="s2">"137.59 USD"</span> on 2014-03-28.
</pre> 
       <p>After installation, you should have <tt>currency_converter</tt> in your <tt>$PATH</tt>:</p> 
       <pre>$ currency_converter <span class="m">100</span> USD -d 2013-12-12
</pre> 
      </div> 
      <div id="python-api-example"> 
       <h2>Python API example</h2> 
       <p>Example:</p> 
       <pre><span class="o">&gt;&gt;&gt;</span> <span class="kn">from</span> <span class="nn">currency_converter</span> <span class="kn">import</span> <span class="n">CurrencyConverter</span>

</pre> 
       <p>Convert from EUR to USD:</p> 
       <pre><span class="o">&gt;&gt;&gt;</span> <span class="n">c</span><span class="o">.</span><span class="n">convert</span><span class="p">(</span><span class="mi">100</span><span class="p">,</span> <span class="s1">'EUR'</span><span class="p">,</span> <span class="s1">'USD'</span><span class="p">)</span> <span class="c1"># doctest: +SKIP</span>

       <p>Default target currency is EUR:</p> 
       <pre><span class="o">&gt;&gt;&gt;</span> <span class="n">c</span><span class="o">.</span><span class="n">convert</span><span class="p">(</span><span class="mi">100</span><span class="p">,</span> <span class="s1">'EUR'</span><span class="p">)</span>

       <p>Change reference date for rate:</p>
