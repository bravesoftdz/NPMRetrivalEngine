       <h1 class="postTitle">Currency conversion using javascript and openexchangerates.org JSON API</h1> 
       <div class="postMetaData">
        <a href="http://www.dynamicguru.com/author/admin/" title="Posts by Admin" rel="author">Admin</a> on

      <div class="entry"> 
       <p>In this short tutorial we will learn how to build a simple currency converter in javascript using free real-time exchange rates provided by <a href="http://openexchangerates.org/" target="_blank">Open Exchange Rates</a>.</p> 
       <p>This tutorial will require a basic understanding of javascript, AJAX, and the JSON data format.<br> Lets begin!<br> <span id="more-511"></span></p> 
       <h2>Real-time currency conversion rates, for free?</h2> 
       <p>OpenExchangeRates.org really does provide real-time updated currency conversion rates for over 160 currencies absolutely free of cost for personal usage. However, for commercial applications, their paid plans arent too expensive either.<br> For the scope of this article, we will be using only the free API which can be accessed once you register and request your App ID for free from <a href="https://openexchangerates.org/signup/free" target="_blank">here</a></p> 
       <p>Once you have received your App ID, lets get coding.</p> 
       <h2>The API and returned data</h2> 
       <p>The URI of the currency API is:</p> 
       <pre class="wp-code-highlight prettyprint">

http://openexchangerates.org/latest.json

</pre> 
       <h3>Parameters</h3> 
       <p><tt>app_id</tt>: your App ID that you received when you signed up.<br> <tt>base</tt>: to specify a base currency (paid feature)</p> 
       <p>The returned JSON data looks something like:</p> 
       <pre class="wp-code-highlight prettyprint">
{
    "disclaimer": "Exchange rates provided for informational purposes only, with no guarantee whatsoever of accuracy, validity, availability, or fitness for any purpose; use at your own risk. Other than that, have fun! Usage subject to acceptance of terms: http://openexchangerates.org/terms/",
    "license": "Data sourced from various providers with public-facing APIs; copyright may apply; not for resale; no warranties given. Usage subject to acceptance of license agreement: http://openexchangerates.org/license/",
    "timestamp": 1349978408,
    "base": "USD",
    "rates": {
        "AED": 3.673269,
        "AFN": 51.400767,
        "ALL": 108.275499,

        ..... many, many currencies ...

        "ZMK": 5146.528026,
        "ZWL": 322.387247
    }
}
</pre> 
       <p>Also a list of currency codes and their name is also available in JSON format at <a href="http://openexchangerates.org/currencies.json" target="_blank">http://openexchangerates.org/currencies.json<br> </a></p> 
       <h2>Querying data using jQuery AJAX</h2> 
       <p>Now we will query the API using jQuery AJAX :</p> 
       <pre class="wp-code-highlight prettyprint">
app_id="xxxxxxxxxxxxxxxxxxxxxxxxxx"; // your unique app id goes here
uri=encodeURI("http://openexchangerates.org/latest.json?app_id="+app_id);

$.get(uri,function(json) {
// the returned data is available in the json object within the scope of this function
// rates are available in json.rates as properties with their 3 letter international currency short-codes
// Eg. get the value of 1 EUR to base currency (by default USD) :
euro=json.rates.EUR;

},"jsonp");
</pre> 
       <p>The returned data has a list of 160 currencies with their relative value to USD by default.<br> The base currency too can be changed by adding the <tt>base</tt> parameter to URI but it is available only for paid users, so we will skip that</p> 
       <h2>Performing any currency to any currency conversion</h2> 
       <p>Once the data is available ,for converting a from a base currency other than USD, it is not neccessary to query the API for base currency, we can still perform the conversion quite easily by their relative values to USD.<br> Here is how:</p> 
       <pre class="wp-code-highlight prettyprint">
// set your new base currency
my_base="EUR";

// set currency to convert to
my_destination="INR";

// base amount to convert
amount=100;

converted_amount=(amount/json.rates[my_base])*json.rates[my_destination];

alert(converted_amount);
</pre> 
       <p>What we are doing is converting the base amount to USD then multiplying it with the current rate of destination currency which is relative to USD.<br> In the above example, we converted amount from EUR to INR.<br> In real world implementation, it may happen that you may read the base currency as well as the destination currency from user using a html drop down or some other way, that time you can access the rate of that particular currency by iterating over the properties of <tt>rates</tt> object using a <tt>for</tt> loop as follows:</p> 
       <pre class="wp-code-highlight prettyprint">
for(currency in json.rates) {
 if(currency===destination_currency) {
// perform conversions now
}
}
</pre>
