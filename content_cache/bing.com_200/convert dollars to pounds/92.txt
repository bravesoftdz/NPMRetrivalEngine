      <li>Compatibility: The plugin works with all major versions of Microsoft Windows and Excel and both on 32 and 64 bit systems (<a href="#compability">Details</a>).</li>

     <h2>Compatibility</h2> 
     <p>The addin is compatible to all Excel versions from Microsoft Excel 2000 … that is Microsoft Excel 2000, 2002, XP, 2003, 2004, 2007, 2010 and 2013! Furthermore it is compatible to any Windows System from Microsoft Windows XP (that is Microsoft Windows XP, 2000, 2003, 2007, 2010, 2013, 7 and 8) and supports both 32-bit as well as 64-bit computers. At this point in time, the plugin is not compatible with Microsoft Office for Mac.<br> <a name="currencies"></a></p> 
     <h2>Usage</h2> 
     <p>The Add-In supports the following functions:</p> 
     <p>=CX(Amount of money, "Source Currency", "Destination Currency")</p> 
     <p>Converts the specified amount of money from one currency to another. E.g.:</p> 
     <p>=CX(100, "USD", "EUR")</p> 
     <p>will convert 100 US $ to European Euro. The plugin always uses the current system date for the conversion. In the professional version you can specify the date using:</p> 
     <p>=CX(100, "USD", "EUR", "2010-01-01")</p> 
     <p>The date can be specified as:</p> 
     <ul> 
      <li>dd.mm.yy(yy), e.g. 20.11.2013 or 20.11.13</li> 
      <li>yy(yy)-mm-dd, e.g. 2010-11-20 or 10-20-11</li> 
      <li>mm/dd/yy(yy), e.g. 11/20/13 or 20/11/2013</li> 
     </ul>
