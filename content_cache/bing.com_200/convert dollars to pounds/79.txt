     <h1>Currency Formatting</h1> 
     <p> Many web pages and documents publish currency amounts. Shopping cart items and totals, for example, and invoices. </p> 
     <p> Some numbers need conversion to the proper currency format. Examples are totals from adding item prices and calculated sales tax, numbers from a database, and numbers provided in a form. </p> 
     <p> As an example of use, existing software that adds those totals and calculates taxes and shipping charges can be modified to incorporate one of the two functions presented in this article for converting the result into currency format. </p> 
     <p> One of the functions is JavaScript and the other is PHP. Use whichever is applicable for your project. </p> 
     <p> Different currencies use different characters to show groups of thousands and the decimal part. With US Dollars, it's a comma and a period (as in 1,234.56). The functions in this article let you specify which characters to use. Also the number of decimal places for the currency. </p> 
     <p> A financial site's <a href="http://www.thefinancials.com/Default.aspx?SubSectionID=curformat">currency format</a> page and a software distributor's <a href="http://trigeminal.fmsinc.com/samples/setlocalesample2.asp">currency example</a> page show formats for various currencies. If you want to study the nitty-gritty of it, see this <a href="http://www.xencraft.com/resources/multi-currency.html">currency internationalization</a> page. </p> 
     <p> The functions presented in this article don't try to format for every currency. But most can be formatted. </p> 
     <p> Both functions do the same work. They have the same function name and similar variable names. The three customizable variables are spelled the same, except the variables in PHP begin with a "$" character. </p> 
     <p> When a number has more decimals than the currency format, it rounds the currency decimal. </p> 
     <p> When you know how to use one you also know how to use the other. </p> 
     <p> It's simple to implement and simple to use. </p> 
     <p> Try it here by typing in a number. It responds with USD currency formatting. (Specify other formatting in the source code of the function for other currency.) </p>

     <script type="text/javascript">
function CurrencyFormat(number)
{
   var decimalplaces = 2;
   var decimalcharacter = ".";
   var thousandseparater = ",";
   number = parseFloat(number);
   var sign = number < 0 ? "-" : "";
   var formatted = new String(number.toFixed(decimalplaces));
   if( decimalcharacter.length && decimalcharacter != "." ) { formatted = formatted.replace(/\./,decimalcharacter); }
   var integer = "";
   var fraction = "";
   var strnumber = new String(formatted);
   var dotpos = decimalcharacter.length ? strnumber.indexOf(decimalcharacter) : -1;
   if( dotpos > -1 )
   {
      if( dotpos ) { integer = strnumber.substr(0,dotpos); }
      fraction = strnumber.substr(dotpos+1);
   }
   else { integer = strnumber; }
   if( integer ) { integer = String(Math.abs(integer)); }
   while( fraction.length < decimalplaces ) { fraction += "0"; }
   temparray = new Array();
   while( integer.length > 3 )
   {
      temparray.unshift(integer.substr(-3));
      integer = integer.substr(0,integer.length-3);
   }
   temparray.unshift(integer);
   integer = temparray.join(thousandseparater);
   return sign + integer + decimalcharacter + fraction;
}
function FormatAsCurrency(n) { document.getElementById("formatted").innerHTML = CurrencyFormat(n) + " USD"; }
</script>

     <p> Here's the source code for the <b>JavaScript function.</b> Implementation and use notes follow. </p>

function CurrencyFormat(number)
{
   var decimalplaces = 2;
   var decimalcharacter = ".";
   var thousandseparater = ",";

   var sign = number &lt; 0 ? "-" : "";

   if( dotpos &gt; -1 )
   {

     <p> <b><i>JavaScript Function Implementation and Use Notes</i></b> </p> 
     <p> Paste the JavaScript anywhere in the source code of your web page where JavaScript can run. Update the 3 customization variables: </p> 
     <ol> 
      <li> <p> <span class="code">var decimalplaces = 2; </span>If needed, change the number 2 for the number of decimal places the currency formatting required. (The number 0 is acceptable.) </p> </li> 
      <li> <p> <span class="code">var decimalcharacter = "."; </span>If needed, replace the period between the quotation marks with another character. The character separates the whole number from the decimal. For no decimal character, remove the period. </p> </li> 
      <li> <p> <span class="code">var thousandseparater = ","; </span>If needed, change the comma between the quotation marks. The character (which may be a space) will separate the groups of thousands. For no separation, remove the comma. </p> </li> 
     </ol> 
     <p> To use the function, call it with the number to be currency formatted. Here's an example, which stores a currency format in the variable amount. It's followed by three ways the variable amount can be used. </p> 
     <pre onclick="containerSelect(this)" class="source-code-box">
var amount = CurrencyFormat(12345.6789);

alert(amount+" USD"); // Alert box with a space and currency code appended to amount.
document.write("&lt;b&gt;amount+" USD&lt;/b&gt;"); // Print amount with space and currency code appended.
document.getElementById("DivOrSpanIDvalue").innerHTML = amount; // Update content of an HTML container tag with amount.

</pre>

}
