    <span class="Quote">I have used AngularJS “currency” filters in my projects, also wrote an article previously on <a href="http://www.encodedna.com/angularjs/tutorial/format-text-using-angularjs-currency-filter.htm" target="_blank" class="quotelink">how to format a string using AngularJS currency filter</a>. Consider this article related to the previous one, as here I am going to show you how to use the currency filter to get the output in Rupee (or perhaps Pound and Euro) instead of dollar.</span> 
    <div class="top_ad" id="top_ad" style="clear:both;float:left;width:100%;margin:10px 0;">
     <script async src="//pagead2.googlesyndication.com/pagead/js/adsbygoogle.js"></script>

    <p>The dollar ($) as you know, is the <a href="https://docs.angularjs.org/api/ng/filter/currency" target="_blank" class="contact" rel="nofollow">default symbol</a> for Angular currency filter. To get the result in other symbol, for example Rupee (?), you must explicitly define it in your app. You can define any other currency symbol (if not just Rupee), using HTML entities for currency. </p> 
    <p>There are two different HTML entities for Rupee.</p> 
    <p><span style="color:Red;">1)</span> For Rs - 8360 <br><span style="color:Red;">2)</span> For ? symbol - 8377</p> 
    <p>Now, let’s see how you can add these symbols in your AngularJS app with the <span style="font-style:italic;color:#000;">currency</span> filter.</p> 
    <h2>AngularJS currency Filter with ? Rupee symbol</h2> 
    <div class="code">
     <pre class="pre">
&amp;ltbody&gt;
    &amp;ltscript src="https://ajax.googleapis.com/ajax/libs/angularjs/1.4.5/angular.min.js"&gt;&lt;/script&gt;

    &amp;ltdiv ng-app&gt;
        &amp;ltp&gt;
            &amp;ltlabel&gt;Enter a number&lt;/label&gt;
            &amp;ltinput type="text" ng-model="bid" /&gt;
        &lt;/p&gt;

        &amp;ltp&gt; {{ bid | <span style="color:#000;">currency</span>:'&amp;#8377;' }} &lt;/p&gt;
    &lt;/div&gt;
&lt;/body&gt;</pre>
    </div> 
    <p>For the rupee symbol I have used decimal “8377” prefixed with symbols “&amp; and #”, just after the <span style="font-style:italic;color:#000;">currency</span> filter.</p> 
    <h3><strong>Try it your self</strong>.</h3> 
    <div class="imagecontainer" style="overflow:hidden;clear:both;"> 
     <div ng-app>
      <p><label>Enter a number: </label><input type="text" ng-model="bid"></p>
      <p style="font-style:italic;"> {{ bid | currency:'?' }} </p>
     </div> 
    </div> 
    <h2>currency Filter with Rupee Sign “Rs”</h2> 
    <p>In the above example, simply replace <span style="font-style:italic;color:#000;">8377</span> with <span style="font-style:italic;color:#000;">8360</span> to get the result in <span style="font-style:italic;color:#000;">Rs</span>.</p> 
    <div class="code">
     <pre class="pre">
 &amp;ltp&gt; {{ bid | <span style="color:#000;">currency</span>:'&amp;#8360' }} &lt;/p&gt;</pre>
    </div>

    <h2>Injecting AngularJS “currency” Filter Rupee in Controller</h2> 
    <p>You can inject the currency filter with the Rupee symbol (?) in your applications controller. Not just the symbol, you can even add signs like <span style="font-style:italic;color:#000;">INR, “Rupee”</span> and <span style="font-style:italic;color:#000;">Rs</span> in the controller.</p> 
    <div class="frmname">
     The Markup and GridView
    </div> 
    <div class="code">
     <pre class="pre">
&lt;!DOCTYPE html&gt;
&amp;lthtml&gt;
&amp;lthead&gt;
    &amp;ltscript src="https://ajax.googleapis.com/ajax/libs/angularjs/1.4.5/angular.min.js"&gt;&lt;/script&gt;
&lt;/head&gt;
&amp;ltbody&gt;
    &amp;ltdiv ng-app="biddingApp" ng-controller="bidController"&gt;
        &amp;ltp&gt;
            &amp;ltlabel&gt;Enter a number&lt;/label&gt;
            &amp;ltinput type="text" <span style="color:#000;">ng-model</span>="bid" /&gt;
        &lt;/p&gt;

        &amp;ltp&gt; {{ result }} &lt;/p&gt;
    &lt;/div&gt;
&lt;/body&gt;</pre>
    </div> 
    <div class="frmname">
     The Controller
    </div> 
    <div class="code">
     <pre class="pre">
&amp;ltscript&gt;
    var bidApp = angular.module('biddingApp', []);

    bidApp.controller(
        'bidController',
        function ($scope, $filter) {

            $scope.$watch('bid', function (val) {
                $scope.result = $filter('currency')(val, '?', 2);
            }, true);
        }
    );
&lt;/script&gt;
&lt;/html&gt;</pre>
    </div>

     <p>I have defined the <span style="font-style:italic;color:#000;">currency</span> filter, with 3 parameters. The <span style="font-style:italic;color:#000;">val</span> has a numeric value, followed by Rupee symbol and assigned a value <span style="font-style:italic;color:#000;">2</span> for two decimals.</p> 
    </div>
