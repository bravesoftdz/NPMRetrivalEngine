         <!--WCS[226]--><br> <strong>Background and Explanation of the code</strong></p> 
        <hr size="2">?
        <a href="http://codingzone.net/wp-content/uploads/2011/05/screenshot4.jpg"><img class="alignright size-medium wp-image-2736" title="screenshot" src="http://codingzone.net/wp-content/uploads/2011/05/screenshot4-300x125.jpg" alt="" width="300" height="125"></a>We have assumed that 1$ = Rs. 44.20.&nbsp;The above program converts the amount in dollars to rupees. For this we have created two classes 
        <strong>rupee and dollar</strong> for maintaining amount in rupees and dollars respectively.In 
        <strong>main ( )</strong> we first create an object 
        <strong>d1</strong> of class 
        <strong>dollar</strong> and initialize it using single parameter constructor which is then displayed.Then we create object
        <strong> r1</strong> of class 
        <strong>rupee</strong>.The statement
        <p></p> 
        <p><strong>r1 = d1;</strong></p> 
        <p>on execution invokes the overloaded typecast operator function<strong> operator rupee ( )</strong> which is defined as conversion function in source class <strong>dollar</strong>.As the above statement involves objects of different classes so there is need of conversion function.The <strong>operator rupee ( )</strong> conversion function converts the object <strong>d1</strong> of&nbsp; which it is a member to <strong>rupee</strong> object which is then returned and assigned to object <strong>r1</strong> in <strong>main ( )</strong>.The statement</p> 
        <p><strong>return (rupee (rs1));</strong></p> 
        <p>first creates an unnamed object of class rupee and initialize it&nbsp; using single parameter constructor <strong>(rupee (double))</strong> which is then returned.</p> 
        <p><strong>OUTPUT</strong></p> 
        <p>Money in dollars = 4</p> 
        <p>Money in Rs. =&nbsp; 176.8</p>
