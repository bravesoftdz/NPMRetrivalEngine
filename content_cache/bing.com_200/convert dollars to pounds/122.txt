       <h1>MS Excel: <span>How to convert currency into words</span></h1> 
       <p>This Excel tutorial explains how to convert currency into words (with screenshots and step-by-step instructions).</p> 
       <p><span class="question">Question:</span> In Microsoft Excel, how can I convert currency to words?</p> 
       <p>For example,</p> 
       <p>$100 should read as "one hundred exactly"</p> 
       <p><span class="answer">Answer:</span> To convert currency into words, you'll need to open your Excel spreadsheet. Press <kbd>Alt</kbd>+<kbd>F11</kbd> and create a new module. (Under the Insert menu, select Module).</p> 
       <p>Then paste into the new module the following two functions:</p> 
       <pre class="notranslate">Function English(ByVal N As Currency) As String

   Const Thousand = 1000@
   Const Million = Thousand * Thousand
   Const Billion = Thousand * Million
   Const Trillion = Thousand * Billion

   If (N = 0@) Then English = "zero": Exit Function

   Dim Buf As String: If (N &lt; 0@) Then Buf = "negative " Else Buf = ""
   Dim Frac As Currency: Frac = Abs(N - Fix(N))
   If (N &lt; 0@ Or Frac &lt;&gt; 0@) Then N = Abs(Fix(N))
   Dim AtLeastOne As Integer: AtLeastOne = N &gt;= 1

   If (N &gt;= Trillion) Then
      Buf = Buf &amp; EnglishDigitGroup(Int(N / Trillion)) &amp; " trillion"
      N = N - Int(N / Trillion) * Trillion
      If (N &gt;= 1@) Then Buf = Buf &amp; " "
   End If

   If (N &gt;= Billion) Then
      Buf = Buf &amp; EnglishDigitGroup(Int(N / Billion)) &amp; " billion"
      N = N - Int(N / Billion) * Billion
      If (N &gt;= 1@) Then Buf = Buf &amp; " "
   End If

   If (N &gt;= Million) Then
      Buf = Buf &amp; EnglishDigitGroup(N \ Million) &amp; " million"
      N = N Mod Million
      If (N &gt;= 1@) Then Buf = Buf &amp; " "
   End If

   If (N &gt;= Thousand) Then
      Buf = Buf &amp; EnglishDigitGroup(N \ Thousand) &amp; " thousand"
      N = N Mod Thousand
      If (N &gt;= 1@) Then Buf = Buf &amp; " "
   End If

   If (N &gt;= 1@) Then
      Buf = Buf &amp; EnglishDigitGroup(N)
   End If

   If (Frac = 0@) Then
      Buf = Buf &amp; " exactly"
   ElseIf (Int(Frac * 100@) = Frac * 100@) Then
      If AtLeastOne Then Buf = Buf &amp; " and "
      Buf = Buf &amp; Format$(Frac * 100@, "00") &amp; "/100"
   Else
      If AtLeastOne Then Buf = Buf &amp; " and "
      Buf = Buf &amp; Format$(Frac * 10000@, "0000") &amp; "/10000"
   End If

   English = Buf
End Function

Private Function EnglishDigitGroup(ByVal N As Integer) As String

   Const Hundred = " hundred"
   Const One = "one"
   Const Two = "two"
   Const Three = "three"
   Const Four = "four"
   Const Five = "five"
   Const Six = "six"
   Const Seven = "seven"
   Const Eight = "eight"
   Const Nine = "nine"
   Dim Buf As String: Buf = ""
   Dim Flag As Integer: Flag = False

   Select Case (N \ 100)
      Case 0: Buf = "": Flag = False
      Case 1: Buf = One &amp; Hundred: Flag = True
      Case 2: Buf = Two &amp; Hundred: Flag = True
      Case 3: Buf = Three &amp; Hundred: Flag = True
      Case 4: Buf = Four &amp; Hundred: Flag = True
      Case 5: Buf = Five &amp; Hundred: Flag = True
      Case 6: Buf = Six &amp; Hundred: Flag = True
      Case 7: Buf = Seven &amp; Hundred: Flag = True
      Case 8: Buf = Eight &amp; Hundred: Flag = True
      Case 9: Buf = Nine &amp; Hundred: Flag = True
   End Select

   If (Flag &lt;&gt; False) Then N = N Mod 100
   If (N &gt; 0) Then
      If (Flag &lt;&gt; False) Then Buf = Buf &amp; " "
   Else
      EnglishDigitGroup = Buf
      Exit Function
   End If

   Select Case (N \ 10)
      Case 0, 1: Flag = False
      Case 2: Buf = Buf &amp; "twenty": Flag = True
      Case 3: Buf = Buf &amp; "thirty": Flag = True
      Case 4: Buf = Buf &amp; "forty": Flag = True
      Case 5: Buf = Buf &amp; "fifty": Flag = True
      Case 6: Buf = Buf &amp; "sixty": Flag = True
      Case 7: Buf = Buf &amp; "seventy": Flag = True
      Case 8: Buf = Buf &amp; "eighty": Flag = True
      Case 9: Buf = Buf &amp; "ninety": Flag = True
   End Select

   If (Flag &lt;&gt; False) Then N = N Mod 10
   If (N &gt; 0) Then
      If (Flag &lt;&gt; False) Then Buf = Buf &amp; "-"
   Else
      EnglishDigitGroup = Buf
      Exit Function
   End If

   Select Case (N)
      Case 0:
      Case 1: Buf = Buf &amp; One
      Case 2: Buf = Buf &amp; Two
      Case 3: Buf = Buf &amp; Three
      Case 4: Buf = Buf &amp; Four
      Case 5: Buf = Buf &amp; Five
      Case 6: Buf = Buf &amp; Six
      Case 7: Buf = Buf &amp; Seven
      Case 8: Buf = Buf &amp; Eight
      Case 9: Buf = Buf &amp; Nine
      Case 10: Buf = Buf &amp; "ten"
      Case 11: Buf = Buf &amp; "eleven"
      Case 12: Buf = Buf &amp; "twelve"
      Case 13: Buf = Buf &amp; "thirteen"
      Case 14: Buf = Buf &amp; "fourteen"
      Case 15: Buf = Buf &amp; "fifteen"
      Case 16: Buf = Buf &amp; "sixteen"
      Case 17: Buf = Buf &amp; "seventeen"
      Case 18: Buf = Buf &amp; "eighteen"
      Case 19: Buf = Buf &amp; "nineteen"
   End Select

   EnglishDigitGroup = Buf

End Function</pre> 
       <p>Now, when you want to convert currency into words, you will reference the English function in your Excel spreadsheet as follows:</p> 
       <pre class="notranslate">=English (100)
<em>Result:</em> "one hundred exactly"

=English (125)
<em>Result:</em> "one hundred twenty-five exactly"

=English (125.3)
<em>Result:</em> "one hundred twenty-five and 30/100"</pre>
