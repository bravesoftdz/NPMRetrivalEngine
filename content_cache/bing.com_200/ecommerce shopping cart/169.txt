     <p>In this tutorial, we are going to see a simple PHP example for creating a shopping cart application. This shopping cart application is purposely kept simple and as minimal as possible. You can download this and quickly customize it for your needs. In this example, we are displaying a&nbsp;list of products from the&nbsp;database. For each product, we can select product quantity and add to cart. The cart items are stored in the session. The cart can be emptied by removing the items from the&nbsp;session.</p> 
     <p><img class="alignnone size-full wp-image-4091" title="php-shopping-cart" src="http://phppot.com/wp-content/uploads/2014/10/phpshoppingcart.png" alt="php-shopping-cart" width="582" height="585" srcset="http://phppot.com/wp-content/uploads/2014/10/phpshoppingcart.png 582w, http://phppot.com/wp-content/uploads/2014/10/phpshoppingcart-150x150.png 150w, http://phppot.com/wp-content/uploads/2014/10/phpshoppingcart-298x300.png 298w" sizes="(max-width: 582px) 100vw, 582px"></p>

     <h2>Adding Products&nbsp; to Shopping Cart</h2> 
     <p>First, we need to&nbsp;create our product catalog page by listing products from the&nbsp;database. The following code retrieves the&nbsp;list of products from the database and displays them in a grid view. Each product has the ‘Add to cart’ option to place it into&nbsp;the cart.</p> 
     <pre class="prettyprint lang-php">&lt;?php
$product_array = $db_handle-&gt;runQuery("SELECT * FROM tblproduct ORDER BY id ASC");
if (!empty($product_array)) { 
foreach($product_array as $key=&gt;$value){
?&gt;
&lt;div class="product-item"&gt;
	&lt;form method="post" action="index.php?action=add&amp;code=&lt;?php echo $product_array[$key]["code"]; ?&gt;"&gt;
	&lt;div class="product-image"&gt;&lt;img src="&lt;?php echo $product_array[$key]["image"]; ?&gt;"&gt;&lt;/div&gt;
	&lt;div&gt;&lt;strong&gt;&lt;?php echo $product_array[$key]["name"]; ?&gt;&lt;/strong&gt;&lt;/div&gt;
	&lt;div class="product-price"&gt;&lt;?php echo "$".$product_array[$key]["price"]; ?&gt;&lt;/div&gt;
	&lt;div&gt;&lt;input type="text" name="quantity" value="1" size="2" /&gt;&lt;input type="submit" value="Add to cart" class="btnAddAction" /&gt;&lt;/div&gt;
	&lt;/form&gt;
&lt;/div&gt;
&lt;?php }} ?&gt;</pre> 
     <p>On clicking the ‘Add to cart’ button, we add the selected product id and its quantity into a shopping cart. We are using PHP SESSION to store the cart items. The following code shows a switch case used for adding products to the cart. If we add the&nbsp;same product multiple times in a cart then the cart item&nbsp;quantity&nbsp;will be incremented.</p> 
     <pre class="prettyprint lang-php">case "add":
	if(!empty($_POST["quantity"])) {
		$productByCode = $db_handle-&gt;runQuery("SELECT * FROM tblproduct WHERE code='" . $_GET["code"] . "'");
		$itemArray = array($productByCode[0]["code"]=&gt;array('name'=&gt;$productByCode[0]["name"], 'code'=&gt;$productByCode[0]["code"], 'quantity'=&gt;$_POST["quantity"], 'price'=&gt;$productByCode[0]["price"]));
		
		if(!empty($_SESSION["cart_item"])) {
			if(in_array($productByCode[0]["code"],array_keys($_SESSION["cart_item"]))) {
				foreach($_SESSION["cart_item"] as $k =&gt; $v) {
						if($productByCode[0]["code"] == $k) {
							if(empty($_SESSION["cart_item"][$k]["quantity"])) {
								$_SESSION["cart_item"][$k]["quantity"] = 0;
							}
							$_SESSION["cart_item"][$k]["quantity"] += $_POST["quantity"];
						}
				}
			} else {
				$_SESSION["cart_item"] = array_merge($_SESSION["cart_item"],$itemArray);
			}
		} else {
			$_SESSION["cart_item"] = $itemArray;
		}
	}
break;</pre> 
     <h2>Removing or Clearing Cart Item</h2> 
     <p>In this example, we allow users to remove each item from the cart by using the ‘Remove Item’ link in each row. Also, we let the users&nbsp;clear the whole cart by using ‘Empty Cart’ option. On clicking ‘Remove Item’, we call remove action with the reference of the item code to clear the particular item of the cart session. We use PHP unset() to clear the cart session. The following code shows the switch cases used to perform ‘Remove Item’ and ‘Empty Cart’ actions.</p> 
     <pre class="prettyprint lang-php">case "remove":
	if(!empty($_SESSION["cart_item"])) {
		foreach($_SESSION["cart_item"] as $k =&gt; $v) {
			if($_GET["code"] == $k)	unset($_SESSION["cart_item"][$k]);				
			if(empty($_SESSION["cart_item"])) unset($_SESSION["cart_item"]);
		}
	}
break;
case "empty":
	unset($_SESSION["cart_item"]);
break;</pre> 
     <p>Import the following SQL script for creating a product table and load your own data to show in your catalog&nbsp;page.</p> 
     <pre class="prettyprint lang-php">CREATE TABLE IF NOT EXISTS `tblproduct` (
  `id` int(8) NOT NULL AUTO_INCREMENT,
  `name` varchar(255) NOT NULL,
  `code` varchar(255) NOT NULL,
  `image` text NOT NULL,
  `price` double(10,2) NOT NULL,
  PRIMARY KEY (`id`),
  UNIQUE KEY `product_code` (`code`)
)</pre>
