        <p><img title="bitcoing" src="/d/file/p/2014/05/02/faca22484ab5c166aaacfb8db267bc6d.jpg" alt="">The rise of Bitcoin has resulted in the development of a variety of services that enable individuals and companies to track Bitcoin, create and manage Bitcoin wallets, and perform transactions using the cryptocurrency.</p>
        <p>One of the most popular Bitcoin websites,Blockchain.info, hasa suite of Bitcoin APIsand in this tutorial, we’ll look at how two of those can be used to build a Bitcoin currency conversion application.</p>
        <h2>Getting started</h2>
        <p>To start, you will need a hosting account that supports PHP 5.0 or greater and has the PHP curl module installed. In the root web directory of your hosting account, create two empty PHP files, index.php and converter.php.</p>
        <h2>The index page</h2>
        <p>Our index page, index.php, should contain a form that we can use to prompt the user for information about the currency conversion desired. In the empty index.php file, paste the following code:</p>
        <pre>&lt;html&gt;&lt;head&gt;&lt;title&gt;Bitcoin Currency Converter&lt;/title&gt;&lt;/head&gt;&lt;body&gt;&lt;form action="converter.php" method="post"&gt;&lt;h1&gt;Bitcoin Currency Converter&lt;/h1&gt;&lt;p&gt;&lt;label for="amount"&gt;Bitcoin Amount&lt;/label&gt;&lt;input type="text" name="amount" id="amount"&gt;&lt;/p&gt;&lt;p&gt;&lt;label for="currency"&gt;Currency&lt;/label&gt;&lt;select name="currency" id="currency"&gt;&lt;option value="USD"&gt;USD&lt;/option&gt;&lt;option value="EUR"&gt;EUR&lt;/option&gt;&lt;option value="CNY"&gt;CNY&lt;/option&gt;&lt;option value="JPY"&gt;JPY&lt;/option&gt;&lt;option value="SGD"&gt;SGD&lt;/option&gt;&lt;option value="HKD"&gt;HKD&lt;/option&gt;&lt;option value="CAD"&gt;CAD&lt;/option&gt;&lt;option value="AUD"&gt;AUD&lt;/option&gt;&lt;option value="NZD"&gt;NZD&lt;/option&gt;&lt;option value="GBP"&gt;GBP&lt;/option&gt;&lt;option value="DKK"&gt;DKK&lt;/option&gt;&lt;option value="SEK"&gt;SEK&lt;/option&gt;&lt;option value="BRL"&gt;BRL&lt;/option&gt;&lt;option value="CHF"&gt;CHF&lt;/option&gt;&lt;option value="RUB"&gt;RUB&lt;/option&gt;&lt;option value="SLL"&gt;SLL&lt;/option&gt;&lt;/select&gt;&lt;/p&gt;&lt;p&gt;&lt;input type="submit" name="submit" value="Submit"&gt;&lt;/p&gt;&lt;/form&gt;&lt;/body&gt;&lt;/html&gt;</pre>
        <p>Now our index page is in place:</p>
        <p><img title="Index page" src="/d/file/p/2014/05/02/9c661bcba1e2190a6ae176eaf2d728f6.jpg" alt=""></p>
        <h2>The converter script</h2>
        <p>To perform the currency conversion, our index page will post data submitted through its form to converter.php. In the empty converter.php file, paste the following code:</p>
        <pre>&lt;html&gt;&lt;head&gt; &lt;title&gt;Bitcoin Currency Converter&lt;/title&gt;&lt;/head&gt;&lt;body&gt;&lt;?php// Make sure the user submitted all of the data requiredif(isset($_POST['amount']) &amp;&amp; is_numeric($_POST['amount']) &amp;&amp; isset($_POST['currency'])) { 	// Use curl to perform the currency conversion using Blockchain.info's currency conversion API	$ch = curl_init();	curl_setopt($ch, CURLOPT_URL, "https://blockchain.info/tobtc?currency=" . $_POST['currency'] . "&amp;value=" . $_POST['amount']);	curl_setopt($ch, CURLOPT_RETURNTRANSFER, 1);	$conversion = curl_exec($ch);	// Use curl to get current prices and 15 minute averages for all currencies from Blockchain.info's exchange rates API	curl_setopt($ch, CURLOPT_URL, "https://blockchain.info/ticker");	curl_setopt($ch, CURLOPT_RETURNTRANSFER, 1);	$prices = json_decode(curl_exec($ch), true);	curl_close($ch);?&gt;&lt;h1&gt;Conversion Results&lt;/h1&gt;&lt;p&gt;&lt;?php echo $_POST['amount']; ?&gt; &lt;?php echo $_POST['currency']; ?&gt; is &lt;?php echo $conversion; ?&gt; BTC.&lt;/p&gt;&lt;h2&gt;Historical Prices&lt;/h2&gt;&lt;p&gt;&lt;strong&gt;Last price:&lt;/strong&gt; &lt;?php echo $prices[$_POST['currency']]['last']; ?&gt;&lt;?php echo $prices[$_POST['currency']]['symbol']; ?&gt;&lt;/p&gt;&lt;p&gt;&lt;strong&gt;Buy price:&lt;/strong&gt; &lt;?php echo $prices[$_POST['currency']]['buy']; ?&gt;&lt;?php echo $prices[$_POST['currency']]['symbol']; ?&gt;&lt;/p&gt;&lt;p&gt;&lt;strong&gt;Sell price:&lt;/strong&gt; &lt;?php echo $prices[$_POST['currency']]['sell']; ?&gt;&lt;?php echo $prices[$_POST['currency']]['symbol']; ?&gt;&lt;/p&gt;&lt;p&gt;&lt;strong&gt;15 minute average price:&lt;/strong&gt; &lt;?php echo $prices[$_POST['currency']]['15m']; ?&gt;&lt;?php echo $prices[$_POST['currency']]['symbol']; ?&gt;&lt;/p&gt;&lt;?php } else { ?&gt;&lt;p&gt;Please fill out the form completely. &lt;a href="index.php"&gt;Go back.&lt;/a&gt;&lt;/p&gt;&lt;?php } ?&gt;&lt;/body&gt;&lt;/html&gt;</pre>
        <p>The code above calls two endpoints associated with Blockchain.info’s Market Prices and Exchanges Rates API. The first endpoint allows us perform the actual currency conversion for the currency and amount supplied by the user. The second endpoint provides real-time currency conversion information, such as the current buy and sell prices and 15-minute average price. We can use this to display more information to the user about the state of the market in their chosen currency:</p>
        <p><img title="Bitcoin currency conversion results" src="/d/file/p/2014/05/02/1135de5b6c33b69202ded4128c477b21.jpg" alt=""></p>
        <h2>Adding a price chart</h2>

&lt;?php echo $prices[$_POST['currency']]['15m']; ?&gt;&lt;?php echo $prices[$_POST['currency']]['symbol']; ?&gt;&lt;/p&gt;&lt;?php// Display the pricing chart if we're doing a US Dollar conversionif($_POST['currency'] == "USD") { // Use curl to get pricing chart data for the past 60 days$ch = curl_init();curl_setopt($ch, CURLOPT_URL, "https://blockchain.info/charts/market-price?showDataPoints=true&amp;timespan=60days&amp;show_header=true&amp;daysAverageString=7&amp;scale=0&amp;format=json");curl_setopt($ch, CURLOPT_RETURNTRANSFER, 1);$chartdata = json_decode(curl_exec($ch), true);?&gt;&lt;div id="chart" style="width: 1000px;         <h2>Ideas for further development</h2>
        <p>Blockchain.info has a set of Wallet APIs that can be used to create new Bitcoin wallets and send and receive payments using them.
