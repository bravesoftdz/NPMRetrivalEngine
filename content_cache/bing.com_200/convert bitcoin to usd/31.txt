    <h1> Exchange Rates API <small>Market Prices and exchanges rates api</small> </h1> 
   </div> 
   <p class="well"> Some API calls are available with <a href="http://en.wikipedia.org/wiki/Cross-origin_resource_sharing" targe="new">CORS headers</a> if you add a &amp;cors=true paramter to the GET request </p> 
   <p> <b>URL:</b> <a href="/ticker">https://blockchain.info/ticker</a> </p> 
   <p> <b>No Parameters</b> </p> 
   <p> Returns a JSON object with the currency codes as keys. "15m" is the 15 minutes delayed market price, "last" is the most recent market price, "symbol" is the currency symbol. </p>

   <p> <b>URL:</b> <a href="/tobtc">https://blockchain.info/tobtc</a> </p> 
   <p> Convert x value in the provided currency to btc. </p> 
   <p> <b> ex: https://blockchain.info/tobtc?currency=USD&amp;value=500</b> </p> 
   <h5>Parameters</h5> 
   <ul> 
    <li><b>currency</b> - A currency code. See list above.</li> 
    <li><b>value</b> - Value to convert.</li> 
   </ul> 
   <p> Returns the value in BTC. </p>
