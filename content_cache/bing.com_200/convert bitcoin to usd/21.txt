      <h1>CoinDesk Bitcoin Price Index API</h1> CoinDesk provides a simple API to make its 
      <a href="/price/">Bitcoin Price Index</a> (BPI) data programmatically available to others. You are free to use this API to include our data in any application or website as you see fit, as long as each&nbsp;page or app that uses it includes the text “Powered by 
      <a href="/price/">CoinDesk</a>”, linking to our 
      <a href="/price/">price</a> page. CoinDesk data is made available through a number of HTTP resources, and data is returned in JSON format. Please do not abuse our service. 
      <h2>BPI real-time data</h2> On the CoinDesk website, we publish the BPI in USD, EUR, and GBP, calculated every minute, based on criteria as discussed on the 
      <a href="/price/">CoinDesk BPI page</a>. This same data can be retrieved using the endpoint: 
      <ul class="endpoint"> 
       <li>https://api.coindesk.com/v1/bpi/currentprice.json</li> 
      </ul> 
      <blockquote>
       Sample JSON Response: 
       <pre>{"time":{"updated":"Oct 3, 2017 17:15:00 UTC","updatedISO":"2017-10-03T17:15:00+00:00","updateduk":"Oct 3, 2017 at 18:15 BST"},"disclaimer":"This data was produced from the CoinDesk Bitcoin Price Index (USD). Non-USD currency data converted using hourly conversion rate from openexchangerates.org","chartName":"Bitcoin","bpi":{"USD":{"code":"USD","symbol":"$","rate":"4,281.0675","description":"United States Dollar","rate_float":4281.0675},"GBP":{"code":"GBP","symbol":"£","rate":"3,231.3497","description":"British Pound Sterling","rate_float":3231.3497},"EUR":{"code":"EUR","symbol":"€","rate":"3,642.2595","description":"Euro","rate_float":3642.2595}}}</pre> 
      </blockquote> We also offer the BPI converted into in any of our 
      <a href="https://api.coindesk.com/v1/bpi/supported-currencies.json" target="_blank">supported currencies</a>. This data can be accessed using the endpoint: 
      <ul class="endpoint"> 
       <li>https://api.coindesk.com/v1/bpi/currentprice/&lt;CODE&gt;.json</li> 
      </ul> Where &lt;CODE&gt; should be replaced by a valid ISO 4217 currency code as per our 
      <a href="https://api.coindesk.com/v1/bpi/supported-currencies.json" target="_blank">supported currency list</a>. 
      <blockquote>
       <em>Sample Request</em> 
       <code>https://api.coindesk.com/v1/bpi/currentprice/CNY.json</code> 
       <em>Sample JSON Response</em> 
       <pre>{"time":{"updated":"Sep 18, 2013 17:27:00 UTC","updatedISO":"2013-09-18T17:27:00+00:00"},"disclaimer":"This data was produced from the CoinDesk Bitcoin Price Index. Non-USD currency data converted using hourly conversion rate from openexchangerates.org","bpi":{"USD":{"code":"USD","rate":"126.5235","description":"United States Dollar","rate_float":126.5235},"CNY":{"code":"CNY","rate":"775.0665","description":"Chinese Yuan","rate_float":"775.0665"}}}</pre> 
      </blockquote> 
      <h2>Historical BPI data</h2> We offer historical data from our Bitcoin Price Index through the following endpoint: 
      <ul class="endpoint"> 
       <li>https://api.coindesk.com/v1/bpi/historical/close.json</li> 
      </ul> By default, this will return the previous 31 days' worth of data. This endpoint accepts the following optional parameters: 
      <ul> 
       <li><strong>?index=[USD/CNY]</strong>The index to return data for. Defaults to USD.</li> 
       <li><strong>?currency=&lt;VALUE&gt;</strong>The currency to return the data in, specified in ISO 4217 format. Defaults to USD.</li> 
       <li><strong>?start=&lt;VALUE&gt;&amp;end=&lt;VALUE&gt;</strong> Allows data to be returned for a specific date range. Must be listed as a pair of start and end parameters, with dates supplied in the YYYY-MM-DD format, e.g. 2013-09-01 for September 1st, 2013.</li> 
       <li><strong>?for=yesterday</strong>Specifying this will return a single value for the previous day. Overrides the start/end parameter.</li> 
      </ul> 
      <blockquote>
       <em>Sample Request:</em> 
       <code>https://api.coindesk.com/v1/bpi/historical/close.json?start=2013-09-01&amp;end=2013-09-05</code> 
       <em>Sample JSON Response:</em> 
       <pre>{"bpi":{"2013-09-01":128.2597,"2013-09-02":127.3648,"2013-09-03":127.5915,"2013-09-04":120.5738,"2013-09-05":120.5333},"disclaimer":"This data was produced from the CoinDesk Bitcoin Price Index. BPI value data returned as USD.","time":{"updated":"Sep 6, 2013 00:03:00 UTC","updatedISO":"2013-09-06T00:03:00+00:00"}}</pre> 
      </blockquote>
