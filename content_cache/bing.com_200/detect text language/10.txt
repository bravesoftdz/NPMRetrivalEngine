            <h1>language_detection</h1> 
            <p><a href="https://travis-ci.org/patrick-schur/language_detection" rel="nofollow"><img alt="Build Status" src="https://travis-ci.org/patrick-schur/language_detection.svg?branch=master"></a> <a href="https://packagist.org/packages/patrick-schur/language-detection" rel="nofollow"><img alt="Version" src="https://img.shields.io/packagist/v/patrick-schur/language-detection.svg?style=flat-square"></a> <a href="https://packagist.org/packages/patrick-schur/language-detection" rel="nofollow"><img alt="Total Downloads" src="https://img.shields.io/packagist/dt/patrick-schur/language-detection.svg?style=flat-square"></a> <a href="https://github.com/patrick-schur/language_detection" rel="nofollow"><img alt="Maintenance" src="https://img.shields.io/maintenance/yes/2016.svg?style=flat-square"></a> <a href="https://opensource.org/licenses/MIT" rel="nofollow"><img alt="License" src="https://img.shields.io/packagist/l/patrick-schur/language-detection.svg?style=flat-square"></a></p> 
            <p>Detect the language from a given text. To do that it generates a language profile based on <a href="https://en.wikipedia.org/wiki/N-gram" rel="nofollow">N-grams</a> for every file in <code>etc</code> directory. Then it generate such language profile for the unknown text and compare the previosly language profiles against the unknown.</p> 
            <h2>Requirements:</h2> 
            <p>Only requirement is a PHP version greater than or equal to 7.1. &gt; <strong>Note:</strong> language_detection requires the <a href="http://php.net/manual/en/book.mbstring.php" rel="nofollow">Multibyte String</a> extension in order to work. </p> 
            <h2>Install via Composer</h2> 
            <pre><code class="language-text">composer require patrick-schur/language-detection
</code></pre>
            <p>Or add the following to <code>composer.json</code></p>
            <pre><code class="language-json">{
  "require": {
     "patrick-schur/language-detection": "*"
  }
}
</code></pre>
            <p></p> 
            <h2>Basic Usage</h2> 
            <p></p> 
            <p>Before we can recognize the language from a given text, we have to generate a language profile for each language. From the beginning it comes with a pre-trained language profile (<a href="etc/_langs.json"><code>etc/_langs.json</code></a>).&lt;br&gt; Also you can add new files to <a href="etc"><code>etc</code></a> or change existing ones.</p> 
            <p>First we have to generate a language profile.</p> 
            <pre><code class="language-php">require_once 'vendor/autoload.php';
 
use LanguageDetector\Trainer;
 
$t = new Trainer;
 
$t-&gt;learn();
</code></pre>
            <p></p> 
            <p>If we have our language profile, we can classify texts by their language. To detect the language correctly, the length of the input text should be at least some sentences.</p> 
            <pre><code class="language-php">require_once 'vendor/autoload.php';
 
use LanguageDetector\LanguageDetector;
 
$ld = new LanguageDetector;
 
var_dump($ld-&gt;detect('Das ist ein deutscher Satz.')); // de
</code></pre>
            <p></p>
