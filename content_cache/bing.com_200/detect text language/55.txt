       <h1 class="entry-title">Detect Text Language – Java&nbsp;Program</h1> 
       <div class="entry-content clear"> 
        <p>In Natural Language Processing many time we need to identify language of given text beore processing ,Apache tika make it simple</p> 
        <p>1. &nbsp;Apche tika -library require in classpath</p> 
        <p>2.Some of the language identified by tika are</p> 
        <p><strong>en</strong>-English<br> <strong>es</strong>-Spanish<br> <strong>fr</strong>-French<br> <strong>da</strong>-Danish<br> <strong>de</strong>-German</p> 
        <p>[ro, ca, no, hu, lt, th, de, fi, sv, fr, be, sl, sk, uk, da, is, it, gl, el, pl, pt, eo, en, ru, et, es, nl]</p> 
        <p>3. If language which your code need to identify is not supported &nbsp;you can train model .</p> 
        <pre class="brush: java; title: ; notranslate" title="">
import org.apache.tika.language.LanguageIdentifier;

/**
 *
 * @author 396460
 */
public class LanguageId
{
 /**
 * Detect language of given text and return its full name
 * @param content
 * @return name of language [ro, ca, no, hu, lt, th, de, fi, sv, fr, be, sl, sk, uk, da, is, it, gl, el, pl, pt, eo, en, ru, et, es, nl]
 */
 public static String detectLang(String content)
 {
 LanguageIdentifier identifier=new LanguageIdentifier(content);
 String lang=identifier.getLanguage();
 if(lang.equals("en"))
 {
 return "English";
 }
 else if(lang.equals("es"))
 {
 return "Spanish";
 }
 else
 {
 return lang;
 }
 }

}
