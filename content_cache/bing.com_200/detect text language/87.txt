        <h2>Project Description</h2> NTextCat is a text classification utility (tool and API).
        <br> The primary target is 
        <b>language identification</b>. So it helps you to recognize (identify) the language of a given text snippet.
        <br> 
        <br> 
        <div style="clear:both; height:0">
         &nbsp;
        </div> 
        <a href="https://www.paypal.com/cgi-bin/webscr?cmd=_donations&amp;business=AML4FBLBWZKRL&amp;lc=GB&amp;item_name=NTextCat%20free%20opensource%20language%20identification%20API&amp;item_number=NTextCat&amp;currency_code=EUR&amp;bn=PP%2dDonationsBF%3abtn_donateCC_LG_global%2egif%3aNonHosted" style="float:right; padding-left:.5em"><img src="https://www.paypalobjects.com/en_US/i/btn/btn_donate_LG.gif" alt="Sponsor features you wish to have." title="Sponsor features you wish to have." style="border:none"></a> 
        <a href="http://ntextcat.com/">ONLINE DEMO</a>
        <br> 
        <h2>Languages available out of the box: </h2> 
        <ul> 
         <li>Wikipedia-based: 
          <ul> 
           <li><i>Default:</i> <b>280+ languages</b> (and flavors) encoded in UTF-8, UTF-16(LE), UTF-32(LE). Additionally 83 most popular lanugages encoded in their respective "legacy" encodings (e.g. 1252, Big5, etc.). Directory: <i>Wikipedia-MostCommon-Legacy__All-Utf8</i> </li>
           <li><b>280+ languages</b> (and flavors) encoded in UTF-8 only. Directory: <i>Wikipedia-All-Utf8</i> </li>
           <li>83 most popular languages encoded in UTF-8, UTF-16(LE), UTF-32(LE) and their respective "legacy" encodings (e.g. 1252, Big5, etc.). Directory: <i>Wikipedia-MostCommon-LegacyAndUtf8</i> </li>
           <li>83 most popular languages encoded in UTF-8 only. Directory: <i>Wikipedia-MostCommon-Utf8</i></li>
          </ul> </li>
         <li><b>74 languages</b> from original <i>TextCat</i> tool (<a href="http://odur.let.rug.nl/~vannoord/TextCat/list.html">full list</a>). Directory: <i>TextCat\LM</i></li>
        </ul> 
        <br> Recommended input: a snippet of text with at least 
        <b>5 words</b> (though it works quite OK with just a couple of words). 
        <a href="http://ntextcat.com/">Try it out yourself.</a>
        <br>

        <h2>Features:</h2> 
        <ol> 
         <li>.Net Framework 4.0 support </li>
         <li>Compatible with <b>Mono 2.10.x</b>). Mono 2.10.5 is shipped with Ubuntu 11.10 and Mono 2.10.8.1 with Ubuntu 12.04 and 12.10 (Linux expert's reviews are highly appreciated). <a href="http://ntextcat.codeplex.com/wikipage?title=Mono&amp;referringTitle=Home">More info on Mono support.</a> </li>
         <li>Pure .Net application (C#) </li>
         <li>Command line interface is available. <i>Mono.Options</i> was used. </li>
         <li>SQL Server 2012 integration via user-defined functions.</li>
        </ol> 
        <h2>Roadmap </h2> 
        <i>(if you need particular items, please <b>vote</b> for them or <b>discuss</b> them)</i>: 
        <ol> 
         <li><a href="http://ntextcat.codeplex.com/workitem/1360">Driver Mode (stdout-CLI for other applications)?</a> -- command line interface for communication with other programming environments via stdout (for Java, C/C++, Python, etc.) </li>
         <li><a href="http://ntextcat.codeplex.com/workitem/1361">Web Service API?</a> (JSON/XML based, HTTP based).</li>
        </ol> Low Priority: 
        <ol> 
         <li><a href="http://ntextcat.codeplex.com/workitem/1359">Silverlight Support?</a> </li>
         <li>Driver mode client (description above) for Java and C/C++ (help needed). </li>
         <li>Web service client for Java (help needed).</li>
        </ol> 
        <h2>Interface available:</h2> 
        <ul> 
         <li>A <b>console application</b> which is capable of training (creating language models) and recognizing which language a snippet of text belongs to. </li>
        </ul> 
        <a href="http://ntextcat.codeplex.com/wikipage?title=How%20to%20identify%20language%20from%20command%20line&amp;referringTitle=Home">How to identify language from command line</a> 
        <ul> 
         <li>A <b>library</b> that you can reference from your application to empower it with language identification capabilities. </li>
        </ul> 
        <a href="http://ntextcat.codeplex.com/wikipage?title=NTextCat.Lib%20Samples&amp;referringTitle=Home">How to identify language using managed API</a>

    function displayRequestAccess() {

    function requestAccess(e) {
        var error = $('#requestAccessError').hide();
        var comment = $('#requestAccessComment').val();
        if ($normalizeText(comment).length === 0) {
            error.text('* Please provide your comments for the project coordinators.').show();
            e.stopImmediatePropagation();
            return false;
        }

        CodePlex.UpdateProgress.show();
        $.post('http://ntextcat.codeplex.com/team/requestAccess',
               { comment: $('#requestAccessComment').val() },
               function(data) {
                   CodePlex.UpdateProgress.show(false);
                   if (data.RedirectUrl && data.RedirectUrl.length) location.href = data.RedirectUrl;
                   else if (data.IsError) {
                       $('#requestAccessError').text(data.Message).show();
                   } else {
                       $('#joinContainer').empty()
                                            .addClass('SuccessMessage')
                                            .html(data.Message)
                                            .show().fadeOut(5000, function() {
                                                $(this).empty()
                                                       .removeClass('SuccessMessage')
                                                       .text('Request to join project is pending')
                                                       .show();
                                            });
                       OpenDialog('#requestAccessContainer', false);
                   }
               }, 'json');
    }

    function cancelRequestAccess() {
        OpenDialog('#requestAccessContainer', false);
        $('#requestAccessError').hide();
        $('#requestAccessComment').val('');
        $('#charCounter').text('250');
    }
