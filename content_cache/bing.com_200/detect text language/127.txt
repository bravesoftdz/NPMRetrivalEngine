  <div class="text-column"> 
   <div id="about" class="text"> 
    <div class="startnow"> 
     <a href="download.php">Download</a> 
    </div> MALLET is a Java-based package for statistical natural language processing, document classification, clustering, topic modeling, information extraction, and other machine learning applications to text. 
   </div> 
   <div id="classification" class="text">
     MALLET includes sophisticated tools for 
    <b>document classification</b>: efficient routines for converting text to "features", a wide variety of algorithms (including Naïve Bayes, Maximum Entropy, and Decision Trees), and code for evaluating classifier performance using several commonly used metrics. [
    <a href="classification.php">Quick Start</a>] [
    <a href="classifier-devel.php">Developer's Guide</a>] 
   </div> 
   <div id="sequence" class="text">
     In addition to classification, MALLET includes tools for 
    <b>sequence tagging</b> for applications such as named-entity extraction from text. Algorithms include Hidden Markov Models, Maximum Entropy Markov Models, and Conditional Random Fields. These methods are implemented in an extensible system for finite state transducers. [
    <a href="sequences.php">Quick Start</a>] [
    <a href="fst.php">Developer's Guide</a>] 
   </div> 
   <div id="topics" class="text">
     Topic models are useful for analyzing large collections of unlabeled text. The MALLET 
    <b>topic modeling</b> toolkit contains efficient, sampling-based implementations of Latent Dirichlet Allocation, Pachinko Allocation, and Hierarchical LDA. [
    <a href="topics.php">Quick Start</a>] 
    <!-- [<a href="">Developer's Guide</a>] --> 
   </div> 
   <div id="optimization" class="text">
     Many of the algorithms in MALLET depend on 
    <b>numerical optimization</b>. MALLET includes an efficient implementation of Limited Memory BFGS, among many other optimization methods. [
    <a href="optimization.php">Developer's Guide</a>] 
   </div> 
   <div id="pipes" class="text">
     In addition to sophisticated Machine Learning applications, MALLET includes routines for transforming text documents into numerical representations that can then be processed efficiently. This process is implemented through a flexible system of "pipes", which handle distinct tasks such as tokenizing strings, removing stopwords, and converting sequences into count vectors. [
    <a href="import.php">Quick Start</a>] [
    <a href="import-devel.php">Developer's Guide</a>] 
   </div> 
   <div id="grmm" class="text">
     An add-on package to MALLET, called GRMM, contains support for inference in general graphical models, and training of CRFs with arbitrary graphical structure. [
    <a href="grmm/index.php">About GRMM</a>] 
   </div>
