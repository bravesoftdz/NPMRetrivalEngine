       <h1 class="entry-title">Day 12: OpenCV — Face Detection for Java Developers</h1> 
       <time class="published" datetime="2013-11-09T00:00:00+00:00">November 9, 2013</time> 
       <p class="byline author vcard">By <a href="https://blog.openshift.com/author/shekhargulati/" rel="author" class="fn">Shekhar Gulati</a></p> 
       <script>
      dataLayer = dataLayer || [];
      dataLayer.push({
        'author': "Shekhar Gulati",
        'datePublished': "2013-11-09"
      });
      </script> 
       <p class="addthis_toolbox addthis_default_style" addthis:url="https://blog.openshift.com/day-12-opencv-face-detection-for-java-developers/" addthis:title="Day 12: OpenCV — Face Detection for Java Developers"> <a class="addthis_button_tweet" data-via="openshift" tw:count="none"></a> <a class="addthis_button_google_plusone" g:plusone:count="false" g:plusone:size="medium"></a> <a class="addthis_button_facebook_like" fb:like:layout="button"></a> <a class="addthis_counter addthis_pill_style"></a> </p> 
      </header> 
      <div class="entry-content"> 
       <p>Today for <a href="/learning-30-technologies-in-30-days-a-developer-challenge">my 30 day challenge</a>, I decided to learn how to do <a href="http://en.wikipedia.org/wiki/Face_detection">face detection</a> using the Java programming language. Face detection helps in recognizing human faces in arbitrary (digital) images. After doing some research, I discovered that the <a href="http://opencv.org/">OpenCV</a> library can help me detect faces in an image. However, I wasn’t able to find a complete beginner tutorial for using OpenCV library with Java, so this blog might help others looking for an introductory text on this subject.</p> 
       <p><img src="/wp-content/uploads/imported/opencv.png" alt="OpenCV logo" style="display: block;margin-left: auto;margin-right: auto"></p> 
       <h2>What is OpenCV?</h2> 
       <p><a href="http://opencv.org/">OpenCV(Open Source Computer Vision)</a> is an open source computer vision algorithms library. It is written in C/C++, and is designed to take advantage of multiple cores. It provides interfaces for the C++, C, Python and Java programming languages and supports all the major operating system platforms including Windows, Linux, Mac OS, iOS, and Android.</p> 
       <h2>Github Repository</h2> 
       <p>The code for today’s demo application is available on <a href="https://github.com/shekhargulati/day12-face-detection">github: day12-face-detection</a>.</p> 
       <h2>Getting Started with OpenCV</h2> 
       <p>In order to get started with OpenCV, the first step is to download the latest OpenCV package for your operating system from the <a href="http://sourceforge.net/projects/opencvlibrary/">official website</a>. In this blog, I will be using the 2.4.7 version.</p> 
       <p>After downloading the package, extract the package using tar command.</p> 
       <div class="geshifilter"> 
        <pre class="text geshifilter-text" style="font-family:monospace;">$ tar xvf opencv-2.4.7.tar.gz</pre> 
       </div> 
       <p>Change to the directory to opencv-2.4.7</p> 
       <div class="geshifilter"> 
        <pre class="text geshifilter-text" style="font-family:monospace;">$ cd opencv-2.4.7</pre> 
       </div> 
       <h2>Build OpenCV jar</h2> 
       <p>It took me lot of time to understand how to get the OpenCV jar file. The Java tutorial in the documentation assumes the OpenCV jar file is in the build folder. The tutorial works for windows users as the OpenCV package for windows includes the jar file. This is not true for Linux and Mac OS users. To build the OpenCV jar, executes the command shown below.</p> 
       <div class="geshifilter"> 
        <pre class="text geshifilter-text" style="font-family:monospace;">$ cd opencv-2.4.7
&nbsp;
$ mkdir build
&nbsp;
$ cd build/
&nbsp;
$ cmake -G "Unix Makefiles" -D CMAKE_CXX_COMPILER=/usr/bin/g++ -D CMAKE_C_COMPILER=/usr/bin/gcc -D WITH_CUDA=ON .. 
&nbsp;
$make -j4 
&nbsp;
$ make install</pre> 
       </div> 
       <p>The above commands will create the <em>opencv-247.jar</em> file in the <em>opencv-2.4.7/build/bin</em> directory. This is the Java binding to the native OpenCV installation.</p> 
       <h2>Download Eclipse</h2> 
       <p>If you do not have Eclipse installed on your machine, download the latest Eclipse package for your operating system from the <a href="http://www.eclipse.org/downloads/">official Eclipse website</a>. At the time of writing this blog, the latest Eclipse package is called Kepler.<br> <img src="/wp-content/uploads/imported/Download_Eclipse.png" height="500" width="600"><br> It is very easy to install eclipse, just extract the downloaded package and you are done. On linux or mac machines, open a new command line terminal and type command shown below.</p> 
       <div class="geshifilter"> 
        <pre class="text geshifilter-text" style="font-family:monospace;">$ tar -xzvf eclipse-jee-kepler-R-*.tar.gz </pre> 
       </div> 
       <p>On windows you can extract the zip file using winzip or <a href="http://www.7-zip.org/download.html">7-zip</a> or any other software.<br> After you have extracted Eclipse, there will be a folder named <em>eclipse</em> in the directory where you extracted the zip file. You can optionally create a shortcut of the executable file.</p> 
       <h2>Add the User Library</h2> 
       <p>Open the Eclipse IDE and then navigate to the project workspace. Go to <em>Windows &gt; Preferences &gt; Java &gt; Build Path &gt; User Libraries</em>, and select to add a new library.</p> 
       <p><img src="/wp-content/uploads/imported/Add_New_User_Libraries.png" alt="OpenCV Add New User Libraries" style="display: block;margin-left: auto;margin-right: auto" width="450" height="450"></p> 
       <p>Give the user library a name like <em>OpenCV-2.4.7</em> and then press OK.</p> 
       <p><img src="/wp-content/uploads/imported/OpenCV-2.4.7_User_Library.png" alt="OpenCV User Libraries" style="display: block;margin-left: auto;margin-right: auto" width="450" height="450"></p> 
       <p>Click on <strong><em>Add External Jars</em></strong>, and add the <em>opencv-247.jar</em> file.</p> 
       <p><img src="/wp-content/uploads/imported/Add_External_Jar_OpenCV_2.4.7.png" alt="Add External Jar" style="display: block;margin-left: auto;margin-right: auto" width="450" height="300"></p> 
       <p>Select the <strong><em>Native library location</em></strong> and then press <strong><em>Edit</em></strong>.</p> 
       <p><img src="/wp-content/uploads/imported/Edit_Native_Library.png" alt="Edit Native Library" style="display: block;margin-left: auto;margin-right: auto" width="400" height="300"></p> 
       <p>Click on <strong><em>External Folder</em></strong></p> 
       <p><img src="/wp-content/uploads/imported/Add_External_Folder_OpenCV_2.4.7.png" alt="Add External Library" style="display: block;margin-left: auto;margin-right: auto" width="400" height="300"></p> 
       <p>Give the location of lib directory under the <em>opencv-2.4.7/build/lib</em> folder.</p> 
       <p><img src="/wp-content/uploads/imported/Lib_directory.png" alt="Add Lib directory" style="display: block;margin-left: auto;margin-right: auto" width="400" height="300"></p> 
       <p>Finally, press <strong><em>OK</em></strong>. Now we have added OpenCV as the user library.</p> 
       <h2>Create New Java Project</h2> 
       <p>Create a new Java project by going to <em>File &gt; New &gt; Other &gt; Java Project</em>. After the project is created, right click on the project and configure the build path.</p> 
       <p><img src="/wp-content/uploads/imported/Project_Configure_Build_Path.png" alt="Configure Project Build Path" style="display: block;margin-left: auto;margin-right: auto" width="400" height="300"></p> 
       <p>Go to the Libraries tab, and click on <strong><em>Add Library</em></strong>.</p> 
       <p><img src="/wp-content/uploads/imported/Add_Library.png" alt="Add Lib directory" style="display: block;margin-left: auto;margin-right: auto" width="400" height="300"></p> 
       <p>Select the <strong><em>User Library</em></strong></p> 
       <p><img src="/wp-content/uploads/imported/Select_User_Library.png" alt="Select User library" style="display: block;margin-left: auto;margin-right: auto" width="400" height="300"></p> 
       <p>Choose the OpenCV-2.4.7 user library we added in last step and click <strong><em>Finish</em></strong>.</p> 
       <p><img src="/wp-content/uploads/imported/Select_OpenCV-2.4.7_User_Library.png" alt="Select OpenCV User Library" style="display: block;margin-left: auto;margin-right: auto" width="400" height="300"></p> 
       <p>Finally, the Java project will include the OpenCV-2.4.7 user library.</p> 
       <p><img src="/wp-content/uploads/imported/Project_with_OpenCV_user_library.png" alt="Project with OpenCV user library" style="display: block;margin-left: auto;margin-right: auto" width="300" height="250"></p> 
       <h2>Write FaceDetector</h2> 
       <p>Create a new class in the java project we created in the last step and add the following code.</p> 
       <div class="geshifilter"> 
        <pre class="text geshifilter-text" style="font-family:monospace;">package com.shekhar.facedetection;
&nbsp;
import org.opencv.core.Core;
import org.opencv.core.Mat;
import org.opencv.core.MatOfRect;
import org.opencv.core.Point;
import org.opencv.core.Rect;
import org.opencv.core.Scalar;
import org.opencv.highgui.Highgui;
import org.opencv.objdetect.CascadeClassifier;
&nbsp;
public class FaceDetector {
&nbsp;
    public static void main(String[] args) {
&nbsp;
        System.loadLibrary(Core.NATIVE_LIBRARY_NAME);
        System.out.println("\nRunning FaceDetector");
&nbsp;
        CascadeClassifier faceDetector = new CascadeClassifier(FaceDetector.class.getResource("haarcascade_frontalface_alt.xml").getPath());
        Mat image = Highgui
                .imread(FaceDetector.class.getResource("shekhar.JPG").getPath());
&nbsp;
        MatOfRect faceDetections = new MatOfRect();
        faceDetector.detectMultiScale(image, faceDetections);
&nbsp;
        System.out.println(String.format("Detected %s faces", faceDetections.toArray().length));
&nbsp;
        for (Rect rect : faceDetections.toArray()) {
            Core.rectangle(image, new Point(rect.x, rect.y), new Point(rect.x + rect.width, rect.y + rect.height),
                    new Scalar(0, 255, 0));
        }
&nbsp;
        String filename = "ouput.png";
        System.out.println(String.format("Writing %s", filename));
        Highgui.imwrite(filename, image);
    }
}</pre> 
       </div> 
       <p>The code shown above does the following :</p> 
       <ol> 
        <li>It loads the native OpenCV library so that we can use it using Java API.<p></p> </li> 
        <li> <p>We create an instance of CascadeClassifier passing it the name of the file from which the classifier is loaded.</p> </li> 
        <li> <p>Next we convert the image to a format which the Java API will accept using the Highui class. Mat is the OpenCV C++ n-dimensional dense array class.</p> </li> 
        <li> <p>Then we call the <em>detectMultiScale</em> method on the classifier passing it the image and MatOfRect object. After processing, the MatOfRect will have face detections.</p> </li> 
        <li> <p>We iterate over all the face detections and mark the image with rectangles.</p> </li> 
        <li> <p>Finally, we write the image to the <em>output.png</em> file.</p> </li> 
       </ol> 
       <p>The output of the program is shown below. This is my pic before and after face detection.</p> 
       <p><img src="/wp-content/uploads/imported/combine_images.jpg" alt="OpenCV Output" style="display: block;margin-left: auto;margin-right: auto" width="450" height="250"></p> 
       <p>That’s it for today. Keep giving feedback.</p>
