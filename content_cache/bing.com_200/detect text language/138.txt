       <h1><span class="editsection">[<a href="/learn-scripts/index.php?title=International_language_support&amp;action=edit&amp;section=1" title="Edit section: Introduction">edit</a>]</span> <span class="mw-headline" id="Introduction">Introduction</span></h1> 
       <p>If you are a non-English speaker, <span class="texhtml" style="font-family: 'CMU Serif', cmr10, LMRoman10-Regular, 'Times New Roman', 'Nimbus Roman No9 L', Times, serif;">L<span style="text-transform: uppercase; font-size: 70%; margin-left: -0.36em; vertical-align: 0.3em; line-height: 0; margin-right: -0.15em;">a</span>T<span style="text-transform: uppercase; margin-left: -0.1667em; vertical-align: -0.5ex; line-height: 0; margin-right: -0.125em;">e</span>X</span> can be configured to typeset in your language. Below is an example of a document in Spanish: </p> 
       <p> </p>
       <div class="example"> 
        <div class="code"> 
         <div dir="ltr" class="mw-geshi mw-code mw-content-ltr">
          <div class="latex source-latex">

<span class="re8">\begin</span><span class="sy0">{</span><span class="re7">document</span><span class="sy0">}</span>
&nbsp;

&nbsp;

       <h1><span class="editsection">[<a href="/learn-scripts/index.php?title=International_language_support&amp;action=edit&amp;section=2" title="Edit section: Input encoding">edit</a>]</span> <span class="mw-headline" id="Input_encoding">Input encoding</span></h1> 
       <p>Modern computer systems allow you to input letters of national alphabets directly from the keyboard. In order to handle a variety of input encodings used for different groups of languages and/or on different computer platforms <span class="texhtml" style="font-family: 'CMU Serif', cmr10, LMRoman10-Regular, 'Times New Roman', 'Nimbus Roman No9 L', Times, serif;">L<span style="text-transform: uppercase; font-size: 70%; margin-left: -0.36em; vertical-align: 0.3em; line-height: 0; margin-right: -0.15em;">a</span>T<span style="text-transform: uppercase; margin-left: -0.1667em; vertical-align: -0.5ex; line-height: 0; margin-right: -0.125em;">e</span>X</span> employs the <b><code>inputenc</code></b> package to set up input encoding. To use this package add the next line to the preamble of your document: </p> 
       <div dir="ltr" class="mw-geshi mw-code mw-content-ltr">
        <div class="latex source-latex">
         <pre class="de1"><span class="sy0">\</span><span class="kw1">usepackage</span><span class="sy0">[</span><span class="re2">encoding</span><span class="sy0">]{</span><span class="re9">inputenc</span><span class="sy0">}</span></pre>
        </div>
       </div> 
       <p><br> The recommended input encoding is <b><code>utf8</code></b>, which supports a lot of national alphabets letters (Inside the brackets, instead of the word "encoding" you must put the name of the encoding you are using). If you want, you can also use other encodings connected with different groups of languages and/or on different computer platforms (see table below). </p> 
       <table class="wikitable"> 
        <tbody>
         <tr> 
          <th>Operating system </th> 
          <th> Western European Latin encoding </th> 
          <th> Central European Latin encoding </th> 
          <th> Cyrillic encoding </th>
         </tr> 
         <tr> 
          <td> Windows </td> 
          <td> cp1252 </td> 
          <td> cp1250 </td> 
          <td> cp1251 </td>
         </tr> 
         <tr> 
          <td> GNU/Linux &amp; Unix-like (*BSD, Mac OS X) </td> 
          <td> latin1 </td> 
          <td> latin2 </td> 
          <td> koi8-ru </td>
         </tr> 
         <tr> 
          <td> Recommended for all systems </td> 
          <td> utf8 </td> 
          <td> utf8 </td> 
          <td> utf8 </td>
         </tr>
        </tbody>
       </table> 
       <p>Note: <i>If you can't input some letters of national alphabets directly from the keyboard, you can use <span class="texhtml" style="font-family: 'CMU Serif', cmr10, LMRoman10-Regular, 'Times New Roman', 'Nimbus Roman No9 L', Times, serif;">L<span style="text-transform: uppercase; font-size: 70%; margin-left: -0.36em; vertical-align: 0.3em; line-height: 0; margin-right: -0.15em;">a</span>T<span style="text-transform: uppercase; margin-left: -0.1667em; vertical-align: -0.5ex; line-height: 0; margin-right: -0.125em;">e</span>X</span> alternative commands for accents and special characters. See the <a href="#Reference_guide">reference guide</a>.</i> </p>

       <h1><span class="editsection">[<a href="/learn-scripts/index.php?title=International_language_support&amp;action=edit&amp;section=3" title="Edit section: Font encoding">edit</a>]</span> <span class="mw-headline" id="Font_encoding">Font encoding</span></h1> 
       <p>To proper <span class="texhtml" style="font-family: 'CMU Serif', cmr10, LMRoman10-Regular, 'Times New Roman', 'Nimbus Roman No9 L', Times, serif;">L<span style="text-transform: uppercase; font-size: 70%; margin-left: -0.36em; vertical-align: 0.3em; line-height: 0; margin-right: -0.15em;">a</span>T<span style="text-transform: uppercase; margin-left: -0.1667em; vertical-align: -0.5ex; line-height: 0; margin-right: -0.125em;">e</span>X</span> document generation you must also choose a font which has to support specific characters for a given language by using <b><code>fontenc</code></b> package: </p> 
       <div dir="ltr" class="mw-geshi mw-code mw-content-ltr">
        <div class="latex source-latex">
         <pre class="de1"> <span class="sy0">\</span><span class="kw1">usepackage</span><span class="sy0">[</span><span class="re2">encoding</span><span class="sy0">]{</span><span class="re9">fontenc</span><span class="sy0">}</span></pre>
        </div>
       </div> 
       <p><br> The default <span class="texhtml" style="font-family: 'CMU Serif', cmr10, LMRoman10-Regular, 'Times New Roman', 'Nimbus Roman No9 L', Times, serif;">L<span style="text-transform: uppercase; font-size: 70%; margin-left: -0.36em; vertical-align: 0.3em; line-height: 0; margin-right: -0.15em;">a</span>T<span style="text-transform: uppercase; margin-left: -0.1667em; vertical-align: -0.5ex; line-height: 0; margin-right: -0.125em;">e</span>X</span> font encoding is <code>OT1</code>, but it contains only the 128 characters. The <b><code>T1</code></b> encoding contains letters and punctuation characters for most of the European languages using Latin script. For languages using Cyrillic script you can use T2A, T2B, T2C, or X2 font encodings. </p>
       <p><a rel="nofollow" class="external text" href="https://www.sharelatex.com/project/new/template?zipUrl=/project/5304cefb6a6237452e006b5e/download/zip&amp;templateName=ArabicExample&amp;compiler=pdflatex"><i class="fa fa-share"></i>&nbsp;&nbsp;Open an example of the pgfplots package in ShareLaTeX</a> </p> 
       <h1><span class="editsection">[<a href="/learn-scripts/index.php?title=International_language_support&amp;action=edit&amp;section=4" title="Edit section: Babel">edit</a>]</span> <span class="mw-headline" id="Babel">Babel</span></h1> 
       <p>The Babel package presented in the <a href="#Introduction">introduction</a> allows to use special characters and also translates some elements within the document. This package also automatically activates the appropriate hyphenation rules for the language you choose. </p>
       <p>You can activate the babel package by adding the next command to the preamble: </p> 
       <div dir="ltr" class="mw-geshi mw-code mw-content-ltr">
        <div class="latex source-latex">
         <pre class="de1"><span class="sy0">\</span><span class="kw1">usepackage</span><span class="sy0">[</span><span class="re2">language</span><span class="sy0">]{</span><span class="re9">babel</span><span class="sy0">}</span></pre>
        </div>
       </div> 
       <p><br> Change the <code>language</code> to the name of the language you need. </p>
       <p>You can see list of the languages built into your LaTeX system every time the compiler is started. In the .log file you can find something similar to this: </p>
