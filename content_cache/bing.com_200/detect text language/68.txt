       <h1>Goslate: Free Google Translate API<a class="headerlink" href="#goslate-free-google-translate-api" title="Permalink to this headline">¶</a></h1> 
       <div class="admonition note"> 
        <p class="first admonition-title">Note</p> 
        <p class="last">Google has updated its translation service recently with a ticket mechanism to prevent simple crawler program like <code class="docutils literal"><span class="pre">goslate</span></code> from accessing. Though a more sophisticated crawler may still work technically, however it would have crossed the fine line between using the service and breaking the service. <code class="docutils literal"><span class="pre">goslate</span></code> will not be updated to break google’s ticket mechanism. Free lunch is over. Thanks for using.</p> 
       </div>

       <p><code class="docutils literal"><span class="pre">goslate</span></code> provides you <em>free</em> python API to google translation service by querying google translation website.</p> 
       <p>It is:</p> 
       <ul class="simple"> 
        <li><strong>Free</strong>: get translation through public google web site without fee</li> 
        <li><strong>Fast</strong>: batch, cache and concurrently fetch</li> 
        <li><strong>Simple</strong>: single file module, just <code class="docutils literal"><span class="pre">Goslate().translate('Hi!',</span> <span class="pre">'zh')</span></code></li> 
       </ul> 
       <div class="section" id="simple-usage"> 
        <h2><a class="toc-backref" href="#id7">Simple Usage</a><a class="headerlink" href="#simple-usage" title="Permalink to this headline">¶</a></h2> 
        <p>The basic usage is simple:</p> 
        <div class="highlight-python">
         <div class="highlight">
          <pre><span class="gp">&gt;&gt;&gt; </span><span class="kn">import</span> <span class="nn">goslate</span>
<span class="gp">&gt;&gt;&gt; </span><span class="n">gs</span> <span class="o">=</span> <span class="n">goslate</span><span class="o">.</span><span class="n">Goslate</span><span class="p">()</span>
<span class="gp">&gt;&gt;&gt; </span><span class="k">print</span><span class="p">(</span><span class="n">gs</span><span class="o">.</span><span class="n">translate</span><span class="p">(</span><span class="s">'hello world'</span><span class="p">,</span> <span class="s">'de'</span><span class="p">))</span>
<span class="go">hallo welt</span>
</pre>
         </div> 
        </div>

       <div class="section" id="installation"> 
        <h2><a class="toc-backref" href="#id8">Installation</a><a class="headerlink" href="#installation" title="Permalink to this headline">¶</a></h2> 
        <p>goslate support both Python2 and Python3. You could install it via:</p> 
        <div class="highlight-bash">
         <div class="highlight">
          <pre><span class="nv">$ </span>pip install goslate
</pre>
         </div> 
        </div> 
        <p>or just download <a class="reference external" href="https://bitbucket.org/zhuoqiang/goslate/raw/tip/goslate.py">latest goslate.py</a> directly and use</p> 
        <p><code class="docutils literal"><span class="pre">futures</span></code> <a class="reference external" href="https://pypi.python.org/pypi/futures">pacakge</a> is optional but recommended to install for best performance in large text translation task.</p> 
       </div> 
       <div class="section" id="proxy-support">

        <p>Proxy support could be added as following:</p> 
        <div class="highlight-python">
         <div class="highlight">
          <pre><span class="kn">import</span> <span class="nn">urllib2</span>
<span class="kn">import</span> <span class="nn">goslate</span>

<span class="n">proxy_handler</span> <span class="o">=</span> <span class="n">urllib2</span><span class="o">.</span><span class="n">ProxyHandler</span><span class="p">({</span><span class="s">"http"</span> <span class="p">:</span> <span class="s">"http://proxy-domain.name:8080"</span><span class="p">})</span>
<span class="n">proxy_opener</span> <span class="o">=</span> <span class="n">urllib2</span><span class="o">.</span><span class="n">build_opener</span><span class="p">(</span><span class="n">urllib2</span><span class="o">.</span><span class="n">HTTPHandler</span><span class="p">(</span><span class="n">proxy_handler</span><span class="p">),</span>
                                    <span class="n">urllib2</span><span class="o">.</span><span class="n">HTTPSHandler</span><span class="p">(</span><span class="n">proxy_handler</span><span class="p">))</span>

<span class="n">gs_with_proxy</span> <span class="o">=</span> <span class="n">goslate</span><span class="o">.</span><span class="n">Goslate</span><span class="p">(</span><span class="n">opener</span><span class="o">=</span><span class="n">proxy_opener</span><span class="p">)</span>
<span class="n">translation</span> <span class="o">=</span> <span class="n">gs_with_proxy</span><span class="o">.</span><span class="n">translate</span><span class="p">(</span><span class="s">"hello world"</span><span class="p">,</span> <span class="s">"de"</span><span class="p">)</span>

       <div class="section" id="romanlization"> 
        <h2><a class="toc-backref" href="#id10">Romanlization</a><a class="headerlink" href="#romanlization" title="Permalink to this headline">¶</a></h2> 
        <p>Romanization or latinization (or romanisation, latinisation), in linguistics, is the conversion of writing from a different writing system to the Roman (Latin) script, or a system for doing so.</p> 
        <p>For example, pinyin is the default romanlization method for Chinese language.</p> 
        <p>You could get translation in romanlized writing as following:</p> 
        <div class="highlight-python">
         <div class="highlight">
          <pre><span class="gp">&gt;&gt;&gt; </span><span class="kn">import</span> <span class="nn">goslate</span>
<span class="gp">&gt;&gt;&gt; </span><span class="n">roman_gs</span> <span class="o">=</span> <span class="n">goslate</span><span class="o">.</span><span class="n">Goslate</span><span class="p">(</span><span class="n">writing</span><span class="o">=</span><span class="n">goslate</span><span class="o">.</span><span class="n">WRITING_ROMAN</span><span class="p">)</span>
<span class="gp">&gt;&gt;&gt; </span><span class="k">print</span><span class="p">(</span><span class="n">roman_gs</span><span class="o">.</span><span class="n">translate</span><span class="p">(</span><span class="s">'China'</span><span class="p">,</span> <span class="s">'zh'</span><span class="p">))</span>
<span class="go">Zh?ngguó</span>
</pre>
         </div> 
        </div> 
        <p>You could also get translation in both native writing system and ramon writing system</p> 
        <div class="highlight-python">
         <div class="highlight">
          <pre><span class="gp">&gt;&gt;&gt; </span><span class="kn">import</span> <span class="nn">goslate</span>
<span class="gp">&gt;&gt;&gt; </span><span class="n">gs</span> <span class="o">=</span> <span class="n">goslate</span><span class="o">.</span><span class="n">Goslate</span><span class="p">(</span><span class="n">writing</span><span class="o">=</span><span class="n">goslate</span><span class="o">.</span><span class="n">WRITING_NATIVE_AND_ROMAN</span><span class="p">)</span>
<span class="gp">&gt;&gt;&gt; </span><span class="n">gs</span><span class="o">.</span><span class="n">translate</span><span class="p">(</span><span class="s">'China'</span><span class="p">,</span> <span class="s">'zh'</span><span class="p">)</span>
<span class="go">('??', 'Zh?ngguó')</span>
</pre>
         </div> 
        </div> 
        <p>You could see the result will be a tuple in this case: <code class="docutils literal"><span class="pre">(Translation-in-Native-Writing,</span> <span class="pre">Translation-in-Roman-Writing)</span></code></p> 
       </div>

<span class="gp">&gt;&gt;&gt; </span><span class="n">gs</span> <span class="o">=</span> <span class="n">goslate</span><span class="o">.</span><span class="n">Goslate</span><span class="p">()</span>
<span class="gp">&gt;&gt;&gt; </span><span class="n">language_id</span> <span class="o">=</span> <span class="n">gs</span><span class="o">.</span><span class="n">detect</span><span class="p">(</span><span class="s">'hallo welt'</span><span class="p">)</span>
<span class="gp">&gt;&gt;&gt; </span><span class="n">language_id</span>
<span class="go">'de'</span>
<span class="gp">&gt;&gt;&gt; </span><span class="n">gs</span><span class="o">.</span><span class="n">get_languages</span><span class="p">()[</span><span class="n">language_id</span><span class="p">]</span>
<span class="go">'German'</span>
</pre>
         </div> 
        </div>

       <div class="section" id="concurrent-querying"> 
        <h2><a class="toc-backref" href="#id12">Concurrent Querying</a><a class="headerlink" href="#concurrent-querying" title="Permalink to this headline">¶</a></h2> 
        <p>It is not necessary to roll your own multi-thread solution to speed up massive translation. Goslate already done it for you. It utilizes <code class="docutils literal"><span class="pre">concurrent.futures</span></code> for concurent querying. The max worker number is 120 by default.</p> 
        <p>The worker number could be changed as following:</p> 
        <div class="highlight-python">
         <div class="highlight">
          <pre><span class="gp">&gt;&gt;&gt; </span><span class="kn">import</span> <span class="nn">golsate</span>
<span class="gp">&gt;&gt;&gt; </span><span class="kn">import</span> <span class="nn">concurrent.futures</span>
<span class="gp">&gt;&gt;&gt; </span><span class="n">executor</span> <span class="o">=</span> <span class="n">concurrent</span><span class="o">.</span><span class="n">futures</span><span class="o">.</span><span class="n">ThreadPoolExecutor</span><span class="p">(</span><span class="n">max_workers</span><span class="o">=</span><span class="mi">200</span><span class="p">)</span>
<span class="gp">&gt;&gt;&gt; </span><span class="n">gs</span> <span class="o">=</span> <span class="n">goslate</span><span class="o">.</span><span class="n">Goslate</span><span class="p">(</span><span class="n">executor</span><span class="o">=</span><span class="n">executor</span><span class="p">)</span>
<span class="gp">&gt;&gt;&gt; </span><span class="n">it</span> <span class="o">=</span> <span class="n">gs</span><span class="o">.</span><span class="n">translate</span><span class="p">([</span><span class="s">'text1'</span><span class="p">,</span> <span class="s">'text2'</span><span class="p">,</span> <span class="s">'text3'</span><span class="p">])</span>
<span class="gp">&gt;&gt;&gt; </span><span class="nb">list</span><span class="p">(</span><span class="n">it</span><span class="p">)</span>
<span class="go">['tranlation1', 'translation2', 'translation3']</span>
</pre>
         </div> 
        </div> 
        <p>It is adviced to install <code class="docutils literal"><span class="pre">concurrent.futures</span></code> backport lib in python2.7 (python3 has it by default) to enable concurrent querying.</p> 
        <p>The input could be list, tuple or any iterater, even the file object which iterate line by line</p> 
        <div class="highlight-python">
         <div class="highlight">
          <pre><span class="gp">&gt;&gt;&gt; </span><span class="n">translated_lines</span> <span class="o">=</span> <span class="n">gs</span><span class="o">.</span><span class="n">translate</span><span class="p">(</span><span class="nb">open</span><span class="p">(</span><span class="s">'readme.txt'</span><span class="p">))</span>
<span class="gp">&gt;&gt;&gt; </span><span class="n">translation</span> <span class="o">=</span> <span class="s">'</span><span class="se">\n</span><span class="s">'</span><span class="o">.</span><span class="n">join</span><span class="p">(</span><span class="n">translated_lines</span><span class="p">)</span>
</pre>
         </div> 
        </div> 
        <p>Do not worry about short texts will increase the query time. Internally, goslate will join small text into one big text to reduce the unnecessary query round trips.</p> 
       </div> 
       <div class="section" id="batch-translation"> 
        <h2><a class="toc-backref" href="#id13">Batch Translation</a><a class="headerlink" href="#batch-translation" title="Permalink to this headline">¶</a></h2> 
        <p>Google translation does not support very long text, goslate bypass this limitation by split the long text internally before send to Google and join the mutiple results into one translation text to the end user.</p> 
        <div class="highlight-python">
         <div class="highlight">
          <pre><span class="gp">&gt;&gt;&gt; </span><span class="kn">import</span> <span class="nn">golsate</span>
<span class="gp">&gt;&gt;&gt; </span><span class="k">with</span> <span class="nb">open</span><span class="p">(</span><span class="s">'the game of thrones.txt'</span><span class="p">,</span> <span class="s">'r'</span><span class="p">)</span> <span class="k">as</span> <span class="n">f</span><span class="p">:</span>
<span class="gp">&gt;&gt;&gt; </span>    <span class="n">novel_text</span> <span class="o">=</span> <span class="n">f</span><span class="o">.</span><span class="n">read</span><span class="p">()</span>
<span class="gp">&gt;&gt;&gt; </span><span class="n">gs</span> <span class="o">=</span> <span class="n">goslate</span><span class="o">.</span><span class="n">Goslate</span><span class="p">()</span>
<span class="gp">&gt;&gt;&gt; </span><span class="n">gs</span><span class="o">.</span><span class="n">translate</span><span class="p">(</span><span class="n">novel_text</span><span class="p">)</span>
</pre>
         </div> 
        </div> 
       </div> 
       <div class="section" id="performance-consideration"> 
        <h2><a class="toc-backref" href="#id14">Performance Consideration</a><a class="headerlink" href="#performance-consideration" title="Permalink to this headline">¶</a></h2> 
        <p>Goslate use batch and concurrent fetch aggresivelly to achieve maximized translation speed internally.</p> 
        <p>All you need to do is reducing API calling times by utilize batch tranlation and concurrent querying.</p> 
        <p>For example, say if you want to translate 3 big text files. Instead of manually translate them one by one, line by line:</p> 
        <div class="highlight-python">
         <div class="highlight">
          <pre><span class="kn">import</span> <span class="nn">golsate</span>

<span class="n">big_files</span> <span class="o">=</span> <span class="p">[</span><span class="s">'a.txt'</span><span class="p">,</span> <span class="s">'b.txt'</span><span class="p">,</span> <span class="s">'c.txt'</span><span class="p">]</span>
<span class="n">gs</span> <span class="o">=</span> <span class="n">goslate</span><span class="o">.</span><span class="n">Goslate</span><span class="p">()</span>

<span class="n">translation</span> <span class="o">=</span> <span class="p">[]</span>
<span class="k">for</span> <span class="n">big_file</span> <span class="ow">in</span> <span class="n">big_files</span><span class="p">:</span>
    <span class="k">with</span> <span class="nb">open</span><span class="p">(</span><span class="n">big_file</span><span class="p">,</span> <span class="s">'r'</span><span class="p">)</span> <span class="k">as</span> <span class="n">f</span><span class="p">:</span>
        <span class="n">translated_lines</span> <span class="o">=</span> <span class="p">[]</span>
        <span class="k">for</span> <span class="n">line</span> <span class="ow">in</span> <span class="n">f</span><span class="p">:</span>
            <span class="n">translated_line</span> <span class="o">=</span> <span class="n">gs</span><span class="o">.</span><span class="n">translate</span><span class="p">(</span><span class="n">line</span><span class="p">)</span>
            <span class="n">translated_lines</span><span class="o">.</span><span class="n">append</span><span class="p">(</span><span class="n">translated_line</span><span class="p">)</span>

        <span class="n">translation</span><span class="o">.</span><span class="n">append</span><span class="p">(</span><span class="s">'</span><span class="se">\n</span><span class="s">'</span><span class="o">.</span><span class="n">join</span><span class="p">(</span><span class="n">translated_lines</span><span class="p">))</span>
</pre>
         </div> 
        </div> 
        <p>It is better to leave them to Goslate totally. The following code is not only simpler but also much faster (+100x) :</p> 
        <div class="highlight-python">
         <div class="highlight">
          <pre><span class="kn">import</span> <span class="nn">golsate</span>

<span class="n">big_files</span> <span class="o">=</span> <span class="p">[</span><span class="s">'a.txt'</span><span class="p">,</span> <span class="s">'b.txt'</span><span class="p">,</span> <span class="s">'c.txt'</span><span class="p">]</span>
<span class="n">gs</span> <span class="o">=</span> <span class="n">goslate</span><span class="o">.</span><span class="n">Goslate</span><span class="p">()</span>

<span class="n">translation_iter</span> <span class="o">=</span> <span class="n">gs</span><span class="o">.</span><span class="n">translate</span><span class="p">(</span><span class="nb">open</span><span class="p">(</span><span class="n">big_file</span><span class="p">,</span> <span class="s">'r'</span><span class="p">)</span><span class="o">.</span><span class="n">read</span><span class="p">()</span> <span class="k">for</span> <span class="n">big_file</span> <span class="ow">in</span> <span class="n">big_files</span><span class="p">)</span>
<span class="n">translation</span> <span class="o">=</span> <span class="nb">list</span><span class="p">(</span><span class="n">translation_iter</span><span class="p">)</span>
</pre>
         </div> 
        </div> 
        <p>Internally, goslate will first adjust the text to make them not so big that do not fit Google query API nor so small that increase the total HTTP querying times. Then it will use concurrent query to speed thing even further.</p> 
       </div> 
       <div class="section" id="lookup-details-in-dictionary"> 
        <h2><a class="toc-backref" href="#id15">Lookup Details in Dictionary</a><a class="headerlink" href="#lookup-details-in-dictionary" title="Permalink to this headline">¶</a></h2> 
        <p>If you want detail dictionary explaination for a single word/phrase, you could</p> 
        <div class="highlight-python">
         <div class="highlight">
          <pre><span class="gp">&gt;&gt;&gt; </span><span class="kn">import</span> <span class="nn">goslate</span>
<span class="gp">&gt;&gt;&gt; </span><span class="n">gs</span> <span class="o">=</span> <span class="n">goslate</span><span class="o">.</span><span class="n">Goslate</span><span class="p">()</span>
<span class="gp">&gt;&gt;&gt; </span><span class="n">gs</span><span class="o">.</span><span class="n">lookup_dictionary</span><span class="p">(</span><span class="s">'sun'</span><span class="p">,</span> <span class="s">'de'</span><span class="p">)</span>
<span class="go">[[['Sonne', 'sun', 0]],</span>
<span class="go"> [['noun',</span>
<span class="go">   ['Sonne'],</span>
<span class="go">   [['Sonne', ['sun', 'Sun', 'Sol'], 0.44374731, 'die']],</span>
<span class="go">   'sun',</span>
<span class="go">   1],</span>
<span class="go">  ['verb',</span>
<span class="go">   ['der Sonne aussetzen'],</span>
<span class="go">   [['der Sonne aussetzen', ['sun'], 1.1544633e-06]],</span>
<span class="go">   'sun',</span>
<span class="go">   2]],</span>
<span class="go"> 'en',</span>
<span class="go"> 0.9447732,</span>
<span class="go"> [['en'], [0.9447732]]]</span>
</pre>
         </div> 
        </div> 
        <p>There are 2 limitaion for this API:</p> 
        <ul class="simple"> 
         <li>The result is a complex list structure which you have to parse for your own usage</li> 
         <li>The input must be a single word/phase, batch translation and concurrent querying are not supported</li> 
        </ul> 
       </div> 
       <div class="section" id="query-error"> 
        <h2><a class="toc-backref" href="#id16">Query Error</a><a class="headerlink" href="#query-error" title="Permalink to this headline">¶</a></h2> 
        <p>If you get HTTP 5xx error, it is probably because google has banned your client IP address from transation querying.</p> 
        <p>You could verify it by access google translation service in browser manully.</p> 
        <p>You could try the following to overcome this issue:</p> 
        <ul class="simple"> 
         <li>query through a HTTP/SOCK5 proxy, see <a class="reference internal" href="#proxy-support">Proxy Support</a></li> 
         <li>using another google domain for translation: <code class="docutils literal"><span class="pre">gs</span> <span class="pre">=</span> <span class="pre">Goslate(service_urls=['http://translate.google.de'])</span></code></li> 
         <li>wait for 3 seconds before issue another querying</li> 
        </ul> 
       </div>

               <td class="field-body"><p class="first"></p><p>the translated text(s)</p> 
                <ul class="simple"> 
                 <li>unicode: on single string input</li> 
                 <li>generator of unicode: on batch input of string sequence</li> 
                 <li>tuple: if WRITING_NATIVE_AND_ROMAN is specified, it will return tuple/generator for tuple (u”native”, u”roman format”)</li> 
                </ul> <p></p> </td> 
              </tr>

                 <li><a class="reference internal" href="#goslate.Error" title="goslate.Error"><code class="xref py py-class docutils literal"><span class="pre">Error</span></code></a> (‘input too large’) if input a single large word without any punctuation or space in between</li>
