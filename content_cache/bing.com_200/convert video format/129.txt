           <div class="title">
            How do I convert a JavaBean to an XML String using XMLEncoder?
           </div> 
           <div>
            Author: Deron Eriksson
           </div> 
           <div class="smaller">
            Description: This Java tutorial shows how to write a JavaBean to an XML String representation using Java's XMLEncoder class.
           </div> 
           <div class="smaller">
             Tutorial created using: Windows XP || JDK 1.5.0_09 || Eclipse Web Tools Platform 2.0 (Eclipse 3.3.0)
           </div> 
          </div> 
          <hr> <br> <p>In <a title="How do I write a JavaBean to an XML file using XMLEncoder?" href="how-do-i-write-a-javabean-to-an-xml-file-using-xmlencoder.html">another tutorial</a>, we saw how we could use the XMLEncoder class to save a JavaBean<a class="extlink" href="http://en.wikipedia.org/wiki/Javabean" title="Go to 'JavaBean' in Wikipedia">W</a> object to an XML<a class="extlink" href="http://en.wikipedia.org/wiki/XML" title="Go to 'XML' in Wikipedia">W</a> file. Suppose we'd rather convert the JavaBean to a String representation rather than to a file. To do this, we can have the XMLEncoder write to a ByteArrayOutputStream rather than a FileOutputStream. </p> <p>The XmlEncodeToString class demonstrates this. It creates a MyBean object, populates it with some data, and then the XMLEncoder class writes the object to a ByteArrayOutputStream. The ByteArrayOutputStream is converted to a String via a call to its toString() method. Then, this String is output to standard output. </p> <h3><a href="../general-java/how-do-i-convert-a-javabean-to-an-xml-string-using-xmlencoder/XmlEncodeToString.java">XmlEncodeToString.java</a></h3> <pre><span class="javaKeyword">package</span> example;

<span class="javaKeyword">import</span> java.beans.XMLEncoder;
<span class="javaKeyword">import</span> java.io.ByteArrayOutputStream;
<span class="javaKeyword">import</span> java.util.Vector;

<span class="javaKeyword">public</span> <span class="javaKeyword">class</span> XmlEncodeToString {

	<span class="javaKeyword">public</span> <span class="javaKeyword">static</span> <span class="javaKeyword">void</span> main(String[] args) <span class="javaKeyword">throws</span> Exception {

		MyBean mb = <span class="javaKeyword">new</span> MyBean();
		mb.setMyBoolean(true);
		mb.setMyString(<span class="javaString">"xml is cool"</span>);
		Vector&lt;String&gt; v = <span class="javaKeyword">new</span> Vector&lt;String&gt;();
		v.add(<span class="javaString">"one"</span>);
		v.add(<span class="javaString">"two"</span>);
		v.add(<span class="javaString">"three"</span>);
		mb.setMyVector(v);

		ByteArrayOutputStream baos = <span class="javaKeyword">new</span> ByteArrayOutputStream();
		XMLEncoder xmlEncoder = <span class="javaKeyword">new</span> XMLEncoder(baos);
		xmlEncoder.writeObject(mb);
		xmlEncoder.close();

		String xml = baos.toString();
		System.out.println(xml);

	}

}
</pre> <p>The MyBean class is shown here: </p> <h3><a href="../general-java/how-do-i-convert-a-javabean-to-an-xml-string-using-xmlencoder/MyBean.java">MyBean.java</a></h3> <pre><span class="javaKeyword">package</span> example;

<span class="javaKeyword">import</span> java.io.Serializable;
<span class="javaKeyword">import</span> java.util.Vector;

<span class="javaKeyword">public</span> <span class="javaKeyword">class</span> MyBean <span class="javaKeyword">implements</span> Serializable {

	<span class="javaKeyword">private</span> <span class="javaKeyword">static</span> <span class="javaKeyword">final</span> <span class="javaKeyword">long</span> serialVersionUID = 1L;
	<span class="javaKeyword">private</span> <span class="javaKeyword">boolean</span> myBoolean;
	<span class="javaKeyword">private</span> String myString;
	<span class="javaKeyword">private</span> Vector&lt;String&gt; myVector;

	<span class="javaKeyword">public</span> MyBean() {
	}
	<span class="javaKeyword">public</span> <span class="javaKeyword">boolean</span> isMyBoolean() {
		<span class="javaKeyword">return</span> myBoolean;
	}
	<span class="javaKeyword">public</span> <span class="javaKeyword">void</span> setMyBoolean(<span class="javaKeyword">boolean</span> myBoolean) {
		<span class="javaKeyword">this</span>.myBoolean = myBoolean;
	}
	<span class="javaKeyword">public</span> String getMyString() {
		<span class="javaKeyword">return</span> myString;
	}
	<span class="javaKeyword">public</span> <span class="javaKeyword">void</span> setMyString(String myString) {
		<span class="javaKeyword">this</span>.myString = myString;
	}
	<span class="javaKeyword">public</span> Vector&lt;String&gt; getMyVector() {
		<span class="javaKeyword">return</span> myVector;
	}
	<span class="javaKeyword">public</span> <span class="javaKeyword">void</span> setMyVector(Vector&lt;String&gt; myVector) {
		<span class="javaKeyword">this</span>.myVector = myVector;
	}
}
</pre> <p>Executing XmlEncodeToString results in the following console output: </p> <h3><a href="../general-java/how-do-i-convert-a-javabean-to-an-xml-string-using-xmlencoder/console.txt">Console Output</a></h3> <pre>
&lt;?xml version="1.0" encoding="UTF-8"?&gt; 
&lt;java version="1.5.0_09" class="java.beans.XMLDecoder"&gt; 
 &lt;object class="example.MyBean"&gt; 
  &lt;void property="myBoolean"&gt; 
   &lt;boolean&gt;true&lt;/boolean&gt; 
  &lt;/void&gt; 
  &lt;void property="myString"&gt; 
   &lt;string&gt;xml is cool&lt;/string&gt; 
  &lt;/void&gt; 
  &lt;void property="myVector"&gt; 
   &lt;object class="java.util.Vector"&gt; 
    &lt;void method="add"&gt; 
     &lt;string&gt;one&lt;/string&gt; 
    &lt;/void&gt; 
    &lt;void method="add"&gt; 
     &lt;string&gt;two&lt;/string&gt; 
    &lt;/void&gt; 
    &lt;void method="add"&gt; 
     &lt;string&gt;three&lt;/string&gt; 
    &lt;/void&gt; 
   &lt;/object&gt; 
  &lt;/void&gt; 
 &lt;/object&gt; 
&lt;/java&gt;
