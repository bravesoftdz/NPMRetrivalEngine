                  There isn't one is the short answer.
                 <br> 
                 <br> Synology's primary consumer boxes are Arm based and woefully underpowered for video transcoding.
                 <br> 
                 <br> The only other option is a media server, for real time transcoding you have Serviio, TVMobili or Plex at your disposal and seeing as your using Plex to begin with that that leaves two but no support for Plex clients as they are DLNA only. TVMobili is more the more light weight of the two but also costs money, Serviio requires installing Java though the dev provides a Java package you can use.
                 <br> 
                 <br> Someone with the skills could likely very well compile something like Handbrake for Synology but I dont think anyone has ever bothered, you could try hunting around or asking on the

			$('.dap-widget').each(function(key){
				if (data[key] != undefined)
				{
					if ($(this).data('dap-type') == 'inline' && data[key]['prices'].length > 0)
					{
						$(this).show();
						$(this).append(
						  $('<div/>')
						    .addClass("inline")
						    .html('<i class="fa fa-shopping-cart"></i> Buy the ').append(
						    	$('<b/>').text(data[key]['name'])
						    ).append($('<span/>').text(', available from '))
						);
	
						var priceList = $('<ul/>').addClass("prices");
						$(this).append(priceList);
	
						$.each(data[key]['prices'], function(val){
							priceList.append(
								$("<li/>").append(
									$("<a/>").attr('href', this['url']).attr('target', '_blank').addClass('retailer').text(this['retailer'])

									$("<span/>").text(': ')
								).append(
									$("<a/>").attr('href', this['url']).attr('target', '_blank').addClass('price').text(this['price'])
								)
							);
						});
	
					}
					
				}
	
			});
			
		});
	}
