         <h2>Can't open a .js file?</h2> 
         <p> If you want to open a <strong>.js</strong> file on your computer, you just need to have the appropriate program installed. If the <strong>.js</strong> association isn't set correctly, you may receive the following error message: </p> 
         <p> <em>Windows can't open this file:<br><br>File: example.js<br><br>To open this file, Windows needs to know what program you want to use to open it. Windows can go online to look it up automatically, or you can manually select from a list of programs that are installed on your computer.</em> </p> 
         <h2>To change file associations:</h2> 
         <ul> 
          <li>Right-click a file with the extension whose association you want to change, and then click <em>Open With</em>.</li> 
          <li>In the <em>Open With</em> dialog box, click the program whith which you want the file to open, or click <em>Browse</em> to locate the program that you want.</li> 
          <li>Select the <em>Always use the selected program</em> to open this kind of file check box.</li> 
         </ul>
