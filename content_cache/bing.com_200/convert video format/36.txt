           <div class="cd">
            <p><em></em> FFmpeg Extension for Video Processing without any php extension Requirement For Windows (for Ubantu install it using "sudo apt-get install ffmpeg" command )</p> 
            <h1>Introduction</h1> 
            <p>FFmpeg Extension for Video Processing without any php extension Requirement For Windows.Using by this plugin you can do versatile operation such as you can convert video into other format,grab screen shot of specific second from video , convert video into single gif file, convert video in to audio , convert video into serialize images.</p> 
            <h1>Developed By :</h1> 
            <p>HARSH PATEL</p> 
            <h1>Version :</h1> 
            <p>1.0</p> 
            <h2>Documentation</h2> 
            <p>This Package Contains Following Files 1.Code 2.input 3.output 4.Commands </p> 
            <p></p> 
            <p>1.Code </p> 
            <p>In this Folder you can find demo.php which contain some ffmpeg commands to convert video into different formate,Capture Image of specific seconds from video,Convert audio from video , etc. You can here one "ffmpeg.exe" please do not delete it is require for execution.</p> 
            <p>2.input </p> 
            <p>This Folder contains sample avi video. </p> 
            <p>3.output</p> 
            <p>In this Folder you will get converted video as per path set in demo.php. </p> 
            <p>4.Commands</p> 
            <p>Here you can find some useful ffmpeg commands.</p> 
            <p>(as per your requirement you can change input output folder path) </p> 
            <h2>Operating system Support</h2> 
            <ul>
             <li>Windows</li> 
             <li>Linux</li> 
             <li>Ubantu (sudo apt-get install ffmpeg)</li> 
            </ul> 
            <h2>License</h2> 
            <p>This Package is an open source project by HARSH PATEL </p> 
            <h2>USEFUL COMMANDS</h2> 
            <p></p> 
            <ol>
             <li>Cut video file into a smaller clip =====================================</li> 
            </ol> 
            <p>You can use the time offset parameter (-ss) to specify the start time stamp in HH:MM:SS.ms format while the -t parameter is for specifying the actual duration of the clip in seconds.</p> 
            <p>==&gt; ffmpeg -i input.mp4 -ss 00:00:50.0 -codec copy -t 20 output.mp4</p> 
            <p></p> 
            <ol>
             <li>Split a video into multiple parts ====================================</li> 
            </ol> 
            <p>If you want to split a large video into multiple smaller clips without re-encoding, ffmpeg can help. This command will split the source video into 2 parts â?? one ending at 50s from the start and the other beginning at 50s and ending at the end of the input video.</p> 
            <p>=&gt; ffmpeg -i video.mp4 -t 00:00:50 -c copy small-1.mp4 -ss 00:00:50 -codec copy small-2.mp4</p> 
            <ol>
             <li>Convert video from one format to another ===========================================</li> 
            </ol> 
            <p>You can use the -vcodec parameter to specify the encoding format to be used for the output video. Encoding a video takes time but you can speed up the process by forcing a preset though it would degrade the quality of the output video.</p> 
            <p>=&gt; ffmpeg -i youtube.flv -c:v libx264 filename.mp4 =&gt; ffmpeg -i video.wmv -c:v libx264 -preset ultrafast video.mp4</p> 
            <ol>
             <li>Join (concatenate) video files =================================</li> 
            </ol> 
            <p>If you have multiple audio or video files encoded with the same codecs, you can join them into a single file using FFmpeg. Create a input file with a list of all source files that you wish to concatenate and then run this command.</p> 
            <p>=&gt; ffmpeg -f concat -i file-list.txt -c copy output.mp4</p> 
            <ol>
             <li>Mute a video (Remove the audio component) ============================================</li> 
            </ol> 
            <p>Use the -an parameter to disable the audio portion of a video stream.</p> 
            <p>=&gt; ffmpeg -i video.mp4 -an mute-video.mp4</p> 
            <ol>
             <li>Extract the audio from video ===============================</li> 
            </ol> 
            <p>The -vn switch extracts the audio portion from a video and we are using the -ab switch to save the audio as a 256kbps MP3 audio file.</p> 
            <p>=&gt; ffmpeg -i video.mp4 -vn -ab 256 audio.mp3</p> 
            <ol>
             <li>Convert a video into animated GIF ====================================</li> 
            </ol> 
            <p>FFmpeg is an excellent tool for converting videos into animated GIFs and the quality isnâ??t bad either. Use the scale filter to specify the width of the GIF, the -t parameter specific the duration while -r specifies the frame rate (fps).</p> 
            <p>=&gt; ffmpeg -i video.mp4 -vf scale=500:-1 -t 10 -r 10 image.gif</p> 
            <ol>
             <li>Extract image frames from a video ====================================</li> 
            </ol> 
            <p>This command will extract the video frame at the 15s mark and saves it as a 800px wide JPEG image. You can also use the -s switch (like -s 400Ã?300) to specify the exact dimensions of the image file though it will probably create a stretched image if the image size doesnâ??t follow the aspect ratio of the original video file.</p> 
            <p>=&gt; ffmpeg -ss 00:00:15 -i video.mp4 -vf scale=800:-1 -vframes 1 image.jpg</p> 
            <ol>
             <li>Convert Video into Images ============================</li> 
            </ol> 
            <p>You can use FFmpeg to automatically extract image frames from a video every â??nâ?? seconds and the images are saved in a sequence. This command saves image frame after every 4 seconds.</p> 
            <p>=&gt; ffmpeg -i movie.mp4 -r 0.25 frames_%04d.png</p> 
            <ol>
             <li>Merge an audio and video file =================================</li> 
            </ol> 
            <p>You can also specify the -shortest switch to finish the encoding when the shortest clip ends.</p> 
            <p>=&gt; ffmpeg -i video.mp4 -i audio.mp3 -c:v copy -c:a aac -strict experimental output.mp4 =&gt; ffmpeg -i video.mp4 -i audio.mp3 -c:v copy -c:a aac -strict experimental -shortest output.mp4</p> 
            <ol>
             <li>Resize a video ==================</li> 
            </ol> 
            <p>Use the size (-s) switch with ffmpeg to resize a video while maintaining the aspect ratio.</p> 
            <p>=&gt; ffmpeg -i input.mp4 -s 480x320 -c:a copy output.mp4</p> 
            <ol>
             <li>Create video slideshow from images ======================================</li> 
            </ol> 
            <p>This command creates a video slideshow using a series of images that are named as img001.png, img002.png, etc. Each image will have a duration of 5 seconds (-r 1/5).</p> 
            <p>=&gt; ffmpeg -r 1/5 -i img%03d.png -c:v libx264 -r 30 -pix_fmt yuv420p slideshow.mp4</p> 
            <ol>
             <li>Add a poster image to audio ===============================</li> 
            </ol> 
            <p>You can add a cover image to an audio file and the length of the output video will be the same as that of the input audio stream. This may come handy for uploading MP3s to YouTube.</p> 
            <p>=&gt; ffmpeg -loop 1 -i image.jpg -i audio.mp3 -c:v libx264 -c:a aac -strict experimental -b:a 192k -shortest output.mp4</p> 
            <ol>
             <li>Convert a single image into a video =======================================</li> 
            </ol> 
            <p>Use the -t parameter to specify the duration of the video.</p> 
            <p>=&gt; ffmpeg -loop 1 -i image.png -c:v libx264 -t 30 -pix_fmt yuv420p video.mp4</p> 
            <ol>
             <li>Add subtitles to a movie ============================</li> 
            </ol> 
            <p>This will take the subtitles from the .srt file. FFmpeg can decode most common subtitle formats.</p> 
            <p>=&gt; ffmpeg -i movie.mp4 -i subtitles.srt -map 0 -map 1 -c copy -c:v libx264 -crf 23 -preset veryfast output.mkv</p> 
            <ol>
             <li>Crop an audio file ======================</li> 
            </ol> 
            <p>This will create a 30 second audio file starting at 90 seconds from the original audio file without transcoding.</p> 
            <p>=&gt; ffmpeg -ss 00:01:30 -t 30 -acodec copy -i inputfile.mp3 outputfile.mp3</p> 
            <ol>
             <li>Change the audio volume ===========================</li> 
            </ol> 
            <p>You can use the volume filter to alter the volume of a media file using FFmpeg. This command will half the volume of the audio file.</p> 
            <p>=&gt; ffmpeg -i input.wav -af 'volume=0.5' output.wav</p> 
            <ol>
             <li>Rotate a video ==================</li> 
            </ol> 
            <p>This command will rotate a video clip 90Â° clockwise. You can set transpose to 2 to rotate the video 90Â° anti-clockwise.</p> 
            <p>=&gt; ffmpeg -i input.mp4 -filter:v 'transpose=1' rotated-video.mp4</p> 
            <p>This will rotate the video 180Â° counter-clockwise.</p> 
            <p>=&gt; ffmpeg -i input.mp4 -filter:v 'transpose=2,transpose=2' rotated-video.mp4</p> 
            <ol>
             <li>Speed up or Slow down the video ===================================</li> 
            </ol> 
            <p>You can change the speed of your video using the setpts (set presentation time stamp) filter of FFmpeg. This command will make the video 8x (1/8) faster or use setpts=4*PTS to make the video 4x slower.</p> 
            <p>=&gt; ffmpeg -i input.mp4 -filter:v "setpts=0.125*PTS" output.mp4</p> 
            <ol>
             <li>Speed up or Slow down the audio ====================================</li> 
            </ol> 
            <p>For changing the speed of audio, use the atempo audio filter. This command will double the speed of audio. You can use any value between 0.5 and 2.0 for audio.</p> 
            <p>=&gt; ffmpeg -i input.mkv -filter:a "atempo=2.0" -vn output.mkv</p>
