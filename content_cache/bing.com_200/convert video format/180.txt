      <h1 class="title">Converting Date and Time Values</h1> 
      <div class="lw_vs"> 
       <div id="curversion"> 
        <strong> SharePoint 2010 </strong>

        <div id="vsseperator" class="cl_lw_vs_seperator"></div>

      <div id="mainSection"> 
       <div id="mainBody"> 
        <div class="introduction"> 
         <p> <strong>Last modified: </strong>April 15, 2010</p> 
         <p> <em> <strong>Applies to: </strong>SharePoint Foundation 2010</em> </p> 
         <p>Microsoft SharePoint Foundation stores date and time values in Coordinated Universal Time (UTC) format, and almost all date and time values that are returned by members of the object model are in UTC format. The one exception is list column values that are obtained through the indexer for the <span><a href="https://msdn.microsoft.com/en-us/library/office/microsoft.sharepoint.splistitem(v=office.14).aspx">SPListItem</a></span> class, which are formatted in the local time for the site. Use the <span><a href="https://msdn.microsoft.com/en-us/library/office/microsoft.sharepoint.spquery.datesinutc(v=office.14).aspx">DatesInUtc</a></span> property of the <span><a href="https://msdn.microsoft.com/en-us/library/office/microsoft.sharepoint.spquery(v=office.14).aspx">SPQuery</a></span> class to use the indexer to return values in UTC format.</p> 
         <p>To convert values from UTC format to local time, use the <span><a href="https://msdn.microsoft.com/en-us/library/office/microsoft.sharepoint.sptimezone.utctolocaltime(v=office.14).aspx">UTCToLocalTime</a></span> method of the <span><a href="https://msdn.microsoft.com/en-us/library/office/microsoft.sharepoint.sptimezone(v=office.14).aspx">SPTimeZone</a></span> class, which can be accessed through the <span><a href="https://msdn.microsoft.com/en-us/library/office/microsoft.sharepoint.spweb.regionalsettings(v=office.14).aspx">RegionalSettings</a></span> property for the current site, as follows: <span class="code">mySite.RegionalSettings.TimeZone.UtcTimeToLocalTime(date)</span>. To convert values from local time to UTC format, instead use the <span><a href="https://msdn.microsoft.com/en-us/library/office/microsoft.sharepoint.sptimezone.localtimetoutc(v=office.14).aspx">LocalTimeToUTC</a></span> method. </p> 
         <p>In addition to converting between local and UTC time formats, you may also need to convert between date and time formats, for example, from ISO8601 format (YYYY-MM-DDTHH:MM:SSZ) to <span class="input">System.DateTime</span> format (mm/dd/yyyy hh:mm:ss AM or PM), or the reverse. The <span><a href="https://msdn.microsoft.com/en-us/library/office/microsoft.sharepoint.utilities.sputility(v=office.14).aspx">SPUtility</a></span> class provides several methods that can be used to convert or modify the format of date and time values, such as the following: </p> 
         <ul> 
          <li> <p> <span> <a href="https://msdn.microsoft.com/en-us/library/office/microsoft.sharepoint.utilities.sputility.createiso8601datetimefromsystemdatetime(v=office.14).aspx">CreateISO8601DateTimeFromSystemDateTime</a> </span> — from system DateTime format to ISO8601 DateTime format (yyyy-mm-ddThh:mm:ssZ). This method is useful, for example, when building a query with a filter that is based on a <span class="input">System.DateTime</span> value. The following example returns all items in a document library that have been modified in the past five days.</p>
