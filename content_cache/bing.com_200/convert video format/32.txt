        <h2>Introduction</h2> 
        <p>The <b>java.lang.String</b> class represents character strings. All string literals in Java programs, such as "abc", are implemented as instances of this class.Strings are constant, their values cannot be changed after they are created</p> 
        <h2>Class Declaration</h2> 
        <p>Following is the declaration for <b>java.lang.String</b> class ?</p> 
        <pre class="result notranslate">
public final class String
   extends Object
      implements Serializable, Comparable&lt;String&gt;, CharSequence
</pre>
