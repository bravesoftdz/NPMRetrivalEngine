              <p>Version 1:</p> 
              <pre><code>cat ~/scripts/compile_movie.sh 
#!/bin/sh
mencoder "mf://*.jpg" -fps $1 -ovc lavc -lavcopts vcodec=mpeg4:vbitrate=1400 -ffourcc DIVX -vf scale=640:360 -o $2
</code></pre> 
              <p>usage: ./compile_move.sh 30 myFile.mp4 editing the bitrate, codec, etc is straightforward. </p> 
              <p>Version 2: </p> 
              <pre><code>cat ~/scripts/compile_pngs_to_flash_movie.sh 
#!/bin/bash
#
# usage:
#./compile_pngs_to_flash_movie.sh  my_funky_movie.flv

output_movie="$1"
mencoder mf://*.png -mf w=736:h=246:fps=20:type=png -of lavf -ffourcc FLV1 -ovc lavc -   lavcopts vcodec=flv -o "${output_movie}"
</code></pre>

            <td class="answercell"> 
             <div class="post-text" itemprop="text"> 
              <p>The best tool I have found that does this is <a href="http://dvd-slideshow.sourceforge.net" rel="nofollow noreferrer">dvd-slideshow</a>. It creates a dvd-compatible mpeg2 video from a bunch of images. You can add music if you want also. Supports several effects like fadein/fadeout/crossfade/crop/kenburns. I use it to make family memories type dvd's.</p> 
              <p>It runs on any *nux easily, though I've never installed it on windows, so if that's your target OS, I don't know what hurdles you might face with it. It's a bash script. On windows, at a minimum you'd want to start by installing <a href="http://www.cygwin.com/" rel="nofollow noreferrer">cygwin</a> and go from there.</p>
