       <div class="wikidoc">
        VideoConverter is a .NET wrapper for FFMpeg tool that can handle almost all known formats and video/audio codecs. Unlike another wrappers it is not statically linked with FFMpeg library and can use GPL-build of FFMpeg.
        <br> 
        <br> 
        <b>features</b> 
        <ul> 
         <li>convert video and audio files with one line of code </li>
         <li>webcam/online RTSP stream/URL/screen capture can be used as input </li>
         <li>can merge several video files into one resulting video </li>
         <li>simple deployment: all in one DLL (no external dependencies, ffmpeg is embedded as assembly resource and automatically extracted on first use) </li>
         <li>can be used from ASP.NET server code (legacy .NET 3.5 / Runtime 2.0 is also supported) </li>
         <li>can extract frame images (get thumbnail from video), append logo or advertisement, apply watermark image </li>
         <li>real-time media stream conversions (stdin/stdout ffmpeg mode, named pipes)</li>
        </ul> 
        <br> 
        <b>supported targets</b> 
        <ul> 
         <li>.NET 2.0-4.x </li>
         <li>Mono (2.0 LT build, requires os-specific ffmpeg binaries) </li>
         <li>.NET Core (.NET Standards 1.5 LT build, requires os-specific ffmpeg binaries)</li>
        </ul> 
        <br> Convert video with one line of code:
        <br> 
        <pre>
(new NReco.VideoConverter.FFMpegConverter()).ConvertMedia(pathToVideoFile, Response.OutputStream, Formats.flv);
</pre> 
        <br> 
        <br> See also: 
        <ul> 
         <li><a href="https://www.nrecosite.com/video_converter_net.aspx">Video Converter for .NET</a> component page with online demo (convert video clip, extract thumbnail, apply watermark), change log and FAQ </li>
         <li><a href="https://www.nrecosite.com/doc/NReco.VideoConverter/">Online API Reference</a></li>
        </ul>
