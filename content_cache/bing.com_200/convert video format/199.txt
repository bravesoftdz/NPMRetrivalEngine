       <p>Imagine the situation where you are working on a huge project and the project manager uses Windows and created a project document using Microsoft word 2007 document and send it to the whole team including you.<br> <code> </code><br> You can call the project manager and ask him to convert it to txt or pdft and send it to you. But, you are not going to do that. Remember.. you are a Geek . So, you have to find a way to convert it to pdf yourself without asking the project manager to convert it using his Microsoft word 2007 software.<br> <code> </code><br> In this article, let us review <strong>how to convert almost any file types</strong> without the need of having the original software.<br> <span id="more-499"></span><br> There are lot of ways to convert files. Some of them are mentioned below.<code> </code></p> 
       <ol> 
        <li>Linux command line utilities such as pdftohtml, pdftotext, ps2pdf.</li> 
        <li>Using the export option in the original software itself to change the type of the file, such as<br> export as pdf in open office. And converting images between different formats using gimp.</li> 
        <li>Specific softwares both free &amp; paid, which might be doing conversion of a particular type to other.</li> 
       </ol> 
       <p><code> </code><br> But there might be cases where you will find nothing fits your file conversion requirement. In that case, you can look-up to the online file conversion tools which will definitely be a platform independent solution.<br> <code> </code><br> <a rel="nofollow" href="http://www.zamzar.com/">Zamzar</a> is an <strong>online file conversion tool</strong> that converts files for free.&nbsp; The various types file conversion at zamzar is very impressive, provided the fact they do it for free. They convert the following types of files.</p> 
       <ul> 
        <li>Document formats</li> 
        <li>Image formats</li> 
        <li>Music formats</li> 
        <li>Video formats</li> 
        <li>Compressed formats</li> 
        <li>CAD formats</li> 
       </ul> 
       <p><code> </code><br> <strong>The file conversion concept is very simple:</strong> Go to zamzar website and upload your source file and specify the destination file format that you want. Zamzar will automatically convert it in the background. Once the conversion is done, it will send an email to you with a link to download the converted file.<br> <code> </code><br> If your source file is 100MB or less you can convert it for free. In most conversion cases 100 MB is really a lot. Personally for me, there is no way my source file will go above 100 MB.<br> <code> </code></p> 
       <h3>Online File Conversion Procedure</h3> 
       <h3>1. Upload the Source File</h3> 
       <ul> 
        <li><strong>Step 1.</strong> Go to <a rel="nofollow" href="http://www.zamzar.com">zamzar.com</a> and upload your source file. In this example, I uploaded Project-Document.docx file as shown below. You can upload a maximum of 5 files at the same time for concurrent conversion. If you want more than 5 files to be converted, repeat the same steps again.</li> 
        <li><strong>Step 2.</strong> Choose your destination format. In this example, I choose pdf as shown below.</li> 
        <li><strong>Step 3.</strong> Enter the email address where you want the link to the converted file to be sent to.</li> 
        <li><strong>Step 4.</strong> Click on convert to begin the conversion.</li> 
        <li><strong>Note:</strong> All the selected file will be displayed at the bottom under “Files to Convert” section as shown in the picture below.</li> 
       </ul> 
       <p style="text-align: center;"><a href="http://static.thegeekstuff.com/wp-content/uploads/2009/06/zamzar-1.jpg"><img class="size-full wp-image-545 aligncenter" title="Online File Converter - jpeg to tiff" src="http://static.thegeekstuff.com/wp-content/uploads/2009/06/zamzar-1.jpg" alt="File Format Converter - pdf to jpg" width="500" height="207" srcset="http://static.thegeekstuff.com/wp-content/uploads/2009/06/zamzar-1.jpg 775w, http://www.thegeekstuff.com/wp-content/uploads/2009/06/zamzar-1-300x124.jpg 300w" sizes="(max-width: 500px) 100vw, 500px"></a><br> <strong>Fig:</strong> Upload the Source file for conversion</p> 
       <h3>2. Confirm File Conversation</h3> 
       <p>Once you’ve clicked on convert, it will display the following conversion confirmation. Click on OK to accept the file conversion.</p> 
       <p style="text-align: center;"><img class="size-full wp-image-546 aligncenter" title="File Converter - jpeg to pdf converter" src="http://static.thegeekstuff.com/wp-content/uploads/2009/06/zamzar-2.jpg" alt="Convert pdf Files to Word" width="457" height="160" srcset="http://static.thegeekstuff.com/wp-content/uploads/2009/06/zamzar-2.jpg 457w, http://www.thegeekstuff.com/wp-content/uploads/2009/06/zamzar-2-300x105.jpg 300w" sizes="(max-width: 457px) 100vw, 457px"><br> <strong> Fig:</strong> File conversation confirmation.</p>

       <p>Once you click OK on the confirmation dialogue, it will start uploading the source file (in this example, Project-Document.docx) to the zamzar website as shown below.</p>
