       <h1 id="Converting_audio_and_video_into_open_formats">Converting audio and video into open formats</h1> 
       <span class="anchor" id="line-3"></span>
       <span class="anchor" id="line-4"></span>
       <p class="line867"> </p>
       <h2 id="Converting_MP3s_to_Ogg-Vorbis">Converting MP3s to Ogg-Vorbis</h2> 
       <span class="anchor" id="line-5"></span>
       <span class="anchor" id="line-6"></span>
       <p class="line874">Please keep in mind that converting between lossy compression formats, such as MP3 and Ogg-Vorbis, will result in some loss of sound quality. You have a few options for converting MP3s to Ogg-Vorbis format. Two techniques are listed below. <span class="anchor" id="line-7"></span><span class="anchor" id="line-8"></span></p>
       <ul>
        <li><p class="line891"><strong>Ubuntu's Universe repository includes an easy to use application for mass-converting MP3 music to Ogg Vorbis format.</strong> <span class="anchor" id="line-9"></span></p>
         <ul>
          <li>In Ubuntu: <span class="anchor" id="line-10"></span>
           <ul>
            <li><p class="line862">Install the <tt>soundconverter</tt> package <span class="anchor" id="line-11"></span></p></li>
            <li><p class="line862">Open it from the <em>Applications-&gt;Sound and Video</em> menu <span class="anchor" id="line-12"></span></p></li>
            <li>Add the files you wish to convert <span class="anchor" id="line-13"></span></li>
            <li>Tweak the settings for quality, filename, etc. <span class="anchor" id="line-14"></span></li>
            <li>Start converting! <span class="anchor" id="line-15"></span></li>
           </ul></li>
          <li>In Kubuntu: <span class="anchor" id="line-16"></span>
           <ul>
            <li><p class="line862">Follow the above instructions using the <tt>soundkonverter</tt> package <span class="anchor" id="line-17"></span></p></li>
            <li><p class="line862">Open it from "Kmenu -&gt; Multimedia &gt; <a class="nonexistent" href="/community/SoundKonverter">SoundKonverter</a>" <span class="anchor" id="line-18"></span><span class="anchor" id="line-19"></span></p></li>
           </ul></li>
         </ul></li>
        <li class="gap"><p class="line891"><strong>You can easily convert MP3s into Ogg Vorbis format using <a class="http" href="http://audacity.sourceforge.net/">Audacity</a></strong> <span class="anchor" id="line-20"></span></p>
         <ul>
          <li><p class="line862">Install the package <strong>audacity</strong> <span class="anchor" id="line-21"></span></p></li>
          <li>Start Audacity <span class="anchor" id="line-22"></span></li>
          <li><p class="line862">To import an MP3: File --&gt; Open <span class="anchor" id="line-23"></span></p></li>
          <li>Select an MP3 and open it <span class="anchor" id="line-24"></span></li>
          <li><p class="line862">To export as an Ogg: File --&gt; Export as Ogg Vorbis <span class="anchor" id="line-25"></span></p></li>
          <li>Give the file a name, and save <span class="anchor" id="line-26"></span><span class="anchor" id="line-27"></span><span class="anchor" id="line-28"></span></li>
         </ul></li>
       </ul>
       <p class="line867"> </p>
       <h2 id="Converting_Videos_To_Ogg_Theora">Converting Videos To Ogg Theora</h2> 
       <span class="anchor" id="line-29"></span>
       <span class="anchor" id="line-30"></span>
       <p class="line862">Ubuntu has full support for the Free video format <a class="http" href="http://www.theora.org/">Ogg Theora</a> out of the box, and there are a couple of methods for converting videos into this format: <span class="anchor" id="line-31"></span><span class="anchor" id="line-32"></span></p>
       <ul>
        <li><p class="line862">The <a class="http" href="http://www.videolan.org">VideoLAN</a> project has given the <strong>VLC</strong> media player (available in Ubuntu's Universe repository) streaming capabilities, including the ability to stream into a file. To encode an Ogg Theora video of a file VLC is capable of playing <span class="anchor" id="line-33"></span></p>
         <ul>
          <li><p class="line862">Install the <tt>vlc</tt> package and run it (if working from command line these instructions are for the <tt>vlc</tt> or <tt>wxvlc</tt> commands, not the <tt>svlc</tt> command) <span class="anchor" id="line-34"></span></p></li>
          <li><p class="line862">Choose <em>File-&gt;Open File</em> (NOT <em>Quick Open File</em>) <span class="anchor" id="line-35"></span></p></li>
          <li><p class="line862">In the <em>File</em> tab select the <em>Browse</em> button and choose the video you wish to convert <span class="anchor" id="line-36"></span></p></li>
          <li><p class="line862">Tick the box next to <em>Stream output</em> then click the <em>Settings</em> button which becomes accessible <span class="anchor" id="line-37"></span></p></li>
          <li><p class="line862">Under <em>Output Methods</em> tick <em>File</em> then click the <em>Browse</em> button and choose what you want to save your output file as (possibly ending the filename with ".ogg", although that is not strictly needed) <span class="anchor" id="line-38"></span></p></li>
          <li><p class="line862">Change <em>Encapsulation Method</em> to <em>Ogg</em> and tick <em>Video codec</em> and <em>Audio codec</em>, changing them to <em>theo</em> and <em>vorb</em> respectively <span class="anchor" id="line-39"></span></p></li>
          <li>Choose your desired bit rates for audio and video (you will need to do some experimentation with these to produce small files of high quality) <span class="anchor" id="line-40"></span></li>
          <li><p class="line862">Press <em>OK</em> in the settings window, then again in the open file window. The file will now stream to an Ogg Theora video instead of being played normally. <span class="anchor" id="line-41"></span><span class="anchor" id="line-42"></span></p></li>
         </ul></li>
        <li class="gap">The program ffmpeg2theora is slightly more reliable than VLC, but it is based on the command line: <span class="anchor" id="line-43"></span>
         <ul>
          <li><p class="line862">Install <tt>ffmpeg2theora</tt> from Universe <span class="anchor" id="line-44"></span></p></li>
          <li>Open a terminal in the same folder as the video you wish to convert <span class="anchor" id="line-45"></span></li>
          <li><p class="line862">Run the command <tt>ffmpeg2theora&nbsp;-v&nbsp;N&nbsp;video_file</tt> where N is the output quality and video_file is the filename of the video. This will produce a file called "video_file.ogg" <span class="anchor" id="line-46"></span></p></li>
          <li><p class="line862">For more options see the ffmpeg2theora manual with the command <tt>man&nbsp;ffmpeg2theroa</tt> <span class="anchor" id="line-47"></span></p></li>
         </ul></li>
        <li><p class="line862">There is a rudimentary GUI available for ffmpeg2theora which allows configuration of most ffmpeg2theora options and provides a useful progress bar. It uses the QT4 toolkit, so you may need to install some extra packages to compile or use it. A pre-compiled binary can be found <a class="http" href="http://p80.free.fr/katiuska-0.1-bin.tar.bz2">here</a> and the corresponding source can be found <a class="http" href="http://p80.free.fr/katiuska-0.1.tar.bz2">here</a>. <span class="anchor" id="line-48"></span></p></li>
        <li><p class="line862">The program <strong>Transmageddon</strong> is a very simple gui for converting videos to Ogg Theora and other formats. It can be installed by typing <strong>sudo apt install transmageddon</strong> into the terminal. The settings are very limited because it targets people with low computer skills, however the default settings provide very reliable encodings that produce the best quality/file size possible. The only drawback is that it doesn't offer batch encoding, you're limited to one file at a time. Just open the program, choose the media file you want, and hit transcode, it will convert directly into the .ogg/theora/vorbis format, at best quality without further adjustments. <span class="anchor" id="line-49"></span></p></li>
       </ul>
