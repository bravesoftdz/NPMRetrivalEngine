        <h2 id="converting-rpm-packages-to-debianubuntu-deb-format-with-alien-">Converting .rpm Packages To Debian/Ubuntu .deb Format With Alien </h2> 
        <p>Version 1.0 <br> Author: Falko Timme <br> </p> 
        <p>This article shows how you can convert .rpm packages to .deb packages with a tool called <span class="system">alien</span> so that you can easily install them on Debian and Ubuntu systems. Sometimes this is quite convenient as not all software projects release their software as Debian/Ubuntu packages.</p> 
        <p>For example, I recently wrote a tutorial about <a mce_real_href="/mysql_zrm_debian_sarge" href="/mysql_zrm_debian_sarge" target="_blank">MySQL Backup And Recovery With mysql-zrm On Debian Sarge</a>. In that tutorial I had to install mysql-zrm from the sources because the developers had released an .rpm and a source package, but no .deb package. The installation was a little bit painful because there were no installation instructions for the source package. Now, with <span class="system">alien</span> I can just convert the .rpm file to a .deb file and simply install that one. </p> 
        <p>However, you should keep in mind what the <span class="system">alien</span> man page says:</p> 
        <p><i>"alien should not be used to replace important system packages, like init, libc, or other things that are essential for the functioning of your system. Many of these packages are set up differently by the different distributions, and packages from the different distributions cannot be used interchangeably. In general, if you can't remove a package without breaking your system, don't try to replace it with an alien version." </i></p> 
        <p>&nbsp;</p> 
        <h3 id="-preliminary-note">1 Preliminary Note</h3> 
        <p>In this example I use a Debian Sarge system, and I will show how to convert the mysql-zrm rpm package to a .deb file and install that one. The procedure is the same for Ubuntu. </p> 
        <p>&nbsp;</p> 
        <h3 id="-alien-installation-">2 Alien Installation </h3> 
        <p><span class="system">Alien</span> is available in the normal Debian repositories, so we can install it like this:</p> 
        <p class="command">apt-get install alien</p> 
        <p>&nbsp;</p> 
        <h3 id="-converting-rpm-to-deb">3 Converting .rpm To .deb</h3> 
        <p>Next we download the current mysql-zrm rpm package (1.1-1 at the time of this writing):</p> 
        <p class="command">cd /tmp<br> wget http://www.zmanda.com/downloads/community/ZRM-MySQL/1.1/Linux/MySQL-zrm-1.1-1.noarch.rpm </p> 
        <p>To convert it into a .deb package, we simply run</p> 
        <p class="command">alien MySQL-zrm-1.1-1.noarch.rpm</p> 
        <p>Afterwards, run</p> 
        <p class="command">ls -l</p> 
        <p>in the <span class="system">/tmp</span> directory, and you'll see that <span class="system">alien</span> has created the file <span class="system">mysql-zrm_1.1-2_all.deb</span>. You'll also notice that <span class="system">alien</span> has counted up the version number, it's now <span class="system">1.1-2</span> instead of <span class="system">1.1-1</span>. If you want to keep the original version number, you must use the <span class="system">-k</span> switch:</p> 
        <p class="command">alien -k MySQL-zrm-1.1-1.noarch.rpm</p> 
        <p>will create the file <span class="system">mysql-zrm_1.1-1_all.deb</span>. </p> 
        <p>To install the new .deb file, we use <span class="system">dpkg -i</span>:</p> 
        <p class="command">dpkg -i mysql-zrm_1.1-1_all.deb</p> 
        <p>Now mysql-zrm is installed and fully functional (you still might have to edit its configuration file though).</p> 
        <p>If you want to save the <span class="system">dpkg -i</span> step, you can have <span class="system">alien</span> install the package. The command </p> 
        <p class="command">alien -i MySQL-zrm-1.1-1.noarch.rpm</p> 
        <p>would convert the original rpm package and immediately install it. </p> 
        <p>You see, converting .rpm files to .deb files is very easy. You can have a look at</p> 
        <p class="command">man alien</p> 
        <p>to learn about what else you can do with <span class="system">alien</span>. </p> 
        <p>&nbsp;</p>
