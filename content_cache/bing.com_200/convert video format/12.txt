       <h2 class="entry-title">How to convert JPG to video (AVI, WMV) in Visual C# and VB .NET using Bytescout Image To Video SDK</h2> 
       <div class="post-content"> 
        <div class="at-above-post addthis_tool" data-url="https://bytescout.com/products/developer/imagetovideosdk/imagetovideosdk_convert_jpg_to_video.html"></div>
        <p><object height="344" width="425"><param name="movie" value="http://www.youtube.com/v/_dC9WSS2c7k&amp;hl=en_US&amp;fs=1&amp;rel=0&amp;color1=0x234900&amp;color2=0x4e9e00"><param name="allowFullScreen" value="true"><param name="allowscriptaccess" value="always"><embed allowfullscreen="true" allowscriptaccess="always" src="http://www.youtube.com/v/_dC9WSS2c7k&amp;hl=en_US&amp;fs=1&amp;rel=0&amp;color1=0x234900&amp;color2=0x4e9e00" type="application/x-shockwave-flash" height="344" width="425"></object></p> 
        <p>Image To Video SDK allows to convert image sequence to video slideshow with transitions, slide effects, background music and color.</p> 
        <p><b>JPG images can be converted to either AVI or WMV video file in Image To Video SDK</b>. What format to choose for JPG to video conversion?</p> 
        <p><b>Difference between AVI and WMV video formats:</b></p> 
        <ul> 
         <li>Videos in AVI format have significantly larger file sizes than video files in WMV.</li> 
         <li>AVI has better video playback quality and performance.</li> 
         <li>There are lots of codecs and file extensions associated with AVI (DivX, xDiv, mJpeg, MP4 and many more) while WMV is a proprietary file format of Microsoft.</li> 
         <li>AVI gives possibility to create uncompressed video and 32-bit RGBA video (i.e. video with transparency channel) while WMV doesn’t.</li> 
        </ul> 
        <p><b>Summary:</b> <b>WMV</b> is preferrable for online streaming while <b>AVI</b> is the best format to use for high quality playback on PCs and editing purposes.</p> 
        <p>To <b>convert JPG to AVI in Visual C# or Visual Basic. NET</b>, use source codes from <a href="https://bytescout.com/products/developer/imagetovideosdk/imagetovideosdk_convert_jpg_to_avi.html">How to convert JPG to AVI</a> tutorial.</p> 
        <p>To <b>convert JPG to WMV in Visual C# or Visual Basic .NET</b>, use the source codes below.</p> 
        <p><strong>VISUAL C#</strong></p> 
        <pre class="brush: csharp; title: ; notranslate" title="">using System;
using System.Diagnostics;

using BytescoutImageToVideoLib;

namespace SimpleSlideshow
{
class Program
{

static void Main(string[] args)
{

Console.Write("Converting JPG slides into video, please wait..");

// Create BytescoutImageToVideoLib.ImageToVideo object instance
ImageToVideo converter = new ImageToVideo();

// Activate the component
converter.RegistrationName = "demo";
converter.RegistrationKey = "demo";

// Add images and set the duration for every slide
Slide slide;
slide = (Slide) converter.AddImageFromFileName("..\..\..\..\slide1.jpg");

slide.Duration = 3000; // 3000ms = 3s
slide = (Slide) converter.AddImageFromFileName("..\..\..\..\slide2.jpg");

slide.Duration = 3000;
slide = (Slide) converter.AddImageFromFileName("..\..\..\..\slide3.jpg");

slide.Duration = 3000;

// Set output video size
converter.OutputWidth = 400;

converter.OutputHeight = 300;

// Set output video file name
converter.OutputVideoFileName = "result.wmv";

// Run the conversion
converter.RunAndWait();

Console.WriteLine("Conversion is done. Press any key to continue..");

Console.ReadKey();

// Open the result video file in default media player
Process.Start("result.wmv");

}
}
} </pre>
