            <h1 class="entry-title ">Convert JSON date to JavaScript date in angular</h1> 
            <div class="entry-meta"> 
             <span class="posted-on"><i class="fa fa-calendar"></i> <a href="http://theshravan.net/blog/convert-json-date-to-javascript-date-in-angular/" rel="bookmark"><time class="entry-date published" datetime="2015-06-10T11:41:00+00:00">June 10, 2015</time><time class="updated" datetime="2016-05-22T00:58:36+00:00">May 22, 2016</time></a></span>
             <span class="byline"> <i class="fa fa-user"></i> <span class="author vcard"><a class="url fn n" href="http://theshravan.net/blog/author/shravan/">Shravan Kumar Kasagoni</a></span></span> 
             <span class="cat-links"><i class="fa fa-folder-open-o"></i> <a href="http://theshravan.net/blog/category/javascript/" rel="category tag">JAVASCRIPT</a> </span> 
            </div>
            <!-- .entry-meta -->

           <div class="entry-content"> 
            <p style="text-align: justify;">Dealing with dates is always a nightmare. In this example I will show how to convert a JSON date to JavaScript date returned from server side to an AngularJS app.</p> 
            <p style="text-align: justify;">In AngularJS we use filters to transform data, in this case we can use angular date filter. But it doesn’t work directly with JSON formatted dates.</p> 
            <p><span id="more-2147"></span></p> 
            <p style="text-align: justify;">JSON Date format: <strong>/Date(milliseconds)/</strong></p> 
            <p style="text-align: justify;">I am going to write a custom filter which uses angular built in date filter and accepts the JSON date as well as the required output format. Filter will read the input and extracts parts of a JSON date using substr() method and coverts into integer and pass it as input to angular date filter along with output date format.</p> 
            <p style="text-align: justify;">Here sample controller sets a date directly to variable on the $scope for simplicity in example.</p> 
            <pre><code class="javascript">
angular.module('jsonDateConverterExample', [])
  .controller('JsonDateController', ['$scope', function($scope) {
    $scope.sampleJsonDate = '\/Date(1433913313004-0800)\/';
  }]);

</code></pre> 
            <p style="text-align: justify;">Here is the angular filter which accepts JSON date.</p> 
            <pre><code class="javascript">
angular.module('jsonDateConverterExample', [])
  .filter('jsonDate', ['$filter', function ($filter) {
    return function (input, format) {
        return (input) 
               ? $filter('date')(parseInt(input.substr(6)), format) 
               : '';
    };
 }]);

</code></pre> 
            <p style="text-align: justify;">Here is the html which uses JSON date filter to display the JSON date in various formats of JavaScript date (Save controller and filter code into script.js file).</p> 
            <pre><code class="html">
&lt;!DOCTYPE html&gt;
&lt;html&gt;
&lt;head&gt;
  &lt;title&gt;Angular JSON Date Converter Filter Example&lt;/title&gt;
  &lt;script data-require="angular.js@*" 
          data-semver="1.4.2" 
          src="https://code.angularjs.org/1.4.2/angular.js"&gt;&lt;/script&gt;
  &lt;script src="script.js"&gt;&lt;/script&gt;
&lt;/head&gt;
&lt;body ng-app="jsonDateConverterExample"&gt;
  &lt;h2&gt;Angular JSON Date Converter Filter Example&lt;/h2&gt;
  &lt;div ng-controller="JsonDateController as dateCtrl"&gt;
    &lt;div&gt;
      Original JSON Date : {{sampleJsonDate}}
    &lt;/div&gt;
    &lt;h3&gt;JavaScript dates in different formats :&lt;/h3&gt;
    &lt;div&gt;{{ sampleJsonDate | jsonDate }}&lt;/div&gt;
    &lt;div&gt;{{ sampleJsonDate | jsonDate : 'yyyy-MM-dd' }}&lt;/div&gt;
    &lt;div&gt;{{ sampleJsonDate | jsonDate :'medium' }}&lt;/div&gt;
    &lt;div&gt;
      {{ sampleJsonDate | jsonDate : 'yyyy-MM-dd HH:mm:ss Z' }}
    &lt;/div&gt;
  &lt;/div&gt;
&lt;/body&gt;
&lt;/html&gt;

</code></pre>
