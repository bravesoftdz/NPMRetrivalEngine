        <h1 class="entry-title">How to Convert One Date Format to Another in PHP?</h1> 
       </header>
       <!-- .entry-header --> 
       <div class="entry-content"> 
        <div class="wwsgd" style="display:none;">
         <p style="background-color:#4965B5; color: #fff; padding:3px;border-top:5px solid #fff;">New here? <a href="http://www.facebook.com/schoolsofweb" style="color:#fff; text-decoration:underline; padding-bottom:2px;" target="_blank">Like SchoolsOfWeb on Facebook</a> to stay up to date with new posts.</p> 
        </div>
        <h2 class="h2-problem">Problem:</h2> 
        <p>You have a date (for ex. in this format, yyyy-mm-dd) and you want to change the date format to another one(ex. mm-dd-yyyy).<span id="more-1972"></span></p> 
        <h2 class="h2-solution">Solution:</h2> 
        <p>You can convert date format in PHP in few ways. In the following we’ll discuss two ways.</p> 
        <h3>Method 1: Using strtotime() and date() function</h3> 
        <p>You can convert one date format to another by the following two steps-</p> 
        <ol> 
         <li>Use s<a href="http://www.php.net/manual/en/function.strtotime.php" target="_blank">trtotime() function</a> to convert the date to its Unix timestamp, then,</li> 
         <li>Use <a href="http://www.php.net/manual/en/function.date.php" target="_blank">date() function</a> to create your desired date format from the Unix timestamp.</li> 
        </ol> 
        <p>See the example below-</p> 
        <pre class="brush: php; title: ; notranslate" title="">
&lt;?php
$timestamp = strtotime("2014-05-26");
echo date("m-d-Y", $timestamp);
?&gt;
</pre> 
        <p><strong>Output:</strong><br> 05-26-2014</p> 
        <p><strong>How it works:</strong><br> strtotime() function takes the date 2014-05-26 as string and converts it to its equivalent timestamp. Then, date() function converts the timestamp to the m-d-Y format.</p> 
        <div class="note-nl"> 
         <p><img src="http://schoolsofweb.com/wp-content/uploads/notes.png" alt=""></p> 
         <div>
          If you want to convert your date to different format check 
          <a href="http://php.net/manual/en/function.date.php" target="_blank">PHP date function</a> page to find out other formatting characters available.
         </div> 
        </div> 
        <div class="clear-nl"></div> 
        <div class="note-nl"> 
         <p><img src="http://schoolsofweb.com/wp-content/uploads/alert.png" alt=""></p> 
         <div>
          Please make sure that you followed the 
          <a href="http://www.php.net/manual/en/datetime.formats.php" target="_blank">supported date and time formats</a> when inserting date time as parameter in strtotime() function.
         </div> 
        </div> 
        <div class="clear-nl"></div> 
        <h3>Method 2: Using createFromFormat() and format() methods of the dateTIme class from SPL</h3> 
        <p>This method is better and smarter solution than the previous one. PHP provides some ready-made classes (called SPL – Standard PHP Library) to solve everyday problems. To solve date related issues, it has DateTime class. We’ll use this class.</p> 
        <pre class="brush: php; title: ; notranslate" title="">
&lt;?php
$dateObj = DateTime::createFromFormat('Y-m-d', '2014-05-26');
echo $new_date_format = $dateObj-&gt;format('m-d-Y');
?&gt;
</pre> 
        <p><strong>Output:</strong><br> 05-26-2014</p> 
        <p><strong>How it works:</strong></p> 
        <table border="0" width="100%" cellspacing="0" cellpadding="0"> 
         <tbody> 
          <tr> 
           <td align="center" valign="middle" width="14%">Line 2</td> 
           <td width="86%">First we inform PHP that the supplied date(second parameter – 2014-05-26) is in “Y-m-d” format(first parameter). The method createFromFormat() will convert the date into a new DateTime object($dateObj) based on the format.</td> 
          </tr> 
          <tr> 
           <td align="center" valign="middle">Line 3</td> 
           <td>The format() function formats the DateTIme object($dateObj) to a date according the date format specified in the parameter(‘m-d-Y’)</td> 
          </tr> 
         </tbody> 
        </table> 
        <div class="note-nl"> 
         <p><img src="http://schoolsofweb.com/wp-content/uploads/notes.png" alt=""></p> 
         <div>
          You can also convert the date format to different one. To know more about the formatting options, see 
          <a href="http://www.php.net/manual/en/datetime.createfromformat.php" target="_blank">createFromFormate() method</a>.
         </div> 
        </div>
