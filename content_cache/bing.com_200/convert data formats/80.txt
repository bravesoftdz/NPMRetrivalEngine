         <div class="entry-content"> 
          <p>Sometimes it isn’t easy to deal with Date format and comparing a date in JavaScript in MS Dynamics CRM.<br> Below is a simple function which takes a UTC date and then return back into a date only in formate YYYYMMDD.<span id="more-274"></span><br> In case you have a string date i.e. not a date then you can cast it into Date by using below<br> var completedOn = crmForm.formatedDate(completeDate);<br> // date format<br> crmForm.formatedDate = function(attr)<br> {<br> var parts = attr.split(/[T]/);<br> var fullDate= parts[0];<br> var dt = new Date();<br> parts = fullDate.split(/[-\/]/);</p> 
          <p>var y = parts[0];<br> var m = parseInt(parts[1]);<br> var d =&nbsp; parts[2];</p> 
          <p>var formatedDate =new Date(y&nbsp; + “/” + m&nbsp; + “/” + d) ;</p> 
          <p>return formatedDate;<br> }</p>
