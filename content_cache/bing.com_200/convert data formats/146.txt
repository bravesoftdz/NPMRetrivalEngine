      <p>MySQLs default DATE field format is</p>
      <blockquote>
       <p>YYYY-MM-DD</p>
      </blockquote>
      <p>and DATETIME type is a date and time combination which stores data in</p>
      <blockquote>
       <p>YYYY-MM-DD HH:MM:SS</p>
      </blockquote>
      <p>format.</p>
      <p>Whereas in Java, the Date class (available in java.util package) default format is,</p>
      <blockquote>
       <p>dow mon dd hh:mm:ss zzz yyyy</p>
      </blockquote>
      <p>where:</p>
      <ul>
       <li>dow is the day of the week (Sun, Mon, Tue, Wed, Thu, Fri, Sat).</li>
       <li>mon is the month (Jan, Feb, Mar, Apr, May, Jun, Jul, Aug, Sep, Oct, Nov, Dec).</li>
       <li>dd is the day of the month (01 through 31), as two decimal digits.</li>
       <li>hh is the hour of the day (00 through 23), as two decimal digits.</li>
       <li>mm is the minute within the hour (00 through 59), as two decimal digits.</li>
       <li>ss is the second within the minute (00 through 61, as two decimal digits.</li>
       <li>zzz is the time zone (and may reflect daylight saving time). Standard time zone abbreviations include those recognized by the method parse. If time zone information is not available, then zzz is empty  that is, it consists of no characters at all.</li>
       <li>yyyy is the year, as four decimal digits.</li>
      </ul>
      <p>So, there is a need to convert the javas default Date format to Mysqls date format. The conversion can be done at the database level (through various mysql functions) or at the program level through some java code. We will discuss only the latter part.</p>
      <p>Java provides a class called SimpleDateFormat available in java.text package which allows for formatting (date -&gt; text) through format() method and parsing (text -&gt; date) through parse() method.</p>
      <p>Here, we will convert the todays date from java Date object to mysql date format.</p>
      <p>1. Create a Date object.</p>
      <pre class="brush: java; title: ; notranslate" title="">Date now = new Date();</pre>
      <p>2. Create a SimpleDateFormat object by using the constructor,</p>
      <blockquote>
       <p>public SimpleDateFormat(String pattern)</p>
       <p>Constructs a SimpleDateFormat using the given pattern and the default date format symbols for the default locale. Note: This constructor may not support all locales. For full coverage, use the factory methods in the DateFormat class.</p>
       <p>Parameters:<br> pattern  the pattern describing the date and time format</p>
      </blockquote>
      <pre class="brush: java; title: ; notranslate" title="">String pattern = "yyyy-MM-dd";
SimpleDateFormat formatter = new SimpleDateFormat(pattern);
</pre>
      <p>3. Now use the format() method to convert the date object to text format provided in the pattern.</p>
      <pre class="brush: java; title: ; notranslate" title="">String mysqlDateString = formatter.format(now);</pre>
      <p>Complete Code:</p>
      <pre class="brush: java; title: ; notranslate" title="">import java.text.SimpleDateFormat;
import java.util.Date;

public class JavaToMysqlDate {
	public static void main(String[] args) {
		Date now = new Date();
		String pattern = "yyyy-MM-dd";
		SimpleDateFormat formatter = new SimpleDateFormat(pattern);
		String mysqlDateString = formatter.format(now);
		System.out.println("Java's Default Date Format: " + now);
		System.out.println("Mysql's Default Date Format: " + mysqlDateString);
	}
}</pre>
