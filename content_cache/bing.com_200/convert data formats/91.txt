          <h1>How Do I Convert a Date/Time String into a Timestamp Constant in LabVIEW?</h1>
          <p></p> 
          <span style="color: #dd4b39"></span>

          <br> 
          <strong>Problem:</strong> 
          <br>I have a Date/Time string in LabVIEW and I want to convert it into a Timestamp constant. How do I do this?
          <br>
          <br> 
          <strong>Solution:</strong> 
          <br>To convert the Date/Time string into a Time Stamp constant you have two primary options: 
          <br> 
          <br>
          <strong>Use the Scan from String VI </strong> 
          <ol> 
           <li>Place a Scan from String VI on the block diagram.</li> 
           <li>Right-click the Scan from String VI and select <strong>Edit Scan String</strong> from the dropdown menu. </li> 
           <li>Select <strong>Scan time stamp</strong> from the <strong>Selected operation</strong> menu. </li> 
           <li>Customize the string format using the <strong>Corresponding scan string</strong> textbox. You can use the <a href="http://zone.ni.com/reference/en-XX/help/371361M-01/glang/codes_for_time_format_str/">Time Stamp Format Codes</a> to correct format the string. Note: If you leave the format string as <code>%T</code>, LabVIEW will guess based on commonly used formats.</li> 
          </ol> 
          <br> 
          <div style="text-align: center;">
           <img src="/public.nsf/$CXIV/ATTACH-AEEE-AK7LGY/$FILE/fig1.png" width="450" height="102" alt="">
           <br>
          </div> 
          <div style="text-align: center;">
           <br>
           <strong>Figure 1: Scan From String Method</strong>
          </div> 
          <br> 
          <strong>Use the <code>NumText.Text</code> property of a Time Stamp control or indicator </strong>
          <br> 
          <ol> 
           <li>Create a Time Stamp control or indicator.</li> 
           <li>Create a Property Node for the control or indicator and select the <code>NumText.Text</code> property.</li> 
           <li>Right-click the Property Node and select<strong> Change All to Write</strong>.</li> 
           <li>Right-click the Time Stamp control or indicator and select <strong>Properties</strong>.</li> 
           <li>Navigate to the <strong>Display Format</strong> tab to customize the string format interactively (see Figure 3) </li> 
          </ol> 
          <br>
