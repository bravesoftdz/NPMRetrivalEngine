          <h1 class="entry-title ">Convertir de String a Date con java</h1> 
          <div class="entry-meta"> 
           <span class="posted-on"><i class="fa fa-calendar"></i> <a href="http://muchocodigo.com/convertir-de-string-a-date-con-java/" rel="bookmark"><time class="entry-date published" datetime="2013-08-29T22:53:26+00:00">2013-08-29</time></a></span>
           <span class="byline"> <i class="fa fa-user"></i> <span class="author vcard"><a class="url fn n" href="http://muchocodigo.com/author/muchocodigo/">muchocodigo</a></span></span>

         <div class="entry-content"> 
          <p> Lo que puede parecer algo simple suele resultar complejo cuando se está empezando con java, y este es un claro ejemplo: pasar de string a date. </p> 
          <p> Lo primero que hacemos es importar las librerías necesarias: </p> 
          <pre class="brush:java;">
import java.text.SimpleDateFormat;
import java.text.ParseException;</pre> 
          <p> &nbsp; </p> 
          <p> Después realizaremos una función con un objeto SimpleDateFormat que será construido con el formato de fecha que nos venga en el String. </p> 
          <p> La conversión se realiza con el método parse de la clase SimpleDateFormat. </p> 
          <p> Por diversas causas, como un mal formato de fecha respecto del String, se pueden dar excepciones, por lo que las controlamos con un bloque try-catch para ParseException<code>.</code> </p> 
          <pre class="brush:java;highlight: [8,11];">
    /**
     * Permite convertir un String en fecha (Date).
     * @param fecha Cadena de fecha dd/MM/yyyy
     * @return Objeto Date
     */
    public static Date ParseFecha(String fecha)
    {
        SimpleDateFormat formato = new SimpleDateFormat("dd/MM/yyyy");
        Date fechaDate = null;
        try {
            fechaDate = formato.parse(fecha);
        } 
        catch (ParseException ex) 
        {
            System.out.println(ex);
        }
        return fechaDate;
    }</pre>
