        <section class="ocpArticleTitleSection"> 
         <h1>Format a date the way you want</h1> 
        </section> 
        <section class="supAppliesToSection" ms.cmpgrp="applies_to" aria-label="This article" dir="ltr"> 
         <div id="supAppliesToTableContainer"> 
          <div id="supAppliesToList" class="appliesTo collapsed"> 
           <span>Applies To: </span> 
           <span class="appliesToItem">Excel 2016</span>

           <span class="appliesToItem">Excel for Mac 2011</span> 
           <a tabindex="0" class="appliesToOverflowControl expand ocHidden font-bold " href="javascript:" ms.interactiontype="11" ms.title="Less applies to" aria-label="Press the Enter key to listen to the other Office versions that this help article applies to.">More...</a>

        <article id="ocpArticleContent" class="ocpArticleContent"> 
         <a name="__top" aria-hidden="true"> </a> 
         <section class="ocpIntroduction"> 
          <p>When you type something like <b class="ocpLiteral">2/2</b> in a cell, Excel knows you’re typing a date and formats it based on the date setting in Control Panel. So for example, Excel might format it as <b class="ocpLiteral">2-Feb</b>. If you change your date setting in Control Panel, the default date format in Excel will change as well. If you don’t like the default date format, you can pick a different one in Excel, like <b class="ocpLiteral">February 2, 2012</b> or <b class="ocpLiteral">2/2/12</b>. You can also create your own custom format in Excel. </p> 
         </section> 
         <section class="ocpSection"> 
          <h2>What do you want to do? </h2> 
          <a name="___toc___1_l" aria-hidden="true"> </a> 
          <dl class="ocpTOC"> 
           <dd> 
            <p> <a href="#__toc320186182" class="ocpInlineLink" aria-label="Skip to section Choose from a list of date formats">Choose from a list of date formats</a> </p> 
           </dd> 
           <dd> 
            <p> <a href="#__toc320186183" class="ocpInlineLink" aria-label="Skip to section Create a custom date format">Create a custom date format</a> </p> 
           </dd> 
           <dd> 
            <p> <a href="#__migbm_0" class="ocpInlineLink" aria-label="Skip to section Tips for displaying dates">Tips for displaying dates</a> </p> 
           </dd> 
          </dl> 
         </section> 
         <a name="__toc320186182" aria-hidden="true"> </a> 
         <section class="ocpSection"> 
          <h2>Choose from a list of date formats</h2> 
          <ol type="1" itemscope itemtype="http://schema.org/ItemList"> 
           <li itemprop="itemListElement" itemscope itemtype="http://schema.org/ListItem"> <p itemprop="name">Select the cells you want to format.</p> 
            <meta itemprop="position" content="1"> </li> 
           <li itemprop="itemListElement" itemscope itemtype="http://schema.org/ListItem"> <p itemprop="name">Press CTRL+1.</p> <p>On a Mac, press Control+1 or Command+1.</p> 
            <meta itemprop="position" content="2"> </li> 
           <li itemprop="itemListElement" itemscope itemtype="http://schema.org/ListItem"> <p itemprop="name">In the <b class="ocpUI">Format Cells</b> box, click the <b class="ocpUI">Number</b> tab.</p> 
            <meta itemprop="position" content="3"> </li> 
           <li itemprop="itemListElement" itemscope itemtype="http://schema.org/ListItem"> <p itemprop="name">In the <b class="ocpUI">Category</b> list, click <b class="ocpUI">Date</b>.</p> 
            <meta itemprop="position" content="4"> </li> 
          </ol> 
          <p> <img src="https://support.content.office.net/en-us/media/4d8a65ab-8eff-46b8-8de8-9bde34e18a11.jpg" alt="Pick Date in the Category list" title="Pick Date in the Category list"> </p> 
          <ol type="1" itemscope itemtype="http://schema.org/ItemList"> 
           <li itemprop="itemListElement" itemscope itemtype="http://schema.org/ListItem"> <p itemprop="name">Under <b class="ocpUI">Type</b>, pick a date format. Your format will preview in the <b class="ocpUI">Sample</b> box with the first date in your data.</p> 
            <meta itemprop="position" content="1"> </li> 
          </ol> 
          <a name="__goback" aria-hidden="true"> </a> 
          <p> <img src="https://support.content.office.net/en-us/media/2d9513de-9959-4182-8073-7e7fbcf4158c.jpg" alt="Pick the date type and then see a preview in the Sample box" title="Pick the date type and then see a preview in the Sample box"> </p> 
          <div class="ocpAlert"> 
           <p class="ocpAlertSection"> <b class="ocpNote">Note:</b>&nbsp; Date formats that begin with an asterisk (*) will change if you change the regional date and time settings in Control Panel. Formats without an asterisk won’t change.</p> 
          </div> 
          <ol type="1" itemscope itemtype="http://schema.org/ItemList"> 
           <li itemprop="itemListElement" itemscope itemtype="http://schema.org/ListItem"> <p itemprop="name">If you want to use a date format based on how another language displays dates, pick the language under <b class="ocpUI">Locale (location)</b>, </p> 
            <div class="ocpAlert"> 
             <p class="ocpAlertSection"> <b class="ocpNote">Tip:</b>&nbsp; Do you have numbers showing up in your cells as #####? This probably means your cell isn’t wide enough to show the whole number. Try double-clicking the right border of the column that contains the cells with #####. This will resize the column to fit the number. You can also drag the right border of the column to make it any size you want.</p> 
            </div> 
            <meta itemprop="position" content="1"> </li> 
          </ol> 
         </section> 
         <a name="__toc320186183" aria-hidden="true"> </a> 
         <section class="ocpSection"> 
          <h2>Create a custom date format</h2> 
          <p>If you want to use a format that isn’t in the <b class="ocpUI">Type</b> box, you can create your own. The easiest way to do this is to start from a format this is close to what you want.</p> 
          <ol type="1" itemscope itemtype="http://schema.org/ItemList"> 
           <li itemprop="itemListElement" itemscope itemtype="http://schema.org/ListItem"> <p itemprop="name">Select the cells you want to format.</p> 
            <meta itemprop="position" content="1"> </li> 
           <li itemprop="itemListElement" itemscope itemtype="http://schema.org/ListItem"> <p itemprop="name">Press CTRL+1.</p> <p>On a Mac, press Control+1 or Command+1.</p> 
            <meta itemprop="position" content="2"> </li> 
           <li itemprop="itemListElement" itemscope itemtype="http://schema.org/ListItem"> <p itemprop="name">In the <b class="ocpUI">Format Cells</b> box, click the <b class="ocpUI">Number</b> tab.</p> 
            <meta itemprop="position" content="3"> </li> 
           <li itemprop="itemListElement" itemscope itemtype="http://schema.org/ListItem"> <p itemprop="name">In the <b class="ocpUI">Category</b> list, click <b class="ocpUI">Date</b>, and then under <b class="ocpUI">Type</b>, pick a date format that is close to the format you want.</p> 
            <meta itemprop="position" content="4"> </li> 
          </ol> 
          <p> <img src="https://support.content.office.net/en-us/media/4d8a65ab-8eff-46b8-8de8-9bde34e18a11.jpg" alt="Pick Date in the Category list" title="Pick Date in the Category list"> </p> 
          <ol type="1" itemscope itemtype="http://schema.org/ItemList"> 
           <li itemprop="itemListElement" itemscope itemtype="http://schema.org/ListItem"> <p itemprop="name">Go back to the <b class="ocpUI">Category</b> list, and pick <b class="ocpUI">Custom</b>. Under <b class="ocpUI">Type</b>, you’ll see the format code for the date format you picked in step 4. The built-in date format can’t be changed, so don’t worry about messing it up. The changes you make will only apply to the custom format you’re creating.</p> 
            <meta itemprop="position" content="1"> </li> 
           <li itemprop="itemListElement" itemscope itemtype="http://schema.org/ListItem"> <p itemprop="name">In the <b class="ocpUI">Type</b> box, make the changes you want using code from the table below.</p> 
            <meta itemprop="position" content="2"> </li> 
          </ol> 
          <table id="tblID0EBBCAAA" class="banded"> 
           <thead> 
            <tr> 
             <td> <p> <b class="ocpLegacyBold">To display</b> </p> </td> 
             <td> <p> <b class="ocpLegacyBold">Use this code</b> </p> </td> 
            </tr> 
           </thead> 
           <tbody> 
            <tr> 
             <td> <p>Months as 1–12</p> </td> 
             <td> <p>m</p> </td> 
            </tr> 
            <tr> 
             <td> <p>Months as 01–12</p> </td> 
             <td> <p>mm</p> </td> 
            </tr> 
            <tr> 
             <td> <p>Months as Jan–Dec</p> </td> 
             <td> <p>mmm</p> </td> 
            </tr> 
            <tr> 
             <td> <p>Months as January–December</p> </td> 
             <td> <p>mmmm</p> </td> 
            </tr> 
            <tr> 
             <td> <p>Months as the first letter of the month</p> </td> 
             <td> <p>mmmmm</p> </td> 
            </tr> 
            <tr> 
             <td> <p>Days as 1–31</p> </td> 
             <td> <p>d</p> </td> 
            </tr> 
            <tr> 
             <td> <p>Days as 01–31</p> </td> 
             <td> <p>dd</p> </td> 
            </tr> 
            <tr> 
             <td> <p>Days as Sun–Sat</p> </td> 
             <td> <p>ddd</p> </td> 
            </tr> 
            <tr> 
             <td> <p>Days as Sunday–Saturday</p> </td> 
             <td> <p>dddd</p> </td> 
            </tr> 
            <tr> 
             <td> <p>Years as 00–99</p> </td> 
             <td> <p>yy</p> </td> 
            </tr> 
            <tr> 
             <td> <p>Years as 1900–9999</p> </td> 
             <td> <p>yyyy</p> </td> 
            </tr> 
           </tbody> 
          </table> 
          <p>If you’re modifying a format that includes time values and you use "m" immediately after the "h" or "hh" code or immediately before the "ss" code, Excel displays minutes instead of the month.</p> 
         </section> 
         <a name="__migbm_0" aria-hidden="true"> </a> 
         <section class="ocpSection"> 
          <h2>Tips for displaying dates</h2> 
          <ul> 
           <li> <p>To quickly use the default date format, click the cell with the date, and then press CTRL+SHIFT+#.</p> </li> 
           <li> <p>If a cell displays <b class="ocpLegacyBold">#####</b> after you apply date formatting to it, the cell probably isn’t wide enough to show the whole number. Try double-clicking the right border of the column that contains the cells with #####. This will resize the column to fit the number. You can also drag the right border of the column to make it any size you want.</p> </li> 
           <li> <p>To quickly enter the current date in your worksheet, select any empty cell, press CTRL+; (semicolon), and then press ENTER, if necessary. </p> </li> 
           <li> <p>To enter a date that will update to the current date each time you reopen a worksheet or recalculate a formula, type <b class="ocpLiteral">=TODAY()</b> in an empty cell, and then press ENTER.</p> </li> 
          </ul> 
         </section>
