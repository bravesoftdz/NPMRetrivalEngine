            <a href="http://www.ejbtutorial.com/java-rmi/a-step-by-step-implementation-tutorial-for-java-rmi" target="_blank"><strong>In case you are new to Java RMI, You need to follow this simple tutorial to learn RMI</strong></a>

          <p>In this tutorial, we will create a simple chat program using two users. One user will be chatting from the server side, whilst the other user would be chatting from the client side of the RMI application. The structure of the files for the projects created using Eclipse throughout this tutorials is shown below:</p> 
          <p><img class="aligncenter size-full wp-image-252" alt="rmi-steps21" src="http://www.ejbtutorial.com/wp-content/uploads/2013/10/rmi-steps21.png" width="390" height="290" srcset="http://www.ejbtutorial.com/wp-content/uploads/2013/10/rmi-steps21.png 390w, http://www.ejbtutorial.com/wp-content/uploads/2013/10/rmi-steps21-300x223.png 300w" sizes="(max-width: 390px) 100vw, 390px"></p> 
          <h3>1. Server Side:</h3> 
          <p><span class="step">1</span> Lets create a new <strong>Java Project</strong> using Eclipse ( or NetBeans or other editor you prefer), and call it : <strong>SimpleChatServer</strong>-&gt; Click <strong>Finish</strong> once done</p> 
          <p><span class="step">2</span> Under the <strong>SimpleChatServer</strong> project, Select <strong>New -&gt; Interface</strong>, set the name for the interface as : <strong>ChatInterface</strong>. The Java code for the interface is as below:</p> 
          <pre class="wp-code-highlight prettyprint linenums:1">import java.rmi.*;

public interface ChatInterface extends Remote{
	public String getName() throws RemoteException;
	public void send(String msg) throws RemoteException;
	public void setClient(ChatInterface c)throws RemoteException;
	public ChatInterface getClient() throws RemoteException;
}</pre> 
          <p><span class="step">3</span> Under the <strong>SimpleChatServer</strong> project, Select <strong>New -&gt; Class</strong>, set the name for the class as : <strong>Chat</strong>. The Java code for the class is as below:</p> 
          <pre class="wp-code-highlight prettyprint linenums:1">import java.rmi.*;
import java.rmi.server.*;

public class Chat extends UnicastRemoteObject implements ChatInterface  {

	public String name;
	public ChatInterface client=null;

	public Chat(String n)  throws RemoteException { 
		this.name=n;   
	}
	public String getName() throws RemoteException {
		return this.name;
	}

	public void setClient(ChatInterface c){
		client=c;
	}

	public ChatInterface getClient(){
		return client;
	}

	public void send(String s) throws RemoteException{
		System.out.println(s);
	}	
}</pre> 
          <p><span class="step">4</span> Under the <strong>SimpleChatServer</strong> project, Select <strong>New -&gt; Class</strong>, set the name for the class as : <strong>ChatServer.</strong> The Java code for the class is as below:</p> 
          <pre class="wp-code-highlight prettyprint linenums:1">import java.rmi.*;
import java.rmi.server.*;
import java.util.*;

public class ChatServer {
public static void main (String[] argv) {
    try {
	    	System.setSecurityManager(new RMISecurityManager());
	    	Scanner s=new Scanner(System.in);
	    	System.out.println("Enter Your name and press Enter:");
	    	String name=s.nextLine().trim();

	    	Chat server = new Chat(name);	

	    	Naming.rebind("rmi://localhost/ABC", server);

	    	System.out.println("[System] Chat Remote Object is ready:");

	    	while(true){
	    		String msg=s.nextLine().trim();
	    		if (server.getClient()!=null){
	    			ChatInterface client=server.getClient();
	    			msg="["+server.getName()+"] "+msg;
	    			client.send(msg);
	    		}	
	    	}

    	}catch (Exception e) {
    		System.out.println("[System] Server failed: " + e);
    	}
	}
}</pre> 
          <p><span class="step">5</span> Compile your program.</p> 
          <p><span class="step">6</span> Run the<strong> rmic</strong> from the console ( cmd ) to generate the stub for the <strong>Chat</strong> class.</p> 
          <p><span class="step">7</span> Create the policy file for the security with the following code:</p> 
          <pre class="wp-code-highlight prettyprint linenums:1">grant {
     	permission java.security.AllPermission;
  };</pre> 
          <p><span class="step">8</span> Set the VM Arguments for your project using the following :<br> </p>
          <div class="snvdshc">
           <div class="synved-box-message synved-box-info">
            <strong>-Djava.security.policy=file:${workspace_loc}/SimpleChatServer/security.policy -Djava.rmi.server.codebase=file:${workspace_loc}/SimpleChatServer/bin/</strong>
           </div>
          </div>
          <p></p> 
          <p><span class="step">9</span> Once your Run your server, you will be asked to type in your name as shown below:</p> 
          <p><img class="aligncenter size-full wp-image-246" alt="rmi-steps19" src="http://www.ejbtutorial.com/wp-content/uploads/2013/10/rmi-steps19.png" width="512" height="184" srcset="http://www.ejbtutorial.com/wp-content/uploads/2013/10/rmi-steps19.png 512w, http://www.ejbtutorial.com/wp-content/uploads/2013/10/rmi-steps19-300x107.png 300w" sizes="(max-width: 512px) 100vw, 512px"></p> 
          <h3>2. Client Side:</h3> 
          <p><span class="step">1</span> Lets create a new <strong>Java Project</strong> using Eclipse and call it : <strong>SimpleChatClient</strong>-&gt; Click <strong>Finish</strong> once done</p> 
          <p><span class="step">2</span> Under the <strong>SimpleChatClient</strong> project, Select <strong>New -&gt; Class</strong>, set the name for the class as : <strong>ChatClient.</strong> The Java code for the class is as below:</p> 
          <pre class="wp-code-highlight prettyprint linenums:1">import java.rmi.*;
import java.rmi.server.*;
import java.util.*;

public class ChatClient {
	public static void main (String[] argv) {
	    try {
		    	System.setSecurityManager(new RMISecurityManager());
		    	Scanner s=new Scanner(System.in);
		    	System.out.println("Enter Your name and press Enter:");
		    	String name=s.nextLine().trim();		    		    	
		    	ChatInterface client = new Chat(name);

		    	ChatInterface server = (ChatInterface)Naming.lookup("rmi://localhost/ABC");
		    	String msg="["+client.getName()+"] got connected";
		    	server.send(msg);
		    	System.out.println("[System] Chat Remote Object is ready:");
		    	server.setClient(client);

		    	while(true){
		    		msg=s.nextLine().trim();
		    		msg="["+client.getName()+"] "+msg;		    		
	    			server.send(msg);
		    	}

	    	}catch (Exception e) {
	    		System.out.println("[System] Server failed: " + e);
	    	}
		}
}</pre> 
          <p><span class="step">3</span> Exactly the same way as the server side, create the interface : ChatInterface, class: Chat, Policy: security.policy within the Client Project.</p> 
          <p><span class="step">4</span> You can run the RMIC from the client also.</p> 
          <p><span class="step">5</span> Set the VM Arguments for the client as :<br> </p>
          <div class="snvdshc">
           <div class="synved-box-message synved-box-info">
            <strong>-Djava.security.policy=file:${workspace_loc}/SimpleChatClient/security.policy -Djava.rmi.server.codebase=file:${workspace_loc}/SimpleChatClient/bin/</strong>
           </div>
          </div>
          <p></p> 
          <p><span class="step">6</span> Run the client side and start Chatting !</p>
