          <p>As a Java developer you probably used to know about reflexion. However, in order to keep your software architecture flexible, some functionalities are sometimes not provided out of the box by the JVM.<br> In my particular case, I needed to find out every Class and Sub-Classes inside a package, thus reparteed within several Jars.<br> Internet has lots of solution, but it remains complicated for everybody to reach this goal. After googleing, I found a link which provided a partial solution. I would like to thank the website author:</p> 
          <p><a href="http://www.java2s.com/Code/Java/Reflection/Attemptstolistalltheclassesinthespecifiedpackageasdeterminedbythecontextclassloader.htm">http://www.java2s.com/Code/Java/Reflection/Attemptstolistalltheclassesinthespecifiedpackageasdeterminedbythecontextclassloader.htm</a></p> 
          <p>Some other solution invited us to deploy external libraries as well. But I was not interested to manage another lib in my soft just for that purpose.<br> So, the solution was to recover all jars from the context classloader and loop on them in order to find out the classes we are looking for.<br> Following, you will see a complete Java class resolving this issue:</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">import java.io.File;</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">import java.io.IOException;</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">import java.io.UnsupportedEncodingException;</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">import java.net.URL;</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">import java.net.URLClassLoader;</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">import java.net.URLDecoder;</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">import java.util.ArrayList;</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">import java.util.Enumeration;</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">import java.util.HashMap;</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">import java.util.List;</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">import java.util.jar.JarEntry;</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">import java.util.jar.JarFile;</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">/**</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp;*</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp;*</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp;*</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp;* @author Philippe Schweitzer dbi services Switzerland</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp;*</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp;*/</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">public class ClassFinder {</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; public static void main(String[] args) throws ClassNotFoundException {</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; List&lt;Class&gt; classes = ClassFinder.getClassesFromPackage(“YOUR PACKAGE NAME”);</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; System.out.println(“START ClassList:”);</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; for (Class c : classes) {</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; System.out.println(c.toString());// + ” ” + c.getCanonicalName());</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; }</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; System.out.println(“END ClassList:”);</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; }</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; /**</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp;*</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp;* Attempts to list all the classes in the specified package as determined &nbsp; &nbsp; *</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp;* by the context class loader…</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp;*</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp;* @param pckgname the package name to search</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp;* @return a list of classes that exist within that package</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp;* @throws ClassNotFoundException if something went wrong</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp;*</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp;*/</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; public static List getClassesFromPackage(String pckgname) throws ClassNotFoundException {</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; ArrayList result = new ArrayList();</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; ArrayList&lt;File&gt; directories = new ArrayList();</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; HashMap packageNames = null;</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; try {</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; for (URL jarURL : ((URLClassLoader) Thread.currentThread().getContextClassLoader()).getURLs()) {</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; System.out.println(“JAR: ” + jarURL.getPath());</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; getClassesInSamePackageFromJar(result, pckgname, jarURL.getPath());</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; String path = pckgname;</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; Enumeration&lt;URL&gt; resources = cld.getResources(path);</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; File directory = null;</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; while (resources.hasMoreElements()) {</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; String path2 = resources.nextElement().getPath();</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; directory = new File(URLDecoder.decode(path2, “UTF-8?));</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; directories.add(directory);</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; }</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; if (packageNames == null) {</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; packageNames = new HashMap();</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; }</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; packageNames.put(directory, pckgname);</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; }</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; } catch (NullPointerException x) {</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; throw new ClassNotFoundException(pckgname + ” does not appear to be a valid package (Null pointer exception)”);</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; } catch (UnsupportedEncodingException encex) {</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; throw new ClassNotFoundException(pckgname + ” does not appear to be a valid package (Unsupported encoding)”);</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; } catch (IOException ioex) {</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; throw new ClassNotFoundException(“IOException was thrown when trying to get all resources for ” + pckgname);</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; }</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; for (File directory : directories) {</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; if (directory.exists()) {</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; String[] files = directory.list();</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; for (String file : files) {</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; if (file.endsWith(“.class”)) {</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; try {</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; // &nbsp; &nbsp; &nbsp;System.out.println(packageNames.get(directory).toString() + ‘.’ + file.substring(0, file.length() – 6));</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; result.add(Class.forName(packageNames.get(directory).toString() + ‘.’ + file.substring(0, file.length() – 6)));</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; } catch (Throwable e) {</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; }</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; }</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; }</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; } else {</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; throw new ClassNotFoundException(pckgname + ” (” + directory.getPath() + “) does not appear to be a valid package”);</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; }</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; }</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; return result;</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; }</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; /**</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp;*</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp;* Returns the list of classes in the same directories as Classes in</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp;* classes.</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp;*</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp;* @param result</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp;* @param classes</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp;* @param jarPath</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp;*</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp;*/</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; private static void getClassesInSamePackageFromJar(List result, String packageName, String jarPath) {</p>

          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; Enumeration&lt;JarEntry&gt; en = jarFile.entries();</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; while (en.hasMoreElements()) {</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; JarEntry entry = en.nextElement();</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; String entryName = entry.getName();</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; packageName = packageName.replace(‘.’, ‘/’);</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; if (entryName != null &amp;&amp; entryName.endsWith(“.class”) &amp;&amp; entryName.startsWith(packageName)) {</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; try {</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; Class entryClass = Class.forName(entryName.substring(0, entryName.length() – 6).replace(‘/’, ‘.’));</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; if (entryClass != null) {</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; result.add(entryClass);</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; }</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; } catch (Throwable e) {</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">// do nothing, just continue processing classes</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; }</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; }</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; }</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; } catch (Exception e) {</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; } finally {</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; try {</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; if (jarFile != null) {</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; jarFile.close();</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; }</p> 
          <p style="margin-bottom: 0cm; background: none repeat scroll 0% 0% #e6e6ff; padding: 0cm;" align="LEFT">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; } catch (Exception e) {</p>
