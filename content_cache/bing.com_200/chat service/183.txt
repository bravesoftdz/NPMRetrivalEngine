              <p><strong>OR:</strong><br><br></p> 
              <p><strong>2.</strong> <strong>Install Install Oracle Java 8 (JDK 8u5 - lastest and stable version) in Ubuntu:</strong></p> 
              <p><strong>Step 1:</strong> Install Java 8 (JDK 8) Add the webupd8team java PPA repository in our system and install Oracle java8 using following set of commands:</p> 
              <pre><code>sudo add-apt-repository ppa:webupd8team/java
sudo apt-get update
sudo apt-get install oracle-java8-installer
</code></pre> 
              <p><br><strong>Step 2:</strong> Verify JAVA Version After successfully installing oracle java using above step verify installed version using following command:</p> 
              <pre><code>java -version
</code></pre> 
              <p><br><strong>Step 3:</strong> Setup JAVA Environment Webupd8team is providing a package to set environment variables, Install this package using following command:</p> 
              <pre><code>sudo apt-get install oracle-java8-set-default
</code></pre>

            <td class="answercell"> 
             <div class="post-text" itemprop="text"> 
              <p>Personally I don't like using package managers to install the JDK(and by extension the JRE) I typically perform the following:</p> 
              <ul> 
               <li>Down jdk from <a href="http://www.oracle.com/technetwork/java/javase/downloads/index-jsp-138363.html#JDK7" rel="nofollow">http://www.oracle.com/technetwork/java/javase/downloads/index-jsp-138363.html#JDK7</a></li> 
               <li>Copy to ~/dev/tools/jdk</li> 
               <li>untar the file</li> 
               <li>Edit your ~/.bashrc file and add <strong>export JAVA_HOME=~/dev/tools/jdk/jdk1.7.0_55</strong>. Update you PATH environment variable in your .bashrc as well <strong>export PATH=$JAVA_HOME/bin:$PATH</strong></li> 
               <li>Close the current terminal and open a new one.</li> 
               <li>Type <strong>java -version</strong></li> 
              </ul>
