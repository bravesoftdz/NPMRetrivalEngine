      <h2>What is JS Charts?</h2> 
      <p> JS Charts is a JavaScript based chart generator that requires little or no coding. With JS Charts drawing charts is a simple and easy task, since you only have to use client-side scripting (i.e. performed by your web browser). No additional plugins or server modules are required. Just include our scripts, prepare your chart data in XML, JSON or JavaScript Array and your chart is ready! </p> 
      <p> JS Charts lets you create charts in different templates like <a href="/how-to-use-bar-charts">bar charts</a>, <a href="/how-to-use-pie-charts">pie charts</a> or simple <a href="/how-to-use-line-graphs">line graphs</a>. </p>
