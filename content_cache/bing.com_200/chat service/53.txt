        <br>
        <p>The Client<br> ———</p> 
        <p><a href="https://linuxjunkies.files.wordpress.com/2011/03/client1.jpg"><img data-attachment-id="219" data-permalink="https://linuxjunkies.wordpress.com/2011/03/10/a-simple-chat-server-in-java-using-threads-part-1/client-2/" data-orig-file="https://linuxjunkies.files.wordpress.com/2011/03/client1.jpg?w=660" data-orig-size="401,395" data-comments-opened="1" data-image-meta="{&quot;aperture&quot;:&quot;0&quot;,&quot;credit&quot;:&quot;&quot;,&quot;camera&quot;:&quot;&quot;,&quot;caption&quot;:&quot;&quot;,&quot;created_timestamp&quot;:&quot;0&quot;,&quot;copyright&quot;:&quot;&quot;,&quot;focal_length&quot;:&quot;0&quot;,&quot;iso&quot;:&quot;0&quot;,&quot;shutter_speed&quot;:&quot;0&quot;,&quot;title&quot;:&quot;&quot;}" data-image-title="client" data-image-description="" data-medium-file="https://linuxjunkies.files.wordpress.com/2011/03/client1.jpg?w=660?w=300" data-large-file="https://linuxjunkies.files.wordpress.com/2011/03/client1.jpg?w=660?w=401" src="https://linuxjunkies.files.wordpress.com/2011/03/client1.jpg?w=660" alt="" title="client" class="aligncenter size-full wp-image-219" srcset="https://linuxjunkies.files.wordpress.com/2011/03/client1.jpg 401w, https://linuxjunkies.files.wordpress.com/2011/03/client1.jpg?w=150 150w, https://linuxjunkies.files.wordpress.com/2011/03/client1.jpg?w=300 300w" sizes="(max-width: 401px) 100vw, 401px"></a></p> 
        <p>The client side first. We import the necessary stuff&nbsp; needed for this program. The java.net.* contains all<br> the necessary packages for the networking part. The swing and awt packages are for the GUI.</p> 
        <pre class="brush: java; title: ; notranslate" title="">
import java.io.*;
import java.util.*;
import javax.swing.*;
import java.net.*;
import java.awt.*;
import java.awt.event.*;

public class Client {
</pre> 
        <p>The JTextArea will be used to display all the chat proceedings in real time. The objective of this program is to have mutiple clients connect to a server and receive the information passed to them in real time. We make use of threads in this case.</p> 
        <p>JTextField is used for passing the chat comments.<br> JButton is ofcourse used for sending the message.</p> 
        <pre class="brush: java; title: ; notranslate" title="">
JTextArea incoming;
JTextField outgoing;
JButton sendbutton;
</pre> 
        <p>BufferedReader is used to buffer the inputstream from another source.<br> We use the PrintWriter to print to the socket. It is similar to using the system println function.</p> 
        <pre class="brush: java; title: ; notranslate" title="">
BufferedReader reader;
PrintWriter writer;
</pre> 
        <p>Sockets<br> ——–<br> Sockets form the important part of this program. Simply put, it is an IP addreass with a port number. Now, port numbers vary from about 0 to 65535. It should be noted that the port numbers from 0 to 1023 are reserved and should not, unless for a reason, be used. In this example we will be using the port number 5000 to connect to the server.</p> 
        <pre class="brush: java; title: ; notranslate" title="">
Socket sock;
</pre> 
        <p>The function go()<br> —————–<br> This is the first method to be called. This does the following:<br> 1. Sets up a GUI environment like I mentioned in previous posts.<br> 2. Calls the setUpNetworking() function which will be explained later. Basically, it sets up the networking environment.<br> 3. Most importantly, we start the threads, which I will explain.</p> 
        <pre class="brush: java; title: ; notranslate" title="">
public void go(){
JFrame frame = new JFrame("Client");
JPanel mainpanel = new JPanel();

incoming = new JTextArea(15,50);
incoming.setLineWrap(true);
incoming.setWrapStyleWord(true);
incoming.setEditable(false);

JScrollPane qScroller = new JScrollPane(incoming);
qScroller.setVerticalScrollBarPolicy(ScrollPaneConstants.VERTICAL_SCROLLBAR_ALWAYS);
qScroller.setHorizontalScrollBarPolicy(ScrollPaneConstants.HORIZONTAL_SCROLLBAR_ALWAYS);

outgoing = new JTextField(20);
sendbutton = new JButton("Send");

sendbutton.addActionListener(new SendButtonListener());

mainpanel.add(qScroller);
mainpanel.add(outgoing);
mainpanel.add(sendbutton);

setUpNetworking();
</pre> 
        <p>Now what this&nbsp; does is set up a mainpanel which is JPanel. On it we add a JTextArea, which itself is added onto a JScrollPane as seen from the code. The different paramenters of this are set accordingly. The JButton is added and ofcourse, the ActionListener() for the button.</p> 
        <pre class="brush: java; title: ; notranslate" title="">
Thread readerThread = new Thread(new IncomingReader());
readerThread.start();
</pre> 
        <p>This is how a thread is created. Here we pass a new object of the class IncomingReader which implements the Runnable class, which I will explain in a moment.</p> 
        <pre class="brush: java; title: ; notranslate" title="">
frame.getContentPane().add(BorderLayout.CENTER, mainpanel);
frame.setSize(400, 400);
frame.setVisible(true);
</pre> 
        <p>}</p> 
        <p>These are just basic frame housekeeping stuff.</p> 
        <pre class="brush: java; title: ; notranslate" title="">
public static void main(String[] args){
Client client = new Client();
client.go();
}
</pre> 
        <p>The program begins here ofcourse. An object of the type client is created and the go() function is called.</p> 
        <pre class="brush: java; title: ; notranslate" title="">
private void setUpNetworking(){

try{

sock = new Socket("127.0.0.1", 5000);
</pre> 
        <p>This step creates a socket at the address “127.0.0.1” which is the localhost, ie, the same computer. And we select arbitary socket number of 5000. This number is known to the server.</p> 
        <pre class="brush: java; title: ; notranslate" title="">
InputStreamReader streamReader = new InputStreamReader(sock.getInputStream());
</pre> 
        <p>The InputStreamReader reads the bytes of data coming from the socket inputstream.</p> 
        <pre class="brush: java; title: ; notranslate" title="">
reader = new BufferedReader(streamReader);
</pre> 
        <p>This is then passed on to the BufferedReader object.</p> 
        <pre class="brush: java; title: ; notranslate" title="">
writer = new PrintWriter(sock.getOutputStream());
</pre> 
        <p>The PrintWriter object is finally responsible for putting the message onto the output stream.</p> 
        <pre class="brush: java; title: ; notranslate" title="">
System.out.println("Networking established!!");

}catch(IOException e){
e.printStackTrace();
}
}
</pre>
