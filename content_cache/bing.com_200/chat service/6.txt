      <section class="step-container">
       <h2 id="intro" class="step-title intro-title">Introduction: Creating a Chat Server Using Java</h2>
       <div class="photoset">
        <img alt="Picture of Creating a Chat Server Using Java" src="https://cdn.instructables.com/FSS/K8A1/ILJWGR24/FSSK8A1ILJWGR24.MEDIUM.jpg">
       </div>
       <div class="step-body">
        <p><strong>Introduction</strong></p>
        <p>Networking is a major branch of programming that is vital to connecting users through devices. As such many programming languages have multiple ways to form connections users and servers or between peers. For starting out programming, Java is one of the first languages many programmers learn, and one of the interesting ways java can handle network connections is through the use of Java Sockets. I have prepared this tutorial to instruct how to use sockets in java by developing a chat server between one server and many users. This tutorial is divided between client side and server side development.</p>
        <p><strong>Requirements:</strong></p>
        <ul>
         <li>Java JDK (I used version 1.8.0_74 for this) </li>
         <li>Java IDE (I used Eclipse Mars for this) </li>
         <li>A package structure with the above classes</li>
        </ul>
        <p><strong>Timer: </strong>About 15-30 minutes</p>
        <p><strong>Skill: </strong>This guide assumes that you have basic Java experience and are familiar with common Java I/O types</p>
       </div>
      </section>
      <section class="step-container">
       <h2 id="step1" class="step-title">Step 1: Setup a ServerSocket in the Server Class</h2>
       <div class="photoset">
        <img alt="Picture of Setup a ServerSocket in the Server Class" src="https://cdn.instructables.com/F97/H6JC/ILV8GY1T/F97H6JCILV8GY1T.MEDIUM.jpg">
       </div>
       <div class="step-body">
        <p>Create a static Server socket at the beginning of main in the ChatServer class and instantiate it with a port number.</p>
        <p>TIP: ServerSockets may be instantiated with a port number other than 4444, but numbers lower than 1000 are more likely to already be in use by the system.</p>
       </div>
      </section>
      <section class="step-container">
       <h2 id="step2" class="step-title">Step 2: Create a Socket in the Login Class</h2>
       <div class="photoset">
        <img alt="Picture of Create a Socket in the Login Class" src="https://cdn.instructables.com/FHH/A9GO/IM55XBVN/FHHA9GOIM55XBVN.MEDIUM.jpg">
       </div>
       <div class="step-body">
        <p>Create a socket in the main method of Login, specifying the host address and port number. Since this is a login window, the socket can be an instance variable as it will be passed to the actual chat window later.</p>
        <p>TIP: "localhost" is used in the socket parameters for debugging purposes to specify that the server is running on the same computer as the client. Later this can be changed to take a host address to connect to a remote server.</p>
       </div>
      </section>
      <section class="step-container">
       <h2 id="step3" class="step-title">Step 3: Create a Loop to Continuously Accept Clients</h2>
       <div class="photoset">
        <img alt="Picture of Create a Loop to Continuously Accept Clients" src="https://cdn.instructables.com/FK2/O5Z9/IM55XOL1/FK2O5Z9IM55XOL1.MEDIUM.jpg">
       </div>
       <div class="step-body">
        <p>After instantiating the serverSocket in chatServer, start a while loop that continuously accepts all clients.</p>
       </div>
      </section>
      <section class="step-container">
       <h2 id="step4" class="step-title">Step 4: Create the Client Threads</h2>
       <div class="photoset">
        <img alt="Picture of Create the Client Threads" src="https://cdn.instructables.com/FVX/2L9R/IM55XGZE/FVX2L9RIM55XGZE.MEDIUM.jpg">
       </div>
       <div class="step-body">
        <p></p>
        <ol>
         <li>With the socket received from serversocket.accept, create a new ClientThread . </li>
         <li>Create a new Thread using the ClientThread, and then call start on the thread.</li>
         <li>In ClientThread, make ClientThread extend ChatServer, and implement Runnable</li>
         <li>Create the ClientThread constructor and add the method run.</li>
        </ol>
        <p></p>
       </div>
      </section>
      <section class="step-container">
       <h2 id="step5" class="step-title">Step 5: Create the Server Thread</h2>
       <div class="photoset">
        <img alt="Picture of Create the Server Thread" src="https://cdn.instructables.com/F96/R1N3/IM55XV0V/F96R1N3IM55XV0V.MEDIUM.jpg">
       </div>
       <div class="step-body">
        <ol>
         <li>Create a new ServerThread using socket. </li>
         <li>Create a new Thread using the ServerThread, and then call start on the thread.</li>
         <li>In ServerThread, make ServerThread implement Runnable</li>
         <li>Create the ServerThread constructor and add the method run.</li>
        </ol>
       </div>
      </section>
      <section class="step-container">
       <h2 id="step6" class="step-title">Step 6: Make the Client Thread Send and Receive Data</h2>
       <div class="photoset">
        <img alt="Picture of Make the Client Thread Send and Receive Data" src="https://cdn.instructables.com/FNI/S9BQ/IM55XHUK/FNIS9BQIM55XHUK.MEDIUM.jpg">
       </div>
       <div class="step-body">
        <p></p>
        <ol>
         <li>In ClientThread make a private BufferedReader in to receive data from clients, and a PrintWriter to write to the client.</li>
         <li>Create a getter for the PrintWriter</li>
         <li>Initialize the PrintWriter in run with the socket's output stream, and the BufferedReader with a new InputStreamReader using the socket's input stream</li>
         <li>Create a while loop in run that checks for any new input and prints the input to all clients using the list of ClientThreads and the getter for the PrintWriter</li>
        </ol>
        <p></p>
       </div>
      </section>
      <section class="step-container">
       <h2 id="step7" class="step-title">Step 7: Make the Server Thread Send and Receive Data</h2>
       <div class="photoset">
        <img alt="Picture of Make the Server Thread Send and Receive Data" src="https://cdn.instructables.com/F29/WMGR/IM55XLOH/F29WMGRIM55XLOH.MEDIUM.jpg">
       </div>
       <div class="step-body">
        <p></p>
        <ol>
         <li>In ServerThread make a private BufferedReader in to receive data from the server, another BufferedReader to receive data from the user, and a PrintWriter to write to the server.</li>
         <li>Initialize the PrintWriter in run with the socket's output stream, the server's BufferedReader with a new InputStreamReader using the socket's input stream, and the user's BufferedReader with a new InputStreamReader using System.In</li>
         <li>Create a while loop in run that checks for any new input from the server and prints the input to the console, and checks for any new input from the user and prints that input to the server.</li>
        </ol>
        <p></p>
       </div>
      </section>
      <section class="step-container">
       <h2 id="step8" class="step-title">Step 8: Test the Server and Add Optional Functionality</h2>
       <div class="step-body">
        <p>After running the server and launching clients, the program should relay all messages from each client to all other clients. This was a small test of what can be done with java sockets. </p>
        <p>You can expand on this project by adding functionality to let the server itself relay messages, archiving messages, allowing other forms of input and more to become more familiar with java sockets and networking in general.</p>
       </div>
