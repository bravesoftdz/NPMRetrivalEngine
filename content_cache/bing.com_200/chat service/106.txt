       <h1 class="h2 no-top">Cómo instalar Java utilizando apt-get</h1>
       <div class="help-articles">
        <p>Java es un lenguaje de programación que se ha diseñado específicamente para ser como plataforma independiente como sea posible. Los desarrolladores pueden escribir sus aplicaciones una vez sin necesidad de volver a compilar una versión independiente de la aplicación para cada plataforma. La herramienta de empaquetado avanzada es el método más habitual de instalar Java en distribuciones de Linux Debian como Ubuntu. APT simplifica la administración de software en estos sistemas mediante la automatización de gran parte de este proceso, incluidos la descarga, configuración e instalación de paquetes de software. En este ejemplo se muestra cómo seleccionar un paquete de instalación del Kit de desarrollo Java (JDK) 7, instalarlo con apt y comprobar que Java se ha instalado correctamente.</p> 
        <style class="notranslate">

        <p>Aquí hay un tutorial rápido sobre cómo instalar Java utilizando apt-get.</p> 
        <p><strong>Seleccione el paquete de Java para instalar</strong></p> 
        <p>Abra una ventana de terminal y el inicio de sesión como root. Ejecute el siguiente comando para enumerar los paquetes de OpenJDK disponibles que pueden instalar Java:</p> 
        <div class="notranslate">
         # apt-cache search openjdk
        </div> 
        <p>Seleccione <strong>openjdk-7-jdk</strong> para este ejemplo, que es el Kit de desarrollo de OpenJDK 7.</p> 
        <p><strong>Instalar OpenJDK</strong> </p>
        <p>Escriba este comando para actualizar la lista de paquetes en los repositorios remotos:</p> 
        <div class="notranslate">
         # apt-get update
        </div> 
        <p>Escriba el siguiente comando para instalar el paquete de software que ha seleccionado.</p> 
        <div class="notranslate">
         # apt-get install openjdk-7-jdk
        </div> 
        <p>Pulse 'y' cuando se le pida para autorizar el espacio de almacenamiento necesario para la instalación.</p> 
        <p><strong>Compruebe que se ha instalado Java</strong></p> 
        <p>Este comando proporciona la versión de Java que se instaló:</p> 
        <div class="notranslate">
         # java -version
        </div> 
        <p>Captura de pantalla siguiente muestra que 1.7.0_65 de java se ha instalado en este ejemplo:</p> 
        <img style="max-width:650px;" src="https://www.writeraccess.com/upload/3e58eda86a49fe880de57d4d00f46b81.png">
