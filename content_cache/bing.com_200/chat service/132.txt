   <article class="hat"> 
    <h1>Play Framework makes it easy to build web applications with Java &amp; Scala.</h1> 
    <p> Play is based on a lightweight, stateless, web-friendly architecture. </p> 
    <p> Built on Akka, Play provides predictable and minimal resource consumption (CPU, memory, threads) for highly-scalable applications. </p> 
   </article> 
   <article class="feature developer"> 
    <h2>Developer friendly.</h2> 
    <p> Make your changes and simply hit refresh! All you need is a browser and a text editor. </p> 
    <ul> 
     <li>Hit refresh workflow</li> 
     <li>Powerful console &amp; build tools</li> 
     <li>Type safety</li> 
     <li>Built in testing tools</li> 
     <li>IDE support for Eclipse and IntelliJ IDEA</li> 
    </ul> 
    <div id="workflow">

    <h2>Scale predictably.</h2> 
    <p> Underneath the covers Play uses a fully asynchronous model built on top of Akka. Combined with being stateless, Play scales simply and predictably. </p> 
    <ul> 
     <li>Stateless Web Tier</li> 
     <li>Non-blocking I/O</li> 
     <li>Built on Akka</li> 
     <li>Real-time enabled</li> 
    </ul> 
   </article> 
   <article class="feature modern"> 
    <div id="modern"> 
     <div></div> 
    </div> 
    <h2>Modern web &amp; mobile.</h2> 
    <p> Play was built for needs of modern web &amp; mobile apps. </p> 
    <ul> 
     <li>RESTful by default</li> 
     <li>Asset Compiler for CoffeeScript, LESS, etc</li> 
     <li>JSON is a first class citizen</li> 
     <li>Websockets, Comet, EventSource</li> 
     <li>Extensive NoSQL &amp; Big Data Support</li> 
    </ul> 
   </article> 
   <article class="feature strong small"> 
    <h2>Solid &amp; fast.</h2> 
    <p> The compiler and runtime (JVM) do the heavy lifting so that your apps run super-fast and keeps running under load. </p> 
    <ul> 
     <li>Code is compiled</li> 
     <li>Runs on the JVM</li> 
     <li>Java NIO via Akka HTTP or Netty</li> 
    </ul> 
   </article> 
   <article class="feature techs small"> 
    <h2>Massive ecosystem.</h2> 
    <p> The ecosystem around Java is huge. There are libraries for everything - most of which can be used in Play. </p> 
    <ul> 
     <li>Use Maven Central libraries</li> 
     <li>Numerous Play plugins</li> 
     <li>Very active Play community</li> 
    </ul> 
   </article> 
   <article class="feature extend small"> 
    <h2>Proven in production.</h2> 
    <p> From startups to enterprises, Play powers some of the most innovative web sites. Play is proven, reliable and scalable. </p> 
    <ul> 
     <li>Production support by Lightbend</li> 
     <li>Many cloud deployment options</li> 
     <li>Used in mission-critical apps</li> 
    </ul> 
   </article>
