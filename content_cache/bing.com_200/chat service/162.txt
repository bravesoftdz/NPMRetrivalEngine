     <h2 id="post-128">Chat room program in&nbsp;java</h2> 
     <div class="entrytext"> 
      <p>It had been my dream to create a chat room using my programming knowledge. I tried it once previously using servlets, I could get it work for fast internet connections, but for slow connections, working out the chat application had been a night mare, even with a lot of effort I couldn’t do it.</p> 
      <p>This time my effort was fruitful. I searched the web for the way in which I can serve multiple clients using java network programming. I got a handy tutorial, which is very easy to follow and explains all the necessary concepts to create the client server application. Here it the line for the tutorial <a href="http://carfield.com.hk/document/java/tutorial/socket_chat.pdf" rel="nofollow">http://carfield.com.hk/document/java/tutorial/socket_chat.pdf</a></p> 
      <p>I made some simple modifications to the program given in tutorial to make it look like a chat room. The added features includes:</p> 
      <ol> 
       <li>Entering your name at the starting of the app</li> 
       <li>Viewing all the people who are&nbsp;available&nbsp;in the Chat room.</li> 
       <li>Change the client program from an applet to a stand alone program.</li> 
      </ol> 
      <p>The application contains 3 classes as described in the tutorial. 2 of them(Server and ServerThread) should run in the server side, one should run in the client side(Client).</p> 
      <p>The code is:</p> 
      <p>Server1.java</p>

message.charAt(i+1)==’:’))<br> {<br> name1 = name1+message.charAt(i);<br> i++;<br> }<br> to.append(name1+”n”);<br> }<br> }<br> }<br> }<br> catch(IOException ie)<br> {<br> System.out.println(ie);<br> }<br> }<br> public static void main(String args[])<br> {<br> Client1 obj = new Client1(“IP Address of server”,portNumber);<br> obj.setSize(new Dimension(500,500));<br> obj.setVisible(true);<br> obj.setTitle(“Chatting Client”);<br> }<br> }</p>
