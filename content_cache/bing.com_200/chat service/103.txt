       <h1 class="page-title "> Programmable Chat Quickstart: Javascript </h1> 
       <div class="block-markdown "> 
        <div> 
         <p>In this guide, we will get you up and running quickly with a sample chat application you can build on as you learn more about Programmable Chat. Sound like a plan? Then let's get cracking!</p> 
         <h2 id="gather-account-information"><a class="toclink" href="#gather-account-information">Gather account information </a><a id="information"></a></h2> 
         <p>The first thing we need to do is grab all the necessary information from our Twilio account. To set up our back-end for Chat, we will need four values from our account:</p> 
         <table> 
          <thead> 
           <tr> 
            <th align="left">Config Values</th> 
            <th align="left">Description</th> 
           </tr> 
          </thead> 
          <tbody> 
           <tr> 
            <td align="left">Service Instance SID</td> 
            <td align="left">A <a href="/docs/api/chat/rest/services">service</a> instance where all the data for our application is stored and scoped. <a href="/console/chat/services">Generate one in the console here</a>.</td> 
           </tr> 
           <tr> 
            <td align="left">Account SID</td> 
            <td align="left">Your primary Twilio account identifier - find this <a href="/console">in the console here</a>.</td> 
           </tr>

         <h2 id="explore-the-starter-app"><a class="toclink" href="#explore-the-starter-app">Explore the starter app </a><a id="explore"></a></h2> 
         <p>This is where the fun begins! When you visit the homepage of the starter application, verify that everything looks good with your settings, and then click the green chat button at the bottom. You should see a page which looks like this:</p> 
         <p><img alt="quick start app screenshot" src="https://s3.amazonaws.com/howtodocs/quickstart/ipm-browser-quickstart.png"></p> 
         <p>You have been assigned a random <a href="identity">user identity</a> and have <a href="channels">joined a channel</a> called <code>general</code>. If you open this page in another browser window or tab, you should be able to have a rousing conversation with yourself!</p> 
         <p>The HTML and CSS in the starter application aren't terribly interesting, but the JavaScript code driving this application has a few very important jobs:</p> 
         <ol> 
          <li>Fetch an <code>AccessToken</code> from the server via Ajax</li> 
          <li>Initialize the Chat JavaScript SDK</li> 
          <li>Join a <a href="channels">channel</a> called "general"</li> 
          <li>Handle UI events to send messages or display new incoming messages</li> 
         </ol> 
         <p>We explore how to manage your users in our <a href="identity">user identity guide</a>, and cover everything you can do with a channel object in the <a href="channels">channels and messages guide</a>.</p>
