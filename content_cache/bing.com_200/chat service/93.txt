            <p>This guide shown you how to build chat server using <span style="text-decoration: underline;">Openfire</span> 3.7.1 in ubuntu server 12.10.&nbsp;<strong>Openfire</strong> is a free and&nbsp;open source real-time collaboration (RTC) instant messaging server.&nbsp;It uses the only widely adopted open protocol for instant messaging, XMPP (also called Jabber). Openfire is incredibly easy to install and configure, but offers rock-solid security and performance. <a href="http://www.igniterealtime.org/projects/openfire/index.jsp" rel="nofollow" onclick="_gaq.push(['_trackEvent', 'outbound-article', 'http://www.igniterealtime.org/projects/openfire/index.jsp', 'Openfire']);" title="openfire" target="_blank">Openfire</a>&nbsp;licensed under the Open Source Apache License.</p> 
            <p>In additonal, <em>openfire</em> also available for major platform ( windows, Mac OS and Linux). Follow the step-by-step to install and configure&nbsp;chat/IM server using Openfire 3.7.1 on ubuntu server edition</p> 
            <h2 style="text-align: justify;">Step-by-Step to Installing and Configure&nbsp;Chat/IM Server using Openfire</h2> 
            <p><span style="color: #993366;"><strong>Step 1.&nbsp;</strong></span>Login using SSH or Login Directly into ubuntu server 12.10.&nbsp;Before installing Openfire server, make sure your system is up to date. Enter the following commands, one by one, and install any available updates:</p> 
            <pre>sudo apt-get update</pre> 
            <pre>sudo apt-get upgrade</pre> 
            <p><strong><span style="color: #993366;">Step 2.</span> &nbsp;</strong>Install latest version of Oracle JRE/JDK, use personal package archieve (PPA) <a href="http://www.webupd8.org/2012/11/oracle-sun-java-6-installer-available.html" rel="nofollow" onclick="_gaq.push(['_trackEvent', 'outbound-article', 'http://www.webupd8.org/2012/11/oracle-sun-java-6-installer-available.html', 'WEBUPD8']);" target="_blank">WEBUPD8</a> to install it.</p> 
            <pre><strong>sudo apt-get install python-software-properties</strong></pre> 
            <pre><strong>sudo add-apt-repository ppa:webupd8team/java</strong></pre> 
            <pre><strong>sudo apt-get update</strong></pre> 
            <p>If you are using OpenJDK, remove it&nbsp;to minimise any Java conflicts</p> 
            <pre><strong>sudo apt-get remove --purge openjdk*</strong></pre> 
            <p>Then install&nbsp;<strong>Oracle&nbsp;Java 7</strong>&nbsp;version with following command</p> 
            <pre><strong>sudo apt-get install oracle-java7-installer</strong></pre> 
            <p><strong><span style="color: #993366;">Step 3.</span> Install MySql Database server</strong></p> 
            <p>Install latest version MySQL server using apt-get</p> 
            <pre><strong>sudo apt-get install mysql-server</strong></pre> 
            <p><strong><strong><span style="color: #993366;">Step 4</span>.&nbsp;</strong>Create new MySQL database for Openfire</strong></p> 
            <p>Log in to MySQL Server as root.</p> 
            <pre><strong>sudo mysql -u root p</strong></pre> 
            <p>Create&nbsp;the&nbsp;database “dbopenfire”, add new user “openfire” and add new password “openfirepwd” for user “openfire”.</p> 
            <pre><strong>mysql&gt;</strong> CREATE DATABASE dbopenfire CHARACTER SET='utf8';
<strong>mysql&gt;</strong> CREATE USER 'openfire'@'localhost' IDENTIFIED BY '<strong>openfirepwd</strong>';
<strong>mysql&gt;</strong> GRANT ALL PRIVILEGES ON dbopenfire.* TO openfire@localhost WITH GRANT OPTION;
<strong>mysql&gt;</strong> FLUSH PRIVILEGES;
<strong>mysql&gt;</strong> quit</pre> 
            <p><strong><span style="color: #993366;">Step 5.</span> Download and Install openfire 3.7.1</strong></p> 
            <p>Download deb. package&nbsp;of Openfire 3.7.1 from&nbsp;<a href="http://www.igniterealtime.org/downloads/index.jsp" rel="nofollow" onclick="_gaq.push(['_trackEvent', 'outbound-article', 'http://www.igniterealtime.org/downloads/index.jsp', 'ignite realtime download page']);" target="_blank">ignite realtime download page</a></p> 
            <pre><strong>cd /tmp</strong></pre> 
            <pre><strong>wget&nbsp;http://download.igniterealtime.org/openfire/openfire_3.7.1_all.deb</strong></pre> 
            <p>Install&nbsp;openfire_3.7.1_all.deb with dpkg command:</p> 
            <pre><strong>sudo dpkg -i openfire_3.7.1_all.deb</strong></pre> 
            <pre>(Reading database ... 85791 files and directories currently installed.)
Preparing to replace openfire 3.6.4 (using openfire_3.7.1_all.deb) ...
Unpacking replacement openfire ...
Setting up openfire (3.7.1) ...
Installing new version of config file /etc/openfire/security/truststore ...
Installing new version of config file /etc/init.d/openfire ...
Processing triggers for ureadahead ...
ureadahead will be reprofiled on next reboot</pre> 
            <p>Ignore all installation errors regarding user and folder permissions.it cause depending on your version&nbsp;of JRE/JDK, Now you need to editing file&nbsp;<strong>/etc/init/d/openfire</strong>&nbsp;in line 27. Replace<strong>&nbsp;<em>java-6-sun</em>&nbsp;</strong>with<strong>&nbsp;</strong><em>java-6-oracle<strong>&nbsp;</strong></em>or<strong>&nbsp;</strong><em>java-7-oracle.</em></p> 
            <pre>sudo apt-get install rpl</pre> 
            <pre>sudo rpl '<strong>6-sun</strong>' '<strong>7-oracle</strong>' /etc/init.d/openfire</pre> 
            <pre>sudo service openfire start</pre> 
            <p><span style="color: #993366;"><strong>Step 6. &nbsp;<span style="color: #000000;">Allow&nbsp;</span></strong></span><strong>ports for Openfire</strong></p> 
            <p>You need to setup firewall and allow some ports for openfire on ubuntu server, use <strong>ufw command</strong>&nbsp;to set firewall openfire</p> 
            <pre>sudo ufw allow 9090/tcp
sudo ufw allow 9091/tcp
sudo ufw allow 5222/tcp
sudo ufw allow 7777/tcp
sudo ufw allow 7443/tcp
sudo ufw allow 7070/tcp
sudo ufw allow 3478/tcp
sudo ufw allow 3479/tcp</pre> 
            <p>This completes the initial installation steps for Openfire. Next, we’ll continue with configuration through a web browser</p>

            <p>Configuring Openfire is very easy and can be completed in just a couple of steps. Here’s how to configure Openfire:</p> 
            <p>From computer in your network open your favorite browser&nbsp;&nbsp;and go to :</p> 
            <pre>http://<strong>mydomain</strong>:9090/setup/index.jsp</pre> 
            <p>Replace&nbsp;<strong>mydomain</strong>&nbsp;above with your FQDN or IP address of your ubuntu server.&nbsp;The webpage shown below appears:</p>

            <p><strong><span style="color: #993366;">Step 8.</span>&nbsp;Download and Install IM client for Linux, Windows and Mac OSX</strong></p> 
            <p>You need to download install IM/VOIP client software on all PC/Laptop&nbsp;&nbsp;to connect with the Openfire server.&nbsp;You can install&nbsp;any IM client that uses the<a href="http://xmpp.org/xmpp-software/clients/" rel="nofollow" onclick="_gaq.push(['_trackEvent', 'outbound-article', 'http://xmpp.org/xmpp-software/clients/', ' XMPP/Jabber protocol']);" target="_blank"> XMPP/Jabber protocol</a>&nbsp;for example the excellent&nbsp;<a href="https://apps.ubuntu.com/cat/applications/pidgin/" rel="nofollow" onclick="_gaq.push(['_trackEvent', 'outbound-article', 'https://apps.ubuntu.com/cat/applications/pidgin/', 'Pidgin']);" target="_blank">Pidgin</a>, it&nbsp;available for Linux Ubuntu, Windows, and Mac OSX.&nbsp;&nbsp;for Ubuntu users pidgin can be installed from the Ubuntu Software Center.</p> 
            <pre><strong>sudo apt-get install pidgin</strong></pre> 
            <p>If you have done install pidgin,Make sure that you&nbsp;select <strong>XMPP</strong></p> 
            <p>The native cross plaform client for Openfire is&nbsp;<strong>Spark</strong>, it also available for&nbsp;<strong>Linux, Windows and Mac&nbsp;OSX</strong>. Download spark from <a href="http://www.igniterealtime.org/downloads/index.jsp#spark" rel="nofollow" onclick="_gaq.push(['_trackEvent', 'outbound-article', 'http://www.igniterealtime.org/downloads/index.jsp#spark', 'Igniterealtime Download Page']);" target="_blank">Igniterealtime Download Page</a>, and install it on ubuntu desktop edition (Ubuntu 11.04, Ubuntu 11.10, ubuntu 12.04 and Ubuntu 12.10)</p> 
            <pre>wget&nbsp;http://download.igniterealtime.org/spark/spark_2_6_3.tar.gz</pre> 
            <pre>tar -zxvf&nbsp;spark_2_6_3.tar.gz</pre> 
            <pre>sudo mkdir /opt/spark</pre> 
            <pre>sudo mv Spark/* /opt/spark/</pre> 
            <p>Create a desktop launcher file, Open terminal and run following command</p> 
            <p><span style="font-family: Consolas, Monaco, monospace; font-size: 12px; line-height: 18px;">sudo nano /usr/share/applications/spark.desktop</span></p> 
            <pre><strong>[Desktop Entry]</strong>
Name=Spark
Version=2.6.3
GenericName=Spark
X-GNOME-FullName=Spark
Comment=ignite realtime Spark IM client
Type=Application
Categories=Application;Utility;
Path=/opt/spark
Exec=/bin/bash Spark
Terminal=false
StartupNotify=true
Icon=/opt/spark/logo-spark.png
TargetEnvironment=Unity</pre> 
            <pre>sudo cd /opt/spark</pre> 
            <pre>sudo wget https://dl.dropbox.com/u/50880014/spark.png</pre> 
            <p>Now, You will then be able to search for&nbsp;<strong>Spark </strong>in you unity dash&nbsp;and launch Spark.</p>
