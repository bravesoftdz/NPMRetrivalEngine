         <p>ws4py is a Python package implementing the WebSocket protocol as defined in <a class="reference external" href="http://tools.ietf.org/html/rfc6455">RFC 6455</a>.</p> 
         <p>It comes with various server and client implementations and runs on CPython 2/3, PyPy and Android.</p>
