                     <div class="body">
                      <p>Hello everyone,</p> 
                      <p align="left">&nbsp;</p> 
                      <p>I have implemented a package, which loads data from a flat file into an OLE DB destination. I have added the properties of the OLE DB Connection Manager to the package configuration (XML file), in order to give me enough flexibility for running the package on different databases.</p> 
                      <p>&nbsp;</p> 
                      <p align="left">Now I wanted to integrate my package into the application that would call it and I ran into these problems.</p> 
                      <p>&nbsp;</p> 
                      <p align="left"><strong><u>A. Call a package from a JAVA application:<br></u></strong>I have a JAVA web application, which should provide the source file and finally trigger the load operation.<br></p> 
                      <p align="left"><strong><em><font color="#ff0000">A1: How can I execute a package from within a JAVA application?</font></em></strong></p> 
                      <p align="left"><strong><em><font color="#ff0000"><br>A2: How can I pass parameters to this package?</font></em></strong></p> 
                      <p>&nbsp;</p> 
                      <p align="left"><strong><u>B. Parallel Package Execution:<br></u></strong>If I was able to trigger the package execution from my application, then I can encounter a situation where the package is triggereed to be executed again with different input data before the first package has finished.</p> 
                      <p align="left">&nbsp;</p> 
                      <p><strong><em><font color="#ff0000">B1: Is it possible (from SSIS point of view)&nbsp;that the same package gets executed parallely with different input?</font></em></strong></p> 
                      <p align="left">&nbsp;</p> 
                      <p><strong><u>C. Run the package against several databases:<br></u></strong>Assuming that I have 5 databases where I load data using this package.<br>I have created 5 configuration.xml files with the corresponding DB connection properties.</p> 
                      <p align="left">&nbsp;</p> 
                      <p><strong><em><font color="#ff0000">C1: How can I tell the package which configuration file to read from, in order to load the source data into the correct destination? Is it possible to pass the path of the configuration.xml file when executing a package?</font></em></strong></p> 
                      <p>&nbsp;</p> 
                      <p align="left">Thanks in advance.</p>

                      <div class="body">
                       <p>Hi vamsidhar,</p> 
                       <p align="left">&nbsp;</p> 
                       <p>I am extremely sorry for my late reply, but I haven't checked the SSIS forum for a long time.</p> 
                       <p>&nbsp;</p> 
                       <p align="left">Well, here is what you should do:</p> 
                       <p align="left"><u><strong>1. DTEXEC command:</strong></u></p> 
                       <p align="left">First construct the dtexec command with wich you would execute the package.<br>For instance to execute an SSIS package that is saved in the file system, use the following code:<br>dtexec /f "D:\myPackage.dtsx" </p> 
                       <p align="left">&nbsp;</p> 
                       <p>For more details about the dtexec utility, please visit the following page:<br><a title="http://msdn2.microsoft.com/en-us/library/ms162810.aspx" href="http://msdn2.microsoft.com/en-us/library/ms162810.aspx">http://msdn2.microsoft.com/en-us/library/ms162810.aspx</a></p> 
                       <p>&nbsp;</p> 
                       <p align="left"><u><strong>2. Call a local package:</strong></u></p> 
                       <p align="left">If you would call a package which is located on the same machine as your JAVA runtime, you can call it directly from your JAVA program by executing a process.</p> 
                       <p align="left"><br>Here is the sample code:</p>
                       <b><font color="#7f0055" size="2"> <p align="left"> </p></font></b>
                       <font></font>
                       <div class="codeseg">
                        <font><b> 
                          <div class="codecontent"> 
                           <div class="codesniptitle">
                            <span style="width:100%">Code Block</span>
                           </div>
                           <b><font color="#7f0055" size="2"> </font></b>
                           <font></font>
                           <p align="left"><font><b>import</b></font><font size="2"> java.io.BufferedReader;</font></p>
                           <b><font color="#7f0055" size="2"> </font></b>
                           <font></font>
                           <p align="left"><font><b>import</b></font><font size="2"> java.io.InputStreamReader;</font></p>
                           <b><font color="#7f0055" size="2"> </font></b>
                           <font></font>
                           <p align="left"><font><b>import</b></font><font size="2"> java.io.DataInputStream;</font></p> 
                           <p align="left"><font size="2">&nbsp;</font></p>
                           <font size="2"> <p align="left"></p></font>
                           <b><font color="#7f0055" size="2"> </font></b>
                           <font></font>
                           <p align="left"><font><b>public</b></font><font size="2"> </font><b><font color="#7f0055" size="2">class</font></b><font size="2"> SSISCaller {</font></p>
                           <font size="2"> </font>
                           <p align="left"><font size="2"></font><b><font color="#7f0055" size="2"></font></b>&nbsp;</p> 
                           <p align="left"><b><font color="#7f0055" size="2">public</font></b><font size="2"> </font><b><font color="#7f0055" size="2">static</font></b><font size="2"> </font><b><font color="#7f0055" size="2">void</font></b><font size="2"> main(String[] args) {</font></p>
                           <font size="2"> </font>
                           <p align="left"><font size="2"></font><b><font color="#7f0055" size="2"></font></b>&nbsp;</p> 
                           <p align="left"><b><font color="#7f0055" size="2">try</font></b><font size="2"> {</font></p>
                           <font size="2"> </font>
                           <p align="left"><font size="2"></font><font color="#3f7f5f" size="2">// create the execution process</font></p>
                           <font size="2"> </font>
                           <p align="left"><font size="2">Process executionProcess = Runtime.<i>getRuntime</i>().exec(</font><font color="#2a00ff" size="2">"dtexec /f \"D:/ReplacementLoad.dtsx\""</font><font size="2">);</font></p> 
                           <p align="left"><font size="2">&nbsp;</font></p>
                           <font size="2"> <p align="left"></p> </font>
                           <p align="left"><font size="2"></font><font color="#3f7f5f" size="2">// create the output reader</font></p>
                           <font size="2"> </font>
                           <p align="left"><font size="2">BufferedReader output = </font><b><font color="#7f0055" size="2">new</font></b><font size="2"> BufferedReader(</font><b><font color="#7f0055" size="2">new</font></b><font size="2"> InputStreamReader(</font><b><font color="#7f0055" size="2">new</font></b><font size="2"> DataInputStream(executionProcess</font></p>
                           <font size="2"> <p align="left">.getInputStream())));</p> <p align="left">String readStr;</p> </font>
                           <p align="left"><font size="2"></font><b><font color="#7f0055" size="2">while</font></b><font size="2"> ((readStr = output.readLine()) != </font><b><font color="#7f0055" size="2">null</font></b><font size="2">) {</font></p>
                           <font size="2"> </font>
                           <p align="left"><font size="2">System.</font><i><font color="#0000c0" size="2">out</font></i><font size="2">.println(readStr);</font></p>
                           <font size="2"> <p align="left">}</p> <p align="left">output.close();</p> <p align="left">&nbsp;</p> <p align="left"></p> </font>
                           <p align="left"><font size="2"></font><font color="#3f7f5f" size="2">// wait for the process to terminate</font></p>
                           <font size="2"> <p align="left">executionProcess.waitFor();</p> <p align="left"></p> </font>
                           <p align="left"><font size="2"></font><font color="#3f7f5f" size="2"></font>&nbsp;</p> 
                           <p align="left"><font color="#3f7f5f" size="2">// check the exit value</font></p>
                           <font size="2"> </font>
                           <p align="left"><font size="2"></font><b><font color="#7f0055" size="2">if</font></b><font size="2"> (executionProcess.exitValue() == 0) {</font></p>
                           <font size="2"> </font>
                           <p align="left"><font size="2">System.</font><i><font color="#0000c0" size="2">out</font></i><font size="2">.println(</font><font color="#2a00ff" size="2">"SSIS execution succeeded\n"</font><font size="2">);</font></p>
                           <font size="2"> </font>
                           <p align="left"><font size="2">} </font><b><font color="#7f0055" size="2">else</font></b><font size="2"> {</font></p>
                           <font size="2"> </font>
                           <p align="left"><font size="2">System.</font><i><font color="#0000c0" size="2">out</font></i><font size="2">.println(</font><font color="#2a00ff" size="2">"SSIS execution failed\n"</font><font size="2">);</font></p>
                           <font size="2"> <p align="left">}</p> <p align="left">&nbsp;</p> </font>
                           <p align="left"><font size="2">} </font><b><font color="#7f0055" size="2">catch</font></b><font size="2"> (Throwable e) {</font></p>
                           <font size="2"> </font>
                           <p align="left"><font size="2">System.</font><i><font color="#0000c0" size="2">out</font></i><font size="2">.println(e.getMessage());</font></p>
                           <font size="2"> <p align="left">}</p> <p align="left">&nbsp;</p> <p align="left">} </p> <p align="left"><font color="#3f7f5f" size="2">// end of main</font></p> <p align="left">&nbsp;</p> <p align="left">}</p></font> 
                           <p align="left">&nbsp;</p>
                          </div></b></font>
                       </div> 
                       <p align="left">&nbsp;</p> 
                       <p></p>
                       <font size="2"></font> 
                       <p align="left">&nbsp;</p> 
                       <p align="left"><strong><u>3. Call a&nbsp;remote package:</u></strong></p> 
                       <p>If you will call a package which is located on a remote machine (the DB server), you should do the following:<br>a- Create a stored procedure in your DB schema, which would be responsible for executing the SSIS package.<br>For more details, please visit the following page:<br><a title="http://www.simple-talk.com/sql/sql-server-2005/executing-ssis-packages-/" href="http://www.simple-talk.com/sql/sql-server-2005/executing-ssis-packages-/">http://www.simple-talk.com/sql/sql-server-2005/executing-ssis-packages-/</a></p> 
                       <p>&nbsp;</p> 
                       <p align="left"><u>Sample Script:</u></p>
                       <font color="#0000ff" size="2"><font color="#0000ff" size="2"> <p align="left"> </p></font>
                        <div class="codeseg">
                         <font color="#0000ff" size="2"> 
                          <div class="codecontent"> 
                           <div class="codesniptitle">
                            <span style="width:100%">Code Block</span>
                           </div>
                           <font color="#0000ff" size="2"><font color="#0000ff" size="2"> </font></font>
                           <p align="left"><font color="#0000ff" size="2"><font color="#0000ff" size="2">Create</font><font color="#000000" size="2"> </font><font color="#0000ff" size="2">procedure</font><font color="#000000" size="2"> [dbo]</font><font color="#808080" size="2">.</font><font size="2"><font color="#000000">[s_ExecutePackage]</font></font></font></p>
                           <font color="#0000ff" size="2"> <p align="left">as</p> </font>
                           <p align="left"><font color="#0000ff" size="2">Declare</font><font color="#000000" size="2"> @cmd </font><font color="#0000ff" size="2">varchar</font><font color="#808080" size="2">(</font><font color="#000000" size="2">1000</font><font color="#808080" size="2">)</font></p>
                           <font size="2"> </font>
                           <p align="left"><font size="2"></font><font color="#0000ff" size="2">select</font><font size="2"> @cmd </font><font color="#808080" size="2">=</font><font size="2"> </font><font color="#ff0000" size="2">'dtexec /f "D:\myPackage.dtsx"'</font></p>
                           <font size="2"> </font>
                           <p align="left"><font size="2"></font><font color="#0000ff" size="2">exec</font><font size="2"> master</font><font color="#808080" size="2">..</font><font color="#800000" size="2">xp_cmdshell</font><font size="2"> @cmd</font></p>
                           <font size="2"> <p align="left">go</p></font> 
                           <p align="left">&nbsp;</p>
                          </div></font>
                        </div> <p align="left">&nbsp;</p></font> 
                       <p></p>
                       <font size="2"> <p align="left"></p></font>
                       <font size="2"></font> 
                       <p><br>b- Call this stored procedure from your JAVA program.<br>Unfortunately, I do not have sample code for calling a stored procedure from JAVA, but this should be very simple.<br></p> 
                       <p align="left">&nbsp;</p> 
                       <p>I hope this helps.</p> 
                       <p align="left">&nbsp;</p>

                              <p align="left">Well, here is what you should do:</p> 
                              <p align="left"><u><strong>1. DTEXEC command:</strong></u></p> 
                              <p align="left">First construct the dtexec command with wich you would execute the package.<br>For instance to execute an SSIS package that is saved in the file system, use the following code:<br>dtexec /f "D:\myPackage.dtsx" </p> 
                              <p align="left">&nbsp;</p> 
                              <p>For more details about the dtexec utility, please visit the following page:<br><a title="http://msdn2.microsoft.com/en-us/library/ms162810.aspx" href="http://msdn2.microsoft.com/en-us/library/ms162810.aspx">http://msdn2.microsoft.com/en-us/library/ms162810.aspx</a></p> 
                              <p>&nbsp;</p> 
                              <p align="left"><u><strong>2. Call a local package:</strong></u></p> 
                              <p align="left">If you would call a package which is located on the same machine as your JAVA runtime, you can call it directly from your JAVA program by executing a process.</p> 
                              <p align="left"><br>Here is the sample code:</p>
                              <b><font color="#7f0055" size="2"> <p align="left"> </p></font></b>
                              <font></font>
                              <div class="codeseg">
                               <font><b> 
                                 <div class="codecontent">

                                  <p align="left"><font><b>import</b></font><font size="2"> java.io.BufferedReader;</font></p>
                                  <b><font color="#7f0055" size="2"> </font></b>
                                  <font></font>
                                  <p align="left"><font><b>import</b></font><font size="2"> java.io.InputStreamReader;</font></p>
                                  <b><font color="#7f0055" size="2"> </font></b>
                                  <font></font>
                                  <p align="left"><font><b>import</b></font><font size="2"> java.io.DataInputStream;</font></p> 
                                  <p align="left"><font size="2">&nbsp;</font></p>
                                  <font size="2"> <p align="left"></p></font>
                                  <b><font color="#7f0055" size="2"> </font></b>
                                  <font></font>
                                  <p align="left"><font><b>public</b></font><font size="2"> </font><b><font color="#7f0055" size="2">class</font></b><font size="2"> SSISCaller {</font></p>
                                  <font size="2"> </font>
                                  <p align="left"><font size="2"></font><b><font color="#7f0055" size="2"></font></b>&nbsp;</p> 
                                  <p align="left"><b><font color="#7f0055" size="2">public</font></b><font size="2"> </font><b><font color="#7f0055" size="2">static</font></b><font size="2"> </font><b><font color="#7f0055" size="2">void</font></b><font size="2"> main(String[] args) {</font></p>
                                  <font size="2"> </font>
                                  <p align="left"><font size="2"></font><b><font color="#7f0055" size="2"></font></b>&nbsp;</p> 
                                  <p align="left"><b><font color="#7f0055" size="2">try</font></b><font size="2"> {</font></p>
                                  <font size="2"> </font>
                                  <p align="left"><font size="2"></font><font color="#3f7f5f" size="2">// create the execution process</font></p>
                                  <font size="2"> </font>
                                  <p align="left"><font size="2">Process executionProcess = Runtime.<i>getRuntime</i>().exec(</font><font color="#2a00ff" size="2">"dtexec /f \"D:/ReplacementLoad.dtsx\""</font><font size="2">);</font></p> 
                                  <p align="left"><font size="2">&nbsp;</font></p>
                                  <font size="2"> <p align="left"></p> </font>
                                  <p align="left"><font size="2"></font><font color="#3f7f5f" size="2">// create the output reader</font></p>
                                  <font size="2"> </font>
                                  <p align="left"><font size="2">BufferedReader output = </font><b><font color="#7f0055" size="2">new</font></b><font size="2"> BufferedReader(</font><b><font color="#7f0055" size="2">new</font></b><font size="2"> InputStreamReader(</font><b><font color="#7f0055" size="2">new</font></b><font size="2"> DataInputStream(executionProcess</font></p>
                                  <font size="2"> <p align="left">.getInputStream())));</p> <p align="left">String readStr;</p> </font>
                                  <p align="left"><font size="2"></font><b><font color="#7f0055" size="2">while</font></b><font size="2"> ((readStr = output.readLine()) != </font><b><font color="#7f0055" size="2">null</font></b><font size="2">) {</font></p>
                                  <font size="2"> </font>
                                  <p align="left"><font size="2">System.</font><i><font color="#0000c0" size="2">out</font></i><font size="2">.println(readStr);</font></p>
                                  <font size="2"> <p align="left">}</p> <p align="left">output.close();</p> <p align="left">&nbsp;</p> <p align="left"></p> </font>
                                  <p align="left"><font size="2"></font><font color="#3f7f5f" size="2">// wait for the process to terminate</font></p>
                                  <font size="2"> <p align="left">executionProcess.waitFor();</p> <p align="left"></p> </font>
                                  <p align="left"><font size="2"></font><font color="#3f7f5f" size="2"></font>&nbsp;</p> 
                                  <p align="left"><font color="#3f7f5f" size="2">// check the exit value</font></p>
                                  <font size="2"> </font>
                                  <p align="left"><font size="2"></font><b><font color="#7f0055" size="2">if</font></b><font size="2"> (executionProcess.exitValue() == 0) {</font></p>
                                  <font size="2"> </font>
                                  <p align="left"><font size="2">System.</font><i><font color="#0000c0" size="2">out</font></i><font size="2">.println(</font><font color="#2a00ff" size="2">"SSIS execution succeeded\n"</font><font size="2">);</font></p>
                                  <font size="2"> </font>
                                  <p align="left"><font size="2">} </font><b><font color="#7f0055" size="2">else</font></b><font size="2"> {</font></p>
                                  <font size="2"> </font>
                                  <p align="left"><font size="2">System.</font><i><font color="#0000c0" size="2">out</font></i><font size="2">.println(</font><font color="#2a00ff" size="2">"SSIS execution failed\n"</font><font size="2">);</font></p>
                                  <font size="2"> <p align="left">}</p> <p align="left">&nbsp;</p> </font>
                                  <p align="left"><font size="2">} </font><b><font color="#7f0055" size="2">catch</font></b><font size="2"> (Throwable e) {</font></p>
                                  <font size="2"> </font>
                                  <p align="left"><font size="2">System.</font><i><font color="#0000c0" size="2">out</font></i><font size="2">.println(e.getMessage());</font></p>
                                  <font size="2"> <p align="left">}</p> <p align="left">&nbsp;</p> <p align="left">} </p> <p align="left"><font color="#3f7f5f" size="2">// end of main</font></p> <p align="left">&nbsp;</p> <p align="left">}</p></font> 
                                  <p align="left">&nbsp;</p>
                                 </div></b></font>
                              </div> 
                              <p align="left">&nbsp;</p> 
                              <p></p>
                              <font size="2"></font> 
                              <p align="left">&nbsp;</p> 
                              <p align="left"><strong><u>3. Call a&nbsp;remote package:</u></strong></p> 
                              <p>If you will call a package which is located on a remote machine (the DB server), you should do the following:<br>a- Create a stored procedure in your DB schema, which would be responsible for executing the SSIS package.<br>For more details, please visit the following page:<br><a title="http://www.simple-talk.com/sql/sql-server-2005/executing-ssis-packages-/" href="http://www.simple-talk.com/sql/sql-server-2005/executing-ssis-packages-/">http://www.simple-talk.com/sql/sql-server-2005/executing-ssis-packages-/</a></p>

                              <font color="#0000ff" size="2"><font color="#0000ff" size="2"> <p align="left"> </p></font>

                                 <div class="codecontent"> 
                                  <div class="codesniptitle">
                                   <span style="width:100%">Code Block</span>

                                  <p align="left"><font size="2"></font><font color="#0000ff" size="2">select</font><font size="2"> @cmd </font><font color="#808080" size="2">=</font><font size="2"> </font><font color="#ff0000" size="2">'dtexec /f "D:\myPackage.dtsx"'</font></p>
                                  <font size="2"> </font>
                                  <p align="left"><font size="2"></font><font color="#0000ff" size="2">exec</font><font size="2"> master</font><font color="#808080" size="2">..</font><font color="#800000" size="2">xp_cmdshell</font><font size="2"> @cmd</font></p>

                           Hi Afrin,
                           <br>
                           <br>1. For tracing which record contains an error, you should redirect any rows that fail to a so-called 
                           <span style="font-weight:bold;font-style:italic">error output</span>. These erroneous rows will be written to a file that you can check later.
                           <br>For more details about this concept, please refer to:
                           <br>
                           <span style="font-weight:bold;font-style:italic">Title:</span> Adding Error Flow Redirection
                           <br>
                           <span style="font-weight:bold;font-style:italic">URL:</span> http://msdn.microsoft.com/en-us/library/ms166707.aspx
                           <br>
                           <br>2. What you are describing in point 2 is called 
                           <span style="font-weight:bold;font-style:italic">Packge Configuration</span>. Package configurations let you set run-time properties and variables from outside of the development environment. Configurations allow you to develop packages that are flexible and easy to both deploy and distribute. SSIS offers the following configuration types:
                           <br># XML configuration file
                           <br># Environment variable
                           <br># Registry entry
                           <br># Parent package variable
                           <br># SQL Server table
                           <br>
                           <br>For more details, please refer to:
                           <br>
                           <span style="font-weight:bold;font-style:italic">Title:</span> Adding Package Configurations
                           <br>
                           <span style="font-weight:bold;font-style:italic">URL:</span> http://msdn.microsoft.com/en-us/library/ms167153.aspx
                           <br>
