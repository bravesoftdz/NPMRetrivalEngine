               <h1 class="entry-title">What’s package-info.java for?</h1>
               <p class="et_pb_title_meta_container"> by <span class="author vcard"><span class="fn"><a href="https://www.intertech.com/Blog/author/jim-white/" title="Posts by Jim White" rel="author">Jim White</a></span></span> | <span class="published updated">Jul 3, 2013</span></p>

               <p>In some recent classes, my students have encountered a package-info.java file tucked inside of some of the sample code we get from courseware providers.&nbsp; “What’s that thing?”, they usually ask.</p> 
               <p><strong>package-info.java’s purpose</strong></p> 
               <p>The package-info.java is a Java file that can be added to any Java source package.&nbsp; Its purpose is to provide a home for&nbsp;package level documentation and package level annotations. Simply create the package-info.java file and add the package declaration that it relates to in the file.&nbsp; In fact, the only thing the package-info.java file must contain is the package declaration.</p>

               <p> <strong>Help&nbsp;adding package-info.java to your packages</strong></p>
