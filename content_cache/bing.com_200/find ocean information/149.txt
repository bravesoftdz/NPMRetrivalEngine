    <h2>How to run MINE in Java</h2> 
    <p>First, make sure <a href="http://www.java.com/en/download/index.jsp" target="_blank">Java</a> is installed. Then <a href="/Downloads/MINE-Application">download </a>MINE.jar, open a command prompt, and type</p> 
    <p><span style="font-family: courier new,courier;">java -jar MINE.jar</span></p> 
    <p>followed by the necessary <a href="/Usage-instructions/Parameters">command-line parameters</a>.</p> 
    <h2>How to run MINE in R</h2> 
    <p>First, make sure both&nbsp;<a href="http://cran.r-project.org/" target="_blank">R</a> and <a href="http://www.java.com/en/download/index.jsp" target="_blank">Java</a>&nbsp;are installed.&nbsp;Next, <a href="/Downloads/MINE-Application">download</a> and save both MINE.jar and MINE.r into the same directory and make that directory the working directory in R. The example below then runs MINE on all the variable pairs in the file example.csv.</p> 
    <p><span style="font-family: courier new,courier;">install.packages("rJava") # 1-time initialization step<br>source("MINE.r")<br>MINE("example.csv","all.pairs")</span></p> 
    <p>The library exposes&nbsp;two functions: MINE, and rMINE. The difference between them is that MINE takes as input a path to a csv file to be analyzed, while rMINE takes as input an R matrix containing the data to be analyzed.&nbsp;Both functions take&nbsp;<a href="/Usage-instructions/Parameters">parameters</a> identical to those in the&nbsp;Java version. For additional information, see the comments in MINE.r.</p>

function mycarousel_initCallback(carousel)
{
    carousel.buttonNext.bind('click', function() {
        carousel.startAuto(0);
    });
    carousel.buttonPrev.bind('click', function() {
        carousel.startAuto(0);
    });
    carousel.clip.hover(function() {
        carousel.stopAuto();
    }, function() {
        carousel.startAuto();
    });
};

		function animator(currentItem) {
			var distance = currentItem.height();
			duration = (distance + parseInt(currentItem.css("marginTop"))) / 0.015;
			currentItem.animate({ marginTop: -distance }, duration, "linear", function() {
				currentItem.appendTo(currentItem.parent()).css("marginTop", 0);
				animator(currentItem.parent().children(":first"));
			});
		};
		animator(ticker.children(":first"));
		ticker.mouseenter(function() {
			ticker.children().stop();
		});
		ticker.mouseleave(function() {
			animator(ticker.children(":first"));
		});
