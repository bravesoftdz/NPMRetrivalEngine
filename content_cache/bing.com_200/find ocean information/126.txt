         <h1 id="JavaBusinessObjectsEnterpriseSDKSamples-JavaBusinessObjectsEnterpriseSDKSamples">Java BusinessObjects Enterprise SDK Samples</h1> 
         <p><a href="#JavaBusinessObjectsEnterpriseSDKSamples-General">#General</a><br> <a href="#JavaBusinessObjectsEnterpriseSDKSamples-Exporting/Printing">#Exporting/Printing</a><br> <a href="#JavaBusinessObjectsEnterpriseSDKSamples-Scheduling">#Scheduling</a><br> <a href="#JavaBusinessObjectsEnterpriseSDKSamples-Viewing">#Viewing</a><br> <a href="#JavaBusinessObjectsEnterpriseSDKSamples-SecurityManagement">#Security Management</a><br> <a href="#JavaBusinessObjectsEnterpriseSDKSamples-ContentManagement">#Content Management</a><br> <a href="#JavaBusinessObjectsEnterpriseSDKSamples-ServerManagement">#Server Management</a><br> <a href="#JavaBusinessObjectsEnterpriseSDKSamples-UserManagement">#User Management</a><br> <a href="#JavaBusinessObjectsEnterpriseSDKSamples-Publications">#Publications</a></p> 
         <h2 id="JavaBusinessObjectsEnterpriseSDKSamples-General">General</h2> 
         <hr> 
         <p><a class="external-link" href="https://www.sdn.sap.com/irj/scn/go/portal/prtroot/docs/library/uuid/80627c67-2f15-2c10-c785-9b344b3a0079" rel="nofollow">BusinessObjects Enterprise XI R2 Java SDK API functionality</a><br class="atl-forced-newline"> The samples in this collection are grouped by the API functionality they demonstrate. Each group is contained in the folder listed below:</p> 
         <ul> 
          <li>Scheduling - Scheduling objects using the Enterprise SDK.</li> 
          <li>Viewing - Viewing/Exporting reports using the Viewer SDK.</li> 
          <li>RAS - Report modification and creation using the Report Application Server (RAS) SDK.</li> 
          <li>Content Management - Managing reports, events, packages, and other objects.</li> 
          <li>REBean - Report modification and creation of Web Intelligence documents using the Report Engine Java SDK and advanced viewing of Web Intelligence documents.</li> 
         </ul> 
         <p><a class="external-link" href="https://www.sdn.sap.com/irj/scn/go/portal/prtroot/docs/library/uuid/c01d832c-3115-2c10-eab7-b211c2d9a46e" rel="nofollow">BusinessObjects Enterprise XI R2 SDK Java sample application</a><br class="atl-forced-newline"> This Java sample demonstrates logging on and off of Business Objects Enterprise XI R2 as well as scheduling, renaming, and viewing reports.</p> 
         <p><a class="external-link" href="https://www.sdn.sap.com/irj/scn/go/portal/prtroot/docs/library/uuid/9067e215-d366-2b10-88a6-ee01b6211e7b" rel="nofollow">BusinessObjects Enterprise Java SDK Feature Samples</a></p> 
         <p>Contains JSP sample code pages that highlight features of the BusinessObjects Enterprise SDK, including authentication, scheduling, publishing, server administration and auditing, and managing repository objects.</p> 
         <h2 id="JavaBusinessObjectsEnterpriseSDKSamples-Exporting/Printing">Exporting/Printing</h2> 
         <hr> 
         <p><a class="external-link" href="https://www.sdn.sap.com/irj/scn/go/portal/prtroot/docs/library/uuid/8016fd66-be66-2b10-6d86-94cdae94c1a9" rel="nofollow">Java_RAS-Managed_BE12_Export_XML</a> (XI 3.x)<br> This Java Server Pages (JSP) sample demonstrates how to export a Crystal Report to XML format using the XI 3.x Report Application Server (RAS) SDK. <br class="atl-forced-newline"></p> 
         <h2 id="JavaBusinessObjectsEnterpriseSDKSamples-Scheduling">Scheduling</h2> 
         <hr> 
         <p><a class="external-link" href="https://www.sdn.sap.com/irj/scn/go/portal/prtroot/docs/library/uuid/00d36d9d-18ba-2b10-9c9f-f222ccc7d5b2" rel="nofollow">Scheduling to Destinations</a><br class="atl-forced-newline"> Scheduling to SMTP, User's Inbox and Disk sample</p> 
         <p><a class="external-link" href="https://www.sdn.sap.com/irj/scn/go/portal/prtroot/docs/library/uuid/f00d1b01-cc66-2b10-f1ab-81f8a56bbbe0" rel="nofollow">Retrieve Email Addresses from existing Schedules </a><br class="atl-forced-newline"> This sample retrieves email addresses that a scheduled report is going to. This works for already scheduled or recurring instances.</p>

         <p><a class="external-link" href="https://www.sdn.sap.com/irj/boc/index?rid=/library/uuid/5007842c-4c57-2c10-7583-9c4fdad410b8" rel="nofollow">Java_Enterprise_BE115_send_report_to_inbox </a> (XI R2, XI 3.x)<br> This Java Server Pages (JSP) sample demonstrates how to send a crystal report to a users inbox on a Business Objects Enterprise XI R2 system.</p> 
         <p><a class="external-link" href="https://www.sdn.sap.com/irj/boc/index?rid=/library/uuid/f00d3ca2-4a57-2c10-a4bb-ec22ebca5b61" rel="nofollow">Java_Enterprise_BE115_send_report_to_ftp</a> (XI R2, XI 3.x)<br> This Java Server Pages (JSP) sample demonstrates how to send a crystal report to ftp on a Business Objects Enterprise XI R2 system.</p> 
         <p><a class="external-link" href="https://www.sdn.sap.com/irj/boc/index?rid=/library/uuid/602f3fbd-4a57-2c10-0aa9-ce1e3d9048e6" rel="nofollow">Java_Enterprise_BE115_send_report_to_disk </a> (XI R2, XI 3.x)<br> This Java Server Pages (JSP) sample demonstrates how to send a crystal report to disk on a Business Objects Enterprise XI R2 system.</p>

         <p><a class="external-link" href="https://www.sdn.sap.com/irj/boc/index?rid=/library/uuid/d028d507-4c57-2c10-dc88-d7507b640111" rel="nofollow">Java_Enterprise_BE115_schedule_webi_now</a> (XI R2, XI 3.x)<br> This Java Server Pages (JSP) sample demonstrates how to schedule a webi report to run immediately on a Business Objects Enterprise XI R2 system.</p>

         <p><a class="external-link" href="https://www.sdn.sap.com/irj/boc/index?rid=/library/uuid/904c18b9-4b57-2c10-2b94-9135936826f1" rel="nofollow">Java_Enterprise_BE115_schedule_report_to_inbox</a> (XI R2, XI 3.x)<br> This Java Server Pages (JSP) sample demonstrates how to schedule a crystal report to a users enterprise inbox on a Business Objects Enterprise XI R2 system.</p> 
         <p><a class="external-link" href="https://www.sdn.sap.com/irj/boc/index?rid=/library/uuid/c0a415a2-4b57-2c10-49ac-efcf67c15de5" rel="nofollow">Java_Enterprise_BE115_schedule_report_to_ftp</a> (XI R2, XI 3.x)<br> This Java Server Pages (JSP) sample demonstrates how to schedule a crystal report to FTP on a Business Objects Enterprise XI R2 system.</p> 
         <p><a class="external-link" href="https://www.sdn.sap.com/irj/boc/index?rid=/library/uuid/f0b3398a-4b57-2c10-498d-b3f175279ae5" rel="nofollow">Java_Enterprise_BE115_schedule_report_to_disk</a> (XI R2, XI 3.x)<br> This Java Server Pages (JSP) sample demonstrates how to schedule a crystal report to disk on a Business Objects Enterprise XI R2 system.</p> 
         <p><a class="external-link" href="https://www.sdn.sap.com/irj/boc/index?rid=/library/uuid/f057fc73-4b57-2c10-1294-f5d63ef4c950" rel="nofollow">Java_Enterprise_BE115_schedule_report_selection_formula</a> (XI R2, XI 3.x)<br> This Java Server Pages (JSP) sample demonstrates how to schedule a crystal report while setting the recordselection formula on a Business Objects Enterprise XI R2 system.</p>

         <p><a class="external-link" href="https://www.sdn.sap.com/irj/boc/index?rid=/library/uuid/405a1725-4b57-2c10-928e-c25c34a5cf43" rel="nofollow">Java_Enterprise_BE115_schedule_report_now</a> (XI R2, XI 3.x)<br> This Java Server Pages (JSP) sample demonstrates how to schedule a crystal report to run immediately on a Business Objects Enterprise XI R2 system.</p>

         <h2 id="JavaBusinessObjectsEnterpriseSDKSamples-Viewing">Viewing</h2> 
         <hr> 
         <p><a class="external-link" href="https://www.sdn.sap.com/irj/scn/go/portal/prtroot/docs/library/uuid/900e3976-b666-2b10-4f91-aca16e5acab6" rel="nofollow">Viewers Java SDK Feature Samples</a><br class="atl-forced-newline"> Contains JSP sample code pages that highlight features of the Viewers SDK, including viewing and exporting a report.</p> 
         <p><a class="external-link" href="https://www.sdn.sap.com/irj/scn/go/portal/prtroot/docs/library/uuid/306f7bf1-3415-2c10-7b9a-f9bf04da9b54" rel="nofollow">Getting Started Tutorials in the Viewer Java SDK Guide (XI R2) Sample Code</a><br class="atl-forced-newline"> This sample web application accompanies the Getting Started Tutorials in the Viewer Java SDK Guide (XI R2). It demonstrates the two principal uses of the Viewer Java SDK---displaying Crystal reports, and exporting them to a different format.</p> 
         <h2 id="JavaBusinessObjectsEnterpriseSDKSamples-SecurityManagement">Security Management</h2> 
         <hr> 
         <p><a class="external-link" href="https://www.sdn.sap.com/irj/scn/go/portal/prtroot/docs/library/uuid/50fe641f-be66-2b10-318f-aab141c9cfe7" rel="nofollow">Java_Enterprise_BE12_SSO_InfoView</a> (XI 3.x)<br> This Java Server Pages (JSP) sample demonstrates how to single sign-on to XI 3.x Java InfoView by creating and passing a serialized Enterprise Session.<span style="color: rgb(102,102,102);">01 Apr 2008</span></p> 
         <p><a class="external-link" href="https://www.sdn.sap.com/irj/scn/go/portal/prtroot/docs/library/uuid/c0b2b62c-db66-2b10-cf9d-a5d18bc0afa8" rel="nofollow">Java_Enterprise_BE12_Assign_Access_Level</a> (XI 3.x)<br> This Java Server Pages (JSP) sample demonstrates how to assign Access Levels (also known as Custom Roles) to a Principal in BusinessObjects Enterprise XI 3.0. Previous versions of Enterprise allowed only a single Access Level to be assigned. XI 3.x allows multiple Access Levels to be assigned.</p>
