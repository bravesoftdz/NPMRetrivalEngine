          <p><span class="seoSummary"><strong>Rhino</strong> is an open-source implementation of <a title="en/JavaScript" href="/en/JavaScript">JavaScript</a> written entirely in Java. It is typically embedded into Java applications to provide scripting to end users. It is embedded in J2SE 6 as the default Java scripting engine.</span></p>
