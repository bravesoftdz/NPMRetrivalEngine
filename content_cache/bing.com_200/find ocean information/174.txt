    <h3>Which Sun/Oracle Java Certification is best suited for your needs?</h3> 
    <p><a href="https://www.whizlabs.com/wp-content/uploads/2014/04/java-certifications.jpg" data-featherlight="image" data-wpel-link="internal"><img src="https://www.whizlabs.com/wp-content/uploads/2014/04/java-certifications.jpg" alt="Java Certifications" width="725" height="1670" class="  wp-image-21270 aligncenter" srcset="https://www.whizlabs.com/wp-content/uploads/2014/04/java-certifications.jpg 1080w, https://www.whizlabs.com/wp-content/uploads/2014/04/java-certifications-130x300.jpg 130w, https://www.whizlabs.com/wp-content/uploads/2014/04/java-certifications-444x1024.jpg 444w, https://www.whizlabs.com/wp-content/uploads/2014/04/java-certifications-460x1060.jpg 460w, https://www.whizlabs.com/wp-content/uploads/2014/04/java-certifications-220x507.jpg 220w" sizes="(max-width: 725px) 100vw, 725px"></a></p> 
    <div class="spacer"></div> 
    <p>There are four types of certification levels are structured under the Oracle’s Java certification path. The four levels are:</p> 
    <ul> 
     <li><strong><strong>Oracle Certified Associate (OCA)</strong></strong></li> 
     <li><strong>Oracle Certified Professional (OCP)</strong></li> 
     <li><strong>Oracle Certified Expert (OCE)</strong></li> 
     <li><strong>Oracle Certified Master (OCM)</strong></li> 
    </ul> 
    <p>The following are the details of different certifications under each level on the certification path. Note that, each certification would have multiple version of the same certification corresponding to the different Java SE or Java EE version.</p> 
    <ul> 
     <li><strong>Associate Level Certification</strong>: If you are new to&nbsp;Java, by becoming a <a href="https://www.whizlabs.com/ocajp-scja/" data-wpel-link="internal">Oracle Certified Associate, Java Programmer I (OCAJP)</a>, you will acquire a base set of knowledge that enables entry into a career in application development or software project management using Java technology.</li> 
    </ul> 
    <ul> 
     <li><strong>Professional Certification</strong>: As a Java Programmer, in order to demonstrate that you’ve acquired required skills,&nbsp;you should take <a href="https://www.whizlabs.com/ocpjp-scjp/" data-wpel-link="internal">Oracle Certified Professional, Java Programmer – II (OCPJP)</a> exam. Earning this certification will take your Java career to next level.</li> 
    </ul> 
    <ul>
