    <h1>Java user input</h1> 
    <p> So far, what we have focused on has been one sided. That is, the programs just display some data and that's it, there is no interaction. But things are now going to get more interesting as we focus on accepting user input and interacting with the user based on that input. </p> 
    <p class="tutSpec"> This tutorial focuses on: </p> 
    <ul class="tutOutline"> 
     <li><span class="outline">User input package</span></li> 
     <li><span class="outline">User input classes</span></li> 
     <li><span class="outline">Handling the exceptions</span></li> 
    </ul> 
    <h2>User input package</h2> 
    <p> The package that needs to be imported to accept user input is <b>java.io</b>. The <b>java.io</b> package contains classes and interfaces used for input and output. </p> 
    <div class="code">
     <span class="example">The setup:</span>
     <div class="theCode">
      <span class="highlight">import java.io.*;</span> class GetUserInput{ }
     </div>
    </div> 
    <h2>User input classes</h2> 
    <p> To get user input, use the <b>BufferedReader</b> and <b>InputStreamReader</b> classes. </p> 
    <ul> 
     <li> <span class="data">The InputStreamReader class</span> - reads the user's input. </li> 
     <li> <span class="data">The BufferedReader class</span> - buffers the user's input to make it work more efficiently. </li> 
    </ul> 
    <div class="code">
     <span class="example">Get some input:</span>
     <div class="theCode">
      <span class="highlight">import java.io.*;</span> class GetUserInput{ public static void main(String[] args){ //the data that will be entered by the user String name; //an instance of the BufferedReader class //will be used to read the data 
      <span class="highlight">BufferedReader reader;</span> //specify the reader variable //to be a standard input buffer 
      <span class="highlight">reader = new BufferedReader(new InputStreamReader(System.in));</span> //ask the user for their name System.out.print("What is your name? "); //read the data entered by the user using //the readLine() method of the BufferedReader class //and store the value in the name variable 
      <span class="highlight">name = reader.readLine();</span> } }
     </div>
    </div> 
    <h2>Handling the exceptions</h2> 
    <p> Using the above code, you still cannot accept user input because one thing is still missing, and that is you have to catch the exception (a situation in which something unexpected might happen). The exception we will be working with in this situation is the <b>IOException</b> which is the exception used when an input error occurs. Exceptions are covered in detail in our <a href="java-exceptions.php">java exceptions</a> tutorial. For now, you should just know that exceptions are handled by executing a specific set of code if there is an error. </p> 
    <div class="code">
     <span class="example">Get some input with exceptions:</span>
     <div class="theCode">
      <span class="highlight">import java.io.*;</span> class GetUserInput{ public static void main(String[] args){ //the data that will be entered by the user String name; //an instance of the BufferedReader class //will be used to read the data BufferedReader reader; //specify the reader variable //to be a standard input buffer reader = new BufferedReader(new InputStreamReader(System.in)); //ask the user for their name System.out.print("What is your name? "); 
      <span class="highlight">try{ //read the data entered by the user using //the readLine() method of the BufferedReader class //and store the value in the name variable name = reader.readLine(); //print the data entered by the user System.out.println("Your name is " + name); } catch (IOException ioe){ //statement to execute if an input/output exception occurs System.out.println("An unexpected error occured."); }</span> } }
     </div>
