     <h1>Build Mobile Apps Using HTML, CSS &amp; JavaScript</h1> 
     <div class="article_meta"> 
      <div style="margin-bottom: 5px;"> 
       <span class="timestamp">??&nbsp;2014-09-05 23:39:15 </span> 
       <span class="from"> <i class="icon-globe"></i> <a class="cut cut28 from" href="/sites/Fna2Qr" target="_blank">Design Aeon </a> </span>

      <div class="source"> 
       <i style="float:left;">??</i>&nbsp; 
       <a class="cut cut70" href="http://www.tuicool.com/articles/hit/JVjEnq" style="display:inline-block;">http://designaeon.com/2014/09/build-mobile-apps-using-html-css-javascript/</a> 
      </div> 
      <div> 
       <span>??</span> 
       <a href="/topics/11060000" target="_blank"> <span class="new-label">HTML</span> </a>

     <div class="article_body article_body_en" id="nei"> 
      <div> 
       <p>Ever since the evolution of mobile phones, there’s been an incredible increase in the demand for mobile applications. Whether it’s about finding the best restaurant in town or gaining an insight into the latest happenings across the world, mobile apps can serve as your right partner in fetching accurate and relevant details. Developing a mobile app is a wholesome experience in itself. Without a good technical know-how, it’s actually impossible to come up with a brilliant mobile app. With HTML, CSS and JavaScript being considered as the traditional programming languages used for building apps, there are frameworks which have paced up the mobile app development process by an enormous leap. This is a blog that will brief you about 6 such outstanding frameworks that have served as remarkable tools for developing portable yet fully-functional mobile apps.</p> 
       <p> <strong> <em>Importance of choosing the right mobile app development framework</em> </strong> </p> 
       <p>Although anyone with a streamlined knowledge of HTML, CSS and JavaScript can easily build a mobile app, it is the portability factor that forces web developers to use a renowned web technology. A portable app is the one with a greater outreach as compared to the one that’s developed using the traditional programming languages. One of the greatest advantages of using a framework for developing an app is that you can run the app equally well both, on the desktops as well as hand-held devices such as smartphones, tablets, netbooks etc.</p> 
       <p> <span> <em>And now, details about 6 excellent frameworks that work as handy tools for building the next million-dollar app for yourself or for your client:</em> </span> </p> 
       <ol> 
        <li> <strong> <em>PhoneGap</em> </strong> </li> 
       </ol> 
       <p>Essentially based on Cordova, PhoneGap comes equipped with a set of JavaScript APIs that allow you to connect the device’s default functions such as Compass, Camera, Contacts and Geo-location. This framework lets you build an application without the need for using any native programming language. Instead you can use a framework such as jQuery Mobile that will compile the app using the platform’s SDK and will hence be ready to install on the app development platform. Supported by popular Mobile operating systems such as Android, iOS, Windows Phone and BlackBerry, PhoneGap is a must-have for every app developer.</p> 
       <div> 
        <img src="http://img1.tuicool.com/euyMzu.jpg!web" alt="phone gap Build Mobile Apps Using HTML, CSS &amp; JavaScript" title="Build Mobile Apps Using HTML, CSS &amp; JavaScript" class="alignCenter"> 
        <p>Phone Gap</p> 
       </div> 
       <ol start="2"> 
        <li> <strong> <em>Ratchet</em> </strong> </li> 
       </ol> 
       <p>Originally used by Twitter for creating mobile app prototype, Ratchet now comes with a collection of HTML, CSS and JavaScript plugins and User Interface that aid in development of simple yet dynamic mobile apps. The latest Ratchet version 2.0 is made available along with its proprietary font icon set called the Ratcheticon and two pre-made UI themes for Android and iOS operating systems.</p> 
       <ol start="3"> 
        <li> <strong> <em>Lungo</em> </strong> </li> 
       </ol> 
       <p>As a lightweight mobile framework based on HTML5 and CSS3, Lungo comes with nice default styles that can be used as the starting point for designing the mobile application. Apart from the traditional mobile UI components, Lungo is equipped with ample number of innovative JavaScript APIs that render you complete control of your app. Offering support to major mobile operating systems such as Android, iOS, Firefox and BlackBerry, Lungo is a mobile app development framework you can’t afford to miss.</p> 
       <div> 
        <img src="http://img0.tuicool.com/ru6RNz.jpg!web" alt="lungo Build Mobile Apps Using HTML, CSS &amp; JavaScript" title="Build Mobile Apps Using HTML, CSS &amp; JavaScript" class="alignCenter"> 
        <p>lungo</p> 
       </div> 
       <ol start="4"> 
        <li> <strong> <em>Famo.us</em> </strong> </li> 
       </ol> 
       <p>As a newbie in the world of open source mobile app development frameworks, Famo.us is a free, open source JavaScript framework that allows you to create complex UIs for any screen dimensions. This framework comes with a promise of eliminating all HTML5 Performance issues cropping in for the mobile devices. Its built-in lightweight JavaScript engine comes as an incredible add-on for developing out-of-the-box mobile apps. As per a report, it has been revealed that Famo.us will soon introduce a cloud-based service that would allow you to package your app so as to publish the same to the AppStore.</p> 
       <ol start="5"> 
        <li> <strong>Ionic</strong> </li> 
       </ol> 
       <p>Ionic is an app development framework that specially focuses on building apps that are capable of performing exceptionally. As an HTML5 mobile framework, Ionic lets you leverage the best potential of your JavaScript components such as Angular.js for building a highly interactive application. Similar to Ratchet framework, Ionic also comprises of font icon sets called Ionicons and a bunch of reusable HTML Classes that allow you to build an excellent mobile UI for a brilliant end user experience.</p> 
       <ol start="6"> 
        <li> <strong> <em>Sencha Touch</em> </strong> </li> 
       </ol> 
       <p>Sencha Touch is yet another amazing framework that’s powered by CSS3 and HTML5. It comes with APIs and components that are compatible with all the popular mobile browsers and platforms. Supporting both PhoneGap as well as Cordova, Sencha Touch allows you to create an app and submit the same to the respective mobile platform’s App stores. And that’s not all, Sencha Touch is also equipped with a set of themes for Android, Windows Phone, iOS, BlackBerry, Tizen and a variety of other mobile platforms, helping you build an app that’s much more advanced in terms of looks and functionality.</p> 
       <div> 
        <img src="http://img2.tuicool.com/zQRRn2.jpg!web" alt="sencha Build Mobile Apps Using HTML, CSS &amp; JavaScript" title="Build Mobile Apps Using HTML, CSS &amp; JavaScript" class="alignCenter"> 
        <p>sencha</p> 
       </div> 
       <p> <strong> <em>Wrapping Up</em> </strong> </p> 
       <p>So that was a collection of mobile app development frameworks worth making a note of. You can go ahead and try them out for building apps that have the potential of standing out from the crowd.</p> 
       <p> <em>Please don’t forget to drop in your feedback/suggestions on the above post.</em> </p>
