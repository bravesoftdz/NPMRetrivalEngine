          <h2>Introduction</h2> 
          <p>Mobile development has been growing since the day Apple introduced iPhone. Mobile browser has never been so robust. It supports HTML5, CSS3 even with CSS animation. After that, Google introduced Android to join existing <a href="http://www.bluewhaleapps.com/" target="_blank">iPhone app developers</a> in ushering in the mobile platform era. That leave a significant mark to the milestone of mobile development.</p> 
          <p>In the mobile industry, it has quite a few of frameworks available aiming to create a mobile web app rapidly. To help you start up your mobile development, I have done some research on all mobile frameworks, and below is the list. I have used jQTouch before, it's pretty easy to implement but definitely has a lot of room of improvement.</p> 
          <p>This page has been translated into <a href="http://www.webhostinghub.com/support/es/misc/empezar-su-carrera">Spanish</a> language by Maria Ramos.</p> 
          <ul class="list"> 
           <li><a href="http://zeptojs.com/">Zepto.js</a> <img src="http://www.queness.com/resources/images/mframework/1.jpg" alt=""><span>Zepto.js is a minimalist JavaScript framework for mobile WebKit browsers, with a jQuery-compatible syntax. The goal: a 2-5k library that handles most basic drudge work with a nice API so you can concentrate on getting stuff done. Zepto.js is currently in early beta, and you can help to make it awesome by contributing code, documentation and demos. </span></li> 
           <li><a href="http://www.dhtmlx.com/touch/">DynamicX</a> <img src="http://www.queness.com/resources/images/mframework/2.jpg" alt=""><span>DHTMLX Touch is an HTML5-based JavaScript library for building mobile web applications. It‚Äôs not just a set of UI widgets, but a complete framework that allows you to create eye-catching, cross-platform web applications for mobile and touch-screen devices. </span></li> 
           <li><a href="http://www.sencha.com/">Sencha</a> <img src="http://www.queness.com/resources/images/mframework/3.jpg" alt=""><span>Sencha Touch, the first HTML5 mobile JavaScript framework that allows you to develop mobile web apps that look and feel native on iPhone and Android touchscreen devices, has just hit the big 1.0. And best of all, it‚Äôs completely free to use.</span></li> 
           <li><a href="http://jquerymobile.com/">jQuery Mobile</a> <img src="http://www.queness.com/resources/images/mframework/4.jpg" alt=""><span>A unified user interface system across all popular mobile device platforms, built on the rock-solid jQuery and jQuery UI foundation. Its lightweight code is built with progressive enhancement, and has a flexible, easily themeable design.</span></li> 
           <li><a href="http://jqtouch.com/">jQTouch</a> <img src="http://www.queness.com/resources/images/mframework/5.jpg" alt=""><span>A jQuery plugin for mobile web development on the iPhone, iPod Touch, and other forward-thinking devices.</span></li> 
           <li><a href="http://www.winktoolkit.org/">Wink ToolKit</a> <img src="http://www.queness.com/resources/images/mframework/6.jpg" alt=""><span>Wink toolkit is a mobile JavaScript framework for building webapps on iPhone, iPad and Android</span></li> 
           <li><a href="http://code.google.com/p/iui/wiki/Introduction">iUI</a> <img src="http://www.queness.com/resources/images/mframework/7.jpg" alt=""><span>iUI is a framework consisting of a JavaScript library, CSS, and images for developing advanced mobile webapps for iPhone and comparable/compatible devices. </span></li> 
           <li><a href="http://snippetspace.com/">iWebkit</a> <img src="http://www.queness.com/resources/images/mframework/8.jpg" alt=""><span>iWebKit is a file package designed to help you create your own iPhone, iPod Touch and iPad compatible website or webapp.</span></li> 
           <li><a href="http://webapp-net.com/">WebAPP.net</a> <img src="http://www.queness.com/resources/images/mframework/9.jpg" alt=""><span>WebApp.Net is a light weight, powerful javascript framework taking advantage of AJAX technology. It provides a full set of ready to use components to help you develop, quickly and easily, advanced mobile web applications. </span></li> 
          </ul>
