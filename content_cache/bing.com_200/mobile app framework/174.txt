          <h1 class="post-title entry-title" itemprop="headline"> <a href="https://www.iprodev.com/12-best-mobile-app-testing-frameworks/" rel="bookmark" title="Permanent Link: 12 Best Mobile App Testing Frameworks">12 Best Mobile App Testing Frameworks <span class="post-format-icon minor-meta"></span> </a></h1>
          <span class="post-meta-infos"><time class="date-container minor-meta updated">April 9, 2015</time><span class="text-sep text-sep-date">/</span><span class="comment-container minor-meta"><a href="https://www.iprodev.com/12-best-mobile-app-testing-frameworks/#comments" class="comments-link"><span class="dsq-postid" data-dsqidentifier="589 http://iprodev.com/?p=589">3 Comments</span></a></span><span class="text-sep text-sep-comment">/</span><span class="blog-categories minor-meta">in <a href="https://www.iprodev.com/category/web-resources/" rel="tag">Web Resources</a> </span><span class="text-sep text-sep-cat">/</span><span class="blog-author minor-meta">by <span class="entry-author-link"><span class="vcard author"><span class="fn"><a href="https://www.iprodev.com/author/admin/" title="Posts by Hemn Chawroka" rel="author">Hemn Chawroka</a></span></span></span></span></span>
         </header>
         <div class="entry-content" itemprop="text">
          <input class="jpibfi" type="hidden">
          <p>Native mobile apps usually work faster, smoother and these result in a better user experience. And, web designers/developers always look for new ways to catch this performance with web technologies. Mobile Application Developers, while able to produce amazing applications, are limited by the native browsers own limited functionality. Basic features available on desktop browsers are completely absent from mobile browsers. The ability to view source code, debug JavaScript, or even execute a JavaScript command from a console is all tools developers have come to expect while developing websites.</p> 
          <p>For this roundup we have compiled a list of <strong>12 Best Mobile App Testing Frameworks</strong> for mobile programmers that will help them to test, debug and execute their web applications on various browsers and devices. If you are aware of some other useful mobile app testing frameworks, please let us know by posting a comment below.</p> 
          <h2>1. Appium</h2> 
          <p><a href="http://appium.io/" target="_blank" class="external" rel="nofollow"><img class="alignnone size-full" src="http://iprodev.com/wp-content/uploads/appium3.jpg" alt="appium" width="625" height="342" data-jpibfi-post-excerpt="" data-jpibfi-post-url="https://www.iprodev.com/12-best-mobile-app-testing-frameworks/" data-jpibfi-post-title="12 Best Mobile App Testing Frameworks" data-jpibfi-src="http://iprodev.com/wp-content/uploads/appium3.jpg"></a></p> 
          <p>Appium is an open source framework which helps automating mobile app testing from any language and any test framework, with full access to back-end APIs and DBs from test code. It works for both iOS + Android apps and tests can be written with Java, Objective-C, JavaScript, PHP, Python, Ruby, C#, Clojure, or Perl. The framework is Mac OS X only and requires Nodejs to run.</p> 
          <h2>2. Mobile Testing Framework</h2> 
          <p><a href="https://gitorious.org/mobile-testing-framework#more" target="_blank" class="external" rel="nofollow"><img class="alignnone size-full" src="http://iprodev.com/wp-content/uploads/Mobile-Testing-Framework-Gitorious.jpeg" alt="Mobile Testing Framework   Gitorious" width="625" height="291" data-jpibfi-post-excerpt="" data-jpibfi-post-url="https://www.iprodev.com/12-best-mobile-app-testing-frameworks/" data-jpibfi-post-title="12 Best Mobile App Testing Frameworks" data-jpibfi-src="http://iprodev.com/wp-content/uploads/Mobile-Testing-Framework-Gitorious.jpeg"></a></p> 
          <p>The Mobile Testing Framework (MTF) is an alpha-state open source project to automate GUI/System tests for iPhone/iPad applications. MTF is based on the automation tool Sikuli which allows interacting with the User interface with python scripts and screenshots.</p> 
          <h2>3. Calabash</h2> 
          <p><a href="http://calaba.sh/" target="_blank" class="external" rel="nofollow"><img class="alignnone size-full" src="http://iprodev.com/wp-content/uploads/calabash.jpeg" alt="calabash" width="625" height="291" data-jpibfi-post-excerpt="" data-jpibfi-post-url="https://www.iprodev.com/12-best-mobile-app-testing-frameworks/" data-jpibfi-post-title="12 Best Mobile App Testing Frameworks" data-jpibfi-src="http://iprodev.com/wp-content/uploads/calabash.jpeg"></a></p> 
          <p>Calabash enables you to write and execute automated acceptance tests of mobile apps. Calabash is cross-platform, supporting Android and iOS native apps. It is open source and free, and has a company, Xamarin, backing and developing it. Calabash consists of libraries that enable test-code to programmatically interact with native and hybrid apps. The interaction consists of a number of end-user actions.</p> 
          <h2>4. KIF</h2> 
          <p><a href="https://github.com/kif-framework/KIF" target="_blank" class="external" rel="nofollow"><img class="alignnone size-full" src="http://iprodev.com/wp-content/uploads/kif.jpeg" alt="kif" width="625" height="415" data-jpibfi-post-excerpt="" data-jpibfi-post-url="https://www.iprodev.com/12-best-mobile-app-testing-frameworks/" data-jpibfi-post-title="12 Best Mobile App Testing Frameworks" data-jpibfi-src="http://iprodev.com/wp-content/uploads/kif.jpeg"></a></p> 
          <p>KIF, which stands for Keep It Functional, is an iOS integration test framework. It allows for easy automation of iOS apps by leveraging the accessibility attributes that the OS makes available for those with visual disabilities. KIF builds and performs the tests using a standard XCTest testing target. Testing is conducted synchronously in the main thread (running the run loop to force the passage of time) allowing for more complex logic and composition.</p> 
          <h2>5. Titanium Jasmine</h2> 
          <p><a href="https://github.com/guilhermechapiewski/titanium-jasmine#readme" target="_blank" class="external" rel="nofollow"><img class="alignnone size-full" src="http://iprodev.com/wp-content/uploads/titanium-jasmine.jpeg" alt="titanium-jasmine" width="625" height="291" data-jpibfi-post-excerpt="" data-jpibfi-post-url="https://www.iprodev.com/12-best-mobile-app-testing-frameworks/" data-jpibfi-post-title="12 Best Mobile App Testing Frameworks" data-jpibfi-src="http://iprodev.com/wp-content/uploads/titanium-jasmine.jpeg"></a></p> 
          <p>Titanium-jasmine provides testing for Appcelerator’s Titanium Mobile applications using Pivotal’s Jasmine as the core testing framework. titanium-jasmine puts together Jasmine and Titanium Mobile, so you can write Jasmine tests that will run inside your iPhone, iPad and Android applications. titanium-jasmine is a really simple testing framework that will help you test your Titanium Mobile applications properly.</p> 
          <h2>6. Testing with Frankk</h2> 
          <p><a href="http://www.testingwithfrank.com/" target="_blank" class="external" rel="nofollow"><img class="alignnone size-full" src="http://iprodev.com/wp-content/uploads/testing-with-frank.jpeg" alt="testing with frank" width="625" height="291" data-jpibfi-post-excerpt="" data-jpibfi-post-url="https://www.iprodev.com/12-best-mobile-app-testing-frameworks/" data-jpibfi-post-title="12 Best Mobile App Testing Frameworks" data-jpibfi-src="http://iprodev.com/wp-content/uploads/testing-with-frank.jpeg"></a></p> 
          <p>Frank allows you to write structured text test/acceptance tests/requirements (using Cucumber) and have them execute against your iOS application, Frank also includes a powerful “app inspector” (called Symbiote) that you can use to get detailed information on your running app.</p> 
          <h2>7. Switchboard</h2> 
          <p><a href="https://github.com/KeepSafe/Switchboard" target="_blank" class="external" rel="nofollow"><img class="alignnone size-full" src="http://iprodev.com/wp-content/uploads/switchboard.jpeg" alt="switchboard" width="625" height="291" data-jpibfi-post-excerpt="" data-jpibfi-post-url="https://www.iprodev.com/12-best-mobile-app-testing-frameworks/" data-jpibfi-post-title="12 Best Mobile App Testing Frameworks" data-jpibfi-src="http://iprodev.com/wp-content/uploads/switchboard.jpeg"></a></p> 
          <p>Switchboard – easy and super light weight A/B testing for your mobile iPhone or android app. This mobile A/B testing framework allows you with minimal servers to run large amounts of mobile users. Switchboard segments your users consistently. Because user segmentation is based only on UUID that is computed once, the experience you switch on and off using Switchboard is consistent across sessions.</p> 
          <h2>8. Flip Test</h2> 
          <p><a href="http://fliptest.io/" target="_blank" class="external" rel="nofollow"><img class="alignnone size-full" src="http://iprodev.com/wp-content/uploads/fliptest.jpeg" alt="fliptest" width="625" height="291" data-jpibfi-post-excerpt="" data-jpibfi-post-url="https://www.iprodev.com/12-best-mobile-app-testing-frameworks/" data-jpibfi-post-title="12 Best Mobile App Testing Frameworks" data-jpibfi-src="http://iprodev.com/wp-content/uploads/fliptest.jpeg"></a></p> 
          <p>FlipTest allows you to run A/B tests on iOS apps without having to re-submit the app to the App Store or make significant modifications to your code. It only requires adding a single line of code to your iOS app. With FlipTest, even non-developers can easily create and monitor A/B tests. No coding necessary.</p> 
          <h2>9. ReactiveUI</h2> 
          <p><a href="http://reactiveui.net/" target="_blank" class="external" rel="nofollow"><img class="alignnone size-full" src="http://iprodev.com/wp-content/uploads/ReactiveUI-Home.jpeg" alt="ReactiveUI  Home" width="625" height="291" data-jpibfi-post-excerpt="" data-jpibfi-post-url="https://www.iprodev.com/12-best-mobile-app-testing-frameworks/" data-jpibfi-post-title="12 Best Mobile App Testing Frameworks" data-jpibfi-src="http://iprodev.com/wp-content/uploads/ReactiveUI-Home.jpeg"></a></p> 
          <p>A MVVM framework that integrates with the Reactive Extensions for .NET to create elegant, testable User Interfaces that run on any mobile or desktop platform. Supports Xamarin.iOS, Xamarin.Android, Xamarin.Mac, WPF, Windows Forms, Windows Phone 8 and Windows Store apps.</p> 
          <h2>10. Catjs</h2> 
          <p><a href="http://catjsteam.github.io/" target="_blank" class="external" rel="nofollow"><img class="alignnone size-full" src="http://iprodev.com/wp-content/uploads/catjs.jpeg" alt="catjs" width="625" height="291" data-jpibfi-post-excerpt="" data-jpibfi-post-url="https://www.iprodev.com/12-best-mobile-app-testing-frameworks/" data-jpibfi-post-title="12 Best Mobile App Testing Frameworks" data-jpibfi-src="http://iprodev.com/wp-content/uploads/catjs.jpeg"></a></p> 
          <p>CatJS is an automation framework for (mobile) web applications testing. It offers a new technology that makes testing easy, we don’t do WebDriver like the other technologies, in our case the applications test themselves. With simple annotations added to your HTML5 code, an automated functional tests will be available as part of your application lifecycle.</p> 
          <h2>11. YSO Mobile Security Framework</h2> 
          <p><a href="https://github.com/ajinabraham/YSO-Mobile-Security-Framework" target="_blank" class="external" rel="nofollow"><img class="alignnone size-full" src="http://iprodev.com/wp-content/uploads/yso.jpg" alt="yso" width="625" height="294" data-jpibfi-post-excerpt="" data-jpibfi-post-url="https://www.iprodev.com/12-best-mobile-app-testing-frameworks/" data-jpibfi-post-title="12 Best Mobile App Testing Frameworks" data-jpibfi-src="http://iprodev.com/wp-content/uploads/yso.jpg"></a></p> 
          <p>YSO Mobile Security Framework is an intelligent, all-in-one open source mobile application (Android/iOS) automated pen-testing framework capable of performing static and dynamic analysis. We’ve been depending on multiple tools to carry out reversing, decoding, debugging, code review, and pen-test and this process requires a lot of effort and time. YSO Mobile Security Framework can be used for effective and fast security analysis of Android APK/Android app source code/iOS app source code.</p> 
          <h2>12. Imagrium</h2> 
          <p><a href="https://github.com/azoft-dev-team/imagrium" target="_blank" class="external" rel="nofollow"><img class="alignnone size-full" src="http://iprodev.com/wp-content/uploads/imagirum.jpeg" alt="imagirum" width="625" height="309" data-jpibfi-post-excerpt="" data-jpibfi-post-url="https://www.iprodev.com/12-best-mobile-app-testing-frameworks/" data-jpibfi-post-title="12 Best Mobile App Testing Frameworks" data-jpibfi-src="http://iprodev.com/wp-content/uploads/imagirum.jpeg"></a></p> 
          <p>Imagrium is a Jython framework for cross-platform testing of mobile applications basing on the image recognition method (and distributed under the MIT License terms).</p>
