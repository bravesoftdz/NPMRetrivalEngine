     <h1 class="entry-title">Adding Audio to netbeans (Java Application)</h1> 
     <div class="entry-content"> 
      <p>If you think, Netbeans and java is all about pages of ever-scrolling codes destined only to create boring applications sans music, sans animations and sans life, you are at the wrong place. Its domain extends far beyond what my words can ever describe. Java is the most powerful programming language and platform for you to begin with. In this blog I will unravel some of the mysteries of java. Please don’t go hysteric…</p> 
      <p>What is life without music?&nbsp;Speech and music are the greatest gifts of God to mankind.Have your watched a movie on mute. I suggest you don’t. It is like food without salt whose presence is unnoticeable but in whose absence the movie loses its flair. Ans that is why the greatest writer in the English language once quoted</p> 
      <p><span style="color:#181818;">“If music be the food of love, play on,</span><br style="color:#181818;"><span style="color:#181818;">Give me excess of it; that surfeiting,</span><br style="color:#181818;"><span style="color:#181818;">The appetite may sicken, and so die.”&nbsp;</span><br style="color:#181818;"><span style="color:#181818;">?&nbsp;</span><a style="color:#666600;" href="http://www.goodreads.com/author/show/947.William_Shakespeare">William Shakespeare</a><span style="color:#181818;">,&nbsp;</span><i style="color:#181818;"><a style="color:#666600;" href="http://www.goodreads.com/work/quotes/3267921">Twelfth Night</a></i></p> 
      <p>In this post I will teach you how to add audio to your form in netbeans through very simple coding. So let’s get started.</p> 
      <p>In order to play an audio in netbeans, you need to first choose an audio file of the <strong>.wav</strong>&nbsp;format. If it is of some other format, please convert it into <strong>.wav&nbsp;</strong>using suitable converter before importing it into netbeans. Once it is done, drag and drop the audio file into the netbeans project.</p> 
      <p><a href="https://mukeshscience.files.wordpress.com/2014/05/sound.png"><img data-attachment-id="18" data-permalink="https://mukeshscience.wordpress.com/2014/05/26/adding-audio-to-java/sound/" data-orig-file="https://mukeshscience.files.wordpress.com/2014/05/sound.png" data-orig-size="1280,800" data-comments-opened="1" data-image-meta="{&quot;aperture&quot;:&quot;0&quot;,&quot;credit&quot;:&quot;&quot;,&quot;camera&quot;:&quot;&quot;,&quot;caption&quot;:&quot;&quot;,&quot;created_timestamp&quot;:&quot;0&quot;,&quot;copyright&quot;:&quot;&quot;,&quot;focal_length&quot;:&quot;0&quot;,&quot;iso&quot;:&quot;0&quot;,&quot;shutter_speed&quot;:&quot;0&quot;,&quot;title&quot;:&quot;&quot;}" data-image-title="sound" data-image-description="" data-medium-file="https://mukeshscience.files.wordpress.com/2014/05/sound.png?w=525&amp;h=329" data-large-file="https://mukeshscience.files.wordpress.com/2014/05/sound.png?w=640" class="wp-image-18 alignnone" src="https://mukeshscience.files.wordpress.com/2014/05/sound.png?w=525&amp;h=329" alt="sound" width="525" height="329" srcset="https://mukeshscience.files.wordpress.com/2014/05/sound.png?w=525&amp;h=329 525w, https://mukeshscience.files.wordpress.com/2014/05/sound.png?w=1050&amp;h=656 1050w, https://mukeshscience.files.wordpress.com/2014/05/sound.png?w=150&amp;h=94 150w, https://mukeshscience.files.wordpress.com/2014/05/sound.png?w=300&amp;h=188 300w, https://mukeshscience.files.wordpress.com/2014/05/sound.png?w=768&amp;h=480 768w, https://mukeshscience.files.wordpress.com/2014/05/sound.png?w=1024&amp;h=640 1024w" sizes="(max-width: 525px) 100vw, 525px"></a></p> 
      <p>&nbsp;</p> 
      <p>Now, right click on any empty space within the frame and select &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;<strong>Events &gt;&gt; Window &gt;&gt; windowOpened</strong></p>

      <p>First create a Input Stream. &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; <strong>InputStream music;</strong> &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; I have named it <strong>music</strong>. It is just a variable name. Once it is done click on the bulb that appears beside it and click on <strong>Add import for java.io.InputStream</strong></p> 
      <p><a href="https://mukeshscience.files.wordpress.com/2014/05/inputstream.png"><img data-attachment-id="23" data-permalink="https://mukeshscience.wordpress.com/2014/05/26/adding-audio-to-java/inputstream/" data-orig-file="https://mukeshscience.files.wordpress.com/2014/05/inputstream.png" data-orig-size="599,110" data-comments-opened="1" data-image-meta="{&quot;aperture&quot;:&quot;0&quot;,&quot;credit&quot;:&quot;&quot;,&quot;camera&quot;:&quot;&quot;,&quot;caption&quot;:&quot;&quot;,&quot;created_timestamp&quot;:&quot;0&quot;,&quot;copyright&quot;:&quot;&quot;,&quot;focal_length&quot;:&quot;0&quot;,&quot;iso&quot;:&quot;0&quot;,&quot;shutter_speed&quot;:&quot;0&quot;,&quot;title&quot;:&quot;&quot;}" data-image-title="inputstream" data-image-description="" data-medium-file="https://mukeshscience.files.wordpress.com/2014/05/inputstream.png?w=469&amp;h=86" data-large-file="https://mukeshscience.files.wordpress.com/2014/05/inputstream.png?w=599" class="alignnone  wp-image-23" src="https://mukeshscience.files.wordpress.com/2014/05/inputstream.png?w=469&amp;h=86" alt="inputstream" width="469" height="86" srcset="https://mukeshscience.files.wordpress.com/2014/05/inputstream.png?w=469&amp;h=86 469w, https://mukeshscience.files.wordpress.com/2014/05/inputstream.png?w=150&amp;h=28 150w, https://mukeshscience.files.wordpress.com/2014/05/inputstream.png?w=300&amp;h=55 300w, https://mukeshscience.files.wordpress.com/2014/05/inputstream.png 599w" sizes="(max-width: 469px) 100vw, 469px"></a></p>
