     <h1 class="title" id="page-title">How to convert a Microsoft Word document into a full text full audio DAISY Book?</h1> 
     <div class="region region-content"> 
      <div id="block-system-main" class="block block-system first last odd"> 
       <div class="content"> 
        <div id="node-13762" class="node node-daisypedia-article view-mode-full clearfix"> 
         <div id="original-author">
           Original Author(s): Prashant Ranjan Verma
         </div> 
         <div class="content"> 
          <span class="print-link"></span>
          <div class="field field-name-body field-type-text-with-summary field-label-hidden">
           <div class="field-items">
            <div class="field-item even">
             <p> This article describes the steps involved in converting a Microsoft Word document into a full-text full-audio DAISY Book. The output of this process will be DAISY 3.0 and DAISY 2.02 standard versions of the document which will contain complete text present in the source document, its text to speech narrated audio and images. </p> 
             <p> <strong>Step 1:</strong> Open a document that needs to be translated into DAISY DTBook.</p> 
             <p> <strong>Step 2:</strong> Markup the document structure – headings, pages etc.</p> 
             <p> <strong>Step 3: </strong>Click on office button (Top left hand corner of the Word 2007 user interface) or hit Alt+F. In Office 2010, click ACCESSIBILITY in the Microsoft Word ribbon and then click Save AS DAISY button. </p> 
             <p> <strong>Step 4: </strong>Clicking on “Full DAISY (from Single docx)” or press (Alt+F) +Y+A, a sub menu “DAISYPipelineTTSNarrator” will be shown. Click on “DAISYPipelineTTSNarrator” sub menu or Hit (Alt+F) +Y+A+D.</p> 
             <p> <img alt="Picture showing Save As DAISY option in the MS Word 2007 File menu" src="http://www.daisy.org/sites/default/files/daisypedia_node/13762/1_png_20317.png"></p> 
             <p> &nbsp;</p> 
             <p> <strong>Step 5: </strong>On click of “Yes” button, the translation process will continue and it translates the current document. The user is presented with the Translation form as shown below after an initialization message. </p> 
             <p> <img alt="Picture showing DAISY Translator form" src="http://www.daisy.org/sites/default/files/daisypedia_node/13762/2_png_11862.png"></p> 
             <p> Translator form contains fields such as Output directory, Document Title, Document creator, Document Publisher, Document UID; all these fields will be auto populated from the document properties modify or change these values if required. Also, the form will have “Show Advance” button, on clicking or on pressing (Alt + V) it will expand the form as shown below.</p> 
             <p><img alt="Picture showing the DAISY Translator Form with advanced options" src="http://www.daisy.org/sites/default/files/daisypedia_node/13762/3_png_54140.png"></p> 
             <p> The "Show Advanced" form contains the following fields:</p> 
             <dl>
              <dt>
                1. DTBook Fix (drop-down list)
              </dt> 
              <dd>
                Selects whether to apply DTBook Fix routines to the input document. These routines try to clean and possibly repair the input DAISY XML document to enforce the best compatibility with the remaining steps. The "Apply DTBook Fix routines" option is recommended.
              </dd> 
              <dt>
                2. Apply sentence detection (check-box)
              </dt> 
              <dd>
                Selects whether to apply sentence detection to the input document text to enable a finer-grained audio synchronization.
              </dd> 
              <dt>
                3. MP3 Bitrate (drop-down list)
              </dt> 
              <dd>
                Selects the bit rate of the generated MP3 audio files. A higher value will result in better sound quality but the audio files will be larger. The default value is 32 kbit/s.
              </dd> 
              <dt>
                4. 2.02 href target (drop-down list)
              </dt> 
              <dd>
                Selects the SMIL element (text or par) to target by href URIs in the content document and navigation control file of the generated DAISY DTB. "text" is the default value but some DAISY players will behave more gracefully if "par" is selected.
              </dd> 
             </dl>
             <p> On click of “Translate” button or on pressing Alt+T buttons on the keyboard the translation process will start.</p> 
             <p> <strong>Step 6:</strong> On click of “Translate” button, Progress bar will show the progress of the translation as shown below.</p> 
             <p> &nbsp;<img alt="Picture showing translation progresss" src="http://www.daisy.org/sites/default/files/daisypedia_node/13762/4_jpg_77500.jpg"></p> 
             <p> <strong>Step 7: </strong>After completing 100% the Progress bar will show “Initializing Audio Files Generation”. When the "Translating to DAISY" step is completed, the Pipeline Lite tool starts the DAISY XML to DAISY DTB translation and the user is presented with the "Pipeline Job Progress" dialog.</p> 
             <p> <strong>Step 8: </strong>A progress bar shows the progress of the translation and a label (below the progress bar) shows the name of the currently running step. Note that the translation will take several minutes (and on some occasion’s even hours) for big documents.</p> 
             <p> <img alt="Picture showing TTS conversion in progress" src="http://www.daisy.org/sites/default/files/daisypedia_node/13762/5_png_20293.png"></p> 
             <p> <strong>Step 9: </strong>A "Cancel" button allows the user to prematurely abort the translation. Note that the Pipeline Lite process might not terminate immediately but wait for the current task to complete. Some temporary files may then be left in the output directory.</p> 
             <p> <strong>Step 10: </strong>When the translation completes successfully, the user is shown a "Pipeline Job Completed" dialog box displaying the message "Job Completed".</p> 
             <p> <img alt="Picture showing job completion dialog" src="http://www.daisy.org/sites/default/files/daisypedia_node/13762/5_png_20293.png"></p> 
             <p> The output directory will open automatically and it will contain DAISY 2.02 and DAISY 3 DTB (Z39.86-2005) format of the document.</p> 
             <p> Note that on some cases dummy headings may be added to the document to conform to the DAISY specification (this happens for instance when no heading were found in the entire input document).</p> 
             <h2>See also</h2> 
             <ul>
              <li>Visit the<a href="http://www.daisy.org/project/save-as-daisy-microsoft/">save As DAISY Microsoft Project Page</a> to download the add-in for Microsoft Word</li> 
              <li><a href="http://www.daisy.org/forums/save-as-daisy-microsoft">Save As DAISY Microsoft Forum</a></li> 
              <li><a href="http://www.daisy.org/daisypedia/video-tutorials-using-save-as-daisy">Video Tutorials on Save As DAISY</a></li> 
              <li><a href="http://www.daisy.org/project/save-as-daisy-openoffice.org/">Save As DAISY - OpenOffice.org Project Page</a></li> 
             </ul>
            </div>
