         <h1 class="page-title"><span class="field field--name-title field--type-string field--label-hidden">Java sound example - how to play a sound file in Java</span> </h1> 
        </div> 
        <div id="block-aadotcom-content" class="block block-system block-system-main-block"> 
         <article data-history-node-id="1528" role="article" about="/java/java-audio-example-java-au-play-sound" class="node node--type-blog node--promoted node--view-mode-full">

          </div> 
          <div class="node__content"> 
           <div class="clearfix text-formatted field field--name-body field--type-text-with-summary field--label-hidden field__item">
            <p><b>My Java sound/audio example:</b> I'm working on a simple "meditation" application that plays a sound after a certain period of time (the sound of a gong), so I thought I'd share some source code out here that demonstrates how to play a sound file in a Java application like this.</p> 
            <p>(Note: I initially found this technique described at JavaWorld.com, but the code below is taken from my project.)</p> 
            <p>In this case the sound file I'm going to play is an "au" file, but I believe this same technique works with most other sound file types as well.</p> 
            <h2>A simple Java "play sound file" example</h2> 
            <p>Here's the source code for my Java sound file example:</p> 
            <pre class="brush: java">import java.io.*;
import sun.audio.*;

/**
 * A simple Java sound file example (i.e., Java code to play a sound file).
 *&nbsp;AudioStream and AudioPlayer code comes from a javaworld.com example.
 * @author alvin alexander, devdaily.com.
 */
public class JavaAudioPlaySoundExample
{
  public static void main(String[] args) 
  throws Exception
  {
    // open the sound file as a Java input stream
    String gongFile = "/Users/al/DevDaily/Projects/MeditationApp/resources/gong.au";
    InputStream in = new FileInputStream(gongFile);

    // create an audiostream from the inputstream
    AudioStream audioStream = new AudioStream(in);

    // play the audio clip with the audioplayer class
    AudioPlayer.player.start(audioStream);
  }
}
</pre> 
            <p>As you can see from this source code, it's pretty easy to create a basic Java sound file player.</p> 
            <h2>My "real world" Java sound file code</h2> 
            <p>I tried to make that example very simple so you can just copy that source code and run it, but my actual code is a little bit different, primarily because I need to read my sound file out of a jar file, instead of reading it as a file on a filesystem.</p> 
            <p>For the sake of completeness, here's the actual method from my current Java application that plays the sound file by reading the file as a resource from the jar file I create when I&nbsp;build my application:</p> 
            <pre class="brush: java">private void playSound() 
{
  try
  {
    // get the sound file as a resource out of my jar file;
    // the sound file must be in the same directory as this class file.
    // the input stream portion of this recipe comes from a javaworld.com article.
    InputStream inputStream = getClass().getResourceAsStream(SOUND_FILENAME);
    AudioStream audioStream = new AudioStream(inputStream);
    AudioPlayer.player.start(audioStream);
  }
  catch (Exception e)
  {
    // a special way i'm handling logging in this application
    if (debugFileWriter!=null) e.printStackTrace(debugFileWriter);
  }
}
</pre> 
            <p>As you can see from that Java code, after I retrieve the sound file from my jar file, the process of playing the sound file is the same in this code as it was in my earlier example.</p> 
            <h2>Java sound/audio API</h2> 
            <p>As I mentioned, I'm just learning about the Java Sound/Audio API capabilities, but it's interesting that these classes come from the <code>sun.audio</code> package (instead of something like a java.sound package). As I've tried to find more Java sound examples, and the Javadoc for the classes I used above, I see there's a <code>javax.sound</code> package that may have power than these classes, while using these two classes seems very simple. So much to learn ...</p> 
            <p>In the meantime, here are a few great Java sound file links I've discovered on this short journey:</p> 
            <ul>
             <li><a href="http://java.sun.com/products/java-media/sound/">Java Sound API</a></li> 
             <li><a href="http://www.jsresources.org/examples/">Java Sound Resources (jsresources), which looks like a terrific Java Sound site</a></li> 
             <li><a href="http://java.sun.com/javase/6/docs/api/javax/sound/sampled/spi/AudioFileReader.html">Java AudioFileReader</a></li> 
             <li><a href="http://java.sun.com/javase/6/docs/api/javax/sound/sampled/AudioInputStream.html">Java AudioInputStream</a></li> 
            </ul>

             <h3 property="schema:name" datatype=""><a href="/comment/253#comment-253" class="permalink" rel="bookmark" hreflang="en">ClassLoader</a></h3> 
             <div property="schema:text" class="clearfix text-formatted field field--name-comment-body field--type-text-long field--label-hidden field__item">
              <p>Why don't you use a ClassLoader?<br> Then the Sound File must not be in the same Directory as this Class File.</p> 
              <p>ClassLoader CLDR = this.getClass().getClassLoader();<br> InputStream soundName = CLDR.getResourceAsStream("yourDirectory/yourSound.wav");<br> AudioStream audioStream = new AudioStream(soundName);<br> AudioPlayer.player.start(audioStream);</p>
