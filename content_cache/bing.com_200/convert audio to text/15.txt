         <h1 class="entry-title">Converting or Transcribing audio to text using C# and .NET System.Speech</h1> 
         <!--BEGIN .entry-meta .entry-header--> 
         <div class="entry-meta entry-header"> 
          <a href="http://ellismis.com/2012/03/17/converting-or-transcribing-audio-to-text-using-c-and-net-system-speech/"><span class="published updated">March 17, 2012</span></a> 
          <span class="comment-count"><a href="http://ellismis.com/2012/03/17/converting-or-transcribing-audio-to-text-using-c-and-net-system-speech/#comments">22 Comments</a></span> 
          <span class="author vcard"> Written by <strong><a class="url fn" href="http://ellismis.com/author/admin/" title="View all posts by admin">admin</a></strong> </span> 
          <!--END .entry-meta .entry-header--> 
         </div> 
         <!--BEGIN .entry-content .article--> 
         <div class="entry-content article"> 
          <p>Recently, I had a project where I needed to convert some audio to text. It took a bit more googling than I was used to in order to find the code, so I went ahead and whipped up a project that demonstrates its usage, so people can more easily find it.</p> 
          <p>This code uses the .NET System.Speech namespace and demonstrates how to transcribe audio using either a microphone or a previously created .wav file using C#.</p> 
          <p>The code can be divided into 2 main parts:</p> 
          <ul> 
           <li>configuring the <code><a title="system.speech.recognition.speechrecognitionengine" href="http://msdn.microsoft.com/en-us/library/system.speech.recognition.speechrecognitionengine.aspx" target="_blank">SpeechRecognitionEngine</a></code> object (and its required elements)</li> 
           <li>handling the <code><a title="system.speech.recognition.speechrecognitionengine.speechrecognized" href="http://msdn.microsoft.com/en-us/library/system.speech.recognition.speechrecognitionengine.speechrecognized.aspx" target="_blank">SpeechRecognized</a></code> and <code><a title="system.speech.recognition.speechrecognitionengine.speechhypothesized" href="http://msdn.microsoft.com/en-us/library/system.speech.recognition.speechrecognitionengine.speechhypothesized.aspx" target="_blank">SpeechHypothesized</a></code> events.</li> 
          </ul> 
          <p><strong>Step 1: Configuring the SpeechRecognitionEngine</strong></p> 
          <p><code>_speechRecognitionEngine = new SpeechRecognitionEngine();<br> _speechRecognitionEngine.SetInputToDefaultAudioDevice();<br> _dictationGrammar = new DictationGrammar();<br> _speechRecognitionEngine.LoadGrammar(_dictationGrammar);<br> _speechRecognitionEngine.RecognizeAsync(RecognizeMode.Multiple);<br> </code></p> 
          <p>At this point your object is ready to start transcribing audio from the microphone. You need to handle some events though, in order to actually get access to the results.</p> 
          <p><strong>Step 2: Handling the SpeechRecognitionEngine Events</strong><br> <code><br> _speechRecognitionEngine.SpeechRecognized -= new EventHandler
            <speechrecognizedeventargs>
             (SpeechRecognized);
             <br> _speechRecognitionEngine.SpeechHypothesized -= new EventHandler
             <speechhypothesizedeventargs>
              (SpeechHypothesizing);
             </speechhypothesizedeventargs>
            </speechrecognizedeventargs></code></p>
          <code> <p> _speechRecognitionEngine.SpeechRecognized += new EventHandler
            <speechrecognizedeventargs>
             (SpeechRecognized);
             <br> _speechRecognitionEngine.SpeechHypothesized += new EventHandler
             <speechhypothesizedeventargs>
              (SpeechHypothesizing);
             </speechhypothesizedeventargs>
            </speechrecognizedeventargs></p> <p> private void SpeechHypothesizing(object sender, SpeechHypothesizedEventArgs e)<br> {<br> ///real-time results from the engine<br> string realTimeResults = e.Result.Text;<br> }</p> </code>
          <p><code> private void SpeechRecognized(object sender, SpeechRecognizedEventArgs e)<br> {<br> ///final answer from the engine<br> string finalAnswer = e.Result.Text;<br> }</code></p> 
          <p> Thats it. If you want to use a pre-recorded .wav file instead of a microphone, you would use <code>_speechRecognitionEngine.SetInputToWaveFile(pathToTargetWavFile);</code> instead of <code>_speechRecognitionEngine.SetInputToDefaultAudioDevice();</code>.</p> 
          <p>There are a bunch of different options in these classes and they are worth exploring in more detail. This covers the bare essentials for a prototype. I have attached a full example and encapsulation <a href="http://ellismis.com/wp-content/uploads/2012/03/MicrosoftSpeechToTextExample.zip">here</a>.</p> 
          <div class="clearfix"></div> 
         </div>
