  <h1>Converting from HTML</h1> 
  <p> Up to <a href="Filters.html">full list of filters</a> </p>
  <h2>Browsers</h2> Most www browsers will convert html to plain text, for example the Linemode browser or Lynx: 
  <p> www -na "some-URL" &gt; my-text </p>
  <p> lynx -dump "some-URL" &gt; my-text </p>
  <p> (See the <a href="http://www.cc.ukans.edu/lynx_help/Lynx_users_guide.html">Lynx documentation</a>) </p>
  <p> Mosaic, Netscape Navigator, Internet Explorer and other browsers will let you "save as" plain text, and in some versions also in other formats including formatted text and PostScript. </p>
  <h2>SGML tools</h2> 
  <p> Some SGML tools will allow you to convert HTML to other formats. For instance: </p>
  <ul> 
   <li> <a href="gf.html">gf</a>. A general-purpose SGML compiler. </li>
   <li> <a href="HyperHelp-Bridge.html">HyperHelp Bridge</a> from <a href="http://www.bristol.com/"> Bristol Technology</a> will convert to RTF. </li>
   <li> <a href="SGML2TeX.html">SGML2TeX</a> will convert SGML to TeX on the PC. </li>
   <li> <a href="Fred.html">Fred</a> will convert SGML to HTML, TeX (PostScript), ASCII etc. </li>
   <li> <a href="sgml2.txt">sgml2</a> will convert SGML to other formats. </li>
   <li> <a href="instant.html">instant</a> from OSF can be used with sgmls to produce various output formats from standard SGML inputs. </li>
   <li>An <a href="http://www.ucla.edu/ICADD/html2icadd-form.html">HTML to ICADD Transformation Service</a> translates HTML into the ICADD DTD, suitable for further translation to Braille, large print or voice synthesis. </li>
  </ul> 
  <p> Further information is available on <a href="http://dxsting.cern.ch/sting/SGML.html">SGML resources and tools</a>. </p>
  <h2>Other tools</h2> 
  <ul> 
   <li>PostScript 
    <ul> 
     <li><a href="http://www.tdb.uu.se/~jan/html2ps.html">html2ps</a> is a Perl script to convert HTML to PostScript. It handles most tags but not in-line images. Contact: jan@tdb.uu.se (Jan Karrman) </li>
     <li><a href="html2ps">Patched version from Arnaud and then patched by Jan</a>. </li>
     <li><a href="htps.html">Here is information</a> about another script to convert HTML to PostScript. </li>
     <li> And here is a <a href="www-print.html">service</a> to do the job for you. Try it! </li>
    </ul> </li>
   <li>MS Word / RTF 
    <ul> 
     <li> <a name="html2rtf" href="http://itzehoe.netsurf.de/~guido.krueger">html2rtf</a> translates HTML to RTF. Using this program and the standard windows help compiler, you can convert hyperlinked Web pages into Windows HLP files. Contact: guido.krueger@itzehoe.netsurf.de </li>
     <li> <a name="hh2rtf" href="http://www.blighty.com/products/hh2rtf/">hh2rtf</a> is a set of freeware perl scripts that converts most HtmlHelp formatted HTML to WinHelp-ready RTF. Contact: steve@blighty.com (Steve Atkins). </li>
     <li> <a href="http://www.cena.dgac.fr/~sagnier/info/formats/conversions/htm2rtf.htm">htm2rtf</a> is another converter for PC/DOS. Contact: sagnier@cena.dgac.fr (Yves Sagnier) </li>
     <li> <a href="bin/html2wrd.zip">html2wrd.zip</a> is a Microsoft Word Basic program to convert HTML documents (including lists, tables and other formatting) into WinWord documents. Contact: yura@kpmgk.kiev.ua (Yuri M. Lesiuk) </li>
     <li> Here is <a href="htmltortf.html">another tool</a> under development </li>
    </ul> </li>
   <li>Frame 
    <ul> 
     <li> <a href="../Frame/fminit2.0/www_and_frame.html">www_and_frame </a> will convert HTML to MML for FrameMaker. See also an <a href="http://www.w3.org/hypertext/Dist/contrib/html2mif-README">HTML to MIF</a>converter and toolkit by the same author. See <a href="http://www.w3.org/hypertext/WWW/People/Connolly/support/www_and_frame.html"> Support Info</a> for current situation and sources. Contact: connolly@w3.org (Dan Connolly) </li>
     <li><a href="ftp://icemcfd.com/pub/">html2mif</a> is an HTML to Framemaker MIF converter written in Tcl. Contact: faustus@remarque.berkeley.edu (Wayne A. Christopher) </li>
     <li>HTML-2-Frame is an HTML import filter for FrameMaker for Windows 95/NT from <a href="http://www.riess.de">.riess gmbh</a> (Web site in German - auf Deutsch). Contact: thomas@riess.de (Thomas Berendonck). </li>
    </ul> </li>
   <li>WordPerfect 
    <ul> 
     <li>An <a href="htwpv101.html">HTML to WP 5.1 macro</a> is available. Contact: bors@netcom.com (E. Daniel Bors Jr.) </li>
    </ul> </li>
   <li>LaTeX etc 
    <ul> 
     <li> <a href="ftp://ftp.tex.ac.uk/tex-archive/support/html2latex/">html2latex</a> is a program based on the NCSA html parser. Contact: Nathan.Torkington@vuw.ac.nz. </li>
     <li>Another <a href="http://home.planet.nl/~faase009/html2tex.html">html2latex</a> can combine several HTML files into a single LaTeX file, converting links between the files to references. External URL's can be converted into footnotes or into a bibliography sorted on URL. Contact: F.J.Faase@cs.utwente.nl (<a href="http://home.wxs.nl/~faase009/Fhome.html">Frans J. Faase</a>) </li>
     <li>Another <a href="http://www.naochan.com/nc/it/soft/nc-html2latex">html2latex</a> implemented on Linux by yacc+lex+C. Also available from the <a href="ftp://tsx-11.mit.edu/pub/linux/sources/usr.bin/">TSX-11 Linux FTP site</a> as nc-html2latex-0.97.tar.gz. Contact: naochan@naochan.com (Naoya Tozuka) </li>
     <li><a href="http://rowlf.cc.wwu.edu:8080/~n9146070/htmlatex.pl">htmlatex.pl</a> is a perl script to do the conversion (may be moving soon). Contact: n9146070@cc.wwu.edu (Jake Kesinger) </li>
     <li>There is also a <a href="html2latex.sed">sed script</a> to convert HTML into LaTeX. </li>
    </ul> </li>
   <li>Plain text and setext 
    <ul> 
     <li><a href="http://www.aquaticmoon.com/markrm.html">Markup Remover</a> from Aquatic Moon Software is a Windows application to convert HTML into plain text: there are several output options. 
      <!-- Contact: ams@aquaticmoon.com --> </li>
     <li><a href="http://www.iinet.net.au/~bwh/removeit.html">Remove-It</a> from GME Systems is a Windows 3.1X based HTML Tag removal utility. </li>
     <li>Here is a <a href="advantage.html">Visual Basic application</a> to do the job. </li>
     <li>HTMSTRIP by <a href="http://www.geocities.com/SiliconValley/Lakes/2414/">Bruce Guthrie</a> for DOS/Windows processes and removes embedded HTML commands from Web pages. Reflows paragraphs, processes tables, etc as straight ASCII text. </li>
     <li><a href="HTMLCon.html">HTMLCon</a> for MSDOS converts HTML to ASCII. </li>
     <li><a href="http://www.printerport.com/klephacks/markdown.html">HTML Markdown</a> is a drag-and-drop Macintosh program that converts HTML files into regular text files. </li>
     <li>An <a href="html-parser.html">html parser</a> in perl is available which will also convert HTML to plain text. </li>
     <li>Here is information about some other <a href="html2ascii.html">html to ascii converters</a>. </li>
     <li>The dehtml option of <a href="htmlchek.html">htmlchek</a> will produce plain ASCII from HTML. </li>
     <li>html2setext will convert HTML to <a href="http://www.bsdi.com/setext">setext</a> structured enhanced text which is human-readable. It is available from <a href="http://www.sericyb.com.au/sc/">Serious Cybernetics</a>. Contact: xanni@aus.xanadu.com (Andrew Pam) </li>
    </ul> </li>
   <li>Other formats 
    <ul> 
     <li>An <a href="html2pcl.html">enhanced html parser</a> handles HP-PCL and other printer formats </li>
     <li><a href="HTMLhelp.html">HTMLhelp</a> converts to WinHelp </li>
     <li>see <a href="#html2rtf">html2rtf</a> for another route to WinHelp </li>
     <li>and <a href="#hh2rtf">hh2rtf</a> for yet another route to WinHelp </li>
     <li><a href="cphtml.html">cphtml</a> converts an annotated HTML file into a perl script as an aid to writing cgi scripts. </li>
     <li><a href="HTMLDBF.html">HTMLDBF</a> converts HTML pages into DBF files (dBase III+). </li>
    </ul> </li>
  </ul>
