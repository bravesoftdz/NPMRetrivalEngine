     <article class="post" itemprop="articleBody">
      <p>One way to make your podcast, screencast or online video more accessible is to convert it to text. For authors, this means providing a transcript, subtitles or close caption so that the hearing impaired get the same context of the audio. The challenge is most content recorded is done without being planned, organized, or rehearsed. Where it gets tricky is most talk to text software programs require you to speak through a microphone and there is not an obvious way to route speech from a recorded file to the program that converts it.</p>
      <p>If we combine Enhanced Dictation feature in MacOS X 10.9 (Mavericks), with Audacity and Soundflower it can easily be integrated. So at a high level, audacity will output audio to enhanced dictation while using soundflower.</p>
      <ul>
       <li>Audacity (output) -&gt; Soundflower -&gt; Dictation (input)</li>
      </ul>
      <p>Lets see how it is done.</p>
      <style>.post img{width:45%;height:auto;border:1px solid #d7d7d7}</style>
      <h1>Download and install soundflower</h1>
      <p>Soundflower is a OS X system extension that allows applications to pass audio to other applications. It presents itself as an audio device, allowing any audio application to send and receive audio with no other support needed. Soundflower is free, open-source, and runs on Mac Intel and PPC computers. Download <a href="http://cycling74.com/soundflower-landing-page/">soundflower</a></p>
      <p><a href="http://cycling74.com/soundflower-landing-page/"><img src="/assets/images/posts/convert-recorded-audio-to-text/soundflower-download-2c82641110a76fa7300993ec4fcbac90086b5cf4c3c35b46ca9315e4ed6f24cc.png" width="1152" height="160" alt="images/posts/convert-recorded-audio-to-text/soundflower-download.png"></a></p>
      <h1>Modify dictation settings</h1>
      <p><a href="http://support.apple.com/kb/ht5449">Dictation</a>, similar to SIRI, uses voice instead of typing text for OS X Mountain Lion. You will want to make three modifications to direct recorded audio to dictation.</p>
      <p>Navigate to Dictation preferences in the control panel</p>
      <ol>
       <li>Choose Apple (?) &gt; System Preferences.</li>
       <li>From the View menu, choose Dictation &amp; Speech.</li>
       <li>Click "On" to turn Dictation on</li>
       <li><p>Check 'Enhanced dictation', Enhanced Dictation allows you to dictate without an active Internet connection.</p><p>a. The first time you turn on Enhanced Dictation, OS X downloads additional content that allows Dictation to work offline. <img src="/assets/images/posts/convert-recorded-audio-to-text/dictation-speech-download-enhanced-28d11a647c5f4f2063d44a5663c2fc04a5c1074c3fb4b988abe8b1f71036a36e.png" alt="Download enhanced dictation" width="960" height="328"></p></li>
       <li><p>Change input device to Sunflower 2ch</p></li>
      </ol>
      <p><img src="/assets/images/posts/convert-recorded-audio-to-text/dictation-speech-to-text-463d87579ceb6ff65477b1facb05dc1e1f8485e809fd00b8e99796909c2ee421.png" alt="Dictation speech to text settings]" width="1332" height="876"></p>
      <h1>Prepare audacity</h1>
      <p>Audacity, is an open source audio editor for recording, slicing, and mixing audio, you select soundflower as an output device.</p>
      <ol>
       <li>Download and install <a href="http://audacity.sourceforge.net/">Audacity</a></li>
       <li>Import recorded audio into audacity</li>
       <li>Change output device to soundflower (2ch)</li>
      </ol>
      <p><img src="/assets/images/posts/convert-recorded-audio-to-text/audacity-settings-fb8e6842518d15c8a633cdb8108f22818ca486fa99cd3f9c5315822e30c741bf.png" alt="Audacity settings]" width="2158" height="972"></p>
      <h1>Bringing it together</h1>
      <p>Next open up your favorite word editing program (text edit, pages or ms word).</p>
      <ol>
       <li>Open audacity and the word program side by side</li>
       <li>Quickly start audio file in audacity</li>
       <li>Flip over to your word editing program and turn on dication by pressing the fn (Function) key twice, or choose Edit &gt; Start Dictation.</li>
      </ol>
      <p>You won't hear any audio but you will see activity in dictation microphone along with the text appear in the word editing program. It should look something like this:</p>
      <p><img src="/assets/images/posts/convert-recorded-audio-to-text/audacity-sunflower-dication-textedit-cf11857a7698b63bb66f2df2fcc4b5c52948e5fd87b75fb4bba6860904f009a7.png" alt="Audacity sunflower dictation" width="2504" height="970"></p>

      <h1>Wrap up</h1>
      <p>Just with any speech to text software, it isn't perfect. Expect to spend time editing the text produced to get it just how you like it. By default it doesn't provide punctuation, line breaks, breaks on paragraphs or inserts example code in our screencasts :). Dictation does understands basic text-related commands such as “all caps,” “new paragraph,” “new line,” and punctions such as “period,” “comma,” “question mark,” or “exclamation point” but most likely these commands will not be contained within your recording.</p>
      <p>A couple other things to be aware of:</p>
      <ul>
       <li>When starting the audio file in audacity you have to quickly jump over to word editing program to kick off the process.<br></li>
       <li>During the conversion process you need to make sure you maintain context of the word editing program otherwise dictation will stop listening when you switch to another window.<br></li>
      </ul>
