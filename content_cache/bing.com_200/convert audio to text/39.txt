       <div class="page-header">
        <h1> <a href="/java-se/file-io/how-to-read-and-write-text-file-in-java"> How to Read and Write Text File in Java</a></h1>
       </div>
       <div class="article-info muted">
        <dl class="article-info">
         <dt class="article-info-term">
          Details
         </dt>
         <dd class="modified"> 
          <span class="icon-calendar"></span> Last Updated on 13 August 2017 &nbsp; | &nbsp; 
          <a href="/java-se/file-io/how-to-read-and-write-text-file-in-java?tmpl=component&amp;print=1&amp;page=" title="Print" onclick="window.open(this.href,'win2','status=no,toolbar=no,scrollbars=yes,titlebar=no,menubar=no,resizable=yes,width=640,height=480,directories=no,location=no'); return false;" rel="nofollow"><span class="icon-print"></span>&nbsp;Print&nbsp;</a> 
          <a href="/component/mailto/?tmpl=component&amp;template=protostar&amp;link=9801ac12ce7f8727b5c9481f580c76b3f4629d0f" title="Email" onclick="window.open(this.href,'win2','width=400,height=350,menubar=yes,resizable=yes'); return false;"><span class="icon-envelope"></span> Email</a>
         </dd>
        </dl>
       </div>
       <div>
        <a href="https://click.linksynergy.com/link?id=d1Fh24*EiQI&amp;offerid=323058.533682&amp;type=2&amp;murl=https%3A%2F%2Fwww.udemy.com%2Fjava-the-complete-java-developer-course%2F&amp;u1=Web20170725" target="_blank" rel="nofllow"><b><font size="+1" color="red">Complete Java Masterclass - 114,841 students enrolled</font></b></a>
       </div>In this tutorial, we show you how to read from and write to text (or character) files using classes available in the 
       <span style="color: #800000;"><strong><span style="font-size: 10pt; line-height: 115%; font-family: 'Courier New';">java.io </span></strong></span>package. First, let’s look at the different classes that are capable of reading and writing character streams.
       <p></p> &nbsp;
       <p></p>
       <h1><strong><span style="font-size: 16pt; line-height: 115%;">1. Reader, InputStreamReader, FileReader and BufferedReader</span></strong></h1> 
       <span style="color: #800000;"><strong><span style="font-size: 10pt; line-height: 115%; font-family: 'Courier New';">Reader</span></strong></span> is the abstract class for reading character streams. It implements the following fundamental methods:
       <p></p>
       <ul>
        <li><span style="color: #800000;"><strong><span style="font-size: 10pt; line-height: 115%; font-family: 'Courier New';">read()</span></strong></span>: reads a single character.</li>
        <li><span style="color: #800000;"><strong><span style="font-size: 10pt; line-height: 115%; font-family: 'Courier New';">read(char[])</span></strong></span>: reads an array of characters.</li>
        <li><span style="color: #800000;"><strong><span style="font-size: 10pt; line-height: 115%; font-family: 'Courier New';">skip(long)</span></strong></span>: skips some characters.</li>
        <li><span style="color: #800000;"><strong><span style="font-size: 10pt; line-height: 115%; font-family: 'Courier New';">close()</span></strong></span>: closes the stream.</li>
       </ul> 
       <span style="color: #800000;"><strong><span style="font-size: 10pt; line-height: 115%; font-family: 'Courier New';">InputStreamReader</span></strong></span> is a bridge from byte streams to character streams. It converts bytes into characters using a specified charset. The charset can be default character encoding of the operating system, or can be specified explicitly when creating an 
       <span style="color: #800000;"><strong><span style="font-size: 10pt; line-height: 115%; font-family: 'Courier New';">InputStreamReader</span></strong></span>.
       <p></p> 
       <span style="color: #800000;"><strong><span style="font-size: 10pt; line-height: 115%; font-family: 'Courier New';">FileReader</span></strong></span> is a convenient class for reading text files using the default character encoding of the operating system.
       <p></p> 
       <span style="color: #800000;"><strong><span style="font-size: 10pt; line-height: 115%; font-family: 'Courier New';">BufferedReader</span></strong></span> reads text from a character stream with efficiency (characters are buffered to avoid frequently reading from the underlying stream) and provides a convenient method for reading a line of text 
       <span style="color: #800000;"><strong><span style="font-size: 10pt; line-height: 115%; font-family: 'Courier New';">readLine()</span></strong></span>.
       <p></p> The following diagram show relationship of these reader classes in the 
       <span style="color: #800000;"><strong><span style="font-size: 10pt; line-height: 115%; font-family: 'Courier New';">java.io</span></strong></span> package:
       <p></p> 
       <img style="display: block; margin-left: auto; margin-right: auto;" src="/images/articles/javase/fileio/text-files/Reader_Hierarchy.png" alt="Reader Hierarchy" width="240" height="116">
       <p></p> &nbsp;
       <p></p>
       <h1><strong><span style="font-size: 16pt; line-height: 115%;">2. Writer, OutputStreamWriter, FileWriter and BufferedWriter</span></strong></h1>
       <div>
        <script async src="//pagead2.googlesyndication.com/pagead/js/adsbygoogle.js"></script> 
        <!-- CJ Inside Content Rectangle --> 
        <ins class="adsbygoogle" style="display:inline-block;width:336px;height:280px" data-ad-client="ca-pub-8129714482020982" data-ad-slot="4442193350"></ins> 
        <script>(adsbygoogle = window.adsbygoogle || []).push({});</script>
       </div>
       <span style="color: #800000;"><strong><span style="font-size: 10pt; line-height: 115%; font-family: 'Courier New';">Writer</span></strong></span> is the abstract class for writing character streams. It implements the following fundamental methods:
       <p></p>
       <ul>
        <li><span style="color: #800000;"><strong><span style="font-size: 10pt; line-height: 115%; font-family: 'Courier New';">write(int)</span></strong></span>: writes a single character.</li>
        <li><span style="color: #800000;"><strong><span style="font-size: 10pt; line-height: 115%; font-family: 'Courier New';">write(char[])</span></strong></span>: writes an array of characters.</li>
        <li><span style="color: #800000;"><strong><span style="font-size: 10pt; line-height: 115%; font-family: 'Courier New';">write(String)</span></strong></span>: writes a string.</li>
        <li><span style="color: #800000;"><strong><span style="font-size: 10pt; line-height: 115%; font-family: 'Courier New';">close()</span></strong></span>: closes the stream.</li>
       </ul> 
       <span style="color: #800000;"><strong><span style="font-size: 10pt; line-height: 115%; font-family: 'Courier New';">OutputStreamWriter</span></strong></span> is a bridge from byte streams to character streams. Characters are encoded into bytes using a specified charset. The charset can be default character encoding of the operating system, or can be specified explicitly when creating an 
       <span style="color: #800000;"><strong><span style="font-size: 10pt; line-height: 115%; font-family: 'Courier New';">OutputStreamWriter</span></strong></span>.
       <p></p> 
       <span style="color: #800000;"><strong><span style="font-size: 10pt; line-height: 115%; font-family: 'Courier New';">FileWriter</span></strong></span> is a convenient class for writing text files using the default character encoding of the operating system.
       <p></p> 
       <span style="color: #800000;"><strong><span style="font-size: 10pt; line-height: 115%; font-family: 'Courier New';">BufferedWriter</span></strong></span> writes text to a character stream with efficiency (characters, arrays and strings are buffered to avoid frequently writing to the underlying stream) and provides a convenient method for writing a line separator: 
       <span style="color: #800000;"><strong><span style="font-size: 10pt; line-height: 115%; font-family: 'Courier New';">newLine()</span></strong></span>.
       <p></p> The following diagram show relationship of these writer classes in the 
       <span style="color: #800000;"><strong><span style="font-size: 10pt; line-height: 115%; font-family: 'Courier New';">java.io</span></strong></span> package:
       <p></p> 
       <img style="display: block; margin-left: auto; margin-right: auto;" src="/images/articles/javase/fileio/text-files/Writer_Hierarchy.png" alt="Writer Hierarchy" width="240" height="116">
       <p></p> &nbsp;
       <p></p>
       <h1><strong><span style="font-size: 16pt; line-height: 115%;">3. Character Encoding and Charset</span></strong></h1> When constructing a reader or writer object, the default character encoding of the operating system is used (e.g. Cp1252 on Windows):
       <p></p>
       <pre class="brush:java">FileReader reader = new FileReader("MyFile.txt");
FileWriter writer = new FileWriter("YourFile.txt");</pre>So if we want to use a specific charset, use an 
       <strong><span style="font-size: 10pt; line-height: 115%; font-family: 'Courier New';">InputStreamReader</span></strong> or 
       <strong><span style="font-size: 10pt; line-height: 115%; font-family: 'Courier New';">OutputStreamWriter</span></strong> instead. For example:
       <p></p>
       <pre class="brush:java">InputStreamReader reader = new InputStreamReader(
					new FileInputStream("MyFile.txt"), "UTF-16");</pre>That creates a new reader with the Unicode character encoding UTF-16.
       <p></p> And the following statement constructs a writer with the UTF-8 encoding:
       <p></p>
       <pre class="brush:java">OutputStreamWriter writer = new OutputStreamWriter(
					new FileOutputStream("YourFile.txt"), "UTF-8");</pre>
       <span style="font-size: 11pt; line-height: 115%; font-family: 'Calibri','sans-serif';">In case we want to use a </span>
       <strong><span style="font-size: 10pt; line-height: 115%; font-family: 'Courier New';">BufferedReader</span></strong>
       <span style="font-size: 11pt; line-height: 115%; font-family: 'Calibri','sans-serif';">, just wrap the </span>
       <strong><span style="font-size: 10pt; line-height: 115%; font-family: 'Courier New';">InputStreamReader</span></strong>
       <span style="font-size: 11pt; line-height: 115%; font-family: 'Calibri','sans-serif';"> inside, for example:</span>
       <p></p>
       <pre class="brush:java">InputStreamReader reader = new InputStreamReader(
		new FileInputStream("MyFile.txt"), "UTF-16");

BufferedReader bufReader = new BufferedReader(reader);</pre>And for a 
       <strong><span style="font-size: 10pt; line-height: 115%; font-family: 'Courier New';">BufferedWriter</span></strong> example:
       <p></p>
       <pre class="brush:java">OutputStreamWriter writer = new OutputStreamWriter(
					new FileOutputStream("YourFile.txt"), "UTF-8");

BufferedWriter bufWriter = new BufferedWriter(writer);</pre>Now, let’s look at some complete examples.

       <h1><strong><span style="font-size: 16pt; line-height: 115%;">4. Reading from Text File Example</span></strong></h1> The following small program reads every single character from the file 
       <span style="font-size: 10pt; line-height: 115%; font-family: 'Courier New';">MyFile.txt</span> and prints all the characters to the output console:
       <p></p>
       <pre class="brush:java">package net.codejava.io;

import java.io.FileReader;
import java.io.IOException;

/**
 * This program demonstrates how to read characters from a text file.
 * @author www.codejava.net
 *
 */
public class TextFileReadingExample1 {

	public static void main(String[] args) {
		try {
			FileReader reader = new FileReader("MyFile.txt");
			int character;

			while ((character = reader.read()) != -1) {
				System.out.print((char) character);
			}
			reader.close();

		} catch (IOException e) {
			e.printStackTrace();
		}
	}

}</pre>The following example reads a text file with assumption that the encoding is UTF-16:
       <p></p>
       <pre class="brush:java">package net.codejava.io;

import java.io.FileInputStream;
import java.io.IOException;
import java.io.InputStreamReader;

/**
 * This program demonstrates how to read characters from a text file using
 * a specified charset.
 * @author www.codejava.net
 *
 */
public class TextFileReadingExample2 {

	public static void main(String[] args) {
		try {
			FileInputStream inputStream = new FileInputStream("MyFile.txt");
			InputStreamReader reader = new InputStreamReader(inputStream, "UTF-16");
			int character;

			while ((character = reader.read()) != -1) {
				System.out.print((char) character);
			}
			reader.close();

		} catch (IOException e) {
			e.printStackTrace();
		}
	}

}</pre>And the following example uses a 
       <strong><span style="font-size: 10pt; line-height: 115%; font-family: 'Courier New';">BufferedReader</span></strong> to read a text file line by line (this is the most efficient and preferred way):
       <p></p>
       <pre class="brush:java">package net.codejava.io;

import java.io.BufferedReader;
import java.io.FileReader;
import java.io.IOException;

/**
 * This program demonstrates how to read characters from a text file
 * using a BufferedReader for efficiency.
 * @author www.codejava.net
 *
 */
public class TextFileReadingExample3 {

	public static void main(String[] args) {
		try {
			FileReader reader = new FileReader("MyFile.txt");
			BufferedReader bufferedReader = new BufferedReader(reader);

			String line;

			while ((line = bufferedReader.readLine()) != null) {
				System.out.println(line);
			}
			reader.close();

		} catch (IOException e) {
			e.printStackTrace();
		}
	}

}</pre>&nbsp;

       <h1><strong><span style="font-size: 16pt; line-height: 115%;">5. Writing to Text File Example</span></strong></h1> In the following example, a 
       <strong><span style="font-size: 10pt; line-height: 115%; font-family: 'Courier New';">FileWriter</span></strong> is used to write two words “Hello World” and “Good Bye!” to a file named 
       <span style="font-size: 10pt; line-height: 115%; font-family: 'Courier New';">MyFile.txt</span>:
       <p></p>
       <pre class="brush:java">package net.codejava.io;

import java.io.FileWriter;
import java.io.IOException;

/**
 * This program demonstrates how to write characters to a text file.
 * @author www.codejava.net
 *
 */
public class TextFileWritingExample1 {

	public static void main(String[] args) {
		try {
			FileWriter writer = new FileWriter("MyFile.txt", true);
			writer.write("Hello World");
			writer.write("\r\n");	// write new line
			writer.write("Good Bye!");
			writer.close();
		} catch (IOException e) {
			e.printStackTrace();
		}

	}

}</pre>Note that, a writer uses default character encoding of the operating system by default. It also creates a new file if not exits, or overwrites the existing one. If you want to append text to an existing file, pass a boolean flag of true to constructor of the writer class:
       <p></p>
       <pre class="brush:java">FileWriter writer = new FileWriter("MyFile.txt", true);</pre>The following example uses a 
       <strong><span style="font-size: 10pt; line-height: 115%; font-family: 'Courier New';">BufferedReader</span></strong> that wraps a 
       <strong><span style="font-size: 10pt; line-height: 115%; font-family: 'Courier New';">FileReader</span></strong> to append text to an existing file:
       <p></p>
       <pre class="brush:java">package net.codejava.io;

import java.io.BufferedWriter;
import java.io.FileWriter;
import java.io.IOException;

/**
 * This program demonstrates how to write characters to a text file
 * using a BufferedReader for efficiency.
 * @author www.codejava.net
 *
 */
public class TextFileWritingExample2 {

	public static void main(String[] args) {
		try {
			FileWriter writer = new FileWriter("MyFile.txt", true);
			BufferedWriter bufferedWriter = new BufferedWriter(writer);

			bufferedWriter.write("Hello World");
			bufferedWriter.newLine();
			bufferedWriter.write("See You Again!");

			bufferedWriter.close();
		} catch (IOException e) {
			e.printStackTrace();
		}

	}

}</pre>This is the preferred way to write to text file because the 
       <strong><span style="font-size: 10pt; line-height: 115%; font-family: 'Courier New';">BufferedReader</span></strong> provides efficient way for writing character streams.
       <p></p> And the following example specifies specific character encoding (UTF-16) when writing to the file:
       <p></p>
       <pre class="brush:java">package net.codejava.io;

import java.io.BufferedWriter;
import java.io.FileOutputStream;
import java.io.IOException;
import java.io.OutputStreamWriter;

/**
 * This program demonstrates how to write characters to a text file using
 * a specified charset.
 * @author www.codejava.net
 *
 */
public class TextFileWritingExample3 {

	public static void main(String[] args) {
		try {
			FileOutputStream outputStream = new FileOutputStream("MyFile.txt");
			OutputStreamWriter outputStreamWriter = new OutputStreamWriter(outputStream, "UTF-16");
			BufferedWriter bufferedWriter = new BufferedWriter(outputStreamWriter);
			
			bufferedWriter.write("Xin chào");
			bufferedWriter.newLine();
			bufferedWriter.write("H?n g?p l?i!");
			
			bufferedWriter.close();
		} catch (IOException e) {
			e.printStackTrace();
		}
		
	}
}</pre>This program writes some Unicode string (Vietnamese) to the specified text file.

       <strong><span style="font-size: 14pt; line-height: 115%;">NOTE: </span></strong>From Java 7, you can use 
       <a href="/java-core/the-java-language/using-try-with-resources-examples-java-7" target="_blank">try-with-resources statement</a> to simplify the code of opening and closing the reader/writer. For example:
       <p></p>
       <pre class="brush:java">try (FileReader reader = new FileReader("MyFile.txt")) {
	int character;

	while ((character = reader.read()) != -1) {
		System.out.print((char) character);
	}
} catch (IOException e) {
	e.printStackTrace();
}</pre>&nbsp;
