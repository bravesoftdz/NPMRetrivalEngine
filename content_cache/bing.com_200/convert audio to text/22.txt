        <h1>How to write to file in Java – BufferedWriter</h1> 
        <div class="post-meta"> 
         <p class="post-meta">By <a href="http://www.mkyong.com/author/mkyong/" title="mkyong" rel="author">mkyong</a> | <time datetime="2010-06-02T10:53:28+00:00">June 2, 2010</time> | Updated : December 1, 2016 | Viewed : 1,263,224 times <span>+4,409 pv/w</span></p> 
        </div> 
        <!-- /.post-meta --> 
        <div class="post-content"> 
         <p>In Java, you can use <code>java.io.BufferedWriter</code> to write content to a file. </p> 
         <div class="note"> 
          <strong>Note</strong>
          <br> The 
          <code>BufferedWriter</code> is a character stream class to handle the character data. Unlike byte stream (convert data into bytes), you can just write the strings, arrays or character data directly to a file. 
         </div> 
         <h2>1. Classic BufferedWriter</h2> 
         <p>A classic <code>BufferedWriter</code> example to write content to a file, create the file if doesn’t exist, the existing content will be overridden.</p> 
         <div class="filename">
          WriteToFileExample1.java
         </div> 
         <pre><code class="language-java">
package com.mkyong;

import java.io.BufferedWriter;
import java.io.FileWriter;
import java.io.IOException;

public class WriteToFileExample1 {

	private static final String FILENAME = "E:\\test\\filename.txt";

	public static void main(String[] args) {

		BufferedWriter bw = null;
		FileWriter fw = null;

		try {

			String content = "This is the content to write into file\n";

			fw = new FileWriter(FILENAME);
			bw = new BufferedWriter(fw);
			bw.write(content);

			System.out.println("Done");

		} catch (IOException e) {

			e.printStackTrace();

		} finally {

			try {

				if (bw != null)
					bw.close();

				if (fw != null)
					fw.close();

			} catch (IOException ex) {

				ex.printStackTrace();

			}

		}

	}

}
</code></pre> 
         <p>Output : A new file is created with the following content :</p> 
         <div class="filename">
          E:\\test\\filename.txt
         </div> 
         <pre><code class="language-java">
This is the content to write into file
</code></pre>

         <h2>2. JDK7 Example</h2> 
         <p><code>try-with-resources</code> example to auto close the file writer.</p> 
         <div class="filename">
          WriteToFileExample2.java
         </div> 
         <pre><code class="language-java">
package com.mkyong;

import java.io.BufferedWriter;
import java.io.FileWriter;
import java.io.IOException;

public class WriteToFileExample2 {

	private static final String FILENAME = "E:\\test\\filename.txt";

	public static void main(String[] args) {

		try (BufferedWriter bw = new BufferedWriter(new FileWriter(FILENAME))) {

			String content = "This is the content to write into file\n";

			bw.write(content);
			
			// no need to close it.
			//bw.close();

			System.out.println("Done");

		} catch (IOException e) {

			e.printStackTrace();

		}

	}

}
</code></pre>

         <h2>3. BufferedWriter – Append</h2> 
         <p>To append content to the end of a file, pass a <code>true</code> to <code>FileWriter</code></p> 
         <pre><code class="language-java">
	// append to end of file
	FileWriter fw = new FileWriter(FILENAME, true);
	BufferedWriter bw = new BufferedWriter(fw);
</code></pre>
