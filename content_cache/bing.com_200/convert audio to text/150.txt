      <h2 class="blog-post-title"><a href="http://javadevnotes.com/java-integer-to-string-examples">Java Integer To String Examples</a></h2> 
      <p class="blog-post-meta"> Feb 15, 2015 <a href="http://javadevnotes.com/category/core-java">Core Java</a>, <a href="http://javadevnotes.com/category/examples">Examples</a>, <a href="http://javadevnotes.com/category/snippet">Snippet</a>, <a href="http://javadevnotes.com/category/core-java-string">String</a> 
       <fb:comments-count href="http://javadevnotes.com/java-integer-to-string-examples/"></fb:comments-count> comments </p> 
      <!-- post body --> This tutorial will show example codes on how to convert 
      <strong>Java Int to String</strong>. Performing conversion from int to String is a common scenario when programming with Core Java. 
      <!--more--> 
      <br> 
      <h5>Here are some quick reference example codes:</h5> 
      <ul> 
       <li><a href="#p1">Convert using Integer.toString(int)</a></li> 
       <li><a href="#p2">Convert using String.valueOf(int)</a></li> 
       <li><a href="#p3">Convert using new Integer(int).toString()</a></li> 
       <li><a href="#p3.1">Convert using String.format()</a></li> 
       <li><a href="#p3.5">Convert using DecimalFormat</a></li> 
       <li><a href="#p4">Convert using StringBuffer or StringBuilder</a></li> 
       <li><a href="#p5">Quick Solution</a></li> 
       <li><a href="#p6">Convert with special radix (not base 10 system)</a></li> 
      </ul> 
      <br> 
      <p></p>
      <h3><a name="p1">Convert using Integer.toString(int)</a></h3> 
      <p>The Integer class has a static method that returns a String object representing the specified int parameter. Using this is an efficient solution.</p> 
      <p></p> 
      <p><strong>Syntax</strong> <br> </p>
      <pre class="wp-code-highlight prettyprint">
public static String toString(int i)
</pre> 
      <p>The argument <i><b>i</b></i> is converted and returned as a string instance. If the number is negative, the sign will be preserved.</p> 
      <p></p> 
      <strong>Example</strong> 
      <br> 
      <pre class="wp-code-highlight prettyprint">
int number = -782;
String numberAsString = Integer.toString(number);
</pre> The code is equivalent to: 
      <br> 
      <pre class="wp-code-highlight prettyprint">
String numberAsString = "-782";
</pre> If you will try and search for solutions, this is one of the most popular ways of converting an int to String. 
      <br> 
      <br>
      <br> 
      <p></p>
      <h3><a name="p2">Convert using String.valueOf(int)</a></h3> 
      <p></p> 
      <p>The String class has several static methods that can convert most primitive types to their String representation. This includes integers.</p> 
      <p><strong>Example</strong> <br> </p>
      <pre class="wp-code-highlight prettyprint">
int number = -782;
String numberAsString = String.valueOf(number);
</pre> or 
      <br> 
      <pre class="wp-code-highlight prettyprint">
String numberAsString = String.valueOf(-782);
</pre> 
      <p></p> This is also an efficient solution like the first option above. And because this is simple and efficient, it is also a very popular method for converting an int to String. 
      <br> 
      <br>
      <br> 
      <p></p>
      <h3><a name="p3">Convert using new Integer(int).toString()</a></h3> 
      <p>Another alternative method is to create an instance of Integer class and then invoke it's toString() method. This is a little less efficient than the first two options shown above. This is because a new instance of Integer is created before conversion is performed.</p> 
      <p></p> 
      <p><strong>Example</strong> <br> </p>
      <pre class="wp-code-highlight prettyprint">
int number = -782;
Integer intInstance = new Integer(number);      
String numberAsString = intInstance.toString();
</pre> 
      <p></p> 
      <p>We can shortened to: <br> </p>
      <pre class="wp-code-highlight prettyprint">
int number = -782;
String numberAsString = new Integer(number).toString();
</pre> 
      <p></p> 
      <p>or just: <br> </p>
      <pre class="wp-code-highlight prettyprint">
String numberAsString = new Integer(-782).toString();
</pre> 
      <p></p> If your variable is of primitive type (int), it is better to use Integer.toString(int) or String.valueOf(int). But if your variable is already an instance of Integer (wrapper class of the primitive type int), it is better to just invoke it's toString() method as shown above. 
      <br> 
      <br>
      <br> 
      <p></p>
      <h3><a name="p3.1">Convert using String.format()</a></h3> 
      <p>String.format() is a new alternative that can be used for converting an Integer to String object. It is first introduced in Java 5 (JDK1.5) and has many cool features. </p> 
      <p></p> 
      <strong>Syntax</strong> 
      <br> 
      <pre class="wp-code-highlight prettyprint">
public static String format(String format, Object... args)
</pre> 
      <p>There are many options on how to use this method. But for conversion purposes, here is a simple example:</p> 
      <strong>Example</strong> 
      <br> 
      <pre class="wp-code-highlight prettyprint">
int number = -782;
String numberAsString = String.format ("%d", number);
</pre> And the variable 
      <i>numberAsString</i> will have the value 
      <i>"-782"</i> 
      <br> 
      <p>If you are using Java 5 or higher, this is also a very simple alternative for converting an int to String object.</p> 
      <br>
      <br> 
      <p></p>
      <h3><a name="p3.5">Convert using DecimalFormat</a></h3> 
      <p></p> 
      <p>The class <b>java.text.DecimalFormat</b> is a class that formats a number to a String representation following a certain pattern. <br> <strong>Example</strong> <br> </p>
      <pre class="wp-code-highlight prettyprint">
int number = 12345;
DecimalFormat decimalFormat = new DecimalFormat("#");
String numberAsString = decimalFormat.format(number);
System.out.println(numberAsString);
</pre> Will output 
      <br> 
      <pre class="wp-code-highlight prettyprint">
12345
</pre> 
      <p></p> 
      <p>This is an example that will convert to String but with commas <br> </p>
      <pre class="wp-code-highlight prettyprint">
int number = 12345;
DecimalFormat decimalFormat = new DecimalFormat("#,##0");
String numberAsString = decimalFormat.format(number);
System.out.println(numberAsString);
</pre> Will output 
      <br> 
      <pre class="wp-code-highlight prettyprint">
12,345
</pre> 
      <p></p> This is my favorite from all the options shown in this post because of the level of control that you can do with the formatting. You can specify the number of decimal places and comma separator for readability. 
      <br> 
      <br>
      <br> 
      <p></p>
      <h3><a name="p4">Convert using StringBuffer or StringBuilder</a></h3> 
      <p></p> StringBuffer is a class that is used to concatenate multiple values into a String. StringBuilder works similarly but not thread safe like StringBuffer. These two classes can be used to convert a Java Integer to String. 
      <br> 
      <p><strong>StringBuffer Example</strong> <br> </p>
      <pre class="wp-code-highlight prettyprint">
int number = -782;
StringBuffer sb = new StringBuffer();
sb.append(number);
String numberAsString = sb.toString();
</pre> 
      <p></p> 
      <p><strong>StringBuilder Example</strong> <br> </p>
      <pre class="wp-code-highlight prettyprint">
int number = -782;
StringBuilder sb = new StringBuilder();
sb.append(number);
String numberAsString = sb.toString();
</pre> 
      <p></p> 
      <p><strong>Shortened Examples</strong> <br> </p>
      <pre class="wp-code-highlight prettyprint">
String numberAsString = new StringBuffer().append(-782).toString();
</pre> 
      <p></p> 
      <pre class="wp-code-highlight prettyprint">
String numberAsString = new StringBuilder().append(-782).toString();
</pre> 
      <br>
      <br> 
      <p></p>
      <h3><a name="p5">Quick Solution</a></h3> If you want something simple that works but not elegant, you can use this code: 
      <br> 
      <pre class="wp-code-highlight prettyprint">
int number = -782;
String numberAsString = "" + number;
</pre> 
      <p></p> When you concatenate a String and an integer, the result is a String. Internally, the code is inefficient because there are intermediate objects that are created behind the scene. 
      <br> 
      <br>
      <br> 
      <p></p>
      <h3><a name="p6">Convert with special radix</a></h3> 
      <p></p> 
      <p>All of the examples above uses the base (radix) 10. There are cases when we wish to convert a Java Integer to String but using another base. There are convenient methods to convert to binary, octal, and hexadecimal system. Arbitrary custom number system is also supported.</p> 
      <p><strong>Examples</strong> <br> </p>
      <ul> 
       <li><u>Binary</u> <br> <pre class="wp-code-highlight prettyprint">
int number = 255;
String binaryString = Integer.toBinaryString(number);
System.out.println(binaryString);
</pre> The output is <br> <pre class="wp-code-highlight prettyprint">
11111111
</pre> <i>11111111</i> is the binary representation of the number 255. <br> </li> 
       <p></p> 
       <p></p>
       <li><u>Octal</u> <br> <pre class="wp-code-highlight prettyprint">
int number = 255;
String octalString = Integer.toOctalString(number);
System.out.println(octalString);
</pre> The output is <br> <pre class="wp-code-highlight prettyprint">
377
</pre> <i>377</i> is the octal representation of the number 255. <br> </li> 
       <p></p> 
       <p></p>
       <li><u>Hexadecimal</u> <br> <pre class="wp-code-highlight prettyprint">
int number = 255;
String hexString = Integer.toHexString(number);
System.out.println(hexString);
</pre> The output is <br> <pre class="wp-code-highlight prettyprint">
ff
</pre> <i>ff</i> is the hexadecimal representation of the number 255. <br> </li> 
       <p></p> 
       <p></p>
       <li><u>Custom Base/Radix</u> <br> We can use any other custom base/radix when converting an int to String. The sample shown below uses the base 7 number system. <br> <pre class="wp-code-highlight prettyprint">
int number = 255;
String customString = Integer.toString(number, 7);
System.out.println(customString);
</pre> The output is <br> <pre class="wp-code-highlight prettyprint">
513
</pre> <i>513</i> is the representation of the number 255 when written in the base 7 system. <br> </li> 
       <p></p> 
      </ul>
