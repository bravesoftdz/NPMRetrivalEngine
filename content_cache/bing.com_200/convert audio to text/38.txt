                  <br> But I doubt you're going to find an automated tool that goes from javascript to vbscript... there wouldn't be much (if any) call for anyone to spend the time to create one. &nbsp;As others have pointed out, Microsoft's scripting engines support jscript so there's almost zero incentive for someone to create a tool that ports javascript to vbscript. &nbsp;If someone has written one, they've either got way too much time on their hands or they did it as part of a school project. &nbsp;The other way (vbscript -&gt; javascript) makes some sense because you've got non-microsoft tools out there that support javascript but not vbscript.
                  <br> 
                  <br> Anyway... wish you luck on finding a tool. &nbsp;But, even if you do, personally don't think that studying whatever code it generates would be the best way to familiarize yourself with vbscript-- that's kind of like trying to learn a foreign language by reading pages translated by the babelfish.
