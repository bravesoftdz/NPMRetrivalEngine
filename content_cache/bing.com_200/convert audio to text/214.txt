           I have some text that i want to convert into an audio file like .wav.
           <br> 
           <br> So far, I can produce the speech from the text using FreeTTS and can create an audio file but couldn't put the converted audio out of text into the created file.
           <br> 
           <br> Code that converts text into speech using FreeTTS:
           <br> -------------------------------------------------
           <br> 
           <br> 
           <br> 
           <a href="http://www.coderanch.com/t/410859/java/java/string-stringbuffer-stringbuilder-performance" class="api" title="String vs. StringBuilder/StringBuffer" target="_new">String</a> voiceClassName = (args.length &gt; 0) ? args[1] : 
           <br> "com.sun.speech.freetts.en.us.CMUDiphoneVoice";
           <br> Class voiceClass = Class.forName(voiceClassName);
           <br> 
           <br> 
           <br> // instantiate the Voice
           <br> 
           <br> Voice helloVoice = (Voice) voiceClass.newInstance();
           <br> 
           <br> if (helloVoice instanceof MbrolaVoice) {
           <br> try {
           <br> (new MbrolaVoiceValidator((MbrolaVoice) helloVoice)).
           <br> validate();
           <br> } catch (
           <a href="http://docs.oracle.com/javase/8/docs/api/javax/xml/bind/ValidationException.html" class="api" title="Java API" target="_new">ValidationException</a> ve) {
           <br> System.err.println(ve.getMessage());
           <br> throw new 
           <a href="http://www.coderanch.com/how-to/java/illegalstateexception" class="api" title="CodeRanch how to" target="_new">IllegalStateException</a>
           <br> ("Problem starting MBROLA voice");
           <br> }
           <br> }
           <br> 
           <br> // sets the lexicon to CMU lexicon
           <br> 
           <br> helloVoice.setLexicon(new CMULexicon());
           <br> 
           <br> 
           <br> // sets the AudioPlayer to the 
           <a href="http://www.javaranch.com" class="api" title="JavaRanch" target="_new">Java</a> clip player
           <br> 
           <br> helloVoice.setAudioPlayer(new JavaClipAudioPlayer());
           <br> 
           <br> 
           <br> // loads the Voice, which mainly is loading the lexicon
           <br> //Voice.allocate();
           <br> //helloVoice.allocate();
           <br> //helloVoice.loader();
           <br> helloVoice.load();
           <br> 
           <br> // does the text-to-speech
           <br> System.out.println("before speak");
           <br> helloVoice.speak("She sells seashells by the seashore");
           <br> System.out.println("after speak");
           <br> 
           <br> 
           <br> 
           <br> 
           <br> Code for creating a file: 
           <br> ------------------------
           <br> 
           <br> 
           <br> 
           <a href="http://docs.oracle.com/javase/8/docs/api/javax/sound/sampled/AudioFormat.html" class="api" title="Java API" target="_new">AudioFormat</a> audioFormat;
           <br> TargetDataLine targetDataLine;
           <br> float sampleRate = 8000.0F;
           <br> int sampleSizeInBits = 16;
           <br> 
           <br> int channels = 1;
           <br> 
           <br> boolean signed = true;
           <br> boolean bigEndian = false;
           <br> audioFormat = new 
           <a href="http://docs.oracle.com/javase/8/docs/api/javax/sound/sampled/AudioFormat.html" class="api" title="Java API" target="_new">AudioFormat</a>(sampleRate,
           <br> sampleSizeInBits,
           <br> channels,
           <br> signed,
           <br> bigEndian);
           <br> 
           <br> DataLine.Info dataLineInfo =
           <br> new DataLine.Info(TargetDataLine.class,audioFormat);
           <br> 
           <br> if (AudioSystem.isLineSupported(dataLineInfo) ==
           <br> false) {
           <br> System.out.println("Line matching " + dataLineInfo +
           <br> " not supported.");
           <br> return;
           <br> }
           <br> 
           <br> targetDataLine = (TargetDataLine)AudioSystem.getLine(dataLineInfo);
           <br> 
           <br> AudioFileFormat.Type fileType = null;
           <br> File audioFile = null;
           <br> System.out.println("Creating file");
           <br> fileType = AudioFileFormat.Type.AU;
           <br> audioFile = new File("junk.au");
           <br> 
           <br> targetDataLine.open(audioFormat);
           <br> targetDataLine.start();
           <br> System.out.println("Writing into file"); 
           <br> AudioSystem.write(
           <br> new 
           <a href="http://docs.oracle.com/javase/8/docs/api/javax/sound/sampled/AudioInputStream.html" class="api" title="Java API" target="_new">AudioInputStream</a>(targetDataLine),
           <br> fileType,
           <br> audioFile);
           <br> System.out.println("Writing Complete"); 
           <br>

           <br> 
           <br> Please suggest..
