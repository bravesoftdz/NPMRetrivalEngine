        <p>Web Essentials extends Visual Studio with a lot of new features that web developers have been missing for many years.</p>
        <p>If you ever write CSS, HTML, JavaScript, TypeScript, CoffeeScript or LESS, then you will find many useful features that make your life as a developer easier.</p>

     <ul class="row-features clearfix list-unstyled">
      <li class="item-feature item-feature-01"><a href="/features/general"><span></span><h4>General</h4><p>General features of Web Essentials. A bunch of features are not specific to a specific language or editor.</p></a></li>
      <li class="item-feature item-feature-02"><a href="/features/browserlink"><span></span><h4>Browser Link</h4><p>Web Essentials ships with a range of Browser Link features that are all available from a menu that is injected onto your website when Browser Link is running.</p></a></li>
      <li class="item-feature item-feature-03"><a href="/features/stylesheets"><span></span><h4>Stylesheets</h4><p>Stylesheets are difficult to work with for many developers. With Web Essentials it has never been easier.</p></a></li>
      <li class="item-feature item-feature-04"><a href="/features/javascript"><span></span><h4>JavaScript</h4><p>Improve productivity and code quality for any JavaScript developer.</p></a></li>
      <li class="item-feature item-feature-05"><a href="/features/html"><span></span><h4>HTML</h4><p>Lots of useful shortcuts and productivity features that makes it fun to work with HTML.</p></a></li>
      <li class="item-feature item-feature-06"><a href="/features/typescript"><span></span><h4>TypeScript</h4><p>Provides additional features to the TypeScript editing experience. Writing TypeScript has never been easier using Web Essentials.</p></a></li>
      <li class="item-feature item-feature-07"><a href="/features/less"><span></span><h4>LESS</h4><p>Builds upon Visual Studio’s amazing LESS editor and provides compilation and other useful tools.</p></a></li>
      <li class="item-feature item-feature-08"><a href="/features/coffeescript"><span></span><h4>CoffeeScript</h4><p>Prefer Iced CoffeeScript? No problem. Web Essentials handles both flavors equally well.</p></a></li>
      <li class="item-feature item-feature-09"><a href="/features/markdown"><span></span><h4>Markdown</h4><p>Full immersive Markdown editing experience with previews, file generation and more.</p></a></li>
      <li class="item-feature item-feature-10"><a href="/features/custom-editors"><span></span><h4>Custom editors</h4><p>Who would have thought to add Intellisense and syntax highlighting for robots.txt files? Yep, we did.</p></a></li>
     </ul>
