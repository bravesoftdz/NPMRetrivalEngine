     <h1>Weka 3: Data Mining Software in Java</h1> 
    </div> 
    <div class="content"> 
     <p>Weka is a collection of machine learning algorithms for data mining tasks. The algorithms can either be applied directly to a dataset or called from your own Java code. Weka contains tools for data pre-processing, classification, regression, clustering, association rules, and visualization. It is also well-suited for developing new machine learning schemes. </p> 
     <p> Found only on the islands of New Zealand, the Weka is a flightless bird with an inquisitive nature. The name is pronounced like <a href="sounds/weka-say.au">this</a>, and the bird sounds like <a href="sounds/weka-long.au">this</a>. </p> 
     <p>Weka is open source software issued under the <a href="http://www.gnu.org/licenses/gpl.html" target="_blank">GNU General Public License</a>.</p> 
     <p> </p>
     <p>We have put together several free online courses that teach machine learning and data mining using Weka. Check out the <a href="http://weka.waikato.ac.nz" target="_blank">website for the courses</a> for details on when and how to enrol. The videos for the courses are available <a href="https://www.youtube.com/user/WekaMOOC" target="_blan">on Youtube.</a></p> 
     <p>Yes, it is possible to apply Weka to <a href="bigdata.html">big data</a>!</p>
