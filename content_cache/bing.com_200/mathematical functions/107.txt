The</span> <strong><em><span style="color: #2a00ff;">Math</span></em> </strong><span> class is used to operate the calculations. There is not necessary to import any package for the </span><strong> <em><span style="color: #2a00ff;">Math</span> </em></strong><span>class because this is already in</span> <strong><span style="color: #2a00ff;">java.lang </span></strong><span> package.</span></p> 
          <p><span>Any expressions can be operated through certain method calls. There are some functions have been used in the given example. All the functions have been explained below with example :</span></p> 
          <p><strong><em><span style="color: #2a00ff;">E</span></em></strong></p> 
          <p><span>This is E field of the Math class which returns you a default exponent value that is closer than any other to e, the base of the natural logarithms.&nbsp;</span></p> 
          <p><strong><em><span style="color: #2a00ff;">PI</span></em></strong></p> 
          <p><span>This is also a field of the <em>Method </em>class which returns you a default pi value, the ratio of the circumference of a circle to its diameter.</span></p> 
          <p><strong><em><span style="color: #2a00ff;">abs()</span></em></strong></p> 
          <p><span>This is the abs() function which returns you the absolute number.</span></p> 
          <p><strong><em><span style="color: #2a00ff;">ceil()</span></em></strong></p> 
          <p><span>This is the ceil() function which returns you the smallest value but greater than the argument.</span></p> 
          <p><strong><em><span style="color: #2a00ff;">exp()</span></em></strong></p> 
          <p><span>This is the exp() function which returns you the exponential value raised to the power of a double value.</span></p> 
          <p><strong><em><span style="color: #2a00ff;">floor()</span></em></strong></p> 
          <p><span>This is the floor() function which returns you the largest value but less than the argument.</span></p> 
          <p><strong><em><span style="color: #2a00ff;">IEEEremainder()</span></em></strong></p> 
          <p><span>This is the IEEEremainder() which returns you the remainder for the given dividend and divisor.</span></p> 
          <p><strong><em><span style="color: #0000ff;">max()</span></em></strong></p> 
          <p><span>This is the max() function which distinguishes the maximum value from the two given value.</span></p> 
          <p><strong><em><span style="color: #0000ff;">min()</span></em></strong></p> 
          <p><span>This is the min() function which distinguishes the minimum value from the two given value.</span></p> 
          <p><strong><em><span style="color: #0000ff;">pow()</span></em></strong></p> 
          <p><span>This is the pow() function which returns you the number raised to the power of a first given value by the another one.</span></p> 
          <p><strong><em><span style="color: #0000ff;">random()</span></em></strong></p> 
          <p><span>This is the random() function which returns you the random number. It is absolutely system generated.</span></p> 
          <p><strong><em><span style="color: #0000ff;">rint()</span></em></strong></p> 
          <p><span>This is the rint() function which returns you a value closest to the given value.</span></p> 
          <p><strong><em><span style="color: #0000ff;">round()</span></em></strong></p> 
          <p><span>This is the round() function which returns you a value that is in the rounded form.</span></p> 
          <p><strong><em><span style="color: #0000ff;">sqrt()</span></em></strong></p> 
          <p><span>This is the sqrt() function which returns you the square root of the specified value.</span></p>
