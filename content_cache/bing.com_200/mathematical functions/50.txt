   <h1>JStats</h1> 
   <h3>An open-source statistics package for Java</h3> 
   <p>Jstats provides classes for calculating statistical values like mean, median, variance, and standard deviation, as well various probability functions. </p>
   <h3>Warning! This package is incomplete, and almost certainly neither usable nor functional!</h3> 
   <p>This is a project a friend and I started working on a few years ago. We never finished it, and I have left it here only for historical purposes, mostly as a reminder to myself of some of my past projects. Feel free to read the documentation and even download and use the package if you like, but no promises whatsoever are made as to its functionality. There is a remote chance I will revisit this someday, but considering all the other Java statistics packages that are available now (see sidebar), it is unlikely. </p>
   <p><b>Authors</b>: David Edelstein and <a href="http://www.justinscheiber.net/">Justin Scheiber</a>. </p>
