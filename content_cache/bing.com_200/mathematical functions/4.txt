             <div class="post-text" itemprop="text"> 
              <p>The <a href="http://commons.apache.org/proper/commons-math/" rel="nofollow">Apache Commons Math</a> library contains many advanced mathematical functions and algorithms that you can use. It has a section, <a href="http://commons.apache.org/proper/commons-math/userguide/analysis.html" rel="nofollow">Numerical Analysis</a> which have many algorithms for common math problems, such as <strong>roots finding</strong>, <strong>derivatives</strong> and <strong>integrals</strong>.</p> 
              <p>Please refer to the hyperlinks for more information and documentation about the library.</p> 
              <hr> 
              <p>The workflow of computation of a derivatives of an expression y=f(x) is the following one. </p> 
              <p>First we configure an input parameter x of type DerivativeStructure so it will drive the function to compute all derivatives up to order 3 for example. </p> 
              <p>Then we compute y=f(x) normally by passing this parameter to the f function.</p> 
              <p>At the end, we extract from y the value and the derivatives we want. </p> 
              <p>As we have specified 3rd order when we built x, we can retrieve the derivatives up to 3rd order from y. </p> 
              <pre><code>int params = 1;
int order = 3;
double xRealValue = 2.5;
DerivativeStructure x = new DerivativeStructure(params, order, 0, xRealValue);
DerivativeStructure y = f(x);
</code></pre> 
              <p>You can then, get the derivatives:</p> 
              <pre><code>System.out.println("y    = " + y.getValue();
System.out.println("y'   = " + y.getPartialDerivative(1);
System.out.println("y''  = " + y.getPartialDerivative(2);
System.out.println("y''' = " + y.getPartialDerivative(3);
</code></pre>
