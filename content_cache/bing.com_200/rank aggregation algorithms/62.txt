  <h1>WEKA Packages </h1> 
  <p><b>IMPORTANT: make sure there are no old versions of Weka (&lt;3.7.2) in your CLASSPATH before starting Weka</b> </p>
  <h3>Installation of Packages</h3> A GUI package manager is available from the "Tools" menu of the GUIChooser
  <br>
  <br>
  <code>java -jar weka.jar</code>
  <p> For a command line package manager type:<br>
   <br<code>
    java weka.core.WekaPackageManager -h
    <br>
    <br> 
   </br<code></p>
  <hr> 
  <h3>Running packaged algorithms from the command line</h3>
  <code>java weka.Run [algorithm name]</code>
  <p>Substring matching is also supported. E.g. try:<br><br><code>java weka.Run Bayes</code></p>
