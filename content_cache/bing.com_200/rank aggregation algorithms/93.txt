  <big> <strong>GAlib<br>A C++ Library of Genetic Algorithm Components</strong> </big> 
  <br> 
  <small><i>the current release is version 2.4.7</i></small> 
  <p> GAlib contains a set of C++ genetic algorithm objects. The library includes tools for using genetic algorithms to do optimization in any C++ program using any representation and genetic operators. The documentation includes an extensive overview of how to implement a genetic algorithm as well as examples illustrating customizations to the GAlib classes. </p>
