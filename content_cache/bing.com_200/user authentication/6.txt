           <p style="text-align: justify;">In this article, we will see how to make a secured LDAP authentication&nbsp;using Java.</p> 
           <p style="text-align: justify;">First thing to do is importing the trust certificate to Java keystore. The default java keystore is named <strong><em>cacerts</em> </strong>and is located in<strong><em> $JAVA_HOME/jre/lib/security/cacerts</em></strong>.</p> 
           <blockquote> 
            <p style="text-align: justify;"><strong><em>$JAVA_HOME</em></strong> is where your JDK&nbsp;is installed (By default : C:/Program Files/Java/jdk1.7.0_79/).</p> 
           </blockquote> 
           <p style="text-align: justify;"><span class="" style="display:block;clear:both;height: 0px;padding-top: 20px;border-top-width:0px;border-bottom-width:0px;"></span> </p>

           <h3><span id="1-_register_your_ssl_certificate_into_java_keystore">1- Register your SSL certificate into Java keystore</span></h3> 
           <p style="text-align: justify;">Java offers the&nbsp;certificate management utility <a href="http://docs.oracle.com/javase/7/docs/technotes/tools/solaris/keytool.html" onclick="__gaTracker('send', 'event', 'outbound-article', 'http://docs.oracle.com/javase/7/docs/technotes/tools/solaris/keytool.html', 'keytool');" target="_blank">keytool</a> to handle certificates into your keystore. To import a certificate, you need to specify three arguments :</p> 
           <ul> 
            <li style="text-align: justify;"><strong>-keystore</strong> : <em>Absolute path to your keystore. (By default : C:\Program Files\Java\jdk1.7.0_79\jre\lib\security)</em></li> 
            <li style="text-align: justify;"><strong>-alias</strong> : <em>Give a name to your certificate&nbsp; The given name should not already exist in the keystore</em></li> 
            <li style="text-align: justify;"><strong>-file</strong> : <em>Absolute path to the certificate you want to import</em></li> 
           </ul>
