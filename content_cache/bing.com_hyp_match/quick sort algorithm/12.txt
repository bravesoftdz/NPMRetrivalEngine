      <h1 class="entrytitle" id="post-10392"> <a href="https://www.programcreek.com/2012/11/quicksort-array-in-java/" rel="bookmark"> Quicksort Array in Java </a> </h1> 
      <div class="entrymeta">
        &nbsp;  
      </div> 
      <div class="entrybody"> 
       <p>Quicksort is a divide and conquer algorithm. It first divides a large list into two smaller sub-lists and then recursively sort the two sub-lists. If we want to sort an array without any extra space, quicksort is a good option. On average, time complexity is O(n log(n)).</p> 
       <p>The basic step of sorting an array are as follows:</p> 
       <ul> 
        <li>Select a pivot, normally the middle one</li> 
        <li>From both ends, swap elements and make left elements &lt; pivot and all right &gt; pivot</li> 
        <li>Recursively sort left part and right part</li> 
       </ul> 
       <p><a href="http://me.dt.in.th/page/Quicksort/" target="_blank">Here</a> is a very good animation of quicksort.</p>

        <p class="theCode" style="display:none;">public class QuickSort { public static void main(String[] args) { int[] x = { 9, 2, 4, 7, 3, 7, 10 }; System.out.println(Arrays.toString(x)); int low = 0; int high = x.length - 1; quickSort(x, low, high); System.out.println(Arrays.toString(x)); } public static void quickSort(int[] arr, int low, int high) { if (arr == null || arr.length == 0) return; if (low &gt;= high) return; // pick the pivot int middle = low + (high - low) / 2; int pivot = arr[middle]; // make left &lt; pivot and right &gt; pivot int i = low, j = high; while (i &lt;= j) { while (arr[i] &lt; pivot) { i++; } while (arr[j] &gt; pivot) { j--; } if (i &lt;= j) { int temp = arr[i]; arr[i] = arr[j]; arr[j] = temp; i++; j--; } } // recursively sort two sub parts if (low &lt; j) quickSort(arr, low, j); if (high &gt; i) quickSort(arr, i, high); } }</p>
