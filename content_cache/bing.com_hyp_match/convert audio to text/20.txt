    <h1>Text to Speech using Java</h1> 
    <p class="post-date"> Posted On October 19, 2010 by Sneha Philipose filed under <a href="/topic/programming-laguage/">Programming</a> , <a href="/topic/java/">Java</a> </p> 
    <p class="post-date"></p>
    <p>&nbsp;</p> 
    <p>Text to speech conversions have fascinated developers for many years. Most modern languages support the technique. In this article Anand demonstrates Java powered text to speech conversion.</p> 
    <p>This tutorial teaches how to convert the text input into Speech using Java. This can be implemented in two ways. They are,<br> <br> 1) JSAPI 1.0(Java Speech Application Programming Interface)<br> 2) Free TTS(Free Text To Speech)<br> <br> <span style="color: #800000; font-size: medium;"><strong>JSAPI 1.0:-</strong><br> </span> JSAPI 1.0 is developed by Sun MicroSystems. Jsapi contains two core technologies, viz. Speech Synthesis and Speech Recognition.&nbsp;<br> The Speech synthesis is a speech engine that converts the text into speech.&nbsp;<br> The Speech Recognition is used for Speech to text conversion. More Details about the JSAPI have been given below.&nbsp;<br> <br> <strong><span style="color: #800000; font-size: medium;">FreeTTS:-</span></strong><br> FreeTTS is an open source package. It is entirely written by making use of Java programming language. This package can also be used to convert the text into speech.&nbsp;<br> <br> <strong>Necessity for the conversion of the Text into Speech:<br> </strong><br> Speech synthesis of text in a word processor is an aid to proof-reading. It is easier to detect grammatical and stylistic problems. In the TTS, if we save the file in audio format, the size of the file will be larger than that of text file. It may be more useful in mobile phones in which we can hear the message that we have received instead of reading the SMS.&nbsp;<br> <br> <strong><span style="color: #800000; font-size: medium;">Methods to convert Text into Speech</span></strong><br> <br> Structure analysis: Structure analysis processes the input text to determine where paragraphs, sentences and other structures start and end.<br> Text pre- processing: This expands any abbreviations or other forms of shorthand.<br> <br> e.g. St. Joseph becomes Saint Joseph.<br> <br> Text-to-phoneme conversion: It converts each word to phonemes which refers to a basic unit of sound in a language.<br> <br> Prosody analysis: It processes the sentence structure, words and phonemes to determine appropriate prosody for the sentence.<br> <br> Waveform production: Finally, the phonemes and prosody information are used to produce the audio waveform for each sentence.</p> 
    <p><br> Requirements for Text to Speech:<br> <br> The following jar files are very important<br> <br> cmu_us_kal.jar,<br> cmulex.jar,&nbsp;<br> en_us.jar;<br> freetts.jar;<br> cmulex.jar;&nbsp;<br> jsapi.jar;<br> <br> The above files are available in FreeTTS-1.2.1-bin. FreeTTS is a open source package. Download freetts-1.2.1-bin.zip from <a href="http://sourceforge.net/projects/freetts/" target="_blank">http://sourceforge.net/projects/freetts/</a>&nbsp;<br> <br> 1. Unzip the freeTTS binary package and check inside the \lib directory, that all the above jar files are available except jsapi.jar.&nbsp;<br> 2. The jsapi.exe file will be available.<br> 3. Run Jsapi.exe, and you will get jsapi.jar.&nbsp;<br> 4. Copy all the Jars (jsapi.jar, freetts.jar, cmu_time_awb.jar, cmu_us_kal.jar, etc.) to that working folder or C:\Program files\java\ jdk1.6.0_03\jre\lib\ext&nbsp;<br> <br> Important Classes in javax.speech package:<br> Import javax.speech.*; import javax.speech.synthesis.*;<br> <br> Speech package contains two important packages namely synthesis and recognizer.<br> <br> <strong><span style="color: #800000; font-size: medium;">Engine:</span></strong><br> The Engine interface is available inside the speech package. The Engine interface is the parent interface for all speech engines including Recognizer and Synthesizer. “Speech engine” is the generic term for a system designed to deal with either speech input or speech output.<br> import javax.speech.Engine;<br> <br> The basic processes for using a speech engine in an application are as follows.<br> <br> <strong>1.</strong> Identify the application’s functional requirements for an engine (e.g, language or dictation capability).<br> <strong>2.</strong> Locate and create an engine that meets those functional requirements.<br> <strong>3.</strong> Allocate the resources for the engine.<br> <strong>4</strong>. Set up the engine.<br> <strong>5.</strong> Begin operation to allocate the engine and resume it.<br> <strong>6.</strong> Use the engine. Deallocate the resources of the engine.<br> <strong><br> <span style="color: #800000; font-size: medium;">Central:</span></strong><br> The Central class is the initial access point to all speech input and output capabilities. Central provides the ability to locate, select and create speech recognizers and speech synthesizers<br> <br> <span style="color: #800000; font-size: medium;">SynthesizerModeDesc</span><br> <br> SynthesizerModeDesc extends the EngineModeDesc with properties that are specific to speech synthesizers. A SynthesizerModeDesc inherits engine name, mode name, locale and running properties from EngineModeDesc. SynthesizerModeDesc adds two properties:List of voices provided by the synthesizer Voice to be loaded when the synthesizer is started.<br> Synthesizer<br> The Synthesizer interface provides primary access to speech synthesis capabilities<br> <br> <strong>Voice:&nbsp;</strong><br> A description of one output voice of a speech synthesizer. Voice objects can be used in selection of synthesis engines (through the SynthesizerModeDesc). The current speaking voice of a Synthesizer can be changed during operation with the setVoice method of the SynthesizerProperties object.<br> <br> <strong>Create a simple program using jsapi speech synthesis.<br> </strong><br> Import javax.speech.*; import javax.speech.synthesis.*;<br> <strong><em><br> Step 1:</em></strong> We first import the following Packages.&nbsp;<br> <strong><em>Step 2 :</em></strong> Create a Synthesizer:-<br> <br> Synthesizer syn = Central.createSynthesizer(null);<br> <br> This method creates a default Systhesizer. This Synthesizer gets the default locale.<br> SynthesizerModeDesc desc=new SynthesizerModeDesc();desc.setLocale(new Locale("de", ""));desc.addVoice(new Voice(null, GENDER_FEMALE, AGE_DONT_CARE, null));Synthesizer synthesizer = Central.createSynthesizer(desc);<br> <br> The above coding is to select a particular Locale and Particular Voice.<br> <br> <strong><em>Step 3:</em></strong><br> The following code is to allocate and resume the systhesizer.<br> synthesizer.allocate(); synthesizer.resume();<br> <br> <strong><em>Step 4:</em></strong><br> <br> Voice[] voices = desc.getVoices();<br> Get the available voices:<br> <br> The get voices method return all the available voices from the selected Synthesizer.<br> synthesizer.getSynthesizerProperties(). setVoice(voice);<br> <br> <strong><em>Step 5:</em></strong><br> The setVoice() Method sets a particular Voice.<br> <br> <strong><em>Step 6:<br> </em></strong>Speak The text<br> synthesizer.speakPlainText(speaktext, null); synthesizer.waitEngineState(Synthesizer.QUEUE_EMPTY);<br> <br> The speakPlainTex(speaktext,null) method speaks the given text until queue is empty.<br> <br> <strong><em>Step 7:</em></strong><br> Deallocate the Synthesizer:<br> synthesizer.deallocate();<br> deallocate() method is deallocate the synthesizer.<br> <br> <strong><em>Demo Programs:</em></strong><br> <br> Two demo programs are given below. The first one uses the jsapi.jar and the second one uses freetts.jar.</p>
