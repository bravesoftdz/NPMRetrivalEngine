                    <h1 class="post-title entry-title" itemprop="name"> How to format Date in Java - SimpleDateFormat Example </h1> 
                    <div class="post-header"> 
                     <div style="float: right; margin:5px 6px 0px 5px; position:relative; z-index:10 "> 
                      <script async src="//pagead2.googlesyndication.com/pagead/js/adsbygoogle.js"></script> 
                      <ins class="adsbygoogle" data-ad-client="ca-pub-5808379456926196" data-ad-slot="7171924747" style="display:inline-block;width:336px;height:280px"></ins>

                     <div dir="ltr" style="text-align: left;" trbidi="on"> 
                      <div class="MsoNormal"> 
                       <span style="font-family: &quot;courier new&quot;; font-size: 9.0pt;">SimpleDateFormat</span>
                       <span style="font-family: &quot;arial&quot;; font-size: 9.0pt;"> in Java is used to format Date in Java. You can format date on any String format based upon various attribute available in </span>
                       <span style="font-family: &quot;courier new&quot;; font-size: 9.0pt;">SimpleDateFormat</span>
                       <span style="font-family: &quot;arial&quot;; font-size: 9.0pt;"> class e.g. </span>
                       <span style="font-family: &quot;courier new&quot;; font-size: 9.0pt;">mm, dd, YY</span>
                       <span style="font-family: &quot;arial&quot;; font-size: 9.0pt;"> etc. You can also put <a href="http://java67.blogspot.sg/2012/12/how-to-display-date-in-multiple-timezone-java.html">timezone information</a> in formatted Date using </span>
                       <span style="font-family: &quot;courier new&quot;; font-size: 9.0pt;">Z</span>
                       <span style="font-family: &quot;arial&quot;; font-size: 9.0pt;"> attribute of </span>
                       <span style="font-family: &quot;courier new&quot;; font-size: 9.0pt;">DateFormat</span>
                       <span style="font-family: &quot;arial&quot;; font-size: 9.0pt;"> class. </span>
                       <span style="font-family: &quot;courier new&quot;; font-size: 9.0pt;">SimpleDateFormat</span>
                       <span style="font-family: &quot;arial&quot;; font-size: 9.0pt;"> is sub class of </span>
                       <span style="font-family: &quot;courier new&quot;; font-size: 9.0pt;">DateFormat</span>
                       <span style="font-family: &quot;arial&quot;; font-size: 9.0pt;"> and provide </span>
                       <span style="font-family: &quot;courier new&quot;; font-size: 9.0pt;">format()</span>
                       <span style="font-family: &quot;arial&quot;; font-size: 9.0pt;"> and </span>
                       <span style="font-family: &quot;courier new&quot;; font-size: 9.0pt;">parse()</span>
                       <span style="font-family: &quot;arial&quot;; font-size: 9.0pt;"> method to <a href="http://javarevisited.blogspot.sg/2011/09/step-by-step-guide-to-convert-string-to.html">convert Date to and from String in Java</a>. Worth noting is that <a href="http://javarevisited.blogspot.sg/2012/03/simpledateformat-in-java-is-not-thread.html">SimpleDateFormat is not thread-safe</a> and should not be shared with others. Avoid using </span>
                       <span style="font-family: &quot;courier new&quot;; font-size: 9.0pt;">static</span>
                       <span style="font-family: &quot;arial&quot;; font-size: 9.0pt;"> </span>
                       <span style="font-family: &quot;courier new&quot;; font-size: 9.0pt;">SimpleDateFormat</span>
                       <span style="font-family: &quot;arial&quot;; font-size: 9.0pt;"> in Java classes. If you want to share </span>
                       <span style="font-family: &quot;courier new&quot;; font-size: 9.0pt;">SimpleDateFormat</span>
                       <span style="font-family: &quot;arial&quot;; font-size: 9.0pt;"> or want to make it thread-safe, you can use <a href="http://javarevisited.blogspot.sg/2012/05/how-to-use-threadlocal-in-java-benefits.html">ThreadLocal variable in Java</a> to avoid sharing </span>
                       <span style="font-family: &quot;courier new&quot;; font-size: 9.0pt;">SimpleDateFormat</span>
                       <span style="font-family: &quot;arial&quot;; font-size: 9.0pt;"> among multiple threads. </span>
                       <span style="font-family: &quot;courier new&quot;; font-size: 9.0pt;">parse()</span>
                       <span style="font-family: &quot;arial&quot;; font-size: 9.0pt;"> method of </span>
                       <span style="font-family: &quot;courier new&quot;; font-size: 9.0pt;">SimpleDateFormat</span>
                       <span style="font-family: &quot;arial&quot;; font-size: 9.0pt;"> throws </span>
                       <span style="font-family: &quot;courier new&quot;; font-size: 9.0pt;">ParseException</span>
                       <span style="font-family: &quot;arial&quot;; font-size: 9.0pt;"> if String input is not a valid date or can not be parsed into mentioned format.</span>
                      </div>

                      <h2> <u><span style="font-style: normal; font-weight: normal; mso-bidi-font-style: italic; mso-bidi-font-weight: bold;">How to format Date in Java</span></u></h2> 
                      <div class="MsoNormal"> 
                       <a href="http://3.bp.blogspot.com/-1lzFJzIgaHk/UF2Ci6kY5pI/AAAAAAAAAes/OYiM7r-DHzc/s1600/17.jpg" style="clear: left; float: left; margin-bottom: 1em; margin-right: 1em;"><img alt="SimpleDateFormat Example in Java - How to format date in Java" border="0" src="https://3.bp.blogspot.com/-1lzFJzIgaHk/UF2Ci6kY5pI/AAAAAAAAAes/OYiM7r-DHzc/s1600/17.jpg" title=""></a>
                       <span style="font-family: &quot;arial&quot;; font-size: 9.0pt;">In order to format dates using </span>
                       <span style="font-family: &quot;courier new&quot;; font-size: 9.0pt;">SimpleDateFormat</span>
                       <span style="font-family: &quot;arial&quot;; font-size: 9.0pt;">, we first needs to define a <a href="http://javarevisited.blogspot.sg/2012/08/how-to-format-string-in-java-printf.html">String date format</a> e.g. </span>
                       <span style="font-family: &quot;courier new&quot;; font-size: 9.0pt;">"dd-MM-yyyy"</span>
                       <span style="font-family: &quot;arial&quot;; font-size: 9.0pt;"> will print dates in that format e.g. </span>
                       <span style="font-family: &quot;courier new&quot;; font-size: 9.0pt;">01-11-2012</span>
                       <span style="font-family: &quot;arial&quot;; font-size: 9.0pt;">. You can defined format based upon identifiers supported by </span>
                       <span style="font-family: &quot;courier new&quot;; font-size: 9.0pt;">SimpleDateFormat</span>
                       <span style="font-family: &quot;arial&quot;; font-size: 9.0pt;"> class. e.g. </span>
