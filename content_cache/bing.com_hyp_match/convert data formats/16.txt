             <p>Summary:&nbsp;This tutorial demonstrates how to use the Java <em>SimpleDateFormat</em> class to convert a Java <code>Date</code> to a formatted <code>String</code>.</p> 
             <p>The Java&nbsp;<a href="https://docs.oracle.com/javase/8/docs/api/java/text/SimpleDateFormat.html">SimpleDateFormat</a> class provides lets you easily (a) convert between a Java <a href="https://docs.oracle.com/javase/8/docs/api/java/lang/String.html">String</a> to a <a href="https://docs.oracle.com/javase/8/docs/api/java/util/Date.html">Date</a>, or (b) perform the opposite conversion, from a <code>Date</code> to a <code>String</code>. In this article I'll at several different ways to convert a Java <code>Date</code> object to a nicely formatted <code>String</code>&nbsp;object, and in another article I'll look at a couple of nice ways to convert in the opposite direction.</p> 
             <a href="#top" class="back-to-top">Back to top</a> 
             <h2 id="a-first-java-simpledateformat-example">A first Java SimpleDateFormat example</h2> 
             <p>In the first example, I start by getting today's date as a <code>java.util.Date</code> object, and then create a <code>SimpleDateFormat</code> object to define the format I want for the date after it's converted to a String.</p> 
             <p>Here's the source code for a complete Java&nbsp;<code>SimpleDateFormat</code> example&nbsp;class that demonstrates this custom date formatting approach:</p> 
             <pre class="brush: java">
import java.util.Date;
import java.text.SimpleDateFormat;
import java.util.Calendar;

/**
 * SimpleDateFormat example: Convert from a Date to a formatted String
 *
 * Get today's date,
 * then convert it to a String, 
 * using the date format we specify.
 */
public class JavaSimpleDateFormatTest
{
  public static void main(String[] args)
  {
    // (1) get today's date
    Date today = Calendar.getInstance().getTime();

    // (2) create a date "formatter" (the date format we want)
    SimpleDateFormat formatter = new SimpleDateFormat("yyyy-MM-dd-hh.mm.ss");

    // (3) create a new String using the date format we want
    String folderName = formatter.format(today);
    
    // (4) this prints "Folder Name = 2009-09-06-08.23.23"
    System.out.println("Folder Name = " + folderName);
  }
}
</pre> 
             <p>As you can see from the comments in the code, the custom date format (or pattern) I specified looked like this:</p> 
             <pre>
yyyy-MM-dd-hh.mm.ss
</pre> 
             <p>and the resulting formatted date output looks like this:</p> 
             <pre>
2009-09-06-08.23.23
</pre> 
             <p>This Java source code was taken from an actual program, where I append this formatted date string to the end of a filename, so I can easily look at the filename and tell the date and time each file was created.</p> 
             <a href="#top" class="back-to-top">Back to top</a> 
             <h2 id="simpledateformat-many-other-custom-date-formats">SimpleDateFormat - many other custom date formats</h2> 
             <p>As you might guess, you can format a date in many, many different ways. Here are some of the most common custom date formats I've used:</p> 
             <pre>
yyyy-MM-dd                 results in    2009-09-06
yyyyMMdd                   results in    20090906
EEE MMM dd hh:mm:ss yyyy   results in    Sun Sep 06 08:32:51 2009
</pre> 
             <a href="#top" class="back-to-top">Back to top</a> 
             <h2 id="java-simpledateformat-easily-get-the-date-time-or-datetime">Java SimpleDateFormat: Easily get the date, time, or datetime</h2> 
             <p>On a related note, if you just need to quickly get the date or time in a nice format, the <code>SimpleDateFormat</code> class also offers several nice static convenience methods that make it easy to get (a) the date, (b) the time, or (c) both date and time in just a line or two of Java code.</p> 
             <p>Here’s the complete source code for another class that demonstrates these static methods of the&nbsp;Java <code>SimpleDateFormat</code>&nbsp;class:</p> 
             <pre class="brush: java">
import java.util.Calendar;
import java.util.Date;
import java.text.DateFormat;
import java.text.ParseException;
import java.text.SimpleDateFormat;

public class JavaSimpleDateFormatTest2
{
  public static void main(String[] args)
  {

    // prints "Sep 6, 2009"
    DateFormat dateInstance = SimpleDateFormat.getDateInstance();
    System.out.println(dateInstance.format(Calendar.getInstance().getTime()));

    // prints "9:03:20 PM"
    DateFormat timeInstance = SimpleDateFormat.getTimeInstance();
    System.out.println(timeInstance.format(Calendar.getInstance().getTime()));
  
    // prints "Sep 6, 2009 9:03:20 PM"
    DateFormat dateTimeInstance = SimpleDateFormat.getDateTimeInstance();
    System.out.println(dateTimeInstance.format(Calendar.getInstance().getTime()));

  }
}
</pre>

These options are all shown on <a href="http://java.sun.com/javase/6/docs/api/java/text/SimpleDateFormat.html">the SimpleDateFormat javadoc page</a>.</p>
