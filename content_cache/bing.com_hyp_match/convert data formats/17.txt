       <h1 id="title" name="title">Format Conversion Packages</h1> 
       <div class="ComparisonTitle"> 
        <img src="img/headers/FormatConversionPackages/header.gif" border="0" alt="Format Conversion Packages" title="Format Conversion Packages" width="398" height="34"> 
       </div> 
       <div class="redlinelayer"> 
        <div class="redline"> 
         <img src="/common/images/spacer.gif" width="1" height="1"> 
        </div> 
       </div> 
       <div class="ComparisonListing">
        (GraphicConverter, Switch, ReaConverter, Image Converter Plus, ImageMagick, ePrint, ...)
       </div> 
       <div class="ComparisonTextFirst" id="92954039">
        In its broad approach to computation, 
        <i>Mathematica</i> has come to support a remarkable range of formats of almost every type—making it a unique hub not only for analysis and visualization but also for general data transformation and format conversion.
       </div> 
       <div class="ComparisonText" id="130700100">
        While there are many packages that handle particular types of formats—such as sounds, images, data or documents—
        <i>Mathematica</i> is unique in immediately being able to systematically convert between all of these. With its unified symbolic architecture, 
        <i>Mathematica</i> can represent all forms of data in a single, extensible framework, so that with a single line of input you can import bioinformatics data and export it as sound, or import an XML document and export an image of its tree structure.
       </div> 
       <div class="ComparisonText" id="237945752">
        <i>Mathematica</i> provides state-of-the-art coverage of formats as well as integrated web and database connectivity, and unique extensibility through technologies such as symbolic XML. With its rich built-in interactive programming language, 
        <i>Mathematica</i> makes arbitrary data transformation and format conversion fully scriptable and customizable—providing a single environment that can manage the complete digital workflow, from initial data acquisition through to final publication or deployment.
       </div> 
       <div class="ComparisonHeader">
        Format Conversion Package Features Built into 
        <i>Mathematica </i>:
       </div> 
       <ul class="ComparisonBulletList"> 
        <li> <a href="http://reference.wolfram.com/mathematica/guide/GraphicsImportingAndExporting.html" class="bulletTextLink" target="_top">25+ vector and raster image formats </a><a href="http://reference.wolfram.com/mathematica/guide/GraphicsImportingAndExporting.html">»</a> </li> 
        <li> <a href="http://reference.wolfram.com/mathematica/guide/AudioFormats.html" class="bulletTextLink" target="_top">15+ sound and video formats, including Flash </a><a href="http://reference.wolfram.com/mathematica/guide/AudioFormats.html">»</a> </li> 
        <li> <a href="http://reference.wolfram.com/mathematica/guide/3DGeometryAndModelingFormats.html" class="bulletTextLink" target="_top">25+ 3D geometry and rendering formats </a><a href="http://reference.wolfram.com/mathematica/guide/3DGeometryAndModelingFormats.html">»</a> </li> 
        <li> <a href="http://reference.wolfram.com/mathematica/guide/TabularAndSpreadsheetFormats.html" class="bulletTextLink" target="_top">All standard tabular, spreadsheet and database file formats </a><a href="http://reference.wolfram.com/mathematica/guide/TabularAndSpreadsheetFormats.html">»</a> </li> 
        <li> <a href="http://reference.wolfram.com/mathematica/guide/ScientificAndMedicalDataFormats.html" class="bulletTextLink" target="_top">Broad range of numerical, scientific, medical and geospatial formats </a><a href="http://reference.wolfram.com/mathematica/guide/ScientificAndMedicalDataFormats.html">»</a> </li> 
        <li> <a href="http://reference.wolfram.com/mathematica/guide/DocumentFormats.html" class="bulletTextLink" target="_top">Full conversion and data extraction in standard document and web formats </a><a href="http://reference.wolfram.com/mathematica/guide/DocumentFormats.html">»</a> </li> 
        <li> <a href="http://reference.wolfram.com/mathematica/guide/XMLImportAndExport.html" class="bulletTextLink" target="_top">Broad support for XML and XML-based formats </a><a href="http://reference.wolfram.com/mathematica/guide/XMLImportAndExport.html">»</a> </li> 
        <li> <a href="http://reference.wolfram.com/mathematica/guide/CompressionAndArchiveFormats.html" class="bulletTextLink" target="_top">Transparent handling of compression and archiving formats and directories </a><a href="http://reference.wolfram.com/mathematica/guide/CompressionAndArchiveFormats.html">»</a> </li> 
        <li>Support for file, pipe and URL import</li> 
        <li> <a href="http://reference.wolfram.com/language/guide/ImportingAndExporting.html" class="bulletTextLink" target="_top">Metadata and element extraction for all formats </a><a href="http://reference.wolfram.com/language/guide/ImportingAndExporting.html">»</a> </li> 
        <li>Full scripting and automation</li> 
       </ul> 
       <div class="ComparisonHeader">
        Key Advantages of 
        <i>Mathematica </i>for Format Conversion:
       </div> 
       <ul class="ComparisonBulletList"> 
        <li>Full interconversion between different types of formats</li> 
        <li> <a href="http://reference.wolfram.com/language/tutorial/VisualizationAndGraphicsOverview.html" class="bulletTextLink" target="_top">Rich data transformation, visualization and analysis language </a><a href="http://reference.wolfram.com/language/tutorial/VisualizationAndGraphicsOverview.html">»</a> </li> 
        <li>Immediate interactive representations of all imported data</li> 
        <li>Full language for customization of format conversions</li> 
        <li>Unified symbolic representation for all forms of data</li> 
        <li>Immediate support for extracting specified elements from large files</li> 
        <li>Detailed programmatic control of quality level and other parameters</li> 
        <li> <a href="http://reference.wolfram.com/mathematica/guide/BinaryData.html" class="bulletTextLink" target="_top">Language primitives for handling all standard binary data types </a><a href="http://reference.wolfram.com/mathematica/guide/BinaryData.html">»</a> </li> 
        <li> <a href="http://reference.wolfram.com/mathematica/guide/XMLImportAndExport.html" class="bulletTextLink" target="_top">Integrated symbolic XML for import and export of arbitrary XML DTDs </a><a href="http://reference.wolfram.com/mathematica/guide/XMLImportAndExport.html">»</a> </li> 
        <li>Broad rendering and document-generation capabilities</li> 
        <li> <a href="https://www.wolfram.com/technology/guide/DocumentCenteredInterface/" class="bulletTextLink" target="_top">Uniform document-centered interface for saving programs and data </a><a href="https://www.wolfram.com/technology/guide/DocumentCenteredInterface/">»</a> </li> 
        <li> <a href="http://reference.wolfram.com/mathematica/guide/SystemsAndUtilityFormats.html" class="bulletTextLink" target="_top">Support for IT datamining (server logs, mailbox formats, databases, etc.) </a><a href="http://reference.wolfram.com/mathematica/guide/SystemsAndUtilityFormats.html">»</a> </li> 
        <li> <a href="http://reference.wolfram.com/mathematica/guide/DatabaseConnectivity.html" class="bulletTextLink" target="_top">Integrated connectivity to database systems (Oracle, Access, mySQL, ...) </a><a href="http://reference.wolfram.com/mathematica/guide/DatabaseConnectivity.html">»</a> </li> 
        <li> <a href="http://reference.wolfram.com/mathematica/guide/WebOperations.html" class="bulletTextLink" target="_top">Support for real-time import and export through web services, RSS feeds, etc. </a><a href="http://reference.wolfram.com/mathematica/guide/WebOperations.html">»</a> </li> 
        <li> <a href="http://reference.wolfram.com/mathematica/guide/WebFormats.html" class="bulletTextLink" target="_top">Structured import and content extraction from web pages, PDF, etc. </a><a href="http://reference.wolfram.com/mathematica/guide/WebFormats.html">»</a> </li> 
        <li> <a href="http://reference.wolfram.com/mathematica/guide/CreatingWebPages.html" class="bulletTextLink" target="_top">Immediate creation of full-function web pages, with image maps, etc. </a><a href="http://reference.wolfram.com/mathematica/guide/CreatingWebPages.html">»</a> </li> 
        <li> <a href="/mathematica/newin6/content/
LoadOnDemandCuratedData/" class="bulletTextLink" target="_top">Immediate access to broad data collections in finance, chemistry, geography, etc. </a><a href="/mathematica/newin6/content/
LoadOnDemandCuratedData/">»</a> </li> 
        <li> <a href="http://reference.wolfram.com/mathematica/guide/MathematicalTypesetting.html" class="bulletTextLink" target="_top">Integrated support for mathematical typesetting, formula and computation formats </a><a href="http://reference.wolfram.com/mathematica/guide/MathematicalTypesetting.html">»</a> </li> 
        <li>Extensive built-in test examples for all format types</li> 
        <li>Full <i>Mathematica</i> system for creating content in all format types</li> 
        <li> <a href="http://reference.wolfram.com/mathematica/guide/InternationalCharacterSets.html" class="bulletTextLink" target="_top">Systemwide Unicode support for content and filenames </a><a href="http://reference.wolfram.com/mathematica/guide/InternationalCharacterSets.html">»</a> </li> 
        <li> <a href="/mathematica/newin6/content/
LanguageForDataIntegration/" class="bulletTextLink" target="_top">Content-based file type recognition </a><a href="/mathematica/newin6/content/
LanguageForDataIntegration/">»</a> </li> 
        <li>Broad corporate relationships ensuring up-to-date format coverage</li> 
       </ul> 
       <div class="ComparisonHeader">
        Format Conversion Package Interoperability Features:
       </div> 
       <ul class="ComparisonBulletList"> 
        <li> <a href="http://reference.wolfram.com/mathematica/guide/ListingOfAllFormats.html" class="bulletTextLink" target="_top">File-level interoperability through hundreds of types of formats and subformats </a><a href="http://reference.wolfram.com/mathematica/guide/ListingOfAllFormats.html">»</a> </li> 
        <li> <a href="http://reference.wolfram.com/language/guide/WSTPAPI.html" class="bulletTextLink" target="_top">General interprocess communication through <i>MathLink</i> </a><a href="http://reference.wolfram.com/language/guide/WSTPAPI.html">»</a> </li> 
        <li> <a href="http://reference.wolfram.com/legacy/language/v11/guide/SystemsInterfacesAndDeploymentOverview.html" class="bulletTextLink" target="_top">Built-in APIs for Java, C/C++, .NET, Python, SQL, ... </a><a href="http://reference.wolfram.com/legacy/language/v11/guide/SystemsInterfacesAndDeploymentOverview.html">»</a> </li> 
       </ul> 
       <div class="ComparisonHeader">
        Interesting Tidbits:
       </div> 
       <ul class="ComparisonBulletList"> 
        <li>The number of possible cross-conversions between formats and subformats in <i>Mathematica</i> is at least in the hundreds of thousands</li> 
        <li>All format conversions for <i>MathWorld</i> and other Wolfram Research websites are done in <i>Mathematica</i> </li>
